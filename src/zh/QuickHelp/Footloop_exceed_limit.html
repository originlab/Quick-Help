4.29 FAQ-388 如果 for-loop 的长度超出限制应该怎么办？

Last Update: 7/8/2018

for-loop-exceed-limit

在LabTalk中，for 循环命令的脚本长度限制为 3072 个字符，其中包含空格。如果脚本太长，您可以使用 define 命令定义宏，然后在循环中调用宏。


例如: 

// Define three macros
define Part1 {ty one;}; 
define Part2 {ty two;}; 
define Part3 {ty three;}; 

// Call macros within for loop
int ii=0; 
for (ii=0;ii<3;ii++) 
{ 
Part1; 
Part2; 
Part3; 
}
