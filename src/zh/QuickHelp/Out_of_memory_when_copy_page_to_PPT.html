1.193 FAQ-893 当复制页面到 PowerPoint 时显示“内存不足”("Out of Memory")的错误提示时，怎么办呢？

OutOfMemory-CopyToPPT

Last Update: 7/24/2018

当你复制一个具有大量数据的图形，并作为 OLE 对象粘贴到 PowerPoint 时，您可能会收到这样的错误提示信息：
"没有足够的内存执行此操作。" （"There isn't enough memory to perform the operation."）

实际上，当你对一个包含大量数据的图形进行复制页面，并作为 Origin Graph Object 粘贴到 PowerPoint，生成的 EMF 文件(保存在 PPT 文件) 可能会大于 23M，就会出现这个错误。

但这个问题并非 Origin 引起，而是一个已证实在 PowerPoint 上存在的问题


若要避免此问题，你可以：

  • 使用菜单 Edit: Copy Graph as Picture 代替之前的操作，以图片形式复制和粘贴页面。
  • 使用菜单 Edit:Copy Page，但在 PowerPoint 中选择以图片形式粘贴页面。
