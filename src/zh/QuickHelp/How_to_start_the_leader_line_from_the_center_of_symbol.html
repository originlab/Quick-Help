1.186 FAQ-879 如何使符号中心作为指引线的起始点？

Start-LeaderLine-from-Symbol-Center

Last Update: 7/24/2018


默认情况下，连接符号和其标签的指引线是从符号边缘作为起始点的, 而且在指引线和符号边缘之间存在间隙。如果你想指引线的起始点是在符号的中心，你可以：

  1. 用相同的源数据在图上增加一个散点图，并设置新图作为图层的第一个绘图（通过Layer Content 对话框 ）。
  2. 设置新图中的符号大小为0 ( 通过 Plot Details 对话框 -> Symbol 选项卡)。
  3. 在新图上启用标签并且显示指引线 （通过 Plot Details 对话框 -> Label 选项卡)。
  4. 设置指引线和符号之间的间隙为0。在 Command window 中，运行 
    set %c -gtl 0;
注：

以上的 Labtalk 脚本只对当前绘图有效。由于两个绘图是由同一个源数据绘制的，如果新图不是图形中的第一个绘图，则无法选为当前数据图， 那么 Labtalk 脚本也无法设置新图指引线和符号之间的间隙。
