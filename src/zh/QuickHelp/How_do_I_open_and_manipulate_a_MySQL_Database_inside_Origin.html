2.10 FAQ-418 如何在Origin 中打开和管理 MySQL 数据？

MySQL-in-Origin

Last Update: 4/24/2019

Origin使用 ADO 接口来连接数据库，在 Origin 的官网上有如何从数据库中导入数据的教程。

Windows 有内置的 OLE DB Provider, 可以直接使用他们来连接像 Oracle, SQL Server 之类的数据库。 对于像 MySql 这样的数据库，如果你无法使用合适的 OLE DB 那么可以考虑利用 ADO 工具中的 ODBC 来连接这些数据库，方法如下：

  1. 首先在你希望连接的数据库所在的电脑安装 MySQL 数据库 ODBC 驱动 (从MySql 官网获取)。
  2. 接着设置 ODBC 连接， 打开 Windows 控制面板中的管理工具. (Vista 用户可能需要切换到经典视图（"Classic View"）才能看到该文件夹) 双击数据源图标，然后点击添加按钮，为数据源起一个容易记住的名字，并在测试数据连接并保存。
  3. 点击 Data: From Database: New 菜单或工具栏中的 Query Builder 或 SQL Editor 按钮打开 ADO 工具。 在 Query Builder 中, 点击 Query: Data Source: New 新建一个数据源. 在 Provider 选项卡下, 选择 Microsoft OLE DB Provider for ODBC Drivers 点击 Next。 选择先前保存的数据源然后输入密码并选择初始目录，点击 OK,这样将在 Query Builder 右侧出现数据表. (在 SQL Editor 中选择 File: New 来打开 Data Link Properties 对话框可以进行类似的操作。 )
