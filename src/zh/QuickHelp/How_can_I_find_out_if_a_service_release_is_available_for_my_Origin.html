6.20 FAQ-22 如何查看我的 Origin 版本是否有新的更新（service release）？

Check-Available-Patch

Last Update: 4/15/2023

在一个版本中，OriginLab 通常会发布一个或多个服务版本来修复在发布版本中发现的问题。 服务版本作为补丁提供。 通常还提供基于该服务版本的新安装程序。

重要：要运行补丁，您必须运行一次 Origin，例如在许可过程中或 Origin 获得许可之后。

获取服务版本的两种方式

  • 获取帮助文件/检查更新
如果您的 Origin 已注册，您可以运行帮助：获取帮助文件/检查更新来检查是否有适用于您的 Origin 版本和补丁。如果有补丁可用，它会显示在检查更新对话框中，您可以选择下载补丁。 在补丁下载过程结束时，系统会询问您是要关闭 Origin 以运行补丁，还是稍后运行补丁。 选择稍后会提供有关如何运行补丁的信息。
  • 从 OriginLab 网站下载补丁
OriginLab 还提供补丁，可从 OriginLab 网站的 Support区域 下载。 在检查补丁是否可用之前，运行 Origin 并选择帮助：关于 Origin。 记下关于 Origin对话框中的版本和完整内部版本号。 OriginLab 网站上的补丁下载页面提供了保存和应用补丁的说明。 还提供了一个链接来查看补丁发行说明。

注意：如果你有一个多用户包，想部署一个Service Release，请参考这个页面。
