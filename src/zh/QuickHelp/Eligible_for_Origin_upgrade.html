6.68 FAQ-905 我可以升级新版本吗？

Eligible-for-Origin-upgrade

Last Update: 4/14/2023

如果有新版本发布，用户会在选择帮助：获取帮助文件/检查更新菜单时收到通知。

单用户/多用户单机版

  • 如果您购买的是永久版的许可证 - 不管您买的 Origin 是哪一年发行的版本 - 请查阅Origin 各版本的发行日期和您所购买的金牌服务 Maintenance 的时间。如果在您的金牌服务有效期内有新的版本发行，那么您可以免费升级到该新版本。
  • 如果您购买的是 N 年期的许可证，请查看所购买的许可证的有效时间。只要许可证没有过期，您就可以免费升级到最新版本。

注: 您可以从 Origin 的菜单 Help: About Origin 查看许可证和金牌服务的过期时间。如果您发现 Help: About Origin 对话框中中显示的金牌服务时间不对，有可能是因为您续费了金牌服务之后没有及时获取新的许可证文件。

请浏览 这个页面 来了解如何把 Origin 自动升级为最新版本

网络版

  • 如果您购买的是永久版的许可证 - 不管您买的 Origin 是哪一年发行的版本 - 请查阅Origin 各版本的发行日期和您所购买的金牌服务 Maintenance 的时间。如果在您的金牌服务有效期内有新的版本发行，那么您可以免费升级到该新版本。
  • 如果您购买的是 N 年期的许可证，请查看所购买的许可证的有效时间。只要许可证没有过期，您就可以免费升级到最新版本。

注: 您可以从 Origin 的菜单 Help: About Origin 查看许可证和金牌服务的过期时间。如果您发现 Help: About Origin 对话框中中显示的金牌服务时间不对，有可能是因为您的管理员在续费了金牌服务之后没有及时更新服务器的许可证文件。

Origin 网络版管理员: 如何查看金牌服务的过期时间

在装了 FLEXnet 的服务器上找到许可证文件并用文本编辑器打开它。

  • 永久版许可证: 注意，列在 FEATURE 行的 Origin 版本号有可能并不是该许可证能够支持的最新的版本。重要的是以 VENDOR_STRING 开头的一串字符所包含的过期时间，例如，VENDOR_STRING=h0316F62A5R7200#MYYMMDD，在这串字符中的 YY = 2 位数字的年份， MM = 月，DD = 日。
  • N 年期许可证: 在 FEATURE 行所列出的许可证过期时间应该与 VENDOR_STRING 开头的一串字符所包含的过期时间一致，例如，VENDOR_STRING=h1D8DA52A5R1440#MyyMMdd，在这串字符中的 YY = 2 位数字的年份， MM = 月，DD = 日。

例: 如果您在2017年5月购买了 Origin 2017，并且购买的时候包括了1年的金牌服务。因为您的金牌服务有效期是到2018年的5月，当 Origin 2018 在2017年11月9日发行的时候，您就可以免费升级到 Origin 2018。

学习版

只要许可证没有过期，都可以在原电脑上直接安装最新版本使用。
