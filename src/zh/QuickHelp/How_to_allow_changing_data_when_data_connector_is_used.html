2.119 FAQ-1053 使用数据连接器时如何允许更改数据？

Change-Data-When-Connector-Used

Last Update: 4/20/2021

默认情况下，由数据连接器导入的数据将受保护：

  • 通过 Text/CSV 和 Excel 连接器导入的数据不可进行编辑，但可以排序。
  • 通过非 Text/CSV 和 Excel 连接器导入的数据既不可进行编辑，亦不可排序。


导入后如何编辑连接的数据：

  • 解锁导入的数据:数据保持解锁状态直到重新点击连接器图标并选择导入(此时数据会被覆盖，编辑保护也会恢复)。
  • 断连工作表: 删除工作表/矩阵和数据源之间的连接。同时删除编辑保护。但仍然保留连接器图标，以便您可以重新连接到数据源（此时将恢复编辑保护）。
  • 删除数据连接器: 永久删除当前工作簿中的所有工作表的数据连接器。编辑保护亦将从所有工作表中删除。此外，清除 保存时清除导入数据 复选标记。


永久删除连接器数据的限制：

需要在导入数据前改变系统变量@IPDC的值


通过非 Text/CSV 连接器导入的数据如需允许排序但禁止编辑，可如下操作：

  1. 选择菜单 设置：系统变量...。
  2. 在 设置系统变量 对话框中，分别在 变量 和 值 列键入 "IPDC" 和 "1" 。
  3. 然后通过数据连接器导入数据即可。


通过任意连接器导入的数据如需允许自由编辑和排序，正如通过 数据: 从文件导入 菜单:下的工具一样，可如下操作：

  1. 选择菜单 设置：系统变量...。
  2. 在 设置系统变量 对话框中，分别在 变量 和 值 列键入 "IPDC" 和 "0" 。
  3. 然后通过数据连接器导入数据即可。


导入后如何编辑连接的数据：

如果你只想删除当前部分的限制，点击 窗口: 脚本窗口, 输入并按确定: 

@IPDC=0
