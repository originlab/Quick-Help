1.65 FAQ-187 如何绘制热图？

create-heat-map

Last Update: 4/15/2023

Origin 支持将 XY 数据、XYZ 数据或矩阵数据绘制成热图，并具有 4 个内置热图图形模板：可从绘图>等高线图菜单打开的 热图, 带有标签的热图 和分条热图；还有从绘图>3D菜单打开的 3D 堆叠热图。
还有其他的一些热图模板，例如三角热图和带有显着标记的热图，可从工具：模板中心在线下载...


当您从 XY 列或 XYZ 列绘图时，Origin 会对 XY 数据进行二维分格以计算统计量，例如每个分格中的计数、平均值、最大值等，以此创建 Heatmap工作表。 热图就是基于这个新创建的 Heatmap工作表创建的。


调整轴刻度标签

  • 如果刻度标签太密集，例如如果分格太多，请双击轴并转到刻度选项卡。
  • 将主刻度类型设置为 按增量 或 按数量 并设置增量值或计数值。


在热图上添加标签

  • 单击热图并使用浮动工具栏按钮打开热图上的标签。
  • 打开绘图详细对话框， 转到标签选项卡以指定要显示为标签的内容。

添加地图作为背景 如果 XY 是经纬度，有几种方法可以在背景上添加地图。

  • 选择“插入”菜单查看是否有一些内置地图可用，例如美国大陆地图或世界地图等
  • 使用 Origin 的 Map Online APP插入地图作为背景。
  • 使用 Origin 的 [Google Map Import aAPP插入 google 地图作为背景。
  • 使用 Shapefile 连接器导入 shapefile 并绘制 shapefile 作为背景。
  • 要将图形设置为等距，请Alt+单击图形并单击等距浮动工具栏按钮。


在热图单元格之间添加间隙

  • 单击热图并打开热图之间的间隙并调整粗细。
  • 打开绘图细节，转到显示选项卡以调整 X 和 Y 方向的间隙。
  • 使用 绘图>等高线图: 分列热图 绘图时，会自动水平添加间隙。
