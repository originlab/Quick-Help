2.82 FAQ-852 是否可以对两个图像或者两个矩阵进行相减？

Subtract-One-Image-from-Another

Last Update: 7/15/2018

可以的。但是如果图像大小不同，操作就会更复杂一些。

将图像导入到矩阵中

  1. 点击基本工具栏中的 New Matrix 按钮 。
  2. 在 Windows 资源管理器中，按住 CTRL + 选择图像文件，并将图像拖放到矩阵窗口中。
  • 如果图像的尺寸相同, 则它们将作为单独的对象 导入到矩阵窗口中。
  • 如果图像具有不同的维度，则每个图都将被导入到矩阵窗口中的单独工作表中。


注意： 也可以将图像从任何外部窗口拖放到 Origin 中。将创建一个新的矩阵窗口。


在同一工作表中减去图像或矩阵
以下操作的前提是假设图像大小相同：

  1. 如果要将输出结果发送到同一工作表中的新矩阵对象，应首先通过单击图像按钮 来创建空对象，然后选择 Add。
  2. 从这个菜单中选择 Image: Arithmetic Transform: Simple Math: Open Dialog.
  3. 通过 Input Image1, Input Image2 和 Output Image 旁边的箭头按钮选择输入和输出的图像，将新建的空白矩阵对象设置为 Output Image。
  4. 点击 Math Function 旁边的下拉菜单，然后定义相关的操作算法 (例如， "Add", "img1 - img2", 等等)。
  5. 根据需要设置 channel, Z offsets 和 crop 等。
  6. 点击 OK。


减去不同工作表或工作簿中的图像或矩阵

  1. 激活含有两个或更多工作表的矩阵工作簿，然后选择Image: Arithmetic Transform: Simple Math: Open Dialog.
  2. 点击 Input Image1 和 Input Image2 旁的箭头按钮，然后使用 Image Browser (使用顶部的 List Images 下拉菜单来筛选) ，然后选择将 Output Image 输出的位置。
  3. 点击 Math Function 旁边的下拉菜单，然后定义相关的操作算法 (例如， "Add", "img1 - img2", 等等)。
  4. 根据需要设置 channel, Z offsets 和 crop 等。
  5. 点击 OK。

另外, 请参阅此博客文章：对齐和减去两个图像.
