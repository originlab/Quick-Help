6.8 FAQ-9 当我安装 Origin 的时候出现了一个错误信息，该怎么解决？

Install-Origin-Error-Message

Last Update: 7/4/2018

如果在安装 Origin 的时候，看到类似 "error installing ikernel.exe" 这样的错误信息，这通常是由于你的 Windows 登陆账号没有安装权限导致的。解决方法如下：

  1. 打开 Flexera 的网页。
  2. 点击 Knowledge Base。
  3. 从 Product Group 的下拉菜单中选择 Installation/Application Readiness Products。
  4. 从 Product 下拉菜单中选择 InstallShield。
  5. 在 Keyword(s) 文本框中输入 Q111519 并点击 Search。
  6. 点击搜索结果中的 Initialization Error Troubleshooting。该页面详细描述了造成这个错误信息的各种可能原因和解决方法。请依据实际情况选择合适的解决办法。
