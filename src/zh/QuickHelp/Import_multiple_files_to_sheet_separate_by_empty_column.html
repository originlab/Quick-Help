2.133 FAQ-1155 如何将多个文件导入工作表并用空列分隔每个文件？

import-multiple-files-to-sheet

Last Update: 4/21/2023

这个需求应该适用于任何 Origin 的 X-Function 导入例程（数据：从文件导入），只要对话框有一个脚本控件（impASC如图所示 ).

  1. 在数据：从文件导入下选择您的导入例程。
  2. 确保对于多文件导入，您为每个导入的文件指定了新建行。
  3. 展开脚本分支，在每个文件导入后运行脚本框中，输入以下内容： 
    @ISE =  1;
wks.ncols+ = 1;
  4. 在所有文件导入后运行脚本框中，输入以下内容： 
    @ISE = 0;
    （这将系统变量 @ISE 设置为默认值）。
  5. 根据需要设置其他导入选项后，单击对话框中的确定。 这将每个文件导入同一个工作表，每个文件的数据由一个空的工作表列分隔。
