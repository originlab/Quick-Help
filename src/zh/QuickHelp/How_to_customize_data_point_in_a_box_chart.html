1.254 FAQ-1047 如何显示和自定义箱线图的数据点?

data-point-box-chart

Last Update: 4/16/2020


显示箱线图的数据点
  1. 双击箱线图打开绘图细节对话框。
  2. 点击箱体选项卡，在类型选项列表里选择数据，箱体 + 重叠数据， 箱体（右）+ 数据（左）， 箱体（左） + 数据（右）， 条形 + 重叠数据，条形（右）+ 数据（左）， 条形（左）+ 数据（右）, 半箱体（右） + 数据（左） 或者 半箱体（左） + 数据（右）。

或者

  1. 点击箱线图显示迷你工具栏。
  2. 点击箱体类型按钮并选择有数据点的选项。


自定义箱线图的数据点
  1. 双击箱线图打开绘图细节对话框。
  2. 点击符号选项卡自定义数据点格式。


如果箱线图类型是数据, 除了上述方法，你也可以在迷你工具栏自定义数据点。

  1. 点击箱线图显示迷你工具栏。
  2. 在迷你工具栏里，使用边框颜色, 填充颜色, 图形符号, 增大数据点和减小数据点按钮自定义数据点格式.
