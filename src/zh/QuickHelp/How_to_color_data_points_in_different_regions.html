1.148 FAQ-771 怎样给不同区域的数据点设置颜色？

color-datapoints-regions

Last Update: 11/29/2018

方法 1

如果你有 Origin 2017 或更高版本，你将能很方便地使用 Data Highlighter App。要使用此插件，从Origin File Exchange 网站上下载并安装此插件到你的 Origin 里。如果你有 Origin 2018，即可按键盘上F10打开 App Center 对话框. 在对话框中你可以搜索并安装此插件。

注意: 要使用此插件，首先要绘制2D散点图，然后单击图形并在 Apps 窗口选择此插件。

方法 2

如果已经准备好一列跟数据点颜色对应的数据（就称为颜色列），并且此颜色列是位于原来用于绘图的工作表中的，可以用颜色列把相应颜色映射/索引到数据点上。(查看教程).

  1. 在图形上双击打开 绘图细节 对话框。
  2. 根据图形类型相应选择 线条 或 符号 或 图案 选项卡。
  3. 选择 颜色 下拉菜单并选择 按曲线。
    • 如果颜色列已有明确的数据类型如整数或文本，那么选择 索引 。在 颜色列表 选项卡 中允许对每个索引值指定颜色。
    • 如果颜色列中包含 RGB 类型颜色值，请选择 RGB 模式。
    • 选择 颜色映射 将以颜色标尺映射到颜色列中的值。

方法 3

如果不用颜色列来控制数据点的颜色，也可以在绘图之后为指定区域的数据点设置颜色。 可以使用快捷分析工具 快捷分析：集群分析... 来实现这个操作。首先在 集群分析 对话框中，选择包含所需数据点的目标区域。然后使用 创建类别 按钮 对当前目标区域包含的数据点创建类别并改颜色。你可以在 绘图细节 对话框中进一步改变颜色设置。更多关于 集群分析... 快捷分析工具的使用请参阅 帮助。

更多关于 集群分析... 快捷分析工具的使用请参见 查看教程。
