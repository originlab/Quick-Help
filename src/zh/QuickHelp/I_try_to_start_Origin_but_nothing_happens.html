6.14 FAQ-16 我运行 Origin 但是没反应！怎么解决？

Fail-to-Start-Origin

Last Update: 7/4/2018

这有可能是因为用文件夹（User Files folder）的设置有问题导致的。如果你把用户文件夹设在一个网络路径或者云端，请先把原来的用户文件夹删掉（若有要保留的文件，在删除文件夹之前务必备份），然后再打开 Origin。Origin 会弹出 User Files Folder 对话框让你重新设置用户文件夹。请选择一个本机上的路径作为用户文件夹。

如果在更改用户文件夹为本地路径之后仍不能打开 Origin，请联系 OriginLab 的技术支持部门解决。
