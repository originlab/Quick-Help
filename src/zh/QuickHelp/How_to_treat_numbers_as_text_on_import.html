2.132 FAQ-1154 如何导入源文件中看起来像数字但需要被视为类别的 Excel 或 ASCII 数据列？

treat-numbers-as-text-on-import

Last Update: 4/19/2023

有时，数据文件可能包含用作类别的数字代码（例如在地图或水质数据中）。在没有干预的情况下，此类数据将在导入时被视为数字，并且通常会删除前导零位。

最简单的解决方案是：

  1. 连接到文件并导入您的数据文件。
  2. 在导入工作表中，找到被错误地视为数字的列，然后右键单击列标题并选择属性。
  3. 在列属性中，设置格式 = 文本并单击 确定。
  4. 单击工作表左上角的绿色连接器图标并选择导入。 这将重新导入现有数据，这一次，您的数字代码应被视为文本（即不会修剪前导零，值将在工作表列中左对齐）。
