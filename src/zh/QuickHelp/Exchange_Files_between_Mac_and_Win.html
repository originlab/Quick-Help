5.36 FAQ-737 在 Mac 上运行 Origin 时，如何在 Mac 和 Windows 之间交换文件？

Exchange-Files-between-Mac-and-Win

Last Update: 7/23/2018

只需在 Mac 和 Windows 分区之间进行拖放文件即可！无论向任何一方进行文件拖放时，虚拟机都会为其创建副本。因此，您可以通过将文件从 Mac 文件夹拖放到 Origin 工作区来轻松地打开OPJ。同样，您可以将图形导出到 Windows 分区中的文件夹，然后将导出的文件拖放到 Mac 的应用程序来进行发布。

请注意，只有 Parallel 和 VM Fusion 支持拖放操作，而 Virtual box，Wine 和 crossover 并不支持。

Keywords:Mac, Windows, exchange files, 交换文件
