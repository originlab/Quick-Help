5.48 FAQ-831 为什么以管理员身份运行 Origin 时，拖拽文件进 Origin 软件无反应？

Why_does_not_drag_and_drop_work

Last Update: 7/15/2018

如果 以管理员身份运行 Origin 时，通过拖放的方式 导入数据文件 或者 安装 App 时，Origin 没有任何反应？

实际上，Windows 会阻止从低权限的进程向高权限的进程发送信息或数据以防止潜在的攻击。如果以管理员身份运行 Origin，则表明 Origin 具有比其它进程更高的权限，包括存放数据和 App 文件的 Windows 资源管理器。所以，如果在这种情况下尝试向 Origin 中拖放文件，则不起作用。

如果希望通过拖放轻松地导入数据或安装 App 程序，请不要以管理员身份运行 Origin。可以双击 Origin 程序图标或右键点击 Origin 程序图标并选择 Open 以启动 Origin。

注意： 请确保没有在属性中将 Run this program as an administrator 设为了默认值。右击 Origin 图标并选择 Properties，然后切换到 Compatibility 选项卡查看 Run this program as an administrator 的复选框是否选中，如果选中了，请去掉勾选。
