1.136 FAQ-732 如何将多 Y 轴图中的 Y 轴在确定的值处对齐？

align-axis-at-certain-value


使用 Origin 绘图菜单中画出的多 Y 轴图，第一个图层默认是父图层，其它图层则是子图层。双击子图层打开 绘图细节 对话框，切换到 关联坐标轴刻度 选项卡，在 关联Y轴 组中选择 对齐，并在 Y= 后的文本框中输入期望值。

Keywords:align, alignment, Axes start from same value, align axes, same axis start, axis beginning，对齐，坐标轴从相同值开始，对齐坐标轴，同一个坐标轴起点，坐标轴起始值
