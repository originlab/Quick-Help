3.147 FAQ-891 如何单独在每个分析工具中控制数字的小数位数？

Control-Digits-in-AnalysisTool

Last Update: 9/6/2018

在分析报告表中指定数字的小数位数

  1. 在报告表中，点击在节点标题旁的下箭头，或者右键单击在表格中的一个单元格。
  2. 选择在弹出菜单中 Digits...
  3. 在 Digits in Report 对话框中, 指定数字位数的设置。

你可以选择 Save as Default 复选框，以保存该数位设置作为此分析工具的默认设置。 往后此分析工具生成的所有报告表，将会使用此位数设置。

对数字的小数位数进行默认系统设定

如果想控制所有分析工具的数字位数设置，你可以在 Options 对话框的 Numeric Format 选项卡中， Digits in Report 下进行设置
