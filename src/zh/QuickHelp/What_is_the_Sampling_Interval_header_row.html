2.22 FAQ-434 列标签行中的采样间隔是什么？

SamplingInterval-Defination

Last Update: 4/24/2019

列标签行中的采样间隔行显示的是相关的 X 值的信息如 X 的起始值和步长（X 是均匀间隔的）。采样间隔行在导入 SPC, WAV 等文件时会自动生成。数值在工作表中的显示精度是由 Preference: Options对话框中的Numeric Format 选项卡决定的, 而在程序内部，这些值是全精度的并记录在 Workbook Organizer的元数据中。如果想了解详细数值， 在工作表窗口标题栏右击选择 Workbook Organizer，或在列右边灰色处右击选择选择 Show Organizer。

Origin 对数据的所有操作如绘图或分析都会优先使用采样间隔的值，即绘图时将使用此采样间隔的设定值而不是原始的 X 列值。如果需要对应采样间隔生成真实的 X 列，选择菜单 Column: Show X Column。

更多详情请参阅： 采样间隔。


Keywords:step, offset, precision，步骤，起始值，精度
