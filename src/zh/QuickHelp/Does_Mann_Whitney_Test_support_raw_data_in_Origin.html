3.74 FAQ-312 Origin 中的 Mann Whitney 检验是否支持原始数据？

Mann-Whitney-Test

Last Update: 7/15/2018

选择菜单 Statistics: Nonparametric Tests: Mann-Whitney Test（仅限专业版），打开对话框。

你可以从 Input Data Form 下拉列表中选择索引数据 Indexed data 或原始数据 Raw data，执行 Mann-Whitney 检验。


Keywords:Manny Whitney, two independent samples, nonparametric tests, 两个独立样本, 非参数检验
