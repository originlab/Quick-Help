1.133 FAQ-720 将堆积图的轴刻度类型从线性转换为 Log10 时, 这些图形看起来很奇怪。如何处理？

Keep-Proportion-Plot-Range

Last Update: 11/28/2018

有时，改变轴刻度类型可能会显著的改变图形窗口中的绘图显示范围，例如从线性改为 Log10 时。如果在一组堆积图进行了此类更改，则不同曲线之间的显示范围可能会不同。换言之，在修改了坐标轴类型后，绘图的比例发生了变化，所以可能会导致图像看起来很奇怪。为了正确地显示堆积图，可以进行以下操作：

  1. 双击图层内的空白区域以打开 绘图细节 对话框。
  2. 确保在对话框的左侧面板中已经选中 Layer，然后在右侧面板中切换到 堆叠 选项卡。
  3. 把 偏移 项设置为 自动，并勾选 保持绘图在非线性刻度下的比例 处的复选框。

Keywords:change axis scale, Log10, stack plots，更改坐标轴类型，堆积图
