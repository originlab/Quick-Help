5.12 FAQ-351 什么是对话框主题？

Dialog-Theme

Last Update: 8/7/2018


对话框主题提供了一种将对话框中的设置保存为外部磁盘文件以供将来使用的方法。您可以为对话框保存多个主题文件，从而在不同情景下自由地选择使用不同的对话框主题。这些主题文件将保存在用户文件夹的 Themes 子文件夹中。

在任意支持使用主题文件的对话框中，单击 Dialog Theme 右侧的三角形按钮，将会出现弹出菜单，您可以借此管理对话框主题。

对于某些可从主菜单直接执行操作的对话框，您保存的自定义主题文件可能会显示在主菜单的弹出项中。除非您按住 Shift 键，否则当您选择该主题文件时将不会打开对话框并立即执行相应的操作。当然，您也可以通过在对话框中的点击三角形按钮后的弹出菜单选择该主题。

您在上一次操作中使用的对话框设置将会被保存于 <last used> 主题中。当您打开某个对话框时，根据您在菜单中选择的不同的打开方式，对话框或许会加载默认的主题，或许会加载上一次使用过的主题，或许会加载某个已命名的主题。

支持使用主题文件的对话框同样具有将您的设置设为默认主题的选项。在对话框中点击 Dialog Theme 右侧的三角形按钮，在弹出菜单中选择择 Save as <Default> 即可将您的设置保存为新的默认主题。另外，您还可使用弹出菜单中的 System Default 选项将默认主题回滚到系统的默认值。
