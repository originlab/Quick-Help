6.29 FAQ-31 当我试图获取许可证文件的时候，网页上显示这样的错误信息：the license has been upgraded and that licenses for previous versions are no longer available。我应该怎么办？

License-Upgraded-Error-Message

Last Update: 7/6/2018

从 Origin 2018 开始，许可证支持跨版本使用，既是，同一台电脑上的不同版本的 Origin 将使用同一个许可证文件，所以不会再有种高低版本的限制。但是，对于 Origin 2017 或更早的版本，如果一个序列号已经升级到更高的版本并且用户已经注册和激活了高版本，低版本的许可证将无法再使用。如果你在获取许可证文件的时候看到这样的错误信息，请安装和使用新的版本。
