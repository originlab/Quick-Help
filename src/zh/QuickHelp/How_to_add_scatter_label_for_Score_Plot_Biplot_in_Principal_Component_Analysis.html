3.88 FAQ-326 如何在主成分分析中为分值图 / 双标图添加标签？

Score-Plot-Biplot-in-PCA

Last Update: 7/17/2018

从 OriginPro 2015 起，如果你在 PCA 对话框的 Input Data 分支下的 Observation Labels 选项中指定了某一列为标签列，则结果报告中的分值图 Score Plot 和双标图 Biplot 将自动使用该列作为数据点的标签。

或者，如果你使用的是 Origin 9.1 或者更早的版本，你可以将标签列放在原数据工作表的第一列并将其格式设置为文本，则在生成的结果工作表中，该列将自动放在绘图数据的旁边。你可以将其设置为得分值图 Score Plot 和双标图 Biplot 的标签，操作如下:

  1. 双击结果报表上的分值图/双标图形将其打开
  2. 双击图形，打开 Plot Details 对话框
  3. 选择 Label 选项卡，勾选 Enable，然后在 Label Form 下拉列表中选择你的标签列

另外，你还可以使用 data reader 更详细地查看每个数据点的信息，如这个教程所示。
