1.44 FAQ-161 图形的模板 (template) 和主题 (theme) 分别是什么？两者有什么区别？

GraphTemplate-and-GraphTheme

Last Update: 4/16/2021


graph template（图形模板） 是一个包括了图形所有规范的文件，具体的会包括图层数量、坐标轴设置和绘图类型。尽管模板文件可以含有例如箭头、矩形、自定义文本框等一系列图形对象，但是模板文件中没有任何数据。所有的 Origin 图形都是基于模板绘制的。

可以对任意模板进行自定义以备以后使用。选择 文件：保存模板为 ，在打开对话框的 模板名称那会显示此模板的名字。 修改名称以创建一个新的模板，或者继续使用此名称并覆盖已有模板。若要使用模板绘图，则可以选择绘图：我的模板 ，然后从菜单列表中选择所需的模板；或者点击 绘图：模板库 来浏览自定义模板。

从 Origin 2021b 开始，Origin 提供一些扩展的图模板来帮助绘画特定类型的图，您可以打开模板库页面查看更多细节内容。


graph theme（图形主题） 是以 XML 文件格式存储在磁盘上的图形属性的集合。图形主题可以随时应用于任何图形，从而改变各种对象属性，如图层大小、页面颜色、绘图符号类型和颜色等。选择菜单 设置：主题管理器 。使用主题管理器, 可以逐一应用多个主题到图形中。也可以选择将多个主题组合为一个来创建一个新的主题。如果把某个主题设为系统主题，则在此后的绘图中将会默认使用此主题中的样式。


关于 图形模板 和 图形主题 之间区别的一种理解是：模板是用来创建图形的，而主题是用来快速改变已有图形的外观属性的。此外，系统主题可以应用于全局而不需要更改不同图的模板设置并重新保存模板，比如设置刻度线向内。

如果您使用 Origin 2019 或更新的版本，Origin 提供了免费的Theme Preview App。这个 App 包括的主题有以下几方面：轴、箱线图、配色方案、等高线图等等，这些主题可以直接应用到您的图中。主题可以单独应用，也可以组合使用形成一个新的主题。另外，由 App 的名字可知，您可以在把主题应用到您的图之前预览应用主题所带来的改变。
