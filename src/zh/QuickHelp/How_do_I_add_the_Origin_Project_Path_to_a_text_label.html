1.35 FAQ-151 如何在图形中添加一个文本来显示此项目文件所在的文件夹？

Add-Project-Path-as-Label

Last Update: 11/13/2018


有时候可能会需要将 Origin 项目文件所在的路径（即在磁盘中的位置）以文本的形式显示在图形中，这样打印出此图形时便于快速获取相关的路径信息。

如要实现此功能，首先需要将项目文件保存到磁盘中（如果项目文件尚未保存的话）。

然后参考以下两种方法进行操作：

方法一

点击 工具 工具栏中的内置的 项目路径 按钮。如下动画所示：

方法二

  1. 在图形中想要放置显示路径信息文本框的位置右击，然后从弹出菜单中右击 添加文本... 。
  2. 在文本编辑框中输入：
    %X%G
    如还有需要，向文本中添加其他所需文字，然后鼠标在外部点击以退出编辑模式。
  3. 在此文本标签对象上右击，在弹出菜单中选择 属性 。
  4. 切换到 程序控制 选项卡，在 连接到变量(%, $)，替换层次为 下拉菜单中选择 1 并点击 确定 。
  5. 在 文本 选项卡中，将其中的文字内容修改为：
    \v(%X%G)
    然后点击确定以关闭对话框。
    注意：必须在 文本对象 对话框的 文本 选项卡中输入 \v() 。如果直接在文本编辑状态输入此信息是不起作用的。
