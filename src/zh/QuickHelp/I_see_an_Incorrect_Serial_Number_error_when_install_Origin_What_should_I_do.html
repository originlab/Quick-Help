6.85 FAQ-1102 我在安装 Origin 时看到 "Incorrect Serial Number" 的错误信息，该怎么办？

Incorrect-Serial-Number

Last Update: 5/26/2021

如果在安装 Origin 时看到 Incorrect Serial Number （序列号不正确）的错误消息，

|


请仔细检查输入的序列号是否与您从 OriginLab 收到的序列号完全一致。Origin 序列号的格式为 XXXXX-XXXX-XXXXXXX。

  • 请尽可能从 OriginLab 发送的邮件中复制序列号，然后使用 Ctrl+V 的方式把序列号粘贴到安装程序中。
  • 如果您必须手动输入序列号，请确保使用英文输入法来输入。特别要注意的是 "-" 这个字符，在中文的 “全角” 输入法中会输入为 “－” ，从而被软件视为错误的序列号。

Keywords:install, invalid SN, wrong SN, 安装, 错误序列号, 无效序列号, 全角, 半角
