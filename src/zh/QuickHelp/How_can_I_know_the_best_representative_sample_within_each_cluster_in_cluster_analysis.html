3.80 FAQ-318 在聚类分析中如何计算每个聚类中的最具代表性观测值？

best-representative-sample

Last Update: 7/24/2018

在 OriginPro 中，最具代表性的观测值可以在 Hierarchical Cluster Analysis（从菜单 Statistics: Multivariate Analysis: Hierarchical Cluster Analysis 打开）输出结果工作表Clustroid Info(聚类中心点信息)表中找到。

表格中会列出最具代表性观测值和最少代表观测值。

Keywords:hierarchical cluster, clustroid, representative, cluster analysis, 聚类，聚类中心，代表性，聚类分析

Minimum Origin Version Required: 2015 SR0
