4.9 FAQ-368 如何使用 LabTalk 修改图例？

Modify-Legend

Last Update: 7/31/2018


图例是一种普通的文本对象，名称为 legend。 此文本对象使用特殊语法来显示绘图符号和文本。 原始文本可能类似于： 

\l(1) %(1)
\l(2) %(2)

如果右键单击图例对象并选择 Properties，就会看到这个内容。

有关用户界面中语法和图例自定义的说明，请参考 Graph Legends 中图例自定义的讨论，并查看与图例相关的 FAQ 文本标签和图例部分。


使用 LabTalk 修改图形图例:

你可以通过修改名为 legend 对象的 text 属性，则可直接修改图例中的文本： 

legend.text$ = Test
\l(1) %(1)
\l(2) %(2);


你可以使用 legendupdate X-Function 修改 %(#) 表示的内容。 例如： 

legendupdate mode:=lname;  // display long name
legendupdate mode:=7 custom:=@WS;  // display sheet name, customized option


还有其他替换符号可用于图例中，X 和 Y 坐标轴标题中，以及文本对象中。 有关详细信息，请参考文本标签选项。
