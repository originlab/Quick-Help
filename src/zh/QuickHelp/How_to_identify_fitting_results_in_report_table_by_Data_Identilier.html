3.109 FAQ-647 如何通过数据集标识在拟合结果报告表中识别每一组数据的拟合结果？

Data-Identilier

Last Update: 7/22/2018

Origin 提供了下面列举的10种数据集标识 Data Identifier，用于指定在拟合结果报告表中用什么样的信息来表示输入数据。

  • 范围 Range
    使用输入数据的范围。
  • 工作簿名称 Book Name
    使用输入数据所在的工作簿的长名称。
  • 工作表名称 Sheet Name
    使用输入数据所在的工作表的短名称。
  • 工作表的显示名称 Sheet Display Name
    使用输入数据所在的工作表当前显示的名称。
  • 名称 Name
    如果 Y 数据集所在的列有长名称，使用长名称，否则使用该列的短名称。
  • 短名称 Short Name
    使用 Y 数据集所在列的短名称。
  • 长名称 Long Name
    使用 Y 数据集所在列的长名称。
  • 单位 Units
    使用 Y 数据集所在列的单位。
  • 注释 Comments
    使用 Y 数据集所在列的注释。
  • 自定义 <custom>
    自定义数据集标识。有关自定义的格式和语法，请参阅“图例文本自定义”文档。



在平铺结构表中显示标识 Show Identifier in Flat Sheet
除了生成可折叠的结果报告表之外，Origin 的许多分析工具还会将结果数据输出到平铺结构的表格中。在平铺结构的表格中使用数据集标识。
