2.127 FAQ-1110 从 Excel 粘贴链接如果 Origin 崩溃了怎么办？

Paste-Link-from-Excel-Make-Origin-Crash

Last Update: 5/10/2022

从 Excel 文件里复制数据，在 Origin 工作表里粘贴链接 （在单元格里点击右键，选择 粘贴链接），如果你发现 Origin 没反应，可能是因为你的 Excel 文件路径超过了微软的路径长度限制。

要避免这个问题，

请缩短 Excel 文件的完整路径。你可以

  • 把 Excel 文件复制到一个路径较短的文件夹，或者，
  • 用一个较短的名字重命名 Excel 文件与工作表。

关键是 Excel 文件路径、文件名以及工作表名总长度必须小于250。


注意

从 Excel 复制数据，直接粘贴到 Origin 工作表也可能会发生类似文件路径太长的问题。这超出了 Origin 的控制，因为 Excel 可能使用内部机制从文件获得数据，而不是把数据直接复制到剪切板。
