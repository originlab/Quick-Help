1.189 FAQ-885 如果光标一直处于缩放-平移模式中，应该怎么处理呢？

Escape-from-Zooming-and-Panning

Last Update: 4/24/2019


如果您发现光标一直处于缩放-平移模式(保持小手状态)中，并且无法通过单击 Tool 工具栏上的 Pointer 按钮 将光标转到常规模式，你可以尝试以下的解决方法：


  • 方法 1:
    按几次 X/Y/Z/A/Shift+A/Space 键, 看能够否回到常规模式。

有时，你可能无意识地按住在键盘上的键。 例如，按住A键，光标将会停留在平移模式中。


如果不是这种情况，或者你在尝试方法1后仍无法摆脱缩放-平移模式，你可以再尝试以下方法：


  • 方法 2:
  1. 选择 Preferences: System Variables... 打开 Set System Variables 对话框。
  2. 设置 SHS 为 0 和 STSM 为 2 以关闭智能提示（Smart Hint）。
  3. 重启 Origin。 查看问题是否已解决。

事实上，到目前为止，我们无法确认引起这个问题的实际原因，因此还没有明确的解决方法。 但是，在某些情况下, 它可能是与智能提示的冲突而引起的。


如果方法2也无法解决问题，可以再尝试以下这些方法:


  • 方法 3:
    重启计算机。 或者,
    最小化 Origin，使用其他应用程序（例如，使用 Excel 约1小时），然后返回 Origin 界面看问题是否仍存在。 或者,
    关闭并重启 Origin 数次，看问题是否存在。
