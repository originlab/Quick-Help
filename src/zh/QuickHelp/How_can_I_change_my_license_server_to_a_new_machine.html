6.36 FAQ-39 我想要更换许可证服务器的机器，应该怎么操作？

Change-Lic-Server-to-new-PC

Last Update: 4/14/2023

更改许可证服务器

如果您的许可证服务器不再运行或正在退役，并且您需要更改服务器主机，请提交系统转移申请表。 我们处理您的请求后，您将收到电子邮件通知。 然后您可以获取新的许可证文件并在新服务器上设置 FLEXnet 服务。 您可以查看有关 FLEXnet 设置 的更多信息进行了解。

如何提交系统转移申请表

  1. 使用您的网络会员帐户登录 OriginLab 网站
  2. 单击 Manage My License(s)。
  3. 勾选您要转移的序列号前面的复选框。
  4. 单击 Report a license server no longer in use 按钮。
  5. 填写表格并阅读 许可和安装支持政策
  6. 提交请求。

如果您的版本不再被支持，请联系 OriginLab 提交您的系统转移请求。

如果您不在美国/加拿大，请联系您当地的 经销商。

处理您的请求后，您可以继续获取新的许可证文件并设置新的服务器。

  • 请参阅 Windows 或 Linux 的 FLEXnet 设置说明。
