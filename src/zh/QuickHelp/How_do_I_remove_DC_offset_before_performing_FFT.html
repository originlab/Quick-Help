3.70 FAQ-307 在进行 FFT 变换之前如何校正直流偏移？

Remove-DC-Offset-Before-FFT

Last Update: 7/14/2018

在执行 FFT 变换之前，有 2 种方法可以从原始信号中消除直流偏移量（DC offset）:

  • 使用 FFT 高通滤波器
    1. 选中原信号列，然后选择菜单 Analysis: Signal Processing: FFT Filters。
    2. 在打开的对话框中，设 Filter Type 为 High Pass，设 Cutoff Frequency 为 0 并清除 Keep DC offset 复选框。
    3. 点击 OK，得到没有直流偏移量的信号。
  • 从原始信号中减去信号的平均值
    1. 假设原始信号存储在 B 列中。添加新列，比如 C 列。
    2. 选中 C 列, 从右键菜单中选择 Set Column Values，打开 Set Values 对话框。
    3. 在 Set Values 对话框下方的 Before Formula Scripts 面板中，输入: 
      stats col(b);
      然后在列公式文本框中，使用以下公式给 C 列赋值: 
      col(b)-stats.mean
    4. C 列就是没有直流偏移量的信号。


如果需要获得直流偏移量的值，请打开在第 1 种方法中提到的 FFT Filters 对话框，然后设 Filter Type 为 Low Pass，并将 Cutoff Frequency设为 0 即可。或者，在 Script Window 中使用 Mean 函数来计算信号的平均值： 

DCOffset = Mean(col(B));
