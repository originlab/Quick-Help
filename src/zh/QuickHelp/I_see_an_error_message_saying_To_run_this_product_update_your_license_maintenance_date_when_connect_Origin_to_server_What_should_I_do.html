6.95 FAQ-1122 当看到错误信息 "To run this product you need to update your license maintenance date"，该如何处理?

Update-Maint-Date-to-Run-This-Product

Last Update: 11/16/2023

当您启动Origin时看到如下错误信息：

License Status: 170

To run this product you need to update your license maintenance date


如果您是连接您的Origin到授权服务器：

请与您的Origin授权服务器管理员确认，客户端安装的Origin版本是否在您的维护计划支持当中。

  • 如果是，请检查服务器上的许可证文件在维护恢复后是否更新。
    • 如果不是，请按这个页面的指引来获得新的维护日期更新许可证文件。
    • 如果许可证文件已经更新，请按LMTools的 Start/Stop/Reread 选项卡上的 Reread License File 按钮。
  • 如果未更新，请卸载当前的 Origin 版本，然后安装支持的版本。


如果您是使用单机版本授权：

  • 请按照 这个页面的指引来更新您的许可证维护日期。
  • 或者，在 Origin 软件中，选择帮助：激活许可证并单击激活。 这将使用新的维护日期更新您的许可证。
