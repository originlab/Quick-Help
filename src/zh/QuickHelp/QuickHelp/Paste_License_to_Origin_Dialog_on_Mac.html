6.45 FAQ-735 当我在苹果机上安装激活 Origin 的时候，我没办法从网页上复制粘贴许可证文本到 Origin 的对话框里面，怎么办？

Paste-License-to-Origin-Dialog-on-Mac

Last Update: 7/6/2018

苹果机 Mac 上复制粘贴的快捷键 (cmd+c 和 cmd+v) 和 Windows 系统中的快捷键 (ctrl+c 和 ctrl+v) 是不同的。如果你在 Mac 系统这边的浏览器上使用快捷键 cmd+c 来复制许可证文本，那么来到 Windows 系统这边，你应该使用 ctrl+v 来粘贴文本到 Origin 对话框中。
