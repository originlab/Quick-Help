4.39 FAQ-400 如何获取 X-Function 的所有分类？

List-XF-Categories

Last Update: 7/13/2018

您可以使用 lc X-Function 来获取 X-Function 的所有分类，或者是在某一个分类中的所有 X-Function。在 Command Window（可通过选择菜单 Window: Command Window 或者按 Alt + 3 键打开窗口），输入 lc 然后按 Enter 键。这时所有的分类将会显示在 Command Window 中。这个列表同时也对应于帮助文档中的列表。您可以通过选择 Help: X-Functions 打开帮助文档。
