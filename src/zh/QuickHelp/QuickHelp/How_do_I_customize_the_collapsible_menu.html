5.11 FAQ-350 如何关闭可折叠菜单以显示完整的菜单？


Last Update: 4/24/2019

collapsible-menu

在 Origin 中通常主菜单和弹出菜单都是可折叠的。如果您想查看完整的菜单，可点击菜单底部的展开图标 。

Contents

  • 1 打开/关闭可折叠菜单功能
  • 2 重设所有菜单
  • 3 设置自动展开时间

打开/关闭可折叠菜单功能

  1. 在菜单中选择 Preference: Options。
  2. 在 Miscellaneous 标签页中取消勾选 Enable Collapsible Menus。
注:. 您也可以使用 Set System Variables 对话框来关闭该项功能。选择 Preferences: System Variables 打开对话框，设置 mpa 的值为 0。

重设所有菜单

当您第一次使用某个未被包含在折叠菜单中的菜单项时，那么它以后将会被添加到折叠菜单项中。随着时间的推移，菜单就会越来越长。如果您想重设所有菜单使其恢复初始的状态，可在 Script Window 执行 menu -cr 命令。

设置自动展开时间

在打开主菜单或者弹出菜单时，该菜单将会在若干时间后自动展开。您可以通过改变系统变量 @mpd 的值(单位 = 秒)，来改变折叠菜单的自动展开时间。
