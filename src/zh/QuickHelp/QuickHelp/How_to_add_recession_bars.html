1.174 FAQ-840 如何在2D 图中标注经济衰退时段？

Add-RecessionBars

Last Update: 11/12/2021

绘制经济数据的时候时，您可能希望在图上标出经济衰退时段，即阴影垂直条纹，这样可以界定衰退时期。Origin 提供两种方法在 2D 图中标注经济衰退时段。

使用线条标签页里的“填充曲线下的区域”

在您感兴趣的曲线窗口里，另一个“虚拟”XY的线图被添加进来。这个添加进来的图使用到的XY数据是这样的：X标注衰退时期的始末位置，Y则是类似于垂直刻度的最大值。

有两种方法布局这样的数据，且都结合 the Plot Details 线条 标签页 填充曲线下的区域 下拉列表：

  • 垂直贯穿填充 - 在缺失值处断开 -- 这种布局如下图的 C(X2) 和 D(Y2) 所示。这种而已依赖于缺失值 -- 注意空出来的单元格 -- 来断开衰退时期。
  • 成对垂直贯穿 -- 这种布局如下图的 E(X3) 和 F(Y3) 所示。与使用缺失值来区分衰退时期不同，这种布局只是成对的使用 X 列来分别标注每个衰退时期的始末位置。



如果要对不同的衰退时期设置不同的颜色，在绘图细节对话框的 图案 标签页里的 填充 框，点击 颜色 按钮，从而打开 颜色选择器，在这里点 按点 标签，然后设置 增量开始于 单选并选择起始颜色。
