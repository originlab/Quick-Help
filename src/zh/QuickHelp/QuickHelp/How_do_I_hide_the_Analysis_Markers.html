3.43 FAQ-275 怎样隐藏分析标记？

Hide-Analysis-Markers

Last Update: 11/6/2018

用数据选择器 选择部分图形进行分析，这将产生分析标记。

有以下四种方法来隐藏分析标记。

  1. 在当前图层（不要选择当前图层）右击在弹出的快捷菜单中选择 Analysis Markers 选项。
  2. 在分析标记上右击在弹出的快捷菜单中选择 Analysis Markers 选项。
  3. 在锁图标上左键单击在弹出的快捷菜单中选择 Analysis Markers 选项。
  4. 选择菜单 Data: Analysis Markers 选项。
