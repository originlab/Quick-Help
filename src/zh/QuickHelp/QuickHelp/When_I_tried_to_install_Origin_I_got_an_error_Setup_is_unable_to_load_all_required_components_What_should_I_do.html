6.98 FAQ-1126 安装Origin时，出现错误消息“Setup is unable to load all required components.”，该怎么办？

Setup-Unable-Load-Comp

Last Update: 5/8/2022

如果你安装Origin失败，并出现以下错误消息：

Setup is unable to load all required components.

它表明你的安装文件可能损坏、不完整或部分安装被本地安全组件阻止。请按照下列步骤排除故障：

  • 从 OriginLab 网站重新下载安装文件。运行前确保下载文件是完整的。
  • 使用完全的安全权限运行安装文件。请用管理员权限登录电脑，安装期间可以临时关闭杀毒软件，确保它不被杀毒软件阻止。
