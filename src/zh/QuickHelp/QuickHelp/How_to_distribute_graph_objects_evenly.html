1.188 FAQ-882 如何使图形对象等间距分布？

Distribute_Graph_Objects

Last Update: 8/14/2018

如果在图形中有多个对象，并且希望他们能在垂直方向或水平方向上均匀分布，你可以：

  1. 选择多个图形对象。
  2. 点击 Distribute Vertically 按钮 或 Distribute Horizontally 按钮.

Origin 会保持所选对象中顶部和底部（最右边和最左边）的对象的位置，然后垂直（水平）分布所选对象，使它们之间有相等的间距。


Keywords:graph object, distribute vertically, distribute horizontally，图形对象，垂直分布，水平分布

Minimum Origin Version Required: 2018 SR0
