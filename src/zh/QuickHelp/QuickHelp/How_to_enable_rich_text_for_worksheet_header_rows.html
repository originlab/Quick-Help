2.43 FAQ-477 如何在表和工作表中添加上下标和特殊字符？

RichText-for-WksHeader

Last Update: 10/23/2019

向表添加上下标和其他格式

要应用复杂的格式，包括向表对象添加上下标，必须开启 富文本模式.

有关更多信息，请参见 富文本.

向2018和更高版本的表和工作表单元格添加特殊字符

如果未应用任何格式，则无需启用富文本模式。 你可以通过输入点代码将Unicode字符直接添加到表或工作表标题或数据单元中。 或使用 Origin 的符号表以图形方式选择它们。

在单元格中编辑时：

  • 输入（Unicode）点代码的4个字符的十六进制代码 (比如 "λ" 是 03BB ), 然后按 ALT+X 插入点代码。
  • 或者，按 CTRL+M 打开字符表, 或右击后选择字符表。 浏览或输入（Unicode）点代码的4个字符的十六进制代码，然后单击插入。

相关问题 FAQ-149 如何用工作表的表头向图例和坐标轴标题中插入希腊字母和上下标？

注意: 如果要将项目保存为较旧的 opj 格式（例如与仅有 Origin2017 或更早版本的人共享），需要启用富文本模式并使用符号表对话框去插入特殊字符。 请参阅下一节。

向2017及更早期的版本的表和工作表单元格添加特殊字符

对于早于 Origin 2018 的版本，需要启用富文本模式，然后使用符号表 或者 转义序列在表和工作表单元格中输入特殊字符（方法取决于软件版本）。

有关信息，请参见 特殊字符，2017及更早版本.
