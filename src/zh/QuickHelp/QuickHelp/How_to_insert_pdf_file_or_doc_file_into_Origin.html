5.73 FAQ-1027 如何在Origin插入pdf文件或者doc文件?

Insert-Word-or-PDF

Last Update: 4/17/2024

  1. 点击工具栏的 新建布局 按钮 新建一个布局窗口。 你可以右击窗口名称栏并选择属性来重新命名。
  2. 从菜单中选择 插入：OLE 对象 。
  3. 选择 由文件创建 并点击 浏览 按钮选择文件。
  4. 点击 确定 把文件作为对象插入到布局窗口。
