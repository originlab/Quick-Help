5.54 FAQ-880 如何增加撤销操作的次数？

increase-number-undo

Last Update: 4/24/2019

从 Origin 2018 SR0 开始，在默认设置下 Origin 支持 100 次撤销操作 (Ctrl + Z)。(Origin 2018 之前默认支持 10 次)。

然而，如果这么多撤销次数依然不能满足要求，则可以使用系统变量 @UN 来增加撤销操作的次数。

  • 如果只需要在当前的 Origin 操作中将撤销操作的次数设为 200，则可以打开 Script Window，并输入以下代码，然后按下 Enter 键：
@UN=200
  • 如果想在未来的 Origin 操作中都能撤销 200 次，则选择 Preferences: System Variables 以打开 Set System Variables 对话框，并设置 UN 的值为 200。
请注意： 增加 Origin 的撤销操作次数可能会影响软件的运行性能。因此建议不要将软件的可撤销次数增加到 200 次以上。
