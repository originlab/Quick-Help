5.37 FAQ-738 如果 Origin 在 Mac 上的虚拟机中运行缓慢应该怎么办？

Virtual-Machine-is-Slow-on-Mac

Last Update: 7/24/2018

请考虑安装 Boot Camp 并在其上运行 Windows。它可以使您能以本机的配置所应有的速度运行 Windows，即 Windows 的应用程序可以充分利用 Mac 的硬件。

或者，您也可以考虑根据 Windows 操作系统 (Vista/7/8/10)所推荐的系统配置来调整虚拟机的配置。

此外，您的虚拟机软件供应商可能会提供改进虚拟机性能的选项。请参阅供应商的文档来调整能帮助提高虚拟机性能的设置。

注: 如果您在 Mac 的虚拟机中运行 Windows，并且发现 Origin 运行开始变得缓慢，请尝试完全关闭 Windows（不仅仅是暂停），然后重新启动 Windows 并检查 Origin 的性能是否有所提高。
