2.37 FAQ-471 怎样使得在工作表中输入的字符串类型数据不要自动转换成数值类型？

Avoid-Data-Numeric

Last Update: 7/13/2018


有时，当你在单元格输入 "001315472" 这样的字符串数据时， 你并不希望它会被识别为数值。要避免这样的情况，你可以打开 Column Properties 对话框，在 Properties 选项卡中勾选 Do not convert text to numeric during data entry 选项。

更多详情请参阅： 数据输入时不自动转换成数值而是保留文本形式
