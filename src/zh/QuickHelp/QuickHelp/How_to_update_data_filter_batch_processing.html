2.84 FAQ-863 如何在批处理操作中更新数据筛选器？

how-to-update-data-filter-batch-processing

Last Update: 7/15/2018

Origin 的批处理对话框允许用户使用现有的工作簿或已保存的 分析模板™ 来进行批处理分析。如果工作表或分析模板中有数据筛选器，则需要筛选器可以自动更新。

直到 Origin 2017 SR1 之前的 Origin 版本都必须在批处理工具 (File: Batch Processing) 的 Script Before Each Process 输入框中添加以下 LabTalk 脚本来在新数据中重新应用筛选器： 

wks.runfilter();

从 Origin 2017 SR1 开始，不再需要进行这样的操作。添加在分析模板™ 或工作表中的筛选器都会在批处理对话框中的数据更改时自动更新。
