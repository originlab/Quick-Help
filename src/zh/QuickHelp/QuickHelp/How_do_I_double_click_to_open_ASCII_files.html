2.8 FAQ-415 如何在Origin中双击打开ASCII文件？

double-click-open-ASCII-files

Last Update: 7/11/2018

将 Origin 设置为打开 ASCII 文件的默认程序后就可以在 Origin 中双击打开 ASCII 文件，方法如下：

  1. 在 Windows 资源管理器中，右键点击 ASCII 文件，在打开方式中选择默认程序(Open With: Choose default program).
  2. 在 Origin 程序文件夹中选择 Origin##.exe 并勾选始终使用选择的程序打开相同的文件 (Always use the selected program to open this kind of file)，然后点击 OK.


对于 Oirgin2017 及之前的版本，你需要通过批处理文件才能在 Origin 中双击打开 ASCII 文件

  1. 打开记事本
  2. 输入以下命令: 
start C:\Progra~1\OriginLab\<Program Folder>\<Version.exe> -r {impasc %1;};
注意: C:\Progra~1\OriginLab\<Program Folder>\ 是 Origin 的安装目录 ("C:\Progra~1" 即 "C:\Program Files").例如安装的是 Origin2016，则上面的命令为 "start C:\Progra~1\OriginLab\Origin2016\Origin93.exe -r {impasc %1;};"
  1. 点击文件：另存为(File : Save As)将文件类型（Save as type:）设为所有文件（All Files (.)）. 以.BAT为扩展名将文件重命名并保存(比如 imp2org.bat).
  2. 在 Windows 资源管理器中，右键点击 ASCII 文件，在打开方式中选择默认程序 (Open With: Choose default program).
  3. 选择第三步中保存的批处理文件， 勾选始终使用选择的程序打开相同的文件（Always use the selected program to open this kind of file）然后点击 OK..
