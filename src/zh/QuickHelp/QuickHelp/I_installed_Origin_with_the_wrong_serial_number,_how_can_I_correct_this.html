6.7 FAQ-8 我在安装 Origin 时填入了错误的序列号，应该怎么更正？

Modify-SN

Last Update: 6/22/2021

如果在安装 Origin 的时候填写了错误的序列号，或者从 Origin 的标准版升级为专业版，请参照以下步骤来修改你的序列号：

  1. 若 Origin(Pro) 扔在运行，先将它关闭。
  2. 在 Windows 7 或者 Vista 系统中，
    • 点击 Windows 的"开始"菜单并选择 All Programs | OriginLab Origin [version] | Repair or Uninstall Origin [version]。Origin 的安装向导将会打开。
    在 Windows 8 系统中，
    • 敲击一下键盘上的 Windows 键。
    • 右键单击桌面的任一空白处。
    • 点击右下角的 All Apps 按钮。
    • 选择 Repair or Uninstall Origin [version]，打开 Origin 的安装向导。
    在 Windows 10 系统中，
    • 点击 Windows 的"开始"菜单，向下滚动应用列表并选择 OriginLab Origin [version] | Repair or Uninstall Origin [version]。Origin 的安装向导将会打开。
  3. 选择 Modify/Change 选项并点击 Next。
  4. 在弹出的"是否更改序列号？"窗口中点击 Yes 确认。
  5. 填入正确的序列号并依指示完成安装。
