3.75 FAQ-313 Origin 是否支持多变量分析？

Multivariate-Analysis

Last Update: 7/15/2018

从 Origin 9.0 开始，Origin 支持 6 种多变量分析

Feature Way to Use
主成分分析 选择 Statistics:Multivariate Analysis: Principal Component Analysis
偏最小二乘 选择 Statistics:Multivariate Analysis: Partial Least Squares
K-均值聚类分析 选择 Statistics: Multivariate Analysis: K-Means Cluster Analysis
系统聚类分析 选择 Statistics: Multivariate Analysis: Hierarchical Cluster Analysis
判别分析 选择 Statistics: Multivariate Analysis: Discriminant Analysis
典型判别分析 选择 Statistics: Multivariate Analysis: Discriminant Analysis

Keywords:multivariate analysis, cluster, discriminant, principal component analysis, k-means, 多变量分析, 聚类, 判别, 主成分分析, k均值
