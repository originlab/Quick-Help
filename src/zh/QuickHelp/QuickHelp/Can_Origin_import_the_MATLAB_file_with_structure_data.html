2.24 FAQ-436 Origin 能导入 MATLAB 的结构性数据吗？

Import-MAT-with-Structure

Last Update: 7/9/2018

选择菜单 File: Import: Matlab(MAT)... 打开 impMatlab 对话框。

确保勾选 Import into Hierarchical Structure 复选框，但首先导入到工作表中的只是变量列表，可以单击在第 1 列中显示为链接型文本的各个变量名称，来将所需变量依次导入新工作表中。


Keywords:MAT, structure data, Hierarchical Structure, MATLAB,结构性数据,层次结构
