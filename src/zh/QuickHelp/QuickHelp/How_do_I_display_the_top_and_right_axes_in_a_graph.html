1.4 FAQ-114 如何显示图中顶端和右侧的坐标轴？ 如何在图中显示边框？

Last Update: 10/24/2019

Display-Top-Right-Axis

设置当前图形显示边框

每个图层均由四个坐标轴组成（底部x轴、顶端x轴、左侧y轴以及右侧y轴）。默认情况下的绘图仅显示底部和左侧的坐标轴，现给出几种不同的方法以显示出顶端和右侧的坐标轴。

  • 从 Origin 2017 Sr1版本开始，选择 设置：主题管理器。在 图形 选项卡下，选择 All Axes On 并单击 立即应用 即可打开所有的坐标轴显示。此时显示的顶端和右侧坐标轴均带有刻度。
  • 在 主题管理器中，预置了 Opposite Lines 主题，应用此主题可显示出不带有刻度的顶端和右侧坐标轴。
  • 对大部分的直角坐标系的 2D 图，也可以使用 查看：显示：框架 菜单来显示出坐标轴四周的边框。但是此边框的粗细和颜色均不可修改。从 Origin 2019b 开始， 边框的颜色和粗细会自动跟坐标轴一致
  • 双击任意一个坐标轴。选择 轴线和刻度线 选项卡。在左侧面板中同时选择 上轴 和 右轴 并勾选 显示轴线和刻度线 选框。
  • 双击任意一个坐标轴。选择 网格 选项卡。在左侧面板中同时选择 垂直 和 水平 并勾选 对面 。

保存设置使得以后的绘图中默认显示边框

  • 从 Origin 2017版本开始，软件预置了一套名为 All Axes On的主题。选择 设置：主题管理器 菜单，在图形 选项卡下找到并右击All Axe on主题，在弹出的菜单中选择 Set as System Theme 。

    也可以按住 CTRL 键同时选中多个主题，然后右击选择合并 将多个主题组合为一个新的主题。对新主题进行适当的命名，并右击将其设置为系统默认的绘图主题。

  • 在旧版本的Origin中，预置的显示所有坐标轴的主题名称是Opposite Lines。同样可以将其设置为默认主题，但是顶端和右侧的坐标轴没有刻度线。

    或者可以根据需要自定义顶端和右侧的坐标和刻度线并设为默认主题。比如，先绘制一个简单的图，设置顶端和右侧坐标轴为自己满意显示状态；再单击图层以选中图层边框，右击边框并选择保存格式为主题。在主题管理中，将其设置为默认的绘图主题。然而，以这种方式创建的主题中包含了大量的与坐标轴无关的其他属性设置，对于不熟悉主题自定义的初学者，建议慎用这种方法。
