1.112 FAQ-659 如何在坐标轴自定义刻度线和刻度线标签？

Last Update: 11/20/2018

Custom-Tick-Marks-and-Labels

通常情况下，Origin 坐标轴上刻度线和刻度线标签是等间距的。如果需要，按照下面的步骤可以在坐标轴指定的位置上生成刻度线和刻度线标签。

  1. 双击坐标轴或点击 格式：轴：X轴 打开轴对话框。
  2. 点击 刻度 选项卡，将主刻度线（主刻度）的 类型 改为 按自定义位置 (旧版本中为 By Dataset )
  3. 在 位置 （旧版本为 Dataset box ）中，输入空格分割的数值或从下拉菜单中选择要作为刻度线标签的列。
  4. 双击 应用 完成自定义刻度线。
  5. 如要自定义刻度线标签，选择 刻度线标签 选项卡。
  6. 选择 显示 子选项卡，将 类型 改为 刻度索引字符串, 输入空格分隔的标签值。如果这些标签是工作表中的某一列，也可以将 类型 改为 来自数据集的文本 然后选择相应的列作为刻度线标签。
  7. 点击 确定 完成修改并关闭对话框。
