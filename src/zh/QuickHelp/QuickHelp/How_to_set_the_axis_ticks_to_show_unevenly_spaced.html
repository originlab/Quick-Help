1.218 FAQ-963 如何设置使得轴刻度非均匀分布?

Unevenly-spaced-ticks


Last Update: 9/13/2018

在图上双击你想设置的坐标轴打开 Axis 对话框，在 Scale 选项卡， 把 Major ticks 和 Minor Ticks 的 Type 设为 By Custom Positions， 然后选择一列， 把该列上的数值作为轴刻度 又或者输入以空格分隔的数值作为轴刻度.

此外， 我们也可以不用真正的刻度位置作为刻度标签， 而是另外设置某个数列作为刻度标签

Keywords:uneven, tick position, customize ticks，非均匀， 刻度位置， 自定义刻度
