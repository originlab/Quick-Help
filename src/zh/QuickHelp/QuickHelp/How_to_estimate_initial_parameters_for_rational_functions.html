3.39 FAQ-268 怎样为有理函数的参数估计初始值？

Estimate-Initial-Parameters-for-Rational

Last Update: 7/21/2018

Origin 提供的内置 Origin C 函数 ocmath_rational_params_init 可以估计有理函数参数的初始值。在 Rational 类别中的内置函数 rational5 显示了怎么样用这个函数来估计函数的初始值，具体代码在 Function Organizer 的 Parameter Initialization 文本框中可以看到。

Keywords:fit, rational, estimate, initial, parameter，拟合，有理数，估计，初始值，参数

Minimum Origin Version Required: 9.1SR0
