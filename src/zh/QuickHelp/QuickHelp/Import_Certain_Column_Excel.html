2.25 FAQ-437 怎样将不同 Excel 表的特定列导入到 Origin 中？

Import-Certain-Column-Excel

Last Update: 4/24/2019

如果想将多个 Excel 表的同一列（如第二列）导入到单个 Origin 工作表中，可以在导入对话框中的 Multi-File Import Mode 和 Partial Import 部分来进行设置达到需求

需注意，对于 Origin 2018版本， Partial Import 部分已经更新，对于导入哪一列或哪一行可以有更详细的设置。详情请参阅 Importing From Excel。

  1. 打开菜单 Data: Import from File: EXCEL(XLS, XLSX, XLSM)... 选择 multiple Excel files 来打开 impMSExcel 对话框 (Origin 2015 和 Origin 2016版本打开的是 impExcel 对话框 界面一样)。
  2. 在 Import Options 部分，将 MultiFile (except 1st) Import Mode 模式设置为 Start New Columns for Each Sheet。
  3. 在 Partial Import 部分, 在 Partial Import 选项中选择 All , 然后设置要怎样导入特定的列。 例如，将 From Column 和 to Column 都设置为 2 则将多个 Excel 表的第二列都导入到同一 Origin 工作表中。
  4. 单击 OK 来导入。
