1.255 FAQ-1049 如何用实际的X值位置绘制堆积柱状图?

Last Update: 4/17/2020

StackedColumn-actual-X

当你使用绘图> 基础2D图：堆积柱状图 菜单，或者堆积柱状图按钮来创建堆积柱状图时，即使X值是数值，Origin会自动转换为行号，然后把轴刻度线标签设为来自数据集的文本。所以你会看到柱状图的分布是均匀的。

如果你要用实际的X值位置绘制堆积柱状图，你可以：

  1. 选择窗口：脚本窗口打开 脚本窗口。
  2. 运行以下脚本设置系统变量 DRX 为 0，然后再绘制堆积柱状图。
@DRX=0;
提示: 要返回默认设置，可以设 DRX 为 16。


或者

  1. 选择数据集绘制点线图（选择绘图> 基础2D图: 点线图菜单）。
  2. 激活图形，选择格式： 绘图属性打开绘图细节对话框。
  3. 在对话框左下角，设 绘图类型 为 柱状图/条形图。
  4. 在左侧面板选择 图层级别。在堆叠选项卡选择 累积 。点击确定。
