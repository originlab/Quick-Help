5.33 FAQ-727 如何将多个图形窗口制作成一个多页的 PDF 文件？

Multi-Page-PDF

Last Update: 8/1/2018

Origin 没有直接的方法将多个图形窗口保存为多页 PDF 文件。但您可以先使用 Sending Graphs to PowerPoint 的功能将图形发送到 PowerPoint。然后，在 PowerPoint 中选择 Save as PDF Format 选项来创建多页 PDF 文件。

注:

  1. Send Graphs to PowerPoint 会将文件夹中的所有图形发送到 PowerPoint。如果您要跳过某些图形窗口，可以在 Project Explorer 中右键单击该图形，然后选择 Skip Slide。
  2. 如果图形不在 Origin 的同一文件夹中，则可以右键单击每个图形窗口，为其在 Favorites 文件夹中创建它们的快捷方式，然后前往 Favorites 文件夹将图形发送到 PowerPoint。
  3. 若要调整幻灯片顺序，请在 Project Explorer 的 Details 视图中，先单击 Slide 列表头来按幻灯片顺序对窗口进行排序，然后，向上或向下拖动图形窗口以调整其幻灯片顺序。
  4. 若要包含布局窗口于幻灯片中，请将它们作为图片发送至 PowerPoint 而不是嵌入的 Origin 对象。
