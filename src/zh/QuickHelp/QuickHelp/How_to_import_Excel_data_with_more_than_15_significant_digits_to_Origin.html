2.19 FAQ-429 怎样将含超过15位有效数字的数据的 Excel 表导入 Origin 中？

get-Excel-data-with-full-precision

Last Update: 4/24/2019


当你从 Microsoft Excel 直接将超过15位有效数字的数据复制到 Origin 中时，由于 Excel 精确度的限制，原始数据有效位数将被限制为15位。

要使得复制时有效位数不被更改，需要将 Excel 表导入到 Origin 中而不是直接复制粘贴数据。选择菜单 File:Import:Excel(XLS, XLSX, XLSM)... 来导入 Excel 表。

导入后，选择菜单 Preference: Options 打开 Options 对话框，在 Numeric Format 选项卡，勾选 Significant Digits 单选按钮后输入数字设置有效位数。或者在工作表中双击一列打开 Column Properties 对话框，在 Options:Digits 中设置有效位数。

Keywords:Excel, import data, decimal places, significant digits, 数据导入，小数位数，有效数字
