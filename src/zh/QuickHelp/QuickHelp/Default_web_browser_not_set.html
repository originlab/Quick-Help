6.49 FAQ-787 当我在线激活 Origin 的时候，我看到一个 Enter License 对话框打开，但是除此之外什么反应也没有，怎么办？

Default-Web-Browser

Last Update: 7/9/2018

当你在线激活 Origin 的时候，浏览器应该自动打开并跳转到 OriginLab 的官网。如果浏览器没有自动打开，这有可能是因为电脑上的默认浏览器没有设置，或者是用来打开默认浏览器的“文件关联”没有设置好。

设置默认浏览器:

从以下列出的3种主流浏览器中选择一种，根据步骤把它设置为默认浏览器:

  • Microsoft Internet Explorer: 从控制面板 Control Panel 或者从 Internet Explorer浏览器的菜单中选择 Tools | Options，打开 Internet Options，在 Programs 选项卡上，点击 "Make Default"。然后重新激活 Origin。
  • Mozilla FireFox: 从浏览器的菜单中选择 Options。在 General 页中勾选 "Always check to see if FireFox is the default web browser on startup" 并把 FireFox 设置为默认浏览器。然后再重新激活 Origin。
  • Google Chrome: 从浏览器的菜单中选择 Settings. 在 Settings 页面的下方的 Default browser 的部分，点击 "Make Google Chrome my default browser" 来设置为默认浏览器。然后再重新激活 Origin。
设置文件关联 File Association:

如果默认浏览器已经设置好但是 Origin 的激活仍然有问题，检查 File Association，确保指定的文件默认都使用设置好的默认浏览器来打开。

Windows Vista, 7, 8, 8.1 and 10

  1. 打开控制面板，点击 Default Programs。
  2. 点击 Associate a file type or protocol with a program。
  3. 在列表中找到 .htm 和 .html。检查 Current Default 中列出的是否是你设置好的默认浏览器。
  4. 如果不是，选中这一行并点击 Change Program 按钮，选择正确的默认浏览器。.htm 和 .html 都要设置。然后拉到列表的最下方，Protocols 这一部分，确保 FTP, HTTP 和 HTTPS 的 Current Default 都设置为你的默认浏览器。
  5. 重新激活 Origin。
