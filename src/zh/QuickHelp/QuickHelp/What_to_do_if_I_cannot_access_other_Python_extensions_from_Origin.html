4.44 FAQ-406 如果无法从 Origin 访问其他 Python 扩展应该怎么办？


Last Update: 7/13/2018

access-python-extensions

在 Origin 中内置的 Python 已包含所有基本的模块。但是如果您安装了其他 Python 的扩展（例如，pyQt4）并且希望从 Origin 内置的 Python 中访问它，那么您在导入之前总是需要将其安装文件夹添加到系统路径列表中。

至于如何将安装文件夹添加到系统路径列表，您可以参考 LabTalk 文档中的 Working with Python 章节的 A Note to Use Python Extensions 的部分。

Keywords:python, python binding, python extension, PyQt4, system path, python 绑定，python 扩展，系统路径
