2.54 FAQ-494 怎样得到多条曲线的平均曲线？

Average-MultipleCuves

Last Update: 7/20/2018

如果每个 X 列值不一样，那么通常以各自 X 列创建的曲线的 X 起始值与 X 终止值也不一样。可以为这些曲线在指定的 X 范围，例如各曲线重叠的范围或所有曲线的最大范围 (从所有 X 的最小值到所有 X 的最大值)，内创建平均曲线。 平均曲线上的每一点的纵坐标都是其他曲线对应点纵坐标的平均值。

如果有此需要，点选菜单 Analysis:Mathematics:Average Multiple Curves 打开对话框并在 Averaged X 下列列表中选择相应的 X 范围。如此，即可生成平均曲线。
