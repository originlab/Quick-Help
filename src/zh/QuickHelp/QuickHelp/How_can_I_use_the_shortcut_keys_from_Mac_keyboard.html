5.6 FAQ-343 是否存在 Mac 键盘的快捷键对照表？

shortcut-mac-keyboard

Last Update: 4/24/2019

当您通过虚拟机在 Mac 上运行 Origin 时，您可能会注意到 Mac 键盘上的某些快捷键与 Windows PC 键盘上的快捷键是不一样的。所以下表对比了这些两者不同的快捷键，而其他在 Origin 的加速器/快捷键列表中的快捷键在 Mac 键盘和 Windows PC 键盘上是一样的。

项目

快捷键 菜单/操作 操作预置/结果
PC 键盘 Mac 键盘
Ctrl + F4 Command + W 关闭当前工作表/图形/矩阵窗口
Ctrl + F6 fn + Ctrl + F6 Go to Next window 激活下一个窗口
Alt + 1 fn + Alt + 1 View : Project Explorer 打开 Project Explorer 窗口
Alt + 2 fn + Alt + 2 View : Results Log 打开 Results Log 窗口
Alt + 3 fn + Alt + 3 View : Command Window
Window : Command Window 		打开 Command Window 窗口
Alt + 4 fn + Alt + 4 View : Code Builder 打开 Code Builder.
Alt + 5 fn + Alt + 5 View : Quick Help 打开 Quick Help 窗口
Alt + 6 fn + Alt + 6 View : Messages Log 打开 Messages Log 窗口
Alt + 7 fn + Alt + 7 View : Smart Hint Log 打开 Smart Hint Log 窗口
Alt + Shift +3 fn + Shift + Alt + 3 Window : Script Window 打开 Script Window 窗口
F3 fn + F3 在 Project Explorer: 于某个 Folder 点击右键并选择 Find 在激活 Project Explorer 的时候，从选择的文件夹中打开 Find 对话框

工具

快捷键 菜单/操作 操作预置/结果
PC 键盘 Mac 键盘
F7 fn + F7 Preferences: Theme Organizer 打开 Preferences: Theme Organizer.
F8 fn + F8 Tools:Fitting Function Builder 打开 Fitting Function Builder.
F9 fn + F9 Tools:Fitting Function Organizer 打开 Fitting Function Organizer.
F10 fn + F10 Tool:X-Function Builder 打开 X-Function Builder.
F11 F11(若不成功，请更改系统快捷键设置) Help:Origin Central 打开 Origin Central.

编辑

快捷键 菜单/操作 操作预置/结果
PC 键盘 Mac 键盘
Ctrl + Alt + B fn + Ctrl + Alt + B Edit : Button Edit Mode 转换 Button Edit Mode.
Del fn + Del Edit : Clear 当一个对象被选中
Alt + Double-click Command + Option + Double-click 在某个对象上点击右键并选择 Programming Control 当一个对象被选中

工作表

快捷键 菜单/操作 操作预置/结果
PC 键盘 Mac 键盘
F4 fn + F4 Format:Worksheet 打开 Worksheet Properties 对话框

图形

快捷键 菜单/操作 操作预置/结果
PC 键盘 Mac 键盘
F2 fn + F2 Format:Page 从当前图形窗口打开 Plot Details 对话框的 Page Properties 页面
F5 fn + F5 Refresh 按钮 在重绘后刷新图形窗口
R + Ctrl + Shift Not Applicable Rotate + Ctrl + Shift 当某个 3D OpenGL 图形处于激活状态时，在 Z 轴方向旋转该图形。
Ctrl + Double-Click fn + Ctrl + Double-Click 选中图形的某个点

缩放和平移

快捷键 菜单/操作 操作预置/操作结果
PC 键盘 Mac 键盘
+ Z + "Home" Not Applicable View:Whole Page 将页面还原到 100% 视图。(如果是布局则还原到 91%). 激活图形窗口并且在工具栏中选中缩放和平移按钮。
