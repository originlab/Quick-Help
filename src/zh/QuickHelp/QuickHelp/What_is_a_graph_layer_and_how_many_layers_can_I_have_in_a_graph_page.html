1.26 FAQ-140 什么是图层？在一个图形页面中最多可以有多少个图层？

Graph-Layer-Defination

Last Update: 11/13/2018

一个典型的图形由以下元素组成：一组 X 和 Y (可能包括 Z) 坐标轴，一组或者多组绘图数据，相关的坐标轴标题，文本标签和绘图对象。


这些元素组成了一个可移动并可以改变大小的单元，就称为 layer （图层）。任意 Origin 绘图必须至少含有一个图层，但至多也不能超过 1024 个图层。(不存在没有图层的图形窗口，当删掉所有图层后，Origin 也会关闭此图形窗口)。


由于图层是一个独立的单元，所以在图形页面上几乎可以无限制的自由移动和排列。在多图层的图形中，各图层之间可以部分叠加 (例如在 "inset" 图中)，完全叠加 (例如在 "double-Y axis" 图中)，或者完全分离 (例如在 "panel" 图中)。


虽然图形可以含有多个图层，但是在同一时刻只有一个图层处于激活状态。处于激活状态的图层由图形窗口左上角的凹陷的 n 按钮表示，其中 n 表示的是图层编号。在下图示例中，图层 3 处于激活状态，其状态可以通过左上角凹陷的 3 得知，或者通过 查看：显示：当前图层指示器 查看。


当鼠标点击图形的不同区域时，具体激活的图层会随着鼠标点击到的图层而发生变化。如果不想用这样的图层切换方式，可以通过配置 Origin 使得必须点击左上角的图层按钮才能切换。设置如下：选择 格式：页面属性... 菜单以在page(graph)层级打开 绘图细节 对话框，切换到 图层 选项卡，然后勾选 只通过点击图层号已激活图层 复选框。（注：这一设置仅对当前图形有效）。
