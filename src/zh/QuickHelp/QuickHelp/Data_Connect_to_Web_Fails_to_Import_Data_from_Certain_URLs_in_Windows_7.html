2.106 FAQ-995 数据：连接到网页失败，Windows 7 无法从某些网址导入数据

Data-Connect-to-Web-Fails-Windows-7

Last Update: 10/23/2019

如果在 Windows 7 上安装了 Origin 2019b 或更高版本，并且通过数据：连接到网页使用数据连接器，则可能会出现从协议为 HTTPS 的网址导入数据失败。 可能（也可能不会）有一条错误消息。

它与 Windows 中某些过时的设置有关。

可以下载并安装此 Microsoft Easy Fix 向导来定位此问题：https://aka.ms/easyfix51044

有关此问题的深入技术说明，请参阅：https://support.microsoft.com/en-us/help/3140245/update-to-enable-tls-1-1-and-tls-1-2-as-default-secure-protocols-in-wi

Keywords:数据连接器, 连接到网页, Windows 7, HTTPS, 下载失败, Error -50
