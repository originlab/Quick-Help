3.86 FAQ-324 如何计算移动中位数？

cal-movmedian-SCV

Last Update: 7/16/2018

使用 Set Value 对话框简单地计算指定范围（由移动窗口的大小定义）的移动中值。

  1. 选择要放置结果的列或单元格。
  2. 从菜单中选择 Column: Set Column Value...（或按 CTRL + Q 键），打开 Set Values 对话框。
  3. 在对话框中的列公式文本框中输入公式 median(B[i:i+50])，计算 B 列的移动中位数，其中 50 表示移动窗口为从当前的点到其后 50 个数据。
  4. 点击 OK 按钮。

更多信息，请参阅这个文档。
