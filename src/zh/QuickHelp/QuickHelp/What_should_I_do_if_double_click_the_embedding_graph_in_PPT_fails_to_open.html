5.25 FAQ-509 如果双击在 Word/PowerPoint 中的嵌入图形却无法在 Origin 中打开应该怎么办？


Last Update: 7/22/2018

embed-graph-fail-open

有时候您可能会遇到双击在 Word/PowerPoint 的嵌入图形却无法在 Origin 中打开的情况。要解决这个问题，您可以右键单击 Origin 的图标并选择 Run as administrator 来修复注册表。

如果上述方法不能解决这个问题，那么请您尝试以下方法以检查 Run this program as an administrator 勾选框是否被误选了：

  1. 前往 Origin 的安装文件夹, 找到 Origin.exe 文件。右键单击该文件并从上下文菜单中选择 Properties...。
  2. 在 Properties 对话框，前往 Compatibility 标签页。取消勾选 Run this program as an administrator 并点击 OK 按钮。
  3. 右键单击 Origin 图标并选择 Run as Administrator 来打开 Origin。
  4. 尝试再次复制粘贴图形到 Word。
