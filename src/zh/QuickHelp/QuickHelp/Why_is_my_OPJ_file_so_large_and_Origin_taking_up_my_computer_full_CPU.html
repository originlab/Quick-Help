5.30 FAQ-710 为什么生成的 OPJ 文件特别大并且 Origin 会使计算机 CPU 满载？

OPJ-so-large-and-take-full-cpu

Last Update: 8/1/2018

如果您遇到了这些问题：

  • 当您正在使用某个 Origin 项目文件（OPJ）时，Origin 使计算机的 CPU 满载。
  • OPJ 文件非常大。
  • 当您点击黄色的 Recalculate 工具栏按钮时，重新计算失败并且按钮变为红色。

这些问题可能是由未响应的操作引起的。要解决此问题，请执行以下步骤来删除未响应的操作。

打开那个 OPJ 文件，然后打开 Script Window (Window:Script Window)，输入以下脚本并按下 ENTER 键。那么 Origin 将在 Script Window 中列出所有的操作，其中未响应的操作是没有输入和输出的。 

list op;

然后输入以下脚本（并按下 ENTER 键）来删除这些未响应的操作从而解决 OPJ 的问题。 

delete -op;
