1.194 FAQ-896 如果工作表中的数据发生变化，如何重新调整颜色映射？

Rescale-Colormap-Data-Worksheet-Changes

Last Update: 7/24/2018

当数据发生变化时，你可以使用以下方法，轻松地重新调整在等高线图或 3D 颜色映射曲面图上的颜色映射：

  1. 右键单击绘图，选择 Colormap: Set Levels... 打开 Set Levels 对话框。 点击 Find Min/Max，根据你的数据自动设置 From 和 To 的值。 点击 OK 应用此修改。
  2. 双击绘图，打开 Plot Details对话框。 在 Colormap 选项卡上，将 Rescale Mode 设为 Auto。 之后，当你更新数据时，颜色映射将会根据新的数据自动重新调整。
  3. 你也可以尝试使用以下其中一个 Labtalk 脚本： 
    • layer -axyz // Rescale colormap only.
    • layer -axy // Rescale both z axi and colormap.
    想了解更多关于图层重新调整（轴刻度，颜色映射）的命令，可以参考这页.
