5.79 FAQ-1096 升级之后，菜单"文件: 打开Excel" 和 "文件: 新建: Excel"不见了。

File-Open-Excel-Missing

Last Update: 4/27/2021

一直以来，在 Origin 工作区中 Origin 一直支持MS Excel的OLE实例，即使它对Origin功能的支持严重受限。随着时间的流逝，用户报告说，与系统有关的问题有所增加，而我们找不到解决方案。 因此，从Origin 2021b开始，OriginLab决定删除直接在Origin中打开Excel文件时的用户界面控件（文件 > 打开Excel）和在Origin中打开新建的Excel工作簿（文件 > 新建 > Excel）这两项功能。

我们敦促您考虑使用 Excel数据连接器去导入Excel数据。 Excel 数据连接器速度很快，您可以全面调用Origin的图形和分析工具。 此外，由于数据连接器会创建到外部数据文件的“实时链接”，因此当外部Excel文件中的数据更改时，您可以轻松更新Origin项目。

如果不是使用 Excel 数据连接器，则可以通过更改LabTalk系统变量@MOE的值来恢复OLE访问。 

@MOE=0; // 0 (default), 1 = 显示 “文件: 打开 Excel”, 2 = 显示 “文件: 新建：Excel”, 3 = 显示打开和新建Excel

关于如何去更改一个系统变量的值，您可以参考这个 FAQ.
