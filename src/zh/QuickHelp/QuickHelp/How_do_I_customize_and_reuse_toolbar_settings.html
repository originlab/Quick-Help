5.1 FAQ-337 如何自定义工具栏的设置并重复使用该设置？

Customize-Reuse-Toolbar

Last Update: 7/15/2018

通过菜单 View: Toolbars 打开 Customize Toolbar 对话框，您即可在这个对话框中控制您想显示的工具栏。在 Button Groups 标签页，您可以访问所有可用的工具栏按钮（包括一些在默认状态下为不可用的按钮）。

例如，在 Button Groups 标签页选择 Column，然后您会看到 Swap Column 按钮。您可以将此按钮拖放到所有现有的 Origin 工具栏中。

如果您只是单纯地想将某一个按钮从一个工具栏移动到另一个工具栏，那么无需打开 Customize Toolbar 对话框。您只需要按住 Alt 键，然后将按钮从一个工具栏拖放到另一个工具栏中。如果您把该按钮拖放到任意除工具栏以外的位置，那么这个按钮将成为一个单独的浮动的按钮。
