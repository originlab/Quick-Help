1.126 FAQ-706 我的一些图形不能正确导出为EPS格式文件。有没有解决办法呢？

alternative-export-EPS-method

Last Update: 11/21/2018

在菜单 文件：导出图片 中选择导出图片为EPS而不能正确导出的几种可能原因： 1.图形包含有上标/下标格式的文本；2.图形中的文本带有像希腊字母这样的字符或是脚本字符；3.图形包含嵌入式或增强型Windows图元文件；4.使用了 裁去图层框架外的数据 功能。

出现不能导出的问题时，首先确认是否打开了透明度功能。虽然在选择导出格式之前可以打开透明度功能，但 EPS 格式本身不支持透明度，可能会因此使得导出的 EPS 图像失真。

如果与透明度功能无关，请使用下面 B 部分阐述的步骤在 Origin 中导出 EPS 文件。请注意，在完成 B 部分步骤之前，必须先在计算机上安装 PostScript 打印机驱动程序。如果计算机上尚未安装 PostScript 打印机驱动程序，则需要按照下面 A 部分的说明进行操作。

注意:
  1. 因为您将为 Print to Port 指定 File ，实际上并不需要 PostScript 打印机连接到计算机来安装 PostScript 打印机驱动程序。因此，您可以选择任何支持EPS格式输出选项的驱动程序。建议您选择支持自定义页面大小的驱动程序，以便在指定输出EPS图像的边框尺寸时具有最大的可操作性。
  2. 这个创建 EPS 文件的替代方案有以下局限：1）图形必须具有纵向方向。不支持横向模式。对于具有自定义页面大小的打印机驱动程序，这意味着页面高度必须始终大于或等于页面宽度。2）使用Adobe PostScript 打印机驱动程序创建的 EPS 文件无法使用任何 Adobe 软件打开。除了这两个限制之外，下面概述的步骤基本适用各种情况。


Part A

  1. 若要安装 PostScript 打印机驱动程序，你可以访问 Adobe 软件公司, 搜索 printer driver 并为Windows系统选择合适的驱动程序下载并安装。安装驱动程序时，务必为 Print to Port 指定 File 。安装完成后，跳到B部分，使用替代方案在 Origin 中导出 EPS 图像。
  1. 在驱动程序安装过程中，将要求你提供* .PPD文件（PostScript打印机格式）。HP DesignJet 3500CP 打印机的 PPD 文件支持 EPS 输出格式和自定义页面大小。当 Adobe PostScript 打印机驱动程序有了 PDD 文件后，可以很好地创建 EPS 文件。Adobe 在你下载打印机驱动程序的网页底部提供此 PPD 文件（以及相关文件）。

Part B

安装了 PostScript 打印机驱动程序后便可以按照下面的步骤在 Origin 里将图形或页面导出为 EPS 文件了。

  1. 选择菜单 文件：打印，打开 打印 对话框。
  2. 从打印机的 名称 下拉列表中选择一个 PostScript 打印机。
  3. 单击 属性 按钮。打开 Printer Properties 对话框。不同的打印机驱动程序具有不同的参数，但通常步骤4到6中的参数是必选的。
  4. 从 Orientation中选择 Portrait radio 单选按钮。
  5. 选择 PostScript 选项卡（或编辑 PostScript 选项）。
  6. 从 PostScript 输出格式下拉列表中选择 Encapsulated PostScript（EPS）。 Warning 对话框可能会弹出以提醒你需要在打印对话框中勾选 Print to File 复选框。
  7. 在 Warning 对话框中单击 OK 。
  8. 选择 Paper 选项卡（或设置 Paper/Page Size Options ）。
  9. 选择纸张大小以设置 EPS 图像边框的大小。选择自定义并根据需要指定尺寸。
  10. 单击 OK 关闭 Printer Properties 对话框。
  11. 在 打印 对话框的Printer组中勾选 打印到文件 复选框。
  12. 单击 确定 关闭 打印 对话框并打开 打印到文件 对话框。
  13. 从 保存到 下拉列表中选择要保存的文件夹。
  14. 在 文件名 文本框中键入文件名（* .EPS）。
  15. 单击 保存 关闭对话框以保存 EPS 文件。

要创建附加图形或布局窗口的 EPS 文件，激活相应的图形窗口。 选择文件：打印 并重复步骤11到15。
