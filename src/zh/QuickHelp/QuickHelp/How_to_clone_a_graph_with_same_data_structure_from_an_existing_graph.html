1.141 FAQ-754 怎样从已有图形使用相同结构的数据绘制新的图形？

Clone-Graph

Last Update: 4/20/2021

激活当前图形窗口，在标题栏右击选择 保存模板为... 或选择菜单 文件：保存模板为... 打开 Utilities\File:template_saveas 对话框。勾选 标记为克隆模板 复选框并在 用...来匹配绘图列 下拉菜单列表中选择一个参数行作为数据结构匹配的标识，单击 确定 按钮保存当前图形样式为可复制模板。

之后便可以使用新数据来创建与此模板相同结构的图形。要做到这些，选择要绘图的数据，选择菜单 绘图：我的模板，再选择已保存的可复制模板，即可绘制与模板样式相同的图形。可复制模板左下角有小绵羊标志，以此区别于普通模板。

也可以直接选择 模板库，在对话框中先预览模板，再选择相应的模板用于绘图。
