3.114 FAQ-665 如何在不删除数据的情况下从分析过程或图形中排除异常值？

Exclude-Outliers

Last Update: 7/29/2018

Origin 提供 Mask 工具，使用这个工具你可以在图形中排除特定的数据点或者特定范围内的数据：

  • 在散点图上，单击异常点两次以选择它，单击右键，然后选择 Mask 来屏蔽该点。
  • 在散点图/线图上，使用 Tools 工具栏中的 Mask Point on Active Plot 或 Mask Point on All Plot 工具，选择并屏蔽选中的图形范围。你可以使用 Space 键， 使工具在矩形或任意形状之间切换。
  • 在工作表中，你可以选中数据范围，右键单击，选择 Mask 以屏蔽数据。

当你屏蔽了数据点时，使用任何拟合或者统计工具在进行数据分析时，不会考虑数据中被屏蔽的数据点。

Keywords:Outliers, Mask Data，异常值，屏蔽数据
