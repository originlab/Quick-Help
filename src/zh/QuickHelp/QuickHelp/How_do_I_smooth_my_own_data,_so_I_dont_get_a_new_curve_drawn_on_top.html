3.65 FAQ-302 如何对原始数据进行平滑处理，而不是在原始数据上新画一条平滑后的曲线？

Smooth-Data-without-NewCurve

Last Update: 7/14/2018

当你选择 Analysis: Signal Processing: Smooth 来进行平滑处理时，输出 Output 默认为 <new>，即平滑的结果会作为一条新的曲线绘制在原曲线上。如果你想对原始曲线进行平滑，请选择 <input>: XY=inputXY。

如果对话框的 Recalculate Mode 设置为 Auto 或 Manual，你将会在 Script Window 窗口看到一条提示信息: "Unable to establish autoupdate. Recalculate mode is set to NONE." 这表示，当输出与输入数据为相同列时，自动更新功能不可用。
