1.100 FAQ-617 我的彩色打印机只能打印出黑白图，怎么办？

color-printer-prints-black

Last Update: 11/19/2018

如果你用彩色打印机打印 Origin 的图，却只能打出黑白图片，可以尝试以下方法解决:

1. 检查彩色墨盒/墨粉是否已空或堵塞

虽然看起来不太可能，但是以防万一我们建议你还是检查一下。大多数彩色打印机的驱程/软件都提供墨盒校准和检查颜色输出的功能。你可以查阅打印机的用户手册，或者点击 Start : Settings : Printers 来打开打印机的驱程，然后右键单击打印机图标并选择 Properties。在 Properties 对话框中查看关于墨盒检测的选项。

2. 查找“仅打印黑白”选项

打开打印机驱动程序的 Properties 对话框（参见上文），然后查找是否有类似“仅使用黑白打印”这样的选项。如果有，取消选择它，或是选择类似“启用彩色打印”这类的选项。某些彩色打印机的彩打和黑白打印的分辨率不同（通常，黑白打印的分辨率更高）。为保证打印质量，有些应用程序会强制打印机使用分辨率最高的模式来打印，也就是黑白模式，这种情况下彩色输出就会被关闭。但 Origin 不会这样做。

3. 启用灰度颜色打印

有时候，在某些版本的操作系统上使用某些牌子的打印机，可能会导致 Origin 不能够正确读取打印机驱动程序的信息。在这种情况下，Origin 可能会把一台彩色打印机“误认为”是一台黑白打印机。在 Origin 的 打印 对话框中，根据 Origin 所读到的打印性能，会显示以下两个选项之一:

  • 启用灰度颜色
彩色打印机选项
  • 使用简化颜色
黑白打印机选项

如果你看到第一个选项并且你使用的正是一台彩色打印机，勾选此项，Origin 将会把图片包含的所有颜色信息传送给打印机，而打印机会根据实际情况做必要的转换和处理。这个选项相当于旧版 Origin 中的 “Print color on black and white printers” 选项。
