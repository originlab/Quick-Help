3.32 FAQ-261 怎样在用户自定义的拟合方程中调用 NAG 库？

NAGLibrary-in-FitttingFunction

Last Update: 7/21/2018

首先选择菜单 Tools: Fitting Function Organizer 或 Tools: Fitting Function Builder 选项打开相应的对话框，单击 New Function 按钮来自定义函数。 当函数形式选项为 Origin C 时，你可以在代码生成器中定义和编辑函数。当你点击函数编辑框右边的 按钮时, 将会打开代码编辑器窗口。直接在此窗口进行编辑，滑到最上方，为所需的任一 NAG 标题文件添加 #include 指令。
