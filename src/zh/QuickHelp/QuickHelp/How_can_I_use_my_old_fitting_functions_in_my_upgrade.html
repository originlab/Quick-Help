6.19 FAQ-21 如何在升级后的 Origin 中使用原来版本中定义的拟合函数？

Transfer-User-Defined-Function

Last Update: 4/24/2019

Transfer User Files 这个工具可以帮助升级用户很方便的把自定义文件，包括自定义的拟合函数，从老版本转移到新版本的用户文件夹中。你可以从菜单 Preferences: Transfer User Files 打开对话框。详见这个文档。

Keywords:user files, upgrade, customization, user-defined, transfer, 用户文件, 自定义文件, 升级, 转移
