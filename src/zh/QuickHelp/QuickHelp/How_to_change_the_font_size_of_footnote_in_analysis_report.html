3.53 FAQ-287 怎样更改结果表中脚注的字体大小？

font-size-footnote

Last Update: 4/24/2019

在一些数据分析（例如假设检验）的内置结果表中，表格中的脚注可以帮助用户得出结论。

要更改脚注的字体大小，可以选择菜单 Preference: Options 来打开 Options 对话框，然后在 Miscellaneous 选项卡 中， 选择 Footnote Size Ratio 选项，然后在下拉菜单中为所有结果表中的脚注选择相应的字体大小。

注意: 为使字体更改生效，你需要对结果表中的表格进行缩放来刷新文本显示。
