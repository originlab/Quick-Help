6.51 FAQ-789 升级系统之后，Origin 弹出对话框让我重新激活，这是怎么回事？

Upgrade-Windows

Last Update: 4/27/2020

系统升级有可能会删除掉原有的 Origin 许可证文件。遇到这种情况，你可以尝试重新激活:

  • 单用户版激活步骤
  • 多用户版激活步骤

如果在获取许可证的时候看到类似 “you have reached the maximum number of licenses” 这样的错误信息，请依照以下步骤填写移机申请:

  1. 打开 OriginLab 官网 http://www.originlab.com/reginfo
  2. 登录。
  3. 选择你要移机的序列号。
  4. 在打开的网页上，点击 Deactivate License or Transfer License 链接。
  5. 点击页面下方的 Note 中的 “request a license transfer” 链接。
  6. 在打开的 System Transfer Request 页面，按要求填写信息并点击 Submit 发送移机申请。
