6.84 FAQ-1099 是否可以在使用 M系列（M1 M2 等）芯片的苹果电脑上安装 Origin？

Install-Origin-on-Mac-M1

Last Update: 4/17/2024

当然可以。M系列（M1 M2 等）芯片的苹果电脑上可以安装和运行 Origin。

  • 你将需要使用苹果兼容的虚拟机软件，在 MAC 上来运行 Origin。 我们强烈推荐使用 Parallels Desktop。
  • 并且，由于 Origin 是基于 Windows 的，你需要在虚拟机上安装 Windows。
  • 由于 M1/M2/M3 芯片是基于 ARM 的，你需要安装 Windows 11 on ARM。

如果按照上述指导操作后，Origin 仍然启动时或工作期间崩溃，或行为异常，如运行速度非常慢、右键单击不起作用等，请参考本页修复这个问题。

Parallels Desktop 中的 Coherence 模式可能会引发 Origin 的一些意外问题，如果您感觉 Origin 不稳定，请退出 Coherence 模式并重试。
