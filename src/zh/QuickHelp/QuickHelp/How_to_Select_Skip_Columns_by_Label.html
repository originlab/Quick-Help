2.69 FAQ-783 是否可以按标签或索引选择和隐藏工作表中的列？

Select_Column_by_Label_or_Index

Last Update: 4/19/2023

可以按列标签或按列索引在激活的工作表中选择列 (或者隐藏列) ：

  1. 单击 编辑：选择 或者 列：选择列 以打开一个对话框，在此对话框中可以进行如下定义：
    • 选择模式。
    • 如果按标签进行选择，可以选择字符串或者条件；如果按索引进行选择，可以从 Start 列开始每选择 N 列后 跳过 M 列。两种模式如下截图所示：
  2. 点击 选择 按钮来选择出满足条件规则的列。
  3. 也可以点选 Hide 按钮来隐藏选定的列。如果希望解除隐藏，请选择Edit: Undo 或 选择整个工作表， 然后右键单击并选择 Hide/Unhide Columns: Unhide。
