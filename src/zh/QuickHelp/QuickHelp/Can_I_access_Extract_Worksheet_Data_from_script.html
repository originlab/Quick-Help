4.23 FAQ-382 如何通过 LabTalk 脚本访问 Worksheet Query？

WksExtract-Script

Last Update: 7/9/2018

Origin 提供了一个 wxt X-Function，您可以通过 LabTalk 脚本来调用此 X-Function。在 Command Window (可通过选择菜单 Window: Command Window 打开) 中, 输入： 

wxt

然后按下空格键即可利用自动补全功能设置函数的各个参数。您也可通过查看该函数的对话框的各个选项以此设置函数相应的参数，输入命令： 

wxt -d

然后按下 Enter 键，则会打开该函数的对话框。您也可以使用该对话框创建一个条件字符串并传入 wxt X-Function。更多的详细资料请查阅此处.

Keywords:subset, reduction, 子集，减少
