3.26 FAQ-254 怎样使用一个拟合函数对单个数据集的多个部分进行拟合分析？

Single-Data-Multiple-Fit

Last Update: 7/23/2018

请遵循以下步骤：

  1. 激活一个工作簿，然后按住 Ctrl 键对同一列数据进行多选。又或者激活图形窗口，使用 Regional Data Selector 工具 来选择图形上的两个及以上的区域。
  2. 选择好相应的区域之后，选择菜单 Analysis: Fitting: Nonlinear Curve Fit选项来打开 NLFit 对话框。
  3. 从 Settings 选项卡中的 Function Selection 选项里选择一个函数。
  4. 在 Settings 选项卡的 Data Selection 部分中，将 Global Fit 改为 Multi-Data Fit Mode 。
  5. 在 Parameters 选项卡中，你可以共享参数，在 Code 选项卡中的 Constraints 选项中，你可以勾选 Enable Linear Constraints 复选框然后输入相应的约束条件。例如，如果拟合函数是每个峰具有相同的宽度的 Gauss 函数，你可以勾选 Share 复选框来共享参数 w。如果拟合函数是 Boltzmann 函数，一般 Boltzmann 函数曲线上部分与下部分是中心对称的关系，所以曲线的顶部切线（A2）与底部切线（A1）是平行的，可以输入此约束条件 A1_2 = A2。

Keywords:replica, peak, repeat，多个，峰，重复
