1.198 FAQ-921 如何使图形看起来更平滑？

Show-Plot-Smoother

Last Update: 4/24/2019

你可以启用抗锯齿功能使图形看起来更平滑。


如果想使抗锯齿功能在 Origin 中默认开启

你可以设置 AntiAliasing 主题为系统默认主题。

  1. 选择菜单 Preferences: Theme Organizer。
  2. 在 Graph 选项卡下, 找到 AntiAliasing 主题。右键单击此主题，选择 Set as System Theme。


如果想对多个图形启用抗锯齿功能
你可以在 Preferences: Theme Organizer 对话框中，定位到 AntiAliasing 主题，并在 Apple Theme To 下拉列表中选择希望主题应用的范围，然后点击 Apply Now 按钮。


如果要创建一个带有抗锯齿功能的用户自定义主题

  1. 选择 Preferences: Theme Organizer。
  2. 在右键单击 Graph 选项卡下的表格，选择 Add New 以添加一个新主题。如果你已经有一个用户自定义的主题，右键点击主题，选择 Edit。
  3. 在打开的 Theme Editing 对话框中，如果不存在 Page 节点，右键单击表格，选择 Add: Page。
  4. 如果在 Page 节点下不存在 Display 子节点，右键单击 Page 节点，选择 Add: Display。
  5. 右键单击 Display 子节点，选择 Add: Anti-alising。
  6. 选上添加 Anti-aliasing 的复选框。
  7. 保存主题。
