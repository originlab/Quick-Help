6.102 FAQ-1150 如何解决 "The activation period for this license has expired" 的问题？

Activation-Period-Expired

Last Update: 10/26/2022

激活 Origin（线上或线下）时，在 Enter License 对话框中粘贴了许可文本之后，反馈了如下的错误信息：

Licensing Problem: The activation period for this license has expired. Please retrieve a license again, and enter it into the Origin license dialog before the activation period expires.

这个问题可能会在 Origin 2016 或更早的版本中出现。

解决办法：

  • 确保输入的许可文本仍处于激活有效期，也就是说，线上方式的为15分钟，线下方式的为24小时。
  • 如果输入的许可文本在激活有效期，但仍旧出现此问题，请尝试以下的方法：
    1. 如果 Origin 已经运行，关掉它。
    2. 打开 Windows 的资源管理器或我的电脑。
    3. 浏览到文件夹： C:\ProgramData\OriginLab\##\ (## 为 Origin 的版本号)。
    4. 删除整个文件夹。
    5. 右击 Origin 程序图标，然后选择以管理员身份运行。
    6. 使用 Getting Started 许可证对话框重新获取许可证，然后从网站上复制许可证文本到 Enter License 对话框。
