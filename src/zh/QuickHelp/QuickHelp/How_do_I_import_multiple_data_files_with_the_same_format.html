2.6 FAQ-413 如何导入多个格式相同的数据文件？

Import-Multi-Data

Last Update: 4/24/2019

在导入向导 (Data: Import from File: Import Wizard) 或多个 ASCII (Data: Import from File: Multiple ASCII) 中自定义好导入设置后可以一次性导入一个、多个甚至全部的数据文件。

  • 关于导入向导的更多内容请看导入向导.
  • 关于如何导入多个 ACSII 文件的详细内容请看 自定义 ASCII 文件导入设置

如果将来要利用相同的导入设置来导入类似的数据，可以在导入向导中创建自定义过滤器（filters，*.OIF files) 或者在 Import ASCII 对话框中 中保存主题文件。这样就可以在导入向导中选择自定义的过滤器或从 "最近使用过" 菜单中选择主题文件来导入数据。
