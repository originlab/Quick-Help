1.195 FAQ-911 如何在导出图形为矢量图 (PDF, EPS, EMF) 时，保持图例的位置/大小？

Keep-Legends-Position-Size

Last Update: 7/11/2018

将图形导出为矢量图（PDF，EPS，EMF）后，你可能会发现导出图像中图例的位置或大小与在 Origin 图形中的有所不同，这可能是因为导出的图像 DPI 分辨率与 Origin 的图形页面分辨率不同而导致的。

为了保持导出的图形中图例位置/大小不变：

  1. 检查 Origin 中的图形页面分辨率。
    选择菜单 Windows:Script Window...，打开 Script Window。将以下 Labtalk 脚本复制到Script Window中， 并选中脚本按 Enter 键。此时，将会输出图形页面的水平分辨率值。
    • 使用与图形页面相同的分辨率值导出图像：
      选择 File: Export Graph 打开对话框，并展开 Export Setting: Advanced 节点。将 Resolution 设为上一步中输出的值。

      • 或者

    • 重置图形页面的分辨率，使其与 Export Graph 对话框中的默认设置相同。
      选择菜单File: Export Graph， 打开 Export Graph 对话框， 默认情况下，你会看到分辨率值是 300。复制下面的脚本到 Script Window，选中脚本并按 Enter 键，以设置图形页面分表率为 300。然后你可以按照 Export Graph 对话框的默认设置导出图形。 
      page.resx=300;
