2.49 FAQ-483 怎样复制列公式？

Copy-with-formula

Last Update: 7/16/2018

首先在 Set Values 对话框中为列设置公式（以 F(x)= 列标签行上的公式为例），选择整列并右击选择 Copy:Copy(Formula) (或按快捷键 Ctrl+J )复制列公式，然后选中另一列粘贴即可。

按快捷键 Ctrl+Shift+U 可快速切换 F(x)= 列标签行的显示状态。

Keywords:column formula, copy, paste，列公式，复制，粘贴

Minimum Origin Version Required: 9.1 SR0
