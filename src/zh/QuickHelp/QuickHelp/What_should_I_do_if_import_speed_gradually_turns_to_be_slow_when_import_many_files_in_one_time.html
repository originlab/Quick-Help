2.16 FAQ-426 怎样解决一次性导入多个文件时会变慢的问题？

Improve-Import-Speed

Last Update: 7/9/2018

导入文件时，文件信息将被记录在工作簿中以启用 Re-Import 功能。导入后，右击工作簿标题栏选择 Show Organizer 打开 Workbook Organizer 对话框，左侧面板将列出这个工作簿中所有曾经被导入文件的路径和名称等信息。这个列表太长(导入过很多文件)时，说明太多信息被储存，导入速度就会变慢。

要提高导入速度，需按如下设置系统变量 (@npi) 来禁用导入文件时记录文件信息到工作簿。 

@npi = 1;  // 禁用导入文件时记录文件信息

注意：Origin 的 Re-Import 功能 依赖于导入文件时文件信息的记录。
