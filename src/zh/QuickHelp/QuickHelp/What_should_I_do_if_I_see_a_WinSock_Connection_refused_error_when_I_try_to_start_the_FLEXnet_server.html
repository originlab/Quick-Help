6.106 FAQ-1187 在尝试启动 FLEXnet 服务器时看到“WinSock: Connection refused”的错误，该怎么解决？

WinSock-Connection-Refused-Server-Error

Last Update: 10/25/2023

如果您尝试启动 FLEXnet 许可证服务失败了，并在 LMTools 的 Server Status 选项卡中看到以下错误消息：

System Error: 10061 “WinSock: Connection refused”

这是由于服务器没有对日志文件所在路径的写入权限造成的。这个问题多见于用非管理员的 Windows 帐户登录并运行服务时。


如何解决这个问题：

您将需要给服务器应用程序必要的访问权限。详细的步骤信息，请参阅Flexera 的这个文档。
