3.28 FAQ-256 怎样在不同拟合情况下编写线性约束条件？

Linear-Constraints-Cases

Last Update: 8/9/2018

不同情况下需使用不同方法来编写线性约束条件。

复测拟合 (Replica Fitting)

用法是 parameter name +__n，其中 n 表示 (n-1)th 个重复测量的副本，注意，此时使用了 两个下划线号 。

假设有 y0 这个参数并且有一个副本，那么可用的表示法是： 

y0
y0__2

峰值分析 (Peak Analyzer)

此时将使用 parameter name +__n 表示法，其中 n 表示 nth 个峰，注意，此时使用了 两个下划线号 。

假设有两个峰和两个变量，a 和 b，那么可用的表示法是： 

a__1
b__1
a__2
b__2

一个典型约束示例是 b__1 = b__2。

如果有共享参数，只需要使用同组中第一个参数来编写约束。例如，如果参数 a__2, a__3 是共享的，此时约束 a__1 = a__2 将有效， a__1 = a__3 则无效。

注意:
  • 峰值分析 (Peak Analyzer) 中约束表示法与复测拟合 (Replica Fitting)表示法之间的区别在于第一组参数。在峰值分析 (Peak Analyzer)中的表示法会对第一组参数添加下划线和数字后缀，而复测拟合 (Replica Fitting)中的表示法则不会。

全局拟合

用法是 parameter name+_n，其中， n 表示 nth 个数据集。

假设 a 是一个参数并且有两个数据集，那么可用的表示法是： 

a
a_2
注意: 表示法中的后缀 "_n" 仅在全局拟合时有效。单独拟合时会分别对每个数据集进行拟合，那么 "_n" 也就没有意义。
