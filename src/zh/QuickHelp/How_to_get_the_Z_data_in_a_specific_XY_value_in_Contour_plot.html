3.169 FAQ-1183 如何获取等高线图中特定X/Y值的Z数据？

Interpolate-Z-From-XY-In-Contour

Last Update: 10/12/2023

想要从 XYZ 等高线图中获取指定 XY 数据 的 Z 值，您可以使用从 XY 插值Z工具。

  1. 选择菜单分析：数学：从 XY 插值 Z。
  2. 在打开的对话框中，
    • 用于插值的 XY 值：选择要计算 Z 值的 XY 列。
    • 输入：选择用于创建轮廓的 XYZ 列。
    • 方法：
      • 不选择等高线平滑选项：使用 方法 = 三角
      • 选择等高线平滑选项：使用方法 = 随机TPS并且TPS平滑因子设置为您在等高线中设置的平滑参数。
