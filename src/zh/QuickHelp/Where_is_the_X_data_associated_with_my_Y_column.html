2.51 FAQ-485 怎样显示与Y列相关联的X列？

AssociatedX-with-YCol

Last Update: 7/19/2018

在进行某些导入操作和执行一些特定的分析功能时，可能只在工作簿中只有 Y 列而没有在 Y 列的左侧添加关联的 X 列。

如果是这种情况，表明 X 列是均匀分布的，X 起始值和步长信息存储 Y 列的 采样间隔。 Origin 后续以此采样间隔作为 X 列来与 Y 列进行绘图和数据分析。因此这种方式也节省了工作簿中列的数量，可以更有效地存储和显示其他数据。

如果采样间隔行不可见，用以下两种方式来显示采样间隔行。在工作簿标题栏右击在右键菜单中选择 View: Sampling Interval 选项；或者用菜单 Format: Worksheet 来打开 Worksheet Properties 对话框，然后点击 Edit Column Label Rows... 按钮，从而来自定义要显示的标签行。

在存在采样间隔行的Y列的的左上角会有一个图标 i。单击此图标打开 Column additional information 对话框，可以在此对话框中设置 X 轴起始值，步长以及其它相关的参数。

选择一个 Y 列，使用菜单 Column: Show X Column 选项即可在此 Y 列左侧添加一个 X 列。但 Y 列的采样间隔信息仍然存在。
