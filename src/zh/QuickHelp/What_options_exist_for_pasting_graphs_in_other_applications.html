1.93 FAQ-446 从 Origin 中复制粘贴到其他应用程序中的图怎么编辑？

Pasting-Graphs-other-Applications

Last Update: 11/16/2018

  1. 在 Origin中，选择菜单 编辑：复制页面。
  2. 在其他应用程序（如 Microsoft Office 的软件）中，选择菜单 Edit: Paste Special，打开 Paste Special 对话框。
  3. 您可以根据需要选择以 Origin Graph Object 或 Picture 形式粘贴。
  4. 完成选择后，单击 确定。
注: Origin Graph Object 是一种 OLE 对象格式，它除了图片之外，还包含了 Origin 绘制这个图形时所需的数据信息，因此，你可以在 Origin 中再次打开和编辑这个图。相对的，Picture 格式只是一个简单的图片。你可以在插入的应用程序中编辑这个图片，但这些编辑和改动可能会破坏 Origin 独有的某些格式。通常来说，以 Origin Graph Object 形式插入图片比以 Picture 形式插入图片产生的文件更大。

编辑插入图形

双击以 Origin Graph Object 形式插入第三方软件中的图片，如果你的电脑上安装了 Origin，这张图将会在 Origin 中打开，你可以进行编辑。编辑完成后，选择文件：退出，退回到第三方软件。

不打开 Origin，直接在第三方软件中编辑插入图形

Origin 也提供了一个“原位激活 in place activation”的方式，可以在不打开 Origin 的情况下，直接在第三方应用程序中编辑插入的 Origin 图形对象。此编辑方法实际是在第三方软件中激活并调用了 Origin 的一些基本的图形编辑工具。

要启用 in-place activation 方式:：

  1. 从 Origin 菜单中 选择 工具：选项，然后选择 图形 选项卡。
  2. 勾选 启用OLE原位激活。
  3. 单击 确定 关闭对话框（如果系统提示你是否保存为 Origin 的启动选项，请点击 是）。
  4. 关闭 Origin。

要使用原位激活方式来编辑图形:

只需要双击插入第三方软件中的 Origin 图片（注意插入的图片不能是与原项目文件链接起来的），就可以直接编辑了。你可以双击图形打开对话框，也可以右键单击某条曲线/某个图形对象，并从弹出的右键菜单中选择你所需要的工具进行编辑。

注: 通常来讲我们并不推荐使用原位激活方式编辑插入的 Origin 图形，因为在这种模式下你所能调用的 Origin 菜单和功能是非常有限的。例如，你无法打开存储着图形原数据的工作表。在这种情况下，你可以右键单击并选择打开Origin，在 Origin 中打开图片进行更进一步的编辑。
