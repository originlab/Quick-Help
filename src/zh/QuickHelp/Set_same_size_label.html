1.127 FAQ-713 怎样使不同图层的刻度线标签字体大小一致？

Set-same-size-label

Last Update: 11/21/2018

要设置图中不同图层的刻度线标签，使得它们的字体大小一致，需进行以下操作：

  1. 首先打开菜单 工具：主题管理器 在 图形 选项卡下选择 Layer No Element Scaling 来确保操作时所见即所得。
  2. 在图中选择任意一个轴的刻度线标签设置它的字体大小。
  3. 右击此标签选择 复制格式。如果有下一级选项如 样式 或 字体，选择一个适合的。
  4. 在图层外空白处右击选择 粘贴格式到：所有：字体 以将此标签字体格式应用到所有图层。
