1.176 FAQ-843 能不能将列标签行公式 F(x)= 保存在 OGG 文件中或者在粘贴到 MS Word 的图形中？

How-to-save-column-label-with-graph

Last Update: 4/24/2019

工作簿中储存着大量的数据相关信息，在用工作簿画图的时候， 这些信息只会有一部分被储存在图中。 为了减少文件大小，会有些信息， 例如列标签行公式 F(x)=，保存在数据文件导入中的用户自定义变量等这些信息，不会包含在用以下方式生成的文件中。，

  • 将图形另存为 OGG 文件
  • 通过 Edit: Copy Page (or CTRL+C) 作为 OLE 可编辑图形对象粘贴到你的 Office 程序

如果你想在 OGG 文件或者嵌入图形中保存所有工作表中的信息，你可以将 Labtalk 系统变量 @sow 的值设为 1 （默认值=0，表示文件大小为最小值）

  • 如果只需要将此设置应用在当前 Origin 会话的所有工作表数据中，点击菜单 Window: Script Window，打开 Script Window，输入一下内容，然后按 Enter 键：
@sow=1

随后，在当前会话中 任何图形保存为 OGG 文件或作为 OLE 对象插入到另一个应用程序中时，都将包含所有工作簿数据。

  • 如果希望图形能一直 保存所有工作簿数据，你可以把 Labtalk 系统变量的设置保存在 Set System Variables 对话框中。 点击菜单 Preferences: System Variables，在 Set System Variables 对话框中，设置 sow 为 1。
