6.37 FAQ-40 当我升级服务器的许可证到新版本之后，旧版本的 Origin 还能够使用吗？

PreVersion-On-NewLic-FLEX

Last Update: 4/27/2020

可以使用。对于网络版的许可证来说，升级服务器上的许可证到更新的版本之后，旧版本的 Origin 仍然能够正常连接到服务器，继续使用（对于 Origin 8.0 版本，需要使用 SR2 及以上版本，对于 Origin 7.5 版本，需要使用 SR7 ）。这就为客户端的逐步升级提供了更为宽裕的过渡时间。例如，如果你升级了服务器上的许可证到 2017 版本，原来的 2016、2015、9.1、8.5... 这些低版本的 Origin 仍然能够继续使用。你可以在适当的时候，将这些客户端上的 Origin 逐步升级到 Origin 2017。

更详细的信息，参阅这个文档。
