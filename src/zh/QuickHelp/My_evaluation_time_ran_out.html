6.1 FAQ-2 我的试用期结束了，可是我还想继续试用Origin，怎么办？

Demo-Expired

Last Update: 7/3/2018

  • 如果你现在正在使用尚未激活的试用版（即只能使用3天的模式），请参照下面的“试用版激活步骤”来激活你的试用版以延长试用期至21天。
  • 如果你已经激活了试用版并且试用期已经结束，请联系 OriginLab 广州分公司。

试用版激活步骤

安装试用版的时候，如果你选择 Install Origin Evaluation 选项，则安装的是标准版；如果选择 Install OriginPro Evaluation 选项，则安装的是专业版。



  • 在线激活 (电脑必须有网络连接)
详见这一页
  • 离线激活 (电脑没有网络连接)
  1. 打开 Origin。
  2. 选择 My computer does not have internet access. How do I obtain an evaluation license? .
  3. 点击 OK。弹出以下信息窗口。点击 OK‘’‘。
  4. 在一台能够联网的电脑上，用浏览器打开 OriginLab 的主页 www.originlab.com。
  5. 点击主页右上角的 "Try" 按钮。
  6. 在打开的页面中，点击页面下方的 Note 2 中的 "Origin Evaluation License" 链接来申请标准版的试用许可证文件，或者，点击 "OriginPro Evaluation License" 链接来申请专业版的试用许可证文件。
  7. 在打开的 Request for 21-day Evaluation License 页面上，点击 Continue。
  8. 登陆 OriginLab 网页。如果你没有 OriginLab 的账号，请先自行创建一个账号。
  9. 检查页面上列出的 email 地址，确认无误后点击Obtain Evaluation License (这个email 地址是用来接收许可证文件的，请务必填写正确有效的地址)。
  10. 在收到含有许可证文件的邮件后，复制许可证文件的内容。
  11. 粘贴到记事本并把它保存到移动设备（例如移动硬盘或者U盘）。
  12. 在安装了 Origin 试用版的电脑上打开上一步所保存的记事本文件。
  13. 在激活窗口中选择 I already have an evaluation license - how do I enter it?
  14. 点击 OK。
  15. 复制粘贴许可证内容到激活窗口的第一个文本框中。
  16. 在第二个文本框中填入用来接收这个许可证文件的 email 地址。
  17. 点击 OK。

将会有一个信息窗弹出，告诉你试用版已经被成功激活。在收到许可证之后的21天内，你可以尽情试用 Origin 的所有功能。
