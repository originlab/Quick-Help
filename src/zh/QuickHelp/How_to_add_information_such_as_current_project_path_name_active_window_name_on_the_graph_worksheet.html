5.34 FAQ-728 如何将信息变量（如当前项目路径/名称，激活窗口的名称）插入到图形/工作表中？

Project-Additional-Information

Last Update: 7/23/2018

若您需要获取某些附加信息，例如当前激活窗口的名称，项目文件的路径和项目名称：您可在 Insert Variables 对话框的右侧面板上选择根文件夹并前往 Info 标签页即可看见。

将信息变量添加到文本对象
  1. 激活工作簿或者图形窗口，利用 Text Tool 添加文本对象。
  2. 双击该文本对象使其进入编辑模式，右键单击选择 Insert Info Variable 打开 Insert Variables 对话框，在右侧面板选择根文件夹，在左侧面板选择所需的信息变量，点击 Insert 插入该信息到文本对象中。
  3. 点击任意处退出编辑模式，那么添加的内容将显示如下：
  4. 当信息变量发生改变时，您可以激活该窗口按下 F5 键更新文本对象中的信息。

若要在多行中插入多个信息变量，请确保在每次插入前已勾选 Start New Line for Each Insert。

将信息变量添加到工作表单元格

激活某一列, 选择 Column: Set Column Values 来打开 Set Values 对话框，点击 Variable: Add Info Variable 来打开 Insert Variables 对话框，然后重复上述第 2 步和第 3步来将信息变量插入单元格中。
