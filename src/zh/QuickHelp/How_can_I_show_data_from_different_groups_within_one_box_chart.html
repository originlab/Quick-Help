3.84 FAQ-322 如何在一个箱线图中区分来自不同组的数据点？

Show-Data-from-Groups-Within-One-Box-Chart

Last Update: 7/16/2018

从 Origin 9.1 开始，你可以使用另一列数据来控制箱线图中的数据点的颜色。


在 Plot Details 对话框的 Symbol 选项卡上，在 Symbol Color 下拉列表中，选择 By Points 选项卡，从 Indexing 或 Color Mapping 列表中选择你用来控制颜色的存有分组信息数据的那一列。


注: 首先，你需要在 the Box 选项卡上设置 Type 为 Data 或者 Box+Data，以显示数据点。

Keywords:indexing, color mapping, box chart, data, groups, 颜色索引, 颜色映射, 箱形图, 数据点, 分组
