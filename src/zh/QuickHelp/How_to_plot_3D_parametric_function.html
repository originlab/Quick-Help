绘制新的函数图

Origin 支持 4 种不同的方程绘图，分别是 2D 方程， 2D 参数方程， 3D 方程，以及 3D 参数方程。这几种绘图功能都可以通过 文件：新建：函数图 菜单或者 函数图 工具栏按钮打开。 一旦选中了函数绘图功能，则会弹出一个对函数进行定义的对话框。操作示例如下动画所示：

若要进行 3D 参数方程绘图，可以通过 文件：新建：函数图：3D 参数函数图 菜单打开 创建3D参数函数图 对话框。在这个对话框中设置 网格 的维度，并为两个参数设置 刻度 ，再输入可用的 X, Y 和 Z 的值。如果有需要，再在底部表中定义方程中的相关常数。设置完成后单击 OK 按钮出图。可查阅 [3D 函数图示例]获取更多信息。

注意: 完成函数图绘制后，还可以双击图形窗口中的函数图，在 绘图细节 对话框的 函数 选项卡 中对相关参数进行重定义或修改。

往已有图形中添加函数图

激活图形窗口，并打开创建函数绘图对话框，在对话框的底部选择下拉菜单中选择 加入当前图，则定义的函数图像将被添加到已有图形中。

注意: 如果已有绘图为 2D 图，则不能向图中添加 3D 函数图，所以打开 3D 函数绘图对话框时不会显示底部的下拉菜单。
