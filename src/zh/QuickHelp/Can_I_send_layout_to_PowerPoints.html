1.166 FAQ-825 可以将布局页面(Layout)发送到PowerPoint吗？

Can_I_send_layout_to_PowerPoint

Last Update: 12/30/2018

布局 (Layout) 窗口可用于图形和工作表的排列布局，这些图形和工作表还可以在独立的子窗口中进行编辑并在布局(Layout )中刷新。但布局(Layout ) 并不是图形。因此，无法将布局(Layout ) 作为OLE对象，粘贴或发送到其他应用程序中（例如Microsoft Word 或PowerPoint），我们只能把它作为图片粘贴或发送。

如果你想把布局( Layout )作为图片粘贴到其他应用程序，你可以：

  1. 点击布局窗口，在Origin按CTRL+C ，然后在其他应用程序中按CTRL + V，将布局(Layout ) 粘贴为图片。


如果你想把布局(Layout )送到PowerPoint，也可以使用发送图片到 PowerPoint 工具:

  1. 点击布局窗口，然后单击 发送图片到 PowerPoint 按钮 ，打开 ped_pptslide 对话框。
  2. 在 插入图片作为 下，选择 图片。
  3. 最后，点击 确认 按钮，布局页面将会作为图片发送到 PowerPoint。
