4.38 FAQ-398 为什么程序会出现一个关于调用 FFT 函数的编译错误？

FFT-Compile-Error

Last Update: 7/10/2018


有时候，当您在编译一个调用了 FFT 函数的 Origin C 函数（例如我们网页上的一个例子）的过程中，可能会出现以下的编译错误：

Error, Function FFT@IAAAAAAAMAAABAAAMAAABAAA was called, but its body could not be located during linking.

这个错误说明 Origin C 找不到 FFT 函数的主体。解决这个问题需要以下两个步骤：

  1. 首先确保您的程序已包含 fft_utils.h 头文件。该头文件存放于<Origin Program Folder>\OriginC\System。
  2. 将该头文件 <Origin Program Folder>\OriginC\Originlab\fft_utils.c 添加到当前工作区域。

Keywords:workspace, fourier transform, 工作区域，傅里叶变换
