3.77 FAQ-315 如何使箱线图中的异常值对齐到一条直线上？

Align-Outliers-in-Line-Box-Chart

Last Update: 7/15/2018

从 Origin 9.1 开始，箱线图中的异常值默认画出来就是对齐的。你可以在 Plot Details 对话框中找到这个控制选项。

  1. 选择 Plot Details: Plot Properties: Box 选项卡
  2. 勾选 Outliers
  3. 勾选 Outliers in Line，使异常值点对齐排列

Keywords:outliers, box chart, in line, align, 异常值, 箱形图, 对齐
