4.6 FAQ-364 什么是命令窗口 （Command Window）？

Command-Window

Last Update: 7/28/2018

命令窗口可用于交互式运行 LabTalk 脚本和 X-Functions，可从菜单 Window: Command Window 打开。它由两个面板组成：历史面板和命令面板。历史记录面板（左侧）用于存储已发出命令的历史记录。 （存储在用户文件夹的文本文件 LTHistory.txt 中。） 命令面板（右侧）包含 Auto Complete 的功能，可更方便地输入 X-Functions：

  • 在命令面板中键入st - Auto Complete将会显示两个 X-Functions
  • 选择 stats 并按 Enter 健。
  • 按 space 健 - Auto Complete 将显示下一个可用的参数。

命令窗口的命令面板中单击鼠标右键，并选择 Preference... 。然后，您可以对 Auto Complete, Prompt and Command History 进行设置。

请参考教程 Command Window 和 X-Functions.
