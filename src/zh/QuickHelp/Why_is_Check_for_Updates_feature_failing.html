6.22 FAQ-24 为什么 Origin 的 Check for Updates 更新不成功？

check-for-updates-in-Origin

Last Update: 7/5/2018

如果你的网络使用的是一个代理服务器，Origin 的 Check for Updates 功能有可能会更新失败。为使 Origin 成功连接到官网的服务器并进行自动更新，请在 Internet Options applet 里填写正确的信息，

如果你使用的是 IE，选择菜单 Tools : Internet Options，否则，选择菜单 Start : Settings : Control Panel 并打开 Internet Options applet。

  1. 选择 Connections 选项卡。
  2. 点击 Lan Settings 按钮。
  3. 勾选 Use a proxy server for your LAN ...
  4. 输入代理服务器的 Address 和 Port（请咨询你的网络管理员以确保填写正确的信息）。
  5. 点击 OK。
  6. 点击 OK。

打开 Origin 并选择菜单 Help : Check for Updates，看是否能够成功更新。
