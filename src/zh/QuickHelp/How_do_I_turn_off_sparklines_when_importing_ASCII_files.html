2.2 FAQ-409 如何在导入 ASCII 文件时取消添加迷你图？

Turn-off-Sparklines-when-Import

Last Update: 11/12/2021

如果你导入有很多列数据的 ASCII 文件，并且在导入时为每列创建了迷你图，导入过程可能会很缓慢。你可以选择是否在导入数据的过程中添加迷你图，以控制导入数据的速度。

在 Import Wizard 对话框的 Data Columns 页面中，或者在 Single和 Multiple ASCII Import对话框的 Import Options 节点下，你可以将 Add Sparklines 设置为Yes, No 或 Yes (if less than 50 columns)。

如果使用 数据连接器 来导入 CSV 或 MS Excel 数据，您可以在导入时设置 @IMPS = 0 来禁止创建迷你图。关于改变系统变量的更多信息，请参阅 FAQ-708 如何使更改的系统变量的值永久地生效？


Keywords:slow，慢
