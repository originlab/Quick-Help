1.259 FAQ-1060 如何设置柱状图/条形图基于轴？

Last Update: 10/16/2020

Column-Start-from-Axis

从Origin 2020b开始，柱状图/条形图会基于附加线，无论此附加线是否显示出来。默认下，附加线设在Y = 0的位置。尽管您并没有把附加线显示出来，柱状图/条形图仍然会以此附加线作为基线进行绘制。这种情况与之前的Origin版本是不一样的，之前的Origin版本中，柱状图/条形图以X/Y轴作为基线进行绘制。


如果需要像之前的Origin版本那样以轴作为基准绘制柱状图/条形图，

  1. 通过菜单设置：系统变量...打开设置系统变量对话框，然后设置系统变量@CBB = 0。
  2. 双击Y轴打开坐标轴对话框，然后点击网格标签页，清除附加线下的"Y = "复选框。
