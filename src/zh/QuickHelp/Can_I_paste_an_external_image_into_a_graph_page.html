1.87 FAQ-218 如何将一幅外部图片插入 Origin 的图形窗口？

Paste-img-to-Graph

Last Update: 11/15/2018

方法一

使用菜单 编辑：粘贴 将一幅图复制粘贴到 Origin 的图形窗口中。图片将会以 EMF 或 DIB (设备无关位图 Device Independent Bitmap) 的格式粘贴进来。可以用以下 LabTalk 命令来切换这两种格式: 

system.display.pastemeta

打开 脚本窗口 (窗口：脚本窗口)，根据需要运行如下命令: 

system.display.pastemeta=1; // 以 EMF 格式粘贴
system.display.pastemeta=0; // 以 bitmap 格式粘贴

想要了解更多关于 system.display object 的使用方法，参见 system.display 文档。

方法二

使用菜单 文件“导入：从文件中插入图像 插入一幅图片到当前图形窗口。注: 这个菜单命令不受 system.display.pastemeta 的影响。

Keywords:picture, annotation, 标注, 图片

Minimum Origin Version Required: 8.5SR0
