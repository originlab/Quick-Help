2.78 FAQ-827 如何在箱线图上屏蔽数据点？

how-to-mask-data-points-on-box-chart

Last Update: 4/20/2021

Origin 2020b 及更新版本

要在箱线图上屏蔽数据点，箱线图必须显示出个别点 -- 要么是离群值或者是点的分布。

  1. 点击 Tools 工具栏 屏蔽活动绘图上的点 或者 屏蔽所有绘图上的点按钮，移动到图形上。
  2. 有必要时使用空格键，切换选择模式。
  3. 拖动鼠标捕捉你想屏蔽的数据点。被屏蔽的数据点被从箱线图上移除，并且箱体统计会重新计算。另外，源工作表或输出工作表的相关的数据也被屏蔽，被标记为当前的屏蔽颜色。
  4. 要显示屏蔽的数据点，可以点击Mask工具栏的禁用屏蔽按钮；要完全解除屏蔽，可以在禁用屏蔽显示屏蔽点时，选择Tools工具栏取消当前绘图的屏蔽或者取消所有绘图的屏蔽 按钮，拖动鼠标捕捉屏蔽点。此外，你也可以在工作表高亮屏蔽点并点击取消屏蔽范围按钮。


Origin 2020 及更早版本

更早版本不支持直接应用数据屏蔽，但是你可以尝试以下变通方法：

  1. 打开一个含有数据点的箱线图，从 Tools 工具栏中选择 Data Reader 工具 。
  2. 使用此工具选中想要屏蔽的数据点，然后 Data Info 窗口将会显示出该点的 X 和 Y 的坐标值。
  3. 在 Data Info 窗口中，右击单击然后在弹出的菜单中选择 Go To Worksheet 。
  4. 在该选中的数据在数据表中的对应行将会高亮显示。
  5. 在数据组中重新选择目标数据单元格，右键单击选中的单元格，然后从弹出的菜单中选择 Mask: Apply 。
