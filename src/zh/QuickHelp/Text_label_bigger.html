2.52 FAQ-486 怎样解决在 Origin 8.5 版本工作表中创建的文本标签在新版本中显示过大的问题？

text-label-bigger

Last Update: 7/19/2018

如果你在 Origin 8.5 版本工作表中创建了文本标签，在8.5 之后版本的 Origin 中打开时候可能会出现文字字体比 8.5 版本中大很多的问题。

要解决此问题，在Origin 8.6 及之后版本中打开这个 OPJ/OTW 文件，然后选择菜单 Window: Script Window 打开脚本对话框，在对话框中输入以下脚本。 

doc -e w
{
	page.textsizecorr=1;
	doc -uw;
}

在 Script Window 中选中此脚本并按 Enter 确认。
