1.202 FAQ-943 是否可以同时对多图层图中的所有图层一起进行设置?

Simultaneous-edits-multi-panel-graph

Last Update: 9/7/2018

从 Origin 2018b 开始， 对多图层图中的一个图层进行设置， 设置可以自动应用到图中的其他图层。 在Origin 内置的所有多面板图中， 这个并行设置的选项是默认打开的。

Plot Details Common Display Controls

这个操作是由 Plot Details 中的 Layers 选项卡中的 Common Display 设置控制的。当这个选项设置后， 我们可以用 Plot Details 对话框又或者 Style 和 Format 工具栏对图进行设置

如需了解更多， 请参考 The (Plot Details) Layers 选项卡
