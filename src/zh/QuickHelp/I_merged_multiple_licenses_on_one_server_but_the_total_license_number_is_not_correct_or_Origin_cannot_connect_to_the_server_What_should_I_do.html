6.93 FAQ-1120 在一台服务器合并了多个许可证，但总的许可证错误，或者Origin连不上服务器。该怎么做?

Merge-Lic-Server-Not-Work

Last Update: 5/9/2022

如果您有多个许可文件（不同的序列号）合并到了一个文件放到服务器上管理，您可能会遇到下面的问题：

  • 从合并的服务器读取到的可用许可证数目与总的数目对不上。
  • 使用这些序列号进行安装的Origin客户端连接不上服务器。看到的错误信息是这样的：
    Your FLEXlm license file does not match your OriginLab serial number
  • 当您检测服务器上的日志文件，您会发现类似于下面的信息：
    UNSUPPORTED: "OriginPro" (0254 3D02 14A6 DBCA )
    ClientName (No such feature exists. (-5,346)

请联系 OriginLab获取正确的许可证文件。联系的ticket中，请包含所有的序列号和相应需要合并的许可证文件。
