6.83 FAQ-1095 当激活 Origin 时看到"Product Key has already been used"的错误信息，该如何办？

Product-Key-Has-Already-Been-Used

Last Update: 4/20/2021

  • 当您激活 Origin 的时候，看到以下的错误信息，这就意味着您需要系统切换（即在旧电脑上释放产品密钥）。
Product Key has already been used

请参阅 这个页面 获取详细指引。

  • 当您激活学习版时看到下面的错误信息：
This Product Key has already been activated at computer XXX,
please deactivate it first to be able to activate this one.

请发送电子邮件到 le@originlab.com 请求系统切换。在邮件中请包含您的产品密钥以及用于获取学习版的电子邮箱地址。
