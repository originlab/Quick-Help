3.30 FAQ-258 怎样进行带有 X 误差的曲线拟合？

NLFit-X-Err

Last Update: 7/20/2018

线性拟合

如果是线性拟合， 可以使用 Analysis: Fitting: Fit Linear with X Error 菜单执行含有 X 误差棒的数据的线性拟合。


非线性拟合

如果要进行带有 X 误差的非线性拟合，需使用正交距离回归 (Orthogonal Distance Regression (ODR) 算法：

  1. 选中要进行拟合的数据或数据图。
  2. 去到菜单 Analysis: Fitting: Nonlinear Curve Fit 来打开 NLFit 对话框。
  3. 在对话框的 Settings 选项卡中，转到 Data Selection 选项，为相应的 X 或 Y 设置加权（或误差），然后选择一个拟合函数，再选择 Orthogonal Distance Regression 作为 Iteration Algorithm。

如果你的误差数据是存放在数据列中, 并且列的(绘图属性为 X 误差或 Y 误差) ,你可以选择 Instrumental 作为你的 加权方式，误差列会被自动选定。如果你的误差数据列被设定为其它的列绘图属性而不是 X/Y 误差，此时应该选择 Arbitrary 作为加权方式并选择相应的列。


隐函数曲线拟合

对于隐函数曲线拟合，可以选择菜单 Analysis: Fitting: Nonlinear Implicit Curve Fit 来打开 NLFit 对话框，然后以相同方式为相应的 X 或 Y 设置加权（或误差）。


注意: 上文中提到的所有方法都只在 OriginPro 中才支持.
