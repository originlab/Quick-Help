2.27 FAQ-454 怎样将 Origin 图形和工作表以 PostScript 文件格式打印出来？

Print-to-PostScript

Last Update: 7/12/2018

PostScript (PS) 是由 Adobe® 公司开发的文档创建的编程语言。 PostScript 打印机包括一个识别 PostScript 代码的驱动程序以及给打印机的打印指引。如果将 PostScript 代码发送到非 PostScript 打印机，则将生成一个含有 PostScript 代码文本的页面。

PostScript files (PS) 不同于 Encapsulated PostScript (EPS) 文件的是，EPS文件是假定在包含在其他文档中使用的（比如Microsoft PowerPoint® 等应用程序）。EPS 文件缺少在 PostScript 打印机上输出打印件所需的命令，这是因为通常假设 EPS 所嵌入的文件能解决此打印问题。 所以 EPS 文件不能直接在 PostScript 打印机上进行打印而 .PS 格式文件可以。

  • 要创建一个PostScript（PS）文件，需按以下步骤操作：
  1. 激活当前窗口，选择菜单 File:Print，打开 Print 对话框。
  2. 在 Name 选项下拉列表中选择一个 PostScript 打印机。（你必须先安装 PostScript 打印机驱动程序。）
  3. 选择 Print to File 选项。
  4. .单击 OK，打开 Print to File 对话框。
  5. 要修改默认扩展名，需要在 Save As Type 下拉列表中选择 All Files (*.*) 。
  6. 在 File Name 文本框里输入文件名和扩展名。
  7. 单击 Save 即可创建一个 PostScript 文件。
  • 将 PostScript 文件发送到打印机：

去到DOS提示符窗口，将当前目录更改为包含 PostScript 文件的目录（或在下一步中指定路径）

在DOS提示符窗口中输入：

COPY FileName PrinterPort

此处 FileName 是 PostScript 文件名和扩展名， PrinterPort 是 PostScript 输出设备的可用打印端口。此命令将输出一份 PostScript 文件的打印件。

例如，若要通过连接 LPT1 端口来打印 GRAPH1.PS 文件，需要在 DOS 窗口输入以下命令:

COPY GRAPH1.PS LPT1
