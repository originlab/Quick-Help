1.14 FAQ-125 能不能把三元图的坐标轴刻度设为 0 到 100？

TernaryPlot-Labels

Last Update: 11/12/2018


可以将三元图的坐标轴刻度设为 0 到 100。

如果所采用的数据已经标准化到 "0 to 100" 区间，或者符合 0 到 100 区间的标准化规则 (也就是说 x, y 和 z 的数据量总和超过 50)，则绘制三元图的数据标签将自动显示为 0 到 100。


Minimum Origin Version Required: Origin 9.0 SR0

对于显示为 0 到 1 的坐标轴刻度，可以通过在 坐标轴 对话框中的 刻度线标签 选项卡下设置 除以因子 为 0.01 使其显示为 0 到 100。

Keywords:Ternary plot, ternary, axis label, normalized data, tick label，三元图，三元相图，坐标轴标签，标准化数据，坐标轴刻度
