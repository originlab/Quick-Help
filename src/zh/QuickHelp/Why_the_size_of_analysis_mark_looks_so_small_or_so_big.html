3.51 FAQ-284 为什么分析标记看起来不是太大就是太小？

Unusual-Marker-Size

Last Update: 7/19/2018


分析标记的大小可在菜单 Data: Analysis Markers:Tiny/Small/Medium/Large 中选择。可以通过四种不同方式来定位到此选项：

  1. 在当前图层（不要选择图层）右击从快捷菜单中选择 Analysis Markers 选项。
  2. 在分析标记上右击从快捷菜单中选择 Analysis Markers 选项。
  3. 左键单击锁的图标，从快捷菜单中选择 Analysis Markers 选项。
  4. 在菜单栏选择菜单 Data: Analysis Markers 选项。


但是，在进行拟合操作后，如果拟发现分析标记在打印输出后不是太大就是太小，这也许是你的默认打印机的分辨率引起的。

首先，你需要在命令窗口中运行此 system.resolution = 代码来获得默认打印机的分辨率。得知分辨率之后，请使用下面的脚本来为页面分辨率设置合适的 x/y 值。 

page.resx= 
page.resy=
