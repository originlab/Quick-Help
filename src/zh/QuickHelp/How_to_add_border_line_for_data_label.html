1.135 FAQ-730 如何为数据标签添加边框线？

add-border-for-label

绘制了启用了数据标签的图形（如 时间线图）后，可能想要在标签的周围添加边框线。则可以按照以下步骤进行操作：

  1. 激活含有标签的图形。然后选择 格式：绘图属性... 以打开 绘图细节 对话框。
  2. 切换到 标签 选项卡，从 框架 的下拉菜单中选择 框 或 阴影 ，然后使用 边框颜色 按钮 修改边框颜色，使用 填充颜色 按钮 更改填充色。
  3. 更改 边距(字体高度的%) 组合框中的值来调整标签四周的边距。
  4. 点击 OK 按钮。

按住 Ctrl 键并单击任一个数据点以打开 绘图细节 对话框。则会对应添加这个点为特殊点。在左侧面板中选择这个特殊点，然后切换到 标签 选项卡，以上所提到的 框架, 边框颜色, 填充颜色 和 边距(字体高度的%) 四个选项的默认值都是 自动 。这表示在此特殊点中的这些选项都与整个绘图保持一致。如有需要也对这个特殊点进行自定义设置。
