6.43 FAQ-711 我的加密狗不能启动 Origin，怎么办？

Dongle-driver-error


Last Update: 10/26/2022

如果你遇到了以下的问题:

  • 当我插入加密狗并打开 Origin 的时候，只打开一个文本窗口，怎么办？
  • 当我插入加密狗的时候，我看到一个 "No driver found" 这样的错误信息，怎么办？
  • 当我插入加密狗的时候，我看到一个 "The system cannot find the file specified" 这样的错误信息，怎么办？
  • 为什么 Origin 会弹出一个“只能用3分钟”的提示窗？


注: 如果近期你的 Windows 操作系统更新过，请在更新完成之后重启电脑，看看加密狗是否能够正常工作。如果还是有问题，请按照下面的步骤更新加密狗的驱动。

新版的驱动修复了以下问题:

  • Origin(Pro) 8.x 和 9.x 如果安装在同一台电脑上，加密狗驱动可能会相互冲突
  • Origin(Pro) 8.x / 9.x 和 Windows XP（64位）、Windows Vista（32 和 64位）、Windows 7（32 和 64位）、以及 Windows 8（32 和 64 位）的兼容性问题.
  • Origin(Pro) 8.5 SR1 的加密狗驱动安装有问题
  • Origin(Pro) 7.5 / 8.x / 9 在 Windows 待机模式下的问题
  • Origin(Pro) 7.5 和 Windows Vista 以及 Windows 64 系统的兼容性问题


加密狗驱动更新步骤:

  1. 如果 Origin/Pro 还在运行，关闭它。
  2. 把加密狗拔出。
  3. 下载并保存 加密狗驱动 到电脑上。
  4. 双击文件 Install.exe，打开驱动安装界面。
  5. 勾选 Uninstall 然后点击 Begin Uninstall，卸载原来的驱动。 **为了避免原来的驱动影响新装驱动，所以请先把旧的驱动卸载掉。
  6. 卸载完成后点击 Finish。
  7. 再一次双击文件 Install.exe，打开驱动安装界面。
  8. 勾选 2 和 3 中的 USB 选项，如果使用 Parallels 软件,只需选取 Parallel 加密狗。然后点击 Begin Install。
  9. 安装完成后点击 Finish。
  10. 插入加密狗，静待 Windows 弹出 Found New Hardware Wizard 的提示信息，这表示驱动已经成功安装。
  11. 打开 Origin。
