6.9 FAQ-10 如何把 64 位 Origin 转换成 32 位 Origin?

Change-64bit-to-32bit

Last Update: 4/27/2020

从 Origin 2020版本开始，Origin不再支持32位系统。本文内容仅适用于Origin2019b及之前版本的用户。

如果您是Origin2020或以上版本的用户，并且需要在32位的计算机上运行Origin，请联系 OriginLab 获取 Origin2019b 的安装包。

在 64 位的系统上，Origin 提供了 32 位和 64 位应用程序这两种选择。如果你安装了 64 位的 Origin 但是想要安装 32 位 Origin，可以参照以下步骤将它转成 32 位的应用程序：

  1. 若 Origin 还在运行状态，先关闭它。
  2. 在 Windows 7 或者 Vista 系统中，
    • 点击 Windows 的"开始"菜单并选择 Programs | OriginLab Origin(Pro) | Repair or Uninstall Origin(Pro)。Origin 的安装向导将会打开。
    在 Windows 8 系统中，
    • 敲击一下键盘上的 Windows 键。
    • 右键单击桌面的任一空白处。
    • 点击右下角的 All Apps 按钮。
    • 选择 Repair or Uninstall Origin(Pro)，打开 Origin 的安装向导。
    在 Windows 10 系统中，
    • 点击 Windows 的"开始"菜单并选择 All apps | OriginLab Origin(Pro) | Repair or Uninstall Origin(Pro)。Origin 的安装向导将会打开。
  3. 选择 Modify/Change 选项并点击 Next。
  4. 在弹出的"是否更改序列号？"窗口中点击 Yes 确认。
  5. 在 Customer Information 页面上会显示你的序列号。点击 Next。
  6. 在 Choose 32 bit, 64 bit, or both 页面上，根据自身需要选择 32 bit Origin 或者 Both 32 bit and 64 bit Origin。如果选择了 Both 32 bit and 64 bit Origin，在 Windows 的 Start|Programs | OriginLab 菜单下面将会有32 和 64 位两套图标。
  7. 点击 Next 并依指示完成安装。
