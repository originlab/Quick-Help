1.269 FAQ-1109 如何在散点图中创建自定义符号?

Create-User-Defined-Symbol

Last Update: 5/9/2022

如果您在内置的符号列表（绘图细节 对话框 > 符号 标签页）中找不到您想要符号，您可以用其它工具创建您自己的符号，或者从网络上搜索您想要的符号，然后在 Origin 中使用。

方法1：用其它工具创建符号然后导入 Origin

  1. 使用画图软件创建您自己的符号。
  2. 保存为 *.SVG 或 *.BMP 文件。
  3. 把自定义符号导入 Origin，步骤为：
    1. 选择菜单 设置: 选项。
    2. 在 选项 对话框的图形标签卡，点空白格子，然后按 Ctrl + F 打开文件浏览器，然后选择保存好的符号文件。
    3. 点击 确定 关闭对话框。
  4. 现在您可以从 绘图细节 对话框的 符号 标签卡中选择这个自定义符号，也可以通过迷你工具栏的 图形符号 按钮选择。

方法2：从网络上搜索并在 Origin 中使用

  1. 从网络上搜索所需的符号。
  2. 右击此符号图像，然后从弹出菜单中选择 复制图像。
  3. 在 Origin 中，双击散点图打开 绘图细节 对话框。
  4. 在 符号 标签卡，勾上 自定义结构，然后选择 用户自定义符号。您会看到 从剪贴板添加 显示您刚才从网络上复制的符号图像。点击选择这个符号。
  5. 点击 确定 按钮把它应用到绘图。
