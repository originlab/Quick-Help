3.36 FAQ-265 怎样自定义一个隐函数？

Implicit-Function

Last Update: 7/21/2018

如果我们想在 隐函数拟合 中使用自定义函数，那么这个函数必须被放到 Implicit 的函数类别

选择菜单 Tools: Fitting Function Builder 来打开 Fitting Function Builder 对话框，在 Name and Type 页面，在 Function Model 选项下拉列表中选择 Implicit ,然后你就可以在此对话框中继续创建隐函数。

Keywords:implicit function, fit，隐函数，拟合

Minimum Origin Version Required: 9.0SR0
