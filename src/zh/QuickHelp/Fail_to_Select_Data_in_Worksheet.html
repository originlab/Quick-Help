5.63 FAQ-924 为什么在工作表中无法选中一列或者范围内的数据？

Fail-to-Select-Data-in-Worksheet

Last Update: 7/12/2018

在极少数情况下，用户会发现其无法选中工作表中的任何数据，其效果如下所示：

  • 在工作表列的短名称处悬停鼠标，光标无法变为选择模式 (光标不会变为向下的箭头)。
  • 单击列标题不会选择该列。
  • 不能在工作表中选择单元格区域。

如果出现这种情况，先激活出现问题的工作表，打开脚本窗口 (Window: Script Window)，输入以下代码并按 Enter: 

worksheet -es

从 Origin 2018b 版本开始，将在加载已保存的文件时自动运行此指令，可以有效防止此类问题的发生。

Keywords:selection, mouse, dragging，选择，鼠标，拖动
