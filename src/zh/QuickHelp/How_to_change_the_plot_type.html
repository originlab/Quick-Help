1.84 FAQ-214 怎么改变绘图类型？

Change-Plot-Type

Last Update: 11/15/2018

  • 对于线图、散点图、点线图和柱状/条形图，你可以选中数据曲线，然后在工具栏上点选别的绘图按钮来直接改变图线的类型。
  • 对于线图、散点图、点线图、柱状/条形图，你还可以右键单击数据曲线，在右键菜单中选择 绘图更改为，在弹出的次级菜单中选择你想要的其他绘图类型。
  • 图表绘制 对话框给我们提供了更多的可以转换的绘图类型：
    • 在下方面板的 绘图列表 列表中选中你想要转换的数据曲线
    • 在中间面板的 绘图类型 列表中选择你想要的绘图类型
    • 点击 替换 按钮
    • 点击 确定，图上的数据曲线就会转换成你想要的类型。

注: 如果多条曲线被组合起来，上述操作会改变整组曲线的类型。如果多条曲线没有被组合起来，你可以单击选中它，然后使用工具栏上的绘图按钮或者 绘图更改为 右键菜单来改变单条曲线的类型。
