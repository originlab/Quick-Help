1.82 FAQ-212 Origin 图形的数据点的符号可以在其他软件里调用吗？

OriginSymbol-in-OtherAppli

Last Update: 11/15/2018


Origin 在安装时，会在 Windows \FONTS 文件夹下安装一组名为 Origin 的全真字体（True Type font）。这组字体包括了你在画点线图的时候在 绘图细节 对话框的 符号 选项卡的符号下拉列表中列出的所有的符号形状，以及在 线条 选项卡上的 样式 选项的下拉菜单中列出的所有曲线的样式。

我们都知道，在默认生成的图例中，是根据曲线实际的样式来标识这条曲线的。你也可以手动添加文本标签，插入 Origin 符号，来更灵活的表达你的意图。双击这个文本标签进入编辑模式后，右键单击并在弹出菜单中选择 符号表，在打开的对话框的下方选择 Origin 字体，在上方表格中找到并插入你需要的符号。

你也可以在其他的软件中使用 Origin 字体所包含的这些符号形状和曲线样式。例如，如果在 Microsoft Word 文档中选择菜单 Insert: Symbol，你会在 Font 下拉列表中找到 Origin 字体。

如果 Origin 字体没有自动安装，你可以在 Origin 的安装目录下找到 Origin.ttf 文件，双击它，用 Windows Font Viewer 打开。然后点击 Install 按钮来安装。
