3.162 FAQ-1057 如何链接本地 CSS 和其他文件到 HTML 报告？

link-css-html-reports

Last Update: 11/4/2020

当使用完整的文件路径链接到本地文件 (*.css, *.js, 图像文件) 时， 您可能会遇到附件无法加载到 HTML 报表中的情况。例如，假定无法加载以下样式表： 

<link rel="stylesheet" href="C:\path\to\files\my_styles.css">

如果您确定路径正确，请尝试添加 file:/// 到路径。例如： 

<link rel="stylesheet" href="file:///C:\path\to\files\my_styles.css">

这样可以解决附件加载问题。如果仍然无法解决，请先确定您是否有权访问该文件。
