1.217 FAQ-960 如何计算 2D 三元相图中图线下面的面积？

Area-Calculation-in-2D-Ternary

Last Update: 11/5/2018

如需计算 2D 三元相图中图线下面的面积(上图中的阴影区域)，

  1. 去到原始数据，X，Y和Z列，选中X和Y，然后选择分析：数学：积分菜单去打开积分工具。
  2. 保持默认设置，点击 OK 按钮去做积分。您可以在结果日志和结果列 “Integrated Y1” 的注释单元格中得到积分面积值。
  3. 将得到的面积值乘以 sqrt(3)/2。这个结果就是 2D 三元相图中的曲线以下的面积值。

Keywords:面积，积分，积分值，三元坐标系，area, integrate, integrated value, ternary coordinate
