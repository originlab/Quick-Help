6.90 FAQ-1116 在安装 Origin 时报 "Location is not available" 的错误，如何解决?

Access-Is-Denied

Last Update: 5/10/2022

如果您在安装 Origin 时，看到以下的 "Location is not available" 错误信息：

C:\Users\username\AppData\Roaming\Microsoft\Windows\StartMenu\Programs\OriginLab Origin #### is not accessible.

Access is denied.

这是由于您使用了非管理员帐户安装 Origin，而且只为当前用户安装。也就是， It is because you installed Origin with a non-administration account and chose to install it for current user only. That is,

  1. 用非管理员帐户登陆Windows。
  2. 右击 Origin 的 setup.exe，然后选择 以管理员身份运行，登陆一个管理员帐户。
  3. 按步骤进行安装向导。到了选择 所有用户或者当前用户？' 这个步骤，选择了 只为当前用户。
这就是上面问题出现的步骤。这时，安装程序把管理员帐户认为"当前用户"，而不是您现在登陆的帐户。因此快捷路径会在管理员帐户的文件夹C:\Users\<AdminUserName>\AppData\Roaming\Microsoft\Windows\StartMenu\Programs\OriginLab Origin2022下面创建，而当前非管理员帐户是不能打开这个路径的。


解决方法：

  1. 双击 Origin 的 setup.exe 来开始 Origin 的安装向导。
  2. 选择 修复/升级.
  3. 到了选择 所有用户或者当前用户？' 这个步骤，选择了 所有用户。

Keywords:location not available, not accessible, non admin account, access deny, Incorrect logged on user，位置不可用，不可访问，非管理员帐户，访问拒绝，登陆用户错误
