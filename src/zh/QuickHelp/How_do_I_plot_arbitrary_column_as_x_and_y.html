4.22 FAQ-381 如何以任意列作为 X 和 Y 绘制图形？

Plotxy-PlotDesignation

Last Update: 7/7/2018

在 Origin 中每一列都有其相对应的绘图设定来帮助我们方便快捷的绘制图形。当您没有选中任何列而进行绘图时，Plot Setup 对话框将会弹出。在 Plot Setup 对话框中您可以重新选择哪一列作为 X 和哪一列作为 Y，并且该操作将不受工作表中的原绘图设定的影响。

另外，Origin 还提供了 plotxy X-Function 帮助您以任意列作为 X 和 Y 绘图。您可以使用 LabTalk 脚本调用此 X-Function。在 Command Window (可通过选择菜单 Window: Command Window 打开) 中，键入 plotxy 并按下空格键，接着就可以利用自动补全功能设置函数的各种参数。

例如: 

// To create a new scatter plot:
plotxy iy:=(col(1), col(2)) plot:=201;

// To create a new layer in the active graph:
plotxy [Book1]Sheet1!(A,C) ogl:=new;


Keywords:graph, layer, create, 图形，图层，创建
