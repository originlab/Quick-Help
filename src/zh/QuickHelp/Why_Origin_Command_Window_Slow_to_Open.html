4.57 FAQ-796 为什么 Origin 的 Command Window 打开很慢？

Origin_Command_Window_Slow_Open

Last Update: 7/15/2018

Origin 的 Command Window 储存了之前执行过的命令。这些命令的历史记录保存于一个在用户文件夹的文本文件 (LTHistory.txt)。

虽然这些命令的历史记录可用于查找或重新运行之前执行过的命令，但是这些历史记录也会随着时间地推移而慢慢地累积。这可能会导致打开 Command Window 时产生较长的延迟。如果出现这种情况，解决的办法是删除或者减小 LTHistory.txt 文件的大小。您也可以通过在 Command Window 的历史面板（左侧面板）中点击右键并选择 Clear 来清除历史记录。

如果您经常在命令行中运行大段的脚本，那么最好在 Script Window 中执行（Script Window 不储存命令的历史记录）。
