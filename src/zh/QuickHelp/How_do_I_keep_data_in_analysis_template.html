3.42 FAQ-273 怎样在分析模板中保留某些数据？

keep-data-in-template


Last Update: 7/21/2018


默认情况下，将工作表保存为 分析模板 时，将清除工作表中所有数据。因此，为了在模板中保留一些数据，你可以：

  1. 选中要保留的单元格。
  2. 在单元格上右击并在快捷菜单中选择 Format Cells... 选项。
  3. 这将打开 Format Cells 对话框，在对话框中勾选 Disable Editing 复选框。
  4. 转到菜单 File:Save Workbook as Analysis Template 将当前工作表保存为分析模板以便日后分析使用。
