3.139 FAQ-826 如何将图形中新报告表 Results ± Errors 转换为旧报告表 Results 和 Errors 显示在两列的形式？

How_to_revert_to_old_report_table

Last Update: 7/30/2018


在 Origin 2016 以及以后的版本，默认情况下，在拟合分析后 Results ± Errors 会显示在 报告表中，而在旧版本中，Results 和 Errors 是分别显示在两个列中的。

注: 有时，你可能会不想以正负号来连接参数值及其误差，那么你可以将新表格还原为旧表格的显示形式，则可以在两列中分别显示参数值和误差。


如何将新表格还原为旧表格的显示形式...
方法 1:

  1. 右键单击在图形中的新报告表，然后选择 Quantities in Table…，打开 Quantities in Table 对话框。
  2. 选中所有项，然后点击 Remove 按钮。
  3. 点击 OK 按钮，则可以恢复到旧报告表的形式。


方法 2:

  1. 重新打开 NLFit 对话框，选择 Settings选项卡下的 Output 子选项卡，删除 Quantities in Table 框中的内容。
  2. 点击 Fit 按钮，再次拟合，则可恢复到旧报告表的形式。


如果想在所有曲线拟合过程中，将新报告表转切换为旧报告表，你可以：

  1. 在新打开的 NLFit 中，选择 Settings选项卡下的 Output 子选项卡，删除 Quantities in Table 框中的内容。
  2. 单击 Dialog Theme 旁边的三角按钮，将这个设置保存为 <默认>.


这样，此设置就会保存到 NLFit 对话框的默认主题中，并且在之后所有的拟合中运用。

如果想在所有曲线拟合过程中，取消生成报告表，你可以：

  1. 在新打开的 NLFit 中，选择 Settings选项卡下的 Output 子选项卡，在 Result Table 下拉列表中选择 None。
  2. 单击 Dialog Theme 旁边的三角按钮，将这个设置保存为 <默认>.
