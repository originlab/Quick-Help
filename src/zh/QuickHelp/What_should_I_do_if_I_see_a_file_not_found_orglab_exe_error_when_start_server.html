6.101 FAQ-1139 启动 FLEXnet 服务器时提示"File not found, orglab.exe"错误，如何解决？

Orglab-File-Not-Found

Last Update: 10/26/2022

如果启动许可证服务器失败，并且在日志文件中有如下错误信息：

9:42:08 (lmgrd) License server manager (lmgrd) startup failed:
9:42:08 (lmgrd) File not found, orglab.exe


请检查 orglab.exe 与 lmgrd.exe 是否在同一个文件夹。在 LMTools 的 Config Services 选项卡里可以找到 lmgrd.exe 所在目录。

如果不是

  1. 请从这里下载 orglab.exe。
  2. 解压，然后把 orglab.exe 放到与 lmgrd.exe 同一个目录下。
  3. 重启 FLEXnet 服务器。


如果已经有 orglab.exe，但 flexlm 服务器找不到

  1. 请用文本编辑器打开 Origin 的许可文件。
  2. 把 orglab.exe 的路径添加到第二行， 添加后，许可文件的内容类似下面： 
    SERVER_NAME HOSTID
VENDOR orglab "C:\OriginLicenseServer\Servers\orglab.exe"
  3. 重启 FLEXnet 服务器。
