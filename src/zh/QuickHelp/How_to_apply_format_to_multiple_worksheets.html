2.89 FAQ-913 如何将格式应用到多个工作表中？

Format-Multi-Worksheets

Last Update: 4/24/2019

用户可以将工作表的格式保存为工作表主题文件，并将其应用到其它的工作表中。

默认情况下，Origin 禁用了 save worksheet format as a theme 的功能。若要开启这个功能，首先选择 Preferences: System Variables... 菜单，在打开的对话框中双击 Variable 列下面的空白单元格，然后输入 WEF，并将值设为 1。

自定义一个工作表的样式，比如，修改行高、列宽等。右击工作表中的空白区域，从弹出的菜单中选择 Edit Format...。在打开的 Worksheet Theme Editor 对话框中，点击 Save As... 按钮以保存这个主题。

要将保存的主题应用于多个工作表，例如当前文件夹中的所有工作表，点击 Preferences: Theme Organizer... 菜单，切换到 Worksheet 选项卡，选择刚刚保存的主题。在 Apply Theme To 下拉列表中，选择 Sheets in Folder。单击 Apply Now 将格式应用于文件夹中的所有工作表。
