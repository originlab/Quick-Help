1.86 FAQ-217 在打印时，如何根据国家标准来设置图片尺寸？

Graph-Page-Size-Country-Dependent

Last Update: 11/15/2018

Origin 可以根据不同国家/地区的标准，设置打印时的默认图片尺寸。标准如下: 美国、加拿大和墨西哥 - 宽度 6400 pixel(10.67 inch)，高度 4800 pixel (8 inch)；在这三个国家以外的地区 - 宽度 6779 pixel (22.6 厘米)，高度 4760 Pixel (15.866 厘米)。

设置方法如下:

  1. 在Windows 系统中设置你的地理位置 (Control Panel\Clock, Language, and Region)。我们假设你选择的当前位置是 Unite State。
  2. 打开 Origin，画一幅图。在图形窗口的灰色区域右键单击，勾选菜单 调整页面以适应打印机。此选项会根据你在系统中设置的国家/地区区域来自动调整图形的打印尺寸。
  3. 要检验图形是否已经被设置为正确的打印尺寸，选择菜单 窗口：脚本窗口，输入以下 LabTalk 命令。全选并回车来运行。 
    page.width=;
page.height=;

    如果你看到如下的返回结果，证明设置是正确的。关闭 脚本窗口。 

    page.width=6400
page.height=4800
  4. 选中你想要打印的图形，选择菜单 文件：打印 来打印。


注:国家/地区标准只有在设置了 调整页面以适应打印机 的图形窗口中才会起作用。其他的图如果没有勾选这个选项，则不会根据国家标准来设置图片的打印尺寸，而是由绘图所用的模板来决定图片的大小。
