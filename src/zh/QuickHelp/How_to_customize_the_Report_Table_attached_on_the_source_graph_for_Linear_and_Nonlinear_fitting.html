3.132 FAQ-770 如何自定义线性拟合(Linear Fit) 和非线性拟合 (Nonlinear fitting) 后在源图形上输出的结果表？

Custom-ReportTable-LR-NLFit

Last Update: 7/30/2018

你可以自定义报告表格式，并将其另存为模板。 Linear Fit 和 Nonlinear Curve Fit 对话框中均有加载此模板的选项，并且可以指定在结果表中的输出量。如果想了解更详细信息，请参考这页

Minimum Origin Version Required: Origin 2016 SR0

Keywords:report table, customize，结果表，自定义
