2.80 FAQ-847 为什么在插入或删除列时工作表中的列公式不能更新？

Why-column-formula-not-update

Last Update: 11/7/2018

Origin 2017 开始，软件引入了一种新的、简洁的电子表格单元格表示法。当启用这种表示法时，工作表的左上角显示 "A-Z" 图标 ，列的 短名称被锁定 且不可更改。 因此，当插入或删除列时，将自动更改偏移列的短名称以保持列按照字母顺序从左向右排列。

当电子表格单元格表示法启用时，在插入、删除或者移动列或表时，列中的公式正常都会自动更新，但如果存在以下情况， 列公式将会无法自动更新：

  • 公式包括传统的列表示法，如 col(A) 或 wcol(1)。
  • 公式包括表的索引引用 (例如， 1!A 不能自动更新，应当考虑使用 Sheet1!A 表示)。
  • 使用 执行公式前的运行脚本 来定义列公式。
  • 公式含有 { } 大括号。

有关使用 Set Values 对话框定义列公式的内容，请参阅设置值 - 快速入门。
