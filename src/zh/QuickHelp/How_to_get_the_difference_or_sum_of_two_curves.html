2.96 FAQ-961 如何对两条曲线求和或求差?

Math-of-two-curves

Last Update: 9/13/2018

如果想对两条曲线求差或者求和

  1. 激活两条曲线所在的图形窗口， 选择菜单 Analysis: Mathematics: Simple Curve Math... 打开 Simple Curve Math 对话框
  2. 把其中一条曲线设置为 Input1 (被减数/被加数) 把 Operator 设为 Subtract/Add, Operand 设为 Reference Data, 然后把第二条曲线设为 Reference Data (减数/加数).
  3. Output 设为<new>: New XY 以储存结果. 点选 OK 键得到结果.

Keywords:difference, sum, subtract a plot, add a plot, math on curves, 求和，求差， 减去曲线，加曲线， 曲线加减
