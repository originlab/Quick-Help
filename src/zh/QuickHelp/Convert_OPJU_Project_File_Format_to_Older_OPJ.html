2.91 FAQ-927 是否有方法可以将新的 OPJU 文件格式转换为以前的 OPJ 格式？

Convert-OPJU-Project-File-Format-to-Older-OPJ

Last Update: 4/14/2023

Origin 2018 引入了一种新的 Unicode-标准 (UTF-8) 的工程文件类型 -- OPJU 文件。这种类型的文件不能够被之前旧版本的 Origin 所打开。 (需要指出的是，Origin 2018 到 Origin 2022b版本均可以将文件保存为之前的 OPJ 格式)。Origin 2023 及更高版本中的新功能需要 OPJU 格式，因此无法另存为 OPJ 格式。

如果目前使用的旧版本的 Origin 无法打开 OPJU 文件，则可以按以下方法进行转换：

  1. 访问 OriginLab 网站上的这个网页 下载 Origin Viewer。
  2. 解压下载的 Origin Viewer。
  3. 双击 EXE 文件运行Viewer (不需要安装)，然后在菜单中选择 File: Save Project As 将工程文件另存为 OPJ 文件。


关于 Origin Viewer的更详细信息，请查看 OriginLab 网站上的相关网页 。
