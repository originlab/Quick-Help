4.42 FAQ-404 在 Origin 中如何运行 Python?


Last Update: 7/13/2018

use-python-in-Origin

Origin 提供了内置的 Python 开发环境使得您可以在 Origin 中运行 Python。Origin 支持从 Command Window 运行 Python 命令行, 或者执行 .py 文件。同时，Origin 也提供了 PyOrigin 模块使得您可以从 Python 中访问 Origin。

如果您想学习在 Origin 中如何运行 Python，您可以参阅 LabTalk 文档中的 Working with Python 的章节。

Keywords:Python, Script, PyOrigin, 脚本
