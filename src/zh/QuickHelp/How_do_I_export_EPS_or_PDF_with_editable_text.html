1.153 FAQ-795 怎样导出 EPS 或 PDF 文件使得图片中文本可编辑？

Exp-Editable-Text-EPS

Last Update: 4/17/2024

一些科技期刊偏爱 EPS 和 PDF 格式的图形文件，因为这样的文件后期具有可编辑性。要导出的 EPS 或 PDF 图片中的文本是可编辑的，需在 expGraph 对话框中进行如下设置：

  • 导出图形为 EPS 文件: 展开到 图像设置：字体 选项并选择 使用内置字体 单选按钮。
  • 导出图形为 PDF 文件: 展开到 图像设置：字体 选项并选择 内嵌 单选按钮。


有时如果 Origin 导出的 EPS 或 PDF 文件中出现奇怪的问题， 我们可以考虑将图形导出为 EMF 或 SVG 图片文件，然后使用 Adobe Illustrator 打开它们并另存为 EPS / PDF 文件。
