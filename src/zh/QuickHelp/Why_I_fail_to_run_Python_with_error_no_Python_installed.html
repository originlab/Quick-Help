4.60 FAQ-842 为什么会运行 Python 失败并出现错误信息“没有安装 Python”？

Why_I_fail_to_run_Python_with_error_no_Python_installed

Last Update: 7/15/2018

在 Origin 2017，Python Console 依赖于某些 VC++ 2015 的 DLL。通常当您安装 Origin 2017 时这些 DLL 也会被同步安装于您的计算机并更新到最新的版本。

然而，在您安装 Origin 2017 时可能您的计算机就已经存在了某些同样依赖于这些 DLL 的应用程序。那么这些 DLL 就会由于许可冲突而没有办法更新到最新的版本。因此当您在 Origin 2017 中运行 Python 的时候就会看见错误信息“没有安装 Python”。

不过，您可以尝试以下的方法修复这个问题：

  1. 打开 Windows Control Panel 然后选择 Programs and Features。
  2. 在列表中找到 Microsoft Visual C++ 2015 Redistributable(x64)-14.0.023026 和 Microsoft Visual C++ 2015 Redistributable(x86)-14.0.023026。
  3. 双击该 DLL 并打开对话框。
  4. 点击 Repair 按钮。
  5. 重新启动 Origin。
