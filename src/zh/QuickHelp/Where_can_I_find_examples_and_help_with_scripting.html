4.12 FAQ-371 我可以在哪里找到编写脚本的示例及帮助文档？

LT-Script-Example

Last Update: 7/5/2018

在<Origin Installation Directory> / Samples / LabTalk Script Examples/ 文件夹中有许多 LabTalk 脚本文件（*.ogs 文件）。您可以通过 Code Builder（可通过选择菜单View: Code Builder打开）打开这些文件并查看脚本。


您可以按照以下步骤在 Command Window 中运行这些示例

  1. 使用 cdX-Function 将当前工作目录定位到 LabTalk 脚本文件所在的文件夹，如: 
    cd 2;
  2. 通过 dirX-Function 列出位于该目录下的所有文件: 
    dir;
  3. (可选步骤）使用 ed.open(fileName.ogs) 在 Code Builder 中打开文件。 
    ed.open(embed_graphs.ogs);
  4. 要运行某个示例，只需从步骤 2 的列表中选择一个文件并在 Command Window 中键入该文件名，然后按 Enter 键，例如：
    embed_graphs;

其实LabTalk Scripting Guide， LabTalk Language Reference和 Script Examples对 LabTalk 初学者而言都很有帮助。学习过程中可多参考。
