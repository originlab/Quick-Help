3.82 FAQ-320 如何绘制 3D 直方图？

Plot-3D-Histogram

Last Update: 7/16/2018

你可以使用 2D Binning 工具绘制3D直方图

  1. 你的工作表中必须包含代表 X 和 Y 变量的两列数据，从菜单中选择 Statistics: Descriptive Statistics: 2D Frequency Counts/Binning
  2. 在打开的对话框中，勾选 Output Matrix
  3. 展开 Matrix Plots 分支，勾选 3D Bars


了解更多信息，你还可以查看官网的 graph gallery 中的这个示例。
