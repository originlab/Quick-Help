6.71 FAQ-916 当我在 Red Hat Enterprise Linux 系统下安装 FLEXlm 许可证服务器的时候，我看到一个错误信息 "bad ELF interpreter"。怎么办？

Bad-ELF-Interpreter-Err-RedHat-Linux-Concurrent

Last Update: 7/12/2018

Flexnet 服务器是一个 32 位的应用程序，因此它需要安装 32 位的操作系统库 Operating System libraries。但是，64 位的 Red Hat Enterprise Linux 操作系统并不会默认安装 32 位的操作系统库。所以，如果你是在 64 位的 Red Hat Enterprise Linux 系统下安装 Flexnet 服务器，你可能会遇到如下的错误信息:

/lib/ld-lsb.so.3: bad ELF interpreter: No such file or directory

要解决这个问题，请在根目录下运行以下命令来安装 redhat-lsb library: 

 yum -y install redhat-lsb
