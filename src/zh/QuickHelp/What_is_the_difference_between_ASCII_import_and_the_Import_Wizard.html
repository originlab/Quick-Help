2.4 FAQ-411 导入 ASCII 工具 (ASCII Import) 和导入向导 (Import Wizard) 有什么不同呢？

ASCIIImport-ImpWiz

Last Update: 4/24/2019


导入向导 Import Wizard 支持导入 ASCII 以及简单二进制（包括可选的文件头部分以及随后的二进制结构）数据文件。导入向导提供各种导入选项，并提供导入预览，使得你可以随时调整设置。 导入向导还可以通过剪贴板 Clipboard 将数据导入到 Origin。通过导入向导可为每种文件类型创建自定义的过滤器。 创建过滤器后，通过拖放操作可以将后续文件轻松导入到 Origin 工作区中。


Data: Import from File: Single ASCII 菜单工具适用于导入数据结构不太复杂的 ASCII 文件。如果你的文件中包含少量标题行 （文件的简短描述，列的名称和单位），并且数据列以有序的方式分隔，那么你可以选用这个菜单项以导入文件。你还可以选择 Show Options Dialog 复选框，打开 impASC 对话框，做一些和导入向导中类似的设置，但此对话框不提供预览功能。

Data: Import from File: Multiple ASCII 菜单工具的工作方式和 Single ASCII 的相同，但其中的 Show Options Dialog 复选框是默认选中的。

通过 Single 和 Multiple ASCII 导入选项对话框，你可以将导入设置保存到工作表或主题文件中。

点击在对话框主题旁边的三角形按钮，在弹出菜单中选择 Save to <Sheet> 项，则可将设置保存到工作表中。 然后，你可以将工作簿另存为模板，即使在没有打开选项对话框的情况下，Origin 也会使用已保存的设置导入 ASCII 文件。


如果 将对话框设置保存为主题, 那么这些主题文件将独立于任何模板保存到磁盘中。 你可以保存多个这样的主题文件，并在导入文件时，从 impASC 对话框中的弹出菜单（点击三角形按钮），为当前文件选择适合的导入主题。 你还可以从 Data - Recent used items 菜单中选用这些主题。
