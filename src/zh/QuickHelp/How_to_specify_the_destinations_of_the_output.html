3.110 FAQ-648 如何指定结果的输出位置？

destinations-output

Last Update: 7/28/2018


如要指定具体怎么把结果输出到工作簿，你可以在 books 旁边的下拉列表中选择以下项目：

  • <none>: 不要输出报告工作表。
  • <auto>: 系统自动选择的工作簿。
  • <source>: 源数据工作簿。
  • <new>: 新工作簿
  • <existing>: 指定的现有工作簿。

如果选择<new>作为目的地，你可以在 BookName 框中定义名称。 更重要的是，我们也可以把结果输出到 Result Log, Script Window, Notes Window 中。
