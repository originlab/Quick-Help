6.81 FAQ-1092 激活 Origin 时看到"Learning Edition data is not right"的错误信息，该怎么办？

Learning-Edition-Data-Is-Not-Right

Last Update: 5/10/2021

激活 Origin 学习版时，看到错误信息 Learning Edition data is not right：

请检测您安装 学习版 时使用的序列号是否正确：

  • 您可以选择菜单 帮助：关于 Origin 来查看所用的序列号。请检测序列号是否与您收到的邮件里的一致，如果不一致，请按这个帮助页面的指引进行更正。
  • 如果您曾多次申请学习版，每次收到的序列号有可能不一样。请使用最后一次收到的邮件里的序列号来安装。

如果使用的序列号正确但仍有这个错误信息，请发电子邮件到 le@originlab.com。请在邮件里附上您的产品密钥和用于申请学习怎么的电子邮箱地址。
