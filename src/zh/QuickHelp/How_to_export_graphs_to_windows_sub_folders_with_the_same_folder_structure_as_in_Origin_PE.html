1.172 FAQ-836 如何把图形导出到与Origin 项目管理器（Project Explorer）有相同结构的Windows子文件夹中？

Export-Graphs-in-SubFolders

Last Update: 9/7/2018

假设在Origin项目中有两个文件夹，folderA 和 folderB。两个文件夹中均有一个图形，分别为 graphA 和 graphB。 有时候，我们会有需求根据项目文件的结构导出图形，获得如下所示内容：

--<project folder>/graphs/folderA/graphA.jpg
--<project folder>/graphs/folderB/graphB.jpg

在Origin 2017 后的版本，导出图形工具引入了新关键词 <PE Path>，为这种情况提供了完美的解决方案。<PE Path> 表示根据 Origin 项目管理器（Project Explorer）中的文件夹结构，将图形导出到 windows 子文件夹中。 以上述情况为例，按照以下步骤可以把图形导出到对应子文件夹中：

  1. 当图形窗口在激活状态，从菜单中选择 File: Export Graphs...，打开expGraph 对话框。
  2. 从 Path 下拉列表中选择 <Project Folder><PE Path>。
  3. 根据需要指定其他设置。点击 OK 导出图形。
