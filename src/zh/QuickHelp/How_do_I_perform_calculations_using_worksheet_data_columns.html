3.57 FAQ-292 怎样对工作簿中的数据列进行计算？

Math-Between-Cols

Last Update: 7/21/2018

首先打开 Set Values 对话框，在对话框中选择 Column:Set Column Values... 选项，或者选择一整列并右击从弹出的快捷菜单中选择 Set Column Values... 选项。

在此对话框中有示例公式和多种内置数据函数。你可以通过点击 Search and Insert Functions 按钮来打开 Search and Insert Functions 对话框，在此对话框中输入关键字来快速找到所需函数。

工作簿中每一列都可以带有与此列相关的公式。要显示其他列的公式，单击 和 按钮来滚动显示每一列的公式。
