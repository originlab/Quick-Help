6.74 FAQ-1031 启动 FLEXnet 服务器时看到“orglab exited with status 53” 错误，怎么办？

FLEXnet-Server-VC2012

Last Update: 10/11/2023

版本 11.19 或更高版本的 FLEXnet 服务器需要 Microsoft Visual C ++ 2015 组件包。如果尚未安装，则在尝试启动FLEXnet服务器时，您将在日志文件中看到以下错误消息： 

orglab exited with status 53 (this status is reserved but will never be generated)

或者： 

9:12:34 (lmgrd) orglab exited with status 255 ()
9:12:34 (lmgrd) Since this is an unknown status, license server 
9:12:34 (lmgrd) manager (lmgrd) will attempt to re-start the vendor daemon.


要解决这个问题

  • 如果您使用的是 FLEXnet11.16 或更旧版本，请升级到版本 11.19 或更高版本。 您随时可以从这里下载最新版本。
  • 如果您运行最新版本，但仍然看到此错误消息，则需要手动下载并安装 Microsoft Visual C++ 2015-2019 Redistributable 组件包：
  • 32 位系统，请下载安装 vcredist_x86-2015-2019.exe
  • 64位系统，请下载安装 vcredist_x64-2015-2019.exe
