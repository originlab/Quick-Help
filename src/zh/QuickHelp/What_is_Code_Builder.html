4.4 FAQ-362 什么是代码生成器 （Code Builder）？

CodeBuilder

Last Update: 7/31/2018

Code Builder是 Origin 的集成开发环境（IDE），用于构建和调试用 LabTalk 或 Origin C 创建的 Origin 代码。在带有颜色编码语法的工作区，可编辑多个文件，除了此以外，Code Builder 还有 Output, Watch, Local Variables, Call Stack, 和 LabTalk Console 面板，以帮助用户开发和调试代码。

你可以从 View 菜单或者按 Alt + 4 键打开 Code Builder。 在某些情况下，Origin 菜单选项是运行 LabTalk 代码实现的，我们可以打开 Code Builder，并定位到源文件中执行该菜单的位置以了解对应的代码。例如，按住 Shift 和 Ctrl 键，同时选择菜单 Plot: Statistics : Histogram，这时，Origin 不会执行对应的绘图操作， 而是在 Code Builder 中打开 PLOT.OGS 并定位到 [Histogram] 部分代码的开头。

请参考 Origin C 和 Code Builder 简介 和 Code Builder 工作区。
