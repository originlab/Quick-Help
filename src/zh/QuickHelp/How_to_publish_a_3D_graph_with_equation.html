1.54 FAQ-173 如何输出包含公式的 3D OpenGL 图形？

export-3d-with-equation

Last Update: 10/27/2022

从 Origin 2017 SR0版本开始，软件支持像 2D 图形一样往3D OpenGL 图形中插入 OLE 对象 (i.e. 数学公式，word 文档，等) 。

激活图形窗口，点击 Insert Equations/Insert Word Objects/Insert Excel Object/Insert Object 工具栏并选择想要插入到图形中的对象。

请注意：若要插入这些对象则必须在电脑中安装了对应的软件或插件。 例如，如果想要插入一个公式，则需安装了 (MathType 公式软件，Microsoft Equation Editor) 插件，或者 OriginLab 免费提供的LaTeX 应用程序。

Keywords:3d, OpenGL, merge, equation, export, publish，3D 图，组合图，公式，导出，出版

Minimum Origin Version Required: 2017 SR0
