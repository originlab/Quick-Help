1.76 FAQ-205 二维和三维瀑布图有什么区别？


Last Update: 11/15/2018

2D-3D-Waterfall

2D 瀑布图由菜单 绘图>3D:瀑布图:瀑布图 画出，而 3D 瀑布图则由菜单 绘图>3D:瀑布图:3D瀑布图 画出。两种图都可以根据 Y / Z 值的变化来变色。

2D 瀑布图实际上是一个“假的” 3D 图。它的绝大多数操作跟其他的 2D 图形类似。它既不能改变视图的投影方式，也不支持在图上直接旋转，更不能在坐标轴上设置断点。

3D 瀑布图是基于 OpenGL 的 3D 图。它的绝大多数操作跟其他的 3D 图形类似。它支持直接拖拽瀑布图来旋转和拉伸。它提供两种视图投影模式：透视图和正交视图，你可以在 绘图细节 对话框的 其他 选项卡上切换这两种模式。3D 瀑布图还支持在坐标轴上设置断点。
