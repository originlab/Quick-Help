4.59 FAQ-1077 为什么使用外部dll的拟合函数在某些电脑失灵？

dll-compatibility

Last Update: 4/16/2021

Origin 只预先安装了vcredist_x64_2008, 2010, 2012, 和 2015-2019。所以如果您使用不同的 Visual Studio 版本来构建dll，比如2013，那么得到的dll在其它没有安装vcredist_x64_2013的电脑就可能会失灵。

这种情况怎么可以做到兼容呢？开发人员需要在 Visual Studio 中作如下设置：在项目的属性设置中，在 Configuration Properties: C/C++: Code Generation 这一页，把 Runtime Library 设置为 Multi-threaded (/MT)。

Keywords:dll, fitting，拟合
