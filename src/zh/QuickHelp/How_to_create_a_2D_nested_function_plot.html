1.152 FAQ-779 怎样创建二元嵌套函数图？

在 Origin 里，我们可以使用矩阵来创建如下二元嵌套函数图。

示例

在此教程中，我们将使用下列二元函数 来创建等高线图。

当且仅当

步骤:

  1. 单击 新建矩阵 按钮 创建一张矩阵工作表。
  2. 选择菜单 矩阵：行列数/标签设置 并按照下图所示设置相应的数值，单击 确定 以设置矩阵行列数。
  3. 接下来我们将要为矩阵对象设置如下函数：
    • 选择菜单 矩阵：设置值 在 Mat(1) 对话框中 输入 以下公式：
    x^2+y^2
  4. 现在再创建一张矩阵表来执行此 函数, 这将在计算中调用 Mat(1) 矩阵。
    • 单击矩阵窗口右上方的 D 按钮 ，从弹出菜单中选择 添加 选项。
    • 再次单击 D 按钮 ，从弹出菜单中选择 2 来激活第二张矩阵窗口。
    • 第二张矩阵窗口激活后，在 Mat(2) 对话框中 输入 以下公式：
    sqrt(Mat(1))+x+y
  5. 最后，从 Mat(2) 创建与下图相似的等高线图。
