2.44 FAQ-478 怎样设置我的工作表的行列数使得它总是 m 行 n 列 （m x n）？

Resize-Wks

Last Update: 7/13/2018

选择菜单 Format: Worksheet 或在工作表空白处右击选择 Properties 选项或按快捷键 F4 都可以打开 Worksheet Properties 对话框。 在 Size 选项卡中，设置 Row Number 和 Column Number。 在 Miscellaneous 选项卡中, 不要勾选 Auto Add Rows 复选框,但要勾选 Fit Sheet to Window 复选框，单击 OK 关闭对话框,这将使得无论工作表窗口是多大， 总会完全显示所设置的 m 行 n 列 （m x n）。


Keywords:report sheet，结果表
