5.40 FAQ-743 在 Project Explorer 下如何按多个属性对窗口进行排序？

Sort-Multiple-Attributes

Last Update: 8/1/2018


您可以通过单击列标题并按住 Shift 键，使得在 Project Explorer 下对窗口按多个属性进行嵌套排序。

假设您要按窗口类型进行升序排序，然后按窗口的长名称进行降序排序。

首先单击 Type 列标题以按类型进行排序。此时将会有一个箭头显示在 Type 旁边。其中向上的箭头表示升序，向下的箭头表示降序。如果你看到的是一个向下的箭头，您可以再次单击 Type 列标题将其改为向上的箭头。然后您可以按住 Shift 键并单击 Long Name 列标题一次或两次，使得向下的箭头出现在 Long Name 旁边。此时长名称就是该例子的辅助排序键。

Note: Layer Contents 对话框和 Column/Dataset 浏览器也具有该项功能。
