3.148 FAQ-894 如何拟合带 X/Y/Z 误差的 3D XYZ 曲面？

Fit-Surface-WithErr

Last Update: 7/26/2018


如果想拟合带误差的 3D XYZ 曲面，你可以参考以下步骤：

  1. 首先，在 Fitting Function Organizer 对话框，将你需要用的 3D 曲面函数从 Surface Fitting 类别移动到 User Defined
  2. 点击带 XYZ 数据的工作表，选中 Z 数据，选择菜单 Analysis: Fitting: Nonlinear Curve Fit ，打开NLFit 对话框。选择你刚刚移动的曲面函数作为拟合函数。

    对于 Iteration Algorithm，
    • 当你选择 Levenberg Marquardt，在 Data Selection 页面中，只包含 Z 方向的误差。
    • 当你选择 Orthoginal Distance Regression(Pro)， 所有方向 (X,Y 和 Z) 的误差都包含在拟合中。

    关于如何设置拟合权重的方法，请参考这个帮助页面

  3. 进行其他必要的设置，完成 3D 拟合。
