2.11 FAQ-419 如何从数据库导入数据并保存和重复使用查询？

DatabaseAccess-Toolbar

Last Update: 7/28/2018

要从数据库中导入数据，请参阅本教程。

要保存自定义的查询，请在 Query Builder 中选择 Query：Save AS ODQ file，或者，在 SQL Editor 中，选择 File: Save Connection and Query as。

要重用保存的查询，请单击 Load ODQ 按钮 ，浏览并加载查询文件。你还可以在 Query Builder 和 SQL Editor 中直接打开 ODQ 文件。
