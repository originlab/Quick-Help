3.131 FAQ-769 可以在每次分析处理前，预处理峰数据，例如排除不需要的数据值吗？

Batch-PA-Script-before-each-Process

Last Update: 7/30/2018

当然是可以的。在每个处理之前运行指定的脚本，那么你就可以在每次峰值分析之前预处理数据。有关详细说明，请参考此教程

请注意， Batch Peak Analysis Using Theme 工具也可以选择是在每个批处理之后运行脚本，还是在所有处理结束后再运行脚本。此外，几乎所有峰值分析设置，包括频谱对齐 (Snap to Spectrum); 相对基线 (Relative Baseline); 逐个峰初始化参数值 (Sequentially Initializing Parameter Values) 等， 都可以保存在主题中，帮助生成批处理结果。

Minimum Origin Version Required: Origin 2016 SR0
