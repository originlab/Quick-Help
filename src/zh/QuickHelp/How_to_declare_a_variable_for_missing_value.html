4.19 FAQ-378 如何声明一个变量为缺失值?

Declare-Variable-for-Missing-Value

Last Update: 7/6/2018

您可以使用表达式 “0/0” 声明一个变量，将其定义为缺失值. 例如: 

misval = 0/0;
misval =;//Should return --

另外，在 LabTalk 中一个缺失值等同于 False. 例如: 

misval = 0/0;
type $(misval?1:0);//Should return 0 (False)


Keywords:missing value, declare, variable, false, 缺失值，声明，变量
