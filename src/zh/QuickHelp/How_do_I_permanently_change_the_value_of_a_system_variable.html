5.29 FAQ-708 如何使更改的系统变量的值永久地生效？

permanent-change-system-variable-value

Last Update: 10/11/2022


您可以在 设置: 系统变量 对话框中键入变量 及其 值来更改任意 LabTalk “@” 系统变量的启动值。您也可以为该变量添加注释，以帮助您记住该 “@” 变量所控制的内容。

注意：
  • 当输入变量名称时，无需加上 "@" 首字符。
  • 变量名称是不区分大小写的。
  • 变量名称按类型进行颜色和加粗区分。
  • 要将系统变量恢复为默认值，(a) 右击设置系统变量中的变量并选择 删除 或(b) 输入默认 值。
  • 有关管理系统变量及其控制的行为的更多信息，请参阅 System Variable List > Introduction.
