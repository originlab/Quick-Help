2.7 FAQ-414 如何从剪贴板或网页中导入数据

Import-from-Clipboard-or-Web

Last Update: 10/17/2020

  1. 从应用程序或网页中复制数据
  2. 在 Origin 中点击 Data: Import from File: Import Wizard 菜单或工具栏中导入向导按钮打开导入向导（Import Wizard）
  3. 如下图所示，在导入向导对话框中将 Data Source 设置为 Clipboard .
  4. 点击 Finish 完成导入.
注意:
  • 新版本的 Origin 支持使用 数据连接器 直接导入 Web 数据。数据连接器支持常见的 Web 数据格式。了解更多信息，请参见

Connect to Web.

  • 如果要将字符串从剪贴板粘贴到 Origin，则应阅读 this FAQ.
