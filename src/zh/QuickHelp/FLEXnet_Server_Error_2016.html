6.53 FAQ-792 当我的 Origin 2016（或者以后版本）连接 FLEXnet 服务器的时候，弹出一个错误信息 "Your FLEXlm license file does not allow the requested feature." 怎么办？

FLEXnet-Server-Error

Last Update: 7/11/2018

Origin 2016 及以后版本需要 FLEXnet server version 11.13.1.2 或以上版本才能够支持。如果你的许可证服务器 FLEXnet server 的版本低于 11.13.1.2，你将会看到这样的错误信息:
"Your FLEXlm license file does not allow the requested feature".

请更新你的 Origin FLEXnet 服务器。详细步骤看这里。
