6.87 FAQ-1105 如何生成日志文件？

generate-log-file

Last Update: 6/17/2021

Origin 会生成一个有助于诊断软件和许可问题的日志文件。如果 OriginLab 客户服务或技术支持代表要求您生成日志文件，您可以按以下步骤进行：

  1. 启动 Origin。
  2. 在脚本窗口 (窗口: 脚本窗口)，输入以下内容并按 ENTER：
    3. @log=1
  3. 关闭 Origin, 再次重启后就会开始生成日志文件。
  4. 重复能产生错误消息的步骤以再次产生该错误消息。
  5. 3个会话后，日志记录将自动关闭。找到日志文件 -- 以 ORIGIN_LOG_###.TXT 命名、存放在你的用户文件夹 (帮助: 打开文件夹: 用户文件夹)。
  6. 把日志文件添加到 support ticket 的附件中。如果没有 support ticket，请 在此 创建一个。

Origin 2019 或更早的版本

  1. 在脚本窗口 (窗口: 脚本窗口)，输入以下内容并按 ENTER： 
    debug_log 2
    就会开始生成日志文件，并且将日志文件位置返回到脚本窗口。
  2. 要停止日志记录，运行脚本：
    3. debug_log 0
  3. 要查看日志状态和调试日志文件名和位置，请运行：
    4. debug_log
  4. 按照上面第6步中的说明将日志文件发送给我们。
