1.185 FAQ-876 如何减少导出图形周边的空白区域？

Export-Region-of-Graph

Last Update: 4/24/2019

图形页面中被复制或者导出的范围，是由 Preference: Options 对话框中 Page 选项卡下的 Margin Control 和 Clip Border Width 设置所控制的。

默认情况下， Margin Control 设为 Page, 表示将复制或导出整个页面。 你可以将其改为 Tight，这样就不会复制/导出图形框架以外的空白区域。在Export Graphs 对话框 中也有这样的控件。 默认情况下，会自动加载 Preference: Options 对话框中的设置，但你也可以在 Export Graphs 对话框中进行修改。


在 Origin 2018，你可以使用交互式工具来定义图形的复制和导出区域，而无需更改 Tools: Option 对话框中的设置。

  1. 选择 Edit: Export Margin Offsets， 此时将会打开 Edit: Export Margin Offsets 对话框，并在图形页面上显示一个边界框。
  2. 使用边界框上的选择手柄，结合对话框中的文本框，来调整图形的导出边距。或者直接输入大于0的边界像素值。点击 OK。
  3. 然后使用 Edit: Copy Page, Edit: Copy Graph as Picture 或者 File: Export Graph，则可以复制/导出你定义的图形区域。
