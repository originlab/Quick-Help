1.236 FAQ-1010 如何创建隐函数图?

create-implict-function-plot

Last Update: 10/22/2019

创建隐函数图，您需要先建立一个隐函数，然后模拟这个隐函数。例如：画以下函数

  1. 通过工具: 拟合函数管理器自定义一个隐函数。
    1. 在"User Defined"类目下创建一个新的自定义函数。
    2. 定义函数如下：
      • 函数名称 = Heart
      • 函数模型 = 隐函数.
      • 变量 = x,y
      • 函数形式 = 公式
      • 函数 = f=x*x+(y-(x*x)^(1/3))^2-r^2.
        注意：对于如何定义隐函数，您也可以参考Implicit类目下的"Circle"函数。
    3. 点击保存按钮进行保存。
  2. 点击模拟按钮来创建这个函数的图形。
