5.43 FAQ-777 如何使用 Ctrl + C 复制全精度的数据？

Full-Precision-Copy

Last Update: 7/28/2018

复制和粘贴数据时，通常会使用 CTRL + C（复制）和 CTRL + V（粘贴）键盘组合。但是，当您需要将数据粘贴到外部应用程序时，这可能会无法保留数字的精度。若您需要保留完整的数字精度时，请使用 CTRL + ALT + C（复制全精度）键盘组合（然后使用标准的 CTRL + V 粘贴）。

请注意，如果您只是在 Origin 工作表之间进行复制和粘贴数据，那么使用 CTRL + C 即可以全精度的方式复制和粘贴数据。但是，如果您要复制并粘贴到外部应用程序并且您觉得使用 CTRL + ALT + C 键盘组合很麻烦，则可以通过修改 Origin 使其能以简单的 CTRL + C 键盘组合复制 Origin 全精度的数据。您可以通过修改系统变量 @CWACC 的默认值来执行此操作。

  • 仅更改当前 Origin 进程中的默认设置，请打开 Script Window，输入以下内容，然后按 Enter 键：
@CWACC=1
通这种更改，Ctrl + C 变为在工作簿中以全精度复制数据的键盘快捷键，Ctrl + Alt + C 则变为以显示的精度复制数据。
