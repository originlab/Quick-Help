1.240 FAQ-1018 如何修正插入到图中的Word对象的字体大小?

Fix-FontSize-for-Word-Obj

Last Update: 10/23/2019

如果你在图中插入了Word对象，而且想要保留在Word OLE中设置的字体大小，那么你可以右击这个Word对象，然后选择属性打开Documentation Properties对话框，去到View选项卡，勾上Relative to Original Size勾选框。
