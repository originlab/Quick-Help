6.56 FAQ-798 当我把许可证复制粘贴回 Origin 的 license 对话框中时，我看到一个 "The activation time has passed" 错误信息。怎么办？

Activation-Time-Pass-Err

Last Update: 7/9/2018

当你选择 "Get a license file now. This computer has internet access." 这一选项来激活 Origin 许可证的时候，你必须在 15 分钟之内把许可证文件粘贴回 Origin，过期失效。如果你看到了失效信息，请重启 Origin 并重新激活。

当你选择 "Get a license file. This computer does NOT have internet access." 这一选项来激活 Origin 许可证的时候，你必须在 24 小时之内把许可证文件粘贴回 Origin，过期失效。如果你看到了失效信息，请重启 Origin 并重新激活。

Keywords:activation time, 许可证失效, 许可证过期
