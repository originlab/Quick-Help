5.82 FAQ-1127 打开同步到 OneDrive 的文件触发错误：此文件已损坏

OneDrive-file-corrupted-project-backup

Last Update: 5/10/2022

当从同步到微软 OneDrive 帐户的文件夹里打开 Origin 文件，有些用户报告看到错误信息说这个文件已经损坏，但前一个版本的文件仍然可以用。用户通过指引，选择 帮助: 打开文件夹: 项目文件备份 拿到备份文件。

用户报告如下：

  • 对错误信息点 确定 后，文件正常打开并且完好无损。
  • 但如果试着保存，Origin默认保存到\Backup。这就需要浏览到 OneDrive 的文件夹，然后保存到那个位置。

当用户在完全的管理权限时，不会看到这个错误信息。我们仍然在研究具体的细节，但用户报告称，从普通用户帐户切换到有高级权限的管理帐户，这个问题得到解决。
