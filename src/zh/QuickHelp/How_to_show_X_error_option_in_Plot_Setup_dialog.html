1.158 FAQ-813 如何在绘图设置 (Plot Setup) 对话框中显示X误差的选项？

Show-XErr-in-PS

Last Update: 12/30/2018

在默认情况下，在 图表绘制 对话框的中间面板上，显示了 X, Y 和 YEr 等绘图设定。如需要显示X误差选项，你可以右击中间面板的表格内部，并选择弹出菜单中选择 X 误差棒 。你还可以通过选择弹出菜单中的 Y 正/负误差帮 以显示 yEr+ 和 yEr- 的选项。

Keywords:Plot Setup, X error, Y error plus, Y error minus, 绘图设置, X 误差, 正向Y 误差, 负向Y 误差
