6.107 FAQ-1189 我看到“HOSTNAME specified in the license file is not available in the local network database”这样的错误信息，应该怎么解决？

Hostname-Not-Available-Linux-Server

Last Update: 10/25/2023

如果您在 Linux 操作系统（例如 RHEL8.8）上设置 FLEXnet 许可证服务器，尝试启动服务失败并在调试日志中看到以下错误消息：

(lmgrd) Unknown Hostname: <HOSTNAME> specified in the license file is not available in the local network database.

这是由于服务器计算机上的主机文件引起的，服务器无法识别主机名并正常启动。This is caused by a problem with the hosts file on the server machine. The server cannot recognize the hostname and start properly.


要修复这个问题：

  • 请在主机文件 (/etc/hosts) 中添加下面这一行:
<Server_Ip_Address >   <HOSTNAME>
