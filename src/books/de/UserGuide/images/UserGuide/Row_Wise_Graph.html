Datenanforderungen

Wählen Sie zwei oder mehrere Y-Spalten oder einen Datenbereich aus mindestens zwei Y-Spalten aus.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen: 2D: Linie + Symbol: Zeilenweise.

oder

Klicken Sie auf die Schaltfläche Zeilenweise auf der Symbolleiste 2D Grafiken.


Einzelheiten zu den Bedienelementen erhalten Sie im Abschnitt unten:

Der Dialog plot_rowwise

Zeilenweise Daten Dieser Zweig wird zum Festlegen der Eingabedaten verwendet.
Y-Beschriftung in Legen Sie eine Spalte als Diagrammlegende fest.
X-Daten in Legen Sie den X-Achsenwert für das Diagramm fest. Die verfügbaren Optionen entsprechen den folgenden:
  • 1. Zeile der Auswahl
Verwenden Sie die Werte der ersten Zeile in der aktuellen Auswahl.
  • 1. Zeile in Arbeitsblatt
Verwenden Sie die Werte der ersten Zeile im Arbeitsblatt.
  • Spaltenbeschriftung
Verwenden Sie die Spaltenbeschriftungen im Arbeitsblatt. Mit dieser Option können Sie eine Spaltenbeschriftungszeile aus der Auswahlliste Spaltenbeschriftung auswählen.
  • Benutzerdefiniert
Mit dieser Option wird die manuelle Datenauswahl mit dem Feld X-Zeilenbereich möglich. Bitte beachten Sie, dass die Werte nur aus dem gleichen Arbeitsblatt ausgewählt werden können.
Spaltenbeschriftung Verfügbar, wenn X-Daten in = Spaltenbeschriftung. Legen Sie Langname, Einheit oder Kommentare als X-Daten fest.
X-Zeilenbereich Verfügbar, wenn X-Daten in = Benutzerdefiniert. Legen Sie einen Zeilenbereich als X-Daten fest.
Diagrammtyp Legen Sie den Diagrammtyp auf Linie, Punkt, Linie+Symbol, Säulen oder Balken fest.
Ausgabe Legen Sie fest, wo die berechneten Daten ausgegeben werden.
