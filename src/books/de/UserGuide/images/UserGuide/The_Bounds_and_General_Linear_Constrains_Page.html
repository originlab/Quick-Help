Die Seite Grenzen und allgemeine lineare Nebenbedingungen

FitFuncBuilder-Bounds-Constraints

Diese Seite bietet Optionen, mit denen Sie die unteren und/oder oberen Grenzen für jeden Parameter festlegen können. Diese Seite ist optional, d.h., Sie können direkt auf Weiter klicken, um zu der nächsten Seite zu gelangen, auf der weitere Einstellungen vorgenommen werden können. Bitte beachten Sie, dass Sie diese Werte auch im Dialogfeld NLFit während des Anpassungsvorgangs festgelegt werden können.

Beachten Sie, dass es nur möglich ist, die allgemeinen linearen Nebenbedingungen zu verwenden, wenn das Funktionsmodell auf Explizit gesetzt ist.


Bedienelemente des Dialogs

Gruppe Grenzen Bietet Bedienelemente, mit denen untere und/oder obere Grenzen für jeden Parameter festgelegt werden können.
  • Untere Grenzen
    Legt die unteren Grenzen für Parameter fest. Sie können nur bearbeitet werden, wenn < oder <= aus der Spalte < oder <= ausgewählt wird.
  • < oder <=
    Klicken Sie doppelt auf die Zellen in der Spalte Operator (< oder <=), um einen Operator auszuwählen. Geben Sie dann die unteren bzw. oberen Grenzwerte in die jeweiligen Bearbeitungsfelder ein.
  • Param
    Führen Sie alle Namen der Anpassungsfunktionsparameter auf.
  • Obere Grenzen
    Legt die oberen Grenzen für Parameter fest. Sie können nur bearbeitet werden, wenn < oder <= aus der Spalte < oder <= ausgewählt wird.
Allgemeine lineare Nebenbedingungen verwenden Aktivieren Sie dieses Kontrollkästchen, wenn Sie lineare Nebenbedingungen für Parameter festlegen möchten. Trennen Sie mehrere Zeilen durch Semikolon.
Allgemeine lineare Nebenbedingungen Mit dieser Option können Sie lineare Nebenbedingungen für die Parameter eingeben. Sie können auch die Nebenbedingungen im Dialog NLFit während des Anpassungsvorgangs unter in Allgemeine lineare Nebenbedingungen festlegen. Das Folgende ist ein Beispiel für lineare Nebenbedingungen:
a > b;
a + 2 * b >= c * 2? d;
a < b < c;
a / 3 < 9;
