Importassistent, Variablenextraktion durch Positionsseite (ASCII)

Diese Seite ist nur verfügbar, wenn Bestimmen des Speicherorts von Variablennamen und Werten unter Verwendung von Positionen verwenden auf der Seite Variablenextraktion aktiviert ist (und es mindestens eine Headerzeile in Ihrer Datendatei gibt). Verwenden Sie die Bedienelemente auf dieser Seite, um Variablen aus dem Dateipfad/Dateinamen und dem Dateiheader zu extrahieren, wobei Informationen durch die Zeichenposition von einem "Variablen"-Vorschaufenster extrahiert werden.

Vorschaufenster Variablen

Verwenden Sie dieses Vorschaufenster, um Variablen für die Extraktion auszuwählen. Der Inhalt dieses Vorschaufensters umfasst vier Standarddateinamen (Pfad + Dateiname.Erweiterung, Pfad, Dateiname.Erweiterung, Dateiname). Ebenso sind jegliche Dateiheaderzeilen enthalten, die auf der Seite Headerzeilen bestimmt wurden. Jedem Variablennamen/-wert geht eine Zeilennummer voran -- der Zeilenwert in der Gruppe Variablen (siehe unten).

Bedienelemente von Variablengruppen

Bei jeder Informationszeile im Vorschaufenster Variablen wird vorausgesetzt, dass sie Variablen und Variablenwerte (Sie entscheiden, welche welche sind) enthält. Beachten Sie, dass jede der folgenden Möglichkeiten unabhängig von jeder Zeile im Vorschaufenster ausgewählt werden kann.

Linie Wählen Sie die Zeilennummer der gewünschten Variablen.
Beginn/Ende Sie können die Zeichenpositionen direkt eingeben oder (einfachere Methode) Sie können die Variable Wert mit Hilfe der Maus markieren und auf die Schaltfläche Aktualisieren rechts neben dem Feld Wert klicken.
Wert Das Feld Wert wird automatisch mit den Zeichenpositionsinformationen Anfang und Ende ausgefüllt.
Geben Sie den Variablennamen ein. Es gibt zwei Möglichkeiten, Variablennamen festzulegen. Wählen Sie diese Schaltfläche, um eine Zeichenkettenkonstante (Leerzeichen und Sonderzeichen sind nicht erlaubt) als Variablennamen zu verwenden. Um den Variablennamen aus den Informationen, die im Vorschaufenster Variablen enthalten sind, zu extrahieren, wählen Sie die Option Variablennamen aus der Datei extrahieren (nächste Option).
Variablennamen aus der Datei extrahieren Wenn Sie diese Schaltfläche wählen, werden zusätzliche Felder zur Seite Extraktion durch Position hinzugefügt. Diese Felder ermöglichen es Ihnen, die Zeichenpositionen einer Zeile, von Anfang und Ende und den Namen für eine gegebene Variable einzugeben (das Feld Name wird automatisch mit den speziellen Zeichenstelleninformationen gefüllt). Diese Option wird verwendet, um der Auswahlliste Wert einen Namen zuzuweisen.

Variablen zum Extrahieren des Fensters

Methode Sie können Variablen durch mehr als eine Methode extrahieren. Sie werden über beide Seiten Trennzeichen und Position gefunden, unabhängig von der Methode/Seite, durch die sie bestimmt wurden. Die Variablen, die auf dieser Seite festgelegt werden, sind vom Typ Methode = Position.
Name Der Name, wie angegeben, verwendet die Bedienelemente der Gruppe Variable.
Wert Die Variable Wert, wie angegeben, verwendet die Bedienelemente der Gruppe Variable.
