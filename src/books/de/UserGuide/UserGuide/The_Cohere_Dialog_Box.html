Dialog Theme See Dialog Themes.
Recalculate

See Recalculating Analysis Results.
Input1

Specifies the first input signal. See Specifying Your Input Data.
Input2

Specifies the second input signal. See Specifying Your Input Data.
Sampling
Interval

Specifies the sampling frequency. The default is <auto>, which corresponds to an automatically-calculated interval.
FFT Length

Specifies the size of FFT section.
Window length

Specifies the window size, which must not be greater than the size of an FFT section.
Overlap

Specifies the number of data points by which the window sections overlap. It should be less than the window size.
Window
Type

Specifies the window type used by FFT.

  • Rectangle
  • Welch
  • Triangular
  • Bartlett
  • Hanning
  • Hamming
  • Blackman
  • Gaussian
  • Kaiser

For more information on window types, see the Coherence Algorithms page.
Window length

Specifies the window size, which must not be greater than the size of an FFT section.
Overlap

Specifies the number of data points by which the window sections overlap. It should be less than the window size.
Alpha

This variable is only available when Window Type is Gaussian. It specifies the Alpha parameter for the Gaussian window.
Beta

This variable is only available when Window Type is Kaiser. It specifies the Beta parameter for the Kaiser window.
Output

Specifies the output. See Output Results.
