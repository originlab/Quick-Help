Die Registerkarte Pfeil (Objekteigenschaften)

Ob-Prop-Arrow-tab

 

Auf der Registerkarte Pfeil können Sie die Anzeige der Pfeilspitze an Anfangs und Ende festlegen.

Wenn eine Anfangs- und eine Endpfeilspitze ausgewählt sind, können Sie das Kontrollkästchen Gleiche Pfeilspitzengröße auswählen, um sicherzustellen, dass Änderungen, die Sie an Breite und Länge der Pfeilspitze entweder in der Gruppe Anfang oder Ende vornehmen, auf beide angewendet werden.

Um den Pfeil für jedes Segment des Objekts Polylinie hinzuzufügen, können Sie das Kontrollkästchen Pfeil für jedes Segment zeigen aktivieren, wenn die Pfeilspitze ausgewählt ist.

Aktivieren Sie das Kontrollkästchen Als Standardfestlegen, um diese Registerkarteneinstellungen auf das aktuell ausgewählte Objekt und alle zukünftigen Objekte des gleichen Typs anzuwenden.

Hinweis: Das Bedienelement Transparenz ist das gleiche wie auf der Registerkarte Linie. Sie enthalten beide den gleichen Wert.

Wenn Sie den Prozentsatz der Transparenz für die Linie ändern, wird der Pfeil auf den gleichen Prozentsatz für die Transparenz angewendet und umgekehrt.
