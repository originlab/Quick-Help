Datenanforderungen

Sie brauchen eine oder mehrere Arbeitsblattspalten, um solch ein Diagramm zu erstellen, außer für Violine aufteilen.

Für die aufgeteilte Violine müssen Sie mindestens zwei Spalten auswählen. Die letzte Spalte sollte nur 2 Kategorien enthalten.

Diagramm erstellen

Violindiagramm

  1. Wählen Sie die gewünschten Daten aus.
  2. Wählen Sie im Menü Zeichnen > Statistisch: Violindiagramm.


Violine mit Box

  1. Wählen Sie die gewünschten Daten aus.
  2. Wählen Sie im Menü Zeichnen > Statistisch: Violine mit Box.
oder
Klicken Sie auf die Schaltfläche Violine mit Box auf der Symbolleiste 2D-Grafiken.


Violine mit Punktdiagramm

  1. Wählen Sie die gewünschten Daten aus.
  2. Wählen Sie im Menü Zeichnen > Statistisch: Violine mit Punkt.


Violine mit Quartildiagramm

  1. Wählen Sie die gewünschten Daten aus.
  2. Wählen Sie im Menü Zeichnen > Statistisch: Violine mit Quartil.


Violin mit Streifen

  1. Wählen Sie die gewünschten Daten aus.
  2. Wählen Sie im Menü Zeichnen > Statistisch: Violine mit Streifen.


Aufgeteilte Violine

  1. Wählen Sie die gewünschten Daten aus.
  2. Wählen Sie im Menü Zeichnen > Statistisch: Aufgeteilte Violine.


Halbe Violine

  1. Wählen Sie die gewünschten Daten aus.
  2. Wählen Sie im Menü Zeichnen > Statistisch: Halbe Violine.
