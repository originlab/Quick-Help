Registerkarte Liste Innen (Details Zeichnung)

PD-dlg-interior-list-tab

Diese Registerkarte ist für 2D-Diagramme verfügbar, wenn das Innere der Symbole über Index definiert wird, d.h. auf der Registerkarte Symbol wird unter Benutzerdefinierte Konstruktion die Option Geometrisch gewählt und die Auswahlliste Innen wird auf eine Datenspalte oder Inkrement gesetzt.

Die Liste Innen kann vom Anwender benutzerdefiniert angepasst werden.

Benutzerdefinierte Inkrementliste verwenden

Dieses Kontrollkästchen ist standardmäßig deaktiviert. In diesem Fall wird die Standardinkrementliste verwendet.

Wenn dieses Kontrollkästchen aktiviert wird, wird die Liste Innen gezeigt. Sie können sie benutzerdefiniert anpassen.

Liste Innen

Sie können die Formliste benutzerdefiniert anpassen, speichern und neu laden. Weitere Optionen finden Sie im Kontextmenü.

Wenn die Form einen Index auf eine Spalte mit kategorialen Daten hat, werden Kategorien neben jedem Stil in der Liste Form angezeigt, wodurch es einfacher ist, Kategorie und Innen in Verbindung zu bringen.

Innen benutzerdefiniert anpassen

Klicken Sie für jede Zeile in der Liste auf die dreieckige Schaltfläche und wählen Sie ein Standardsymbol für Innen aus.

Liste Innen neu anordnen

Sie können ausgewählte Zeile(n) in der Liste Innen verschieben, hinzufügen und löschen sowie die Liste umkehren. Dazu verwenden Sie die gleichen Methoden wie zum Neuordnen der Farbliste.

Inkrementliste speichern und laden

Sie können mit der rechten Maustaste klicken und im Kontextmenü wählen, ob eine Inkrementliste gespeichert oder geladen werden soll. Dazu verwenden Sie die gleichen Methoden wie zum Speichern und Laden einer Inkrementliste.

Inkrementliste kopieren und einfügen

Um eine benutzerdefinierte Farbinkrementliste auf andere Zeichnungen anzuwenden, können Sie mit der rechten Maustaste auf die Liste klicken und Kopieren wählen. Wechseln Sie dann zur Registerkarte Liste Innen der anderen Zeichnungen, klicken Sie mit der rechten Maustaste auf die Liste und wählen Sie die Option Einfügen.
