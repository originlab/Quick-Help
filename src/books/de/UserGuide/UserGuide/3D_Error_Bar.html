3D-Punktdiagramm mit Fehlerbalken

3D-ErrBars

 

Datenanforderungen

  • Arbeitsblatt: Sie müssen 4 Spalten aus einem Arbeitsblatt auswählen und die Zuordnung auf XYZZ setzen.

oder

  • Matrix: Sie benötigen zwei Matrixobjekte in dem gleichen Matrixblatt. Das Matrixobjekt, das die Punktdaten enthält, sollte aktiv sein.

Sie können zu dem bestehenden 3D-Punktdiagramm auch Fehlerbalken über den Dialog Details Zeichnung hinzufügen.

Diagramm erstellen

Aktivieren Sie das Matrixblatt oder wählen Sie die gewünschten Daten im Arbeitsblatt aus.

Wählen Sie im Menü für die Arbeitsblattdaten Zeichnen: 3D: 3D-Punktdiagramm mit Fehlerbalken; wählen Sie für die Matrixdaten Zeichnen: 3D: 3D-Punktdiagramm mit Fehlerbalken.

oder

Klicken Sie auf die Schaltfläche 3D-Punkt + Fehlerbalken der Symbolleiste 3D- und Konturdiagramme.

Vorlage

Arbeitsblatt

  • gl3DERROR.OTP (OpenGL)
  • 3DERROR.OTP

Matrix

  • gl3DSCATTERMATERR.OTP (OpenGL)
  • 3DSCATTERMATERR.OTP

(installiert im Origin-Programmordner)
