Einführung

Für das Boxdiagramm kann eine spezielle Art von Legende erzeugt werden. In diesen Legenden bezieht sich jeder Eintrag auf eine der Boxkomponenten,, wie die Box selbst, die Whisker, die Median- oder Mittelwertlinie, das Perzentil etc.

Die Legende für Boxkomponenten kann für die folgenden Diagrammtypen erzeugt werden:

  • Boxdiagramme
  • Gruppiertes Boxdiagramm aus Indexdaten
  • Gruppiertes Boxdiagramm aus Rohdaten

Sie können die Legende der Boxdiagrammkomponenten sehr einfach benutzerdefiniert anpassen, um Standardparameter von Boxdiagrammen entweder zu zeigen oder zu verbergen.

Legende für Boxkomponenten erstellen

In einer Grafik mit Boxdiagrammen (siehe den Abschnitt oben) können Sie eine Legende für die Boxkomponenten erstellen und die vorhandene Legende erstellen (für ein Legendenobjekt der Datenzeichnung).

Die Standardlegende für Boxkomponenten werden automatisch aktualisiert, wenn die Boxdiagramme sich ändern.

Um eine Legende für Boxkomponenten zu erstellen, müssen Sie vielleicht den Dialog legendbox öffnen und verwenden. Auf diesen Dialog kann durch einen der folgenden Schritte zugegriffen werden:

  • Wählen Sie bei aktivem Diagrammfenster Grafik: Legende: Boxdiagrammkomponenten im Menü.

oder

  • Klicken Sie mit der rechten Maustaste auf der Diagrammseite und wählen Sie Legende: Boxdiagrammkomponenten im Kontextmenü.

oder

  • Klicken Sie mit der rechten Maustaste auf das Legendenobjekt und wählen Sie Legende: Boxdiagrammkomponenten.

Hinweis: Wenn die vorhandene Legende bereits Legendeneinträge für Boxdiagrammkomponenten enthält, verwenden Sie diesen Dialog, um diese Legendeneinträge zu aktualisieren.

Alternativ können Sie eine Legende für die Komponenten des Boxdiagramms erstellen, ohne den Dialog legendbox zu öffnen:

  • Verwenden Sie direkt den Menübefehl Grafik: Legende: Legende rekonstruieren oder drücken Sie Strg + L.

 

Der Dialog legendbox

Dieser Dialog wird verwendet, um eine erweiterte Legende für die Komponenten des Boxdiagramms zu erstellen. Wenn es bereits ein Legendenobjekt in dem Diagramm gibt, werden die Komponenten des Bocdiagramms in die existierende Legende eingebunden. Beachten Sie, dass einige Optionen im Dialog nur für bestimmte Varianten des Boxdiagramms gezeigt werden.

Dieser Dialog verwendet die X-Funktion legendbox. Er wird verwendet, um festzulegen, welche Boxelemente in der Legende gezeigt werden sollen. Die Bedienelemente des Dialogs werden in der untenstehenden Tabelle zusammengefasst.

Boxdiagramm Diese Option ist nur verfügbar, wenn der Boxtyp Box, Box + Datenüberschneidung, Box (Rechts) + Daten (Links), Box (Links) + Daten (Rechts), Balken (Links) + Daten (Rechts), Halbe Box (Rechts) + Daten (Links) oder Halbe Box (Links + Daten (Rechts). Sie dient dazu festzulegen, ob der Eintrag für die Box in der Legende angezeigt wird. Der Symbolstil der Datenzeichnung folgt den Einstellungen des Boxstils auf der Registerkarte Boxdiagramm und der Registerkarte Muster. Der Legendentext verwendet die Variable, die in der Auswahlliste Boxidentifizierer und durch das Kontrollkästchen Boxbereich zeigen festgelegt ist.
Boxenidentifizierer Diese Auswahlliste ist nur verfügbar, wenn Box aktiviert ist. Wählen Sie die Option <Benutzerdefiniert>, um ein benutzerdefiniertes Format festzulegen.
  • Kurzname
  • Langname
  • Einheiten
  • Kommentare
  • Beliebige kategoriale Gruppenspalten: Spalten mit kategorialen Daten, die zum Definieren der Boxdiagrammgruppen verwendet werden
  • <Benutzerdefiniert>
  • <Kein>: Kein Boxidentifizierer Standardoptionen für normales Boxdiagramm und gruppiertes Boxdiagramm
Benutzerdefiniertes Format des Identifizierers Diese Option ist nur verfügbar, wenn für Boxidentifizierer die Option <Benutzerdefiniert> gewählt wurde. Legen Sie Ihren Identifizierer mit einer folgenden Optionen fest:
  • Wählen Sie ein benutzerdefiniertes Standardformat aus dem Auswahlmenü.
  • Klicken Sie auf die Schaltfläche mit dem Pfeil rechts vom Kombinationsfeld.
  • Geben Sie Ihre eigenen Zeichenketten für den Identifizierer direkt ein. Bitte lesen Sie Einzelheiten dazu in der vollständigen Liste von @Optionen.
Boxbereich zeigen Dieses Kontrollkästchen ist nur verfügbar, wenn Box aktiviert ist. Durch Aktivierung wird die Information des Boxbereichs neben dem Text des Boxidentifizierer angehängt. Der Boxbereich wird von der Auswahlliste Bereich auf der Registerkarte Boxdiagramm bestimmt.
Whisker Dieses Kontrollkästchen steht nur zur Verfügung, wenn Whisker aktiviert ist. Es legt fest, ob in der Legende Whisker angezeigt werden. Wenn sie angezeigt werden, folgt das Diagrammsymbol den Stileinstellungen für Whisker auf der Registerkarte Linie des Dialogs Details Zeichnung. Der Legendentext verwendet die Einstellung des Whiskerbereichs (in der Auswahlliste Bereich auf der Registerkarte Boxdiagramm).
Balken Diese Option ist nur verfügbar, wenn der Boxtyp Balken, Balken + Datenüberschneidung, Balken(Rechts) + Daten(Links) oder Balken(Links) + Daten(Rechts) ist. Sie dient dazu festzulegen, ob der Eintrag für die Balken in der Legende angezeigt wird.

Der Symbolstil der Datenzeichnung folgt den Einstellungen des Balkenstils auf der Registerkarte Boxdiagramm und der Registerkarte Muster. Der Legendentext verwendet die Variable, die in der Auswahlliste Boxidentifizierer und durch das Kontrollkästchen Boxbereich zeigen festgelegt ist.
Balkenidentifizierer Diese Auswahlliste ist nur verfügbar, wenn Balken aktiviert ist. Wählen Sie die Option <Benutzerdefiniert>, um ein benutzerdefiniertes Format festzulegen.
  • Kurzname
  • Langname
  • Einheiten
  • Kommentare
  • Beliebige kategoriale Gruppenspalten: Spalten mit kategorialen Daten, die zum Definieren der Boxdiagrammgruppen verwendet werden
  • <Benutzerdefiniert>
Benutzerdefiniertes Format des Identifizierers Diese Option ist nur verfügbar, wenn für Balkenidentifizierer die Option <Benutzerdefiniert> gewählt wurde. Legen Sie Ihren Identifizierer mit einer folgenden Optionen fest:
  • Wählen Sie ein benutzerdefiniertes Standardformat aus dem Auswahlmenü.
  • Klicken Sie auf die Schaltfläche mit dem Pfeil rechts vom Kombinationsfeld.
  • Geben Sie Ihre eigenen Zeichenketten für den Identifizierer direkt ein. Bitte lesen Sie Einzelheiten dazu in der vollständigen Liste von @Optionen.
Balkenbereich zeigen Dieses Kontrollkästchen ist nur verfügbar, wenn Balken aktiviert ist. Durch Aktivierung wird die Information des Boxbereichs neben dem Text des Balkenidentifizierers angehängt. Der Boxbereich wird von der Auswahlliste Bereich auf der Registerkarte Boxdiagramm bestimmt.
Fehlerbalken Diese Option ist nur verfügbar, wenn der Fehlerbalken aktiviert ist. Hier wird gesteuert, ob in der Legende Fehlerbalken angezeigt werden.
Medianlinie Dieses Kontrollkästchen ist nur verfügbar, wenn die Medianlinie auf der Registerkarte Linie aktiviert ist. Es legt fest, ob der Legendeneintrag für die Medianlinie gezeigt wird. Das Diagrammsymbol folgt den Stileinstellungen der Medianlinie auf der Registerkarte Linie, gefolgt von dem Text Medianlinie.
Mittelwertlinie Dieses Kontrollkästchen ist nur verfügbar, wenn die Mittelwertlinie auf der Registerkarte Linie aktiviert ist. Es legt fest, ob der Legendeneintrag für die Mittelwertlinie gezeigt wird. Das Diagrammsymbol folgt den Stileinstellungen der Mittelwertlinie auf der Registerkarte Linie, gefolgt von dem Text Mittelwertlinie.
Max.

Dieses Kontrollkästchen ist nur verfügbar, wenn das Kontrollkästchen Max auf der Registerkarte Prozentangaben für das Boxdiagramm aktiviert ist. Es legt fest, ob der Legendeneintrag für den Perzentilpunkt beim Maximum gezeigt wird. Der Stil des Diagrammsymbols folgt den Einstellungen von Max auf der Registerkarte Prozentangaben, und der Legendentext ist Max.
99% Dieses Kontrollkästchen ist nur verfügbar, wenn das Kontrollkästchen 99 % auf der Registerkarte Prozentangaben für das Boxdiagramm aktiviert ist. Es legt fest, ob der Legendeneintrag für den Perzentilpunkt bei 99% gezeigt wird. Der Stil des Diagrammsymbols folgt den Einstellungen von 99% auf der Registerkarte Prozentangaben, und der Legendentext ist 99%.
Mittelwert Dieses Kontrollkästchen ist nur verfügbar, wenn das Kontrollkästchen Mittelwert auf der Registerkarte Prozentangaben für das Boxdiagramm aktiviert ist. Es legt fest, ob der Legendeneintrag für den Perzentilpunkt beim Mittelwert gezeigt wird. Der Stil des Diagrammsymbols folgt den Einstellungen des Mittelwerts auf der Registerkarte Prozentangaben, und der Legendentext ist Mittelwert.
Mittelwertidentifizierer Dieses Kontrollkästchen ist nur verfügbar, wenn das Kontrollkästchen Mittelwert auf der Registerkarte Prozentangaben und das Kontrollkästchen Mittelwert oben aktiviert ist. Die ersten vier verwenden die Variable in der entsprechenden Spaltenbeschriftungszeile der Quellendatenspalte.
  • Kurzname
  • Langname
  • Einheiten
  • Kommentare
  • Beliebige kategoriale Gruppenspalten: Spalten mit kategorialen Daten, die zum Definieren der Boxdiagrammgruppen verwendet werden
  • <Benutzerdefiniert>: Wird mit dem Benutzerdefinierten Identifiziererformat verwendet, um die Textzeichenkette zu erstellen.
  • <Mittelwert>: Das Mittelwertsymbol der ersten Zeichnung wird verwendet.
Median Dieses Kontrollkästchen ist nur verfügbar, wenn das Kontrollkästchen Median auf der Registerkarte Prozentangaben für das Boxdiagramm aktiviert ist. Es legt fest, ob der Legendeneintrag für den Perzentilpunkt beim Medianwert gezeigt wird. Der Stil des Diagrammsymbols folgt den Einstellungen von Median auf der Registerkarte Prozentangaben, und der Legendentext ist Median.
Medianidentifizierer Dieses Kontrollkästchen ist nur verfügbar, wenn das Kontrollkästchen Median auf der Registerkarte Prozentangaben und das Kontrollkästchen Median oben aktiviert ist. Die ersten vier verwenden die Variable in der entsprechenden Spaltenbeschriftungszeile der Quellendatenspalte.
  • Kurzname
  • Langname
  • Einheiten
  • Kommentare
  • Beliebige kategoriale Gruppenspalten: Spalten mit kategorialen Daten, die zum Definieren der Boxdiagrammgruppen verwendet werden
  • <Benutzerdefiniert>: Wird mit dem Benutzerdefinierten Identifiziererformat verwendet, um die Textzeichenkette zu erstellen.
  • <Median>: Das Mediansymbol der ersten Zeichnung wird verwendet.
1% Dieses Kontrollkästchen ist nur verfügbar, wenn das Kontrollkästchen 1% auf der Registerkarte Prozentangaben für das Boxdiagramm aktiviert ist. Es legt fest, ob der Legendeneintrag für den Perzentilpunkt bei 1% gezeigt wird. Der Stil des Diagrammsymbols folgt den Einstellungen von 1% auf der Registerkarte Prozentangaben, und der Legendentext ist 1%.
Min. Dieses Kontrollkästchen ist nur verfügbar, wenn das Kontrollkästchen Min. auf der Registerkarte Prozentangaben für das Boxdiagramm aktiviert ist. Es legt fest, ob der Legendeneintrag für den Perzentilpunkt beim Minimum gezeigt wird. Der Stil des Diagrammsymbols folgt den Einstellungen von Min auf der Registerkarte Prozentangaben, und der Legendentext ist Min.
Benutzerdefinierte Perzentile Dieses Kontrollkästchen ist nur verfügbar, wenn der Boxtyp nicht Daten ist. Es legt fest, ob der Legendeneintrag für den Perzentilpunkt beim benutzerdefinierten Wert gezeigt wird. Der Stil des Diagrammsymbols folgt den Einstellungen von Benutzerdefinierte Perzentile auf der Registerkarte Prozentangaben, und der Legendentext ist Benutzerdefinierte Perzentile. Wenn zum Beispiel P=5 festgelegt ist, lautet der angezeigte Legendentext 5%, 95%.
Daten Dieses Kontrollkästchen ist nur verfügbar, wenn der Boxtyp nicht Box/ Balken ist oder es sich um ein Säulen-Punktdiagramm handelt. Es dient dazu festzulegen, ob der Eintrag für die Daten in der Legende angezeigt wird. Der Stil des Datenpunkts folgt den Einstellungen auf der Registerkarte Symbol, und der Text ist Datenidentifizierer.

Hinweis: Wenn die Zeichnung für die Datenpunkte die Anforderungen zum Zeichen der Legende der kategorialen Werte oder Punkt-für-Punkt-Legende erfüllt, werden die Legendeneinträge, die auf diesen beiden Legendenobjekten basieren, an die Legende für Boxkomponenten angehängt.
Datenidentifizierer Dieses Kontrollkästchen ist nur verfügbar, wenn der Boxtyp nicht Box/ Balken ist oder es sich um ein Säulen-Punktdiagramm handelt und das Kontrollkästchen Daten oben aktiviert ist. Die ersten vier verwenden die Variable in der entsprechenden Spaltenbeschriftungszeile der Quellendatenspalte.

Benutzerdefiniertes Format des Identifizierers Diese Option ist nur verfügbar, wenn für Datenidentifizierer die Option <Benutzerdefiniert> gewählt wurde. Legen Sie Ihren Identifizierer mit einer folgenden Optionen fest:
  • Wählen Sie ein benutzerdefiniertes Standardformat aus dem Auswahlmenü.
  • Klicken Sie auf die Schaltfläche mit dem Pfeil rechts vom Kombinationsfeld.
  • Geben Sie Ihre eigenen Zeichenketten für den Identifizierer direkt ein. Bitte lesen Sie Einzelheiten dazu in der vollständigen Liste von @Optionen.
Ausreißer Diese Option ist nur verfügbar, wenn Ausreißer auf der Registerkarte Box im Dialog Details Zeichnung aktiviert ist. Erstellt einen Legendeneintrag für Ausreißer. Das Legendensymbol ist das gleiche wie für den Ausreißerdatenpunkt. Der Legendentext ist Ausreißer.
Extremwerte Diese Option ist nur verfügbar, wenn Ausreißer auf der Registerkarte Box im Dialog Details Zeichnung aktiviert ist. Erstellt einen Legendeneintrag für extreme Werte. Das Legendensymbol ist das gleiche wie für den extremen Datenpunkt. Der Legendentext ist Extreme Werte.
Mittelwert verbinden Diese Option ist nur verfügbar, wenn Mittelwert verbinden auf der Registerkarte Verbindungslinien im Dialog Details Zeichnung aktiviert ist. Erstellt einen Legendeneintrag für Linien, die die Mittelwertpunkte der Box verbinden. Das Legendensymbol folgt den Einstellungen von Mittelwert verbinden auf der Registerkarte Verbindungslinien, und der Legendentext ist Mittelwert.
Medianlinie verbinden Diese Option ist nur verfügbar, wenn Mittelwert verbinden auf der Registerkarte Verbindungslinien im Dialog Details Zeichnung aktiviert ist. Erstellt einen Legendeneintrag für Linien, die die Medianpunkte der Box verbinden. Das Legendensymbol folgt den Einstellungen der Gruppe Median verbinden auf der Registerkarte Verbindungslinien, und der Legendentext ist Median.
Datenpunkte verbinden Diese Option ist nur verfügbar, wenn Datenpunkte verbinden auf der Registerkarte Verbindungslinien im Dialog Details Zeichnung aktiviert ist. Erstellt einen Legendeneintrag für Linien, die die Datenpunkte für jede Variable verbinden. Das Legendensymbol folgt den Einstellungen der Gruppe Datenpunkte verbinden auf der Registerkarte Verbindungslinien, und der Legendentext ist Daten. Wenn die Farbe für die Linien, die die Datenpunkte verbnden, auf eine Spalte mit kategorialen Daten indexiert ist, listet der Legendentext die Kategorien auf.
Perzentile verbinden Diese Option ist nur verfügbar, wenn Perzentile verbinden auf der Registerkarte Verbindungslinien im Dialog Details Zeichnung aktiviert ist. Erstellt einen Legendeneintrag für Linien, die bestimmte Perzentile verbinden. Das Legendensymbol folgt den Einstellungen von Perzentile verbinden auf der Registerkarte Verbindungslinien, und der Legendentext ist Perzentile.
