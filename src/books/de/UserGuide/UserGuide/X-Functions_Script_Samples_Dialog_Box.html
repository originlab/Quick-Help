Dialogfeld Skriptbeispiel für X-Funktionen

Das Dialogfeld XF Script ist eine Sammlung von allen Skripts für X-Funktionen oder Beispielen zur Implementierung bestimmter Aufgaben. Auf das Dialogfeld XF Script kann auf zwei Arten zugegriffen werden:

  1. Drücken Sie F11.
  2. Wählen Sie im Menü Hilfsmittel: X-Funktion Skriptbeispiele.

Es ist möglich, eine X-Funktion im linken Bedienfeld auszuwählen und dann den Gebrauch der X-Funktionen oder Beispielskripts (nur X-Funktionen von Anwendern) im rechten Bedienfeld anzuzeigen bzw. zu modifizieren.

Hinweis: X-Funktion, die über Skriptbeispiele verfügen, verwenden fette Namen im linken Bedienfeld. Bei X-Funktionen des Systems können Sie die Gebrauchs- und Skriptbeispiele in diesem Dialogfeld nicht bearbeiten. Diese X-Funktionsnamen verwenden blaue Schrift.

Die Beschreibungen der Bedienelemente sind wie folgt:

Auswahllisten

Ort

Legt fest, welche Art von X-Funktionen Sie anzeigen möchten, einschließlich standardmäßige X-Funktionen des Systems, benutzerdefinierte X-Funktionen und innerhalb einer Gruppe gemeinsam genutzte X-Funktionen.

Beispiel

Legt fest, welche Arten von Beispielen Sie anzeigen möchten, Gebrauch von X-Funktion oder Skriptbeispiel.
Schaltflächen

Speichern

Legt fest, welche Art von X-Funktionen Sie anzeigen möchten, einschließlich standardmäßige X-Funktionen des Systems, benutzerdefinierte X-Funktionen und innerhalb einer Gruppe gemeinsam genutzte X-Funktionen.

Ausführen

Ausführen des Skriptbeispiels

Schließen

Schließen Sie das Dialogfeld.
Oder

Erweitern oder verkleinern Sie das untere Bedienfeld usage / variable information.
