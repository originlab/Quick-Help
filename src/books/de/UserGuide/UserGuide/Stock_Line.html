Datenanforderungen

Markieren Sie eine oder mehrere Y-Spalten (oder einen Bereich aus einer oder mehreren Y-Spalten). Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie Zeichnen: Spezialisiert: Liniendiagramm.

oder

Klicken Sie auf die Schaltfläche Liniendiagramm (Gruppe Kursdiagramme) auf der Symbolleiste 2D Grafiken.

Vorlage

StockLine.otp (im Origin-Programmordner installiert)

Hinweise

Die Vorlage des (Kurs-)Liniendiagramms ist für das Zeichnen von Finanzdaten leicht benutzerdefiniert angepasst:

  • Die Achse Skalierungstyp ist auf Diskret gesetzt, um handelsfreie Daten auszuschließen.
  • Die Option Rand neu skalieren (%) ist auf 0 gesetzt.
