Die Registerkarte Zeichensätze (Optionen)

Options-Dialog-TextFont-Tab


Verwenden Sie diese Bedienelemente, um verschiedene Optionen für Zeichensatzformate der Beschriftungen anzugeben.

Inhalt

  • 1 Gruppe Text-Hilfsmittel
  • 2 Gruppe Aktueller Zeichensatz (in %)
  • 3 Zusätzliche Bedienelemente

Gruppe Text-Hilfsmittel

Schriftart

Wählen Sie die Standardschrift aus, die angezeigt werden soll, wenn Sie eine Beschriftung mit dem Hilfsmittel Text erstellen.
Griechisch

Wählen Sie den Zeichensatz aus, der verwendet werden soll, wenn Sie auf die Schaltfläche Griechisch auf der Symbolleiste Format oder auf die Schaltfläche Griechisch in dem Dialog Texteinstellungen klicken.
Farbe

Wählen Sie die Standardfarbe aus, die angezeigt werden soll, wenn Sie eine Beschriftung mit dem Hilfsmittel Text erstellen.
Größe

Wählen Sie die Standard-Textgröße aus, die verwendet werden soll, wenn Sie eine Beschriftung mit Hilfe des Hilfsmittels Text erstellen, oder geben Sie diese ein.

Informationen zum Festlegen von Schriftarten, die auf der Origin-Bedienoberfläche verwendet werden, siehe LabTalks system.font(object).

Gruppe Aktueller Zeichensatz (in %)

Zeilenabstand

Dieser Wert des Auswahlfeldes steuert den Abstand zwischen Linien in einer Beschriftung wie einen Prozentsatz der aktuellen Zeichensatzgröße + der aktuellen Zeichensatzgröße.

Beispielsweise erzeugen eine Zeichensatzgröße von 20 und ein Zeilenabstand von 10% einen Abstand von 22 Punkten zwischen den Textzeilen innerhalb einer einzelnen Beschriftung.

Der Einstellung des Linienabstands kann mit dem Projekt gespeichert werden.
Größe tiefgestellt und hochgestellt

Dieser Wert des Auswahlfeldes steuert die Größe von tief- und hochgestellten Zeichen in einer Beschriftung wie einen Prozentsatz der aktuellen Zeichensatzgröße.
Verschiebung tiefgestellt

Dieser Wert des Auswahlfeldes steuert den Versatz eines tiefgestellten Zeichens von der Basislinie einer Beschriftung wie einen Prozentsatz der aktuellen Zeichensatzgröße. Tiefgestellt erscheint in einem angegebenen Prozentsatz unterhalb der Basislinie.
Verschiebung hochgestellt

Dieser Wert des Auswahlfeldes steuert den Versatz eines hochgestellten Zeichens von der Oberkante einer Beschriftung wie einen Prozentsatz der aktuellen Zeichensatzgröße. Hochgestellt erscheint in einem angegebenen Prozentsatz oberhalb der Oberkante.

Zusätzliche Bedienelemente

Standard

Wählen Sie den Zeichensatz aus, der verwendet werden soll, wenn Standard im Dialog Texteinstellung ausgewählt ist.

Hinweis: Wenn Sie ein Projekt eines Kollegen öffnen, bei dessen Erstellung der "Standard"-Zeichensatz verwendet wurde, beachten Sie bitte, dass eventuell einige Unterschiede bestehen, da Origin die Textbeschriftungen mit Hilfe Ihres Standardzeichensatzes anzeigt..
Schrift mit Basislinie verbinden

Dieses Kontrollkästchen wird auf alle Beschriftungen angewendet, die mehrere Zeichensätze verwenden. Wenn dieses Kontrollkästchen aktiviert ist, werden die Unterkanten der Buchstaben und Symbole einer Beschriftungszeile horizontal ausgerichtet (bei den meisten Zeichensätzen). Dies ist nützlich, wenn verschiedene Zeichensätze in derselben Beschriftungszeile verwendet werden, besonders wenn griechische Symbole verwendet werden.
Größenverhältnis der Fußnote Ändern Sie das Größenverhältnis der Fußnote in allen Analyseberichten. Der Wert ist ein Prozentsatz der Standardschriftgröße. Lesen Sie in dieser FAQ Informationen zur Verwendung.
Text für direkte Bearbeitung horizontal setzen

Aktivieren Sie dieses Kontrollkästchen, um die Beschriftungen während des direkten Bearbeitens in eine horizontale Position zu drehen. Die ursprüngliche Ausrichtung wird wieder hergestellt, wenn die Auswahl der Beschriftung deaktiviert wurde.

Deaktivieren Sie dieses Auswahlfeld, um die wirkliche Textausrichtung während des Bearbeitens beizubehalten.
Proben Auswahlmodus

Aktivieren Sie dieses Kontrollkästchen, um den Zugriff auf die Dreh- und die Verzerrpunkte einer Beschriftung durch einen Mausklick zu deaktivieren.
Direktes Bearbeiten deaktivieren

Aktivieren Sie dieses Kontrollkästchen, um den direkten Bearbeitungsmodus zu deaktivieren. Wenn der Modus direktes Bearbeiten ausgeschaltet ist, öffnet ein Doppelklick auf die Beschriftung das Dialogfeld Texteigenschaften.
