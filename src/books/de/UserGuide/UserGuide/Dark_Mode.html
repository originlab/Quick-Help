Dunkelmodus

Dark-Mode


Seit Origin 2024 wird der Dunkelmodus für die Origin-Bedienoberfläche unterstützt.

Wir verwenden die Symbolleiste Dunkelmodus oder die Systemvariable, um zu steuern, ob die Bedienoberfläche in den Dunkelmodus versetzt wird.

Dunkelmodus von GUI umschalten

Sie können das Menü bzw. die Schaltfläche verwenden oder die Systemvariable festlegen, um die GUI auf Dunkelmodus umzuschalten.

  • Wählen Sie im Menü Einstellungen: Dunkelmodus.
Wenn die Symbolleiste Dunkelmodus in diesem Moment verborgen wird, werden Sie per Hinweis gefragt, ob sie angezeigt werden soll.
  • Klicken Sie auf die Schaltfläche GUI umschalten auf der Symbolleiste Dunkelmodus.
  • Um die Origin-Oberfläche in den Dunkelmodus zu versetzen, legen Sie für die Systemvariable @CTP = 1 fest. Um die Origin-Oberfläche in den Dunkelmodus zu deaktivieren, legen Sie für die Systemvariable @CTP = 2 fest.

Dunkeldesign

Origin wendet mehrere Designs für den Dunkelmodus an. Wenn die Origin GUI sich im Dunkelmodus befindet, können Sie Einstellungen: Dunkeldesign auswählen, um den die Designstilliste im Menü zu ändern.


Dunkeldesign benutzerdefiniert anpassen

Entsprechend dem Standarddunkeldesign können Sie sie benutzerdefiniert anpassen und ein benutzerdefiniertes Dunkeldesign hinzufügen.

  1. Wählen Sie im Menü ein Standarddunkeldesign aus.
  2. Führen Sie das folgende Skript im Skriptfenster aus:
    mem -pw;

    Die aktuellen Farben des Dunkelmodus werden in eine Arbeitsmappe geladen.

  3. Klicken Sie die Farbzelle und verwenden Sie die Minisymbolleiste, um die Farbe zu ändern. Hier ändern Sie die WindowTitleBarActive.
  4. Führen Sie im Skriptfenster das folgende Skript aus, um das Design als MyDark zu speichern.
    mem -pe "MyDark";
  5. Wählen Sie das benutzerdefinierte Design im Menü Einstellungen: Dunkeldesign.
  6. Sie könnten die Titelleiste des aktiven Fensters im Dunkelmodus aktualisiert vorfinden.


Dunkelmodus für Fenstertyp umschalten

Wenn sich die Origin-Oberfläche im Dunkelmodus befindet, können Sie Schaltfläche oder Tastenkombination verwenden, um den Dunkelmodus des Fensters nach Fenstertyp global umzuschalten.

  • Schaltfläche Dunkelmodus nach Fenstertyp auf der Symbolleiste Dunkelmodus.
  1. Aktivieren Sie ein/e Arbeitsmappe/Matrix/Diagramm oder Layout-/Bild-/Notizfenster.
  2. Wenn Sie auf diese Schaltfläche klicken, werden Fenster des gleichen Typs in den Dunkel- bzw. Hellmodus geschaltet.
  • Tastenkombination: Alt + 5
  1. Aktivieren Sie ein/e Arbeitsmappe/Matrix/Diagramm oder Layout-/Bild-/Notizfenster.
  2. Drücken Sie die Tasten Alt + 5. Fenster des gleichen Typs werden in den Dunkel- bzw. Hellmodus geschaltet.


Dunkelmodus für aktuelles Fenster umschalten

Wenn sich die Origin-Oberfläche im Dunkelmodus befindet, können Sie diese Methoden verwenden, um das aktuelle Fenster in den Dunkelmodus umzuschalten.

  • Schaltfläche Aktuelles Fenster auf der Symbolleiste Dunkelmodus.
  1. Aktivieren Sie ein Fenster.
  2. Klicken Sie auf diese Schaltfläche. Dieses Fenster wird dann in den Dunkel- bzw. Hellmodus geschaltet.
  • Tastenkombination: Strg + 5
  1. Aktivieren Sie ein/e Arbeitsmappe/Matrix/Diagramm oder Layout-/Bildfenster.
  2. Drücken Sie die Tasten Strg + 5. Dieses Fenster wird dann in den Dunkel- bzw. Hellmodus geschaltet.
  • Labtalk-Skript
  1. Aktivieren Sie ein/e Arbeitsmappe/Matrix/Diagramm oder Layout-/Bildfenster.
  2. Führen Sie dieses Skript im Skriptfenster aus. Dieses Fenster wird dann in den Hellmodus geschaltet.
    page.nodark=1;
  3. Führen Sie das folgende Skript aus. Dieses Fenster wird dann in den Dunkelmodus geschaltet.
    page.nodark=0;

Diagramm-/Layoutfenster

Dunkelmodus unabhängig umschalten

Diagramm-/Layoutfenster können nicht nur in den Dunkelmodus geschaltet werden, wenn sich die Origin-Bedienoberfläche im Dunkelmodus befindet. Sie können unabhängig von der GUI in den Dunkelmodus geschaltet werden. Wenn Diagramm-/Layoutfenster aktiviert sind, sind die Schaltflächen Dunkelmodus nach Fenstertyp und Aktuelles Fenster verfügbar.

Farbe für Dunkelmodus

Hintergrundfarbe

Wenn das Diagramm-/Layoutfenster in den Dunkelmodus geschaltet wird, ändert sich der Seitenhintergrund (automatische Farbe) automatisch in die Farbe des Dunkelmodus. Um die Farbe des Dunkelmodus benutzerdefiniert anzupassen, können Sie:

  • Klicken Sie auf die Schaltfläche Hintergrundfarbe auf der Symbolleiste Dunkelmodus, um die Hintergrundfarbe des Diagramms im Dunkelmodus benutzerdefiniert anzupassen.


oder

  • Einstellungen: Optionen auswählen und zur Registerkarte Seite gehen. Passen Sie dann die Hintergrundfarbe im Abschnitt Automatische Farbe für Dunkelmodus benutzerdefiniert an.
  • Wenn Sie die Anzeigefarbe des Diagramms im Dialog Details Zeichnung benutzerdefiniert angepasst haben (keine automatische Farbe), behält es die von Ihnen angepasste Farbe beim Umschalten in den Dunkelmodus.
  • Wenn die dynamische Farbumkehrung auf Nur Schwarz oder Schwarz und Weiß gesetzt ist, wird Schwarz in Weiß umgekehrt. Das wird als Vordergrundfarbe bezeichnet. Sie können die Vordergrundfarbe auf der Registerkarte Seite im Dialog Einstellungen: Optionen benutzerdefiniert anpassen.


Dynamische Umkehrung steuern

Die dynamische Farbumkehrung wird durch die Leuchtdichte der Hintergrundfarbe des Dunkelmodus ausgelöst. Sie kann mit der Systemvariable @GLD gesteuert werden.

  • Der Standardwert ist @GLD = 28, so dass die Farbe nur für sehr dunklen Hintergrund umgekehrt wird.

Wenn Sie den Hintergrund des Dunkelmodus zum Beispiel Dunkelgrau setzen, werden schwarze Zeichnungen, Achsen und Beschriftungen etc. in Weiß umgekehrt.

  • Wenn Sie den Hintergrund des Dunkelmodus allerdings auf Grau setzen, wird die Farbe nicht umgekehrt. Wenn Sie @GLD = 60 festlegen, auch für graue Hintergründe, werden die Farben umgekehrt.


Dynamische Farbumkehrung

Wenn sich das Diagrammfenster im Dunkelmodus befindet, wird die Farbe der Zeichnungen und Objekte im Diagramm umgekehrt. Origin bietet mehrere Stufen für die Farbumkehrung, um verschiedenen Anforderungen zu entsprechen. Sie kann auf der Registerkarte Seite im Dialog Einstellungen: Optionen angepasst werden. Es gibt auch eine Systemvariable @DCR, mit der gesteuert werden kann, ob die Farbe beim Umschalten in den Dunkelmodus umgekehrt wird.

Wert für @DCR Option Farbumkehrung
0 Kein Die Farbumkehrung ist deaktiviert.
1 Nur Schwarz Es wird nur Schwarz in Weiß umgekehrt (Vordergrundfarbe).
2 Schwarz und Weiß Nur Schwarz wird in Weiß umgekehrt (Vordergrundfarbe) und Weiß wird in Schwarz umgekehrt.
3 Graue Farben Alle grauen Farben werden von Schwarz in Weiß umgekehrt, z. B. Dunkelgrau wird in Hellgrau umgekehrt.
4 Alle Farben Alle Farben werden umgekehrt, ähnlich wie in MS Word. Falls Sie z. B. Hellgelb wählen, wird dies eventuell als Dunkelgelb gezeigt.
5 Benutzerdefiniert (Standard) Wenn die Farbe im Arbeitsblatt Farbbildung im Dunkelmodus bearbeiten aufgeführt ist (klicken Sie auf Einstellungen: Farbbildung des Dunkelmodus, um es zu öffnen), wird diese Farbe beim Umschalten vom Hell- in den Dunkelmodus in die Abbildungsfarbe im Arbeitsblatt umgekehrt, einschließlich der Zeichnungs- und Diagrammobjektfarbe.


  • Wenn Sie die umgekehrte Farbe steuern möchten, können Sie im Arbeitsblatt Farbbildung im Dunkelmodus bearbeiten den Hex-Farbwert (HTML) hinzufügen, um die Farbe im Hellmodus und ihren Gegenpart im Dunkelmodus festzulegen. Klicken Sie nach der Anpassung auf die Schaltfläche Speichern. Sie wird in der Datei DarkColors.ini im Ordner \User Files\Themes\Graph\ gespeichert.
  • Sie können auch auf die Schaltfläche Zurücksetzen klicken, um sie in der Standardfarbliste zurückzusetzen.

Wenn Sie die Farbabbildung in der Zeichnung verwenden, wird die Farbe nicht in jeder Situation umgekehrt.

Tabelle/ Arbeitsblatt im Diagramm

Per Standard folgen die/das eingefügte Tabelle/Arbeitsblatt im Diagramm dem Dunkelmodus des Diagramms. Wenn Sie ändern möchten, dass die eingefügte Tabelle dem Dunkelmodus des Arbeitsblatts folgt, können Sie die Systemvariable @TCP = 1 setzen (Standardwert dieser Systemvariable ist 0).

Seite im Dunkelmodus wie auf Bildschirm kopieren

Angenommen, der Hintergrund ist dunkel oder sogar ein dunkles Bild in Ihrem Word, Power Point usw. und Sie möchten die Farbe der Zeichnungen und Objekte in heller Farbe wie im Dunkelmodus beibehalten, dann können Sie die Option Einstellungen: Seite im Dunkelmodus wie auf Bildschirm kopieren aktivieren, bevor Sie die Seite kopieren. Der eingefügte Diagrammhintergrund ist transparent, der Hintergrund kann in Word/PPT weiterhin gezeigt werden.

Hinweise: Wenn Sie die Anzeigefarbe des Diagramms im Dialog Details Zeichnung benutzerdefiniert angepasst haben (keine automatische Farbe), behält das Diagramm diese Hintergrundfarbe und wird nicht transparent.

Diagramm im Dunkelmodus exportieren

  • Wählen Sie bei aktiviertem Dunkelmodus Datei: Grafiken exportieren oder Grafiken exportieren (Erweitert), um Diagramme zu exportieren. Es gibt die Option Farbe des Dunkelmodus folgen, mit der Sie steuern können, ob die exportierten Diagramme und Diagrammobjekte mit der Farbe im Dunkelmodus übereinstimmen.
  • Wenn Sie im Dunkelmodus ein Diagramm mit Farbe des Dunkelmodus folgen exportieren, können Sie die Option Transparenter Hintergrund verwenden, um zu bestimmen, ob der exportierte Diagrammhintergrund die Farbe wie im Dunkelmodus beibehalten oder zum transparenten Hintergrund zurückkehren soll.
Hinweise: Wenn Sie die Anzeigefarbe des Diagramms im Dialog Details Zeichnung benutzerdefiniert angepasst haben (keine automatische Farbe), behält das Diagramm diese Hintergrundfarbe, unabhängig davon, ob Sie die Option Transparenter Hintergrund aktiviert haben.


Es gibt eine Systemvariable @GEFD, mit der Sie bestimmen können, ob der Grafikexport dem Dunkelmodus folgt. Standardmäßig ist @GEFD = 1, das heißt, die exportierten Grafiken folgen der Farbe im Dunkelmodus. Wenn sie auf Null gesetzt ist, wird der Dunkelmodus ignoriert.

Code Builder

Wenn sich die Origin-GUI im Dunkelmodus befindet, folgt der Code Builder diesem Design der GUI.

Wenn Sie die Textfarbe im Code Builder ändern möchten, wählen Sie Hilfsmittel: Optionen im Menü des Code Builders, um den Dialog zu öffnen. Die Farbe kann auf der Seite Schriftarten und Farben im Dialog Optionen benutzerdefiniert angepasst werden. Wenn Sie die Farbe zurücksetzen möchten, wählen Sie Hilfsmittel: Optionen zurücksetzen.
