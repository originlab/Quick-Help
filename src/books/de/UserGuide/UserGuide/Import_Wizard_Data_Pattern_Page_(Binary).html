Importassistent, Datenmusterseite (Binär)

Auf dieser Seite definieren Sie die Struktur Ihrer Binärdatei, indem Sie die Reihenfolge, den Typ und das Datum pro Aufzeichnung festlegen.

Muster

Spalte

Diese Spalte kann nicht bearbeitet werden. Es erscheinen die Spaltenindexnummern.
Typ

Klicken Sie auf einen Datentyp in der Auswahlliste und wählen Sie diesen aus (die Auswahlmöglichkeiten sind ganze Zahl, ganze Zahl ohne Vorzeichen, Real und String).
Größe

Sobald ein Dateityp ausgewählt wurde, wird die Spalte Größe automatisch mit dessen typischer Größe ausgefüllt (mit Ausnahme von String; in diesem Fall bleibt die Spalte leer). Der Spaltentext Größe kann bearbeitet werden. Beispielsweise ist die Standardlänge für int 4, Sie können sie jedoch auch durch 2 ersetzen, wenn Sie einen short int möchten. Für Zeichenketten sollten Sie die Länge in Zeichen festlegen.
Anzahl

Nutzen Sie die Spalte Anzahl, um festzulegen, wie häufig Datenspalten diesen Typs in der Aufzeichnung wiederholt werden.
Einfügen

Zeilen werden automatisch hinzugefügt, wenn Sie Ihre Datenstruktur festlegen. Klicken Sie auf diese Schaltfläche, um eine Zeile hinzuzufügen.
Löschen

Zeilen werden automatisch hinzugefügt, wenn Sie Ihre Datenstruktur festlegen. Klicken Sie auf diese Schaltfläche, um eine Zeile hinzuzufügen.

Datenansicht

Hex

Wählen Sie diese Option, wenn Sie die Daten in Byteform sehen möchten.
ASCII

Wählen Sie diese Option, wenn Sie die Daten in ASCII-Form sehen möchten.

Vorschaufenster

Wenn eine Strukturdefinition abgeschlossen ist, zeigt das Vorschaufenster die importierten Daten auf Basis der bisher festgelegten Aufzeichnungsstruktur so an, wie sie im sich ergebenden Arbeitsblatt aussehen werden. Beachten Sie, dass Sie mit einem Klick der rechten Maustaste auf die Spaltenheader im Vorschaufenster die Diagrammzuordnung und das Format der Spalten des Zielarbeitblatts einstellen können.

Zielspaltenformat beibehalten: Verwenden Sie dieses Kontrollkästchen, um festzulegen, ob das Zielspaltenformat beibehalten werden soll. Wenn dieses Kontrollkästchen aktiviert ist, ist das Kontextmenü Format setzen, das aufgeht, wenn Sie mit der rechten Maustaste auf den Spaltenheader des Vorschaufensters klicken, nicht verfügbar.
