Datenanforderungen

Wählen Sie eine Y-Spalte oder einen XY-Bereich aus einem Arbeitsblatt aus.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen: Kontur: 2D-Kernel-Dichtediagramm.

oder

Klicken Sie auf die Schaltfläche 2D-Kernel-Dichtediagramm in der Symbolleiste 2D Grafiken.

Origin öffnet den Dialog plot_kde2.

Vorlage

kdCONTOUR.OTP (installed to the Origin program folder)
