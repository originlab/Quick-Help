Die Registerkarte (Details Zeichnung) Steigender/Fallender Preis

PD-Dialog-AdvDecPrice-Tab

Die Registerkarten Steigender/Fallender Preis bietet Bedienelemente für Kursdiagramme wie Japanische Candlesticks, OHLC-Balkendiagramm und OHLC-Volumendiagramm. Die Kursdiagramme enthalten normalerweise zwei verschiedene Diagrammelemente, erstellt von 4 Spalten, die die Eröffnungs-, Höhst-, Tiefst- und Schlusspreise enthalten. Die erste Elementart bedeutet, dass der Schlusswert höher ist als der Eröffnungswert. Seine Darstellung wird auf der Registerkarte Steigender Preis festgelegt. Die zweite Elementart bedeutet, dass der Schlusswert niedriger ist als der Eröffnungswert. Seine Darstellung wird auf der Registerkarte Fallender Preis festgelegt. Die Bedienelemente der zwei Registerkarten sind die gleichen, haben aber unterschiedliche Standardeinstellungen.

Inhalt

  • 1 Rahmen/Linie
  • 2 Füllung
  • 3 Vorschau
  • 4 Gradientenfüllung

Rahmen/Linie

Farbe

Legen Sie eine Rahmen-/Linienfarbe fest.
Stil

Wählen Sie einen Rahmen-/Linienstil aus.
Breite

Wählen Sie oder geben Sie eine Rahmen-/Linienbreite aus. Die Breite wird in Punkten gemessen, wobei 1 Punkt=1/72 Zoll entsprechen.

Hinweis: Sie können die gestrichelten Linienmuster von Origin benutzerdefiniert anpassen, indem die Gruppe der gestrichelten Linien auf der Registerkarte Grafik des Dialogs Optionen (Einstellungen: Optionen) bearbeiten.

Füllung

Diese Gruppenoptionen können nur bearbeitet werden, wenn ein Diagramm mit Japanischen Candlesticks oder ein OHLC-Volumendiagramm erstellt wurde.

Farbe

Legen Sie eine Füllfarbe fest. Wählen Sie Kein aus, um eine transparente Füllung anzuzeigen.
Muster

Wählen Sie ein Füllmuster.
Musterfarbe

Wenn ein Muster in der Liste Muster ausgewählt wird, ist die Schaltfläche Musterfarbe verfügbar. Wählen Sie eine Farbe über diese Schaltfläche aus. Wenn Automatisch ausgewählt ist, verwendet Origin die Schaltfläche Randfarbe zum Festlegen der Musterfarbe.
Breite

Legt die Breite der Linien in dem Füllmuster fest. Die Linienbreite wird in Punktgröße gemessen, wobei 1 Punkt = 1/72 Zoll ist.

Vorschau

Origin zeigt Ihre aktuellen Registerkarteneinstellungen in diesem Vorschaufeld an.

Gradientenfüllung

Modus Legt den Modus der Gradientenfüllung fest.
  • Keine
Keine Gradientenfarbänderung
  • Eine Farbe
Eine Farbe verwandelt sich allmählich von schwarz in weiß.
  • Zwei Farben
Die Füllung verwandelt sich allmählich von eine Farbe in eine andere.
Helligkeit Diese Option ist nur verfügbar, wenn der Modus auf eine Eine Farbe gesetzt ist. Die Helligkeit (in %) der Gradientenfüllfarbe wird hier mit einem Schiebeelement eingestellt und variiert zwischen schwarz und weiß.
Farbe Diese Option ist nur verfügbar, wenn der Modus auf eine Zwei Farben gesetzt ist. Wählen Sie die zweite Farbe, die zum Erstellen des Gradienten verwendet werden soll.
Richtung Diese Option ist verfügbar, wenn der Modus auf Eine Farbe oder Zwei Farben festgelegt ist. Mit ihr können Sie die Richtung festlegen, entlang der sich die Füllfarbe allmählich verändert.
