Eingabe

Wählen Sie Daten für die Partiellen kleinsten Quadrate.

Unabhängige Variablen Wählen Sie Daten, um die unabhängigen Variablen festzulegen.
Abhängige Variablen Wählen Sie Daten, um die abhängigen Variablen festzulegen.
Beobachtungsbeschriftungen Wählen Sie die Daten aus, um die Beschriftung für die Beobachtung festzulegen. Die Beschriftungen werden im Ergebnis in den Scorediagrammen und XY-Scorediagrammen gezeigt.
Antworten prognostizieren Bestimmen Sie, ob aus dem erzeugten Modell der partiellen kleinsten Quadrate Antworten prognostiziert werden sollen. Wenn dieses Kontrollkästchen aktiviert ist, wird Unabhängige Variablen für Prognose angezeigt.
Unabhängige Variablen für Prognose Wählen Sie daten, um die unabhängigen Variablen für die Prognose festzulegen.

Einstellungen

Legen Sie die Einstellungen für die Partiellen kleinsten Quadrate fest.

Methode Wählen Sie die Methode, um die extrahierten Faktoren zu berechnen.
  • SVD
  • Wolds Iteration

Hinweis: Weitere Einzelheiten zu diesen beiden Methoden finden Sie auf der Seite Algorithmus.
Skalierungsvariablen Wählen Sie die Methode, um unabhängige Variablen durch die Standardabweichung zu teilen.
Maximale Anzahl von Faktoren Legen Sie die maximale Anzahl von Faktoren fest, die im Regressionsmodell enthalten sind.
Kreuzvalidierung Legen Sie fest, ob das Regressionsmodell der partiellen kleinsten Quadrate mit Hilfe der Kreuzvalidierung erzeugt werden soll.

Eigenschaften

Deskriptive Statistik Legen Sie fest, ob grundlegende Statistiken (N in Analyse, N fehlende Werte, Mittelwert, Standardabweichung) für jede unabhängige und abhängige Variable durchgeführt werden sollen. Die Ergebnisse werden im Blatt PLSResultsn ausgegeben.
Koeffizienten Legen Sie fest, ob die Koeffizienten des angepassten Modells ausgegeben werden. Die Koeffizienten werden im Blatt PLSResultsn eingefügt.
Standardisierte Koeffizienten Die Option ist nur verfügbar, wenn das Kontrollkästchen Skalierungsvariablen aktiviert ist. Legen Sie fest, ob die Koeffizienten für die skalierten Variablen ausgegeben werden. Die Ergebnisse werden im Blatt PLSResultsn ausgegeben.
Prozent der Varianz erläutert für jedes X
Prozent der Varianz erläutert für jedes Y
 Legen Sie fest, ob die Prozent der Varianz für jede X- oder Y-Variable ausgegeben werden soll. Die Ergebnisse werden im Blatt PLSResultsn ausgegeben.
VIP Legen Sie fest, ob die Statistik des Variableneinflusses auf Projektionen ausgegeben werden soll. Die Ergebnisse werden im Blatt PLSResultsn ausgegeben.
Ladungen Legen Sie fest, ob die Ladungen für jede X- oder Y-Variable ausgegeben werden sollen. Die Ergebnisse werden im Blatt PLSLoadingsn ausgegeben.
Scores Legen Sie fest, ob die Scores für jede X- oder Y-Variable ausgegeben werden sollen. Die Ergebnisse werden im Blatt PLSScoresn ausgegeben.
Prognostizierte Antworten für Schulungsdaten Legen Sie fest, ob die angepassten Werte ausgegeben werden. Die Ergebnisse werden im Blatt PLSResultsn ausgegeben.
Residuen Legen Sie fest, ob die Residuen ausgegeben werden sollen. Die Ergebnisse werden im Blatt PLSScoresn ausgegeben.
Distance Legen Sie fest, ob die Distanz zum X/Y-Modell für jede Beobachtung ausgegeben werden soll. Die Ergebnisse werden im Blatt PLSResultsn ausgegeben.
T-Quadrat für X-Scores Legen Sie fest, ob die Statistik des T-Quadrats für jede Beobachtung ausgegeben werden soll. Die Ergebnisse werden im Blatt PLSResultsn ausgegeben.
X-Gewichtungen Legen Sie fest, ob die Gewichtungen für jede X-Variable ausgegeben werden sollen. Die Ergebnisse werden im Blatt PLSLoadingsn ausgegeben.

Diagramme

Legen Sie fest, ob in den Partiellen kleinsten Quadraten (PLS) Zeichnungen gezeigt werden. Die Zeichnungen werden zu dem Blatt PLSn hinzugefügt.

Koeffizientenzeichnungen Legen Sie fest, ob der Screeplot für Eigenwerte gezeigt werden soll.
Wichtigkeitsdiagramm für Variablen Legen Sie fest, ob das Wichtigkeitsdiagramm für Variablen gezeigt werden soll oder nicht.
Komponentendiagramm Legen Sie fest, ob Zeichnungen für Faktoren gezeigt werden sollen.
  • Zu zeichnende Hauptkomponenten auswählen
    Legen Sie die Faktoren für X-, Y- und Z-Achse in Zeichnungen fest.
    • Faktor für X-Achse
      Legen Sie den Faktor für die X-Achse in Komponentenzeichnungen fest.
    • Faktor für Y-Achse
      Legen Sie den Faktor für die Y-Achse in Komponentenzeichnungen fest.
    • Faktor für Z-Achse
      Legen Sie den Faktor für die Z-Achse in Komponentenzeichnungen fest.
  • X-Ladungsdiagramm
    Legen Sie fest, ob das Ladungsdiagramm für Eigenvektoren von X gezeigt werden soll.
  • Y-Ladungsdiagramm
    Legen Sie fest, ob das Ladungsdiagramm für Eigenvektoren von Y gezeigt werden soll.
  • X-Scorediagramm
    Legen Sie fest, ob das Scorediagramm für Scores von X gezeigt werden soll.
  • Y-Scorediagramm
    Legen Sie fest, ob das Scorediagramm für Scores von Y gezeigt werden soll.
Diagnostikzeichnungen Legen Sie fest, ob die Diagnostikzeichnungen gezeigt werden sollen, einschließlich
  • Diagramm der prognostizierten Werte vs. abhängige Werte
  • Diagramm der Residuen vs. Diagramm der prognostizierten Werte
  • Residuum vs. Unabhängige Diagramme
  • P-P-Diagramm der Residuen
Distanzzeichnungen Legen Sie fest, ob die Distanzzeichnung gezeigt werden soll.
T-Quadrat-Diagramm Legen Sie fest, ob das T-Quadrat-Diagramm gezeigt werden soll.

Ausgabe

Legen Sie das Ziel der Ausgabeergebnisse fest.

PLS-Bericht Legen Sie das Blatt für den Bericht der partiellen kleinsten Quadraten fest. Der Standardwert ist ein neues Blatt in der Arbeitsmappe der Eingabedaten mit dem Namen PLSn.
Berichtsdaten (Angepasste Ergebnisse) Legen Sie das Blatt für die Ergebnisse der ausgewählten Option im Zweig Eigenschaften fest. Der Standardwert ist ein neues Blatt in der Arbeitsmappe der Eingabedaten mit dem Namen PLSResultsn.
Datenbericht (Ladungen und X-Gewichte) Legen Sie das Blatt für die Ladungswerte und die X-Gewichte fest. Der Standardwert ist ein neues Blatt in der Arbeitsmappe der Eingabedaten mit dem Namen PLSLoadingsn.
Datenbericht (Scores und Residuen) Legen Sie das Blatt für die Scorewerte und die Residuen fest. Der Standardwert ist ein neues Blatt in der Arbeitsmappe der Eingabedaten mit dem Namen PLSScoresn.
