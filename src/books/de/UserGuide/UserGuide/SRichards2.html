Brief Description

Sigmoidal logistic function, type 2.

Reference: Seber, G. A. F., Wild, C. J. 1989. Nonlinear Regression. John Wiley & Sons, Inc. pp. 332 - 337

Sample Curve

Parameter

Number: 4

Names: a, xc, d, k

Meanings: a = top asymptote, xc = center, d = unknown, k = coefficient

Lower Bounds: a > 0.0, k > 0.0

Upper Bounds: none

Script Access
