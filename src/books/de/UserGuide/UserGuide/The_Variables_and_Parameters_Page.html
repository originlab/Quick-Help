Die Seite Variablen und Parameter

FitFuncBuilder-Var-Para

Diese Seite bietet Optionen, die es Ihnen erlauben, Variablen und Parameter für die Anpassungsfunktion festzulegen. Nachdem Sie alle Einstellungen auf dieser Seite gemacht haben, könnten Sie auf die nächsten Seiten gehen, um den Anpassungsfunktionskörper zu schreiben, Parameter festzulegen, Initialisierungscodes zu schreiben usw.

Bedienelemente des Dialogs

Wenn das Funktionsmodell auf Explizit gesetzt ist:

Unabhängige Variablen Geben Sie hier die unabhängigen Variablen ein. Wenn Sie mehrere Variablen eingeben müssen, dann trennen Sie diese mit einem Komma.
Abhängige Variablen Geben Sie hier die abhängigen Variablen ein. Wenn Sie mehrere Variablen eingeben müssen, dann trennen Sie diese mit einem Komma.
Parameter Geben Sie hier die Parameter ein. Wenn Sie mehrere Parameter eingeben müssen, dann trennen Sie diese mit einem Komma.
Derived Parameters Legen Sie hier die Abgeleiteten Parameter fest. Abgeleitete Parameter sind zusätzliche Parameter, die aus den Parameterwerten der Funktion nach dem Anpassungsprozess berechnet werden.
Constants Legen Sie hier die Konstanten fest. Konstanten sind feste Werte, die entweder im Funktionsausdruck oder im Parameterinitialisierungscode verwendet werden können.
Peakfunktion Aktivieren Sie dieses Kontrollkästchen, wenn Sie Ihre Funktion zum Anpassen von Peaks im Assistenten Peaks analysieren oder zum Anpassen von Wiederholungen im Hilfsmittel NLFit verwenden möchten.

Beachten Sie, dass Parameter für Peakfunktionen ein spezielles Peakattribut haben, die auf der nächsten Seite festgelegt werden müssen. Der erste Parameter in der Parameterliste muss als "y0" bezeichnet werden und sollte das Peakattribut "Versatz/Basislinie" zugeordnet bekommen. Abgesehen von "y0" müssen Sie mindestens drei weitere Parameter definieren und das Peakattribut auf "Zentrum X", "Breite X" und "Amplitude/Fläche" setzen. Im Folgenden sehen Sie ein Beispiel für eine Peakanpassungsfunktion:

y = y0 + (2*A/PI)(w/(4*(x-xc)^2 + w^2))


Wenn das Funktionsmodell auf Implizit gesetzt ist:

Variablen Geben Sie hier die unabhängigen Variablen ein. Wenn Sie mehrere Variablen eingeben müssen, dann trennen Sie diese mit einem Komma.
Formelzahl Geben Sie hier die Zahl der Formal f ein. Für eine Formelzahl zwischen 1 und 9 können Sie die Auswahlliste nutzen.
Schätzungen Geben Sie hier den/die Name(n) der Formel ein. Die Namen werden automatisch entsprechend der Formelzahl erzeugt.
