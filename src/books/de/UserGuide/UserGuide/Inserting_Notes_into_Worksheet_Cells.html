Einfügen von Notizen in Arbeitsblattzellen

InsNote-to-WksCell

Jede Arbeitsblattzelle -- Datenzeile oder Spaltenbeschriftungszeile -- kann eine Zellennotiz enthalten, sogar diejenigen, die Daten oder andere Objekte enthalten wie Bilder oder eingebettete Diagramme.

  • Der Inhalt von Notizen wird als Popup angezeigt, wenn Sie über die Registerkarte in der oberen rechten Ecke der Zelle fahren.
  • Die Notizen können in einem Notizfenster bearbeitet werden, indem die Zelle markiert und dann auf die Schaltfläche Im Notizfenster öffnen auf der Minisymbolleiste geklickt wird.

Auch wenn die Anzeige von großen Mengen gemischten Inhalts (d. h. Text, verknüpfte Diagramme, Bilder etc.) in Popup-Notizen nicht praktisch ist (Text ist für Popups am besten geeignet), können Sie eine beliebige Anzahl von Elementen zu einem Notizfenster hinzufügen, um diese anzuzeigen oder zu exportieren).

Um Popups von Zellnotizen zu verhindern, setzen Sie @NCP = 0. Eine Anleitung, wie Sie den Wert einer LabTalk-Systemvariablen ändern, finden Sie in dieser FAQ.

Notizen hinzufügen

  1. Um eine Notiz hinzuzufügen, wählen Sie die Arbeitsblattzelle und klicken Sie in der Minisymbolleiste auf Notiz hinzufügen.
  2. Geben Sie Ihren Text ein. Falls Sie mehr als einen kurzen, einfachen Text hinzufügen, klicken Sie in der Minisymbolleiste auf die Schaltfläche Im Notizfenster öffnen und öffnen Ihre Notiz in einem Notizfenster. Das Notizfenster unterstützt Origin Rich Text, wodurch sie komplexere Formatierungen mit Bildern, LaTeX-Gleichungen etc. kombinieren können.
  3. Während der Eingabe von Text in die Zellnotiz können Sie Text mit Hilfe von Schaltflächen auf der Minisymbolleiste Format formatieren.

Es gibt in einem Origin-Projekt mehrere Stellen, um Ihre Notizen hinzuzufügen. Drei von ihnen -- Ordnernotizen, Zellnotizen (hier beschrieben) und die alleinstehenden Notizfenster -- unterstützen gemischte Objekte (z. B. Text und Bilder) und komplexe Formatierung wie Origin Rich Text. Zusätzlich können Sie Notizen im Klartext zum Arbeitsblatt hinzufügen.

Syntax des Notizfensters

Origins Notizfenster unterstützt vier Syntaxen: Text, Origin Rich Text (Standard), HTML und Markdown. Zellnotizen unterstützen Text und Origin Rich Text.

  1. Um die Syntax zu wechseln, wählen Sie Notizen: Syntax: Text.
  2. Wenn Sie Text ausgewählt haben, können Sie eine der anderen Syntaxen frei im Menü Notizen auswählen.


Per Standard ist für alle neuen Zellnotizen und alle neuen Notizfenster (diejenigen, die per Klick auf Neue Notizen auf der Symbolleiste Standard hinzugefügt wurden) die Verwendung von Origin Rich Text festgelegt. Sie können den Standard für die neuen Zellnotizen ändern, indem Sie eine Skriptzeile zu \User Files\Origin.ini hinzufügen. Sie können dies durch eine indirekte Bearbeitung tun, indem Sie folgende Werte verwenden: 0 = Text, 1 = HTML, 2 = Markdown, 3 = Origin Rich Text.

  1. Um beispielweise Text als Standard für alle neuen Zellnotizen festzulegen, klicken Sie auf Fenster: Skriptfenster, geben Sie Folgendes ein und drücken Sie Enter ...
ed -syn 0

Es wird eine Textzeile zum Abschnitt [Options] von \User Files\Origin.ini hinzugefügt:

CellNoteSyntax=0

Sie können prüfen, ob alle Zellnotizen jetzt standardmäßig Text verwenden, indem (1) Sie eine neue Zellnotiz einfügen, (2) die Zelle auswählen und auf die Minitoolschaltfläche Im Notizfenster öffnen klicken und (3) bei aktivem Notizfenster Notizen: Syntax wählen. Die Option Text sollte ausgewählt sein.
