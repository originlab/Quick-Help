Anwenderfilter (nur OriginPro)


Die Funktion imgUserfilter filtert ein Bild basierend auf einer benutzerdefinierten Filter (Maske). Mit dieser Funktion können Sie einen benutzerdefinierten Filter erstellen und auf ein Bild anwenden. Um diese Funktion zu verwenden:

Wählen Sie Bild: Räumliche Filter: Benutzerfilter im Origin-Menü.

Weitere Informationen über diese Funktion finden Sie in der Dokumentation der X-Funktion imgUserfilter in der Origin-Hilfe. Diese kann über das Befehlsfenster mit folgendem Befehl aufgerufen werden:

help imgUserfilter
