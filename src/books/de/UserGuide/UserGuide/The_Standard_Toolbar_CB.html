Die Symbolleiste Standard

CB-Standard-Toolbar

Die Schaltflächen der Symbolleiste Standard bieten Zugriff auf das Öffnen, Speichern und Drucken des Arbeitsbereichs sowie auf das Durchführen von Operationen im Code Builder.

Öffnet ein Dialogfeld für eine neue Datei, um eine neue Quelle, einen Header, LabTalk-Skript oder eine Textdatei zu erstellen.

Öffnet eine Quelle, einen Header, LabTalk-Skript oder eine Textdatei und fügt dem Arbeitsbereich die Datei hinzu.

Speichert die Code Builder-Projektdatei.

Speichert alle im Arbeitsbereich geöffneten Dateien.

Sendet das aktive Fenster ohne Nachfrage an den Drucker; die aktuellen Einstellungen von Seite einrichten und vom Standarddrucker werden verwendet.

Schneidet die Auswahl aus und legt Sie in der Zwischenablage ab.

Kopiert die Auswahl und legt Sie in der Zwischenablage ab.

Fügt den Inhalt der Zwischenablage ein.

Kompiliert eine aktuelle Quelldatei aus dem Code Builder heraus.

Kompiliert alle Quelldateien, die zu dem Arbeitsbereich hinzugefügt wurden, und führt die Verknüpfung durch.

Beendet den Compiler.

Kompiliert alle Dateien neu und führt die Verknüpfung durch, wenn es seit dem letzten Erstellen keine Dateiveränderung gegeben hat.
