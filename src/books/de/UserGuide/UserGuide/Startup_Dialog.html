Übersicht

Der Dialog Lernzentrum hilft Anwendern beim Einstieg in Origin. Er wird mit dem Start von Origin 2017 geöffnet.

Um das Lernzentrum zu öffnen:
Wählen Sie Hilfe: Lernzentrum im Origin-Menü.
oder
Drücken Sie die Taste F11.

Verwendung

Diagrammbeispiele und Beispielprojekte öffnen

Wählen Sie Diagrammbeispiele oder Beispielprojekte im Abschnitt Öffnen, um von Origin bereitgestellte Beispiele anzuzeigen.

Es stehen eine Vielzahl von Diagramm- und Analysebeispielen zur Verfügung. Sie können die Miniaturansicht des Diagramms (im Fall von Diagrammbeispielen) oder ein Beispiel aus der Liste (im Fall von Analysebeispielen) auswählen und dann doppelt darauf klicken. Daraufhin wird nur der entsprechende Ordner mitsamt der Anweisungen geöffnet. Sie können Ihre Daten hier einfügen, um zu sehen, wie sich das Diagramm und die Analyse aktualisieren.

Sie können nach Diagrammbeispielen suchen, indem Sie Schlüsselwörter und Phrasen im Feld Suche eingeben.

Erste Schritte mit Origin

Der Abschnitt Lernressourcen enthält eine Liste von Tutorials und Videos, die Anwender beim Einstieg in Origin unterstützen.

Anwenderdateien übertragen

Die Schaltfläche Anwenderdateien übertragen unten im Dialog ermöglicht Anwendern das Verschieben ihrer Dateien aus ihrem vorherigen Anwenderdateiordner in den neuen Ordner.
