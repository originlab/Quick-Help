Übersicht

Mit dem Minitool 2D-Integration (verwendet die X-Funktion addtool_integ2) können Sie die 2D-Integration für ein Kontur- oder Bilddiagramm berechnen, das aus gleichverteilten Daten in einer Matrix oder nicht gleichverteilten Daten einer virtuellen Matrix oder XYZ-Arbeitsblattdaten erstellt wird.

Um dieses Minitool zu verwenden, erstellen Sie ein Konturdiagramm aus Ihren Daten, aktivieren Sie das Diagramm und wählen Sie Minitools: 2D-Integration im Hauptmenü.

Hinweise: Die Optionen für die X-Funktion des Minitools sind in einer Baumstruktur als Variablen strukturiert. Die Namen der Baumknoten, die für jede Option/jeden Zweig aufgerufen werden, sind in den Klammern neben dem Optionsnamen (Baumknotenbeschriftungen) aufgelistet.

Grafische Datenauswahl (ROI)

Das Feld der grafischen Datenauswahl (ROI - Region of Interest) definiert die Fläche für die Berechnung der Oberflächenintegration.

Form

Definieren Sie die Form des ROI-Felds.

Rechteck Legen Sie die Form und Position des rechteckigen ROI-Felds unten in Formposition fest.
Kreis Legen Sie die Form und Position des kreisförmigen ROI-Felds unten in Formposition fest.
Beliebige Form Nach Klicken auf OK wird der Minidialog ROI erstellen geöffnet, mit dem Sie das Zeichenhilfsmittel wählen, um die beliebige Form auf dem Diagramm zu zeichnen und so das ROI-Feld zu erstellen.

  • Schaltflächen Rechteck einfügen und Kreis einfügen: Zeichnen Sie ein quadratisches/rechteckiges/kreisförmiges/ovales Objekt, um ein ROI-Feld hinzuzufügen. Details zur Verwendung dieser Hilfsmittel finden Sie auf dieser Seite.
  • Schaltflächen Polygon einfügen und Bereich einfügen: Zeichnen Sie eine Freihandobjekt, um das ROI-Feld hinzuzufügen. Details zur Verwendung dieser Hilfsmittel finden Sie auf dieser Seite.
  • Schaltfläche Kurvenhilfsmittel : Zeichnen Sie eine geschlossene Kurve, um ein ROI-Feld zu zeichnen. Details zur Verwendung dieses Hilfsmittels finden Sie auf dieser Seite.


Hinweis: Nach Hinzufügen eines ROI-Felds zum Diagramm können Sie immer auf die dreieckige Schaltfläche oben rechts vom ROI-Feld klicken und ROI modifizieren wählen, um den Dialog ROI erstellen zu öffnen und die ROI-Form erneut zu ändern.

Position der Form (shapeProp)

Sie können die Position der grafischen Datenauswahl definieren, indem Sie Werte für XY-Koordinaten angeben.

X1 (xFrom) Geben Sie einen Wert für die X-Koordinate für die am weitesten links im ROI-Feld liegende Linie ein.
X2 (xTo) Geben Sie einen Wert für die X-Koordinate für die am weitesten rechts im ROI-Feld liegende Linie ein.
Y1 (yFrom) Geben Sie einen Wert für die Y-Koordinate für die am weitesten unten im ROI-Feld liegende Linie ein.
Y2 (yTo) Geben Sie einen Wert für die Y-Koordinate für die am weitesten oben im ROI-Feld liegende Linie ein.
Festgelegt (Verschieben durch graf. Datenauswahl nicht möglich) (fix_roi_loc) Legen Sie fest, ob das manuelle Verschieben des ROI-Feldes möglich sein soll.

 

Toolname zeigen (toolname)

Legen Sie fest, ob ein Toolname oben links im ROI-Feld gezeigt werden soll. Der Toolname kann indem Textfeld weitergehend festgelegt werden, wenn dieses Kontrollkästchen aktiviert ist.

 

In grafischer Datenauswahl in der Mitte oben zeigen (showtop)

Legen Sie fest, ob die Ergebnisse der entsprechenden Eigenschaft oben in der Mitte des ROI-Felds angezeigt werden sollen.

Volumen (topVolume) Volumen, das von der Basis zur Quelloberfläche integriert wurde
Halbwertsbreite (topFwhm) Die volle Breite bei halber Höhe der Quelloberfläche
N (topN) Gesamtanzahl der Datenpunkte
Minimum (topMinimum) Der minimale Z-Wert
Maximum (topMaximum) Der maximale Z-Wert
Mittelwert (topMean) Der Mittelwert der Z-Werte
Median (topMedian) Der Mittelwert der Z-Werte
StdAbw. (topSD) Die Standardabweichung aller Z-Werte
Summe (topSum) Die Summe der Z-Werte

Füllfarbe (rectcolor)

Legen Sie die Füllfarbe der grafischen Datenauswahl fest.

Basislinie (baseline)

Modalwert (method) Wählen Sie einen Modalwert für die Basis.
  • Kein
    Es gibt keine Basislinie. Die Integration wird von der Nullebene aus durchgeführt.
  • Konstante
    Verwenden Sie eine flache Ebene parallel zu der XY-Ebene bei dem vordefinierten Z-Wert.
  • Ebene
    Erstellen Sie eine angepasste Ebene als Basislinie mit Hilfe der Grenzpunkte.
Konstant (constantMode) Diese Option ist nur dann verfügbar, wenn der Modalwert auf Konstante gesetzt wurde. Sie wird verwendet, um den Z-Wert für die Ebene der Basislinie festzulegen.
  • Benutzerdefiniert
    Verwenden Sie einen benutzerdefinierten Z-Wert. Dieser wird in dem Textfeld Z= definiert.
  • Minimum
    Verwenden Sie den minimalen Z-Wert.
  • Maximum
    Verwenden Sie den maximalen Z-Wert.
Z= (zvalue) Diese Option ist nur verfügbar, wenn für Konstant die Option Benutzerdefiniert ausgewählt wurde. Sie wird verwendet, um einen benutzerdefinierten Z-Wert für die Ebene der Basislinie festzulegen.
Grenztoleranz (%) (boundtor) Diese Option ist nur dann verfügbar, wenn der Modalwert auf Ebene gesetzt wurde. Die Grenze wird von der Option bestimmt, die für Bereich ausgewählt wurde. Die Toleranz in Prozent bestimmt eine Toleranzbreite bzw. -höhe. Die Punkte, deren Abstand von der Grenze kleiner ist als die Toleranzbreite oder -höhe Toleranzbreite/-höhe = Toleranz (%) * Breite/Höhe.
Bereich (range) Diese Option ist nicht verfügbar, wenn der Modalwert auf Kein oder Benutzerdefiniert in der Auswahlliste Konstant ausgewählt ist. Sie wird verwendet, um den Datenbereich festzulegen, der beim Bestimmen der Basisebene berücksichtigt wird.
  • Bereich des gesamten Diagramms
    Die Grenze des gesamten Diagrammbereichs wird zum Anpassen der Basisebene verwendet oder der Minimal-/Maximalwert wird innerhalb des gesamten Diagrammbereichs gefunden.
  • Zeichnung innerhalb ROI
    Nur die Grenze der Zeichnung innerhalb des ROI-Felds wird zum Anpassen der Basisebene verwendet oder der Minimal-/Maximalwert wird innerhalb des durch die grafische Datenauswahl definierten Bereichs gefunden.

Integration (integration)

Sie können die Optionen für die Integration auf dieser Registerkarte festlegen.

Wenn Interpolieren nach dem fehlenden Wert (interpmissing) aktiviert ist, berücksichtigt die Interpolation auch fehlende Werte.

Wenn Basislinie zeigen - Subtrahierte Oberfläche (showsurface) aktiviert ist, wird die subtrahierte Oberfläche (subtrahiert aus Quelldaten und Basislinienebene) gezeigt.

Ausgabe (Output)

Eigenschaften ausgeben in (outputQuantitiesTo)

Hier kann die Ausgabe der Ergebnisse benutzerdefiniert angepasst werden.

Skriptfenster (script) Legen Sie fest, ob die Ergebnisse im Skriptfenster ausgegeben werden.
Ergebnisfenster (reslog) Legen Sie fest, ob das Ergebnis im Ergebnisfenster ausgegeben wird.
An Arbeitsblatt anhängen (appendwks) Legen Sie fest, ob die Ergebnisse an das Arbeitsblatt angehängt werden.
Ergebnisblattname (wbkName) Diese Option ist nur verfügbar, wenn An Arbeitsblatt anhängen (appendwks) ausgewählt ist.

Wenn Sie sich dafür entscheiden, die Ausgabe an ein Arbeitsblatt zu senden, haben Sie folgende Optionen:

  • Wenn Sie eine neue Ausgabe erzeugen, werden die Ergebnisse standardmäßig in [%H-Integ2d]Result (hier bezeichnet %H den Kurznamen des Quelldiagramms) ausgegeben, aber es können auch andere Mappen und Blätter festgelegt werden. Falls die Mappe und das Blatt nicht existieren, werden sie bei der Ausgabe erstellt.
  • Alternativ können Sie auf die Ausklappschaltfläche rechts von Ergebnisblattname klicken und Blatt in Eingabemappe wählen. Das Bearbeitungsfeld wird mit [<input>]Result gefüllt. Wenn Sie eine neue Ausgabe erzeugen, werden die Ergebnisse in ein Blatt mit dem Namen Result in der Quellmappe ausgegeben.

Zweig Eigenschaften (quantities)

Legen Sie die auszugebenden Eigenschaften fest.

Datensatzidentifizierer Legen Sie einen Datensatzidentifizierer in der Auswahlliste fest.
Indexbereich (index_rng) Legen Sie fest, ob die Indizes für die Dimension auf das ROI-Feld beschränkt ausgegeben werden sollen. Diese Option ist nur verfügbar, wenn die Eingabedaten kein XYZ-Bereich sind.
X-Bereich (x_rng) Legen Sie fest, ob die Indizes für den X-Bereich ausgegeben werden sollen.
Y-Bereich (y_rng) Legen Sie fest, ob die Indizes für den Y-Bereich ausgegeben werden sollen.
Max. Höhe (max_height) Legen Sie fest, ob die maximale Höhe von der Quelloberfläche bis zur Basislinie ausgegeben werden soll.
Volumen (volume) Legen Sie fest, ob das Volumen, das von der Basislinie bis zur Quelloberfläche integriert wurde, ausgegeben werden soll.
Fläche der Oberfläche (area) Legen Sie fest, ob die Fläche der Quelloberfläche ausgegeben werden soll.
Zentroid (centroid) Legen Sie fest, ob die Koordinaten des Zentroids für den ausgewählten Bereich ausgegeben werden sollen.
Halbwertsbreite (fwhm) Legen Sie fest, ob die Halbwertsbreite, d.h. die volle Breite auf halber Höhe der Quelloberfläche, ausgegeben werden soll.
Halbe Breite (half_width) Legen Sie fest, ob die halbe Breite ausgegeben wird.
Z-Max. (zmax) Legen Sie fest, ob der maximale Z-Wert ausgegeben werden soll.
Index bei Z-Max. (index_zmax) Legen Sie fest, ob der Spalten-/Zeilenindex für die Daten bei dem maximalen Z-Wert ausgegeben werden soll.
(x,y) bei Z-Max. (xy_zmax) Legen Sie fest, ob die X- und Y-Werte für den Datenpunkt bei dem maximalen Z-Wert ausgegeben werden sollen.
Statistik (QStatistics) Legen Sie fest, die Statistikergebnisse basierend auf Z-Werten, ausgegeben werden sollen. Verfügbare Eigenschaften sind:
  • N (qsN): Anzahl der Datenpunkte
  • Minimum (qsMinimum): Minimum der Z-Werte
  • Maximum (qsMaximum): Maximum der Z-Werte
  • Mittelwert (qsMean): Mittelwert der Z-Werte
  • Median (qsMedian): Median der Z-Werte
  • StdAbw. (qsSD): Standardabweichung der Z-Werte
  • Summe (qsSum): Summe der Z-Werte

Basislinie (baselineOutput)

Basislinie (baseline) Legen Sie fest, ob die Basislinieninformationen ausgegeben werden sollen.
Ausgabe in Legen Sie fest, wo die Ergebnisse ausgegeben werden.

 

Hinweise: Der Name in den Klammern ist der Baumknotennamen, und der Name außerhalb ist die Baumknotenbeschriftung. Wenn die X-Funktion mit dem Skript ausgeführt wird, müssen Sie den Baumknoten Werte mit Hilfe ihrer Namen zuweisen. Sonderzeichen und Leerzeichen sind in Baumknotennamen nicht zugelassen. Sie können Sonder- und Leerzeichen in den Baumknotenbeschriftungen verwenden.

Das Ausklappmenü

Klicken Sie auf die dreieckige Schaltfläche in der oberen rechten Ecke der grafischen Datenauswahl, um das Ausklappmenü mit den folgenden Optionen zu öffnen:

Neue Ausgabe Das Ergebnis wird ausgegeben.
Zum Berichtsarbeitsblatt gehen Es wird zum Berichtsblatt gewechselt, sobald das Ergebnis im Arbeitsblatt ausgegeben wurde.
Ausgabe in Zwischenablage Wenn diese Option ausgewählt ist (Menüelement aktiviert), wird die Neue Ausgabe in der Zwischenablage abgelegt.
Daten ändern Die Eingabedaten werden geändert.
  • Per Standard ist der Modus Auto aktiviert. Wenn Auto aktiviert ist, wird die Zeichnungsauswahl gesteuert, indem auf eine Zeichnung im Diagrammfenster oder in der Objektverwaltung geklickt wird. Vor Origin 2019 unterstützte Origin nicht Auto. Um die Zielzeichnung/-daten in älteren Versionen zu ändern, müssen Sie eine Zeichnung im Ausklappmenü auswählen.
  • Positionieren Sie das Häkchen vor einer bestimmten Zeichnung, um diese Zeichnung zu aktivieren.
  • Klicken Sie auf Auswählen oder Mehr..., um den Dialog Diagramme auswählen aufzurufen und die Auswahl zu ändern.
Auf gesamten Bereich der Zeichnung(en) erweitern Die grafische Datenauswahl wird auf den gesamten Diagrammbereich ausgestreckt.
Auf gesamte Layerbreite erweitern Die grafische Datenauswahl wird auf die gesamte Layerbreite ausgestreckt.
Auf gesamte Layerhöhe erweitern Die grafische Datenauswahl wird auf die gesamte Layerhöhe ausgestreckt.
Design speichern Die Einstellungen in diesem Dialog werden als Dialogdesign gespeichert.
Speichern als <Standard> Die Einstellung in diesem Dialog wird als Design <Standard> gespeichert.
Design laden Das vorgespeicherte Dialogdesign wird geladen.
Einstellungen Der Dialog Einstellungen wird geöffnet.
