Die Registerkarte Allgemeines (Details Zeichnung)

PD-Dialog-Misc-Tab


Die Registerkarte Allgemeines im Dialog Details Zeichnung der Seite enthält Bedienelemente zum Ansichtsmodus und die Aktualisierungskriterien eines Diagrammfensters (wenn dieses nicht aktiv ist). Außerdem können hier Einstellungen zur Anwendung des Entwurfsmodus' für den Grafikexport (einschließlich der in die Zwischenablage kopierten Diagramme) vorgenommen werden. Sie finden Kriterien für die Übereinstimmung von Zeichnungseigenschaften, die Datennormierung von XYZ-Kontur- und Oberflächendiagrammen sowie Informationen der Diagrammvorlage zu den Modifizierern der Zeichnungsattribute, die in den Spalten des Arbeitsblatt gespeichert sind.

Performance

Ansichtstyp

Wählen Sie einen Ansichtsmodus aus dieser Auswahlliste aus. Die Optionen umfassen Druckansicht, Seitenansicht, Fensteransicht und Entwurfsansicht.

Weitere Informationen können Sie unter Verfügbare Ansichtsmodi der Seite nachlesen.
Im Entwurfsmodus auch Grafikexport beschleunigen

Mit diesem Kontrollkästchen können Sie den Entwurfsmodus anwenden, wenn Ihr Diagramm in eine Datei exportiert oder in die Zwischenablage kopiert wird. Der Entwurfsmodus wird auf der Registerkarte Anzeige und Performance des Dialogs Details Zeichnung auf Ebene des Layers eingestellt. Durch die Aktivierung des Entwurfsmodus' wird die Aktualisierungsgeschwindigkeit Ihres Diagrammlayers erhöht, indem die Anzahl der im Layer angezeigten Datenpunkte begrenzt wird.

Autom. Farbe der Beschriftung entspricht dem Symbol

Dieses Bedienelement ist nur für 2D-Linien- und Symboldiagramme verfügbar.

Wenn die Farbe der Datenbeschriftung auf der Registerkarte Beschriftung auf Auto gesetzt ist, verwenden Sie dieses Kontrollkästchen, um das Verhalten der automatische Farbe der Beschriftung festzulegen.

Wenn dieses Kontrollkästchen aktiviert ist, folgt die Beschriftungsfarbe der Symbolfarbe.

Wenn es deaktiviert ist, wird die Beschriftungsfarbe gewählt, so dass sie im höchsten Kontrast zur Hintergrundfarbe angezeigt wird.

Daten für XYZ-Kontur-/Oberflächendiagramm normieren

Ein Algorithmus zur Konturtriangulation, der X- und Y-Rohdaten zeichnet, wurde für Origin 2016 implementiert. Vorgängerversionen führten eine Normierung der X- und Y-Daten vor dem Zeichnen durch. Wenn daher große Skalierungsunterschiede zwischen X- und Y-Daten bestehen, kann es sein, dass Kontur- und Oberflächendiagramme, die mit Version 2016 oder höher gemacht wurden, anders aussehen als Diagramme, die mit älteren Versionen von Origin erstellt wurden.

Sie können Origin veranlassen, mit dem älteren Algorithmus zu zeichnen, indem Sie das Kontrollkästchen Daten für XYZ-Kontur-/Oberflächendiagramm normieren aktivieren.

  • Informationen zu den Algorithmen der Konturerstellung finden Sie unter diesem Thema.
  • Das LT-Skriptsteuerung dieser Einstellung finden Sie unter the page.triangulationnormalizedata property.

Modifizierer stimmen überein mit (für Vorlage und Format kopieren)

Verwenden Sie dieses Bedienelement, um Zeichnungsattribute mit den modifizierenden Spalten zu verbinden, wenn Sie Diagrammvorlagen speichern oder das Format kopieren. Einige Zeichnungsattribute können modifiziert werden, indem Sie auf eine Spalte von Werten abgebildet werden. Die abgebildete Spalte kann über ihre Spaltenindexnummer identifiziert werden (Versatz) oder über Kurzname, Langname, Einheiten, Kommentare der Spalte, (Standard-) Parameter(n) oder den Benutzerdefinierten Parameter. Weitere Einzelheiten finden Sie unter Modifizierer stimmen überein mit (für Vorlage und Format kopieren).

Versatz Die Zeichnung wird nach Spaltenversatz auf die Modifiziererdaten abgebildet.
Kurzname Die Zeichnung wird nach Kurzname auf die Modifiziererdaten abgebildet. Anders als die anderen Beschriftungsmodifizierer ist der Kurzname im Arbeitsblatt immer einzigartig.
Langname Die Zeichnung wird nach Langname auf die Modifiziererdaten abgebildet.
Einheiten Die Zeichnung wird nach Einheiten auf die Modifiziererdaten abgebildet.
Kommentare Die Zeichnung wird nach Kommentaren auf die Modifiziererdaten abgebildet.
Parameter n Die Zeichnung wird nach Parameternn auf die Modifiziererdaten abgebildet.
Benutzerdefiniertn Die Zeichnung wird nach Benutzerdefinierten Parameternn auf die Modifiziererdaten abgebildet.

Bitte beachten Sie, dass dieser Optionswert die Standardauswahl des Bedienelements Modifizierer stimmen überein mit des Stapelzeichnens beeinflussen.
