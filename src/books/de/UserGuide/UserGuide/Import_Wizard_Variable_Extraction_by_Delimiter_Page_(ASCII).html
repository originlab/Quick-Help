Importassistent, Variablenextraktion durch Trennzeichen (ASCII)

 

Diese Seite ist nur verfügbar, wenn Bestimmen Sie den Speicherort von Variablennamen und Werten unter Verwendung von Trennzeichen auf der Seite Variablenextraktion aktiviert ist (und es mindestens eine Headerzeile in Ihrer Datendatei gibt). Verwenden Sie die Bedienelemente auf dieser Seite, um Variablen aus dem Dateipfad/Dateinamen und aus den Dateiheaderinformationen durch Analyse der Informationen mit Hilfe eines oder mehrerer benutzerdefinierter Trennzeichen zu extrahieren.

 

 

Vorschaufenster Variablen

Verwenden Sie dieses Vorschaufenster, um Variablen für die Extraktion auszuwählen. Die Inhalte dieses Vorschaufensters umfassen vier Standarddateinamen für Variablen (Pfad + Dateiname.Erweiterung, Pfad, Dateiname.Erweiterung, Dateiname). Ebenso sind jegliche Dateiheaderzeilen enthalten, die auf der Seite Headerzeilen bestimmt wurden. Jedem Variablennamen/-wert geht eine Zeilennummer voran -- der Zeilenwert in der Gruppe Variablen (siehe unten).

 

Bedienelemente von Variablengruppen

Bei jeder Informationszeile im Vorschaufenster Variablen wird vorausgesetzt, dass sie Variablen und Variablenwerte (Sie entscheiden, welche welche sind) enthält. Beachten Sie, dass jede der folgenden Möglichkeiten unabhängig von jeder Zeile im Vorschaufenster ausgewählt werden kann.

 

Linie Wählen Sie die Zeilennummer der gewünschten Variablen.
Trennzeichen Bestimmen Sie ein Trennzeichen, das verwendet wird, um die Variablennamen und -werte zu analysieren. Die Optionen in der Auswahlliste umfassen Komma, Register, Leerzeichen, Gleich, Doppelpunkt, Andere und Mehrere. Die Optionen Andere und Mehrere werden hier verwendet, um ein oder mehr Zeichen in dem angrenzenden Textfeld festzulegen. Wenn Sie mehrere Zeichen eingeben, benötigen Sie keine Trennzeichen; jedes eingegebene Zeichen wird einzeln betrachtet.
Wert Jede Zeile wird unterbrochen durch der Verwendung von bestimmten Trennzeichen. Mögliche Werte können aus der Auswahlliste Wert ausgewählt werden.
Geben Sie den Variablennamen ein. Es gibt zwei Möglichkeiten, Variablennamen festzulegen. Wählen Sie diese Schaltfläche, um eine Zeichenkettenkonstante (Leerzeichen und Sonderzeichen sind nicht zugelassen) als Variablennamen zu verwenden. Um den Variablennamen aus den Informationen, die im Vorschaufenster Variablen enthalten sind, zu extrahieren, wählen Sie die Option Variablennamen aus der Datei extrahieren (nächste Option).
Variablennamen aus der Datei extrahieren Wenn Sie diese Schaltfläche wählen, werden zusätzliche Felder zur Seite Extraktion durch Trennzeichen hinzugefügt. Diese Felder ermöglichen es Ihnen, eine Zeile, ein Trennzeichen und einen Namen für eine gegebene Variable zu bestimmen. Diese Schaltfläche wird dazu verwendet, Namen für die Auswahl aus der Auswahlliste Wert zuzuweisen.

Variablen zum Extrahieren des Fensters

Methode Sie können Variablen durch mehr als eine Methode extrahieren. Sie werden über beide Seiten Trennzeichen und Position gefunden, unabhängig von der Methode/Seite, durch die sie bestimmt wurden. Die Variablen, die auf dieser Seite bestimmt werden, sind vom Typ Methode = Trennzeichen.
Name Der Name, wie angegeben, verwendet die Bedienelemente der Gruppe Variable.
Wert Die Variable Wert, wie angegeben, verwendet die Bedienelemente der Gruppe Variable.
Hinzufügen Fügen Sie den Variablen zum Extrahieren des Fensters eine benutzerdefinierte Variable hinzu, nachdem Sie einen Variablennamen im Textfeld Variablenname eingegeben haben.
Ersetzen Klicken Sie auf diese Schaltfläche, um die gewählte Variable mit einer neuen Variable zu ersetzen.
Löschen Löschen Sie gewählte Variable(n) aus Variablen zum Extrahieren des Fensters.
