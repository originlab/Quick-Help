Partieller Import

Partieller Import

Die Auswahlliste verfügt über drei Optionen: Kein, Alle Dateien und Ab der 2. Datei. Die Optionen Alle Dateien und Ab der 2. Datei aktivieren die Kontrollkästchen der Bereiche Spalte Von/Bis und Zeile Von/Bis für den partiellen Import.

  • Kein: Die Einstellungen des partiellen Imports werden deaktiviert.
  • Alle Dateien: Die Einstellungen des partiellen Imports werden auf alle ausgewählten Dateien angewendet.
  • Ab der 2. Datei: Die Einstellungen des partiellen Imports werden auf alle ausgewählten Dateien außer die 1. Datei angewendet.
Spalte Von/Bis

Geben Sie zwei Spaltennummern ein, um den Spaltenbereich für den teilweisen Import zu bestimmen. Die Standardeinstellungen in Von und Bis sind 1 bzw. 0 (hat einen Import aller Spalten und Zeilen zur Folge).
Zeile Von/Bis

Geben Sie zwei Zeilennummern ein, um den Zeilenbereich für den teilweisen Import zu bestimmen. Die Standardeinstellungen in Von und Bis sind 1 bzw. 0 (hat einen Import aller Spalten und Zeilen zur Folge).
Spalten überspringen

Spalten überspringen benötigt einen ganzzahligen Wert größer oder gleich 0. Wenn Spalten überspringen = 0, werden keine Zeilen beim Import ausgelassen.
Spalten lesen

Dieses Textfeld kann nur bearbeitet werden, wenn Spalten überspringen nicht null ist. Es muss ein ganzzahliger Wert größer oder gleich 1 sein.

Wenn Spalten überspringen > 0 ist, dann wiederholt der Importassistent den Import von Spalten mit Spalten lesen und das Überspringen von Spalten mit Spalten überspringen solange, bis entweder das Dateiende oder die Bereichseinstellungen des teilweisen Imports erreicht sind.
Zeilen überspringen

Datenzeilen übergehen benötigt einen ganzzahligen Wert größer oder gleich 0. Wenn Datenzeilen übergehen = 0, werden keine Zeilen beim Import ausgelassen.
Zeilen lesen

Dieses Textfeld kann nur bearbeitet werden, wenn Datenzeilen übergehen nicht null ist. Es muss ein ganzzahliger Wert größer oder gleich 1 sein.

Wenn Datenzeilen übergehen > 0 ist, dann wiederholt der Importassistent den Import von Zeilen mit Zeilen lesen und das Überspringen von Zeilen mit Datenzeilen übergehen solange, bis entweder das Dateiende oder die Bereichseinstellungen des teilweisen Imports erreicht sind.
Anwenden

Klicken Sie auf diese Schaltfläche, um die Bereichseinstellungen anzuwenden.

Nicht-numerische Daten in einem numerischen Feld (ASCII)

Dies ist nur für ASCII-Dateien verfügbar. Wenn zwischen numerischen Daten nicht-numerische Daten eingefügt sind, können Sie dieses Bedienelement verwenden, um zu entscheiden, wie mit nicht-numerischen Daten verfahren werden soll (typischerweise werden solche nicht-numerischen Daten dazu verwendet, verschiedene Datensätze in einer Datei zu identifizieren und voneinander zu trennen).

Diese Auswahlliste hat vier Optionen:

Import abbrechen

Der Import wird beendet, sobald das erste nicht-numerische Datenmerkmal in den Daten gefunden wird.
Überspringen

Nicht-numerische Daten werden übersprungen (siehe Als Text lesen).
Als Text lesen

Nicht-numerische Daten werden als Text gelesen. Wenn das Spaltenformat auf Text & Numerisch eingestellt wurde, sieht man den Text auf der Seite des sich ergebenden Arbeitsblatts. Wenn das Spaltenformat auf Numerisch eingestellt wurde, wird der Text als fehlende Werte angezeigt.
Neue (verborgene) Arbeitsmappe öffnen

Eine neue verborgene Arbeitsmappe wird begonnen, sobald nicht-numerische Daten gefunden werden.
Neues Arbeitsblatt öffnen

Ein neues Arbeitsblatt wird begonnen, sobald nicht-numerische Daten gefunden werden.
Neue Spalte öffnen

Eine neue Spalte wird im selben Arbeitsblatt geöffnet, sobald nicht-numerische Daten gefunden werden.

Vorschaufenster Spalten

Das Fenster zeigt die importierten Daten so, wie sie im sich ergebenden Arbeitsblatt aussehen werden. Beachten Sie, dass Sie mit der rechten Maustaste auf den Titel jeder Spalte klicken und das Format und Ziel mit dem Kontextmenü festlegen können.
