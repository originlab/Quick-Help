Die Registerkarte Segmente (Details Zeichnung)

PD-Dialog-Wedges-Tab

Segmente kombinieren nach

Legen Sie fest, wie die Segmente kombiniert werden sollen nach

  • Prozent: Es werden die Segmente kombiniert, deren Prozentangabe kleiner oder gleich einem festgelegten Prozentsatz sind.
  • Wert: Es werden die Segmente kombiniert, deren Wert kleiner oder gleich einem festgelegten Wert sind.
  • Benutzerdefiniert: Es werden die Segmente kombiniert, die in der Spalte Kombinieren der Tabelle unten ausgewählt wurden.

Bitte beachten Sie, dass die kombinierten Segmente als das Segment Sonstige behandelt werden, sowohl in der Zeichnung als auch in der Legende.

Prozent/Wert <=

Legen Sie einen Schwellenwert in Prozent oder als Wert fest, um die Segmente zu kombinieren. Wenn Sie eine Angabe als Prozent/Wert in diesem Feld eingegeben haben, werden alle Segmente, deren Prozent/Wert kleiner als oder gleich diese Angaben sind, in der Spalte Kombinieren der Tabelle unten ausgewählt.

Kombinierte zeigen als

Wenn Sie im 2D-Kreisdiagramm, dessen Ansichtswinkel auf 90 gesetzt ist, Segmente kombiniert haben, können Sie ein weiteres Diagramm hinzufügen (Kreis, Säulen oder Ring), um die Einzelheiten der kombinierten Segmente zu zeigen.

Die zweite Zeichnung, mit der die Einzelheiten der kombinierten Segmente gezeigt werden, kann sein:

  • Kreisdiagramm: Um ein Kreisdiagramm für das Segment Sonstige (kombinierte Segmente) zur Grafik hinzuzufügen, so dass ein Kreis-von-Kreisdiagramm erstellt wird.
  • Säulendiagramm: Um ein Säulendiagramm für das Segment Sonstige (kombinierte Segmente) zur Grafik hinzuzufügen, so dass ein Balken-von-Kreisdiagramm erstellt wird.
  • Ringdiagramm: Um ein Ringdiagramm für das Segment Sonstige (kombinierte Segmente) zur Grafik hinzuzufügen, so dass ein Ring-von-Kreisdiagramm erstellt wird.

Größe der 2. Zeichnung (% von Radius)

Legen Sie die Größe der zweiten Zeichnung in Prozent des Radius fest.

  • Für das Kreis-von-Kreisdiagramm ist der Radius des zweiten Kreises die Größe.
  • Für das Balken-von-Kreisdiagramm ist die Höhe der gestapelten Säulen die Größe.
  • Für das Ring-von-Kreisdiagramm ist die Segmentdicke des Rings die Größe.

Abstand zwischen Zeichnungen (% des Radius)

Legen Sie den Abstand zwischen den Zeichnungen in Prozent des Radius fest.

Farbe der kombinierten Segmente

Legen Sie eine Farbe für die anderen, im ersten Diagramm kombinierten Segmente fest.

Tabelle zum Explodieren oder Kombinieren von Segmenten

In dieser Tabelle können Sie das Kontrollkästchen in den Spalten Explodieren, Mit Nächstem explodieren und Kombinieren verwenden, um zu entscheiden, wie die Segmente explodiert und/oder kombiniert werden.

Segmente explodieren

Um einige Segmente mehr hervorzuheben, können Sie diese Segmente explodieren, indem Sie die Segmente aus der Mitte rücken.

Aktivieren Sie die Kontrollkästchen in der Spalte Explodieren, um die Segmente festzulegen, die Sie explodieren möchten. Bitte beachten Sie, dass in diesem Fall alle ausgewählten Segmente separat explodieren.

Wenn Sie die nebeneinander liegenden N Segmente zusammen explodieren möchten, können Sie das Kontrollkästchen Mit Nächstem explodieren für den ersten N-1 Segmente aktivieren.

Falls Sie Segmente kombiniert haben, werden diese Segmente explodiert, wenn Sie das Kontrollkästchen "Explodieren" für diese Segmente aktiviert haben.

Segmente kombinieren

Wenn Sie sich entschieden haben, einen Schwellenwert als Prozent oder als Wert zu verwenden, um die Segmente mit Hilfe des Bedienelements Segmente kombinieren nach zu kombinieren, werden die Segmente unter dem Schwellenwert (festgelegt durch Prozent/Wert <=) automatisch in dieser Spalte Kombinieren aktiviert. Diese Spalte ist nicht bearbeitbar.

Wenn Sie Segmente kombinieren nach auf Benutzerdefiniert gesetzt haben, können Sie die Segmente optional in dieser Spalte Kombinieren wählen.

Explodierte Versetzung (% von Radius)

Legen Sie die Versetzung für die explodierten Stücke fest. Die Einheiten werden in % des of the Kreisdiagrammradius angegeben.

Reihenfolge der Segmente

Legen Sie fest, wie die Segmente angeordnet werden sollen. Hier unterstützen wir vier Möglichkeiten, die Reihenfolge festzulegen:

  • Standard: Diese Option ist standardmäßig ausgewöhlt, das heißt, die Segmente werden gemäß dem Zeilenindex angeordnet.
  • Aufsteigend: Die Segmente werden in aufsteigender Reihenfolge der Werte angeordnet.
  • Absteigend: Die Segmente werden in absteigender Reihenfolge der Werte angeordnet.
  • Nach Spalte: Die Reihenfolge der Segmente wird mittels einer Spalte festgelegt. Wenn Sie diese Option gewählt haben, ist die Auswahlliste Reihenfolge nach verfügbar, mit der Sie eine Spalte aus dem aktuellen Arbeitsblatt auswählen können.

Bitte beachten Sie, dass Sie Startposition (Grad) verwenden können, um den ersten Azimut des ersten Stücks anzugeben.

Wenn Sie Nach Spalte für die Reihenfolge der Segmente wählen, können Sie die daneben befindliche Auswahlliste verwenden, um die von Ihnen gewünschte Spalte auszuwählen. Bitte beachten Sie, wenn Sie eine kategoriale Spalte/Textspalte wählen, dass die Reihenfolge der Zeilenindex ist.

Segmente gesamt nach

Legen Sie einen benutzerdefinierten prozentualen Gesamtanteil bzw. einen Wert für alle Segmente fest. Standardmäßig liegen die Gesamtprozent bei 100, und der Gesamtwert ist der Summenwert der Stücke.

Wenn Sie diese Segmente gesamt nach Prozent auf 50 setzen, ist die Summe der Prozentanteile aller Segmente 50 % und das Kreisdiagramm zeigt kein ganzes Kreisdiagramm, sondern einen Halbkreis.

Wenn Sie diese Segmente gesamt nach Wert auf Y setzen, das größer ist als der Summenwert Sum aller Stücke, wird der Kreis als ein Sektor gezeigt, θ = (Sum/Y)*360.

Rest als Segment zeichnen

Wenn Sie Segmente gesamt benutzerdefiniert haben, kann das Kreisdiagramm ein Sektor sein. Sie können dieses Bedienelement dann verwenden, um den verbleibenden Teil des Kreises als letztes Segment mit einer festgelegten Farbe zu zeigen.
