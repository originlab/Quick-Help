Datenanforderungen

Wählen Sie eine oder mehrere Y-Spalten oder einen Datenbereich aus mindestens einer Y-Spalte aus. Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen: Einfache 2D: 2-Punkte-Segment.

oder

Klicken Sie die Schaltfläche 2-Punkte-Segment in der Symbolleiste 2D Grafiken an.

Vorlage

Das Diagramm 2-Punkte-Segment wurde aus der Vorlage LINESYMB.OTP (im Origin-Programmordner installiert) erstellt.

Hinweise

Die Linienverbindung zwischen den Punkten wechselt zwischen einer graden Linie und keiner Linie (die Verbindung ist auf 2-Punkte-Segment auf der Registerkarte Linie des Dialogfelds Details Zeichnung gesetzt (Format: Zeichnung)). ). Demzufolge sind immer zwei aufeinanderfolgende Datenpunkte durch eine Linie verbunden (der erste und zweite Datenpunkt, der dritte und vierte Datenpunkt etc.). Die Datenpunkte werden als Symbole angezeigt.
