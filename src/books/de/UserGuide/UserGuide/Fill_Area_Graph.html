Datenanforderungen

Wählen Sie mindestens zwei Y-Spalten aus (oder einen Bereich aus mindestens zwei Y-Spalten). Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen: Einfache 2D: Füllfläche.

oder

Klicken Sie auf die Schaltfläche Füllfläche auf der Symbolleiste 2D-Grafiken.

Vorlage

FILLAREA.OTP (im Origin-Programmordner installiert).

Hinweise

Die Fläche zwischen den Datenzeichnungen (die zwei Y-Spalten) wird gefüllt. Standardmäßig wird nur ein Füllmuster verwendet. Sie können zur Registerkarte Linie des Dialogs Details Zeichnung gehen, um den Modus der Füllfläche auf Bis Datenzeichnung füllen - Obige und untere Farben setzen und dann die zwei unterschiedlichen Muster benutzerdefiniert anpassen. Dies ist nützlich, wenn Schnittpunkte für die zwei Y-Kurven existieren und sich diese nicht nur auf den X-Grenzen befinden.

Das Füllflächendiagramm unterstützt das Füllen mit Farbabbildung. Klicken Sie doppelt auf das Diagramm, um den Dialog Details aufzurufen, erweitern Sie den Zweig Layer1 und markieren Sie den ersten Unterknoten. Gehen Sie dann zur Registerkarte Muster und wählen Sie im Abschnitt Füllung die Option Farbpalette in der Auswahlliste Farbe. Die entsprechenden Registerkarten Farbpalette und Numerisches Format werden angezeigt. Sie können die Einstellungen der Farbpalette auf der Registekarte Farbplatte benutzerdefiniert anpassen.
