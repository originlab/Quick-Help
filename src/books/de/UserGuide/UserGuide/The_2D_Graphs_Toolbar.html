Die Symbolleiste 2D-Grafiken

2D-Graph-Toolbar

Die Schaltflächen der Symbolleiste 2D-Grafiken sind verfügbar, wenn ein Arbeitsblatt, eine Excel-Arbeitsmappe oder ein Diagrammfenster aktiv sind. Schaltflächen gewährleisten einen schnellen Zugriff auf allgemeine Vorlagen der 2D-Diagramme. Verwandte Diagrammtypen sind gruppiert und können über ein Popup-Menü geöffnet werden, indem Sie auf die Schaltfläche auf der rechten Seite der Schaltfläche klicken. Die letzte Schaltfläche öffnet die Template-Bibliothek, die Ihnen Zugriff auf benutzerdefinierte Diagrammvorlagen bietet.

Um die Schaltflächen der Symbolleiste 2D-Grafiken zu verwenden:

  • Wenn ein Arbeitsblatt oder eine Excel-Arbeitsmappe aktiv ist, markieren Sie die zu zeichnenden Daten und klicken Sie dann auf die gewünschte Schaltfläche der Symbolleiste 2D-Grafiken, um ein Diagramm zu erstellen.
  • Wenn ein Diagrammfenster aktiv ist, können Sie den Diagrammtyp (beispielsweise von einem Punktdiagramm in ein Säulendiagramm) für die aktive Datenzeichnung ändern.
