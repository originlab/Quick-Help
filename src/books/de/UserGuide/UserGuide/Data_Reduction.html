Datenmengen, die sich bereits in einem Arbeitsblatt befinden, können mit Hilfe verschiedener Hilfsmittel in Origin reduziert werden. Die folgende Tabelle bietet eine Übersicht der verfügbaren Hilfsmittel. Auf diese Hilfsmittel kann über das Menü Analyse: Datenbearbeitung bei aktivem Arbeitsblatt über das Menü Worksheet zugegriffen werden. Beachten Sie, dass einige der Hilfsmittel auch bei aktivem Diagrammfenster verfügbar sind.

Tool X-Funktionsname Beschreibung
Auf äquidistante X-Werte reduzieren reduce_ex Wird auf eine einzelne XY-Kurve angewendet. Ermöglicht das Erstellen einer reduzierten Ausgabe mit äquidistanten X-Werten. Verwendet Interpolation vor der Reduktion, wenn Eingabedaten nicht äquidistant sind.
Duplizierte X-Daten reduzieren reducedup Wird auf eine einzelne XY-Kurve angewendet. Reduziert alle XY-Paare mit duplizierten X-Werten und bietet Optionen zum Bearbeiten der Y-Werte. Beachten Sie auch das Menü Worksheet: Duplizierte Zeilen entfernen, das auch mit Textdaten arbeitet.
Zeilen reduzieren

wdelrows

reducerows

wreducerows

wdelrows: Wird auf alle oder Teilbereiche von Spalten des Arbeitsblatts angewendet. Eine festgelegte Anzahl von Zeilen wird jedes Mal übersprungen, um die Datengröße zu reduzieren.

reducerows: Wird auf einen Auswahl von Spalten angewendet. Alle N Datenpunkte mit festgelegtem einfachen Statistikwert wie dem Durchschnitt werden ersetzt.

wreducerows: Wird auf alle oder Teilbereiche von Spalten des Arbeitsblatts angewendet. Diese X-Funktion kombiniert die Funktionen von wdelrows und reducerows.

Spalten reduzieren

wreducecols

Wird auf alle oder Teilbereiche von Spalten des Arbeitsblatts angewendet und reduziert Arbeitsblattspalten durch Löschen oder statistisches Zusammenfügen.

Nach Gruppe reduzieren reducexy Wird auf eine einzelne XY-Kurve angewendet. Ermöglicht die Gruppierung der Daten nach Zahl der Gruppe, jedem N-ten Punkt oder nach X-Inkrement und dann das Ersetzen von X und Y durch Größen wie Mittelwert, Minimum und Maximum.

Daten können auch beim Importieren einer Datei in ein Arbeitsblatt reduziert werden. Der Dialog Optionen für verschiedene (nicht alle) Dateitypen sowie der Importassistent bieten Bedienelemente für den Teilweisen Import, wobei Anwender n Zeilen importieren und m Zeilen überspringen und das Muster wiederholen können, um die Datengröße zu reduzieren. Später kann die Datei ggf. vollständig erneut importiert werden.
