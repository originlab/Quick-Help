Ausgabe Ergebnisprotokoll

Aktivieren Sie das Kontrollkästchen, um die Ergebnisse im Ergebnisprotokoll auszugeben.

Berechnen

Trennschärfe Die Trennschärfe für den festgelegten Stichprobenumfang wird berechnet.
Stichprobenumfang Der Stichprobenumfang für eine vorgeschlagene Trennschärfe wird berechnet.

Testspezifikationen

1. Gruppenmittelwert Mittelwert der Gruppe 1
2. Gruppenmittelwert Mittelwert der Gruppe 2
Standardabweichung Gepoolte Standardabweichung für die zwei Stichproben
Alpha Wahrscheinlichkeit der Zurückweisung der Nullhypothese, wenn es wahr ist
Hypothetische Trennschärfe(n) Legen Sie eine durch Leerzeichen getrennte Liste der hypothetischen Trennschärfe(n) fest. Verfügbar, wenn Berechnen = Stichprobenumfang.
Hypothetische/r Stichprobenumfang/-umfänge Die Option legt eine durch Leerzeichen getrennte Liste der hypothetischen Stichprobenumfänge fest. Verfügbar, wenn Berechnen = Trennschärfe.
Seite

Wählen Sie einen ein- oder beidseitigen Test für einen t-Test bei zwei Stichproben aus.

1-seitig
Ein einseitiger Test wird durchgeführt.
2-seitig
Ein beidseitiger Test wird durchgeführt.

Optionen

Die zu verwendenden Bedienelemente, wenn zusätzliche Werte für Trennschärfe und Stichprobenumfang durch den Arbeitsblattdatensatz zur Verfügung stehen:

Beim Berechnen der Trennschärfe (Berechnen = Trennschärfe):

Weitere Hypothetische Stichprobenumfänge Aktivieren Sie diese Option, um die Eingabeelemente Spalte der Hypothetischen Stichprobenumfänge zu aktivieren.
Spalte der Hypothetischen Stichprobenumfänge Legen Sie die Spalte der Werte für den Stichprobenumfang fest. Nur verfügbar, wenn Weitere Hypothetische Stichprobenumfänge aktiviert ist.

Beim Berechnen des Stichprobenumfangs (Berechnen = Stichprobenumfang):

Weitere Hypothetische Trennschärfen Aktivieren Sie diese Option, um die Eingabeelemente Spalte der Hypothetischen Stichprobenumfänge zu aktivieren.
Spalte der Hypothetischen Trennschärfen Legen Sie die Spalte für die Trennschärfewerte fest. Nur verfügbar, wenn Weitere Hypothetische Trennschärfe(n) aktiviert ist.

Ergebnisausgabe

Bestimmen Sie, wo das auszugebende Berichtsblatt erstellt wird. Weitere Einzelheiten finden Sie unter: Analyseberichtsblätter.
