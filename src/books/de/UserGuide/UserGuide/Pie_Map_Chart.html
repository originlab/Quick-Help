Karte mit Kreisdiagrammen

Pie-Map

Die Karte mit Kreisdiagrammen ist eine Reihe von Kreisdiagrammen, deren Lage von einem XY-Datensatz bestimmt wird. Der Kartenhintergrund in dieser Grafik wird mit der App Google Map Import importiert.

Datenanforderungen

Wählen Sie X mit mindestens 3 Y-Spalten aus, um eine Karte mit Kreisdiagrammen zu erstellen. Die ersten zwei Spalten (X und Y) entscheiden die Position von jedem Kreis. Weitere Y-Spalten bestimmen die Kreissegmente. Die Größe des Kreises wird von der Summe aller Kreissegmente bestimmt, die weiterhin auf der Registerkarte Symbol im Dialog Details Zeichnung benutzerdefiniert angepasst werden kann.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen > Balken, Kreis, Fläche: Karte mit Kreisdiagrammen.

Vorlage

PieMap.OTPU (installiert im Origin-Programmordner).

Notizen

Die Karte mit Kreisdiagrammen ist eine Reihe von Kreisdiagrammen, die XY-Daten verwenden, um die Kreise zu positionieren. Jedes Kreisdiagramm wird durch eine Zeile der Y-Spalten für die Kreissegmente erstellt. Außerdem können Sie die Kreisgröße steuern, indem Sie eine weitere Spalte oder die Segmente gesamt verwenden, ähnlich wie beim Blasendiagramm.

Um festzulegen, wie die Kreisdiagramme in Bezug auf Farbe und Größe aussehen, können Sie zur Registerkarte Symbol im Dialog Details Zeichnung gehen.

Sie können die Strg-Taste drücken, um doppelt auf ein Kreisdiagramm zu klicken und einen speziellen Punkt hinzuzufügen bzw. zu ziehen, um diesen Kreis neu zu positionieren.

Im Fall weiterer Bedienelemente, z. B. beim Hinzufügen von Beschriftungen, behandelt Origin die Kreise als Symbole.
