Die Liste des Linienstils (Details Zeichnung)

PD-line-style-list-tab

Die Registerkarte Liste des Linienstils wird hinzugefügt, wenn Sie (a) eine Arbeitsblattspalte der kategorialen Werte auswählen, um auf der Registerkarte Linie den Linienstil zu steuern (d. h., Sie wählen eine Arbeitsblattspalte aus der Auswahlliste) oder Sie (b) den Linienstil = Inkrement setzen.

Per Standard verwendet Origin die LabTalk-Liste der Linientypen. Sie können diese Liste jedoch anpassen, neu anordnen oder Stile aus ihr entfernen, damit sie ihren Anforderungen entspricht. Aktivieren Sie das Kontrollkästchen Benutzerdefinierte Inkrementliste verwenden, um Bedienelemente freizugeben, und befolgen Sie die Anweisungen auf dieser Seite, um die Liste benutzerdefiniert anzupassen.

Beachten Sie, dass Sie seit Origin 2022b benutzerdefinierte Strichlinien "nebenbei" definieren können und dass diese benutzerdefinierten Definitionen beim Erstellen Ihrer Linienstilliste verwendet werden können.
