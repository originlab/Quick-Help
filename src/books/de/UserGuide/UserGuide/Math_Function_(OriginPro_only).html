Mathematische Funktionen (nur OriginPro)


Die Funktion imgMathfun färbt das Bild nach einer mathematischen Funktion neu ein. Um diese Funktion zu verwenden:

Wählen Sie Bild: Arithmetische Transformation: Mathematische Funktionen im Origin-Menü.

Weitere Informationen über diese Funktion finden Sie in der Dokumentation der X-Funktion imgMathfun in der Origin-Hilfe. Diese kann über das Befehlsfenster mit folgendem Befehl aufgerufen werden:

help imgMathfun
