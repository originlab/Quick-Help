Die meisten von Origins Dialogen für die Analyse und Datenverarbeitung enthalten die Funktion Dialogdesign. Dialogdesigns speichern die Einstellungen von Analysedialogen in einer externen Datei auf der Festplatte für die zukünftige Verwendung. Außerdem können Sie mehrere Designdateien für einen gegebenen Dialog speichern. Dies ermöglicht Ihnen, verschiedene Kombinationen von Eingabe-, Analyse- und Ausgabeoptionen für verschiedene Datensätze oder verschiedene Endanwender festzulegen.

Wenn solche Dialoge über Menü zugänglich sind, können diese benutzerdefinierten Designs als benutzerdefinierte Elemente in einem Ausklappmenü neben den Menübefehlen des Dialogs angezeigt werden. Zusätzlich können diese Analysedesigns aus der Liste Dialogdesign im oberen Bereich des Dialogs ausgewählt werden (siehe unten).

Erfahrene Anwender sollten außerdem wissen, dass gespeicherte Designs auch programmatisch aufgerufen werden können. Dies ermöglicht Ihnen, Designs in Ihre Skripts aufzunehmen. Weitere Informationen finden Sie im Abschnitt LabTalk-Programmierung in der Programmierhilfe (Hilfe: Programmierung).

Spezielle Dialogdesigns

Es gibt für jedes Hilfsmittel zwei Designs:

  • <Standard>
  • <Zuletzt verwendet>

Sie können Einstellungen als <Standard> speichern. Jedes Mal, wenn Sie dieses Hilfsmittel aus dem Origin-Menü öffnen, werden die Einstellungen aus diesem Design geladen, sobald sich der Dialog öffnet. Beachten Sie, dass das Design <Standard> durch Zurücksetzen entfernt werden kann (Klicken Sie auf das Dreieck rechts neben Dialogdesign und wählen Sie Zurücksetzen im Kontextmenü). Sobald das Design <Standard> entfernt ist, werden die standardmäßigen Werkseinstellungen geladen, wenn Sie das Dialogfeld öffnen.

Das Design <Zuletzt verwendet> wird automatisch mit den Einstellungen aktualisiert, die Sie zuletzt verwendet haben, als Sie auf OK im Dialogfeld des Hilfsmittels geklickt haben. Wenn Sie das Hilfsmittel im Menü "Zuletzt verwendet" öffnen, werden die Einstellungen in diesem Design geladen.

Speichern von Dialogdesigns

Um Ihr Design zu speichern, passen Sie die Einstellungen im Dialog des Hilfsmittels an. Führen Sie anschließend Folgendes aus:

  1. Klicken Sie auf die dreieckige Schaltfläche rechts neben Dialogdesign.
  2. Verwenden Sie verschiedene Menübefehle, um ein Design nach Ihren Anforderungen zu speichern:
    • Wenn Sie die Einstellungen als Standarddesign speichern möchten, wählen Sie Speichern als <Standard>.
    • Wenn Sie die Einstellungen in ein Zielarbeitsblatt speichern möchten (diese Funktion wird nur von wenigen X-Funktionen, wie impASC, unterstützt), wählen Sie Speichern als <Blatt>.
    • Wenn Sie die Einstellungen in eine Zielarbeitsmappe speichern möchten (diese Funktion wird nur von wenigen X-Funktionen unterstützt), wählen Sie Speichern als <Mappe>.
    • Wenn Sie die Einstellungen im aktuellen Design speichern möchten, klicken Sie Speichern.
    • Wenn Sie die Einstellungen als ein anderes Design speichern möchten, wählen Sie Speichern unter.
  3. Wenn Sie im letzten Schritt Speichern unter gewählt haben, öffnet sich ein Dialog. Geben Sie den Namen des Designs in das Textfeld Designname ein.

Sie können die Designs auch nach Ausführung einer Operation speichern, indem Sie auf das grüne Schlosssymbol auf Ihrem Ausgabeblatt klicken und Design speichern unter... im Kontextmenü auswählen.

Anwenden von Dialogdesigns

Es gibt verschiedene Möglichkeiten, ein Dialogdesign anzuwenden.

Designs in den Dialog laden

Sie können ein Design aus der Liste Dialogdesign auswählen. Diese erscheint, wenn Sie auf das Dreieck rechts neben Dialogdesign klicken. Hierdurch werden die Einstellungen dieses Designs geladen. Beachten Sie, dass die Designliste auch die oben erwähnten speziellen Designs enthält.

Anwenden von Designs mit einem Menübefehl

Wenn ein Dialogdesign gespeichert wird, wird es außerdem zu einem Menübefehl, der es Ihnen erlaubt, eine Analyseroutine mit vordefinierten Einstellungen auszuführen. Angenommen, Sie haben bereits einige Dialogdesigns für das Hilfsmittel Glätten gespeichert. Sie können diese im Ausklappmenü des Hilfsmittels sehen:

Um ein Design zum Ausführen des Hilfsmittels zu verwenden, ohne den Dialog zu öffnen, markieren Sie Ihre Eingabedaten und klicken Sie anschließend auf das Dialogdesign im Ausklappmenü des Hilfsmittels. Wenn Sie den Dialog öffnen und dann das Design laden möchten, klicken Sie bei gedrückter Umschalttaste auf das Dialogdesign im Ausklappmenü.

Anwenden von Designs von der Befehlszeile aus

Wenn Sie ein Design für eine X-Funktion, die mit LabTalk aufgerufen werden kann, gespeichert haben, können Sie den Befehl -t verwenden, um das Design vom Befehlsfenster aus anzuwenden. Angenommen, Sie haben ein Design mit dem Namen "MyTheme" für die X-Funktion Glätten gespeichert. Mit folgendem Befehl können Sie das Design verwenden, um die Daten in Spalte B des aktiven Worksheets zu glätten:

smooth Col(B) -t MyTheme;

Weitere Informationen über das Ausführen von LabTalk-Befehlen über die Befehlszeile finden Sie in der Programmierhilfe für LabTalk-Skript (wählen Sie Hilfe: Programmierung im Origin-Menü).
