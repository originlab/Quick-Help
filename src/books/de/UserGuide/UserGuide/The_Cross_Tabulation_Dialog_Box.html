Eingabedaten

Wählen Sie Daten für Kreuztabelle und Chi-Quadrat.

Zeile Wählen Sie Daten aus einer Spalte aus, um eine Zeilenvariable festzulegen.
Spalte Wählen Sie Daten aus einer Spalte aus, um eine Spaltenvariable festzulegen.
Layer Wählen Sie Daten aus einer Spalte aus, um eine Layervariable festzulegen. Wenn eine Layervariable festgelegt ist, wird die Kreuztabelle zwischen der Zeilen- und Spaltenvariable auf jeder Ebene der Layervariable erstellt.
Häufigkeiten Wenn Daten im Häufigkeitendatenmodus zusammengefasst werden, wählen Sie die Häufigkeitenspalte, um die Häufigkeitenvariable festzulegen.

Statistik

Kontingenztabelle Dieser Zweig bestimmt, welche Statistiken in der Kontigenztabelle berechnet und ausgegeben werden.
  • Anzahl: Anzahlen, Erwartete Anzahlen
  • Prozentanteile: Prozentanteile der Zeilenanzahl, Prozentanteil der Spaltenanzahl, Prozentanteil der Gesamtanteil
  • Residuen: Residuen, Standardisierte Residuen, Korrigierte Residuen

Hinweis: Um weitere Einzelheiten zu diesen Statistiken zu erfahren, lesen Sie bitte die Einführungsseite und die Seite Algorithmus.
Signifikanzniveau Legen Sie das Signifikanzniveau für den folgenden Hypothesentest fest. Der Standardwert ist 0,05.

Tests

Chi-Square Test Legen Sie fest, ob Chi-Quadrat-Tests durchgeführt werden sollen, um zu bestimmen, ob eine signifikante Beziehung zwischen Variablen in der Kontingenztabelle besteht.

Hinweis: Um weitere Einzelheiten zu diesen Statistiken zu erfahren, lesen Sie bitte die Seiten Einführung, Algorithmus und Ergebnisse interpretieren.
Fishers exakter Test Legen Sie fest, ob Fishers Exakter Test durchgeführt werden soll, um zu bestimmen, ob eine signifikante Beziehung zwischen Variablen in der Kontingenztabelle besteht.

Hinweis: Um weitere Einzelheiten zu diesen Statistiken zu erfahren, lesen Sie bitte die Seiten Einführung, Algorithmus und Ergebnisse interpretieren.
Assoziationsmaße Dieser Zweig bestimmt, welche Statistiken berechnet werden sollen, um die Stärke der Beziehung zwischen den Variablen in der Kontingenztabelle zu bewerten. Es gibt die folgenden zwei Kontrollkästchen.
  • Gamma
  • Kendalls Tau-b und Tau-c
  • Phi
  • Cramérs V
  • Kontingenzkoeffizient
  • Somers D
  • Lambda
  • Unsicherheitskoeffizient

Hinweis: Um weitere Einzelheiten zu diesen Statistiken zu erfahren, lesen Sie bitte die Seiten Einführung, Algorithmus und Ergebnisse interpretieren.
Übereinstimmungen Dieser Zweig bestimmt, welcher Test durchgeführt werden soll, um die Übereinstimmung der verglichenen Datenpaare zu messen und zu testen. Es gibt die folgenden zwei Kontrollkästchen.
  • Kappa
  • Bowkers Test (McNemar für [2*2])

Hinweis: Um weitere Einzelheiten zu diesen Statistiken zu erfahren, lesen Sie bitte die Seiten Einführung, Algorithmus und Ergebnisse interpretieren.
Andere Messwerte Legen Sie fest, ob andere Tests durchgeführt und weitere Statistiken berechnet werden sollen. Es gibt die folgenden zwei Kontrollkästchen.
  • Cochran-Mantel-Haenszel:
    Legen Sie fest, ob die Beziehung von mehreren Kontingenztabellen analysiert werden soll, einschließlich Test auf bedingte Unabhängigkeit, Homogenitätstest des Quotenverhältnisses und allgemeines Quotenverhältnis.
  • Relatives Risiko:
    Legen Sie fest, ob das relative Risiko berechnet werden soll, um das Verhältnis der Wahrscheinlichkeit, dass ein Ereignis in einer Gruppe eintritt, zu der Wahrscheinlichkeit, dass das Ereignis in einer Vergleichsgruppe eintritt, zu messen.
  • Quotenverhältnis:
    Legen Sie fest, ob das Quotenverhältnis berechnet werden soll, um das Verhältnis der Wahrscheinlichkeit, dass ein Ereignis oder Ergebnis eintritt, zu der Wahrscheinlichkeit, dass das Ereignis nicht eintreten wird, zu messen.

Hinweis: Um weitere Einzelheiten zu diesen Statistiken zu erfahren, lesen Sie bitte die Seiten Einführung, Algorithmus und Ergebnisse interpretieren.

Ausgabe

Ausgabe Ergebnisprotokoll Mit diesem Kontrollkästchen wird festgelegt, ob das Ergebnisprotokoll angezeigt wird oder nicht.
Mosaikdiagramm Legen Sie fest, ob ein Mosaikdiagramm, bei dem die Fläche jedes Rechtecks proportional zu den Anteilen der Y-Variable auf jeder Stufe der X-Variable ist, angezeigt werden soll.
Bericht der Kreuztabelle Legen Sie das Blatt für das Ergebnis der Kreuztabelle fest. Der Standardwert ist ein neues Blatt in der Arbeitsmappe der Eingabedaten.
Daten zeichnen Legen Sie das Blatt für die Daten des Mosaikdiagramms fest. Der Standardwert ist ein neues Blatt in der Arbeitsmappe der Eingabedaten.
