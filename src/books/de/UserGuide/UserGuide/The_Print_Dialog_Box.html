Das Dialogfeld Drucken

Das Dialogfeld Drucken wird zum Anpassen der Druckoptionen und Durchführen des Drucksvorgangs verwendet. Beachten Sie, dass seine Bedienelemente von dem aktiven Fenstertyp (Grafik, Arbeitsmappe, Excel-Arbeitsmappe etc.) abhängen.

Die Gruppe Drucker

Wählen Sie einen Drucker in der Auswahlliste Name aus. Befindet sich der gewünschte Drucker nicht in der Liste der verfügbaren Drucker, fügen Sie ihn über die Windows-Systemsteuerung ein.

Klicken Sie auf die Schaltfläche Einstellungen, um die Optionen für den der Auswahlliste Name ausgewählten Druckertreiber einzustellen.

Aktivieren Sie das Kontrollkästchen In Datei umleiten, um das (die) gewählte(n) Fenster in eine Datei zu drucken. Diese Option muss ausgewählt sein, wenn Sie eine PostScript-Datei erstellen möchten. Sie ist nur verfügbar, wenn Sie in der Auswahlliste Current (Aktuell) in der Gruppe Grafik drucken unten ausgewählt ist bei aktivem Grafik-, Funktionsgrafik- oder Layoutseitenfenster.

Die Gruppe Grafik drucken, Drucken oder Bereich drucken

  • Bei aktivem Grafik-, Funktionsgrafik- oder Layoutseitenfenster
Auswahlliste Objektauswahl

Legt die zu druckenden Grafik-, Funktionsgrafik- oder Layoutseitenfenster in dem Projekt fest. Hier können Sie die aktuelle Datei, die Dateien im Projekt (geöffnete oder alle), die Dateien im aktiven Ordner (geöffnete oder alle) oder alle Dateien im aktiven Ordner und seinen Unterordnern auswählen.
Optionen

Diese Schaltfläche öffnet das Dialogfeld Weitere Druckoptionen.
Punkte auslassen

Worksheet-Daten, Punkte übergehen steuert die Anzahl der Arbeitsblattdatenpunkte, die in der (den) gedruckten Grafik(en) angezeigt werden. Wenn dieses Kontrollkästchen aktiviert ist, erscheint das Textfeld Max. # der Punkte pro Kurve. Sie können dann die maximale Anzahl der für jedes Diagramm zu druckenden Punkte in diesem Textfeld festlegen.

Das Auslassen von Punkten erhöht die Druckgeschwindigkeit von Diagrammen, die umfangreiche Datensätze enthalten. Diese Option hat keine Auswirkungen auf die Arbeitsblatt- oder Bildschirmdarstellung der Datensätze. Wenn dieses Kontrollkästchen aktiviert wurde, deaktiviert Origin auch komplizierte Linientypen und Schraffur-Muster, um sie in eine vom Drucker unterstützte Form zu konvertieren.

Analog zum Kontrollkästchen und Textfeld Worksheet-Daten, Punkte übergehen steuern auch das Kontrollkästchen und Textfeld Matrix Daten, Maximale Punktzahl die Anzahl der Matrix-Datenpunkte, die im gedruckten Diagramm dargestellt werden.
Farben als Graustufen/Nur reine Farben

Origin stellt zwei Bedienelemente zum Drucken von Farbdiagrammen als Schwarzweiß-Ausdruck bereit: die Kontrollkästchen Farben als Graustufen und Nur reine Farben. Nur eine der beiden Kontrollkästchen ist auswählbar, wenn Sie Datei: Drucken wählen. Origin aktiviert eine der beiden Optionen, nachdem es beim Standarddrucker abgefragt hat, wie viele Farben er unterstützt.

Für Drucker, die nur zwei Farben unterstützen, steht das Kontrollkästchen Farben als Graustufen zur Verfügung.

  • Wenn Sie dieses Kontrollkästchen deaktivieren, druckt Origin Schwarz-auf-Weiß. In diesem Fall werden Achsen und Beschriftungen als Schwarz-auf-Weiß ausgedruckt, unabhängig von ihrer Farbe. (In Origin 6.0 waren sowohl der Hintergrund als auch die Achsen nicht sichtbar, weil sie beide weiß gedruckt wurden, wenn der Hintergrund schwarz und die Achsen weiß waren.)
  • Wenn Sie dieses Kontrollkästchen aktivieren, sendet Origin die Farben so zum Drucker, wie sie am Bildschirm dargestellt werden, und lässt sie durch den Drucker bestmöglich umwandeln.

Für Drucker, die mehr als zwei Farben unterstützen (das impliziert nicht, dass es sich um einen Farbdrucker handelt, weil der Drucker mehrere nicht-gerasterte Grauschattierungen unterstützen könnte), steht das Kontrollkästchen Nur reine Farben zur Verfügung.

  • Wenn Sie dieses Kontrollkästchen deaktivieren, sendet Origin die Farben so zum Drucker, wie sie am Bildschirm dargestellt werden, und lässt sie durch den Drucker bestmöglich umwandeln.
  • Wenn Sie dieses Kontrollkästchen aktivieren, druckt Origin Schwarz-auf-Weiß wie es für das Kontrollkästchen Farben als Graustufen beschrieben wurde.
  • Bei aktivem Arbeitsblatt- oder Matrixfenster
Auswahl

Aktivieren Sie das Kontrollkästchen Auswahl, um einen Bereich der Daten zu drucken. Geben Sie den gewünschten Spalten- und Zeilenbereich in den entsprechenden Textfeldern ein.

Alternativ markieren Sie die gewünschten Arbeitsblatt- oder Matrixdaten, bevor dieses Dialogfeld geöffnet wird. Aktivieren Sie das Kontrollkästchen Auswahl, um den in den zugehörigen Textfeldern ausgewählten Bereich anzuzeigen (und zu drucken).
Option

Diese Schaltfläche öffnet das Dialogfeld Optionen .
  • Bei aktivem Notizfenster
Alle

Sollte der Text in einem Notizfenster über eine Seite hinausgehen, aktivieren Sie die Option Alle, um alle Seiten des Fensters auszudrucken.
Seiten

Um eine bestimmte Seite des Notizfensters zu drucken, aktivieren Sie die Option Seiten und geben die gewünschten Seitenzahlen in die zugehörigen Textfelder ein.
  • Bei aktiver Excel-Arbeitsmappe
Druckbereich

Um einen Bereich von Seiten zu drucken, aktivieren Sie die Option Seiten und legen dann den Bereich in den Listenfeldern Von und Bis fest.
Drucken

Wählen Sie die gewünschte Option aus, um ein Blatt, einen ausgewählten Bereich oder eine ganze Arbeitsmappe zu drucken.

Die Gruppe Kopien

Im Drehfeld Anzahl geben Sie die Anzahl der zu druckenden Kopien ein oder wählen diese hier aus.

Aktivieren Sie das Kontrollkästchen Sortieren, wenn Sie möchten, dass die Kopien sortiert werden.

Die Schaltfläche OK

Klicken Sie auf diese Schaltfläche, um mit den aktuellen Einstellungen des Dialogfelds Drucken zu drucken.
