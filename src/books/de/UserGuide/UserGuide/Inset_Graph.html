Eingefügtes Diagramm

Inset-Plot

Ein eingefügtes Diagramm ist ein Layer, der zu einem schon bestehenden Layer in einem Diagrammfenster hinzugefügt wird. Das eingefügtes Diagramm ist kleiner, so dass das Diagramm im größeren Layer des Diagrammfensters zumindest teilweise sichtbar bleibt.

Diagramm erstellen

Es gibt zwei Möglichkeiten, um ein eingesetztes Diagramm zu erstellen.

  • Um einen eingefügten Layer zu einem 2D-Diagramm hinzuzufügen:
  1. Aktivieren Sie das Diagrammfenster mit dem Hauptlayer.
  2. Klicken Sie entweder auf die Schaltfläche Eingefügtes Diagramm hinzufügen oder Eingefügtes Diagramm mit Daten hinzufügen auf der Symbolleiste Diagramm.

ODER

  • Um einen anderen Diagrammlayer zu kopieren und in den Hauptlayer einzufügen:
  1. Erstellen Sie zwei separate Diagrammfenster von Hauptlayer und eingefügtem Layer.
  2. Aktivieren Sie das Diagrammfenster des eingefügten Layers, klicken Sie auf den weißen Bereich des Layers (d.h. auf eine beliebige Stelle innerhalb des Achsenrahmen ohne Datenzeichnung), um den Layer zu markieren. (Wenn der Layer ausgewählt ist, werden um den Achsenrahmen herum Ankerpunkte angezeigt.)
  3. Wählen Sie bei markiertem Layer Bearbeiten: Kopieren oder Strg + C, um den eingefügten Layer zu kopieren.
  4. Aktivieren Sie das Diagrammfenster des Hauptlayers und drücken Sie Strg + V, um den eingefügten Layer zu dem Hauptlayer hinzuzufügen. Sie können den eingefügten Layer erneut in dem neuen Diagramm markieren und die Ankerpunkte verwenden, um den eingefügten Layer zu verschieben bzw. in der seiner Größe zu verändern.

Notizen

Die Verknüpfung von Hauptlayer und eingefügtem Layer, die Position und/oder die Größe im Dialog Layermanagement verändert werden.

Der Hauptlayer und der eingefügte Layer können getrennt voneinander im Dialog Details Zeichnung benutzerdefiniert angepasst werden.
