Datenanforderungen

Wählen Sie genau eine Y-Datenspalte (oder einen Bereich aus einer Spalte) aus.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen > Einfache 2D: Ring mit Segmenten gesamt = 100.

Vorlage

DonutTranspose.OTP (installiert im Origin-Programmordner).

Hinweise

Das Diagramm Ring mit Segmenten gesamt = 100 ist ein sehr beliebtes Mittel, den Fortschritt in Bezug auf das Ziel zu messen.

Dieses Ringdiagramm (mit Gesamt =100) kann von einem Wert aus gezeichnet werden oder von einem Datensatz/einer Spalte aus. Jeder ausgewählte Wert wird in einen separaten Kreis als ein Segment gezeichnet, dass den Prozentanteil von dem festgelegten Gesamtwert darstellt. Zum Teil könnte man denken, dass diese Art von Ringdiagramm ein bogenförmiges Säulen-/Balkendiagramm ist, das die Prozentanteile des Fortschrttts relativ zur gesamten Aufgabe zeigt.

Bei einem allgemeinen Ringdiagramm wird jeder Datensatz/jede Spalte als ein voller Kreis mit mehreren Segmenten als Prozentanteile der Summe des Datensatzes/der Spalte gezeichnet.

Referenz

Um Ansichtswinkel, Drehung, Radius und Mitte benutzerdefiniert anzupassen, siehe Die Registerkarte Geometrie (Details Zeichnung).

Um Segmentbeschriftungen benutzerdefiniert anzupassen, siehe Die Registerkarte Beschriftungen (Details).
