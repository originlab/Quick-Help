Datenanforderungen

Wählen Sie genau eine Y-Datenspalte (oder einen Bereich aus einer Spalte) aus.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen > Einfache 2D: Ringdiagramm.

Vorlage

Doughnut.OTP (installiert im Origin-Programmordner).

Hinweise

Ein Ringdiagramm mit einem einzelnen Ring ist eine Variante des 2D-Kreisdiagramms, das über ein Innenloch mit benutzerdefinierter Größe verfügt. Wenn Sie mehrere Datensätze zusammen zeichnen, werden mehrere Ringdiagramme abhängig voneinander erstellt und in radialer Richtung gestapelt mit einem ausgeschnittenen Bereich in der Mitte.

Für das Ringdiagramm mit einzelnem Ring werden alle benutzerdefinierten Anpassungen von 2D-Kreisdiagrammen unterstützt, wie das Festlegen des Inkrementradius für die Segmente, das Kombinieren und Explodieren von Segmenten usw.

Für das Ringdiagramm mit mehreren Ringen können Sie den Inkrementradius für die Segmente festlegen, Segmente kombinieren usw., aber Sie können die Segmente nicht explodieren, da alle Ringe abhängig voneinander sind. Falls Sie diese Ring auf der Registerkarte Gruppe unabhängig voneinander machen, können Sie nur die Segmente des äußeren Rings explodieren.

Referenz

Um Ansichtswinkel, Drehung, Radius und Mitte benutzerdefiniert anzupassen, siehe Die Registerkarte Geometrie (Details Zeichnung).

Um die Segmente, wie explodierende und kombinierte Segmente, benutzerdefiniert anzupassen, siehe Die Registerkarte Segmente (Details Zeichnung).

Um Segmentbeschriftungen benutzerdefiniert anzupassen, siehe Die Registerkarte Beschriftungen (Details).
