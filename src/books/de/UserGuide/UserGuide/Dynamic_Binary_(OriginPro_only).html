Dynamisch Binär (nur OriginPro)


Die Funktion imgDynamicBinary konvertiert ein bestimmtes Bild zu einem Schwarzweißbild ohne den Wert seiner Bits pro Pixel zu ändern, indem ein Schwellenwert für jedes Pixel des Bildes verwendet.

Um diese Funktion zu verwenden,

wählen Sie Bild: Konvertierung: Dynamisch Binär im Origin-Menü.

Weitere Informationen über diese Funktion finden Sie in der Dokumentation der X-Funktion imgDynamicBinary in der Origin-Hilfe. Diese kann über das Befehlsfenster mit folgendem Befehl aufgerufen werden:

help imgDynamicBinary
