FAQ-412 Wie passe ich Drag&Drop für verschiedene Importdateitypen benutzerdefiniert an?

Drag-and-Drop-Import

Letztes Update: 20.09.2022


Wenn Sie eine Datei per Drag& Drop in eine neue Origin-Arbeitsmappe ziehen, folgt der Umgang mit diesen Datendateien dieser Logik:

  1. Falls im Blatt Importeinstellungen gespeichert wurden, regeln diese Einstellungen den Import.
  2. Falls keine Einstellungen des Importdialogs im Blatt gespeichert wurden, sucht Origin im Datenordner nach einem passenden Origin-Importfilter (*.oif). Falls einer gefunden wird, regelt dieser Importfilter den Import.
  3. Falls es keinen Importfilter im Datenordner gibt, prüft Origin den Ordner Anwenderdateien\Filters auf einen geeigneten Importfilter. Falls ein USER-Filter existiert und das entsprechende Kontrollkästchen Drag&Drop unterstützen im Dialog Importfilter verwalten aktiviert ist, regelt dieser Filter den Import.
  4. Wenn kein passender Filter im Ordner Anwenderdateien\Filters gefunden wurde, prüft Origin, ob es einen installierten Datenkonnektor gibt, der dem Dateityp entspricht. Wenn ein passender Datenkonnektor gefunden wird, regelt der Datenkonnektor den Import.
  5. Falls kein passender Datenkonnektor existiert, prüft Origin den das Verzeichnis <Programmverzeichnis>\Filters auf einen Filter. Falls ein Filter existiert und das entsprechende Kontrollkästchen Drag&Drop unterstützen im Dialog Importfilter verwalten aktiviert ist, wird der Systemfilter verwendet.
  6. Falls die vorherigen Schritte fehlschlagen, fragt Origin über eine Achtungsmeldung, ob Sie den Importassistent zum Importieren der Datei öffnen möchten.

Wenn Sie mit Ihrem Origin-Importfilter (*.oif) zufrieden sind, aber einige der Vorteile beim Verwenden eines Datenkonnektors mögen, können Sie die beiden Methoden kombinieren, indem Sie Daten: Mit Datei verbinden: Importfilter wählen. Für Drag&Drop versuchen Sie bitte, Ihre *.oif-Datei im Datenordner abzulegen.

Schlüsselwörter:Importfilter, *.oif, Importeinstellungen, Import benutzerdefiniert anpassen, Datenkonnektor, verbinden
