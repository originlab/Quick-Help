Datenanforderungen

Wählen Sie mindestens zwei Y-Spalten aus (oder einen Bereich aus mindestens zwei Y-Spalten). Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen > Kategorial: Banddiagramm, 100 %.

Vorlage

RibbonChart.OTPU (installiert im Origin-Programmordner).

Hinweise

Das 100-%-Banddiagramm ist eine Variante des gestapelten Säulendiagramms, 100 %, bei dem Bänder als Verbindungslinien dienen. Sie können die Bändern auf der Registerkarte Linie des Dialogs Details Zeichnung benutzerdefiniert anpassen.
