XYY 3D-Balkendiagramm mit Fehlerbalken

3DBar-Graph-with-ErrBar

Datenanforderungen

Wählen Sie mindestens eine Y-Spalte und eine verbundene Y-Spalte für den Fehlerbalken (oder einen Bereich aus mindestens einer Y-Spalte und einer verbundenen Y-Spalte für den Fehlerbalken). Sie müssen die Diagrammzuordnungen der Arbeitsblattspalte definieren, bevor sie die Art von Zeichnung erstellen. Jeder zugeordnete Fehlerbalken-Datensatz muss sich rechts neben dem Y-Datensatz befinden, auf den er sich bezieht (Beispiel: Y1, yEr1, Y2, yEr2, Y3, yEr3 usw.).

Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Sie können zu dem bestehenden XYY 3D-Balken auch Fehlerbalken über den Dialog Diagrammeinstellungen oder den Dialog Details Zeichnung hinzufügen.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen: 3D: XYY 3D Balken.

oder

Klicken Sie auf die Schaltfläche XYY 3D Balkendiagramm der Symbolleiste 3D- und Konturdiagramme.

Vorlage

  • glBar3D.OTP (OpenGL)
  • BAR3D.OTP

(installiert im Origin-Programmordner)
