Der Dialog 3D-OpenGL-Einstellungen

GraphRef-3DOpenGL-Dialog

Der Dialog 3D-OpenGL-Einstellungen kann entweder durch Auswahl im Menü über Einstellungen: 3D-OpenGL-Einstellungen geöffnet werden oder durch Ausführen des folgenden LabTalk-Skripts im Befehls- oder Skriptfenster:

system.opengl();

OpenGL-Info

Dieser Zweig zeigt nur lesbare Informationen zu OpenGL auf der lokalen Festplatte.

Der Hersteller ist der Lieferant der verwendeten Grafikkarte.

Der Hersteller ist der Name der verwendeten Grafikkarte.

Die Version ist die verwendete OpenGL-Version.

Weitere Einzelheiten zu OpenGL können Sie erhalten, in dem Sie ein 3D-OpenGL-Diagramm erzeugen und das folgende Skript im Befehls- oder Skriptfenster ausführen, um die Ausgabe zu erstellen:

type -glv;

Einstellungen

Dieser Zweig enthält Optionen zum Festlegen der OpenGL-Eigenschaften.

Transparenz aktivieren Legen Sie fest, ob Transparenz aktiviert wird. Dies kann auch mit der Systemvariablen @OGLT festgelegt werden.
  • Aktivieren
  • Deaktivieren

Hinweis: Die Funktion der Transparenz wird für OpenGL-Versionen, die älter als 2.1 sind, deaktiviert.
Anti-Aliasing aktivieren Legen Sie fest, ob Anti-Aliasing aktiviert werden soll. Unten wird eine Meldung angezeigt, wenn eine Option aktiviert ist. Dies kann auch über die Systemvariable @OGLFA gesteuert werden.
  • Kein Anti-Aliasing
  • Anti-Aliasing bei mehreren Stichproben
Diese Option bietet Anti-Aliasing mit der höchsten Qualität, verringert aber die Geschwindigkeit entsprechend.
  • Schnelles approximatives Anti-Aliasing
Diese Option bietet eine höhere Geschwindigkeit als das Anti-Aliasing von mehreren Stichproben, die Qualität leidet jedoch.

Hinweis: Die Funktion des Anti-Aliasing funktioniert möglicherweise nicht in allen OpenGL-Versionen, die älter sind als 2.1.0 (einschließlich 2.1.0).
Anti-Aliasing für Export/Druck aktivieren Legen Sie fest, ob Anti-Aliasing aktiviert werden soll, wenn Diagramm exportiert oder gedruckt werden. Dies kann auch über die Systemvariable @OGLFAE gesteuert werden.

Die Optionen sind die gleichen wie in der Auswahlliste Anti-Aliasing aktivieren.
Anzeigeliste aktivieren Legen Sie fest, ob die Anzeigeliste verwendet werden soll. Dies kann auch mit der Systemvariablen @OGLDL festgelegt werden.

Diese Option ist nur nützlich, wenn die OpenGL-Version höher als Intel 3.3 ist. Sie sollten dies verwenden, wenn das Diagramm durch Aktivierung der Transparenz nicht mehr angezeigt wird.

Hinweis: @OGLDL=0 bedeutet, dass die Anzeigeliste deaktiviert wurde, und @OGLDL=1 bedeutet das Gegenteil.
