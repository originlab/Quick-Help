The Decomposition Dialog Box

DWT-Dialog

  • Menu Command: Analysis: Signal Processing: Wavelet: Decompose
  • Window Types: Workbook, Graph
  • X-Function Scripting and GUI Examples: dwt(Pro)
Dialog Theme See Dialog Themes.
Recalculate

See Recalculating Analysis Results.
Input

Specifies the input signal.
Wavelet Type & Order

Specifies the type and order of the wavelet.

  • Haar: Haar wavelet
  • DBN: Daubechies wavelet (N = wavelet order and no. of vanishing moments)
  • BiorNr.Nd: Biorthogonal (Nr = wavelet order or scaling functions used for reconstruction; Nd = order of the functions used for decomposition)
Extension Mode

Used to extend the input signal.

  • Periodic: The input signal will be treated as periodic.
  • Zero-padded: The data points outside the input range will be treated as zeroes.
Approx Coefficient

Specifies the output for the approximation coefficients.
Detail Coefficient

Specifies the output for the detail coefficients.
