Brief Description[edit]

Gamma cumulative distribution function

Sample Curve[edit]

Parameters[edit]

Number: 4

Names: y0, A1, a, b

Meanings: y0 = offset, A1 = Amplitude, a = Shape, b = Scale

Lower Bounds: A1 > 0.0, a > 0.0, b> 0.0

Upper Bounds: none

Derived Parameters[edit]

Mean: mu=a*b

Standard Deviation: sigma=sqrt(a)*b
