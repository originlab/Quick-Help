Registerkarte Skript (Arbeitsblatteigenschaften)

WksProperties-Dialog-Script-Tab

Hinweis: Worksheet Skript wird für die Rückwärtskompatibilität mit älteren Versionen von Origin bereitgestellt, die keine Neuberechnung von Werte setzen unterstützen und/oder keine Importfilter haben, in denen Skripts zur Ausführung nach dem Import eingerichtet werden können (z. B. Skripte im ASCII-Import). Angesichts der Tatsache, dass es vielen neuen Stapelfunktionen wie Analysevorlagen und Neuberechnung vorausgeht, sollte es nur für die Nachbearbeitung von importierten Daten verwendet werden, wenn zum Beispiel ein Datenkonnektor für den Datenimport verwendet wird.

 

Um die Registerkarte Skript des Dialogs Arbeitsblatteigenschaften zu öffnen:

  1. Wählen Sie Worksheet: Worksheet-Skript, wenn das gewünschte Arbeitsblatt aktiv ist.

oder

  1. Wählen Sie Format: Worksheet im Menü oder drücken Sie F4. Gehen Sie dann zur Registerkarte Skript.


Das in diesem Skriptfeld eingegebene LabTalk-Skript ist mit dem aktiven Arbeitsblatt verbunden. Die Methode der Skriptausführung wird innerhalb der Registerkarte bestimmt. Speichern Sie das Arbeitsblatt als Vorlage zusammen mit dem entsprechenden Arbeitsblatt-Skript.

Inhalt

  • 1 Skript ausführen
  • 2 Unterstützung von einzeiligen Befehlen ohne Abschluss mit ";"
  • 3 Das Skriptfeld

Skript ausführen

Nach Import

Aktivieren Sie das Kontrollkästchen, um das Arbeitsblattskript jedes Mal auszuführen, wenn Sie Daten in das Arbeitsblatt importieren.
Auf Änderungen in Bereich(en) (durch Leerzeichen getrennt)

Aktivieren Sie dieses Kontrollkästchen, um das Skript im Skriptfeld auszuführen, wenn Änderungen an dem Datensatz, der im entsprechenden Textfeld festgelegt wurde, vorgenommen werden. Die folgenden Notationen für den Datensatzbereich werden in dem zugehörigen Textfeld unterstützt. Beachten Sie, dass sowohl Leerzeichen als auch Kommas hier als Trennzeichen unterstützt werden.

A,C,E    // Column A, Column C and Column E A C E        // Column A, Column C and Column E A:E        // Column A to Column E A:B, D:E   // Column A to Column B and Column D to Column E A:B D:E   // Column A to Column B and Column D to Column E 1:end      // All Columns 1:2, 4:end // All Columns except the third column 1:2 4:end // All Columns except the third column

Unterstützung von einzeiligen Befehlen ohne Abschluss mit ";"

Aktivieren Sie dieses Kontrollkästchen, um ein einzelnes Anweisungsskript im Skriptfeld auszuführen, ohne es mit einem Semikolon abzuschließen.

Das Skriptfeld

Geben Sie das gewünschte LabTalk-Skript in dieses Textfeld ein. Das Skript sollte der gleichen Syntax wie eine LabTalk-Skriptdatei folgen. Weitere Informationen finden Sie im Abschnitt LabTalk der Programmierhilfe (Hilfe: Programmierung).

Klicken Sie auf die Schaltfläche Ausführen , um das Skript sofort auszuführen. Klicken Sie außerdem auf , oder , um das Skript und die Ausführungsmethode mit dem Arbeitsblatt zu speichern und sie als Standardeinstellungen festzulegen.
