Lookup-Tabelle der Funktionen (nur OriginPro)


Die Funktion imgFuncLUT berechnet eine Lookup-Tabellenanordnung mit Hilfe der vordefinierten Funktion. Es wird eine LookUp-Tabelle zum Verändern der Intensitätswerte eines Bitmaps verwendet. Um diese Funktion zu verwenden:

Wählen Sie Bild: Anpassungen: Funktion LUT im Origin-Menü.

Weitere Informationen über diese Funktion finden Sie in der Dokumentation der X-Funktion imgFuncLUT in der Origin-Hilfe. Diese kann über das Befehlsfenster mit folgendem Befehl aufgerufen werden:

help imgFuncLUT
