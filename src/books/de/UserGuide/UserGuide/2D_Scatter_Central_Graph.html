2D-Punktdiagramm mit zentriertem Ursprung

2DScatterCentral-Graph

Datenanforderungen

Wählen Sie eine oder mehrere Y-Spalten oder einen Datenbereich aus mindestens einer Y-Spalte aus. Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen: Einfache 2D: Punktdiagramm zentriert.

oder

Klicken Sie auf die Schaltfläche Punktdiagramm in der Symbolleiste 2D-Grafiken.

Vorlage

SCATTERCENTRAL.OTP (im Origin-Programmordner installiert).
