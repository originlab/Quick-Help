Windrose - Eingeteilte Daten

WindRose-Binned-Graph

Datenanforderungen

Markieren Sie eine oder mehrere Y-Spalten (oder einen Bereich aus einer oder mehreren Y-Spalten). Die X-Spalte zeigt die Windrichtung an, und die Y-Spalte(n) enthalten die Werte der Windgeschwindigkeit. Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen: Spezialisiert: Windrose - Eingeteilte Daten.

oder

Klicken Sie auf die Schaltfläche Windrose - Eingeteilte Daten auf der Symbolleiste 2D Grafiken.

Vorlage

WindRose.OTP (im Origin-Programmordner installiert)
