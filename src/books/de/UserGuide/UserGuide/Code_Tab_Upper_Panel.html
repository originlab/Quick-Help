Parameter Startwert

Wählen Sie Auto. Parameterinitialisierung aktivieren, um die Parameterinitialisierungsroutinen zu aktivieren.

Sie können die Initialisierungsroutinen in dem Textfeld rechts sehen. Die Initialisierungsroutine verwendet die ausgewählten abhängigen und unabhängigen Datensätze zum Bestimmen angemessener Initialisierungsparameterwerte für die ausgewählte Anpassungsfunktion. Die Routine wird automatisch ausgeführt, nachdem Sie eine Anpassungsfunktion für die Eingabe ausgewählt haben.

Origin bietet Parameterinitialisierungsroutinen für alle Standardfunktionen. Für Standardfunktionen können Sie benutzerdefinierte Routinen erstellen oder einfach einen passenden Parameterinitialisierungswert festlegen.

Siehe: Parameterinitialisierung.

Nebenbedingungen

Wählen Sie Lineare Bedingung aktivieren, um die Bedingungen zu aktivieren. Das Textfeld rechts legt allgemeine lineare Nebenbedingungen für die Anpassungsparameter fest. Mehrere Nebenbedingungen sollten durch Semikolons getrennt werden.

Beispielsweise definiert die folgende Zeichenkette mehrere Nebenbedingungen:

a>b; a+2*b>=c*2-d; a<b<c; a/3<9;

Beachten Sie, dass nichtlineare Nebenbedngungen wie "a^2>c; a*b>3; 1/b>c+2; sin(a)<c;" nicht zugelassen sind.

Siehe: Schranken und Nebenbedingungen.

Abgeleitete Parameter

Abgeleiteter Parameter ist ein Prozess, der nach dem Fit ausgeführt wird. Er verwendet LabTalk zur Berechnung von Werten aus angepassten Parametern. Diese zusätzlichen Parameter werden am Ende der Anpassungsitzung mit Hilfe der sich ergebenden Anpassungsparameterwerte berechnet.
