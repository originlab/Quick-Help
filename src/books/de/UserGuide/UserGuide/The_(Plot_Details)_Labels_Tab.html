Schriftart

Legen Sie die Schriftart für die Beschriftungen für die Stücke des Kreises fest, einschließlich Schriftfarbe und Schriftgröße.

Bitte beachten Sie, dass, wenn die Schriftfarbe auf Auto gesetzt wurde, die Schriftfarbe nach der folgenden intelligenten Methode verfährt:

  • Wenn die Beschriftungen sich innerhalb der Segmente von Kreis-/Ringdiagramms befinden, wird die Schriftfarbe nach der Tiefe der aktuellen Segmentfarbe entschieden, gesetzt auf Weiß oder Schwarz.
  • Wenn Sie die Beschriftungen außerhalb der Segmente positioniert haben, entspricht die Schriftfarbe der Farbe ihrer Segmente.

 

Die Regel von Auto ist in der Minisymbolleiste und im Dialog Eigenschaften der Textobjekte unterschiedlich. Wenn die Schriftfarbe oben an zwei Stellen auf Auto gesetzt ist, folgt die Schriftfarbe der Einstellung des Dialogs Details Zeichnung.

Bedienelemente Format

Beschriftungsinhalt

Diese Option legt fest, was als Beschriftung gezeigt wird.

Werte

Die Arbeitsblattzellenwerte werden als Beschriftungen angezeigt.
Prozentanteile

Der Prozentanteil gesamt wird für jeden Arbeitsblattzellenwert angezeigt.
Wenn Kombinierte zeigen als auf der Registerkarte Segmente aktiviert wurde, wird die zweite Zeichnung (Kreis, Säulen oder Ring) hinzugefügt, um alle kombinierten Segmente zu zeigen. Eine Auswahlliste wird für dieses Bedienelement gezeigt:

  • Wählen Sie Gesamt, um die tatsächlichen Prozent der kombinierten Segmente auf der ersten Zeichnung zu zeigen.
  • Wählen Sie Einzeln, um die jeweiligen Prozentanteile auf allen kombinierten Segmenten zu zeigen. Alle Prozentanteile in der zweiten Zeichnung sind bis 100.
Kategorien

Anzeigen der zugehörigen X-Werte für jedes Segment Wenn es keine zugehörige X-Spalte gibt, werden Zeilennummern angezeigt.
Benutzerdefiniert Passen Sie die Anzeige der Beschriftungen benutzerdefiniert an, indem Sie eine Notation eingeben oder eine Notation aus der Ausklappliste auswählen. Wenn Sie dieses Kontrollkästchen aktivieren, werden alle Kästchen oben deaktiviert. Die Formatbedienelemente unten werden ebenfalls deaktiviert.

Es gibt einige spezifische Notationen für ein Kreisdiagramm.

*c: Categories *v: Wedge Value *p: Wedge Percentage *rv: Radius Value //not show if Increment Radius is not enabled *rp: Radius Percentage //not show if Increment Radius is not enabled *tv: Thickness Value //not show if Increment Thickness is not enabled *tp: Thickness Percentage //not show if Increment Thickness is not enabled

Für die Notationen, die mit Radius und Dicke verbunden sind, sollten Sie zuerst einen Datensatz für den Inkrementradius und die Inkrementdicke auf der Registerkarte Geometrie festlegen.

Beschriftungsformat festlegen

Verwenden Sie die Auswahlliste Format, um das numerische Format festzulegen: †

  • Dezimal: 1000: 1, 1000, 1E6, 1E9
  • Wissenschaftlich: 10^3: 10^0, 10^3, 10^6, 10^9
  • Technisch: 1k: 1.0, 1.0k, 1.0M, 1.0G
  • Dezimal: 1.000: 1, 1.000, 1E6, 1E9
  • Wissenschaftlich: 1E3: 1E0, 1E3, 1E6, 1E9

Dezimalstellen

Verwenden Sie die Option in Kombination mit dem Listenfeld, um Dezimalstellen festzulegen.

Signifikante Stellen

Verwenden Sie die Option in Kombination mit dem Listenfeld, um signifikante Stellen festzulegen.

† Der Schwellenwert zur Konvertierung in die wissenschaftliche Schreibweise wird über die Registerkarte Zahlenformat des Dialogs Optionen festgelegt (Einstellungen: Optionen).

Bedienelemente Position

Wenn das Kontrollkästchen Mit Segment verbinden aktiviert ist, kann die Position der Kreisstückbeschriftungen in zwei Richtungen verändert werden: Radiale Richtung und Winkelrichtung.

In radialer Richtung können Sie die Beschriftungen außerhalb des Radiusendes positionieren, innerhalb des Radiusendes, in der Mitte des Radius oder an der Kreiskante.

Dann können Sie die radiale Verschiebung verwenden, um den Versatz der Beschriftungen in radialer Richtung festzulegen. Die Einheit ist der Prozentanteil des Radius.

In Winkelrichtung können Sie die Beschriftungen am Anfang, in der Mitte oder am Ende des Bogens der Einzelstücke positionieren.

Beschriftung verbergen, wenn Prozent weniger als (%)

Aktivieren Sie dieses Bedienelement, um ein prozentualen Schwellenwert festzulegen. Die Beschriftungen für die Segmente, deren Prozentanteil kleiner ist als dieser Schwellenwert, werden verborgen.

Bedienelemente Verbindungslinie

Unabhängige Verbindungslinie Wählen Sie diese Option, um den Stil von einer oder mehreren Verbindungslinien unabhängig festzulegen.
Verbindungslinie zeigen, wenn Versatz (%) überschreitet Die Verbindungslinien werden gezeigt, wenn der Abstand zwischen Datenbeschriftungen und Datenpunkten eine kritische Länge überschreitet, die definiert wird mit...

Kritische Länge = Festgelegter Prozentwert % * (Layerhöhe + Layerbreite)/2.

Beachten Sie, dass Sie, wenn Sie beabsichtigen, Beschriftungen auf Ihrem Diagramm zu verschieben, das Kontrollkästchen Mit Kreissegmenten verbinden deaktivieren müssen.
Verbindungslinie in Segmente verlängern (%) Legen Sie fest, ob die Verbindungslinie in die Segmente verlängert wird und wie lang diese Erweiterung sein soll. Die Einheit ist der Prozentanteil des Radius.
Verbinden Wählen Sie Gerade Linie, 2-Segment-Polylinie oder 3-Segment-Polylinie.
Stil Wählen Sie durchgezogene, gestrichelte, gepunktete Linie etc.
Breite Die Linienbreite wird in Punkten gemessen, wobei 1 Punkt=1/72 Zoll entsprechen.
Farbe Wählen Sie Auto (Standard, folgt der Beschriftungsfarbe) oder legen Sie eine Farbe fest.
