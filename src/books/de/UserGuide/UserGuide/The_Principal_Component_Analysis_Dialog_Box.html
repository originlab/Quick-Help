Neu berechnen

Legen Sie fest, wie das Ergebnis neu berechnet und aktualisiert wird, wenn sich die Eingangsdaten oder Einstellungen ändern.

Keine Die Ausgabe wird nicht mit den Quelldaten verbunden, und Änderungen an den Quelldaten führen zu keiner Aktualisierung des Ergebnisses. Außerdem können die Einstellungen nicht geändert werden, um das Ergebnis neu zu berechnen.
Auto Die Ergebnisse werden automatisch aktualisiert, wenn sich die Quelldaten ändern. Die Einstellungen können geändert werden, um das Ergebnis neu zu berechnen.
Manuell Die Ausgabe wird nicht automatisch aktualisiert, wenn sich die Quelldaten ändern. Sie müssen die Aktualisierung manuell durchführen, indem Sie auf die Schaltfläche Neu berechnen auf der Symbolleiste Standard klicken. Die Einstellungen können geändert werden, um das Ergebnis neu zu berechnen.

Eingabe

Wählen Sie Daten für die Hauptkomponentenanalyse.

Unabhängige Variablen Wählen Sie Daten aus, um die unabhängigen Variablen festzulegen. Die Daten in jeder Spalte entsprechen einer Variablen und jede Zeile einer Beobachtung.
Beobachtungsbeschriftungen Wählen Sie die Daten aus, um die Beschriftung für die Beobachtung festzulegen. Die Beschriftungen werden im Ergebnis in den Score-Diagrammen und Biplots dargestellt Beachten Sie, dass die Beschriftungsspalte als Kategorial gesetzt wird, wenn es sich um eine Textspalte handelt.

Einstellungen

Legen Sie die Einstellungen für die Hauptkomponentenanalyse fest.

Analysieren Wählen Sie den Matrixtyp, der in der Hauptkomponentenanalyse analysiert werden soll.
  • Korrelation
    Verwenden Sie eine Korrelationsmatrix zum Durchführen der Hauptkomponentenanalyse.
  • Kovarianz
    Verwenden Sie eine Kovarianzmatrix zum Durchführen der Hauptkomponentenanalyse.
Anzahl der zu extrahierenden Komponenten Legen Sie die Anzahl der zu extrahierenden Hauptkomponenten fest. Diese sollten zwischen 1 und der Anzahl der Variablen liegen.
Scores standardisieren Bestimmen Sie, ob Scores standardisiert werden sollen. Für den listenweisen Ausschluss von fehlenden Werten werden Scores als Varianz der Einheiten standardisiert. Für den paarweisen Ausschluss werden Scores mit der Quadratwurzel des Eigenwerts skaliert. Die Scoredaten in Score-Diagramm und Biplot werden standardisiert, wenn Scores standardisieren aktiviert ist.
Fehlende Werte ausschließen Legen Sie die Art und Weise fest, wie fehlende Werte für die Analyse ausgeschlossen werden.
  • Listenweise
    Eine Beobachtung, die einen oder mehrere fehlende Werte enthält, wird aus der Analyse ausgeschlossen.
  • Paarweise
    Eine Beobachtung wird aus der Berechnung der Kovarianz oder Korrelation zwischen zwei Variablen nur ausgeschlossen, wenn fehlende Werte in einer der zwei Variablen für die Beobachtung existieren.

Deskriptive Statistik

Legen Sie fest, ob die grundlegenden statistischen Kennwerte für die Eingabedaten durchgeführt werden sollen.

Einfache deskriptive Statistik Legen Sie fest, ob einfache Statistiken für die Eingabedaten durchgeführt werden sollen, u.a. Mittelwert, Standardabweichung und Anzahl der Beobachtungen in der Analyse für jede Variable.
Korrelationsmatrix Legen Sie fest, ob die Korrelationsmatrix berechnet werden soll. Beachten Sie, dass die Methode zum Berechnen der Korrelationsmatrix von der Methode abhängt, die zum Ausschließen der fehlenden Werte ausgewählt haben.

Zu berechnende Eigenschaften

Legen Sie die Ergebnisse fest, die in der Hauptkomponentenanalyse berechnet werden sollen.

Eigenwerte Legen Sie fest, ob Eigenwerte ausgegeben werden sollen, einschließlich Anteil und kumulativer Anteil. Das Ergebnis des Bartletts Tests wird gezeigt, wenn die Analysematrix eine Kovarianzmatrix ist.
Eigenvektoren Legen Sie fest, ob Eigenwerte für die Hauptkomponenten, die unter Anzahl der zu extrahierenden Komponenten festgelegt wurden, ausgegeben werden sollen.
Scores Legen Sie fest, ob Scores für die Hauptkomponenten, die unter Anzahl der zu extrahierenden Komponenten festgelegt wurden, ausgegeben werden sollen. Beachten Sie, dass dieses Kontrollkästchen aktiviert und ausgeblendet angezeigt wird, wenn Score-Diagramm oder Biplot in der Gruppe Zeichungen aktiviert ist.

Diagramme

Legen Sie fest, ob in der Hauptkomponentenanalyse Zeichnungen gezeigt werden.

Scree-Diagramm Legen Sie fest, ob der Screeplot für Eigenwerte gezeigt werden soll.
Komponentendiagramm Legen Sie fest, ob 2D- oder 3D-Diagramme für Hauptkomponenten gezeigt werden sollen.
  • Komponentendiagrammtyp
    Legen Sie das Zeichnen eines 2D- oder 3D-Komponentendiagramms fest.
  • Zu zeichnende Hauptkomponenten auswählen
    • Hauptkomponente für XYZ-Achse
      Legen Sie die Hauptkomponente für die XYZ-Achse in Komponentendiagrammen fest.
  • Ladungsdiagramm
    Legen Sie fest, ob das Ladungsdiagramm für Eigenvektoren gezeigt werden soll.
  • Score-Diagramm
    Legen Sie fest, ob das Score-Diagramm für Scores gezeigt werden soll.
  • Biplot
    Legen Sie fest, ob der Biplot für Eigenvektoren und Scores gezeigt werden soll.

Ausgabe

Legen Sie das Ziel der Ausgabeergebnisse für die Hauptkomponentenanalyse fest.

PCA-Bericht Legen Sie das Blatt für den Bericht der Hauptkomponentenanalyse fest. Der Standardwert ist ein neues Blatt in der Arbeitsmappe der Eingabedaten.
Score-Daten Legen Sie das Blatt für die Scores fest. Der Standardwert ist ein neues Blatt in der Arbeitsmappe der Eingabedaten. Beachten Sie, dass diese Option deaktiviert wird, wenn Scores in der Gruppe Zu berechnende Mengen deaktiviert ist.
