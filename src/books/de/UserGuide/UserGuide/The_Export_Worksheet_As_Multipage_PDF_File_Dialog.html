Blatt/Blätter auswählen

Legen Sie die zu exportierende(n) Seite(n) fest. Die Standardoption ist Aktuelles Blatt.

Aktuelles Blatt Das aktuelle Arbeitsblatt wird exportiert.
Blätter in Mappe Alle Arbeitsblätter in der aktiven Arbeitsmappe werden exportiert.
Blätter im Ordner Alle Arbeitsblätter in dem aktuellen Ordner werden exportiert, nicht einschließlich derjenigen in den Unterordnern.
Blätter im Ordner (Rekursiv) Alle Arbeitsblätter in dem aktuellen Ordner werden exportiert, einschließlich derjenigen in den Unterordnern.
Blätter im Projekt Alle Arbeitsblätter in dem Projekt werden exportiert.
Festgelegt Das festgelegte Arbeitsblatt wird in das Textfeld Arbeitsblätter exportiert.

Arbeitsblätter

Das Textfeld listet die zu exportierenden Arbeitsblätter auf. Klicken Sie auf die Pfeilschaltfläche unten rechts vom Textfeld, um das Textfeld zu erweitern/zu minimieren. Das Textfeld kann nur bearbeitet werden, wenn die Option Festgelegt unter Blatt/Blätter auswählen ausgewählt ist. Sie können den Blattnamen entsprechend der Form [Mappenname]Blattname in das Textfeld eingeben oder auf die Schaltfläche klicken, um den Dialog Blattbrowser zu öffnen, mit dem das Blatt für den Export ausgewählt werden kann.

Dateiname(n)

Legen Sie den/die Name(n) für die exportierten PDF-Datei(en) fest. Sie können den gewünschten Namen entweder unter den Optionen wählen oder manuell eingeben.

<Langname> Der Langname des Arbeitsblatts wird als Dateiname verwendet.
<Kurzname> Der Kurzname des Arbeitsblatts wird als Dateiname verwendet.
<Blattname> Der Blattname des Arbeitsblatts wird als Dateiname verwendet.
wks## "wks + index" wird als Dateiname verwendet.
wks<Raute># "wks + index" und das Zeichen "#" (dargestellt durch die <Raute>) wird als Dateiname verwendet.
Hinweis:
  1. Der Dateiname unterstützt Stichwortkombinationen wie <Langname>-multi. Der Langname des Arbeitsblatts und die Zeichenkette -multi werden als Exportdateiname verwendet.
  2. Die <Raute>, die das Zeichen "#" darstellt, kann auch verwendet werden, ohne mit dem Arbeitsblattindex kombiniert zu werden. Setzen Sie Dateiname(n) beispielsweise auf Wks<Raute>2. Dann wird der Dateiname für die exportierte PDF-Datei als Wks#2 angezeigt.

Pfad

Legen Sie den Dateipfad fest, um die exportierten PDF-Dateien zu speichern. Abgesehen von der Optionenauswahl können Sie den Pfad auch manuell in das Textfeld eingeben oder auf die Schaltfläche klicken, um ihn im Dialog Nach Ordnern suchen festzulegen.

<Anwenderdateiordner> Die exportierte(n) PDF-Datei(en) werden im Anwenderdateiordner gespeichert.
<Projektordner> Die exportierte(n) PDF-Datei(en) werden in dem Ordner gespeichert, in dem sich die aktuelle Projektdatei befindet. Sie können den gesamten Pfad im Origin-Header sehen.

Bestehende Dateien überschreiben

Legen Sie fest, wie mit den exportierte(n) Datei(en) umgegangen werden soll, wenn eine PDF-Datei mit demselben Namen im gleichen Ordner existiert.

Automatisches Umbenennen Die exportierte PDF-Datei wird automatisch umbenannt und einer Nummerierung versehen.
Ersetzen Die existierende PDF-Datei wird ersetzt.
Überspringen Der Export der PDF-Datei wird übersprungen und die vorhandene Datei wird behalten.
Fragen Ein Nachrichtenfeld wird angezeigt, das nachfragt, ob die vorhandene PDF-Datei überschrieben werden soll.

Bildeinstellungen

Legen Sie die Eigenschaften für die exportierte PDF-Datei fest.

PDF-Optionen

Legen Sie die Exporteinstellungen der PDF-Datei fest.

Basisobjekt

Farbübersetzung
  • Normal
    Modus Normal
  • Inverse
    Inverse Farben
  • Nur inverse Graustufen
    Nur inverse Graustufen
  • Graustufe
    In Graustufendiagramm konvertieren
  • Inverse Graustufen
    In Graustufen und inverse Farben konvertieren

Linienmodus

Der Linienmodus wird bestimmt.

  • Gerät
    Der Gerätemodus wird verwendet.
  • Anschlag
    Der Modus Anschlag wird verwendet.

Füllmodus

Der Füllmodus wird festgelegt, der beim Export des Diagramms verwendet wird.

  • Gerät
    Der Gerätemodus wird verwendet.
  • Anschlag
    Der Modus Anschlag wird verwendet.

Hintergrund Rechteck

Es wird bestimmt, ob Rechtecke im Hintergrund verwendet werden.

Handling

Es wird bestimmt, ob oder ob die Rasterformatkonvertierung nicht durchgeführt wird, und falls ja, welche Methode verwendet wird.

  • Ignorieren
    Das Rasterformat wird nicht konvertiert.
  • Kopie der Quelle
    Die Kopie der Quelle wird verwendet.
  • Individuelle Bitmaps
    Individuelle Bitmaps der Anwender
  • Alle Bitmaps kombinieren
    Alle Bitmaps werden kombiniert.

Auflösung

Die Auflösung wird bestimmt, die für die Konvertierung verwendet wird, z.B. 72, 96, 100, 150, 300, 600 oder 1200.

Datenkomprimierung

Farbbilder

Der Farbkomprimierungsmodus wird bestimmt.

  • Keine
    Keine Komprimierung
  • JPEG Hoch
    Die JPEG hohe Komprimierung wird verwendet.
  • JPEG Hoch-Mittel
    Die JPEG hoch-mittlere Komprimierung wird verwendet.
  • JPEG Mittel
    Die JPEG mittlere Komprimierung wird verwendet.
  • JPEG Mittel-Niedrig
    JPEG mittlere-niedrige Komprimierung wird verwendet.
  • JPEG Niedrig
    JPEG niedrige Komprimierung wird verwendet.
  • Flach
    Die flache Komprimierung wird verwendet.

Einfarbige Grafiken

Der Modus für die Schwarzweißkomprimierung wird festgelegt.

  • Keine
    Keine Komprimierung
  • CCITT Gruppe 3
    Kompatibel mit CCITT Gruppe 3
  • CCITT Gruppe 4
    Kompatibel mit CCITT Gruppe 4
  • Laufzeit-Länge verschlüsselt
    Die Kodierungsmethode der Laufzeitlänge wird verwendet.
  • Flach
    Flache Komprimierung

Grafik und Text

Der Textkomprimierungsmodus wird festgelegt.

  • Keine
    Keine Komprimierung
  • Flach
    Flache Komprimierung

ASCII-Kodierung

Es wird bestimmt, ob die ASCII-Kodierung verwendet werden soll.

Zeichensätze
Einbettungsmodus der Zeichensätze

Wählen Sie die Option, um den Einbettungsmodus der Zeichensätze festzulegen.

  • Nur Deskriptor
    Es ist nur der Deskriptor enthalten.
  • Eingebettet
    Die Zeichensätze werden im Diagramm eingebettet.
  • Umrandung
    Es werden Zeichensätze mit Umrandung verwendet.

Modus Umrandung

Der Modus Umrandung, z.B. Kompakt oder Adobe Type 3, wird festgelegt.

Anschlag Breite

Die Breite des Anschlags für die Zeichensätze, die Dünn, Normal, Halbfett oder Fett sein können, wird festgelegt.

Ersatzschrift

Es wird bestimmt, ob die Schrift ersetzt werden soll.

True Typ verwenden

Es wird bestimmt, ob TrueType-Zeichensätze verwendet werden sollen.

Mehrere Seiten

Legen Sie fest, ob ein Deckblatt und Seitenzahlen zu der exportierten PDF-Datei hinzugefügt werden soll.

Deckblatt

Aktivieren Sie das Kontrollkästchen neben der Option Deckblatt, um die Einstellungen für das Deckblatt einzublenden.

Text

Die Texte, die in dieses Textfeld eingegeben werden, werden auf dem Deckblatt angezeigt.

Schriftart
  • Name
    Legen Sie die Schriftart der Texte fest.
  • Größe
    Wählen Sie die gewünschte Punktgröße der Texte aus bzw. geben Sie sie ein. Die Standardgröße ist 18.
  • Fett
    Legen Sie fest, ob die Texte fett sein sollen oder nicht. Dieses Kontrollkästchen ist standardmäßig aktiviert.
  • Kursiv
    Legen Sie fest, ob die Texte kursiv sein sollen oder nicht. Dieses Kontrollkästchen ist standardmäßig deaktiviert.
  • Unterstrichen
    Legen Sie fest, ob Texte unterstrichen sein sollen oder nicht. Dieses Kontrollkästchen ist standardmäßig deaktiviert.

Seitennummerierung

Aktivieren Sie das Kontrollkästchen neben der Option, um die Einstellungen für die Seitennummerierung einzublenden.

Schriftart
  • Name
    Legen Sie die Schriftart der Texte fest.
  • Größe
    Wählen Sie die gewünschte Punktgröße der Texte aus bzw. geben Sie sie ein. Die Standardgröße ist 18.
  • Fett
    Legen Sie fest, ob die Texte fett sein sollen oder nicht. Dieses Kontrollkästchen ist standardmäßig aktiviert.
  • Kursiv
    Legen Sie fest, ob die Texte kursiv sein sollen oder nicht. Dieses Kontrollkästchen ist standardmäßig deaktiviert.
  • Unterstrichen
    Legen Sie fest, ob Texte unterstrichen sein sollen oder nicht. Dieses Kontrollkästchen ist standardmäßig deaktiviert.

Oben

Legen Sie fest, ob Seitennummern oben auf den Seiten der exportierten PDF-Datei hinzugefügt werden sollen. Dieses Kontrollkästchen ist standardmäßig deaktiviert.

Unten

Legen Sie fest, ob Seitennummern unten auf den Seiten der exportierten PDF-Datei hinzugefügt werden sollen. Dieses Kontrollkästchen ist standardmäßig deaktiviert.

Horizontale Position

Legen Sie die horizontale Position der Seitennummern fest. Wenn sowohl Oben als auch Unten deaktiviert sind, ist die hier festgelegte horizontale Position irrelevant, da es keine Seitennummern in der PDF-Datei gibt. Optionen:

  • Links auf gerade, rechts auf ungerade Seiten
    Die Seitennummern werden links auf geraden Seiten und rechts auf ungeraden Seiten gezeigt.
  • Links
    Die Seitennummern werden links auf allen Seiten gezeigt.
  • Rechts
    Die Seitennummern werden rechts auf allen Seiten gezeigt.
  • Zentrum
    Die Seitennummern werden zentriert auf allen Seiten gezeigt. Diese Option ist standardmäßig aktiviert.

Druckeinstellungen

Legen Sie die Druckeinstellungen der exportierten PDF-Datei fest.

Eigene Einstellungen des Worksheets verwenden

Legen Sie fest, ob die eigenen Einstellungen des Arbeitsblatts (horizontale Gitternetzlinien, Zeilenheader etc.) verwendet werden sollen, um die exportierte PDF-Datei zu drucken. Wenn dieses Kontrollkästchen nicht aktiviert ist, können Sie die Optionen im Zweig Druckoptionen für Worksheet festlegen.

Druckoptionen für Worksheet

Diese Optionen sind nur verfügbar, wenn das Kontrollkästchen Eigene Einstellungen des Worksheets verwenden deaktiviert ist. Dieser Zweig kann verwendet werden, um festzulegen, ob einige Arbeitsblattelemente gedruckt werden sollen.

  • Horizontale Gitternetze
    Legen Sie fest, ob die horizontalen Gitternetze der PDF-Datei gedruckt werden sollen.
  • Vertikale Gitternetze
    Legen Sie fest, ob die vertikalen Gitternetze der PDF-Datei gedruckt werden sollen.
  • Zeilenheader
    Legen Sie fest, ob der Zeilenheader der PDF-Datei gedruckt werden soll.
  • Spaltenheader
    Legen Sie fest, ob der Spaltenheader der PDF-Datei gedruckt werden soll.
  • Seitenkopfzeile
    Legen Sie fest, ob die Seitenkopfzeile der PDF-Datei gedruckt werden soll.
  • Seitenfußzeile
    Legen Sie fest, ob die Seitenfußzeile der PDF-Datei gedruckt werden soll.
  • Hintergrundfarbe
    Legen Sie fest, ob die Hintergrundfarbe der PDF-Datei gedruckt werden soll.
  • Rahmen
    Legen Sie fest, ob der Rahmen der PDF-Datei gedruckt werden soll.
  • Leere Spalten
    Legen Sie fest, ob die leeren Spalten der PDF-Datei gedruckt werden sollen.
  • Leere Zeilen
    Legen Sie fest, ob die leeren Zeilen der PDF-Datei gedruckt werden sollen.
