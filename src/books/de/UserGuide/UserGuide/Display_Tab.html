Zeichnungsreihenfolge

Zeichnungsreihenfolge

Verwenden Sie diese Auswahlliste, um die Zeichnungsreihenfolge der Layer festzulegen. Optionen:

  • Jeden Layer als eigene Einheit zeichnen
    Dies bringt beide Layer dazu, Daten anzuzeigen, auch wenn einer oder beide Layer eine Hintergrundfarbe besitzen.
  • Layer nacheinander zeichnen
    Angenommen, es sind m Layer im Diagramm enthalten. Dann gilt für n=1, 2, ..., m. Der Layer n+1 wird gezeichnet, wenn Layer n vollständig gezeichnet wurde, einschließlich dem Hintergrund für Layer n. Wenn Sie eine Hintergrundfarbe für Layer n+1 festlegen, wird dieser Hintergrund einige Elemente von Layer n verdecken. Wenn Sie für Hintergrund Kein ausgewählt haben, sind die Daten aus Layer n in Layer n+1 sichtbar.

Option

Es gibt hier drei Optionen (Farbe?Rahmenabmessungen?Skalierungselemente). Sie können eine dieser Optionen auf der Registerkarte Anzeige auswählen.

Farbe

Hintergrundfarbe

Rahmenfüllfarbe

Verwenden Sie diese Farbliste, um die Rahmenfüllfarbe für den ausgewählten Layer festzulegen.
Rahmenfarbe

Verwenden Sie diese Farbliste, um die Rahmenfarbe für den ausgewählten Layer festzulegen.
Anwenden

Wenn auf diese Schaltfläche geklickt wird, werden die Farbeinstellungen, die Sie geändert haben, auf Diagramme angewendet und im Vorschaufeld angezeigt.

Skalierungselemente

Skalierungsmodus

Bestimmt, wie Achsen, Achsenbeschriftungen, Legenden, Diagramme und andere Objekte, die mit dem Layerbereich oder dem Layer und den Skalierungen verbunden sind, von einer Größenänderung eines Layers in der neuen Diagrammseite beeinflusst werden.

  • Skalierung mit Layerrahmen
    Skalieren Sie die Layerelemente proportional zum Layer.
  • Fester Faktor
    Bestimmt die Größe der Layerelemente in Relation zu ihrer ursprünglichen Größe (Schriftgröße, Symbolgröße und Linienbreite usw.). Die Größe dieser Elemente wird durch Multiplizieren ihrer ursprünglichen Größe mit dem Skalierungsfaktor, der im Textfeld Fester Faktor eingegeben wurde, bestimmt.
Fester Faktor

Diese Variable ist nur verfügbar, wenn die Variable für den Skalierungsmodus auf Fester Faktor gesetzt ist. Sie bestimmt den festen Faktor der Skalierung.
Anwenden

Wenn auf diese Schaltfläche geklickt wird, werden die Skalierungseinstellungen, die Sie geändert haben, auf Diagramme angewendet und im Vorschaufeld angezeigt.
