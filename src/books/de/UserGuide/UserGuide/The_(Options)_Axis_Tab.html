Max. Anzahl der Teilstriche

Numerische Skalierungen

Das Auswahlfeld Numerische Skalierung legt die maximale Anzahl an großen Hilfsstrichen fest, die in einem Diagrammfenster angezeigt werden können, wenn Numerisch in der Auswahlliste Typ auf der Registerkarte Hilfsstrichsbeschriftung des Dialogs Achse ausgewählt ist.

Wenn die Option # Große Hilfsstriche auf der Registerkarte Skalierung des Dialogs Achse ausgewählt ist und der Wert im verbundenen Textfeld den Wert des Kombinationsfeldes Numerische Skalierung überschreitet, dann zeigt Origin ein Maximum von sechs großen Hilfsstrichen im Diagrammfenster an.
Zeitskalierung Text/Datum

Das Kombinationsfeld Zeitskalierung Text/Datum legt die maximale Anzahl an großen Hilfsstrichen fest, die im Diagrammfenster angezeigt werden, wenn Text aus Datensatz, Zeit, Datum, Monat oder Wochentag in der Auswahlliste Typ auf der Registerkarte Hilfsstrichsbeschriftung des Dialogs Achse ausgewählt ist. Der Standardwert ist 400.

Wenn die Anzahl von großen Hilfsstrichen den Wert des Kombinationsfelds Zeitskalierung Text/Datum überschreitet, zeigt Origin ein Maximum von sechs großen Hilfsstrichen im Diagrammfenster an. (Wird beeinträchtigt, wenn der Typ der Großen Hilfsstriche auf der Registerkarte Skalierung des Dialogs Achsen auf die Option Nach Inkrement oder Nach Anzahl gesetzt ist.)

Hinweis: Wenn der Typ der Großen Hilfsstriche auf Nach Datensatz gesetzt ist, wird die maximale Anzahl der großen Hilfsstriche von den Befehlen @ATDM und @ATDS bestimmt und nicht von diesem Bedienelement.

Abstand der Gruppe Hilfsstrichsbeschriftung (%)

Innere Hilfsstriche

Geben Sie den Standardabstand eines "inneren Hilfsstrichs" von der Hilfsstrichsbeschriftung in Prozent der Hilfsstrichslänge an oder wählen Sie diesen aus.
Äußere Hilfsstriche

Geben Sie den Standardabstand eines "äußeren Hilfsstrichs" von der Hilfsstrichsbeschriftung in Prozent der Hilfsstrichslänge an oder wählen Sie diesen aus.
Achsentitel

Geben Sie den Standardabstand des Achsentitels von der Hilfsstrichsbeschriftung in Prozent der Hilfsstrichslänge an oder wählen Sie diesen aus.

Anzeige der Hilfsstrichsbeschriftungen

Null ist '0'

Die Beschriftung des Nullpunkts wird immer auf der Achsenskalierung als '0' angezeigt, womit andere Formatoptionen (z. B. Setze Dezimalstellen = 2) überschrieben werden.

Bitte beachten Sie, dass eine Systemvariable @TLC zum Steuern dieser Option verwendet werden kann.

Keine anhängende Null

Die anhängenden Nullen werden immer von den Beschriftungen der Achsenhilfsstriche entfernt, womit andere Formatoptionen (z. B. Setze Dezimalstellen = 2) überschrieben werden.

Bitte beachten Sie, dass eine Systemvariable @TLC zum Steuern dieser Option verwendet werden kann.

1 und 10 als 10^n im Wissenschaftlichen Format: 10^3 anzeigen

Aktivieren Sie dieses Kontrollkästchen, um 1 und 10 als 10^n anzuzeigen, wenn die Anzeige der Hilfsstrichsbeschriftungen auf Wissenschaftlich: 10^3 festgelegt wurde.

Wissenschaftlich:1x10^3 Anzeige

Verwenden Sie diese Auswahlliste, um festzulegen, wie das Wissenschaftliche Format (10^3) gezeigt wird, wenn das Anzeigeelement der Hilfsstrichsbeschriftungen folgendermaßen aussieht:1×10^3,1 × 10^3, 1x10^3, 1·10^3.

Hinweis: Vor Origin 2021 können Sie nur das Kontrollkästchen 1×10^3 als 1·10^3 im Wissenschaftlichen Format:10^3 anzeigen verwenden, um das Wissenschaftliche Format (10^3) mit 1×10^3 oder 1·10^3 zu zeigen, wie bei:


Seit Origin 2021 unterstützt Origin, dass das Wissenschaftliche Format (10^3) auch mit 1×10^3 oder 1 × 10^3 angezeigt wird. Beide Optionen enthalten das reale Multiplikationszeichen × (Unicode x(00d7)), nicht den Buchstaben x.

Wissenschaftlich: 1E3 Anzeige als

Verwenden Sie diese Auswahlliste, um festzulegen, wie das wissenschaftliche Format angezeigt werden soll (1E3): 1E+03, 1E+3 oder 1E3. Die Standardeinstellung ist 1E+03.

Hinweis: Sie können die Systemvariable verwenden, um die Einstellung für diese Auswahlliste zu ändern:

  • Wenn Sie @ete auf 1 setzen, verändert sich die Auswahlliste Wissenschaftlich: 1E3 Anzeige als in 1E+3.
  • Wenn Sie @sss auf 0 setzen, verändert sich die Auswahlliste Wissenschaftlich: 1E3 Anzeige als in 1E3.

Anzeige der Einheiten für Achsentitel

Um die Anzeige der Einheiten für Achsentitel zu steuern, (a) wählen Sie eine Option aus dem Ausklappmenü oder (b) geben Sie eine benutzerdefinierte Notationszeichenkette direkt ins Bearbeitungfeld ein. Die benutzerdefinierte Zeichenkette kann literalen Text einschließen.

Hinweise:

  • Die Einstellungen auf der Registerkarte Optionen: Achsen werden als Standard verwendet. Das Verwenden von Systemvariablen, wie in diesen Hinweisen beschrieben, ist im Allgemeinen nur notwendig, wenn Sie vorübergehend ein anderes Standardverhalten brauchen (wenn Sie z. B. eine Reihe von Zeichnungen per Skript erzeugen wollen und dafür einen temporären Wert für @LUQ nur für die Sitzung festlegen).
  • Sie können die Systemvariable @LUQ verwenden, um die Anzeige der Einheiten festzulegen: -2 = Benutzerdefiniert, -1 = Auto, 0 = Deaktivieren, 1 = (Einheit), 2 = [Einheit], 3 = {Einheit}, 4 = / Einheit, 5 = - Einheit, 6 = in Einheit. In den meisten Fällen sind die Werte von @LUQ und @LUQO die gleichen, aber in einigen Spezialfällen wird @LUQ in den realen Wert übersetzt, während die andere Systemvariable @LUQO den ursprünglich festgelegten Wert behält. Wenn Einheitsanzeige für Achsentitel auf <Auto> gesetzt ist und die aktuelle Sprache Deutsch ist, wird @LUQ in 2 übersetzt, während @LUQO gleich -1 bleibt.
  • Wenn Sie sich für die Anzeige der Einheit / <U> oder - <U> entscheiden, wird vor dem Trennzeichen standardmäßig eine Leerstelle eingefügt. Um diese Leerstelle auszuschalten, setzen Sie die Systemvariable @ATSS auf 0.
  • Um Einheiten als Präfix/Suffix zu Hilfsstrichsbeschriftungen hinzuzufügen, verwenden Sie Optionen auf der Registerkarte Beschriftung der Hilfsstriche des Dialogs Achsen.
