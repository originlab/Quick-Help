Eingabe

Eingabedaten

Legen Sie die Eingabedatenform mit Roh oder Indiziert fest.

Indiziert

In diesem Modus enthält eine Spalte Daten der Faktorvariablen und die anderen Spalten die Stufen dieser Faktoren.

FaktorA/FaktorB/FaktorC
Wählen Sie Daten für die Variablen Faktor 1, Faktor 2 und Faktor 3. Beachten Sie, dass die Faktorspalten wird als kategorial festgelegt werden, wenn es sich um ein Textspalten handelt.
Daten
Wählen Sie Daten für die Antwortvariablen aus.
Roh

In diesem Modus sind die Antwortdaten jeder Stufe der Faktorvariable in einer separaten Spalte gespeichert.

FaktorA/FaktorB/FaktorC
Name
Legen Sie die Stufennamen des Faktors fest.
Anzahl der Stufen
Legen Sie die Anzahl der Stufen des Faktors fest. Standardeinstellung ist 2. Jede Stufe hat eigene Bedienelemente für Stufei. Beachten Sie, dass die Systemvariable @AML dazu verwendet werden kann, die max. Anzahl der unterstützten Stufen für ANOVA zu steuern, die standardmäßig 25 beträgt. Sie können die maximale Anzahl der Stufen ändern, indem Sie der @AML-Variablen neue Werte zuweisen.
Stufei Name
Legen Sie die Stufennamen jedes Faktors fest.
Daten
Sie können Daten über das Bearbeitungsfeld unten festlegen oder auf klicken, um den Dialog für die Daten der dreifachen ANOVA zu öffnen und sie festzulegen.

Signifikanzniveau

Legen Sie das Signifikanzniveau der dreifachen ANOVA fest.

Homogenitätstests (Levenes Test)

Aktivieren Sie diese Option, um den Test auf Varianzhomogenität durchzuführen.

Modell

Legen Sie das ANOVA-Modell fest. Wählen Sie alle für ein vollfaktorielles Modell. Die Haupteffekte werden immer gezählt.

Deskriptive Statistik

Legen Sie fest, ob die deskriptive Statistik für die Haupteffekte und Wechselwirkungen berechnet werden soll.

Mittelwertevergleich

Legen Sie das Signifikanzniveau des paarweisen Mittelwertevergleichs im Nachhinein fest. Standardeinstellung ist 0,05.

Analyse der Trennschärfe

Berechen Sie die Trennschärfeanalyse:

Tatsächliche Trennschärfe

Aktivieren Sie diese Option, um die tatsächliche Trennschärfe zu berechnen.
Hypothetische Trennschärfe

Aktivieren Sie diese Option, um die hypothetische Trennschärfe aus dem/n hypothetischen Stichprobenumfang/umfängen, die unten eingegeben werden, zu berechnen.

Signifikanzniveau

Legen Sie das Signifikanzniveau () für die Berechnung der Trennschärfe fest. Diese Option ist nur verfügbar, wenn entweder die tatsächliche oder die hypothetische Trennschärfe aktiviert ist.
Hypothetische/r Stichprobenumfang/-umfänge

Legen Sie den hypothetischen Stichprobenumfang mit Hilfe von Leerzeichen zum Trennen der Einträge fest. Die Standardeinstellung ist immer 50 100 200.

Mittelwertdiagramm

Legen Sie fest, ob das Mittelwertdiagramm für Haupteffekte und Wechselwirkungen in dem Berichtsblatt erzeugt werden soll.

Hinweis: Wenn die Spalte Faktor auf Kategorisch gesetzt ist, wird das ausgegebene Diagramm entsprechend der kategorialen Ordnung gezeichnet, die auf der Registerkarte Kategorien (Spalteneigenschaften) basierend auf der Spalte Faktor gezeichnet.
