FAQ-1139 Ich erhalte beim Start des FLEXnet-Servers die Fehlermeldung "Datei nicht gefunden, orglab.exe". Was soll ich tun?

Orglab-File-Not-Found

Letztes Update: 20.09.2022

Wenn das Starten des Origin-Lizenzservers fehlschlägt und Sie die folgende Fehlermeldung in der Log-Datei erhalten:

9:42:08 (lmgrd) License server manager (lmgrd) startup failed:
9:42:08 (lmgrd) File not found, orglab.exe


Bitte prüfen Sie, ob die Datei orglab.exe sich im gleichen Ordner befindet wie lmgrd.exe. Den Pfad zur lmgrd.exe können Sie auf der Registerkarte Config Services der LMTools finden.

Falls nicht:

  1. Laden Sie bitte die Datei orglab.exe hier herunter.
  2. Extrahieren Sie die Datei und legen Sie orglab.exe im Verzeichnis von lmgrd.exe ab.
  3. Starten Sie den FLEXnet-Server neu.


Falls Sie orglab.exe haben, aber der FLEXlm-Server die Datei nicht finden konnte:

  1. Bitte öffnen Sie Ihre Origin-Lizenzdatei mit einem Texteditor.
  2. Fügen Sie den Pfad zur Datei orglab.exe auf der 2. Zeile hinzu, so dass Ihre Lizenzdatei sich folgendermaßen liest: 
    SERVER_NAME HOSTID VENDOR orglab "C:\OriginLicenseServer\Servers\orglab.exe"
  3. Starten Sie den FLEXnet-Server neu.

Schlüsselwörter:Lizenz, orglab, Datei nicht gefunden, License server manager failed, Start des Lizenzservermanagers fehlgeschlagen, lmgrd
