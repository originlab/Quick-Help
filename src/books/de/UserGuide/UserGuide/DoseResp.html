Function

Brief Description

Dose-response curve with variable Hill slope given by parameter 'p'.

Sample Curve

Parameters

Number: 4

Names: A1, A2, LOGx0, p

Meanings: A1 = bottom asymptote, A2 = top asymptote, LOGx0 = center, p = hill slope

LOGx0 = -5.0 (vary), p = 1.0 (vary)

Lower Bounds: none

Upper Bounds: none

Script Access

nlf_response1(x,A1,A2,LOGx0,p)
