Eingabe

Legt den Datenbereich für die Analyse fest, einschließlich des Zeitdaten-, Zensordaten- und Gruppierungsbereichs.

Zeitbereich Legen Sie die Spalte der Zeitdaten fest.
Zensorbereich Legen Sie die Spalte der Zensordaten fest.
Gruppierungsbereich (Optional) Legen Sie den Gruppendatenbereich fest. Beachten Sie, dass die Gruppierungsspalte als Kategorial gesetzt wird, wenn es sich um eine Textspalte handelt.

Zensierungswert(e)

Legt den Zensorsubjektwert fest, der identifiziert, welche Zeitwerte in dem beobachteten Datensatz zensiert werden.

Lebensdauertabellen

Verwenden Sie diesen Zweig, um die auszugebende Berichtstabelle festzulegen.

Zusammenfassung von Ereignis- und Zensorwerten Kontrollkästchen für die Ausgabe der Zusammenfassung des Ereignis- und Zensorwerteergebnisses
Überlebensschätzer Kontrollkästchen für die Ergebnisausgabe des Überlebensschätzers
Quartilschätzer Kontrollkästchen für die Ergebnisausgabe des Quartilschätzers
Mittelwertschätzer Kontrollkästchen für das Ergebnis des Mittelwertschätzers
Konfidenzniveau in % für Quartil-/Mittelwertschätzer In dieses Textfeld wird ein Konfidenzniveau eingegeben, das verwendet wird, wenn Fehler für die Überlebensfunktion und obere und untere Grenzen der Quartilschätzungen berechnet werden.

Überlebensdiagramm

Legt den auszugebenden Diagrammtyp fest

Überleben Kontrollkästchen für die Ausgabe der Überlebenskurve
Konfidenzintervalle hinzufügen Fügt Konfidenzintervalle zur Überlebenskurve hinzu.
1- Überlebensfunktion Kontrollkästchen für die Ausgabe einer 1-Minus-Überlebenskurve, d.h. die kumulative Verteilungskurve
Hazard Kontrollkästchen für die Ausgabe der Hazardkurve
Log-Überlebenskurve Kontrollkästchen für die Ausgabe der Log-Überlebenskurve
Zeichne alle Überlebenskurven in einem Diagramm Dieses Kontrollkästchen wird aktiviert, um alle Überlebenskurven in einem Diagrammfenster zu zeichnen.

Test auf Gleichheit

Legt die Methode fest, die zum Testen der Gleichheit verwendet werden soll.

Log-Rang Die Log-Rang-Methode wird zum Testen der Gleichheit von Überlebensverteilungen für verschiedene Gruppen verwendet.
Breslow Die Breslow-Methode wird zum Testen der Gleichheit von Überlebensverteilungen für verschiedene Gruppen verwendet.
Tarone-Ware Die Tarone-Ware-Methode wird zum Testen der Gleichheit von Überlebensverteilungen für verschiedene Gruppen verwendet.
Paarweise Vergleiche Wenn Sie einen oder mehrere Gleichheitstests von oben gewählt haben, aktivieren Sie das Kontrollkästchen für paarweise Vergleiche der drei oder mehr Gruppen.

Ausgabe

Ausgabe Ergebnisprotokoll

Ist dieses Kontrollkästchen aktiviert, wird das Ergebnis im Ergebnisfenster ausgegeben.

Legt das Ausgabeziel fest.

Überlebensfunktionen Legt das Datenziel der Überlebensfunktion fest.
Ergebnisse ausgeben Legt das Ziel des Ausgabeberichtsblatts fest.
