Veröffentlichen und Exportieren von Bildern

Image-Publish-Export-Images

Ein Bild aus einem Bildfenster exportieren

  1. Wählen Sie im Menü Datei: Bild exportieren.... Der Dialog CvExport wird geöffnet.
  2. Wählen Sie einen Dateityp und legen Sie den Dateiname und -pfad fest.
  3. Wenn der Dateityp = AVI, wählen Sie einen Codec.
  4. Klicken Sie auf die Schaltfläche Speichern, um das Bild oder Video zu exportieren.

Ein Bild aus einem Matrixfenster exportieren

  1. Wählen Sie im Menü Datei: Export: Bild.
  2. Wählen Sie im Dialog imageExp einen Dateityp in der Auswahlliste Speichern als aus und legen Sie den Dateinamen fest. Aktivieren Sie Optionendialog zeigen, wenn Sie mehr Optionen festlegen möchten. Klicken Sie auf die Schaltfläche Speichern.
  3. Wenn Optionendialog zeigen aktiviert ist, wird der Dialog expImage geöffnet. Legen Sie die Exportoptionen fest und klicken Sie auf OK, um das Bild zu exportieren.
