Aus Origin-Projekt importieren

Import-OPJ

Sie können nur einige Spalten aus dem gleichen Arbeitsblatt in eine Origin-Projektdatei (*.OPJ oder *.ORG) oder Arbeitsmappendatei (*.OGW) importieren.


Um festgelegte Spalte(n) aus einer Origin-Projektdatei zu importieren.

  1. Wählen Sie Daten: Aus Datei importieren: Origin-Projekt (OPJ, ORG, OGW).... Dieser Menübefehl öffnet den Dialog Excel.
  2. Wählen Sie die Datei und stellen Sie sicher, dass das Kontrollkästchen Optionendialog zeigen aktiviert ist. Klicken Sie auf die Schaltfläche Öffnen, um den Dialog impOPJ zu öffnen.
  3. Nehmen Sie notwendige Einstellungen im Dialog impOPJ vor. Klicken Sie dann auf die Schaltfläche OK, um die Dateien in Origin zu importieren.


Der Dialog impOPJ enthält mehr Bedienelemente:

Dateiname

Das Feld OPJ-Dateiname listet alle Dateien auf, die im Dialog Origin-Importausgewählt worden sind. Beachten Sie, dass Sie auf die Schaltfläche Durchsuchen rechts klicken können, um Ihre Dateien erneut auszuwählen.
Dateiinfo und Datenauswahl
  • Arbeitsmappe
Wählen Sie eine Arbeitsmappe in der Origin-Projektdatei aus der Auswahlliste aus.
  • Arbeitsblatt
Wählen Sie ein Arbeitsblatt in der ausgewählten Arbeitsmappe aus dieser Auswahlliste aus.
  • Spalten
Wählen Sie eine oder mehrere Spalte(n) zum Importieren.
Importoptionen

Importmodus

Diese Einstellung bestimmt, wo die importierten Daten hinzugefügt werden:

  • Bestehende Daten ersetzen: Die Daten der Spalte(n) werden in das festgelegte Fenster importiert. Die im Zielfenster bestehenden Daten werden gelöscht.
  • Neue Mappen öffnen: Die Daten der Spalte(n) werden in eine neue Mappe importiert.
  • Neue Blätter öffnen: Die Daten der Spalte(n) werden in ein neues Blatt der festgelegten Arbeitsmappe importiert.
  • Neue Spalten öffnen: Die Daten der Spalte(n) werden in neue Spalten des festgelegten Arbeitsblatts importiert.


Einschließen

Legen Sie die Spalteninformationen fest, die Sie mit den Daten importieren müssen.

  • Spaltentyp
  • Langname
  • Einheit
  • Kommentar
  • Parameter
  • Benutzerdefinierte Beschriftungen
Ausgabe Legen Sie die Position fest, wohin die Daten importiert werden sollen.
