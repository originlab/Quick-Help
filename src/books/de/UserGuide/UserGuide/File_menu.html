Menü Datei

Das Menü Datei wird zum Erstellen, Öffnen und Speichern von X-Funktionen verwendet.

Neuer X-Funktionen-Assistent

Öffnet den Neuen X-Funktionen-Assistenten. Der Assistent führt Sie durch die Schritte der Erstellung einer neuen X-Funktion. Beachten Sie, dass dieser Menübefehl die Symbolleistenschaltfläche Neuer X-Funktionen-Assistent dupliziert .
Benutzer-X-Funktionen öffnen

Mit diesem Menübefehl können Sie nach einer zu bearbeitenden X-Funktion suchen. Er dupliziert X-Funktion öffnen von der Symbolleistenschaltfläche OXF-Datei .
System-X-Funktionen öffnen

Mit diesem Menübefehl wird der Unterordner \X-Functions im Origin-Programmordner geöffnet und Sie können nach einer standardmäßigen X-Funktion suchen.
Speichern unter

Speichern der aktuellen X-Funktion in eine OXF-Datei. Beachten Sie, dass Sie eine X-Funktion nicht im Origin-Programmordner (reserviert für standardmäßige X-Funktionen) gespeichert werden kann.
Speichern

Speichern der aktuellen X-Funktion Beachten Sie, dass Sie eine X-Funktion nicht im Origin-Programmordner (reserviert für standardmäßige X-Funktionen) gespeichert werden kann. Dieser Menübefehl dupliziert die Symbolleistenschaltfläche OXF-Datei speichern .
