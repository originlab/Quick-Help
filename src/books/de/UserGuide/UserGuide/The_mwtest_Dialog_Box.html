Mann-Whitney-Test

Nullhypothese Die Nullhypothese Die zwei Stichproben stammen aus Grundgesamtheiten, die die gleiche Verteilung haben (nur lesbar).
Alternativhypothese
  • Median1 <> Median2
Ein beidseitiger Mann-Whitney-Test wird durchgeführt.
  • Median1 > Median2
Ein oberer Mann-Whitney-Test wird durchgeführt.
  • Median1 < Median2
Ein unterer Mann-Whitney-Test wird durchgeführt.
Signifikanzniveau

Das Alpha (α) des Tests
Genauer P-Wert

Geben Sie an, ob der genaue P-Wert ausgegeben werden soll. Der P-Wert im Mann-Whitney-Test kann eine exakte Wahrscheinlichkeit oder eine normale Approximation sein. Für einen großen Datensatz kann die exakte Berechnung sehr aufwändig werden, insbesondere dann, wenn die Beobachtungen Beziehungen enthalten. Bei einer großen Stichprobe wird empfohlen, dass Sie dieses Kontrollkästchen deaktivieren und eine Approximation ausgeben. Andernfalls sollte die exakte Wahrscheinlichkeit benutzt werden.
