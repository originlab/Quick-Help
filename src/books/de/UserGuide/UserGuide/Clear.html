Löschen


Die Funktion imgClear bereinigt das Bild. Sie setzt alle Pixelwerte auf 0. Diese Funktion kann über das Befehlsfenster mit folgendem Befehl ausfgerufen werden:

imgClear

Weitere Informationen über diese Funktion finden Sie in der Dokumentation der X-Funktion imgClear in der Origin-Hilfe. Diese kann über das Befehlsfenster mit folgendem Befehl aufgerufen werden:

help imgClear
