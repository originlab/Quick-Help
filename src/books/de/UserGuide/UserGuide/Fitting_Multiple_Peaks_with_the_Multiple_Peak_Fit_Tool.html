Mehrere Peaks mit dem Hilfsmittel Mehrere Peaks anpassen anpassen

Fit-MultiPeakFit-Tool

Das Hilfsmittel zum Anpassen mehrerer Peaks bietet eine interaktive und einfache Möglichkeit zum Auswählen mehrerer Peaks in einem Diagramm und ihrer Anpassung mit einer Peakfunktion.

Wählen Sie Analyse: Peaks und Basislinie: Mehrere Peaks anpassen. Dadurch wird der Dialog nlfitpeaks geöffnet.

Wählen Sie im Dialog die Eingabedaten und die Peakfunktion zum Durchführen der Anpassung. Beachten Sie, dass die Eingabedaten der aktiven Datenzeichnung initialisiert werden, wenn das Hilfsmittel von einem Diagrammfenster aus geöffnet wird.
Bitte beachten Sie: Seit Origin 2020b können Sie, wenn Sie von einem Diagrammfenster ausgegangen sind, auf die Pfeilschaltfläche hinter diesem Feld klicken, um X-Skalierungsbereich verwenden auszuwählen und den X-Skalierungsbereich des Quelldiagramms auf den Eingabebereich anzuwenden.

Durch Klicken auf OK wird der Dialog geschlossen und das Diagramm aktiviert. Wenn das Hilfsmittel von einem Arbeitsblatt aus geöffnet wurde, wird ein Diagramm erstellt. Der Mauszeiger ändert sich zu und der Dialog Punkte erhalten wird geöffnet. Verwenden Sie die Maus oder die Pfeiltasten der Tastatur, um den Zeiger auf ein gewünschtes Peakzentrum zu richten. Klicken Sie dann doppelt oder drücken Sie die Eingabetaste, um diesen Punkt als Peakzentrum auszuwählen. Wenn Sie den Zeiger bewegen, werden seine X- und Y-Koordinaten im Dialog Punkte erhalten und im Fenster Datenanzeige angezeigt.. Der Dialog Punkte erhalten zeigt auch die Anzahl der gewählten Peakzentren an.

In einigen Fällen müssen Sie das Minitool Peakanalyse verwenden, um zuvor die Zielpeaks zu finden. Sie können die Peaks dann zuerst mit Peakmarkierungen beschriften.

Sobald sich Peakmarkierungen auf dem Diagramm befinden, ist das Kontrollkästchen Peakmarkierung verwenden verfügbar. Aktivieren Sie dieses Kontrollkästchen, um die Peaks direkt mit den Markierungen anzupassen. In diesem Fall müssen Sie Peaks nicht manuell auswählen wie oben.

Wenn Sie die gewünschte Anzahl der Peakzentren ausgewählt haben, können Sie auf zwei Arten fortfahren:

  • Klicken Sie auf die Schaltfläche Fit im Dialog Punkte erhalten.
    Dies wird die Daten anhand Ihrer gewählten Punkte anpassen, um die Peaks zu initialisieren. Dem Diagramm wird eine Tabelle mit Fit-Ergebnissen und Parameterwerten hinzugefügt.
  • Klicken Sie auf die Schaltfläche NLFit öffnen im Dialog Punkte erhalten.
    Dies öffnet den Dialog NLFit, in dem die Parameter der Peakfunktion wie Peakhöhe und Peakzentrum anhand der von Ihnen gewählten Peakzentren initialisiert werden. Sie können die Einstellung weiter anpassen und Grenzen oder Bedingungen hinzufügen oder Parameter gemeinsam nutzen und dann mit der Anpassung manuell fortfahren. Einzelheiten zu den Einstellungen des Anpassungsprozesses lesen Sie bitte unter Der Dialog NLFit.

Bitte verwenden Sie für eine weitergehende Peakanpassung einschließlich Basislinienerkennung bitte die Option Peaks anpassen im Hilfsmittel Peaks analysieren, verfügbar in OriginPro.
