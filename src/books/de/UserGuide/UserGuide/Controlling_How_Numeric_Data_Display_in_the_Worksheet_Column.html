Zusätzlich zum Benennen und Festlegen der Diagrammzuordnung einer Spalte können Sie das Format (Text & Numerisch, Numerisch usw.) und den Datentyp (z. B. numerische Daten werden als "Double", "Real" usw. gespeichert) festlegen. Standardmäßig wird das Format der Spalten so festgelegt, dass Text & Numerische Daten bearbeitet werden können, aber wenn die Systemleistung begrenzt ist, kann es auf Numerisch beschränkt werden. Zusätzlich zum Festlegen des Datentyps als Numerisch können Sie die Größe des Speichers, der dem numerischen Datenspeicher zugeordnet wird, weiter verringern (siehe unten Vorsicht).

Numerisch vs. Text & Numerisch

Das Format Text & Numerisch benötigt ungefähr 20% mehr Speicherkapazität als der Typ Numerisch Doppelt(8). In den meisten Fällen bereitet dieser Unterschied keine Probleme mit der Computerleistung. Wenn Sie sich Gedanken über das Maximieren der Systemressourcen machen, können Sie Origin konfigurieren, so dass es Numeric Double(8) für alle Spalten von neuen Arbeitsblättern verwendet:

  1. Wählen Sie im Menü Einstellungen: Optionen.
  2. Klicken Sie auf die Registerkarte Sonstiges.
  3. Wählen Sie Standardformat für Spalten = Numerisch.

Falls Sie außerdem benutzerdefinierte Formate auf eine Spalte mit numerischen Daten anwenden, sollten Sie die Spalte Format von Text & Numerisch in Numerisch ändern. Während Text & Numerisch im Allgemeinen Origins benutzerdefinierte Anzeigeformatierung unterstützt, gibt es kleine Macken. Das Festlegen des Formats auf Numerisch beseitigt einige Probleme.

Numerische Datentypen und Speicherzuweisung

Wenn Sie mit numerischen Daten arbeiten, können Sie die Größe des Speichers einstellen, der jedem Datenwert zugeordnet wird, indem Sie die Einstellungen Datentyp im Dialogfeld Spalteneigenschaften ändern. Wenn Sie beispielsweise noch mehr Speicherplatz sparen wollen (bis zu weiteren 50%), können Sie die Spalten auf Format = Numerisch und den Datentyp auf Real(4) setzen (lesen Sie die untenstehende Anmerkung):

Um den Dialog Spalteneigenschaften zu öffnen:

  1. Klicken Sie doppelt auf den Spaltenkopf.

oder

  1. Klicken Sie mit der rechten Maustaste auf die Spalte und wählen Sie Einstellungen.

oder

  1. Wählen Sie bei ausgewählter Spalte im Origin-Menü Format: Spalte.
    Anschließend:
  2. Wählen Sie im Dialog Spalteneigenschaften das Format = Numerisch.
  3. Wählen Sie in der Auswahlliste Datentyp.

Format und numerische Anzeige

Wenn die Arbeitsblattspalte auf Datentyp = Numerisch oder Text & Numerisch eingestellt wurde, wird die Anzahl der dargestellten Ziffern durch eine Kombination dieser Einstellungen im Dialog Spalteneigenschaften bestimmt.

  • Anzeige Legen Sie Dezimal-, wissenschaftliche bzw. technische Formate oder ein benutzerdefiniertes Format fest.
  • Stellen Dies ist die Anzahl der Dezimalstellen oder gültigen Stellen, die in den Spalten angezeigt werden (Datensatz).

Wenn nichts anderes angegeben wird (z.B. die numerische Anzeige auf Standard Dezimalstellen eingestellt ist), verwendet Origin die voreingestellte Anzahl der Dezimalstellen auf der Registerkarte Zahlenformat des Dialogs Optionen.

Beachten Sie, dass die Einstellungen für Stellen im Dialog Spalteneigenschaften Vorrang haben vor den allgemeinen Einstellungen im Dialog Optionen.


Um die Anzahl der standardmäßig voreingestellten Dezimalstellen einzurichten (allgemein gültig, falls dies nicht im nächsten Schritt anders festgelegt werden sollte):

  1. Wählen Sie im Menü Einstellungen: Optionen.
  2. Klicken Sie auf die Registerkarte Zahlenformat.
  3. Legen Sie die Anzahl der Nachkommastellen fest.


Um die Anzahl der gültigen Stellen bzw. der Dezimalstellen für einen oder mehrere Datensätze eines Arbeitsblatts festzulegen:

  1. Wählen Sie im aktiven Arbeitsblatt Format: Spalte aus oder klicken Sie doppelt auf die Spaltenüberschrift des Arbeitsblatts. Der Dialog Spalteneigenschaften wird geöffnet.
  2. Ändern Sie Stellen von Standard Dezimalstellen in Dezimalstellen setzen = oder Signifikante Stellen = und geben Sie eine Zahl in das Textfeld unten ein (max. Dezimalstellen = 12; max. gültige Stellen = 15).
  3. Optional wählen Sie Auf alle Spalten rechts anwenden aus.

Ein Hinweis zu angezeigten vs. tatsächlichen Werten

Wenn Sie Ihre Daten in ein Arbeitsblatt importieren oder eingeben, verwendet Origin die Einstellungen für die Anzahl der Ziffern/Dezimalstellen, um die Daten jeder einzelnen Zelle anzuzeigen. Beachten Sie, dass dies lediglich eine visuelle Darstellung des Datensatzes ist. Die tatsächlichen Datenwerte werden gespeichert. Wenn Sie Berechnungen durchführen, werden die tatsächlichen Werte -- übereinstimmend mit den Grenzen des Datentyps und der zugeordneten Speichergröße -- verwendet, nicht die in der Zelle angezeigten Werte.

Dezimale, wissenschaftliche und Ingenieursschreibweise

Origin unterstützt die Anzeige von Arbeitsblattdaten in den Formaten Dezimal:1000, Wissenschaftlich:1E3, Ingenieur:1k und Dezimal:1,000. Wenn Sie Dezimal:1000, Ingenieur:1k oder Dezimal:1.000 auswählen, verwendet Origin zusätzliche Einstellmöglichkeiten (die Einstellungen Oberer Grenzwert und Unterer Grenzwert auf der Registerkarte Zahlenformat im Dialog Optionen) für die numerische Anzeige.

Standardmäßig sind die Einstellungen Oberer Grenzwert und Unterer Grenzwert 6 beziehungsweise -3. Solange Werte in einem Bereich von 1x10-3 bis 1x106 liegen, werden Daten mit Hilfe des im Dialog Spalteneigenschaften in der Auswahlliste Anzeige ausgewählten Formats angezeigt (Gruppe Optionen). Liegen die Werte entweder ober- oder unterhalb der Konvertierungsschwelle (kleiner als 1x10-3 oder größer als 1x106), werden die Daten in der wissenschaftlichen Schreibweise angezeigt, und zwar unabhängig von den Einstellungen in der Auswahlliste Format.


Um die standardmäßigen Grenzwerteinstellungen zu ändern:

  1. Wählen Sie im Menü Einstellungen: Optionen.
  2. Klicken Sie auf die Registerkarte Zahlenformat.
  3. In der Gruppe In wissenschaftliche Schreibweise konvertieren legen Sie den Oberen Schwellenwert und Unteren Schwellenwert fest.
