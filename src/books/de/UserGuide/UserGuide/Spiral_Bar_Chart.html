Spiralförmiges Balkendiagramm

Spiral-Bar-Chart

Ein spiralförmiges Balkendiagramm zeichnet zeitbasierte Daten als Balken entlang einer archimedischen Spirale. Es beginnt im Zentrum einer Spirale und entwickelt sich dann nach außen.

Datenanforderungen

Wählen Sie eine X-Spalte mit einer Y-Spalte aus. Die X-Spalte enthält meistens Zeit-/Datumsdaten, gezeichnet als eine spiralförmige Achse, und die Y-Spalte wird als Balken basierend auf der spiralförmigen Achse gezeichnet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen > Balken, Kreis, Fläche: Spiralförmiges Balkendiagramm.

Vorlage

SpiralMap.OTPU (installiert im Origin-Programmordner).

Notizen

Spiralförmige Balkendiagramme sind ideal, um große Datensätze zu zeigen, typischerweise Trends über einen langen Zeitabschnitt.

Um all diese Datenpunkte zu zeichnen, können Sie entscheiden, welches Layout zum Anordnen der Punkte verwendet werden soll:

  • Nach Zeitabschnitt: Die Datenpunkte werden als periodische Muster gezeichnet. Jedem Zeitabschnitt kann eine Farbe zugewiesen werden, um sie zu unterbrechen und einen Vergleich zwischen den Zeitabschnitten zu ermöglichen.
  • Nach Wicklungen: Die Datenpunkte werden auf mehreren Wicklungen gezeichnet, jede Wicklung hat jeweils die gleiche Farbfüllung.

Sie können zur Registerkarte Spirale im Dialog Details Zeichnung gehen, um weitere Anpassungen vorzunehmen.
