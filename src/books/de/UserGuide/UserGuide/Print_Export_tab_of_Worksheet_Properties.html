Registerkarte Druck/Export (Arbeitsblatteigenschaften)

Ansicht

Legen Sie fest, was gezeigt soll, wenn das aktuelle Arbeitsblatt exportiert oder gedruckt wird. Rahmendicke und Rand (%) werden für das Menü Bearbeiten: Kopieren: Als Tabelle kopieren (HTML/EMF) verwendet.

Rand

Legen Sie den Rand für die gedruckte/exportierte Seite in Millimetern (mm) oder Zoll (Zoll) fest.

Hinweis: Wählen Sie Datei: Drucken und klicken Sie dann auf die Schaltfläche Optionen. Auf diesem Weg wird der Dialog Druckoptionen geöffnet. In diesem Dialog können Sie auch die obenstehenden Einstellungen vornehmen.
