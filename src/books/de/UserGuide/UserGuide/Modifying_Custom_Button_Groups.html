Die Bedienelemente der Registerkarte Schaltflächengruppen des Dialogfelds Symbolleiste anpassen (Ansicht: Symbolleisten) ermöglichen es Ihnen, die Bitmap für die benutzerdefinierten Schaltflächengruppen zu entfernen oder zu verändern und die Schaltflächen aus der benutzerdefinierten Schaltflächengruppe zu entfernen oder Schaltflächen hinzuzufügen.

Löschen einer benutzerdefinierten Schaltflächengruppe

Um eine benutzerdefinierte Schaltflächengruppe zu entfernen:

  1. Wählen Sie die Schaltflächengruppe, die Sie entfernen möchten, aus dem Listenfeld Gruppen aus.
  2. Klicken Sie auf die Schaltfläche Löschen in der Gruppe Schaltflächengruppe.

Eine Warnmeldung bittet um Bestätigung, bevor die Gruppe entfernt wird. Wenn Sie auf Ja klicken, entfernt Origin die benutzerdefinierten Schaltflächengruppen aus dem Listenfeld Gruppen. Falls sich irgendwelche Schaltflächen aus dieser (entfernten) Gruppe in der Symbolleiste befinden, besitzen diese Schaltflächen nach der Entfernung der Gruppe keine Funktion mehr.

Hinweis: Sie können keine standardmäßigen Schaltflächengruppen entfernen.

Verändern der Bitmap für eine benutzerdefinierte Schaltflächengruppe

Wenn Sie eine neue Schaltflächengruppe hinzugefügt haben, können Sie die Bitmap für die Gruppe verändern:

  1. Wählen Sie die gewünschte Gruppe aus dem Listenfeld Gruppen aus.
  2. Geben Sie einen neuen Pfad und Dateinamen in das Textfeld Bitmap ein oder klicken Sie auf die Schaltfläche Durchsuchen, um die neue Bitmap-Datei zu finden.

Hinweis: Sie können auch die Bitmap für die Kategorie Benutzerdefiniert bearbeiten.

Entfernen einer Schaltfläche aus einer benutzerdefinierten Schaltflächengruppe

Um eine Schaltfläche aus einer benutzerdefinierten Schaltflächengruppe (inklusive der benutzerdefinierten Schaltflächengruppe) zu entfernen:

  1. Wählen Sie die Schaltfläche aus, die Sie entfernen möchten.
  2. Klicken Sie auf die Schaltfläche Löschen in der Gruppe Schaltfläche.

Hinzufügen einer Schaltfläche in eine benutzerdefinierte Schaltflächengruppe

Um eine Schaltfläche zu einer benutzerdefinierten Schaltflächengruppe (einschließlich der benutzerdefinierten Schaltflächengruppe) hinzuzufügen:

  1. Wählen Sie die Schaltflächengruppe aus, zu der Sie die Schaltfläche hinzufügen möchten.
  2. Klicken Sie auf die Schaltfläche Hinzufügen in der Schaltflächengruppe.
  3. Das Dialogfeld Schaltfläche Hinzufügen wird geöffnet.
  4. Wählen Sie die Indexnummer, die dem Schaltflächenbild in der Bitmapdatei entspricht, aus oder geben Sie die Indexnummer ein.
  5. Klicken Sie auf OK. Die Schaltfläche wird zu der benutzerdefinierten Schaltflächengruppe hinzugefügt.
