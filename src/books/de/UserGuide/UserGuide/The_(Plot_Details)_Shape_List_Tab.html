Die Registerkarte Formliste (Details Zeichnung)

PD-dlg-shape-list-tab


Sie können diese Registerkarte verwenden, um eine Inkrementformliste zu erstellen/benutzerdefiniert anzupassen/zu laden. Diese Registerkarte ist verfügbar:

  • Für 2D-Diagramme, wenn die Symbolform von der Inkrementliste oder dem Index auf der Registerkarte Symbol gesteuert wird (d.h. die Form wird von den Werten in einer Datenspalte gesteuert), wenn für die Benutzerdefinierte Konstruktion die Option Geometrisch gewählt und in der Auswahlliste Form eine Datenspalte oder Inkrement festgelegt ist.
  • Für 3D-XYY-/-XYZ-Balkendiagramme, wenn die Balkenform durch die Inkrementliste oder den Index auf der Registerkarte Muster gesteuert wird (d.h., die Form wird durch die Werte in einer Datenspalte gesteuert).

Benutzerdefinierte Inkrementliste verwenden

Dieses Kontrollkästchen ist standardmäßig deaktiviert. In diesem Fall wird die Standardinkrementliste verwendet.

Wenn dieses Kontrollkästchen aktiviert wird, wird die Liste Formen gezeigt, die Sie benutzerdefiniert anpassen können.

Liste der Formen

Sie können die Formliste benutzerdefiniert anpassen, speichern und neu laden. Weitere Optionen finden Sie im Kontextmenü.

Wenn die Form einen Index auf eine Spalte mit kategorialen Daten hat, werden Kategorien neben jedem Stil in der Liste Formen angezeigt, wodurch es einfacher ist, Kategorie und Form in Verbindung zu bringen.

Eine Form benutzerdefiniert anpassen

Klicken Sie für jede Zeile in der Liste auf die dreieckige Schaltfläche und wählen Sie ein Standardsymbol/eine Balkenform aus.

Formliste neu anordnen

Sie können ausgewählte Zeile(n) in der Formliste verschieben, hinzufügen und löschen sowie die Liste umkehren. Dazu verwenden Sie die gleichen Methoden wie zum Bearbeiten der Farbliste.

Inkrementliste speichern und laden

Sie können mit der rechten Maustaste klicken und im Kontextmenü wählen, ob eine Inkrementliste gespeichert oder geladen werden soll. Dazu verwenden Sie die gleichen Methoden wie zum Speichern und Laden einer Inkrementliste.

Inkrementliste kopieren und einfügen

Um eine benutzerdefinierte Farbinkrementliste auf andere Zeichnungen anzuwenden, können Sie mit der rechten Maustaste auf die Liste klicken und Kopieren wählen. Wechseln Sie dann zur Registerkarte Formliste der anderen Zeichnungen, klicken Sie mit der rechten Maustaste auf die Liste und wählen Sie die Option Einfügen.
