Datenanforderungen

Wählen Sie eine oder mehrere Y-Spalten oder einen Datenbereich aus mindestens einer Y-Spalte aus. Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen: Einfache 2D: Horizontale Stufen.

oder

Klicken Sie auf die Schaltfläche Horizontale Stufen in der Symbolleiste 2D Grafiken.

Vorlage

LINE.OTP (im Origin-Programmordner installiert)

Notizen

Jeder Datenpunkt im Diagramm ist mit einer Linie verbunden. Der Verbindungstyp der Linie ist auf eine horizontale Stufenverbindung (Stufe Horiz.) auf der Registerkarte Linie des Dialogs Details Zeichnung festgelegt (Format: Zeichnung). Die Verbindung Horizontale Stufen erstellt eine rechwinklige Verbindung zwischen den Datenpunkten mit einer horizontalen Anfangslinie. Die Datenpunkte werden nicht angezeigt.

Wenn Sie möchten, dass der Datenpunkt mittig des horizontalen Verlaufs gezeichnet wird, können Sie zur Registerkarte Linie der Registerkarte Linie im Dialog Details Zeichnung gehen, um die Verbindung auf Stufe H Zentr. zu setzen.

Binäre Spalte zum Zeichnen eines Stufendiagramms

Wenn die Quellspalte binäre Werte ("0"/"1", "Ja"/"Nein" bzw. "Wahr"/"Falsch") speichert, können Sie ihr Format auf Binär setzen und die Werte dann als Liniendiagramm zeichnen, um den binären Status zu zeigen.

Die Linienverbindung wird auf die zentrale Stufenverbindung begrenzt: Wenn die Y-Spalte binär ist, zeichnen wir mit Stufe H Zentr.; wenn die X-Spalte binär ist, zeichnen wir mit Stufe V Zentriert.
