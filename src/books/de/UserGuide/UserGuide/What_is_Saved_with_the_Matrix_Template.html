Die Seitenansicht des Matrixfensters

Das Matrixfenster besitzt drei Bedienbereiche, die einzeln angezeigt werden können:

  • Der Organizer
  • Der Skriptbereich
  • Die Matrixminiaturbilder

Beachten Sie, dass geöffnete Bereiche mit der Matrixvorlagendatei (OTM) gespeichert werden.

Die Blätter und Objekte des Matrixfensters

Das Matrixfenster kann wie die Arbeitsmappe mehrere Blätter enthalten. Jedes Matrixblatt enthält ein oder mehrere Matrixobjekte. Matrixblätter und Blattnamen und Matrixobjekte und Objektnamen werden mit der Matrixmappenvorlage gespeichert.

Siehe Benennungsregeln für Matrixblätter und Matrixobjekte und Grundlegende Operationen von Matrixmappe und Matrixblatt.

Dialogfeld Matrixeigenschaften

Das Dialogfeld Matrixeigenschaften bietet Bedienelemente zum Festlegen von Spaltenbreite, Datentyp und Anzeigeformat. Die Einstellungen werden mit der Matrixvorlage gespeichert.

Um diesen Dialog zu öffnen, führen Sie eine der folgenden Operationen aus:

  • Wählen Sie Matrix: Eigenschaften setzen im Origin-Menü.
  • Klicken Sie auf das Objektsymbol und wählen Sie im Kontextmenü Einstellungen aus.
  • Klicken Sie auf die obere linke Ecke, um die gesamte Matrix auszuwählen, klicken Sie dann mit der rechten Maustaste auf die Matrix und wählen Sie Eigenschaften der Matrix setzen im Kontextmenü.

Siehe Eigenschaften des Matrixblatts.

Dialogfeld Matrixdimensionen

Das Dialogfeld Matrixdimensionen bietet Möglichkeiten, um die Matrixabmessungen und die XY-Abbildungsbeziehungen einzustellen.

Um diesen Dialog zu öffnen, führen Sie eine der folgenden Operationen aus:

  • Wählen Sie Matrix: Eigenschaften setzen im Origin-Menü.
  • Klicken Sie auf die obere linke Ecke, um die gesamte Matrix auszuwählen, klicken Sie dann mit der rechten Maustaste auf die Matrix und wählen Sie Eigenschaften der Matrix setzen im Kontextmenü.

Siehe Matrixblattdimensionen und Koordinaten.

Dialog Matrixwerte setzen

Der Dialog Matrixwerte setzen ermöglicht es Ihnen, eine Formel einzugeben, um festzulegen, wie die Matrix gefüllt werden soll.

Um diesen Dialog zu öffnen, führen Sie eine der folgenden Operationen aus:

  • Wählen Sie Matrix: Eigenschaften setzen im Origin-Menü.
  • Klicken Sie auf die obere linke Ecke, um die gesamte Matrix auszuwählen, klicken Sie dann mit der rechten Maustaste auf die Matrix und wählen Sie Eigenschaften der Matrix setzen im Kontextmenü.
  • Verwenden Sie die Tastenkombination Strg + Q.

Siehe Matrixwerte setzen.

Dialoge der ASCII-Importoptionen

Die Einstellungen im Dialog ASCII bestimmen, wie mit den Daten beim Importieren in die Matrix umgegangen wird.

Um den Dialog impASC zu öffnen:

  1. Wählen Sie Daten: Aus Datei importieren und aktivieren Sie das Kontrollkästchen Optionendialog zeigen (unter der Dateiliste). Wählen Sie eine Datei und klicken auf Öffnen. Dadurch wird der Dialog impASC geöffnet.
  2. Klicken Sie auf die Schaltfläche Dialogdesign und wählen Sie In <Blatt> speichern. Übernehmen Sie Ihre Einstellungen und klicken Sie auf OK. Die aktuellen Importeinstellungen werden im Blatt gespeichert und die ausgewählte Datei importiert.

Um die impASC-Optionen als Dialogdesign zu speichern, klicken Sie auf die Schaltfläche Dialogdesign und wählen Sie Speichern unter, bevor Sie auf OK klicken und den Dialog schließen.

Hinweis: Diese Aktionen speichern die Importoptionen nicht in der Vorlage. Um die Optionen in der Matrixvorlage zu speichern, müssen Sie die Matrixvorlage neu speichern.

Der Dialog ASCII Export in

Der Dialog für den ASCII Export in legt das Standardexportverhalten für die Matrix fest, inklusive der Einstellungen für den Export der Spaltennamen und Datentrennzeichen.

Um den Dialog des ASCII Exports zu öffnen:

  1. Wählen Sie im Menü Datei: Export: ASCII.
  2. Aktivieren Sie das Kontrollkästchen Optionendialog zeigen.
  3. Geben Sie einen Dateinamen ein und Speichern Sie die Datei.

Durch Klicken auf Speichern öffnet Origin den Dialog expMatASC. Wählen Sie anschließend Speichern in <Blatt> für Dialogdesign. Dies erlaubt Ihnen, die Exportoptionen, die im Blatt gespeichert werden, anzuzeigen und zu ändern. Wenn Sie die Einstellungen in einem Dialogdesign speichern möchten, wählen Sie Speichern unter.

Hinweis: Diese Aktionen speichern die Importoptionen nicht in der Vorlage. Um die Optionen in der Matrixvorlage zu speichern, müssen Sie die Matrixvorlage neu speichern.
