Datenanforderungen

Wählen Sie mindestens eine Y-Spalte aus (oder einen Bereich aus mindestens einer Y-Spalte). Idealerweise wählen Sie mindestens zwei Y-Spalten aus (oder einen Bereich aus mindestens zwei Y-Spalten). Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Klicken Sie auf die Schaltfläche Wasserfall Z: Farbabbildung in der Symbolleiste 2D-Grafiken.

Vorlage

WATERFAL.OTP (im Origin-Programmordner installiert)

Notizen

Das Wasserfalldiagramm ist ideal, um Abweichungen zwischen mehreren Datensätzen, die unter ähnlichen Bedingungen erstellt wurden, zu vergleichen. Das Diagramm besitzt einen Pseudo-3D-Effekt, der es Ihnen ermöglicht, Abweichungen in Z-Richtung zu sehen. Die Farbe wird entlang der Z-Richtung abgebildet.

Jeder Datensatz wird als Liniendiagramm angezeigt, das um einen angegebenen Betrag sowohl in X- als auch in Y-Richtung versetzt ist. Weitere Einzelheiten zum Anpassen dieses Diagramms finden Sie unter folgendem Thema in der Hilfe: Wasserfalldiagramm benutzerdefiniert anpassen.
