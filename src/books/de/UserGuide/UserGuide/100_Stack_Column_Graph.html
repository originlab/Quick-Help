Datenanforderungen

Wählen Sie mindestens eine Y-Datenspalte (oder einen Bereich aus mindestens einer Spalte) aus. Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen: Einfache 2D: Gestapeltes 100%-Säulendiagramm.

oder

Klicken Sie auf die Schaltfläche Gestapeltes 100%-Säulendiagramm auf der Symbolleiste 2D-Grafiken.

Vorlage

COLUMN.OTP (im Origin-Programmordner installiert).

Hinweise

Dieser Diagrammtyp kann auch aus einem gestapelten Säulendiagramm erstellt werden, indem das Kontrollkästchen Für kumulierte Häufigkeit auf Prozent normieren auf der Registerkarte Stapeln des Dialogs Details Zeichnung - Layereigenschaften aktiviert wird. Sie können die Datenbeschriftungen als Prozentwerte zeigen, indem Sie auf der Registerkarte Beschriftung des Dialogs Details Zeichnung das Beschriftungsformat auf Benutzerdefiniert und Formatzeichenkette = $(p,.0)% setzen.

Sie können Fehlerbalken zu Ihrem gestapelten 100%-Säulendiagramm hinzufügen, indem Sie Y- und yEr-Spalten im Arbeitsblatt verbinden. Weitere Informationen finden Sie unter Fehlerbalken zu Diagramm hinzufügen, Methode 1 - Diagrammzuordnung vordefinieren.
