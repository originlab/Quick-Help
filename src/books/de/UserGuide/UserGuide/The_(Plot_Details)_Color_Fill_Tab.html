Vordere Oberfläche

Bei Aktivierung gibt es zwei Modi zum Anwenden von Farbe auf die vordere Oberfläche.

Stückweise füllen

Die Gitternetzlinien werden als Farbfülllgrenze verwendet.

Eine allgemeine Erklärung der Farblisten finden Sie unter Diagrammfarben benutzerdefiniert anpassen.

Informationen zu Index, Direkt RGB und Farbabbildung finden Sie unter Datensatz zum Einstellen der Diagrammfarbe verwenden.

Bis Konturlinien füllen

Farbabbildung

Wenn die Oberfläche aus XYZ-Daten erstellt wurde, können Sie eine weitere Spalte als Farbabbildung wählen, um Farbe bis zu den Konturlinien zu füllen. Wenn das Kontrollkästchen Identisch aktiviert ist, wird die aktuelle Z-Spalte als Farbabbildung verwendet, um die Oberfläche zu füllen; deaktivieren Sie das Kontrollkästchen Identisch. Sie können eine andere Spalte (außer der aktuellen Z-Spalte) im aktuellen Arbeitsblatt als Farbabbildung auswählen.

Kontur über Matrix füllen

Wenn die Oberfläche aus einen Matrixobjekt erstellt wurde, können Sie eine Matrix festlegen, um Farbe bis zu den Konturlinien zu füllen. Beachten Sie, dass Sie jedes Matrixobjekt in der Matrix mit Hilfe der Auswahlliste wählen können; oder aktivieren Sie Identisch, um Werte im aktuellen Matrixobjekt zu verwenden.

Hintergrundfarbe

Legen Sie die Hintergrundfarbe aus der Farbliste fest.

Transparenz

Aktivieren Sie das Kontrollkästchen Auto. Die Einstellung der Transparenz folgt der Einstellung auf der Registerkarte Oberfläche.

Deaktivieren Sie das Kontrollkästchen Auto, um die Transparenz manuell anzupassen.
