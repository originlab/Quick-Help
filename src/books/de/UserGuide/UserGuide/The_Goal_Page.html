Die Seite Ziel

FitFuncBuilder-Goal

Die Seite Start ist die erste Seite des Dialogs Fitfunktionen erstellen. Sie können auf dieser Seite wählen, ob eine benutzerdefinierte Anpassungsfunktion erstellt oder bearbeitet werden soll. Abhängig von dem gewählten Ziel zeigen die folgenden Seiten des Dialogs einige kleinere Differenzen auf.

Bedienelemente des Dialogs

Eine neue Funktion erstellen Wählen Sie das Ziel zum Erstellen einer neuen Funktion. Die Funktion befindet sich im Ordner Anwenderdateien\ fitfunc.
Eine benutzerdefinierte Funktion bearbeiten Wählen Sie dieses Ziel, um eine benutzerdefinierte Funktion zu bearbeiten. Wenn sich im Ordner Anwenderdateien\ fitfunc benutzerdefinierte Funktionen befinden, erzeugt das Hilfsmittel eine Auswahlliste mit allen verfügbaren benutzerdefinierten Funktionen als Optionen unter Eine benutzerdefinierte Funktion bearbeiten, so dass Sie eine von ihnen zur Bearbeitung auswählen können. Gibt es keine benutzerdefinierte Funktion in dem Ordner, werden Sie durch eine Nachricht darauf aufmerksam gemacht: Keine benutzerdefinierten Funktionen gefunden! Bitte zuerst eine Funktion erstellen.

Mit der App Fitting Function Library können Sie nach Add-on-Fitfunktionen suchen. Diese App ist auf den neuesten Versionen von Origin vorinstalliert. Wenn Sie keine besondere Fitfunktion innerhalb von Origins Standardbibliothek finden, prüfen Sie, ob sie zum Download auf der Webseite von OriginLab zur Verfügung steht. Starten Sie die App, indem Sie (1) auf den Link der Seite Ziel klicken oder (2) gehen Sie zu Ihrer Apps-Galerie, die sich rechts in Ihrem Arbeitsbereich befindet, und klicken Sie auf das Symbol der App.
