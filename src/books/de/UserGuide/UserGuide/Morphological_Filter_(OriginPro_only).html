Morphologischer Filter (nur OriginPro)


Die Funktion ImgMorph ist ein Hilfsmittel für die morphologische Bildverarbeitung. Sie kann auf numerischen Matrizen, Graustufenbildern und binären Bildern verwendet werden. Um diese Funktion zu verwenden:

Wählen Sie Bild: Arithmetische Transformation: Morphologische Filter im Origin-Menü.

Weitere Informationen zu dieser Funktion finden Sie in der Dokumentation der X-Funktion ImgMorph in der Origin-Hilfe für die X-Funktionen. Diese kann über das Befehlsfenster mit folgendem Befehl aufgerufen werden:

help imgMorph
