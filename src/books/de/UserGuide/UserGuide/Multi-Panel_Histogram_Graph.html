Datenanforderungen

Wählen Sie eine oder mehrere Y-Spalten oder einen Bereich aus einer Arbeitsblattspalte.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen > Statistisch: Histogramm mit mehreren Feldern.

oder

Klicken Sie auf die Schaltfläche Histogramm mit mehreren Feldern auf der Symbolleiste 2D-Grafiken.

Vorlage

Keine Vorlage für diesen Diagrammtyp

Hinweise

Der Menübefehl Gestapelte Histogramme zeichnet jeden ausgewählten Datensatz in seinem eigenen Layer in einem Grafikfenster (dabei werden für jeden Layer die gleichen Klasseneinteilungsgrenzen verwendet).
