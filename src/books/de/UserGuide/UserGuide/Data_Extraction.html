Datenextraktion

Gadget-DataExtraction

Das Minitool Datenextraktion kann dabei helfen, die Datenpunkte in einem ROI-Feld im 2D-Diagramm in ein Zielarbeitsblatt zu extrahieren. Es unterstützt die Extraktion von verschiedenen Punktteildatensätzen aus unterschiedlichen ROI-Feldern.

Um das Minitool Datenextraktion zu verwenden:

  • Wählen Sie im Hauptmenü Minitools: Datenextraktion, wenn ein Diagramm aktiv ist. Der Dialog wird geöffnet:

Bedienelemente des Dialogs

Registerkarte Grafische Datenauswahl

Auf dieser Registerkarte können Sie das ROI-Feld benutzerdefiniert anpassen:

Füllfarbe

Legen Sie die Füllfarbe der grafischen Datenauswahl fest.

Wenn Sie Auto auswählen, wird die Füllfarbe automatisch geändert, wenn Sie mehr ROI-Felder mit der Minisymbolleiste hinzufügen. Wenn Sie eine einzelne Farbe ausgewählt haben, teilen alle ROI-Felder die gleiche Füllfarbe.

Punktkennzeichnungsmodus

Legen Sie fest, wie die inneren/äußeren Punkte gekennzeichnet werden, wenn das ROI-Feld hinzugefügt wurde.

  • Kein: Daten werden nicht gekennzeichnet.
  • Innere Punkte färben: Die Punkte innerhalb des ROI-Felds erhalten eine Farbe.
  • Äußere ausblenden: Die Punkte außerhalb des ROI-Felds werden ausgeblendet.

Form

Legen Sie die Form der grafischen Datenauswahl fest. Sie kann entweder rechteckig oder kreisförmig sein oder eine Beliebige Form haben. Siehe Details über diese Formen hier.

Position der Form

Mit dieser Gruppe von Einstellungen können Sie die Position der grafischen Datenauswahl genau setzen. Wenn Kreis in der Auswahlliste Form ausgewählt ist, definieren die Einstellungen unten ein Rechteck, dessen Seiten tangential zum Kreis der grafischen Datenauswahl liegen.

Registerkarte Ausgabe

  • Modus Daten hinzufügen: Legen Sie fest, wie die ausgewählten Datenpunkte ausgegeben werden: Zeilen anhängen, Zeilen mit Abstand anhängen, Neues Blatt öffnen oder Neue Spalten öffnen. Der Standard ist Neue Spalten öffnen.
  • Datenanordnung für mehrere Zeichnungen im gleichen ROI-Feld: Die Auswahloptionen sind identisch mit den obigen. Sie können dies verwenden, um festzulegen, wie die Datenpunkte für mehrere Zeichnungen innerhalb des gleichen ROI-Felds angeordnet werden.
  • Zusätzliche Infos: Legen Sie fest, welche zusätzlichen Informationen zusammen ausgegeben werden.
    • Mappenname: Für den Modus Neues Blatt öffnen oder Neue Spalten öffnen befinden sich Mappenname und Blattname in den Anwenderparametern von Quellmappe, Quellblatt.
    • Blattname: Wie Mappenname. Wenn der Modus Zeilen anhängen oder Zeilen mit Abstand anhängen ist, befinden sich Mappenname und Blattname in der Spalte.
    • Zeilenindex: Der Zeilenindex der Punkte im Quellarbeitsblatt.
    • Modifiziererinfos: Dieses Kontrollkästchen ist standardmäßig aktiviert. Dies bedeutet, dass Sie die entsprechende Zeile der Modifiziererspalten(n) erhalten und keine zusätzliche Spalte, falls es keinen Modifizierer gibt.
  • Arbeitsblattname des Teildatensatzes: Standard ist [%H-Subset]Subset. Wenn der Modus Neues Blatt öffnen ist, heißen die Blattnamen Subset1, Subset2 ...

Elemente der Minisymbolleiste

Wenn Sie im Dialog Datenextraktion: addtool_data_extract auf OK klicken, wird ein ROI-Feld hinzugefügt und eine Minisymbolleiste wird angezeigt:

ROI-Feld hinzufügen

Klicken Sie, um den Dialog ROI erstellen zu öffnen und ein weiteres ROI-Feld hinzuzufügen.

Sie können die Form für das ROI-feld wählen. Die Füllfarbe wird sich automatisch für das neue ROI-Feld ändern, falls Füllfarbe auf Auto gesetzt ist.

ROI modifizieren

Klicken Sie, um den Modus Punkt bearbeiten für das ausgewählte ROI-Feld zu öffnen und weitere Änderungen vorzunehmen. Durch einen Einzelklick auf die leere Fläche verlassen Sie den Modifizierungsmodus.

Daten auswählen

Klicken Sie auf die Ausklappschaltfläche, um die einzelnen Zeichnungen, die aktiviert/deaktiviert sein können, aufzulisten. Die ersten beiden Optionen sind Aktive Zeichnung und Alle Zeichnungen (Trennzeichen zwischen einzelnen Zeichnungen zeigen). Standard ist eine aktive Zeichnung. Es werden Daten für alle ROI-Felder ausgewählt.

Teildatensatzblatt für aktives ROI/Alle ROIs erstellen

Klicken Sie auf diese Schaltfläche, um Daten der ausgewählten Zeichnung im aktiven ROI-Feld/in allen ROI-Feldern in ein bestimmtes Blatt zu kopieren.

Einstellungen

Klicken Sie auf diese Schaltfläche, um die Registerkarte Ausgabe im Dialog Einstellungen Datenextraktion zu öffnen.
