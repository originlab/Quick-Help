Die Registerkarte (Details Zeichnung) Bearbeiten

Die Registerkarte Bearbeiten wird auf Ebene der Daten im Dialogfeld Details Zeichnung für 3D-XYZ-Diagramme angezeigt und steuert das Bearbeitungsverhalten von Diagrammsymbolen und ihren Projektionen. Sie können bei 3D-XYZ-Punktdiagrammen Eigenschaften von Linien, Symbolen und Ankerlinien bearbeiten. Sie können zusätzlich Projektionen auf den XY-, YZ- und ZX-Ebenen anzeigen, von denen alle bearbeitbare Linien-, Symbol- und Ankerlinieneugenschaften haben. Diese Registerkarte bietet Ihnen Optionen zum Bearbeiten aller Eigenschaften einer Gruppe und aller Eigenschaften unabhängig voneinander oder zum Bearbeiten aller Projektionen als eine Gruppe.

Um auf diese Registerkarte zuzugreifen, müssen Sie den XYZ-Datensatz links im Dialogfeld Details Zeichnung markieren.

Bearbeiten der abhängigen Gruppe

Alle zusammen

Aktivieren Sie diese Option, um die Einstellungen auf den Registerkarten Linie, Symbol und Ankerlinien des Dialogfelds Details Zeichnung im (ursprünglichen) 3D-Diagramm und in den drei Projektionen anzuzeigen, zu bearbeiten und anzuwenden. Wenn diese Option ausgewählt ist, können Sie die XY-, ZX- und YZ-Projektionsfelder aktivieren, die Auswahl wird jedoch immer wieder zur ursprünglichen Auswahl zurückkehren, da alle Eigenschaften als eine Gruppe bearbeitet werden.
Vollständig unabhängig

Aktivieren Sie diese Option, um Elemente für jedes 3D-Diagramm getrennt zu bearbeiten (Original + drei Projektionen).
Original unabhängig von Projektionen

Aktivieren Sie diese Option, um die Elemente der 3D-Diagramme (Original) getrennt und alle Projektionen als eine Gruppe zu bearbeiten. Wenn diese Option ausgewählt ist, können Sie die XY-, ZX- und YZ-Projektionsfelder aktivieren, die Auswahl wird jedoch immer wieder zur ursprünglichen Auswahl zurückkehren, da alle Projektionseigenschaften als eine Gruppe bearbeitet werden.

Versatz in Z nach Prozent des Skalierungsbereichs, 0 = unten, 100 = oben

Legen Sie fest, ob das 3D-Diagramm in Z-Richtung verschoben wird, nach Prozent des Skalierungebereichs, 0 = unten,100 = oben.
