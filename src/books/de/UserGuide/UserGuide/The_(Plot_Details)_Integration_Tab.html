Die Registerkarte Integration (Details Zeichnung)

PD-Dialog-Integration-Tab


Diese Registerkarte ist nur verfügbar, wenn Sie ein Feldliniendiagramm erstellen. Sie enthält Bedienelemente, mit denen Sie die Richtung der Integration, die Stufengröße und die maximale Stufen für die Feldlinie benutzerdefiniert festlegen können.

Inhalt

  • 1 Richtung
  • 2 Stufengröße
  • 3 Maximale Stufen

Richtung

Legen Sie die Richtung der Integration vom Anfangspunkt aus fest. Sie können in dieser Auswahlliste Vorwärts, Rückwärts und Beides auswählen.

Stufengröße

Geben Sie die gewünschte Stufengröße in dieses Textfeld ein. Per Standard ist die Stufengröße = 0,1. Wenn Sie einen Wert festgelegt haben, der kleiner ist als die Beschränkung, setzen Sie ihn zurück auf die beschränkte Stufengröße.

Maximale Stufen

Geben Sie die gewünschten maximalen Stufen in dieses Textfeld ein.
