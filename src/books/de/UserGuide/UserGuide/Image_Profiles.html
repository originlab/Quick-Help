Bildprofile


Das Hilfsmittel Bildprofil bietet einen schnellen Weg zur Analyse der Bilddaten und erstellt Bildprofile. Sie können Profile für horizontale, vertikale oder beliebige Liniensegmente der Bilddaten dynamisch.

Um ein Bildprofildiagramm zu erzeugen,

wählen Sie Diagramme: Bild: Profile im Origin-Menü.

Beachten Sie, dass das Eingabebild für das Bildprofildiagramm graustufig oder 8-Bit farbig sein sollte. Wenn die Farbtiefe des Eingabebilds dem nicht entspricht, müssen Sie das Bild zuerst konvertieren.
