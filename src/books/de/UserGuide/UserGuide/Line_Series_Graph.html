Hinweise

Dieser Diagrammtyp wurde für Origin 2017 verbessert. Er basiert nun auf dem Boxdiagramm. Die Zeichnung kann einfacher benutzerdefiniert angepasst werden, einschließlich Festlegen der Zeichnungsattribute über andere Spalten aus dem Arbeitsblatt.

  • Es wird kein verborgenes Arbeitsblatt erstellt.
  • Sie sind nicht länger auf das Zeichnen von zwei oder drei Y-Spalten beschränkt.
  • Mehrere Y-Spalten werden als Zeichnungsgruppe betrachtet, wodurch es einfacher wird, sie anzupassen.
