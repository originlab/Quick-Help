Die Registerkarte Linie (Objekteigenschaften)

Ob-Prop-Line-tab

 

Auf der Registerkarte Linie können Sie Farbe, Breite, Typ und Transparenz für das Linienobjekt festlegen.

Informationen zum Definieren von benutzerdefinierten Strichlinien (Element Typ) finden Sie auf der Hilfeseite zur Registerkarte Linie im Dialog Details Zeichnung.

Für das Objekt der geraden Linie (gerade Linie mit Pfeil) können Sie die Ausrichtung auch auf Vertikal oder Horizontal setzen und dies auf den Gesamten Layer, Alle Layer oder die Gesamte Seite strecken.

Aktivieren Sie das Kontrollkästchen Als Standardfestlegen, um diese Registerkarteneinstellungen auf das aktuell ausgewählte Objekt und alle zukünftigen Objekte des gleichen Typs anzuwenden.

Hinweis: Das Bedienelement Transparenz ist das gleiche wie auf der Registerkarte Pfeil. Sie teilen den gleichen Wert.

Wenn Sie den Prozentsatz der Transparenz für die Linie ändern, wird der Pfeil auf den gleichen Prozentsatz für die Transparenz angewendet und umgekehrt.
