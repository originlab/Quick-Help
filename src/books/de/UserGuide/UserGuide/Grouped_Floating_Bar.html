Datenanforderungen

Wählen Sie mindestens 4 Y-Spalten aus. Verwenden Sie eine Spaltenbeschriftungszeile oder eine Anzahl von Spalten, um die Gruppierung zu entscheiden. Jeder frei bewegliche Balken beginnt ab der ersten Spalte und endet bei der letzten Spalte innerhalb der Gruppe.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen: Kategorial: Gruppierte freie Balken.

In dem sich öffnenden Dialog plot_gfloatbar wählen Sie den Gruppenbereich und das Gruppendesign aus, um die gruppierten frei beweglichen Balken zu erstellen.

Der Dialog "plot_gfloatbar"

  • Eingabe: Legen Sie den Eingabebereich fest, um das Diagramm zu zeichnen. Standardmäßig werden die markierten Spalten aus dem Quellarbeitsblatt eingegeben.
  • Teilgruppe: Legen Sie fest, wie die Spalten in Teilgruppen aufgeteilt werden. Sie können
    Nach Größe auswählen, um die Spalten mit einer konstanten Anzahl von Spalten, die in das Feld Größe eingegeben wird, in Teilgruppen aufzuteilen.
    Nach Spaltenbeschriftung auswählen, um die Spalten gemäß der Werte in der Spaltenbeschriftungszeile, die in der folgenden Auswahlliste Spaltenbeschriftung festgelegt sind, in Teilgruppen aufzuteilen.
  • Diagrammtyp: Legen Sie fest, dass frei bewegliche Spalten oder Balken gezeichnet werden.
  • Ausgabe: Legen Sie das Ausgabeblatt fest, um die gestapelten Daten zu speichern.

Vorlage

gFloatBar.otpu, gFloatCol.otpu (installiert im EXE-Ordner von Origin)

Hinweise

Dieser Diagrammtyp ist eine erweiterte Version des frei beweglichen Balkendiagramms, gezeichnet aus Rohdaten (nicht Indexdaten) mit Gruppierungsmechanismen wie Gruppierte Boxdiagramme - Rohdaten.

Wie Sie aus diesem Flussdiagramm zum Zeichnen eines gruppierten frei beweglichen Balkendiagramms aus Rohdaten ersehen können, wird die Einstellung Teilgruppe im Dialog plot_gfloatbar verwendet, um über die Werte zu entscheiden, die im gleichen Balken gestapelt werden (z. B. können alle Zellen "A-R" in der gleichen Zeile zusammen gestapelt werden). Sie können die Einstellungen Teilgruppe auf der Registerkarte Abstände im Dialog Details Zeichnung verwenden, um Teildatensätze innerhalb der X-Achse (wie Bereich1, Bereich2 und Bereich3 in diesem Beispiel) festzulegen.
