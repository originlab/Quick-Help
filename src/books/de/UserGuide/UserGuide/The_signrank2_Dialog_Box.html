Alternativhypothese
  • Median1 <> Median2
Führt einen beidseitigen Wilcoxon-Vorzeichen-Rangtest bei verbundenen Stichproben durch.
  • Median1 > Median2
Führt einen oberen Wilcoxon-Vorzeichen-Rangtest bei verbundenen Stichproben durch.

Führt einen unteren Wilcoxon-Vorzeichen-Rangtest bei verbundenen Stichproben durch.
