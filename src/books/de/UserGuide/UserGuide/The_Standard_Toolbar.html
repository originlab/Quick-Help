Die Symbolleiste Standard

Standard-Toolbar


Die Symbolleiste Standard gewährleistet einen schnellen Zugriff auf das Öffnen, Speichern und Drucken von Objekten. Klicken Sie auf die Schaltfläche Neue Arbeitsmappe, um eine neue Arbeitsmappe zu erstellen. Die Schaltfläche Neu berechnen auf der Symbolleiste Standard zeigt ebenfalls an, wenn eine Aktualisierung notwendig ist.
