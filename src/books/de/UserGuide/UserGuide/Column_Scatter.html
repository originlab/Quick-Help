Datenanforderungen

Wählen Sie eine oder mehrere Spalten oder einen Datenbereich aus mindestens einer Spalte aus.

Diagramm erstellen

Wählen Sie die Daten aus und:

  • Wählen Sie im Origin-Menü Zeichnen: Einfache 2D: Punkt-Säulendiagramm.

ODER

  • Klicken Sie auf die Schaltfläche Punkt-Säulendiagramm auf der Symbolleiste 2D-Grafiken.

Vorlage

ColumnScatter.otp (installiert im Origin-Programmordner)

Hinweise

Tatsächlich ist das Punkt-Säulendiagramm ein Boxdiagramm, dessen Boxtyp auf Daten gesetzt ist, und Zufällig verstreute Punkte (Jitter-Punkte) auf der Registerkarte Daten ausgewählt wurde.

Um ein gruppiertes Punkt-Säulendiagramm zu erstellen, können Sie ein zuerst ein gruppiertes Boxdiagramm mit Indexdaten oder Rohdaten erstellen und dann Box_Column Scatter in der Auswahlliste Diagrammdesigns auswählen.
