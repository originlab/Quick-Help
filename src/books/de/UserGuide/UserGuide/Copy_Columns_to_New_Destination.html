Spalten in neue Ziele kopieren

copy-columns

Die Funktion Kopieren-Einfügen ist eine direkte Methode, um Daten in einer oder mehreren Spalten in neue Spalten zu kopieren. Um aber das Spaltendatenformat oder Beschriftungen bzw. einen angewendeten Datenfilter zu kopieren, müssen Sie die X-Funktion colcopy verwenden. Über die Bedienoberfläche haben Sie Zugriff auf diese X-Funktion, indem Sie im Hauptmenü Worksheet: Spalten kopieren in: Dialog öffnen... auswählen (oder mit der rechten Maustaste auf eine Spalte klicken und Spalten kopieren in... im Kontextmenü wählen).

Die folgende Tabelle fasst die Bedienelemente des Dialogs zusammen:

Neu berechnen Legen Sie den Modus der Neuberechnung für diese Operation fest.
Quellspalte(n) Legen Sie die Eingabedaten fest.
Zielspalte(n) Legen Sie fest, wo die eingefügten Daten ausgegeben werden.
Daten kopieren Legen Sie fest, ob Daten kopiert werden oder nicht.
Abtastintervall kopieren Legen Sie fest, ob das Abtastintervall der Daten kopiert wird.
Verborgene Zeilen ignorieren Legen Sie fest, ob verborgene Zeilen ignoriert werden sollen.
Maskierung kopieren Legen Sie fest, ob der Maskierungsstatus der Datenpunkte kopiert wird.
Format kopieren Legen Sie fest, ob das Spaltenformat und alle Zellenformate der Daten kopiert werden.
Beschriftung kopieren Legen Sie fest, ob die Spaltenbeschriftungszeilen kopiert werden.
Ausgabe sortieren Die Ausgabe wird nach Werten in einer Referenzspalte sortiert. Standard ist unsortiert. Sie können entscheiden, ob die Werte in auf- oder absteigender Reihenfolge in einer Referenzspalte mit Namen sortiert werden.

Wenn der Neuberechnungsmodus für die Operation Spalten kopieren in auf Auto oder Manuell gesetzt ist, wird ein grünes Schlosssymbol in der/n Zielspalte/n angezeigt. Durch einen Mausklick auf das Schloss mit der linken Maustaste wird ein Menü geöffnet. Wenn die Zielspalte(n) sich nicht dem gleichen Arbeitsblatt befindet/n wie die Quellspalten, werden in diesem Menü drei Optionen angeboten, die mit dem Arbeitsblattfilter verbunden sind.

Anfänglich ist nur der Menübefehl Arbeitsblattfilter: Blockieren verfügbar. Zu diesem Zeitpunkt löst eine Änderung der Filterbedingung des Quellarbeitsblatts eine Aktualisierung der Daten in der/n Zielspalte/n aus. Wenn Sie auf Arbeitsblattfilter: Blockieren klicken, werden die Zielarbeitsblattspalten gesperrt und Änderungen an der Filterbedingung des Quellarbeitsblatts lösen keine Aktualisierung der Zielspalte(n) mehr aus.

Arbeitsblattfilter: Blockieren

Wenn diese Option ausgewählt ist, wird das Ergebnis für eine Änderung der Datenfilterbedingung bezüglich der Quellspalte(n) gesperrt. Das heißt, wenn die Filterbedingung im Quellspaltenarbeitsblatt geändert wird, wird keine Aktualisierung der Zielspalte(n) ausgelöst.
Arbeitsblattfilter: Neu laden

Diese Option ist nur verfügbar, wenn Arbeitsblattfilter: Blockieren bereits ausgewählt ist. Die Datenfilterbedingung wird von den Quellspalten für die Zielspalten neu geladen. Das heißt, nachdem Sie die Bedingung für den Datenfilter der Quellspalte(n) geändert haben, klicken Sie auf diese Option, um die automatische Aktualisierung der gesperrten Zielspalte(n) auszulösen, um die gleiche Filterbedingung auch auf die Zielspalte(n) anzuwenden.
Arbeitsblattfilter: Auf Quellarbeitsblatt anwenden

Diese Option ist nur verfügbar, wenn Arbeitsblattfilter: Blockieren bereits ausgewählt ist. Die ursprüngliche Datenfilterbedingung wird auf die Quellspalte(n) angewendet. Das heißt, nachdem Sie die Datenfilterbedingung des Quellarbeitsblatts geändert haben, klicken Sie auf diese Option, und die jüngste Datenfilterbedingung, die von den Quellspalten auf die Zielspalten angewendet wurde, wird für das Quellarbeitsblatt verwendet. Beachten Sie, dass, wenn Sie einen Datenfilter direkt auf die Zielspalte(n) angewendet haben, er nicht wieder auf die Quelle angewendet wird.
