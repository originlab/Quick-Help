Mehrere Kurven unabhängig anpassen

MultiCurve-IndepFit

Diese Funktion wird zum Anpassen individueller Datensätze unter Verwendung des gleichen Modells verwendet. Der Anpassungsprozess von mehreren Kurven kann gleichzeitig oder aufeinander folgend, aber vollkommen unabhängig voneinander stattfinden. Sie können festlegen, ob die Ergebnisberichte für jede Kurve separat erzeugt oder in einem Arbeitsblatt durch Wählen verschiedener Anpassungsmodi für mehrere Datensätze zusammengefasst werden sollen.
