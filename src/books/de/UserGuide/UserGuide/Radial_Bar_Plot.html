Data Requirements

Select at least one pair of XY columns.

Creating the Graph

Select Plot > Specialized: Radial Bar Plot/Radial Stacked Bar Plot

Template

RadialBar.otpu ; RadialStackedBar.otpu (All above are installed to the Origin program folder).

Notes

  • These two plot types are variants of Stacked Radial Plot.
  • Radial Bar plot has labels by default. And, the Spoke Style is Sector by default.
  • The Stack Offset for the Radial Stacked Bar plot is Cumulative.
