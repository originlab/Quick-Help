Datenanforderungen

Wählen Sie zwei Y-Spalten (oder einen Bereich aus zwei Y-Spalten) aus. Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen: Einfache 2D: Blasendiagramm.

oder

Klicken Sie auf die Schaltfläche Blasendiagramm auf der Symbolleiste 2D-Grafiken.

Vorlage

Bubble.OTP (im Origin-Programmordner installiert)

Hinweise

Für jede Zeile legen die beiden Y-Werte die Anzeige der Datenpunkte fest. Die erste Y-Spalte liefert die Y-Datenpunktwerte. Die zweite Y-Spalte steuert die Symbolgröße der Datenpunkte in Punkten.

Standardmäßig wird eine Blasenskala zum Blasendiagramm bei der Erstellung hinzugefügt. Sie dient als Legende.
