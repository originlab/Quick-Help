Datenanforderungen

Wählen Sie sechs Spalten (oder einen Bereich aus diesen Spalten), die als XYZXYZ zugewiesen wurden.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie Zeichnen: Spezialisiert: Ternäres Vektordiagramm.

Vorlage

TernaryVector.otpu (installiert im Origin-Programmordner).

Notizen

Das ternäre Vektordiagramm ist ein spezielles Vektordiagramm, das im ternären Koordinatensystem gezeichnet wird, ähnlich dem polaren Vektordiagramm.

Eingabedaten in ternäre Koordinaten zeichnen

Da die Vektoren in einem ternären Koordinatensystem gezeichnet werden, wird angenommen, dass die Eingabedaten nach Gruppen normalisiert sind, das bedeutet, dass die Summe der absoluten Zellenwerte in jeder Zeile von jeder XYA-Gruppe gleich 1 oder 100: |x|+|y|+|z|=1 oder 100 ist. Mehr Einzelheiten zur ternären Normalisierung können Sie auf der Seite Ternärdiagramm nachlesen.

Um ternäre Achsen benutzerdefiniert anzupassen, können Sie zur

  • Registerkarte Ternär im Dialog Details Zeichnung gehen.

XYZXYZ-Spalten als Vektoren zeichnen

Die am weitesten links stehenden X-,Y- und Z-Spalten bestimmen die ternäre XYZ-Koordinaten am Ende des Vektors. Der zweite Satz X-, Y- und Z-Spalten bestimmt die ternären XYZ-Koordinate der Vektorspitze.

Um die Vektoren benutzerdefiniert anzupassen, können Sie zur

  • Registerkarte Vektor im Dialog Details Zeichnung gehen.
  • Registerkarte Beschriftung im Dialog Details Zeichnung gehen.
