Umwandeln von Datenmatrix in Bild


Die X-Funktion m2img wandelt ein numerisches Matrixobjekt in ein Graustufenbild mit Abbildung um. Sie können dann andere Hilfsmittel zur Bildverarbeitung verwenden, um das Bild zu verarbeiten. Um diese Funktion zu verwenden:

Wählen Sie Bild: Konvertierung: In Bild umwandeln im Origin-Menü.

Weitere Informationen über diese Funktion finden Sie in der Dokumentation der X-Funktion m2img in der Origin-Hilfe für die X-Funktionen. Diese kann über das Befehlsfenster mit folgendem Befehl aufgerufen werden:

help imgC2gray
