Form der Eingabedaten

Indiziert Eine Spalte enthält die Gruppenvariable und eine andere Spalte enthält die entsprechenden Antwortdaten.
Roh Die Daten für jeden Datenbereich sind in getrennten Spalten enthalten.

Eingabe

  • Eingabedatenbereich, wenn Eingabedatenform = Index
Gruppenbereich Die Gruppierungsspalte Beachten Sie, dass die Gruppierungsspalte als Kategorial gesetzt wird, wenn es sich um eine Textspalte handelt.
Datenbereich Die Datenspalte
  • Eingabedatenbereich, wenn Eingabedatenform = Roh Die Daten für jede Gruppe sind in getrennten Spalten enthalten. Wählen Sie mindestens 2 Spalten aus.
