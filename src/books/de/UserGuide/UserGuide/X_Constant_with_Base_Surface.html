Datenanforderungen

  • Arbeitsblatt: Ein Block von Arbeitsblattzellen (virtuelle Matrix)

oder

  • Matrix: Eine Matrix von Z-Werten

Diagramm erstellen

Aktivieren Sie das Matrixblatt oder wählen Sie die gewünschten Daten im Arbeitsblatt aus.

Wählen Sie im Menü Zeichnen: 3D: 3D X konstant mit Basis.

oder

Klicken Sie auf die Schaltfläche 3D X konstant mit Basis der Symbolleiste 3D- und Konturdiagramme.

Vorlage

  • glXCONST.OTP (OpenGL)
  • XCONST.OTP

(installiert im Origin-Programmordner)

Hinweise

Der Z-Wert bestimmt eine Oberfläche von Gitternetzlinien parallel zur Y-Achse. Die vordere Oberfläche zeigt eine Füllfarbe. X- und Y-Seitenwände sind ebenfalls aktiviert.
