Brief Description

Five parameters logistic function.

Sample Curve

Parameter

Number: 5

Names: Amin, Amax, x0, h, s

Meanings: Amin = Lower Asymmetry, Amax = Upper Asymmetry, x0 = X of Half Y, h = Hill Slope, s = Control Factor

Lower Bounds: s > 0.0

Upper Bounds: none

Script Access
