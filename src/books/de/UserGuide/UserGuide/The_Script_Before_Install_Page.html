Die Seite Skript vor Installation

FitFuncBuilder-Python-PackCheck

Bedienelemente des Dialogs

Python-Paket-Prüfung
(Labtalk-Skript)		 Fügen Sie Skript zu diesem Feld hinzu, um Anwender, die benutzerdefinierte Python-Fitfunktionen aus erforderlichen Python-Paketen installieren, aufzufordern, die Fitfunktion zu verwenden. Wenn ein Endanwender die FDF-Datei auf den Origin-Arbeitsbereich zieht, wird das Skript ausgeführt. Die Seite kann nicht leer sein (nur Python-Funktionen).
  • Wenn die erforderlichen Pakete nicht installiert sind, wird der Endanwender aufgefordert, die Installation des Pakets zuzulassen.
  • Wenn die erforderlichen Pakete bereits installiert sind, wird der Endanwender aufgefordert, die Fitfunktion zu einer Kategorie hinzuzufügen.

Es folgt ein Beispiel. Ersetzen Sie Ihre erforderlichen Pakete innerhalb der Klammern und doppelten Anführungszeichen nach Bedarf:

[BeforeInstall] if(Python.chk("pandas cv2(opencv-python)") > 1) return 1;//should not install FDF return 0;//proceed
