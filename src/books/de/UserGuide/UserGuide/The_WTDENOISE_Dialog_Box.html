Schwellenwerttyp

Legt die Methode fest, um den Schwellenwert zum Kürzen der Detailkoeffizienten von jeder Stufe zu bestimmen.

Benutzerdefiniert

Der Schwellenwert wird mit dem Bedienelement Schwellenwert jeder Stufe (%) eingestellt.

sqtwolog

Der Schwellenwert wird aus der Länge der Detailkoeffizienten berechnet.
Stufe mit Schwellenwert

Legt die Anzahl der Stufen fest, für die der Schwellenwert der Detailkoeffizienten bestimmt werden muss. Eine große Stufe hat mehr zu entfernendes Rauschen zur Folge; die Wahrscheinlichkeit jedoch, dass das nützliche Signal verzerrt wird, wird ebenfalls größer.
Schwellenwert von jeder Stufe (%)

Diese Option ist nur verfügbar, wenn der Schwellenwerttyp Benutzerdefiniert ist. Er legt einen Schwellenwert in Prozent fest.
