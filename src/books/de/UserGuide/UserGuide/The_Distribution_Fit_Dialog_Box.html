Eingabedaten

Hilfe zum Festlegen von Bereichen finden Sie hier: Eingabedaten festlegen

Datenbereich

Der Eingabedatenbereich Sie können eine oder mehrere Spalten eingeben.
Gruppierungsbereich

Der Datenbereich, der die Gruppeninformationen enthält Verschiedene Gruppierungswerte weisen darauf hin, dass die Daten in den entsprechenden Zellen aus verschiedenen Gruppen sind. Sie können nur eine Spalte für den Gruppierungsbereich eingeben.

Grundlagen der Statistik

Legen Sie die Ausgabe der deskriptiven Statistik fest. Einzelheiten zur Berechnung finden Sie auf der Seite Spaltenstatistik.

Quantile

Legen Sie die Werte der Quantile fest. Dazu gehören Minimum, 1. Quantil (Q1), Median, 3. Quantil (Q3), Maximum, Interquartilbereich (IQR = Q3 - Q1), Spannweite (Maximum - Minimum).
Momente

Legen Sie fest, ob die Momente ausgegeben werden sollen. Dazu gehören N gesamt, N fehlend, Mittelwert, Standardabweichung, SE des Mittelwerts, unteres/oberes 95% KI des Mittelwerts.

Weitere Momente

Legen Sie fest, ob weitere Momente ausgegeben werden sollen, z.B. Varianz, Summe, Schiefe, Kurtosis, unkorrigierte Summe der Quadrate, korrigierte Summe der Quadrate, Variationskoeffizient, Modalwert.

Grundlegende Tests

Mittelwertetest

Wählen Sie eine Option aus der Auswahlliste, um einen t-Test oder Z-Test für dem Mittelwert durchzuführen oder den Mittelwertetest nicht durchzuführen.

t-Test/Z-Test

Testmittelwert Der zu testende hypothetische Mittelwert Wenn Sie einen Wert eingeben, werden die Null- und Alternativhypothese entsprechend angepasst.
Standardabweichung Legen Sie den Wert der Standardabweichung fest, der in dem Z-Test verwendet wird. Diese Option ist nur verfügbar, wenn der Z-Test ausgewählt ist.
Nullhypothese Die zu testende Nullhypothese

(Mittelwert = Testmittelwert)
Alternativhypothese Die Alternativhypothese

Optionen:

Mittelwert <>Testmittelwert

Wird für beidseitigen Test verwendet. Diese Option testet, ob der Stichprobenmittelwert sich von dem hypothetischen Mittelwert unterscheidet.

Mittelwert >Testmittelwert

Wird für den einseitigen Test nach oben verwendet. Diese Option testet, ob der Stichprobenmittelwert größer ist als der hypothetische Mittelwert.

Mittelwert <Testmittelwert

Wird für den einseitigen Test nach unten verwendet. Diese Option testet, ob der Stichprobenmittelwert kleiner ist als der hypothetische Mittelwert.
Konfidenzintervall(e) Legen Sie fest, ob das/die Konfidenzintervall(e) für den Mittelwert berechnet wird/werden.

Einzelheiten zur Berechnung finden Sie unter Algorithmen: Verteilungsanpassung.
Konfidenzniveau(s) in % Legen Sie die Konfidenzniveaus (durch Komma getrennt) fest, (1-)%. Diese Option ist verfügbar, wenn Konfidenzintervall(e) aktiviert wird.

Chi-Quadrat-Test auf Varianz

Testvarianz Die zu testende hypothetische Varianz
Nullhypothese Die zu testende Nullhypothese (d.h. Stichprobenvarianz = Testvarianz).
Alternativhypothese Optionen:
Varianz <>

Die Stichprobenvarianz unterscheidet sich von der hypothetischen Varianz.

Varianz >

Die Stichprobenvarianz ist größer als die hypothetische Varianz (einseitiger Test nach oben).

Varianz <

Die Stichprobenvarianz ist kleiner als die hypothetische Varianz (einseitiger Test nach unten).
Konfidenzintervall(e) Legen Sie fest, ob die Konfidenzintervalle für den Mittelwert berechnet werden.
Konfidenzniveau(s) in % Legen Sie die Konfidenzniveaus (durch Komma getrennt) fest (%). Diese Option ist nur verfügbar, wenn Konfidenzintervall(e) aktiviert ist.

Verteilungen

Verteilungstyp

Legen Sie fest, ob die stetige Verteilung oder diskrete Verteilung durchgeführt werden soll.

Stetige Verteilung Im Zweig Verteilungen sind 5 Typen für stetige Verteilungen verfügbar.
Diskrete Verteilung Im Zweig Verteilungen sind 2 Typen für diskrete Verteilungen verfügbar.

Verteilungen

Wählen Sie eine oder mehrere Verteilungen, die für die Eingabedaten angepasst werden.

Stetige Verteilung

Normal LogNormal Weibull Exponential Gamma

Diskrete Verteilung

Binomial Poisson

Diagramme

Boxdiagramm Das Boxdiagramm wird im Ergebnisblatt ausgegeben.
Histogramm Das Histogramm wird im Ergebnisblatt ausgegeben.
Funktionsdiagramm der kumulativen Verteilung (CDF-Diagramm) Das CDF-Diagramm wird im dem Ergebnisblatt ausgegeben.
Wahrscheinlichkeitsdiagramm Das P-P-Diagramm wird im dem Ergebnisblatt ausgegeben.
Signifikanzniveau Legen Sie das Konfidenzniveau für die Konfidenzbänder des Wahrscheinlichkeitsdiagramm (P-P) fest.
Diagramme des gleichen Typs in einem Graph anordnen Wählen Sie diese Option, um Zeichnungen aus mehreren Datenbereichen in einer Grafik anzuordnen. Der Vergleich mehrerer Datenbereiche wird hierdurch einfacher. Bitte beachten Sie, dass die Histogramme in der Grafik gestapelt werden und die gleiche Klassengröße haben.

Anpassungsgüte

Kolmogorov-Smirnov Der Kolmogorov-Smirnov-Test wird durchgeführt.
Kolmogorov-Smirnov (Modifiziert) Der modifizierte Kolmogorov-Smirnov-Test wird durchgeführt.
Anderson-Darling Der modifizierte Anderson-Darling-Test wird durchgeführt.

Ausgabeeinstellungen

Bericht der Verteilungsanpassung Legen Sie das Arbeitsblatt fest, in dem die Ergebnisse der Verteilungsanpassung ausgegeben werden.
Daten zeichnen Legen Sie ein Arbeitsblatt fest, in dem gezeichneten Daten ausgegeben werden (Origin erzeugt tabellarische Daten, wenn Zeichnungen erstellt werden).
