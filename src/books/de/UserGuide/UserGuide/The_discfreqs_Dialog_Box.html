Zu berechnende Eigenschaften

Anzahl

Die Zählung wird bei jedem diskreten Wert vorgenommen.
Kumulative Anzahl

Es wird die kumulative Anzahl von jedem diskreten Wert berechnet.
Relative Häufigkeit

Es wird die relative Häufigkeit von jedem diskreten Wert berechnet. Weitere Informationen zum Algorithmus finden Sie hier.
Kumulative Häufigkeit

Es wird die kumulative Häufigkeit von jedem diskreten Wert berechnet. Weitere Informationen zum Algorithmus finden Sie hier.
Häufigkeit in

Legen Sie die Form der Häufigkeit fest. Diese Option ist nur verfügbar, wenn entweder das Kontrollkästchen Relative Häufigkeit oder Kumulative Häufigkeit aktiviert ist.

  • Bruchteil
    Die Häufigkeit wird mit einer Bruchzahl angegeben.
  • Prozent
    Die Häufigkeit wird mit einer Prozentzahl angegeben.

Groß-/Kleinschreibung beachten

Legen Sie fest, ob Groß- und Kleinschreibung berücksichtigt wird, wenn Origin entscheiden muss, ob zwei Werte gleich sind.

Fehlende Werte ausschließen

Aktivieren Sie diese Option, um sowohl fehlende Werte und leere Werte auszuschließen.

Kategorien mit Anzahl Null zeigen

Legen Sie fest, ob die Kategorien aufgelistet werden sollen und, wenn eine Kategorie nicht gefunden wird, eine 0 als Anzahl der Daten gezeigt wird.

Daten sortieren nach

Daten sortieren nach

Diese Option steuert die Sortierungsordnung der Ausgabespalten:

  • Aufsteigende Anzahlen
  • Absteigende Anzahlen
  • Aufsteigende Werte
  • Absteigende Werte
  • Kategorische Ordnung

Wenn Kategorische Ordnung ausgewählt ist, folgt die Ausgabespalte der kategorischen Ordnung, die auf der Registerkarte Kategorien (Spalteneigenschaften) benutzerdefiniert angepasst wird.

Ergebnisse ausgeben

Legen Sie fest, wo die Ergebnisse ausgegeben werden. Der Identifizierer im Ergebnisblatt verwendet den Langnamen der Eingabespalten, wenn es keinen duplizierten Langnamen gibt.
