Neu berechnen

Eingabe

Legt den zu glättenden XY-Eingabedatenbereich fest.
Hinweis: Ab Origin 2024 können mehrere Datensätze im gleichen Arbeitsblatt in einem Verarbeitungsschritt geglättet werden, zum Beispiel die Datenspalten XYYY und XYXYXY.

Siehe Eingabedaten festlegen.
Methode

Legt die zu verwendende Glättungsmethode fest. Die Standardmethode ist Savitzky-Golay.

  • Gleitender Durchschnitt: Diese Methode nimmt den Durchschnitt einer bestimmten Anzahl (die Sie festlegen) von Datenpunkten um jeden Punkt Ihrer Daten herum und ersetzt diesen Punkt mit dem neuen Durchschnittswert.
  • Savitzky-Golay: Diese Methode führt eigentlich eine lokale polynomielle Regression durch. Sie eignet sich am besten dafür, Rauschen zu entfernen, ohne die Gesamtform des Signals zu beeinträchtigen. Die ursprünglichen Datenwerte werden mit angepassten Werten ersetzt.
  • Perzentilfilter: Diese Methode wird auch als Rangordnungsfilter bezeichnet. Sie sucht im Wesentlichen das festgelegte Perzentil der Datenpunkte in dem Datenfenster um jeden Punkt Ihrer Daten herum und ersetzt diesen Punkt mit dem Perzentil. Sie eignet sich am besten zum Entfernen von Schrotrauschen.
  • FFT-Filter: Diese Methode basiert auf dem Tiefpassfilter der FFT. Sie sollte verwendet werden, wenn die Rauschfrequenz höher ist als das wahre Signal. Sie entfernt im Wesentlichen die hochfrequenten Komponenten mit einem parabolischen Fenster. Frequenzen, die nicht geringer sind als die Abschneidefrequenz, werden verworfen.
  • LOWESS und LOESS: Der Name Lowess bzw. Loess ist die Abkürzung für "locally weighted scatter plot smoothing" - die lokal gewichtete Glättung von Punktdiagrammen. Am besten verwendet man sie im Fall einer großen Anzahl von Datenpunkten. Bitte lesen Sie weitere Einzelheiten unter diesem Algorithmus.
  • Binomial: Der Binomialfilter ist ein gewichteter Filter des gleitenden Durchschnitts, dessen Gewichtung von den binomialen Koeffizienten abgeleitet wird. Es handelt sich hierbei um einen Tiefpassfilter zum Filtern von hochfrequentem Rauschen.
Interpolation nach gewichteter Durchschnitt

Wenn für die Methode der Gleitende Durchschnitt ausgewählt ist, aktivieren Sie dieses Kontrollkästchen, um den gewichteten Durchschnitt zu verwenden.
Bereich festlegen mit

Wenn die Methode entweder Lowess oder Loess ist, können Sie den Bereich mit diesen drei Optionen festlegen:

  • Punkte des Fensters: Die Punkte des Fensters werden für den Bereich verwendet.
  • Anteil: Der Anteil (0-1) wird für den Bereich verwendet.
  • X-Wert: Der festgelegte X-Wert wird für den Bereich verwendet.
Punkte des Fensters

Wenn Punkte des Fensters für Bereich festlegen mit ausgewählt ist, legen Sie die Anzahl der Punkte im beweglichen Fenster fest. Ansonsten kann dieser Parameter verwendet werden, um die Abschneidefrequenz für den FFT-Filter zu steuern. Größere Werte erzeugen eine größere Glättung. Es muss ein positiver Wert sein, der nicht null ist. Sehen Sie sich den Abschnitt zum Algorithmus für die Berechnung der Abschneidefrequenz an.
Datenbereich (0-1)

Wenn Anteil für Bereich festlegen mit ausgewählt ist, steuern Sie den Anteilswert für den Bereich. Größere Werte erzeugen eine größere Glättung.
X-Wert

Wenn X-Wert für Bereich festlegen mit ausgewählt wird, legen Sie den X-Wert fest. Standardmäßig ist dieser X-Wert 0,1*(x max-x min).
Grenzbedingung

Nicht verfügbar, wenn unter Methode die Option FFT-Filter/Lowess/Loess ausgewählt ist.

  • Keine: Die Enden werden nicht aufgefüllt.
  • Reflektieren: Die Daten werden durch Reflexion des Signals an beiden Enden aufgefüllt.
  • Wiederholen: Die Daten werden aufgefüllt, indem die Endpunkte an beiden Enden wiederholt werden.
  • Periodisch: Die Daten werden als periodisches Signal behandelt.
  • Extrapolieren: Die lineare Extrapolation wird verwendet, um das Signal an beiden Enden aufzufüllen.
Polynomielle Ordnung

Verfügbar, wenn unter Methode die Option Savitzky-Golay ausgewählt ist.
Perzentil

Diese Option bestimmt das Perzentil, das bei der Perzentilfilterung oder Rangordnungsfilterung verwendet wird. Standardmäßig sind es 50%, was dem Ersetzen des Signalpunkts mit dem Medianwert im Datenfenster entspricht. Nur verfügbar, wenn unter Methode der Rangordnungsfilter ausgewählt ist.
Abschneidefrequenz

Ein nur lesbares Textfeld zeigt die Abschneidefrequenz des Tiefpassfilters. Dieser Parameter ist nur verfügbar, wenn die Methode der FFT-Filter oder Binomial ist.
Abtastintervall

Abtastintervall Deaktivieren Sie das Kontrollkästchen Auto, um das Abtastintervall manuell festzulegen. Diese Option ist nur verfügbar, wenn unter Methode die Option Periodisch ausgewählt ist.
Ordnung

Der Standardwert ist 1. Deaktivieren Sie das Kontrollkästchen Auto, um die Ordnung manuell festzulegen. Diese Option ist nur verfügbar, wenn unter Methode die Option Periodisch ausgewählt ist.
Ausgabe

Bestimmen Sie, wo das geglättete Signal ausgegeben werden soll. Siehe Ergebnisse ausgeben.
