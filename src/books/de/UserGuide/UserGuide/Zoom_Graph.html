Datenanforderungen

Wählen Sie mindestens eine Y-Datenspalte (oder einen Bereich aus mindestens einer Spalte) aus. Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen > Mehrere Felder/Achsen: Vergrößern.

oder

Klicken Sie auf die Schaltfläche Zoom auf der Symbolleiste 2D-Grafiken.

Vorlage

Zoom1.OTP (im Origin-Programmordner installiert)

Hinweise

Das Zoom-Diagramm übernimmt jeden Datensatz in beide Layer als Liniendiagramm. Es kann auch über die kontextabhängigen Menüs oder das Dialogfeld Details Zeichnung in ein Punkt-, Punkt-Linien- oder Säulen-/Balkendiagramm geändert werden.

Ein Rechteck in dem oberen Layer wurde kodiert (d.h., es verfügt über ein Skript), so dass der durch seine Grenzen definierte Bereich automatisch im unteren Layer angezeigt wird, wenn es verschoben wird. Das Rechteck kann unterschiedliche Größen haben, um den angezeigten Bereich weiter einzuschränken bzw. zu erweitern.

Sie können die Schaltfläche Achsenskalierung vergrößern verwenden, um einen vergrößerten Teil Ihres 2D-Diagramms in ein neues Fenster zu kopieren. Dies funktioniert auch für Grafiken, die mehrere verknüpfte Layer haben, solange die Layer sich überschneiden und eine gemeinsame Achse teilen (für % des verknüpften Layers ist Links/Oben auf 0 gesetzt und Breite/Höhe ist auf 100 gesetzt plus Gerade (1 zu 1) ist mit dem Hauptlayer verknüpft).

  1. Drücken Sie bei aktiver Grafik die Strg-Taste und klicken Sie dann auf die Schaltfläche Achsenskalierung vergrößern (Symbolleiste Hilfsmittel). Ziehen Sie auf Ihrem 2D-Diagramm ein Rechteck auf.
  • Verschieben Sie das Rechteck auf Ihrem ursprünglichen Diagramm oder verändern Sie seine Größe. Das vergrößerte Diagramm wird aktualisiert.
  • Wenn Sie den Skalierungsbereich (XY-Achse) des vergrößerten Diagramms ändern, klicken Sie auf das Rechteck im ursprünglichen Diagramm und wählen Sie Abmessungen aktualisieren, um seine Form so zu verändern, dass der gleiche Skalierungsbereich abgedeckt wird.
  • Markieren Sie das Rechteck und drücken Sie die Taste Entfernen, um das Rechteck zu löschen und gleichzeitig das vergrößerte Diagrammfenster zu löschen.

Weitere Informationen zum Zoomen, Schwenken und Skalieren Ihres Diagramms finden Sie auf dieser Seite.
