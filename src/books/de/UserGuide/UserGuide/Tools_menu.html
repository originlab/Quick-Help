Menü Hilfsmittel

Tools-Menu

Das Menü Hilfsmittel kann verwendet werden, um zwischen zwei Ansichten des Dialogs X-Funktionen erstellen zu wechseln und den Code Builder zu öffnen.

Baumansicht

Dieser Menübefehl kann verwendet werden, um zwischen der Standardansicht und Baumansicht zu wechseln. Bei aktivierter Baumansicht befindet sich ein Häkchen neben dem Menüelement. Wählen Sie erneut das Menüelement, um das Häkchen zu entfernen und zur Standardansicht zurückzukehren.
Im Code Builder bearbeiten

Öffnet die aktuelle X-Funktion in der Code Builder IDE (Entwicklungsumgebung). Hier können Sie den Code bearbeiten oder kompilieren. Dieses Menü dupliziert die Schaltfläche X-Funktion im Code Builder bearbeiten auf der Symbolleiste.
Dokument löschen

Wählen Sie diesen Menübefehl, um den Inhalt eines Dokuments zu löschen. Die für das Dokument relevanten Elemente umfassen:

  1. Alle Knoten unter dem Zweig Japanische Hilfe/Deutsche Hilfe - Beschreibung, Benutzerdefinierte Schaltflächen des Dialogs GetN, Automatische Vervollständigung für Hinweise;
  2. Alle Knoten unter dem Unterzweig Variablen: x: Englische Hilfe/Japanische Hilfe/Deutsche Hilfe - Anzeigename, Beschreibung, Automatische Vervollständigung für Hinweise, Erläuterungen der Optionen.

Diese Option ist nützlich, wenn Sie Ihre eigene benutzerdefinierte X-Funktion durch Duplizieren einer Standardfunktion und deren Bearbeitung erstellen möchten.
