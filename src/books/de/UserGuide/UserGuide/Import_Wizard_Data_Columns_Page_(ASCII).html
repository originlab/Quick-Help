Spaltentrennzeichen

Trennzeichen

Wenn die Option Trennzeichen ausgewählt ist, wird Origin die Spalten mit dem gewählten Trennzeichen trennen. Um ein benutzerdefiniertes Trennzeichen festzulegen, wählen Sie Andere und geben das Trennzeichen im benachbarten Textfeld ein. Beachten Sie, dass das Bearbeitunsgfeld Andere nur einzelne Zeichen unterstützt.
Aufeinanderfolgende Trennzeichen als eins betrachten

Wenn dieses Kontrollkästchen aktiviert ist, wird jede Anzahl an fortlaufenden Trennzeichen als nur ein Trennzeichen betrachtet.
Feste Breite

Die Option Feste Breite sollte ausgewählt werden, wenn die Spalten eine feste Breite haben. Sie können die Breite in das Textfeld neben der Option eingeben.

Wenn Feste Breite verwendet und die Anzahl der Spalten auf 0 gesetzt wird, wird die letzte Zahl, die in Feste Breite eingegeben wurde, wiederholt, bis alle Spalten importiert wurden. Wenn zum Beispiel Feste Breite auf 6,5,13 gesetzt wurde und die Anzahl der Spalten 0 ist, dann ist die feste Breite der ersten Spalte 6, die der zweiten 5 und die der dritten und aller weiteren Spalten 13 bis alle Spalten importiert wurden. Um eine Reihe von Zahlen zu wiederholen und nicht nur den Wert der letzten Spalte, können Sie Klammern verwenden. Wenn zum Beispiel für Feste Breite 5,6,(2,13) eingegeben wird und die Anzahl der Spalten 0 ist, dann ist die Breite der Spalten nach Spalte 2 abwechselnd 2 und 13, bis alle Spalten importiert wurden.
Hinweis: In Origin 8.6 werden Tab/Leerzeichen als Trennzeichen unterstützt. Dies unterscheidet sich von Tab oder Leerzeichen. Tab/Leerzeichen bezieht sich auf einzelne oder mehrere Tab(Leerzeichen) als Trennzeichen.

Sie können zum Beispiel zwei Leerzeichen zwischen zwei Werten haben. Wenn Sie beim Importieren dieser Daten Leerzeichen als Trennzeichen wählen, werden diese zwei Leerzeichen als zwei separate Trennzeichen betrachtet und drei Zellen importiert. Wenn Sie Tab/Leerzeichen als Trennzeichen wählen, werden zwei aufeinanderfolgende Leerzeichen als leerer Raum betrachtet, d.h. als einzelnes Trennzeichen. Es werden also nur zwei Zellen importiert. Wenn Sie Tab/Leerzeichen als Trennzeichen verwenden, sind keine anderen Trennzeichen verfügbar.

Spaltenzuordnung

Dieses Bedienelement kann für die Einstellung von Diagrammzuordnungen für alle Datenspalten während des Datenimports verwendet werden. Treffen Sie eine Wahl in der Auswahlliste oder geben Sie die Zuordnungssymbole direkt ein. Klicken Sie dann auf die Schaltfläche Anwenden, um Ihre Zuordnungen auf die Spalten der Datei anzuwenden.

Beachten Sie, dass Sie auch das Spaltenformat und die Spaltendiagrammzuordnung einzelner Spalten einstellen können, indem Sie mit der rechten Maustaste auf eine Spaltenüberschrift klicken und Setze Format oder Setze Zuordnung im Kontextmenü wählen. Die Einstellungen in Format und Diagrammzuordnung werden beim Datenimport auf die Spalten des Zielarbeitsblatts angewendet.

Sonstige Bedienelemente

Anzahl der Spalten

Legen Sie die Anzahl der Spalten fest, die beim Dateiimport im Zielarbeitsblatt erstellt werden soll. Geben Sie 0 ein, wenn Sie alle Spalten importieren möchten.

Wenn Trennzeichen ausgewählt ist (Bedienelemente Spaltentrennzeichen), wird die Spaltenanzahl auf den Wert gesetzt, den Origin "sieht", basierend auf Ihren Angaben zu Trennzeichen (Tab, Komma, Leerzeichen usw.).

Wenn Feste Breite ausgewählt wurde, wird die Spaltenanzahl mit Hilfe des Werts für Feste Breite errechnet. Sie können diesen Wert überschreiben, indem Sie einen Wert eingeben und auf Anwenden klicken. Wenn der Wert in der Spaltenanzahl den gesetzten Wert der Festen Breite übersteigt, wird die letzte Breite, die in der Zeichenkette bestimmt wurde, wiederholt. Das untere Anzeigebedienfeld wird aktualisiert, um den eingegebenen (überschreibenden) Wert anzuzeigen, nachdem Sie auf Anwenden geklickt haben.
Benutzerdefiniertes Datumsformat

Wählen Sie ein Datumsformat aus der Auswahlliste oder geben Sie eine benutzerdefinierte Zeichenkette für das Datumsformat ein und klicken Sie auf die Schaltfläche Anwenden.

Weitere Einzelheiten finden Sie unter Benutzerdefinierte Datums- und Zeitformate.
Benutzerdefiniertes Zeitformat

Wählen Sie ein Zeitformat aus der Auswahlliste oder geben Sie eine benutzerdefinierte Zeichenkette für das Zeitformat ein und klicken Sie auf die Schaltfläche Anwenden.

Weitere Einzelheiten finden Sie unter Benutzerdefinierte Datums- und Zeitformate.
Textvermerk

Wählen Sie das Zeichen aus, das die Werte in der Quelltextdatei einschließt. Der Text, der in jedem Paar des Abgrenzungszeichens eingeschlossen ist, wird als ein Wert angesehen, auch wenn der Text Trennzeichen enthält. Aktivieren oder deaktivieren Sie das Kontrollkästchen Entfernen, um zu bestimmen, ob Sie das Textabgrenzungszeichen entfernen möchten.
Numerische Trennzeichen

Wählen Sie ein numerisches Trennzeichen aus der Auswahlliste aus.
Führende Nullen in Zahlen entfernen

Aktivieren Sie diese Option, wenn Sie am Anfang stehende Nullen von importierten Daten entfernen möchten (z.B. 0050 wird zu 50).
Sparklines hinzufügen

Verwenden Sie diese Auswahlliste, um festzulegen, ob Sparklines zum Zielarbeitsblatt hinzugefügt werden sollen.
Zeilen durch Auffüllen mit fehlenden Werten gleich groß machen

Wenn dieses Kontrollkästchen aktiviert ist, können fehlende Werte hinzugefügt werden, um sicher zu stellen, dass jede Zeile gleich groß ist.
Zielspaltenformat behalten

Verwenden Sie dieses Kontrollkästchen, um festzulegen, ob das Zielspaltenformat beibehalten werden soll. Wenn dieses Kontrollkästchen aktiviert ist, ist das Kontextmenü Format setzen, das sich öffnet, wenn Sie mit der rechten Maustaste auf den Spaltenheader des Vorschaufensters klicken, nicht verfügbar.
Vorschau Spaltenbreite

Diese Option arbeitet in Verbindung mit der Schaltfläche Feste Breite. Verwenden Sie dieses Bedienelement, um das Muster der Spaltenbreite in einer Datei mit fester Breite (ohne Trennzeichen) festzulegen. Falls Sie dies auswählen, wechselt die Anzeige in den Modus Vorschau Spaltenbreite. In diesem Modus können die angezeigten Spalten mit der Maus auf die gewünschte Spaltenbreite gezogen werden. Spalten lassen sich mit Hilfe der benachbarten Schaltflächen Spalten hinzufügen und Spalten löschen hinzufügen oder löschen. Wenn Sie einzelne Spaltenbreiten durch Ziehen verändern, wird das Textfeld neben der Schaltfläche Feste Breite aktualisiert und zeigt die Spaltenbreiten als eine durch Komma getrennte Liste an. Sobald die Spaltenbreiten festgelegt wurden, deaktivieren Sie dieses Kontrollkästchen, um die Datenspalten so anzuzeigen, wie sie im Zielarbeitsblatt erscheinen werden.

Vorschaufenster

Wenn eine Strukturdefinition abgeschlossen ist, zeigt das Vorschaufenster die importierten Daten auf Basis der bisher festgelegten Aufzeichnungsstruktur so an, wie sie im Ergebnisarbeitsblatt aussehen werden. Beachten Sie, dass Sie mit einem Klick der rechten Maustaste auf die Spaltenköpfe im Vorschaufenster die Diagrammzuordnung und das Format der Spalten des Zielarbeitsblatts einstellen können.
