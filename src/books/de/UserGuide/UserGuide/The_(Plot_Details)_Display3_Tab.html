Die Registerkarte Anzeige für Ridgeline (Details Zeichnung)

PD-Dialog-Display-Tab-Ridgeline

Diese Registerkarte Anzeige wird verwendet, um zu steuern, wie die jeweilige Verteilung gezeigt wird.

Inhalt

  • 1 Typ
    • 1.1 Datenpunkte unter Kurve
  • 2 Quantillinien
  • 3 Linienstil
  • 4 Position
  • 5 Rugs

Typ

Legen Sie fest, welcher Typ der Ridgelines gezeigt wird:

  • Verteilungskurve
    Es werden nur die Verteilungskurven für das Ridgeline-Diagramm gezeigt.
  • Verteilungskurve mit Datenpunkten
    Es wird die Verteilungskurve mit Datenpunkten gezeigt. Sie können die folgende Option Datenpunkte unter Kurve aktivieren, um die Datenpunkte unter den Kurven anstatt die auf den Kurven zu zeigen. Die Registerkarte Daten wird angezeigt. Hier können Sie entscheiden, wie die Datenpunkte gezeigt werden sollen.
  • Verteilungskurve mit Rugs
    Die Verteilungskurven werden mit Rugs unter den Kurven gezeigt. Sie können die Rugs mit Hilfe der Gruppe Rugs benutzerdefiniert anpassen.
  • Balkendiagramm mit Einzelblock
    Histogramme werden als Balken mit Einzelblock für alle Zeichnungen gezeigt.

Datenpunkte unter Kurve

Diese Option wird nur angezeigt, wenn Sie Verteilungskurve mit Datenpunkten als Typ auswählen. Wenn Sie dieses Kontrollkästchen aktivieren, werden die Datenpunkte unter den Verteilungskurven gezeichnet. Ansonsten überschneiden sich die Datenpunkte mit den Kurven.

Quantillinien

Legen Sie fest, ob und welche Quantillinien gezeigt werden sollen. Aktivieren Sie das Kontrollkästchen. Sie können die Zahlen in das folgende leere Feld eingeben. Die Zahlen, die Sie eingeben, sollten zwischen 0 ~100 liegen.

Linienstil

Sobald Sie Quantillinien hinzugefügt haben, wird die Option Linienstil verfügbar und lässt Sie den Linienstil festlegen mitsamt Linienfarbe und Linienbreite.

Position

Wählen Sie eine Spaltenbeschriftungszeile, um die Position der Ridgelines in Y-Richtung zu steuern. Standardmäßig wird der Spaltenindex verwendet, um die Ridgelines anzuordnen.

Rugs

Sobald Sie Verteilungskurve mit Rugs als Typ ausgewählt haben, wird das Bedienfeld Rugs angezeigt, in dem Sie die Anzeige der Rugs unter den Ridgelines bestimmen können.

Sie können Farbe, Dicke, Größe (%) und Abstand zwischen Kurven für die Rugs festlegen.
