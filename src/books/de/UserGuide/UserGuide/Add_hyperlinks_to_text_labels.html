Hyperlinks zu Textbeschriftungen hinzufügen

Hyperlink-TextLabel

Origin unterstützt verschiedene Typen von Hyperlinks in Textbeschriftungen. Die Syntax für Links ist folgende:

range://: Kennzeichnet einen Bereich innerhalb des Projekts.
http:// oder https://: Kennzeichnet einen externen Link.
help://: Kennzeichnet einen Link zu der Origin-Hilfedokumentation.
note://: Kennzeichnet einen Link zu einem Notizfenster.

In jedem Fall wird der Link in das Textfeld in den Objekteigenschaften auf der Registerkarte Programmierung eingegeben (Alt+Doppelklick auf die Textbeschriftung). Der angezeigte Text wird dagegen auf der Registerkarte Text (Doppelklick) eingegeben. Die Textbeschriftung wird unterstrichen, wodurch angezeigt wird, dass es sich um einen Hyperlink handelt.

Eine Textbeschriftung kann auch in eine Schaltfläche verwandelt werden, die eine Operation durchführt, wenn man auf sie klickt. Siehe die Anweisungen für die Beispiele 3 und 4.

Beispiel 1 Link für Bereich

  1. Geben Sie Folgendes in die Textbeschriftung eines Diagramms ein:
    Gehen Sie zu Spalte 1, Zeile 100
  2. Klicken Sie mit der rechten Maustaste auf die Textbeschriftung und wählen Sie Eigenschaften im Kontextmenü. Gehen Sie im Dialog zur Registerkarte Programmierung.
  3. Geben Sie den Folgendes in das Skriptfeld unten im Dialog ein und klicken Sie auf OK. 
    range://[book1]sheet1!col(1)[100]
  4. Klicken Sie auf die Textbeschriftung, um Book1, Sheet1 zu öffnen und lassen Sie es bis zu der Zelle in Spalte 1, Zeile 100 scrollen.

Hinweis: Der Link tut nichts, der Zielbereich nicht existiert.

Beispiel 2 Externer Link

  1. Geben Sie Folgendes in die Textbeschriftung eines Diagramms ein:
    originlab.com
  2. Klicken Sie mit der rechten Maustaste auf die Textbeschriftung und wählen Sie Eigenschaften im Kontextmenü. Gehen Sie im Dialog zur Registerkarte Programmierung.
  3. Geben Sie Folgendes in das Skriptfeld ein und klicken Sie auf OK.
    http://www.originlab.com
  4. Klicken Sie auf die Beschriftung, um einen Browser auf der Startseite von OriginLab zu öffnen.

Beispiel 3 Verknüpfte Schaltfläche zur Hilfeseite

  1. Geben Sie Folgendes in die Textbeschriftung eines Diagramms ein:
    Klicken Sie, um die Hilfe zur Registerkarte Abstände anzuzeigen.
  2. Klicken Sie mit der rechten Maustaste auf die Textbeschriftung und wählen Sie Eigenschaften im Kontextmenü. Gehen Sie im Dialog zur Registerkarte Programmierung.
  3. Geben Sie das Folgende im Skriptfeld ein und stellen Sie sicher, dass Sie den Link in doppelt Anführungszeichen setzen: 
    run -LK "help://origin.chm/UserGuide/The_(Plot_Details)_Spacing_Tab.html"
  4. Wählen Sie die Option Mausklick unter Kriterien für Skriptausführung und klicken Sie auf OK.
  5. Klicken Sie auf die Schaltfläche, um die Origin-Hilfe auf der Seite "Die Registerkarte (Details Zeichnung) Abstände" zu öffnen.

Beispiel 4 Verknüpfte Schaltfläche zu Notizfenster

  1. Erstellen Sie ein neues Notizfenster Notes1.
  2. Geben Sie Folgendes in die Textbeschriftung eines Diagramms ein:
    Wechseln Sie zu Notes1.
  3. Klicken Sie mit der rechten Maustaste auf die Textbeschriftung und wählen Sie Eigenschaften im Kontextmenü. Gehen Sie im Dialog zur Registerkarte Programmierung.
  4. Geben Sie das Folgende im Skriptfeld ein und stellen Sie sicher, dass Sie den Link in doppelt Anführungszeichen setzen: 
    run -LK "notes://Notes1"
  5. Wählen Sie die Option Mausklick unter Kriterien für Skriptausführung und klicken Sie auf OK.
  6. Klicken Sie auf die Schaltfläche, um zum Fenster Notes1 zu wechseln.

Hinweis: Die Schaltfläche tut nichts, wenn das festgelegte Notizfenster nicht existiert.
