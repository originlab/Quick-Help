Projekteigenschaften

Project-Properties

Verwenden Sie den Dialog Projekteigenschaften, um bestimmte Einstellungen mit der Projektdatei zu speichern.

EinstellungTrennzeichen im Projekt speichern

Aktivieren Sie diese Option, um die aktuelle Listenoption der Trennzeichen (Einstellungen: Optionen: Zahlenformat) im Projekt zu speichern. Während das Projekt in Origin geöffnet ist, wird diese Option in Einstellungen gezeigt. Wenn das Projekt geschlossen ist, kehren die Einstellungen zurück zur benutzerdefinierten Option der Trennzeichen.
Einstellung des Linienabstands im Projekt speichernProject

Aktivieren Sie diese Option, um die aktuelle Listenoption des Linienabstands (Einstellungen: Optionen: Zeichensätze) im Projekt zu speichern. Während das Projekt in Origin geöffnet ist, wird diese Option in Einstellungen gezeigt. Wenn das Projekt geschlossen ist, kehren die Einstellungen zurück zur benutzerdefinierten Option des Linienabstands.
Zugriffsprotokoll

Nur lesbares Bedienelement, das nur dazu dient, den Anwender davon ins Kenntnis zu setzen, dass das Zugriffsprotokoll für das Projekt aktiviert wurde. Das Zugriffsprotokoll muss über Hilfsmittel: Schutz aktiviert oder deaktiviert werden.
Öffnen ist passwortgeschützt

Nur lesbares Bedienelement, das nur dazu dient, den Anwender davon in Kenntnis zu setzen, dass das Projekt passwortgeschützt ist. Passwortschutz muss über Hilfsmittel: Schutz verwaltet werden.

Sie können Dateien an ein Projekt anhängen. Öffnen Sie den Code Builder und fügen Sie Dateien zum Projektordner hinzu.
