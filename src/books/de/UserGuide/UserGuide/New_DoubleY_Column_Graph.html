Datenanforderungen

Wählen Sie mindestens eine Y-Spalte aus (oder einen Bereich aus mindestens einer Y-Spalte). Wählen Sie zwei Y-Spalten (oder einen Bereich aus zwei Y-Spalten) aus. Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie Zeichnen > Mehrere Felder/Achsen: Säulen mit doppeltem Y im Menü.

Vorlage

DblY_Col.OTP (im Origin-Programmordner installiert)

Notizen

Dieser Diagrammtyp hat nur einen Layer und 2 Y-Achsen mit unterschiedlichen Achsenskalierungen wie beim Diagramm mit doppelter Y-Achse. In diesem Diagramm werden alle Y-Spalte(n) als Säulen gezeichnet, unabhängig davon auf welcher Y-Achse sie basieren.

  • Dies ist eine Diagrammvorlage mit einem einzelnen Layer mit zwei unabhängigen vertikalen (Y)-Achsen.
  • Das Kontrollkästchen Jede Achse unabhängig neu skalieren auf der Registerkarte Zeigen im Dialog Achsen ist aktiviert.
  • Die Vorlage hat die gleichen Datenanforderungen und erstellt das gleiche Diagramm wie das Säulendiagramm mit 2Y mit zwei Layern.
  • Wenn eine gerade Anzahl von Spalten größer als 2 (Y-Spalten) ausgewählt ist, werden Datensätze abwechselnd gegen die vertikalen Achsen gezeichnet, erst die linke, dann die rechte.
