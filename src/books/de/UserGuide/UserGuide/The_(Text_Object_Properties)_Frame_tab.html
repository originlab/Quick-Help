Rahmen

Legen Sie fest, ob ein Rahmen um den Text herum hinzugefügt werden soll. Drei Auswahlmöglichkeiten sind für diese Auswahlliste verfügbar: Kein, Box, Schatten und Marmor.

Kein Diese Option ist standardmäßig ausgewählt; Rahmenfarbe ist währenddessen deaktiviert.
Boxdiagramm Wenn Box aktiviert ist, wird eine Box als Rahmen um den Text herum hinzugefügt. Sie können Füllfarbe, Rahmenfarbe, Rahmenbreite und Rand ändern.
Schatten Wenn Schatten aktiviert ist, wird eine Box und sein Schatten um den Text herum hinzugefügt. Sie können Füllfarbe, Rahmenfarbe, Rahmenbreite, Schattenfarbe, Schattenfarbe und Rand ändern.
Marmor Wenn Marmor ausgewählt ist, wird der graue Marmorstil für den Text verwendet. Sie können Füllfarbe, Rahmenfarbe, Rahmenbreite und Rand ändern.

Füllfarbe

Wenn Box, Schatten oder Marmor in der Auswahlliste Rahmen ausgewählt ist, können Sie die Füllfarbe des Rahmens benutzerdefiniert anpassen.

Transparenz

Wenn in der Auswahlliste Rahmen die Option Box ausgewählt ist, können Sie die Transparenz der Füllfläche benutzerdefiniert anpassen, indem Sie den Schieber bewegen oder eine ganze Zahl zwischen 0 bis 100 im Kombinationsfeld eingeben. Beachten Sie, dass das Symbol bei 0 gar nicht und bei 100 vollkommen transparent ist.

Hinweis: Sie können die Transparenz des Legendenobjekts zurzeit nicht in OpenGL-basierten Diagrammen steuern.

Rahmenfarbe und -breite

Wenn Box, Schatten oder Marmor in der Auswahlliste Rahmen ausgewählt ist, können Sie die Rahmenfarbe und die Rahmenbreite benutzerdefiniert anpassen.

Schattenfarbe und -breite

Wenn Schatten in der Auswahlliste Rahmen ausgewählt ist, können Sie die Schattenfarbe und die Rahmenbreite benutzerdefiniert anpassen.

Rand (% Schrifthöhe)

Wenn Box, Schatten oder Marmor in der Auswahlliste Rahmen ausgewählt ist, können Sie den Rand des Rahmens benutzerdefiniert anpassen. Sie können das Textfeld neben Rand verwenden, um die vier Seitenrändern zusammen festzulegen, oder Sie legen die Seitenränder unter Links, Rechts, Oben bzw. Unten fest.

Anwenden auf

Sie können das Textobjekt auf Objekte, Zeichnungen, Gruppierte Zeichnungen, Layer und Fenster anwenden.
