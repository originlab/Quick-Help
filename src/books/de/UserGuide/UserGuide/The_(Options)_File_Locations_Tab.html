Bedienelemente der Dateiverfolgung

Folge Änderungen im Dialog Datei

Standardmäßig behält Origin den Dateipfad für Origin-Projekte, Importdateien, grafische Exportdateien und andere unterstützte Dateitypen. Daher wird jedes Mal, wenn Sie ein neues Projekt öffnen (oder ein Diagramm in eine Datei exportieren), der Pfad der zuletzt geöffneten Projektdatei (oder der Pfad der zuletzt exportierten Diagrammdatei) standardmäßig in dem Dialogfeld Öffnen (oder Speichern unter) ausgewählt.

Um die Dateiverfolgung für alle Dateitypen auszuschalten, deaktivieren Sie das Kontrollkästchen Folge Änderungen im Dialog Datei.
Bearbeiten

Um das Pfadverfolgungsverhalten für bestimmte Dateitypen zu verändern:

  1. Lassen Sie das Kontrollkästchen Folge Änderungen im Dialog Datei aktiviert.
  2. Wählen Sie die Dateigruppe aus dem oberen Listenfeld aus.
  3. Klicken Sie auf die Schaltfläche Bearbeiten (obere rechte Ecke). Dies öffnet den Dialog Standard für Erweiterungen der Dateigruppen für die spezifische Gruppe.
  4. Nach dem benutzerdefinierten Anpassen der Einstellungen in dem Dialogfeld, klicken Sie auf OK.

Der Dialog Standard für Erweiterungen der Dateigruppen

Dieses Dialogfeld öffnet sich, wenn Sie auf die Schaltfläche Bearbeiten in der Gruppe Folge Änderungen im Dialog Datei klicken. Es kann dazu verwendet werden, das Verhalten der Pfadverfolgung für einen bestimmten Dateityp zu ändern.

Gruppe:

Zeigt die Gruppe der Dateitypen an, deren Pfadverfolgungsverhalten durch das Dialogfeld bearbeitet werden kann.
Verz. öffnen

Die Optionen unter Verz. öffnen steuern die Dateiverfolgung, wenn der bestimmte Dateityp geöffnet oder importiert wird.

Verwenden Sie die Auswahliste, um den Standardpfad im Dialogfeld Öffnen festzulegen. Optionen:

  • Benutzerdefinierter Pfad:
    Ein festgelegter Pfad, der unten angezeigt wird. (Beachten Sie, dass der benutzerdefinierte Pfad die Standardeinstellung ist. Daher wird der Pfad in dem entsprechenden Textfeld angezeigt, wenn Sie bereits eine Datei dieser Gruppe geöffnet haben.)
  • Projektpfad:
    Der Pfad des momentan geöffneten Projekts
  • Origin exe Pfad:
    Der Origin-Installationsordner
  • Anwenderdateiordner:
    Der Origin Anwenderdateiordner

Auf alle Gruppen anwenden

Wenn dieses Kontrollkästchen aktiviert ist, verwendet es den ausgewählten geöffneten Pfad als Verfolgungspfad, wenn anderen Dateigruppen geöffnet oder importiert werden.
Pfad speichern

Die Optionen unter Verz. speichern steuern die Dateiverfolgung, wenn der bestimmte Dateityp gespeichert oder exportiert wird. Die Optionen in der Auswahlliste haben die gleiche Bedeutung wie ihre Gegenstücke in der Auswahlliste der Gruppe Verz öffnen. Die Auswahl in dieser Gruppe definiert jedoch den Standardpfad im Dialogfeld Speichern. Das Kontrollkästchen Auf alle Gruppen anwenden legt fest, ob der ausgewählte gespeicherte Pfad als Verfolgungspfad verwendet wird, wenn andere Dateigruppen gespeichert oder exportiert werden.
Geben Sie

Diese Auswahlliste führt alle Dateitypen in der aktuellen Gruppe auf. Die Auswahl in dieser Auswahlliste legt die Standardtypen fest, die in den Dialogfeldern Öffnen und Speichern unter für spezifische Gruppen aufgelistet werden. Falls die aktuelle Gruppe ein Bild ist und Sie Joint Photographic Experts Group (*.JPG) aus der Auswahlliste Typ auswählen und falls Sie ein Diagramm durch die Auswahl Datei: Seite exportieren exportieren, dann wird Joint Photographic Experts Group (*.JPG) standardmäßig in der Auswahlliste Speichern als Dateityp angezeigt.

Bedienelemente des ASCII-Dateityps

Hinzufügen/Bearbeiten/Löschen

Dieses Listenfeld zeigt die ASCII-Dateibeschreibungen an, die in der Auswahlliste Dateityp angezeigt werden, wenn Origin ASCII-Dateien über Daten: Aus Datei importieren importiert.

Klicken Sie auf die Schaltflächen, um eine Dateibeschreibung hinzuzufügen/zu bearbeiten/zu löschen.

Um einen Dateityp hinzuzufügen, drücken Sie auf Hinzufügen. Hierdurch öffnet sich das Dialogfeld Dateierweiterungstyp. Geben Sie einen Text in das Textfeld Beschreibung ein, der in der Auswahlliste Dateityp angezeigt wird, wenn ASCII-Dateien importiert oder exportiert werden. Geben Sie im Textfeld Anforderung(en) die Dateianforderungen für die Dateien ein, die im Ansichtsfeld Suchen angezeigt werden, wenn Dateien importiert oder exportiert werden.

Angenommen Sie verwenden beispielsweise ein Drittprogramm, dass Daten in eine ASCII-formatierte Datei ausgibt. Dieser formatierten Datei wird ein Name mit der Endung .XYZ gegeben. Sie können Origin verwenden, um nur diese XYZ-Dateien in dem Dialogfeld ASCII-Datei Import und Export anzuzeigen, indem Sie einen ASCII-Dateityp hinzufügen.

Im Textfeld Beschreibung können Sie den Namen des Drittprogramms eingeben. Sie können ebenfalls (*.XYZ) nach dem Namen eingeben, um zu zeigen, welche Dateitypen mit dem Drittprogramm verbunden sind.

Im Textfeld Anforderung(en) würden Sie *.XYZ eingeben. Mehrere Anforderungen können angegeben werden, indem diese mit einem Semikolon getrennt werden. Beispiel: *.XYZ;*.XY;*.ZZ. Es können außerdem beide Platzhalter "*" und "?" verwendet werden.

Wenn Sie nun ASCII-Dateien über das Menü Daten: Aus Datei importieren importieren, können Sie Ihren neuen Dateitypen aus der Auswahlliste Dateityp auswählen und nur die Dateien, die mit diesem Programm im Ansichtsfeld Suchen verbunden sind, sehen.

Hinweise:

Diese Einstellungen sind für das Menü Daten: Aus Datei importieren. Wenn Sie einen Datenkonnektor verwenden und die Dateierweiterungen des CSV-Konnektors benutzerdefiniert anpassen möchten, können Sie mit der rechten Maustaste auf das Symbol des CSV-Konnektors in der Apps-Galerie klicken und Dateierweiterungen benutzerdefiniert anpassen im Kontextmenü auswählen. Weitere Einzelheiten finden Sie in diesem Abschnitt zum CSV-Konnektor.
