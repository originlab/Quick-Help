Datenanforderungen

Wählen Sie mindestens zwei Y-Spalten aus (oder einen Bereich aus mindestens zwei Y-Spalten). Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Mehrere Y-Spalten können durch zwei X-Spalten in zwei Gruppen unterteilt werden. Ansonsten werden sie nach Abfolge verschiedenen Y-Achsen zugewiesen.

Diagramm erstellen

Wählen Sie die erforderlichen Daten aus

Wählen Sie Zeichnen > Mehrere Felder/Achsen: Doppelte Y-Achse.

Vorlage

DblY.otpu (installiert im Origin-Programmordner).

Notizen

Dieses Diagramm mit doppelter Y-Achse hat nur einen Layer, zeigt aber zwei Y-Achsen (linke Y und rechte Y) mit unterschiedlichen Skalierungen. Die Vorlage hat die gleichen Datenanforderungen und erstellt das gleiche Diagramm wie das Diagramm 2Y, Y-Y mit zwei Layern. Falls zwei Wertespalten ausgewählt sind (oder ein Bereich aus beiden Spalten), dann werden beide in einem einzelnen Layer angezeigt. Jeder Datenpunkt im Diagramm ist mit einer Linie verbunden. Die Standardlinienverbindung zwischen den Punkten ist eine gerade Linie. Die Datenpunkte werden als Symbole angezeigt.

Es wurden zwei Kontrollgruppen hinzugefügt, um diesen Diagrammtyp zu unterstützen:

  • Die Option Daten zeichnen in auf der Registerkarte Anzeige im Dialog Details Zeichnung kann verwendet werden, um zu entscheiden, auf welcher Y-Achse das aktuelle Diagramm gezeichnet werden soll.
  • Auf der Registerkarte Zeigen im Dialog Achsen können Sie entscheiden, ob die rechte und linke Y-Achse beide im gleichen Layer gezeigt werden und ob diese zwei Y-Achsen unabhängige Skalierungen haben. Sobald es zwei Y-Achsen mit unabhängigen Achsenskalierungen gibt, können Sie die Achsenskalierung, Gitternetzlinien, Referenzlinien und Unterbrechungen für diese zwei Y-Achsen unabhängig voneinander benutzerdefiniert anpassen.
