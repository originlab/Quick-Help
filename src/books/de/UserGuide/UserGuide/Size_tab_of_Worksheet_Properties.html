Registerkarte Größe (Arbeitsblatteigenschaften)

WksProperties-Dialog-Size-Tab

Bedienelemente zum Modifizieren der Gitternetzabstände im Arbeitsblatt

Größe

Zeilennummer

Bestimmt die Anzahl der Zeilen im aktuellen Arbeitsblatt. Wenn das Kontrollkästchen Automatisches Hinzufügen von Zeilen auf der Registerkarte Verschiedenes aktiviert ist, dann bestimmt die Größe des Fensters die minimale Zeilenanzahl (deaktivieren Sie Automatisches Hinzufügen von Zeilen, um die Zeilenanzahl ohne Einschränkung einstellen zu können).

Spaltennummer

Bestimmt die Anzahl der Spalten im aktuellen Arbeitsblatt.

Abmessungen Arbeitsblatt

Höhe des Spaltenkopfs

Höhe des Spaltenkopfs = X% der Standardzeilenhöhe von 16px (z.B. 150% * 16px = 24px)

Breite des Zeilenkopfs

Breite der Zeilenkopfs = X% der Standardspaltenbreite von 67px (z.B. 128% * 67px = 86px)

Abstand von oben

Sie können eine Lücke zwischen der Fenstertitelleiste der Arbeitsmappe und der oberen Kante des Arbeitsblatts einfügen. Verwenden Sie diese Fläche für solche Dinge wie das Platzieren von Textobjekten oder benutzerdefinierten Schaltflächen, die Skripte ausführen. Die Lücke wird in Einheiten der Textpunktgröße gemessen (z.B. erstellt die Eingabe "20" eine Lücke, die einen 20-Punkt-Text unterbringt).

Höhe aller Zeilen

Höhe aller Zeilen = X% der Standardzeilenhöhe von 16px (z.B. 100% * 16px = 16px)

Breite aller Spalten

Breite aller Spalten = X% der Standardspaltenbreite von 67px (z.B. 100% * 67px = 67px)
