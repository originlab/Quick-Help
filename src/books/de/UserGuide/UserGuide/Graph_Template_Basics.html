Diagramm mit Hilfe einer bestehenden Diagrammvorlage erstellen

Alle Origin-Diagramme werden aus einer Vorlage erstellt. Wenn Origin neu für Sie ist, sind alle Ihre Diagrammvorlagen Systemvorlagen und mit ausgelieferte erweitere Vorlagen. Über die Zeit werden Sie diese Systemvorlagen wahrscheinlich benutzerdefiniert anpassen und diese benutzerdefinierten Vorlagen in Ihrem Anwenderdateiordner oder an anderer geeigneter Stelle speichern. Sie können auf diese benutzerdefinierten Vorlagen und die mit ausgelieferten erweiterten Vorlagen dann über Origins Diagrammvorlagenbibliothek zugreifen und sie verwalten.

In Origin 2022 wurde neues Hilfsmittel eingeführt, das sogenannte Vorlagencenter, mit dem Sie die Arbeitsmappen- und Diagrammvorlagen aus dem Dateiaustausch der Webseite von OriginLab wie beim App-Center herunterladen können.

Alle Diagrammvorlagen von Origin enthalten einen Satz von Standardeigenschaften, mit denen Sie in nur wenigen schnellen Schritten ein einfaches Diagramm erstellen können. Dazu verwenden Sie eine dieser Methoden:

  • Wählen Sie Ihre Arbeitsblatt-, Matrix- oder Excel-Arbeitsmappendaten und klicken Sie auf eine der Schaltflächen auf den Grafiksymbolleisten (2D-Grafiken, 3D- und Konturdiagramme).
  • Wählen Sie Ihre Arbeitsblatt-, Matrix- oder Excel-Arbeitsmappendaten und im Origin-Menü Zeichnen > Kategorial: Diagrammtyp.
  • Wählen Sie Ihre Arbeitsblatt-, Matrix- oder Excel-Arbeitsmappendaten und dann Zeichnen: Meine Vorlagen oder Zeichnen: mit Template.
  • Öffnen Sie eine leere Diagrammvorlagendatei, indem Sie (a) auf die Schaltfläche Neue Grafik auf der Symbolleiste Standard klicken oder (b) Datei: Neu: Diagramm wählen. Fügen Sie dann Daten zu der leeren Vorlage hinzu, entweder durch (a) Drag&Drop, (b) den Dialog Diagrammeinstellungen oder (c) den Dialog Layer n (beachten Sie, dass Diagrammtypen, die eine Art von Berechnung erfordern (z. B. Histogramme, Boxdiagramme, Ternärdiagramme etc.), auf diese Weise nicht erstellt werden können. Verwenden Sie stattdessen das Menü Zeichnen oder eine geeignete Schaltfläche auf einer Grafiksymbolleiste).


Wenn die Zeichnung, die Sie mit den Standardeigenschaften der Vorlage erstellt haben, gut für Sie aussieht, sind Sie fertig. Wenn Sie das Diagramm benutzerdefiniert anpassen und, optional, Ihre Anpassungen in eine Vorlagendatei speichern möchten, lesen Sie den nächsten Abschnitt.

Diagramm benutzerdefiniert anpassen

Alle Zeichnungsanpassungen (ausschließlich Schaltflächenobjekte mit Anmerkungen etc.) können über Origins Dialog Details Zeichnung, die Dialoge Achsen oder die zugehörigen Minisymbolleisten durchgeführt werden.

Innerhalb des Dialogs Details Zeichnung werden Eigenschaften hierarchisch nach dem Schema Seite > Layer > Zeichnung organisiert. Informationen dazu, wie die Eigenschaften des Diagrammfensters organisiert sind und wie sie bearbeitet werden können, finden Sie unter diesen Themen:

  • Elemente der Seite benutzerdefiniert anpassen
  • Elemente des Layers benutzerdefiniert anpassen
  • Diagrammelemente benutzerdefiniert anpassen

 

Alle Anpassungen der Diagrammachsen können im Dialog Achsen festgelegt werden. Einige spezifische Diagramme haben ihren eigenen Achsendialog, wie ternäre und polare Koordinatendiagramme.

Die Minisymbolleiste ist eine Sammlung von Hilfsmitteln zur "schnellen Bearbeitung" durch interaktives Modifizieren von allgemeinen Diagrammobjekteigenschaften. Diese Symbolleisten hängen von dem ausgewählten Objekttyp ab. Die Schaltflächen im Popup bieten Zugriff auf alle allgemeinen Anpassungsoptionen, so dass schnelle Änderungen durchgeführt werden können, ohne die komplexen Dialoge öffnen zu müssen.

Weitere Einzelheiten zu den Anpassungen, die mit der Vorlage gespeichert werden, finden Sie unter Was wird mit der Diagrammvorlage gespeichert?

Diagramm als benutzerdefinierte Diagrammvorlage speichern

Wenn Sie ein Basisdiagramm erstellt haben, dieses Diagramm benutzerdefiniert angepasst haben und diese Anpassungen jetzt zur späteren Verwendung speichern möchten, dann besteht der nächste Schritt darin, Ihr Diagramm als benutzerdefinierte Diagrammvorlage zu speichern.

Bei aktivem benutzerdefinierten Diagrammfenster:

  1. Wählen Sie Datei: Template speichern unter im Origin-Menü. Oder klicken Sie mit der rechten Maustaste auf die Titelleiste des aktuellen Diagrammfensters, um im Kontextmenü Template speichern unter auszuwählen. Der Dialog template_saveas wird geöffnet.


Informationen zu den Dialogelementen von template_saveas finden Sie unter Ein benutzerdefiniertes Fenster als eine Vorlage speichern.

Klonbare Vorlagen

Klonbare Vorlagen sollen für das "intelligente Zeichnen" von Daten verwendet werden, die immer eine gemeinsame Struktur anzeigen. Klonbare Vorlagen erfordern keine Vorauswahl der Daten vor dem Auswählen einer Vorlage (Datenauswahlen werden beim Zeichnen in eine klonbare Vorlage tatsächlich ignoriert). Wenn die Datenstruktur der aktiven Arbeitsmappe oder Matrixmappe mit der Datenstruktur übereinstimmt, die mit der klonbaren Vorlage gespeichert ist, dann ist das Symbol mit dem Schaf links unten in der klonbare Vorlage blau. Sie können die Vorlage in der Vorlagenbibliothek auswählen und auf die Schaltfläche Zeichnung klicken. Ohne die Vorlagenbibliothek zu benutzen, können Sie auch einfach Ihre mit dem blauen Schafsymbol markierte Vorlage in der Liste Zeichnen: Meine Vorlagen auswählen.

Ein Diagramm als klonbare Vorlage speichern

Bei aktivem Diagrammfenster:

  1. Wählen Sie Datei: Template speichern unter oder klicken Sie mit der rechten Maustaste auf die Titelleiste des Diagrammfensters und wählen Sie Template speichern unter, um den Dialog Utilities\File: template_saveas zu öffnen.
  2. Standardmäßig werden neu erstellte Vorlagen in der Kategorie UserDefined gespeichert. Sie können sie aber auch in einer neuen Kategorie ablegen, indem Sie einen neuen Kategorienamen im Bearbeitungsfeld Kategorie eingeben. Kategorien helfen Ihnen, Ihre Vorlagen zu organisieren, so dass Sie diese leichter wieder finden können.
  3. Um der Vorlage einen Namen zu geben, geben Sie einen neuen Namen in dem Bearbeitungsfeld Vorlagenname ein. Um eine bestehende Vorlage zu überschreiben, wählen Sie bzw. geben Sie diesen Vorlagennamen in der Auswahlliste/in dem Bearbeitungsfeld Vorlagenname ein und und klicken Sie auf OK. Klicken Sie für die Aufforderung auf OK.
  4. Sie können den Lang-/Kurznamen des neu geklonten Diagramms mit Headerinformationen mit Hilfe von LabTalk-Substitutionsnotationen im Bearbeitungsfeld Diagrammlang-/-kurzname definieren vordefinieren. Wenn Sie nicht hinreichend vertraut sind mit der LabTalk-Syntax, können Sie im Ausklappmenü zu den verfügbaren Elementen navigieren.
  5. Durch Aktivieren des Kontrollkästchens Als klonbare Vorlage markieren speichern Sie dieses Diagramm als eine klonbare Vorlage und nicht als Standardvorlage. Wenn diese Option aktiviert ist, können Sie die Methode weiter spezifizieren, mit der untersucht wird, wie Arbeitsmappendaten mit dem Prototypdiagramm abgeglichen werden. Wenn Sie Kurz-/Langname für die Option Spalte/Blatt stimmt überein mit wählen, müssen Sie sicherstellen, dass jeder Kurz-/Langname einer Spalte/eines Blatts in der aktuellen Arbeitsmappe einzigartig ist.
  6. Wenn Sie Ihre Diagrammvorlagen an einem anderen Speicherort verwalten müssen, können Sie auf die Schaltfläche für weitere Optionen klicken und zum Dateipfad navigieren.
  7. Nachdem Sie alle Einstellungen vorgenommen haben, können Sie auf OK klicken, um die klonbare Vorlage zu erstellen.

Ein weiteres intelligentes Zeichenhilfsmittel ist das Stapelzeichnen, das kein Speichern oder Aurufen einer Vorlagendatei erfordert.
