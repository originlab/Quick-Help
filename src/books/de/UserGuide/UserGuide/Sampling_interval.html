Was ist ein Abtastintervall?

Für das Zeichnen und viele Analyseroutinen sind X-Werte der Daten erforderlich. Normalerweise werden die X-Werte einer Datenspalte in einer anderen Arbeitsblattspalte gespeichert. Für Daten mit äquidistanten X-Werten (zum Beispiel Waveform-Daten) gibt es jedoch eine andere Option, mit der die Verschiebung, das Inkrement und die Einheit der X-Werte in der Kopfzeile Abtastintervalls der Datenspalte gespeichert werden.

Ein Vorteil bei der Verwendung der Kopfzeile des Abtastintervalls besteht darin, dass sie Speicher spart. Außerdem kann Origin die Informationen verwenden, die in dieser Kopfzeile für alle Operationen wie Zeichnen und Analyse gespeichert werden. Zum Beispiel können Sie im Dialog Diagrammeinstellungen bestimmen, dass eine Datenspalte gegen <AutoX> gezeichnet wird, die X-Werte, die von den Informationen in der Kopfzeile des Abtastintervalls definiert werden. Wenn Sie die Spalte mit den X-Werten haben möchten, können Sie ganz einfach eine mit den Informationen des Abtastintervalls erzeugen.

Die Informationen in der Kopfzeile des Abtastintervalls können automatisch durch einige Import- und Analyseroutinen erzeugt werden. Sie können sie auch manuell durch die Nutzung von Origin-Hilfsmitteln erstellen und bearbeiten. Die Aktualisierung der Informationen löst eine Neuberechnung aus, wenn die Datenspalte eine Eingabespalte der Operation ist.

Hinweise:

  • Die Genauigkeit der in der Kopfzeile des Abtastintervalls angezeigten Anzahl wird von der Registerkarte Zahlenformat des Dialogs Einstellungen: Optionen gesteuert. Intern sind die Werte sehr genau und befinden sich in den Metadaten, die im Arbeitsmappenorganizer angezeigt werden können.
  • Sie können das Abtastintervall für X-, Y- und Z-Spalten festlegen.

Abtastintervallinformationen festlegen und löschen

Sie können die Informationen in der Kopfzeile des Abtastintervalls nicht direkt bearbeiten. Sie können jedoch die folgenden Hilfsmittel verwenden, um sie zu erstellen/modifizieren/löschen.

Abtastintervall festlegen

Die X-Funktion colint kann die Informationen in der Kopfzeile des Abtastintervalls hinzufügen, modifizieren und entfernen.

Der Dialog dieser X-Funktion kann geöffnet werden durch:

  • Auswählen von Spalte: Abtastintervall festlegen im Hauptmenü,
  • Doppelklick auf die Zelle Abtastintervall der Spalte.

Der X-Initialisierungswert, der in diesem Dialog festgelegt wird, wird als x0 in der Kopfzeile des Abtastintervalls angezeigt. Das X-Inkrement wird als dx gezeigt. Die Einheit, die in diesem Dialog festgelegt wird, ist auch in der Kopfzeile des Abtastintervalls enthalten. Zusätzlich können Sie den Langnamen für die X-Spalte festlegen. Er wird dann später aus den Informationen in der Kopfzeile des Abtastintervalls erstellt.

Um die existierenden Informationen in der Kopfzeile zu modifizieren, öffnen Sie einfach wieder diesen Dialog, nehmen die notwendigen Änderung vor und klicken Sie auf OK.

Der Dialog kann auch dazu verwendet werden, um die Informationen in der Kopfzeile zu löschen. Öffnen Sie den Dialog und legen Sie den X-Initialisierungswert und das X-Inkrement mit 1 (der Standardwert) fest.

Zusätzliche Spalteninformationen

Wenn die Kopfzeile des Abtastintervalls nicht leer ist, sehen Sie ein gelbes Symbol mit dem Buchstaben "i" oben links. Durch Klicken auf dieses Symbol wird der Dialog Zusätzliche Spalteninformationen geöffnet, mit dem Sie die Abtastintervallinformationen bearbeiten und die Kopfzeileninformationen anzeigen können, wenn die Daten in dieser Spalte aus einer SPC-Datei importiert werden.

Das Bearbeitungsfeld EvenX Initialisierungswert im Dialog entspricht der Information x0 in der Kopfzeile, während Sie mit dem Bearbeitungsfeld EvenX Inkrement die dx-Information in der Kopfzeile bearbeiten können. Indem beide Werte auf 1 gesetzt werden, können die Informationen in der Kopfzeile entfernt werden, weil 1 der Standardwert ist.

Der Bereich Headerinfo im Dialog Zusätzliche Spalteninformationen zeigt die Headerinformationen an, die aus der SPC-Datei importiert wurden, aus der die Daten stammen. Wenn die Daten in der Spalte nicht aus einer SPC-Datei importiert werden, bleibt dieser Bereich leer.

X-Spalte aus einem Abtastintervall erstellen

Sie können eine X-Spalte (eine Spalte mit X-Werten) aus den Informationen in der Kopfzeile des Abtastintervalls erstellen. Beachten Sie, dass die Diagrammzuordnung dieser Spalte standardmäßig auf Ignorieren eingestellt ist.

Um eine X-Spalte aus einem Abtastintervall zu erstellen, markieren Sie die Datenspalte und wählen Sie dann Spalte: X-Spalte zeigen im Origin-Menü.
