Form der Eingabedaten

Indiziert Eine Spalte enthält die Gruppenvariable und eine andere Spalte enthält die entsprechenden Antwortdaten.
Roh Die Daten für die zwei Stichproben sind in getrennten Spalten enthalten.

Eingabe

Hilfe zum Festlegen von Bereichen finden Sie hier: Eingabedaten festlegen

  • Eingabedatenbereich, wenn Eingabedatenform = Index
Gruppenbereich Die Gruppierungsspalte Beachten Sie, dass die Gruppierungsspalte als Kategorial gesetzt wird, wenn es sich um eine Textspalte handelt.
Datenbereich Die Datenspalte
  • Eingabedatenbereich, wenn Eingabedatenform = Roh
Erster Datenbereich Die erste Stichprobenspalte
Zweiter Datenbereich Die zweite Stichprobenspalte

Nullhypothese

Nullhypothese Die Nullhypothese: Die zwei Stichproben stammen aus Grundgesamtheiten, die die gleiche Verteilung haben (nur lesbar).

Alternativhypothese

Median1 <> Median2 Ein beidseitiger Test wird durchgeführt.
Median1 > Median2 Ein oberer Test wird durchgeführt.
Median1 < Median2 Ein unterer Test bei zwei Stichproben wird durchgeführt.
