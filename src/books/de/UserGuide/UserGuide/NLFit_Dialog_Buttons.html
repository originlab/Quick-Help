Zusammenfassung der Schaltflächen

Die Schaltflächen im mittleren Bereich des Dialogs NLFit bieten die Steuerung des Anpassungsprozesses und den Zugriff auf das Hilfsmittel Fitfunktionen verwalten. Dieses Hilfsmittel wird verwendet, um neue Anpassungsfunktionen zu bearbeiten oder zu definieren.

Funktionen suchen und einfügen

Klicken Sie, um eine angemessene Funktion zu suchen und sie mit Hilfe von Fitfunktion suchen in den Dialog einzufügen.
Anpassungsfunktionen bearbeiten

Wenn Sie eine Standardfunktion ausgewählt haben, öffnet diese Schaltfläche den Dialog Fitfunktionen verwalten. Wenn Sie eine benutzerdefinierte Funktion ausgewählt haben, öffnet diese Schaltfläche den Dialog Fitfunktionen erstellen.
Neue Anpassungsfunktionen erstellen Klicken Sie auf diese Schaltfläche, um eine neue Anpassungsfunktion im Dialog Fitfunktionen erstellen zu erstellen.

Fit-Parameterwerte im Modus Parameter ändern

Wenn Sie eine Kurvenanpassung durchführen, bei der der Modus Neu berechnen auf Manuell oder Auto gesetzt ist, können Sie den Dialog NLFit von einem ausgegebenen Arbeitsblatt oder einer Grafik einfach wieder öffnen. Klicken Sie einfach mit der linken Maustaste auf das Schlosssymbol im Arbeitsblatt oder auf der Grafik und klicken Sie dann Parameter ändern.

Die Parameterwerte, die auf der Registerkarte Parameter angezeigt werden, sind die gleichen wie beim letzten Schließen des Dialogfelds. Da es mehr als einen Weg gibt, das Dialogfeld während einer Anpassungsoperation zu schließen, hängen die angezeigten Werte von den Aktivitäten des Anwenders ab:

  • Verwendung der Schaltfläche Fit
    • Wenn keine Iterationen durchgeführt werden und die Schaltfläche Fit verwendet wird, um die Daten anzupassen und das Dialogfeld zu schließen, dann entsprechen die angezeigten Werte den Anfangswerten der Parameter.
    • Wenn mehrere Iterationen durchgeführt werden, der Fit aber noch nicht konvergiert, und die Schaltfläche Fit verwendet wird, führt Origin die restlichen Iterationen durch, um die Daten anzupassen, und schließt dann den Dialog. Im Modus Parameter ändern zeigt die Registerkarte Parameter jedoch die noch nicht konvergierten Werte an, die vor dem Klicken auf die Schaltfläche Fit vorhanden waren.
  • Verwendung der Schaltfläche OK
    • Wenn die Schaltfläche OK angezeigt wird und Sie darauf klicken, um den Dialog zu schließen, zeigt der Modus Parameter ändern die Parameterwerte, so wie sie in den Berichtsblatttabellen aufgeführt sind.
  • Verwendung der Schaltfläche Fertig
    • Wenn der Dialog mit der Schaltfläche Fertig geschlossen wird, zeigt Origin Anpassungsergebnisse und Parameterwerte zu diesem Zeitpunkt an, ohne weitere Iterationen durchzuführen. Dies geschieht unabhängig davon, ob der Fit konvergiert hat. Der Modus Parameter ändern zeigt die Parameterwerte zu dem Zeitpunkt, an dem auf die Schaltfläche Done geklickt wurde.
