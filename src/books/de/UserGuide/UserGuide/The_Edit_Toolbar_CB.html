Die Symbolleiste Bearbeiten

CB-Edit-Toolbar

Mit den Schaltflächen der Symbolleiste Bearbeiten können Sie Dateien kommentieren, suchen und wiederherstellen, außerdem Lesezeichen setzen.

Kommentiert ausgewählte Textzeilen.

Entfernt Kommentare von ausgewählten Textzeilen.

Sucht festgelegte Dateien.

Sucht das vorherige Auftreten der festgelegten Dateien.

Sucht das nächste Auftreten der festgelegten Dateien.

Ersetzt bestimmten Text mit einem anderen.

Wiederholt die letzte Aktion.

Öffnet den Dialog der Texteditoroptionen.

Setzt die Optionen auf Ihre Standardwerte zurück.

Die letzte Aktion wird rückgängig gemacht.

Wiederholt die zuvor rückgängig gemachte Aktion.

Wechselt zur entsprechenden Klammer.

Wechselt zwischen Lesezeichen.

Löscht Lesezeichen.

Wechselt zum nächsten Lesezeichen.

Wechselt zum vorherigen Lesezeichen.
