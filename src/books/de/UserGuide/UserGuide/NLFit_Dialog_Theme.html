Dialogdesign des NLFit

NLFit-Dialog-Theme

Wie bei den meisten Analysedialogen in Origin können Sie die gewählten Einstellungen im Dialog NLFit als Designdesign speichern. Um eine bestimmte Gruppe von Einstellungen aufzurufen, laden Sie einfach den Dialog Design neu.

Das Bedienelement Dialogdesign befindet sich oben im Dialog NLFit. Der aktuelle Dialogdesignname wird im Textfeld rechts von Dialogdesign angezeigt. Wenn kein Text angezeigt wird, wird kein Design verwendet. Die dreieckige Schaltfläche rechts vom Textfeld öffnet ein Kontextmenü, mit dem Sie Dialogdesigns speichern, laden oder löschen bzw. die Dialogeinstellungen zurücksetzen können.

Speichern als <Standard>

Speichert das aktuelle Design als Standarddesign für den Dialog NLFit.

Speichern

Speichert die Einstellungen im aktuellen Design.

Speichern unter...

Speichert das aktuelle Design unter einem anderen Namen. Ein Dialog wird geöffnet, in dem Sie den Designnamen vergeben und mit dem Design zu speichernde Komponenten auswählen können.

Designeinstellung

Wählen Sie Komponenten aus, die mit dem Design gespeichert werden sollen (gleiche Optionen wie unter Speichern unter). Weitere Einzelheiten finden Sie hier.

Systemstandard

Das Systemdesign wird in den Dialog NLFit gespeichert.

Löschen

Das Dialogdesign wird gelöscht.

Vergleichen

Die aktuellen Einstellungen werden mit den Einstellungen von Designname verglichen.

<Zuletzt verwendet>

Einstellungen aus der zuletzt durchgeführten Analyse
