Helligkeit


Die Helligkeit beschreibt die relative Helligkeit oder Dunkelheit einer bestimmten Farbe. Sie reicht von Schwarz bis Weiß. Die X-Funktion imgBrightness macht das Bild mit einem Anpassungswert, der die prozentuale Änderung jedes Pixels darstellt, dunkler oder heller. Ein negativer Wert hat ein dunkleres Bild zur Folge, während ein positiver Wert ein helleres Bild zur Folge hat. Um diese Funktion zu verwenden:

Wählen Sie Bild: Anpassungen: Helligkeit im Origin-Menü.

Weitere Informationen über diese Funktion finden Sie in der Dokumentation der X-Funktion imgBrightness in der Origin-Hilfe. Diese kann über das Befehlsfenster mit folgendem Befehl aufgerufen werden:

help imgBrightness
