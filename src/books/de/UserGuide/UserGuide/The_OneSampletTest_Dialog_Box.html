Eingabe

Geben Sie die Eingabedaten ein.

t-Test für Mittelwert

Testmittelwert Der zu testende hypothetische Mittelwert Wenn Sie einen Wert eingeben, werden die Null- und Alternativhypothese entsprechend angepasst.
Nullhypothese Die zu testende Nullhypothese

(Mittelwert = Testmittelwert)
Alternativhypothese Die Alternativhypothese

Optionen:

Mittelwert <>Testmittelwert

Wird für beidseitigen Test verwendet. Diese Option testet, ob der Stichprobenmittelwert sich von dem hypothetischen Mittelwert unterscheidet.

Mittelwert >Testmittelwert

Wird für den einseitigen Test nach oben verwendet. Diese Option testet, ob der Stichprobenmittelwert größer ist als der hypothetische Mittelwert.

Mittelwert < Testmittelwert

Wird für den einseitigen Test nach unten verwendet. Diese Option testet, ob der Stichprobenmittelwert kleiner ist als der hypothetische Mittelwert.
Signifikanzniveau Legen Sie das Signifikanzniveau fest.
Konfidenzintervall(e) Legen Sie fest, ob das/die Konfidenzintervall(e) für den Mittelwert berechnet wird/werden.

Berechnungseinzelheiten finden Sie unter Algorithmen (t-Test bei einer Stichprobe).
Konfidenzniveau(s) in % Legen Sie die Konfidenzniveaus (durch Komma getrennt) fest, (1-)%. Diese Option ist verfügbar, wenn Konfidenzintervall(e) aktiviert wird.

Trennschärfeanalyse

Tatsächliche Trennschärfe Berechnen Sie die Trennschärfe eines Tests (die Wahrscheinlichkeit, die Nullhypothese abzulehnen, wenn sie falsch ist).
Signifikanzniveau Diese Option ist nur verfügbar, wenn Hypothetische Trennschärfe und/oder Tatsächliche Trennschärfe aktiviert ist/sind. Dies ist das Signifikanzniveau (Alpha-Wert) für die Berechnung der Trennschärfe.
Hypothetische Trennschärfe Berechnen Sie die Trennschärfe des Tests für verschiedene Stichprobenumfänge.
Hypothetische/r Stichprobenumfang/-umfänge Nur verfügbar, wenn Hypothetische Trennschärfe aktiviert ist. Legen Sie den/die hypothetischen Stichprobenumfang/-umfänge für die Berechnung der Trennschärfe fest.
