Die Registerkarte Auswerten

Die Registerkarte Auswerten erlaubt es Ihnen, jeden Mathematica-Ausdruck, der als Skalar gelöst werden kann, auszuwerten. Diese Registerkarte ist dem "Notebook" von Mathematica sehr ähnlich.

Das Textfeld Eingabe

Das Textfeld Eingabe sollte zur Eingabe von Mathematica-Ausdrücken verwendet werden. Beachten Sie, dass in dieses Bearbeitungsfeld mehrere Zeilen eingegeben werden können.

Das Textfeld Ausgabe

Das Textfeld Ausgabe zeigt die Ergebnisse von Mathematica an. Die Ausgabe von nachfolgenden Auswertungen während einer Sitzung werden an die vorherige Ausgabe angehängt. Sie können klicken und mehrere Ausgabezeilen auswählen und diese anschließend in eine andere Anwendung kopieren oder die Zeilen löschen.

Die Schaltfläche Auswerten

Die Schaltfläche Auswerten sendet die Ausdrücke in das Eingabetextfeld Input in Mathematica und zeigt dann die Ergebnisse in dem Ausgabetextfeld Output an.
