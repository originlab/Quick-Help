Vorlage

ridgeline.otpu (installiert im Origin-Programmordner).

Hinweise

Das Ridgeline-Diagramm wird verwendet, um die Verteilung von mehreren numerischen Variablen zu visualisieren - einfach eine Reihe von vielen Dichtediagrammen, die in Y-Richtung gestapelt sind. Man sat auch, dass das Ridgeline-Diagramm dem Halben Violindiagramm ähnelt, nur mit vertauschter X- und Y-Achse.

Origin unterstützt vier Typen Ridgeline-Diagramm:

  • Verteilungskurve
    Es wird nur die Verteilungskurve gezeigt. Aber Sie können den Kurventyp und das Füllmuster ändern.
  • Verteilungskurve mit Datenpunkten
    Sie können entscheiden, wo die Datenpunkte gezeigt werden, auf der oder unter der Kurve.
  • Verteilungskurve mit Rugs
    Rugs werden unter der Kurve gezeigt. Dies ist eine Reihe von Verteilung + Rugs-Diagrammen.
  • Balkendiagramm mit Einzelblock
    Eine Reihe von Histogrammen.

Um ein Ridgeline-Diagramm benutzerdefiniert anzupassen, können Sie folgende Registerkarten verwenden:

  • Registerkarte Anzeige
  • Registerkarte Muster
  • Registerkarte Daten
  • Registerkarte Verteilung
