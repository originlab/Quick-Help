Der Dialog Arbeitsblattnavigation

Nav-Wks-Dialog

Verwenden Sie der Dialog In Arbeitsblättern navigieren, um Arbeitsblätter in der aktuellen Arbeitsmappe zu verwalten. Klicken Sie mit der rechten Maustaste auf einen der Reiter und klicken Sie dann auf Navigieren, um den Dialog zu öffnen.

Arbeitsblatttabelle

In der Arbeitsblatttabelle können Sie:

  • auf die Spaltenheader klicken, um Arbeitsblätter zu sortieren,
  • Klicken Sie doppelt auf das Arbeitsblatt, um das Blatt zu aktivieren.
  • Benennen Sie das Arbeitsblatt in der Zelle Name um.
  • Geben Sie einen Kommentar für das Arbeitsblatt in der Zelle Notizen ein.
  • Zeigen oder verbergen Sie ein Arbeitsblatt in der Spalte Zeigen.
  • Weitere Operationen sind möglich über das Kontextmenü, das Sie über einen Rechtsklick öffnen, einschließlich Duplizieren von Arbeitsblättern mit/ohne Daten, Ausschließen von Arbeitsblättern vom Zeichnen oder Importieren in Excel, Verschieben/Löschen/Leeren eines Arbeitsblatts. Siehe Details unten.

Sie können die Strg/Shift-Taste drücken, um mehrere Blätter auszuwählen, und diese Operationen dann gleichzeitig im Menü Aktionen oder im Kontextmenü per Rechtsklick durchführen.
