Datenanforderungen

Wählen Sie mindestens eine Y-Datenspalte (oder einen Bereich aus mindestens einer Spalte) aus. Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie Zeichnen: Einfache 2D: Fläche.

oder

Klicken Sie auf die Schaltfläche Flächendiagramm in der Symbolleiste 2D Grafiken.

Vorlage

AREA.OTP (im Origin-Programmordner installiert)

Hinweise

Die Fläche zwischen dem Diagramm und dem Y-Achsenwert "Von" wird standardmäßig für jedes Diagramm gefüllt. Wenn mehr als ein Diagramm zusammen in einem Layer gezeichnet wird, könnte eines das andere blockieren. Um ein Flächendiagramm gestapelt mit dem vorherigen Flächendiagramm zu zeichnen (wobei Sie die oberen Grenzen des vorherigen Flächendiagramms als Basislinie für das nächste Flächendiagramm verwenden), wählen Sie Grafik: Versatz gruppierter Daten im Layer: Kumulativ.
