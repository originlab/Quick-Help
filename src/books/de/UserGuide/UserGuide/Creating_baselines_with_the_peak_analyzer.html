Erstellen von Basislinien mit dem Impulsanalysator

Wenn die Option Basislinien erzeugen in der Gruppe Ziel auf der Seite Beginn des Impulsanalysators ausgewählt ist, dann können Sie den Impulsanalysator verwenden, um eine Basislinie für die Eingabedaten zu erstellen. Wenn der Impulsanalysator in diesem Modus verwendet wird, dann sind nur die Basislinienmodi Benutzerdefiniert und XPS verfügbar. Sie können die Ankerpunkte der Basislinie auf der Seite Modus Baseline definieren und sie dann verbinden, um eine benutzerdefinierte Basislinie auf der Seite Baseline erzeugen zu erstellen.

Weitere Informationen finden Sie auf der Seite Modus Baseline und der Seite Baseline erzeugen .
