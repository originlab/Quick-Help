Datenanforderungen

Wählen Sie mindestens eine Y-Spalte aus (oder einen Bereich aus mindestens einer Y-Spalte). Wählen Sie zwei Y-Spalten (oder einen Bereich aus zwei Y-Spalten) aus. Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Mehrere Y-Spalten können durch zwei X-Spalten in zwei Gruppen unterteilt werden. Ansonsten werden sie nach Abfolge verschiedenen Y-Achsen zugewiesen.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie Zeichnen > Mehrere Felder/Achsen: Säulen/Linie+Symbol mit doppeltem Y im Hauptmenü.

Vorlage

DblY_ColSymb.OTP (im Origin-Programmordner installiert)

Notizen

Dieser Diagrammtyp hat nur einen Layer und 2 Y-Achsen mit unterschiedlichen Achsenskalierungen wie beim Diagramm mit doppelter Y-Achse. In diesem Diagramm werden die Y-Spalte(n), die auf der linken Y-Achse basieren, als Säulen gezeichnet und die Y-Spalte(n), die auf der rechten Y-Achse basieren, als Linie + Symbol.

  • Dies ist eine Diagrammvorlage mit einem einzelnen Layer mit zwei unabhängigen vertikalen (Y)-Achsen.
  • Das Kontrollkästchen Jede Achse unabhängig neu skalieren auf der Registerkarte Zeigen im Dialog Achsen ist aktiviert.
  • Die Vorlage hat die gleichen Datenanforderungen und erstellt das gleiche Diagramm wie das Säulen-/Linien+Symboldiagramm mit 2Y mit zwei Layern.
