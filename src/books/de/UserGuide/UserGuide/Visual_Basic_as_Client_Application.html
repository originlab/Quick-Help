Microsoft® Visual Basic als Client-Anwendung verwenden

VB-client-app

Ein Hilfsmittel von Microsoft® Visual Basic® kann mit Hilfe der Unterstützung von Origins Automatisierungsserver als Client-Anwendung verwendet werden.

Beispielhaft wurde das Hilfsmittel unten erzeugt, um die simple Aufgabe der Weitergabe von zufälligen Daten an Origin zu erfüllen.

In den obigen Beispielbildern ist das Hilfsmittel von Visual Basic die Client-Anwendung. Dieses Hilfsmittel erzeugt einfach einige zufällige numerische Daten und Texte und platziert diese dann in einer Origin-Arbeitsmappe, die vom Anwender festgelegt wird. Sollte die Arbeitsmappe nicht existieren, wird eine neue Arbeitsmappe erstellt.

Das Hilfsmittel bietet Ihnen die Wahl, entweder die Daten im Blatt zu ersetzen oder sie an vorhandene Daten anzuhängen. Sie können sogar die Anzahl der Zeilen und Spalten des Blatts festlegen, das die Daten erhalten soll.

Die Origin-Anwendung kann zwischen sichtbaren und verborgenen Zuständen wechseln. Das Origin-Projekt kann auch beim Schließen der Visual Basic-Anwendung gespeichert werden.

Um dieses Beispiel in Origin zu testen, führen Sie SendTextDataToWks.exe aus, die sich in dieser zip-Datei im Unterordner \COM Server and Client\VB\SendTextDataToWks befindet.
