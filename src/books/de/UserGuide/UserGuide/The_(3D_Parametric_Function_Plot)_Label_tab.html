Die Registerkarte Beschriftung (Parametrisches 3D-Funktionsdiagramm)

PD-Dialog-3DParaFunc-Label-Tab

Festlegen von Funktionsheadern für unabhängige und abhängige Variablen auf der Registerkarte Beschriftung. Diese Registerkarte ist als Tabelle strukturiert. Es können X-, Y- und Z-Header (z.B. Langname, Einheiten und Kommentare) eingegeben werden. Diese Header können im Funktionsdiagramm angezeigt werden. Beispielsweise können Langname und Einheiten als Titel verwendet werden, und der Z-Kommentar kann als Legende angezeigt werden.
