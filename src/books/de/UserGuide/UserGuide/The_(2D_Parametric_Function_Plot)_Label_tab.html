Die Registerkarte (Parametrisches 2D-Funktionsdiagramm) Beschriftung

PD-Dialog-2DParaFunc-Label-Tab

 


Festlegen von Funktionsheadern für unabhängige und abhängige Variablen auf der Registerkarte Beschriftung. Diese Registerkarte ist als Tabelle strukturiert. Es können X- und Y-Header (z. B. Langname, Einheiten und Kommentare) eingegeben werden. Diese Header können im Funktionsdiagramm angezeigt werden. Beispielsweise können Langname und Einheiten als X- und Y-Achsentitel verwendet werden, und der Y-Kommentar kann als Legende angezeigt werden.
