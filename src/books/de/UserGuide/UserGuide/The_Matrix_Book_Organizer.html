Es gibt drei Hilfsbereiche, die mit dem Matrixfenster angezeigt werden können.

Der Organizer

Der Organizer kann zum Ansehen und Ändern von Kurzname, Langname und Kommentar der Matrixmappe verwendet werden. Sie können außerdem wählen, ob Sie den Datenmodus oder den Bildmodus zur Ansicht der Matrixdaten verwenden möchten. Wenn eine Datei (Bild, ASCII usw.) in die Matrix importiert wurde, behält der Organizer die mit der importierten Datei verbundenen Informationen bei. Zusätzlich können Sie einen Anwenderbaum im Organizer hinzufügen, um alle von Ihnen benötigten Informationen zu erfassen.

Der Skriptbereich

Der Skriptbereich wird zum Speichern von Skriptbefehlen mit einer Matrix oder Matrixvorlage verwendet. Wenn Sie auf die Schaltfläche Alle ausführen klicken, wird der Skript im Skriptbereich ausgeführt. Falls das Kontrollkästchen Ausführen von Skript aktiviert ist (Sie können diese Funktion im Kontextmenü des Skriptbereichs aktivieren oder deaktivieren), können Sie einen Befehl auch ausführen, indem Sie ihn in den Skriptbereich eingeben und auf Enter drücken.

Es wird ein Befehlsverlauf im Skriptbereich gespeichert, so dass Sie einen Überblick über die bereits ausgeführten Befehle erhalten.

Die Objektminiaturbilder

Die Miniaturbilder der Matrixobjekte ist nützlich, wenn sich mehrere Matrixobjekte in demselbem Matrixblatt befinden. Sie ermöglicht Ihnen eine einfache Navigation zwischen diesen Objekten.
