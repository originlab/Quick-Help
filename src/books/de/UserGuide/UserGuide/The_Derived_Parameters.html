Die abgeleiteten Parameter

Diese Seite bietet Optionen zum Definieren von Parametern, die zusätzlich aus den Funktionsparameterwerten berechnet werden. Diese Seite ist optional, d.h., Sie können direkt auf Weiter klicken, um zu der nächsten Seite zu gelangen, auf der weitere Einstellungen vorgenommen werden können. Bitte beachten Sie, dass Sie diese Werte auch im Dialogfeld NLFit während des Anpassungsvorgangs festgelegt werden können.

Bedienelemente des Dialogfelds

Gruppe Abgeleitete Parameter Bietet Optionen zum Festlegen von Bedeutung und Einheit für jeden abgeleiteten Parameter. Sie enthält die folgenden Spalten:
  • Name
    Führt die Namen aller abgeleiteten Parameter auf.
  • Bedeutung
    Optional können die Bedeutungen der abgeleiteten Parameter festgelegt werden.
  • Einheit
    Optional können die Einheiten der abgeleiteten Parameter festgelegt werden.
Abgeleitete Parametergleichungen Sie können die abgeleiteten Parametergleichungen hier eingeben. Im Folgenden fnden Sie ein Beispiel zum Schreiben von abgeleiteten Parametergleichungen. Angenommen, Sie haben einen Funktionsparameter mit der Bezeichnung w, dann können Sie zwei zusätzliche abgeleitete Parameter wie folgt definieren:
sigma = w/2;
FWHM = sqrt(2*ln(2)) * w;
