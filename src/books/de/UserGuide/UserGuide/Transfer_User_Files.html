Automatische Übertragung der Anwenderdateien nach Upgrade

Wir haben in Origin 2018 ein neues Verhalten zum automatischen Übertragen der Anwenderdateien aus Origin 2017 eingeführt.

Wenn Sie Origin 2017 bereits installiert und ein Upgrade auf eine höhere Version (wie Origin 2018) durchgeführt haben, wird beim ersten Start der höheren Version der folgende Popup-Dialog bei der Einrichtung Ihres Anwenderdateiordners (UFF) aufgerufen:

  • Wenn Sie auf Ja klicken, wird das Layout des Arbeitsbereichs von Origin 2017, wie die Anpassungen der Symbolleisten und der Status der angedockten Fenster, kopiert und angewendet.
  • Wenn Sie auf Nein klicken, wird das Standardlayout des Arbeitsbereichs verwendet.

Als Nächstes wird ein weiterer Popup-Dialog angezeigt, in dem Sie entscheiden, ob Sie die Anwenderdateien aus dem Anwenderdateiordner von Origin 2017 in den Anwenderdateiordner der aktuellen Origin-Version übertragen möchten.

  • Wenn Sie auf Ja klicken, werden alle Ihre Anpassungen von Origin 2017 automatisch auf die aktuelle Version übertragen.
Neben der direkten Übertragung der vom Benutzer gespeicherten/benutzerdefinierten Dateien (wie .opj-Dateien, neue Vorlagen-/Designdateien, .FDF-Dateien und .oif-Dateien...) werden auch die benutzerdefinierten Abschnitte in den Dateien der Standardeinstellungen (wie .ini-Dateien) an die entsprechenden Einstellungsdateien der aktuellen Version geliefert. Bitte beachten Sie, dass während der Lieferung alle exe/UFF-Pfade in den Einstellungsdateien jeweils in die aktuelle Version umgewandelt werden, damit alle betroffenen Dateien korrekt und auf dem neuesten Stand sind.
  • Wenn Sie auf Nein klicken, wird keine Anwenderdatei automatische übertragen. Sie können Anwenderdateien aber immer manuell mit Hilfe des Dialogs Anwenderdateien übertragen unten übertragen.

Anwenderdateien manuell übertragen

Der Dialog Anwenderdateien übertragen wurde für Anwender entwickelt, um Dateien aus dem Anwenderdateiordner älterer Versionen von Origin in den Anwenderdateiordner der aktuellen Version von Origin zu übertragen.

Zum Öffnen dieses Hilfsmittels:

Wählen Sie Einstellungen: Anwenderdateien übertragen im Origin-Menü.

Dateien anzeigen

Es gibt sechs Kontrollkästchen in dem Dialog, um festzulegen, ob bestimmte Dateitypen in den Listen aufgeführt werden sollen.

Projekte & Vorlagen OPJ, OGW, OTW, OTP, OTM
Einstellungen (Menü, Schaltflächen...) OMC, XML, INI, TXT
Anpassungsfunktionen FDF
Designs, Importfilter, Paletten OIS, ODT, OTH, OIF, ODQ, ODS
Code, Hilfsmittel OGS, H, C, CPP, OXF, OPK, OPX
Weitere DOC, DOCX, PDF

Feld der Quellordner (linkes Bedienfeld)

Auswahlliste Speicherort

In dieser Auswahlliste können Sie den Anwenderdateiordner von jeder Origin-Version, die Sie installiert haben, oder Festgelegten Ordner auswählen.

Um einen festgelegten Ordner für das Feld auszuwählen,

  1. wählen Sie Festgelegter Ordner in der Auswahlliste.
  2. Klicken Sie auf die Schaltfläche Durchsuchen auf der rechten Seite und wählen Sie einen gewünschten Ordner im geöffneten Dialog Nach Ordnern suchen.
  3. Wenn Sie eine Festgelegten Ordner gewählt haben, ist das Kontrollkästchen Rekursiv verfügbar. Aktivieren Sie dieses Kontrollkästchen. Die unten aufgelisteten Dateien umfassen dann auch die Dateien in allen Unterordnern des festgelegten Ordners.

Pfad

Der detaillierte Pfad des ausgewählten Ordners für den Quellspeicherort wird gezeigt.

Liste der Quelldateien

Dieses Bedienfeld führt die detaillierten Informationen (einschließlich Dateiname, Dateityp, Dateigröße und Änderungsdatum) für die Dateien auf, die die aktuellen Kriterien von Dateityp und Quellspeicherort erfüllen.

Der aktuelle Status der aufgelisteten Dateien im Vergleich zu ihren Gegenstücken in der aktuellen Version wird durch die Farbe des Dateinamens gekennzeichnet.

Farbe Bedeutet
Grün Die Datei ist älter als ihr Gegenstück im Anwenderdateiordner der aktuellen Version.
Rot Die Datei ist neuer als ihr Gegenstück im Anwenderdateiordner der aktuellen Version.
Hinweis: Wenn das Kontrollkästchen Nur neuere Dateien zeigen aktiviert ist, werden nur die Dateien gezeigt, die neuer sind als ihre Gegenstücke im aktuellen Anwenderdateiordner.
Blau Die Datei ist mit der Datei mit dem gleichen Namen in dem anderen Ordner identisch.
Schwarz Die Datei ist neu. Die Datei existiert nur im Quellordner. Origin kann ihr Gegenstück im Anwenderdateiordner der aktuellen Version nicht finden.

In diesem Bedienfeld können Sie festgelegte Dateien im Quellspeicherort auswählen und auf die Schaltfläche Hinzufügen klicken, um sie zur Liste der Übertragungsdateien hinzuzufügen.

Übergeordneten Ordner öffnen

Um den übergeordneten Ordner für eine bestimmte Datei zu öffnen:

  1. Klicken Sie mit der rechten Maustaste auf eine Datei, entweder in der Quellordnerliste oder in aktuellen Anwenderdateiordnerliste.
  2. Wählen Sie im Kontextmenü Übergeordneten Ordner öffnen.

Liste der Übertragungsdateien

Wenn das Kontrollkästchen Aktuelle Anwenderdateien zeigen (grau) aktiviert ist, zeigt diese Liste die Dateien im Anwenderdateiordner der aktuellen Version in grau an.

Es gibt eine temporäre Liste für die Dateien, die in den Anwenderdateiordner der aktuellen Version übertragen werden. Sie zeigt die Dateien an, die von der Quelldateiliste hinzugefügt wurden. Sie können durch die Farbgebung der Dateinamen sagen, ob die aufgelisteten Dateien neuer oder älter als ihre Gegenstücke im Anwenderordner der aktuellen Version von Origin sind.

Farbe Bedeutet
Grün Die Datei ist älter als ihr Gegenstück im Anwenderdateiordner der aktuellen Version.
Rot Die Datei ist neuer als ihr Gegenstück im Anwenderdateiordner der aktuellen Version.
Blau Die Datei ist mit der Datei mit dem gleichen Namen im Anwenderdateiordner der aktuellen Version identisch.
Schwarz Die Datei ist neu. Die Datei existiert nur im Quellordner. Origin kann ihr Gegenstück im Anwenderdateiordner der aktuellen Version nicht finden.

 

Hinweis: Die Schaltfläche Übertragen ist verfügbar, wenn es Dateien mit dem Status Neuer als aktuell, Älter als aktuell oder Neue Datei gibt. Nachdem die Dateien in den Anwenderdateiordner der aktuellen Version übertragen wurden, wird die Farbe der Dateinamen Blau, sowohl in der Liste der Quelldateien als auch in der Liste der Übertragungsdateien.


Sobald Sie die Datei(en) zur aktuellen Version übertragen haben, indem Sie auf die Schaltfläche Übertragen klicken, wird das Fenster Meldungsprotokoll automatisch geöffnet und eine Nachricht mit allen übertragenen Dateien angezeigt.

Schaltfläche Hinzufügen/Entfernen

Dateien, die Sie in der Liste der Quelldateien ausgewählt haben, können Sie in die Liste der Übertragungsdateien kopieren, indem Sie auf die Schaltfläche Hinzufügen klicken. Wenn die Dateien zur Liste der Übertragungsdateien hinzugefügt werden, können sie vor der Übertragung in den Anwenderdateiordner der aktuellen Version durch Klicken auf die Schaltfläche Entfernen entfernt werden.

Schaltfläche Löschen

Sie können Dateien, die im Feld der Liste der Übertragungsdateien ausgewählt sind, aus dem Anwenderordner der aktuellen Version von Origin entfernen, indem Sie auf die Schaltfläche Löschen klicken.

Export an OPX

Um Dateien an OPX zu exportieren:

  1. Aktivieren Sie das Kontrollkästchen Export an OPX zulassen. Anschließend wird der Hinweis n Datei(en) werden exportiert unten links gezeigt.
  2. Klicken Sie auf die Schaltfläche Export . Die Dateien werden in eine OPX-Datei exportiert, die Sie mit anderen Anwendern teilen können.
  3. Nachdem die Dateien exportiert werden, wird der Dateipfad im Meldungsprotokoll ausgegeben.
Hinweis: Die ausgeblendeten Dateien können nicht exportiert werden.
