Das hierarchische Clusterberichtsblatt

Deskriptive Statistik

Die Tabelle der deskriptiven Statistik ist nützlich beim Beobachten der Mittelwerte und Standardabweichungen der Variablen, die aufdecken können, ob die Variablen mit unterschiedlichen Skalierungen gemessen wurden. In diesem Fall sind Mittelwerte und Standardabweichungen sehr unterschiedlich. Deshalb möchten Sie während der Analyse vielleicht die Variablen standardisieren.

Distanzmatrix

Die Distanzmatrix enthält die tatsächlichen Distanzen, wodurch die Ähnlichkeiten aufgezeigt werden, die für ein beliebiges Paar von Beobachtungen/Variablen berechnet werden.

Clusterstufen

Die Tabelle der Clusterstufen führt die Einzelheiten zu den Clustern von Beobachtungen und Variablen auf. Sie kann verwendet werden, wenn es nur wenige Variablen und Beobachtungen gibt. Wenn der Stichprobenumfang groß ist, wird empfohlen, das Dendrogamm zu verwenden, das die Clusterstufen grafisch darstellt.

Dendrogramm

Das Dendrogramm stellt das wichtigste Ergebnis der Clusteranalyse dar. Es führt alle Stichproben auf und gibt an, auf welcher Ähnlichkeitsstufe zwei beliebige Cluster miteinander verbunden wurden. Die Position der Linie auf der Skala deutet auf die Distanz hin, auf die Cluster miteinander verbunden wurden. Das Dendrogramm wird auch als sinnvolles Hilfsmittel zum Bestimmen der Anzahl der Cluster eingesetzt. Beachten Sie einen plötzlichen Anstieg der Differenz zwischen zwei nebeneinander liegenden Stufen, da dies ein Hinweis auf die angemessene Anzahl von zu berücksichtigenden Clustern ist. Einzelheiten lesen Sie bitte im Tutorial nach.

Wenn Sie den Dialog der hierarchischen Clusteranalyse (hcluster) verwenden, können Sie einen phylogenetischen Baum mit auswählbaren Knoten ausgeben, der mit Hilfe eines Kontextmenüs bearbeitet werden kann:

  • Verwenden Sie Unterbäume vertauschen, um Cluster, die sich direkt unter dem aktuell ausgewählten Knoten befinden, zu vertauschen.
  • Mit diesem Knoten rerooten ist für die Ausgabe des Dialogs hcluster deaktiviert.
  • Baum zurücksetzen macht die Austauschoperationen rückgängig und stellt die ursprüngliche Baumstruktur wieder her.
  • Zweig in neues Fenster duplizieren gibt die aktuell ausgewählten Cluster in einem neuen Diagrammfenster aus.
