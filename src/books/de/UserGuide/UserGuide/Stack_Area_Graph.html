Datenanforderungen

Wählen Sie mindestens eine Y-Datenspalte (oder einen Bereich aus mindestens einer Spalte) aus. Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie Zeichnen > Einfache 2D: Stapelfläche im Menü.

oder

Klicken Sie auf die Schaltfläche Stapelfläche auf der Symbolleiste 2D Grafiken.

Vorlage

STACKAREA.OTP (im Origin-Programmordner installiert)

Hinweise

Falls eine X-Datenspalte (oder ein Bereich einer Spalte) ausgewählt ist, wird die Fläche zwischen dem Diagramm und dem Y-Achsenwert "Von" standardmäßig gefüllt. Falls mehrere Y-Spalten (oder Bereiche) ausgewählt sind, wird jede aufeinanderfolgende Wertespalte als ein Flächendiagramm angezeigt, das auf das vorherige Flächendiagramm gestapelt ist. Die oberen Grenzen der ersten Fläche werden als Basislinie des nächsten Flächendiagramms betrachtet.
Um jedes Diagramm so zu zeichnen, dass die Fläche zwischen dem Diagramm und dem Y-Achsenwert "Von" gefüllt wird, wählen Sie Grafik: Versatz gruppierter Daten im Layer: Keine.

Um ein Stapelflächendiagramm mit 100 % zu erstellen, klicken Sie doppelt auf das Diagramm, um den Dialog Details Zeichnung zu öffnen. Aktivieren Sie das Kontrollkästchen Für Kumulativ/Inkrementell auf Prozent normieren auf der Registerkarte Stapeln auf Layerebene.
