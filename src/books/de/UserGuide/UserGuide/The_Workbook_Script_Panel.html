Der Skriptbereich der Arbeitsmappe

Arbeitsmappen und Matrixmappen haben jetzt einen Skriptbereich, der auf der rechten Seite angezeigt wird. Dieser Bereich kann ein- oder ausgeblendet werden, indem Sie mit der rechten Maustaste auf den Seitentitel klicken und das Kontextmenü "Skriptbereich" verwenden.

Skript kann in der unteren Hälfte dieses Bereichs eingegeben werden. Die obere Hälfte zeigt den Verlauf des ausgeführten Skripts. Die Mappe kann anschließend als Fenster (OGW) oder als Vorlage (OTW) gespeichert werden. Außerdem kann das gesamte OPJ zusammen mit der Mappe gespeichert werden. In allen Fällen wird der Skript mit dem Objekt gespeichert und ist in Zukunft verfügbar.

Um das Skript auszuführen, wählen Sie eine oder mehrere Zeilen im unteren Bereich und drücken ENTER. Sie können auch alle Zeilen ausführen, indem Sie auf die Schaltfläche "Alle Ausführen" klicken.

Sie können Skript, das Sie häufige rmit einem speziellen Typ Mappe/Daten ausführen möchten, in diesen Bereich eingeben und von dort aus ausführen.
