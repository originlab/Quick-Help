Die Registerkarte Beschriftung für spiralförmige Balkendiagramme (Details Zeichnung)

PD-Dialog-Spiral-Label-Tab

Titel

Legen Sie fest, ob der Diagrammtitel in der Mitte des Diagramms angezeigt werden soll.

Quelle

Legen Sie fest, welche Informationen im Titel gezeigt werden: Spaltenüberschriften, Kommentare, Einheiten oder Abtastintervall.

Wenn Sie Spaltenüberschriften auswählen, wird der Spaltenlangname als Titel gezeigt. Wenn es keinen Langnamen gibt, wird der Spaltenkurzname gezeigt.

Schriftart

Passen Sie die Schriftart des Titels benutzerdefiniert an.

Textumbruch nach Anzahl der Zeichen

Legen Sie eine ganze Zahl fest, um den Text umzubrechen.

Beschriftungen der Hilfsstriche

Legen Sie fest, wie die Beschriftungen der Hilfsstriche auf der spiralförmigen Achse gezeigt werden.

Geben Sie

Legen Sie den Werttyp für die Hilfsstrichsbeschriftungen fest. Weitere Einzelheiten zu diesen Typen finden Sie unter Optionen der Hilfsstrichsbeschriftungstypen im Dialog Achsen.

Schriftart

Passen Sie die Schriftwart der Hilfsstrichsbeschriftungen auf der spiralförmigen Achse benutzerdefiniert an.

Anzeige

Legen Sie fest, wie die Werte als Hilfsstrichsbeschriftungen angezeigt werden. Weitere Einzelheiten zu diesen Typen finden Sie unter Optionen der Hilfsstrichsbeschriftungstypen im Dialog Achsen.

Versatz

Legen Sie fest, ob die Hilfsstrichsbeschriftungen mit einem Versatz von den Hilfsstrichen gezeigt werden.

Zeigen für

Legen Sie fest, welcher Teil der Hilfsstriche über Beschriftungen verfügt, nur die äußere Wicklung, die innere Wicklung, äußere und innere Wicklung, oder alle Hilfsstriche.

Im Kreis

Wenn Sie Nur außen für die Option Zeigen für auswählen, können Sie dieses Kontrollkästchen aktivieren, um die Hilfsstrichsbeschriftungen der äußeren Wicklung im Kreis zu zeigen.

Zwischen Hilfsstrichen

Legen Sie fest, ob die Beschriftungen der Hilfsstriche zwischen nebeneinander liegenden Hilfsstrichen gezeigt wird.

Hilfsstriche

Legen Sie fest, wie die Hilfsstriche gezeigt werden.

Inkrement der großen Hilfsstriche

Legen Sie das Inkrement für die großen Hilfsstriche fest.

Länge

Legen Sie die Länge der großen Hilfsstriche auf der spiralförmigen Achse fest.

Linie

Passen Sie die spiralförmige Achsenlinie benutzerdefiniert an.

Anzahl der kleinen Hilfsstriche

Legen Sie fest, wie viele kleine Hilfsstriche zwischen zwei nebeneinander liegenden großen Hilfsstrichen gezeigt werden.

Länge

Legen Sie die Länge von den gezeigten kleinen Hilfsstrichen fest.

Farbe

Legen Sie die Linienfarbe der spiralförmigen Achse fest.

Verankern

Legen Sie fest, die Position der großen Hilfsstriche zu verankern. 0 bedeutet, dass der Hilfsstrich sich am Anfang des Balkens befindet. Die Einheit ist die gleiche wie die Einheit der Lücke zwischen Balken.

Y-Achse zeigen

Aktivieren Sie diese Option, um eine Y-Achse am Ende der spiralförmigen Achse zu zeigen.

Wert von Minimum/Maximum

Standardmäßig ist das Kontrollkästchen Auto daneben aktiviert. Der minimale und der maximale Y-Wert werden in diese zwei Felder eingegeben. Deaktivieren Sie das Kontrollkästchen Auto, um den von Ihnen gewünschten Wert einzugeben und damit den Achsenanfang und das Achsenende festzulegen. Diese Einstellung hat keinen Einfluss auf den Zeichnungsbereich des Balkens. Nur die Y-Achse wird in einem anderen Bereich gezeigt.

Anzeige

Legen Sie den Werttyp für die Hilfsstrichsbeschriftungen der Y-Achse fest. Weitere Einzelheiten zu diesen Typen finden Sie unter Optionen der Hilfsstrichsbeschriftungstypen im Dialog Achsen.

Schriftart

Passen Sie die Schriftart der Hilfsstrichsbeschriftungen auf der Y-Achse benutzerdefiniert an.

Anzahl der kleinen Hilfsstriche

Legen Sie fest, wie viele kleine Hilfsstriche auf der Y-Achse gezeigt werden.

Versatz

Fügen Sie einen Versatz für die Hilfsstrichsbeschriftungen von den Hilfsstrichen hinzu.
