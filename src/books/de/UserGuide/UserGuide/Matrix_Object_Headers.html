Matrixobjektheader

Matrixobjektheader können im Dialogfeld Matrixdimensionen und Beschriftungen festgelegt werden (wählen Sie Matrix: Dimensionen/Beschriftungen festlegen im Menü). Die Registerkarte Z-Beschriftungen ist ein tabellenähnliches 2D-Gitter, in das Sie Matrixheader wie Langname, Einheiten, Kommentare etc. eingeben können. Diese Header wie z.B. Langname kann in Matrixminiaturbildern gezeigt oder als Z-Achsenbeschriftungen in 3D-Diagrammen verwendet werden.

Beim Bearbeiten von Matrixobjektbeschriftungen können Sie mit der rechten Maustaste auf die Zelle klicken und die folgenden Optionen aufrufen:

Wert nach rechts anwenden Wendet den aktuellen Wert auf alle Zellen zur rechten Seite an.
Wert nach links anwenden Wendet den aktuellen Wert auf alle Zellen zur linken Seite an.
Systemparameter zeigen Zeigt alle Systemparameter.
Benutzerdefinierte Parameter hinzufügen Fügt einen benutzerdefinierten Parameter hinzu.
Löschen Löscht einen benutzerdefinierten Parameter.
Zur Spalte des aktiven Objekts gehen Wechselt zu der Spalte in dieser Tabelle, die dem aktiven Matrixobjekt entspricht.
