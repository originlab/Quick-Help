Mehrere .omc-Dateien verwalten

Manage-Multi-OMCFile

Die Konfiguration der benutzerdefinierten Menüs kann als .omc-Datei gespeichert werden, die ganz leicht mit anderen geteilt werden kann. .omc-Dateien sollten in den Ordnern Anwenderdateiordner, Gruppenordner und Installationsordner von Origin abgelegt werden. Standardmäßig werden alle .omc-Dateien in diesen Ordnern automatisch unter Einstellungen: Menü aufgeführt. Origin listet zuerst die .omc-Dateien im Anwenderdateiordner auf, dann im Gruppenordner und dann im Installationsordner von Origin. Die Reihenfolge der Dateien im gleichen Ordner wird alphabetisch entsprechend der Dateinamen bestimmt.

Die omc.ini-Datei kann Ihnen beim Verwalten Ihrer .omc-Dateien helfen. Mit der omc.ini-Datei können Sie:

  • Die maximale Anzahl der Dateien setzen, die unter Einstellungen: Menü aufgeführt werden.
  • Die Anzeigereihenfolge dieser Dateien unter Einstellungen: Menü ändern.
  • Eine .omc-Datei als Standardoption unter Einstellungen: Menü festlegen.
Hinweis:

Die omc.ini kann im Anwenderdateiordner oder Gruppenordner oder Installationsordner von Origin gespeichert werden. Sie kann nur die .omc-Dateien im gleichen Ordner verwalten.

Die Organisation einer omc.ini-Datei

Eine omc.ini-Datei sollte zwei Abschnitte enthalten. Der erste Abschnitt wird als Config bezeichnet und kann dazu verwendet werden, die maximale Anzahl der Dateien in einem Ordner festzulegen, die unter Einstellungen: Menü aufgeführt werden, sowie die Anzeigereihenfolge der Dateien. Der andere Abschnitt hat die Bezeichnung Optionen. Hier kann eine .omc-Datei als Standardmenüoption unter Einstellungen: Menü angegeben werden.

Angenommen, wir haben vier .omc-Dateien, ImageProcessing.omc, Curve Fititng.omc, Statistics.omc und SignalProcessing.omc im Anwenderdateiordner. Wir wollen, dass nur die ersten drei Dateien gezeigt werden. Die gewünschte Anzeigereihenfolge ist Statistics.omc, ImageProcessing.omc und CurveFitting.omc. Außerdem soll ImageProcessing.omc als Standard festgelegt sein. Die folgende omc.ini-Datei sollte sein:

[Config] NumOMCLevels=3 ;Menu will ordered as the file list below OMCFile1=Statistics OMCFile2=ImageProcessing OMCFile3=CurveFitting [Options] ; Default selected menu OMCLevel=ImageProcessing
Hinweis:
  1. Origin behält automatisch die letzte Menüauswahl unter Einstellung: Menü als Standardeinstellung bei. Die Standardoption in der omc.ini-Datei ist nur wirksam, wenn auf keine .omc-Datei unter Einstellung: Menü zugegriffen wurde.
  2. Es wird keine omc.ini -Datei mit Origin ausgeliefert. Wenn Sie .omc-Dateien verwalten möchten, sollten Sie sich selbst eine omc.ini-Datei erstellen.

Beispiel zum Erstellen einer omc.ini-Datei

In diesem Abschnitt wird ein Beispiel zum Erstellen einer omc.ini-Datei unter dem Betriebssystem Windows erläutert, die im vorherigen Abschnitt verwendet wird.

  1. Erstellen Sie ein neues Textdokument im Anwenderdateiordner. Die Dateierweiterung sollte .txt sein (Aufgrund Ihrer Betriebssystemeinstellungen wird die Dateierweiterung möglicherweise nicht angezeigt. Sie können Extras: Ordneroptionen in Windows 7 auswählen, um den Dialog zu öffnen, und dann zur Registerkarte Ansicht gehen. Deaktivieren Sie in der Gruppe Erweiterte Einstellungen das Kontrollkästchen Erweiterungen bei bekannten Dateitypen ausblenden). Ändern Sie die Dateierweiterung dann von .txt in .ini. Benennen Sie danach die Datei in omc.ini um.
  2. Öffnen Sie die omc.ini-Datei mit Notepad. Kopieren Sie den Text, der im vorherigen Abschnitt gezeigt wurde, und fügen Sie ihn in die omc.ini-Datei ein. Speichern Sie dann die omc.ini-Datei und schließen Sie sie.
  3. Starten Sie Origin erneut. Sie können sehen, dass jetzt nur die ersten 3 Menüs unter Einstellungen: Menü aufgelistet werden.
