Dialog Anwenden auf

AxesRef-Apply-To

Dieser Dialog wird durch Klicken auf die Schaltfläche Anwenden auf... unten im Dialog Achsen geöffnet. Sie können das Format der Hilfsstrichsbeschriftungen, Linie und Hilfsstriche sowie Gitternetzlinien festlegen, um diese von der Quellachse auf die Zielachse zu übertragen.

Inhalt

  • 1 Anwenden von
  • 2 Anwenden bis
  • 3 Alle unten
  • 4 Beschriftungen der Hilfsstriche
  • 5 Linie und Hilfsstriche
  • 6 Gitternetzlinien

Anwenden von

Legen Sie die Quellachse fest, deren Einstellungen übernommen werden sollen.

Anwenden auf

Legen Sie die Zielachse fest, auf die die Einstellungen angewendet werden sollen.

Alle unten

Alle untenstehenden Elemente werden ausgewählt, um Sie von der Quellachse auf die Zielachse zu übertragen.

Beschriftungen der Hilfsstriche

Wählen Sie das Format der Hilfsstrichsbeschriftungen aus, um es von der Quellachse auf die Zielachse zu übertragen.

Linie und Hilfsstriche

Wählen Sie das Format für Linie und Hilfsstriche aus, um es von der Quellachse auf die Zielachse zu übertragen.

Gitternetzlinien

Legen Sie fest, ob das Format der Gitternetzlinien von der Quellachse auf die Zielachse übertragen werden soll.
