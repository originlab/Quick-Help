Datenanforderungen

Wählen Sie mindestens eine Y-Spalte aus (oder einen Bereich aus mindestens einer Y-Spalte). Die X-Spalte wird als Titel der Radialachse um das Diagramm herum angezeigt. Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

  • Linie + Symbol, Radar
Wählen Sie im Menü Zeichnen: Spezialisiert: Radar.
oder
Klicken Sie auf die Schaltfläche Radar auf der Symbolleiste 2D Grafiken.
  • Radar mit Linienfüllung: Wählen Sie im Menü Zeichnen > Spezialisiert: Linienfüllung.
  • Radar mit Linie: Wählen Sie im Menü Zeichnen > Spezialisiert: Linie.
  • Radar mit Symbol: Wählen Sie im Menü Zeichnen > Spezialisiert: Symbol.

Vorlage

Radar.OTP (im Origin-Programmordner installiert)
