Datenanforderungen

Wählen Sie mindestens eine XY-Spalte aus.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen > Statistisch: Rand.

Origin öffnet den Dialog plot_marginal.

Passen Sie die Optionen im Dialog benutzerdefiniert an und klicken Sie dann auf OK, um das Diagramm zu erstellen. Bitte lesen Sie weitere Einzelheiten zum Erstellen und benutzerdefinierten Anpassen von Randdiagrammen.

Vorlage

Marginal_Scatter.OTPU; Marginal_Contour.OTPU (installiert im Origin-Programmordner).

Hinweise

  1. Sie können marginale Boxdiagramme und Randverteilungshistogramme direkt im Menü Zeichnen erstellen.
  2. Nachdem Sie ein Randdiagramm gezeichnet haben, können Sie doppelt auf jeden Layer klicken, um Details Zeichnung zu öffnen und es weiter benutzerdefiniert anzupassen oder die Zwischendaten zu überprüfen.
