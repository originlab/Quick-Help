Parameter der partiellen Ableitungen

Das Festlegen partieller Ableitungen Ihrer Funktion kann die Zeit reduzieren, die für die Durchführung einer Reihe von Iterationen notwendig ist. Allerdings ist es bei der verbesserten Anpassungsgeschwindigkeit durch Origin C möglicherweise nicht erforderlich, partielle Ableitungen festzulegen.

Wenn Sie partielle Ableitungen festlegen möchten, beachten Sie, dass Sie die Ableitungen der Funktion der abhängigen Variablen entsprechend der Anpassungsparameter festlegen müssen. Außerdem müssen partielle Ableitungen gemäß aller Anpassungsparameter festgelegt werden.

Wenn Sie sich entschieden haben, die Funktion mit Origin C zu definieren, werden die Variablen für die partiellen Ableitungen im Code-Builder vordefiniert. Sie müssen für sie nur den Ausdruck eingeben.

Wenn die Funktion z.B. lautet:

Wenn Sie dann eine Origin C-Anpassungsfunktion definieren, sind die Variablen dy_P1, dy_P2 und dy_P3 verfügbar und können die Funktion mit partiellen Ableitungen definieren durch:

y = P1 + P2 * x + exp(P3 * x);
dy_P1 = 1;
dy_P2 = x;
dy_P3 = x * exp(P3 * x);

Beachten Sie, dass, wenn Sie nicht Origin C verwenden und Ihre Funktion mit LabTalk definieren mit Hilfe der auf Y-Skript gesetzten Auswahlliste Form (die einzige Form, die mit der Verwendung von Ableitungen kompatibel ist), die Schreibweise für die partiellen Ableitungen unterschiedlich ist. In diesem Fall wären die partiellen Ableitungen für das oben stehende Beispiel:

y = P1 + P2 * x + exp(P3 * x);
y'P1 = 1;
y'P2 = x;
y'P3 = x * exp(P3 * x);
