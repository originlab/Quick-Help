Form der Eingabedaten

Form der Eingabedaten
  • Zusammengefasst
Führen Sie den Test mit Hilfe einer zusammenfassenden Statistik durch, die Sie manuell in den Dialog eingeben.
  • Roh
Führen Sie den Test mit Hilfe eines Datensatzes durch, der aus "Erfolgs-" bzw. "Fehler"werten besteht. Der Datensatz darf nicht mehr als zwei eindeutige Werte enthalten.

Test von Anteilen

Testanteil Der zu testende hypothetische Anteil
Nullhypothese Die zu testende Nullhypothese (Stichprobenanteil = Testanteil).
Alternativhypothese Optionen:
Proportion <> Testanteil
Ein beidseitiger Test von Anteilen, der zu beantworten versucht, ob der Anteil der Stichprobe = Testanteil.
Proportion > Testanteil
Ein einseitiger Test von Anteilen nach oben, der zu beantworten versucht, ob der Anteil der Stichprobe größer ist als der Testanteil.
Proportion < Testanteil
Ein einseitiger Test von Anteilen nach unten, der zu beantworten versucht, ob der Anteil der Stichprobe kleiner ist als der hypothetische Anteil.
Signifikanzniveau Das Signifikanzniveau des Tests
Konfidenzintervall(e) Berechnen Sie das Konfidenzintervall für den wahren (Grundgesamtheit) Anteil.
Konfidenzniveau(s) in % Legen Sie die Konfidenzniveaus fest, für die die Konfidenzintervalle berechnet werden. Diese Option ist nur verfügbar, wenn Konfidenzintervall(e) aktiviert ist.

Testmethode

Origin verwendet zwei Methoden zum Berechnen von p-Werten und Konfidenzintervallen für Tests von Anteilen bei einer Stichprobe (Dokumentation zur Approximation der Normalverteilung und zum Binomialtest finden Sie auf der Seite zum Algorithmus):

Approximation der Normalverteilung Die Approximation der Normalverteilung wird immer berechnet. Im Allgemeinen erhalten Sie ein befriedigendes Ergebnis, wenn Erfolge ≥ 10 und wenn Stichprobenumfang - Erfolge ≥ 10. Die Qualität der Approximation neigt dazu, sich mit zunehmendem Stichprobenumfang zu verbessern.
Binomialtest Der Binomialtest wird verwendet.

Analyse der Trennschärfe

Tatsächliche Trennschärfe Berechnen Sie die Trennschärfe eines Tests (die Wahrscheinlichkeit, die Nullhypothese abzulehnen, wenn sie falsch ist).
Signifikanzniveau Diese Option ist nur verfügbar, wenn Hypothetische Trennschärfe und/oder Tatsächliche Trennschärfe aktiviert ist/sind. Dies ist das Signifikanzniveau (Alpha-Wert) für die Berechnung der Trennschärfe.
Hypothetische Trennschärfe Berechnen Sie die Trennschärfe des Tests für verschiedene Stichprobenumfänge.
Hypothetische/r Stichprobenumfang/-umfänge Nur verfügbar, wenn Hypothetische Trennschärfe aktiviert ist. Legen Sie den/die hypothetischen Stichprobenumfang/-umfänge für die Berechnung der Trennschärfe fest.
