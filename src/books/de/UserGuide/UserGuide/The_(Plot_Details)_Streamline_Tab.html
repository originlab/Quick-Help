Die Registerkarte Feldlinien (Details Zeichnung)

PD-Dialog-Streamline-Tab


Diese Registerkarte ist nur verfügbar, wenn Sie ein Feldliniendiagramm erstellen. Sie enthält Bedienelemente, um die Linie, den Pfeil und Zusätzliche Anfangspunkte für das Feldliniendiagramm benutzerdefiniert anzupassen.

Linie

Farbe

Wählen Sie aus dieser Auswahlliste die gewünschte Linienfarbe aus.

Diese Option kann andere Matrixobjekte verwenden, um die Farbabbildung der Feldlinien zu steuern. Informationen zur Farbabbildung finden Sie unter Datensatz zum Einstellen der Diagrammfarbe verwenden.

Stil

Wählen Sie die gewünschte Verbindungslinie aus dieser Auswahlliste aus.

Breite

Wählen Sie oder geben Sie die gewünschte Linienbreite in dieses Auswahlfeld ein. Die Linienbreite wird in Punkten gemessen, wobei 1 Punkt=1/72 Zoll entsprechen.

Dichte

Wählen Sie oder geben Sie die gewünschte Dichte der Linien in dieses Kombinationsfeld ein.

Segmentierung für jeden Pfeil

Die Verknüpfung wird nach Pfeil unterbrochen, so dass jedes Segment nur einen Pfeil zeigt.

Wenn dieses Kontrollkästchen aktiviert ist, ist die Option Abstand zwischen Segmenten (%) aktiviert.

Abstand zwischen Segmenten (%)

Wählen Sie oder geben Sie in dieses Kombinationsfeld einen Wert ein, der den Abstand zwischen den Segmenten definiert. Der Wert ist ein Prozentwert der Segmente.

 

Pfeil

In diesem Bedienfeld können Sie die Form, die Breite, die Länge und den Abstand (%) des Pfeils für die Feldlinien festlegen.

 

Zusätzliche Anfangspunkte

X/ Y

Wählen Sie eine Spalte oder ein Matrixobjekt oder geben Sie Werte als X/Y für die zusätzlichen Anfangspunkte ein.

Hinweise: Geben Sie mit Leerzeichen getrennte Zahlen oder Zahlen aus einem Datensatz ein.
