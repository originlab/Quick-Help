Datenanforderungen

Arbeitsblatt: Wählen Sie XYZZZ-/XYZXYZ-Spalten (voller oder Teilbereich) oder XYY als virtuelle Matrix aus.

Matrixblatt: Ein Matrixblatt mit mehreren Objekten

Diagramm erstellen

  1. Aktivieren Sie ein Arbeitsblatt mit virtuellen Matrixdaten oder den Spalten XYZXYZ.../XYZZ... oder ein Matrixblatt mit mehreren Objekten.
  2. Wählen Sie im Menü Zeichnen > Kontur: Heatmap mit aufgeteilten Kacheln.

Quelldaten: Virtuelle Matrizen

Wenn auf Grundlage eines Arbeitsblatts mit virtuellen Matrizen (keine Z-Spalten hier) gezeichnet wird, wird der Dialog plotmultivm geöffnet:

Drei Optionen für Datentyp:

  • Mehrere Blätter: Sie können mehrere Blätter als Eingabe auswählen. Alle Blätter sollten die gleiche Dimension haben. X-Werte und Y-Werte sollten in der gleichen Position der Blätter und mit den gleichen Werten gespeichert werden.
  • Mehrere Bereiche im gleichen Blatt: Sie können mehrere Datenblöcke mit gleicher Dimension im gleichen Arbeitsblatt auswählen, um sie als virtuelle Matrizen mehrfach einzugeben. Um diese Blöcke auszuwählen, müssen Sie durchgehend die Strg-Taste drücken, um sie getrennt zu markieren.
  • Vorhandene virtuelle Matrizen: Sie können mehrere virtuelle Matrizen auswählen. Wählen Sie Virtuelle Matrizen auswählen... im Ausklappmenü des Felds Eingabe und der Virtuelle Matrixmanager wird geöffnet.

Sobald Sie die Eingabedaten ausgewählt haben, sind die anderen Einstellungen in diesem Dialog den Bedienelementen im Dialog plotvm ähnlich.

Quelldaten: XYZZ... oder XYZXYZ...

Wenn Sie aus mehreren XYZ-Spalten oder Spalten des Formats XYZZ... zeichnen, wird der Dialog plot_heatmapxyz geöffnet:

Hinweis: Dieser Dialog macht keinen Unterschied zwischen aus einer XYZ-Datensatz erstellten Heatmap und Heatmap mit aufgeteilten Kacheln. Mehrere Z-Spalten werden als Teile in die gleiche Heatmap-Zelle gezeichnet.

Quelldaten: Matrixblatt mit mehreren Objekten

Wenn Sie die Quelldaten in mehreren Matrixobjekten in einem Matrixblatt gespeichert haben, wird durch Auswahl von Zeichnen > Kontur: Heatmap mit aufgeteilten Kacheln eine Heatmap mit aufgeteilten Kacheln direkt mit den Werten der gleichen Stelle aus verschiedenen Objekten gezeichnet, wodurch die gleiche Heatmapzelle geteilt wird.

Notizen

Dieses Diagramm ist eine erweiterte Variante der Heatmap, die das Zeichnen von gruppierten Heatmaps in der gleichen Zelle unterstützt.

  • Über eine spezielle Registerkarte Gruppe im Dialog Details Zeichnung können Sie festlegen, wie die Heatmap-Zelle entsprechend der Anzahl von Z-Werten aufgeteilt werden soll.
  • Die Registerkarte Anzeige und Registerkarte Beschriftung können verwendet werden, um die Heatmap mit aufgeteilten Kacheln benutzerdefiniert anzupassen genauso wie die Heatmap.
  • Schaltflächen der Minisymbolleiste werden auch hinzugefügt, um die Anzeige der Heatmap mit aufgeteilten Kacheln benutzerdefiniert anzupassen.
