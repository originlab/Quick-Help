LabTalk

  • LabTalk Programming
    Der "Labtalk Scriping Guide" richtet sich an Anwender, die allgemein vertraut sind mit dem Programmieren in einer Programmiersprache und daher vollen Nutzen aus den Programmiermöglichkeiten von Origin ziehen können. Die Anleitugn enthält ein ausreichendes Maß an Informationen für einen Anwender mit Basiswissen von Origin, um die Software auf seine individuellen Bedürfnisse anpssen zu können.
  •  
    Die LabTalk Language Reference bietet Informationen zu LabTalk-Befehlen, -Objekten, -Funktionen und Makros.
  • LabTalk Script Examples
    In diesem Abschnitt werden Beispiele von vollständigen Skripts aufgeführt, die verschiedene Programmieraufgaben erfüllen.

Origin C

  •  
    Der "Programming Guide" für Origin C beschreibt viele der wichtigsten Themen zur Programmierung in Origin, einschließlich der Programmierung mit Origin C, der Entwicklung einer Benutzeroberfläche sowohl in der Standard- als auch in der Profiversion von Origin sowie der COM-Client-Programmierung, und erläutert die Verteilung Ihrer benutzerdefineirten Applikationen nach Fertigstellung. Die Beschreibungen enthalten viele kleine, leicht verständliche Beispiele mit kurzen Erklärungen. Es werden Quelldateien zur Verfgung gestellt, die einen Großteil des Beispielcodes beinhalten. Sie befinden sich im Origin-Unterordner \Samples\Origin C Examples.
  •  
    Die Hilfe zur Origin C-Sprache bietet detaillierte Informationen zu jeder Mitglieder- und globalen Funktion der Origin C-Klassen. Die Referenz umfasst den Funktionsprototyp, Anmerkungen, eine Zusammenfassung der Parameter und Rückgabewerte, Beispielcode und den Namen der Headerdatei, die wiederum den Prototyp und Links zu verwandten Funktionen enthält.
  • Origin C Examples
    Hier finden Sie vollständige Beispiele von Origin C-Code für die Durchführung verschiedenster Programmieraufgabe.

X-Function Programming

Der X-Function Programming Guide zeigt Ihnen detailliert, wie Sie ein benutzerdefiniertes Hilfsmittel mit X-Funktionen erstellen. Er enthält außerdem einige praktische Beispiele inklusive Erläuterungen.

Automation Server

Der von Origin unterstützte Automatisierungsserver ermöglicht es Anwendern, Origin von anderen Applikationen aus aufzurufen. Die Anleitung zeigt in detaillierten Beispielen (Excel VBA, VB, VC, C#...), wie Sie mit verschiedenen Bereichen in Origin kommunizieren können. Weiterhin werden ausführliche Referenzinformationen zu den Methoden und Eigenschaften des Automatisierungsservers bereit gestellt. Die Referenzinformationen zu den Methoden umfassen den Funktionsprototyp, Anmerkungen, eine Zusammenfassung der Parameter und die Rückgabewerte, VB-Beispielcode sowie Links zu verwandten Eigenschaften und Funktionen. Die Referenzinformationen zu den Eigenschaften umfassen den Eigenschaftstyp und -namen, Anmerkungen, VB, VC-Beispielcode sowie Links zu verwandten Eigenschaften und Funktionen.

Custom VIs for LabVIEW

Die Anleitung der benutzerdefinierten VIs für LabVIEW enthält Einzelheiten zu verschiedenen benutzerdefinierten VI-Bausteinen, die mit Origin installiert sind. Mit diesen benutzerdefinierten VIs können LabVIEW-Anwender ihre eigenen VIs erstellen, um mit Origin zu kommunizieren. Sie können für Operationen verwendet werden wie das Öffnen und Beenden der Kommunikation mit Origin, das Austauschen von Daten zwischen Origin und LabVIEW und das Senden von Befehlen an Origin.

Code Builder User's Guide

Der Code Builder User's Guide bietet umfassende Erklärungen zur Verwendung des Code Builders. Der Code Builder ist eine integrierte Entwicklungsumgebung (IDE) in Origin, die dazu verwendet werden kann, mit Origins zwei Programmiersprachen LabTalk und Origin C Programme zu schreiben und zu debuggen.
