Spalten reduzieren

Reduce-Columns

Origin verfügt über ein Hilfsmittel zum Reduzieren von Daten durch Löschen oder Zusammenfügen von Arbeitsblattspalten. Wird auf alle oder Teilbereiche von Spalten des Arbeitsblatts angewendet und reduziert Arbeitsblattspalten durch Löschen oder statistisches Zusammenfügen.


Um diesen Dialog zu öffnen:

  1. Aktivieren Sie die Arbeitsmappe.
  2. Klicken Sie auf Worksheet: Spalten reduzieren und öffnen Sie den Dialog wreducecols, der die X-Funktion wreducecols verwendet.

Reduktionsmethoden
  • N Spalten löschen, dann M Spalten überspringen
N Spalten löschen, dann M Spalten überspringen
  • N Spalten mit zusammengefügten Werten reduzieren
Alle N Datenpunkte werden in einem Datenpunkt zusammengefügt entsprechend dem Wert Zusammenfügen mit.
Spalten hinzufügen/löschen

Die Option hängt von Ihrer Auswahl der Reduktionsmethode: N Spalten löschen... (Spalten löschen) oder N Spalten reduzieren... (Spalten zusammenfügen) ab. Anzahl der zusammenzufügenden oder zu löschenden Spalten
Zusammenfassen mit

Verfügbar, wenn Reduktionsmethode = N Spalten reduzieren.... Legen Sie den Wert fest, der zusammengefügte Datenpunkte ersetzen soll.

  • Erster Punkt
Alle N Datenpunkte werden mit dem Wert des ersten zusammengefügten Punkts ersetzt.
  • Letzter Punkt
Alle N Datenpunkte werden mit dem Wert des letzten zusammengefügten Punkts ersetzt.
  • Durchschnitt
Alle N Datenpunkte werden mit dem Wert des Durchschnitts der zusammengefügten Punkte ersetzt.
  • Min.
Alle N Datenpunkte werden mit dem Wert des Minimums der zusammengefügten Punkte ersetzt.
  • Max.
Alle N Datenpunkte werden mit dem Wert des Maximums der zusammengefügten Punkte ersetzt.
  • Summe
Alle N Datenpunkte werden mit dem Wert der Summe der zusammengefügten Punkte ersetzt.
  • StAbw
Alle N Datenpunkte werden mit der Standardabweichung der zusammengefügten Punkte ersetzt.
  • Median
Alle N Datenpunkte werden mit dem Median der zusammengefügten Punkte ersetzt.
  • Benutzerdefiniert
Verwenden Sie das hinzugefügt Bedienelement Benutzerdefiniert, um die Größen festzulegen (siehe nächstes Element).
Benutzerdefiniert

Wählen Sie die Größen im Ausklappmenü. Mehrere Größen sollten mit einem Leerzeichen getrennt werden. Jede Größe wird in einer separaten Spalte ausgegeben und mit der Größe zum Zusammenfügen mit beschriftet.
Spalten überspringen

Nur verfügbar, wenn Reduktionsmethode = N Spalten löschen....
Erste zu löschen/zusammenzufügende Spalte

Die erste Spalte, die zusammengefügt bzw. gelöscht werden soll
Ausgabearbeitsblatt

Legen Sie den Ausgabebereich fest. Für das Zusammenfügen werden die Ausgabespalten mit der Größe zum Zusammenfügen mit beschriftet.
