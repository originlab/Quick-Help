Datenanforderungen

Die Eingabedaten sollten aus Arbeitsblattspalten stammen. Es ist auch erforderlich, mindestens eine anderen Datenbereich als gruppierte Spalte(n) zu haben.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen: Kategorial: Gruppiertes Intervalldiagramm, Index.

In dem sich öffnenden Dialog plot_gboxindexed wählen Sie den Gruppenbereich und das Gruppendesign aus, um das gruppierte Intervalldiagramm zu erstellen.

Vorlage

gBox.otp (im Origin-Ordner installiert)

Hinweise

Dieser Diagrammtyp ist eine Variante des Gruppierten Boxdiagramms, Index, wobei das Diagrammdesign im Dialog plot_gboxindexed auf Box_Interval Plot gesetzt ist.
