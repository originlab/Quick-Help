Der Dialog Projektvariablen

ProjVar-Dialog

Dieser Dialog wird verwendet, um Projektvariablen anzuzeigen, so dass die gewünschten Variablen in die Bearbeitungsfelder Spaltenformel oder Skript vor Anwenden der Formel in dem Dialog Werte setzen eingefügt werden können. Änderungen in Projektvariablen können durch Halten der Tasten Strg + F5 aktualisiert werden.

Dialog öffnen
  • Wählen Sie im Dialog Werte setzen die Menüoption Variablen: Projektvariablen.

Der Dialog Projektvariablen ist eine kurze Version des Dialogs LabTalk-Variablen und -Funktionen und zeigt nur Variablen der Projektebene. Daher besteht nur Zugriff auf Funktionen, die einen Bezug zur Projektvariablen haben. Um alle Variablen anzuzeigen, verwenden Sie bitte den Dialog LabTalk-Variablen und -Funktionen im Skriptfenster.

Inhalt

  • 1 Dialog öffnen
  • 2 Schaltflächen
  • 3 Variablenliste
  • 4 Kontrollkästchen
  • 5 Wertefeld

 

Schaltflächen

Hinzufügen Fügen Sie eine Variable hinzu. Der Dialog Variable definieren wird geöffnet.
Löschen Die ausgewählte Variable wird gelöscht.
Name kopieren Kopieren Sie den Namen der ausgewählten Variable, um Tippfehler zu vermeiden, wenn Sie ihn in das Bearbeitungsfeld Spaltenformel oder Skript vor Anwenden der Formel im Dialog Werte setzen eingeben.
Werte zeigen/verbergen Das Wertefeld wird gezeigt/verborgen.

Variablenliste

Es werden alle Informationen der Projektvariablen aufgeführt, einschließlich Namen, Werte und Datentyp (Numerisch, Zeichenkette, Konstante, Ungebundener Datensatz etc.).

Kontrollkästchen

Aktivieren oder deaktivieren Sie die Kontrollkästchen, um die gewünschten Variablentypen zu zeigen oder zu verbergen.

Der Dialog Projektvariablen listet nur Variablen der Projektebene auf:

Numerisch Zeichenkette Ungebundener Datensatz

Wertefeld

Es werden die Werte der ausgewählten Variablen in der Variablenliste gezeigt. Nur Ungebundener Datensatz zeigt Werte in diesem Feld.
