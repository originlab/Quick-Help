Senden und Empfangen von Arbeitsblattdaten

Sie können Daten zwischen Origin-Arbeitsblättern und Mathematica austauschen, indem Sie die Registerkarte Spalte des Hilfsmittels verwenden. Diese Registerkarte besitzt Einstellungen, die Ihnen erlauben, die Mathematica-Variable und die Arbeitsblattspalte von Origin zum Senden und Empfangen von Daten sowie den Datenbereich, der ausgetauscht werden soll, festzulegen. Eine Arbeitsblattspalte von Origin oder ein bestimmter Bereich aus Zeilen in einer Arbeitsblattspalte entspricht dann einer benutzerdefinierten Variablen in Mathematica.

Das Textfeld Mathematica-Variable sollte den Namen der Variable enthalten, die die von Origin gesendeten Arbeitsblattdaten empfängt. Der Variablenname ist benutzerdefiniert und sollte nicht mit einer Zahl beginnen. Außerdem sollte er keine Operatoren oder Punkte enthalten.

Das Textfeld Origin Datensatz sollte den Namen des Origin-Datensatzes enthalten, der an Mathematica gesendet wird. Beachten Sie die neue Datenbereichsschreibweise, wie [Book1]Sheet1!Col(B), die Ihnen erlaubt, eine bestimmte Spalte in einem bestimmten Blatt in einer bestimmten Mappe zu adressieren. Diese kann dazu verwendet werden, die gewünschte Arbeitsblattspalte auszuwählen. Es wird die Verwendung dieser neuen Bereichsschreibweise an Stelle der alten Datensatznamenschreibweise, wie Book1_B, die in früheren Versionen unterstützt wurde, empfohlen. Beachten Sie weiterhin, dass Sie bei aktivem Origin-Worksheet auf Spalten im aktiven Arbeitsblatt verweisen können, indem Sie einfach nur Spalteninformationen, wie Col(1), Col(Druck) usw. verwenden.

Die Textfelder Erste Zeile und Letzte Zeile können dazu verwendet werden, um einen Bereich aus Indizes oder Zeilen für die Origin-Datensatzspalte festzulegen. Die Standardwerte sind 0 und, wenn diese unverändert bleiben, wird der gesamte Datensatz übertragen.

Die Schaltfläche Senden sendet die Daten von der Origin-Arbeitsblattspalte an die Mathematica-Variable.

Die Gruppe Empfange Spalte:

Das Textfeld Mathematica-Variable sollte den Namen der Variable enthalten, die an Origin gesendet wird.

Das Textfeld Origin Datensatz sollte den Namen der Origin-Arbeitsblattspalte enthalten, die die Daten von Mathematica empfängt. Die Spalte wird nicht erstellt und muss daher bereits in Origin bestehen.

Das Textfeld Erste Zeile kann zum Festlegen des ersten Index' oder Zeilenwertes für die Origin-Datenspalte verwendet werden, ab der die Daten eingefügt werden sollen. Der Standardwert ist 0, was dazu führt, dass die Daten ab der ersten Zeile eingefügt werden. Die Schaltfläche Empfangen sendet die Daten von der Mathematica-Variable und speichert sie in der Arbeitsblattspalte von Origin.
