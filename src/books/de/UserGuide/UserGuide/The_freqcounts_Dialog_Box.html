Klassifizierungsbereich festlegen mit

Klassenmittelpunkte

Der Klassenmittelpunkt wird im Allgemeinen berechnet, indem die halbe Klassengröße an den Klassenenden subtrahiert wird. Außerdem können Sie durch die benutzerdefinierte Anpassung von Minimum der Einteilung Mittelpunkt und Maximum der Einteilung Mittelpunkt im Abschnitt Steuerung Berechnung die Beschränkungen der Klassenmittelpunkt definieren. Weitere Klassenmittelpunkte werden in einer Klassengröße hinzugefügt, basierend auf der vorherigen.

Klassenendwerte

Definieren Sie den Einteilungsbereich mit Hilfe der Klassenendwerte von Minimum und Maximum.

Anwenderdefinierte Grenzen Klassengrenzen werden durch einen vom Benutzer festgelegten Datensatz definiert, von minimalem Start bis zum maximalen Endwert der Einteilungen. Die Werte müssen monotonisch steigen. Wählen Sie den Datensatz für die benutzerdefinierten Klassengrenzen unter Datenbereich der Klasseneinteilung.
Benutzerdefinierte Zentren Klassenmittelpunkte werden durch einen vom Benutzer festgelegten Datensatz definiert. Die Werte müssen monotonisch steigen. Wählen Sie den Datensatz für die benutzerdefinierten Klassenmittelpunkte unter Datenbereich der Klasseneinteilung.

Steuerung Berechnung

Minimum der Einteilung - Mittelpunkt/Minimum der Einteilung - Startwert Legen Sie das Minimum des Klassenmittelwerts oder den Klassenstartwert fest, abhängig davon, welche Option unter Bereich der Klasseneinteilungen festlegen mit ausgewählt wurde. Deaktivieren Sie Auto, um das Minimum manuell bearbeiten zu können.
Maximum der Einteilung - Mittelwert/Maximum der Einteilung - Endwert Legen Sie das Maximum des Klassenmittelwerts oder den Klassenendwert fest, abhängig davon, welche Option unter Bereich der Klasseneinteilungen festlegen mit ausgewählt wurde. Deaktivieren Sie Auto, um das Maximum manuell bearbeiten zu können.
Schritt nach Legen Sie fest, wie Eingabedatensätze in Klassen eingeteilt werden sollen.

Optionsliste:

  • Klassenbreite
    Die Eingabedaten werden nach einer festen Klassenbreite, die in Klassengröße festgelegt ist, eingeteilt.
  • Anzahl der KIassen
    Die Eingabedaten werden in n Klassen eingeteilt. n wird durch Eingabe in Anzahl der Klassen festgelegt.
Klassenbreite Legen Sie die Klassenbreite zum Sortieren der Eingabedaten fest. Die Anzahl der Klassen wird entsprechend berechnet.
Anzahl der KIassen Legen Sie die Anzahl der Klasseneinteilungen zum Sortieren der Eingabedaten fest. Die Klassenbreite wird entsprechend berechnet.
Ausreißer einbinden < Minimum Ausreißer, die kleiner als das Minimum in der niedrigsten Klasse sind, werden eingeschlossen.
Ausreißer einbinden >= Maximum Ausreißer, die größer als oder gleich das Maximum in der höchsten Klasse sind, werden eingeschlossen.
Minimum separat zählen Betrachten Sie das Minimum als eine separate Klasse.
Maximum separat zählen Betrachten Sie das Maximum als eine separate Klasse.
