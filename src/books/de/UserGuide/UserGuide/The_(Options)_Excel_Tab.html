Auswahl der Gruppe Zeichnen

Nach Standardeinstellungen zeichnen

Aktivieren Sie dieses Kontrollkästchen, um durch Hervorheben der Arbeitsmappendaten und Annahme der entsprechenden Spaltenzuordnung gemäß dem ausgewählten Diagrammtyp zu zeichnen. Falls keine Arbeitsmappendaten hervorgehoben sind, wenn der Diagrammtyp ausgewählt ist, öffnet Origin das Dialogfeld Daten zum Zeichnen auswählen.
Legende suchen

Wählen Sie dieses Kontrollkästchen aus, um eine Diagrammlegende durch automatisches Durchsuchen der zugehörigen Arbeitsmappe nach Text zu erstellen. Bei jedem Diagramm beginnt Origin die Suche nach Legendentext in der Zelle unmittelbar über der Zelle, die die ersten Y-Daten enthält. Origin führt die Suche solange die Spalten aufwärts fort, bis weitere Textwerte gefunden werden.
Zeichnen in

Wenn mehrere Datensätze zum Zeichnen ausgewählt sind, können Sie die Anzeige der Diagramme von dieser Auswahlliste aus steuern.

  • Um die ausgewählten Datensätze in einem einzigen Layer eines Diagrammfensters unterzubringen, wählen Sie Einzelnes Layer.
  • Wählen Sie Mehrere Layer, um jeden ausgewählten Datensatz in seinen eigenen Layer zu zeichnen. Alle Layer werden im selben Diagrammfenster angezeigt.
  • Wählen Sie Mehrere Seiten, um jeden ausgewählten Datensatz in seinen eigenen Layer zu zeichnen. Jeder Layer wird in seinem eigenen Diagrammfenster angezeigt.

Gruppe Abfrage vor

Excel-Dateien per Drag&Drop öffnen

Aktivieren Sie dieses Kontrollkästchen, um ein Dialogfeld anzuzeigen, das Anwendern die Möglichkeit gibt auszuwählen, ob eine Excel-Datei in Origin geöffnet wird, wenn die Datei per Drag&Drop in Origin gezogen wird. Wenn ?Ja? in diesem Dialogfeld gewählt ist, wird die Excel-Datei in eine Origin-Arbeitsmappe importiert. Wenn ?Nein? gewählt ist, wird die Excel-Datei als eine Excel-Arbeitsmappe in Origin geöffnet.

Wenn dieses Kontrollkästchen deaktiviert ist, öffnet Origin die Excel-Datei als eine Excel-Arbeitsmappe.
Excel-Arbeitsmappen speichern

Diese Auswahlliste steuert die Anzeige eines Dialogfelds, das festlegt, wie interne und externe Excel-Arbeitsmappen gespeichert werden, wenn Sie ein Projekt speichern, das die Arbeitsmappe enthält.

  • Wählen Sie Nie aus, damit Origin das Dialogfeld Excel-Arbeitsmappen speichern unter allen Umständen öffnet. Standardmäßig werden externe Excel-Arbeitsmappen automatisch aktualisiert, wenn das Projekt, das die Arbeitsmappe enthält, gespeichert wird.
  • Wählen Sie Vor dem Speichern aus, damit Origin das Dialogfeld Excel-Arbeitsmappen speichern öffnet, sobald Sie Datei: Projekt speichern auswählen (oder sobald Sie auf die Schaltfläche Speichern in der Standard-Symbolleiste klicken), vorausgesetzt, dass das Projekt eine Excel-Arbeitsmappe enthält . Das Dialogfeld Excel-Arbeitsmappen speichern ermöglicht es Ihnen festzulegen, welche (externen) Excel-Arbeitsmappen beim Speichern des Projekts aktualisiert werden sollen. Standardmäßig werden alle externen Excel-Arbeitsmappen aktualisiert.
  • Wählen Sie Vor Projekt speichern unter, um Origin anzuweisen, das Dialogfeld Excel-Arbeitsmappen speichern jedes Mal zu öffnen, wenn Sie Datei:Projekt speichern unter auswählen, vorausgesetzt, das Projekt enthält eine Excel-Arbeitsmappe. (Lesen Sie die Option Vor dem Speichern.)
