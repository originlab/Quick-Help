Spaltenkurznamen zurücksetzen

Reset-Column-Short-Names

Wenn Sie Spalten in einem Arbeitsblatt eingefügt bzw. verschoben haben, haben Sie vielleicht festgestellt, dass die Spaltenkurznamen durcheinander geraten können. Der Dialog Spaltenkurznamen zurücksetzen (wmvsn) setzt die Kurznamen für alle Spalten im Arbeitsblatt zurück. Sie können vorhandene Kurznamen bewahren, indem Sie sie in andere Arbeitsblattkopfzeilen verschieben. Der Dialog wmvsn verwendet die X-Funktion wmsvn.

Um diesen Dialog zu öffnen:

  1. Aktivieren Sie das entsprechende Arbeitsblatt.
  2. Klicken Sie auf Worksheet: Spaltenkurznamen zurücksetzen, um den Dialog wmvsn zu öffnen.

Dialogoptionen

Arbeitsblatt

Legen Sie das zurückzusetzende Arbeitsblatt fest.
Präfix des Kurznamens

Legen Sie das Präfix einer Zeichenkette für Spaltenkurznamen fest. Wenn der Dialog wmvsn geschlossen ist, werden die Spaltenkurznamen zurückgesetzt, indem Ihr Präfix angewendet und die Spalten von links nach rechts innerhalb des Arbeitsblatts nummeriert werden. Beachten Sie, dass das Präfix auf drei Zeichen begrenzt war.
Verschieben in Beschriftungszeile

Bewahren Sie die vorhandenen Zeichenketten der Kurznamen, indem Sie sie in eine Spaltenbeschriftungszeile verschieben.

Optionsliste:

  • Kein
Die aktuellen Kurznamen werden nicht in eine Spaltenbeschriftungszeile verschoben.
  • Langname
Die aktuellen Kurznamen werden in die Spaltenbeschriftungszeile der Langnamen verschoben.
  • Einheiten
Die aktuellen Kurznamen werden in die Spaltenbeschriftungszeile der Einheiten verschoben.
  • Kommentare
Die aktuellen Kurznamen werden in die Spaltenbeschriftungszeile der Kommentare verschoben.
  • Parameter 1
Die aktuellen Kurznamen werden in die Spaltenbeschriftungszeile des ersten Parameters verschoben.
  • Parameter 2:
Die aktuellen Kurznamen werden in die Spaltenbeschriftungszeile des zweiten Parameters verschoben.
  • Benutzerparameter 1
Die aktuellen Kurznamen werden in die Zeile des ersten benutzerdefinierten Parameters verschoben.
  • Benutzerparameter 2
Die aktuellen Kurznamen werden in die Zeile des zweiten benutzerdefinierten Parameters verschoben.
Rückgängig aktivieren

Die Option Bearbeiten: Rückgängig wird aktiviert.

Beispiel

Angenommen, Sie möchten Kurznamen der Arbeitsblattspalten in die Zeile der Langnamen verschieben und die Kurznamen in "abc1", "abc2" etc. umbenennen:

  1. Aktivieren Sie das Arbeitsblatt.
  2. Wählen Sie Worksheet: Spaltenkurznamen zurücksetzen im Hauptmenü, um den Dialog Data Manipulation\Worksheet: wmvsn zu öffnen.
  3. Das aktive Arbeitsblatt wird automatisch als Eingabearbeitsblatt ausgewählt.
  4. Geben Sie abc in Präfix des Kurznamens ein.
  5. Wählen Sie Langname in der Auswahlliste Verschieben in Beschriftungszeile.
  6. Klicken Sie auf OK.
