Datenanforderungen

Wählen Sie mindestens zwei Y-Spalten oder einen Wertebereich aus mindestens zwei Spalten aus. Die Subjektspalte ist optional. Die Diagrammzuweisung macht keinen Unterschied; ausgewählte Spalten werden wie Y-Spalten behandelt.

Diagramm erstellen

Wählen Sie die erforderlichen Daten aus

Wählen Sie Zeichnen > Statistisch: Bland-Altman.

Origin öffnet den Dialog plot_BA.

Vorlage

BA_NOSUBJECT.OTPU (installiert im Origin-Programmordner).

Hinweise

Bland-Altman-Diagramme zeigen die Korrelation zwischen zwei unterschiedlichen Methoden an (zum Beispiel die traditionelle Methode und eine neue), wenn sie verwendet werden, um die gleiche Eigenschaft der gleichen Reihe von Proben zu messen. Den ausführlichen Algorithmus finden Sie auf dieser Seite.
