Datenanforderungen

Eine Stiff Map ist eine Variante des Stiff-Diagramms mit einer weiteren XY-Eingabe zum Festlegen der Position der Stiff-Polygone.

Diagramm erstellen

Markieren Sie die Quelldaten und öffnen Sie den Dialog plotstiff durch eine der folgenden Methoden:

  • Wählen Sie Zeichnen > Spezialisiert: Stiff Map im Hauptmenü.

Vorlage

StiffMap.otpu (installiert im Origin-Programmordner).

Bedienelemente

Die Stiff Map ist ein spezielles Stiff-Diagramm, das die Stiff-Polygone als Streupunkte entsprechend der Eingabepositionsdaten zeichnet.

Für diese Art von Diagramm können Sie die Stiffs immer noch benutzerdefiniert anpassen wie beim Stiff-Diagramm, z. B. das Füllmuster der Stiffs, die Beschriftungen der Ionen und Sample-IDs und den Abstand zwischen den Stiffs.

Außerdem passen Sie die Map des Stiffs benutzerdefiniert an. Siehe

  • Registerkarte Map
Hinweis: Sie können doppelt auf einen der Stiff-Punkte klicken, wobei Sie die Strg-Taste gedrückt halten, um diesen Stiff-Punkt als einen speziellen Punkt hinzuzufügen. Sie können dann auf der Registerkarte Map des Dialogs Details Zeichnung die Optionen Horizontaler Versatz und Vertikaler Versatz festlegen, um diesen Punkt neu zu positionieren, und festlegen, ob die Verbindungslinien zum Markieren der ursprünglichen Position dieses Punktes gezeigt werden.

Bequemerweise können Sie doppelt auf einen Stiff-Punkt im Diagramm klicken, um ihn direkt auszuwählen und dann per Drag&Drop schnell neu zu positionieren. Standardmäßig gibt es eine Verbindungslinie, die ihn mit der ursprünglichen Position verbindet.
