The 2D FFT Dialog Box

FFT2-Dialog

  • Menu Command: Analysis: Signal Processing: FFT: 2D FFT
  • Window Type: Matrix, 3D/Contour Plots
  • X-Function Script and GUI Examples: fft2(Pro)
Dialog Theme See Dialog Themes.
Recalculate

See Recalculating Analysis Results.
Input Matrix

Specifies the complex matrix or the real matrix of the input signal. See Specifying Your Input Data.
Imaginary Matrix

Specifies the imaginary matrix of the input signal, if the input signal is complex, and is separated into real matrix and imaginary matrix. See Specifying Your Input Data.
Normalize Amplitude Matrix

Specifies whether or not to normalize the amplitudes to 1.
DC Shift Center

If this is selected, the DC component will be put in the center of the result matrix. Otherwise, the DC component will locate at four corners of the result matrix.
Output

Specifies the matrices to output. See Output Results.

Complex Matrix
Amplitude Matrix
Phase Matrix
Power Matrix
Log Amplitude Matrix: Output the amplitude matrix for Log10 values.
Log Power Matrix: Output the power matrix for Log10 values.
