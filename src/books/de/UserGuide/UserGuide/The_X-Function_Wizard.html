Der Assistent der X-Funktionen

Das Hilfsmittel Neuer X-Funktionen-Assistent führt Sie durch die Schritte zum Erstellen einer X-Funktion. Sie können es öffnen durch:

Auswahl von Datei: Neuer X-Funktionen-Assistent im Menü des X-Funktion-Builders.

oder

Klicken Sie auf die Schaltfläche Neuer X-Funktionen-Assistent auf der Symbolleiste des X-Funktion-Builders.

Es gibt sechs Seiten zum X-Funktionen-Assistenten:

Anzahl der Eingabevariablen

Festlegen der Anzahl von Eingabevariablen
Eingabevariablen

Festlegen der Namen und Typen für alle Eingabevariablen
Anzahl der Ausgabevariablen

Festlegen der Anzahl der Ausgabevariablen
Ausgabevariablen

Festlegen der Namen und Typen für alle Ausgabevariablen
Funktionskörper

Geben Sie den Funktionscode auf dieser Seite ein oder lassen Sie sie leer und bearbeiten Sie die X-Funktion später im Code Builder.
Standarddaten für Variablen

Festlegen der Standardwerte für die Eingabe- und Ausgabevariablen

Weitere Informationen finden Sie im Kapitel X-Funktionen in der Programmierhilfe.
