Exportieren der Symbolleisteneinstellungen


Die Einstellungen einer Symbolleiste wie die Anzahl an Schaltflächen und ihre Anordnung kann als .opx-Datei exportiert werden. Dann können die Einstellungen durch Ziehen der .opx-Datei in den Arbeitsbereich von Origin importiert werden. Dies ermöglicht einen einfachen Tausch der Symbolleisteneinstellungen mit anderen Origin-Anwendern.

Beachten Sie, dass die Informationen zur Schaltflächengruppe in dieser Art von .opx-Datei nicht enthalten sind. Bevor Sie eine .opx-Datei mit Symbolleisteneinstellungen installieren, vergewissern Sie sich, dass die zugehörigen Informationen zur Schaltflächengruppe bereits installiert wurden.

Um die Einstellungen einer Symbolleiste als .opx-Datei zu exportieren:

  1. Wählen Sie im Menü Ansicht: Symbolleisten, um das Dialogfeld Symbolleiste anpassen auszuwählen.
  2. Wählen Sie die Registerkarte Symbolleisten.
  3. Wählen Sie eine Symbolleiste aus dem Listenfeld Symbolleisten aus.
  4. Klicken Sie auf die Schaltfläche Exportieren. Hierdurch öffnet sich das Dialogfeld Speichern unter.
  5. Geben Sie den Dateinamen und den Pfad ein, wenn nötig*.
  6. Klicken Sie auf Speichern, um eine .OPX-Datei zu erstellen.

*Hinweis: Die Endung des Dateinamens muss .OPX lauten. Sie müssen diese Endung nicht eingeben, da Origin sie automatisch einfügen wird. Geben Sie eine andere Endung als .OPX ein, wird .OPX an den Dateinamen angehängt.
