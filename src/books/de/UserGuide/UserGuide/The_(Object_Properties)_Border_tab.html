Die Registerkarte Grenze (Objekteigenschaften)

Ob-Prop-Border-tab

 

Auf der Registerkarte Grenze können Sie Farbe, Breite, Typ und Transparenz für das Formobjekt festlegen.

Für das rechteckige Objekt können Sie auch die Ausrichtung auf Vertikal, Horizontal, Keine Ecken und Nur Ecken festlegen. Wenn Sie das Objekt vertikal oder horizontal ausrichten, kann sich dies auch über den Gesamten Layer, Alle Layer oder die Gesamte Seite strecken.

Hinweis: Das Bedienelement Transparenz ist das gleiche wie auf der Registerkarte Füllung. Sie enthalten beide den gleichen Wert.

Wenn Sie den Prozentsatz der Transparenz für die Grenze ändern, wird die Füllung auf den gleichen Prozentsatz für die Transparenz angewendet und umgekehrt.
