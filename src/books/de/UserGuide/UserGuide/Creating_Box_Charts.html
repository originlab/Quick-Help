Das Boxdiagramm, das auch als Boxplot, Box-Whisker-Plot oder Kastengrafik bezeichnet wird, ist eine grafische Darstellung der Schlüsselwerte aus einer zusammenfassenden Statistik.

Ein Boxdiagramm erstellen:

  1. Markieren Sie eine oder mehrere Y-Spalten (oder einen Bereich aus einer oder mehreren Y-Spalten).
  2. Wählen Sie Zeichnen: Statistikdiagramme: Boxdiagramm.

Jede Y-Datenspalte wird in einer separaten Box dargestellt. Die Spaltennamen und Beschriftungen liefern die Hilfsstrichsbeschriftungen der X-Achse. Standardmäßig wird die Box durch das 25. und das 75. Perzentil bestimmt. Die Whisker werden durch das 5. und das 95. Perzentil bestimmt. In einem Boxdiagramm von Origin können zusätzliche Werte dargestellt werden, einschließlich Minimum, Median, Mittelwert, Maximum, 1. und 5. Perzentil und 95. und 99. Perzentil.

Sie können auch ein gruppiertes Boxdiagramm erstellen, entweder aus Indexdaten oder Rohdaten.

Zum Anpassen der Darstellung des Boxdiagramms:

  1. Klicken Sie doppelt auf das Boxdiagramm.

oder

  1. Klicken Sie mit der rechten Maustaste und wählen Sie Details Zeichnung.

Beide Aktionen öffnen den Dialog Details Zeichnung. Dabei ist auf der linken Seite des Dialogs das Diagrammsymbol für das Boxdiagramm aktiv. Die Bedienelemente des Boxdiagramms sind auf der rechten Seite des Dialogs über die Registerkarten verfügbar.

Informationen zu weiteren Anpassungen von Boxdiagrammen finden Sie unter den folgenden Themen:

  • Registerkarte Boxdiagramm (Details Zeichnung)
  • Registerkarte Daten (Details Zeichnung)
  • Die Registerkarte Prozentangaben (Details Zeichnung)
  • Die Registerkarte Muster (Details Zeichnung)
  • Die Registerkarte Abstände (Details Zeichnung)
  • Die Registerkarte Linien (Details Zeichnung)
