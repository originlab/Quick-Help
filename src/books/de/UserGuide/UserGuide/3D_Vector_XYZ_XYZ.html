Datenanforderungen

Sie müssen 6 Spalten aus einem Arbeitsblatt auswählen und setzen die Zuordnung auf XYZ XYZ.

Sie können 3D-Vektoren aktivieren, nachdem Sie ein 3D-Punktdiagramm gezeichnet haben. Bitte lesen Sie weitere Informationen auf dieser Hilfsseite.

Diagramm erstellen

Aktivieren Sie ein Arbeitsblatt und wählen Sie die gewünschten Daten.

Wählen Sie im Menü Zeichnen: 3D: 3D-Vektor XYZ XYZ.

oder

Klicken Sie auf die Schaltfläche 3D-Vektor XYZ XYZ auf der Symbolleiste 3D- und Konturdiagramme.

Vorlage

  • gl3DVector.OTP (OpenGL)
  • 3DVECTOR.OTP

(installiert im Origin-Programmordner)

Notizen

Das 3D-Vektordiagramm wird auch als 3D-Köcherdiagramm bezeichnet.

Der erste XYZ-Datenpunkt stellt den Anfangspunkt des Vektors und der zweite XYZ-Datenpunkt den Endpunkt des Vektors dar. Sie können Farbe, Länge etc. auf der Registerkarte 3D-Vektor im Dialog Details Zeichnung benutzerdefiniert anpassen.
