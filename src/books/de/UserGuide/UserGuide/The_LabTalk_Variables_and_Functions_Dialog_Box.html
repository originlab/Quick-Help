Der Dialog LabTalk-Variablen und -Funktionen

LTVar-Fun-Dialog

Dieser Dialog wird verwendet, um LabTalk-Variablen und -funktionen in Origin anzusehen.

Dialog öffnen
  1. Geben Sie den Befehl ed in das Befehls- oder Skriptfenster ein, um
  2. das Skriptfenster aufzurufen oder zu aktivieren, und wählen Sie Hilfsmittel: Variablen.

Inhalt

  • 1 Dialog öffnen
  • 2 Schaltflächen
  • 3 Variablenliste
  • 4 Kontrollkästchen
  • 5 Wertefeld

Schaltflächen

Aktualisieren Klicken Sie, um die neueste Variablenliste nach Ändern/Hinzufügen von Varablen im Skriptfenster zu sehen.
Hinzufügen Fügen Sie eine Variable hinzu. Der Dialog Variable definieren wird geöffnet.
Löschen Die ausgewählte Variable wird gelöscht.
Name kopieren Kopieren Sie den Namen der ausgewählten Variablen, um Tippfehler zu vermeiden.
Werte zeigen/verbergen Das Wertefeld wird gezeigt/verborgen.

Variablenliste

Informationen aller Variablen und benutzerdefinierten Labtalk-Funktionen werden aufgelistet, einschließlich Namen, Werte, Datentyp (Numerisch, Zeichenkette und Ungebundener Datensatz etc.), Variablentyp (System, Projekt und Sitzung), Zeichnungsinformationen (wenn der ungebundene Datensatz verwendet wird, um eine Zeichnung zu erstellen) und erforderliche Beschreibungen.

Kontrollkästchen

Aktivieren oder deaktivieren Sie die Kontrollkästchen, um die gewünschten Variablentypen zu zeigen oder zu verbergen.

Variablentyp System Projekt Sitzung
Datentyp Numerisch Zeichenkette Ungebundener Datensatz Konstante Zeichenkettenregister Funktion Baum Bereich Makro Zeichenkettenarray

Wertefeld

Werte oder Skripte (Makro der ausgewählten Variablen in der Variablenliste zeigen. Nur Werte/Skripte der Datentypen Funktion, Ungebundener Datensatz, Baum, Bereich und Makro werden in diesem Feld gezeigt.
