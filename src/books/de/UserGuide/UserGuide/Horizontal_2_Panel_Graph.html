Datenanforderungen

Wählen Sie mindestens eine Y-Datenspalte (oder einen Bereich aus mindestens einer Spalte) aus. Idealerweise wählen Sie mindestens zwei Y-Spalten aus (oder einen Bereich von mindestens zwei Y-Spalten). Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Bitte beachten Sie, dass die Anzahl der X-Spalten nicht mehr als zwei für dieses Diagramm mit 2 Feldern sein sollte:

  • Wenn es mehr eine X-Spalte gibt, wird diese X-Spalte für beide Feldern verwendet und alle Y-Spalten werden gleichmäßig in zwei Gruppen geteilt und in diese zwei Layer gegen die gleichen X-Daten gezeichnet;
  • Wenn es zwei X-Spalten gibt, wird die zweite X-Spalte als Trennzeichen verwendet, um die Y-Spalten in zwei Gruppen zu teilen, und diese zwei Gruppen Y-Spalten werden in zwei Layer gegen die angrenzenden X-Daten gezeichnet;
  • Wenn es mehr zwei X-Spalten gibt, können Sie diese Art von Diagrammen mit zwei Feldern nicht zeichnen.

Diagramm erstellen

Wählen Sie die erforderlichen Daten aus

Wählen Sie im Menü Zeichnen: Mehrere Felder/Achsen: 2-fach Horizontal.

oder

Klicken Sie auf die Schaltfläche 2-fach Horizontal in der Symbolleiste 2D Grafiken.

Vorlage

PAN2HORZ.OTP (im Origin-Programmordner installiert)

Hinweise

Jede Spalte mit Y-Werten wird in ein separates Layer gezeichnet. Die zwei Layer werden in zwei Spalten mit einer Zeile angeordnet.
