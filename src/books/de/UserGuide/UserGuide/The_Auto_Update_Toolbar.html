Die Symbolleiste Automatisch aktualisieren

Auto-Update-Toolbar

Über die Schaltfläche auf der Symbolleiste Automatisch aktualisieren können Sie die Operationen Automatische Aktualisierung/Neuberechnung pausieren. Dies ist eine allgemeine Einstellung, daher wird jede Analyse, die auf automatisch aktualisieren gesetzt ist, zuerst die Eigenschaft dieser Schaltfläche prüfen. Wenn diese Operationen pausieren, entspricht die Symbolleistenschaltfläche und die Schaltfläche Neuberechnen auf der Standardsymbolleiste wird ausgegraut . Auch wenn der Neuberechnungsmodus für eine besondere Analyse auf Auto gesetzt ist, wird also kein automatisches Update durchgeführt, bis erneut auf die Schaltfläche Automatische Aktualisierung/Neuberechnung pausieren geklickt wurde.

Da eine konstante Neuberechnung und automatische Aktualisierung der Ergebnisse Ihre Arbeit verlangsamen kann, möchten Sie vielleicht Automatische Aktualisierung/Neuberechnung pausieren, wenn Ihr Projekt viele solche Operationen enthält.
