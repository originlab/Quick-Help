Fenster Anfragen erzeugen

Das Erzeugen von Anfragen ist eine komplexe Aufgabe, die häufig nur für den Unternehmensspezialisten verfügbar ist. Origin kann bereits vorher erstellte Anfragen verwenden (wie in einer Access-Datenbank) oder sie mit dem Dialogfeld Anfrage erzeugen erstellen. Wenn eine vorgepackte Anfrage verfügbar ist, können Sie das Kontextmenü Objekt hinzufügen im Hauptfenster verwenden und die Anfrage auf der Registerkarte Sichten auswählen. Beachten Sie, dass 'Provider' und 'Dialekt' (siehe Einstellung: Optionen) für einen ordnungsgemäßen Vorgang übereinstimmen sollten.

Sie können eine neue Anfrage auch erstellen, indem Sie die Quellobjekte und die abgeleiteten Tabellen wählen, Verknüpfungen zwischen ihnen erstellen und die Daten für die Ausgabe auswählen. Dieses Fenster ermöglicht Ihnen das Bearbeiten der Eigenschaften von Verknüpfungen und Objekten. Sie können hier z.B. ein Alias für ein Objekt definieren.

Sie können in diesem Fenster ein Objekt oder eine abgeleitete Tabelle hinzufügen, indem Sie mit der rechten Maustaste auf eine leeren Bereich des Fensters klicken und dann Objekt hinzufügen oder Abhängige Tabelle hinzufügen auswählen. Wenn eine abhängige Tabelle hinzugefügt ist, wird eine entsprechende Registerkarte erstellt. Sie können Unteranfragen, die zu der abgeleiteten Tabelle gehören, visuell in der gleichen Weise bearbeiten, wie Sie die Hauptanfrage bearbeiten. Das Drücken der Schaltfläche Maximieren auf dem Objekt, das diese abgeleitete Tabelle darstellt, bringt Sie auch zu der entsprechenden Registerkarte.

Sie können auch Tabellen und Ansichten (Anfragen) aus dem rechten Fenster in das Hauptfenster ziehen.

Aktivieren Sie das Kontrollkästchen auf der linken Seite eines Feldnamens, wenn Sie das Feld ausgeben möchten. Wenn alle Felder des Objekts ausgegeben werden sollen, sollten Sie das Kontrollkästchen links vom Sternchenelement des Objekts aktivieren. Um ein Feld aus der Ausgabe zu entfernen, deaktivieren Sie einfach das Kästchen vor dem Feldnamen.

Um die Eigenschaft eines Quellobjekts zu bearbeiten, können Sie mit der rechten Maustaste darauf klicken und dann Editieren im Kontextmenü auswählen. Die bearbeitbaren Eigenschaften eines Quellobjekts variieren abhängig von dem Server. Aber Sie können immer das Alias definieren oder ändern.

Um zwei Objekte miteinander zu verbinden, wählen Sie ein Feld in dem einem Objekt aus und ziehen Sie es in das entsprechende Feld des anderen Objekts. Es wird eine Linie zwischen den beiden Feldern gezogen. Dies erstellt eine "innere Verbindung". Um andere Typen von Verbindungen zu erzeugen, klicken Sie mit der rechten Maustaste auf die Verbindungslinie und wählen Sie Editieren im Kontextmenü aus. Um eine Verbindung zu löschen, wählen Sie Entfernen im Kontextmenü aus oder klicken Sie auf die Verbindungslinie und drücken Sie auf ENTF.

Über dem Bereich Anfragen erzeugen ist eine Seitenbedienelement. Sie können es verwenden, um einfach zwischen Haupt- und Unteranfragen zu wechseln.

Bedienelement der Mengenoperation Unteranfragen

Das Symbol rechts oben im Fenster Anfragen erzeugen ist das Bearbeitungsbedienelement der Mengenoperation Unteranfragen. Mit diesem Bedienelement können Sie neue Mengenoperation Unteranfragen hinzufügen und Operationen durchführen.

Um eine neue Mengenoperation Unteranfragen hinzuzufügen, klicken Sie mit der rechten Maustaste auf das Bedienelement und wählen Sie Neue Mengenoperation Unteranfragen im Kontextmenü aus. Dann sehen Sie ein weiteres Symbol und eine Vereinendes Element zwischen ihnen, wie unten gezeigt wird:


Wenn es mehr als eine Unteranfrage gibt, können Sie mit diesem Kontextmenü ebenfalls:

  • eine Unteranfrage entfernen,
  • eine Unteranfrage in Klammern setzen,
  • eine Unteranfrage oder Klammer vor- oder zurückbewegen.

Um die vereinende Operation zu konfigurieren, können Sie mit der rechten Maustaste auf das Symbol Vereinen (Union) klicken . Dann sehen Sie ein anderes Kontextmenü, mit dem Sie die den vereinenden Operator aus einem der folgenden auswählen können:

  • Union (Gesamtmenge)
  • Union All (Gesamtmenge mit Duplikaten)
  • Except (Menge ohne Überschneidungen)
  • Intersect (Überschneidungsmenge)

Spaltenfenster

In dem Spaltenfenster können Sie die Felder auswählen, die ausgegeben werden sollen, die Pseudonyme für die Felder setzen und die Sortierung und die Gruppierung einrichten.

Vor jedem Feldnamen befindet sich ein Kontrollkästchen. Wenn dieses aktiviert ist, wird das dazugehörige Feld in das Ergebnis der Anfrage eingeschlossen. Daher müssen Sie nur das Kontrollkästchen vor dem Feldnamen deaktivieren, wenn Sie ein Feld aus der Ausgabe der Anfrage entfernen möchten. Dies ist besonders nützlich, wenn Sie ein Feld in dem Spaltenausschnitt lediglich zum Einstellen von Sortierung und Gruppierung behalten, es aber nicht ausgeben möchten.

Die Spalte Ausdruck zeigt den Namen der Felder, die der aktuellen Anfrage hinzugefügt worden sind. Sie können immer die Auswahlliste verwenden, um ein Feld auszuwählen und es diesem Fenster hinzuzufügen. Zusätzlich können Sie eine Unteranfrage erstellen, indem Sie "(Select)" in diese Spalte eingeben.

Um Alias für die Ausgabefelder zu erstellen, können Sie die Alias in die Spalte Alias eingeben.

Die Spalte Aggregat ermöglicht Ihnen, aggregierte Funktionen (Avg, Count, Sum, ...) für das Feld auszuwählen.

Die Spalten Sortierungstyp und Sortierungsreihenfolge werden verwendet, um die Sortierung der Anfrageergebnisse einzurichten. Sortierungstyp könnte entweder Aufsteigend oder Absteigend sein. Die Spalte Sortierungsreihenfolge wird verwendet, wenn es mehr als ein Feld gibt, das sortiert werden soll. Sie legt die Reihenfolge fest, in der die Felder sortiert werden.

Die Spalte Gruppe unterstützt Sie beim Einrichten der Gruppierung. Wenn das Kontrollkästchen in Spalte aktiviert ist, wird das aktuelle Feld zu dem Ausdruck "Gruppe von" hinzugefügt. Außerdem wird die Spalte Kriterien für gezeigt. Wenn Sie Kriterien für die gruppierten Daten einrichten möchten, können Sie in der Spalte Kriterien für die Option "Für Gruppen" auswählen und dann die Kriterien in die Spalte Kriterien eingeben.

In der Spalte Kriterien können Sie die Kriterien definieren, die mit dem Ausdruck Where verwendet werden (wenn entweder das Kontrollkästchen Gruppe deaktiviert oder Kriterien für auf "Für Werte" eingestellt ist, während das Kontrollkästchen Gruppe aktiviert ist) oder dem Ausdruck Having, der zusammen mit Gruppe von verwendet wird (wenn das Kontrollkästchen Gruppe aktiviert ist und Kriterien für auf "Für Gruppen" eingestellt ist). Beachten Sie, dass der Ausdruck (der Name des Feldes) ausgelassen werden sollte. Angenommen, Sie möchten das folgende Kriterium in der Anfrage verwenden:

where (field>10)

Dann geben Sie Folgendes in die Spalte Kriterien ein:

>10

Diese Spalte ermöglicht es Ihnen ebenfalls, eine Unteranfrage zu erstellen. Klicken Sie mit der rechten Maustaste auf die Spalte und wählen Sie Unteranfrage einfügen. Eine Unteranfrage wird hinzugefügt. Sie können zu ihr wechseln, indem Sie auf die Registerkarte der Unteranfrage klicken, die dem Fenster Anfrage erzeugen hinzugefügt wurde.

Wenn Sie mehrere Kriterien festlegen möchten, verwenden Sie bitte die Spalten Oder.... Die Kriterien werden über den ODER-Operator in der Anfragenanweisung verbunden. Dieses erlaubt Ihnen, eine Mehrfachbedingung aufzustellen.

Fenster der Verzeichnisbaumansicht Anfrage

In dem Fenster der Verzeichnisbaumansicht Anfrage können Sie die gesamte Anfrage ansehen, die wie eine Übersicht organisiert wird. Mit dieser können Sie schnell zu jedem Teil der Anfrage wechseln.

Fenster Tabellen/Ansichten/Prozeduren (Funktionen)

Dieses Fenster ermöglicht es Ihnen, der gegenwärtigen Anfrage alle möglichen Quellobjekte wie Tabellen, Ansichten oder Prozeduren hinzuzufügen. Um ein Quellobjekt hinzuzufügen, können Sie entweder doppelt darauf klicken oder in das Fenster Anfragen erzeugen ziehen.

Vorschau Schaltfläche/Fenster

Das Vorschaufenster ermöglicht es Ihnen, sich die Ergebnisse vorher anzuschauen und Ihre aktuelle Anfrage einzuschätzen. Sie können dieses Fenster mit den Schaltflächen Vorschau zeigen oder Keine Vorschau ein- und ausblenden.

Wenn das Vorschaufenster eingeschaltet ist, aktualisiert ein Klick auf die Schaltfläche Vorschau den Vorschauausschnitt.

SQL zeigen

Sie können auf diese Schaltfläche klicken, um die SQL Anfragenzeichenkette anzusehen. Ein Dialog wird geöffnet, das die Zeichenkette anzeigt. Sie können die Zeichenkette auch ändern oder die Änderungen mit dem Dialogfeld übernehmen.

Import

Indem Sie auf diese Schaltfläche klicken, können Sie die Anfrage übernehmen und das Ergebnis in das aktive Arbeitsblatt importieren.
