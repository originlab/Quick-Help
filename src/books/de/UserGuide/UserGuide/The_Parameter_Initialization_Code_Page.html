Die Seite Parameterinitialisierungscode

FitFuncBuilder-ParaINICode

Diese Seite bietet Optionen, mit denen Sie Code eingeben und ausführen können, um Initialisierungswerte für Anpassungsfunktionsparameter zu berechnen. Diese Seite ist optional, d.h., Sie können direkt auf Weiter klicken, um zu der nächsten Seite zu gelangen, auf der weitere Einstellungen vorgenommen werden können.

Bedienelemente des Dialogs

Initialisierungscode immer vor Anpassung ausführen Deaktivieren Sie dieses Kontrollkästchen, wenn Sie nicht möchten, dass der Initialisierungscode standardmäßig ausgeführt wird. Der Code kann ggf. im Dialog der nichtlinearen Anpassung später ausgeführt werden.
Initialisierung der festen Parameter zulassen

Aktivieren Sie dieses Kontrollkästchen, um den Parameterinitialisierungscode, der im Bearbeitungsfeld Initialisierungscode angegeben ist, zu aktivieren, auch wenn die Parameter auf der Seite "Variablen und Parameter" festgelegt sind.
Vordefinierte Variablen verwenden
Benutzerdefinierten Code verwenden		 Wählen Sie die Option Vordefinierte Variablen verwenden, um die Parameterinitialisierungswerte mit Hilfe von vordefinierten Variablen (wie Spaltenbeschriftungszeilen oder vordefinierte Statistikwerte) in der unten angezeigten Tabelle Vordefinierte Variablen festzulegen.

Wenn Sie die Option Benutzerdefinierten Code verwenden auswählen, schreiben Sie Ihren Initialisierungscode in das Bearbeitungsfeld Initialisierungscode.
Initialisierungscode Diese Option ist verfügbar, wenn Benutzerdefinierten Code verwenden ausgewählt ist. Wenn Sie in dieses Bearbeitungsfeld Initialisierungscode eingeben und das Kontrollkästchen Initialisierungscode immer vor Anpassung ausführen aktiviert ist, ersetzen die Werte, die Ihr Code erzeugt, die statischen Initialisierungswerte, die Sie auf der Seite "Variablen und Parameter" eingegeben haben. Weitere Einzelheiten zum Schreiben von Initialisierungscode finden Sie unter Parameterinitialisierung.
Tabelle Vordefinierte Variablen Sie ist verfügbar, wenn Vordefinierte Variablen verwenden ausgewählt ist. Initialisieren Sie Parameter durch eine Initialisierungsformel oder durch Metadaten aus Spalten in dieser Tabelle. Klicken Sie auf die dreieckige Schaltfläche in der Spalte Initialisierungsformel, um die gewünschte Initialisierungsformel oder die gewünschten Metadaten im Ausklappmenü auszuwählen. Auf diese Weise können einfache mathematische Operationen auf die Formel oder Metadaten durchgeführt werden.

Hinweis: Initialisierungsformel und vordefinierte Metadaten sind Origin C-Code, so dass sie Groß- und Kleinschreibung berücksichtigen.
LabTalk verwenden
Origin C verwenden		 Diese Option ist verfügbar, wenn Benutzerdefinierten Code verwenden ausgewählt ist. Wählen Sie die Option Origin C verwenden, um den Code mit Hilfe von Origin C zu schreiben. Es gibt viele verfügbare globale Funktionen zur Datenuntersuchung, um geeignete Initialisierungswerte abhängig von den anzupassenden Daten zu erzeugen.

Wenn die Option LabTalk verwenden aktiviert ist, wird angenommen, dass es sich bei dem Code um LabTalk-Skript handelt.
