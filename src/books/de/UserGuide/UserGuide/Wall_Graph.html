Datenanforderungen

Wählen Sie mindestens eine Y-Spalte aus (oder einen Bereich aus mindestens einer Y-Spalte). Idealerweise wählen Sie mindestens zwei Y-Spalten aus (oder einen Bereich aus mindestens zwei Y-Spalten). Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie im Menü Zeichnen: 3D: 3D-Wände.

oder

Klicken Sie auf die Schaltfläche 3D Wände der Symbolleiste 3D- und Konturdiagramme.

Vorlage

  • glWalls.OTP (OpenGL)
  • WALLS.OTP

(installiert im Origin-Programmordner)

Hinweise

  • Der Y-Wert jedes Datenpunktes wird als die Höhe einer Wand angezeigt. Jede Wand besitzt eine feste Breite und ist mit seinem Spaltennamen an der Z-Achse beschriftet.
  • Seit Origin 2018 können Sie Beleuchtungseffekte zum 3D-Wanddiagramm hinzufügen, indem Sie das Kontrollkästchen Beleuchtungseffekte hinzufügen auf der Registerkarte Muster im Dialog Details Zeichnung (Format: Zeichnung) aktivieren. Um den Dialog der zugehörigen Beleuchtung zu öffnen, klicken Sie auf die Schaltfläche Beleuchtungssteuerung auf der Symbolleiste Stil.
