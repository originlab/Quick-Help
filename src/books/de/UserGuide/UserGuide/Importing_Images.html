Einen unterstützten Bildtyp können Sie in ein Bild- oder Matrixfenster importieren.

Ein Bild ins Bildfenster importieren
  1. Wählen Sie im Menü Datei: Neu: Bild.
  2. Wählen Sie die Dateierweiterung aus der Auswahlliste Dateityp aus. Wechseln Sie in das Verzeichnis, in dem sich das Bild befindet, und klicken Sie doppelt darauf, um die Datei zu wählen. Dies öffnet das ausgewählte Bild in einem neuen Bildfenster mit Pixel-für-Pixel-Anzeige.

ODER

Für die Bildtypen .bmp, .gif, .jpg, .png, .tif können Sie ein Bildfenster direkt öffnen. Das heißt:

  1. Wählen Sie Datei: Öffnen.
  2. Wählen Sie Bilder aus der Auswahlliste Dateityp aus. Wechseln Sie in das Verzeichnis, in dem sich das Bild befindet, und wählen Sie die Datei. Klicken Sie auf OK.
Ein Bild ins Matrixfenster importieren
  1. Wählen Sie bei aktivem Matrixfenster Daten: Aus Datei importieren: Bild in Matrix.
  2. Wählen Sie die Dateierweiterung aus der Auswahlliste Dateityp aus. Wechseln Sie in das Verzeichnis, in dem sich das Bild befindet, und wählen Sie die Datei. Klicken Sie auf OK. Der Dialog der X-Funktion impImage wird geöffnet.
  3. Die Importoptionen können im Zweig Importoptionen eingestellt werden. Klicken Sie auf OK, um die Datei zu importieren.

Eine Datei ins Bildfenster importieren
  1. Wählen Sie im Menü Datei: Neu: Bild.
  2. Wählen Sie die Dateierweiterung aus der Auswahlliste Dateityp aus. Wechseln Sie in das Verzeichnis, in dem sich das Bild/Video befindet, und klicken Sie doppelt, um die Datei zu wählen. Die ausgewählte Datei wird in einem neuen Bildfenster geöffnet.
