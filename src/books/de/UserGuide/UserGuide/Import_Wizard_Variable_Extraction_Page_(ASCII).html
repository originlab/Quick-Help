Importassistent, Seite Variablenextraktion (ASCII)

Bedienelemente auf dieser Seite werden hauptsächlich dazu verwendet, um eine Methode zum Extrahieren von Variablen aus dem Headerbereich Ihrer Importdateien festzulegen. Jedes der ersten drei Kontrollkästchen fügt eine Seite zum Importassistenten hinzu. Sie können Variablen extrahieren, indem Sie die Bedienelemente auf jeder einzelnen Seite verwenden oder die Bedienelemente/Seiten zusammen verwenden.

Extrahieren von Variablen aus dem Dateinamen und Dateiheader

Bestimmen des Speicherorts von Variablennamen und Werten, die Trennzeichen verwenden

Wenn Sie diese Option auswählen, wird die Seite Variablenextraktion durch Trennzeichen zum Assistenten hinzugefügt.
Bestimmen des Speicherorts von Variablennamen und Werten, die Trennzeichen verwenden

Wenn Sie diese Option auswählen, wird die Seite Variablenextraktion durch Position zum Assistenten hinzugefügt.
Bestimmen von Trennzeichen und Assistentenüberprüfungen für Variablennamen und Werte

Wenn Sie diese Option auswählen, wird die Seite Headervariablen zum Assistenten hinzugefügt. Dies ist die Methode von Origin 7.5, die es erforderlich macht, ein einzelnes Trennzeichen festzulegen. Dann sucht Origin automatisch nach den Variablennamen und Werten.
Aufrufen einer benutzerdefinierten Origin C-Funktion

Die Auswahl dieser Option ermöglicht es Ihnen, eine Quelldatei und eine Origin C-Funktion in der Datei auszuwählen, die die Variablen auf die gewünschte Weise extrahiert.
Hinzufügen von Variablen zur Seiteninformation

Die Auswahl dieser Option speichert die Variablen in der Zielarbeitsmappe (nur verfügbar, wenn eine der anderen Seitenoptionen Variablenextraktion ausgewählt ist).
Anwenderparameterzeilen für jede Variable erstellen

Durch Auswahl dieser Option werden Anwenderparameterzeilen für jede Variable in der Zielarbeitsmappe erstellt. (nur verfügbar, wenn eine der anderen Optionen der Seite Variablenextraktion ausgewäht ist).

Dateiinfo

Dateiinfo in Arbeitsmappe speichern

Legt fest, ob die Dateiinformationen in der Arbeitsmappe gespeichert werden, die im Arbeitsmappen-Organizer angezeigt wird. Das Speichern der Dateiinformationen erfordert zusätzliche Zeit. Daher erwägen Sie vielleicht diese Option zu deaktivieren, um die Importgeschwindigkeit beim Import mehrerer Dateien zu beschleunigen.
