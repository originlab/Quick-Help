Hinweise

Dieses Diagramm ist eine Variation der Linienkurvenschar. Wenn Sie einige Spalten ausgewählt haben, werden all diese Spalten als eine Linienkurvenschar gezeichnet, die alle Datenpunkte einer Zeile als eine Linie darstellt, es werden dabei jedoch nur alle zwei Datenpunkte verbunden. Falls Sie eine ungerade Anzahl Spalten ausgewählt haben, wird die letzte Spalte als Punkt ohne Verbindung gezeichnet.

Sie können eine Linienkurvenschar auch in ein Vorher-und-Nachher-Diagramm umwandeln:

  • Öffnen Sie den Dialog Details Zeichnung, gehen Sie zur Registerkarte Gruppe, wählen Sie die Option Nach Größe unter Untergruppe aktivieren und geben Sie 2 im Feld Untergruppengröße ein.
