Datenanforderungen

Wählen Sie mindestens eine Spalte mit Werten oder einen Bereich aus mindestens einer Spalte aus.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

  • Wählen Sie Zeichnen: Statistisch: QC (X bar R) Diagramm.
Oder
Klicken Sie auf die Schaltfläche QC (X bar R) Diagramm auf der Symbolleiste 2D Grafiken.

Origin öffnet den Dialog X-bar R Chart.

  • Legen Sie die Teilgruppengröße für den ausgewählten Datensatz fest.

Origin erstellt ein Arbeitsblatt und das Diagrammfenster zeigt zwei Layer an. Das Arbeitsblatt beinhaltet den Mittelwert, die Spannweite und die Standardabweichung für jede Teilgruppe des ausgewählten Datensatzes.

Vorlage

Xbarr.OTP (installiert im Origin-Programmordner)

Hinweise

Der obere Layer der Regelkarte ist die X-quer-Karte. Dieser Layer zeigt die mittleren Werte für jede der Teilgruppen an als ein Punktdiagramm mit Falllinien zum durchschnittlichen Mittelwert aller Gruppen (X-quer). Dieser Layer zeigt auch zwei Grenzlinien an, die im Abstand von Num Sigma Standardabweichungen von X-quer positioniert werden. Num Sigma wird im Fenster des QC1-Worksheets definiert und liegt standardmäßig bei drei.

Der untere Layer des QC-Diagramms stellt die R-Karte dar. Dieser Layer zeigt die Spannweite jeder Teilgruppe als Spaltendiagramm an. Die Spannweite jeder Teilgruppe wird ab der durchschnittlichen Linie (oder R-quer) gezeichnet. Diese Linie repräsentiert die durchschnittliche Spannweite der Gruppen oder den Durchschnitt der mittleren Spannweite innerhalb jeder Teilgruppe. Dieser Layer zeigt auch zwei Grenzlinien an - die OEG- und UEG-Linien. Informationen zur Berechnung von OEG- und UEG-Linien, siehe "Statistical Quality Control" 5th edition (1980) von Grant, E.R. und Leavenworth, R.S., McGraw Hill, Seite 81.
