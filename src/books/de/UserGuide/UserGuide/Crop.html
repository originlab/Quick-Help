Schneiden


Die Funktion imgCrop extrahiert eine rechteckige Fläche aus dem Bild. Dieses Hilfsmittel wird normalerweise mit dem Hilfsmittel Interessenbereiche verwendet, mit dem Bereiche unmittelbar ausgewählt werden können.

Um ein Bild zu schneiden:

Stellen Sie sicher, dass vor dem Hilfsmittel Interessenbereich im Menü Hilfsmittel ein Häkchen gesetzt ist. Wenn dieses Menüelement nicht aktiviert ist, holen Sie das bitte nach.
Klicken Sie auf die Schaltfläche Rechteck auf der Symbolleiste Tools.
Ziehen Sie ein Rechteck auf dem Bild.
Klicken Sie mit der rechten Maustaste auf das Rechteck und wählen Sie Schneiden im Kontextmenü aus.

Weitere Informationen über diese Funktion finden Sie in der Dokumentation der X-Funktion imgCrop in der Origin-Hilfe. Diese kann über das Befehlsfenster mit folgendem Befehl aufgerufen werden:

help imgCrop
