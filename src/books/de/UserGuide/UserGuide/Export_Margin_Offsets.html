Dialog Randverschiebung für Export

Export-Margin_Offsets

Randverschiebung für Export ist ein kleiner Dialog, der mit Rahmenstärke unter Einstellungen: Optionen: Seiteneinstellungen kopieren oder im Zweig Exporteinstellungen des Dialogs Grafiken exportieren zusammenarbeitet, um Ihnen mehr Kontrolle darüber zu geben, welcher Bereich eines Diagramms kopiert bzw. exportiert wird.

Über Randverschiebung für Export

Wenn Sie normalerweise auf Bearbeiten: Seite kopieren oder Bearbeiten: Diagramm als Bild kopieren klicken oder wenn Sie ein Diagramm als Bilddatei exportieren (Datei: Grafiken exportieren), werden die Ränder Ihres kopierten oder exportierten Diagramms über Rahmenstärke in der Gruppe Seiteneinstellungen kopieren unter Einstellungen: Optionen oder den Zweig Exporteinstellungen im Dialog Grafiken exportieren bestimmt.

Die Werte in Randverschiebung für Export sind numerische Werte (ausgedrückt in Pixel), die die Ränder modifizieren, so dass Sie den genauen Bereich eines Diagramms beim Kopieren oder Exportieren benutzerdefiniert anpassen können. Die Verschiebungswerte sind relativ zu den jeweiligen Einstellungen unter Seiteneinstellungen kopieren.

Hinweis: Diese Ränder (ob Verschiebungen nun festgelegt wurden oder nicht) können in den Grafiken durch Verwenden des Menüs Ansicht und Auswahl der Option Zeige: Seite/Exportbereich kopieren angezeigt werden. Alternativ können Sie die Tastaturkombination Strg + 4 verwenden.

Randverschiebung für Export festlegen

Um Randverschiebung für Export für das aktive Diagramm festzulegen, navigieren Sie zum Menü Bearbeiten und wählen Sie Randverschiebung für Export. Der Dialog wird mitsamt dem Rechteck ROI, Region of Interest, im Diagramm angezeigt.

Sie können entweder die Elemente zum Verändern der Größe im Rechteck verwenden, um die Verschiebungen visuell anzupassen, oder ganzzahlige Pixel-Werte direkt im Dialog eingeben. Wenn Sie die Verschiebungen nach Ihren Wünschen eingestellt haben, klicken Sie auf OK im Dialog, um sie im Diagramm festzulegen. Hinweis: Beim Eingeben von Werten im Dialog können Sie auf die Schaltfläche Anwenden klicken, um die Wirkung auf Ihr Diagramm zu prüfen.

Sie können auch die zuvor festgelegten Randverschiebungen entfernen, indem Sie den Dialog öffnen, auf die Schaltfläche Zurücksetzen und dann auf OK klicken.

Hinweis: Um den Dialog schnell wieder zu öffnen, können Sie doppelt auf die schattierten Ränder klicken. Es gibt jedoch keine Randverschiebung für die Diagrammseite. Dieser Dialog wird nicht geöffnet, wenn Sie doppelt auf den Seitenrand klicken.

Weitere Informationen zu: Randverschiebungen sind relativ zu den Einstellungen unter Seiteneinstellungen kopieren

Wie zuvor erwähnt sind Randverschiebungen relativ zu dem jeweiligen Wert unter Seiteneinstellungen kopieren: Rahmenstärke. Dies bedeutet, dass die Ergebnisse unterschiedlich sein können, wenn nacheinander zwei unterschiedliche Werte für Rahmenstärke verwendet wurden.

Allgemein erhält man die besten Ergebnisse, wenn die Rahmenstärke auf Eng gesetzt ist. Ist dies der Fall, werden die tatsächlichen Rändern angepasst, sobald Sie die Größe oder Position eines Layers im Diagramm ändern. Während das Festlegen der Rahmenstärke auf Seite für gute Ergebnisse sorgt, bleibt die Fläche statisch und wird nicht aktualisiert, wenn die Layergröße oder -position geändert wird.

Die Einstellungen von Randverschiebungen funktioniert nicht sehr gut, wenn die Rahmenstärke auf Grenze gesetzt ist. Sie sollten es vermeiden, diese Einstellung vorzunehmen, bevor Sie die Randverschiebungen festgelegt haben.
