Mit Python Bilder verarbeiten

Using-Python-in-Image-Processing

Sie können vom Bildfenster aus mit Hilfe des Pakets originpro, das in der eingebetteten Python-Funktion in Origin enthalten ist, auf Bilder zugreifen und sie verarbeiten.

Im Folgenden sehen Sie ein einfaches Codebeispiel, mit dem ein Bildstapel aus einer externen Datei in ein Bildfenster importiert wird und eines der Bilder dann mit Hilfe des Python-Pakets Scikit-Image umgekehrt wird:

import originpro as op import numpy as np from skimage.util import invert #load image stack fn = op.path('e') + r'Samples\Image Processing and Analysis\*.tif' iw=op.new_image() iw.from_file(fn) print(iw.frames) #get the 3 image im2 = iw.to_np2d(2) im2 *= 2 im2 = invert(im2) #put it back into 2nd image iw.from_np2d(im2, 1) iw.set_int('NAV',1)
