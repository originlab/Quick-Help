Die Symbolleiste Import

Import-Toolbar

Die Schaltflächen der Symbolleiste Import bieten schnellen Zugriff auf das Importieren von Dateien.

Öffnet den Importassistenten.

Öffnet den Dialog CSV-Importoptionen (TXT, CSV, DAT).

Öffnet den Dialog Mehrfachimport ASCII (TXT, DAT, CSV, *.*).

Öffnet den Dialog Excel-Importoptionen (XLS, XLSX, XLSM).

Verwendet die letzte Importeinstellung und den Dateipfad des Arbeitsblatts zum erneuten Importieren der gleichen Daten, ohne nachzufragen.

Zeigt die letzten Importeinstellungen und den Dateipfad im entsprechenden Dialog für den erneuten Import.

Verwendet die letzte Importeinstellung des Arbeitsblatts zum Importieren von neuen Dateien in neuen Arbeitsblättern.

Öffnet einen Dialog zum Auswählen der Dateien für die Stapelverarbeitung.

Importiert alle verbundenen Daten.

Importiert eine Datendatei von Drittanbietern online mit Hilfe des Datenkonnektors.

Verbindet mehrere Datendateien mit einem Datenkonnektor.
