Eingabedaten

Erster Datenbereich Legen Sie den Datenbereich (mehrere Spalten) der ersten Stichprobe fest. Jede Zeile stellt eine Datenvariable dar.
Zweiter Datenbereich Legen Sie den Datenbereich (mehrere Spalten) der zweiten Stichprobe fest. Jede Zeile stellt eine Datenvariable dar.


Testoptionen

Testmittelwert Der zu testende hypothetische Mittelwert (0, falls die Gleichheit der Mittelwert getestet wird)
Alternativhypothese Die Alternativhypothese:
Mittelwert1 - Mittelwert2 <> Testmittelwertdifferenz

Wird für den beidseitigen Test bei zwei Stichproben verwendet. Mittelwert1 und Mittelwert2 unterscheiden sich.

Mittelwert1 - Mittelwert2 > Testmittelwertdifferenz

Wird für den oberen Test bei zwei Stichproben verwendet. Mittelwert1 ist größer als Mittelwert2.

Mittelwert1 - Mittelwert2 > Testmittelwertdifferenz

Wird für den unteren Test bei zwei Stichproben verwendet. Mittelwert1 ist kleiner als Mittelwert2.

Ausgabe

Hilfe zum Festlegen der Bereiche finden Sie unter: Ergebnisse ausgeben

T-Statistik Wählen Sie, ob die T-Statistik ausgegeben werden soll. Sie können die Spalte für die Werte festlegen. Jede Zeile stellt die T-Statistik der Daten in der Zeile dar.
Freiheitsgrade Wählen Sie, ob der Freiheitsgrad ausgegeben werden soll. Sie können die Spalte für die Werte festlegen. Jede Zeile stellt den Freiheitsgrad der Daten in der Zeile dar.
p-Wert Wählen Sie, ob der p-Wert ausgegeben werden soll. Sie können die Spalte für die Werte festlegen. Jede Zeile stellt den p-Wert der Daten in der Zeile dar.
Signifikant Wählen Sie, ob die Markierung der Entscheidung ausgegeben werden soll. Sie können die Spalte für die Werte festlegen. Ein Wert gleich 1 zeigt an, dass die Differenz der Mittelwerte auf dem 0,05-Niveau signifikant ist. 0 bedeutet, dass die Mittelwerte sich auf einem Niveau von 0,05 nicht signifikant unterscheiden.
