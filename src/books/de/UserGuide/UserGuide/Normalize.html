Beschreibung


Es gibt zwei Hilfsmittel der Bedienoberfläche zum Normieren von Daten: eines für Arbeitsblattdaten und eines für gezeichnete Daten. Die Ausgabe von beiden ist ähnlich, und sie werden beide in diesem Kapitel erläutert. Für spezielle Informationen ziehen Sie die Referenzseiten für diese zwei X-Funktionen zurate:

  • Der Dialog rnormalize dient der Verarbeitung von Arbeitsblattdaten.
  • Der Dialog cnormalize dient der Verarbeitung von gezeichneten Daten.
Daten normieren
  1. Klicken Sie bei aktivem Arbeitsblatt auf Analyse: Mathematik: Spalten normieren

oder

  1. klicken Sie bei aktivem Diagrammfenster auf Analyse: Mathematik: Kurven normieren.

Eingabe Definieren Sie Ihre Eingabedaten.

Hilfe zum Festlegen von Bereichen finden Sie hier: Eingabedaten festlegen
Eingabe als einen zusammengesetzten Datensatz behandeln Legen Sie fest, ob alle Eingabedatensätze als ein zusammengesetzter Datensatz behandelt werden sollen. Diese Option ist nicht verfügbar, wenn für Normierungsmethoden die Option Geteilt durch einen festgelegten Wert, Geteilt durch Referenzzelle und Auf den aus dem Diagramm ausgewählten Punkt normieren gewählt ist.

Wenn Sie zum Beispiel zwei Spalten (Spalte A und B) als Eingabedatensätze auswählen können und hoffen, die Daten durch den maximalen Wert zu teilen.
Wenn diese Option nicht aktiviert ist, werden die Daten in Col(A) durch den maximalen Wert in Col(A) geteilt und die Daten in Col(B) durch den maximalen Wert in Col(B).
Wenn diese Option aktiviert ist, erhalten Sie die Statistikwerte aus dem zusammengesetzten Datensatz. Die Daten der zwei Spalten werden dann durch den maximalen Wert aus zwei Spalten geteilt.
Normierungsmethoden
Geteilt durch einen festgelegten Wert
Die Spalte oder die Kurve wird durch den Benutzerdefinierten Wert (siehe unten) geteilt.
Auf (0,1) normieren
Es wird auf den Bereich 0 bis 1 normiert.
Normieren auf (0,100)
Es wird auf den Bereich 0 bis 100 normiert.
Normieren auf [v1, v2]
Die Daten werden auf einen Bereich von benutzerdefinierten Werten normiert.
Z-Score (auf N (0,1) normieren)
Die Daten werden auf die Standardnormalverteilung normiert.
Geteilt durch Max.
Die Spalte oder die Kurve wird durch das Maximum des Datensatzes geteilt.
Geteilt durch Min.
Die Spalte oder die Kurve wird durch das Minimum des Datensatzes geteilt.
Geteilt durch Mittelwert
Die Spalte oder die Kurve wird durch den Mittelwert des Datensatzes geteilt.
Geteilt durch Median
Die Spalte oder die Kurve wird durch den Medianwert des Datensatzes geteilt.
Geteilt durch SD
Die Spalte oder die Kurve wird durch die Standardabweichung des Datensatzes geteilt.
Geteilt durch Norm
Die Spalte oder die Kurve wird durch die Norm des Datensatzes geteilt.
Geteilt durch Modus
Die Spalte oder die Kurve wird durch den Modus des Datensatzes geteilt.
Geteilt durch Summe
Die Spalte oder die Kurve wird durch die Summe des Datensatzes geteilt.
Referenzdiagramm/Referenzspalte verwenden
Die Eingabedaten werden zu den Bedingungen, die in der Liste Normieren auf (siehe unten) festgelegt worden, normiert.
Geteilt durch Referenzzelle
Die Spalte wird durch einen Referenzzellenwert geteilt.
Auf den aus dem Diagramm ausgewählten Punkt normieren
Die Kurve wird durch einen aus einer Kurve ausgewählten Punkt geteilt.
Benutzerdefinierter Wert

Geben Sie einen Wert ein. Verwenden Sie diese Option, wenn Normierungsmethoden = Geteilt durch einen festgelegten Wert.
v1, v2

Verfügbar, wenn Normierungsmethoden = Normieren auf [v1, v2]. Definieren Sie einen Wertebereich, einschließlich v1 und v2.
Referenzspalte/Referenzdiagramm

Benennen Sie eine Arbeitsblattspalte oder ein Referenzdiagramm.
Normieren auf

Diese Option wird verwendet, wenn Normierungsmethoden = Referenzspalte verwenden oder Referenzdiagramm verwenden:

  • Min.: Die Eingabe wird normiert, indem das Minimum des Datensatzes auf das der/s Referenzspalte/-diagramms gesetzt wird.
  • Max.: Die Eingabe wird normiert, indem das Maximum des Datensatzes auf das der/s Referenzspalte/-diagramms gesetzt wird.
  • Mittelwert: Die Eingabe wird normiert, indem der Mittelwert des Datensatzes auf das der/s Referenzspalte/-diagramms gesetzt wird.
  • Median: Die Eingabe wird normiert, indem der Median des Datensatzes auf das der/s Referenzspalte/-diagramms gesetzt wird.
  • Summe: Die Eingabe wird normiert, indem die Summe des Datensatzes auf das der/s Referenzspalte/-diagramms gesetzt wird.

Weitere Informationen zu den Methoden finden Sie im Abschnitt Algorithmus unten.
