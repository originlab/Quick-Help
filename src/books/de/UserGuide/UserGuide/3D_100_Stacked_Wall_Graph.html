Datenanforderungen

Wählen Sie mindestens eine Y-Spalte aus (oder einen Bereich aus mindestens einer Y-Spalte). Idealerweise wählen Sie mindestens zwei Y-Spalten aus (oder einen Bereich aus mindestens zwei Y-Spalten). Die Y-Werte müssen das gleiche Vorzeichen haben.

Wenn es eine verbundene X-Spalte gibt, stellt die X-Spalte die X-Werte bereit; ansonsten wird ein Abtastintervall der Y-Spalte oder Zeilennummer verwendet.

Diagramm erstellen

Wählen Sie im Menü Zeichnen: 3D: Gestapelte 3D-Wände, 100%.

Vorlage

glStackWallsP.OTP (installiert im Origin-Programmordner)

Hinweise

Gestapelte 3D-Wanddiagramme, 100%, stapeln Y-Spalten in einer Wand, deren Höhe der Kumulation der Y-Spalten normiert in [0, 100] entspricht.
