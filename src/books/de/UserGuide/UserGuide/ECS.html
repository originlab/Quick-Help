Function

wobei

Brief Description

Edgeworth-Cramer peak function for use in chromatography.

Sample Curve

Parameter

Number: 6

Names: y0, xc, A, w, a3, a4

Meanings: y0 = offset, xc = center, A = area, w = half width, a3 = unknown, a4 = unknown

Lower Bounds: A > 0.0, w > 0.0

Upper Bounds: none

Script Access

gcas(x,y0,xc,A,w,a3,a4)

Function File

FITFUNC\EDGWTHCR.FDF

Category

Peak Functions, PFW, Chromatography
