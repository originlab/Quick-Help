Symbolleistenschaltflächen

Neuer X-Funktionen-Assistent

Öffnet den Neuen X-Funktionen-Assistenten. Der Assistent führt Sie durch die Schritte der Erstellung einer neuen X-Funktion. Dupliziert den Menübefehl Datei: Neuer X-Funktionen-Assistent.
Öffnen der X-Funktion mit der OXF-Datei

Diese Schaltfläche öffnet den Ordner der zuletzt aufgerufenen X-Funktion. Sie können auch nach einer zu bearbeitenden X-Funktion suchen. Dupliziert den Menübefehl Datei: Anwender-X-Funktionen öffnen.
Speichern der OXF-Datei

Speichern der aktuellen X-Funktion Beachten Sie, dass X-Funktionen nicht im Ordner \X-Functions im Origin-Programmordner gespeichert werden können. Dupliziert den Menübefehl Datei: Speichern.
Bearbeiten der X-Funktion im Code Builder

Öffnet die X-Funktion im Code Builder. Dupliziert den Menübefehl Hilfsmittel: X-Function im Code Builder bearbeiten.
Baumansicht

Umschalten zwischen Standardansicht und Baumansicht Wenn der X-Funktion-Builder sich im Modus Baumansicht befindet, ist die Schaltfläche gedrückt angezeigt. Dupliziert den Menübefehl Hilfsmittel: Baumansicht.
