Datenanforderungen

Wählen Sie genau eine Y-Datenspalte (oder einen Bereich aus einer Spalte) aus.

Diagramm erstellen

Wählen Sie die erforderlichen Daten aus

Wählen Sie Zeichnen > Statistisch: Wahrscheinlichkeitsdiagramm.

Oder

Klicken Sie auf die Schaltfläche Wahrscheinlichkeitsdiagramm auf der Symbolleiste 2D Grafiken.

Origin öffnet den Dialog plot_prob.

Passen Sie die Optionen im Dialogfeld benutzerdefiniert an und klicken Sie dann auf OK, um das Diagramm zu erstellen. Bitte lesen Sie weitere Einzelheiten zum Erstellen und benutzerdefinierten Anpassen von Wahrscheinlichkeitsdiagrammen.

Vorlage

PP.OTP (im Origin-Programmordner installiert)
