Brief Description

Exponential growth 2 with offset.

Sample Curve

Parameters

Number: 6

Names: y0, x0, A1, t1, A2, t2

Meanings: y0 = offset, x0 = center, A1 = amplitude, t1 = width, A2 = amplitude, t2 = width

Lower Bounds: none

Upper Bounds: none

Note: different initial parameters are used for t1 and t2 so as to make sure t2 > t1 in the fitting results.

Script Access
