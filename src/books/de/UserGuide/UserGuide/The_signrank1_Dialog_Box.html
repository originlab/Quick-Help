Testmedian

Nullhypothese

Die zu testende Nullhypothese

Alternativhypothese
  • Median <> 0
Führt einen beidseitigen Wilcoxon-Rangtest mit Vorzeichen durch.
  • Median > 0
Führt einen oberen Wilcoxon-Rangtest mit Vorzeichen durch.
  • Median < 0
Führt einen unteren Wilcoxon-Rangtest mit Vorzeichen durch.
