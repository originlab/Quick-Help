Datenanforderungen

Wählen Sie mindestens ein Paar XY-Spalten. Wird Polar θ(X) r(Y) zum Zeichnen eines Diagramms verwendet, stellt X den Winkel (Einheiten in Grad) und Y den Radius dar. Bei Verwendung von Polar θ(X) r(Y) ist X der Radius und Y stellt den Winkel (Einheiten in Grad) dar.

Diagramm erstellen

Polar Theta(X) r(Y) soll hier als Beispiel dienen.

  1. Wählen Sie die gewünschten Daten aus.
  2. Wählen Sie im Menü Zeichnen > Spezialisiert: Polar θ(X) r(Y)/Polar r(X) θ(Y).
oder
Klicken Sie auf die Schaltfläche Polar Theta(X) r(Yr) auf der Symbolleiste 2D Grafiken.

 

  • Balken, Polar: Wählen Sie im Menü Zeichnen > 2D: Spezialisiert: Balken θ, r/Balken r, θ.
  • Linie + Symbol, Polar: Wählen Sie im Menü Zeichnen > 2D: Spezialisiert: Linie + Symbol θ, r / Linie+Symbol r, θ.
  • Symbol, Polar: Wählen Sie im Menü Zeichnen > 2D: Spezialisiert: Symbol θ, r/Symbol r, θ.

Vorlage

POLAR.otpu für Polar θ(X) r(Y).

PolarXrYTheta.otpu für Polar r(X) θ(Y).

POLARTHETARLINE+SYMBOL.otpu für Linie + Symbol θ, r

PolarRThetaLine+Symbol.otpu für Linie + Symbol r, θ

POLARTHETARSCATTER.otpu für Symbol θ, r

PolarRThetaScatter.otpu für Symbol r, θ

POLARTHETARBAR.otpu für Balken θ, r

PolarRThetaBar.otpu für Bar r, θ

(Alle oben genannten Vorlagen sind im Origin-Programmordner installiert).
