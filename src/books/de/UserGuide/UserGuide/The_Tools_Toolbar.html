Symbolleiste Hilfsmittel

Tools-Toolbar

Die Symbolleiste Hilfsmittel bietet Schaltflächen zur Datenauswahl, Maskierung, Vergrößerung eines Teils der Achsenskalierung, Anmerkung und zum Zeichnen. Verwandte Hilfsmittel sind gruppiert und können aus einem Popup-Menü heraus geöffnet werden, indem Sie auf die Schaltfläche auf der rechten Seite der Schaltfläche klicken.

Wird im Standardmodus der Objektauswahl verwendet.

Vergrößert einen Teil einer Achsenskalierung.

Sie können ein gleichmäßiges Rechteck aufziehen (das Symbol der Vergrößerung wird mit angezeigt), um das Diagramm zu vergrößern und nur den gewünschten Bereich des Diagramms anzuzeigen.

Drücken Sie die ESC-Taste, um den Vergrößerungsmodus zu verlassen. Ein Doppelklick in diesem Vergrößerungsmodus auf die Schaltfläche stellt die Diagrammskalierung wieder her. Ein Doppelklick auf das Diagramm erstellt ein vergrößertes Diagramm.

Stellt die Achsenskalenwerte auf ihre vorherigen Werte Von und Bis zurück.

Erstellt ein Viertel einer Bogenkurve (Radiant = 90 Grad).

Verwenden Sie dieses Hilfsmittel, um ein Rechteck zu zeichnen. Die Breite und Länge des Rechtecks werden als Halbachsenlänge und Halbachsenbreite einer Ellipse betrachtet. Standardmäßig zeichnet dieses Hilfsmittel immer 1/4 der Ellipse innerhalb des Rechtecks. Halten Sie die Shift-Taste, um ein Quadrat zu zeichnen. Es wird 1/4 eines Kreises erstellt.

Nach Erstellen einer Bogenkurve können Sie die Endelemente verwenden, um den Bogen von einem Maximum von 360 Grad auf ein Minimum von 0 Grad zu verlängern oder zu kürzen.
