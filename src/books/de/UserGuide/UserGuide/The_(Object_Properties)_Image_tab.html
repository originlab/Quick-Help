Die Registerkarte Bild (Objekteigenschaften)

Ob-Prop-Image-tab


Wählen Sie das Kästchen Platzhalter verwenden, um den Platzhalter zu verwenden. Dieses Kontrollkästchen ist verfügbar, wenn das Objekt ein Arbeitsblatt ist.

Um einen Hintergrund für das ausgewählte Objekt hinzuzufügen, aktivieren Sie das Kontrollkästchen Hintergrund und wählen Sie eine gewünschte Option aus der Kombinationsliste. Deaktivieren Sie dieses Kontrollkästchen, um das Objekt ohne einen Hintergrund anzuzeigen.

Das Kontrollkästchen Interpolation aktivieren kann verwendet werden, um zu entscheiden, ob die Interpolation zum Bildrendern aktiviert werden soll. Dieses Bedienelement ist der Systemvariablen @ii gleichzusetzen.
