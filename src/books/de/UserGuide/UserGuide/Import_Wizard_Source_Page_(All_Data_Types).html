Importassistent, Quellseite (Alle Datentypen)


Verwenden Sie die Bedienelemente dieser Seite, um Importdateinamen, Dateityp, Zielvorlage und Importfilter festzulegen. Diese Seite haben alle Datentypen: ASCII, Binär und Benutzerdefiniert gemeinsam. Beachten Sie, dass die Verfügbarkeit einiger Bedienelemente abhängig ist von anderen Einstellungen, die auf dieser Seite vorgenommen wurden. Von daher empfiehlt es sich, dieses Dialogfeld systematisch von oben nach unten zu bearbeiten.

Datentyp

ASCII

ASCII-Dateien, einschließlich derjenigen, die Erweiterte ASCII-Zeichen enthalten.
Binär

Dateien mit einem optionalen Headerbereich gefolgt von einer sequenziellen binären Struktur
Benutzerdefiniert

Hauptsächlich jede Datei, die nicht von den ASCII- oder binären Importroutinen gelesen werden kann. Der Anwender muss Importanweisungen in der Form von Origin C-Code eingeben.

Datenquelle

Datei

Um Daten aus einem ASCII-, Binär- oder Benutzerdefinierten Dateityp zu importieren, wählen Sie die Option Datei aus. Geben Sie den Dateinamen ein (oder verwenden Sie die Schaltfläche Durchsuchen, um die Datendatei zu finden).
Zwischenablage

Um Daten zu importieren, die in die Zwischenablage kopiert wurden, wählen Sie die Option Zwischenablage aus.

Importfilter

Filter, die auf Datentyp und Dateinamen anwendbar sind, auflisten

Wenn dieses Kontrollkästchen deaktiviert ist, werden alle Filter, die auf den Datentyp angewendet werden in der Auswahlliste Importfilter für aktuellen Datentyp angezeigt. Ist dieses Kontrollkästchen aktiviert, zeigt die Auswahlliste Importfilter für aktuellen Datentyp nur die Filter an, die mit den Einstellungen Dateinamen angeben... auf der Seite Filter speichern des Importassistenten übereinstimmen (einschließlich Filter, die aufgrund von Platzhaltereinstellungen übereinstimmen).
Filter für aktuellen Dateityp importieren

Die Auswahlliste Importfilter für aktuellen Datentyp zeigt alle Filterdateien (.OIF) an, die sich an folgenden Orten befinden:

  • Derselbe Unterordner wie die Datei Diese Filternamen beginnen mit dem Präfix für Datenordner.
  • im Unterordner \Filter des Anwenderdateipfads Diese Namen enthalten das Präfix für das Anwenderdateiverzeichnis.
  • Der Unterordner \Filter im Origin-Programmverzeichnis Diese Namen enthalten das Präfix für Origin-Ordner.
Beschreibung

Dieses nicht-bearbeitbare Textfeld listet jeden beschreibenden Text auf, der in dem Feld Filterbeschreibung auf der Seite Filter speichern des Importassistenten bei der Erstellung des Filters (.OIF) eingegeben wurde.

Zielfenster

Zielfenster

Legen Sie ein Zielfenster -- Worksheet, Matrix oder Keine -- für den Datenimport fest.
Vorlage

Geben Sie der Vorlage, die Sie beim Erstellen neuer Zielfenster verwenden wollen, einen Namen. Neue Fenster werden erstellt:

  • Während Drag&Drop-Vorgängen
  • Bei Verwendung des Menüs Datei:Öffnen zum Öffnen von Datendateien
  • Gemäß den Einstellungen der Auswahlliste Import Modus (siehe Folgendes).

Wenn keine Vorlage festgelegt wurde, wird die Origin-Standardvorlage der Arbeits- oder Matrixmappe verwendet (dies wäre die Arbeitsmappe oder die Matrixmappe, die mit den Schaltflächen Neu... in der Symbolleiste Standard geöffnet würde).

Existiert ein Filter für den benannten Dateityp und besitzt dieser Filter eine Vorlagenspezifikation, so wird das neue Fenster mit Hilfe der Vorlage erstellt, die in der Auswahlliste Importfilter für aktuellen Datentyp festgelegt ist.
Import-Modus

Das Bedienelement der Auswahlliste Import Modus legt fest, wohin die importierten Daten geschrieben werden (in Übereinstimmung mit den Spezifikationen Zielfenster und Vorlage).

Die Daten werden in das aktive Fenster importiert, wenn der Import Modus auf Bestehende Daten ersetzen oder Neue Datenblätter/Spalten/Zeilen öffnen eingestellt ist.

Die Daten werden in die neuen Fenster importiert und mit der benannten Vorlage erstellt, wenn der Import Modus auf Neue Arbeitsmappen öffnen gesetzt und das aktive Fenster nicht leer ist oder nicht zum Typ des Zielfensters gehört.
