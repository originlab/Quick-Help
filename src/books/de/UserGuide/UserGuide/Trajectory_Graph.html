3D-Punktdiagramm + Ankerlinie

Trajectory-Graph

Datenanforderungen

Wählen Sie eine Z-Spalte (oder einen Bereich einer Z-Spalte) aus. Falls die Z-Spalte verbundene XY-Spalten besitzt, werden die XY-Spalten verwendet; ansonsten werden die XY-Standardwerte des Arbeitsblatts verwendet.

Diagramm erstellen

Wählen Sie die erforderlichen Daten im Arbeitsblatt.

Wählen Sie im Menü Zeichnen: 3D: 3D-Punkt + Ankerlinie.

oder

Klicken Sie auf die Schaltfläche 3D Punktdiagramm + Ankerlinie auf der Symbolleiste 3D- und Konturdiagramme.

Vorlage

  • glTraject.OTP (OpenGL)
  • TRAJECT.OTP

(installiert im Origin-Programmordner)
