Einführung

Die deskriptive Statistik wird zur Beschreibung der grundlegenden Merkmale von Datensätzen verwendet. Sie fasst Aspekte zusammen wie Mittelwerte, Standardabweichung, Quantile, Anzahl der Stichproben etc.

Die Zeilenstatistik wird verwendet, um Aspekte von Daten, die in Zeilen gespeichert sind, zusammenzufassen.

Umgang mit fehlenden Werten

Die fehlenden Werte in dem Datenbereich werden aus der Analyse ausgeschlossen. Der Dialog Zeilenstatistik enthält die Option, die Anzahl der fehlenden Werte in jeder Zeile auszugeben.

Zeilenstatistik durchführen

Um die Zeilenstatistik für einen Datenbereich zu erzeugen:

  1. Wählen Sie Statistik: Deskriptive Statistik: Zeilenstatistik. Dadurch öffnet sich der Dialog der Zeilenstatistik.
  2. Legen Sie im Zweig Eingabedaten den Datenbereich fest, der analysiert werden soll. Im Zweig Zu berechnende Eigenschaften legen Sie deskriptive Statistiken fest.
  3. Optional bieten Auflistungen unter dem Zweig Steuerung Berechnung Optionen zur Berechnung der Momente und der Quantile.
  4. Nachdem Sie auf OK geklickt haben, wird ein Analysebericht erstellt, der die gewählten Statistiken anzeigt.
