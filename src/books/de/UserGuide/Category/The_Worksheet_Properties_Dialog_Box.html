Dialog Arbeitsblatteigenschaften

WksProperties-Dialog

Die Elemente des Arbeitsblatts -- die Zeilen- und Spaltenüberschriften, die Gitternetzlinien, die Größe und das Format usw. -- werden in erster Linie im Dialog Arbeitsblatteigenschaften festgelegt. Der Dialog Arbeitsblatteigenschaften kann durch Auswahl von Format: Worksheet im Menü oder durch Drücken von F4 geöffnet werden.

  • Registerkarte Format (Arbeitsblatteigenschaften)
  • Registerkarte Verschiedenes (Arbeitsblatteigenschaften)
  • Registerkarte Größe (Arbeitsblatteigenschaften)
  • Registerkarte Druck/Export (Arbeitsblatteigenschaften)
  • Registerkarte Ansicht (Arbeitsblatteigenschaften)
  • Registerkarte Skript (Arbeitsblatteigenschaften)
