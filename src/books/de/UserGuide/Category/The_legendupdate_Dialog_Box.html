Dialog legendupdate

 

legendupdate-dlg

Der Dialog legendupdate wird verwendet, um Legenden auf der Diagrammseite/im Layer zu aktualisieren oder zu rekonstruieren. Der Zugriff auf ihn erfolgt bei aktivem Diagrammfenster durch Auswahl von Grafik: Legende: Legende aktualisieren. Dieser Dialog verwendet die X-Funktion legendupdate.

Die folgende Tabelle enthält Referenzen für die Bedienelemente in diesem Dialog:

Aktualisierungsmodus

Legt fest, ob die Legende rekonstruiert oder aktualisiert werden soll.

  • Aktualisieren
    Aktualisiert die Daten der vorhandenen Legende(n) über Automatischer Übersetzungsmodus der Legende. Das Format und die Struktur der vorhandenen Legende(n) werden nicht verändert.
  • Rekonstruieren
    Die Legende(n) wird entsprechend der Einstellungen für Automatischer Übersetzungsmodus der Legende und Ordnung neu erstellt. Das Legendenformat wird auf die Standardeinstellungen zurückgesetzt.
Ordnung Diese Option ist nur dann verfügbar, wenn der Aktualisierungswert auf Rekonstruieren gesetzt wurde. Hiermit wird die Zeichnungsordnung in der rekonstruierten Legende festgelegt. Sie kann entweder Aufsteigend oder Absteigend sein. Beachten Sie, dass Sie für eine Legende, die nie manuell aktualisiert wurde, auch Grafik: Legende: Umgekehrte Ordnung verwenden können, um die Einträge der Datenzeichnungen umzukehren, ohne die Legende rekonstruieren zu müssen.
Automatischer Übersetzungsmodus der Legende

Zum Erstellen der Legenden werden die Notationen der LabTalk-Variablen ("%(1)", "%(2)" etc.) verwendet. Origin "übersetzt" beim Erstellen einer Legende die Notationen automatisch in Informationen. Diese Variable legt die Informationen fest, die verwendet werden, um die Notationen zu übersetzen. Wenn die Option Benutzerdefiniert gewählt ist, können Sie eine benutzerdefinierte Notation verwenden. Ansonsten können Sie den Datenbereichsnamen oder eine Variable in den Spaltenbeschriftungszeilen verwenden. Einzelheiten zu den unterstützten Variablennotationen erhalten Sie in der Tabelle der Textbeschriftungsoptionen.
Benutzerdefiniertes Legendenformat (@D,@LU etc.) Diese Option ist nur verfügbar, wenn Benutzerdefiniert unter Automatischer Übersetzungsmodus der Legende gewählt wurde. Es wird ein Auswahlmenü angezeigt, das mehrere häufig verwendete Notationskombinationen enthält. Die Kombination der Legendennotationen mit einem beliebigen Buchstaben vor, nach und dazwischen ist erlaubt. Beispiel:
@LL\+(o)C: Spaltenlangname der Datenzeichnung plus Grad Celsius.

Sie können auf die Schaltfläche neben dem Kombinationsfeld klicken, um die Liste der @Options anzuzeigen. Sie können in der Tabelle der Textbeschriftungsoptionen die unterstützten Notationen nachlesen.
Zusätzliches Format (für @L, @V etc.) Dies ist nur verfügbar, wenn Benutzerdefiniert für Automatischer Übersetzungsmodus der Legende und @L, @V oder @WT für Benutzerdefiniertes Legendenformat (@D, @LU etc) ausgewählt ist. Sie können für das zusätzliche Format Folgendes eingeben: n-te X-Wert für @L, n-te Y-Wert für @V und Zelle der n-ten Spalte sowie m-ten Zeile im Quellarbeitsblatt für @WT. Einzelheiten erhalten Sie in der Tabelle der Textbeschriftungsoptionen.
Legende nur für sichtbare Zeichnungen zeigen

Dieses Kontrollkästchen legt fest, ob die verborgene Zeichnung in der Legende gezeigt wird oder nicht. Wenn das Kontrollkästchen aktiviert wird, wird die verborgene Zeichnung nicht in der Legende gezeigt. Beachten Sie, dass Sie auch mit der rechten Maustaste auf die Legende klicken und Legende: Legende nur für sichtbare Zeichnungen zeigen wählen können, um die verborgenen Zeichnungen zu verbergen.
Legende für angepasste Kurven verbergen

Dieses Kontrollkästchen legt fest, ob die angepasste Kurve in der Legende verborgen wird oder nicht. Wenn das Kontrollkästchen aktiviert wird, wird die Anpassungskurve nicht in der Legende gezeigt. Beachten Sie, dass Sie auch mit der rechten Maustaste auf die Legende klicken und Legende: Legende für angepasste Kurven verbergen wählen können, um die Kennzeichen für den aktiven Datensatz zu verbergen.
Aktiven Datensatz kennzeichnen Dieses Kontrollkästchen bestimmt, ob der aktive Datensatz in der Legende gekennzeichnet wird. Wenn es aktiviert ist, wird das Zeichnungssymbol des aktiven Datensatzes in der Legende mit einem roten Rechteck umrahmt.

Beachten Sie, dass Sie beide der folgenden Methoden wählen können, um die Kennzeichnung des aktiven Datensatzes anzuzeigen:

  • Klicken Sie mit der rechten Maustaste auf die Legende und wählen Sie Legende: Aktiven Datensatz kennzeichnen.
  • Wählen Sie Format: Seiteneigenschaften und gehen Sie zur Registerkarte Legenden/Titel. Aktivieren Sie unten Aktiven Datensatz kennzeichnen.
