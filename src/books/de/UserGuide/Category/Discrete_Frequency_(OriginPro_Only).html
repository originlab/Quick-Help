Einführung

Die diskrete Häufigkeitsanalyse ist eine häufig verwendete Methode der deskriptiven Statistik, die verwendet wird, um diskrete Variablen zu analysieren. Die diskrete Häufigkeitsanalyse kann eine Häufigkeitstabelle ausgeben, aus der die Zusammenfassung der Daten ganz einfach und intuitiv abgelesen werden kann.

Die Zusammenfassung enthält eine Anzahl, die relative Häufigkeit und die kumulative Häufigkeit der Daten, vorausgesetzt, die entsprechenden Kontrollkästchen sind aktiviert.

Umgang mit fehlenden Werten

Die fehlenden Werte in dem Datenbereich werden in die Analyse eingeschlossen, und die Anzahl der fehlenden Werte wird in dem Ergebnis ausgegeben.

Die fehlenden Werte in dem Gruppierungsbereich werden als eine Gruppe betrachtet.

Analyse der diskreten Häufigkeit durchführen

Um die Anzahl einer diskreten Häufigkeit für einen Wertebereich zu bestimmen:

  1. Wählen Sie Statistik: Deskriptive Statistik: Diskrete Häufigkeit. Dieser Menübefehl öffnet den Dialog discfreq. Origin durchsucht den ausgewählten Bereich der Daten, um die Anzahl der Beobachtungen bei jedem eindeutigen Wert zu erfassen.
  2. Wird auf OK geklickt, erstellt Origin ein Ausgabearbeitsblatt. Das Ausgabearbeitsblatt führt die eindeutigen Werte und die entsprechende Anzahl auf. Die relative Häufigkeit und kumulative Häufigkeit können auch berechnet werden.
