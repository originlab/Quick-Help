Glätten

Glättung ist eine Methode der Signalverarbeitung, die normalerweise verwendet wird, um Rauschen von Signalen zu entfernen. Diese Funktion führt Glättung durch, indem Detailkoeffizienten des Signals abgeschnitten werden. Die Option Abschneiden, Cutoff, dieser Funktion bestimmt den abzuschneidenden Prozentsatz der Detailkoeffizienten. Je größer der Wert ist, desto glatter wird das Signal.

Wenn die Eingabe aus einem Diagramm kommt, wird das entrauschte Signal dem Eingabediagramm hinzugefügt.

Um die Wavelet-Glättung zu verwenden:

  1. Aktivieren Sie eine Arbeitsmappe oder ein Diagramm.
  2. Wählen Sie Analyse: Signalverarbeitung: Wavelet: Glätten im Origin-Menü.
