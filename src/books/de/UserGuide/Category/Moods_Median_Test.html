Einführung

Der Mood-Median-Test ist ein nicht-parametrischer Test zum Vergleichen von zwei oder mehreren unabhängigen Stichproben. Er stellt eine Alternative zur ANOVA dar. Anders als einen ANOVA setzt er jedoch keine Normalverteilung der Stichproben voraus. Daher ist er nützlich, wenn Mediane verglichen werden sollen, deren Normalverteilung zweifelhaft ist. In diesem Test besteht die Nullhypothese darin, dass die Mediane gleich sind. Die Alternativhypothese ist, dass mindestens zwei der Mediane der Stichproben unterschiedlich sind.

Umgang mit fehlenden Werten

Die fehlenden Werte in dem Datenbereich werden aus der Analyse ausgeschlossen.

Ab Origin 2015 werden fehlende Werte im Gruppierungsbereich und die entsprechenden Datenwerte aus der Analyse ausgeschlossen. In den älteren Versionen wurden die fehlenden Werte in dem Gruppierungsbereich als eine Gruppe betrachtet.

Mood-Median-Test durchführen

Um einen Mood-Median-Test durchzuführen:

  1. Wählen Sie Statistik: Nicht-parametrische Tests: Moods Mediantest. Der Dialog mediantest wird geöffnet.
  2. Legen Sie die Eingabedaten fest.
  3. Wenn Sie auf OK klicken, wird ein Analysebericht erstellt, der die Häufigkeitstabelle, die Freiheitsgrade, die Chi-Quadrat-Statistiken, den zugehörigen p-Wert und die Testschlussfolgerung anzeigt.
