Der Dialog Werte setzen kann verwendet werden, um Daten für eine oder mehrere Arbeitsblattspalten zu erzeugen.

Um den Dialog Werte setzen zu öffnen, können Sie:

  1. eine oder mehrere Spalten oder einen Bereich von Zellen in einer bzw. mehreren Arbeitsblattspalte/n auswählen.
  2. Wählen Sie dann im Menü Spalte: Spaltenwerte errechnen oder Spalte: Werte für mehrere Spalten festlegen.

oder

  1. eine oder mehrere Spalten oder einen Bereich von Zellen in einer bzw. mehreren Arbeitsblattspalte/n auswählen.
  2. Klicken Sie dann mit der rechten Maustaste auf die Arbeitsblattspalte/n und wählen Sie im Kontextmenü Spaltenwerte errechnen oder Werte für mehrere Spalten festlegen.

oder

  1. Klicken Sie mit der rechten Maustaste auf die Spaltenbeschriftungszelle F(x) und wählen Sie Dialog öffnen. Diese Option ist nur für die einzelne Spalte verfügbar.
