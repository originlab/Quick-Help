Einführung

Diese Funktion führt einen Chi-Quadrat-Test auf Varianz bei einem gegebenen Signifikanzniveau durch, um zu bestimmen, ob eine aus einer Normalverteilung ausgewählte Stichprobe einen gegebenen hypothetischen Varianzwert besitzen kann oder nicht.

Umgang mit fehlenden Werten

Die fehlenden Werte in dem Datenbereich werden aus der Analyse ausgeschlossen.

Test auf Varianz bei einer Stichprobe durchführen

Um einen Test auf Varianz bei einer Stichprobe durchzuführen:

  1. Wählen Sie Statistik: Hypothesentest: Test auf Varianzen bei einer Stichprobe. Dadurch öffnet sich das Dialogfeld OneSampleTestVar.
  2. Legen Sie Ihre Eingabedaten, die Testvarianz und die Alternativhypothese fest.
  3. Indem Sie auf OK klicken, wird ein Analysebericht erstellt, der die Freiheitsgrade, die Chi-Quadrat-Statistiken, den zugehörigen p-Wert und die Testschlussfolgerung anzeigt. Zusätzlich können Sie Konfidenzintervalle für die Stichprobenvarianz, ein Histogramm und ein Box-Diagramm erstellen.
