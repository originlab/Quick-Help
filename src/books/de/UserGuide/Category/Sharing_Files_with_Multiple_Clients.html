Dateien mit mehreren Anwendern gemeinsam nutzen

Origin 8 bietet auch die Funktion Gruppenordner, bei der ein Gruppenleiter oder Serverrechner verschiedene Origin-Dateitypen mit mehreren Gruppenmitgliedern oder Client-Rechnern teilen kann. Die Begriffe Server und Client werden hier in einer sehr allgemeinen Bedeutung verwendet.

Die Funktion Gruppenordner kann von jedem Betrieb/ jeder Organisation verwendet werden, die mehrere Origin-Anwender mit einer gleichzeitigen FLEXlm-Lizenz verwalten.

Weitere Informationen zu diesem Thema lesen Sie bitte auf der Supportseite der Webseite OriginLab.
