Das Dialogfeld X-Funktion Builder

Das Dialogfeld X-Funktion Builder (siehe Bild unten) wird zum Erstellen und Bearbeiten der X-Funktionen -- System und anwenderdefiniert -- verwendet. Es wird im Menü Hilfsmittel geöffnet (Hilfsmittel: X-Funktion Builder).

Das Dialogfeld X-Funktion Builder bietet zwei Ansichten: Standardansicht und Baumansicht. Die Standardansicht ermöglicht Ihnen, die X-Funktion zu benennen und die X-Funktionsvariablen zu bearbeiten, während die Baumansicht Zugriff auf eine größere Vielzahl von X-Funktion-Eigenschaften bietet. Sie können zwischen den Ansichten durch Klicken auf die Schaltfläche Baumansicht oben im Dialigfeld wechseln.

Es gibt zwei Menüs in diesem Dialogfeld: das Menü Datei und das Menü Hilfsmittel. Das Menü Datei wird zum Erstellen, Öffnen und Speichern von X-Funktionen verwendet. Es bietet zudem Shortcuts zu den zuletzt geöffneten X-Funktionen. Das Menü Hilfsmittel kann zum Wechseln zwischen den zwei Ansichten verwendet werden und zum Öffnen der aktuellen X-Funktion im Arbeitsbreich des Code Builders. Die Dialogfeldsymbolleiste ist in beiden Ansichten verfügbar.

  • Menü Datei
  • Menü Hilfsmittel
  • Symbolleistenschaltflächen
  • Weitere Bedienelemente der Standardansicht
  • Weitere Bedienelemente der Baumansicht
  • Der X-Funktionen-Assistent
