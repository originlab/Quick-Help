Das Funktionsdiagramm

 

Ein Funktionsdiagramm zeigt eine 2D-Kurve oder 3D-Oberfläche, die von einer Formel oder einer Reihe von Formeln beschrieben werden. Funktionen lassen sich in eine der Standarddiagrammvorlagen von Origin oder in ein aktives Diagrammfenster zeichnen, das zusätzliche benutzerdefinierbare Einstellungsmöglichkeiten enthält.

Origin unterstützt jetzt das Zeichnen von 2D-Funktionen, parametrischen 2D-Funktionen und 3D-Funktionen.
