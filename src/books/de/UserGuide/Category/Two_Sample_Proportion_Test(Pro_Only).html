Einführung

Im Allgemeinen betrachten Tests von Anteilen bei zwei Stichproben die proportionale Darstellung von zwei Gruppen in einer Stichprobe, die untersucht werden soll. Zum Beispiel können Sie nach Unterschieden in den Antworten von Männern und Frauen auf eine neue Medikation suchen; oder Sie sind vielleicht an den Einstellungen der Bürger interessiert, die sich mit einer von zwei politischen Parteien hinsichtlich einer ausstehenden Gesetzgebung identifizieren.

Zusätzlich zum Hypothesentest bietet Origins Test von Anteilen bei zwei Stichproben eine flexible Dateneingabe (es werden 3 Formen unterstützt), die Trennschärfeanalyse sowie zwei Methoden für die Berechnung von Konfidenzintervallen.

Umgang mit fehlenden Werten

Die fehlenden Werte in dem Datenbereich werden aus der Analyse ausgeschlossen.

Ab Origin 2015 werden fehlende Werte im Gruppierungsbereich und die entsprechenden Datenwerte aus der Analyse ausgeschlossen. In den älteren Versionen wurden die fehlenden Werte in dem Gruppierungsbereich als eine Gruppe betrachtet.
