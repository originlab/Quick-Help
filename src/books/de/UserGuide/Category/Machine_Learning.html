Machine-Learning

Einführung

Machine Learning (oder Maschinenlernen) ist eine Methode zur Datenanalyse, die Informationen direkt den Daten entnimmt und "erlernt", um das Erstellen von analytischen Modellen zu automatisieren. Die Algorithmen des Maschinenlernens nutzen verschiedene Optionen, um ihr Wissen aufzubauen: direkt aus den Daten, durch das Identifizieren von Mustern und durch das Fällen von Entscheidungen.

Es gibt zwei Arten von Aufgaben im Machine Learning, das überwachte Lernen (Supervised) und das unüberwachte Lernen (Unsupervised).

  • Überwachtes Lernen: Prädiktive Modelle werden auf Grundlage von Eingabedaten entwickelt und neue Beobachtungen werden vorher definierten Gruppen zugeordnet.
  • Unüberwachtes Lernen: Daten werden basierend auf den Eingabedaten interpretiert und gruppiert. Beobachtungen bzw. Variablen werden anhand von Gruppen klassifiziert.
