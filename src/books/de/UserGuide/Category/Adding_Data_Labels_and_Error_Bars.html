Datenbeschriftungen und Fehlerbalken hinzufügen

Add-DataLabel-ErrBars

Es gibt zahlreiche Optionen zum Hinzufügen von Beschriftungen und Fehlerbalken zu Ihrem Diagramm:

  • Die älteste und einfachste Methode besteht darin, die Arbeitsblattspalten, die Beschriftungen oder Fehlerbalken enthalten, vorher schon zuzuweisen und sie dann im Arbeitsblatt zusammen mit den XY-Datensätzen auszuwählen, bevor Sie Ihren Diagrammtyp wählen.
  • Bei den meisten 2D-Diagrammen können Sie Beschriftungen hinzufügen, nachdem das Diagramm erstellt wurde. Dies tun Sie einfach, indem Sie den Dialog Details Zeichnung öffnen, zur Registerkarte Beschriftung des Datensatzes gehen und dann Ihre Beschriftungen aktivieren und formatieren.
  • Sie können Datenbeschriftungen zu vielen Diagrammen hinzufügen, indem Sie das Diagramm im Grafikfenster auswählen und dann auf die Schaltfläche Datenbeschriftungen zeigen auf der angezeigten Minisymbolleiste klicken.
  • Sie können Fehlerbalken zu einem einfachen 2D-Diagramm hinzufügen, indem Sie Ihre Daten auswählen, dann auf Zeichnen: Einfache 2D klicken und Y-Fehler oder XY-Fehler wählen. Alternativ klicken Sie auf die entsprechende Symbolleistenschaltfläche.
  • Sie können auch Diagramme über den Dialog Diagrammeinstellungen erstellen. In dem Fall weisen Sie Ihre Datensätze der Beschriftungen bzw. Fehlerbalken im Dialog selbst zu. Diese Option wird nicht in diesem Abschnitt erläutert. Informationen dazu finden sie aber hier.
