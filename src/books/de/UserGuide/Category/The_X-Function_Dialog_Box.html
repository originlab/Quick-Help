Das Dialogfeld X-Function

Eine X-Funktion kann als interaktiver Dialog programmiert werden, was es dem Anwender ermöglicht, Informationen wie Ein- und Ausgabedaten und Werte für verschiedene Eingabeparameter zu bestimmen. Das Dialogfeld wird basierend auf den Informationen in der X-Funktionsdatei automatisch von Origin erstellt. Daher haben die Dialogfelder für alle X-Funktionen ein einheitliches Aussehen und bieten dem Anwender konsistente Einstellmöglichkeiten zum Festlegen verschiedener Parameter. Die folgende Abbildung zeigt die X-Funktion smooth (Kategorie = Signalverarbeitung) als gerendertes Dialogfeld.

Die Anzeige eines der X-Funktionsdialogfelder wird von einer Fahne im Definitionsbaum der X-Funktionen gesteuert. In Situationen, in denen es für X-Funktionen nicht erforderlich oder angemessen ist, ein Dialogfeld anzuzeigen, kann diese Fahne ausgeschaltet werden. Weitere Einzelheiten finden Sie in der Programmierhilfe.
