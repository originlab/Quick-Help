Python-Konsole

Python-Console-Page

Die Python-Konsole ist nützlich für das einfache Ausführen von Python-Skript. Eine Entwicklungsumgebung mit größerem Funktionsumfang finden Sie unter Python-Code ausführen > Code Builder.

Um die Python-Konsole zu öffnen, wählen Sie Konnektivität: Python-Konsole.

Um die Python-Umgebung in Origin zurückzusetzen, wählen Sie Fenster: Skriptfenster und führen Sie den folgenden Befehl aus:

run -pyr;

Seit Origin 2021b ist Intellisense (automatische Vervollständigung) per Standard aktiviert. Sie können Intellisense durch Festlegen der Systemvariable @NPYC = 0 deaktivieren. Informationen zum Ändern des Werts einer LabTalk-Systemvariablen finden Sie unter FAQ-708 Wie ändere ich permanent den Wert einer Systemvariablen?.
