Einführung

Die deskriptive Statistik wird zur Beschreibung der grundlegenden Merkmale von Datensätzen verwendet. Aspekte wie Mittelwerte, Standardabweichung, Quantile, Anzahl der Stichproben etc. werden berechnet.

Umgang mit fehlenden Werten

Die fehlenden Werte in dem Datenbereich werden aus der Analyse ausgeschlossen. Der Dialog Spaltenstatistik enthält die Option, die Anzahl der fehlenden Werte in jeder Spalte/Gruppe auszugeben.

Ab Origin 2015 werden fehlende Werte im Gruppierungsbereich und die entsprechenden Datenwerte aus der Analyse ausgeschlossen. In den älteren Versionen wurden die fehlenden Werte in dem Gruppierungsbereich als eine Gruppe betrachtet.

Spaltenstatistik durchführen

Um die deskriptive Statistik für einen Datenbereich zu berechnen:

  1. Wählen Sie Statistik: Deskriptive Statistik: Spaltenstatistik. Der Dialog Spaltenstatistik wird geöffnet.
  2. Legen Sie im Zweig Eingabedaten den Datenbereich fest, der analysiert werden soll. Im Zweig Zu berechnende Eigenschaften legen Sie deskriptive Statistiken fest.
  3. Optional können Sie unter Gruppierungsbereich eine Gruppierungsvariable eingeben, falls Sie zusammenfassende Statistiken für Gruppen erzeugen möchten. Wenn Sie in Gewichtungsbereich eine Gewichtung eingeben, werden die zusammenfassenden Statistiken durch den Faktor in der entsprechenden Zelle gewichtet. Auflistungen unter dem Zweig Steuerung Berechnung bieten Optionen zur Berechnung der Momente und der Quantile.
  4. Indem Sie auf OK klicken, wird ein Berichtsblatt erstellt, das alle gewählten Statistiken anzeigt.

 

Das gesperrte Ergebnisblatt, das von der Spaltenstatistik ausgegeben wird, kann sortiert werden, indem die Referenzsortierspalte markiert und über einen Rechtsklick die Option Arbeitsblatt sortieren gewählt wird.
