Benutzerdefinierte Symbolleisten

 

Die Konfigurationen von Symbolleisten und andockbaren Fenstern werden nach der Beendigung von Origin gespeichert. Sie können die Symbolleistenkonfigurationen auf verschiedene Weisen anpassen:

  • Anzeigen oder Verbergen von Symbolleisten
  • Hinzufügen oder Entfernen von Schaltflächen in der Symbolleiste
  • Programmieren Sie eine Schaltfläche in der Schaltflächengruppe Benutzerdefiniert, um einen benutzerdefinierten Aufgabe durchzuführen.
  • Erstellen Sie Ihre eigenen Bitmaps, verwenden Sie diese, um Symbolleistenschaltflächen zu erstellen, und programmieren Sie die Symbolleistenschaltflächen so, dass sie Ihre benutzerdefinierten Aufgaben durchführen.
  • Sie können benutzerdefinierte Schaltflächengruppen (Symbolleisten) bearbeiten.
