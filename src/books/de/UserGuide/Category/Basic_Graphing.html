Auf grundlegender Ebene werden Diagramme in Origin in zwei Schritten erstellt:

  1. Wählen Sie die gewünschten Daten aus. Origin verwendet die Diagrammzuordnung der Arbeitsblattspalte, um festzulegen, ob eine Spalte mit Werten gegen die X- und Y-Achsen gezeichnet werden soll oder ob es zum Erstellen von Fehlerbalken, Datenbeschriftungen usw. verwendet werden soll.
  2. Klicken Sie auf eine Schaltfläche. Eine Diagrammvorlage (und optional ein Design) wird aufgerufen, das den Diagrammtyp und die Anzeige einstellt (Farbe, Linien- und Symbolstil, Füllflächen usw.). Ihre Daten werden mit den in der Vorlage gespeicherten Einstellungen gezeichnet..

Um ein einfaches 2D-Diagramm mit den Diagrammzuordnungen der Arbeitsblattspalten zu erstellen:

  1. Klicken Sie doppelt auf die Kopfzeile jeder Datenspalte und setzen Sie im Dialog Spalteneigenschaften die Diagrammzuordnung auf X, Y, Z, X-Fehler, Y-Fehler, Beschriftung oder Ignorieren.
  2. Markieren Sie die Daten im Arbeitsblatt (oder einen Bereich aus einer oder mehreren Datenspalten) und führen Sie anschließend eine der folgenden Operationen aus:
  3. Wählen Sie Zeichnen > Registerkarte der Kategorie: Diagrammtyp (Linie, Punkt, Punkt-Linie usw.).
  4. Klicken Sie mit der rechten Maustaste auf die ausgewählten Arbeitsblattdaten und wählen Sie Zeichnen > Registerkarte der Kategorie: Diagrammtyp im Kontextmenü.
  5. Klicken Sie auf eine Schaltfläche auf eine der 2D-Diagramm-Symbolleisten (2D-Grafik oder Erweiterte 2D-Grafik).

Themen, die in diesem Abschnitt behandelt werden:

  • Diagrammseite vergrößern und verkleinern
  • Seite an Layer anpassen und Layer an Seite anpassen
  • Ansicht auf Diagrammseite und innerhalb des Layers schwenken
  • Mathematische Funktionen zeichnen
  • Stapelzeichnen
