Werte der Koordinaten im Diagramm lesen

Read-Coordinate-in-Graph

 

Origin bietet verschiedene Hilfsmittel zur grafischen Analyse von Datenzeichnungen.

  • Bildschirmkoordinaten: Die Bildschirmkoordinaten sind ein älteres Origin-Hilfsmittel, dass das Fenster Datenanzeige verwendet, um die XY-Koordinaten (und die Z-Koordinaten in Ternär- und Konturdiagrammen) auf der Diagrammseite oder im Diagrammlayer abzulesen. Es gibt die Koordinaten der gezeichneten Punkte nicht aus, sondern gibt einfach die Koordinaten in Einheiten der Achsenskalierung an.

Weitere Informationen finden Sie im Abschnitt zum Hilfsmittel Bildschirmkoordinaten.

  • Datenanzeige: Die Datenanzeige wird von verschiedenen Analysehilfsmitteln verwendet oder beim Durchführen bestimmter Analyseoperationen. Weitere Informationen finden Sie im Abschnitt Hilfsmittel Datenanzeige in der Origin-Hilfedatei.
  • Tooltipp der Datenpunkte: Seit Origin 2018 können Sie die Werte der Datenpunktkoordinaten in einem 2D-Diagramm (Punkt, Linie + Symbol, Balken etc.) ablesen, indem Sie die Maus über den Datenpunkt bewegen. Außerdem gibt es ein Hilfsmittel Punkte auswählen, das über einen Rechtsklick auf den Tooltipp geöffnet wird. Klicken Sie doppelt auf die Datenpunkte, die Sie interessieren und geben Sie die Koordinatenwerte und weitere Informationen in einer Arbeitsmappe aus.

    Seit Origin 2019 ist dieser Tooltipp der Datenpunkte, der durch Bewegen der Maus sichtbar wird, benutzerdefiniert anpassbar und bietet damit im Grunde die kombinierte Funktionalität der Fenster Datenkoordinaten + Dateninfo, ohne dass Sie ein Hilfsmittel auf der Symbolleiste Hilfsmittel auswählen und auf einen Datenpunkt klicken müssen. (Beachten Sie, dass dies nicht für alle Diagrammtypen funktioniert, für die die Fensterkombination Datenkoordinaten + Dateninfo verwendbar ist. In den folgenden Beschreibungen der Fenster Datenkoordinaten und Dateninfo finden Sie weitere Informationen.
Weitere Informationen finden Sie im Abschnitt zum Tooltipps der Datenpunkte im Origin-Handbuch.
  • Datenkoordinaten: Das Hilfsmittel Datenkoordinaten verwendet das weitreichend benutzerdefiniert anpassbare Fenster Dateninfo, um die X- und Y-Koordinaten (und für Konturdiagramme die Z-Koordinaten) der Datenpunkte anzugeben, plus im Grunde alle anderen Arbeitsblattinformationen, die mit einem bestimmten Datenpunkt verbunden sind (siehe Dateninfo). Weitere Informationen finden Sie im Abschnitt zum Hilfsmittel Datenkoordinaten des Origin-Handbuchs.

Für die Anwender, die die Datenkoordinaten mit den Pfeiltasten der Tastatur steuern, gibt es eine Anzahl von Systemvariablen, die kontrollieren, wie das Hilfsmittel sich entlang der Zeichnung bewegt. Weitere Informationen finden Sie unter LabTalk-Systemvariablen.

@DRC Inkrement mit der Strg-Taste für Datenkoordinaten-Pfeiltaste. Hinzugefügt in Version 2022b.
@DRD Steuert den Schwellenwert für die Anzahl der Datenpunkte, bei der die Beschleunigung der Datenkoordinaten mit der SHIFT-Taste beginnt, einen beachtenswerten Effekt zu haben. Hinzugefügt in Version 8.0987.
@DRI Steuert das Stufeninkrement der Datenkoordnaten in Prozent der vorherigen Stufe für die Beschleunigung mit der SHIFT-Taste. Hinzugefügt in Version 8.0987.
@DRM Steuert die maximale Stufengröße in Prozent der Größe der Datenzeichnung. Hinzugefügt in Version 8.0987.
@DRS Inkrement mit der Shift-Taste für Datenkoordinaten-Pfeiltaste. Hinzugefügt in Version 2022b.
@DRSC Inkrement mit Shift- + Strg-Taste für Datenkoordinaten-Pfeiltaste. Hinzugefügt in Version 2022b.
@DRT Steuert den Schwellenwert (in Millisekunden), wenn Datenkoordinaten beginnt, beim Halten der SHIFT- + Pfeil-Tasten zu beschleunigen. Hinzugefügt in Version 8.0987.
  • Dateninfo: Wie erwähnt, wird das Fenster der Dateninfo geöffnet, wenn Sie das Hilfsmittel Datenkoordinaten, Anmerkung oder Datencursor über die Symbolleiste Hilfsmittel auswählen. Das Fenster Dateninfo ist in hohem Maß anpassbar und kann so konfiguriert werden, dass verschiedene Typen von Metadaten zusätzlich zu den Koordinatenwerten gemeldet werden können. Weitere Informationen finden Sie im Abschnitt Hilfsmittel Dateninfo in der Origin-Hilfedatei.
