Eingabe

Bestimmen Sie den Eingabedatenbereich. Siehe Eingabedaten festlegen.

Umhüllungstyp

Legen Sie den zu suchenden Umhüllungstyp fest. Optionsliste:

  • Obere Hüllkurve: Es wird nur die obere Hüllkurve gesucht.
  • Untere Hüllkurve: Es wird nur die untere Hüllkurve gesucht.
  • Beide Hüllkurven: Es wird die obere und die untere Hüllkurve gesucht.

Glättungspunkte

Die Glättung wird während der Umhüllungserkennung durchgeführt. Diese Variable steuert den Glättungsgrad. Je größer der Wert ist, desto glatter wird die Umhüllungkurve. Einzelheiten lesen Sie bitte im Abschnitt zum Algorithmus Umhüllung nach.

Ausgabe

Legt den Ausgabebereich fest. Siehe Ergebnisse ausgeben.
