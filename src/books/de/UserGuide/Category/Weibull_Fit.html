Einführung

Es gibt drei Methoden, die Lebensdaueranalyse durchzuführen: eine nicht-parametrische Methode, eine halbparametrische Methode und eine parametrische Methode. Der Weibull-Fit ist eine Art der parametrischen Methode zum Analysieren des Verhältnisses zwischen Lebensdauerfunktion (Überlebenszeitfunktion) und Ausfallzeit.

Umgang mit fehlenden Werten

Wenn es fehlende Werte im Zeit-/Zensorbereich gibt, wird der gesamte Fall aus der Analyse ausgeschlossen.

Weibull-Fit durchführen

Um einen Weibull-Fit durchzuführen:

  1. Wählen Sie Statistik: Lebensdaueranalyse: Weibull-Fit. Dieser Menübefehl öffnet den Dialog weibullfit.
  2. Legen Sie die Eingabedaten, einschließlich Zeit- und Zensorvariablen, fest. Optional kann ein Weibull-Diagramm erstellt werden.
  3. Nachdem Sie auf OK geklickt haben, wird ein Analysebericht erstellt.
