Der Dialog Mehrfach-Skala DWT

MDWT-Dialog

  • Menübefehl: Analyse: Signalverarbeitung: Wavelet: Mehrfach-Skala DWT
  • Fenstertypen: Arbeitsmappe, Diagramm
  • Skript der X-Funktion und GUI-Beispiele: mdwt(Pro)
Dialogdesign Siehe Dialogdesigns.
Neu berechnen

Siehe Neu berechnen von Analyseergebnissen.
Eingabe

Legen Sie das Eingabesignal fest. Siehe Eingabedaten festlegen.
Wavelet-Typ

Legt den Typ des Wavelets fest.

  • Haar: Haar-Wavelet
  • DBN: Daubechies-Wavelet (N = Wavelet-Klasse und Wavelet-Anzahl der verschwindenden Momente)
  • BiorNr.Nd: Biorthogonal (Nr = Wavelet-Klasse oder -Skalierungsfunktionen, die zur Rekonstruktion verwenden werden; Nd = Ordnung der Funktionen, die zur Zerlegung verwendet wurden)
Erweiterungsmodus

Legt den Enderweiterungsmodus fest.

  • Periodisch: Das Eingabesignal wird als periodisch betrachtet.
  • Mit Nullen aufgefüllt: Datenpunkte außerhalb des Eingabebereichs werden als Null betrachtet.
Zerlegungsstufe

Legt die Anzahl der Stufen fest, für die die Eingabesignale zerlegt werden.
Mehrere Koeffizientendaten

Legt die Berichtsdaten für die Ergebnisse fest. Siehe Ergebnisse ausgeben.
