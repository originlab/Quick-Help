2D-FFT-Filter

Filterung ist ein Prozess, der Frequenzkomponenten aus einem Signal auswählt. OriginPro enthält 2D-FFT-Filter für 2D-Signale, einschließlich Matrizen und Bilder. Die Filter führen zunächst eine zweidimensionale schnelle Fourier-Transformation durch (2D-FFT), wenden dann ein Frequenzdomainfilter auf sie an und führen zuletzt eine 2D-IFFT durch, um sie zurück in die räumliche Domain zu konvertieren.

Es gibt fünf Filtertypen in der Funktion des 2D FFT-Filters: Tiefpass, Hochpass, Bandpass, Bandblock und Schwellenwert. Tiefpassfilter blocken die Frequenzkomponenten über der Grenzfrequenz, so dass nur Tieffrequenzkomponenten durchgelassen werden. Hochpassfilter sind genau das Gegenteil: Sie blocken die Frequenzkomponenten, die unter der Grenzfrequenz liegen. Bandpassfilter lassen nur Frequenzen innerhalb eines spezifischen Bereichs, der von der unteren und oberen Grenzfrequenz bestimmt wird, die Filter passieren, während Bandblockfilter alle Frequenzen innerhalb des gewählten Bereichs entfernen. Schwellenwertfilter lassen nur Frequenzkomponenten zu, deren Amplituden zwischen dem unteren Schwellenwert und dem oberen Schwellenwert liegen.

Es stehen vier Fenstertypen zur Auswahl: Butterworth, Ideal, Gaussian und Blackman. Das Butterworth-Fenster ist ein Filtertyp der Signalverarbeitung, der entwickelt wurde, um eine möglichst flache Frequenzantwort im Passband zu erhalten. Daher wird es auch als ein Filter mit maximal flachem Verlauf des Betragsfrequenzgangs bezeichnet. Das Ideal-Fenster lässt alle Frequenzen durch, die die Anforderungen erfüllen, und entfernt alle Frequenzen, die das nicht tun. Das Gaussian-Fenster ist ein Filter, dessen Impulsantwort eine 2D Gaussian-Funktion ist. Der Gaussian-Filter modifiziert das Eingabesignal durch Faltung mit einer Gaussian-Funktion. Das Blackman-Fenster ist ein Filter, dessen Impulsantwort eine 2D Blackman-Funktion ist.

Außerdem können Sie den Grenzwert durch Bruch, Fourier Pixel, Wellenlänge und Hertz festlegen.

Bei angezeigter Vorschau können Sie die Grenzwerte oder den Schwellenwert aus dem Vorschaufeld des Dialogfelds intuitiv auswählen. Setzen Sie zum Beispiel Filtertyp auf Hochpass, den Fenstertyp auf Butterworth, Ordnung auf 1, Grenze festlegen mit auf Fourier Pixel und aktivieren Sie das Kontrollkästchen Automatische Vorschau. Wenn der Wert im Bearbeitungsfeld Grenzfrequenz geändert wird, wird das durch 2D-FFT-Filter verarbeitete Bild automatisch in der Vorschau aktualisiert.

Beachten Sie, dass die 2D-FFT-Filter nicht verwendet werden können, um die Bilder direkt zu verarbeiten. Nach Erstellen einer neuen Matrix und Importieren des Bilds muss das Bild zuerst in Daten konvertiert werden. Origin bietet ein Hilfsmittel zum Konvertieren eines Bilds in Daten. Um auf dieses Hilfsmittel zuzugreifen, wählen Sie Bild: Konvertierung: In Daten konvertieren.

Um die 2D-FFT zu verwenden:

  1. Aktivieren Sie eine Matrixmappe.
  2. Wählen Sie Analyse: Signalverarbeitung: FFT-Filter im Origin-Menü.
