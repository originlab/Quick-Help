Glättung der Matrix

 

Für die Matrix kann eine Glättung durchgeführt werden.

Um die Glättung der Matrix zu verwenden:

  1. Aktivieren Sie eine Matrix.
  2. Wählen Sie Analyse: Signalverarbeitung: Glätten im Origin-Menü.

Die Origin-Projektdatei:

  • Das Dialogfeld MSmooth
  • Algorithmus (MSmooth)
