Neu berechnen

Siehe Neu berechnen von Analyseergebnissen.
Eingabematrix

Legt die komplexe Matrix oder die reale Matrix des Eingabesignals fest. Siehe Eingabedaten festlegen.
Imaginäre Matrix

Legt die imaginäre Matrix des Eingabesignals fest, wenn das Eingabesignal komplex ist, und wird in reale Matrix und imaginäre Matrix unterteilt. Siehe Eingabedaten festlegen.
Normieren der Amplitudenmatrix

Legt fest, ob die Amplituden auf 1 normiert werden oder nicht.
DÜ in die Mitte verschieben

Wenn diese Option aktiviert ist, wird die DÜ-Komponente in die Mitte der Ergebnismatrix geschoben. Ansonsten wird die DC-Komponente an vier Ecken der Ergebnismatrix positioniert.
Ausgabe

Legen Sie die Matrizen für die Ausgabe fest. Siehe Ergebnisse ausgeben.
