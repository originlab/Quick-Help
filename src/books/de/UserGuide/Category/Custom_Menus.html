Benutzerdefinierte Menüs


Wenn Sie einige Skripts besonders häufig verwenden, haben Sie die Möglichkeit, diese in einem Menü zu sammeln, auf das Sie schnell zugreifen können. Origin enthält ein Hilfsmittel namens Manager für benutzerdefinierte Menüs, mit dem Sie diese Skripts Menüelementen ganz einfach zuweisen können. Mit diesem Hilfsmittel können Sie bis zu 3 Hauptmenüs und ihre Untermenüs erstellen. Falls Sie möchten, können Sie auch standardmäßige Menüs verbergen. Dazu kommt, dass die Konfiguration der benutzerdefinierten Menüs als eine .omc-Datei gespeichert werden kann, die Sie ganz einfach mit Ihren Freunden und Kollegen teilen können.

Themen, die in diesem Abschnitt behandelt werden:

  • Das Dialogfeld Manager des benutzerdefinierten Menüs
  • Benutzerdefinierte Menüs erstellen
  • Verwalten mehrerer .omc-Dateien
