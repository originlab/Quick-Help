Alle Origin-Diagrammfenster -- auch leere, die keine gezeichneten Daten enthalten -- werden mit einer Diagrammvorlage erstellt. Jedem der über 100 Diagrammtypen ist eine Standardvorlage zugeordnet. Wenn Sie Daten aus einem Arbeitsblatt oder einer Matrix auswählen und auf eine der Schaltflächen für die Diagrammtypen auf Origins Grafiksymbolleisten klicken, rufen Sie diese Standardvorlagendatei (Dateinamenerweiterung = .OTP) zusammen mit einigen gespeicherten Diagrammeinstellungen auf. Diese Einstellungen steuern Elemente wie Linienfarbe, Symboltyp, Seitengröße, Anzahl und Position der Layer etc.

Der grundlegende Ablauf zum Erstellen einer Diagrammvorlage ist gleich, egal, ob eine neue Vorlage erstellt oder eine bereits bestehende modifiziert wird:

  1. Sie erstellen ein Diagramm mit Hilfe einer Diagrammvorlage.
  2. Sie passen das Diagramm an.
  3. Sie speichern Ihr Diagramm als Vorlage.
