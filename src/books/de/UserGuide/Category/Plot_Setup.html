Dialog Diagrammeinstellungen

GraphRef-PlotSetup-Dialog

 

Der Dialog Diagrammeinstellungen ist bei einer Vielzahl von Zeichenaufgaben nützlich, einschließlich dem Erstellen von Diagrammen, Bearbeiten des Diagrammtyps, Hinzufügen oder Entfernen von Zeichnungen aus dem Diagramm, Gruppieren oder Aufheben der Gruppierung von Diagrammen und Bearbeiten des Zeichnungsbereich.

Dieser Dialog besteht aus drei vertikalen Bedienfeldern (sehen Sie dazu den Screenshot unten): das obere, das mittlere und das untere Bedienfeld. Sie können die Schaltflächen und verwenden, um die Bedienfelder nach Bedarf zu zeigen bzw. zu verbergen.

 

Themen, die in diesem Abschnitt behandelt werden:

  • Dialogeinstellungen: Verfügbare Daten im oberen Bedienfeld
  • Diagrammeinstellungen: Diagrammzuordnungen im mittleren Bedienfeld
  • Diagrammeinstellungen: Diagrammliste im unteren Bedienfeld
