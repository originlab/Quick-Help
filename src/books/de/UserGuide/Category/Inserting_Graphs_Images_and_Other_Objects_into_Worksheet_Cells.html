Einfügen von Diagrammen, Bildern und anderen Objekten in Arbeitsblattzellen

InsObject-to-WksCell

 

Relevantes Video: Diagramme einfügen

In Origin-Arbeitsblättern können verschiedene Arten von Informationen abgelegt werden über den üblichen tabellarischen Text und die numerischen Daten hinaus. Zum Beispiel:

  • Arbeitsblätter können Analysergebnisse und Diagramme in einem präsentationsreifen Bericht kombinieren.
  • Arbeitsblätter können Skripte speichern.
  • Arbeitsblätter können Datenbankabfragen speichern.
  • Arbeitsblätter können programmierbare Schaltflächenobjekte enthalten.


Dieses Thema betrifft nur die Dinge, die in Arbeitsblattzellen eingefügt werden können, entweder in die Headerzeilen (alias "Spaltenbeschriftungszeilen") oder die Datenzeilen.

Die folgende tabelle listet die Dinge aus, die in Arbeitsblattzellen eingefügt werden können. Einige Elemente können entweder in Headerzeilen oder in Datenzeilen eingefügt werden (gekennzeichnet durch die Verfügbarkeit von Kontext- oder Menübefehl).

Die Möglichkeit zum Verbinden von Arbeitsblattzelleninhalten mit Variablenwerten, Werten in anderen Origin-Arbeitsmappen oder anderen Diagrammfenstern, kann beim Automatisieren von Routineaufgaben und Erstellen von Berichten nützlich sein.

Pfeil

Nützlich zum Kommentieren von Arbeitsblättern

  1. Klicken Sie mit der rechten Maustaste auf die Zelle und wählen Sie Pfeil einfügen > Typ.

Diagramm als bearbeitbares Diagramm†

Diese Option bettet ein bearbeitbares Diagramm in der Arbeitsblattzelle ein. Bearbeiten Sie das Diagramm durch einen Doppelklick auf die Arbeitsblattzelle. Wenn Sie fertig sind, klicken Sie auf die Schaltfläche Schließen in der oberen rechten Ecke des bearbeiteten Diagramms. Das Diagrammfenster wird geschlossen und Ihre Änderungen im eingebetteten Diagramm gespeichert.

  1. Klicken Sie mit der rechten Maustaste auf die Zelle und wählen Sie Diagramm einfügen.
  2. In dem aufgerufenen Dialog Diagramm einfügen (insertGraph) wählen Sie Ihr Diagramm über den Diagrammbrowser aus und klicken auf OK.
  3. Aktivieren Sie Diagramm in Zelle verschieben (Einbetten) (Zweig Optionen) und klicken Sie auf OK.

Diagramm als Bild†

Diese Option fügt ein Diagramm als Bildobjekt ein. Sie können das Bild mit dem Quelldiagramm verknüpfen, so dass jede Änderung am Quelldiagramm das Bild aktualisiert.

  1. Klicken Sie mit der rechten Maustaste auf das Diagramm und wählen Sie Diagramm als Bild kopieren/Seite kopieren oder klicken Sie auf Bearbeiten: Diagramm als Bild kopieren/Bearbeiten: Seite kopieren im Hauptmenü.
  2. Klicken Sie mit der rechten Maustaste in die Arbeitsblattzelle und wählen Sie Verknüpfung einfügen oder klicken Sie auf die Zelle. Klicken Sie auf Bearbeiten: Verknüpfung einfügen im Hauptmenü. Um ein statisches Bild (unverknüpft) einzufügen, wählen Sie einfach Einfügen.

Sie können mit der Maus über die Zelle fahren, um eine Vorschau anzuzeigen.

Hyperlinks

Sie können Hyperlinks in Arbeitsblattzellen einfügen. Weitere Informationen finden Sie unter Verknüpfungen in Arbeitsblattzellen einfügen.

Bild aus Datei†

Diese Option fügt ein Bild ein (.JPG, .BMP, .GIF etc.). Durch Doppelklick auf die Zelle wird das Bild im neuen Bildfenster geöffnet.

  1. Klicken Sie mit der rechten Maustaste auf die Zelle und wählen Sie Bilder aus Dateien einfügen.

Wenn das eingefügte Bild in einem Bildfenster geöffnet ist, kann es mit einer externen Quelldatei verbunden. Dies unterstützt beim Steuern der Projektgrößen, wenn viele (oder große) Bilder involviert sind. Informationen finden Sie unter den folgenden Links zum (1) Verknüpfen und (2) Wiederherstellen der extern verknüpften Bilder mit dem Arbeitsbereich der geöffneten Projektdateien.

Notizen

Um eine Notiz zu einer Zelle hinzuzufügen, wählen Sie eine Zelle aus und klicken Sie dann auf die Schaltfläche Notiz einfügen auf der Minisymbolleiste.

  • Ein Popup mit dem Notiztext wird angezeigt, wenn Sie die Maus über den roten Notizanker in der oberen rechten Ecke der Zelle bewegen.
  • Notizen können zu Zellen hinzugefügt werden, die andere Informationen enthalten, einschließlich eingebettete Bilder.
  • Um existierende Notizen zu verwalten, wählen Sie die Zelle aus und klicken Sie auf die Schaltflächen Notiz leeren oder Im Notizfenster öffnen auf der Minisymbolleiste.
  • Sie können eine Notiz bereits im Voraus mit Ihrem benutzerdefinierten Text ausfüllen:
  • Öffnen Sie \Anwenderdateien\Origin.ini, suchen Sie den Abschnitt [Text] und fügen Sie CellNote="Hier Ihren Text" hinzu.
  • Wenn Sie Origin.ini nicht bearbeitet haben, können Sie die Systemvariable @CNN verwenden, um die Standardelemente zu jeder Notiz hinzuzufügen
    (z. B. @CNN=2 ==> "user 1/7/2022"). Eine Anleitung, wie Sie den Wert einer Systemvariablen ändern, finden Sie in dieser FAQ.
0 = "hier eingeben"
1 = Benutzer
2 = Benutzer + Datum
3 = Benutzer + Datum + Zeit

Gleichung

Eine LaTeX-Gleichung kann in den Datenbereich und die Beschriftungszeilen eingefügt werden (z. B. Langname, Einheit, Kommentar).

  • Wählen Sie die gewünschte Arbeitsblattzelle und dann auf die Schaltfläche LaTeX einfügen auf der aufgerufenen Minisymbolleiste. Oder
    Klicken Sie doppelt in die gewünschte Zelle, um den direkten Bearbeitungsmodus zu öffnen, klicken Sie mit der rechten Maustaste und wählen Sie im Kontextmenü Einfügen: LaTeX-Gleichung.
    Der LaTeX-Gleichungseditor wird geöffnet.
  • LaTeX-Gleichung wird mit Hilfe von \q escape eingefügt. Um die Frage anzuzeigen, müssen Sie die Syntax Rich Text über den Dialog Spalteneigenschaften oder das Kontextmenü Datenstil festlegen: Rich Text aktivieren (Rich Text ist standardmäßig für die Beschriftungszeile Einheit eingeschaltet).

Sparklines†

Eine Sparkline stellt eine Miniaturzeichnung der Daten dar, die sich in einer Arbeitsblattspalte befinden.

  1. Klicken Sie mit der rechten Maustaste auf eine Zelle in der Headerzeile Sparklines und wählen Sie Sparklines hinzufügen oder aktualisieren. Eine Sparkline der Spaltendaten wird in die Headerzeilenzelle eingefügt. Weitere Informationen finden Sie unter der X-Funktion sparklines.
  1. Klicken Sie mit der rechten Maustaste in eine Datenzelle und wählen Sie Sparklines einfügen. Eine Sparkline eines Datensatzes aus dem Projekt wird in die Zelle der Datenzeile eingefügt. Weitere Informationen finden Sie unter der X-Funktion insertSparklines.

Variablen

Sie können Variablen, die im binären Speicherbereich des Arbeitsblatts gespeichert sind (z. B. als Ergebnis des Imports von Datendateien mit wichtigen Headerinformationen), in eine Arbeitsblattzelle einfügen, entweder als statische Werte oder als ein verknüpfter Wert. Wenn er als verknüpfter Wert eingefügt wurde, wird der Zellenwert aktualisiert, sobald sich der Wert der verknüpften Variable ändert (wie zum Beispiel beim Importieren einer neuen Datendatei).

Weitere Informationen finden Sie unter: Der Dialog Variablen einfügen für Arbeitsblätter.

†Hinweis: Ab Origin 2018 wird jedes eingefügte Diagramm eine Popup-Vorschau erzeugen, wenn Sie mit dem Cursor darüber fahren, sei es ein bearbeitbares Diagramm, ein Diagramm, das als verknüpftes oder statisches Bild eingefügt wurde, ein aus einer Datei eingefügtes Bild oder eine Sparkline.

 

Um die Bilddatei in die Arbeitsblattszelle einzufügen, können Sie sie kopieren und direkt einfügen.

  • Eine Bilddatei (.JPG, .BMP, .GIF, etc.) aus Origin kopieren
  • Klicken Sie auf eine Arbeitsblattzelle.
  • Drücken Sie die Tasten Strg + V oder klicken Sie mit der rechten Maustaste auf die Zelle, um Einfügen auszuwählen.
  • Durch Doppelklick auf die Zelle wird das Bild im neuen Bildfenster geöffnet.
