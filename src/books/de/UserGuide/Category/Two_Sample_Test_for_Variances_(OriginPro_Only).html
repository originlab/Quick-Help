Einführung

Diese Funktion führt den F-Test durch, um zu testen, ob die Varianz aus einer Grundgesamtheit gleich, größer oder kleiner als eine andere ist, und erstellt dann ein Konfidenzintervall für das Verhältnis der Stichprobenvarianzen.

Die Statistik des F-Tests ist eine Verallgemeinerung der Statistik des t-Tests und stellt eine skalare Zufallsvariable dar. Die Statistik des F-Tests kann als Verhältnis der Varianzen zweier Stichproben berechnet und daher verwendet werden, um zu testen, ob Datenstichproben aus Grundgesamtheiten stammen, deren Varianzen gleich sind.

Die Statistik des F-Tests und der p-Wert werden berechnet, so dass Sie entscheiden können, ob die Nullhypothese zurückgewiesen wird. Ein kleiner p-Wert, der kleiner als ein Signifikanzniveau ist, weist darauf hin, dass Sie die Nullhypothese zurückweisen können.

Umgang mit fehlenden Werten

Die fehlenden Werte in dem Datenbereich werden aus der Analyse ausgeschlossen.

Ab Origin 2015 werden fehlende Werte im Gruppierungsbereich und die entsprechenden Datenwerte aus der Analyse ausgeschlossen. In den älteren Versionen wurden die fehlenden Werte in dem Gruppierungsbereich als eine Gruppe betrachtet.

Test auf Varianzen bei zwei Stichproben durchführen

Um einen Test auf Varianzen bei zwei Stichproben durchzuführen:

  1. Wählen Sie Statistik: Hypothesentest: Test auf Varianzen bei zwei Stichproben. Der Dialog TwoSampleTestVar wird geöffnet.
  2. Legen Sie die Form der Eingabedaten (Indiziert oder Roh) fest und geben Sie dann die Eingabedaten, den F-Test für das Varianzverhältnis und die Alternativhypothese an.
  3. Wenn Sie auf OK klicken, wird ein Analysebericht erstellt, der die Freiheitsgrade, die F-Statistiken, den zugehörigen p-Wert und die Testschlussfolgerung auflistet. Zusätzlich können Sie die Konfidenzintervalle für das Verhältnis der Stichprobenvarianzen, ein Histogramm und ein Boxdiagramm sowie eine Analyse der Trennschärfe erstellen.
