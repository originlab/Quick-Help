Einführung

Das Hilfsmittel 2D-Klasseneinteilung teilt Daten ein, die zwei Variablen (zwei Dimensionen) haben. Wenn diese Funktion auf Daten angewendet wird, die in zwei Spalten gespeichert sind, wird eine Matrix mit der jeweiligen Anzahl in zwei Dimensionen erstellt. Gegebenenfalls wird ein 3D-Balkendiagramm und/oder ein Bilddiagramm des Ergebnisses erzeugt, um bei der Visualisierung der der Datenverteilung zu helfen.

Umgang mit fehlenden Werten

Die fehlenden Werte in den X/Y-Daten werden aus der Analyse ausgeschlossen.

2D-Häufigkeitszählung/Klasseneinteilung durchführen

Zum Durchführen der 2D-Klasseneinteilung:

  1. Wählen Sie eine Y-Spalte oder einen Bereich der Y-Daten in Ihrem Arbeitsblatt aus (die andere Spalte muss als X-Spalte zugewiesen werden und sollte, falls nicht ausgewählt, die erste X-Spalte links von der Y-Spalte sein).
  2. Wählen Sie Statistik: Deskriptive Statistik: 2D Häufigkeitszählung/Klasseneinteilung. Dadurch öffnet sich der Dialog TwoDBinning.

Wenn Sie dies tun, Origin erstellt eine Satz von Klasseneinteilungen in der XY-Ebene, wobei das Programm bei dem Wert für das Minimum der X- und Y-Werte nach Inkrementgröße beginnt. Origin erfasst die Anzahl der XY-Paare, die in jede einzelne Klasseneinteilung fallen, und erstellt eine Matrix mit Ausgabedaten. Die Koordinaten der Ausgabematrix bestehen aus den Extremwerten. Jede Zelle zeigt die Anzahl der "Treffer" an, die für jede Einteilung erfasst wurde.
