Zusammenfassung

In diesem Tutorial erfahren Sie die grundlegenden Schritte, um eine Shapefile zu importieren und eine Farbabbildung auf die Grenzen innerhalb der Datei mit Drittanbieterdaten anzuwenden.

Was Sie lernen werden

Sie lernen, wie Sie die Apps Import Shapefile und Colormap for Map Data verwenden, um ein kartenbasiertes Diagramm aus Shapefile-Daten zu erstellen.

Schritte

Bevor Sie beginnen, tun Sie bitte Folgendes:

  1. Installieren Sie die App Import Shapefile über das App Center.
  2. Installieren Sie die App Colormap for Map Data über das App Center.
  3. Laden Sie die zu importierende Shapefil herunter (extrahieren Sie den Ordner mit den relevanten Dateien an einem Ort Ihrer Wahl wie dem Desktop).

Schritt 1 - Daten importieren

Klicken Sie in der Apps-Galerie auf der rechten Seite des Origin-Arbeitsbereichs auf das Symbol der App Import Shapefile. Wählen Sie dann die *.SHP-Datei aus, die Sie gerade heruntergeladen haben. Stellen Sie sicher, dass Output Data Points aktiviert ist (siehe unten) und klicken Sie auf OK.

Sie haben jetzt eine Arbeitsmappe mit Daten und einem Diagramm (siehe unten). Für dieses Tutorial können Sie das Diagramm löschen. Später wird ein anderes erstellt.

Schritt 2 - Daten nützlich machen

Klicken Sie nun bei aktivem Arbeitsblatt mit den Shapefile-Daten auf das Symbol der App Colormap for Map Data in der Apps-Galerie. Eine kleine Symbolleiste mit zwei Schaltflächen wird angezeigt. Klicken Sie auf die erste Schaltfläche Reorder Shapefile Data. Klicken Sie im Dialog auf die Schaltfläche > neben Names for ID und wählen Sie J(Y): NAME (siehe unten). Klicken Sie auf OK.

Ein neues Ausgabeblatt mit dem Namen Shapefile Data Reordered wird erstellt und aktiviert. Achten Sie auf die Spalte Value, die aktuell leer ist (siehe unten).

Fügen Sie die folgenden Daten zur Armutsrate 1 in diese Spalte.

16,8 10,9 14 17,2 12,8 9,6 10,4 12,5 16,2 13,6 14,3 8,8 11,8 12,1 13,1 11,2 12 16,9 18,6 11,6 9 10 14,1 9,6 19,7 13,2 13 11 12,9 7,6 9,5 19,5 13,6 14 10,7 13,9 15,6 12,6 12,2 43,1 12,9 15,3 13,1 15,3 14,9 9 11 10,7 10,3 17,8 11 11,1

1 Quelle: List of U.S. states and territories by poverty rate

Klicken Sie in der Symbolleiste Colormap for Map Data auf die zweite Schaltfläche Create Colormap Map Plot. Sie sehen ein neues Diagramm. Ändern Sie die Werte der Horizontalen Achsenskalierungen Von und Bis in -175 bzw. -60. Ändern Sie dann die Werte der Vertikalen Achsenskalierungen Von und Bis in 17 bzw. 72 (siehe unten).

Das Diagramm ist jetzt in einem Zustand, in dem es benutzerdefiniert angepasst werden kann (siehe unten).

Diagramm benutzerdefiniert anpassen

Unter den vielen Anpassungen, die am Ende zu dem oben in diesem Tutorial abgebildeten Diagramm führen, sind hier einige wichtige, die Sie selbst durchführen können:

  1. Verbergen Sie die Achsenlinien, Hilfsstriche, Beschriftungen und Titel.
  2. Wählen Sie im Menü Grafik die Option Seite an Layer anpassen, um die Diagrammgröße anzupassen.
  3. Passen Sie die Ebenen der Farbabbildung für das Diagramm an.
  4. Passen Sie die Einstellungen der Farbskala an.
  5. Passen Sie einzelne Beschriftungspositionen an.

Zuletzt beachten Sie, dass Alaska, Hawaii und Puerto Rico sich verschoben haben. Dieser Schritt beinhaltet grundlegend das Erstellen von Kopien des Hauptdiagrammlayers und ihr erneutes Einfügen in den Hauptlayer, während die Werte für Von und Bis der Achsenskalierung für jeden eingefügten Layer (Zustand) angepasst wird. Die Größe der Layer wird dann verändert und sie werden neu positioniert.
