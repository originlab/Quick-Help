Wichtige Hinweise!

  • Die Studentenlizenz erlaubt die Installation von OriginPro auf EINEM persönlichen Computer.
  • Wenn Sie eine 1-Jahres-Studentenlizenz im Online-Shop von OriginLab/ADDITIVE kaufen, können Sie immer ein Softwareupgrade durchführen, wenn ein neues Produkt vor Ablauf Ihrer 1-Jahres-Lizenz veröffentlicht wird. Studentenlizenzen über eine Laufzeit von sechs Monaten, die über OnTheHub gekauft wurden, erhalten KEIN Upgrade.
  • Wenn Sie ein Upgrade für Ihre Studentenversion durchführen möchten, müssen Sie die neue Version auf dem Computer mit der alten installierten Version installieren und lizenzieren. Wenn Sie Ihre Lizenz vor dem Upgrade auf einen neuen Computer übertragen möchten, müssen Sie die Lizenz auf dem alten Computer deaktivieren, bevor Sie das Upgrade auf dem neuen Computer installieren (siehe Lizenzdeaktivierung unten).

Produktschlüssel erhalten

Wenn Sie das Produkt von OriginLab oder einen seiner Vertriebspartner, wie ADDITIVE, gekauft haben

  1. Nach dem Kauf erhalten Sie eine E-Mail mit dem Betreff <email title>, die Ihre Origin-Seriennummer und einen Link für die Registrierung enthält,
    oder
    ein gedrucktes Lizenzzertifikat mit Ihrer Origin-Seriennummer und einem Registrierungscode
    oder
    einen Registrierungscode von Ihrem Origin-Vertriebshändler.
    • Wenn Sie die E-Mail erhalten:
      1. Klicken Sie auf den Link zur Registrierung. Sie gelangen zur Origin-Webseite. Sie werden dann aufgefordert, sich anzumelden oder sich zu registrieren. Melden Sie sich an, wenn Sie bereits über ein Konto verfügen, oder erstellen Sie sich ggf. ein Konto.
      2. Nach der Anmeldung auf der Webseite werden der Code Ihrer Produktregistrierung und Ihre Seriennummer angezeigt. Klicken Sie auf die Schaltfläche Registrieren, um diese Seriennummer zu registrieren und Ihren Produktschlüssel zu erzeugen.
      3. Auf der aufgerufenen Seite zum Anforderungsverlauf der Registrierung|Lizenzierung Ihres Origin-Produkts wird ein Produktschlüssel erzeugt.
    • Wenn Sie das gedruckte Lizenzzertifikat oder den Registrierungscode von Ihrem Vertriebshändler erhalten,
      1. Rufen Sie die OriginLab-Webseite (www.OriginLab.com) manuell auf. Melden Sie sich an Ihrem Konto an oder erstellen Sie sich ggf. eines.
      2. Klicken Sie auf der Willkommensseite auf den Link "Mein Produkt registrieren - Version 2017 (9.4)".
      3. Geben Sie auf der Seite Mein Produkt registrieren, wie angegeben, den Registrierungscode ein und klicken Sie auf die Schaltfläche Weiter, um diese Seriennummer zu registrieren und Ihre Produktschlüssel zu erzeugen.
      4. Auf der aufgerufenen Seite zum Anforderungsverlauf der Registrierung|Lizenzierung Ihres Origin-Produkts wird ein Produktschlüssel erzeugt.

Wenn Sie das Produkt von OntheHub gekauft haben (nur USA und Kanada)

Lizenzaktivierung

  1. Sie erhalten Ihre Seriennummer und den zugehörigen Produktschlüssel vom Origin-Administrator.
  2. Starten Sie Origin. Der Dialog Lizenzaktivierung wird geöffnet.
  3. Wählen Sie die Option Produktschlüssel unten eingeben und auf Aktivieren klicken und geben Sie Ihren Produktschlüssel ein. Klicken Sie auf Aktivieren.
  4. Der Lizenzprozess wird nun im Hintergrund durchgeführt. Geschieht dies erfolgreich, erhalten Sie eine entsprechende Nachricht.

Sollte der Lizenzierungsprozess im Hintergrund fehlschlagen, fahren Sie bitte mit den folgenden Schritten fort:

  1. Wenn der Lizenzierungsprozess aus irgendeinem Grund fehlschlägt (wenn es zum Beispiel keine Internetverbindung gibt oder wenn originlab.com von einer Firewall geblockt wird etc.), dann wird ein Notepad-Fenster mit einem Link darin geöffnet.
  2. Kopieren Sie diesen Link in einen Webbrowser, der eine Internetverbindung hat. Der Link leitet Sie zur OriginLab-Webseite.
  3. Der Lizenztext wird erzeugt und auf der Webseite angezeigt.
  4. Kopieren Sie den Lizenztext und fügen Sie ihn in Origins Dialog Lizenzaktivierung ein. Klicken Sie auf OK.

Lizenzdeaktivierung

Wenn Sie planen, Ihre lizenzierte Origin-Software auf einen anderen Computer zu übertragen, müssen Sie sie auf dem ursprünglichen Computer deaktivieren und dann eine neue Lizenz für den neuen Computer anfordern.

Falls Origin weiterhin auf Ihrem alten Computer ausgeführt werden kann

  1. Starten Sie Origin. Wählen Sie Hilfe: Lizenz deaktivieren, um den Dialog Lizenzdeaktivierung zu öffnen.
  2. Klicken Sie auf die Schaltfläche Jetzt deaktivieren.
  3. Der Lizenzdeaktivierungsprozess wird nun im Hintergrund durchgeführt. Geschieht dies erfolgreich, erhalten Sie eine entsprechende Nachricht.

Sollte der Deaktivierungsprozess im Hintergrund fehlschlagen, fahren Sie bitte mit den folgenden Schritten fort:

  1. Wenn der Deaktivierungsprozess im Hintergrund aus irgendeinem Grund fehlschlägt (wenn es zum Beispiel keine Internetverbindung gibt oder wenn originlab.com von einer Firewall geblockt wird etc.), dann wird ein Notepad-Fenster mit einem Link darin geöffnet.
  2. Kopieren Sie diesen Link in einen Webbrowser, der eine Internetverbindung hat.
  3. Die Webseite zeigt Ihre Lizenzinformationen an. Notieren Sie sich den Deaktivierungscode in Schritt 1.
  4. Kehren Sie zu Origin zurück. Geben Sie den Deaktivierungscode im Bearbeitungsfeld des Deaktivierungscodes unten im Dialog Lizenzdeaktivierung ein. Klicken Sie auf die Schaltfläche Lizenz deaktivieren.
  5. Es wird ein Notepad-Fenster mit einem Link darin geöffnet. Kopieren Sie diesen Link in einen Webbrowser, der eine Internetverbindung hat.
  6. Klicken Sie auf die Schaltfläche Weiter auf der Webseite. Sie bekommen dann eine Erfolgsmeldung, dass eine Lizenz für die Wiederverwendung freigegeben wurde.

Upgrade für Ihr Origin auf eine neuere Version durchführen

Sie sind berechtigt auf ein Softwareupgrade, wenn vor dem Ablauf Ihrer Studentenlizenz ein neues Produkt veröffentlicht wird. Führen Sie die folgenden Schritte durch, um ein Upgrade für Ihr Origin durchzuführen und es zu aktivieren.

Wenn Sie ein Upgrade für Ihre Studentenversion durchführen möchten, müssen Sie die neue Version auf dem Computer mit der alten installierten Version installieren und lizenzieren.

  1. Laden Sie die neue Version von der OriginLab-Webseite herunter. Installieren Sie sie.
  2. Starten Sie Origin. Der Dialog Lizenzaktivierung wird geöffnet.
  3. Der Produktschlüssel ist bereits eingetragen. Klicken Sie auf die Schaltfläche Aktivieren.
  4. Der Lizenzprozess wird nun im Hintergrund durchgeführt. Geschieht dies erfolgreich, erhalten Sie eine entsprechende Nachricht. Schlägt die Aktivierung fehl, wird ein Notepad-Fenster ausgerufen, das weitere Anweisungen enthält. Befolgen Sie die Anweisungen, um Origin manuell zu aktivieren.

Wenn Sie Ihre Lizenz vor dem Upgrade auf einen neuen Computer übertragen möchten, müssen Sie die Lizenz auf dem alten Computer deaktivieren, bevor Sie das Upgrade auf dem neuen Computer installieren.

Ausführliche Anweisungen zur Deaktivierung finden Sie im Abschnitt Lizenzdeaktivierung.
