Rechnergebundene Einzellizenz (Node-Locked)

Single-User-Node-Locked-Lic

Sie haben eine rechnergebundene Einzellizenz (Node-Locked), wenn Ihre Seriennummer folgendermaßen aussieht:

xFxxx-9xxx-70/71/72/73xxxxx oder
xFxxx-6xxx-70/71/72/73xxxxx.

Sollte Ihre Seriennummer folgendermaßen aussehen:

xxxxx-2xxx-79xxxxx oder
xxxxx-3xxx-79xxxxx,

dann haben Sie eine Concurrent-Netzwerklizenz und die Informationen auf dieser Seite sind NICHT relevant für Sie.

Ein rechnergebundenes Einzellizenzpaket (Node-Locked) berechtigt Sie dazu, Origin auf einem Computer zu installieren. Sie müssen für diesen Computer einen Primären (Primary) Produktschlüssel von der OriginLab-Webseite anfordern. Die Lizenz ist rechnergebunden. Bestimmte Lizenzpakete können einen (1) Home-use-Produktschlüssel für Ihr Notebook oder Ihren Heimcomputer erhalten. Diese Home-Use-Lizenz ist Teil des Softwarepflegevertrages. Sie bleibt für den Zeitraum der aktiven bzw. verlängerten Softwarepflege aktiv.

Wenn Sie den Computer mit der Origin-Lizenz austauschen möchten, müssen Sie die Lizenz deaktivieren, bevor Sie eine neue Lizenz für den neuen Computer bekommen.

HINWEIS: Diese Anweisungen gelten für Origin 2017 und höher. Wenn Sie Version 2016 oder älter verwenden, lesen Sie bitte diese Seite.

Lizenzaktivierung

Sehen Sie sich das Video-Tutorial an - Rechnergebundenes Origin (Node-Locked) lizenzieren

Wichtige Hinweise!

  • Ab Origin 2017 verfügt Origin über ein neues Lizenzsystem. Sie benötigen keine Administratorrechte, um den Lizenzprozess für Node-Locked-Lizenzen durchzuführen.

Falls Ihre Origin älter als Version 2017 ist, lesen Sie bitte diese Seite.

  • Sobald der Lizenzprozess abgeschlossen ist, wird die Lizenz auf dem lokalen Computer gespeichert. Ein Internetzugang ist danach nicht mehr erforderlich.

Produktschlüssel erhalten

Wenn Sie das Produkt neu erwerben ...
oder
Wenn Sie ein Upgrade für Ihr Origin (2016 oder ältere Version) auf eine neuere Version (2017 oder höher) während der Softwarepflege durchführen ...

  1. Sie erhalten eine E-Mail, die Ihre Origin-Seriennummer und einen Link für die Registrierung enthält,
    oder
    ein gedrucktes Lizenzzertifikat mit Ihrer Origin-Seriennummer und einem Registrierungscode
    oder
    einen Registrierungscode von Ihrem Origin-Vertriebshändler.
    • Wenn Sie die E-Mail erhalten:
      1. Klicken Sie auf den Link zur Registrierung. Sie gelangen zur Origin-Webseite. Sie werden dann aufgefordert, sich anzumelden oder sich zu registrieren. Melden Sie sich an, wenn Sie bereits über ein Konto verfügen, oder erstellen Sie sich ggf. ein Konto.
      2. Nach der Anmeldung auf der Webseite werden der Code Ihrer Produktregistrierung und Ihre Seriennummer angezeigt. Klicken Sie auf die Schaltfläche Registrieren, um diese Seriennummer zu registrieren und Ihre Produktschlüssel zu erzeugen.
      3. Auf der nun aufgerufenen Seite zum Origin-Produktregistrierung | Verlauf der Lizenzabrufe werden zwei (2) Produktschlüssel erzeugt, einer (1) für den Bürocomputer und einer (1) für den Home-Use-Computer.
    • Wenn Sie das gedruckte Lizenzzertifikat oder den Registrierungscode von Ihrem Vertriebshändler erhalten,
      1. Öffnen Sie die OriginLab-Seite zur Produktregistrierung https://www.originlab.com/reg manuell. Melden Sie sich an Ihrem Konto an oder erstellen Sie sich ggf. eines.
      2. Geben Sie auf der Seite Mein Produkt registrieren, wie angegeben, den Registrierungscode ein und klicken Sie auf die Schaltfläche Weiter, um diese Seriennummer zu registrieren und Ihre Produktschlüssel zu erzeugen.
      3. Auf der nun aufgerufenen Seite Origin-Produktregistrierung | Verlauf der Lizenzabrufe werden zwei Produktschlüssel erzeugt, einer für den Bürocomputer (primary) und einer für den Home-Use-Computer (home).

Sie können die Produktschlüssel jederzeit überprüfen, indem Sie auf Meine Lizenz(en) verwalten unter Ihrem Portal klicken und den Link der Seriennummer auswählen. Sie können dann die Produktschlüsselinformationen auf der aufgerufenen Seite zum Anforderungsverlauf der Registrierung|Lizenzierung Ihres Origin-Produkts einsehen.

Wenn Sie ein Upgrade für Ihr Origin (2017 oder höher) auf eine neuere Version während der Softwarepflege durchführen

Seit Origin 2017 verwenden verschiedene Versionen den gleichen Produktschlüssel. Daher besteht keine Notwendigkeit, einen neuen Produktschlüssel für die neuere Version zu suchen. Siehe hier-> Im Abschnitt "Falls ja..." finden Sie Einzelheiten zur Durchführung des Upgrades.

Origin mit Produktschlüssel aktivieren

  1. Starten Sie Origin. Der Dialog Lizenzaktivierung wird geöffnet.
  2. Wählen Sie die Option Produktschlüssel unten eingeben und auf Aktivieren klicken und geben Sie Ihren Produktschlüssel ein. Klicken Sie auf Aktivieren.
  3. Der Lizenzprozess wird nun im Hintergrund durchgeführt. Geschieht dies erfolgreich, erhalten Sie eine entsprechende Nachricht.

Sollte der Deaktivierungsprozess im Hintergrund fehlschlagen, fahren Sie bitte mit den folgenden Schritten fort:

  1. Wenn der Lizenzierungsprozess aus irgendeinem Grund fehlschlägt (wenn es zum Beispiel keine Internetverbindung gibt), wird ein Notepad-Fenster mit einem Link darin geöffnet. Fahren Sie mit dem nächsten Schritt fort.
    Falls Ihnen KEIN Notepad-Fenster angezeigt wird, kann dies ein Hinweis auf ein Problem mit der Firewall sein. In diesem Fall sollte der Benutzer die Verbindung zum Internet trennen (via das Betriebssystem oder physisch durch Trennen des Netzwerkkabels von Ihrem Computer). Wenn keine Internetverbindung erkannt wird, wird das Öffnen des Notepad-Fensters erzwungen und Sie können mit dem nächsten Schritt fortfahren.
  2. Kopieren Sie diesen Link in einen Webbrowser, der eine Internetverbindung hat. Der Link leitet Sie zur OriginLab-Webseite.
  3. Wenn Sie sich an Ihrem Konto anmelden, wird der Lizenztext erzeugt und auf der Webseite angezeigt.
  4. Kopieren Sie den Lizenztext und fügen Sie ihn in Origins Dialog Lizenzaktivierung ein. Klicken Sie auf OK. Sollte dieser Schritt fehlschlagen, führen Sie die Software bitte als Administrator aus (klicken Sie mit der rechten Maustaste auf das Origin-Programmsymbol und wählen Sie Als Administrator ausführen) und versuchen Sie es erneut.

Lizenzdeaktivierung

Sehen Sie sich das Video-Tutorial an - Eine rechnergebundene Lizenz (Node-Locked) deaktivieren


Wenn Sie Version 2016 oder älter verwenden, lesen Sie bitte diese Seite.

Nach der erfolgreichen Lizenzierung wird der Produktschlüssel im Portal als “verwendet” markiert und kann nicht für einen anderen Computer verwendet werden.

Wenn Sie planen, Ihre lizenzierte Origin-Software auf einen anderen Computer zu übertragen, müssen Sie sie auf dem ursprünglichen Computer deaktivieren und dann eine Lizenz für den neuen Computer mit dem gleichen Produktschlüssel aktivieren.

Falls Origin weiterhin auf Ihrem alten Computer ausgeführt werden kann

  1. Starten Sie Origin. Wählen Sie Hilfe: Lizenz deaktivieren, um den Dialog Lizenzdeaktivierung zu öffnen.
  2. Klicken Sie auf die Schaltfläche Jetzt deaktivieren.
  3. Der Lizenzdeaktivierungsprozess wird nun im Hintergrund durchgeführt. Geschieht dies erfolgreich, erhalten Sie eine entsprechende Nachricht.

Sollte der Deaktivierungsprozess im Hintergrund fehlschlagen, fahren Sie bitte mit den folgenden Schritten fort:

  1. Wenn der Lizenzierungsprozess aus irgendeinem Grund fehlschlägt (wenn es zum Beispiel keine Internetverbindung gibt oder wenn originlab.com von einer Firewall geblockt wird etc.), dann wird ein Notepad-Fenster mit einem Link darin geöffnet.
  2. Kopieren Sie diesen Link in einen Webbrowser, der eine Internetverbindung hat. Der Link leitet Sie zur Webseite von OriginLab, auf der die Lizenzinformationen angezeigt werden.
  3. Notieren Sie sich den Deaktivierungscode.
  4. Kehren Sie zu Origin zurück. Geben Sie den Deaktivierungscode im Bearbeitungsfeld des Deaktivierungscodes unten im Dialog Lizenzdeaktivierung ein. Klicken Sie auf die Schaltfläche Lizenz deaktivieren.
  5. Es wird ein Notepad-Fenster mit einem Link darin geöffnet. Kopieren Sie diesen Link in einen Webbrowser, der eine Internetverbindung hat.
  6. Klicken Sie auf die Schaltfläche Weiter auf der Webseite. Wenn Sie eine Erfolgsmeldung bekommen, ist eine Lizenz für die Wiederverwendung freigegeben.

Falls Ihr lizenzierter Origin-Computer nicht länger verfügbar ist

  1. Gehen Sie zur Webseite von OriginLab (www.OriginLab.com) und melden Sie sich an.
  2. Klicken Sie auf der Willkommensseite auf den Link "Meine Lizenz(en)".
  3. Klicken Sie auf der Seite Meine Lizenz(en) verwalten auf die Seriennummer, die Sie deaktivieren, und dann auf den Link "Go" (Los) in der Spalte Deaktivierung.
  4. Überprüfen Sie auf der Seite zum Deaktivieren von Origin auf einem Gerät die Informationen und klicken Sie auf Weiter. Falls die Deaktivierung erfolgreich ist, erhalten Sie eine Benachrichtigung und der Produktschlüssel kann erneut verwendet werden.
  5. Sollte die Deaktivierung fehlschlagen, leitet die Seite Sie automatisch an das Anfrageformular für Systemübertragung weiter. Füllen Sie das Formular aus und senden Sie ein.

Lizenz nach Verlängerung der Softwarepflege aktualisieren

Seit Origin 2017 haben verschiedene Versionen die gleiche Lizenz. Das bedeutet, dass die neuere Version alle Lizenzinformationen der alten Version, einschließlich der Produktschlüssel und dem Nutzungsstatus, übernimmt. Auf diese Weise wird das Upgrade einfacher.

Nachdem Ihre Softwarepflege verlängert wurde, befolgen Sie die folgenden Schritte, um eine aktualisierte Lizenz zu erhalten.

  1. Gehen Sie zur OriginLab-Webseite (www.OriginLab.com). Melden Sie sich an.
  2. Klicken Sie auf der Willkommensseite auf den Link "Meine Lizenz(en)".
  3. Überprüfen Sie auf der Seite Meine Lizenz(en) die Spalte Version in der Tabelle der Seriennummern auf die Verfügbarkeit einer neueren Version für ein Upgrade.
Falls ja, befolgen Sie die untenstehenden Anweisungen, um ein Upgrade auf eine neuere Version durchzuführen.
  • Wenn Sie die neue Version auf dem Computer installieren möchten, auf dem die aktuelle Origin-Version lizenziert ist, führen Sie bitte die Schritte zum Installieren und Aktivieren der neuen Version durch.
  1. Laden Sie die neue Version von der OriginLab-Webseite herunter. Installieren Sie sie.
  2. Starten Sie Origin. Der Lizenzprozess wird dann automatisch im Hintergrund durchgeführt. Geschieht dies erfolgreich, erhalten Sie eine entsprechende Nachricht. Schlägt die Aktivierung fehl, wird ein Notepad-Fenster ausgerufen, das weitere Anweisungen enthält. Befolgen Sie die Anweisungen, um Origin manuell zu aktivieren.
  • Wenn Sie die neue Version auf einem neuen Computer installieren möchten, führen Sie bitte die Schritte zum Installieren und Aktivieren der neuen Version durch.
  • Wenn Sie nicht alle Produktschlüssel der aktuellen Version verwendet haben, können Sie die neue Version auf Ihrem neuen Computer installieren und den verfügbaren Schlüssel verwenden, um sie zu aktivieren. Ausführliche Anweisungen für die Aktivierung finden Sie im Abschnitt Origin mit Produktschlüssel aktivieren.
  • Wenn Sie alle Produktschlüssel der aktuellen Version aufgebraucht haben, müssen Sie mindestens eine Lizenz deaktivieren, die nicht mehr verwendet wird, bevor Sie den Produktschlüssel neu verwenden können, um die neue Version von Origin auf dem neuen Computer zu aktivieren. Ausführliche Anweisungen zur Deaktivierung finden Sie im Abschnitt Lizenzdeaktivierung.
Wenn keine neuere Version zur Verfügung steht, bedeutet dies, dass Sie bereits die neueste Version registriert haben. Sie können die Anweisungen unten befolgen, um eine aktualisierte Lizenz für Ihr Origin zu erhalten.
  1. Starten Sie Origin. Wählen Sie Hilfe: Lizenz aktivieren.
  2. Der Produktschlüssel ist im Dialog Lizenzaktivierung bereits eingetragen. Klicken Sie auf die Schaltfläche Aktivieren.
  3. Nachdem der Aktivierungsvorgang beendet ist, wählen Sie Hilfe: Über Origin, um zu prüfen, ob das aktualisierte Ablaufdatum der Softwarepflege korrekt ist.
Hinweis:

Ein Upgrade auf die neueste Version wird Ihnen während der Laufzeit der Softwarepflege immer empfohlen.
