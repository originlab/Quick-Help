Lizenzaktivierung

Lic-Activation-Dlg

Um Origin zu aktivieren, brauchen Sie einen Produktschlüssel.

Dieser Dialog Lizenzaktivierung wird jedes Mal geöffnet, wenn Sie Origin starten, bis Sie einen Produktschlüssel eingeben und den Lizenzierungsprozess abschließen. Der Dialog kann auch manuell aufgerufen werden. Dazu verwenden Sie das Menü Hilfe: Lizenz aktivieren.

HINWEIS: In den meisten Fällen funktioniert der Lizenzprozess im Hintergrund. Sollte er aufgrund einer Firewall, einem Webserver oder einem anderen Problem mit dem Internetzugang fehlschlagen, verwenden Sie das zweite Bearbeitungsfeld, um den Lizenztext manuell einzugeben.

Was ist ein Produktschlüssel und wie erhalte ich ihn?

Der Produktschlüssel ist mit Ihrer Seriennummer verbunden. Er wird an den OriginLab-Server gesendet, um eine Lizenz für Ihren Computer zu erzeugen.
