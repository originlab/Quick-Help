Internetlizenz

Internet-Lic

Ähnlich wie die Concurrent-Lizenz erlaubt die Internetlizenz Anwendern, Origin auf einer beliebigen Anzahl von Computern in ihrer Organisation zu installieren. Sie können Origin gleichzeitig auf der gleichen Anzahl von Computern ausführen, wie sie an Lizenzen erworben haben.

Die Unterschied ist, dass Sie im Fall der Internetlizenz keinen Lizenzserver in Ihrer Organisation pflegen müssen.

Wenn Sie n Lizenzen kaufen, werden Ihnen n Produktschlüssel bereitgestellt. Mit diesen Produktschlüsseln können Sie Origin auf jedem PC installieren und aktivieren, das heißt, Sie können Origin gleichzeitig auf n PCs ausführen. Die Lizenz wird automatisch auf dem PC deaktiviert, auf dem der Produktschlüssel vorher verwendet wurde.

Diese Lizenz macht es erforderlich, dass Ihre PCs alle mit dem Internet verbunden sind. Es gibt jedoch die Option, einen Produktschlüssel für einen Zeitraum von 7 Tagen am Stück an einen bestimmten PC zu binden. Dadurch gewinnen Sie die Flexibilität, die Lizenz ohne Unterbrechung verwenden zu können, wenn Sie sich außerhalb Ihrer Organisation befinden, z. B. auf Reisen.

SEHR WICHTIGE HINWEISE!!!

Dieser Lizenztyp erfordert, dass der Computer IMMER MIT DEM INTERNET VERBUNDEN IST. Stellen Sie sicher, dass Ihr Netzwerk über eine stabile Internetverbindung mit geringem Datenverkehr verfügt.

Seriennummer registrieren und Produktschlüssel erhalten

Wenn Sie eine EinzelIizenz mit Seriennummer haben:

xFxxx-6xxx-70/71/72xxxxx

Befolgen Sie bitte die Anweisungen auf dieser Seite, um Ihre Seriennummer zu registrieren und den oder die Produktschlüssel zu erzeugen.

Wenn Sie Administrator einer Mehrbenutzerlizenz mit Seriennummer sind:

xFxxx-6xxx-76xxxxx

Befolgen Sie bitte die Anweisungen auf dieser Seite, um Ihre Seriennummer zu registrieren und den Verteilungsmechanismus der Produktschlüssel einzurichten, BEVOR Sie Origin an Ihre Anwender verteilen.

Lizenzaktivierung

  1. Starten Sie Origin. Der Dialog Lizenzaktivierung wird geöffnet.
  2. Wählen Sie die Option Produktschlüssel unten eingeben und auf Aktivieren klicken und geben Sie Ihren Produktschlüssel ein. Klicken Sie auf Aktivieren.
  3. Der Lizenzprozess wird nun im Hintergrund durchgeführt. Geschieht dies erfolgreich, erhalten Sie eine entsprechende Nachricht.

Sollte der Deaktivierungsprozess im Hintergrund fehlschlagen, fahren Sie bitte mit den folgenden Schritten fort:

  1. Wenn der Lizenzierungsprozess aus irgendeinem Grund fehlschlägt, wird ein Notepad-Fenster mit einem Link darin geöffnet.
  2. Kopieren Sie diesen Link in einen Webbrowser, der eine Internetverbindung hat. Der Link leitet Sie zur OriginLab-Webseite.
  3. Wenn Sie sich an Ihrem Konto anmelden, wird der Lizenztext erzeugt und auf der Webseite angezeigt.
  4. Kopieren Sie den Lizenztext und fügen Sie ihn in Origins Dialog Lizenzaktivierung ein. Klicken Sie auf OK.

Lizenzdeaktivierung

Die Lizenz MUSS NICHT deaktiviert werden. Wenn der Produktschlüssel auf einem anderen Computer aktiviert wird, wird das Origin mit diesem Produktschlüssel auf dem bisherigen Computer deaktiviert.

Verbindung von Origin und Internet trennen

Sie können eine Lizenz für einen begrenzten Zeitraum von einer Woche an einen bestimmten Computer binden. Während dieses Zeitraums kann der Computer vom Internet getrennt werden und Origin wird noch immer erfolgreich ausgeführt.

Hinweis: Sie müssen die Lizenz an den Computer binden, BEVOR Sie ihn vom Internet trennen.

  1. Öffnen Sie Origin. Wählen Sie Hilfe: Über Origin.
  2. Klicken Sie auf die Schaltfläche Lizenz.
  3. Geben Sie den Produktschlüssel ein, wenn er nicht bereits automatisch angegeben wurde.
  4. Aktivieren Sie das Kontrollkästchen "Binden Sie den Produktschlüssel an diesen PC, um ihn 7 Tage ohne Internetzugang zu verwenden.".
  5. Klicken Sie auf die Schaltfläche Aktivieren.
  6. Entfernen Sie die Verbindung zwischen Computer und Netzwerk.
