Einen Gruppenschlüssel für eine große rechnergebundene Gruppenlizenz (Node-Locked) erzeugen und verwenden

Group-Product-Key

Für den Administrator einer großen Gruppe, z. B. einer kampusweiten Site-Lizenz, stellen wir einen speziellen Gruppenproduktschlüssel zur Verfügung, der das Verteilen von Origin auf eine große Anzahl von Computern einfacher machen soll.

Um den Gruppenproduktschlüssel zu erzeugen:

  1. Melden Sie sich auf der Webseite von OriginLab an.
  2. Klicken Sie auf den Link "Meine Lizenz(en)" und dann auf die gewünschte Seriennummer.
  3. Klicken Sie auf den Link "Alle Produktschlüssel". Auf der aufgerufenen Seite ist die Schaltfläche Gruppenproduktschlüssel anfragen zu sehen.


Der Zweck des Gruppenproduktschlüssels besteht darin, dass Sie den Schlüssel während der Aktivierung nicht mehr manuell eingeben müssen.

Um den Gruppenproduktschlüssel zu verwenden:

  1. Speichern Sie den Gruppenproduktschlüssel in einer Textdatei und geben Sie der Datei den Namen PK.txt.
  2. Legen Sie die Datei PK.txt in dem Ordner ab, in dem sich die anderen Origin-Installationsdateien befinden.
  3. Führen Sie die Origin-Installationsdateien aus, um die Software zu installieren.
  4. Wenn die Installation abgeschlossen ist, starten Sie Origin. Beachten Sie, dass der Gruppenproduktschlüssel im Dialog Lizenzaktivierung bereits eingegeben und die Lizenz automatisch aktiviert ist.

Der Gruppenproduktschlüssel* in Kombination mit der Methode der stillen Installation ermöglicht Ihnen, Origin automatisch auf jedem Benutzerrechner zu installieren und zu lizenzieren. (*nicht für individuelle/interaktive Installation)
