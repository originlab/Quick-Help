Origin 2016 erfordert den FlexLM-Server Version 11.13 oder höher in Windows

Concurrent-2016-Require-High-Version

Für Origin und OriginPro 2016 muss der FlexLM-Lizenzserver (FlexNet) für Windows auf 11.13.0.2 oder höher aktualisiert sein. Die folgende Installationsdatei, die auch in einem 64-Bit-System verwendet werden kann, kann heruntergeladen werden:

FlexLMserverSetup32bit.exe

Server auf einem neuen Windows-Rechner einrichten[bearbeiten]

Führen Sie einfach die Installationsdatei oben aus.

Vorhandenen FlexLM-Lizenzserver aktualisieren[bearbeiten]

Wenn ein FLEXnet-Server bereits für Origin 2015 oder eine Vorgängerversion installiert wurde, müssen Sie lmgrd.exe, orglab.exe und lmtools.exe mit den Dateien in der zip-Datei unten ersetzen.

flexnet_win32.zip

Alternativ können Sie den alten FLEXnet-Server deinstallieren und dann den neuen Server mit der Installationsdatei oben installieren.

Bitte halten Sie den Server an, bevor Sie die 3 exe-Dateien ersetzen oder den Server neu installieren, Führen Sie dazu die folgenden Schritte aus:

  1. Öffnen Sie LMTools.exe und klicken Sie die Registerkarte Start/Stop/Reread an. Klicken Sie auf Stop Server.
  2. Schließen Sie LMTools.

Wenn Sie 64-Bit-FLEXnet verwenden müssen, ersetzen Sie die 3 exe-Dateien mit den Dateien in der untenstehenden zip-Datei.

flexnet_win64.zip
