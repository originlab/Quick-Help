FAQ-39 Wie kann ich meinen Lizenzserver für einen neuen Rechner einrichten?

Change-Lic-Server-to-new-PC

Letztes Update: 10.03.2023

Lizenzserver ändern

Sollte Ihr Lizenzserver nicht mehr funktionieren oder stillgelegt sein und Sie müssen den Server-Host wechseln, senden Sie bitte ein Anfrageformular für eine Systemübertragung ein. Nachdem Ihre Anfrage bearbeitet wurde, werden Sie per E-Mail benachrichtigt. Sie können sich dann die neue Lizenzdatei besorgen und den FLEXnet-Dienst auf dem neuen Server einrichten. Hier finden Sie weitere Informationen zur FLEXnet-Installation.

So senden Sie ein Anfrageformular für die Systemübertragung ein

  1. Melden Sie sich über Ihr Online-Konto auf der Webseite von OriginLab an.
  2. Klicken Sie auf den Link Meine Lizenz(en) verwalten.
  3. Aktivieren Sie das Kontrollkästchen vor der Seriennummer, die Sie übertragen möchten.
  4. Klicken Sie auf die Schaltfläche Einen nicht mehr verwendeten Lizenzserver melden.
  5. Vervollständigen Sie das Formular und lesen Sie die Policy zum Lizenzierungs- und Installationssupport.
  6. Anfrage senden

Bitte kontaktieren Sie OriginLab wegen der Anfrage Ihrer Systemübertragung, falls Ihre Version nicht mehr unterstützt wird.

Wenn Sie sich außerhalb der USA und KANADA befinden, wenden Sie sich bitte an einen lokalen Vertriebspartner

Sobald Ihre Anfrage verarbeitet wurde, können Sie damit fortfahren, sich eine neue Lizenzdatei zu besorgen und den neuen Server einzurichten.

  • Siehe die FLEXnet-Installationsanweisungen für Windows oder Linux.
