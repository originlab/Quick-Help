FAQ-461 Wie kann ich eine Datenmatrix normieren?

Normalize-Matrix

Letztes Update: 04.02.2015

Sie können bei aktivem Matrixobjekt folgenden Skript ausführen. Dieser Skript normalisiert die Daten, so dass sie einen minimalen Wert von 0 und einen maximalen Wert von 1 besitzen. Sie können das Skript nach Ihren Anforderungen für verschiedene Normalisierungswerte anpassen.

// Normalize to be between 0 and 1 in all cells range rm=<active>; sum(rm); // Get statistics rm -= sum.min; // subtract the minimum, resulting in new minimum of zero rm /= (sum.max - sum.min); // divide by range, new maximum of one

Wählen Sie Fenster: Skriptfenster und fügen Sie das oben angegebene Skript in dieses Fenster ein. Markieren Sie den gesamten Skript und drücken Sie Eingabe.

Schlüsselwörter:Skript, transformieren, skalieren, normalisieren
