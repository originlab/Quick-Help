FAQ-404 Wie kann ich Python in Origin ausführen?

 

Letztes Update: 16.08.2015

use-python-in-Origin

Origin bietet eine eingebettete Python-Umgebung, so dass Sie entweder Python in Origin ausführen können (Unterstützung sowohl von Befehlszeile aus als auch über .py-Datei), oder ein PyOrigin-Modul verwenden, um von Python aus auf Origin zuzugreifen.

Erste Informationen zur Verwendung von Python in Origin erfahren Sie im Kapitel Working with Python in der LabTalk-Anleitung.

Schlüsselwörter:Python, Skript, PyOrigin

Origin-Version mind. erforderlich: 2016 SR0
