FAQ-792 Why do I see "Your FLEXlm license file does not allow the requested feature." when I use Origin 2016 to connect with FLEXnet Server?

FLEXnet-Server-Error

Last Update: 11/2/2015

Origin 2016 concurrent network package requires FLEXnet server version 11.13.0.2 or later.

If your FLEXnet server version is older, you will see an error message:
"Your FLEXlm license file does not allow the requested feature".

Please update your Origin FLEXnet server by following the instruction in the page.

http://www.originlab.com/index.aspx?go=support&pid=3215
