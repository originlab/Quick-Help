FAQ-1163 Wie füge ich eine Dateierweiterung zu einem Datenkonnektor hinzu?

Add-File-Extension-Data-Connector

Letztes Update: 28.01.2023


Öffnen Sie Apps: Konnektoren, um eine Liste der installierten Datenkonnektoren zu sehen. Klicken Sie mit der rechten Maustaste auf den Datenkonnektor und wählen Sie Benutzerdefinierte Dateierweiterungen....

Sie können im Dialog Ihre Erweiterung hinzufügen, indem Sie den Platzhalter * (Sternchen), gefolgt von der Erweiterung, verwenden. Jede Erweiterung in der Liste wird durch ein Semikolon ; getrennt.

Achten Sie darauf, keine überflüssigen Leerstellen oder leeren Zeichen zu verwenden. Es sollte keine Leerstelle zwischen ; und * sein.

Schlüsselwörter:Dateityp, benutzerdefiniert
