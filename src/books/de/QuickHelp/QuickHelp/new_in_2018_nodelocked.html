FAQ-898 Welche Neuerungen gibt es in Origin 2018 für rechnergebundene Lizenzen (Node-Locked)?

New-in-2018-NodeLocked

Letztes Update: 12.11.2017

An Version 2018 können verschiedene Versionen der rechnergebundenen Node-Locked-Lizenz parallel installiert werden, ähnlich wie bereits bei Netzwerklizenzen. Daher können Sie den gleichen Produktschlüssel zum Aktivieren von mehreren Versionen auf dem gleichen Computer verwendet werden. Vorerst handelt sich dabei um den gleichen Produktschlüssel für Version 2017 und 2018.

Lesen Sie diese Seite-> Im Abschnitt "Falls ja..." finden Sie Einzelheiten zur Durchführung des Upgrades.
