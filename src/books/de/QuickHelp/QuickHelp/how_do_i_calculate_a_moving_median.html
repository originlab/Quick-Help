FAQ-324 Wie kann ich einen gleitenden Median berechnen?

cal-movmedian-SCV

Letztes Update: 15.07.2018

Verwenden Sie den Dialog Werte setzen, um einen einfachen gleitenden Median für einen bestimmten Bereich (definiert durch die Fenstergröße) zu berechnen.

  1. Markieren Sie die Spalte oder die Zellen, in die Sie das Ergebnis einfügen möchten.
  2. Wählen Sie im Menü Spalte: Spaltenwerte errechnen, um den Dialog zu öffnen (oder drücken Sie Strg + Q auf der Tastatur).
  3. Geben Sie in dem aufgerufenen Dialog median(B[i:i+50]) in das Textfeld Spaltenformel ein, um den gleitenden Median der Spalte B zu berechnen, wobei sich das gleitende Fenster vom aktuellen Punkt auf den 50. Punkt zubewegt.
  4. Klicken Sie auf OK.

Weitere Informationen finden Sie in dieser FAQ.

Schlüsselwörter: gleitender Median, gleitende Statistikwerte
