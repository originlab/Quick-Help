FAQ-900 Ich bin berechtigt für ein automatisches Upgrade. Wie aktiviere ich mein kostenloses Upgrade auf die neue Version?

Auto-Upgrade-to-New-Version

Letztes Update: 04.05.2019

Wenn Sie Origin auf dem gleichen Computer installieren:

Installieren Sie die neue Version mit der gleichen Seriennummer. Origin wird nun aktiviert, ohne dass ein Produktschlüssel manuell eingegeben werden muss.

Wenn Sie die neue Version auf einem anderen Computer installieren möchten:

Melden Sie sich auf der Webseite von OriginLab an und navigieren Sie zu den verfügbaren Produktschlüsseln unter der Seriennummer.

Wenn keine Produktschlüssel verfügbar sind, müssen Sie die alte Version auf dem anderen Computer deaktivieren, um diesen Produktschlüssel für die neue Version erneut zu verwenden.
