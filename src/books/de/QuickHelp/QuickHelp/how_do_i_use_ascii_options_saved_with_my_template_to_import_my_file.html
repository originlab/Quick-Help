FAQ-410 Wie kann ich Optionen des ASCII-Imports in meiner Vorlage speichern?

Save-Import-Settings

Letztes Update: 06.01.2019

  1. ASCII-Import benutzerdefiniert anpassen
    1. Wählen Sie Daten: Aus Datei importieren: Einzelnes ASCII oder Daten: Aus Datei importieren: Mehrere ASCII.
    2. Stellen Sie im Dialog ASCII, der zum Auswählen der Datei geöffnet wird, sicher, dass das Kontrollkästchen Optionendialog zeigen aktiviert ist.
    3. Wählen Sie Ihre Datei(en) und klicken auf Öffnen. Dadurch wird der Dialog ImpASC geöffnet.
    4. Passen Sie die Importeinstellungen nach Bedarf an.
    5. Klicken Sie auf OK. (Hinweis: In Origin 9.0 und älter klicken Sie bitte auf den Pfeil rechts vom Feld Dialogdesign und wählen Speichern in <Blatt>, bevor Sie auf OK klicken.
    6. Nach dem Import befinden sich die Importeinstellungen im Arbeitsblatt.
    7. Sie können Ihre Analyse weiterführen.
  2. Importeinstellungen in Vorlage speichern
    • Wählen Sie Datei: Vorlage speichern unter.
    • Oder wählen Sie Datei: Mappe als Analysevorlage speichern..., wenn nach dem Importieren der Daten Analysen und Berechnungen im Arbeitsblatt durchgeführt werden, so dass im Arbeitsblatt grüne Schlösser angezeigt werden.
  3. Importieren Sie neue Daten mit benutzerdefinierten ASCII-Einstellungen.
    1. Wählen Sie Datei: Öffnen, um die gespeicherte Vorlage zu laden.
    2. Wählen Sie Daten: Aus Datei importieren: Einzelnes ASCII oder Daten: Aus Datei importieren: Mehrere ASCII.
    3. Im Dialog ASCII wird <Blatt> neben Optionendialog zeigen angezeigt. Das heißt, dass im Arbeitsblatt Importeinstellungen enthalten sind. Wenn Sie diese Option deaktivieren und importieren, werden die Daten mit benutzerdefinierter Einstellung automatisch importiert. Wenn Sie die Option aktivieren und auf Öffnen klicken, wird der Dialog ImpASC mit den geladenen benutzerdefinierten Einstellungen geöffnet.
