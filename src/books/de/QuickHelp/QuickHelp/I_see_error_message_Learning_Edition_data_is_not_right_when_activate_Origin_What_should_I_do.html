FAQ-1092 Wenn ich Origin aktiviere, sehe ich die Fehlermeldung "Learning Edition data is not right". Was soll ich tun?

Learning-Edition-Data-Is-Not-Right

Letztes Update: 06.04.2021

Wenn Sie die Fehlermeldung Learning Edition data is not right sehen, wenn Sie die Lernausgabe aktivieren:

Bitte überprüfen Sie zweimal, ob Sie die Lernausgabe mit der korrekten Seriennummer installiert haben:

  • Sie können Hilfe: Über Origin auswählen, um die installierte Seriennummer anzuzeigen. Prüfen Sie, ob es die gleiche Nummer ist, die Sie per E-Mail erhalten haben. Falls nicht, befolgen Sie bitte die Anweisungen auf dieser Seite der Schnellhilfe, um sie zu korrigieren.
  • Wenn Sie eine Lernausgabe mehrfach beantragt haben, kann die Seriennummer eine andere sein. Stellen Sie sicher, dass Sie Origin mit der Nummer installiert haben, die Sie in der letzten E-Mail erhalten haben.

Wenn die installierte Seriennummer korrekt ist, das Problem aber nach wie vor auftritt, senden Sie bitte eine E-Mail an le@originlab.com. Bitte geben Sie Ihren Produktschlüssel und die E-Mail-Adresse an, die Sie zum Anwender der Lernausgabe verwendet haben.

Schlüsselwörter:Lizenz, LE, Lernausgabe, Lernversion, Fehler, kann nicht aktiviert werden
