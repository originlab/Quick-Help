FAQ-1047 Wie kann ich Datenpunkte in einem Boxdiagramm zeigen und benutzerdefiniert anpassen?

data-point-box-chart

Letztes Update: 16.03.2020

 

Um die eingeteilten Daten im Boxdiagramm anzuzeigen:
  1. Klicken Sie zum Öffnen des Dialogs Details Zeichnung doppelt auf das Boxdiagramm.
  2. (Wählen Sie auf der Registerkarte Boxdiagramm eine der Optionen Daten, Box + Datenüberschneidung, Box (Rechts) + Daten (Links), Box (Links) + Daten (Rechts), Balken + Datenüberschneidung, Balken (Rechts) + Daten (Links), Balken (Links) + Daten (Rechts), Halbe Box (Rechts) + Daten (Links) oder Halbe Box (Links) + Daten (Rechts) in der Auswahlliste Typ.)

Oder

  1. Klicken Sie auf das Boxdiagramm und die Minisymbolleiste wird angezeigt.
  2. Klicken Sie auf die Schaltfläche Boxtyp und wählen Sie die Option mit Datenpunkt.

 

Um Datenpunkte im Boxdiagramm benutzerdefiniert anzupassen:
  1. Klicken Sie zum Öffnen des Dialogs Details Zeichnung doppelt auf das Boxdiagramm.
  2. Gehen Sie zur Registerkarte Symbol, um das Format der Datenpunkte anzupassen.


Falls der Typ des Boxdiagramms Daten ist, können Sie neben der obigen Methode die Datenpunkte auch über die Minisymbolleiste benutzerdefiniert anpassen.

  1. Klicken Sie auf das Boxdiagramm und die Minisymbolleiste wird angezeigt.
  2. Auf der Minisymbolleiste können Sie die Schaltflächen Randfarbe, Füllfarbe, Zeichnungssymbol, Symbolgröße vergrößern und Symbolgröße verkleinern verwenden, um das Format der Datenpunkte anzupassen.
