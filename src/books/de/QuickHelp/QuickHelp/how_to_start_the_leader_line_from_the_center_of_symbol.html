FAQ-879 Wie verlege ich den Anfang der Verbindungslinien in die Mitte eines Symbols?

Start-LeaderLine-from-Symbol-Center

Letztes Update: 23.07.2018

Die Verbindungslinie, die das Symbol und seine Beschriftungen verbindet, beginnt standardmäßig am Symbolrand. Es gibt einen Abstand zwischen der Linie und dem Symbolrand. Wenn Sie den Anfang der Verbindungslinie in die Mitte eines Symbols verlegen möchten, können Sie

  1. ein weiteres Symboldiagramm der gleichen Quelldaten hinzuzufügen (über den Dialog Layerinhalt oder den Dialog Diagrammeinstellungen).
  2. Setzen Sie die Symbolgröße des neuen Diagramms auf 0 (über den Dialog Details Zeichnung -> Registerkarte Symbol).
  3. Aktivieren Sie die Beschriftung des neuen Diagramms und blenden Sie die Verbindungslinien ein (über den Dialog Details Zeichnung -> Registerkarte Beschriftung).
  4. Führen Sie Folgendes im Befehlsfenster aus: 
    set %c -gtl 0;
    Der Abstand zwischen der Verbindungslinie und dem Symbol wird auf 0 gesetzt.
Hinweise:

Das obige Labtalk-Skript kann nur für das aktive Diagramm wirksam werden. Da die zwei Diagramme aus den gleichen Daten gezeichnet werden, kann das Symboldiagramm nicht als das aktive Diagramm festgelegt werden, wenn es nicht die erste Zeichnung in der Grafik ist, und das LabTalk-Skript wird nicht funktionieren.
