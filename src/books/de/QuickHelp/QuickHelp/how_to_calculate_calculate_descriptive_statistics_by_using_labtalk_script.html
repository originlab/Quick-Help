FAQ-660 Wie berechne ich deskriptive Statistiken mit Hilfe von LabTalk-Skript?

Descriptive_Statistics_LabTalk

Letztes Update: 27.03.2015

Bereich oder spezifische Spalten auswählen

Sie können die deskriptive Statistik für Datensätze in der Arbeitsmappe durchführen, indem Sie die X-Funktion Stats auf einen ausgewählten Bereich oder bestimmte Spalten anwenden.

Die Syntax ist folgende:

stats ix:=<active> //on the selecting range stats ix:=wcol(1) //on the 1st column

Die folgenden statistischen Werte werden berechnet:

Statistik Gespeichertes Ergebnis
Mittelwert stats.mean
Gesamtsumme stats.sum
Minimum stats.min
Maximum stats.max
Standardabweichung stats.sd
Anzahl der Punkte stats.n

Zum Beispiel:

stats ix:=wcol(1) //view the results stats.mean=; stats.sd=; stats.sum=;
Hinweise:

Sie können auch Beschriftungen für die Variablen definieren und diese in späteren Operationen verwenden, zum Beispiel:

stats ix:=col(a) mean:=m sd:=std sum:=mysum //view the results m=; std=; mysum=;
Zeilen

Die X-Funktion Rowstats kann über mehrere Spalten verwendet werden, um einen neuen Datensatz auszugeben, der die Statistikwerte (Mittelwert, Standardabweichung, SE des Mittelwerts, Minimum, Maximum, N, Summe) von jeder Zeile enthält.

Zum Beispiel:

rowstats irng:=col(1):col(3) mean:=<new> sd:=<new>; //mean and sd of each row will be generated in the new columns rowstats irng:=col(1):col(3) mean:=wcol(4) sd:=<optional>; //output mean of each row to the 4th column
