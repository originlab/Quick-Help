FAQ-426 Was soll ich tun, wenn die Importgeschwindigkeit beim gleichzeitigen Importieren von vielen Dateien allmählich abnimmt?

Improve-Import-Speed

Letztes Update: 04.02.2015


Beim Import von Datendateien werden die Dateiinformationen in der Arbeitsmappe festgehalten, um einen Neuimport zu ermöglichen. Nach dem Import der Datendateien können Sie den Arbeitsmappenorganizer öffnen (mit einem Rechtsklick auf die Titelleiste der Arbeitsmappe und Auswahl von Zeige Organizer). Im linken Bedienfeld des Organizers werden alle importierten Dateipfade und -namen aufgeführt. Wenn viele Einträge vorhanden sind (das bedeutet viele Dateien), wird die Importgeschwindigkeit allmählich langsamer.

Um die Importgeschwindigkeit zu verbessern, sorgt eine Systemvariable (@npi) dafür, dass die Importdateiinformarionen in der Arbeitsmappe nicht mehr aufgezeichnet werden. Ohne diese Informationen funktioniert der Neuimport allerdings nicht.

@npi = 1;  // prevent recording the import files info
