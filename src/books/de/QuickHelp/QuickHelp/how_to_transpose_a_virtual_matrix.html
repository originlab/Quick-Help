FAQ-838 Wie kann ich eine virtuelle Matrix transponieren?

Transpose-VM

Letztes Update: 19.09.2016

Alle virtuellen Matrizen in einem Projekt werden im Dialog Virtueller Matrixmanager aufgelistet. Um eine virtuelle Matrix zu transponieren:

  1. Wählen Sie im Menü Hilfsmittel: Virtueller Matrixmanager, um den Dialog Virtueller Matrixmanager zu öffnen. Alle virtuellen Matrizen werden im Feld Virtuelle Matrizen auf der linken Seite aufgeführt.
  2. Markieren Sie die gewünschte virtuelle Matrix und klicken Sie mit der rechten Maustaste. Wählen Sie Als XY-Transponierte duplizieren im Kontextmenü. Die ausgewählte virtuelle Matrix wird dupliziert und transponiert.

Schlüsselwörter:Virtuelle Matrix, transponieren, duplizieren
