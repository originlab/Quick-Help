FAQ-766 Wie erzeuge ich einen benutzerdefinierten Word-Bericht nach der Stapelverarbeitung?

Word-repo-Batch-Process

Letztes Update: 27.08.2015

Sie können eine Word-Vorlage erstellen, so dass die stapelverarbeiteten Ergebnisse in dieser Vorlage ausgegeben werden können und automatisch einen benutzerdefinierten Word-Bericht erzeugen können.
Sie erfahren in diesem Tutorial, wie Sie eine Word-Vorlage erstellen und die erstellte Vorlage im Stapelprozess verwenden.
