FAQ-167 Kann ich meine Formatierung für zusammengefügte Diagrammfenster speichern?

MergedGraph-Save-Formatting

Letztes Update: 04.02.2015

Die Menüoption Grafik: Grafikfenster zusammenfügen lässt Sie mehrere Grafikfenster zu einem neuen Grafikfenster zusammenfügen. In dem sich öffnenden Dialog können Sie die Grafikfenster auswählen, die Sie zusammenfügen möchten. Dieser Dialog enthält eine Vorschau, wie das neue Diagramm aussieht.

Wenn Sie immer die offenen Grafikfenster in einem aktiven Ordner zusammenfügen, würden Sie Zeit sparen, da Sie nicht den Dialog öffnen müssten, um die zusammenzufügenden Grafiken festzulegen. Mit diesem Dialog - wie mit anderen Origin-Dialogen auch - können Sie die Einstellungen speichern und ein Design erstellen, das ein neues Ausklappmenü für Sie erzeugt. Wenn Sie das nächste Mal Grafiken zusammenfügen möchte, können Sie Ihr Design aus dem Menü auswählen und brauchen auf diese Weise weniger Zeit.

Sollten Sie ein Design erstellen wollen, dass ein neues Diagramm mit 2x2 Feldern aus Ihren 4 einzelnen Diagrammen erzeugt, dann befolgen Sie die unten stehenden Schritte.

  1. Wählen Sie Grafik: Grafikfenster zusammenfügen.
  2. Stellen Sie sicher, dass die Auswahlliste Zusammenfügen auf Alle in aktivem Ordner (Offen) eingestellt ist.
  3. Erweitern Sie den Zweig Anordnungseinstellungen und setzen Sie die Anzahl der Zeilen und Anzahl der Spalten jeweils auf 2.
  4. Nehmen Sie Ihre Einstellungen an Abstand etc. vor.
  5. Klicken Sie auf die Schaltfläche mit dem Pfeil oben im Dialog und rechts neben dem Feld Dialogdesign. Wählen Sie Speichern unter... und geben Sie einen Designnamen wie 2x2 ein.
  6. Klicken Sie auf OK, um den Vorgang des Zusammenfügens jetzt durchzuführen, oder auf Abbrechen, wenn Sie im Moment nichts zusammenfügen möchten.

Wenn Sie nächstes Mal eine 2x2-Anordnung durchführen möchten, können Sie einfach Grafik: Grafikfenster zusammenfügen: 2x2 verwenden. Der Dialog öffnet sich nicht, die Einstellungen in Ihrem Design werden verwendet und alle offenen Diagramme in dem aktiven Ordner werden zusammengefügt, so dass ein neues Grafikfenster erstellt wird.
