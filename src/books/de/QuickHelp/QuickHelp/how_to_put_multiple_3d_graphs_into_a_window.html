FAQ-950 Wie kann ich mehrere 3D-Diagramme in einem Fenster anzeigen?

Combine-3D-graphs

Letztes Update: 15.06.2018

Zurzeit unterstützt das 3D-Diagrammfenster von Origin nicht mehrere Layer. Wenn Sie mehrere 3D-Diagramme in einem Fenster zusammenfügen möchten, können Sie das Fenster Layout verwenden, um all diese 3D-Diagramme anzuordnen.

Schlüsselwörter:3D-Diagramme zusammenfügen, mehrere 3D-Diagrammlayer
