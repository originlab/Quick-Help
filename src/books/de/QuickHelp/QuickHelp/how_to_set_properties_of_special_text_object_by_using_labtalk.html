FAQ-888 Wie lege ich Eigenschaften für ein spezielles Textobjekt (wie Achsentitel) mit Hilfe von LabTalk fest?

Set-Special-Text-ByLT

Letztes Update: 31.08.2017

Origin hat einige spezielle Textobjekte wie die Achsentitel unten:

Objekt Objektname
Titel der unteren X-Achse xb
Titel der oberen X-Achse xt
Titel der linken Y-Achse yl
Titel der rechten Y-Achse yr
Titel der hinteren Z-Achse zb
Titel der vorderen Z-Achse zf

Für diese speziellen Objekte können Sie auch objectName.= ausführen, um alle Eigenschaften zu ermitteln bzw. festzulegen.

xb.fsize = 20; //set the font size of bottom X axis title.

yl.fillcolor=2; //Add a box frame to left Y axis title and set the fill color to red. yl.transparency=50;//set the transparency to 50%.
Hinweis: Führen Sie objectName.= aus, um fast alle Eigenschaften des Objekts anzuzeigen. Auch die allgemeinen Eigenschaften werden auf dieser Seite aufgeführt.
