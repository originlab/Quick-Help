FAQ-1046 Wie füge ich eine Verbindungslinie für 2D- und 3D-XYY-Balken hinzu?

Letztes Update: 21.02.2020

Add-Connect-Line-Column


Sie können in einem Origin-Diagramm Verbindungslinien für die 2D- und 3D-XYY-Balken auf der Registerkarte Anzeige im Dialog Plot Details hinzufügen.

Sie können Folgendes tun:

  1. Klicken Sie doppelt auf das Balken-/Säulendiagramm in der Grafik, um den Dialog Details Zeichnung zu öffnen.
  2. Gehen Sie zur Registerkarte Anzeige und aktivieren Sie die Option Verbindungslinie. Die Registerkarte Linie wird im Dialog Details Zeichnung gezeigt.
  3. Auf der Registerkarte Linie können Sie den Stil der Verbindungslinie benutzerdefiniert anpassen.

 

Die Linie verbindet alle Balken in der gruppierten Grafik.

Standardmäßig ist die Verbindungslinie innerhalb des Teildatensatzes im gruppierten 2D-Balkendiagramm verbunden.
Wenn Sie die Linie so festlegen möchten, dass Sie alle Balken im Diagramm verbindet, können Sie die Option Nach Teildatensatz verbinden auf der Registerkarte Linie im Dialog Details Zeichnung deaktivieren.

Stichwörter:Balken, Säulen, Verbindungslinie, Linie, innerhalb von Teildatensatz verbinden
