FAQ-362 Was ist der Code Builder?

CodeBuilder

Letztes Update: 04.02.2015

Der Code Builder ist Origins Integrated Development Environment (IDE) zur Erstellung und Fehlerbehebung von Origin-Code, der entweder mit LabTalk oder mit OriginC erzeugt wird. Zusätzlich zu einem Arbeitsbereich zum Bearbeiten von mehreren Dateien mit farbkodierter Syntax verfügt der Code Builder über Bedienfelder für Ausgabe, Beobachten, Lokale Variablen, Stack aufrufen und LabTalk-Konsole, die alle die Entwicklung und Fehlerbehebung Ihres Codes unterstützen.

Sie können den Code Builder über das Menü Ansicht oder über Alt+4 öffnen. In einingen Fällen führen Origin-Menüoptionen LabTalk-Code aus, und Sie können den Code Builder automatisch an der Stelle der Quelldatei öffnen, an der das Menü ausgeführt wird. Halten Sie beispielsweise die Umschalt- und Strg-Taste gedrückt, während Sie den Menübefehl Zeichnen: Statistikdiagramme: Histogramm auswählen. Anstatt diesen Befehl auszuführen, öffnet Origin die Datei PLOT.OGS im Code Builder im Abschnitt [Histogramm], wo der Code beginnt.

Siehe die Tutorials Einführung in Origin C und den Code Builder und Der Arbeitsbereich des Code Builder.
