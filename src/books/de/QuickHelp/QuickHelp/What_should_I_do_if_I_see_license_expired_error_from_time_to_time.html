FAQ-1089 Welche sollte ich tun, wenn ich ab und zu den Fehler sehe, dass die Lizenz abgelaufen sei?

FLEXnet-TIMEOUT-Environment-Variable

Letztes Update: 14.11.2022

Wenn zwischen den Maschinen Ihres Clients und dem Lizenzserver eine instabile oder langsame Netzwerkverbindung besteht, können Sie Origin eventuell nicht mit dem Lizenzdienst verbinden oder Sie erhalten, während Origin läuft, ab und zu die Nachricht, dass die Lizenz abläuft.

Um dieses Problem zu beheben:

Sie können eine Systemumgebungsvariable FLEXLM_TIMEOUT auf der Client-Maschine erstellen, um die Verbindungszeit zum Lizenzserver zu verlängern.

Methode 1:

Führen Sie bei Origin 2023 oder höher Origin als Administrator aus und öffnen Sie das Skriptfenster über Fenster: Skriptfenster im Hauptmenü. Geben Sie

@FLTO=4000000

ein und drücken Sie ENTER.

Dies setzt die Systemumgebungsvariable FLEXLM_TIMEOUT auf 4 Sekunden. Der Wert 4.000.000 ist in Mikrosekunden (ist gleich 4 Sekunden). Sie können den Wert erhöhen, wenn die Verbindung des Clients noch immer abläuft. Wenn Sie jedoch den Wert auf 10.000.000 erhöhen und immer noch ein Problem haben, wird es wahrscheinlich kein Timeoutfehler sein.

Wenn Sie dies getan haben, starten Sie Origin normal.

Es ist möglich, die Datei config.ini zu bearbeiten, um FLEXLM_TIMEOUT mit dem Installationsprogramm Setup.exe voreinzustellen. Weitere Einzelheiten finden Sie auf dieser Seite.

Methode 2:

Fügen Sie die Systemumgebungsvariable FLEXLM_TIMEOUT hinzu bzw. aktualisieren Sie sie auf 4000000 (4 s). Starten Sie Origin dann neu.

Einzelne Schritte:

  1. Geben Sie im Suchfeld auf der Taskleiste von Windows "Systemsteuerung" ein und wählen Sie sie aus den Ergebnissen aus, um sie zu öffnen.
  2. Suchen Sie in der Systemsteuerung die "Umgebungsvariablen" im Suchfeld und wählen Sie Umgebungsvariablen bearbeiten aus den Ergebnissen.
  3. Wählen Sie im Dialog Systemeigenschaften die Registerkarte Erweitert. Klicken Sie auf die Schaltfläche Umgebungsvariablen.
  4. Wenn Sie in der Gruppe Systemvariablen keine Variable FLEXLM_TIMEOUT haben, klicken Sie bitte auf die Schaltfläche Neu..., um eine neue Umgebungsvariable, wie folgt, zu erstellen:
    • Variablenname: FLEXLM_TIMEOUT
    • Variablenwert: 3000000 oder größer.
  5. Klicken Sie auf OK.
