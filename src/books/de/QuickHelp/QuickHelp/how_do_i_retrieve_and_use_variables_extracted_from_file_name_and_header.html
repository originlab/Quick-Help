FAQ-423 Wie kann ich Variablen, die aus dem Dateinamen und -header extrahiert wurden, aufrufen und verwenden?

Retrieve-and-Use-Variables

Letztes Update: 07.08.2015

Es gibt mehrere Möglichkeiten, mit denen Dateimetadaten extrahiert und in Ihrem Origin-Projekt verwendet werden können. Diese FAQ legt den Schwerpunkt auf die flexibelsten Methoden der Bedienoberfläche zum Aufrufen und Verwenden von Variablen, die Sie aus Ihren importierten Datendateien extrahiert haben.

  1. Zuerst müssen Sie Metadaten aus Ihrer Datendatei extrahieren. Das beste Origin-Hilfsmittel für dies ist der Importassistent. Um heruaszufinden, wie das im Einzelnen aussieht, sehen Sie sich bitte Teil 3 dieses Video-Tutorials über den Import von Daten an.
  2. Sobald Sie Ihre Variablen identifiziert und Ihre Datendateien importiert haben, gibt es mehrere Optionen zum Aufrufen und Verwenden von extrahierten Variablen:
    • Um einen Variablenwert in eine Arbeitsblattzelle einzufügen, klicken Sie mit der rechten Maustaste auf die Zelle und wählen Sie Variablen einfügen (im Bearbeitungsmodus können Sie Strg+H drücken). Der Dialog Variablen einfügen wird für Arbeitsblätter geöffnet. Weitere Informationen zu diesem Dialog finden Sie unter diesem Thema
    • Zum Einfügen eines Variablenwerts in ein Textobjekt auf einer Diagrammseite wählen Sie Text einfügen auf der Symbolleiste Hilfsmittel, klicken Sie auf eine leere Stelle in dem Diagrammfenster, klicken Sie dann mit der rechten Maustaste und wählen Sie Infovariable einfügen (oder drücken Sie Strg+H). Weitere Informationen zu der Verwendung dieses Dialogs finden Sie unter diesem Thema.
    • Die Variablen in dem Bereich page.info einer Arbeitsmappe können zum Beispiel in Berechnungen für Werte setzen in dieser Arbeitsmappe verwendet werden. Diese Variablen werden auf Seitenebene in beiden Dialogen, Arbeitsmappenorganizer oder Variablen einfügen, aufgeführt (Datenzeichnungen, Arbeitsblätter). Weitere Informationen finden Sie in diesem Eintrag des OriginBlogs.
