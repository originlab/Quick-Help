FAQ-323 Wie führe ich eine Spaltenberechnung über gleitende Fenster durch, wie einen gleitenden Durchschnitt oder eine gleitende Standardabweichung?

cal-movavg-SCV

Letztes Update: 15.12.2016

Gleitende Standardfunktionen verwenden

Origin enthält mehrere Standardfunktionen, mit denen Sie die Fenstergröße rückwärts und/oder vorwärts festlegen können:
Movavg, Emovavg, Mmovavg, Movcoef, Movrms, Movslope, Tmovavg und Wmovavg.

Wenn Sie zum Beispiel die Funktion Movavg im Dialog Werte setzen verwenden möchten, tun Sie Folgendes:

  1. Markieren Sie die Spalte oder die Zellen, in die Sie das Ergebnis einfügen möchten.
  2. Wählen Sie im Menü Spalte: Spaltenwerte errechnen, um den Dialog zu öffnen (oder drücken Sie Strg + Q auf der Tastatur).
  3. Klicken Sie in dem geöffneten Dialog auf Funktion: Statistik: Movavg(vd,back,forward), um die Funktion movavg automatisch in das Textfeld Spaltenformel einzufügen.
  4. Ersetzen Sie vd mit dem Bereich, für den Sie einen gleitenden Durchschnitt berechnen möchten.
  5. Ersetzen Sie back mit dem Rückwärts-Versatz in Bezug zur aktuellen Position und forward mit dem Vorwärts-Versatz in Bezug zur aktuellen Position.
  6. Klicken Sie auf OK.

 

Andere Funktionen modifizieren und zu gleitenden Funktionen machen

Sie können Unterbereiche in Ihrer Formel zum Errechnen von Spaltenwerten verwenden, um ein gleitendes Fenster festzulegen.

Sie können zum Beispiel zum Berechnen der Standardabweichung über einer Fenstergröße von 11 den Unterbereich in Ihrer Formel festlegen:

StdDev(A[i-5:i+5])

Verwenden Sie Formel: Sample laden im Menü des Dialogs Werte setzen für Beispiele zu Berechnungen von Spaltenwerten.



Hinweis: Zum Berechnung des gleitenden Durchschnitts einer Zeichnung können Sie Analyse: Signalverarbeitung: Glättung verwenden und als Methode den gleitenden Durchschnitt bestimmen.
