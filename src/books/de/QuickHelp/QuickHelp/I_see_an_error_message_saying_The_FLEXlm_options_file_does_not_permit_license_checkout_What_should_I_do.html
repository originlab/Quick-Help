FAQ-1121 Ich erhalte die Fehlermeldung "Die FLEXlm-Optionsdatei erlaubt keine Lizenzausbuchung. Was soll ich tun?

Opt-File-Not-Permit-Lic-Checkout

Letztes Update: 30.12.2021

Falls Sie die folgende Meldung sehen, wenn Sie versuchen, Ihr Origin mit dem Lizenzserver zu verbinden:

License Status: 112

The FLEXlm options file does not permit license checkout.

Sie wird dadurch verursacht, dass Ihr Lizenzservermanager eine Optionsdatei eingerichtet hat, um zu bestimmen, wer auf den Server zugreift, und Ihrer Maschine keine Zulassung hat, den Vorgang zu Ende zu führen.

Um dieses Problem zu beheben, kontaktieren Sie bitte Ihren Manager, um die verwendete Optionsdatei zu prüfen.

Schlüsselwörter:err 112, keine Verbindung zum Server, 3 Minuten, Lizenz läuft ab, Netzwerk, Concurrent-Lizenz, freie Lizenz, Ausbuchung nicht erlaubt
