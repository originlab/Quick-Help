FAQ-9 Warum wird mir eine Fehlermeldung angezeigt, wenn ich versuche, Origin zu installieren?

Install-Origin-Error-Message

Letztes Update: 04.02.2015

Wenn Ihnen beim Start von Origin die Fehlermeldung "Fehler beim Installieren von ikernal.exe" ("error installing ikernal.exe") angezeigt wird, wird dies normalerweise durch Ihren Login-Account von Windows verursacht, der nicht über die notwendigen Installationsrechte verfügt. Versuchen Sie bitte folgende Lösung:

  1. Gehen Sie auf die Webseite von Flexera.
  2. Klicken Sie auf Knowledge Base.
  3. Wählen Sie Installation/Application Readiness Products in der Liste Product Group.
  4. Wählen Sie InstallShield in der Auswahlliste Product.
  5. Geben Sie Q111519 in das Suchfeld ein und klicken Sie auf Search.
  6. Der Eintrag Initialization Error Troubleshooting beschreibt die möglichen Ursachen für diese Fehlermeldung. Befolgen Sie die Schritte zur Behebung des Problems.
