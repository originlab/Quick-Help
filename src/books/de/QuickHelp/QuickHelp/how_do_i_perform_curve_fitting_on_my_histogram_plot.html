FAQ-253 Wie kann ich einen Kurvenfit auf mein Histogramm anwenden?

CurveFit-on-Histogram

Letztes Update: 04.02.2015

Wenn Sie Daten aus einem Histogramm anpassen, werden nicht die eingeteilten Daten, sondern die Rohdaten angepasst.

Wenn Sie bereits wissen, dass Sie die eingeteilten Daten anpassen möchten, führen Sie anstatt der Erstellung eines Histogramms eine Häufigkeitszählung in Ihrer Datenspalte durch. Wählen Sie Statistik: Deskriptive Statistik: Häufigkeiten zählen, um eine Häufigkeitszählung durchzuführen. Ein neues Arbeitsblatt wird erstellt, von dem aus Sie die Spalte Zahlen als Säulendiagramm zeichnen können. Nun können Sie bei aktiver Zeichnung den Dialog NLFit öffnen und die Daten anpassen.

Alternativ können Sie mit der rechten Maustaste auf Ihr Histogramm klicken und Gehe zu Klassierungsdaten im Kontextmenü auswählen. Ein neues Arbeitsblatt wird erstellt, dass die eingeteilten Daten enthält. Seit Version 8.5.1 können Sie auf die Schaltfläche Los auf der Registerkarte Daten des Dialogs Details Zeichnung klicken, um zu dem Arbeitsblatt mit den eingeteilten Daten zu gelangen. Sie könen dann Ihr Säulendiagramm von diesem Arbeitsblatt aus erstellen. Ist dieses neue Säulendiagramm aktiv, können Sie die Daten anpassen.

Schlüsselwörter:Histogramm, nichtlineare Kurvenanpassung, nichtlinearer Kurvenfit, Häufigkeiten zählen, Datenmodus
