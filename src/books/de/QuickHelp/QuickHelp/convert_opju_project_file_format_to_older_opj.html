FAQ-927 Gibt es eine Möglichkeit, das neue OPJU-Dateiformat in das ältere OPJ-Format umzuwandeln?

Convert-OPJU-Project-File-Format-to-Older-OPJ

Letztes Update: 05.01.2023

Origin 2018 führte einen neuen Unicode-kompatiblen Projektdateityp (UTF-8) ein -- die OPJU-Datei. Diese Dateien können nicht in älteren Versionen von Origin geöffnet werden (es sollte jedoch angemerkt werden, dass die Versionen Origin 2018 bis 2022b die Option behalten, Dateien im älteren OPJ-Format zu speichern). Neue Funktionen in Origin 2023 und höher erfordern das OPJU-Format und können daher keine OPJ-Dateien mehr speichern.

Wenn Sie eine OPJU-Datei nicht mit Ihrer Origin-Software öffnen können, tun Sie Folgendes:

  1. Laden Sie den Origin Viewer auf dieser Seite der OriginLab-Webseite herunter.
  2. Extrahieren Sie den Inhalt der Downloaddatei des Viewers.
  3. Klicken Sie doppelt auf die .exe-Datei, um den Viewer auszuführen (es keine Installation erforderlich). Wählen Sie dann Datei: Projekt speichern unter im Menü Viewer und speichern Sie das Projekt als eine OPJ-Datei neu.


Weitere Informationen zum Origin Viewer können Sie auf dieser Seite der OriginLab-Webseite nachlesen.
