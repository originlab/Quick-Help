FAQ-655 Welche Art von Fitparametern (Fehleranalyse) kann in der linearen Anpassung ermittelt werden?

Fit-Parameters

Letztes Update: 28.07.2018

Um eine lineare Anpassung in Origin durchzuführen, können Sie die folgenden Statistikkennwerte der Anpassungsparameter erhalten:

  • Wert
    Parameterwert
  • Standardfehler
    Standardfehler der Parameter
  • UEG
    Die untere Konfidenzgrenze
  • OEG
    Die obere Konfidenzgrenze
  • Konfidenzniveau für Parameter (%)
    Das Konfidenzniveau für Regression
  • t-Wert
    Wert des t-Tests der Parameter
  • Wahrsch.>|t|
    p-Wert der Parameter
  • KI halbe Breite
    Halbe Breite des Konfidenzintervalls

Um mehr darüber zu lernen, wie diese Statistiken nach der Anpassung interpretiert werden, beziehen Sie sich bitte auf dieses Dokument: Regressionsergebnisse interpretieren.
