FAQ-472 Wie lege ich das Datumsformat fest, wenn die Daten nicht mit dem gewünschten Anzeigeformat konsistent sind?

Convert-Date-Format

Letztes Update: 14.10.2020

Die letzten Versionen von Origin machten den Import von Datums-/Zeitdaten viel einfacher, insbesondere mit der Erweiterung durch den Text/CSV-Datenkonnektor, der die Formatierung von gängigen Datums-/Zeitformaten automatisch verarbeitet.

Trotzdem kann es passieren -- insbesondere wenn Sie nicht standardisierte Datums-/Zeitformate verwenden --, dass Ihre Datums-/Zeitdaten nicht automatisch beim Import formatiert werden. Sie können Probleme schnell an der Ausrichtung der Daten in der Arbeitsblattzelle erkennen: wenn die Datums-/Zeitdaten links ausgerichtet sind, werden sie als Text betrachtet und nicht als Datum/Zeit (die intern als numerisch gespeichert werden). Diese FAQ zeigt Ihnen, wie Sie Ihre Datums-/Zeitdaten im gewünschten Format anzeigen.


Um Textdaten in numerische Daten umzuwandeln oder das Anzeigeformat der Daten zu ändern, tun Sie Folgendes:

  1. Klicken Sie doppelt auf die Spaltenüberschrift, um den Dialog Spalteneigenschaften zu öffnen und das Format auf Datum zu setzen.
  2. Klicken Sie auf die Liste Anzeige und wählen Sie die entsprechende Zeichenkette der Datums-/Zeitsyntax aus der Liste und klicken Sie auf Anwenden. Anweisungen zur Syntax von Datum und Zeit finden Sie unter Daten und Zeiten im Arbeitsblatt.
  3. Wenn ein geeignetes Format nicht in der Liste verfügbar ist, können Sie Ihr eigenes auf eine von zwei Weisen definieren (beziehen Sie sich auf die Syntax im vorherigen Link):
    • Definieren Sie die Zeichenkette unter Benutzerdefinierte Datumsformate im Dialog Einstellungen: Optionen (Registerkarte Sonstiges) vor. Die Zeichenkette wird dann als eine der zwei "globalen benutzerdefinierten" Anzeigeformate angezeigt, die in der Liste Anzeige verfügbar sind.
    • Setzen Sie die Anzeige auf Benutzerdefinierte Anzeige und geben Sie Ihre Zeichenkette direkt im nun verfügbaren Feld Benutzerdefinierte Anzeige ein. Ihr benutzerdefiniertes Format wird zur Liste Benutzerdefinierte Anzeige hinzugefügt und kann in zukünftigen Sitzungen verwendet werden.
