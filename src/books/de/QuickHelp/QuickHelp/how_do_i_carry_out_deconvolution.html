FAQ-309 Wie führe ich eine Entfaltung durch?

deconvolution-methods

Letztes Update: 04.02.2015


Die Entfaltung ist eine weit verbreitete Methode in verschiedenen Forschungsgebieten, wie z.B. der supramolekularen Chemie, der Berechnung von Fluoreszenzlebensdauern etc. In Origin gibt es mehrere Ansätze zur Durchführung einer Entfaltung für verschiedene Szenarien:

  • Die Daten sind eine Faltung von Gauss- und Exponentialfunktionen:
    Passen Sie die Daten direkt mit der Standardanpassungsfunktion GaussMod im Dialog NLFit an.
  • Die Daten sind eine Faltung von zwei Funktionen (nicht Gauss und Exponential):
    Befolgen Sie das Tutorial Zwei Funktionen mit Faltung anpassen.
  • Die IRF (Instrument Response Function - Instrument-Antwort-Funktion) ist sehr komplex und kann nicht durch eine Funktion definiert werden:
    Befolgen Sie das Tutorial Mit Faltung anpassen.
  • Verwenden Sie die Entfaltung, um Peaks zu analysieren:
    Lesen Sie in dieser FAQ nach oder verwenden Sie die FFT, um die Entfaltung durchzuführen.
