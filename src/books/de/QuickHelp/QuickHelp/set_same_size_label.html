FAQ-713 Wie kann ich die gleiche Schriftgröße für Beschriftungen in verschiedenen Layern festlegen?

Set-same-size-label

Letztes Update: 04.01.2019

Wenn Sie die gleiche Schriftgröße für alle Layer in einem Diagramm festlegen möchten:

  1. Öffnen Sie zuerst Einstellungen: Designs verwalten. Wählen Sie auf der Registerkarte Diagramm Layer No Element Scaling, um WYSIWYG sicherzustellen.
  2. Wählen Sie eine Beschriftung im Diagramm und legen Sie die Schriftgröße fest.
  3. Klicken Sie mit der rechten Maustaste auf sie und wählen Sie Format kopieren. Wenn darunter Untermenüs zu finden sind, wählen Sie ein geeignetes wie Stil oder Schriftart.
  4. Um die gleiche Schriftgröße auf alle anderen Layer anzuwenden, klicken Sie mit der rechten Maustaste außerhalb des Layer und wählen Sie Format einfügen in: Alle: Schrift.

Schlüsselwörter:Beschriftung, Mehrere Layer, konsistent, Text
