FAQ-855 Warum funktioniert meine DLL zur Anpassung auf einem PC, aber nicht auf einem anderen?

fittingDLL-fail-in-anotherPC

Letztes Update: 29.07.2018

Wenn Sie eine DLL zum Anpassen in einer höheren Version von Visual Studio erstellen (z. B. VS2015, Dialog Property: Configuration Properties: General: Platform Toolset), kann es sein, dass sie nicht auf einem anderen PC funktioniert, wenn dieser nicht die MS VC Redistributable dieser Version installiert hat.

Beachten Sie, dass Origin 2016/2017 und höhere Versionen automatisch MS VC 2012 Redistributable installieren.

Schlüsselwörter:DLL, Kurvenanpassung, Visual Studio
