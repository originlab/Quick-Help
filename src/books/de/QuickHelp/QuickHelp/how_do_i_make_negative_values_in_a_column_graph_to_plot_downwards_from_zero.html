FAQ-178 Wie kann ich negative Werte in einem Säulendiagramm von Null abwärts zeichnen lassen?

Plot-Negative-Cloumn-from-0

Letztes Update: 04.02.2015

Wenn Sie möchten, dass Ihre X-Achse bei der Position gezeigt wird, an der Y=0:

  1. Wählen Sie Format: Achsen: X-Achse oder klicken Sie doppelt auf die X-Achse, um den Dialog Achsen zu öffnen.
  2. Wechseln Sie zur Registerkarte Linie und Hilfsstriche und stellen Sie sicher, dass das Symbol Unten im linken Bedienfeld ausgewählt ist.
  3. Wählen Sie die Option An Position= aus der Auswahlliste Achsenposition.
  4. Geben Sie 0 in das Textfeld Prozent/Wert ein.
  5. Klicken Sie auf OK.


Wenn Sie Ihre X-Achse unten anzeigen möchten, die negativen Balken aber nach unten weisen, müssen Sie sicherstellen, dass die zusätzliche Linie bei Y=0 aktiviert ist:

  1. Wählen Sie Format: Achsen: X-Achse oder klicken Sie doppelt auf die X-Achse, um den Dialog Achsen zu öffnen.
  2. Wechseln Sie zur Registerkarte Linie und Hilfsstriche und stellen Sie sicher, dass das Symbol Unten im linken Bedienfeld ausgewählt ist.
  3. Wählen Sie Unten aus der Auswahlliste Achsenposition.
  4. Wechseln Sie zur Registerkarte Gitternetze.
  5. Stellen Sie sicher, dass das Kontrollkästchen Y = 0 in der Gruppe Zusätzliche Linien aktiviert ist. Sollte es deaktiviert sein, zeigen die negativen Balken vom Wert des Y-Minimums aus nach oben.
