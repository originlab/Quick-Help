FAQ-321 Wie kann ich ein Histogramm mit den Anzahlen als Beschriftungen zeichnen?

histogram-with-counts-label

Letztes Update: 18.02.2019


Ab Origin 2017 SR0 können Sie nach Erstellen eines Histogramms die Auswahlliste Zeigen auf der Registerkarte Beschriftung des Dialogs Details Zeichnung verwenden, um Anzahl/Prozentanteil direkt als Beschriftung hinzuzufügen.

Bei Origin 2016 oder früheren Versionen müssen Sie aus den eingeteilten Daten ein Histogramm erstellen, dessen Beschriftungen aus Anzahlen bestehen.

Methode 1: Sie können das Hilfsmittel Häufigkeitszählung verwenden, um die eingeteilten Daten zu berechnen.

  1. Wählen Sie Statistik: Deskriptive Statistik: Häufigkeiten zählen, um eine Häufigkeitszählung durchzuführen.
  2. Markieren Sie im Ergebnisarbeitsblatt die Spalte Anzahlen und wählen Sie im Menü Zeichnen: Einfache 2D: Säulen + Beschriftung.

Methode 2: Alternativ können Sie die eingeteilten Daten einem Histogramm entnehmen.

  1. Zeichnen Sie zuerst ein Histogramm, indem Sie Zeichnen: Statistisch: Histogramm wählen.
  2. Klicken Sie dann mit der rechten Maustaste auf Ihr Histogramm und wählen Sie Gehe zu Klassifizierungsdaten im Kontextmenü aus.
  3. Markieren Sie im Einteilungsarbeitsblatt die Spalte Anzahlen und wählen Sie im Menü Zeichnen: Einfache 2D: Säulen + Beschriftung.
