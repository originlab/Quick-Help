FAQ-128 Kann ich externe Farbpalettendateien verwenden, um auf meine Diagramme Farbe anzuwenden?

Use-External-Color-Palette

Letztes Update: 11.12.2020

Origin wird mit einer Anzahl von integrierten Palettendateien (.PAL) ausgeliefert, die verwendet werden können, um Farbe auf farbkodierte Diagramme wie Konturdiagramme, gruppierte Diagramme und Diagrammen, in denen Grafikelemente (z. B. Datenpunkte) auf Datenwerte aus einer anderen Spalte abgebildet werden, anzuwenden.

In Origin 2021 wurden grundlegende Verbesserungen vorgenommen, zum Beispiel die Unterstützung von mehr Palettenformaten, die Installation von Paletten per Drag&Drop und ein neuer Farbmanager, um das Hinzufügen, Modifizieren und Verwalten von Palettendateien zu vereinfachen. Diese FAQ deckt die letzten Versionen der Origin-Software ab.

Paletten zu Origin 2020b und älteren Versionen hinzufügen

Sie können Ihre eigenen Palettendateien zu Origin hinzufügen. Origin unterstützt die Dateien Microsoft binary .PAL und JASC ASCII .PAL. Die Hilfsmittel unten ermöglichen Ihnen, zu einer externen .PAL-Datei zu navigieren und sie auf das aktuelle Diagramm anzuwenden.

  1. Die Farbschaltfläche Palette auf der Symbolleiste Stil
  2. Die Bedienelemente der Farbauswahl im Dialog Details Zeichnung
  3. Aus den Dialog Palette anwenden wird über das Menü Grafik: Palette auf Farbabbildung anwenden zugegriffen.


Wenn Sie diese .PAL-Datei zu Ihrer Origin-Software hinzufügen möchten, müssen Sie die .PAL-Datei in Ihren <Anwenderdateiordner>\Palettes\ kopieren. (Sollte der Ordner noch nicht existieren, erstellen Sie ihn.) Ihre Palettendatei wird jetzt in jeder Palettenliste in Ihrem Origin angezeigt.

Paletten zu Origin 2021 und höheren Versionen hinzufügen

Origin 2021 unterstützt die folgenden Typen von Farbpaletten: Scribus (.xml), CorelDraw und Corel PhotoPaint (.xml), Office Color Table (.soc), Adobe Color (.aco), Adobe Color Table (.act), Adobe Swatch Exchange (.ase), JASC PaintShopPro (.pal) und GNUPlot Palette (.pal).

Außerdem macht ein neuer Farbmanager (Hilfsmittel: Farbmanager) das Hinzufügen von Paletten viel einfacher:

  • Die Schaltfläche Aus Web importieren zum Hinzufügen von Paletten mit gegebener URL oder zum Herunterladen von OriginLabs Webseite Dateiaustausch.
  • Die Schaltfläche Aus Dateien importieren zum Hinzufügen von Paletten aus lokalen Dateien.
  • Die Schaltfläche Neu zum Erzeugen von eigenen benutzerdefinierten Paletten.

Paletten per Drag&Drop hinzufügen

Sie können die erweiterte Liste der Palettenformate mit Ausnahme derjenigen im XML-Format sämtlichst per Drag&Drop hinzufügen. Um eine Palette in einem der unterstützten XML-Formate hinzuzufügen, verwenden Sie die Schaltflächen Importieren aus ... im Farbmanager.
