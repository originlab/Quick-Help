FAQ-298 Wie setze ich die Schranke bei der Anpassung einiger Funktionen?

Set-Bound-for-Fitting

Letztes Update: 04.02.2015

Falls eine Anpassung nicht konvergiert, überdenken Sie die Möglichkeit, Grenzen für die ausgewählten Anpassungsparameter festzulegen. Obwohl es keine Garantie dafür gibt, dass eine Anpassung konvergieren wird, können die untenstehenden Vorschläge in bestimmten Fällen nützlich sein.

Sobald der begrenzte Fit konvergiert, ergibt auch die Anpassung mit zurückgesetzten oder zurückgenommenen Grenzen oftmals eine konvergierte Anpassung.

Die folgenden Anpassungsfunktionen werden meistens im Hilfsmittel Peaks analysieren verwendet. Einige von ihnen können auch im NLFit verwendet werden.

DoniachSunjic Sie können eine kleine obere Grenze für den Parameter a, d.h a<=2, einrichten, wenn es zu viele Schwankungen in der angepassten Kurve nach dem ersten Anpassungsversuch gibt.
GaussianLorenCross Legen Sie eine Grenze für s fest, z.B. 0,25<=s<=0,75. Setzen Sie die Grenze zurück, sobald die Anpassung konvergiert.
FraserSuzuki Es ist nicht empfehlenswert, einen negativen Peak anzupassen, falls die X-Koordinate des Peakzentrums nicht positiv ist.
Pulse Diese Funktion wird zum Anpassen von positiven Peaks verwendet. Beachten Sie, dass die Amplitude A so festgelegt ist, dass sie eine positive Grenze hat. Bei einem negativen Peak kann es sein, dass das Konvergieren der Anpassung fehlschlägt.
ECS Diese Funktion eignet sich mehr zum Anpassen von positiven Peaks. Sie können die Grenze aber auch für die Amplitude A für die Anpassung des negativen Peaks lösen.
PearsonVII Diese Funktion eignet sich mehr zum Anpassen von positiven Peaks. Sie können die Grenze aber auch für die Amplitude A für die Anpassung des negativen Peaks lösen.
Weibull3 Um eine geeignete konvergente Anpassung zu erhalten, müssen Sie w2 auf einen geeigneten Wert setzen, da w2 die Form der angepassten Kurve beeinflusst. Beispielsweise können Sie w2=3,4 festlegen, um eine Form wie Gaussian zu bekommen. Es wird empfohlen, w2 zu befreien. Für eine asymmetrische Peakanpassung können Sie eine geeignete Grenze für w2 um 3,4 setzen.
