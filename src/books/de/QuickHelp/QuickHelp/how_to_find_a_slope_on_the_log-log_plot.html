FAQ-233 Wie suche ich eine Steigung auf dem Log-Log-Diagramm?

Find-Slop-on-Log2-Plot

Letztes Update: 04.02.2015

Nachdem Sie Daten auf einer Log-Log-Skalierung gezeichnet haben, öffnen Sie den Dialog Lineare Anpassung durch die Menüauswahl von Analyse: Anpassen: Linearer Fit. Aktivieren Sie in diesem Dialog das Kontrollkästchen Scheinbarer Fit im Zweig Fit-Optionen, um die exponentiellen Daten mit einer geraden Linie anzupassen. Sie können die Steigung dann im Ergebnisblatt ablesen.

Schlüsselwörter:log-Skalierung, Steigung, Log-Log-Diagramm anpassen
