FAQ-374 Was ist der aktuelle Arbeits- und Favoritenordner?

Current-Directory

Letztes Update: 04.02.2015

Die Standardeingabeaufforderung im Befehlsfenster (Öffnen über Fenster: Befehlsfenster oder Alt + 3) lautet ">>". Sie können aber auch den aktuellen Arbeitsordner als Befehlsaufforderung verwenden, indem Sie mit der rechten Maustaste in das rechte Bedienfeld des Befehlsfensters klicken und Einstellungen im Kontextmenü wählen. Unabhängig davon, welche Eingabeaufforderung Sie verwenden, ist es dieses aktuelle Verzeichnis, "Current Directory" (CD), wo Origin zuerst nach der Skriptdatei sucht, die Sie ausführen. Wenn das aktuelle Verzeichnis nicht geändert ist, sucht Origin die Skriptdatei zuerst im Anwenderdateiordner und dann im Origin-Programmordner.

Um den aktuellen Arbeitsordner zu ändern, können Sie die X-Funktion cd verwenden. Außerdem verfügt Origin über die X-Funktion cdset zum Verwalten von vordefinierten Pfaden bzw. Favoritenordnern, so dass der aktuelle Arbeitsordner einfach geändert werden kann. Zum Beispiel:

cd;  // output the current working folder cd c:\aaa;  // set c:\aaa as the current working folder cdset 4;  // set c:\aaa to pre-defined paths list, with index of 4 cdset;  // output the pre-defined paths cd 2;  // set the 2nd pre-defined path as the current working folder


Hinweis:

  • Möglicherweise ist es für die vordefinierten Pfade am besten, den Index 1 für den Anwenderdateiordner zu reservieren, da es sich hierbei um eine Standardzuweisung handelt. Diese Zuweisungen werden im Abschnitt [cdset] der Datei origin.ini file gespeichert.
  • Wenn der Index 1 nicht für den Anwenderdateiordner zugewiesen ist, können Sie stattdessen das Zeichenkettenregister %Y verwenden, das diesen Pfad ebenfalls speichert.
