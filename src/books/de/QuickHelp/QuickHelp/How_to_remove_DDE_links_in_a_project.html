FAQ-1134 Wie werden DDE-Links in einem Projekt entfernt?

Remove-DDE-Link

Letztes Update: 05.05.2023

Wenn Ihr Projekt über DDE-Links (Dynamic Data Exchange) verfügt, z. B. Arbeitsblattzellen mit Zellen der externen Excel-Datei verknüpft, ist es eventuell nicht einfach, das OPJU mit anderen zu teilen, die keinen Zugriff auf die Excel-Datei haben.

Stattdessen können Sie den aktualisierten Dialog Formel/Links entfernen verwenden, um alle DDE-Links im Projekt in Werte umzuwandeln und dann mit anderen zu teilen.

  1. Wählen Sie im Menü Bearbeiten: Links entfernen ....
  2. Stellen Sie sicher, dass das Kontrollkästchen Excel-Links (DDE) aktiviert ist.
  3. Wählen Sie Projekt in der Auswahlliste Links entfernen in und klicken Sie auf OK.
  4. Speichern Sie das Projekt.

Hinweis: Dieser Dialog kann auch dazu verwendet werden, Zellformeln/Links und Spaltenformeln in Werte umzuwandeln.


Wenn Sie die Person sind, die eine Projektdatei wie diese ohne Zugriff auf die Excel-Datei erhält, dann fragt der Dialog Start DDE Link beim Öffnen der Projektdatei in Origin nach Wiederherstellen oder Entfernen des Links. Sollte Ihr Projekt sehr viele solcher Links haben, wird der Dialog wiederholt angezeigt, was sehr nervig werden und sogar verhindern kann, das Projekt zu öffnen.

Um solch eine Projektdatei öffnen zu können, ohne den Dialog Start DDE Link zu zeigen, setzen Sie die Systemvariable @dden = 1, bevor Sie die Projektdatei öffnen. Einzelheiten zu der Systemvariable @dden hier.
