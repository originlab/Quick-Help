FAQ-692 Wie lege ich Werte für einen Teilbereich fest?

 

Letztes Update: 04.02.2015

Um Werte für einen Teilbereich einer Spalte oder mehrere Spalten zu setzen, können Sie den Teilbereich auswählen, mit der rechten Maustaste klicken und entweder Spaltenwerte errechnen oder Werte für mehrere Spalten festlegen auswählen. Verwenden Sie die Bedienelemente von Bereich/Spalte oben, um den Teilbereich festzulegen und die Formel in das Bearbeitungsfeld Spaltenformel einzugeben.

Bitte lesen Sie im Dokument Dialog Werte setzen Einzelheiten.

Origin-Version mind. erforderlich: 9.1 SR0

Schlüsselwörter:Dialog Werte setzen
