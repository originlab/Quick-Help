FAQ-345 Wie löse ich Probleme der Dialoganzeige?

Dialog-Dislay-Problem

Letztes Update: 05.07.2019

Wenn Sie ein Problem bei der Dialoganzeige bemerken - das obere Bedienfeld des Dialogs NLFit wird z.B. zusammengepresst angezeigt oder alle Symbolleistenschaltflächen sind extrem klein -, dann versuchen Sie bitte Folgendes:

  1. Bitte lesen Sie unter Wie ändere ich die Größe von Dialogen wie Dialogen mit mehreren Feldern?, wie Sie die Größe des Dialogs verändern, um alle Bedienelemente anzuzeigen.
  2. Setzen Sie die DPI-Einstellung auf "kleiner" oder passen Sie die Auflösung an.
  3. Wenn Schritt 1 und Schritt 2 das Problem nicht lösen können, löschen Sie den Registry Key:
"HKEY_CURRENT_USER\Software\OriginLab\Origin 9.0\International\Dialogs\"
und starten Sie Origin neu.
Hinweise: Origin 9.0 im Registry Key sollte abhängig von der installierten Origin-Version geändert werden.
