FAQ-1058 My tick labels are chopped by table border. How can I fix it?

Label-Chopped-by-Axis-Table

Last Update: 5/24/2020

If the tick label text is too long that goes beyong the tabel cell, it will be chopped by the table borders.

To show the full label text beyond table, set system variable @ATLC=0 to disable cutting the tick label by table border. Refer to this page to see how to change the value of a system variable.
