FAQ-742 Wie füge ich die Linie für den Mittelwert zu meiner Kurve hinzu?

Add-Meanline

Letztes Update: 17.06.2015

Sie können das Minitool Statistik verwenden, um eine Mittellinie zu einer Kurve hinzuzufügen.

  1. Aktivieren Sie das Diagrammfenster und wählen Sie Minitool: Statistik im Hauptmenü.
  2. Klicken Sie in dem aufgerufenen Dialog auf OK, um das Rechteck der grafischen Datenauswahl (ROI) hinzuzufügen. Eine Linie für den Mittelwert wird gezeigt. Passen Sie das Feld so an, dass der Bereich, für den Sie den Mittelwert berechnen möchten, eingefasst wird.
  3. Klicken Sie auf die dreieckige Schaltfläche oben rechts und wählen Sie Kennzeichnungslinien im Ausklappmenü. Auf diese Weise wird eine Linie für den Mittelwert als ein Objekt hinzugefügt. Durch Schließen der grafischen Datenauswahl (ROI) wird die Linie des Mittelwerts nicht gelöscht.
