Über die Statusleiste

Wenn Sie im Arbeitsblatt eine Spalte, mehrere Spalten oder einen Datenbereich markieren, werden die grundlegenden Statistikwerte der Auswahl in der Statusleiste unten im Origin-Arbeitsbereich gezeigt.

Minimum und Maximum gehören nicht zu den Standardwerten, die gezeigt werden. Sie können mit der rechten Maustaste auf die Statusleiste klicken und Minimum, Maximum etc. aktivieren. Sie können auch über den Rechtsklick die Statistikwerte kopieren.

Weitere Informationen finden Sie unter  Die Statusleiste.

Mit dem Hilfsmittel Spalten-/Zeilenstatistik

Wenn der Datensatz sich in einer Spalte oder Zeile befindet, können Sie sich die Statistikwerte anzeigen lassen und die Werte für Minimum oder Maximum ermitteln. Um diese Hilfsmittel aufzurufen, wählen Sie Statistik: Deskriptive Statistik: Spaltenstatistik/Zeilenstatistik. In beiden Dialogen befinden sich die Kontrollkästchen für Minimum und Maximum im Zweig Zu berechnende Eigenschaften >> Quantile.

LabTalk-Funktionen verwenden

Die folgenden LabTalk-Funktionen (max() und min()) dienen zum Suchen von Maximum und Minimum. Die Funktion list() sucht die Indexnummer, und die Funktion xvalue() sucht den entsprechenden X-Wert einer gegebenen Indexnummer. Sie können das LabTalk-Skript unten im Skriptfenster ausführen (markieren Sie eines der unten bereitgestellten Skripte und drücken Sie Enter):
 

  • To find the maximum and minimum in column B:
    bMax = max(col(B));
    bMin = min(col(B));

  • To find the corresponding index number:
    maxIndex = list(max(col(B)),col(B));
    minIndex = list(min(col(B)),col(B));

  • To find the corresponding xvalue:
    1) Find corresponding value from x column:
    xvalue(list(max(col(B)),col(B)),col(B)) xvalue(list(min(col(B)),col(B)),col(B))
    2) Find corresponding value from a different column C:
    v1 = col(C)[list(max(col(B)),col(B))];
    v2 = col(C)[list(min(col(B)),col(B))];
    v3 = Table(col(B), col(C), Max(col(B)))
    v4 = Table(col(B), col(C), Min(col(B)))

 

Sie können die LabTalk-Funktionen in F(x)= Zelle oder im Dialog Werte setzen verwenden, um die Werte für Maximum und Minimum, Indexnummer, den entsprechenden X-Wert oder den entsprechenden Wert in einer beliebigen Spalte zu suchen.

Seit Origin 2017 unterstützt Origin die einfache Zellennotation in der Spaltenformel, so dass Sie direkt B anstatt von col(B) verwenden können.

Fügen Sie beispielsweise eine neue Spalte im Arbeitsblatt ein.

Fügen Sie Folgendes in die Zelle F(x) oder den Dialog Werte setzen ein:
Zum Suchen von Maximum und Minimum in Spalte B:
max(B) min(B) Zum Suchen der entsprechenden Zeilenindexnummer von Maximum bzw. Minimum in Spalte B:
Wenn es duplizierte Werte für Max./Min. in Spalte B gibt, wird die Zeilenindexnummer des als Erstes auftretenden Werts für Max./Min ausgegeben.
list(max(B),B)
list(min(B),B)
Um den entsprechenden X-Wert von Maximum oder Minimum in Spalte B zu suchen, nehmen Sie an B ist eine Y-Spalte.
xvalue(list(max(B),B),B)
xvalue(list(min(B),B),B)
Um den entsprechenden Wert in Spalte C in derselben Zeile zu finden, in der auch das Maximum bzw. das Minimum von Spalte B gefunden wurde:
C[list(max(B),B)]
C[list(min(B),B)]
Table(B, C, Max(B))
Table(B, C, Min(B))
Note: Sie müssen col(B) statt B für Origin 2016 und älter eingeben.
