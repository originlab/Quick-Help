FAQ-1111 Ich habe die Softwarepflege gekauft, erhalte aber beim Öffnen von Origin noch immer die Nachricht, dass meine Softwarepflege abgelaufen ist. Wie kann ich die Softwarepflegeinformationen aktualisieren?

Update-License-After-Renew-Maintenance

Letztes Update: 12.12.2021

Nach dem Kauf/der Verlängerung Ihrer Softwarepflege müssen Ihre Lizenz aktualisieren, um die neuen Softwarepflegeinformationen anzuzeigen.

Wenn Sie eine rechnergebundene Lizenz haben:

  1. Starten Sie Origin. Gehen Sie zu Hilfe: Lizenz aktivieren.
  2. Drücken Sie im Dialog auf Aktivieren, um die neue Lizenz zu erhalten. Dies schließt die aktuellen Softwarepflegeinformationen ein.

Sie können die Änderung überprüfen, indem Sie im Menü Hilfe: Über Origin wählen. Das korrekte Datum sollte Ihnen angezeigt werden.


Wenn Sie eine Concurrent-Lizenz haben:

Bitte lesen Sie diese Seite.
