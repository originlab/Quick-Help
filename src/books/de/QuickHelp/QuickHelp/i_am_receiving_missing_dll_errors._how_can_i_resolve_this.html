FAQ-18 Ich erhalte Fehler zu fehlenden DLLs. Wie kann ich dieses Problem lösen?

Resolve-Missing-DLL

Letztes Update: 16.05.2022

Origin wurde mit Hilfe von Microsoft (MS) Visual C++ erstellt. Daher sind bestimmte MS DLLs erforderlich, um es auszuführen. Diese DLLs müssen auf dem Computer vorhanden sein, auf dem Origin ausgeführt wird. Möglicherweise wird Ihnen ein MS DLL-Fehler beim Start von Origin auf einem Computer angezeigt, der nicht über diese DLLs verfügt, insbesondere wenn Sie MSI zum Installieren von Origin verwenden.

Um dieses Problem zu lösen, befolgen Sie die Schritte in dieser FAQ, um die erforderlichen DLLs erneut zu installieren.

Schlüsselwörter:Fehlermeldung, Origin-Version, weitere Software, Origin-Installation, vcruntime 140_1, keine DLL, MSVCR110.dll
