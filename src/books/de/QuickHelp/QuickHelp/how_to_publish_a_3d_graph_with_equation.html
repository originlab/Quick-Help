FAQ-173 Wie veröffentliche ich ein 3D-OpenGL-Diagramm mit Gleichung(en)?

export-3d-with-equation

Letztes Update: 21.06.2022

Seit 2017 SR0 unterstützt Origin das direkte Einfügen von OLE-Objekten (d. h. mathematische Gleichungen, Word-Objekte etc.) in 3D-OpenGL-Diagramme, wie es auch bei 2D-Diagrammen möglich ist.

Klicken Sie bei aktivem Diagrammfenster auf der Symbolleiste auf Gleichungen einfügen/Word-Objekte einfügen/Excel-Objekt einfügen/Objekt einfügen. um das gewünschte Objekt in das Diagrammfenster einzufügen.

Bitte beachten Sie: Um diese Art von Objekten einzufügen, müssen Sie die entsprechende Software/Plug-in muss vorher installiert haben. Um zum Beispiel eine Gleichung einzufügen, sollte ein Gleichungstool wie MathType, Microsoft Equation Editor oder OriginLabs kostenlose App LaTeX auf Ihrem System installiert sein.

Schlüsselwörter:3d, OpenGL, zusammenfügen, Gleichung, exportieren, veröffentlichen

Origin-Version mind. erforderlich: 2017 SR0
