FAQ-1129 Wie lege ich für die Achsenskalierung Log basiert auf 1000 fest?

Set-Axis-Scale-to-Log-Base-on-1000

Letztes Update: 22.02.2022

Die logarithmische Achsenskalierung in Origin basiert per Standard auf 10. Falls Sie den logarithmischen Skalierungstyp auf eine Basis von 1000 setzen wollen, verwenden Sie bitte eine benutzerdefinierte Formel. Das heißt:

  1. Klicken Sie doppelt auf die Achse, um den Dialog Achsen zu öffnen.
  2. Wählen Sie auf der Registerkarte Skalierung die Option Benutzerdefinierte Formel für Typ. In der Auswahlliste Direkte Formel finden Sie eine Beispielformel log(x)/log(1000). Wählen Sie es aus. Die inverse Formel wird automatisch mit 1000^x gefüllt.
  3. Um die Achsenhilfsstriche bei der integralen Position anzuzeigen, setzen Sie die großen Hilfsstriche > Typ auf Nach benutzerdefinierter Position und geben dann die Positionswerte ein, die Sie in der Reihe zeigen möchten.
    Hinweise: Wenn Sie die Hilfsstrichsposition mit den Werten der Direkten Formel festlegen möchten, das heißt im obigen Beispiel 0, 1, 2, 3, können Sie @CLFT = 1 setzen, bevor Sie die Hilfsstrichsposition festlegen.



Ebenso können Sie die logarithmische Achsenskalierung ändern, um sie auf einem anderen Wert als 1000 zu basieren, müssen die Direkte Formel und Inverse Formel aber entsprechend ändern.
