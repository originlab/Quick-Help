FAQ-405 Wo finde ich Beispiele zum Arbeiten mit Python in Origin?

 

Letztes Update: 04.02.2015

python-in-Origin-example

Origin enthält einige Beispiele zum Arbeiten mit Python. Sie liegen im Verzeichnis <Origin>\Samples\Python.

Es gibt zwei .py-Dateien, die direkt von Origin ausgeführt werden können (Einzelheiten können Sie in der LabTalk-Referenz zu run -pyf nachlesen).

Die anderen Origin-Projektdateien zeigen verschiedene Beispiele der Verwendung von Python, um auf Origin-Objekte zuzugreifen (Daten, Diagramm etc.). Sie können die Projektdatei öffnen und auf die Schaltfläche Python-Code zeigen klicken, um Zugriff auf die .py-Datei zu erlangen. Die .py-Datei befindet sich tatsächlich in dem Ordner Projekt des Arbeitsbereichs im Code Builder.

Schlüsselwörter:Skript, Python, PyOrigin, Beispiel

Origin-Version mind. erforderlich: 2015 SR0
