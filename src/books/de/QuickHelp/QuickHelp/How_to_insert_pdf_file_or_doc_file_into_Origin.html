FAQ-1027 Wie füge ich eine PDF-Datei bzw. .doc-Datei in Origin ein?

Insert-Word-or-PDF

Letztes Update: 19.03.2024

  1. Öffnen Sie ein neues Layoutfenster, indem Sie auf die Schaltfläche Neues Layout auf der Symbolleiste Standard klicken. Sie können mit der rechten Maustaste auf die Fenstertitelleiste klicken und Eigenschaften auswählen, um sie Ihrem Wunsch gemäß umzubenennen.
  2. Wählen Sie Einfügen: OLE-Objekt im Menü.
  3. Wählen Sie Aus Datei erstellen und dann die Datei, indem Sie auf die Schaltfläche Browser klicken.
  4. Klicken Sie auf OK, um das Dokument als ein Objekt in das Layoutfenster einzufügen.
