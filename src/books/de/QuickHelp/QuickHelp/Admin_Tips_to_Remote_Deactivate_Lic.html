FAQ-903 Ich bin der Administrator einer großen Organisation. Wenn meine Anwender eine Origin-Lizenz anfragen, ist es schwer zu sagen, wer Origin verwendet. Gibt es eine Möglichkeit, Lizenzen an/von Anwendern zu vergeben/zurückzuholen?

Admin-Tips-to-Remote-deactivate-Lic

Letztes Update: 12.11.2017

Ein Vorteil der Softwarepflege ist, dass Sie die Option der “Ferndeaktivierung”, um den Produktschlüssel/die Lizenz sofort zu vergeben. Das alte Origin wird deaktiviert, und ein neuer Anwender kann den Produktschlüssel verwenden und sofort eine Lizenz erhalten. Hinweis: Dazu muss Origin auf das Internet Zugriff haben, um die Gültigkeit der Lizenz zu überprüfen.

Lesen Sie den zweiten Tipp auf dieser Seite -> Im Abschnitt "Wenn Sie der Administrator der Software sind" finden Sie detaillierte Schritte zur Ferndeaktivierung eines Produktschlüssels.

Schlüsselwörter:Lizenzieren, node-locked, Gruppenlizenz, Neuerungen in 2018, Schlüssel deaktivieren
