FAQ-488 Wie erhalte ich die Polygonfläche innerhalb einer Konturlinie?

Polygon-Area-Within-Contour-Lines

Letztes Update: 11.05.2016


Um die Polygonfläche innerhalb einer einzelnen Konturlinie schnell zu erhalten, können Sie sie im Diagramm auswählen und die Fläche im Fenster Datenanzeige ablesen.

Klicken Sie dreimal auf eine einzelne Konturlinie, um sie auszuwählen:

  1. Der erste Klick wählt alle Konturen in dem Layer aus.
  2. Mit dem zweiten Klick werden alle Konturlinien auf der gleichen Z-Ebene ausgewählt.
  3. Der dritte Klick wählt nur eine einzelne Konturlinie aus.

Wenn eine einzelne Konturlinie ausgewählt wird, können Sie die Polygonfläche ablesen, die von der Konturlinie eingeschlossen ist. Beachten Sie, dass der Wert Z für die Ebene auch im Fenster Datenanzeige mitsamt der Polygonfläche gezeigt wird. Sie können mit der rechten Maustaste auf das Fenster Datenanzeige klicken und Text kopieren auswählen, um die Werte zu kopieren und an anderer Stelle einzufügen.

Seit Origin 2016 SR1 können Sie mit der rechten Maustaste auf Ihre Auswahl (alle, gegebener Z-Wert, einzelne Kontur) klicken und die Option Konturlinien extrahieren wählen. Daten, einschließlich Polygonfläche, werden in einer neuen Arbeitsmappe ausgegeben. Weitere Informationen finden Sie bei diesem Thema.
