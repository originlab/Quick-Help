FAQ-700 Wie kann ich die aktive Aktualisierung der Zeichnungen ausschalten?

Turn-Off-Update-Plot

Letztes Update: 06.02.2015

Wenn Sie Datenwerte in einem Arbeitsblatt/einer Matrix ändern, wird die entsprechende Datenzeichnung standardmäßig automatisch aktualisiert, sollte sie existieren, ohne dass das Diagrammfenster aktiviert werden muss.
Wenn Sie die automatische Aktualisierung der Zeichnung deaktivieren möchten, können Sie das Element im Menü Ansicht: Zeichnungen aktualisieren deaktivieren. Wenn Sie das Projekt erneut gespeichert haben, werden die Einstellungen für Zeichnungen aktualisieren beibehalten.

Schlüsselwörter:Zeichnung aktualisieren, automatisch aktualisieren, deaktivieren
