FAQ-382 Kann ich per Skript auf Arbeitsblattanfragen zugreifen?

WksExtract-Script

Letztes Update: 04.02.2015

Origin besitzt eine X-Funktion wxt, die Sie mit Skript verwenden können. Geben Sie in das Befehlsfenster, das über Fenster: Befehlsfenster geöffnet wurde, ein:

wxt

und drücken Sie dann die Leertaste, um sich von der automatischen Vervollständigung bei den verschiedenen Argumenten für die Funktion helfen zu lassen. Eine weitere sinnvolle Möglichkeit, sich die unterschiedlichen Optionen für die Funktion anzeigen zu lassen, ist die Eingabe von:

wxt -d

Drücken Sie dann Enter und ein Dialogfeld öffnet sich. Sie können einen Bedingungszeichenkette erstellen und an die X-Funktion weitergeben. Bitte sehen Sie hier ein entsprechendes Tutorial.

Schlüsselwörter:Teilmenge, Reduktion
