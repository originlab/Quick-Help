FAQ-922 Wie verhindere ich, dass Origin-Dateien beim Senden per E-Mail beschädigt werden?

Email-incompatible-file-type-corruption

Letztes Update: 28.02.2018

Einige Anwender haben berichtet, dass Origin-Dateien, die als E-Mail-Anhänge versendet wurden, nicht mehr in Origin geöffnet werden konnten. Oder sie erhielten Origin meldete Fehler, die folgendem ähneln:

File corruption: marker not at structure end: 000000000000001F File format error. Incompatible file type or file may be corrupted.

Dieses Problem kann auftreten, weil einige E-Mail-Clients und/oder Virusscanner Origin-Dateien, die als E-Mail-Anhänge verschickt wurden, leicht verändern, so dass in Origin nicht mehr verwendet werden können.

Um dieses Problem zu vermeiden, wird sehr empfohlen, dass Sie Origin-Dateien zippen oder in irgendeiner anderen Form komprimieren, bevor Sie sie in E-Mails verschicken.
