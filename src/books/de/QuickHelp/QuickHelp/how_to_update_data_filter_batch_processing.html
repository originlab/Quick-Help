FAQ-863 Wie aktualisiere ich einen Datenfilter während einer Stapeloperation?

how-to-update-data-filter-batch-processing

Letztes Update: 27.03.2017

Mit Origins Stapelverarbeitung können Anwender eine vorhandene Arbeitsmappe oder gespeicherte Analysevorlagen verwenden, um Analysen in Stapelverarbeitung durchzuführen. Wenn es einen Datenfilter im Arbeitsblatt oder in der Analysevorlage gibt, ist das automatische Aktualisieren von Filtern besonders wichtig.

Bis zur Version Origin 2017 SR1 mussten Sie die folgende Zeile des LabTalk-Skripts im Feld Skript vor jedem Prozess im Hilfsmittel Stapelverarbeitung (Datei: Stapelverarbeitung) einfügen, um den Filter auf neue Daten erneut anzuwenden:

wks.runfilter();

Ab Origin 2017 SR1 ist nicht mehr notwendig. Jegliche Filter, die zur Analysevorlage oder zum Arbeitsblatt hinzugefügt werden, werden automatisch bei Datenänderung im Dialog Stapelverarbeitung aktualisiert.

Schlüsselwörter:Stapelverarbeitung, Arbeitsblatt, Zeilen verbergen, importieren, erneut importieren
