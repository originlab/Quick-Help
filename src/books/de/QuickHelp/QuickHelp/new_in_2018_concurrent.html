FAQ-904 Welche Änderungen gibt es für Netzwerklizenzen der Version 2018?

New-in-2018-Concurrent

Letztes Update: 12.11.2017

Wir nehmen das Datum der Softwarepflege, um zu prüfen, ob Sie berechtigt sind, die neue Version zu erhalten, so dass Sie die neue Lizenz reibungslos anwenden können. Wenn auf einen Computer Origin 2017 installiert ist und er vorher mit einem Server verbunden war, verwendet Origin 2018 auf dem gleichen Computer die Serverinformationen von 2017, um eine Lizenz zu beziehen, ohne dass die Serverinformationen manuell eingegeben werden müssen. Ansonsten gibt es keine Änderungen bei der Verteilung/Installation etc.

Schlüsselwörter:Lizenzierung, Concurrent, Netzwerk, Neuerungen in 2018
