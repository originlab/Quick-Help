FAQ-787 Wenn ich versuche, Origin online zu aktivieren, wird mir 'Lizenz eingeben' angezeigt, es passiert aber nichts.

Default-Web-Browser

Letztes Update: 03.01.2019

Entweder haben Sie keinen Standard-Webbrowser festgelegt oder Ihre Dateiverbindung zum Öffnen Ihres Standard-Webbrowsers ist auf Ihrem Computer nicht korrekt eingerichtet.

Um den Standard-Webbrowser festzulegen:

Wählen Sie aus den untenstehenden Webbrowsern denjenigen aus, den Sie als Ihren Standard-Webbrowser verwenden möchten (dies sind am meisten verwendeten) und befolgen Sie die Anweisungen, um ihn als Standard einzurichten:

  • Microsoft Internet Explorer: Öffnen Sie die Internetoptionen, entweder von der Systemsteuerung aus oder mit Hilfe des Menüs Extras | Internetoptionen im Internet Explorer. Klicken Sie auf die Registerkarte Programme, dann auf die Schaltfläche "Als Standard" und versuchen Sie erneut, Origin zu aktivieren.
  • Mozilla FireFox: Im Menü Einstellungen | Allgemein. Klicken Sie auf Allgemein und deaktivieren Sie unter Start das Kontrollkästchen "Immer überprüfen, ob Firefox Standardbrowser ist" und versuchen Sie erneut, Origin zu aktivieren.
  • Google Chrome: Klicken Sie auf die drei horizontalen Balken oben rechts von Chrome, um das Menü zu öffnen, und wählen Sie Einstellungen. Scrollen Sie nach unten. Klicken Sie im Standardbrowser auf "Chrome als Standardbrowser festlegen" und versuchen Sie erneut, Origin zu aktivieren.
Um die Dateiverbindung festzulegen:

Wenn Sie feststellen, dass das Problem weiterhin besteht, nachdem Sie sichergestellt haben, dass der Standard-Webbrowser festgelegt ist, muss die Dateiverbindung zum Öffnen Ihres Standardbrowsers korrigiert werden.

Windows Vista, 7, 8, 8.1 und 10

  1. Öffnen Sie die Systemsteuerung und klicken Sie auf Standardprogramme.
  2. Klicken Sie auf Dateityp oder Protokoll einem Programm zuordnen.
  3. Scrollen Sie nach unten zu .htm und .html. Stellen Sie sicher, dass die Spalte Aktueller Standard auf Ihren Standardbrowser gesetzt ist.
  4. Sollte das nicht der Fall sein, markieren Sie die Zeile und klicken Sie auf Programm ändern. Wählen Sie Ihren Standardbrowser für sowohl .htm als auch .html. Scrollen Sie dann nach unten, wo Sie den Abschnitt Protokolle sehen. Stellen Sie sicher, dass FTP, HTTP und HTTPS auf Ihren Standardbrowser eingestellt sind.
  5. Versuchen Sie erneut Origin zu aktivieren.
