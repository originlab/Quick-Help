FAQ-755 Wie lege ich das Anzeigeformat der Einheiten für die Achsentitel global fest?

Global-Unit

Letztes Update: 29.03.2021

Wählen Sie im Hauptmenü Einstellungen: Optionen, um den Dialog Optionen zu öffnen. Gehen Sie zur Registerkarte Achsen, wählen Sie ein einheitliches Anzeigeformat, indem Sie auf die Schaltfläche neben dem Bearbeitungsfeld Einheitsanzeige für Achsentitel klicken und den gewünschten Stil im Ausklappmenü auswählen.

Beachten Sie, dass Sie auch die Notation benutzerdefiniert anpassen können, indem Sie die angezeigte Zeichenkette bearbeiten. Wenn Sie zum Beispiel die Einheit wie in Einheit anzeigen möchten, können Sie in <Einheit> in dem Bearbeitungsfeld eingeben.

Einzelheiten finden Sie auf dieser Seite.

Origin-Version mind. erforderlich: 2016 SR0

Schlüsselwörter:Einheit, Anzeigestil
