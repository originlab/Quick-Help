FAQ-788 Wie übertrage ich Origin auf einen anderen Computer?

System-Transfer

Letztes Update: 02.02.2023


Die Origin-Lizenz muss zuerst auf dem ursprünglichen Computer deaktiviert werden. Dann kann die Lizenz auf dem neuen Computer unter Verwendung des gleichen Produktschlüssels wieder aktiviert werden.

Diese Seite ist für Node-Locked-Lizenzen. Sollten Sie eine Concurrent-Netzwerklizenz haben (xFxxx-xxxx-79xxxxx), lesen Sie bitte: FAQ-39 Wie kann ich meinen Lizenzserver für einen neuen Rechner einrichten?

Haben Sie Zugriff auf den Computer, auf dem Origin installiert ist?

Ja, ich habe Zugriff auf den Computer.

  1. Starten Sie Origin.
  2. Verwenden Sie das Menü Hilfe -> Lizenz deaktivieren ....

Nein, ich habe keinen Zugriff auf den Computer.

Rechnergebundene Einzellizenz (Node-Locked)

Wenn Sie eine Seriennummer für eine Einzellizenz verwenden (xFxxx-xxxx-70/71/72/73xxxxx)...

  1. Öffnen Sie eine Webbrowser auf der Seite http://www.originlab.com/reginfo und melden Sie sich an,
  2. Klicken Sie auf die Seriennummer.
  3. Unter der Deaktivierungsspalte sehen Sie Ihren Produktschlüssel und den Link "Go" ("Los").
    Falls Sie über eine gültige Softwarepflege verfügen, klicken Sie auf diesen Link, um die Deaktivierung durchzuführen.
    Falls Ihre Softwarepflege abgelaufen ist, werden Sie durch Klicken auf "GO" zum Formular Anfrageformular für Systemübertragung weitergeleitet. Füllen Sie das Formular bitte aus, lesen Sie die Policy zum Lizenzierungs- und Installationssupport und senden Sie das Formular ein.

Rechnergebundene Gruppenlizenz (Node-Locked)

Wenn Sie eine Seriennummer für eine Gruppenlizenz verwenden (xFxxx-xxxx-76/78xxxxx)...

  • Wenn Sie kein Administrator sind, wenden Sie sich bitte an den Origin-Lizenzadministrator in Ihrer Organisation, um Hilfe bei der Übertragung zu erhalten.
  • Wenn Sie der Administrator sind:
  1. Öffnen Sie eine Webbrowser auf der Seite http://www.originlab.com/reginfo und melden Sie sich an,
  2. Klicken Sie auf die Seriennummer.
  3. Klicken Sie auf der aufgerufenen Seite auf den Link “Alle Produktschlüssel”.
  4. Unter der Deaktivierungsspalte sehen Sie Ihre Liste mit den Produktschlüsseln und den Link "Go" ("Los") für jeden aktivierten Schlüssel.
    Sie können Filter auf diese Liste anwenden, um einen bestimmten Benutzer oder Produktschlüssel zu finden.
  5. Falls Sie über eine gültige Softwarepflege verfügen, klicken Sie auf diesen Link "Go", um die Deaktivierung durchzuführen.
    Falls Ihre Softwarepflege abgelaufen ist, werden Sie durch Klicken auf "GO" zum Formular Anfrageformular für Systemübertragung weitergeleitet. Füllen Sie das Formular bitte aus, lesen Sie die Policy zum Lizenzierungs- und Installationssupport und senden Sie das Formular ein.
