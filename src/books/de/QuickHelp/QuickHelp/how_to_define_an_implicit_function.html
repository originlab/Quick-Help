FAQ-265 Wie definiere ich eine implizite Funktion?

Implicit-Function

Letztes Update: 04.02.2015

Wählen Sie Hilfsmittel: Fitfunktionen erstellen, um den Dialog Fitfunktionen erstellen zu öffnen. Wählen Sie dann auf der Seite Name und Typ die Option Implizit für das Funktionsmodell. Danach können Sie eine implizite Funktion mit diesem Assistenten definieren.

Schlüsselwörter:implizite Funktion, Fit, Anpassung

Origin-Version mind. erforderlich: 9.0 SR0
