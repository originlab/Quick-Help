FAQ-1075 Wie lade ich Python-Pakete standardmäßig von einem Mirror-Repository herunter?

Download-Python-Package-from-MirrorRepository

Letztes Update: 11.11.2020

Wenn Sie ein Python-Paket herunterladen und installieren und dabei:

  • LabTalk-Befehl pip install SomePackage
  • Menü Konnektivität: Python-Pakete > Schaltfläche Installieren

verwenden, können Sie den Schritten unten folgen, um das Standard-Mirror-Repository zu wechseln.

  1. Suchen Sie die Datei pip.ini (auch wenn sie versteckt ist) unter: C:\ProgramData\pip\pip.ini
    Falls keine existiert, erstellen Sie selbst eine.
  2. Öffnen Sie pip.ini und konfigurieren Sie das Mirror-Repository folgendermaßen: 
    [global] index-url = https://mirrors.aliyun.com/pypi/simple
  3. Speichern Sie die ini-Datei.
