FAQ-261 Wie kann ich die NAG-Bibliothek in meiner benutzerdefinierten Fit-Gleichung aufrufen?

NAGLibrary-in-FitttingFunction

Letztes Update: 04.02.2015

Neue Funktionen werden durch Auswahl von Hilfsmittel: Fitfunktionen verwalten oder Hilfsmittel: Fitfunktionen erstellen und die Schaltfläche Neue Funktion definiert. Sie können eine Funktion im Code Builder definieren und bearbeiten, wenn Funktionsform = Origin C. Wenn Sie auf die Schaltfläche auf der rechten Seite des Feldes zur Funktionsbearbeitung klicken, öffnet sich ein spezielle Instanz des Arbeitsbereichs des Code Builders. Bearbeiten Sie dieses Fenster direkt, scrollen Sie ganz nach oben und geben Sie #include directives für jede NAG-Headerdatei, die Sie benötigen, ein.
