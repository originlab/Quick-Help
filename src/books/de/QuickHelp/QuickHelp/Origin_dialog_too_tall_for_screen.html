FAQ-955 Was tue ich, wenn ein Dialog für meinen Bildschirm zu groß ist?

Origin-dialog-too-tall-for-screen

-

Letztes Update: 06.03.2019

Auf einigen Computern mit geringerer Auflösung (z. B. 1920 X 1080) kann es vorkommen, dass ein Dialogfeld zu groß ist, so dass es nicht in seiner Gesamtheit auf einem Monitor angezeigt werden kann.

Dialogfeld Details Zeichnung

Falls der Dialog zu groß ist, um auf die Schaltflächen unten zu klicken, etwas, das beim Dialog Details Zeichnung häufig vorkommt, führen Sie bitte ein Upgrade Ihres Origin auf Version 2019 oder höher durch.
Wir haben die Dialoggröße in Origin 2019 verbessert. Im Moment wird der Dialog automatisch auf die Größe gebracht, die der kleinstmöglichen Größe der aktiven Registerkarte entspricht. Falls die aktive Registerkarte zu groß ist, können Sie zu einer anderen Registerkarte mit weniger Optionen wechseln wie der Registerkarte Ankerlinien. Der Dialog wird automatisch "verkleinert", so dass Sie auf “OK” oder “Anwenden” klicken können.

Erweiterbare Dialoge wie "Diagrammeinstellungen"

Falls der Dialog mehrere Bedienfelder hat und das Erweitern aller Bedienfelder den Dialog zu groß macht, wie die Dialoge Nichtlinearer Kurvenfit und Diagrammeinstellungen, minimieren Sie bitte ein oder zwei Bedienfelder, so dass Sie die unteren Schaltflächen bedienen können, oder passen Sie die Größe des Dialogs durch Ziehen des Cursors an.
Lesen Sie diese Schnellhilfe, um zu erfahren, wie Sie die Größe von Dialogen mit mehreren Feldern verändern.
