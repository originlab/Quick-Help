FAQ-794 Wie importiere ich Daten aus einer Datenbank?

DB-Import-Date

Letztes Update: 03.01.2019

Wenn Sie Datenspalten aus einer Datenbank importieren, ist möglicherweise das Datumsformat verloren gegangen. Um das Datumsformat beim Importieren beizubehalten:

  1. Wählen Sie Einstellungen: Optionen, um den Dialog Optionen zu öffnen.
  2. Gehen Sie zur Registerkarte Numerisches Format und wählen Sie das geeignete Format in der Auswahlliste Datumsformat für Datenbankimport. Sie können auch das benutzerdefinierte Datumsformat direkt im Bearbeitungsfeld eingeben.

Schlüsselwörter:Importieren, Datumsformat, ODCB, Datenbank, Oracle, MySql, SQL
