FAQ-1153 Das Skriptfenster wird geöffnet und zeigt die Meldung "err saving <object>" an.

file-err-saving

Letztes Update: 26.09.2022

Problem:
Sie versuchen eine Projektdatei zu speichern und erhalten eine Warnmeldung, die besagt: "Dateifehler beim Speichern in <Pfad>. Datei nicht gespeichert. Prüfen Sie, um sicherzustellen, dass die Festplatte nicht voll ist."

Gleichzeitig wird das Skriptfenster geöffnet und zeigt einen Fehler wie den folgenden an:


err saving #1781A405628YLNx:3

Lösung:
Sie müssen das Objekt, das in der Fehlermeldung im Skripfenster genannt wird, löschen. Der Objektname schließt das Zeichen # ein, aber nicht das Zeichen :. Für die obige Beispielfehlermeldung müssten Sie den folgenden Befehl im Skriptfenster eingeben und Enter drücken:

del #1781A405628YLNx

Schlüsselwörter:speichern, Datei nicht gespeichert, Fehler, Festplatte voll
