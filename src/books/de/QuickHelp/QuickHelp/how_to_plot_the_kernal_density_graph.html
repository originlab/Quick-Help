FAQ-330 Wie zeichne ich das Kernel-Dichtediagramm?

kernel_density_graph

Letztes Update: 26.04.2023


Für das 1D-Kernel-Dichte-Diagramm

  1. Erstellen Sie zuerst ein Histogramm (im Menü Zeichnen: Statistisch: Histogramm).
  2. Klicken Sie zum Öffnen des Dialogs Details Zeichnung doppelt auf das Diagramm.
  3. Stellen Sie sicher, dass das Diagramm auf der linken Seite des Dialogs ausgewählt ist. Sie sehen dann auf der rechten Seite die Registerkarte Verteilung. Wählen Sie Kernel-Glättung in der Auswahlliste Verteilungskurventyp.
  4. Gehen Sie zur Registerkarte Daten und aktivieren Sie Einteilungen verbergen.


Für das 2D-Kernel-Dichtediagramm

  • Wählen Sie im Menü Zeichnen: Kontur: 2D-Kernel-Dichte.
