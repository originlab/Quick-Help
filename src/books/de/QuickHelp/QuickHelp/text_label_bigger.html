FAQ-486 Was soll ich tun, wenn die Textgröße in einem Arbeitsblatt, das in Origin 8.5 erstellt wurde, beim Öffnen mit einer höheren Version viel größer angezeigt wird?

text- label-bigger

Letztes Update: 04.02.2015

Wenn Sie in Origin 8.5 ein Arbeitsblatt erstellt haben, das Text enthält, und Sie dieses Arbeitsblatt mit einer höheren Version von Origin öffnen möchten, kann es vorkommen, dass der Text viel größer angezeigt wird als in Origin 8.5.

Um dieses Problem zu lösen, können Sie die jeweilige OPJ/OTW-Datei öffnen und auf Fenster: Skriptfenster klicken. Fügen Sie folgendes Skript in das Skriptfenster ein:

doc -e w { page.textsizecorr=1; doc -uw; }

Wählen Sie das Skript im Skriptfenster und drücken Sie Enter. Bitte beachten Sie, dass diese Lösung nur für Origin 8.6 und höhere Versionen funktioniert.
