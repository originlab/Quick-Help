2 单用户单机版(Node-Locked)许可证

Single-User-Node-Locked-Lic

如果您的序列号如下，那么您正在使用的是单用户单机版许可证：

xFxxx-6xxx-70/71/72xxxxx.

如果您的序列号是:

xxxxx-2xxx-79xxxxx, 或
xxxxx-3xxx-79xxxxx,

那么您使用的是网络版(Concurrent Network) 许可证，以下信息则不适用于您的情况。

单用户单机版允许您在 (1) 台计算机上安装 Origin。您需要在 OriginLab 官网上为该台计算机获取一个许可证，即工作环境许可证(Office License)。这个许可证是该计算机专用的。在您的金牌服务期(Maintenance)内，除了一个工作环境许可证外，您还可以为您的笔记本电脑或家用电脑获取 (1) 个家庭环境许可证(Home-Use)。这个许可证与金牌服务一同生效，只要您续订我们的金牌服务，您就可以持续使用该许可证。

如果您需要更换计算机，则需要先解除当前计算机上的许可证，然后才能为新计算机获取新许可证。

激活许可证

重要提示！

  • Origin 许可证系统从 2017 版开始全新升级，单机版许可证的激活不再需要计算机的管理员权限。
  • 一旦成功激活许可证，这个许可证将保存到当前计算机中，此后不再需要访问互联网。

获取产品密匙

如果您是新购买我们产品的客户
或，
如果您是在金牌服务期内从 Origin2016 或更早的版本升级到 Origin2017 或更高的版本的老客户

  1. 您将会收到一封含有 Origin 序列号和注册链接的电子邮件，
    或
    一份许可证书(PDF文档或纸质文件)，包含 Origin 序列号和注册码，
    • 如果您收到的是注册链接
      1. 点击注册链接将打开 OriginLab 网站。登录。如果您还没有 OriginLab 网站的帐户，请先自行创建帐户。
      2. 登录后，您的序列号和注册码信息将显示在网页上。点击注册(Register)按钮注册此序列号并生成您的产品密钥。

  2. 在 Origin Product Registration|License request history 页面上将会显示 (2) 个产品密钥：(1) 个为工作环境许可证(Office License)，(1) 个为家庭环境许可证(Home-Use License)。

您可以随时登陆 OriginLab 官网来查看产品密匙。登录后，点击 “Manage My License(s)” 链接，再点击相应的序列号链接。在 Origin Product Registration|License request history 页面上会列出该序列号的产品密匙及其使用情况。


如果您是在金牌服务期内从 Origin2017 或以后版本升级到最新版本的老客户

从 Origin2018 开始，不同的版本使用相同的产品密匙。所以您无需获取新版本的产品密匙。请参照这一小节-> "如果有，请按照以下步骤升级到新的版本"，了解升级的详细步骤。

使用产品密匙激活 Origin

  1. 打开 Origin，许可证激活对话框将自动弹出。
  2. 选择 在下面输入产品密钥，然后单击激活，输入产品密匙，然后点击激活。
  3. 激活过程将在后台完成。如果成功，您将收到成功激活的信息。

如果在后台的激活过程失败，请继续以步骤：

  1. 如果激活过程因为一些原因失败（例如，没有网络连接，originlab.com 被防火墙阻止访问等），一个带有 URL 链接的记事本窗口会自动打开。
    如果记事本没能正常打开，这很有可能是本机的防火墙导致的。若碰到这种情况，请您在系统的网络设置中强制切断网络或者直接拔掉网线。当网络被彻底断开之后，点击激活按钮就能打开带有链接的记事本。
  2. 在网络链接正常的情况下，将此链接复制到浏览器。该链接将会打开 OriginLab 的网页。
  3. 许可证文本将生成并显示在网页上。
  4. 复制许可证文本，并粘贴到 Origin 的许可证激活对话框。点击 确定。

停用许可证

许可证被成功激活后，产品密匙会被标记为“used”，且不能再被其他计算机使用。

如果您想将激活了的 Origin 软件从一台计算机转移到另一台计算机上，您需要先停用原来计算机上的许可证，然后才能在新的计算机上使用相同的产品密匙来激活。

如果原计算机上的 Origin 仍能正常运行

  1. 启动 Origin。选择菜单 帮助: 停用许可证，打开停用许可证对话框。
  2. 点击 立即停用 按钮。
  3. 许可证停用过程将在后台自动完成。 如果成功，您会收到成功停用的消息

如果后台停用许可证失败，请继续以下步骤：

  1. 如果停用许可证进程因为一些原因失败（例如，没有网络连接，originlab.com 被防火墙阻止访问等），一个带有 URL 链接的记事本窗口会自动打开。
  2. 在网络链接正常的情况下，将此链接复制到浏览器。该链接会打开 OriginLab 的网站。
  3. 该网页上会显示您的许可证信息。记下停用编码 Deactivation Code (DC)。
  4. 返回 Origin。在许可证停用对话框底部的 停用编码 文本框中输入停用编码。点击 停用许可证 按钮。
  5. 另一个带有 URL 链接的记事本窗口自动打开。在网络链接正常的情况下，将此链接复制到浏览器。
  6. 在网页上点击 Continue 按钮。当您看到成功的消息时，则表示这个许可证已经被成功释放，可以使用在其他计算机上了。

如果原来的电脑上的Origin已经无法正常运行，或者原电脑已报废

  1. 登录官网 www.originlab.com/reginfo。
  2. 点击您想要停用许可证的序列号。在打开的网页上，点击表格中 Deactivation 一列中的 "Go" 链接。
  3. 在 Deactivate Origin from a device 页面上，核对信息无误后，点击 Continue 停用许可证。若操作成功，您将会收到一个成功信息，该产品密匙可在新电脑上重新使用。
  4. 若操作失败，网页会自动跳转到 System Transfer Request Form 页面。请按要求填写表格并点击 Submit 提交移机申请。

FAQ

  • 在开始使用新版本之前是否需要停用旧的许可证？

金牌服务续约后更新您的许可证

从 Origin2018 开始，不同版本共享同一许可证，这意味着新版本会沿用之前版本所有的许可证信息，包括产品密匙及其使用情况。这使得升级更简单。

当您的金牌服务续费完成后，请按照以下步骤更新您的许可证。

  1. 打开 OriginLab 官网(www.OriginLab.com)。登录。
  2. 在 Welcome 页面，点击 "Manage my License(s)" 链接。
  3. 在 Manage my License(s) 页面，查看序列号表格中的 Version 列，看看是否有新版本可以升级。
如果有，请按照以下步骤升级到新的版本
  • 如果您想要在已经安装了旧版 Origin 的计算机上安装新版本，
    1. 从 OriginLab 网站下载新的版本，安装。
    2. 打开 Origin。许可证会在后台自动激活。如果成功，你将收到成功激活的消息。如果失败，一个带有 URL 链接的记事本窗口会自动打开。把 URL 复制到浏览器手动激活 Origin。
  • 如果您想要在新的计算机上安装新版本，
    • 如果您的当前版本还有尚未使用的产品密匙，在新的计算机上安装新版 Origin 后，您可以用尚未使用的密匙激活它。详细步骤见使用产品密匙激活Origin小节。
    • 如果当前版本所有的产品密匙均已被使用，您需要先停用至少一个许可证，然后才能重新使用该产品密匙去激活新的计算机上的新版 Origin。详细步骤见停用许可证小节。
如果没有新的版本，这意味着您已经注册了最新的版本。您可以按照以下步骤更新您的许可证。
    1. 打开Origin。选择菜单 帮助: 激活许可证。
    2. 在许可证激活对话框中，产品密匙已经预填好。点击激活按钮。
    3. 激活完成后，选择 帮助: 关于Origin，检查金牌服务截止日期是否已经更新。
