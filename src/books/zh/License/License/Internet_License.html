7 浮动单机版许可证（Internet License）

Internet-Lic

浮动单机版许可证 与 网络版许可证 类似，它也可以在任意数量的计算机上安装 Origin，但能够同时运行 Origin 的计算机的数量必须与许可证的购买数量一致。两者的关键区别在于，使用浮动单机版许可证，您无需自己搭建和维护许可证服务器。

例如，如果您购买了 n 个许可证，则您将获得 n 个产品密钥。您可以使用这些产品密钥在任意计算机上安装和激活 Origin，但是同一时刻最多只能有 n 台计算机上的 Origin 是成功激活并可以正常运行的。当您使用同一产品密钥在另一台计算机上激活 Origin 的时候，前一台计算机上的许可证将自动停用。

该许可证要求计算机必须连接网络，但是同时也提供了一个选项，可以选择将产品密钥锁定到该计算机上，为期7天，这使得您在不方便联网的情况下，例如出差或旅行时，也可以不间断地使用 Origin。

重要提示！！！

此类型许可证要求计算机必须始终保持联网状态，请确保您的网络具有较稳定的互联网连接。

注册序列号并获取产品密钥

如果您是单机版用户，即您的序列号为:

xFxxx-6xxx-70/71/72xxxxx

请参见此页面上的步骤来注册序列号并生成产品密钥。



如果您是多用户单机版的管理人员，即您管理的序列号为r::

xFxxx-6xxx-76xxxxx

请参见此页面上的步骤来注册序列号并设置好产品密钥获取方式，再将激活 Origin 所需的信息（序列号，获取产品密钥的方法等等）告知最终用户。

激活许可证

  1. 打开 Origin，许可证激活对话框将自动弹出。
  2. 选择 在下面输入产品密钥，然后单击激活，输入产品密匙，然后点击激活。
  3. 激活过程将在后台完成。如果成功，您将收到成功激活的信息。

如果在后台的激活过程失败，请继续以步骤：

  1. 如果激活过程因为一些原因失败（例如，没有网络连接，originlab.com 被防火墙阻止访问等），一个带有 URL 链接的记事本窗口会自动打开。
    如果记事本没能正常打开，这很有可能是本机的防火墙导致的。若碰到这种情况，请您在系统的网络设置中强制切断网络或者直接拔掉网线。当网络被彻底断开之后，点击激活按钮就能打开带有链接的记事本。
  2. 在网络链接正常的情况下，将此链接复制到浏览器。该链接将会打开 OriginLab 的网页。
  3. 许可证文本将生成并显示在网页上。
  4. 复制许可证文本，并粘贴到 Origin 的许可证激活对话框。点击 确定。

停用许可证

无需停用许可证，当您使用同一产品密钥在另一台计算机上激活了 Origin，原始计算机上的 Origin 就会自行停用。

断开 Origin 的网络连接

您可以在有限的一段时间内（最长一周）将许可证锁定到您的计算机上。在此期间，您可以断开计算机与互联网的连接，而 Origin 仍然可以正常运行。

注意：请务必在断开计算机的网络连接之前，执行以下操作锁定许可证。

  1. 打开 Origin，选择 帮助：关于Origin。
  2. 单击 许可证 按钮。
  3. 一般而言，产品密钥一栏会自动填写。如果没有，请手动输入您的产品密钥。
  4. 选择 将产品密钥锁定到此计算机... 复选框。
  5. 单击 激活 按钮。
  6. 现在，您可以断开计算机与网络的连接了。
