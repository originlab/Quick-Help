统计分析 Apps

从Origin 2016 开始， Origin引入了 Apps 的概念. 许多新的统计分析工具作为 Apps 发布在 OriginLab 的File Exchange 网页， OriginLab 也仍在不断地创建和发布新的 App。

有一部分 Apps 会预安装在 Origin 中。截至 2019 年秋季， 所有的 Apps 都可以免费下载使用。您可以选择菜单 统计: 查找App 来打开 App Center。这样打开的对话框中仅显示与统计相关的 App。 浏览可用的 App，然后单击 Download and Install 按钮 以安装该 App。

现在Origin 中预安装了一个名为 统计向导 的 App。此应用程序使用交互式“向导”界面来帮助您选择适当的统计测试，并告诉您在Origin软件中找到它们的位置，或者 - 如果测试可从其他附加应用程序获得 - 在the OriginLab File Exchange中的哪里可以找到它。

对选定数据进行统计分析

Origin提供了简单快捷的方法来帮忙用户得到工作簿或者图上选定数据点的简单统计量

状态栏

统计, 选择数据 在工作簿中选择一部分数据。您可以选择一个或者多个列中的数据，甚至可以使用CTRL键来选择非连续的数据 统计, 状态栏

这些选定数据的基本统计量(平均值, 求和, 计数) 将会显示在状态栏上. 鼠标右键点击状态栏， 您还可以进一步设置希望在状态栏上显示什么数据.

命令窗口和脚本窗口

  1. 在工作簿中选择部分数据.
  2. 从窗口菜单中打开命令窗口和脚本窗口, 键入下面的命令行, 按下ENTER键执行命令:
stats
moments
quantiles

在命令窗口, 对应结果会在命令行下输出. 在脚本窗口, 您还需要输入下面的命令来得到结果:

stats;
stats.=;


注: 想了解更多的信息，请浏览我们的帮助文档:

帮助: 编程: X-Functions > X-Function Reference > Statistics
