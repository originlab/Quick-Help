Origin的工作区域

Origin工作区域

这个章节我们将简单介绍一下Origin的工作区域。Origin的老用户可以跳过这一章节。

窗口大小和位置会自动调整以适应不同的显示。 这样您就可以更轻松地在笔记本电脑和更高分辨率的显示器之间切换，而无需对每个窗口和对话框进行调整大小/重新定位。同时添加了一个系统变量来控制重新缩放行为：@SRWS (0 = 以前的行为；1 = 适合宽度，缩放 y；2 = 适合高度，缩放 x；3（默认）= 适合宽度和高度）。

Windows 文件资源管理器预览

虽然不是 Origin 用户界面的一部分, 但您可以使用 Windows 文件资源管理器 (以前是 "windows 资源管理器") 预览项目文件中包含图形.

  • Windows 资源管理器的大或超大图标中可以显示文件中最后激活的图形的预览图片 (在文件资源管理器中右键单击, 然后选择 查看: 大图标或超大图标或从文件管理器查看菜单中选择")
  • 在预览窗口(ALT + P) 可以显示更大的图形预览图片, 并利用其垂直滚动条，以浏览OPJU项目文件中所有图形的图片.
  • 如果您不希望将图形预览保存到文件中（在图标视图中或在资源管理器预览窗格中），则可以设置LabTalk系统变量@PESS=0或者=-1。 有关更改系统变量的值的信息，请参见FAQ-708如何永久更改系统变量的值？

Origin的菜单和菜单命令

菜单

  • 不同使用环境下的菜单栏和菜单：菜单栏和菜单命令会根据使用环境变化；它们会根据窗口类型变化而变换（比如，从工作表转换到图形再到矩阵）。只有跟活动窗口匹配的菜单会显示。
  • 菜单位图，工具栏按钮和快捷键：一些菜单命令带有位图，显示在菜单命令的左侧。另外，你可能会在菜单右侧看到一个热键组合。按键式位图和热键表明您可以用不同的方法来打开该功能。
  • 快捷菜单命令：很多命令可从快捷菜单打开。如果想要打开快捷菜单，您需要右击一个Origin对象（比如工作表窗口，图的轴。文本对象等）。当然，也只有与当前对象相关联的命令才会显示。
  • 可折叠菜单：可折叠菜单全菜单Origin的主菜单和快捷菜单会默认折叠起来，也就是只有简单的菜单序列会显示。当一个最初“被隐藏”的菜单命令被使用，它便会在折叠菜单可见。如果您想要总是看到所有的菜单项，点击设置：选项 > 其他选项卡，清除启用折叠菜单项。
  • 最近使用的分析菜单命令：最近使用的菜单命令会显示在分析菜单（比如，工作表，分析，统计，图像等）的底部。这样您便可以快速的打开工具进行重复性操作。
  • 最近使用的文件: 多个 文件 子菜单 (例如 文件: 最近使用的项目) 列出了你最近打开或保存的文件。因此很容易找到或打开你当前使用的一些文件。
  • 自定义菜单管理器：从Origin 8.1 开始，用户可以使用自定义菜单管理器菜单, 管理器去添加或者移除菜单项。如果想了解更多，请查看这个章节定制Origin。

所有Origin文件操作都记录于一个文本日志文件。文件名为"OriginProjHistory.txt"，保存在用户文件夹(帮助: 打开文件夹: 用户文件夹)。日志最大条目数由系统变量@MHF控制（默认值为1000）。

Origin工具栏

工具栏, 自定义

使用自定义工具栏对话框（查看:工具栏菜单）控制工具栏的显示和重新初始化工具栏。

  • 定制工具栏按钮的外观：使用选项选项卡去调整按钮和工具栏的外观特性。
  • 靠边/悬浮工具栏：工具栏可悬浮于工作区域内。将鼠标光标停留在工具栏的左端，当光标变为一个四箭头，拖动工具栏离开工具栏区域；双击悬浮状态的工具栏，它将会回置到之前停靠的位置。
  • 隐藏/显示工具栏：点击浮动工具栏的关闭按钮，工具栏会被隐藏。如需恢复，需打开自定义工具栏对话框，在工具栏选项页，在选定工具栏的左边打上勾。
  • 隐藏/显示按钮： 点击工具栏尾部的小箭头，在弹出菜单里打开/关闭当前工具栏中的某些按钮。
  • 移动/隐藏按钮： Alt+拖动可以移动单个按钮：你可以在当前工具栏内调整按钮的位置，也可以将该按钮拖到另外的工具栏，你还可以将按钮拖到工作区域，从而新建一个新的工具栏。当你将按钮拖到Origin子窗口上（如图形窗口，不能在空的工作区），并且×出现时，松开鼠标，该按钮会从工具栏中被移除。

有关定制Origin的标准工具栏，详见定制工具栏和按钮。

浮动工具栏

今后 OriginLab 也将持续增加浮动工具栏。当您选中一个对象或单击页面内的某些关键区域时，对应的工具栏就会出现，其支持的控件取决于选定的对象，窗口类型等等。

  • 在使用浮动工具栏之前，请确保查看：浮动工具栏被勾选（默认）。
  • 选择一个对象 —— 例如一条曲线、一个标签文本或者一个单元格，带有相关按钮的浮动工具栏将会出现。
  • 此外，还有用于页面和层级自定义的工具栏，你可以通过将鼠标悬停在窗口的边缘附近来显示这些工具栏。当鼠标变成 ，则提示可以单击当前点来显示相应的浮动工具栏。
  • 大多数浮动工具栏都有一个属性按钮，该按钮可以访问更复杂的设置（例如，图形窗口的“绘图细节”对话框）。
  • 如果您移开或动作比较慢，浮动工具栏会消失。如需要在浮动工具栏消失后恢复显示，请按SHIFT键。
  • 单击浮动工具栏底部中心的三个点以打开 自定义浮动工具栏 对话框，清除或选中对应的复选框来隐藏或显示工具栏上的任何按钮。

搜索器菜单

在查找栏中输入关键字和短语，以返回相关的菜单项，Apps，帮助文档，视频和X-Functions。打开最近打开过的文件，菜单和Apps。

如何打开 搜索器： (1)（没有打开对话框时）按F1， (2) 或单击Origin工作区左下角的放大镜图标，或者 (3) 在主菜单点击 帮助: 激活开始菜单。

要缩小搜索范围，请在搜索的关键字前键入以下字母:

v = 仅搜索视频
x = 仅搜索X-Functions
p = 搜索您最近打开过的项目
s = 搜索示例项目
e = 搜索菜单 + Apps +（可执行的）X-Functions

或者...

单击搜索栏右边的“齿轮”图标以打开设置对话框。您可以在此处限制搜索范围并调整其他搜索设置。

Learning Center

Learning Center是Origin启动时显示的一个对话框，可以帮助您快速调用绘图和分析示例，还有一些学习材料(教程，视频，OriginLab官方的Blog)。

  1. 选择帮助: Learning Center或者按下F11键可打开它。
  2. 选中或清除在启动时显示复选框，以便在每次启动 Origin 时显示或隐藏此对话框。
  • 浏览和载入绘图示例，包括原始数据和可帮助你重建该图的备注。
  • 在不关闭当前项目的情况下加载分析示例。
  • 在学习资源选项卡浏览教程和视频列表，查看Origin博客，用户论坛和其他Origin社交媒体页面的链接。
  • 转移之前的用户文件夹 到新的用户文件夹。

如需查看新版本的新图形, 打开 Learning Center ， 点击图形示例选项卡，设置绘图过滤器下拉菜单为 所有图型 ， 并查找版本号，例如 2024。

项目管理器

项目管理器 (PE)是帮助你组织Origin项目文件（OPJ）内的各个（子）文件夹和窗口文件。

项目管理器分为两个面板：主面板显示OPJ的文件夹结构；主面板中被选中的文件夹内的窗口文件将列在子面板中。

在2018之前，子面板列出了所选文件夹下的子文件夹。 如果要恢复以前的行为，可以 更改系统变量的值 @PEF = 1。

使用项目管理器管理您的项目

  • 右键单击标题栏，或者点击下拉菜单，选择垂直对齐或水平对齐去叠放或者并列两个面板。
  • 两个面板的右键菜单都提供了删除，隐藏，打印等选项。菜单内选项根据你选中的项目或者右击的区域而变化。
  • 你可以在子面板中进行多选。（按住Shift或者Ctrl键，配合鼠标进行多选。）
  • 子面板支持多种查看模式：详细信息（默认）， 列表，小图标，大图标，超大图标。
  • 在任何查看模式（除了“超大图标”），将鼠标停留在图形窗口图标上可以弹出式预览图形，工作表，矩阵或者布局窗口。注意，窗口内容会在提示信息中被显示。

要禁用PE预览，请设置系统变量 @PESS。

  • 通过在子面板中单击右键并勾选按住 Ctrl 键显示预览图项来控制弹出预览的显示。此后，弹出窗口只会在您按 CTRL 时出现。
  • 当显示窗口文件为超大图标，子面板中会显示图形，布局窗口，工作薄和矩阵薄的缩略图。注意，窗口内容会在提示信息中被显示。
  • 您可以右键单击PE文件夹（上面板）并选择复制--或选择文件夹并执行CTRL + C和CTRL + V --复制并粘贴文件夹结构和内容到（a）当前项目中的另一个文件夹或（b）第二个Origin会话中运行的另一个项目。 要复制单个工作簿窗口，请右键单击窗口（下方面板）并复制，然后将窗口粘贴到（a）同一项目中的另一个文件夹，或（b）到第二个Origin会话中运行的新项目。
  1. 选择帮助：打开文件夹：程序文件夹，点击Samples子文件夹，然后将Tutorial Data.opj拖到Origin工作区域。
  2. 默认状态下，项目管理器是停靠在Origin工作区域的左侧，并自动隐藏。将鼠标停留在项目管理器选项卡上， 它就会展开。
  3. 点击状态栏上的“图钉”按钮将任务管理器固定在当前停靠的位置。
  4. 子面板默认处在详细信息查看模式。 在这种模式下，窗口的元数据比如名称，大小和注释等会按列排列在面板中。右击列标题，你可以删减和添加列。拖动列标题你可以重排这些列。
  5. 右击列标题，显示或者隐藏任意一列，通过勾选或者清除每一项菜单旁边的复选框。
  6. 如需添加注释在注释区域，点击一次， 停顿一下，然后再次点击。这样就会进入编辑模式，然后输入注释内容。
  7. 如需改变窗口的名称，右击对应的行，选择重命名。默认的，名称列显示窗口的短名称；或者如果一个窗口有长名称，则显示窗口的长名称（每个窗口必须有短名称，而长名称则是可选的）。如需显示长名称，在名称格处于编辑模式下点击右侧的SN或者LN按钮去编辑短名称和长名称。如果没有长名称，你可以输入一个。
  8. 在项目管理去的上面板，滚动查找文件夹3D Surface with Point Label然后点击它。子面板会显示文件夹中的3个窗口。注意：工作表eastcali3053用粗体标记，说明它是激活窗口 - 正在执行的窗口以及确定可用操作的窗口。 还有一个Notes窗口和一个名为MBook4的矩阵窗口。
  9. 在子面板，双击MBook4窗口，注意它现在用粗体标识，并有一个玫红的光晕，显示它现在是激活窗口。您也会看到在工作区域，该窗口会整个被光晕标识(光晕的颜色可以用系统变量@AWC控制) -- 另外一个活动窗口的标识。双击这个MBook4按钮，注意这个按钮会变暗。同时需注意，实际的窗口会从工作区域消失。这是因为MBook4现在处于隐藏状态。再次双击MBook4按钮 ，窗口会重载，并被激活。
  10. 保持MBook4处于激活状态，点击绘图 > 3D: 曲面图: 3D颜色映射曲面。这会生成一个3D映射曲面图（关闭提醒窗口）。同时注意，一个新的窗口 -- Graph1 -- 被添加到项目管理器的子面板中。
  11. 右击子面板的空白区域，选择查看: 超大图标。注意， Graph1现在显示一个缩略图。

    12. 再次右击，选择查看: 大图标。注意，窗口现在显示为按钮，但是，当您把鼠标停顿在按钮上，你可以看到在弹出窗口查看窗口内容。

文件夹备注

你可以为PE文件夹添加文件夹备注。

  1. 在PE文件夹上点击右键，选择文件夹备注。
    • 如果文件夹不含注释，将会为文件夹添加一个空的文件夹备注窗口（有关增加和显示备注窗口内容，详见此页）。
    • 如果文件夹包含注释(提示: 在文件夹上点击右键，选择属性)，添加文件夹备注将会把注释的内容复制到文件夹备注窗口。
    • 当在不同文件夹切换时，例如使用新的文件夹和窗口工具栏，这些文件夹备注不管是窗口被固定或最小化，总会在工作区显示。
    • 如果关闭文件夹备注（通过点击关闭按钮 ）,它会被嵌入文件夹里。只有在PE文件夹上右键点击，选择文件夹备注，文件夹备注窗口才会再次显示。


此外 ...

  • 按住Ctrl键，把鼠标放到PE文件夹上，可以预览文件夹备注内容。
  • 增加了系统变量@FNS用来设定文件夹备注的默认语法(= 0 (文本), = 1 (HTML), = 2 (Markdown), 或 = 3 Origin 富文本 (默认))。
  • 增加了浮动工具栏（通过在PE文件夹右边点击打开）包括文件夹备注， 复制文件夹 以及 打开文件夹属性对话框。

文件夹和窗口工具栏

添加了一个文件夹与窗口工具栏用于对项目管理文件夹以及文件夹里的窗口操作。当第一次运行Origin，你也许会发现这个新的工具栏“浮”在Origin工作区里。你可以把它拖到(按住标题栏)工具栏间隔的空白处；或者从菜单选择查看: 工具栏...并点击重置按钮来把工具栏重设为标准布局。

文件夹和窗口工具栏使得下列操作更容易：在不同PE文件夹间切换(Seesaw)，添加当前窗口到上一个文件夹并显示为快捷方式，固定当前窗口，以及在当前文件夹排列窗口（菜单 窗口: 排列窗口 <上次使用的>）。

  • 前一个文件夹: 从当前文件夹回到前一个文件夹。
  • 后一个文件夹: 从前一个文件夹回到当前文件夹。
  • Seesaw: 在两个文件夹间切换的另一种方法(Ctrl + Alt + X)。也可以从菜单打开，窗口: Seesaw。
  • 添加当前窗口为快捷方式用于Seesaw: 把当前窗口作为快捷方式添加到上一个文件夹(Ctrl + Shift +F7)。也可以通过在窗口标题栏点击右键，选择添加快捷方式到(使用Seesaw来查看)：添加快捷方式。
  • 从上一个文件夹添加快捷方式: 根据窗口类型，添加快捷方式到当前文件夹。
  • 排列窗口(上次使用的): 与菜单窗口: 排列窗口 <上次使用的>功能相同。
  • 固定活动窗口: 固定当前窗口，当点击排列窗口(上次使用的)按钮时，当前窗口不会重新排列。另外也可以通过在窗口标题栏点击右键，选择固定窗口或取消固定。

窗口快捷方式

在当前的项目管理(PE)文件夹里，你可以把鼠标放在另一个文件夹上，点击右键，选择添加快捷方式...。会弹出一个对话框，可以根据窗口类型添加为快捷方式到当前文件夹。

  • 当一个窗口快捷方式被添加到当前文件夹，快捷方式窗口的标题栏右边会出现主页按钮（跳转至源窗口）。点击该按钮，将会跳转到源窗口所在的文件夹，源窗口也会变为活动窗口。
  • 快捷方式窗口的标题栏同时显示源窗口所在的PE文件夹名，并用小括号括上。
  • 在快捷方式窗口的标题栏点击关闭按钮，将会在当前文件夹隐藏该快捷方式窗口。
  • 需要从当前文件夹删除快捷方式，可以通过在项目管理器里右键点击该文件夹，选择删除快捷方式（如果没有快捷方式可删，该选项会变灰色）。
  • 快捷方式窗口的最小化和向上还原按钮可以使窗口最小化或恢复窗口大小，从而使窗口可见但不会覆盖整个工作区。

Origin中的数字

  • 单元格的显示值vs实际值：当你导入或者输入数据到一个工作薄或者矩阵薄，Origin使用一个用户指定的设置组件去决定如何显示每个单元格的数据。显示值是一个可见的数据呈现值；实际值被存储为全精度，其精度符合数据集的数据类型（参看下方列表）。当你对数据进行运算时，其实是其实际值参与了运算，而不是其显示值。
  • 当内容超出单元格宽度：在Origin 2020b之前，如果工作表单元格中的字符序列超出了当前单元格（列）宽度上可显示字符，Origin将单元格内容表示为一系列磅符号（######）。现在已更改为类似MS Excel的工作表行为：
  • 不管列宽如何，文本字符串都不会触发显示为######。要查看完整的字符串，必须增​​加列宽。
  • 超出列宽的数值将首先四舍五入到最接近的十进制，并且只有在不足以显示完整的整数值时，才会在显示的地方加上######。
  • 这种四舍五入的数值仅影响显示的值。内部存储的值不变。
  • 要恢复单元格内容未完全显示时显示######的旧行为，请设置系统变量@WPE = 1。有关更改系统变量的值的信息，请参见这个FAQ。
  • 工作薄中的缺失值：当使用电子表格存储数据时，有一种需求就是去表达和自动生成那些在运算过程中不是数字的特殊数字。Origin支持这种数据，它的内部值是-1.23456789E-300。因为Origin认为这个值是一个特殊值，你可以使用它在工作薄或者矩阵薄中输入一个缺失值，该值可以在运算或者脚本中调用（比如，返回一个缺失值除非符合某些条件）。 Origin在工作表或矩阵表中显示缺失值为"--"。这样就可以把缺失值区分开来。然而，你不需要因为该值没有像你输入的数值一样，而感到困惑。
  • 十进制，科学记数法，工程: 你可显示工作薄和矩阵薄的数据为十进制:1000, 科学记数法:1E3, 工程:1k, 和十进制:1,000。你可以手动从这些选项中选择，但是你也可以预定义自动切换到科学记数法的阀值。默认阀值是6 (向上)和-3 (向下)。
  • Origin中的日期和时间数据: Origin根据格里高里历（公历）来解析时间数据。Origin的日期时间的长期数学体系基于天文Julian日期数字，并滞后12小时。它定义1/1/-4712(1月1日, 4713 BCE)，格林威治标准时间12：00为时间零点。 近期，Origin增加了两套时间系统: (1) 真正的Julian日期数字（0是中午12点而非次日午夜）； (2) “2018”系统，在该系统中，0 定义为2018年1月1日00:00。2018系统在使用例如导入向导导入数据时，能够支持更高的精度（例如，自定义日期格式可支持"MM'/'dd'/'yyyy HH':'mm':'ss'.'######"）。而以前版本的精度为0.0001秒。有关更多信息，请参阅Origin中的日期和时间。
