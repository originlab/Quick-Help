母版页面

母版页面功能用于添加一个或者多个通用对象到输出或者打印图片上。 使用母版页面去添加一系列的属性和对象 -- 定义的背景，公司Logo，文本标签，图片等等 -- 到项目中被选的图形窗口。

要使用此功能，您必须先创建母版页面，仅包含要在选定的图形窗口上显示的属性和对象：

  1. 从菜单中选择文件：新建：母版页面。 如果您在选择此菜单项时不在项目根文件夹中，则会要求您切换到项目根文件夹。
  2. 自定义母版。 请注意，显示在'保存母版模板按钮下方的注释是“隐藏的”，不会应用到您的图形窗口。
  3. 点击保存母版按钮。 保存时，母版将作为“活动窗口的名称”保存到用户文件夹并命名为"激活窗口的名字".otpu (e.g. "C:\Users\YourName\Documents\OriginLab\User Files\Master.otpu").


有关母版，需要知道的信息：

  • 母版只会在与模板相同尺寸的图形页面显示。 将尺寸应用于所有图将母版页面尺寸应用于相同宽高比的所有图形。
  • 如果您创建图形并修改页面大小，然后选择文件：新建：母版并将母版加载到工作区中，则母版模板将加载您修改的图形页面的页面尺寸。
  • 旋转按钮交换母版页面尺寸（横向和纵向切换）。
  • 单击将尺寸应用于所有图形时，会提示您，仅将尺寸应用于具有相同纵横比的图形，或者应用于具有相同方向（纵向或横向）的所有图形。
  • 纵向定向的母版页面项，不会显示在横向定向的图形上，反之亦然。
  • 不使用母版时，编辑：复制页面(CTRL + J) 将在剪贴板上放置“一个 OLE 可编辑的Origin图形对象”。 使用母版时，编辑：复制页面将在剪贴板上放置一个“图像”； 在单击“复制”之前，图形复制为图像对话框打开，要求您是否接受或调整图像设置。
  • 编辑：复制页面通常会将原始图形对象复制到剪贴板。将对象粘贴到MS Word文档中后，用户随后双击图形对象时，该图形会在Origin的OLE实例中打开，并在其中进行编辑。当图形中出现母版时，编辑：复制页将图像对象（EMF）放置在剪贴板上，而不是通常的图形对象。
  • 如果使用母版，发送图形到PowerPoint 不支持 插入图形为嵌入对象,建议使用 将图形插入为图片。
  • 还有一种机制，可以有选择地排除导出或打印页面中的母版项。如果有对象您不希望它出现在导出或打印页面中，可以设置它的名字以“NoExport”开头（例如，打开一个箭头对象的属性对话框，并在编程选项卡上，将“箭头”重命名为“NoExport_Arrow”）。
  • 当您自定义并将您的模板保存到用户文件夹时，下次选择文件：新建：母版，加载你的自定义模板。
  • 如果您不想在工作区中显示母版，请点击查看：显示：母版并删除复选标记。导出的图形仍将包含主项目（它们不会显示在工作区中）。
  • 如果您想从工作区和导出图像中的图形中删除母版项，请单击格式：页面属性，显示选项卡并清除使用母版复选框。

您可以使用文本工具将项目和系统信息作为静态文本对象添加到您的母版中，但由于该想法是为了构建用于重复使用的模板，因此请考虑文本标签中的软编码信息。 要了解更多信息，请参阅以下主题：

  • 文本标签中的替换变量值

导出边距调整对话框

默认情况下，你复制或者输出图形的边距由在下面工具中决定：设置：选项的复制页面设置组件中的页边距控制选项 (复制操作)；或者导出图形对话框中的导出设置分支（导出操作）。您可以使用 导出边距调整工具在图上直接拖拉来设置导出或复制图形的页边距。

  1. 点击激活图形窗口，选择编辑：设置导出边距去打开导出区域微调对话框。
  2. 拖动页面边框上的蓝色控件，或者在对话框输入数值。点击应用按钮。

调整页面至图层大小以及调整图层至页面大小

有两个对话框可以帮助我们在导出之前去掉页面的空白部分或者调整页边距

  • 调整页面至图层大小: 选择菜单 图: 调整页面至图层大小 打开 pfit2l 对话框.
  • 调整图层至页面大小: 选择菜单 图: 调整图层至页面大小 打开 gfitp 对话框.

这两个对话框都可以用于去掉页面的空白部分. 无论是否改变图形纵横比，两者都可以用于调整大小。可以在这里查看两者的具体文档 来决定哪种方法更适合你。基本上如果您需要保留特定的页面大小，我们建议使用调整图层至页面大小; 如果不是的话，那么使用哪种工具都可以。

图形, 复制/粘贴或嵌入 复制/粘贴图形 嵌入图形，第三方应用程序

把Origin图形粘贴到另一个应用程序的文件中 (例如：Microsoft Word的.docx 文件)，有两种方法：

  • 作为图像
  • 作为可编辑的OLE Graph 对象 对象链接和嵌入 (OLE).

当您粘贴为图像时，您正在粘贴静态对象。例如，在 MS Word 中，您可以使用 Word 的图片编辑工具修改图像，但不能在 Origin 中编辑图形并让图形在 Word 中更新（除非删除现有图像并用一个新的替换它）。

相比之下，当您粘贴图形为对象时，您可以稍后在宿主应用程序中双击该对象以启动 Origin 的新实例。图形和任何相关的数据表将可以被编辑。编辑完成后，使用 Origin 的 文件：退出并返回... 菜单项更新嵌入对象，关闭 Origin 并返回到宿主应用程序。

复制图形作为图像

  1. 点击选中您的Origin图形，然后 ...
  2. 选择编辑：复制图为图像（CTRL+ALT+J）。 或者，单击页边距附近的图形窗口，然后单击 浮动工具栏 上的“将图形复制为图像”按钮。
  3. 在您的宿主应用程序中，选择粘贴或按CTRL+V键。


如果将图形复制为图像 (CTRL+ALT+J) 时，您现在可以选择不同图像格式（PNG、EMF、DIB、HTML、JPG）。此外，您可以快速调整大小并指定背景透明度 (PNG)。 复制图为图像 (copyimg) 对话框包括一个 对话框主题 控件，它允许重复 <上次使用的> 设置，将当前设置保存为默认 保存为<默认> 或将设置保存到命名主题（另存为...）。

布局窗口

布局窗口，如名所示，是可以布局不同元素的窗口，然后用于发表出版或导出。使用布局窗口可以把图形、结果表格、文本对象等组合到一个版面。

通常，您会先创建一个空白的版面：

  1. 点击标准工作栏上的 新建布局 按钮。
  2. 右击空白布局，然后就可以粘贴剪切板的内容，添加文本，插入图像文件，添加图形、表格等。
  3. 从文件插入图像时，在要插入的区域单击鼠标右键。 添加图形或工作表时，在浏览器中选择一个窗口并用鼠标拖出布局上的一个区域。 请注意，可以通过用鼠标拖动来移动添加的对象或调整其大小。
    • 您可以通过双击图像，对插入的图像进行编辑。 这将在 图像窗口 中打开图像。
    • 您可以通过双击插入的图形对象来编辑图形。 这将打开源图形进行编辑。 对源图形的编辑将更新布局页面上的图形中。
    • 插入进来的表格可以被编辑，双击它即可。选中一个表格对象，然后使用弹出的 迷你工具栏 可以调整表格的样式。
  4. 注意，使用 对象编辑 工具栏上的按钮可以定位布局窗口里的选定元素。

我们支持将布局页面导出为 .SVG 文件。在布局中添加图表时，从布局快捷菜单中选择添加图表，您的 SVG 文件的元素（例如文本）将可以在 Inkscape 或 Adobe Illustrator 等应用程序中进行编辑。


更多关于布局窗口的信息，请参阅 Origin 帮助文件的使用布局窗口。

组合图形到布局

作为手动创建布局的替代方案，Origin 提供了一种工具，可以将选择的图形合并到一个等间距的布局页面中。 在这种情况下，合并的图形是图像，因此不能使用上一节第 3 步中描述的过程进行编辑。 然而当您不需要进一步的图形编辑，可以使用此方法快速生成漂亮的布局。

除了 组合图形到布局 之外，Origin 在其他几个对话框（合并图形窗口, 排列图层 和 图层管理）中也加入了“合并图形”这一功能。


组合图形图像到单一布局的步骤：

  1. 图形窗口处于激活状态，点击菜单 图：组合图形到布局窗口。
  2. 利用 选择 下拉列框指定如何选择图形。如要选择单一图，点选 指定，然后点 图 后面的三个点的浏览按钮打开 图形浏览器 进行图形选择。

调整页面至对象大小 菜单命令（格式：调整页面至对象大小）可以应用统一的边距到当前布局页面。这与前面提到的 调整页面至图层大小 对图形窗口的作用相同

文件：导出图形

文件：导出图作为图像 (X-Function = expG2img) 是导出激活图形为以下格式的一种的简易方法，PNG, BMP, JPEG, TIFF 或者 EMF。它并没有提供像 文件：导出图形(高级) （请看下一节）那样的层级控制或灵活性（如不能用于布局窗口），但它对于单一图形的光栅格式图像的导出非常好用。


使用简化的 expG2img 对话框将图形或布局页面导出为栅格图像：

  1. 当图形窗口处于激活状态，选择菜单 文件：导出图形。
  2. 对于EMF图像，设置一个关于报告的对话框底部的图像大小的大小因子。 对于光栅格式，选择输出DPI设置。 或者，清除 自动 复选框并指定一个 点阵宽度。单击预览 可在 Origin 自带的查看器中预览要导出的图形。

文件：导出图形(高级)

文件：导出图形(高级) (X-Function = expGraph) 是 Origin 导出图形和布局页面的主要工具。它既支持光栅图，也支持矢量图，导出多图灵活（如 当前文件夹的所有图），DPI 和快速模式的控制，等等。


导出图形或布局页面为图像文件的方法：

  1. 选择菜单 文件：导出图形(高级)。


expGraph 方式的导出图形支持15种文件格式：

光栅格式 BMP, GIF, JPG, PCX, PNG, TGA, PSD, TIF;

当您选择 图像类型 = TIF/TIFF，并选择导出多个图形（例如，当前文件夹中的所有图）时，对话框中将会出现一个 导出为多帧TIF文件 复选框。选中该框将所选的图形窗口导出为单个 TIF 图像（TIF 动图）。

矢量格式 CGM, DXF, EPS, EMF, PDF, SVG, WMF

  • 保存当前设置但 不 导出，选择 保存到<图形>。
  • 当您在导出对话框已经修改了设置，要再读出保存的设置，选择菜单 <图形>。

将数据浏览绘图导出为视频文件

将数据浏览绘图导出为视频（GIF、TIFF、AVI）：

将图像导出为栅格图像或视频文件

菜单 文件: 导出图像 (X-Function = cvExport) 可以将图像窗口导出为多种栅格图像格式：BMP, GIF, JPG, PNG, TIF。


使用 cvExport 导出（图像窗口的）图像：

打印或导出为 PDF

您可以将富文本内容输出到打印机；或使用第三方驱动程序（例如 Microsoft Print to PDF）打印到 PDF。

要创建漂亮的输出，您需要注意以下设置：

  • 纸张尺寸：这是在文件：页面设置或打印机驱动程序的高级选项中设置的。
  • 边距：这些在文件：页面设置中设置。
  • Notes显示边距：默认情况下，Notes 显示边距 = 5（单位 = 点数）。因此，您会注意到在渲染模式 (CTRL + M) 中，Notes 页面的左侧有一点边距。此显示边距通过设置系统变量 @NDM 来控制。
  • Notes窗口页面宽度：页面宽度由属性note.width (units = in/cm, etc）。默认值 = 0，即不控制宽度。
  • 不控制宽度 (note.width = 0) 时，Notes 窗口页面宽度 大约等于 纸张尺寸 - 边距。通常情况下这样导出的 PDF 或打印输出的效果都比较令人满意。
  • 为获得最佳打印效果，您需要设置 notes.width。但是在设置的时候，请注意不要超过 纸张大小 - 边距，否则文本内容可能会被截断。例如，一个"8.5"英寸宽的页面，左右边距为“1”英寸； 0 点阵的显示边距，应"打印出来"为 6.5 英寸宽。因此，为获得最佳效果，请在打印前设置 page.width = 6.5。

HTML 和 Markdown

您可以使用备注窗口创建HTML报告。您可以使用链接到报表元素（如元数据，统计结果或图表）的占位符来构建HTML报表。生成的HTML报告可以打印或导；或者它可以合并到分析模板中。也可以使用Markdown语法而不是HTML来创建报告。如果使用Markdown语法，则还可以包含可选的HTML语法以进行高级自定义。

要使用此功能，用户应该对HTML或Markdown有一些基本的了解，并且与大多数事情一样，用户越熟练，创建具有专业性外观的输出报表就越容易。无论如何，生成基本报告所需的技能是很容易掌握的。有关指导，请参阅此OriginLab的blog。

向 HTML 报告中添加内容

使用Origin备注窗口创建报告的基本步骤如下：

  1. 创建项目，包括图形和分析结果。
  2. 单击新建备注按钮添加一个备注窗口。
  3. 从菜单中选择HTML: 语法，然后选择HTML或Markdown（提示：您可以切换文本/视图和标记模式，通过按CTRL + M热键，您可以通过设置LabTalk系统变量来选择默认语法@NPS）。
  4. 您可以在备注窗口的标记模式下编写代码，也可以使用Notepad++等文本编辑器，然后将其复制并粘贴到备注窗口。如果您没有使用HTML或Markdown的经验，您可能需要激活备注窗口，然后选择HTML：加载示例(HTML = ".html", Markdown = ".md")来加载示例。
  5. 您可能希望将当前项目的图形和分析结果合并到报告中。最简单的方法是在您的代码中为每个需要呈现的Origin对象插入一个“占位符”。这些占位符可以使用任何对你有意义的字符串命名，它们应该用双花括号括起来（例如，{{MyR-SquareValue}}）。
  6. 右键单击备注窗口标题栏，然后选择编辑占位符。您已合并到代码中的占位符将被选中并添加到工作簿HTML的Link工作表中。该表将在A列中列出代码中所有占位符。第二列 - Col(B) - 为空白，保留用于添加要在报告中呈现的关联的Origin对象（即变量，分析和图形结果等）。
  7. 对于A列中的每个占位符，找到关联的Origin对象并将其复制到剪贴板（例如，如果它是由某些统计分析得到的工作表中的数字数量，请选择单元格数量，右键单击并选择复制，或按键盘上的CTRL + C）。
  8. 返回占位符表，右键单击B列，在相关占位符旁边，然后选择粘贴链接。这会将占位符与预期的Origin对象相关联。
  9. 如果要将HTML报告包含在分析模板中，可以右键单击分析模板中的工作簿选项卡，然后选择添加备注作为工作表。这会将备注窗口插入到工作簿中，您现在可以选择文件：保存工作簿为分析模板。

当备注窗口处于活动状态时，您可以单击HTML菜单并将示例HTML文件加载到窗口中。

使用您的HTML报告

在填写占位符表时，您应该会在备注窗口中看到您的HTML报告。完成后，您可以使用HTML报告执行各种操作，包括：

  • 打印到硬拷贝或PDF（文件：打印）。
  • 将窗口导出为网页（备注：导出为网页）。
  • 单击备注窗口右上角的嵌入按钮将内容嵌入到工作簿中，然后将工作簿另存为分析模板（文件：保存工作簿为分析模板）。
