2.56 FAQ-497 如何在 Set Column Values 对话框中调用参数行或工作表变量？

Use-Variable-in-SetColValue

Last Update: 7/14/2018

我们可以在 Set Values 对话框 中调用用户自定义的公式来给列赋值。您可以在用户定义的表达式中调用 工作表标题行 中的数据。

首先在 Set Values 对话框 的Before Formula Scripts 面板中声明一个 double 类型的变量。 

double d1 = %(col(A)[C]$);

变量定义后，就可以直接在列公式面板中使用该变量，如下示例表达式所示： 

A/d1;//or col(A)/d1 in older versions


有关调用工作表标题行数据的语法的使用教程，请参见 列标签行字符 。


请查阅以下相关主题：

  • 可以从导入的文件中提取元数据，并将此数据保存在工作簿中。有关详细信息，请参见 FAQ-423 如何检索和使用从文件名和标题中提取的变量？。
  • 要在 Set Values 计算中使用 分析报告工作表 中包含的数据，有关详细信息，请参见 FAQ-498: 如何在设置值 (Set Values) 对话框中使用分析结果？
  • Beginning with Origin 2016, Project Variables are listed in the Variables menu of Set Values, making it easier to track and use them in your Set Values formulas.

从 Origin 2016开始，Set Values 的 Variables 菜单中已经默认列出项目变量 (Project Variables)，因此在 Set Values 公式可以更容易的跟踪和使用这些变量。
