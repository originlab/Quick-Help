6.94 FAQ-1121 当看到错误信息 "The FLEXlm options file does not permit license checkout"。该如何做?

Opt-File-Not-Permit-Lic-Checkout

Last Update: 4/30/2024

当您尝试连接您的 Origin 到授权服务器，如果您看到下面的信息：

License Status: 112

The FLEXlm options file does not permit license checkout.

这是由于您的授权服务器管理设置了选项文件来限制谁能访问服务器，而您的机器是不被允许访问的。

请联系您的管理员来检查正在使用的选项文件来解决。
