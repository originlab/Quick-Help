3.83 FAQ-321 如何使用计数作为直方图的标签？

histogram-with-counts-label

Last Update: 7/16/2018

从 Origin 2017 SR0 开始，绘制直方图后，你可以从 Plot Details 对话框的 Label 选项卡的 Show 下拉列表中直接选择 Counts / Percentage 作为标签。

对于 Origin 2016 及以前的版本，你需要从分格数据中绘制一幅以计数作为标签的直方图。

方法 1: 使用 frequency count 工具计算出分格数据

  1. 选择 Statistics: Descriptive Statistics: Frequency Counts，计算频数分布以得到分格数据
  2. 从结果工作表中选中 Counts 列，选择菜单 Plot:2D:Bar:Column+Label 来绘图。

方法 2: 直接从直方图中找到分格数据工作表

  1. 选择菜单 Plot:2D:Histogram:Histogram，绘制一幅直方图
  2. 右键单击直方图，从弹出菜单中选择 Go to Bin Worksheet
  3. 在打开的分格工作表中选中 Counts 列，选择菜单 Plot:2D:Bar:Column+Label 来绘图
