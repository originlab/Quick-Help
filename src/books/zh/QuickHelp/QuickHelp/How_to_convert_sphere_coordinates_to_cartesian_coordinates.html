3.58 FAQ-293 怎样将球面坐标转换为平面坐标？

Sphere-Coordinates-to-Cartesian-Coordinates

Last Update: 7/21/2018

Origin 提供了一个 X-Function sph2cart 脚本，用于将球面坐标转换为平面坐标。球面坐标数据通常是三列数据，或虚拟矩阵，亦或是三个矩阵对象。

  1. 去到菜单 Windows 打开 Command Window 对话框或 Script Window 对话框。
  2. 输入以下命令并按 ENTER 键确定。
sph2cart -d


Keywords:sphere coordinates, cartesian coordinates，球面坐标，平面坐标
