1.25 FAQ-139 如何将多个图形组合为一个？

MergeGraphs

Last Update: 10/14/2022

通过工作区右侧的 合并 工具栏按钮 或选择 图: 合并图表 菜单可将多个图形组合为一个。对于内嵌图表，激活包含内嵌图表的工作表，选择 编辑: 合并(内嵌) 图表 菜单。

  1. 在 图 编辑框使用下移或上移按钮排列图表顺序，或使用菜单选择图表。
  2. 点击 "预览" 按钮或勾选 "自动预览" 复选框以显示和更新预览。
  3. 设置组合后的图形的行数和列数。注意：当设为 1x1 时所有的图形将相互覆盖整合为 1 个。
  4. 勾选 关联源图 后面的复选框，则所有图形将关联到组合后的图形中，所以在此后改变图层大小或者移动图层时，其它图形也将一起变化。
  5. 使用 添加标签 功能为每个图形添加标签。
  6. 组合之后，可以根据需要将各图形中的图例组合为一个：
    1. 右击图例，选择 图例: 更新图例。
    2. 更新模式 设为重构，图例 设为整页只添加一个总图例，然后点确定。
    3. 按住 Ctrl 键并拖动图例的边缘以重新排列。

有时候会发现 合并图表 对话框中的相关操作反应很慢，这通常是因为在对很多图形进行组合时，右侧的预览窗口反应较慢导致的。此时可以通过在命令窗口中输入命令行进行合并操作，而不是打开对话框，则反应速度会有明显提升。

若要使用命令进行图形合并，则通过以下方法进行操作：

  1. 在标准工具栏中点击 新建文件夹 按钮，则在项目管理器中创建了一个新的文件夹。
  2. 把想要进行组合的所有图形移动到这个文件夹中，并确保所有图形都是打开的 (可见且未隐藏)。(直接在两个文件夹之间拖拽图形即可实现图形位置的移动)
  3. 选择 窗口: 命令窗口 菜单打开命令窗口，然后输入 merge_graph。
  4. 按下 Enter 键，则可以创建一个包含了此文件夹下所有图形的组合窗口。
  5. 如需对组合后的图形位置进行调整，则可以通过 图: 图层管理 菜单下的 图层管理 对话框进行操作。
