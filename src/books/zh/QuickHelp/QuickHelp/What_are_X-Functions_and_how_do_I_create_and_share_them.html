4.40 FAQ-401 什么是 X-Function？我应该如何创建并与他人分享？

Create-and-Share-XF

Last Update: 7/13/2018

Origin 提供了大量的 X-Function，而这些 X-Function 可用于执行各种数据处理任务。大部分的 X-Function 可以被 LabTalk 脚本访问或调用。LabTalk 脚本可在 Command Window 或者 Script Window 中调用 X-Function 执行数据处理的任务。

大量的可调用的数据处理函数，以及灵活的获取数据的方法，给 Origin 用户为其日常数据处理创建自定义的脚本代码提供了强有力的支持。如果您想了解更多关于创建 X-Function 的内容，请参考 X-Function 入门介绍。如果你想创建一个向导文件，请参考如何创建一个向导。 如果您想和其他 Origin 用户分享这些文件，您可以上传到 Origin 网页的 File Exchange 区域。

请参阅 X-Function 编程指导了解更多关于 X-Function 的详细内容。
