4.11 FAQ-370 如何列出所有脚本变量？

List-Variables

Last Update: 7/5/2018

若要列出该进程的所有变量，请在 Script Window 或者 Command Window 中键入以下命令，然后按 Enter 键： 

list a

若要列出该项目和进程的所有数值变量，请使用： 

list v

若要列出该项目和进程的所有字符串变量，请使用： 

list vs


请参阅更多关于LIST命令的详细信息。


注意：ed 命令可以打开 LabTalk Variables and Functions 对话框。在此对话框中，您可以查看并选择需要显示的变量，例如系统变量，项目变量，字符串变量等。您也可以在此对话框中进行删除变量的操作。


Keywords:edit, session, project, 编辑，进程，项目
