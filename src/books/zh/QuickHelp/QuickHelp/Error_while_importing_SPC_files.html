2.99 FAQ-970 导入 SPC 文件时产生错误应该怎么处理?

Error-while-importing-SPC-files

Last Update: 9/21/2018

如果您导入 SPC 文件时候失败并看到以下错误信息:

Error: File "C:\some\folder\MyFile.spc" cannot be opened!

这很有可能是因为，在安装 Origin 的时候，或者由于某些未知的理由，没能正确注册用于 SPC 文件导入的 DLL 文件，

可以用管理员身份运行一次 Origin来修复这个错误 。鼠标右键单击 Origin 应用程序图标，然后选择 Run as Administrator 或者 More>Run as Administrator. (您可能需要在这之后重新启动 Windows， 但也有可能不需要)

如果这不能解决问题， 也有可能是您的 SPC 文件实际上不是 Thermo-Galactic SPC 文件， 而是其他格式的文件， 只不过碰巧也使用了 *.SPC 作为文件扩展名。 在这种情况下， Origin 无法导入这些文件。
