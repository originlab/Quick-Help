5.61 FAQ-922 通过邮件发送 Origin 文件时如何防止文件损坏？

Email-incompatible-file-type-corruption

Last Update: 7/15/2018

一些用户反馈说下载了邮件附件中的 Origin 文件后无法正常打开，或者打开时 Origin 弹出以下类似的提示： 

File corruption: marker not at structure end: 000000000000001F
File format error. Incompatible file type or file may be corrupted.

这种问题的出现是因为有些邮件客户端或者病毒扫描程序可能会巧妙的更改邮件附件中的 Origin 文件，致使 Origin 打开文件出错。

为了解决这个问题，我们强烈建议在使用邮件发送 Origin 文件时先对文件进行压缩。
