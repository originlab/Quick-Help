3.129 FAQ-765 如何在设置值（Set Values）对话框中快速定义和访问变量？

Define-ProVar-in-SetValues

Last Update: 7/30/2018

若要在 Set Values 对话框中定义一个变量，你可以：

  1. 选择 Variables: Project Variables: More... 打开 Project Variables 对话框。你可以在这里查看或删除所有已定义的变量。（注：这些是只存在于当前项目中的变量）
  2. 点击 + 按钮，在 Define Variables 中定义变量。


当项目变量被定义后，你可以在 Variables: Project Variables 菜单下找到该变量，并可以快速访问变量。

Minimum Origin Version Required: Origin 2016 SR0

Keywords:define project variable, set column values，定义项目变量，设置列值
