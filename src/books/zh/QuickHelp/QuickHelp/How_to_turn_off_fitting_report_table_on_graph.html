3.153 FAQ-965 怎么使得拟合后的图上不再生成拟合结果表?

Turn_off_fit_report_table

Last Update: 9/21/2018

如果拟合后我们想使拟合结果表默认在图上不要输出

  1. 打开 NLFit 对话框, 在 Settings 选项卡的 Output 页面. 在 Result Table 下拉列表中选择 None.
  2. 点击 Dialog Theme 旁边的箭头， 选择 Save as <default> 把这个对话框设置保存为默认设置.

Keywords:remove report table, delete report table，关闭结果表， 删掉结果表
