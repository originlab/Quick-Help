6.13 FAQ-15 Origin 让我设置一个 User Files Folder。这是啥？

UserFilesFolder

Last Update: 4/24/2019

“用户文件夹” User Files Folder (简称 UFF)是 Origin 在保存和打开相关文件时的默认文件夹。

打开用户文件夹:
  1. 选择菜单 Help: Open Folder: User Files Folder，直接打开。

或，

  1. 选择菜单 Preference: Options，然后选择 System Path 选项卡。
  2. 在列表中选中 User Files folder 这一行，点击 Copy 按钮来复制这个文件夹路径，然后在资源管理器中打开 UFF。


改变用户文件夹的路径:
  1. 选择菜单 Preference: Options，然后选择 System Path 选项卡，在列表中选中 User Files Folder 这一行，点击 Change 按钮。或者，在 Script Window 中运行命令：
    changeUFF -d;
    打开 Change User Files Folder 对话框。
  2. 点击 Browse 按钮，选择一个新的路径并点击 OK。
  3. 重启 Origin，使得新设置的用户文件夹生效。
