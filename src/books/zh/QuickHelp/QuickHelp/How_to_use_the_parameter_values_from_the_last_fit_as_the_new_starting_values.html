3.4 FAQ-224 如何使用最后一次拟合的参数值作为新的起始值？

Fitted-Result-as-Initial-Value

Last Update: 7/15/2018

分析重新计算功能允许更改参数并更新分析。当您执行非线性曲线拟合时，重新计算模式将设置为手动。单击拟合按钮后，可以通过单击绿色锁来 Change Parameters 。

可以在参数选项卡上看到使用的初始参数值，它们取决于之前的拟合中点选的是 Fit 按钮还是 Fit till converged 和 OK 按钮。

如果您知道您可能希望以后查看拟合的准确性是否增加，并且希望使用上一次拟合收敛后得到的参数值作为初始值来进一步拟合，则先单击 Fit till converged 按钮，然后单击 OK 按钮。

如果只是单击了 Fit 按钮，当重新打开对话框以更改参数时，将看到所使用的是与上一次执行拟合前相同的初始值。
