3.113 FAQ-655 在线性拟合中可以获得哪些拟合参数的统计量（误差分析）？

Fit-Parameters

Last Update: 7/29/2018

在 Origin 中进行线性拟合， 拟合参数的统计量包括如下这些：

  • Value
    参数值
  • Standard Error
    参数的标准误差
  • LCL
    置信区间下限
  • UCL
    置信区间上限
  • Confidence Level for Parameters (%)
    参数的置信度。
  • t-Value
    参数的 t-测试值
  • prob>|t|
    参数的 p-值.
  • CI Half-Width
    置信区间半宽

如果想进一步了解这些统计量的意义， 请参考这个帮助文档：拟合统计量的解释
