2.94 FAQ-939 有没有办法禁用项目中的单元格公式？

Turn-off-cell-formula

Last Update: 5/10/2022

Origin 2018 引入了一种电子表格单元格表示法的 单元格公式。您可以在编辑模式下直接在工作表单元格中输入公式，然后点单元格的外面，计算得到的结果就会立即显示。但是，您可能会发现有时候需要禁用单元格公式才能更加适合当前的工作流程。


禁用单元格公式：

  • 方法1
    选择菜单 编辑: 移除链接...，并确认 单元格公式和链接 勾上。这会把单元格的公式用当前值替代。请记住这是永久性的改变，不可逆。
  • 方法2
    临时性禁用单元格公式计算，然后查看所有单元格公式。 单击 编辑: 编辑模式。在重新选择 编辑: 编辑模式之前 Origin 都将暂停对公式计算。
  • 方法3
    还有一种方法可以使用 LabTalk 系统变量临时禁用单元格公式。与前面的方法类似，这不会阻止在单元格中输入单元格公式，但在使用此方法时，公式将被简单地视为文本，并且在系统变量的值还原为其默认值 (@esc= 1) 之前不会进行任何计算。 
    @esc=0 // disable cell formulas
@esc=1 // default, allow cell formulas

    有关更改 LabTalk 系统变量值的指导，请参阅此常见问题解答。


当怀疑是单元格公式导致 Origin 崩溃时：

在极少数情况下，我们注意到了与单元格公式相关的程序崩溃问题，当怀疑 Origin 的崩溃可能时由此导致的时，请尝试进行以下操作：

  1. 直接从 Windows 的 Start 菜单点击 Origin 或者双击桌面上的快捷方式图标以运行 Origin 软件。 不要 通过双击项目文件的方式启动 Origin。
  2. 选择 Edit: Edit Mode 来暂停单元格的公式计算。
  3. 通过点击 File: Open 或者 File: Recent Projects 的方式打开项目文件。

如果此时 能够 打开之前不能打开的项目文件，那么有很大的概率时由于单元格公式中的计算导致的程序崩溃。则在 Edit Mode 下删除公式，或者将不包含单元格公式的部分项目另存为一个新的窗口或者新的项目。
