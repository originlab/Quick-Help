2.125 FAQ-1106 如何将多个 NetCDF 文件合并为一个？

Merge-multiple-NetCDF-files

Last Update: 11/11/2021

  1. 在菜单选择 连接: Python 包....
  2. 在 Python 包 对话框，点击 安装 按钮去打开 安装包 对话框。
  3. 在 安装包 对话框输入以下脚本并点击 确定 去进行安装。 
    netcdf4 dask[complete] xarray
  4. 安装 Python 包后，单击 按钮打开 代码编译器。将concatenate.py文件拖放到 代码编译器 中，然后单击 F5 键。
    • 在弹出的 选择输入文件夹 对话框指定合并 NetCDF 文件的文件夹，然后单击 选择文件夹 按钮。
    • 在弹出的 选择输出文件夹 对话框指定输出文件夹路径并输入合并文件的新名称，然后单击 保存 按钮。

    脚本窗口 会在合并成功后提示完成信息。你可以在指定的输出文件夹中找到新文件。
