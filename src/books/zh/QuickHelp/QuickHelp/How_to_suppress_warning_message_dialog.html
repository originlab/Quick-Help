3.145 FAQ-883 如何禁用提示信息对话框？

Suppress-WarningMessage

Last Update: 4/24/2019

当你执行，更改或删除操作时，Origin 会弹出不同类型的提示信息（包括智能提示）。 有时你可能会觉得这些提示信息很烦扰，并想不再显示这些信息。

禁用提示信息对话框选项

  • 有一些提示信息的底部有一个 "don't show this dialog" 的复选框，你可以选中复选框以禁止再次显示此消息对话框。
  • 有一些提示信息的提供了 "don't bother me with this again" 的选项，选择后，将会设置禁止显示这些信息。

对于上述两种情况，您可以通过 Help: Reactivate Reminder Messages 菜单重新激活信息对话框。

智能提示 (Smart Hint)

  • 如果想禁用智能提示，请参考这个页面。如果想重新激活智能提示，可以选择 Help: Reactivate All Hints 菜单。

删除操作的提示信息

  • 如果从工作簿或图形中删除分析操作，将会弹出提示信息对话框，要求确认删除。如果想禁用此提示，可以将系统变量 @DOM 设为 1.
  1. 在 Origin 菜单选择 Preferences: System Variables
  2. 在Variable 下输入 DOM ，并设其 Value 为1。
