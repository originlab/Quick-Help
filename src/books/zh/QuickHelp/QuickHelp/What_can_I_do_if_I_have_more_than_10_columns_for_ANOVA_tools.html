3.93 FAQ-331 如何对10列以上数据上执行 ANOVA？

more-than-10-columns-for-anova

Last Update: 7/26/2018

默认情况下，在 ANOVA 工具中，在原始数据模式（Input Data = Raw）中，Number of Levels 下拉列表最多只可以选择 10 组群组。

如果要分析的列数超过 10 列，可以在工作表中预选数据，然后再打开对话框。

或者

使用系统变量 @AML 来更改 Number of levels 下拉列表中的最大值。例如

  1. 从 Windows 菜单打开 Command Window 或者 Script Window
  2. 在下面键入命令，然后按 ENTER 键Type command below and press ENTER
@AML = 15
