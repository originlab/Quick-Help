1.159 FAQ-814 当我利用MS Word创建PDF时，数据显示超出了Origin图形上坐标轴的范围。

Data_Outside_Graph_Axes_Using_MS_Word_PDFMaker

Last Update: 11/11/2021

如果你执行了以下操作 ...

  1. 缩小了 Origin 绘图上轴刻度范围，可以是通过在改变 坐标轴 对话框中 刻度 选项卡下的 起始 和 结束 的值来修改，也可以是使用在工具栏中的 放大 工具来修改。
  2. 用 CTRL + J (菜单 编辑：复制页面 ) 复制图。
  3. 在 MS Word, 选择 选择性粘贴 粘贴为 图片 (Windows Metafile or Enhanced Metafile).
  4. 使用 MS Word PDFMaker 插件创建一个 PDF 文件...


... 你可能会看到以下的问题:


如需解决这个问题:

  1. 选择菜单 设置: 系统变量.
  2. 设置系统变量@GDIP 为 0
  3. 现在，在 Origin 中按 CTRL+J，粘贴你的 WMF 或 EMF 文件到 Word 中，并再用 PDFMaker 重新创建 PDF 文件。
