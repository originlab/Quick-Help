3.138 FAQ-812 如何在结果图形中绘制减去基线后的数据？

Plot-Subtracted-Data-in-Graph

Last Update: 7/30/2018

当你通过 Peak Analyzer 检测并减去光谱中的基线时，在点击 Finish 按钮之前，预览窗口中会显示减去的曲线。但是一旦点击 Finish 按钮，你会发现曲线还是原来的曲线。如果想在结果图中显示减去基线的光谱，你可以：

  1. 在 Peak Analyzer 对话框的 Subtract Baseline 页面上，单击 Subtracted Data 控件旁边的三角形按钮，然后选择 <Input>: XY = input XY。
  2. 然后点击 Finish 按钮。带有基线校正的数据将替换源数据并显示在结果图中。

注：默认情况下，Subtracted Data 会输出在是源工作表的新列中。默认设置不会在源数据中减去基线，是因为结果图是用源数据绘制的，我们不想覆盖源数据。在峰值分析过程中，点击 Subtract 按钮后看到的图形只是一个预览图窗口。 在峰值分析器关闭后，该窗口将消失。


Keywords:peak analyzer, subtract baseline, baseline correction，峰值分析，减去基线，基线校正
