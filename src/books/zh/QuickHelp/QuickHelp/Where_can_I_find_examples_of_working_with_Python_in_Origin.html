4.43 FAQ-405 在哪里可以找到在 Origin 中运行 Python 的例子？


Last Update: 7/13/2018

python-in-Origin-example

Origin 提供了一些内置的运行 Python 的例子，他们存放在 <Origin EXE Folder>\Samples\Python 路径下。

其中，有两个 .py 文件可在 Origin 中直接执行（请查看 LabTalk 文档中的 run -pyf 了解详细的信息）。

其他的 Origin 项目文件展示了各种利用 Python 访问 Origin 对象（例如数据，图形等）的例子。您可以打开项目文件，点击 Show Python Code 按钮查看 .py 文件。您也可以在 Code Builder 的 Workspace 中的 Project 文件下找到该 .py 文件。
