1.91 FAQ-444 什么是快速模式？它对我的图形和图形导出有何影响？

SpeedMode-GraphExport

Last Update: 11/16/2018

快速模式(Speed Mode) 是 Origin 在重新绘图的时候用以提高速度的一种机制。通常，含有大量数据点的图形窗口重绘的速度会比较慢，此时，“Speed Mode”就会启动，用来控制显示在图层中的数据点的数量，以提高重绘速度。Speed Mode 的设置可以在 绘图细节 对话框的“layer”级查看和修改（从 格式：图层属性 > 大小/速度 选项卡打开）。如果是二维图形，在 工作表数据，每条曲线的最大点数 文本框中输入一个合适的数值（该值应该小于图层中数据最多的那条曲线的总的数据点数），以减少显示的点数从而提高重绘速度。对于三维图形，在 矩阵数据，每行/列的最大点数 的 X 和 Y 文本框中输入合适的数值，以控制在曲面或等高线图中 X 和 Y 方向上显示的数据点数。

使用 X-Function speedmode (菜单 图：快速模式...) 来设置多幅图或特定的某一幅图的快速模式的状态，如关闭在当前项目中的所有图形的快速模式。

快速模式不会对打印或导出的图形造成影响，所有的数据点都会被导出。如果想要在打印的图形中启用快速模式，选择菜单 文件：打印，设置 工作表数据。跳过点 或 矩阵数据，最大点。如果想要在导出的图形中启用快速模式，选择菜单 格式：页面属性。在 其他 选项卡上， 勾选 如果启用快速模式显示，导出图形时也适用该模式。

此外，如果你想默认关闭所有图形的快速模式，可以选择菜单 工具：主题管理器 并将主题 Speed Mode OFF 设为系统主题（右键单击它并选择 设置为系统主题 ）。注: 此主题不会影响3D OpenGL图形。
