5.84 FAQ-1161 项目文件的预览图在Windows资源管理器中不显示怎么办？

OPJU-Preview-Image-Not-Display

Last Update: 4/24/2023

Windows 资源管理器的大图标或超大图标视图将显示项目文件保存前最后一个活动图形的 预览图像。

如果您在 Windows 资源管理器中没有看到预览图像，请首先确保在文件夹选项对话框（通过菜单文件）中取消选中始终显示图标，从不显示缩略图复选框选项）

如果始终显示图标，从不显示缩略图没有勾选，请尝试注册OPJHandler.dll，如下：

  1. 单击以打开 Windows 开始菜单
  2. 在搜索框中键入 cmd。 命令提示符 项目将显示。
  3. 右键单击它并选择“以管理员身份运行”。
  4. 在打开的命令窗口中，键入 
     regsvr32 "<Origin Program Folder>\OPJHandlers.dll"

    例如，如果您在默认路径中安装了 Origin 2023，您可以键入 

    regsvr32 "C:\Program Files\OriginLab\Origin2023\OPJHandlers.dll"
  5. 如果弹出消息框说成功，请重新启动 PC。
注意

如果尝试上述解决方案后仍无法显示预览图像，请检查是否有杀毒软件阻止了 OPJHandler.dll，例如 Kaspersky。
