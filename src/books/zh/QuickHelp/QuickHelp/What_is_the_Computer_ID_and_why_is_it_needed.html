6.23 FAQ-25 Computer ID 是什么？

Computer-ID-for-Licensing

Last Update: 7/5/2018

Computer ID 是一串由数字和字母组成的16位的字符串，它是 Origin 基于电脑的网卡地址和其他配置生成的，仅在激活 Origin 时用于识别你的电脑。

正常情况下，Origin 的激活过程都放在后台自动完成，绝大多数的用户不必知道本机的 Computer ID 是什么。只有在极少数的情况下，你可能需要提供这个 Computer ID。例如，当你需要把 Origin 移到新的电脑上，但是却无法使用 Origin 的 Deactivate License 工具来停用旧电脑上的许可证文件时，你就需要填写 System Transfer Request，这时候你需要知道旧电脑的 Computer ID 是什么。

如何查看 computer ID:

  1. 打开 Origin。
  2. 选择菜单 Window: Script Window 打开 Script Window。
  3. 输入以下命令并回车: 
help -c

Script Window 会显示出 Computer ID。如果返回多个 Computer ID，那个带有星号 (*) 的 Computer ID 就是当前 Origin 所使用的 ID。
