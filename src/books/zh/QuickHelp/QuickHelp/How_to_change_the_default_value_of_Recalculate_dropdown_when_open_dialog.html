3.54 FAQ-288 怎样更改分析对话框中重新计算下拉列表的默认值？

recalculate-dropdown-default

Last Update: 10/26/2022

分析

当你对工作表或数据列执行分析时，打开相应的分析对话框时， Recalculate 模式默认值是 Manual，在结果表中会产生一个绿色锁状图标。当源数据和参数发生改变时，你需要手动点击此图标选择 Recalculate 选项来进行重计算。

如果希望把系统默认的 Recalculate 模式设定为 Auto，需要在做分析之前在脚本窗口或命令窗口设置 系统变量 @DAM=1 (默认是 2) 。然后，当打开相应的分析对话框时，Recalculate 下拉列表中的默认值将是 Auto 。

注意: 系统变量 @DAM 将会影响分析操作相关的工具。 Worksheet Query 工具则不受此系统变量影响，它的默认值将保持为 None。


列公式

当您使用 F(x) 单元格执行列计算时，重新计算 锁在列标题旁边显示为绿色锁图标。控制列函数重新计算的系统变量是@AUFL。@AUFL 默认为 0，它会检查列中是否存在带重新计算锁的公式，如果有则保持重新计算模式，否则设置重新计算模式为自动。

要将重新计算模式全局默认设置为自动、手动，需要在输入公式之前在脚本窗口或命令窗口中设置 @AUFL=1 或 @AUFL=2。

将 @AUFL=3 设置为 无 模式，这样就不会添加重新计算锁。当输入改变时，输出不会自动更新。其好处是您可以自由编辑输出列中的值。

我们也可以永久改变 @DAM 和 @AUFL 的值, 如需了解详情，请看这个帮助文档
