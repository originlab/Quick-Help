5.22 FAQ-456 如何将整个工作表导出为图像文件或者 PDF 文件？

ExpWks-as-img-PDF

Last Update: 8/1/2018

导出为图像格式

若需将整个工作表导出为图像格式，您可激活工作表并在 Origin 菜单中选择 File: Export: As Image File 。

这将打开 expWks X-Function 对话框。这个对话框与 Export Graph 对话框基本相同，但是还添加了 Export Area 下拉列表来设置工作表的导出区域。

在这个对话框中，从下拉列表选择导出文件类型为 Image Type。

展开 Export Settings 分支并从 Export Area 下拉列表中选择 Whole。（这个选项还支持仅导出工作表的可见区域，或仅导出工作表的已选择区域。）

注意，您可以为输出图像设置所需的宽度，但是图像的高度则在保持纵横比的前提下自动设置。

您可以任意合并工作表中的单元格组用于放置图形，图像，报告工作表和文本，由此创建自定义的报告工作表。

这个自定义报告工作表的创建功能可与导出任意宽度的工作表的功能相结合，由此创建包括大型海报在内的演示文稿。

导出为多页的 PDF 文件

若要将整个工作表导出为 PDF 文件，您可激活工作表并在 Origin 菜单中选择 File: Export: As Multipage PDF File。

这将会打开 expPDFw X-Function 对话框。

其中， Image Settings: Multiple Pages 分支包括添加 Cover Page 和 Page Numbering 的选项。

如果您保留勾选 Use Worksheet's Own Settings，那么输出的文件将与工作表在 File: Print Preview 模式下看起来一样。这个视图模式可以通过选择 File: Print 并点击 Options 按钮来对打印选项进行编辑。点击 OK 按钮退出 Option 对话框，然后点击 Cancel 按钮退出 Print 对话框。

如果您取消勾选 Use Worksheet's Own Settings，那么您就选择使用与打印选项中相同的设置来显示的工作表的内容。
