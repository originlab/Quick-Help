5.15 FAQ-354 如何改变操作锁的颜色？

Last Update: 8/8/2018

change-lock-color

在 Origin 中， 如果在某次操作中使用了重新计算，那么在相关联的结果中将会出现一个锁头的图标。这个锁头的图标通过显示不同的颜色来表示不同的状态。

若需要改变锁头不同状态下的默认颜色，打开用户文件夹并找到 Origin.ini 文件。用 Notepad 打开该文件并找到 [Color] 的部分，将以下定义锁头颜色的注释行的注释标志“//”去掉 

PadlockClean=0 255 0
PadlockDirty=255 255 0
PadlockBroken=255 0 0

然后将“=”号右边的定义锁头颜色的值替换为新的颜色。其中那三个数字分别代表介于 0 到 255 之间的红色，绿色和蓝色值。 而：

PadlockClean 代表无需进行重新计算，默认为绿色。

PadlockDirty 代表等待进行重新计算，默认为黄色。

PadlockBroken 代表该操作的重新计算已被破坏，默认为红色。

Minimum Origin Version Required: 9.1 SR0
