2.23 FAQ-435 怎样在 Origin 和 MATLAB 中导入彼此的数据？

Matlabconsole-Importdata

Last Update: 4/24/2019

使用 Origin 的 MATLAB Console，一方面，可以通过图形化的界面和命令来从 MATLAB 中导入数据到 Origin 工作表或矩阵中。另一方面，也可以从 Origin 中导出数据到 MATLAB 中并在 MATLAB 中以变量来调用。请注意使用 MATLAB Console 这个工具， 需要同一台电脑上同时安装有 Origin 和 MATLAB 。

选择菜单 Connectivity: Matlab Console 打开 MATLAB console对话框来使用 MATLAB Console 工具。可以在面板中运行 desktop ，来打开标准 MATLAB 工作界面。在 MATLAB 中准备好数据后，转回到 MATLAB Console 工具，单击 Import 打开 MATLAB Import 对话框，选择所需的数据单击 Import as Matrices 或 Import as Worksheet 来将数据导入到 Origin 中。

同样的，要从 Origin 导出数据到 MATLAB 中，选择菜单 Connectivity: Matlab Console 打开 MATLAB Console 对话框来使用 MATLAB Console 工具，在面板中运行 desktop 来启用标准 MATLAB 工作界面，然后点击 Export 将数据导入到 MATLAB 中。
