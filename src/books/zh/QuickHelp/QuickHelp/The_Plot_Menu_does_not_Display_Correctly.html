5.51 FAQ-853 绘图菜单不能正确显示

The-Plot-Menu-does-not-Display-Correctly

Last Update: 7/15/2018

会有一些用户报告说 Origin 2017 中新增加的 Plot 弹出菜单 (缩略图菜单) 操作异常。这个问题的产生可能与最近的 Windows 10 Fall Creators Update, 2017 (Version 1709) 相关。请检查当前使用的 Windows 10 是不是这个版本。

微软随后发布了 Windows 10 Fall Creators 更新的补丁，并解决了程序响应缓慢的问题。

  • 转到 Windows Settings > Update & Security > Check for updates 并应用所有的可用更新。


如有需要， 也可以选择将绘图菜单回滚到之前的样式：

  • 打开脚本窗口 (Window: Script Window)，输入以下代码并按 Enter：
@DPPM = 1

如果这解决了问题，则可以 通过 Set System Variables 对话框永久禁用绘图弹出菜单。
