6.60 FAQ-886 如何获取产品密匙？

如果你是新近购买 Origin 的用户并且还没有注册序列号

如果你购买的是单用户版本

购买之后，我们将会以电子邮件的形式给你发送 Origin 的序列号和注册链接。

  1. 点击邮件中的注册链接，打开 OriginLab 的官网。请登录。
  2. 登录之后，你的注册码 registration code 和序列号 serial number 将会显示在网页上。点击 Register 按钮注册这个序列号，同时生成产品密匙 product keys。


如果你得到的是一个注册码 Registration Code 而非一个注册链接:

  1. 登录 originlab.com。
  2. 在登录后的 Welcome 页面上，点击链接 Register my product -version 2018 (9.5)。
  3. 在 Register my product 页面上，输入你的注册码 Registration Code 并点击 Continue 按钮注册这个序列号，同时生成产品密匙 product keys。


了解更多信息，参见这个文档。


如果你是一个多用户版的管理员

在将序列号和安装文件发给最终用户之前，请先在官网上注册这个序列号并且建立一个产品密匙（和许可证）的获取机制。详细步骤参见这个文档。


如果你的公司/单位购买了多用户版 Origin，并且你是 Origin 的最终用户

请向贵司的 Origin 软件管理员或者 IT 部门申请产品密匙。

如果你已经注册了序列号

参见这个文档。

如果你是从 Origin 2016 或者更早的版本升级到 2017 或以上版本的升级用户

  1. 登录 originlab.com。
  2. 在登录后的 Welcome 页面上，点击链接 Maintenance Center。
  3. 在 Origin Maintenance Center 页面上，点击链接 Registration for Auto-upgrading to Newest Version。
  4. 在 My Registration List 页面上会列出所有可以升级的序列号。选择你想要升级的那个序列号并点击它的 Register Product 链接。
  5. 你的注册码 registration code 和序列号 serial number 会显示在页面上。点击 Register 按钮注册这个序列号，同时生成产品密匙 product keys。

如果你是从 Origin 2017 升级到 2018 或以上版本的升级用户

从 Origin2018 开始，不同的版本之间共享同一个许可证文件，也就是说，新版本会沿用旧版本的所有许可证信息，包括产品密匙及它们的使用情况。

如果你购买的是单用户版本

新版本的产品密匙与旧版本相同。


如果你是一个多用户版的管理员

你的序列号将会自动升级到新版本。而旧版本的所有许可证信息（包括你在旧版本中设置的许可证 & 产品密匙的获取机制）都会自动应用到新版本中去。你可以随时通知最终用户进行升级。另外，你可以在官网查看到所有的产品密匙及其使用情况:

  1. 登陆 originlab.com。
  2. 点击链接 Manage My License(s)
  3. 点击你要查看的序列号的链接。注，它的版本号已经自动更新为新版本。
  4. 点击链接 you can see all product keys here。在打开的页面上，你可以查看到所有的产品密匙和它们的使用情况。


如果你的公司/单位购买了多用户版 Origin，并且你是最终用户

新版本的产品密匙与旧版本相同。
