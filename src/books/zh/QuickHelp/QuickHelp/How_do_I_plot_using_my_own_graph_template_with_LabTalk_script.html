1.118 FAQ-677 如何利用自己的图形模板和 LabTalk 脚本绘制图形？

Last Update: 11/21/2018

Graph-Template-LabTalk

图形编辑完后可将其设置为图形模板，以后可以通过点击 绘图：模板 菜单或在工作表（worksheet）结合 Labtalk 脚本绘制类似的图形。

保存图形模板时是否勾选 标志为可复制的模板 决定了用 LabTalk 脚本绘图的写法：

  • 如果 标志为可复制的模板 没勾选，则LabTalk的写法为： 
    worksheet -p n templatename

    脚本中 n 的值代表绘图类型的 ID 且绘图类型须与图形模板类似（也就是说如果图形模板是基于2D图则无法绘制3D图，反之亦然），点击 不同绘图类型的的ID查询相应的ID。

    例如，在工作表中选定数据然后运行以下命令： 

    worksheet -p 200 MyTemplate;

    这样在图形模板的基础上以你选定的数据绘制了一条折线图。

  • 如果勾选 模板被标志为可复制的模板 则写法为： 
    worksheet -pa ? templatename

    对于可克隆模板(cloneable template), 只需在工作表中直接运行脚本命令而无需先选择数据。 

    worksheet -pa ? MyMulti_Panel.otp

    也就说可克隆模板(cloneable template)知道该用工作表中的何种数据进行绘图，并使用这些数据绘制图形。
