2.71 FAQ-794 如何从数据库导入日期数据？

DB-Import-Date

Last Update: 4/24/2019

从数据库导入日期列时，可能会发现导入后的日期格式丢失。若要保留日期格式，则在导入时需要进行如下设置：

  1. 从 Origin 菜单中选择 Preference: Options 以打开 Options 对话框。
  2. 切换到 Numeric Format 选项卡，从 Date Format for Database Import 下拉列表选择适当的日期格式。也可以直接在编辑框中输入自定义日期格式。
