打印页面图层的尺寸不正确

如果您正在打印图形窗口，并且打印页面中的图层与 Plot Details 对话框的 Size/Speed 标签页上指定的尺寸不匹配，可以修改 Options 对话框（Preference: Options）的 Miscellaneous 标签页上的 X Scale Factor 和 Y Scale Factor 的组合框的值。

图上的标签在打印时位置发生移动

Origin 默认的查看模式是 Page View (View:Page View)。Page View 为用户提供了快速的屏幕更新，但是并不保证屏幕上文本的位置是其确切的位置，除非您使用的是字体缩放软件(如 Adobe Type Manager)。

  • 若要在打印前确保您的标签位于准确的位置：
  1. 可以打开 Print Preview 模式 (File:Print Preview) 或者 Print View 模式 (View:Print View)。

这两个视图模式均提供了真正的 WYSIWYG 显示（注意，Print Preview 模式更为可靠）。

图上的标签部分被打印至页面外

如果在打印时，图形中的标签被截断，那么请您重新检查图形窗口中的标签位置。超出页面的标签（或标签的一部分）不会显示在打印页面中的。页面的左边框和上边框由图形窗口的左边缘和上边缘所决定。页面的右边和下边框由图形窗口中的灰色区域所决定。

更改打印机会更改图形窗口中的对象

当 Origin 在您的计算机屏幕上绘制图形时，它会利用您的打印机驱动查询对象大小和位置等信息。如果您使用某一个打印机驱动绘制图形，当您更改了默认的打印机驱动（或在具有不同的默认打印机驱动的另一台计算机中打开该项目），那么 Origin 将重新调整在图形窗口中对象的大小和位置。

这不是 Origin 自身的问题，而是因为不同的打印机驱动具有不同的大小和位置的特征。解决此问题的唯一方法是使用相同的打印机驱动创建和显示/打印图形，或者在更换打印机驱动后根据需要重新调整对象的大小和位置。

在另一台计算机上打开项目会更改所选字体

如果您打开在另一台计算机上创建的 Origin 项目文件，并且该项目文件所使用的字体并未被安装于您的计算机，那么 Origin 将为您选择替换字体。如果您不喜欢替换的字体，唯一的解决方法是从计算机加载的字体列表中手动选择一种新字体。例如，要更改图形窗口中文本对象的字体，您可双击该文本对象，然后从 Font 下拉列表中选择一种新字体。大多数应用程序（包括 Origin）无法显示或者打印计算机中未安装的字体。

默认情况下，使用其他打印机打开项目文件时，Origin 不会更新图形页面方向

如果您打开了使用其他打印机驱动创建的 Origin 项目文件时，则需要手动更新图形页面的方向使其适应于当前的打印机驱动。

  • 更新项目文件使其适应于新的打印机：
  1. 选择 File:Page Setup.
  2. 选择 Portrait 或者 Landscape 方向.
  3. 点击 OK。Origin 将根据您当前打印机驱动的设置更新您的图形窗口。
