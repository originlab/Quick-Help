1.178 FAQ-856 为什么无法在图形中插入方程式？

Fail-to-Insert-Euation

Last Update: 11/16/2023

Origin 2023b 内建了基于SVG的LaTeX公式工具。这个新的 插入LaTeX公式 工具不需要手动安装额外的任何库或软件。为了摆脱由 LaTeX App 带来的问题，强烈建议您升级 Origin 到 2023b 版本或更高版本，然后体验这个新的 插入LaTeX公式 工具。请参阅此文档以获取更多信息。

如果你还想继续使用LaTeX App，请继续阅读下面的文字。

对于 Origin 2023 或更旧的版本

要使用 插入公式 工具，需要预先安装公式编辑器(Equation Editor)，在 Microsoft Word 以往的版本中曾经包含此公式编辑器。 如果您在 Origin 中打开 插入公式 工具时遇到以下错误时：

Origin provides an app LaTeX for equation editing.

It requires applications MiKTeX and OAutotrace be installed.

Would you like to use it?

See more details on LaTeX page. (here "LaTeX" shows as a link: https://www.originlab.com/FileExchange/details.aspx?fid=221)

You can later set system variable @IEE=0 to toggle back.

请检查以下内容：

如果你安装了2018年1月的公共更新(PU)

现在所有版本的 Microsoft Office 产品中都不包含公式编辑器 Equation Editor (Microsoft Equation 3.0)。 如果想在 Origin 中使用 Insert Equation 工具，您可以考虑使用在信息中提及的LaTex App，或者安装第三方的 App MathType。

使用免费的 Latex App
  1. 激活一个图形，然后从 工具 工具栏中点击 插入公式 按钮。
  2. 当您看到一个对话框，点击 Yes。
  3. Latex App和两个应用（Mikitex和OAutotrace）会自动安装。安装完成后，重新启动 Origin。
  4. 激活一个图形，然后从 工具 工具栏中点击 插入公式 按钮。
使用 Mathtype （不免费）
  1. 事先安装好 MathType。
  2. 激活一个图形，然后从 工具 工具栏中点击 插入公式 按钮。
  3. 当您看到一个对话框，点击 No。
注意：如需决定插入方程按钮的行为，您可以设置@IEE=0以切换提醒对话框。
