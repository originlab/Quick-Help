5.42 FAQ-756 如何浏览 File Exchange 页面并安装插件工具？

Fetch-Install-Apps

Last Update: 10/23/2019

File Exchange 页面提供了用户创建并共享的各种插件。这一集中的列表使得您不必在 Origin 网站中苦心寻找。

在 Origin 2018 及以后的版本, 您可以单击在 Apps Gallery 的 Add Apps 的图标；或者选择菜单 Help: Apps Center；或者按下 F10 键来打开 App Center，并在此寻找和安装 App

  1. 您可以查看最新的或者已更新的 App。
  2. 您可以通过搜索栏搜索所需的 App。
  3. 您可以通过一键点击下载并安装 App。
  4. 您可以一键更新所有已安装的 App。

从Origin 2020开始，你可以通过以下菜单下的查找App...来搜索相关的Apps：

  • 分析: 拟合
  • 分析: 峰值及基线
  • 统计

这个菜单选项会打开App Center。在App Center对话框里，点击App右边的Download and Install图标来安装对应的App。

而在刚刚引入 App 概念的 Origin 2017 和 Origin 2016中，您可以：

  1. 选择菜单 Tools: Apps on File Exchange 然后从子菜单中选择一个 App。
  2. 那么Origin将自动帮您打开该选中的 App 的 File Exchange 的页面，点击 Download Now 按钮将该 App 的 OPX 文件保存于本地磁盘。
  3. 将已下载的 OPX 文件拖入 Origin 的工作区，当 App 安装成功则会弹出消息说明该 App 已成功安装。

所有已安装的 App 都可以在停靠在 Origin 工作区右方的 Apps Gallery 中找到。点击其图标即可使用该 App。您也可以右键单击该图标来选择 Send Feedback 发送反馈或者 Open File Exchange Page 浏览该 App 的指引。另外，您还可以通过右键弹出菜单隐藏或者卸载该 App。
