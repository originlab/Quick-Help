2.129 FAQ-1134 如何删除项目中的 DDE 链接？

Remove-DDE-Link

Last Update: 10/13/2023

如果您的项目有 DDE（动态数据交换Dynamic Data Exchange）链接，例如 如果工作表单元格链接到外部 Excel 文件的单元格，而对于无法访问 Excel 文件的其他人可能没办法共享这个 opju。

这样的话，您可以使用我们更新的删除公式/链接对话框将项目中的所有 DDE 链接转换为值，然后与其他人共享。

  1. 选择编辑：移除链接...菜单。
  2. 确保选中Excel(DDE) 链接复选框。
  3. 在移除链接在下拉列表中选择项目，然后单击确定。
  4. 保存项目。

注意：此对话框还可用于将单元格公式/链接和列公式转换为值。

如果您是没有权限访问 Excel 文件但是获得此类项目文件的人，在 Origin 中打开该项目文件时，请打开启动 DDE 链接对话框，要求重新建立或删除链接。 如果你的项目中有太多这样的链接，就会不断弹出对话框，这会很烦人，甚至无法打开项目。

为了能够打开此类项目文件而不显示启动 DDE 链接对话框，请在打开项目文件之前设置系统变量 @dden=1。 请参阅系统变量 @dden 的详细信息。
