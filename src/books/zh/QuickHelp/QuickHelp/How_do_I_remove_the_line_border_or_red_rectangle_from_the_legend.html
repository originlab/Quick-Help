1.182 FAQ-868 如何删除图例中的边框或红框？

Customize-Legend-Border

Last Update: 4/24/2019

删除整个图例的黑线边框

  1. 按住 CTRL 键，并双击图例。
    或者
    单击右键，然后选择 Properties...
  2. 在打开的对话框中，选择 Frame 选项卡
  3. 在 Frame 下拉列表中选择 None。

注: 在一些旧的版本中，打开的对话框中没有选项卡，你可以在 Background 下拉列表中进行选择。

删除图例中符号上的红框

在 Origin 附带的某些图形模板中，用红框标示当前数据集的图例项。 在默认情况下，曲线拟合等分析工具都是对当前数据进行操作的，因此，这个红色的边框对选取数据集做分析操作很有帮助。并且这个红色框不会包含在图表导出中。

如果想关闭这红色矩形边框，可以按以下步骤：

在 Origin 2018 以及更新的版本中

  • 右键单击图例，取消勾选 Legend: Indicate Active Dataset

或者

  • 右键单击图例，选择 Legend: Update Legend...。在打开的对话框中，取消选择 Indicate Active Dataset 复选框。


在 Origin 2017 以及更早的版本中

  1. 选择菜单 Format: Page，打开 Plot Details 对话框。选择 Legends/Titles 选项卡，取消选择 Indicate Active Dataset
  2. 可选择通过 File: Save Template As 菜单，将该设置保存到模板，以供将来使用。
