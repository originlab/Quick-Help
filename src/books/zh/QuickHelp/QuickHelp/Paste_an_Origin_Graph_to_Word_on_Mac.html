5.35 FAQ-736 在 Mac 上运行 Origin 时，可以将 Origin 图形作为 OLE 嵌入对象粘贴到 Word 吗？

Paste-an-Origin-Graph-to-Word-on-Mac

Last Update: 7/23/2018


对象的链接与嵌入（OLE）允许用户将 Origin 图形作为嵌入对象复制并粘贴到其他应用程序（如 Microsoft Word 或 PowerPoint）。往后，您可以通过双击嵌入对象在 Origin 中打开它来进行进一步编辑。

目前仅支持在 Windows 操作系统中嵌入 OLE 对象，而在 Mac 操作系统中并不可以。尽管在 Mac 上运行的 Microsoft Office 应用程序组支持嵌入 OLE 对象，但它并不能作为其他应用程序（例如 Origin）的通用机制。

当您在 Mac 上运行 Origin 时您有以下两种选择：

  1. Microsoft Office 安装在 Windows 分区下
    在这种情况下，无论您是使用 Boot Camp 还是使用 Parallels 等虚拟机，都可以将 Origin 图形作为 OLE 嵌入对象粘贴到 Microsoft Office 应用程序中。
  2. Microsoft Office 或其他文档编辑应用程序安装在 Mac 下
    在这种情况下，您可以使用 Origin 中的 Edit: Copy Page as Picture 菜单项，然后将图片粘贴到另一个应用程序中。请注意，此新菜单项仅在 Origin 2016或更高的版本中可用。对于旧版本的 Origin，您需要将图形导出为图像文件，然后将图像插入到其他应用程序中。
