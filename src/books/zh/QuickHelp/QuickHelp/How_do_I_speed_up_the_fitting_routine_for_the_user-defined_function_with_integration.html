3.38 FAQ-267 怎样加快使用用户自定义积分函数的拟合进程？

Call-GSL-to-SpeedUp-IntegrationFit

Last Update: 11/7/2018

使用 Origin C 并且调用 NAG 库来构建积分函数，如果你觉得这样定义的用户自定义拟合函数运行的不够快。那么你也可以调用 GSL 库 来执行积分函数拟合操作。

关于怎么样用 Origin C 来定义这样的拟合函数，请参阅： 此教程。

关于怎样用外部 DLL 库来创建一个拟合函数，请参阅 使用外部 DLL 库。


FittingWithExternalDLL.opx 是一个 OPX 文件包, 在此目录中可以找到： <Origin Installation Directory>/Samples/Curve Fitting/。

  1. 将其拖放到 Origin 上以安装。
  2. 那么将在你的用户文件夹中创建名为 Fitting with External DLL 的文件夹，此文件夹中包括示例代码，帮助文件等。
  3. 在 FitFuncs.c 中注释掉的第三个示例拟合函数，将指导你怎样调用 GSL 库从而使用外部 DLL 库来执行积分函数拟合操作。
