3.55 FAQ-290 在哪可以找到可用的数学函数列表？

MathFunctions-in-SetColValue

Last Update: 7/21/2018

在 Set Values 对话框中可以定义数学表达式，此表达式用于填充列值或执行更多复杂的脚本。在工作簿中选择一列或特定的数据范围，然后去到菜单 Column: Set Column Values...来打开 Set Values 对话框。或者，在工作簿的列上右击从弹出的快捷菜单中选择 Set Column Values... 选项。

在此对话框中，可以单击 Search and Insert Functions 按钮来搜索函数，或转到菜单 Function: Search and Insert... 来打开 Search and Insert Functions 对话框，输入关键词搜索，并拖动滚动条来浏览搜索结果，然后在所需要的函数上双击以插入到 Formula 文本框中。你还可以单击函数描述（函数左边小箭头图标）去到函数说明文档页面。
