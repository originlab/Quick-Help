2.3 FAQ-410 如何将导入 ASCII 的设置保存到模板中？

Save-Import-Settings

Last Update: 4/24/2019

  1. 自定义导入 ASCII:
    1. 选择 Data: Import from File: Single ASCII 或 Data: Import from File: Multiple ASCII
    2. 在打开选择文件的 ASCII 对话框中, 并确认选上 Show Options Dialog 复选框。
    3. 选择文件，并点击 Open 按钮. 打开 impASC 对话框。
    4. 根据需要自定义导入设置。
    5. 点击 OK. （注: 在 Origin 9.0 以及更旧的版本中, 在点击 OK 前, 你需要点击 Dialog Theme 框右侧的三角按钮，并选择Save to <Sheet>。）
    6. 导入数据后, 导入设置则会保存在工作表中。
    7. 你可以进行进一步的分析操作。
  2. 连导入设置一起保存模板
    • 选择 File: Save Template As...
      或者
    • 如果在导入数据后，在工作表中进行了一些分析和计算，工作表带有绿色锁，就需要选择 File: Save Book as Analysis Template...
  3. 使用自定义的 ASCII 设置导入新数据。
    1. 选择 File: Open 加载已保存的模板
    2. 选择 Data: Import from Filet: Single ASCII 或 Data: Import from File: Multiple ASCII
    3. 在 ASCII 对话框中, 你会看到 <sheet> 显示在 Show Options Dialog 旁边, 这表示在工作表中存有导入设置。如果取消选择这个复选框然后导入文件，数据将会按照之前的自定义设置自动导入。如果选中复选框并单击 Open，则会打开 ImpASC 对话框并加载已保存的自定义设置。


关于保存模板的详细信息，请参考 [1].
关于分析对话框主题的详细信息，请参考 Dialog Themes.
