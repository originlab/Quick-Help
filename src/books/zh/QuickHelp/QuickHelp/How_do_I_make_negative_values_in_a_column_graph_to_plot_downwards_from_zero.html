1.57 FAQ-178 如何把负数柱状图绘制到 0 刻度线的下方？

Plot-Negative-Cloumn-from-0

Last Update: 11/14/2018

如果期望 X 轴位于 Y=0 处而不是最底端，可进行如下操作：

  1. 点击 格式：轴：X轴 菜单或者双击 X 轴以打开 坐标轴 对话框。
  2. 切换到 轴线和刻度线 选项卡，并确保在左侧面板中已经选中 下轴 图标。
  3. 在 轴位置 下拉菜单中选择 "在位置=" 。
  4. 在 百分比/值 框中输入 0 。
  5. 点击 确定 进行应用。


如果期望 X 轴显示在最底端，但同时负数柱状图显示为由 0 刻度线向下，则需要把 附加线 设置放到 Y=0 位置，操作如下：

  1. 点击 格式：坐标轴： X轴 菜单或者双击 X 轴以打开 坐标轴 对话框。
  2. # 切换到 轴线和刻度线 选项卡，并确保在左侧面板中已经选中 下轴 图标。
  3. 在 轴位置 下拉菜单中选择 下轴 。
  4. 点击切换到 网格 选项卡。
  5. 确保已经勾选 附加线 组中的 Y = 处的复选框，并输入 0。

如果期望的是当柱状图的数据小于某个特定值时绘制的柱子就朝下， 附加线 组中的 Y = 处输入期望的数值。
