1.151 FAQ-776 怎样编辑图形或分组图形的标注？

edit-annot-group

Last Update: 11/30/2018

要快速编辑同一个图或同一组分组图形内的所有标注(annotation), 首先按照 此页 的步骤对标注(annotation)的样式进行设置, 然后在 标注 对话框的 应用到 选项的下拉列表中，根据需要选择 绘图 还是 分组图形 即可。

Minimum Origin Version Required: 2016 SR0

Keywords:grouped plots, annotations, data labels, text labels,tags，图形分组，标注，数据标签，文本标签，标签
