3.3 FAQ-223 如何在拟合多个数据集时对参数设置相同的边界？

Fitting-Multiple-Data-Param-Bounds

Last Update: 7/15/2018

我们可以先共享多个数据集的拟合参数，设置边界， 然后取消共享，从而快速对多个数据集的拟合参数设置相同的边界

在 Parameters 选项卡中，如果想要共享 y0 参数，则可以为任意的 y0 参数勾选 Share 复选框 (只有在 Multi-Data Fit Mode 被选为全局拟合 (Global Fit)时才可用) 例如， y0 或 y0_2。则在勾选了 Share 处的复选框时，参数列表会更新为只显示 y0 。然后，如果点击 Bounds 选项卡，则可以从禁用状态点击 "< 或 <=" 控制或者改变 < 或 <= 的值，然后双击下边界来输入数值。对上边界重复此操作。然后回到 Parameters 选项卡，并去掉 Share 复选框中的勾。在 Bounds 选项卡中，将会看到 y0, y0_2, 等全部都有了相同的上边界和下边界值。
