5.47 FAQ-811 如何把 X 函数 (X-Function) 显示在菜单上？例如反卷积运算

Turn-on-XF-Menu


Last Update: 9/7/2018

有时你会发现一些内置的 X 函数 (X-Function) 很有用，但它们却没有出现在 Origin 菜单中，比如反卷积运算。若要打开这一功能：

  1. 通过选择 Tools: X-Function Builder 打开 X-函数编辑器。
  2. 选择 File: Open System X-Functions...。找到并载入想要启用的 X 函数。
  3. 选择 Tools: Tree View 以在树形模式下查看该函数。
  4. 展开 Usage Context 分支，从 Menus 下拉菜单中选择 Main menus。
  5. 选择 File: Save As... 将其保存为用户自定义的 X-函数。

注意：也可以用同样的方式在菜单中显示用户自定义的 X 函数。

Keywords:X-Function,menu，X 函数，菜单
