5.80 FAQ-1097 升级之后，导入工具栏上的导入单个ASCII文件和导入Excel两个按钮打不的对话框不一样

ascii-import-excel-button-change

Last Update: 4/19/2021

数据连接器是 Origin 的最新导入机制。由于用户已经过渡到使用连接器代替基于 X-Function 的老旧方式，OriginLab 决定重新实现 导入 工具栏上的 导入单ASCII文件 和 导入 Excel 两个按钮，让它们分别直接打开Text/CSV 和 Excel 的数据连接器。这个修改将随着 Origin 2021b 的发布而开始生效。

如果想让这些按钮回到原来基于 X-Function 的老旧的导入方式，可以通过改变系统变量@TIB。 

@TIB = 1; // 用 "导入单个ASCII文件" 按钮打开impASC
@TIB = 2; // 用 "导入 Excel" 按钮打开impMSExce
@TIB = 3; // 把impASC和impMSExcel重新连接到工具栏的按钮

关于修改系统变量的更多信息，请参阅此 FAQ。
