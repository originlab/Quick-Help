1.16 FAQ-128 是否可以载入外部调色板文件(.PAL)，并将颜色应用到绘图中？

Use-External-Color-Palette

Last Update: 4/15/2021

Origin 内置了大量的调色板文件 (.PAL)，使用调色板可以将颜色应用到带颜色映射的图中，例如等值线图，分组图， 还有那些图形元素 (例如数据点) 映射到另一个列的图。

Origin 2021进行了重大改进，包括支持更多调色板格式，拖放调色板安装以及新的颜色管理器，以帮助添加，修改和管理调色板文件。 该FAQ将涵盖了Origin软件的所有最新版本。

在Origin 2020b或者更早版本添加调色板

我们可以在 Origin 中调用外部的调色板文件。Origin 支持 Microsoft binary .PAL 和 JASC ASCII .PAL 两种格式的文件。使用下面工具， 可以浏览及应用外部的 .PAL 文件到当前绘图中。

  1. 调色板 调色板 的按钮 (在 样式 工具栏上)。
  2. 颜色选择器 (位于 绘图细节 对话框中)
  3. . 应用调色板 工具 (图：应用调色板到颜色映射 菜单打开)。


如果想将 .PAL 永久性的添加到 Origin 软件中，则需要将 .PAL 文件复制到 <User Files Folder>\Palettes\ 文件夹中 (如果此文件夹不存在，则以此名称新建一个文件夹)。在此后绘图的颜色选择中，此调色板颜色将会出现在选择列表中。

在Origin 2021以及之后的版本中添加调色板

Origin 2021 增加了对以下颜色调色板类型的支持：Scribus (.xml), CorelDraw 和 Corel PhotoPaint (.xml), Office Color Table (.soc), Adobe Color (.aco), Adobe Color Table (.act), Adobe Swatch Exchange (.ase), JASC PaintShopPro (.pal) 和 GNUPlot Palette (.pal).

另外，一个新的颜色管理器 (工具：颜色管理器) 是添加调色板更加简单：

  • 从网页导入 按钮通过输入URL添加调色板；或者从OriginLab的文件分享站点下载。
  • 从文件导入 按钮从本地文件去添加调色板。
  • 新建 按钮可以去重组你自己的自定义调色板。

拖拽添加调色板

您还可以通过拖拽来添加扩展的调色板格式列表，XML格式的“除外”。 如果要以被支持的XML格式添加调色板，请使用颜色管理器的从...导入按钮。
