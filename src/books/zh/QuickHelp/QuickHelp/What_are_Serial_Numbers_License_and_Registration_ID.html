6.42 FAQ-671 序列号 Serial Numbers，许可证 License 和 Registration ID 分别是什么？

SN-License-RegID

Last Update: 4/23/2023

在很老很老的版本里，你只需要一个7位的序列号就能够安装 Origin 并正常使用。但是，这样的好日子一去不复返了！现在，Origin 不仅有序列号 Serial Number，还有许可证 License，还有 Registration ID，等等等等……

这都是些啥玩意儿？

现在的序列号 Serial Number 是一个由数字和字母组成的16位字符串。新版的序列号不仅继承了原来的7位序列号机制，而且包含了更多的产品信息，例如版本、标准/专业版和语言（Origin 现在有英文版、日文版、德文版和中文版这 4 种语言版本）。当你向 OriginLab 寻求技术帮助的时候，或者咨询一个合适的升级方案的时候，我们会要求你提供有效的序列号。这串号码会告诉我们你现在所拥有的 Origin 是一个什么样的产品，以便给你提供更好的服务。

如果这世界少一些盗版，这样的一串字符或许已经足够。然而，当我们把我们的软件放在网上供全世界的用户下载的时候，我们不得不考虑日趋严重的盗版问题。为了保护我们的知识产权，我们引入了许可证文件 License。许可证文件除了包含序列号所包含的所有产品信息外，还包含了某些与电脑配置相关的信息。与序列号性比，许可证文件将这些信息进行了加密处理，确保他人无法直接读取。虽然许可证文件的机制还远远不够完善，但我们会不断改进，让软件激活过程变的更加简便快捷。

行吧。我们用序列号来判断产品类型，又把这些信息加密编码存储到许可证文件里，那为什么还需要一个 Registration ID？

Registration ID 实际上是你的 Origin 的“官网身份证”，如果你需要向 OriginLab 寻求技术帮助，我们要求你必须要先注册你的 Origin。

OriginLab 官网为用户提供了很多有用的资源。比如，你可以从我们的 在线帮助文档自己探索问题的答案，你也可以到我们的 用户论坛，与更多的用户交流 Origin 的使用经验。你可以在我们的官网上自行创建一个账号 User ID。如有需要，你可以随时登陆我们的网站，在 User Profile 页面修改个人资料。
