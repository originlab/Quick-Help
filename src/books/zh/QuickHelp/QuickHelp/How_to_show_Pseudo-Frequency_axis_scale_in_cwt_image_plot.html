3.123 FAQ-698 如何在连续小波 (cwt) 图像绘图中显示伪频率 (Pseudo-Frequency) 坐标轴刻度？

Pseudo-Frequency-in-cwt

Last Update: 7/31/2018

从 Origin 2015 开始，在 Continuous Wavelet 的输出图像绘图中，Origin 提供伪频率 (Pseudo-Frequency) 坐标轴的选项。如果想选用伪频率作为坐标轴刻度，在 cwt 对话框中选择 Use Pscudo-Frequency 复选框，

如果你觉得自动生成的伪频率刻度标签并不太理想， 你也可以手动计算出刻度线位置和标签，然后在 cwt 图像中使用这些刻度线位置和标签。 我们可以进行以下操作来达到这个效果。

  1. 在 cwt 对话框选择 Use Pscudo-Frequency 复选框，将会在源数据工作表中输出伪频率列。添加两个新列，例如，A 列和 B 列。在 A 列中填入你想用作为主刻度线标签的值。
  2. 计算出相应刻度值作为标签的位置，并将其填入 B 列。设伪频率列为 X，原刻度值（Scale）列为对应的 Y。打开 interp1 对话框，选择伪频率列和原刻度值（Scale）列作为 Input，然后设刻度线标签列，即 A 列为 X Values to interplate，执行插值以获得相应的刻度位置。 将插值结果输出到 B 列。
  3. 现在选择 cwt 图像图形，双击 Y 坐标轴，打开 Axis 对话框。在 Scale 选项卡， Major Ticks 分支下，设置 Type = By Custom Positions，并选择 B 列作为 Position。选择 Tick Labels 选项卡，在 Display 子选项卡中， 设置 Type = Tick-indexed dataset，并选择列 A 作为 Dataset Name。 此时，图像图形将显示正确的刻度线和标签。

Keywords:Wavelet，小波

Minimum Origin Version Required: 2015
