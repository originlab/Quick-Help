4.33 FAQ-393 如何通过对话框输入 Origin C 程序（LabTalk 脚本）的参数？

Dialog-input-parameter

Last Update: 7/9/2018

有时候您在使用自己的代码进行数值计算时需要输入一些参数，但在 Origin C 中不支持 C++ 的 cin 或 C 中的 scanf 等输入命令。在这种情况下，您可以使用 InputBox 或 GetN dialog 对话框来输入或简单或复杂的参数。

此外，如果您需要使用 LabTalk 脚本来构建一个获取输入数值的对话框，推荐使用 GetNumber 命令。

Keywords: InputBox, GetN, GetNumber, input, cin, scanf, 输入
