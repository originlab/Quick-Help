2.50 FAQ-484 Origin 为什么要为列指定列的绘图属性以及怎样使用列的绘图属性？

Col-Types

Last Update: 11/10/2021

工作表中每一列都有 绘图属性，以此告诉 Origin 在绘图或分析数据时怎样使用这些列数据。列的绘图属性有 X, Y, Z, Label, Disregard, X Error (X 误差棒), Y Error (X 误差棒), Group 和 Subject.。这些属性将在列标题的括号里显示。

  • 当工作表只包含一个 X 列时，Origin 会在 X 列和工作表中的所有在此 X 列右边的 Y 列之间创建关联。因此，当选择一个或多个 Y 列来绘图时，即使没有选择 X 列，Origin 默认使用关联的 X 列值来绘图。
  • 当工作表有多个 X 列时，Origin 会在 X 和 Y 列之间创建以下规则的关联：
    1. X 列右侧的所有Y列（直到下一个 X 列为止）都自动与该 X 列关联。Origin 通过将第一个 X 列表示为 X1 来说明关联规则，其中关联的 Y 列指定为 Y1。第二个 X 列表示为 X2，其关联的 Y 列为 Y2。
    2. 当选择任何 Y1 列进行绘图时，默认此 Y 列将与其相关联的 X1 列进行绘图，同样，当选择仅任何 Y2 列进行绘图时，默认此 Y 列将与其相关联的 X2 列进行绘图。
    3. 当绘画简单的2D图时，对 X1Y1，X2Y2 的关联方案会有例外情况，这时允许在工作表中任意选择一列 X 和一列 Y （提示：选择时按下 CTRL 键）而不需要理会 X 是否在 Y 的左边或右边。但要注意，这只适用于最简单的绘图类型。

请注意，当你从 Plot 菜单或工具栏按钮中选择绘图类型来绘图时， 如果你在工作表 选择 好了相关的列，Origin 将自动使用绘图属性对应的列进行绘图；如果在工作表中未选择任何数据，将打开 Plot Setup 对话框。在此对话框中可以对数据列指定列的绘图属性，而不是由系统自动指定。
