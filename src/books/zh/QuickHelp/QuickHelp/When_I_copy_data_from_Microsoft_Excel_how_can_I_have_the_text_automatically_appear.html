2.21 FAQ-433 怎样将 Excel 数据或纯文本复制粘贴到 Origin 工作表中并设置列标签行？

Copy-Header-from-Excel

Last Update: 4/16/2021

要有效地复制Excel数据或是无格式文本并粘贴到 Origin中，首先在 Excel 或文本文件中复制文本，然后点选 Origin 工作表的左上角的格子(第一列的第一个格子)进行粘贴（不要点击列标题行再粘贴）。如果这时候一起复制了 Excel 或纯文本文件中的标题行，那么粘贴时这些标题行也会位于工作表的数据部分。按照下面步骤可将数据的标题行设置为合适的列标签行：

  1. 单击一整行数据以选择此列包括标题行在内的全部数据。
  2. 在右键菜单中点选 Set As <label row> 或 Append to <label row>。

至此，在工作表中所选行的所有数据将被移动到各自的标签行中。
