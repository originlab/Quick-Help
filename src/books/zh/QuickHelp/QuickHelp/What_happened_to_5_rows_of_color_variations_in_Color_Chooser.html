1.19 FAQ-1076 颜色选择器中的5行颜色变化是怎么回事？

5-Rows-OF-Colors

Last Update: 4/17/2021

在颜色选择器的 更多颜色 那栏，如果您没看到5行颜色变化，那是因为颜色列表中的总颜色数大于16，这是 Origin 的局限。

例如，内置的 System Color List 只有13个颜色，所以它显示了5行颜色变化。

如果您在 自定义 那栏加载它，然后增加更多的颜色，使得总颜色数大于16。Origin 会自动保存它，于是下次您打开颜色选择器，在 System Color List 会多于16种颜色，因此5行颜色变化就会消失。

解决方法

  1. 点 自定义 那栏的向下箭头，然后选择 另存为，保存为一个新名字，因为我们不希望与内置的颜色列表搞混。
  2. 选择菜单 帮助：打开文件夹：用户文件夹，去到 \Themes\Graph\ 子文件夹，然后删除自动保存的 System Color Lists.oth 文件。


Keywords:more colors, color variations, system color list， 更多颜色，颜色变化

Minimum Origin Version Required: Origin 2017
