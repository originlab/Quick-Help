6.78 FAQ-1089 如果我时常看到许可证过期错误，我该怎么办？

FLEXnet-TIMEOUT-Environment-Variable

Last Update: 4/24/2023

如果您的客户端机器和许可证服务器机器之间的网络连接不稳定或缓慢，您可能无法将 Origin 连接到许可证服务，或者在 Origin 运行期间，您会不时收到许可证过期消息。

为了解决这个问题，

您可以在客户端计算机上创建系统环境变量 FLEXLM_TIMEOUT 以延长连接到许可服务器的时间。

方法1

对于 Origin 2023 或更高版本，以管理员身份运行 Origin，然后从主菜单中通过窗口：脚本窗口打开脚本窗口。 输入 

@FLTO=4000000

然后按 ENTER。

这会将系统环境变量 FLEXLM_TIMEOUT 设置为 4 秒。 值 4000000 是以微秒为单位（等于 4 秒）。 如果客户端仍然超时，您可以继续增加该值。 但是，如果您将该值增加到 10000000 并且仍然有问题，则问题的原因可能不是超时错误。

完成后，您可正常重启 Origin。

可以通过 Setup.exe 安装程序编辑 config.ini 文件来预设 FLEXLM_TIMEOUT。 有关详细信息，请参阅 此页。

方法2

添加或更新系统环境变量 FLEXLM_TIMEOUT 为 4000000（4 秒）。 然后重新启动起源。

详细步骤：

  1. 在 Windows 任务栏的搜索框中，键入控制面板，然后选择从结果中将其打开。
  2. 在控制面板的搜索框中搜索环境变量，然后从结果中选择编辑系统环境变量。
  3. 在系统属性对话框中，转到高级选项卡。 单击环境变量按钮。
  4. 在系统变量组中，如果您没有FLEXLM_TIMEOUT变量，请点击新建...按钮创建一个新的环境变量，如下 :
    • 变量名： FLEXLM_TIMEOUT
    • 变量值： 3000000 或更大。
  5. 单击“确定”。
