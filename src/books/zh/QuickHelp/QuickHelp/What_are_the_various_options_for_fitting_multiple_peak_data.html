3.63 FAQ-299 分析峰值数据的工具都有哪些？

Tools-for-PeakAnalysis

Last Update: 7/14/2018

Origin 提供多种峰分析的工具。下面为你列出了哪种工具适用于哪种类型的数据的一般准则。有关每种工具的更多详细信息，请参阅帮助文件。

Quick Peaks Gadget

  • 当图形窗口为当前窗口时，可从 Gadgets 菜单打开
  • 非常适合处理包含少量峰的数据图
  • 在图上选择感兴趣的区域，对这一段数据进行峰分析
  • 提供多种方来来检测和处理基线。
  • 可使用峰值函数来进行峰值拟合，并提供选项来控制拟合过程（仅限 OriginPro）
  • 将设置保存为主题 Theme，用以分析类似数据
  • 不支持批处理功能



Multiple Peak Fit Tool

  • 无论是原数据的工作表或是图形窗口为当前窗口时，都可以从菜单 Analysis: Peaks and Baseline: Multiple Peak Fit 打开。
  • 非常适合处理具有少量峰的数据，特别是那种峰与峰很好地分离开来的数据
  • 单击并选择峰中心，然后执行峰值拟合，如果你想更多的控制拟合进程，也可以切换到 NLFit 对话框中进行设置
  • 不支持基线处理功能



Peak Analyzer

  • 无论是原数据的工作表或是图形窗口为当前窗口时，都可以从菜单 Analysis: Peaks and Baseline: Peak Analyzer 打开
  • 非常适合处理具有很多峰的数据
  • 带有预览窗口的向导型的界面，可以逐步查看结果
  • 提供多种分析目标: 寻峰，峰积分，峰拟合（仅限 OriginPro）
  • 提供多种方法检测和处理基线
  • 将设置保存为主题 Theme，用以分析类似数据
  • 在拟合峰值的输出结果中，包含超过 15 个峰值属性
  • 可以选择使用不同的拟合函数来拟合不同的峰
  • 在拟合过程中，能够自由地在不同的峰之间共享拟合参数，也可以设定参数的边界和拟合的约束条件
  • 可以把 Peak Analyzer 的设置保存成主题 Theme，并在菜单 Analysis: Peaks and Baseline: Batch Peak Analysis using Theme 中调用以进行批量分析
