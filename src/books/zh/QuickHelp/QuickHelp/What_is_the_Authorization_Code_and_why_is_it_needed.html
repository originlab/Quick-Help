6.24 FAQ-26 Authorization Code 是什么？

Authorization-Code

Last Update: 7/5/2018

Authorization Code 是一个8位的字符串，它是 Origin 用来向官网申请许可证文件的一个代码。当使用 licensing 对话框来激活 Origin 的时候，Authorization Code 会自动传输到 OriginLab 的官网服务器上，并生成许可证文件。但是在需要手动获取许可证的情况下，例如，如果本机没有网络连接，这时候你就必须在官网上输入这个 Authorization Code 才能获得许可证。

注: Authorization Code 只在 Origin 2016 及以前的版本中才会使用到。
