1.83 FAQ-213 新版本的 Origin 怎么调用在旧版本中自定义的颜色？

share-custom-color

Last Update: 11/15/2018

如果你在 Origin 里自定义了一些颜色，然后在同一台电脑上升级了你的 Origin，现在想在新版本中使用这些颜色:

  1. 选择菜单 工具：转移用户文件，打开 转移用户自定义文件 对话框。
  2. 从 来源位置 下拉列表中选出老版本的用户文件夹（User Files Folder），并在 查看 中勾选 设置（菜单，按钮...）。
  3. 在左边的列表中选择文件 CustomColors.ini。点击 按钮把它添加到右边的列表中去。
  4. 点击 转移，把 CustomColor.ini 从旧的用户文件夹转移到新版本的用户文件夹里去。


你也可以直接打开用户文件夹（User Files Folder），手动复制文件 CustomColors.ini，粘贴到新版 Origin 的用户文件夹（User Files Folder）中。然后你就可以在新版本中调用这些自定义颜色了。
