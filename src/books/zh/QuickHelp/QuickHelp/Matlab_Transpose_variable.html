2.26 FAQ-438 导入 MATLAB 数据到 Origin 中时怎样转置变量？

Matlab-Transpose-Variable

Last Update: 7/12/2018

将 MATLAB 数据导入到 Origin 工作表窗口或矩阵窗口时，如果需要将变量进行转置，可以使用导入选项中的转置功能。可以按照以下步骤导入并转置一个或多个变量:

  1. 选择菜单 File: New:Worksheet (或者 File: New:Matrix) 创建一个矩阵窗口。
  2. 单击 Import: Matlab(Mat), 选择一个 MATLAB 文件。单击 OK 打开对话框。
  3. 展开对话框里的 Import Option 选项，在 Transpose Variables 下拉列表中选择 All Variables 或 1*N dimension Variables 。单击 OK 关闭对话框。


Keywords:MATLAB, data import, transpose，数据导入，转置

Minimum Origin Version Required: 2015 SR0
