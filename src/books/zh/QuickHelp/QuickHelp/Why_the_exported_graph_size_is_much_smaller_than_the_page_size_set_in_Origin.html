1.233 FAQ-1005 为什么导出的图片尺寸比Origin设置的页面尺寸小很多？

export-graph-size-small

Last Update: 10/24/2019

当导出图为矢量格式比如emf，pdf，eps或者wmf，你会发现导出的图片尺寸变小了。

解决方法 1

在导出图形 对话框展开输出设置：高级分支，选择的保持尺寸。

解决方法 2

在很少见的情况下上述方法可能不起作用。

原因可能是

图形是由第三方软件比如Microsoft Word或者PowerPoint的内嵌图形重新保存的。也就是说，图形是从Origin复制到Word/PPT，再从Word/PPT重新打开并由Origin另外保存。

要重设页面尺寸,

打开窗口：脚本窗口，在问题图形为当前图层情况下运行以下LabTalk代码： 

doc -e p { page.info.system.export.auto=1;}
