2.42 FAQ-476 怎样控制列标签行的显示与隐藏？

Customize-HeaderRow

Last Update: 7/16/2018

在工作簿标题栏或空白区域右击从弹出菜单中选择 View 选项，你可以选择要显示的列标签行，如长名称，单位等。

或者，去到菜单 Format: Worksheet （或按 F4 键）打开 Worksheet Properties 对话框。在 View 选项卡点击 Edit Column Label Rows 按钮，打开 Column Label Rows 对话框，然后选择要显示的列标签行或者通过拖动 Labels 左侧的空白单元格来对列标签行进行排序。注意，你可以在此对话框中添加或删除 用户自定义参数的标签行 。

以下是你在 Origin 工作表中的任意行右击时可选的选项：

  • 清除，隐藏 和 清除并隐藏列标签行。
  • 使用更有意义的名称来命名列标签行，这将使得列标签行变成用户自定义列标签行。
  • 将行属性设置为长名称，单位，添加到注释等。
