4.32 FAQ-392 如何加密 Origin C 文件？

Encrypt-Origin-C-File


Last Update: 7/9/2018

如果您需要加密 Origin C 文件， 您可以直接创建一个加密的 OCZ 文件或者将已有的 *.c 或者 *.cpp 文件保存为 *.ocz 文件。

强烈建议您备份未加密的 Origin C 文件，因为在加密后一旦您忘记密码，是无法通过已加密的 *.ocz 文件恢复源文件的。

创建一个加密的 OCZ 文件

在 Code Builder 中选择 File:New 或者在 Standard 工具栏中点击 New 按钮 。这一操作将会打开 New File 对话框，此时您可在左上方的面板上选择 OCZ File，然后在 File Name: 中输入文件名称，最后点击 OK 按钮。这时将会弹出一个 Password 对话框， 在 New Password 中输入密码并且在 Confirm New Password 再次输入确认密码，点击 OK 按钮完成密码设置。Origin 将会默认把新创建的 OCZ 文件保存到用户文件夹的 Origin C 子文件夹中。

将已有的源文件保存为加密文件

首先在 Code Builder 中打开源文件，然后选择 File:Save As... 打开 Save as 对话框。在文件名右边直接加上 .ocz 后缀并点击 Save 按钮。此时将会弹出 Password 对话框，输入密码和确认密码后，则可将此源文件转为加密的 *.ocz 文件。


当您在 Origin 中打开加密的 OCZ 文件，您可以通过选择菜单 File: Save As 并在 Save as type 下拉列表中选择保存的类型将该 *.ocz 文件重新保存为非加密的 *.c 或者 *.cpp 文件
