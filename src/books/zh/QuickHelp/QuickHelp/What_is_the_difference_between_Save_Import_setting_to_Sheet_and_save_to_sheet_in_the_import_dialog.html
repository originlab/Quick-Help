2.98 FAQ-969 在导入对话框中的 "Save Import setting to Sheet" 和 "save to <sheet>" 有什么区别?

Save-to-Sheet-Theme-Diff

Last Update: 9/20/2018

您可能会注意到，在使用导入对话框导入数据时， Dialog Theme 弹出菜单中有两个类似的选项可将导入设置保存到工作表。 例如，当您通过 ImpASC 对话框导入 ASCII 文件并单击 Dialog Theme 右侧的三角形按钮时，可以看到以下两个菜单项：

  • Save Import setting to Sheet
  • Save to <sheet>

这两者有什么区别？

“Save to <sheet>” 将导入设置保存为 X-Function 对话框主题，当您使用这个 X-Function 导入时，可以重新加载该主题。 但是，如果您不使用 X-Function 导入数据，而是使用其他方式，例如，拖放数据文件到 Origin 以进行导入，则无法使用这些设置。

于是，Origin 引入了 “Save to <sheet>” 来弥补这一不足。 它将导入设置保存到工作簿页面，可以在将文件拖放到工作簿窗口时自动使用
