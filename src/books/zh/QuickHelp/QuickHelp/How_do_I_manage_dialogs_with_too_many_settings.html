5.13 FAQ-352 如何管理对话框中所有的可用选项？

Manage-Complex-Settings

Last Update: 8/7/2018

相对于旧版本 7.x 而言，Origin 8.x 的对话框具有更多的设置选项。这些选项初看起来可能会觉得数量太多，所以在这里向您介绍一些管理和利用这些选项的方法：

  1. 使用默认设置。
    这个方法适用于大部分情况。当您打开一个对话框时，可以在 Dialog Theme 菜单中选择 System Default 将对话框恢复为系统的默认设置。
  2. 使用 Change Parameters 更新结果。
    如果您对获得的结果不满意，无需再次对数据执行分析。您可以点击图形或者输出列中的锁头图标（当您在至少一列数据中将重新计算模式设为自动或者手动，您即可看见该锁头图标），选择 Change Parameters 再次打开该对话框，修改该对话框设置即可更新现有的结果。
  3. 仅展开您所需的分支。
    对话框的设置是按照树形节点进行排布的，您可以很容易地展开或者折叠这些选项。在某些对话框中，它们还可能被排布在多个选项卡中。您可以仅展开您所需的分支或者仅访问某些选项卡来设置您想修改的某个项目。
  4. 保存您的设置使其能重复使用。
    一旦您为您所需修改了某些选项，那么您可以将这些设置存为一个主题文件使其日后能重复使用。请参考这个教程了解更多相关的内容。
  5. 保存已含有结果的工作簿为分析模板将其应用于相似的数据进行重复分析。请参考这个教程了解更多相关的内容。
  6. 使用 Batch Processing 为多个文件或者多组数据执行分析。请参考这个教程了解更多相关的内容。
