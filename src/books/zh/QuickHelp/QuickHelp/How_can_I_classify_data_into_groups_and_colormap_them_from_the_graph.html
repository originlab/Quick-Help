3.125 FAQ-721 如何在图形中直接对数据进行分组，并按分组设置颜色？

Classify-Data-from-Graph

Last Update: 7/30/2018

在 Origin 中，可以很容易地对图形上的数据进行分类和设置。如需要对图形上的数据点分类，并按分组设置颜色，我们可以：

  1. 点击激活图形窗口，在菜单上选择 Gadgets: Cluster ，将感兴趣区域(ROI) 框放置在图形上，并打开 Cluster Gadget 对话框。
  2. 移动并调整 ROI 框以圈出是某个组的数据点群集。点击在 Cluster Gadget 对话框上的 Create Categories 按钮 ，以打开 Create Categorical Value 对话框。 在Category Group Name 中输入类别组名称，并在 Category 输入类别标识符值。
  3. 选择 Color Data by Category 复选框，则可以对分组中的数据点设置颜色映射。
  4. 点击 OK。分组列将添加到源数据表中，在 ROI 内的数据点被标记为 Category 值，并且数据点将使用分组列的值作为颜色索引。（你可以通过双击绘图打开 Plot Details 对话框，来查看颜色设置。）
  5. 重复步骤2，直到所有数据点都被分类到不同的组 （由不同的 Category 值标识）。


如果想了解更详细的信息，请参考 此教程。

Minimum Origin Version Required: 2018 SR0
