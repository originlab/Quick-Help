5.16 FAQ-355 什么是用户文件夹？我的个人设置将存放在哪里？

UFF-and-Personal-Settings

Last Update: 10/11/2022

本台计算机的每个用户在第一次运行 Origin 时都会要求其指定某个路径作为用户文件夹，而该文件夹将成为这位 Origin 用户的默认位置。首选项以及用户自定义文件（如主题，模板和项目文件）都将保存在此文件夹。注意，设置的路径必须具有用户的读写权限。

注:

如果您将用户文件夹设置在一个可移动硬盘中，那么必须保证该硬盘在 Origin 使用的时间里始终可用。如果您将用户文件夹设置在一个网络硬盘，那么网络的性能将影响 Origin 的性能。

如何找到用户文件夹?

以下的方法可帮助您找到用户文件夹：

  1. 从 Origin 2017 开始，选择菜单 帮助: 打开文件夹: 用户文件夹，那么该文件夹将会在 Windows 资源管理器中被打开。
  2. 选择菜单 设置: 选项。用户文件夹的路径将显示在 系统路径 标签页。
  3. 选择菜单 窗口: 脚本窗口，输入 %Y= 并按下 Enter 键，那么该路径将显示在脚本窗口里。
  4. 从 Windows 的开始菜单选择 All Programs: OriginLab: Origin ##: User Files Folder 这一快捷方式将可打开用户文件夹。
