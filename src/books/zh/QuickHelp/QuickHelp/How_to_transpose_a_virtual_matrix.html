1.173 FAQ-838 如何转置虚拟矩阵？

Transpose-VM

Last Update: 7/3/2018

在项目中的所有虚拟矩阵(Virtual Matrix) 都列在 Virtual Matrix Manager 对话框中。如果要转置一个虚拟矩阵，你可以：

  1. 从菜单上选择 Tools: Virtual Matrix Manager，打开 Virtual Matrix Manager 对话框。你将会见到所有的虚拟矩阵都列在左边的虚拟矩阵框中。
  2. 选择需要转置的虚拟矩阵，右击并选择 Duplicate as XY Transpose，则可以复制并转置所选择的虚拟矩阵。


Keywords:Virtual Matrix, transpose, duplicate，虚拟矩阵，转置，复制

Minimum Origin Version Required: Origin 2015
