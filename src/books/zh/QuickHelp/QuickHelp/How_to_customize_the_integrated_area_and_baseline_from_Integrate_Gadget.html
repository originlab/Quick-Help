3.122 FAQ-696 如何自定义曲线下的阴影积分区域？

Customize-IntegratedArea-from-Gadget

Last Update: 10/23/2019


当我们使用积分快捷工具 (Gadgets: Integrate...) 来计算曲线下的积分面积，你可以选择是否把积分区域作为阴影区域显示在图上

注, Origin 2018b 以及以后版本，可以支持画出一次画出多个阴影区域；对于之前的版本，积分快捷工具每次只能画出一个阴影区域。

  1. 点击黄色感兴趣区域 (ROI) 框右上角的三角按钮，并选择 Preferences。
  2. 在 Integration Preferences 对话框中，选择 Integration 选项卡。启用 Keep the shading color after New Output ，并选择所需的阴影颜色
  3. 点击 OK。阴影区域将显示在图形区域上，标记 ROI 框下的积分区域。
  4. 点击三角按钮，选择 New Output(O) (或其他新的输出选项)，此时，即使移动 ROI 框，阴影区域仍会保留在图上。
  5. 你可以移动 ROI 框到另一个感兴趣的范围，添加类似的积分区域。



在 Origin 2016 SR1 以及更高的版本中，如果想自定义灰色阴影区域的外观，你可以：

  • 点击选中阴影区域，然后使用 Style 工具栏的按钮更改 Fill Color, Fill Pattern, 等;

或者

  • 右键单击选择阴影区域，然后选择 Properties ，通过 Object Properties 对话框改变属性。


在较早的 Origin 版本中，如果想自定义灰色阴影区域的外观，你可以：

  1. 选择菜单 Edit: Button Edit Mode ，然后点击选中阴影区域。
  2. 使用 Style 工具栏的按钮更改 Fill Color, Fill Pattern, 等； 或者，右键单击选择阴影区域，然后选择 Properties ，通过 Object Properties 对话框改变属性。
  3. 完成自定义阴影区域后，重新选择 Edit: Button Edit Mode，关闭按钮编辑模式。
