4.66 FAQ-1075 怎样默认从镜像存储库下载Python包？

Download-Python-Package-from-MirrorRepository

Last Update: 4/20/2021

通过以下步骤下载和安装Python包时：

  • LabTalk 命令pip install SomePackage
  • 菜单连接: Python包 > 安装 按钮

可以按照下述方法改变默认镜像存储库。

  1. 在C:\ProgramData\pip\pip.ini找到 pip.ini 文件(尽管是隐藏的):
    如果不存在请新建一个。
  2. 打开 pip.ini, 按如下配置镜像存储库： 
    [global]
index-url = https://mirrors.aliyun.com/pypi/simple
  3. 保存 ini 文件。
