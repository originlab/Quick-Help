1.183 FAQ-869 为什么导出图形中的线看起来很不平滑呢？

why-line-jagged

Last Update: 4/24/2019

Origin 支持抗锯齿功能，使图形中的线条更为平滑。但在某些情况下，在导出图形时会关闭抗锯齿功能。

导出 2D 图形时
  • 如果将 2D 图形导出为栅格图像类型 (JPG, PNG, PCX, BMP, PSD, GIF, TIFF 等)
导出的图形是否具有抗锯齿的效果，取决于你是否对图形中的线启用抗锯齿功能。 例如，如果将 2D 图形导出为.TIFF 图像类型，并希望其中的线条看起来平滑，那么可以在 Origin 中点击 Enable/Disable Anti-Aliasing 按钮 )， 为 2D 图形启用抗锯齿，然后再导出图形。
  • 如果将 2D 图形导出为矢量图像类型 (WMF, EMF, DXF, EPS 等)
即使关闭抗锯齿功能，你也会发现导出图形的质量已足够好。反而，当抗锯齿启用时，图像质量并没有什么改善，但尺寸却会明显增大。 因此，在默认情况下，当你导出矢量图像类型，无论图形中的抗锯齿功能是否启用，Origin 都会在导出时关闭此功能。
注: 如果你想在导出矢量图像类型时仍启用抗锯齿功能，你可以选择菜单 Preferences: System Variables 打开 Set System Variables 对话框。 在对话框中，设 EAA 为 1。
导出 3D OpenGL 图形时

无论将 3D 图形导出为栅格还是矢量图像类型，都会使用抗锯齿功能。
