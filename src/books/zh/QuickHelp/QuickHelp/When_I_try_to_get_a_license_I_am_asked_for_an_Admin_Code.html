6.32 FAQ-35 当我在获取许可证文件的时候，要求我填入一个 Access Code。这是什么？

Admin-Code-for-Licensing

Last Update: 7/9/2018

Access Code 是多用户单机版许可证在激活时候必须提供的一个验证码。它可以在您购买 Origin 时我们提供给您的 License Certificate 上面查找到。Origin 软件的管理者在首次注册这个序列号并建立许可证获取机制 License Access Restrictions 的时候，必须要提供 Access Code。Origin 软件的最终用户在获取许可证文件来激活 Origin 的时候，也必须提供 Access Code。如果在创建许可证获取机智的过程中有任何疑问，请联系 OriginLab 技术支持部门。
