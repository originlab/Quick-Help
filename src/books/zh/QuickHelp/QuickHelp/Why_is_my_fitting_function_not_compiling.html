3.15 FAQ-240 怎样解决拟合函数编译失败的问题？

FitFuction-Not-Compile

Last Update: 7/20/2018

使用 Origin C 自定义函数时，意味着在定义函数时需要遵循 C 语言语法。请注意，此时字母区分大小写。因此，在 参数名称 编辑框中输入的内容必须与 函数 列表框中的内容相匹配。

例如，如果参数名称为 P1，则不能在函数代码中使用 p1。此外，Origin C 中的大多数数学函数名称都是小写的，因此编译 Sin（）函数将失败（应使用 sin（））。但是，Origin 预先定义了 pi，Pi 和 PI 都代表圆周率 。
