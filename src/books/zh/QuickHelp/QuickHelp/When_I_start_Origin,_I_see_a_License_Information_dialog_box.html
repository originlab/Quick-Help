6.34 FAQ-37 当我打开 Origin 的时候，我看到一个 License Information 对话框弹出，让我输入 Host Name 或者 IP Address，这是怎么回事？

Lic-Info-Dialog-Shows-when-Start

Last Update: 7/10/2018

当你打开 Origin，看到一个 License Information 对话框弹出，要求填入服务器的 Host Name 或者 IP Address，这证明你的 Origin 是一个网络版，需要连接到服务器才能正常使用。你需要在这个对话框中填入许可证服务器的 Host Name 或者 IP Address，点击 Updadte 按钮成功连接到服务器才能完成激活。如果你已经成功连接到服务器，此前也能正常运行 Origin，但是现在忽然又看到这个 License Information 对话框弹出，这有可能是因为服务器出了问题，或者是网络连接不畅导致你的电脑无法连接到服务器。请联系你的 Origin 管理员解决这个问题。
