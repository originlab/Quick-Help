2.13 FAQ-423 如何使用从文件名和头文件中提取的变量？

Retrieve-and-Use-Variables

Last Update: 7/24/2018

Origin 提供了许多方法可以提取和使用文件的元数据。本文档将重点介绍如何使用带有对话框的工具从导入的数据文件中提取变量并使用它们，这也是最灵活简便的方法。

  1. 首先，你必须从数据文件中提取出元数据。为此，我们推荐使用的工具是 Import Wizard。想要了解详细的使用步骤，请参阅这个视频教程的第 3 部分，“如何导入数据”。
  2. 定义好变量并导入数据文件后，有几种方法可用调用所提取的变量:
    • 要将变量值插入工作表的单元格中，请右键单击单元格并选择 Insert Variables （在编辑模式下，也可以按 CTRL + H 键）打开 Insert Variables 对话框。有关使用此对话框的详细信息，请参阅这个文档。
    • 要将变量值插入图形页面上的文本对象中，选择 Tools 工具栏上的 Text 工具 ，在图形窗口的任意空白处单击，然后右键单击并选择 Insert Info Variable（或按 CTRL + H 键）。有关使用此对话框的更多信息，请参阅这个文档。
    • 存储在工作簿的 page.info 节点中的变量可在某些计算操作中使用，例如，在使用 Set Values 来给该工作簿中的某些列赋值时，可以调用某些变量。这些变量将在 Workbook Organizer 或（数据图形或者工作表的） Insert Variables 对话框的 page 层列出。有关更多信息，请参阅这个博客。
