6.38 FAQ-41 我已经安装了 FLEX 许可证服务器并用它来管理另一个软件，我怎么把 Origin 的许可证服务加进来？

Use-Existing-FLEXnet-Server

Last Update: 4/27/2020

  1. 下载 Origin 供应商守护进程 vendor daemon: Origin vendor daemon (orglab)
  2. 按照 基本安装指南把 Origin 的许可证服务管理加进已有的 FLEX 服务器中去。
  3. 想获取更多信息，参阅: 这个文档的 "Using an existing FLEXnet Server to manage Origin" 这一部分。


Keywords:licensing, concurrent, Flexnet, server, existing FLEX server, 网络版, 浮动版, 服务器

Minimum Origin Version Required: 8.5SR0
