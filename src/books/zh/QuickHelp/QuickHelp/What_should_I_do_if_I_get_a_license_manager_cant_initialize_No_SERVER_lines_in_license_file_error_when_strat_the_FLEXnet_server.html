6.100 FAQ-1131 当启动 FLEXnet 服务器时报错 "license manager: can't initialize:No SERVER lines in license file."，如何处理?

LIC-Server-Error-1366

Last Update: 5/10/2022

如果您不能启动授权服务器，并且日志文件里报了以下的错误：


12:07:37 (lmgrd) license manager: can't initialize:No SERVER lines in license file.
12:07:37 (lmgrd) License Path: "C:\OriginLicenseServer\Licenses\orglab.lic"
12:07:37 (lmgrd) FlexNet Licensing error:-13,66


请试一下以下的解决方法：

  1. 用 记事本（或其它文本编辑器）打开授权文件。确认 SERVER 这一行存在。如果不存在，请从 OriginLab 的官网获取授权文件。
  2. 通过菜单 文件: 另存为 重新保存授权文件。在 另存为 对话框，确保编码为 ANSI。

Keywords:license, -13,66, can't initialize:No SERVER lines, Cannot connect to license server system，授权，不能初始化，没有服务器，连接不到服务器系统
