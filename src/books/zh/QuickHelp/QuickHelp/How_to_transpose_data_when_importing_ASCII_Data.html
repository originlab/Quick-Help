5.41 FAQ-746 如何在导入 ASCII 数据时进行数据转置？

Origin-Import-Transpose

Last Update: 10/27/2022

Origin 支持在导入时转置 ASCII 数据。

您可以:

  1. 点击 文件: 新建:工作簿 (或者 文件: 新建:矩阵) 创建一个新的工作簿（或者矩阵簿）。
  2. 选择菜单 数据: 从文件导入: 单个ASCII文件 或者 数据: 从文件导入: 多个ASCII文件，然后选中需要导入的 dat/txt 文件。注意，在该选择文件窗口中请务必勾选 显示选项对话框。点击 确定 按钮
  3. 此时在打开的对话框中，在 File Structure 分支下，勾选 Transpose Data 并点击 OK 按钮关闭对话框。

Origin 2019b 之前的版本，ASCII导入对话框的菜单在 文件: 导入: 单个/多个ASCII文件

而在 2019 之前，新工作簿创建的菜单在 文件: 新建: 工作表。

注意: 导入向导 并不支持该项功能

Keywords:Import ASCII,data import, transpose, 导入 ASCII 数据，数据导入，转置
