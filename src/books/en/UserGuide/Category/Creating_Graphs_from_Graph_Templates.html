8.3.1 Creating Graphs from Graph Templates

Plot-Graph-Template

See more related video:Making Multi-Y Plots Using Templates

Every Origin graph window -- even an "empty" one that contains no plotted data -- is constructed using an Origin graph template. Each of Origin's 100+ graph types is assigned a default graph template. When you select worksheet or matrix data and click one of the graph type buttons on Origin's graph toolbars, or a plot type icon under Plot Menu, you call that default template file (file extension = .OTP or OTPU), invoking a suite of saved plot settings. These settings control such things as line color, symbol type, page size, number and position of layers, etc.
