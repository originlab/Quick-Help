6 Importing and Exporting Data

Import-Export-Data

See more related video:Importing Data

You can import ASCII and binary files into Origin for graphing or analysis. In addition, Origin ships with many X-Functions that are capable of importing a number of third party files. If your files are not of any supported format, you can use the Import Wizard to import them as User Defined files or write your own Origin C routines or X-Functions to perform the import.
