8.12.8 The Plot Setup Dialog Box

GraphRef-PlotSetup-Dialog


The plot setup dialog box is useful for a variety of plotting tasks including creating graphs, modifying of plot type, adding plots to or removing plots from the graph, grouping or ungrouping of plots, and editing of the plot range.

This dialog consists of three, vertically arrayed panels (see the screenshot below): the top panel, the middle panel and the bottom panel. You can use the and buttons to show/hide panels as needed.
