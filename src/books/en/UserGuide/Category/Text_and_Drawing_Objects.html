8.6.5 Object Properties Reference

Ref-TextLabel-DrawingObjects

The Object Properties dialog may be opened by these methods:

  • Press CTRL while double-clicking on the object,

or

  • Right-click on the object, and select Properties... in the context menu,

or

  • Click on the object to select it, and then choose Format: Object Properties... from the menu.
