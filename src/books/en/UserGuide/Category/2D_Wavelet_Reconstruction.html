18.13.2 2D Wavelet Reconstruction

2D-Wavelet-Reconstruct

This function reconstructs a 2D signal from four matrices: the approximation coefficients, the horizontal detail coefficients, the vertical detail coefficients and the diagonal detail coefficients.

For correct reconstruction, the wavelet type chosen for this function should correspond with the choice for the wavelet decomposition that is used to produce the input coefficient matrices.

To use 2D wavelet reconstruction:

  1. Make a matrix book active.
  2. Select Analysis: Signal Processing: Wavelet: 2D Reconstruction from the Origin menu.
