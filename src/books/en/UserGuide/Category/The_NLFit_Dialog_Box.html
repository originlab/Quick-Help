15.3.2 The NLFit Dialog Box

NLFit-Dialog

The NLFit dialog box contains two panels (see the picture below): The upper panel is used to do the fitting settings, while the lower panel is used to monitor the fitting procedure. The buttons between these two panels are used to define/edit new functions and control the fitting procedure. The top and bottom panels are toggled on/off with these buttons .
