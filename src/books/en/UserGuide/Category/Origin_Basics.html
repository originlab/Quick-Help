Origin-Basics

See more related videos: Overview of Origin, Introduction to Origin, Introduction to OriginPro

This section opens with a tour of the Origin workspace. We point out key features of the workspace, briefly discuss their function, and -- where appropriate -- point you to more in-depth discussions of use of these features. We then introduce you to Origin's child window types, each of which has specific applications within the Origin workspace. Thirdly, we include a brief synopsis of Origin's file types, again, noting the specific uses for each of those file types.

This section is intended primarily to orient the new user, though a few topics -- Numbers in Origin or UserGuide:Dates and Times in Origin -- may provide useful information for the long-time Origin user.
