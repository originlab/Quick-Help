17.1.10 Partial Correlation Coefficient (Pro Only)

Partial-Corr-Coef


Introduction

Partial correlation measures the linear relationship between two random variables, after excluding the effects of one or more control variables.

Handling Missing Values

When there are missing values in your data, the Partial Correlation Coefficient dialog provides the option to delete the cases pairwise or listwise.

Performing Partial Correlation Coefficient

To open the Partial Correlation Coefficients dialog box from the menu:

  • Select Statistics: Descriptive Statistics: Partial Correlation Coefficient...
