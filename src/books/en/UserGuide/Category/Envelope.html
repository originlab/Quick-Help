18.16 Envelope (Pro Only)

Envelope

Envelope is a curve “enveloping” the source data. It is tangent to every peak in the source dataset. Origin obtains the upper, lower, or both envelopes of the source data by applying a local maximum method combined with a cubic spline interpolation.


To Use the Envelope Tool
  1. Make a workbook or a graph active.
  2. Select Analysis: Signal Processing: Envelope from the Origin menu.
