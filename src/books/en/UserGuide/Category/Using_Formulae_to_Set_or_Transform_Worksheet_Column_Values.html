4.6.8 Using Formulae to Set or Transform Worksheet Column Values

Wks-Set-Column-Values

See more related videos:Setting Column Values, Set Column Values

The Set Values dialog can be used to generate data for a worksheet column or multiple worksheet columns.

To open the Set Values dialog box, you can:

  1. Select one column or multiple columns or a range of cells in worksheet column(s).
  2. Then, from the menu, choose Column: Set Column Values or Column: Set Multiple Column Values.

or

  1. Select one column or multiple columns or a range of cells in worksheet column(s).
  2. Right-click on the worksheet column(s) and choose Set Column Values or Set Multiple Column Values from the shortcut menu.

or

  1. Right click on the F(x) column label cell and select Open Dialog. This is only available for the single column.
Note: Beginning with Origin 2017, you can use a simplified "spreadsheet cell notation" in the Set Values dialog and the F(x)= label row. Please see this topic for more information.
