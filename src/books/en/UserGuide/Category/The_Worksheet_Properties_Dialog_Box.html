4.8.3 The Worksheet Properties Dialog Box

WksProperties-Dialog

The elements of the worksheet -- the row and column headings, the grid lines, the size and the format, etc. -- are controlled primarily from the Worksheet Properties dialog box. The Worksheet Properties dialog may be opened by selecting Format: Worksheet from the menu or pressing F4.
