8.3.6 Plotting Mathematical Functions

FuncPlot

A function plot displays a 2D curve or 3D surface described by a formula or set of formulas. Functions can be plotted into one of Origin's standard graph templates or into an active graph window that provides additional customized settings.

Origin now supports plotting 2D functions, 2D parametric functions, 3D functions and 3D parametric functions
