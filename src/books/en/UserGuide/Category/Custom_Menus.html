27.5 Custom Menus

Custom-Menus

If you have scripts that are frequently used, consider putting them on the menu for easy access. Origin 8.1 offers a Custom Menu Organizer tool which allows you to assign these scripts to menu items easily. With this tool, you can create up to 3 main menus and corresponding sub-menus. You can also hide the built-in menus as you wish. What's more, you can save the configuration of the customized menus as an .omc file, which can be easily shared with your friends and co-workers.
