5.6.1 Setting Matrix Values

Set-Mat-Value

This Set Values dialog can be used to generate data for a matrix object.

To open the Set Values dialog box, you can

  1. Activate a matrix.
  2. From the Origin menu, choose Matrix: Set Values.

Or

  1. Highlight a matrix.
  2. Right-click on the matrix and choose Set Matrix Values from the shortcut menu.
