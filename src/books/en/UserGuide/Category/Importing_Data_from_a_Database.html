6.9 Importing Data from a Database

Import-From-Database


See more related videos:Import Data from a Database

Origin supports connecting to databases by ODBC and OLE DB. Additionally, Origin includes a simple SQL Editor to add connection strings and generate SQL queries within Origin.

For a synopsis of Origin's database connectivity features, see these OriginLab blog posts:

  • The SQL Editor Dialog
  • Connecting to a Database and Importing Data
  • Things to Know About Origin Database Connectivity

Earlier 32-bit versions of Origin supported a "Query Builder" dialog. This dialog is no longer supported.
