12 Common Analysis Features

Common-Analysis-Feature

Origin's analysis dialog boxes, and a number of other data processing dialog boxes, share a common set of controls and features. These include interactive data range selection controls, various output options including a general analysis results reporting mechanism, automatic recalculation of results (where applicable) and the saving or selection of a dialog box "Theme" -- a suite of settings that you expect to use repeatedly.
