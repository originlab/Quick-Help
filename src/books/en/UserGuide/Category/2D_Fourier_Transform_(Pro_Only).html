18.11 2D Fourier Transform (Pro Only)

2D-Fourier-Trans-Pro

Fourier transforms can be used to analyze the frequency spectra of 2D signals. Conversely, inverse 2D Fourier transforms can be applied to 2D frequency spectra in order to reconstruct 2D signals in the time-domain. In OriginPro, the 2D discrete Fourier transform (2D DFT) and its inverse transform (2D IDFT) are implemented using the fast Fourier algorithms, 2D FFT and 2D IFFT, respectively.
