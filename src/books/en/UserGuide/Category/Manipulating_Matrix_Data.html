5.6 Manipulating Matrix Data

Manipulate-MatData

As is the case with any Origin dataset, you may perform a wide variety of math operations on a matrix, either from the command line, using the Set Matrix Values dialog box, or programmatically. The following topics discuss a number of routine matrix operations that are available from the Matrix and Analysis menus.
