20.5.1 Image Analysis

Image-Analysis

Origin provides histogram, profiling and information tool for image analyzing.

The histogram tool is a simple yet useful tool for image analyzing. By analyzing the distribution of the levels, the histogram tells us whether the digitizing of the image is good. In some cases, where the intensity of an object is significantly different from the background, the histogram tool could help to decide the threshold for separating the object from the background.

Image profiles plot can be used to analyze the changes along a line, while the image information tool provides the basic information, such as width, height and color-depth, of the image.
