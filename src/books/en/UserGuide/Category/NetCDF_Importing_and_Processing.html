21 NetCDF Importing and Processing

NetCDF-Importing-Processing

A NetCDF file is a self-describing, scalable format file for storing arrays of variables. Designed as a flexible format for exchange of scientific data, it is readily adaptable to geographic studies and is widely used in atmospheric and climate science. Starting with version 2021b, Origin's NetCDF Connector has been improved to handle the kinds of datasets used in geographic-based studies.


If there are multiple NetCDF files, and you hope to merge them to one before import and manipulate, you can refer to this page.
