5.2 Matrixbooks

Matrix-Books

The matrixbook window is created from a customizable template. All Origin matrixbook templates are named with the OTM file extension.

  • The matrixbook template stores, among other things, the number of sheets in the matrixbook, the names of the matrixsheets, the matrix dimensions, data type, math functions, data import-handling instructions and worksheet style and format options.
  • The Matrix Organizer is designed to help you keep track of the information that is stored in the matrixbook window.
