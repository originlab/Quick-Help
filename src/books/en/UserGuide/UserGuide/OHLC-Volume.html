29.14.5 OHLC-Volume

Data Requirements

Select exactly five Y columns (or a range from five columns). The leftmost Y column contains the opening prices, the second Y column contains the highest prices, the third Y column contains the lowest prices, the fourth Y column contains the closing prices, and the rightmost Y column contains the trade volume. If there is an associated X column, X column supplies X values; otherwise, sampling interval of the Y column or row number is used.

Creating the Graph

Select desired data.

Select Plot > Specialized : OHLC-Volume.

or

Click the OHLC-Volume button on the 2D Graphs toolbar.

Template

OHLCVOL.otp (installed to the Origin program folder).

Notes

The OHLC-Volume plot is actually a stacked plot containing an upper Japanese Candlestick plot that displays the opening, highest, lowest, and closing prices of a security over a given time interval, and a lower column plot that shows the trade volume.
