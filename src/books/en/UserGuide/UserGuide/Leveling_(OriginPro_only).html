20.5.2.12 Leveling (Pro only)

Image-Leveling

The imgLevel function further darkens the dark values and lightens the light values. With this function, users can specify the increasing percentage of shadows and light within their image. An image's contrast and intensity are also enhanced by using the histogram of the image and a color distribution-based algorithm. To use this function

select Image: Adjustment: Leveling from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function imgLevel in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help imgLevel
