20.5.5.1 Average

Image-Average

Average filtering is a routine that blurs the image. When an image is averaged, the color of each pixel is changed to the average color of its neighboring pixels. You can specify the length of the square region for the averaging neighborhood. To use this function,

select Image: Spatial Filters: Average from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function imgAverage in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help imgAverage
