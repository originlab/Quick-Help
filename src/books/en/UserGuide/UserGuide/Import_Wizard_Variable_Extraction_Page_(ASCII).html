6.4.8.3.2 Import Wizard, The Variable Extraction Page (ASCII)

ImpWiz-ASC-ExtractVarPage

Controls on this page are used primarily to specify a method for extracting variables from the header section of your import files. Each of the first three check boxes adds a page to the Import Wizard. You may extract variables using the controls on any single page or you may use the controls/pages in concert.

Contents

  • 1 Extract Variables from File Names and Headers
  • 2 File Info

Extract Variables from File Names and Headers

Specify location of variable names and values using delimiters

Selecting this option adds the Variable Extraction by Delimiter page to the Wizard.

Specify location of variable names and values using character positions

Selecting this option adds the Variable Extraction by Position page to the Wizard.

Specify delimiter and Wizard scans for variable names and values

Selecting this option adds the Header Variables page to the Wizard. This is the Origin 7.5 method which requires you to specify a single delimiter and then Origin will scan for the variable names and values automatically.

Call a user defined Origin C function

Selecting this option allows you to choose a source file and an Origin C function in the file, which extracts the variables in the way you want.

Add variables to page info

Selecting this option stores variables in the page info storage area of the target workbook (available only if one of the other Variable Extraction page options is selected).

Create User Parameter rows for each variable

Selecting this option create user parameter rows for each variable in the target workbook. (available only if one of the other Variable Extraction page options is selected).

File Info

Save file info. in workbook

Specify whether to save the file information to the workbook, which can be seen from the Workbook Organizer. Storing the file information requires extra time. So, you may consider clearing this option to improve the importing speed when importing multiple files.
