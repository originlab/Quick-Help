28.1.13 The Object Edit Toolbar

Object-Edit-Toolbar

The Object Edit toolbar is available when one or more graph objects are selected in the active window, or when more than one picture is selected in a layout page, even when one or more layers are selected in a graph page. The toolbar provides buttons to align the selected objects/pictures, and to change the drawing order of the objects/pictures/layers. Buttons are provided to group objects so that you can move or align the group. Additionally, buttons are provided to change the drawing order of objects relative to data plots.

Another way to select layers in a multi-layer graph is to press CTRL or SHIFT while clicking on the layer icons in the upper left-corner of the graph page.

If a single graph layer is selected, clicking this toolbar will align it relative to page.
