29.2.11 Error Band Graph

Error-Band-Graph

Data Requirements

Select Y and YEr columns of values (or subranges from Y and YEr columns). If there is an associated X column, X column supplies X values; otherwise, sampling interval of the Y column or row number is used.

Creating the Graph

Select required data.

From the menu, choose Plot > Basic 2D: Error Band.

Template

ERRORBAND.OTP (installed to the Origin program folder).

Notes

If you want to plot different upper and lower error bands, prepare two columns for YEr+ and YEr-, respectively.

  1. Highlight all Y, YEr+ and YEr- columns and create a error band graph.
  2. Double click on the plot to open the Plot Details dialog.
  3. Select the error band plot on the left, and activate the Error Bar tab on the right. Select Plus or Minus in the Direction group.

Since Origin 2020, we have adjusted this graph template to make the color of error connected lines to be Auto(which means it will follow the symbol color). And the fill color of the band has also been set to Auto(which means its color will follow the color of error connected lines in Error Bar tab) but with the transparency 50%.
