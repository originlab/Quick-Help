20.5.4.3 Flip

Image-Flip

The imgFlip function flips the image horizontally or vertically. To use this function,

select Image: Geometric Transform: Flip: Horizontal from the Origin menu;

or,

select Image: Geometric Transform: Flip: Vertical from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function imgFlip in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help imgFlip
