13.2.1 The File Menu

XFBuilder-FileMenu

The File menu is used for creating, opening and saving X-Functions:

New X-Function Wizard

Opens the New X-Function Wizard. The wizard guides steps you through the process of creating a new X-Function. Note that this menu command duplicates the New X-Function Wizard toolbar button .

Open User X-Functions

This menu command lets you to browse for an X-Function to edit. It duplicates the Open X-Function from OXF File toolbar button.

Open System X-Functions

This menu command opens the \X-Functions subfolder under the Origin program folder and lets you to browse for a built-in X-Function.

Save As

Save the current X-Function to an OXF file. Note that you cannot save an X-Function to the Origin program folder (reserved for built-in X-Functions).

Save

Save the current X-Function. Note that you cannot save an X-Function to the program folder (reserved for built-in X-Functions). This menu command duplicates the Save OXF File toolbar button.

Close

Close the X-Function Builder dialog box.
