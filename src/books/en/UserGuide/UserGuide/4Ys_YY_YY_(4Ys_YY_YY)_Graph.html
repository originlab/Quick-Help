29.6.10 4Ys YY-YY

Data Requirements

Select at least one Y column (or a range from at least one Y column). Ideally, select at least four Y columns (or a range of at least four Y columns). If there is an associated X column, X column supplies X values; otherwise, sampling interval of the Y column or row number is used.

Creating the Graph

Select the required data.

Select Plot > Multi-Panel/Axis: 4Ys YY-YY.

or

Click the 4Ys YY-YY button on the 2D Graphs toolbar.

Template

4Ys_YY-YY.OTP (installed to the Origin program folder).

Notes

The four Y axis graph contains four layers:

Layer 2, layer 3 and layer 4 are linked to layer 1.

The X axis in layer 2, layer 3 and layer 4 has a straight one-to-one link with the X axis in layer 1.

If four columns of values are selected (or a range of four columns), then one data plot displays in each layer. Each data point in the data plot is connected by a line. The default line connection between points is a straight line. The data points are displayed as symbols.
