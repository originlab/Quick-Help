Data Requirements

  • Matrix: A matrix sheet. Multi-object sheet is supported.

or

  • Worksheet: worksheet data arranged in XYZ columns, or virtual matrix.

or

  • Image: An Image window. Multi-frame image is supported. Refer to this page below to see how to flip through all frames.

Creating the Graph

Activate the matrixsheet or select required data from worksheet or the image window

From the menu, choose Plot > Contour: Image Profiles

or

Click the Image Profiles button on the 3D and Contour Graphs toolbar.

Template

PROFILES.OTP (installed to the Origin program folder).

Notes

  • Matrix images/images are plotted as an image plot which includes X and Y projections.
  • The interface allows you to dynamically create horizontal, vertical, and arbitrary line segment profiles of the image data. Click and drag the text label of the profile line to select and move the lines.
  • For more information on profiling images, please refer to this page in the User Guide.
