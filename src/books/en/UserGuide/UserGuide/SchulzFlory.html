Function

Brief Description

Schulz Flory distribution function to describe relative ratios of polymers after a polymerization process.

Sample Curve

Parameters

Number: 4

Names: y0, xc, w, A

Meanings: y0 = base, xc = center, w = width, A = amplitude

Lower Bounds: xc > 0, w > 0

Upper Bounds: none

Script Access

nlf_SchulzFlory(x,y0,xc,w,A)
