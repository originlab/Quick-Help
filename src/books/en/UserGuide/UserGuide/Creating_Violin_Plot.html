Create-Violin-Plot

Creating Violin Plots

Origin supports seven violin plot graph template, you can create these violin graph type by the memu directly.


To create a violin plot:

  1. Highlight one or more Y worksheet columns (or a range from one or more Y columns).
  2. Select Plot: 2D: Violin Plot: Violin Plot/ Violin with Box/ Violin with Point/ Violin with Quartile/ Violin with Stick/ Split Violin/ Half Violin

Each Y column of data is represented as a separate violin plot. The column names or labels supply the X axis tick labels.


Customizing Violin Plots

To customize the violin plot display:

  1. Double-click on the violin plot.

or

  1. Right-click and select Plot Details.

Both actions open the Plot Details dialog with the violin data plot icon active on the left side of the dialog. The violin plot controls are available on tabs on the right side of the dialog.

For more information on Box Chart customizations, see the following topics:
