9.3.7.34 The (Plot Details) Layout Tab for Chord and Hierarchical Edge Bundling

Direction

Specify how to arrange the annular nodes in the angular direction, clockwise or counter-clockwise.

Nodes Start at

Specify the angle of the start node. When you set it to 0, the nodes will start at the position of 3 o'clock.

Radius(%)

Specify the radius of the diagram. When you set it to 100, it will be equal to the height of the layer.

Ratio Layout(Chord Only)

Show input and output in one link, so you can check this to show the exchange between two nodes within one link.

Pre Mode, Post Mode(Chord Only)

When charting flows with Ratio Layout, one can chart weights prior to assessing net flows; or weights that result from net flows. The Pre Mode and Post Mode radio buttons allow display of either scenario.

One can see in the simple example above that in Post Mode, node B gains 5 units (going from 40 to 45), node A loses 5 units (going from 40 to 35) while node C shows no net change.

Order(Chord Only)

Controls to arrange the nodes and links.

Node Order by

Specify how to order the nodes on the circle.

Order the nodes by:

  • None:Just arrange the nodes according to the original order in the source data.
  • Name:Arrange the nodes according to their name in the alphabetical order, ascending or descending.
  • Total Weight:Arrange the nodes according to the total weight(including the input and output both), ascending or descending.
  • Output Total First:Arrange the nodes according to their input total, ascending or descending.
  • Input Total First:Arrange the nodes according to their output total, ascending or descending.
  • Output/Input Ratio:Arrange the nodes according to the ratio of output and input of each node, ascending or descending.
  • Input/Output Ratio:Arrange the nodes according to the ratio of input and output of each node, ascending or descending.

Link Order by

Specify how to order the links outputting from and inputting into current node. If you selected Ratio Layout, this control will be unavailable.

  • Output First:Arrange the output links firstly, starting from the first tick of current node.
  • Input First:Arrange the input links firstly, starting from the first tick of current node.


Once you've selected Input First or Output First, you can decide to arrange the input/output links according to:

  • Weight Ascending:Arrange the input/output links in ascending order, according to the weight value of the links.
  • Weight Descending:Arrange the input/output links in descending order, according to the weight value of the links.
  • Target Nodes Position:Arrange the input/output links according to the position of target node. In this case, the self-links will be arranged at the first place.

Link Drawing Order

Specify the drawing order of the links within the circle of the chord diagram.

  • By Node and Link Order: Follow the order of nodes and links in source data.
  • Larger Link on Top: Put the wider links on top
  • Smaller Link on Top: Put the narrower links on top

Bubbles for Leaves(Hierarchical Edge Bundling Only)

Specify whether show bubbles for all leaves. Once you checked this box, the Symbol tab will show to let you control the bubbles.

Nodes for Subset(Hierarchical Edge Bundling Only)

Specify whether show nodes for leaves or subsets. Once you checked this box, the Node tab will show to let you control the nodes.
