4.8.3.4 The (Worksheet Properties) Print/Export tab

View

Specify what to show when export or print current worksheet. Frame Thickness and Margin (%) are used for menu Edit: Copy: Copy as Table(HTML/EMF).

Margin

Specify the margin for printed/exported page by millimeters (mm) or by inches (inch).

Note: Select File :Print and then click Options button, you can open the Print Options dialog Box. In this dialog, you can do above settings too.
