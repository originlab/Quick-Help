20.5.2.6 Function Look up Table (Pro only)

Image-Function-Look-up-Table

The imgFuncLUT function calculates a lookup table array using the predefined function. It then uses the lookup table to change the intensity values of a bitmap image. To use this function,

select Image: Adjustment: Function LUT from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function imgFuncLUT in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help imgFuncLUT
