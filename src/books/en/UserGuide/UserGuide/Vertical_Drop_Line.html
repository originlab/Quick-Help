Data Requirements

Select one or more Y columns or a range of data from at least one Y column. If there is an associated X column, X column supplies X values; otherwise, sampling interval of the Y column or row number is used.

Creating the Graph

Select required data.

From the menu, select Plot > Basic 2D : Vertical Drop Line.

or

Click the Vertical Drop Line button on the 2D Graphs toolbar.

Template

DROPLINE.OTP (installed to the Origin program folder).

Notes

Drop lines can be turned on or off from the Drop Lines tab in the Plot Details dialog box.
