Option

There are four radio boxes (Resize、Move、Swap and Align)here. You can check one of them to set the Size/Position tab.

Resize

Unit

Use this drop-down list to choose the unit for the Resize branch. Options are:

  • % of Page
    When this is selected, the Width and Height values are in percentages of the graph page width and height respectively.
  • inch, cm, mm, pixel, point
    One of these units is selected, this unit will be the unit of both Width and Height.
  • % of Reference Layer
    When this is selected, the Width and Height values are in percentages of the Reference Layer's width and height respectively.
Reference Layer

This drop-down list specifies an existing layer as the reference layer. It is available when the Unit is % of Reference Layer.

Width

This textbox is used to set the width of the selected layer(s).

Height

This textbox is used to set the height of the selected layer(s).

Apply

This button is only enabled when you have changed some settings of Resize. When this button is clicked, the settings that you changed will be applied in preview graphs, and you can see it in the Preview panel.

Move

Unit

Use this drop-down list to choose the unit for the Move branch. Options are:

  • % of Page
    When this is selected, the Left and Top values are in percentages of the graph page width and height respectively.
  • inch, cm, mm, pixel, point
    One of these units is selected, this unit will be the unit of both Left and Top.
  • % of Reference Layer
    When this is selected, the Left and Top values are in percentages of the Reference Layer's width and height respectively.
Reference Layer

This drop-down list specifies an existing layer as the reference layer. It is available when the Unit is % of Reference Layer.

Left

This determines the position of the layer(s) relative to the left side of the page or the reference layer, depending on your selection in the Unit drop-down list. The value of this textbox is used to set the left distance to move.

Top

This determines the position of the layer(s) relative to the top of the page or the reference layer, depending on your selection in the Unit drop-down list. The value of this textbox is used to set the top distance to move.

Apply

This button is only enabled when you have changed some settings of Move. When this button is clicked, the settings that you changed will be applied in preview graphs, and you can see it in the Preview panel.

Swap

First Layer

Use these drop-down lists to select the index numbers of the first layer to be swapped.

Second Layer

Use these drop-down lists to select the index numbers of the second layer to be swapped.

Reorder

Check this box to swap the order of two layers in addition to swap the positions of dataplots within each layer.

Apply

This button is only enabled when you have changed some settings of Swap. When this button is clicked, the settings that you changed will be applied in preview graphs, and you can see it in the Preview panel.

Align

Reference Layer

Use this drop-down list to specify the index number of a layer, with reference to which the selected layer(s) in Layer Selection panel can be aligned.

Direction

The drop-down list determines align direction. Options include Left, Top, Right and Bottom.

Apply

This button is only enabled when you have changed some settings of Align. When this button is clicked, the settings that you changed will be applied in preview graphs, and you can see it in the Preview panel.
