Data Requirements

Select at least one Y column (or a range from at least one Y column). Ideally, select at least three Y columns (or a range of at least three Y columns). If there is an associated X column, X column supplies X values; otherwise, sampling interval of the Y column or row number is used.

Creating the Graph

Select the required data.

Select Plot > Multi-Panel/Axis: 3Ys Y-YY.

or

Click the 3Ys Y-YY button on the 2D Graphs toolbar.

Template

3Ys_Y-YY.OTP (installed to the Origin program folder).

Notes

The three Y axis graph contains three layers:

Layer 2 and layer 3 are linked to layer 1.

The X axis in both layer 2 and layer 3 has a straight one-to-one link with the X axis in layer 1.

If three columns of values are selected (or a range of three columns), then one data plot displays in each layer. Each data point in the data plot is connected by a line. The default line connection between points is a straight line.
