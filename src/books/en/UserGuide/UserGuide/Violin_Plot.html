Data Requirements

You need to have one or more worksheet columns to create such graph, except Split Violin.

For Split Violin, you need to select at least two columns, and last column should be contains 2 categories only.

Creating the Graph

Violin Plot

  1. Select required data.
  2. From the menu, select Plot > Statistical: Violin Plot.


Violin with Box

  1. Select required data.
  2. From the menu, select Plot > Statistical: Violin with Box.
or
Click the Violin with Box button on the 2D Graphs toolbar.


Violin with Point

  1. Select required data.
  2. From the menu, select Plot > Statistical: Violin with Point.


Violin with Quartile

  1. Select required data.
  2. From the menu, select Plot > Statistical: Violin with Quartile.


Violin with Stick

  1. Select required data.
  2. From the menu, select Plot > Statistical: Violin with Stick.


Split Violin

  1. Select required data.
  2. From the menu, select Plot > Statistical: Split Violin.


Half Violin

  1. Select required data.
  2. From the menu, select Plot > Statistical: Half Violin.

Notes

  • Each Y column of data is represented as a separate Violin plot. The column names or labels supply the X axis tick labels.
