Function

Brief Description

Two sites competition function to describe the competition of a ligand for two types of receptors.

Sample Curve

Parameters

Number: 5

Names: A1, A2, log(x0_1), log(x0_2), f

Meanings: A1 = top asymptote, A2 = bottom asymptote, log(x0_1) = first center, log(x0_2) = second center, f = fraction

Lower Bounds: none

Upper Bounds: none

Derived Parameters

The span:

Span=abs(A1-A2)

The concentration at half response:

EC50_1=10^(logx0_1)

EC50_2=10^(logx0_2)

Script Access

 nlf_competition2(x,A1,A2,LOGx0_1,LOGx0_2,f)

Function File

FITFUNC\COMP2.FDF
