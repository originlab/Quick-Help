Data Requirements

  • Worksheet: Select XY columns or XYZ columns (full or subrange); or a virtual matrix.

or

  • Matrix: A matrix of Z values.

Creating the Graph

Activate the matrixsheet or select required data from worksheet.

From the menu, choose Plot > Contour: Heatmap with Labels.

or

Click the Heatmap with Labels button on the 3D and Contour Graphs toolbar.

Template

Notes

This graph is constructed in the same way as the basic Heatmap, but each block is labeled.

  • When the Heatmap is created from XY worksheet data, default labels are bin counts. Change the Quantity to Compute by clicking the green lock icon on the Heatmap and choosing Change Parameters.
  • When the Heatmap is created from XYZ worksheet data, default labels are mean Z values in each bin. Change the Quantity to Compute by clicking the green lock icon on the Heatmap and choosing Change Parameters.
  • When the Heatmap is created from a virtual matrix or matrix, labels are the Z values for each block (row/column intersection).
  • To format numeric labels on the Heatmap (e.g. to control decimal places, display as %, etc.), double-click on the plot to open Plot Details, click the Label tab and set Numeric Display Format.
