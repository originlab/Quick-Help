20.5.2.10 Hue

Image-Hue

Colors perceived by the human eyes can be described using the following terms:Hue, Saturation and Brightness. Hue determines what the color is and the specific tone of color. It can be changed from 0 to 360 degrees on the color wheel. To change the hue of an image, you can use the imgHue X-Function. To open the dialog of this function,

select Image: Adjustments: Hue from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function imgHue in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help imgHue
