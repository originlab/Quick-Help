25.3.2 The Export button

MATLABConsole-Export

The Export button in the top right corner of this dialog brings up the MATLAB Export dialog (see figure).

This dialog lists all variables in the current Origin project. You can assign a name for the variable in MATLAB using the Name column, then select desired Origin worksheet columns or matrices, and transfer the selected Origin data to MATLAB. The Column tab lists all variables in Origin worksheet, and the Matrix tab lists variables in Origin matrix.

  • Browse button, this button opens the Column Browse dialog, and you can select the variables to be exported to Matlab manually.
  • Browse button, this button opens the Column Browse dialog, and you can select the variables to be exported to Matlab manually.
  • Remove button, you can remove the Origin variables which you don't want to exported to Matlab with this button.
  • Export button, exports the selected variables to the Matlab.
  • Close button, closes the Matlab Export dialog.
