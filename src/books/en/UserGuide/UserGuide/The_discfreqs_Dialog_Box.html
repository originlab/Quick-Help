Recalculate

Recalculate

Controls recalculation of analysis results

  • None
  • Auto
  • Manual

For more information, see: Recalculating Analysis Results

Input Data Form

Indexed

The input data consists of one or more sample data columns and one factor (grouping) column.

Raw

The input data consists of one or more sample data columns.

Input

If Indexed is selected for Input Data Form,

Group Range

Enter the group data. The grouping column will be set as categorical if most of the column values are text. So you can easily reorder the output columns.

For help with range controls, see: Specifying Your Input Data

Data Range

Enter the data to perform discrete frequency analysis.

For help with range controls, see: Specifying Your Input Data

If Raw is selected for Input Data Form

Input

Enter the data to perform discrete frequency analysis.

Quantities to Compute

Count

Tally counts on each discrete value.

Cumulative Count

Calculate the cumulative Count on each discrete value.

Relative Frequency

Calculate the relative frequency of each discrete value. Please see the algorithm here.

Cumulative Frequency

Calculate the cumulative frequency of each discrete value. Please see the algorithm here.

Frequency in

Specify the form of the frequency. Available when either the Relative Frequency or the Cumulative Frequency check box is checked.

  • Fraction
    The frequency is in the form of fraction.
  • Percent
    The frequency is in the form of percent.

Case Sensitive

Distinguish between uppercase and lowercase when deciding whether two values are the same.

Exclude Missing Values

Check this option to exclude both missing values and empty values.

Show Zero Count Categories

Specify whether to list the category and show 0 as its count data when a category is not found.

Sort Data by

Sort Data by

Controls output columns sorting order:

  • Ascending Counts
  • Descending Counts
  • Ascending Values
  • Descending Values
  • Categorical Order

When Categorical Order is selected, the output column order will follow categorical order customized through (Column Properties) Categories tab.

Output Results

Specify where to output the results. The identifier in the result sheet will use Long Name of input columns if there is no dupliacted Long Name.

For help with the range controls, see: Output Results
