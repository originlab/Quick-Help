Recalculate

Recalculate

Controls recalculation of analysis results

  • None
  • Auto
  • Manual

For more information, see: Recalculating Analysis Results

Input Data

For help with range controls, see: Specifying Your Input Data

Data Range

The input data range. You can input one or multiple columns

Grouping Range

The data range containing grouping information. Different grouping values indicate that the data in the corresponding cells are from different groups. You can input only one column for the grouping range.

The grouping column will be set as categorical if most of the column values are text. So you can easily reorder the output columns.

Basic Statistics

Specify the output the descriptive statistics. For computation details, see the statistics on column page

Quantiles

Specify to output the quantiles values, which include minimum, 1st quartile(Q1), Median, 3rd quartile(Q3), maximum, interquartile range (Q3 - Q1), range (maximum - minimum)

Moments

Specify to output the moments, which include N, N missing, mean, standard deviation(SD), SE of mean, lower/upper 95% CI of mean

More Moments

Specify the output more moment values, which include variance, sum, skewness, kurtosis, uncorrected sum of squares, corrected sum of squares, coefficient of variance, mode

Basic Tests

Mean Test

Select from the drop down list to perform t-test or z-test for mean or not to perform the mean test

T-Test/Z-Test

Test Mean The hypothetical mean to be tested. When you enter a value, null and alternate hypotheses are adjusted accordingly.
Standard Deviation Specify the standard deviation value, which will be used in the z-test. Available only when Z-Test is selected
Null Hypothesis The null hypothesis to be tested.

(Mean = Test Mean)

Alternate Hypothesis The alternate hypothesis.

Options:

Mean <>Test Mean

Used for two-tailed test. Test whether the sample mean differs from the hypothesized mean.

Mean >Test Mean

Used for upper-tailed test. Test whether the sample mean is greater than the hypothesized mean.

Mean <Test Mean

Used for lower-tailed test. Test whether the sample mean is less than the hypothesized mean.
Confidence Interval(s) Compute confidence interval for the mean.

For computation details, see Algorithms (Distribution Fit)

Confidence Level(s) in % Specify space-separated confidence levels, (1-)%. Available when Confidence Interval(s) is selected.

Chi-Square Test for Variance

Test Variance The hypothetical variance to be tested.
Null Hypothesis The null hypothesis to be tested (i.e. sample variance = Test Variance).
Alternate Hypothesis Options:
Variance <>

The sample variance differs from the hypothetical variance.

Variance >

The sample variance is larger than the hypothetical variance (upper-tailed test).

Variance <

The sample variance is less than the hypothetical variance (lower-tailed test).

Confidence Interval(s) Compute confidence intervals for sample variance.
Confidence Level(s) in % Specify space-separated confidence levels (%). This option is only available when Confidence Interval(s) is selected.

Distributions

Distribution Type

Specify whether continuous distribution or discrete distribution to be performed

Continuous Distribution 5 type of continuous distribution are available from the Distribution branch
Discrete Distribution 2 type of discrete distribution are available from the Distribution branch

Distributions

Select one or multiple distribution to be fit for the input data

Continuous Distribution

Normal LogNormal Weibull Exponential Gamma

Discrete Distribution

Binomial Poisson

Plots

Box Chart Output the box chart(s) to the result sheet
Histogram Output the histogram(s) to the result sheet
Cumulative Distribution Function Plot Output CDF plot(s) to the result sheet
Probability Plot Output the P-P plot(s) to the result sheet
Significant Level Specify the confidence level for confidence bands of the Probability(P-P) Plot
Arrange Plots of Same Type in One Graph Select this option to arrange plots from multiple data ranges in one graph, which helps to compare the multiple data range. Please note that in the graph histogram will be stacked and with same bin size.

Goodness of Fit

Komogorov-Smirnov Perform the Komogorov-Smirnov test.
Komogorov-Smirnov (Modified) Perform the Modified Komogorov-Smirnov test.
Anderson-Darling Perform the Modified Anderson-Darling test.

Output Settings

Distribution Fit Report Specify the worksheet to output the distribution fit results.
Plot Data Specify a worksheet to output the plotted data (Origin generates tabular data when plots are created).
