27.7.3.10 The (Options) System Path Tab

Options-Dialog-SysPath-Tab


This tab displays useful information about system paths.

Change, copy and Reset the Path

Change

Change the path of the specified system folder. Note: Only the User File Folder, Auto save, and Group Folder paths can be changed.

Copy

Copy the path of the specified system folder. All paths can be copied.

Reset

The path of the specified system folder returns to its default.

System Path

User File Folder (UFF)

Select this path, and then click Change to browse to or enter a new User File Folder (UFF). You may choose to copy all files from the original User Files Folder to the new one. To use the newly specified User Files Folder, restart Origin.

To open the current UFF, you can click Copy button to copy and paste the path to Windows Explorer, or select Help: Open Folder: User Files Folder menu.

Autosave

Select this path, and then click Change to browse to or enter a new Autosave Folder path.

To open the current Autosave folder, you can click Copy button to copy and paste the path to Windows Explorer, or select Help: Open Folder: Autosave Folder menu.

Application Data

Show the path of the Application Data Folder.

Program Folder

Show the path of the Program Folder.

To open the current Program folder, you can click Copy button to copy and paste the path to Windows Explorer, or select Help: Open Folder: Program Folder menu.

Group Folder

Select this path, and then click Change to browse to or enter a new Group Folder path.

License

Show the path of License Folder.

To open the License folder, you can click Copy button to copy and paste the path to Windows Explorer, or select Help: Open Folder: Program Data Folder menu and then open the \License\ sub-folder.

Project Backup

Show the path of Origin Project Backup Folder. The Project Backup folder is used to save the previous version(the one before the last save) of current saved project. There is a system variable @USKT which can be used to set the keeping time of these backup project files. The default time is 7 days.

To open the Backup folder, you can click Copy button to copy and paste the path to Windows Explorer, or select Help: Open Folder: Project Backup Folder menu and then open the \Backup\ sub-folder.

Unsaved Files

Show the path of Unsaved Files Folder. The unsaved files folder is used to save the projects which has not been saved when it is closed. There is a system variable @USKT which can be used to set the keeping time of these unsaved project files. The default time is 7 days.

To open the Unsaved folder, you can click Copy button to copy and paste the path to Windows Explorer to open the \Unsaved\ sub-folder.
