19.1.10.7 Peak Analyzer, Baseline Treatment Page

PeakAnalyzer-BaselineTreat


This page is available when Goal = Integrate Peaks, Find Peaks or Fit Peaks. Use page controls to subtract the baseline from the spectrum data and specify whether to perform fitting to the baseline. Note that the latter is only available when the Start Page Goal = Fit Peaks.

  • Menu Command: Analysis: Peaks and Baseline: Peak Analyzer: Open Dialog
  • Window Types: Workbook, Graph

Dialog Theme

Dialog Theme See Peak Analyzer Themes.

Baseline Treatment Controls

Auto Subtract Baseline If this check box is selected, clicking the Next button or the icons in the wizard map to step forward in the Peak Analyzer will cause the subtraction to be performed automatically.
Auto Rescale Specify whether to auto rescale the preview graph of Peak Analyzer. The status of this check box can be remembered in a theme file.
Subtract Now Click this button to perform the baseline subtraction manually.
Undo Click this button to undo the baseline subtraction.
Rescale Rescale the preview graph of Peak Analyzer.
Fit Baseline with Peaks Specifies whether to perform fitting to the baseline when performing fitting to the peaks. It is available only when the following two conditions are satisfied.
  • Fit Peaks is selected for Goal in the Start page
  • Straight line is selected for Baseline Mode drop-down list in the Baseline Mode page, or User Defined is selected in the Baseline Mode page and Fitting is selected for Connect By in the Create Baseline page.
Fit Function It shows the function which is used to fit the baseline. It is available only when the following two conditions are satisfied.
  • Fit Peaks is selected for Goal in the Start page
  • Constant is selected for Baseline Mode drop-down list in the Baseline Mode page, or User Defined is selected in the Baseline Mode page and Fitting is selected for Connect By in the Create Baseline page.
Fix Baseline Parameters Specifies whether to fix the parameters of the function which is used to fit the baseline. It is available only when the following two conditions are satisfied.
  • Fit Peaks is selected for Goal in the Start page
  • Constant is selected for Baseline Mode drop-down list in the Baseline Mode page, or User Defined is selected in the Baseline Mode page and Fitting is selected for Connect By in the Create Baseline page.
