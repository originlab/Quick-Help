28.3.1 The Standard Toolbar

XFB-Standard-Toolbar

The X-Function Builder toolbar buttons allow you to perform X-Function operations.

Opens the New X-Function Wizard dialog box, which guides you through steps needed to create an X-Function.

Opens a X-Function file(*.oxf).

Saves the current file.

Edits the current X-Function in the Code Builder.

Toggle the Tree View on/off.
