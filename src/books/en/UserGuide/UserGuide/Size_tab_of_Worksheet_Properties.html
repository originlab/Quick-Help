4.8.3.3 The (Worksheet Properties) Size tab

WksProperties-Dialog-Size-Tab

Controls for modifying grid spacings in the worksheet.

Size

Row Number

Determines the number of rows in the current worksheet. If the Auto Add Rows box on the Miscellaneous tab is checked, then the size of the window will determine minimum row number (clear Auto Add Rows to have full control over row number).

Column Number

Determines the number of columns in the current worksheet.

Worksheet Measurement

Column Header Height

Column Header Height = X% of default row height of 16px (e.g. 150% * 16px = 24px).

Row Header Width

Row Header Width = X% of default column width of 67px (e.g. 128% * 67px = 86px).

Gap From Top

You can insert a gap between the workbook window title bar and the top of the worksheet. Use this area to do such things as place text objects or custom buttons that run script. Gap is measured in units of text point size (e.g. entering "20" creates a gap that will accommodate 20 point text).

All Rows Height

All Rows Height = X% of default row height of 16px (e.g. 100% * 16px = 16px).

All Columns Width

All Columns Width = X% of default column width of 67px (e.g. 100% * 67px = 67px).
