28.2.1 The Standard Toolbar

CB-Standard-Toolbar

The Standard Toolbar buttons provide access to opening, saving and printing workspace, as well as to performing some operations within Code Builder.

Opens a new file dialog box to create a new source, header, LabTalk script, or text file.

Opens a source, header, LabTalk script, or text file and add the file to the workspace.

Saves the Code Builder project file.

Saves all the open files in workspace.

Sends the active window to the printer without prompting; the current Page Setup and default printer settings are used.

Cuts the selection and puts it on the Clipboard.

Copys the selection and puts it on the Clipboard.

Inserts Clipboard contents.

Compiles a current source file from within Code Builder.

Compiles all the source files that have been added to the workspace and to perform linking.

Stops the compiler.

Recompiles all files and performs linking if no files have changed since the last build.
