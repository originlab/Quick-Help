Data Requirements

Select six columns (or a range from these columns) which have been designated as XYZXYZ.

Creating the Graph

Select required data.

Select Plot > Specialized : Ternary Vector

Template

TernaryVector.otpu (installed to the Origin program folder).

Notes

Ternary vector graph is a special vector graph which is plotted in the ternary coordinate, similar with the Polar Vector.

Plotting Input Data into Ternary Corrdinate

As the vectors are plotted in the ternary coordinate, the input data will be assumed to be normalized by groups, which means that the sum of the absolute values of cells in every row of each XYA group equals to 1 or 100: |x|+|y|+|z|=1 or 100. You can read more details about ternary normalization in the Ternary graph page.

To customize the ternary axes, you can go to the

  • Ternary tab of Plot Details dialog

Plotting XYZXYZ columns as Vectors

The leftmost X, Y and Z columns determine the XYZ ternary coordinate of the tail of the vector. The second set of X, Y and Z columns determine XYZ ternary coordinate of the head of the vector.

To customize the vectors, you can go to the

  • Vector tab of Plot Details dialog
  • Label tab of Plot Details dialog
