Function

Brief Description

Pseudo-Voigt function, linear combination of Gaussian and Lorentzian with different FWHM.

Sample Curve

Parameters

Number: 6

Names: y0, xc, A, wG, wL, mu

Meanings: y0 = offset, xc = center, A =area, wG=Gaussian FWHM, wL=Lorentzian FWHM, mu = profile shape factor

Lower Bounds: wG > 0.0, wL > 0.0

Upper Bounds: none

Script Access

nlf_psdvoigt2(x,y0,xc,A,wG,wL,mu)
