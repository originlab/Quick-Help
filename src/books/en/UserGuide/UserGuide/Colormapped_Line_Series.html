29.1.6 Colormapped Line Series

Data Requirements

Select at least one Y column (or a range from at least one Y column). Ideally, select at least two Y columns (or a range of at least two Y columns).

If there is an associated X column, X column supplies X values; otherwise, sampling interval of the Y column or row number is used.

Creating the Graph

Select required data.

From the menu, choose Plot >Basic 2D: Colormapped Line Series.

Template

LineCmap.otpu (installed to the Origin program folder).

Notes

This graph is a 2D line plot with multiple curves, and all of the curves are colored by a specified colormap. By default, Plot Index has been used to colormap the curve colors, you can go to the Line tab of Plot Details dialog and use the Color Control to select other column label row to colormap the curves.

Note:
  • For Origin 2023b or any lower version, as the old template has been used, this graph is a flatten 2D waterfall with Z values as colormap. When plotting, one of the column identifiers located at any column label row will be treated as Z values.You can see the details in the Waterfall tab at the layer level of Plot Details dialog. If there is no any column identifier except column short name, the column indices will be token as Z values automatically.
  • For Origin 2017 or any lower version, there is no such menu entry to create this graph by just one-clicking. However, you can try the workaround in this blog page.
