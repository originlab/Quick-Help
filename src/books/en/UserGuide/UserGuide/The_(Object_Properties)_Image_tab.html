8.6.5.12 The (Object Properties) Image tab

Ob-Prop-Image-tab


Select Use picture holder checkbox to use picture holder. This check box is available when the object is a worksheet.

To add a background for the selected object, check Background check box and select a desired option from the combo drop-down list. Uncheck this check box to display the object without a background.

Enable Interpolation check box can be used to decide whether to enable the interpolation for image rendering. This control is equal to the system variable @ii.
