28.1.3 The Graph Toolbar

Graph-Toolbar


The Graph toolbar buttons are available when a graph is active. They provide easy access for zooming, rescaling the axes to show all plotted data, and adding tables. The toolbar is also useful for quick layer management including adding layers or merging graph windows.

Enable/Disable anti-aliasing for lines on the graph.

Exchange the X and Y axes on the graph.

Zoom in on the specified area.

Zoom out on the specified area.

Display the full page within the window.

Rescales the axes to show all data points on all axis. Useful for restoring the graph after using the Enlarger Tool to enlarge a portion of the graph axis. For general 2D graph in a Cartesian coordinate system, this option will also consider and show the reference lines on X and Y axis.

Rescales the X axes to show the full X range. For general 2D graph in a Cartesian coordinate system, this option will also consider and show the reference lines on X axis.

Rescales the Y axes to show the full Y range. For general 2D graph in a Cartesian coordinate system, this option will also consider and show the reference lines on Y axis.

Enable/Disable the speedmode for all layers in active graph.

Extracts data plots in the active layer to separate layers.

Extracts all layers in the active graph window to separate graphs.

Merges all open graph windows onto a single graph page. Graphs may be superimposed or merged as separate panels.

Open Batch Plotting tool to replicate the current graph by replacing data in all the plots with the data from other columns/sheets/books.

Adds a new layer to your graph window.

Adds a linked top X layer to your graph window.

Adds a linked right Y layer to your graph window.

Adds a linked top X, right Y layer to your graph window.

Adds an inset layer to your graph window.

Adds an inset layer, plotting same data as active layer, to your graph window.
