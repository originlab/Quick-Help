20.5.3.8 RGB Merge (Pro Only)

Image-RGB-Merge

The imgRGBmerge function merges three images as Red, Green and Blue channels to one image. To use this function,

elect Image: Conversion: RGB Merge from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function imgRGBmerge in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help imgRGBmerge
