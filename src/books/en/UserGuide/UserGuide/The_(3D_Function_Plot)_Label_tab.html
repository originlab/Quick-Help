8.3.6.3.2 The (3D Function Plot) Label Tab

PD-Dialog-3DFunc-Label-Tab


The 3D Function Plot's Label tab is arranged as a table, allowing you to set Long Name, Units, and Comments headers for the x, y, and Z columns. These headers can be displayed in the Function Plot graph. (For example, you might use the Long Name and Units headers for x, y, and z axis titles, and the Z comment in the legend.)
