20.5.3.7 Image Scale (Pro Only)

Image-Image-Scale

The ImgScale function applies the new XY coordinate scales either on full image or a selected rectangle area to rescale the image. To use this function,

select Matrix: Set Dimension... from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function ImgScale in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help imgScale
