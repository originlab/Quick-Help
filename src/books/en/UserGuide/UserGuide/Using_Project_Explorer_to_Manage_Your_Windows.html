3.3 Using Project Explorer to Manage Child Windows

OPJ-ManageWindowByPE

Use the Project Explorer (PE) interface to organize and keep track of folders and windows in your project file -- your graphs, workbooks, matrixbooks, layout windows, notes windows and Excel workbook windows. You can also save a folder as a new project file, or append an existing project file to a folder in Project Explorer.

To see how to use Project Explorer, go to Origin Basics > Project Explorer.
