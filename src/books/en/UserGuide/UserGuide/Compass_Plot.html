29.13.10 Compass Plot

Compass-Plot


Contents

  • 2 Creating the Graph
  • 3 Template
  • 4 Notes

Data Requirements

Select at least one pair of XY columns.

Creating the Graph

Select Plot > Specialized: Compass Plot

Template

Compass.otpu (All above are installed to the Origin program folder).

Notes

This is a XYXY vector in the polar coordinate, a variant of the Polar Vector theta r theta r graph, its start point is the center of the polar coordinate, the selected XY columns contain the angle and radial coordinates of the end points.

You can go to the Vector tab of Plot Details to customize the vectors.
