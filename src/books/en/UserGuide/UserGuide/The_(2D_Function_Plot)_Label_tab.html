8.3.6.1.2 The (2D Function Plot) Label Tab

PD-Dialog-2DFunc-Label-Tab

Set function headers for independent and dependent variables and function name on the Label tab. This tab is organized as a table, allowing you to enter X and Y headers (e.g., Long Name, Units, and Comments). These headers can be displayed in the Function Plot graph. For example, independent and dependent variables, Long Name and Units can be used as X and Y axis titles respectively, and Y Comments can be displayed as a legend. Moreover, you can specify the function plot Short Name in the Function Short Name edit box.

After creating the function plot, you can also rename the Long Name and Short Name of function plot on the left panel of the Plot Details dialog. To switch from Short Name to Long Name, click the toggle to switch to LN.
