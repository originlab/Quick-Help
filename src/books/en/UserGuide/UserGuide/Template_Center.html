Overview

The Template Center dialog helps you search and download the online templates. It works similar with the App Center.

Note: All downloaded templates will be treated as Extended templates and listed in the Templete Library

To open Template Center
Click on the Open Template Center icon in Template Library.

Or

Select Tools: Template Center menu.

Registration is needed to use Template Center. If you haven't registered your Origin, follow the prompt to complete registration.

Usage

Search tab: overview of online templates and find the required templates

On this tab, you canï¼š

  • see the basic information, including the developer and the minimum required version.
  • see the introduction
  • when the mouse hovers over the template icon, a brief description will pop up.
  • clicking the template title can open the file exchange page to see detail description
  • check the status of the templates, and then choose to download and install the template
  • sort the template list by date update, download frequency, ratings and min version.
  • search templates by keywords, category, and compatible versions.

Note: Min Version lists All and Compatible two options. Compatible means set min version to current Origin version, so you will only see the templates that can really install and used in your Origin. However, it won't filter out Pro version if you have a standard Origin.

Updates tab: manage and update existing templates

Choose Updates tab from left to view all templates available for updates. You can click button to the right of an Template to update it, or click Update All Compatible button to update all at once.

You can set @AUA = 1 in the System Variables dialog to turn on auto-update for installed templates/Apps that have an update available.

Request a new Template and submit your Template

- click to request a template that you cannot accomplish by the existing templates.

- if you have created an template by yourself, welcome to share it via File Exchange.
