8.12.1 The Arrange Layers Dialog Box

GraphRef-ArrangeLayers

The Arrange Layers tool is used to change a graph to m*n layers.

To open this dialog:

  • Click on the Arrange Layers button on Page level Mini Toolbar
  • Select Graph: Arrange Layers menu
  • Right-click on the layer icon, and then select Arrange Layers
  • Right-click on the graph level of Object Manager, and then select Arrange Layers


Number of Rows

This tool assumes that you want to arrange your layers in an N by M grid. You can use this box to specify M--the number of rows in the grid.

Number of Columns

This tool assumes that you want to arrange your layers in an N by M grid. You can use this box to specify N--the number of columns in the grid.

Direction

This tool specifies the direction for arranging graph layers in N by M grid.

  • Horizontal First
    Arrange layers in horizontal direction first
  • Vertical First
    Arrange layers in vertical direction first
Use Alternate Side to Show Ticks and Labels

Specifies whether to show Axis Ticks and Labels on left and right side (bottom and top) alternately in the graph layers.

Note: This variable is only available in Stack graph. That means either the Number of Rows or Number of Columns must be 1, but not both to be 1.
When Number of Columns=1 and Number of Rows>1, the layers are stacked in vertical, and the layers shows Y axis on left and right alternately.
When Number of Rows=1 and Number of Columns>1, the layers are stacked in horizontal, and the layers shows X axis on bottom and top alternately.

Keep Layers Size and Auto Fit Page Size Specifies whether to keep layers size and auto fit page size.
Page Size

When unchecked Keep Layers Size and Auto Fit Page Size, Page Size branch is showed. You can customize the graph page size.

Unit
Specifies the unit for both Width and Height.
Width
This textbox is used to set the width of the selected graph page.
Height
This textbox is used to set the height of the selected graph page.
Spacing
Unit
Specifies the unit for the graph spacing.
Note: If Keep Layers Size and Auto Fit Page Size is selected, spacing unit can be set to inch, cm, mm, pixel, point. If Keep Layer Size and Auto Fit Page Size is not selected, spacing unit also can be % of page.
Horizontal Gap
Use this textbox to specify the horizontal gaps between two adjacent layers.
Vertical Gap
Use this textbox to specify the vertical gap between two adjacent layers.
Note:When the graph is a Tile Grid Map, Number of Rows and Number of Columns will change to Index of Rows and Index of Columns which are drop-down lists to let you select a column label row of source data to arrange the layers. And the Direction option will be hidden.
