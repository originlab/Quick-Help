9.3.7.40 The (Plot Details) Labels Tab for Hierarchical Edge Bundling

PD-Labels-tab-for-HEB

This tab can be used to control the labels for leaves and nodes in a Hierarchical Edge Bundling diagram.

Show Labels for Leaves

Specify whether show labels for leaves and how to show them.

Font

Specify the font and display format for the labels. Besides changing font, this control group allows you to

  • set color to Auto which means the label color will follow the leaf symbols. And, you also can apply colors to labels By Points.
  • select Angular or Radial for Rotate dropdown list to show labels along angular or radial axis.

Offset

Specify the offset of the labels, from the leaf nodes and along the radius. The unit of this control is % of font height.

Display Name

Specify to show what as the labels. You can select the name of leaf nodes or the text in a column.

Show Labels for Nodes

Specify whether show labels for the nodes of subsets. This group is only available when Nodes for Subset has been checked in the Layout tab.

Outside/Inside Node

Specify show the titles outside or inside nodes.

Font

Specify the font and display format for the node labels. This control group is same as that of leaf labels.

Offset

Specify the offset of the labels, from the leaf nodes and along the radius. The unit of this control is % of font height. This control is only available when Postion = Outside Node.

Wrap Text by Number of Characters

Specify whether wrap the text when the text is longer than the number of characters entered in the Minimum Width box.
