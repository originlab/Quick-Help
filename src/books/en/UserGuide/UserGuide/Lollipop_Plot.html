Data Requirements

Select two or more Y columns or a range of data from at least two Y columns. If there is an associated X column, X column supplies X values; otherwise, sampling interval of the Y columns or row number is used.

Creating the Graph

Select required data.

From the menu, select Plot > Basic 2D : Lollipop Plot.

Template

Lollipop.OTPU (installed to the Origin program folder).

Notes

Lollipop Plot is a variation of scatter plot with vertical drop lines, which has been exchanged X and Y axis and made the drop lines drop to next plot.

When you selected more than two Y columns, the drop lines will drop to next plot one by one and finally connect the first Y value and last Y value at same X value.

You can use the color control Y Value: Plus-Minus to set different color for the drop lines at different directions.
