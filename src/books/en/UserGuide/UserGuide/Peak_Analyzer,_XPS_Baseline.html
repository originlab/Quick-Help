19.1.10.3 Peak Analyzer, XPS Baseline

PeakAnalyzer-XPSBaseline

This page is added to the Peak Analyzer when Baseline Mode is set to XPS. This allows creating a special X-Ray photoemission spectrum baseline.

  • Menu Command: Analysis: Peaks and Baseline: Peak Analyzer: Open Dialog
  • Window Types: Workbook, Graph

Dialog Theme

Dialog Theme See Peak Analyzer Themes.

The Computation Range Group Controls

Range Specify a method to set the computation range of the spectrum data.
  • Row: Specify the computation range by the row indices of the spectrum data points.
  • Energy: Specify the computation range by the energy values of the spectrum data. Energy can be viewed as the x values of the spectrum data.
Row1 This is available only when Row is selected in the Range drop-down list. It specifies the starting row index of the computation range. When the Auto check box is selected, Row1 will be determined by the Peak Analyzer automatically and Row value is Read Only.
Row2 This is available only when Row is selected in the Range drop-down list. It specifies the ending row index of the computation range. When the Auto check box is selected, Row2 will be determined by the Peak Analyzer automatically and Row value is Read Only.
Energy(X) 1 This is available only when Energy is selected in the Range drop-down list. It specifies the starting energy value(or x value) of the computation range. When the Auto check box is selected, Energy(X) 1 will be determined by the Peak Analyzer automatically and Energy value is Read Only.
Energy(X) 2 This is available only when Energy is selected in the Range drop-downlist. It specifies the ending energy value(or x value) of the computation range. When the Auto check box is selected, Energy(X) 2 will be determined by the Peak Analyzer automatically and Energy value is Read Only.

Other Dialog Box Controls

Method Specifies a method to create baseline. For more information, see Peak Analyzer Algorithms.
  • Shirley
  • Tougaard
Option This is available only when the Method drop-down list is set to Tougaard. Specifies a method to adjust the parameters which are related to create baseline.
  • Final Height: This value is used to set the Y value of the user-specified ending data point (baseline has been subtracted). And it is also used to adjust the baseline.
  • Adjustable Parameter: Specify the value of the parameter which can be used to affect the creation of the baseline.
Final Height This specifies the Y value of the user-specified ending data point (baseline has been subtracted). And it is also used to adjust the baseline. When Auto check box is selected, Final Height will be determined by the Peak Analyzer automatically. If you want to specify Final Height manually, uncheck Auto.
Adjustable Parameter It is available only when Adjustable Parameter is selected in Option dropdown list. Specify the value of adjustable parameter. When Auto check box is selected, Adjustable Parameter will be determined by the Peak Analyzer automatically. If you want to specify Adjustable Parameter manually, uncheck Auto.
