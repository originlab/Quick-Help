Input Data

Select data for Cross Tabulation and Chi-square.

Row Select data from a column to specify row variable.
Column Select data from a column to specify column variable.
Layer Select data from a column to specify layer variable. When a layer variable is specified, the Cross Tabulation between the row and column variable will be created at each level of the layer variable.
Frequencies If data is summarized as frequency data mode, select the frequency column to specify frequencies variable.

Statistics

Contingency Table This branch determines which statistics to calculate and output in the Contingency Table.
  • Counts: Counts, Expected Counts.
  • Percentages: Percentages of Row Counts,Percentages of Column Counts,Percentages of Total Counts.
  • Residuals:Residuals,Standardized Residuals,Adjusted Residuals.

Note: To learn more details for these statistics, see the the introduction page and the algorithm page

Significant Level Specify the significant level for the following hypothesis testing. The default value is 0.05.

Tests

Chi-Square Test Specify whether to perform Chi-Square Tests to determine whether there is a significant relationship between variables in the contingency table.

Note: To learn more details for these statistics, see the introduction, algorithm and the interpreting results page

Fisher's Exact Test Specify whether to perform Fisher's Exact Test to determine whether there is a significant relationship between variables in the contingency table.

Note: To learn more details for these statistics, see the introduction, algorithm and the interpreting results page

Measures of Association This branch determines which statistics to calculate to assess the strength of the relationship between the variables in the contingency table. It includes the following check boxes.
  • Gamma
  • Kendall's tau-b tau-c
  • Phi
  • Cramer's V
  • Contingency Coefficient
  • Somer's D
  • Lambda
  • Uncertainty Coefficient

Note: To learn more details for these statistics, see the introduction, algorithm and the interpreting results page

Agreements This branch determines which test to perform to measure and test the agreement of matched-pairs data. It includes the following check boxes.
  • Kappa
  • Bowker's Test(Mcnemar for [2*2])

Note: To learn more details for these statistics, see the introduction, algorithm and the interpreting results page

Other Measures Specify whether to perform other tests and calculate other statistics. It includes the following check boxes.
  • Cochran-Mantel-Haenszel:
    Specify whether to analysis the relationship of multiple contingency tables, including conditional independence test, odds radiosâ€™homogeneity test and common odds radio.
  • Relative Risk:
    Specify whether to calculate Relative Risk to measures the ratio of the odds of an event occurring in an group to the odds of the event occurring in a comparison group.
  • Odds Ratio:
    Specify whether to calculate Odds Ratio to measures the ratio of the odds that an event or result will occur to the odds of the event not happening.

Note: To learn more details for these statistics, see the introduction, algorithm and the interpreting results page.

Output

Result Log Output Specify whether to display the Result Log.
Mosaic Plot Specify whether to display Mosaic Plot whose area of each rectangle is proportional to the proportions of the Y variable in each level of the X variable.
Corsstab Report Specify the sheet for the crosstab result. The default value is a new sheet in the workbook of input data.
Plot Data Specify the sheet for the mosaic plot data. The default value is a new sheet in the workbook of input data.
