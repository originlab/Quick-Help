Data Requirements

Select at least one Y column of values (or a range from at least one column). Ideally, select four or more columns of values (or a range of four or more Y columns). If there is an associated X column, X column supplies X values; otherwise, sampling interval of the Y column or row number is used.

Please note, the number of X columns should not be more than four for this 4 panels graph:

  • if there are only one X column, this X column will be used for all panels and all Y columns will be arranged into these four layers against the same X data; The plot arrangement rule is to plot all Y columns one by one into the layer in order and cyclically.
  • if there are more than two X columns, the X columns will be used as the separator to divide the Y columns into groups, and these groups of Y columns will be plotted into into the layers in order, against the adjacent X data;
  • if there are more than four X columns, you are not allowed to plot such four panels graph.

Creating the Graph

Select required data.

Select Plot > Multi-Panel/Axis : 4 Panel.

or

Click the 4 Panel button on the 2D Graphs toolbar

Template

PAN4.OTP (installed to the Origin program folder).

Notes

  • Each column of Y values plots into a separate layer. The layers are arranged in two columns by two rows.
  • Beginning with Origin 2018b, you can use the Plot Details Common Display controls for simultaneous editing of layer, plot and axis properties in a multi-layer graph. For more information, see The (Plot Details) Layers tab controls.
