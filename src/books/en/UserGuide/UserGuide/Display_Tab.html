Drawing Order

Drawing Order

Use this drop-down list to specify the drawing order of the layers. Options include:

  • Draw Composite
    Force display of data in both layers even if one or more layers have a background color.
  • Draw Layer by Layer
    Suppose that there are m layers in the graph. Then for n=1, 2, ..., m. Layer n+1 is drawn after layer n is completely drawn, including the background of layer n. If you specify a background color for layer n+1, then the background of this layer will mask elements of layer n. If you leave the background as None then the data in layer n will be visible in layer n+1.

Option

There are three radio boxes (Color、Border Dimensions、Scale Elements)here. You can check one of them to set the Display tab.

Color

Background Color

Use this color list to choose the background color for the selected layer(s).

Border Fill Color

Use this color list to choose the border fill color for the selected layer(s).

Border Color

Use this color list to choose the border color for the selected layer(s).

Apply

When this button is clicked, the Color settings that you have changed will be applied in preview graphs, and you can see it in the Preview panel.

Border Dimensions

Border Dimensions(% of Layer)

If the checkbox of Border Dimensions in Option is checked, you can modify the border position with the following four text boxes:

  • Left border
  • Top border
  • Right border
  • Bottom border
Apply

When this button is clicked, the Border Dimensions settings that you have changed will be applied in preview graphs, and you can see it in the Preview panel.

Scale Elements

Scale Mode

Specify how the axes, axes labels, legend, data plots, and other objects that are attached to the Layer Frame or Layer and Scales are affected when a layer in the new graph page is resized.

  • Scale with Layer Frame
    Scale the layer elements proportionally with the layer.
  • Fixed Factor
    Determine the size of the layer elements relative to their original size (font size, symbol size, and line width, etc.). The size of these elements is determined by multiplying their original size by the scale factor specified in the Fixed Factor edit box.
Fixed Factor

This variable is available only when the Scale drop-down is set to Fixed Factor. It specifies the fixed scale factor.

Apply

When this button is clicked, the Scale Elements settings that you have changed will be applied in preview graphs, and you can see it in the Preview panel.
