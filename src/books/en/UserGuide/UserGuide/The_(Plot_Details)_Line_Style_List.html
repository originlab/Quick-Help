9.3.7.44 The (Plot Details) Line Style List

PD-line-style-list-tab

The Line Style List tab is added when (b) you select a worksheet column of categorical values to control the Line tab line Style (i.e. you choose a worksheet column from that drop-down list) or (b) you set line Style = Increment.

By default, Origin uses the LabTalk list of Line Types. However, you can customize this list, rearranging or eliminating styles to suit your needs. Check the Use custom increment list box to expose controls and follow instructions on this page for customizing the list.

Note that beginning with Origin 2022b, you can define custom dashed lines "on-the-fly" and these custom definitions can be used in building your line style list.
