Standard Controls

  • None
  • Auto

For more information, see: Recalculating Analysis Results

Test Options

Input

1st Data Range

The first sample column.

2nd Data Range

The second sample column.

For help with range controls, see: Specifying Your Input Data

Null Hypothesis

The null hypothesis: The two samples are drawn from the same population (read only).

Alternative Hypothesis
  • Median1 <> Median2
Performs a 2-tailed paired sample Wilcoxon signed rank test.
  • Median1 > Median2
Performs an upper-tailed paired sample Wilcoxon signed rank test.
  • Median1 < Median2
Performs a lower-tailed paired sample Wilcoxon signed rank test.
Significance Level

The alpha (α) value.
