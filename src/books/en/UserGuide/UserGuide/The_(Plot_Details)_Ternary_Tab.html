9.2.11 The (Plot Details) Ternary Tab

PD-Dialog-Ternary-Tab

Triangle Type

Specify how to plot the normalized XYZ values, in a Equilateral triangle as a general ternary graph, or a Right triangle as a right-triangle ternary

Direction

This option is available when you select Equilateral as Triangle Type, is used to switch the direction of 2D ternary axis, clockwise or counter-clockwise coordinates.

Start Angle

This option can be used to further set the starting angle with previously-selected direction, to rotate the ternary.
