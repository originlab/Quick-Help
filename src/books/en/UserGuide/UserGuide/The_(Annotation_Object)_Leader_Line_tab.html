Show Leader Line if Offser Exceeds(%)

The leader line is a line object connecting the Annotation object and its reference position This check box controls display of the leader line. Leader lines will only be shown if the offset between Annotation object and data points exceeds a certain critical length. The critical length is defined using the Percent Value in the text box next to the check box.

Critical Length = Percent Value% * (Layer Height + Layer Width)/2.

Color

Determine the color of leader line either by system color or custom color.

Connect

Select the leader line connection type. The seven options are Straight, Polyline-Horz, Polyline-Vert, Step-Horz, Step-Vert, Horizontal and Vertical . Polyline-Horz and Polyline-Vert create three segment lines where the middle segment is angled. Step-Horz and Step-Vert create three segment lines where the middle segment is perpendicular.

If Polyline-Horz or Polyline-Vert is chosen, you can change the percent of horizontal or vertical distance between two inflection points in the text box after the Connect drop-down list(see the below image(1)). Similarly, if Step-Horz or Step-Vert is chosen, you can specify the position of the inflection point by the percent of horizontal or vertical distance in the text box after the Connect drop-down list(see the below image(2)).

Width

Determine the width of the leader line.

Style

Select the line style of the leader line.

Gap between Leader Line and Point(%)

Type in a value to define the offset between the graph point and the adjacent point of the leader line. The value is percent of font height of the text, which is defined in the Font size in the Text tab.

Gap between Leader Line and Text Frame(%)

Type in a value to define the offset between the adjacent point of leader line and the text frame. The value is percent of font height of the text.

Arrow

Width If arrow head shape is selected, this option is avaliable. Specify the width of the arrow head.
Length If arrow head shape is selected, this option is avaliable. Specify the length of the arrow head.
