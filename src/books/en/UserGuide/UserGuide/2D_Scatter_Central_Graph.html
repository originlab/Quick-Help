29.2.2 2D Scatter Central Graph

2DScatterCentral-Graph

Data Requirements

Select one or more Y columns or a range of data from at least one Y column. If there is an associated X column, X column supplies X values; otherwise, sampling interval of the Y column or row number is used.

Creating the Graph

Select required data.

From the menu, select Plot > Basic 2D: Scatter Central.

or

Click the Scatter Central button on the 2D Graphs toolbar.

Template

SCATTERCENTRAL.OTP (installed to the Origin program folder).
