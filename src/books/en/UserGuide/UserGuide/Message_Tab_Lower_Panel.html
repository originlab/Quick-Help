15.3.2.10 Message Tab (Lower Panel)

NLFit-Dialog-MessageTab

Use the Messages tab to monitor fitting procedures and results. For example, you can see whether or not the fit has converged, how many iterations were performed and what the current chi-square value is.

This information is potentially very useful. For example, if you think the reported chi-square value is too large (you can see the reduced Chi-square by clicking the Calculate Chi-Square button ), you can adjust the initial parameter values and perform the fit again.
