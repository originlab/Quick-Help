27.1.2.2 Theme Editing

ThemeRef-Edit


The Theme Editing dialog can be used to edit user-created themes. You can modify all the properties that are stored in the theme file. To open it, you can select Preferences: Theme Organizer from the main menu of Origin and then right-click on the target theme to select Edit.

Theme Editing Dialog Controls

File Displays the full path of the Theme file being edited.
Description This editable box can be input some text for the theme which will be shown as a tip when your mouse moving over the name of the theme in the Theme Organizer list.
Total Number of Properties (Size) This non-editable text shows the total number of object properties that are stored in the Theme which will decrease or increase as object properties are removed from or added to the Theme.
Date Modified This non-editable text shows the date of the last modification of the theme file.
Property/Value list control This table lists the all the properties that stores in the theme file. The default style of the list is in the tree view. You can change to list view by enabling the List View check box at the right of the dialog.

You can edit any property by clicking on the Value column and then

  • selecting a new value or
  • entering a new value or
  • selecting or clearing a check box.

You can also right-click on a property or a branch of the tree, then choose Delete from shortcut menu to delete it. You can right-click on the icon of a folder (the "tree node") to select the Add from the context menu to add more properties/branches to the tree. After you've deleted or added a property, the Total Number above this table will be updated automatically.

In addition, you can use the shortcut menu to expand or collapse the selected node.

OK Write all the changes to the theme file and close the dialog.
Cancel Cancel all the changes and close the dialog box.
Filter Click this button to bring up the Theme Properties Filtering dialog box to edit/modify the Theme using filters. For more details about this dialog, please refer to Theme Properties Filtering. Please note that the button will disappear when edit on a user-created worksheet theme.
List View Enable this checkbox to show the properties in List View.
Show Empty Enable this checkbox to show the empty branches in a tree. By default, empty branches are hidden. So after all the empty branches have been shown, you can right-click on the empty branches to add more properties to the theme file.
Show Collections Select this checkbox to display the branches of the tree which have collection members (for example a Layer collection which could contain Layer1, Layer2 etc).
