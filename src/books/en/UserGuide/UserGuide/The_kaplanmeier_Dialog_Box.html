Input

Specify data range for analysis, including time data range, censor data range and grouping range.

Time Range Specify time data column
Censor Range Specify censor data column
Grouping Range (Optional) Specify group data range. Note that the grouping column will be set as categorical if Text column.

Censoring Value(s)

Specify the censor subject value, which identifies which time values in the observation data set are censored

Survival Tables

Use this branch to specify report table to be output

Event and Censor Values Summary Check box for output the summary of event and censored value result.
Survival Estimates Check box for output the survival estimates result.
Quartile Estimates Check box for output the quantile estimates result.
Mean Estimate Check box for the mean estimates result
Confidence Level in % for Quartile/Mean Estimate Text box to enter a confidence level to be used when computing errors for the survival function and upper and lower limits of the quartile estimates.

Survival Plots

Specify the plot type to be output

Survival Check box for output survival curve.
Add Confidence Intervals Add confidence intervals to the survival curve.
One Minus Survival Check box for output of one minus survival curve, i.e. the cumulative distribution curve.
Hazard Check box for output of the hazard curve.
Log Survival Check box for output of the log survival curve.
Plot all survival curves in one graph Check to plot all the survival curves in one graph window

Equality Test

Specify the method to be used in equality testing

Log Rank Use the Log Rank method to test the equality of the survival distributions for different groups.
Breslow Use the Breslow method to test the equality of the survival distributions for different groups.
Tarone-Ware Use the Tarone-Ware method to test the equality of the survival distributions for different groups.
Pairwise Comparisons When you have chosen one or more of the equality tests above, check the box for pairwise comparisons of three or more groups.

Outputs

Results Log Output

If this check box is checked, the result will be output to Results log.

Specify the output detination.

Survival Functions Specify the survival function data destination
Output Results Specify the report worksheet destination
