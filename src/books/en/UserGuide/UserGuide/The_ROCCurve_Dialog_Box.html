17.10.1 The ROC Curve Dialog Box

ROCCurve-Dialog

Contents

  • 1 Input Data
  • 2 Computation Control
  • 3 Output Results

Input Data

Specifies the input dataset

Data Range

Input Data Range. You can select no less than 1 column

Grouping Range

The range that contains grouping information. Note that the grouping column will be set as categorical if Text column.

Computation Control

Use this branch to specify computation settings

State Value To specify which states to be positive
Threshold Method

This option specified what cutoffs to be used in ROC Curve

Interpolation of Data Points
The method fixes the set of cutoffs to be the set defined by the values half the distance between each successive pair of observed test scores, plus and
Speed Mode
Cutoffs are bounds for 100 equally spaced interval if Speed Mode is used
Test Direction

Choose a test direction.

  • Positive v.s. High
    Larger test measurement values indicates more positive test.
  • Positive v.s. Low
    Smaller test measurement values indicates more positive test.
ROC Curve

Select to decide whether to output ROC curve

Optimal Cutpoint
Select to mark the optimal decision threshold on the ROC curve
With Diagonal Reference Line
Select to decide whether have diagonal reference line in ROC curve
Standard Error and Confidence Interval

Select to decide whether to output SE and Conf. Interval of area under curve

Confidence Levels in %
To specify conf. level of confidence interval

Output Results

Custom the output results and destination.

Dataset Identifier Specify the identifier for the source datasets.
Identifier
Select a type to specify the source datasets information.
  • Range
Use the range syntax.
  • Book Name
Use the workbook long name.
  • Sheet Name
Use the worksheet name.
  • Name
Use the long name of the corresponding column if there is a long name, otherwise use the short name of column.
  • Short Name
Use the short name of the corresponding column.
  • Long Name
Use the long name of the corresponding column.
  • Units
Use the unit of the corresponding column.
  • Comments
Use the comment of corresponding column.
  • <Custom>
Use custom formats to define a data identifier. For its usage, please refer to Advanced legend text customizations.
Show Identifier in Flat Sheet
Specify whether to use the dataset identifier in flat sheet.
Report Tables

Specifies the destination of report worksheet tables

Book
Specifies the destination workbook.
  • <none>: Do not output report worksheet tables
  • <source>: The source data workbook
  • <new>: A new workbook.
  • <existing>: A specified existing workbook
Sheet
The destination worksheet, always <new>.
Result Log
Use this checkbox to Specify whether to output the report to Result Log
Script Window
Use this checkbox to specify whether to output the report to Script Window.
Notes Window
Use this drop-down list to specify the destination Notes Window:
  • <none>: Do not output to any Notes window.
  • <new>: Output to a new Notes window. You can enter the desired name of the Note window here.
Plot Data

Specifies the destination of values of ROC Curve

Book
Specifies the destination workbook.
  • <none>: Do not output report worksheet tables
  • <source>: The source data workbook
  • <new>: A new workbook.
  • <existing:A specified existing workbook
Sheet
The destination worksheet, always <new>.
Optional Report Tables

Specifies what is output to report worksheet

Notes
Notes table
Input Data
Table for input data
Masked Data
