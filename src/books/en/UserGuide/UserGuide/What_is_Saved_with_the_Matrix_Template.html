The matrix window page view

The matrix window has three auxiliary panels which can be displayed singly:

  • The Organizer
  • The Script Panel
  • The Matrix Image Thumbnails

Note that opened panels are saved with the matrix template (OTM) file.

The matrix window sheets and objects

The matrix window, like the workbook window, can have multiple sheets. Each matrixsheet contains one or more matrix objects. Matrixsheets + sheet names and matrix objects + object names are saved with the matrixbook template.

See Matrixsheet and Matrix Object Naming Conventions and Basic Matrixbook and Matrixsheet Operations.

Matrix Properties dialog box

The Matrix Properties dialog box provides controls to specify the column width, data type and data display format. Settings are saved with the matrix template.

To open this dialog, do one of the following:

  • Select Matrix: Set Properties from the Origin menu.
  • Click on the object icon and choose Properties from the short-cut menu.
  • Click on the upper left corner to select the whole matrix, then right-click on the matrix and select Set Matrix Properties from the shortcut menu.

See Matrixsheet Properties.

Matrix Dimensions dialog box

The Matrix Dimensions dialog box provides options for customizing the matrix dimensions and the XY mapping relationship.

To open this dialog, do one of the following:

  • Select Matrix: Set Dimensions from the Origin menu.
  • Click on the upper left corner to select the whole matrix, then right-click on the matrix and select Set Matrix Dimensions from the shortcut menu.

See Matrixsheet Dimensions and Coordinates.

Set Matrix Values dialog box

The Set Matrix Values dialog box allows you to specify a formula to fill the matrix.

To open this dialog, do one of the following:

  • Select Matrix: Set Values from the Origin menu.
  • Click on the upper left corner to select the whole matrix, then right click on the matrix and select Set Matrix Values from the shortcut menu.
  • Use hot keys Ctrl + Q.

See Setting Matrix Values.

ASCII Import Options dialog box

Settings in the ASCII Import dialog box determine how data are handled when importing into the matrix.

To open the impASC dialog box:

  1. Select Data: Import from File: Single ASCII and select the Show Options Dialog check-box (below the file list). Pick a file and click Open. This opens the impASC dialog box.
  2. Click the Dialog Theme button and choose Save to <Sheet>. Adjust your settings and click OK. This saves the current import settings to the sheet and imports the selected file.

To save impASC options as a dialog Theme, click the Dialog Theme button and choose Save As before clicking OK to close the dialog.

Note: These actions do not save import options to the template. To save options to the matrix template, you must resave the matrix template.

ASCII Export Into dialog box

The ASCII Export Into dialog box sets export behavior for the matrix, including controlling the export of column names and specifying the data separator.

To open the ASCII Export Into dialog box:

  1. Select File: Export: ASCII.
  2. Select the Show Options Dialog check box
  3. Specify a File name and Save the file.

Upon clicking Save, Origin opens the expMatASC dialog box. Then select Save to <Sheet> for Dialog Theme. This allows you to view and change the export options saved in the sheet. If you want to save the settings to a dialog Theme, select the Save As option.

Note: These actions do not save export options to the template. To save options to the matrix template, you must resave the matrix template.
