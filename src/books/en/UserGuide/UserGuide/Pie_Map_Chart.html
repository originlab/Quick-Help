29.11.2 Pie Map Chart

Pie-Map

Pie Map Chart is a set of pie charts, the location of which are decided by a XY data. The map background in this graph is imported by using the Google Map Import app.

Contents

  • 1 Data Requirements
  • 2 Creating the Graph
  • 3 Template
  • 4 Notes

Data Requirements

Select X with at least 3 Y columns to create pie map. The first two columns(X and Y) decide the location of each pie. Other Y columns decide pie wedges. Size of pie is decided by total value of all pie wedges, which can be further customized in the Symbol tab of Plot Details dialog.

Creating the Graph

Select required data.

From the menu, choose Plot > Bar, Pie, Area: Pie Map.

Template

PieMap.OTPU (installed to the Origin program folder).

Notes

Pie Map chart is a set of pie charts, using a X-Y data to locate the pies. Each pie chart is created by one row of the Y columns for pie wedges. Furthermore, you can control the pie size by using another column or the wedge total, much like the Bubble Plot.

To control how to show the pie charts in color and size, you can go to the Symbol tab of Plot Details dialog.

You can press Ctrl key to double-click one pie chart to add a special point and drag to reposition this pie.

For more controls such as adding labels, Origin will treat the pies as symbols.
