20.5.3.5 Gray to Data Matrix

Image-Gray-to-Data-Matrix

This img2m function converts the specified grayscale image to a numeric data matrix. To use this function,

select Image: Conversion: Gray to Data Matrix from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function img2m in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help img2m
