19.1.6 Integrate Peaks with the Peak Analyzer

IntegratePeak-PA


If the Integrate Peaks radio button is selected in the Goal group in the Start page of the Peak Analzyer, you can use the Peak Analyzer to create a baseline, subtract the baseline from the input data, find peaks and then calculate the peak areas. A typical wizard map corresponding to this mode is as follows:

The Baseline Mode page allows you to choose a baseline mode and create a baseline. Then you can use the Baseline Treatment page for subtracting the baseline from the input data and the Find Peaks page for detecting the peaks. Finally, The Integrate Peaks page helps you to calculate the peak areas and generate a customized report.
