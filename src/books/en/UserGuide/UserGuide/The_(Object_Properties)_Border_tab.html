8.6.5.7 The (Object Properties) Border tab

Ob-Prop-Border-tab


In the Border tab, you can specify the Color, Width, Type and Transparecy for the shape object.

For the Rectangle and Circle object, you can also set the Alignment to Vertical, Horizontal, No corners and Corners only.

  • If align to vertical or horizontal, the object also can be specified stretching to Whole Layer, All Layers or Whole Page.
  • If align to No corners, the resize indicators only show on lines of object, and the corners of object can not be dragged to resize it.
  • If align to Corners only, the resize indicators only show on corners of object, and the lines of object can not be dragged to resize it.


Note: The Transparecy control is the same one in Fill tab, and they share the same value.

When you change the transparecy percentage for the border, the fill will be applied to the same transparecy percentage, and vice versa.
