29.15.5 XYY 3D Bar Graph with Error Bar

3DBar-Graph-with-ErrBar

Data Requirements

Select at least one Y column and one associated Y error bar column (or a range from at least one Y column and one associated Y error bar column). You need to preset the worksheet's column Plot Designations prior to creating this kind of plot. Each designated error bar dataset must be to the right of the data of the Y dataset with which it is associated (example: Y1, yEr1, Y2, yEr2, Y3, yEr3, etc).

If there is an associated X column, X column supplies X values; otherwise, sampling interval of the Y column or row number is used.

You also can add error bars to the existing XYY 3D bar graph, by using Plot Setup dialog or Plot Details dialog.

Creating the Graph

Select required data.

From the menu, choose Plot > 3D : XYY 3D Bars.

or

Click the XYY 3D Bars button on the 3D and Contour Graphs toolbar.

Template

  • glBar3D.OTP (OpenGL)
  • BAR3D.OTP

(Installed to the Origin program folder).
