20.5.2.3 Brightness

Image-Brightness

Brightness indicates the relative lightness or darkness of a specific color, and ranges from black to white. The X-function imgBrightness can be used to make the image darker or lighter, with the adjustment value representing the percentage change of each pixel. A negative value will result in a darker image while positive values result in a lighter image. To use this function,

select Image: Adjustments: Brightness from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function imgBrightness in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help imgBrightness
