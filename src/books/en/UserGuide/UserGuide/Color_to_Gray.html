20.5.3.2 Color to Gray

Image-Color-to-Gray

In Origin 8, the X-Function imgC2gray converts a 1-, 4-, 8-, 16-, 24-, or 32-bit bitmap image to an 8-, 12-, or 16-bit grayscale bitmap image. To use this function,

select Image: Conversion: Color to Gray from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function imgC2gray in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help imgC2gray
