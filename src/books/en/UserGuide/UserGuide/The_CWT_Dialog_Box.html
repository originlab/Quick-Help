18.12.1.1 The Continuous Wavelet Transform Dialog Box

CTW-Dialog

  • Menu Command: Analysis: Signal Processing: Wavelet: Continuous Wavelet
  • Window Types: Workbook, Graph
  • X-Function Scripting and GUI Examples: cwt(Pro)
Dialog Theme See Dialog Themes.
Recalculate

See Recalculating Analysis Results.

Discrete Signal

Specifies the input signal. See Specifying Your Input Data.

Scale

Specifies the scale vector. All scales should be positive integers. See Specifying Your Input Data.

Wavelet Type

Specifies the type of wavelet that is used for computation.

  • Morlet
  • DGauss
  • MexHat

For more information, see Origin Wavelet Types

Wave Number

If wavelet type is Morlet, use this variable to specify the wave number, which should be not less than 5 and not greater than 20.

Use Pseudo-Frequency

When this checkbox is selected, the image plot created by the coefficient matrix would use the value of pseudo-frequency as the vertical tick label.

Pseudo-Frequency

Specifies a range for outputting the pseudo-frequency

Derivative Order

If wavelet type is DGauss, use this variable to specify the order of derivative, which should be a positive integer less than 9.

Coefficient

Specifies a range for outputting the coefficients. See Output Results.

Coefficient Matrix

Check before this option if you want to output the coefficients to a matrix. Then you can specify the output matrix to the right of the checkbox. See Output Results.

Create Image Plot

The option specifies whether an image plot of the result will be generated.
