20.5.5.3 Edge

Image-Edge

Edge detection is a basic operation required by image segmentation and recognition. Edge detection operators examine the gradient magnitude of the image points by convoluting with a directional derivative mask. Then the points with greater gradient magnitude are viewed as edge points. Several operators are available in Origin 8, depending upon the image to be processed. To use this function,

select Image: Spatial Filters: Edge from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function imgEdge in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help imgEdge
