Brief Description

Substrate inhibition model at high concentrations.

Sample Curve

Parameters

Number: 3

Names: Vmax, Km, Ki

Meanings: Vmax = Maximum Enzyme Activity, Km = Michaelis-Menten Constant for Enzyme Activity, Ki = Dissociation Constant

Lower Bounds: Km > 0.0, Ki > 0.0

Upper Bounds: none

Script Access

nlf_SubstrateInhib(x,Vmax,Km,Ki)
