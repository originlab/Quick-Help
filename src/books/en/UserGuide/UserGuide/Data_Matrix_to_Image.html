20.5.3.3 Data Matrix to Image

Image-Data-Matrix-to-Image

This X-Function m2img converts a numeric matrix object to a grayscale image with mapping. Then you can use other image processing tools to process the image. To use this function,

select Image: Conversion: Convert to Image from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function m2img in the Origin X-Function reference file. This can be accessed from the Command Window by typing 

help imgC2gray
