29.16.10 Colormap Surface with Error Bar

Color-Map-Surface-ErrBar

Data Requirements

You need to have two matrix objects in the same matrixsheet. The matrix object that contains the surface data should be active.

You also can add error bars to the existing Color Map Surface graph, by using Plot Details dialog.

Creating the Graph

Activate the matrix object that contains the surface data.

From the menu, choose Plot > 3D : 3D Colormap Surface with Error Bar.

or

Click the 3D Color Map Surface with Error Bar button on the 3D and Contour Graphs toolbar.

Template

  • gl3dcmaperror.OTP (OpenGL)
  • 3DCMAPERROR.OTP

(Installed to the Origin program folder).
