Input

Input

Enter your input data.

For help with range controls, see: Specifying Your Input Data

Correlation Types

Pearson Compute Pearson product moment correlation coefficient.
Spearman Compute Spearman Rank correlation coefficient.
Kendall Compute Kendall correlation coefficient.

Plots

Scatter plots Produce scatter plots for each pair of variables.
Add Confidence Ellipse Plot a confidence ellipse.
Confidence Level for Ellipse(%) Specify the confidence level for ellipses.

Exclude Missing Values

Pairwise When computing correlation between two columns, the corresponding two entries will be excluded if there is a missing value.
Listwise Exclude the entire row for all datasets if there are any missing values.

Output

For help with the range controls, see: Output Results

Plot Data Specify target worksheet for output of scatter/ellipse data.
Plots Specify target report worksheet for output of scatter plots with confidence ellipse.
Result Table Specify target worksheet for output of report worksheet.
Pearson Sheet Specify target worksheet for output of Pearson Correlations.
Spearman Sheet Specify target worksheet for output of Spearman Correlations.
Kendall Sheet Specify target worksheet for output of Kendall Correlations.
