Data Requirements

Select exactly one column of values or a range from one column.

Creating the Graph

Select the required data.

Select Plot > Statistical: Q-Q Plot...

Or

Click the Q-Q Plot... button on the 2D Graphs toolbar.

Origin opens the plot_prob dialog box.

Please see more details on creating and customizing the Q-Q Plot.

Template

QQ.OTP (installed to the Origin program folder).

Notes

For information about the calculation for Q-Q Plots, please refer to Probability_Plot_and_Q-Q_Plot.
