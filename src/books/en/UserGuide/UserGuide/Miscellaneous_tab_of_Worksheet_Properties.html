4.8.3.2 The (Worksheet Properties) Miscellaneous tab

WksProperties-Dialog-Misc-Tab

Controls on this tab are used to auto add rows to the worksheet or to resize cells to fit cell contents.

Miscellaneous

Auto Add Rows

Specifies that rows be added automatically. If selected, clicking the Enter key in the last row will add 10 rows to the worksheet.

Fit Sheet to Window

Make the worksheet fit the window, when the workbook is resized. When enabled, the window does not use scroll bars but fits all columns and rows into the window, reducing row height and column width, as needed.

Ignore Hidden Rows on Plotting and Analysis

Ignore hidden rows during subsequent plotting and analysis operations.

Resize Cells

These controls affect resizing of worksheet cells when the Resize Cells to Fit Content button is clicked. The size unit for these controls is roughly that of a character whose font is Courier New 9PT.

  • Min Width
Controls minimum width of empty worksheet cells. The size unit for these controls is roughly that of a character whose font is Courier New 9PT.
  • Max Width
Controls maximum width of filled worksheet cells. The size unit for these controls is roughly that of a character whose font is Courier New 9PT.
  • Min Height
Specify the minimum height of cells in the worksheet. The size unit for these controls is roughly that of a character whose font is Courier New 9PT. To automatically adjust cell height to cell contents, set Min Height and Max Height to -1, then click Resize Cells...
  • Max Height
Specify the maximum height of cells in the worksheet. The size unit for these controls is roughly that of a character whose font is Courier New 9PT. To automatically adjust cell height to cell contents, set Min Height and Max Height to -1, then click Resize Cells...
  • Resize Selection Only
Apply resize settings to selected worksheet cells only.
Resize Cells to Fit Content

Click this button to resize worksheet cells, within the boundaries set by the selected Resize Cells controls.

The Resize Cells controls are difficult to master. Most users will not need to concern themselves with these controls. However, if you have a worksheet with sharply-contrasting cell sizes and you want to make some global adjustment so that all cells show full contents, the strategy that seems to work best is to set Min Height and Max Height to -1, then set Max Width so that the cell with the largest content displays all contained data. Click Resize Cells to Fit Content and check to see that the cell contents display properly.
