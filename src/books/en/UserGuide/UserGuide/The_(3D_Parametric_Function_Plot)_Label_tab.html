8.3.6.4.2 The (3D Parametric Function Plot) Label tab

PD-Dialog-3DParaFunc-Label-Tab

Set function headers for independent and dependent variables on the Label tab. This tab is organized as a table, allowing you to enter X , Y and Z headers (e.g., Long Name, Units, and Comments). These headers can be displayed in the Function Plot graph. For example, Long Name and Units can be used as titles, and Z Comment can be displayed as a legend.
