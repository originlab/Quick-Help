Data Requirements

Select exactly one column of values or a range from one column.

Creating the Graph

Select the required data.

Select Plot > Statistical: Probability Plot...

Or

Click the Probability Plot... button on the 2D Graphs toolbar.

Origin opens the plot_prob dialog box.

Customize the options in the dialog and then click OK to create the plot. Please see more details on creating and customizing the Probability Plot.

Template

PP.OTP (installed to the Origin program folder).

Notes

For information about the calculations for Propability Plot, please refer to Probability_Plot_and_Q-Q_Plot.
