Specify Binning Range by

Bin Centers

Bin center is generally calculated by subtracting half bin size at its bin end. Besides, by customizing Minimum Bin Center and Maximum Bin Center under Computation Control section, you can define the limitation of bin centers, other bin centers will be added one bin size on the basis of its previous one.

Bin Ends

Define bin range using minimum beginning bin and maximum ending bin values.

User-Defined Boundaries Bin boundaries are defined by an user specified dataset, from minimum beginning bin value to maximum ending bin value. Values must increase monotonically. Select user defined bin boundaries dataset from Binning Range Data.
User-Defined Centers Bin centers are defined by an user specified dataset. Values must increase monotonically. Select user defined bin centers dataset from Binning Range Data.

Binning Range Data

Binning Range Data

Use controls to specify bin dataset.

For help with range controls, see: Specifying Your Input Data

Computation Control

Minimum Bin Center/Minimum Bin Beginning Specify minimum value of bin center or bin beginning, depending on Specify Binning Range by setting. Clear Auto to manually edit minimum value.
Maximum Bin Center/Maximum Bin End Specify maximum value of bin center or bin end, depending on Specify Binning Range by setting. Clear Auto to manually edit the maximum value.
Step by Specify how to divide input dataset into bins.

Option list:

  • Bin Size
    Divide input data by a fixed bin size specified in Bin Size.
  • Number of Bins
    Divide input data into n bins. n is specified in Number of Bins.
Bin Size Specify the bin size for sorting input data. The Number of Bins is computed accordingly.
Number of Bins Specify the number of bins for sorting input data. The Bin Size is computed accordingly.
Include Outliers < Minimum Include outliers that are less than minimum in the lowest bin.
Include Outliers >= Maximum Include outliers that are greater than or equal to maximum in the highest bin.
Separately Count Minimum Consider the minimum as a separate bin.
Separately Count Maximum Consider the maximum as a separate bin.

Quantities to Compute

Bin Center Compute and output the bin center.
Bin End Compute and output bin end.
Count Compute and output counts for each bin.
Cumulative Count Compute and output cumulative counts for each bin.
Relative Frequency Compute and output the relative frequency for each bin.
Cumulative Frequency Compute and output the cumulative frequency for each bin.
Frequency in Display frequency as fractional or decimal. Only available when Relative Frequency or Cumulative Frequency is checked.
