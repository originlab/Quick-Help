29.12.6 Kite Diagram

Kite-Diagram


Contents

  • 2 Creating the Graph
  • 3 Template
  • 4 Notes

Data Requirements

Select at least two Y columns of values (or a range from at least two columns). If there is an associated X column, X column supplies X values; otherwise, row index number is used. Individual plots are are labeled along the vertical axis using column Long Name.

Creating the Graph

Select required data.

From the menu, select Plot > Bar, Pie, Area: Kite Diagram.

Template

kite.otpu (installed to the Origin program folder).

Notes

  • Adjust the padding of top and bottom plots by double-clicking the vertical axis and modifying the Scale tab From and To values.
  • Vertical grid lines are added by double-clicking the horizontal axis and checking Show under Major Grid Lines and/or Minor Grid Lines.
