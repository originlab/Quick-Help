23.2.6 Export Graph with Highlight Effect

Export-Highlight-Effect

Origin supports to export the Plot Selection effect and Data Highlighter effect in a graph by Export Graph tool or Copy Graph as Image tool.


  • Use Data Highlighter tool to highlight parts of the graph to export.
  • Click to select a plot in a graph with multiple plots and export.
  • Use Object Manager to select multiple plots in a graph and export.
  • Export Graph (Advanced) tool does not support this highlight Effect.
  • Besides Copy Graph as Image tool, when you use Copy Graph and paste it as picture, the highlight effect will be kept.
