8.12.4 The Extract Layers to Graphs Dialog Box

GraphRef-ExtractLayer

Contrary to merging layers to a graph, Extract to Graphs is used to extract the layers to graphs.

This dialog can be opened in three methods,

  1. Click the Extract to Graphs button,
  2. Run layextract -d in the script window,
  3. Right-click the graph to select Extract to Graphs from the context menu.

More details about this dialog, please refer to this XF page.
