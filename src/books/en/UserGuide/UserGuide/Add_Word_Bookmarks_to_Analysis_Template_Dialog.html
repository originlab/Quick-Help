27.4 Add Word Bookmarks to Analysis Template Dialog

AddWord-Bookmarks-Dialog

The Add Word Bookmarks to Analysis Template dialog box is used to add bookmarks from a Word template to an Analysis Template, so that batch processed results can be directly outputted to corresponding bookmarked cells in Word template for reporting.

To open the Add Word Bookmarks to Analysis Template dialog, go to menu File: Add Word Bookmarks to Analysis Template....



The Add Word Bookmarks to Analysis Template dialog allows you to add bookmarks from a Word template to an analysis template. When bookmarks are added to an analysis template, a new worksheet Bookmarks will be automatically created and appended to the analysis template. The added bookmarks are then placed sequentially into the column Bookmarks (Long Name) in worksheet Bookmarks. Column Links (Long Name) in Bookmarks worksheet is used to store the corresponding data/graph links from report sheet. Graph Width (Long Name) column is used to adjust the size of inserted graphs in units of points. The path of used Word template is appended to Word Template label row.

To make an one-time Word report, you can import file into the data sheet and when the analysis results are ready on Bookmarks sheet, you can simply click Export to Word button on the top left corner to create it.
