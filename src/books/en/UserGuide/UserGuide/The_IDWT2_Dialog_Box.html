18.13.2.1 The 2D Wavelet Reconstruction Dialog Box

IDWT2-Dialog

  • Menu Command: Analysis: Signal Processing: Wavelet: 2D Reconstruction
  • Window Type: Matrix, 3D/Contour Plots
  • X-Function Script and GUI Examples: idwt2(Pro)
Dialog Theme See Dialog Themes.
Recalculate

See Recalculating Analysis Results.

Approximation Coefficients

Specifies the approximation coefficients. See Specifying Your Input Data.

Detail (Horizontal)

Specifies the horizontal detail coefficients. See Specifying Your Input Data.

Detail (Vertical)

Specifies the vertical detail coefficients. See Specifying Your Input Data.

Detail (Diagonal)

Specifies the diagonal detail coefficients. See Specifying Your Input Data.

Wavelet Type

Specifies the type of the wavelet.

  • Haar: Haar wavelet
  • DBN: Daubechies wavelet (N = wavelet order and no. of vanishing moments)
  • BiorNr.Nd: Biorthogonal (Nr = wavelet order or scaling functions used for reconstruction; Nd = order of the functions used for decomposition)
Output Matrix

Specifies the output for the reconstructed signal. See Output Results.
