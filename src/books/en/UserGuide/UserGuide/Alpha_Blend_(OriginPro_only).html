20.5.6.1 Alpha Blend

Image-AlphaBlend

The imgBlend function combines image data from two images with a constant opacity. To use this function,

select Image: Arithmetic Transform: Alpha Blend from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function imgBlend in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help imgBlend
