8.3.6.5 Using theme in Function Plot

FuncPlot-UseTheme

When you've defined/customized a function on Create Function Plot dialog or the Function tab of the Plot Details dialog, you can use the Copy and Paste theme flyout menus or buttons to share the definition/customization between the Create Function Plot dialog or Function tab of Plot Details dialog.

Example:

  1. After defining or modifying a function on the Function tab, click the Copy button in the Theme group.
  2. Select File: New: Function Plot from the menu to open the function plot dialog.
  3. In the function plot dialog that opens, click the triangle button to the right of Theme, and then select Paste from the context menu. This will apply all settings from the Function tab of the Plot Details dialog.
