19.1.3 Creating Baselines with the Peak Analyzer

CreateBaseline-PA


If the Create Baseline radio button is selected in the Goal group in the Start page of the Peak Analzyer, you can use the Peak Analyzer to create a baseline for the input data. A typical wizard map corresponding to this mode is as follows:

When the Peak Analyzer is used in this mode, only User-Defined, XPS and End Points Weighted baseline models are available. You can define the baseline anchor points in the Baseline Mode page and then connect them to form a user-defined baseline in the Create Baseline page.

See the reference for more details on the Baseline Mode page and the Create Baseline page.
