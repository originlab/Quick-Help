29.17.6 3D Bar Graph with Categorical/Text XY

3DXYZBar-Text-XY

Data Requirements

  • Select XYZ columns in the worksheet. X and Y columns should contain categorical/text data.

Or

  • A block of worksheet cells (virtual matrix); the X and Y values are text.

Creating the Graph

Select required data.

From the menu, choose Plot> 3D : 3D Bars.

or

Click the 3D Bars button on the 3D and Contour Graphs toolbar.

Template

  • gl3DBARS.OTP (OpenGL)
  • 3DBARS.OTP

(Installed to the Origin program folder).
