18.9.1 The Matrix Smoothing Dialog Box

MSmooth-Dialog

Input Matrix Specify the matrix that contains the input data.
Column Factor The number of column cells to be averaged†.
Row Factor The number of row cells to be averaged†.
Output Matrix Specify the destination matrix to output the smoothed data.
