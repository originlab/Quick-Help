20.5.6.2 Extract to XYZ

Image-Extract-to-XYZ

The Img2xyz function extracts the intensity data and the corresponding X, Y coordinate data of a selected area on an image into an XYZ data of the worksheet. To use this function,

select Image: Arithmetic Transform: Extract to XYZ... from the Origin menu.

For more specific information on this function, see documentation provided for the X-Function Img2xyz in the Origin X-Function reference file. This can be accessed from the Command Window by typing

help img2xyz
