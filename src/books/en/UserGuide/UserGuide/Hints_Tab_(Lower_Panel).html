15.3.2.13 Hints Tab (Lower Panel)

NLFit-Dialog-HintTab

The Hints tab may offer useful information about use of NLFit dialog box controls when controls are activated (note that hints are only offered for selected actions).

For example, if you were to open the NLFit dialog box and fail to choose a fitting function, the Hints tab will display the message seen above.
