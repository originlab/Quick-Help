Brief Description

Damped sine wave, a sinusoidal function whose amplitude decays as time increases.

Sample Curve

Parameters

Number: 5

Names: y0, xc, w, t0, A

Meanings: y0 = offset, xc = phase shift, w = period, t0 = decay constant, A = amplitude.

Lower Bounds: w > 0.0 , t0>0.0

Upper Bounds: none
