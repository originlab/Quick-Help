11.14 Image/Contour Profile Gadget

gadget-image-contour-profile

The image/contour profile gadget allows you to view and profiling matrix or virtual matrix data in an image plot, a contour plot.

To use this tool, make sure the active graph is an image/contour/heatmap plot created from a matrix or virtual matrix and then select Gadget:Image/Contour Profile from the Origin menu.

For more details on the tool, please read the Viewing and Profiling Matrix Data page in the Matrixbooks, Matrixsheets, and Matrix Objects chapter.
