29.13.9 Polar Vector theta r theta r graph

Polar-Vector-theta-r


Contents

Data Requirements

Select two pair of XY columns, with the order theta r theta r.

Creating the Graph

Select Plot > Specialized: Polar Vector theta r theta r

Template

PolarVect.otpu (All above are installed to the Origin program folder).

Notes

This is a XYXY vector in the polar coordinate, the XYXY columns contain the angle and radial coordinates of start and end points.

You can go to the Vector tab of Plot Details to customize the vectors.
