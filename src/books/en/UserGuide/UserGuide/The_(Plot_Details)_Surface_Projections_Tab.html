9.5.2.2 The (Plot Details) Surface/Projections Tab

PD-Dialog-SurfaceProjection-Tab

The Surface/Projections tab provides controls to edit the surface display of a color map surface graph, as well as the top and bottom contour projections. This tab also provides controls to hide data above or below a specified Z value.

Contents

  • 1 Surface, Top Contour and Bottom Contour
  • 2 Z Clipping

Surface, Top Contour and Bottom Contour

To display a fill color on the 3D surface, select the Fill Color check box in the Surface column. To display the fill color as top or bottom contours, select the Fill Color check box in the Top Contour or Bottom Contour column.

Similarly, to display contour lines with the fill color, select the check box associated with the surface or the desired contour. Select the contour line color and width from the associated drop-down lists. The contour line width is in units of points.

Show Major Levels Only

This check box is available when one of the Contour Line check boxes above is selected. If this check box is checked, the contour lines on surface, top contour lines and bottom contour lines are only shown on major levels. If not checked, contour lines will show on both major and minor levels.

Z Clipping

The Z clipping controls allow you to clip the color map surface at the lower or upper axis scale range. Both controls are in units of % of the Z axis scale.

When the Low (%) text box is set to 0, the surface is not clipped at the low end. If you type n in this text box, the surface that is associated with the lower n% of the Z axis scale is clipped from view.

When the High (%) text box is set to 100, the surface is not clipped at the high end. If you type n in this text box, the surface above n% of the Z axis scale is clipped from view. For example, type 75 to clip the surface associated with the top 25% of the Z axis scale range.
