Data Requirements

Select at least one X column and one Y column.

Creating the Graph

Select required data.

From the menu, select Plot > Statistical: Marginal Box Charts.

or

Click the Marginal Box Charts button on the 2D Graphs toolbar.

Template

MARGINBOX.OTP (installed to the Origin program folder).

Notes

This graph is composed of three layers. One is a 2D scatter plot. A horizontal box chart represents the X-dimension distribution of the scatter data. A vertical box chart represents the Y-dimension distribution of the scatter data.
