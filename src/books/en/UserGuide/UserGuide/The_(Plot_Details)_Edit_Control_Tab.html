9.3.7.15 The (Plot Details) Edit Control Tab

PD-Dialog-EditControl-Tab

The Edit Control tab appears at the Data level in Plot Details for 3D XYZ plots and controls the editing behavior of plot symbols and their projections. For 3D XYZ Scatter plots, you can edit Line, Symbol, and Drop Line properties. You can additionally display projections onto the XY, YZ and ZX planes, each of which also has editable Line, Symbol, and Drop Line properties. This tab gives you options of editing all properties as a group, all properties independently or editing all projections as a group.

To access this tab, you have to highlight the XYZ dataset in the left side of the Plot Details dialog box.

Edit Dependencies Group

All Together

Select to display and edit and apply the settings on the (Plot Details) Line, Symbol, and Drop Lines tabs to the (Original) 3D data plot and to the three Projections. When this option is selected, you will be able to check the XY, ZX, and YZ Projection boxes, but the selection will then bounce back to 'Original' since all properties are edited as a group.

Fully Independent

Select to edit elements for each of the 3D data plots (Original + three projections) separately.

Original Independent of Projections

Select to edit the 3D data plot (Original) element separately and edit all projections as a group. When this option is selected, you will be able to check the XY, ZX, and YZ Projection boxes, but the selection will then bounce back to 'XY Projection' since all projection properties are edited as a group.

Shift in Z by percent of scale range, 0=bottom, 100=top

Specify whether to shift the 3D bar graph in the z direction by a percent of the scale range.
