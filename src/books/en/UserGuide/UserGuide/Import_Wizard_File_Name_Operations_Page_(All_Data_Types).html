6.4.8.2 Import Wizard, The File Name Options Page

ImpWiz-FileNamePage

Contents

  • 1 Rename
  • 2 Append

Rename

Auto

Automatically rename worksheets according to the import mode and filenames

Worksheet with File Name

Rename worksheet with file name upon import.

Worksheet with variable

Choose a variable to rename the worksheet from the drop-down list which is extracted from the file header in Variable Extraction page.

Workbook with File Name

Set workbook long name with filename upon import.

Workbook with variable

Choose a variable to rename the workbook from the drop-down list which is extracted from the file header in Variable Extraction page.

Rename Long Name for Book only

Only rename the Long Name for Worksheet and maintain the Short name as default.

Include Path When Renaming Workbook

If this checkbox is selected, full pathname is used to rename the target workbook.

Append

File Name to Workbook Comments

Store the import file name with the workbook Comments (accessible in both the Comments section of the Window Properties dialog box and the Workbook Organizer).

File Name to Column Comments

Store the import file name with the column Comments (accessible in both the Comments section of the column's Properties dialog box and the Comments row of the worksheet).

Add file name user parameters row

When this is checked, the file name is put to user parameter row SourceFile.

Include Path When Appending File Name

When this is checked, the import file path is included with the file name and the file extension is also appended.

Note: This page will not show, when choose User-Defined as Data Type in the Source page
