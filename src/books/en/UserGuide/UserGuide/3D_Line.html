29.17.3 3D Line Graph

3DLine

Data Requirements

Select at least one Z column (or a range from at least one Z column) in worksheet. If the Z column has associated XY columns, then the XY columns are used; otherwise, the worksheet's default XY values are used.

Creating the Graph

Select required data from worksheet

From the menu, choose Plot > 3D: 3D Line.

Template

gl3DLINE.OTP (Installed to the Origin program folder).
