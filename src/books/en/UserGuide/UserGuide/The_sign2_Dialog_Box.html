Standard Controls

Results Log Output

Select to output results to the Results Log.

Recalculate

Controls recalculation of analysis results:

  • None
  • Auto
  • Manual

For more information, see: Recalculating Analysis Results

Test Options

Input

1st Data Range

The first sample column.

2nd Data Range

The second sample column.
Null Hypothesis

The null hypothesis: the median difference between paired samples is zero(0). Read only.

Alternate Hypothesis

Median1 <> Median2'

Performs a 2-tailed paired sample sign test.

Median1 > Median2

Performs an upper-tailed paired sample sign test.

Median1 < Median2

Performs a lower-tailed paired sample sign test.
Significance Level

The alpha(α) value.

Output Results

Specify the output data range.

For help with the range controls, see: Output Results.
