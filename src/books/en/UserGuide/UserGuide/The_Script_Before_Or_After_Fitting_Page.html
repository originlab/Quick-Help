15.3.4.4.11 The Script Before Or After Fitting Page

FitFuncBuilder-Script-Before-After

This page offers options to input scripts which will be executed before and/or after the fitting process. This page is optional, so you can directly click the Finish button to finish the defining/editing fitting funtion process.


Dialog Box Controls

Script Before Fitting Enter Scripts which will be executed before the fitting process.
Script After Fitting Enter Scripts which will be executed after the fitting process.
