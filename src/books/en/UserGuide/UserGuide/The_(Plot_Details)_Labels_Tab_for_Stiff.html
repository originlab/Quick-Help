9.3.7.33 The (Plot Details) Labels Tab for Stiff Diagram

PD-Labels-tab-for-Stiff

This tab can be used to control all kinds of labels in a Stiff diagram.

Show for inons

Controls for the labels of ions.

Label Form

Specify a column label row to show the ion name. If there are nodes plotting combined ions, such as Na⁺ and K⁺, HCO₃⁻ and CO₃⁻, Origin will show them as Na⁺ + K⁺, HCO₃⁻ + CO₃⁻ at the node.

Font

Specify the font and format for the title text.

Align to Outside End

Specify whether align the labels to the outside end.

Label position for Cations is on the left of symbols, for Anions it is on the right of symbols by default. Select this control, Origin will align the labels for cations to left and align the labels for anions to right.

Horizontal Offset

Specify the horizontal offset of the labels from the nodes(symbols) of the polygons. The unit of this control is % of font height.

Vertical Offset

Specify the vertical offset of the labels from the nodes(symbols) of the polygons. The unit of this control is % of font height.

Show for Sample ID

Controls for the labels of the sample ID.

If you don't set Sample ID in the plotstiff dialog, this group is not available. You can check the check box before this item to enable the labels from the sample ID column.

Label Form

Specify a column to apply as Sample ID for the polygons.

Font

Specify the font and general format for the tick labels.

Position

Specify the position of the sample ID.

Horizontal Offset

Specify the horizontal offset of the sample ID, from the position you set. The unit of this control is % of font height.

Vertical Offset

Specify the vertical offset of the sample ID, from the position you set. The unit of this control is % of font height.
