29.16.12 Multiple Color Map Surfaces

Multi-Color-Map-Surface

Data Requirements

You need to have at least two matrix objects in the same matrixsheet. When you select any matrix object, all the matrix objects will be included in the same layer of the graph.

Creating the Graph

Activate the matrixsheet and select a matrix object.

From the menu, choose Plot > 3D : Multiple Colormap Surfaces.

or

Click the Multiple Color Map Surfaces button on the 3D and Contour Graphs toolbar.

Template

  • glCMAP.OTP (OpenGL)
  • CMAP.OTP

(Installed to the Origin program folder).
