Data Requirements

You need to have at least three matrix objects in the same matrixsheet.

Creating the Graph

Activate the matrixsheet.

From the menu, choose Plot > 3D : Parametric Surface.

Template

  • GLparafunc.OTPU

(Installed to the Origin program folder).

Notes

To make this plot, you need a matrix sheet with at least 3 matrix objects.

The 1st object contains Z values, and the 2nd & 3rd objects should contain X & Y values. The 4th optional object will be for coloring.
