29.6.5 2Ys Column-Line Symbol

Data Requirements

Select at least one Y column (or a range from at least one Y column). Ideally, select two Y columns (or a range of two Y columns). If there is an associated X column, X column supplies X values; otherwise, sampling interval of the Y column or row number is used.

Multiple Y columns will be sepereated into 2 groups by 2 X columns; otherwise, these Y columns will be assigned to different Y axis by sequence.

Creating the Graph

Select required data.

Select Plot > Multi-Panel/Axis: 2Ys-Column-LineSymbol from the main menu.

Template

2Ys_ColSy.OTP (installed in the Origin program folder).

Notes

This graph type has two layers and each layer has a Y axis, same as the 2Ys Y-Y graph.
