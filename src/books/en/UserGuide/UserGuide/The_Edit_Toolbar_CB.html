28.2.2 The Edit Toolbar

CB-Edit-Toolbar

The Edit toolbar buttons allow you to comment,find and reset files,in addition to bookmark.

Comments selected lines of text.

Uncomments selected lines of text.

Finds the specified files.

Finds the previous occurrence of the specified files.

Finds the next occurrence of the specified files.

Replaces specific text with different one.

Repeat the last action.

Opens text editor options dialog box.

Resets options to their defaults.

Undo the last action.

Redo the previously undone action,

Goes to matching brace.

Toggles bookmark.

Clears bookmark.

Goes to next bookmark.

Goes to previous bookmark.
