27.6.5 Exporting Toolbar and Other Workspace Settings

Export-Toolbar-Setting

The configuration of menus, built-in toolbars and buttons and dockable windows can be exported as an .OPX file. This allows you to move or share a particular workspace configuration by dropping the .OPX file into the Origin workspace.

Note that button group information is not included when you export toolbar settings in this way. If you have added custom button groups to your current Origin installation, then you need to export those separately as individual .OPX files in order to add those to another installation of Origin.

To export menu, toolbar and dockable window settings as an .OPX file:

  1. From the menu, choose View: Toolbars to open the Customize Toolbar dialog box.
  2. Select the Toolbars tab.
  3. Select a toolbar to be exported from the Toolbars list box.
  4. Click the Export button. This opens the Save As dialog box.
  5. Edit the file name and path, as needed*.
  6. Click Save to create the .OPX file.

*Note: The file name extension must be .OPX. You do not need to enter this extension as Origin will add it automatically. If you enter an extension other than .OPX, .OPX will be appended on the file name.
