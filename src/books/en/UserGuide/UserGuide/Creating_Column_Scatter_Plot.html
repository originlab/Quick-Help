17.2.6 Creating Column Scatter Plot

Create-Column-Scatter-Plot


To create a column scatter plot, you can select one or more Y data and use the Plot:Symbol:Column Scatter menu entry.

To create a grouped column scatter plot, you can create a grouped box chart with indexed data or raw data first and select Box_Column Scatter from Graph Theme drop-down list, the menu entry is Plot:Statistics:Grouped Box Chart - Indexed Data or Plot:Statistics:Grouped Box Chart - Raw Data.
