20.3 Publishing and Exporting Images

Image-Publish-Export-Images

Exporting an image from Image window

  1. Select menu File: Export Image.... This opens the CvExport dialog.
  2. Select a File Type and specify the file name and path.
  3. If File Type = AVI, choose a Codec.
  4. Click the Save button to export the image or video.

Exporting an image from Matrix window

  1. Select menu File: Export: Image.
  2. In the imageExp dialog, select a file type in the Save as type drop-down list and specify the file name. Check Show Options Dialog if you want to specify more options. Click Save button.
  3. If Show Options Dialog is checked, the expImage dialog opens. Specify the export options and then click OK to export the image.
