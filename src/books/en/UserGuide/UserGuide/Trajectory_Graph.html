29.17.4 Trajectory Graph

Trajectory-Graph

Data Requirements

Select one Z column (or a range from one Z column). If the Z column has associated XY columns, then the XY columns are used; otherwise, the worksheet's default XY values are used.

Creating the Graph

Select required data from worksheet.

From the menu, choose Plot > 3D: 3D Trajectory.

or

Click the 3D Trajectory button on the 3D and Contour Graphs toolbar.

Template

  • glTraject.OTP (OpenGL)
  • TRAJECT.OTP

(Installed to the Origin program folder).
