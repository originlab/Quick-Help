8.4.2.1.11 The Apply To dialog

AxesRef-Apply-To

This dialog can be opened by clicking the Apply To... button at the bottom of the Axis dialog. You can specify format of Tick Labels and Line and Ticks, and Grib Lines to apply from source axis to objective axis.

Contents

  • 1 Apply From
  • 2 Apply To
  • 3 All Below
  • 4 Tick Labels
  • 5 Line and Ticks
  • 6 Grid Lines

Apply From

Specify the source axis to apply from.

Apply To

Specify the objective axis to apply to.

All Below

Select all items below to apply from source axis to objective axis.

Tick Labels

Select the format of tick labels to apply from source axis to objective axis..

Line and Ticks

Select the format of line and ticks to apply from source axis to objective axis..

Grid Lines

Specify whether to apply the format of Grid Lines of source axis to objective axis.
