29.13.4 Contour + Streamline

Contour-Streamline

Contents

Data Requirements

Matrix: A matrix of Z values.

Creating the Graph

Select required data.

Select Plot > Specialized : Contour + Streamline.

Template

ContStream.otpu (Installed to the Origin program folder).

Note

Contour + Streamline graph is a graph with a streamline plot overlaying on the contour line plot.

The X-Function plot_mquiver is used to create such graph, same as the graph Contour Line + Gradient Vector.
