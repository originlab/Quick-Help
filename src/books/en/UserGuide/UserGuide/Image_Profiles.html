20.5.1.2 Image Profiles

Image-Profiles

The image profiles plot provides a quick way to analyze the image data and generate profiles. You can create horizontal, vertical and arbitrary line profiles of the image data dynamically.

To create an image profiles plot,

select Plot: Image: Profiles from the Origin menu.

Note that the image profiles plot requires the input image should be grayscale or of 8-bit color. If the color depth of the input image does not meet such requirement, you will have to convert the image first.
