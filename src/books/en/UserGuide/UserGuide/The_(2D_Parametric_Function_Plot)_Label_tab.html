8.3.6.2.2 The (2D Parametric Function Plot) Label Tab

PD-Dialog-2DParaFunc-Label-Tab



Set function headers for independent and dependent variables on the Label tab. This tab is organized as a table, allowing you to enter X and Y headers (e.g., Long Name, Units, and Comments). These headers can be displayed in the Function Plot graph. For example, Long Name and Units can be used as X and Y axis titles, and Y Comment can be displayed as a legend.
