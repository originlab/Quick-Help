2.1.5 Arrange Windows

Arrange-Windows


Overview

In Origin workspace, you can arrange Workbook, Matrix, Graph, Notes windows by Cascade, Tile Horizontally and Tile Vertically under Windows main menu.

From 2022b, Origin supports more arrange operations in the Arrange Windows. To open this dialog, select Windows: Arrange Windows

  • See Snapping Windows
  • Pin Window to skip from arrange windows
  1. Right click on the window header and select Pin Window in the context menu. Or active the window, and run @LWP=1 in the Script Window to pin it.
  2. Click the Remove Window Pin button in the header of the window to unpin it.
  • After arranging windows, if you want to enlarge a window for better viewing. You can restore the window to its previous size and position by clicking Window: Restore Window Size (Ctrl + Shift + Z).

Arrange Windows dialog

Columns Specify the number of columns when you sorting the windows.
Sorting Order
  • By Window Activated Order
    Sorting the Windows by the activated order
  • By Project Explorer Order
    Sorting the Windows by the order listed in the Project Explorer
  • By Window Name Ascending
    Sorting the Windows by the name ascending
  • By Window Name Descending
    Sorting the Windows by the name descending
Window Name When you select By Window Name Ascending or By Window Name Descending as Sorting Order, you can select use the Short Name or Long Name as the order.
  • Short Name
  • Long Name if Exists
First by Window Type When you select By Window Name Ascending or By Window Name Descending as Sorting Order, you can check this option to use Window Type as the highest priority sorting level.
Direction
  • Horizontal First
    Sorting the windows in horizontal direction firstly.
  • Vertical First
    Sorting the windows in vertical direction firstly.
Overlap Each Column Specify whether the window shows overlap. If it is not selected, the windows will be tiled.
Offset (Title Bar Height=10) When select the Overlap Each Column option, you can customize the offset of each window.

Note: this is the vertical offset. The horizontal offset is fixed value.

Include Minimized Windows When check this option, the minimized windows will be arranged as other windows.
Share Active Window Size Set other windows size the same as the active windows when arrange Windows.
