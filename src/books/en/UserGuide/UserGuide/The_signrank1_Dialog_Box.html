Standard Controls

  • None

For more information, see: Recalculating Analysis Results

Test Options

Input

Enter your input data.

For help with range controls, see: Specifying Your Input Data

Test Median

Specify the value for null hypothesis.

Null Hypothesis

The null hypothesis to be tested.

Alternate Hypothesis
  • Median <> 0
Performs a two-tailed Wilcoxon Signed-Rank test.
  • Median > 0
Performs a upper-tailed Wilcoxon Signed-Rank test.
  • Median < 0
Performs a lower-tailed Wilcoxon Signed-Rank test.
Significance Level

The alpha(α) value.

Output Results

Specify the output data range.

For help with the range controls, see: Output Results.
