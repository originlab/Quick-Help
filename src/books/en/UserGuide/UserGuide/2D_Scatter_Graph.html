29.2.1 2D Scatter Graph

2DScatter-Graph

Data Requirements

Select one or more Y columns or a range of data from at least one Y column. If there is an associated X column, X column supplies X values; otherwise, sampling interval of the Y column or row number is used.

Creating the Graph

Select required data.

From the menu, select Plot > Basic 2D: Scatter.

or

Click the Scatter button on the 2D Graphs toolbar.

Template

SCATTER.OTP (installed to the Origin program folder).
