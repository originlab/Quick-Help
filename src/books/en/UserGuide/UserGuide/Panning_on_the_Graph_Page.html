8.3.9 Panning on the Graph Page and inside Layer

Pan-GraphPage

Panning is very useful on graph page and inside layer especially when you have zoomed in/out on a graph and want to move the graph to view different portions. In Origin/OriginPro, panning mode should be able to be in use together with other tools, such as Data Reader.

Panning on the Graph Page

Click the Zoom-Panning Tool button to enable panning, then press the left button of mouse to drag the graph to the interesting portion.

Note: Panning tool will only move the graph page, has nothing to do with the axis scale.

Panning inside the Graph Layer

Use the Scale Zoom-Panning Tool button or hold down the X character key to enable panning inside the active layer, then press the left button of mouse to move to the interesting range of the layer. The arrow key can also be used with the X key to perform the same action.

Pan horizontally
  • Scale Zoom-Panning Tool button + press X key + Mouse Wheel
  • Scale Zoom-Panning Tool button + press X key + +/- key
  • Scale Zoom-Panning Tool button + Right/ Left Arrow key
  • Hold down X key + Mouse Wheel/ Mouse-drag
  • Hold down X key + +/- key
  • Hold down X key + Right/ Left Arrow key


Drag axis to rescale and pan:

  • Click X Axis to show two red dots at the ends of axis, drag the red dot horizontally
  • Press Scale Zoom-Panning Tool button or Hold down X key + Drag and Drop Y Axis Line horizontally
Pan vertically
  • Scale Zoom-Panning Tool button + press X key + hold down Shift key + Mouse Wheel
  • Scale Zoom-Panning Tool button + press X key + hold down Shift key + +/- key
  • Scale Zoom-Panning Tool button + Up/ Down Arrow key
  • Hold down X key and Shift key + Mouse Wheel/ Mouse-drag
  • Hold down X key and Shift key + +/- key
  • Hold down X key + Up/ Down Arrow key


Drag axis to rescale and pan:

  • Click Y Axis to show two red dots at the ends of axis, drag the red dot vertically
  • Press Scale Zoom-Panning Tool button or Hold down X key + Drag and Drop X Axis Line vertically
Pan both directions
  • Scale Zoom-Panning Tool button + Mouse-drag
Note: Rescale Tool button can also auto rescale the axis range.

Toggle Panning Mode

When the panning tool is active, pressing the Space button can toggle panning mode, and the cursor shape will also be changed.
