6.5 FAQ-6 I installed the Origin (or OriginPro) trial version but I really want to evaluate OriginPro (or Origin). How do I fix this?

Convert-to-ProDemo

Last Update: 10/9/2018

If you installed the Origin (or OriginPro) trial version but you really want to evaluate OriginPro (or Origin), you can check (or uncheck) Help: Run Trial as OriginPro to convert it to a Pro (or Standard) version.
