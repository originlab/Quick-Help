2.99 FAQ-974 How do I display Degrees, Minutes and Seconds in Origin?

Display-Degrees-Minutes-Seconds-in-Origin

Last Update: 4/25/2023

Origin supports display of latitude and longitude measurements in common formats such as DMS (degrees-minutes-seconds) or decimal degrees. Custom formatting of such geographic data is supported for worksheets, graphs, text labels, and in data point tooltips and the Data Info tool (see bullet list below).

As the following image shows, it is possible to apply a different custom format wherever your geographic data are displayed.

For specific information on customizing the display of geographic data, see:

  • Custom Numeric Formatting of Worksheet Data: Degrees, Minutes and Seconds
  • Graph Axis Dialog > Tick Labels Display > Custom Display
  • Origin Formats (including examples)
  • The Data Info Window and Data Point Tooltip


For an overview of controlling numeric display in Origin, see the blog post "Custom Formats of Numeric Display in a Worksheet or a Graph."
