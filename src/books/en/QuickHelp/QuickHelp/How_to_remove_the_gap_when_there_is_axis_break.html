1.230 FAQ-997 How to change the shape of axis break?

Remove-Gap-Axis-Break

Last Update: 3/16/2020

The axis break is added as diagonal symbol by default.

To change the axis symbol to other shape
  1. After adding the break, click on it and select Axis Break Symbol botton on the mini toolbar that appears.
  2. Select a desired symbol from the sub-menu list.
To remove break gap as below graph

  1. With the graph active, open the Script Window (from menu, Window: Script Window) if it is not already open.
  2. Copy and paste below LabTalk command layer.x.breaks.gap in the Script Window. Press Enter to run it.
layer.y.breaks.gap=0
