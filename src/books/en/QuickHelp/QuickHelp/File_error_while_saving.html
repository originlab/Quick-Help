5.83 FAQ-1153 Script Window opens, reports "err saving <object>"

file-err-saving

Last Update: 9/26/2022

Problem:
When trying to save a project file, you get an Attention message that says "File error while saving to <path>. File not saved. Check to make sure the disk is not full."

Simultaneously, the Script Window opens and reports an error such as:


err saving #1781A405628YLNx:3

Solution:
You need to delete the object mentioned in the error message in the Script Window. The object name includes the # character but does not include the : character. So, for the above sample error message, you would need to type the following command in the Script Window and press Enter: 

del #1781A405628YLNx

Keywords:saving, save, file not saved, error, disk full
