6.80 FAQ-1092 I see error message "Learning Edition data is not right" when activate Origin. What should I do?

Learning-Edition-Data-Is-Not-Right

Last Update: 5/5/2021

If you see error message Learning Edition data is not right when activate the Learning Edition,

Please double check if you installed Learning Edition with the correct serial number.

  • You can select Help: About Origin to know about the installed serial number. Check if it is the same number you received from the email. If not, please follow instruction on this quick help page to correct it.
  • If you applied for the Learning Edition multiple times, the serial number may be different. Make sure you installed Origin with the number in the last email.

If the installed serial number is correct but the problem still happens, please send email to le@originlab.com. Please include your product key and email address you used to apply Learning Edition.
