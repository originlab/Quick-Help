1.152 FAQ-795 How do I export graphs as EPS or PDF file with editable text?

Exp-Editable-Text-EPS

Last Update: 4/1/2024

Some scientific journals prefer graphs in EPS and PDF formats which can be edited further. To export an Origin graph as EPS and PDF with editable text labels, please set options in the expGraph dialog as below:

  • Export as EPS files: Expand branch Image Settings: Fonts and select Use Built-In Fonts radio button.
  • Export as PDF files: Expand branch Image Settings: Fonts and select Embedded radio button.


Sometimes there is unexpected problem in exported EPS or PDF file. you can also consider to export graphs as EMF or SVG, open them with Adobe Illustrator and then save them as EPS/PDF.
