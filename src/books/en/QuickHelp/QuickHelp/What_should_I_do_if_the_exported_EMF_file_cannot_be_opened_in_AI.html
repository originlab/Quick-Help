1.271 FAQ-1125 What should I do if the exported EMF file cannot be opened in AI?

Export-EMF-Cant-Open-in-AI

Last Update: 5/8/2022

If you use menu File: Export Graph (Advanced) to export an Origin graph as EMF format, but cannot open it in Adobe Illustrator, please try following solutions to workaround it.

  • In the expGraph dialog, under the Export Settings, change Advanced radio choice from Set Resolution to Keep Size.

Or,

  • Use File: Export Graph instead. This expG2img tool internally uses Keep Size.
