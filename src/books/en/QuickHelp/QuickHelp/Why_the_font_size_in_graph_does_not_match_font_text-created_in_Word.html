1.96 FAQ-452 Why does the font size in Origin graph not match the same font text created in Word?

font_size_does_not_match_font_text


Last Update: 2/4/2015

When you want to insert an Origin graph in a Word file, you may find that even if you customize the same font size in Origin, it still did not match that font size in Word.

To use the standard font size in Origin graph, you should first activate the graph, click Graph: Convert to Standard Font Size. The font size in graph would be as the same as those in Word. Then export the graph again and insert it back to the Word file.

Please note that this option is only available for those graphs created before 8.5.1 SR0 version and reopened in Later Origin version. For graphs created in 8.5.1 SR0 or other later versions, the font size will be automatically converted to standard font size.
