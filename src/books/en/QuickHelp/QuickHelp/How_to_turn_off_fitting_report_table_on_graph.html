3.153 FAQ-965 How to turn off the fitting report table on graph?

Turn_off_fit_report_table

Last Update: 6/29/2018

To turn off the fitting report table globally for all curve fitting processes,

  1. Open the NLFit dialog newly, go to the Output page under the Settings tab. Set Result Table drop-down list to None.
  2. Click on the arrow to the right of Dialog Theme and select Save as <default> to save this dialog setting as <default>.

Keywords:remove report table, delete report table
