3.93 FAQ-331 What can I do if I have more than 10 columns to be run for ANOVA?

more-than-10-columns-for-anova

Last Update: 2/4/2015


By default in ANOVA tools, In Raw data mode the Number of Levels is constrained to 10.

If you have more than 10 columns to be analyzed, you can preselect the data in worksheet and then open the dialog.

or

Use the system variable, @AML, to change the maximum value in the Number of levels drop list. For example

  1. Open the Command Window or the Script Window from the Windows menu
  2. Type command below and press ENTER
@AML = 15
