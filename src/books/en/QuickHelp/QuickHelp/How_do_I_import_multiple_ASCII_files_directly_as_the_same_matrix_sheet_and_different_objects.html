2.72 FAQ-809 How do I import multiple ASCII files directly as the same matrixsheet and different objects?

import-multiASCII-different-Matrixobj

Last Update: 3/5/2020

A matrix object in a matrixsheet is similar to a column in a worksheet. To import multiple ASCII files directly as the same matrixsheet and different objects, you will need to set Import Mode to Start New Columns in the impASCII dialog.

Keywords:import , ASCII, multiple, same matrix, matrix object
