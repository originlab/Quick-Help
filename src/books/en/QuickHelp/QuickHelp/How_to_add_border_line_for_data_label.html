1.134 FAQ-730 How to add border line for data label?

add-border-for-label

Once you created a plot with labels enabled, such as timeline plot, you might want to add border line around the data labels. You are advised to follow the below steps:

  1. Activate the graph with labels. Select Format: Plot... to open the Plot Details dialog.
  2. Go to Label tab, select Box or Shadow for the Frame drop-down list, and change the border color and fill color with the Border Color button and Fill Color button .
  3. Change the value for Margin(% Font Height) combo box to adjust the top bottom left right margin for the labels.
  4. Click OK button.

Press Ctrl key and double-click on one data point to open the Plot Details dialog. A special point will be added. With this special point selected in left panel, go to Label tab, the four options mentioned above, Frame, Border Color, Fill Color and Margin(% Font Height) will be set to Auto by default. This means these settings of this special point will follow that of whole plot. You can customize these options for your own needs.
