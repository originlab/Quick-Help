1.291 FAQ-1172 Why short dot rectangle is missing from exported PNG graph?

Short-Dot-Rectangle-Missing-In-PNG

Last Update: 4/17/2023

If your graph has rectangle object with border line set as “Short Dot” type, like the below image, when you export the graph as PNG format, the rectangle (or part of the frame) may disappear in the exported PNG image.

It is caused by a API bug when the line length is 1 (short dot) with square end cap


To workaround this problem:

  • Set system variable @LCR = 2. See this FAQ of how to change a system variable value.
