4.57 FAQ-796 Why is my Origin Command Window so Slow to Open?

Origin_Command_Window_Slow_Open

Last Update: 11/5/2015

The Origin Command Window stores previously-issued commands. This command history is written to a text file (LTHistory.txt) in your User Files Folder.

While this command history can be useful for looking up or re-running previously issued script, be aware that it accumulates over time. This can lead to long delays in opening the Command Window. When this happens, the fix is to delete or drastically reduce the size of LTHistory.txt. You can also clear the command history by right-clicking in the History panel (left-panel) of the Command Window and choosing Clear.

If you routinely run large blocks of script from the command line, it is probably best to do so from the Script Window (which does not store a command history).
