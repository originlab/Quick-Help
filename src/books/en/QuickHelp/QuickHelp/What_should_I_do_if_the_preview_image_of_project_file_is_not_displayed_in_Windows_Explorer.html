5.84 FAQ-1161 What should I do if the preview image of project file is not displayed in Windows Explorer?

OPJU-Preview-Image-Not-Display

Last Update: 12/14/2022

Windows Explorer's Large or Extra-large icons views will show a preview image of the last active graph before the project file was saved.

If you donâ€™t see the preview image in Windows Explorer, please first make sure Always show icons, never thumbnails checkbox is unchecked in the Folder Options dialog (by menu File: Options)

If Always show icons, never thumbnails is unchecked, please try to register OPJHandler.dll as follow:

  1. Click to open Windows Start menu
  2. Type cmd in the search box. Command Prompt item will show.
  3. Right click on it and select Run as Administrator.
  4. In the Command window that opens, type 
     regsvr32 "<Origin Program Folder>\OPJHandlers.dll"

    For example, if you installed Origin 2023 in the default path, you can type 

    regsvr32 "C:\Program Files\OriginLab\Origin2023\OPJHandlers.dll"
  5. If a message box pops up saying successful, restart PC.
Notes:

If the preview image is still not shown after trying above solution, please check if an anti-virus software blocks OPJHandler.dll, like Kaspersky.
