2.35 FAQ-471 How to avoid the input data string converting to a numerical value automatically in the worksheet?

Avoid-Data-Numeric

Last Update: 10/13/2016


Sometimes, you type in a string like "001315472" into cell and do not want the string to be recognized as numerical values. You can check on Do not convert text to numeric during data entry in the Properties tab of Column Properties dialog.

Please refer to this section for details about Do not convert text to numeric during data entry

Keywords:data string, numerical value, display

Minimum Origin Version Required: 2015 SR0
