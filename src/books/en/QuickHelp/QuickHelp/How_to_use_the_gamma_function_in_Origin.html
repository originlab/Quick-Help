3.111 FAQ-653 How to use the gamma function in Origin?

use-gamma-function

Last Update: 2/4/2015

Origin provides a built-in gamma function. You may also notice that in the build-in function list other two functions called gammaln and log_gamma, respectively. These two functions represent the natural log of gamma(x). They are useful when running with very large numbers, typically values larger than 163.264 to avoid runoff.
