1.279 FAQ-1143 How to create colorblind-friendly graphs?

Colorblind-Friendly-Colors

Last Update: 4/3/2024

Origin provides colorblind friendly color schemes. The build-in color list and palettes in the table below are colorblind friendly. You can load and apply these lists or palettes from any color-control GUI to make a colorblind friendly graph.

†: The color lists and palettes which are not shown on GUI by default.


To add them to GUI

  1. Choose Tools: Color Manager menu.
  2. Select Color Lists or Palettes for Type as you want. Find the desired list/palette from left panel, click ⍈ button to add it to the right panel.
