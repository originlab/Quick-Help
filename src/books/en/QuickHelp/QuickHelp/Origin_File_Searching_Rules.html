5.75 FAQ-1071 What's the file searching rules in Origin?

OriginFile-Searching-Rules

Last Update: 10/14/2020


By default Origin put build-in files such as template, fdf files in System folder (installation folder). However, if you save a file with the same name in User Files Folder(UFF). It will search and use the file in UFF first.

UFF > System folder

If there is group folder, Origin's searching order is:

UFF > group folder > System folder

Keywords:relative path, full path, absolute path, search order, default path, default folder, user files folder
