3.91 FAQ-329 How to plot a p-p plot with confidence band?

Plot-weibull-P-P-Plot

Last Update: 2/18/2019

To plot a p-p plot with confidence band, you can

  1. Choose menu Plot > Statistical: Probability Plot
  2. In the opened dialog, make sure Confidence Band check box is selected

Or, you can plot a p-p plot with confidence interval with the Distribution Fit tool (Statistics: Descriptive Statistics: Distribution Fit)


Following 5 distributions are available in either way

  • Normal
  • LogNormal
  • Weibull
  • Exponential
  • Gamma
