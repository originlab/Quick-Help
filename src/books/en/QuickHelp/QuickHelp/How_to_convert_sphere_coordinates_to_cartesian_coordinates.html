3.58 FAQ-293 How to convert sphere coordinates to cartesian coordinates?

Sphere-Coordinates-to-Cartesian-Coordinates

Last Update: 2/4/2015

Origin provides an X-Function sph2cart to convert Sphere coordinate to Cartesian coordinate. Sphere coordinate data can be in three columns, a virtual matrix or three matrix objects.

  1. Open the Command Window or the Script Window from the Windows menu
  2. Type command below and press ENTER
sph2cart -d


Keywords:sphere coordinates, cartesian coordinates
