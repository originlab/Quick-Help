1.75 FAQ-204 How to reverse the order of curves in a waterfall plot?

Reverse-Order-in-Waterfall

Last Update: 3/17/2024


Origin has both "2D" and "3D" waterfall templates and the differences between them sometimes cause confusion.


For 3D waterfall plots
Plot > 3D: 3D Waterfall, Y Color Mapped 3D Waterfall , Z Color Mapped 3D Waterfall

  1. Double-click the axis of the scale that you want to reverse. This opens the Axis dialog box.
  2. Click the Scale tab, then clear or check the Reverse box.


For 2D waterfall plots
Plot > 3D: Waterfall, Y Color Mapped Waterfall, Z Color Mapped Waterfall

  1. Double-click the axis of the scale that you want to reverse. This opens the Axis dialog box.
  2. Exchange the From and To values and change the Increment value to a negative value (e.g. 2 becomes -2).
