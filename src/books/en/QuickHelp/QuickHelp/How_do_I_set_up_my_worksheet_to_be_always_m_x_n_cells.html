2.42 FAQ-478 How do I set up my worksheet to be always m x n cells?

Resize-Wks

Last Update: 10/2/2015

Open the Worksheet Properties dialog by selecting Format: Worksheet or right-click inside the worksheet, to the right of the last column in the sheet, and choose Properties or press F4. On the Size tab, set the Row Number and Column Number. On the Miscellaneous tab, clear the Auto Add Rows box and enable Fit Sheet to Window. Click OK to close the dialog, then resize the workbook window.
