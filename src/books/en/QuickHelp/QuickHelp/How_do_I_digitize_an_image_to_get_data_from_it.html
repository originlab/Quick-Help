2.12 FAQ-421 How do I digitize an image to get data from it?

Digitize-Image

Last Update: 1/12/2023

You can use the Digitizer Tool which is accessible from the Tools: Digitizer menu, or from the Digitize Image button in the Standard toolbar. It supports auto-tracing and digitizing in 2D Cartesian, polar and ternary coordinate systems.

See the Enhanced Digitizer video tutorial.

Digitize an External Image File

Click the Digitize Image button and pick the image file or first choose Tools: Digitizer... menu and then click Import button to pick the image file. Then start digitizing.

Digitize an Image on Clipboard

First choose Tools: Digitizer... menu and then click Import from Clipboard button to pick image on clipboard. Then start digitizing.

Digitize an Image in Image Window

There is no direct way to digitize image in Image window. So please refer to Digitize an External Image File. Or Ctrl+C to copy the image from Image window, then refer to Digitize an Image on Clipboard.
