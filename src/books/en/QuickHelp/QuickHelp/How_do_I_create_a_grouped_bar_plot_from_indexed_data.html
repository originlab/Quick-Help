1.60 FAQ-181 How do I create a grouped bar plot from indexed data?

grouped-bar-plot-indexed

Last Update: 2/4/2015


In order to create a grouped bar plot from indexed data, you can first create a grouped column plot from indexed data and then choose Graph: Exchange X-Y Axis.

Keywords:grouped, bar, indexed, data

Minimum Origin Version Required: 9.1 SR0
