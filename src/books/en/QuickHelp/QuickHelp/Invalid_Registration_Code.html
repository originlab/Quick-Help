6.68 FAQ-910 I've entered the Registration Code provided by OriginLab but the webpage tells me that this code is invalid.

Invalid-Registration-Code

Last Update: 12/1/2022

Please note that the Registration ID (9 characters) is not the same as the Registration Code (16 characters). Please ensure you are entering the correct information.

Registration Code is sent by email and is used to register your Serial Number to your account.

This problem may occur with both copied/pasted Registration Codes or with direct links to the OriginLab website.
Please make sure that the registration code is sixteen (16) characters long. If it is not, please contact us.
