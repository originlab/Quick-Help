3.79 FAQ-317 How do I get data to draw on top of the box/bar + data charts?

Draw-Data-on-Top-of-Box

Last Update: 3/16/2020


For box chart variants Box + Data Overlap and Bar + Data Overlap, you can force data to be drawn on top of the box or bar:

  1. Double-click on the plot to open Plot Details.
  2. Click on the Box tab and check the Draw Data on Top of Box/Bar check box.


  • For more information, see Data and Whisker Drawing Options.
  • For information on creating box charts and variants (bar, violin, column scatter, etc.), see Creating Box Charts.
