2.16 FAQ-427 How to combine data from multiple Origin projects?

Import-data-from-Origin-OPJ

Last Update: 5/17/2023

Depending upon your needs, there are several ways to combine data from separate Origin files.

Copying and Pasting Project Folders Between Origin Sessions

  1. Run two Origin sessions, one with one project and another with a second project.
  2. In the first session, right-click on a Project Explorer folder and choose Copy from context menu.
  3. Go to the second Origin session, right click on a folder and choose Paste from the context menu. The folder including all contained windows, will be added as a subfolder in the second session's current project (Hint: Don't forget to save the current project to preserve changes).

Note: We use the words "session" and "instance" to mean the same thing: The user runs initiates the Origin program twice so as to be able to transfer data between two simultaneously opened projects.

Once you have pasted a project folder into a second project, you could create a new folder specifically to combine only the windows that you want to work with in the current session. See Project Explorer.

Copying and Pasting Origin Workbook Windows Within or Between Origin Sessions

  1. Open Project Explorer (PE) and in the bottom panel, right-click on the desired workbook and choose Copy.
  2. Highlight another folder in the same project (within session) or highlight a folder in another project (between sessions) and in a blank space in the bottom panel of PE, right-click and choose Paste.

Importing Data from another Project or Worksheet Window

This allows import of selected worksheet data from another project.

  1. With a workbook active, choose Data: Connect to File: Origin File to open Origin Connector.
  2. Browse to your project file (.OPJ/.OPJU). All sheets in the project will be listed as branch structure on the left panel. If you then select a particular sheet, you get a list of columns of this sheet on the right panel.
  3. Double click the selected sheet to add it to the buttom panel and click OK to import.
  4. After import, you can always import other sheets by Data Navigator.

There is another X-Function impOPJ to import selected columns of worksheet data from another project.
For help with the impOPJ dialog box, see Importing an Origin Project.

Append All Data in Another Project

This option doesn't allow you to selectively bring in data. Rather, you will append the entire contents of another OPJ file, to the current (open) OPJ file.

  1. Choose File: Append or right-click on a Project Explorer folder and choose Append Project. A Reminder Message prompts you to create a folder for the appended file or append the file to the current folder. In either case, the subfolder structure of the appended file is preserved.

Opening an Origin Window File

This is a way to bring data saved as a standalone worksheet (OGW) or graph window (OGG), into the current OPJ file.

  1. Choose File: Open (or File: Recent Books or Recent Graphs) and browse to your Origin Window file. You can also drag and drop window files.

Note that when you open a graph window file in this way, you bring in supporting data, as well. Right-click on your data plot and choose Create worksheet worksheetName; or double-click on the plot and at the bottom of Plot Details, click the Workbook button.
