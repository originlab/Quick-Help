5.77 FAQ-1087 What should I do if my GUI customization is lost EACH TIME on start up

Toolbar-WindowSettings-not-Save

Last Update: 4/22/2021

Origin will save the window and interface settings you used last time. If you see the following problems:

  • Toolbar "disappeares" on each time Origin startup
  • Origin doesn't remember my window settings of last used
  • Settings made in the Project Explorer are lost each time Origin starts

Please try to remove the registry key and restart Origin:

  1. Press Windows + R to open Run dialog of Windows
  2. Enter regedit and click OK to open Registry Editor
  3. Go to the folder HKEY_CURRENT_USER\Software\OriginLab\Origin 9.8\International\ToolBars\
    Change Origin version# if needed.
  4. Right-click the folder in the left panel and select Delete
  5. Restart Origin


Notes:

Please note that the solution above is for GUI customizations lost EACH TIME. If you only want to restore the default GUI setting, please try steps as below

  1. Select from menu View: Toolbars
  2. On the Toolbar tab, click Reinitialize button
