6.48 FAQ-787 When I try to activate Origin online I see Enter License, but nothing happens

Default-Web-Browser

Last Update: 1/3/2019

Either you do not have a default web browser set, or your File Association for opening your default web browser is not properly set on your computer.

To set the default web browser:

Select from the below web browsers the one which you would like to use as your default web browser (these are the most common) and follow the instructions to set it to be the default:

  • Microsoft Internet Explorer: Open Internet Options either from Control Panel or using the Preference| Options menu in Internet Explorer, then on the Programs tab click on "Make Default" and try to activate Origin again.
  • Mozilla FireFox: From the menu select Options | Options. Click on Advanced (tab) and on the General Tab under System Defaults check off "Always check to see if FireFox is the default web browser on startup" and try to activate Origin again.
  • Google Chrome: Click on the three horizontal bars in the top right of Chrome to open the menu and select Settings. Scroll to the bottom. Under Default browser click on "Make Google Chrome my default browser" and try to activate Origin again.
To set File Association:

If after making sure the default web browser is set and still have a problem, then the File Association must be corrected for your default web browser to open.

Windows Vista, 7, 8, 8.1 and 10

  1. Open Control Panel and click on Default Programs.
  2. Click on Associate a file type or protocol with a program.
  3. Scroll down to .htm and .html. Make sure the Current Default column is set to your default web browser.
  4. If not, highlight and click on Change Program and select your default web browser for both .htm and .html. Then scroll to the bottom where you will see the Protocols section. Make sure FTP, HTTP and HTTPS are all set to your default web browser.
  5. Try to activate Origin again.
