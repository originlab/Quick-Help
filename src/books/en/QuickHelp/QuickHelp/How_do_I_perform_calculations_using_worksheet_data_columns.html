3.57 FAQ-292 How do I perform calculations using worksheet data columns?

Math-Between-Cols

Last Update: 2/4/2015

Use the Set Values dialog. To open the dialog, select Column:Set Column Values... or highlight a column and right click to select Set Column Values... from the context menu.

This dialog has sample formulas as well as a rich avenue of built-in math functions. You can click on the Search and Insert Functions button to bring up Search and Insert Functions dialog and input keywords to easily search for a desired function.

Each column in your worksheet can have a formula associated with it. To view the formulas of other columns, click on the and buttons to scroll through the columns.


Keywords:dataset, vector, math, simple
