3.160 FAQ-1036 How to go back to the source sheet or graph?

Go-to-source


Last Update: 3/16/2020

Sometimes, you had already got the results in the result sheets and want to check the source data of the current result, you can

  • If the operation starts from a sheet, click on the green lock icon to select Go to Source to go back to the source sheet.
  • If the operation starts from a graph, click on the green lock icon to select Go to Graph to go back to the source graph window.

Keywords:go to source, source worksheet, source data, result sheet
