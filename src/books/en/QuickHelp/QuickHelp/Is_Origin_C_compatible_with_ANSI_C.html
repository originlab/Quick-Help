4.35 FAQ-395 Is Origin C compatible with ANSI C?

ANSI-C-Compatibility

Last Update: 2/4/2015

Yes, with minor exceptions Origin C supports a nearly complete ANSI C language syntax including support of most ANSI C elements (program structure, declarations and types, expressions and assignments, statements, functions, macros, enums, typedefs, and syntax). Please refer to Origin C Help for more information.
