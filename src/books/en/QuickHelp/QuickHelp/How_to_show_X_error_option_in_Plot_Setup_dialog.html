1.157 FAQ-813 How to show X Error option in Plot Setup dialog?

Show-XErr-in-PS

Last Update: 1/5/2018

The Plot Setup dialog shows options of X, Y and YEr plot designation in the middle panel by default. To show X Error option, you can right click inside the middle panel table and select X Error Bars from the fly-out menu. You can also turn on yEr+ and yEr- options by checking Y Error Bars Plus/Minus from the fly-out menu.
