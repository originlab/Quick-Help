Method 1: Ctrl + Zoom to select region on graph and then copy layer

  1. With the graph window activate, click the Scale In button (on Tools toolbar docked at the left-hand side of the workspace). Press Ctrl key and drag out a rectangle area on the graph. It will create an enlarged graph. Moving the rectangle box in the source graph will update the enlarged graph dynamically.
  2. On the enlarged graph, right click on a blank place inside the layer and select Copy. Go to the source graph and click Ctrl + V to paste it.

Method 2: Zoom Inset App

You can download Zoom Inset App from File Exchange page.

Method 3: Zoom plot type from Plot menu

Origin provides a build-in zoom graph template. You can access it by selecting menu Plot: Multi-Panel/Axis.
