5.39 FAQ-740 Why can't I use right button when runing virtualized Win 7 on Mac?

Fail-to-Use-Right-Button-in-Virtualized-Win-7

Last Update: 6/16/2015

There are some circumstance that you might fail to use the right button when runing virtualized Win 7 on Mac. To troubleshoot the problem, you can run Origin as Administrator by typing OriginPro 2015 (or any other corresponding versions) in the Start Menu search box of Win 7 and pressing Ctrl + Shift + Enter.
