1.61 FAQ-182 How do I create a cumulative histogram plot?

cumulative-histogram

Last Update: 10/15/2018


Prior to creating the cumulative histogram, use the Frequency Counts tool (from menu Statistics: Descriptive Statistics: Frequency Counts) to compute the Cumulative Count.

Then go to the FreqCounts sheet, use the data in column Cumulative Count to plot a column graph, a cumulative histogram is created.

Keywords:cumulative, histogram, cumulative count
