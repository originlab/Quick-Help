6.49 FAQ-788 How do I transfer Origin to another computer?

System-Transfer

Last Update: 2/2/2023


The Origin license must first be deactivated from the original computer and then the license can be reactivated on the new computer by using the same Product Key.

This page is for Node-Locked licenses. If you have a Concurrent Network License (xFxxx-xxxx-79xxxxx), please see: FAQ-39 How can I change my license server to a new machine?

Do you have access to the computer where Origin is installed?

Yes, I have access to the computer

  1. Launch Origin
  2. Use the menu: Help->Deactivate License...

No, I don't have access to the computer

Single-User Node-Locked

If you are using a Single-User Serial Number (xFxxx-xxxx-70/71/72/73xxxxx)...

  1. Open a web browser to http://www.originlab.com/reginfo and log in
  2. Click on the Serial Number
  3. You will see your Product Key and a "Go" link under the Deactivation column.
    If you have valid Maintenance, click on that link to deactivate.
    If your Maintenance has expired, clicking the "Go" will redirect you to the System Transfer Request form. Please fill out the form, read the licensing and installation support policy, and submit

Multi-User Node-Locked

If you are using a group Serial Number (xFxxx-xxxx-76/78xxxxx)...

  • If you are not the administrator, please contact your organization's Origin license administrator for help with the transfer
  • If you are the administrator:
  1. Open a web browser to http://www.originlab.com/reginfo and log in
  2. Click on the Serial Number
  3. On the page that opens, click on the “you can see all product keys here” link
  4. You will see your list of Product Keys and a "Go" link under the Deactivation column for each activated key.
    You can apply filters to this list to find a particular user or Product Key.
  5. If you have valid Maintenance, click on the "Go" link to deactivate.
    If your Maintenance has expired, clicking the "Go" will redirect you to the System Transfer Request form. Please fill out the form, read the licensing and installation support policy, and submit
