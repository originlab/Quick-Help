1.83 FAQ-213 How to use the custom colors defined in old Origin version?

share-custom-color

Last Update: 1/7/2019

If you defined custom color(s) in an old Origin version and would like to use it in the new Origin version on the same machine, you can do the following:

  1. Select the Preferences: Transfer User Files to open the Transfer User Files dialog.
  2. Set the Source Location as the User Files Folder of the old version and check the Settings(Menus,Buttons...) box in the View section.
  3. Select the CustomColors.ini file in the left panel, click the button to add it to the right panel.
  4. Click the Transfer button to transfer the CustomColor.ini file to the User Files Folder of the current Origin installation.


You can also open the User Files Folder, copy the CustomColors.ini file and paste it to the User Files Folder of another Origin installation to share the custom color settings.

Minimum Origin Version Required: 9.0 SR0
