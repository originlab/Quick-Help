1.158 FAQ-814 When I create a PDF from MS Word, the data extend beyond the axes of my Origin graph.

Data_Outside_Graph_Axes_Using_MS_Word_PDFMaker

Last Update: 6/4/2021

If you ...

  1. Narrow the axis scale on your Origin graph, either by changing the From and To values on the Scale tab of the Axis dialog or by using the Scale In tool on the Tools toolbar,
  2. Do a CTRL+J to (Edit: Copy Page) to copy the graph.
  3. In MS Word, choose Paste Special and paste as Picture (Windows Metafile or Enhanced Metafile).
  4. Create a PDF from this document using the MS Word PDFMaker add-in ...


... you will likely see this:

To fix this problem:

  1. Select menu Preferences: System Variables.
  2. Set the value of system variable @GDIP to 0
  3. Now do a CTRL+J in Origin, paste your WMF or EMF file to Word and recreate the PDF using PDFMaker.
