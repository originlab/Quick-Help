3.41 FAQ-272 How do I automate repetitive tasks?

Automate-Repetitive-tasks

Last Update: 12/4/2018

  • If you import one data and did analysis on it and everything is inclusive in the workbook, with the data sheet active, you can choose Clone Import to repeat same analysis on more data files.
    • Please follow Import Multiple ASCII Files into Cloned Workbooks
    • Or this video: Import Multiple Data into Cloned Book or Sheets for Batch Processing
  • You can also first save your analysis as analysis template and then perform batch processing on multiple data files or datasets using the template. This one gives you the option to get summary sheet, or send report to Word or PDF files
    • Please follow these tutorials: Batch Processing
    • Or this video: Batch Processing
    • Or this video: Creating Batch Reports using Origin Analysis Template and Microsoft Word
  • For fitting, if you have done a fitting on a plot, you can copy and paste an operation for repeating the same analysis/operation on other selected curves or graph windows.
  • For some analysis tools, especially those that don't support multiple input, you can repeat the operation/analysis of the current Y column/plot to all Y columns or all plots.
  • To perform batch peak analysis, you can customize your peak analysis settings in the Peak Analyzer tool and save it as a theme. Then load it in the Batch Peak Analysis using Theme dialog to perform batch processing on multiple datasets.
  • For Gadgets, you can batch analysis of multiple curves in a graph. After determining region of interest and customizing settings in gadgets, choose New Output for All Curves or New Output for All Layers to do the analysis on all curves and output the results.
