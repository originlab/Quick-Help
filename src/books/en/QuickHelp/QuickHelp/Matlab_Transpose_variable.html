2.24 FAQ-438 How can I transpose variable when importing data from MATLAB?

Matlab-Transpose-Variable

Last Update: 10/2/2015

When importing data from MATLAB into Origin worksheets or matrices, you might want to transpose the variables. This feature enables you to do transpose operation to all or part of the variables in the imported files.

To start with:

  1. Create a new workbook (matrix) by clicking File: New:Worksheet (or File: New:Matrix).
  2. Click Import: Matlab(Mat), choose a MATLAB file then add it to the bottom panel. Click OK open the dialog.
  3. Under Import Option section, select All Variables or 1*N dimension Variables in the Transpose Variables drop-down list. Click OK to close the dialog.


Keywords:MATLAB, data import, transpose

Minimum Origin Version Required: 2015 SR0
