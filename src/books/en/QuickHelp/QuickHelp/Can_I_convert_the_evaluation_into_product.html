6.6 FAQ-7 Can I convert the trial into my purchased product?

DemoLic-to-Product

Last Update: 5/16/2019

Yes - 3 ways to convert the Trial to a Product (requires serial number):

Contents

  • 1 If you have a Node-Locked serial number
    • 1.1 Method 1 - if you don't get an evaluation license yet
    • 1.2 Method 2
  • 2 If you have a Concurrent/Dongle serial number

If you have a Node-Locked serial number

Method 1 - if you don't get an evaluation license yet
  1. In the Evaluation License Options dialog that opens when you start the evaluation, choose the last option Activate for this computer and click OK.
  2. In the License Activation dialog, enter the product key to convert evaluation to product and activate the product.
Method 2
  1. Select Help: Convert Evaluation to Product from menu.
  2. Enter the product key in the License Activation dialog that opens. This will convert evaluation to product and activate the product.

If you have a Concurrent/Dongle serial number

  1. Close Origin(Pro) if it is open.
  2. If you are using Windows 7 and Vista,
    • Click the Windows Start button and select Programs | OriginLab Origin(Pro) | Repair or Uninstall Origin(Pro). This starts the InstallShield wizard.
    If you are using Windows 8,
    • Press the Windows key on your keyboard.
    • Right-click in a blank area of the desktop.
    • Click the All Apps button at the bottom right corner of the desktop.
    • Select Repair or Uninstall Origin(Pro) that displays on the desktop. This starts the InstallShield wizard.
    If you are using Windows 10,
    • Click the Windows Start button and select All apps | OriginLab Origin(Pro) | Repair or Uninstall Origin(Pro). This starts the InstallShield wizard.
  3. Select the Modify/Change option and click Next.
  4. Click "Yes" when asked if you would like to change your serial number.
  5. Enter your serial number and complete the Modify/Change process, clicking Finish when done.
