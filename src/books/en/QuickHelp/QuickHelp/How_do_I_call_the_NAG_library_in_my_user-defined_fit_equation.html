3.32 FAQ-261 How do I call the NAG library in my user-defined fit equation?

NAGLibrary-in-FitttingFunction

Last Update: 2/4/2015

New functions are defined by selecting Tools: Fitting Function Organizer or Tools: Fitting Function Builder and clicking the New Function button. You can define and edit a function in Code Builder when Function Form = Origin C. When you click the button, to the right of the Function edit box, a special instance of the Code Builder workspace will open. Edit directly in this window, scrolling up to the very top, and adding #include directives for any NAG header files that you need.
