1.218 FAQ-963 How to set the axis ticks to show unevenly spaced?

Unevenly-spaced-ticks

Last Update: 6/28/2018

With the graph window active, double-click on the axis you want to customize to open the Axis dialog. On the Scale tab, set Type of Major ticks and Minor Ticks as By Custom Positions and select a column to use its values as tick position, or directly enter a set of space-separted position values.

Furthermore, you can specify a dataset to display as tick labels instead of the actual tick position values.

Keywords:uneven, tick position, customize ticks
