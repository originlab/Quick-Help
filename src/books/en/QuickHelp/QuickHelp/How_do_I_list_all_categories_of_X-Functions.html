4.39 FAQ-400 How do I list all categories of X-Functions?

List-XF-Categories

Last Update: 2/4/2015

Use the lc X-Function to list all X-Function categories, or all X-Functions in a specified Category. In the Command Window, opened by selecting Window: Command Window or alternatively, Alt+3, type lc and then press Enter. All categories are displayed in the Command Window. This list matches what you see in the Help file, opened by selecting Help: X-Functions.


Keywords:command, xf
