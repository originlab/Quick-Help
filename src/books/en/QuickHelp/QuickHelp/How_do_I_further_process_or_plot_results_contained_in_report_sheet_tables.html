2.53 FAQ-495 How do I further process or plot results contained in report sheet tables?

Copy-ResultTable-as-NewWks

Last Update: 1/4/2018

To do this, go to your analysis report sheet, which contains multiple report tables, and right-click on any report table heading i.e., Parameters, Statistics, Summary, etc. and choose Create Copy as New Sheet or Create Transposed Copy as New Sheet from the context menu. You will then have a new sheet of data to further manipulate. The value in new sheet are linked to original table still.
