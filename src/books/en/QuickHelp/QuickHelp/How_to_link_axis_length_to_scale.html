1.1 FAQ-111 How to set equal axis length?

axis-isometric

Last Update: 1/8/2019


By default, axis length in an Origin graph is linked to the layer dimensions, so the same scale in different axes might not be displayed in identical length.

In order to set equal axis length, you could link axis length to axis scale range. This makes it possible to:

  • Keep 1:1(or identical) scale for x y and/or z axis
  • Make equal(or same) x y and/or z scale
  • Keep aspect ratio of the scale and axis length
  • Make a squared image with the same x y scale
  • Have the axes proportional to length

To link axis length to axis scale range:

  • For 2D Graphs
    Click Format:Layer to open the Plot Details dialog and go to the Size tab and check the Link Axis Length to Scale with X:Y Ratio box and leave the edit box set at the default value of 1.

  • For 3D Graphs
    Click Format:Layer Properties to open the Plot Details dialog and go to the Axis tab to select a desired option from the Link Axis Length to Scale(isometric) drop-down list.
