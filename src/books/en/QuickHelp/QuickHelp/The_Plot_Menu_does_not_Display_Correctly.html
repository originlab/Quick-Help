5.51 FAQ-853 The Plot Menu does not Display Correctly

The-Plot-Menu-does-not-Display-Correctly

Last Update: 7/14/2018

There have been a few reports of problems displaying the Plot popup menu (the icon-based menu) that was added in Origin 2017. Problems might be related to a recent Windows 10 Fall Creators Update, 2017 (Version 1709). Check and make sure your Windows 10 is not at this version.

Microsoft has subsequently released updates to the Windows 10 Fall Creators Update and applying these may resolve slowness problems.

  • Go to Windows Settings > Update & Security > Check for updates and apply all available updates.


Or you can choose to rollback to old plot menu:

  • Open the Script window (Window: Script Window), type the following, then press Enter:
@DPPM = 1

If this resolves the problem, you can permanently disable the Plot popup menu via the Set System Variables dialog box.


See Also: LabTalk System Variable @DPPM
