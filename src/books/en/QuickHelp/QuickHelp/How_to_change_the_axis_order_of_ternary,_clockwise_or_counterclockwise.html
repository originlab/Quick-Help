1.7 FAQ-117 How to change the axis order of ternary, clockwise or counterclockwise?

Last Update: 1/8/2019

Change-Axis-Order-of-Ternary

Select Format:Layer to open the Plot Details dialog. Go to the Ternary tab, select Counter Clocwise or Clockwise to change the axis order.

Keywords:Counter Clocwise, Clockwise, Ternary

Minimum Origin Version Required: 9.0SR0
