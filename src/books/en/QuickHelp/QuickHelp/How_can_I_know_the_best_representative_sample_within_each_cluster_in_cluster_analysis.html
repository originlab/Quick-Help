3.80 FAQ-318 How can I know the best representative sample within each cluster in cluster analysis?

best-representative-sample

Last Update: 2/4/2015


In OriginPro, the best representative sample will be output in the Clustroid Info table of result sheet of Hierarchical Cluster Analysis (Statistics: Multivariate Analysis: Hierarchical Cluster Analysis menu).

The observation/variable which is the most close to and most far away from the the cluster center will be output in the table.

Keywords:hierarchical cluster, clustroid, representative, cluster analysis

Minimum Origin Version Required: 2015 SR0
