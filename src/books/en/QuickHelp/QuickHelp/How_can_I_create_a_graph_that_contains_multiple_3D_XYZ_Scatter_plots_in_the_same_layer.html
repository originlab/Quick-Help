1.124 FAQ-702 How can I create a graph that contains multiple 3D XYZ Scatter plots in the same layer?

multiple-3D-XYZ

Last Update: 2/19/2019

You can construct the datasets in worksheet with designation XYZXYZ... or XYZZ..., then highlight the worksheet, choose Plot > 3D : 3D Scatter menu to plot the multiple 3D XYZ Scatter plots in the same layer.
