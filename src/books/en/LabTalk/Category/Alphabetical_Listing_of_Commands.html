3.3.2 Alphabetical Listing of Commands

Alphabetical-Listing-of-Commands

A complete listing of all LabTalk commands is provided in subsequent Help topics.

Two general points to remember:

  • While the complete command name is provided for each command, command names can be abbreviated down to two letters as long as the abbreviation is unique. For example, instead of using the print command in your script, you could use pr.
  • LabTalk commands are case-insensitive.
