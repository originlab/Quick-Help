Syntax

wks.DC.Source$

Access

Read/write
string

Description

Relative path of data source.

Examples

//This example shows how to import a CSV file by Data Connector
newbook;
wbook.dc.add("CSV");
wks.dc.source$=System.path.program$+"Samples\Batch Processing\T275K.csv";
wks.dc.import();
