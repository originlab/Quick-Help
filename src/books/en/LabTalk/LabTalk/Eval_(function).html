Description

Execute LabTalk script code.

Syntax

int eval( string strLT$ )

Parameters

strLT

the LabTalk script code to execute.

Return

1 if success, 0 if LabTalk execution leads to error

Example

eval("Worksheet -s 2;"); // Select B(Y) column
