3.6.5.7 BeforePointProc

BeforePointProc-AfterPointProc-macro

Two macros BeforePointProc and AfterPointProc can be used for control the plot styles when using Draw Data Tool. BeforePointProc will be called before PointProc macro, and AfterPointProc will be called after PointProc macro. Please refer to the example for how to set the connecting line style for Draw Data Tool by using the two macros.

Minimum Origin Version Required: 2015 SR0
