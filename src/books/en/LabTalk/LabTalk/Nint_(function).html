Description

This function returns the nearest integer of x. It functions identically as round(x, 0).

Syntax

int nint( double x)

Parameter

x

is any numeric you want to get its nearest integer.

Return

returns the nearest integer of x.

Example

aa = nint(0.5);
aa = ; //should return 1
bb = nint(-0.5);
bb = ; //should return -1
