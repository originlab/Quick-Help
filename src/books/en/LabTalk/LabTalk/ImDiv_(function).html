Description

Calculate the complex division.

Syntax

complex ImDiv(complex cY1, complex cY2)

Parameters

cY1

The dividend.

cY2

The divisor.

Return

Return the result of complex division.

Example

imdiv(2+2i, 2) = ;  // 1+1i

complex c1 = 3+4i;
complex c2 = 5+6i;
complex c3 = imdiv(c1, c2);
c3 = ;  // 0.63934426229508+0.032786885245902i
