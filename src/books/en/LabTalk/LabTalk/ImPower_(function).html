Description

Calculate the given complex to the power of the specified value.

Syntax

complex ImPower(complex cX, double dn)

Parameters

cX

The given complex number.

dn

The power number.

Return

Return the power of the specified complex.

Example

imPower(3+4i, 0.58) = ;  // 2.1842664533957+1.3028830443443i
 
complex c1 = 1+1i;
complex c2;
c2 = imPower(c1, -2);
c2 = ;  // 3.0616169978684E-17-0.5i
