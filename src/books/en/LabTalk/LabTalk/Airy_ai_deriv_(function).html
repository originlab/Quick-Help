3.5.1.3.2 airy_ai_deriv

Airy-ai-deriv-func

Definition:

evaluates an approximation to the derivative of the Airy function Ai(x). It is based on a number of Chebyshev expansions.

For more information please review the s17ajc function in the NAG document.


Parameters:

x (input, double)
The argument x of the function.
(x) (output, double)
Approximation of the Airy function
