3.3.2.8 Dataset

Dataset-cmd

Create a dataset


Contents

  • 1 Syntax:
  • 2 Options:
    • 2.1 no option; Declare a numeric(double) dataset
    • 2.2 -a; Declare a Text & Numeric dataset
    • 2.3 -t; Declare a Text dataset
  • 3 See Also:

Syntax:

dataset [option] {...}

Options:

no option; Declare a numeric(double) dataset

Syntax: dataset {...} 

//Declare dataset with only numeric(double) values
dataset ds1={1,2,"A","B","A"};
col(A) = ds1; ///only 1, 2 will be set in column A

-a; Declare a Text & Numeric dataset

Syntax: dataset -a {...} 

// create a Text & Numeric dataset
dataset -a ds1={1,2,"A","B","A"};
col(A) = ds1; ///1, 2, A, B, A will be set in column A

-t; Declare a Text dataset

Syntax: dataset -t {...} 

//Create a dataset with only text values
dataset -t ds3={1,2,"A","B","A"};
col(A) = ds3;
