3.3.2.29 Help

Help-cmd


The help command gives the information of x-functions, or reports the computer id where the Origin software is running.

Contents

  • 1 Syntax:
  • 2 Options:
    • 2.1 no option;Displays the help message for the specified x-function
    • 2.2 -c;Return the computer id to the Command window

Syntax:

help  [option] [x-function]

Options:

no option;Displays the help message for the specified x-function

Syntax:x-function

This notation displays the help message for the specified x-function such as: 

 help impASC;

The summarized help message can be obtained by the -h option of the x-function such as: 

 impASC -h;

-c;Return the computer id to the Command window

Syntax: help -c

This option is used to return the computer id to the Command window. The computer id must be known to the user before retrieving a FLEXlm license file from the OriginLab website.
