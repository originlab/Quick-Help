Description

This function is used to convert LT color to OColor.

Syntax

int LT_to_oc( int nLTcolor, bool bCvtAutoColor = true )

Parameters

nLTcolor

the COLOR value to be converted.

bCvtAutoColor

true to convert auto-color

Return

Return OC OCOLOR

Example

layer.cmap.ColorMiss = LT_to_oc( wks.nancolor );
