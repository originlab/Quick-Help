3.6.5.8 PointProc

PointProc-macro

The Data Reader tool or the Screen Reader tool must first be selected before the PointProc macro can be triggered. These tools can be selected by

  • clicking on the Data Reader or Screen Reader buttons (on the Tools toolbar).
  • executing the getpts [options] npts (message)] or dotool tool# commands.

Note that the getpts command defines the PointProc macro for use when it is executed. Consequently, if you use the getpts command and you want to define your own PointProc macro, you must define your version of the PointProc macro after you issue the getpts command.

After the Data Reader tool or the Screen Reader tool is selected, the PointProc macro is triggered whenever you (a) double-click on or (b) select with the left or right ARROW key and press ENTER

  • on a data point of the active dataset (for the Data Reader tool).
  • on any point in the active graph window (for the Screen Reader tool).
