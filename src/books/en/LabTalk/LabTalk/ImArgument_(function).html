Description

Get the argument (theta) of a complex.

Syntax

double ImArgument(complex cX)

Parameters

cX

The complex number.

Return

Return the argument (theta) of a complex.

Example

imargument(1+1i) = ;  // 0.78539816339745 = pi/4

complex c = 0-5i;
imargument(c) = ;  // -1.5707963267949 = -pi/2
