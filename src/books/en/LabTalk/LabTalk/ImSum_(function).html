Description

Get sum of two specified complexes.

Syntax

complex ImSum(complex cY1, complex cY2)

Parameters

cY1

The first addend.

cY2

The second addend.

Return

Return the sum of two specified complexes.

Example

imSum(3+4i, 1+2i) = ;  // 4+6i
 
complex c1 = 1+1i;
complex c2 = 4+1i;
complex c3;
c3 = imSum(c1, c2);
c3 = ;  // 5+2i
