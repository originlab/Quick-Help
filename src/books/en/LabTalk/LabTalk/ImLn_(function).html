Description

Calculate the natural logarithm of the complex.

where ImAbs computes the modulus of the complex, atan2 returns the angle, and .

Syntax

complex ImLn(complex cX)

Parameters

cX

The complex number.

Return

Return the natural logarithm of the specified complex.

Example

imLn(3+4i) = ;  // 1.6094379124341+0.92729521800161i
 
complex c1 = 1+1i;
complex c2;
c2 = imLn(c1);
c2 = ;  // 0.34657359027997+0.78539816339745i
