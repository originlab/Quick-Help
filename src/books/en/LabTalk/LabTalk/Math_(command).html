Syntax:

math [option] dataset1 dataset2

Options:

-o option; Open the Math on/between Dataset dialog box

Syntax: math -o dataset1 dataset2 [dataset3]

Open the Math on/between Dataset dialog box, with dataset1 selected as the Y1 dataset and dataset2 selected as the Y dataset.If dataset3 is specified, it is selected as Y2. Otherwise, the Y2 dataset is blank.

-o; Select an operator to display in the 'operator' text box of the Math on/between Dataset dialog box

Syntax: math -o operator [dataset1 dataset2 dataset3]

Select an operator to display in the 'operator' text box of the Math on/between Dataset dialog box.The possible operators are: +, -, *, /, and ^. Dataset1 is selected as the Y1 dataset and dataset2 is selected as the Y dataset. If dataset3 is specified, it is selected as Y2. Otherwise, the Y2 dataset is blank.
