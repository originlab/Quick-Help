Description

This function provides COM support for ActiveX controls that are placed into Origin windows or Dialog Builder windows.

Syntax

_VARIANT GetActiveXControl( )

Parameters

Return

Examples

EX1

Download this zip file, extract it. 

#include <Dialog.h>
#define IDC_CWKNOB1 1001
// Please reference to "\Dialog Builder\DotNet" or 
// "\Dialog Builder\ActiveXNI" (For Labview)

void Window_GetActiveXControl_ex1()
{   
	Dialog MyDlg;
	Control ctrl = MyDlg.GetItem(IDC_CWKNOB1);
	Object CWKnob1 = ctrl.GetActiveXControl();	
}
