Description

get a thumbnails's show state in a matrix page.

Syntax

BOOL IsShowImageThumbnail( )

Parameters

Return

TRUE if ImageThumbnail shows, otherwise FALSE

Examples

EX1 

void IsShowImageThumbnail_Ex1()
{
    MatrixPage mp;
    
    mp.Create("origin");
    if ( !mp )
        return;
    mp.ShowImageThumbnails(TRUE);//set ImageThumbnails show,it is hide default.
    if(mp.IsShowImageThumbnail())
        out_str("MatrixPage show ImageThumbnail");
    else
        out_str("MatrixPage hide ImageThumbnail");
}
