Description

Update GetN tree's range string, and other theme values

Syntax

int okutil_getn_update_ranges( TreeNode * tr, BOOL bUpdateRowRange = false, IntArray * pvnIDs = NULL, StringArray * pvsValues = NULL )

Parameters

tr
[input] pointer to GetN tree
bUpdateRowRange
[input] true to update range string value
pvnIDs
[input] theme ids, optional
pvsValues
[input] theme values to update, optional

Return

0 if no update, 1 if update successfully

Examples

EX1 

#include <GetNbox.h>
void okutil_getn_update_ranges_ex()
{    
    GETN_BOX(trRoot)

    GETN_XYRANGE(Input, "Input Data", 1, "[Book1]Sheet1!((A,B),(A,C))")
    okutil_getn_update_ranges(&trRoot);

    if ( GetNBox(trRoot) )
        out_tree(trRoot);
     
     return;
}
