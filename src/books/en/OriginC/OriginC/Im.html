Name

Im

Declaration

#define Im(_CC)	(_CC.m_im)

Remark

returns the imaginary part of the complex value.

Parameters

_CC
the complex value.

Return

the imaginary part

Examples

EX1 

void    run_complex_Im()
{
    complex        cc(4.5, 7.8);
    
    // Display the imaginary part:
    out_double("imaginary part = ", Im(cc));  // shoulde be 7.8
}
