Description

Compute the Bessel function of the first kind of order n.

Syntax

double Jn( int n, double x )

Parameters

n
[input] order
x
[input] independent variable

Return

Jn(x)

Examples

EX1 

void	Jn_ex1()
{
	double		x = 4.;
	int		n = 2;
	double		val = Jn(x, n);
			
	printf("Jn(%f, %d) = %f\n", x, n, val);
}
