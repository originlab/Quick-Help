Description

Read fcf file header

Syntax

int ocmsp_readfcf_header( int * ptype, const char * filename )

Parameters

ptype
[output] filter type in fcf file, OMSP_FCF_TF, OMSP_FCF_SS, OMSP_FCF_SOS or OMOSP_FCF_UNKNOW
filename
[input] file name

Return

Return OE_NOERROR if succeed. non-zero error code is returned (OE_OPEN_FILE_FAILED, OE_READ_FILE_FAILED)
