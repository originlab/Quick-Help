Name

FORCE_DELETE_LOOSE_DATASET_WITH_GRAPH

Declaration

#define FORCE_DELETE_LOOSE_DATASET_WITH_GRAPH		LTVarTempChange _DLWG("@DLWG", 1)

Examples

void DeleteGraphWithLooseDataset()
{
	GraphPage gp = Project.Pages();
	if (gp)
	{
		FORCE_DELETE_LOOSE_DATASET_WITH_GRAPH;
		gp.Destroy();
	}
}

Remark

OC can use this convenient macro FORCE_DELETE_LOOSE_DATASET_WITH_GRAPH to delete loose dataset with graph deletion.
