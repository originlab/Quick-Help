Description

set legend update mode.

Syntax

BOOL SetLegendsUpdateMode( int nMode )

Parameters

nMode
[input] Legend update mode.
enum
{
LEGEND_UPDATE_MODE_UPDATE = 0,//update when adding.
LEGEND_UPDATE_MODE_RECONSTRUCT = 1,//Reconstruct when adding/removing.
LEGEND_UPDATE_MODE_NO = 2,//none.
};

Return

Return TRUE on successful set legend update mode and FALSE on failure.

Examples

EX1 

void SetLegendsUpdateMode_Ex1()
{
    GraphPage gp = Project.ActivePageBase();
    if(!gp)
        return;
    if(gp.SetLegendsUpdateMode(LEGEND_UPDATE_MODE_NO))
        out_str("Update Mode For Legends is set in GraphPage");
    else
        out_str(" Set update Mode  For Legends is fail");
}
