Description

This function is to get specified file's last wirtten time.

Syntax

BOOL okutil_get_file_last_write_time_as_julian( LPCSTR lpcszFile, double * lpdLastWriteTime )

Parameters

lpcszFile
[input] File path
lpdLastWriteTime
[output] julian date of the Last written time of the specified file

Return

TRUE if success, otherwise FALSE.

Examples

EX1 

void okutil_get_file_last_write_time_as_julian_Ex1()
{
    string strFile = GetAppPath(TRUE) + "Filter.ini";
    if(strFile.IsFile())
    {
        double dLastWriteTime;
        if( okutil_get_file_last_write_time_as_julian(strFile, &dLastWriteTime) )                
            out_str( get_date_str(dLastWriteTime) );
    }
    else
        out_str("File not existed.");
}
