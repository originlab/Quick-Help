Description

Incomplete Gamma functions (a,x)

Syntax

double incomplete_gamma( double a, double x )

Parameters

a
[input] the argument a of the function
x
[input] the argument x of the function

Return

Returns the values for the incomplete gamma function P(a,x)

Examples

EX1 

void incomplete_gamma_ex1()
{
	double a[5] = {2,7,0.5,20.0,21.0};
	double x[5] = {3.0,1.0,99.0,21.0,20.0};
	double dResult[5];
	for(int ii=0;ii<5; ii++)
	{
		dResult[ii] = incomplete_gamma(a[ii],x[ii]);
		printf("%4.1f	%4.1f:	%12.4e\n",a[ii],x[ii],dResult[ii]);
	}
}
