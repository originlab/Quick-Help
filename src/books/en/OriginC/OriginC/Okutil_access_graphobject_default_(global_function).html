Description

Access graph object's properties default value

Syntax

BOOL	okutil_access_graphobject_default( int nType, void * pSetting, int nOptions )

Parameters

nType
[input] graph object type, see GROBJTYPENAME
pSetting
[input] struct pointer to access object properties. See GrObjCommonProp, LineProp, FillProp.
nOptions
[input] AGRD_DEFAULT_SET to set properties, AGRD_DEFAULT_GET to get properties, AGRD_PROPERTY_LINE for line proerties, etc.

Return

return false if nType and pSetting and nOptions do not match.

Examples

EX1 

BOOL OC_EnableRescaleIncludeGraphObjs(BOOL bEnable)
{
	GrObjCommonProp prop;
	okutil_access_graphobject_default(GROBJ_TN_UNKNOWN, &prop, AGRD_DEFAULT_GET | AGRD_PROPERTY_COMMON);
	BOOL bOldVal = prop.bRescaleInc;
	prop.bRescaleInc = bEnable;
	okutil_access_graphobject_default(GROBJ_TN_UNKNOWN, &prop, AGRD_DEFAULT_SET | AGRD_PROPERTY_COMMON);
	return bOldVal;
}
