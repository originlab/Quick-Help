Description

Checks if a string is a date in the form "yyyyMMdd"

Syntax

int is_str_date(LPCSTR lpcsz)

Parameters

[input] lpcsz, string

Return

Integer returns non-zero if string is in expected form: "yyyyMMdd"

Examples

EX1 

void isdate(string str = "19860302")
{
	int nn = is_str_date(str);
	out_int("result ", nn);
}
