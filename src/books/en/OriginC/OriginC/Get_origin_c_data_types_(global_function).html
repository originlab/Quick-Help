Description

Get Origin C basic data types, returns the number of data types

Syntax

int get_origin_c_data_types( vector<string> & vsDataTypes )

Parameters

vsDataTypes
[output] String array which contents all data types in Origin C.

Return

the number of data types.

Examples

EX1 

int get_origin_c_data_types_ex1()
{
    vector<string> vsDataTypes; 
    if(get_origin_c_data_types(vsDataTypes))
        for(int ii=0; ii<vsDataTypes.GetSize(); ii++)
            printf("%s\n",vsDataTypes[ii]);
    else
        printf("error");
    return 0;
}
