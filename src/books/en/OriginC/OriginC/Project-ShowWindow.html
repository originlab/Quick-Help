Description

Show/Hide the sepcified control window.

Syntax

void ShowWindow( int nWinID = CWT_PROJECT_EXPLORER, BOOL bShow = TRUE )

Parameters

nWinID
[input]Control window's ID, can be one of the followings
CWT_PROJECT_EXPLORER,
CWT_SCRIPT_WINDOW,
CWT_CODE_BUILDER,
CWT_RESULTS_LOG,
CWT_COMMAND_WINDOW,
bShow
[input]TRUE, show the window, otherwise, hide the window.

Return

void

Examples

EX1 

// test if Script Window open, hide it if yes
void Project_ShowWindow_ex1()
{
    if(Project.IsVisible(CWT_SCRIPT_WINDOW))
        Project.ShowWindow(CWT_SCRIPT_WINDOW, FALSE);
}
