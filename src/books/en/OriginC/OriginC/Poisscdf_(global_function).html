Description

Poisson distribution cdf

Syntax

double poisscdf( const int iK, double rlamda, int iTail = TAILED_TEST_DISC_LOWER, int * nFail = NULL )

Parameters

iK
[input]integer which defines the required probabilities.
rlamda
[input]parameter of the Poisson distribution.
iTail
[input] int,
TAILED_TEST_DISC_LOWER. lower tail. .
TAILED_TEST_DISC_UPPER. upper tail. .
TAILED_TEST_DISC_POINT. point probability. .
nFail
[output] on successful exit, it returns the NAG error code NE_NOERROR; if an error or warning has been detected, then it returns the specific error or warning code.

Return

Returns Poisson distribution cdf

Examples

EX1 

void poisscdf_ex1()
{
	double iK[]= {0.25, 1.5, 2.75};
	double r = 0.75;
	for(int ii = 0; ii<3; ii++)
		printf("%f	%f:	%f\n", iK[ii], r, poisscdf(iK[ii],r));
}
