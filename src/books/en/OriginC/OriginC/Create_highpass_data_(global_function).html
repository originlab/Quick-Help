Description

Generate highpass Responses data.

Syntax

int create_highpass_data( int nSize, double * pResponse, double dFc1 )

Parameters

nSize
[input] number of points of the highpass Responses data
pResponse
[output] the result of highpass Responses data
dFc1
[input] the lower bound of the frequency(has been normalized, so 0<dFc1<0.5).

Return

Returns 0 for success or error codes for failure.

Examples

EX1 

#include <..\OriginLab\fft.h>
void create_highpass_data_ex1()
{
    vector vRes(8);
    create_highpass_data(8, vRes, 0.3);
}
