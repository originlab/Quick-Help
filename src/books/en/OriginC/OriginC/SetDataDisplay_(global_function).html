Description

Show or hide, docked or floating the Data Display control. if nValType = DD_SHOW, nVal = TRUE for show, FALSE for hide if nValType = DD_DOCK, nVal = TRUE for dock, FALSE for floating

Syntax

void SetDataDisplay( int nValType, int nVal )

Parameters

nValType
[input] DD_SHOW, DD_DOCK
nVal
[input] TRUE, FALSE

Return

Examples

EX1 

void SetDataDisplay_ex1()
{
    if( !GetDataDisplay(DD_SHOW) )
        SetDataDisplay(DD_SHOW, TRUE);

    if( GetDataDisplay(DD_DOCK) ) // Data display is dock now
        SetDataDisplay(DD_DOCK, FALSE); // change to floating
    else
        SetDataDisplay(DD_DOCK, TRUE); // change to dock
}
