Description

Divides double x by double y and returns the remainder.

Syntax

double rmod( double x, double y )

Parameters

x
[input] dividend
y
[input] divisor

Return

the remainder

Examples

EX1 

void    rmod_ex1()
{
    double        x = 6.9, y = 2.1;
    double        val = rmod(x, y);
    printf("rmod(%f, %f) = %f\n", x, y, val);
}
