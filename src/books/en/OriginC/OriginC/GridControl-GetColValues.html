Description

Gets one column of cells' text.

Syntax

bool GetColValues( int nCol, vector<string> & vsVals, int nRow1 = 0 )

Parameters

nCol
[input] column index
vsVals
[output] the cells' text
nRow1
[input] begin row index

Return

true if nCol is a valid column index, otherwise false

Examples

Example codes can be pasted to doExample() function in GridControl's examples to run 

		int nCol = m_GridCtrl.GetColOffset();
		
		vector<string> vsItems = {"Sun", "Mon", "Tue", "Wed", "Thu", "Fri", "Sat"};
		m_GridCtrl.CheckExpandRows( vsItems.GetSize() - m_GridCtrl.GetRowOffset() );
		m_GridCtrl.SetCells(vsItems, nCol);
		
		vector<string> vs;
		m_GridCtrl.GetColValues(nCol, vs);
		out_str( str_combine(vs, " ") );
