Description

get current language, E for English, J for Japanese and G for German.

Syntax

char okutil_get_current_lang( BOOL bUser = FALSE )

Parameters

bUser
[input]Internal use only

Return

a char for current language, E for English, J for Japanese and G for German.

Examples

EX1 

void okutil_get_current_lang_Ex1()
{
    char ch;
    ch = okutil_get_current_lang();
    printf("Current Language = %c\n", ch);//print out Current Language
}
