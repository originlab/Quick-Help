Description

Default constructor for a DatasetObject.


This constructor will create a DatasetObject given a series name.


This constructor will create a DatasetObject from another DatasetObject or a Column object.

Syntax

DatasetObject( )


DatasetObject( LPCSTR lpcszName )


DatasetObject( OriginObject & obj )

Parameters

lpcszName
[input] name of the series.


obj
[input] an OriginObject wrapping a DatasetObject or a Column object

Return

Examples

EX1 

void DatasetObject_DatasetObject_Ex2()
{
    MatrixObject    mobj;
    MatrixLayer        mlay = Project.ActiveLayer();
    if( mlay )
    {
        mobj = mlay.MatrixObjects(0);
    }
}
