Description

GraphPage to load page template.

Syntax

bool page_load( GraphPage & pg, LPCSTR lpcszTemplate, DWORD dwClicks = NOCLICK_USE_DEFAULT, bool * pbLoadSystemTheme = NULL )

Parameters

pg
[Input] Page need to load template
lpcszTemplate
[Input] template of the GraphPage
dwClicks
[Input] page disable mouse clicking setting. default NOCLICK_USE_DEFAULT means follow system settings
pbLoadSystemTheme
[Input] whether to apply system theme to page

Return

Examples

Ex1 

void page_load_Ex1()
{
    GraphPage gp;
    gp.Create();
    if( page_load(gp, STR_NONE_PREVIEW_TEMPLATE))
		out_str("Page_load success!");
}
