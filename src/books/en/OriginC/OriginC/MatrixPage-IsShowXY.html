Description

Get X/Y show state in MatrixPage.

Syntax

BOOL IsShowXY( )

Parameters

Return

if X/Y show in MatrixPage return TRUE else return FALSE.

Examples

EX1 

void IsShowXY_Ex1()
{
    MatrixPage  mp = Project.Pages();
    if(!mp)
        return;
    if(mp.IsShowXY())
        out_str("MatrixPage show XY");
    else
        out_str("MatrixPage hide XY");
}
