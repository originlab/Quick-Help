Description

Reset Matrix or Worksheet

Syntax

BOOL Reset( BOOL bReduceSize = TRUE )

Parameters

bReduceSize
[input] in Matrix, reduce to 2x2 whether bReduceSize = TRUE or not.
in Worksheet, same as ClearWorksheet macro.
Its default value is TRUE.

Return

TRUE if reset successful, FALSE otherwise

Examples

EX1 

//Reset the active matrix layer.
int Datasheet_Reset_Ex1()
{
    MatrixLayer ml = Project.ActiveLayer();
        
    if(!ml)
        return -1;

    return (int) ml.Reset();
}

EX2 

//Reset the worksheet.
int Datasheet_Reset_Ex2()
{
    Layer lay = Project.ActiveLayer();
        
    if(!lay)
        return -1;

    Worksheet wks(lay);
    return (int)wks.Reset(true);
}

Remark

Reset Matrix or Worksheet.

Existing data will be deleted.
