Description

Gets property of found object

Syntax

int GetNames( vector<string> & vs, QUERYRESULTNAME nType )

Parameters

vs
[output] property of found object
nType
[input] property type

{

	QUERYRESULTNAME_FOLDER			= QUERYRESULTNAME_FIRST,
	QUERYRESULTNAME_BOOKSHEET,
	QUERYRESULTNAME_BOOK,
	QUERYRESULTNAME_BOOKLNAME,
	QUERYRESULTNAME_BOOKLNAME_EMPTY_THEN_SNAME,
	QUERYRESULTNAME_SHEET,
	QUERYRESULTNAME_SHEETLNAME,
	QUERYRESULTNAME_SHEET_DISPLAY_NAME,
	QUERYRESULTNAME_BOOK_COMMENTS,
	QUERYRESULTNAME_SHEET_COMMENTS,
	QUERYRESULTNAME_SNAME,
	QUERYRESULTNAME_LNAME,
	QUERYRESULTNAME_RANGE,
	QUERYRESULTNAME_INDEX,
	QUERYRESULTNAME_NAMEDRANGE_SCOPE_RANEG,
	QUERYRESULTNAME_COMMENTS,
};

If successful then 1 else 0
