1.13.4 Report Sheet

Worksheet, Report SheetOutput to Report Sheet The Datasheet class has the GetReportTree and SetReportTree methods for getting and setting a report into a worksheet or matrixsheet. 

if( wks.SetReportTree(tr.MyReport) < 0 )
    out_str("Failed to set report sheet into worksheet.");
if( wks.GetReportTree(tr.MyReport) )
    out_tree(tr.MyReport);
else
    out_str("Failed to get report tree from worksheet.");
