1.2 Language Fundamentals

Language Fundamentals Origin C is closely based on the ANSI C/C++ programming languages. This means Origin C supports the same data types, operators, flow control statements, user defined functions, classes and error and exception handling. The next sections will elaborate on these areas of Origin C.
