1 Origin C Programming Guide

guide

Origin provides two programming languages: Origin C and LabTalk.

This guide covers the Origin C Programming language. It also shows you how to create and control Dialog Builder dialogs. Dialog Builder allows you to create and control custom dialogs such as floating tools, dialog boxes and wizards.

It is assumed the reader is familiar with C/C++ language including object-oriented programming concepts.

This guide should be used in conjunction with the Origin C Language Reference.
