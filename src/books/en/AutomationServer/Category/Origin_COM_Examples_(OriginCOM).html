1 Examples

Origin provides COM examples that demonstrate how to communicate with the different areas of Origin.

Build-in Examples

Examples of ADO, MS Office, Matlab, Labview and XML are installed with Origin. These examples can be found in the Samples subfolder of the Origin installation folder.

The following Samples folders contain a diverse set of examples:

Samples\COM Server and Client\ADO
Samples\COM Server and Client\MS Office\
Samples\COM Server and Client\MATLAB\
Samples\COM Server and Client\LabVIEW\
Samples\COM Server and Client\XML\

Additional Examples

Examples for CSharp, VB.NET and VC can be found in the zip file.

Notes: The examples are created in Origin 2021, they should be re-built to make them work in current Origin version. Please refer to the page for more details
