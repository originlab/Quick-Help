Description

Query objects inside Origin

Syntax

VB: Function GetFinder() As OriginFinder

C++: OriginFinder GetFinder()

C#: OriginFinder GetFinder()

Parameters

Return

The GetFinder method returns an instance of OriginFinder object.

Remark

Examples

Python

import OriginExt as O
app = O.Application(); app.Visible = app.MAINWND_SHOW
pageName = app.CreatePage(app.OPT_WORKSHEET)
finder = app.GetFinder()
finder.Find("1")
print(finder.GetObject(0).Name)
