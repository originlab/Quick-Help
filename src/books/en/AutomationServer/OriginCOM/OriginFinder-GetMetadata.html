Remark

DWORD BITS

0x00008000, Notes Text

0x00010000, Long Name

0x00020000, Short Name

0x00040000, Comments

0x00400000, Graph Text Object

0x00080000, Info Variables

0x00800000, Info Variable

0x00100000, Column Datas

0x00200000, Column Data

0x01000000, Column Units

0x02000000, Column Formula

0x04000000, Column Parameters

0x08000000, Column User Parameters

0x10000000, Column Sample Rates

0x20000000, Column Filters

0x40000000, Column Categories
