Description

The collection of DataPlot on this graphlayer

Syntax

VB: Property Get DataPlots As DataPlots

C++: DataPlots DataPlots

C#: DataPlots DataPlots

Remark

Examples

VB

    Sub DataPlots()
        Dim app As Origin.IOApplication
        Dim gl As Origin.GraphLayer

        app = New Origin.ApplicationSI
        gl = app.FindGraphLayer("")
        If gl Is Nothing Then
            MsgBox("Can not find active worksheet")
            Exit Sub
        End If
        Dim nNum As Integer
        nNum = gl.DataPlots.Count
        MsgBox("DataPlots number in GraphLayer is " + nNum.ToString())
    End Sub
