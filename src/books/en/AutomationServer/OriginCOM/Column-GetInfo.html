Description

Get column information. This function is out of date.

Syntax

VB: Function GetInfo As String

C++: LPCSTR GetInfo

C#: string GetInfo

Return

Remark

Examples

VBA

Function GetInfo() As String
    Dim app As Origin.ApplicationSI
    Dim Wks As Origin.Worksheet
    Dim strInfo As String
 
    Set app = New Origin.ApplicationSI
    Set Wks = app.FindWorksheet("")   'Set Wks as the active Worksheet
 
    strInfo = Wks.Columns(0).GetInfo()
    
    Range("A1") = strInfo
 
End Function
