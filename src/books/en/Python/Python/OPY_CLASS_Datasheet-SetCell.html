8.1.7.8 Datasheet::SetCell

It is recommended that you switch to the originpro package. PyOrigin is primarily for users who need to work with Origin version prior to 2021.

Description

Set the numeric value of a cell in the DataSheet

Syntax

SetCell(Row,  Col, Val)

Parameters

Row
row number
Col
column number
Val
value

Return

True if succeed, otherwise False

Examples

EX1 

#Create Book1 with one sheet with 3column and 5row, set some numbers in sheet
import PyOrigin
wks=PyOrigin.FindWorksheet('Book1')
#Set the number in col(3)[2]=0.5
wks.SetCell(1,2,0.5)
