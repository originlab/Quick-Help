8.2.20 GraphPages

It is recommended that you switch to the originpro package. PyOrigin is primarily for users who need to work with Origin version prior to 2021.


Contents

  • 1 Description
  • 2 Syntax
  • 3 Parameters

Description

Get the GraphPage.

Syntax

GraphPages(PageName)
GraphPages()

Parameters

  1. PageName
  2. no-argument

Return

Graphpage

Examples

EX1 

import PyOrigin
PyOrigin.GraphPages('Graph1').AddLayer('Layer2')

EX2 

import PyOrigin
if PyOrigin.GraphPages().GetCount() > 0:
    # Loop for each graph page.
    print('Prall graphs name:')
    for graph in PyOrigin.GraphPages():
                print(graph.GetName())
