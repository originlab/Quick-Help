8.1.3.10 DataObject::GetDisplayFormat

It is recommended that you switch to the originpro package. PyOrigin is primarily for users who need to work with Origin version prior to 2021.

Description

Gets the display format of the DataObject

Syntax

GetDisplayFormat()

Parameters

Return

0 to display as "Decimal:1000";
1 to display as "Scientific:1E3";
2 to display as "Engineering:1K";
3 to display as "Decimal:1,000";

Examples

import PyOrigin
wks=PyOrigin.WorksheetPages('Book1').Layers(0)
Col=wks.Columns(0)
Col.SetDisplayFormat(2)
if Col.GetDisplayFormat()==0:
    print('Decimal:1000')
elif Col.GetDisplayFormat()==1:
    print('Scientific:1E3')
elif Col.GetDisplayFormat()==2:  
    print('Engineering:1K')
elif Col.GetDisplayFormat()==3:  
    print('Decimal:1,000')
