Contents

Description

Bundle an x column and a y column of a worksheet to a cluster.

Connector Pane

Controls and Indicators

Origin.Worksheet is a reference to Origin.Worksheet.
error in(no error) contains error information that occur before this VI or function runs.
xCol is the index of a x column.
yCol is the index of a y column.
Origin.Worksheet Dup is a reference to Origin.Worksheet.
Bundled Cols is a cluster which has bundled two 1D arrays, one of which is gotten from a x column and another is gotten from a y column.
error out contains error information.
Note: This VI is available since Origin 8 SR4.
