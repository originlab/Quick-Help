Description

Get text array from a Column.

Connector Pane

Controls and Indicators

Origin.Column is a reference to Origin.Column.
nStart is the starting index to set array, default is 0.
nEnd is the ending index to set array, default is -1.
Error In contains error information that occur before this VI or function runs.
Origin.Column is a reference to Origin.Column.
Data is a 1D array of string.
Error Out contains error information.
Note: This VI is available since Origin 8 SR2.
