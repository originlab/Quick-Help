1.4.1 OA Col-GetWave


Contents

  • 1 Description
  • 2 Connector Pane
  • 3 Controls and Indicators

Description

Get a waveform from a column. If the column has sampling interval info, then the waveform attributes will also be properly set.

Connector Pane

Controls and Indicators

Origin.Column is a reference to Origin.Column.
error in(no error) contains error information that occur before this VI or function runs.
Origin.Column Dup is a reference to Origin.Column.
Waveform Out returns waveform which transfered from a column.
error out contains error information.
Note: This VI is available since Origin 8 SR4.
