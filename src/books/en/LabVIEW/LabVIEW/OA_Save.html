Description

Save an .opj file to disk.

Connector Pane

Controls and Indicators

Origin.IOApplication is a reference to Origin.ApplicationSI.
Path accepts a full path of the opj file that you are saving.
Error In contains error information that occur before this VI or function runs.
Origin.IOApplication.Out
Save is TRUE if file is saved successfully, otherwise, saved unsuccessfully.
Error Out contains error information.
Note: This VI is available since Origin 8 SR2.
