2 LabVIEW Tutorials

tutorials

These tutorials serve as a step-by-step guide for a new LabVIEW user to create simple VIs, such as to generate some simulated data and save them into an Origin OPJ file.

More experienced LabVIEW users can look directly at the samples provided in the following Origin installation folder: 

<exe>\Samples\COM Server and Client\LabVIEW\
Notes: All of these Tutorials are written with LabVIEW 7 Express.


Tutorials included in this section:

  • Creating a Simple VI to Save Data to Origin
  • Sending Data to Save into OGW
  • Plotting with a User-Defined Template
  • Working with Origin Analysis Template
  • Getting Data from Worksheet Range
  • Working with Origin Com And LabVIEW SubVIs
  • Setting and Getting Matrix Data
  • Using LabTalk to Get Worksheet Information
