2.9 Sequential Fit (Pro)

Sequential-Fit

Summary

The Sequential Fit app can be used to fit a large number of datasets from a worksheet sequentially, and create a simple report table of parameter values and fit statistics.

Tutorial

  1. Start this tutorial with the app Sequential Fit installed. If you have not installed this app, please click Add Apps button in Apps Gallery to open App Center to search and install the app.
  2. New a worksheet, and then click Data> Connect to File: Text/CSV menu to import the sample data <Origin Porgram Folder>\Samples\Curve Fitting\Step01.png. Delete the first column, select all columns and right-click on them to select Set as : XY XY to set the designation.
  3. As you can see, there are 8 input XY dataset. We will perfrom fitting on them one by one sequentially, by initializing the parameters with fit values of the previous dataset. Click the app icon to open the app dialog.
    • The active worksheet has been selected as Input by default.
    • Set Category to Exponential and Function to ExpGro1.
    • Select Use Fit Values from Previous Dataset for Parameter Initializaiton .
    • Click Init Parameters button to open the Init Parameters dialog. Set the bounds for parameter "A1" and "t1"
  4. Click OK button return to the app dialog.
  5. Click OK button to perfrom the fitting. Two result sheets and one result graph will be created. You can use the navigation buttons at right-bottom connor of the fitted curve graph to check the fitting results of each input curve.
