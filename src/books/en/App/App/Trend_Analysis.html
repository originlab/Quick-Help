2.2.3.4.1 Tutorial for Trend Analysis Tool

TSA-TD-TA-Tutorial

Steps in Origin

  1. Open the sample project file in Origin, go to Folder Transform and Decompose using the Project Explorer. Activate the workbook Annual electricity sales for South Australia from 1989 to 2008.
  2. Highlight column B in worksheet. Click the Time Series Analysis App icon in the Apps Gallery window.
  3. Choose Transform and Decompose tab. Click Trend Analysis icon to open the dialog.
  4. In the Setting tab,choose Exponential Growth model type. Enter 5 in Number of Forecasts box.
  5. Click Preview button to display fitted curve with forecasts against actual data.
  6. Accept all other default settings and click OK button
