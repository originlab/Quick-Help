1.38 Torsional Angle Wheel Plot (Pro)

Torsional-Angle-Wheel-Plot

Summary

The Torsional Angle Wheel Plot app can be used to calculate bins of torsional angles for each group and create a polar contour for them.

Tutorial

  1. Start this tutorial with the app Torsional Angle Wheel Plot installed. If you have not installed this app, please click Add Apps button in Apps Gallery to open App Center to search and install the app.
  2. Right-click on the app icon to select Show Samples Folder to open the folder where the sample project "TAWPlotEx.opju" is stored.
  3. Drag-and-drop the opju file "TAWPlotEx.opju" into Origin workspace to open it.
  4. Highlight col(A)~col(D), and then click Torsional Angle Wheel Plot icon in the Apps Gallery window. A dialog will open. Set Number of Bins for Angles to 36(It will divide the interval 0-360 degrees into 36 evenly spaced intervals). Click OK button to create the plot.
  5. A Matrix book with a 36Ã—4 matrix and a torsional angle wheel plot will be created. As you can see, there are four wheels in the polar coordinate which are from the four source columns. And each wheel is divided into 36 pieces.
