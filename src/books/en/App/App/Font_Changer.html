1.19 Font Changer

Font-Changer

Summary

The Font Changer app can be used to batch modify fonts in a graph.

Tutorial

  1. Start this tutorial with the app Font Changer installed. If you have not installed this app, please click Add Apps button in Apps Gallery to open App Center to search and install the app.
  2. Select Help: Learning Center to open Learning Center. Under the category Multi-Panel Graphs, select the sample "Multi-Panel Graphs - Experimental XES and XAS Spectra" to open it.
  3. Activate the graph window, click the app icon to open the Font Changer.
  4. Check Axis Titles and Other Text Objects and set Font = Cambria and Font Size = 14. This time, we only want to change the font for the first 5 layers. Enter 1 2 3 4 5 into the Layer Index box. Click Preview button to preview the changes in Preview panel.
  5. Click Apply button to apply the font changes into the source graph. As you can see, only the axis titles and text obejcts in the first five layers have been changed in font.
    If you don't want to change the source graph, you can click the button Create to create a copy graph with all the font changes.
