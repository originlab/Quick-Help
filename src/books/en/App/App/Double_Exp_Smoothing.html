2.2.4.3.1 Tutorial for Double Exponential Smoothing

TSA-Smooth-DoubleExp-Tutorial

Summary

This Double Exponential Smoothing tool is supported in the Time Series Analysis App. It is suggested to smooth the time series with trend.

Notes for Starting

  • Start this tutorial with the app Time Series Analysis installed. If you have not installed the app, see Apps for Origin to find and install the app.

Topics for Further Reading:

  • Introduction: Double Exponential Smoothing
  • Algorithm: Double Exponential Smoothing


Steps in Origin

  1. Open the sample project file in Origin, go to Folder Smooth using the Project Explorer. Activate the workbook Annual electricity sales for South Australia from 1989 to 2008.
  2. Highlight column B in worksheet. Click the Time Series Analysis App icon in the Apps Gallery window.
  3. Choose Smooth tab. Click Double Exp Smoothing icon to open the dialog.
  4. In the Setting tab,set Weights to Optimal ARIMA. To predict beer sales in the next 3 months, set Number of Forecasts to 3.
  5. Click Preview button to display fitted curve with forecasts against actual data.
  6. Accept all other default settings and click OK button.
