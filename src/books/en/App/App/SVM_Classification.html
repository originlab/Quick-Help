2.48 SVM Classification

SVM-Classification

Contents

  • 1 Summary
  • 2 Tutorial

Summary

The SVM Classification app can be used to perform the support vector machine algorithm for classification.

Tutorial

  1. Import Fisher's Iris Data.dat in the <Origin EXE Folder>\Samples\Statistics\ folder.
  2. Select Column A and B, then click on SVM Classification icon from the Apps panel to open the dialog.
  3. In the dialog, set Column E as Training Y. Uncheck Predict checkbox.
  4. Check the Territorial Map option . For 2 Dimensions' X data, territorial map plot is available for output. Click OK.

    Then you will get the SVMReport with Territorial Map
