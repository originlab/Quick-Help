1.26 3D Wind Rose

3D-Wind-Rose

Summary

The 3D Wind Rose app can be used to create a 3D Wind Rose graph.

Tutorial

  1. Start this tutorial with the app 3D Wind Rose installed. If you have not installed this app, please click Add Apps button in Apps Gallery to open App Center to search and install the app.
  2. Download this sample file and open it in Origin workspace.
  3. Click the app icon to open the dialog 3D Wind Rose.
  4. Drag-and-drop the column A, B and D into the Input area, set Sphere Scale to 1.2, and check the Auto Preview box to show the 3D Wind Rose plot in the preview panel.
  5. You can use the Rotation Preview Graph controls to rotate the sphere to check the details of the wind rose plots. And, you are also allowed to increase the grid lines on both Auzimuth and Elevation direction.
