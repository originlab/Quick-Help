Custom X Values for Fitted Curve

 (ORG-7459)

 

Origin 9.1 supports choosing an existing dataset as the X dataset of the fitted curve, which enable user to create fitted curve in ranges with different sampling interval or in any non-evenly-spaced data range.

Following curve fitting tools support this option:

  • Linear Fit
  • Linear Fit with X Error
  • Polynomial Fit
  • Nonlinear Curve Fit

The option is available in the the Fitted Curves Plot tab/branch, X Data Type drop down list. User can specify an existing column for the X value of Fitted Curve.
