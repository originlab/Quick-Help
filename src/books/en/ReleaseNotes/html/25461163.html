Unicode Support in Origin

(ORG-7743)

Support unicode font allows international users to create and display text using characters from languages other than English. 

  • Copy and paste unicode text from clipboard into Origin worksheet (cell format to be rich text)  (ORG-7793, ORG-7744)

 

  • Copy and paste unicode text from clipboard into text object (ORG-3262)

 

Copy and paste is driven by new system variable @PSTU, @COPU.
