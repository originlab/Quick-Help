PA: Copy and Paste for Anchor Info and Peak Info

(ORG-7728)

In Peak Analyzer tool, Anchor Info dialog in Baseline Mode page and Peak Info dialog in Find Peaks page support Copy and Paste.

You can paste data two columns of data into Anchor Info dialog or Peak Info dialog, and use these points as baseline's anchor points or peak's locations, or copy baseline's anchor points or peak's locations into the clipboard.
