User Defined Axis Scale Type

Sample OPJ to download to try: Multi breaks and Other Axis Related.opj

(ORG-3964)

(ORG-7980)

Note:

  • User must provide both direct and mutual inverse formula. 
  • From and To value should not exceed the range of them.

How to:

  1. Double click the axis you want cunsomlized to open Axis Dialog.
  2. Go to X Axis/Y axis: Scale node, change Type to be Custom Formula in dragdown list.
  3. Input direct and inverse formula in the two new shown editbox.
