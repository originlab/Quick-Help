New Asterisk Bracket Tool

Ready in Beta3

(ORG-8703) 

Many users ask for easy to indicate sigificant difference between two bars with asterisk in Column Graph.




 


We have separated out the buttons that add objects on a graph into new toolbar called Add Object to Graph  and placed it underneath the Tools toolbar

(ORG-9539)

How to

  • Click to add a new asterisk bracket on Graph. The moment you click this, the new object is added and placed near the center of the graph.
  • Click to select the object. Controls will show in horizontal link and bracket legs.
  • Drag the horizontally line to move the whole object, including asterisk.
  • Drag the control on horizontal line to adjust height.
  • Drag the control of bracket legs to move it.
  • Multiple Asterisk Brackes can be added to the graph.
  • Double click the line to open Object Properties dialog to customize the lines.
  • On Dimension tab, user can set Unit to be Scale and enter integers so the line will drop on center of bars.
  • Drag the asterisk to move the position. Double click it to edit it.
