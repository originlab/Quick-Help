BIC Test for Model Comparison

 (ORG-6495)

 

In addition to Akaike information criterion (AIC) and F-test, Origin 9.1 supports Bayesian information criterion (BIC) for model comparison.

It is available in the Fit Model Comparison tool. To use the tool

  1. Perform fitting twice using two different fitting functions for the same data, and create two fitting reports.
  2. Select Analysis: Fitting: Compare Models from Origin menu. Then check Bayesian information criterion (BIC) checkbox in Comparison Method Branch.
