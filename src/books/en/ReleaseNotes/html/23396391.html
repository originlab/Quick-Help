Import File Name into Worksheet Long Name and Short Name by default

(ORG-9438)

By default when importing data into Origin, the file name will be part to Sheet Name. But we don't support special characters such as {, }, `, |, <, >, %, |, [ or ] in sheet name so user can't use the meaningful file names in other places in Origin.

In Origin 9.1, user can import the file name with special characters in Worksheet Long Name. And by default Worksheet Long name shows if it's availble.

User can right click the sheet name tab and choose Sheet Name to Show to switch to show Short Name.

User can double click the Sheet tab to rename short name or long name.

Import File Name into Worksheet Comments

(ORG-9229)

In Origin 9.1, user can also import File Name into Comments and use it in other places, e.g. in the following case, use the File name as Legend by @WM substitution notation. See how to customize legend in Legend Improvements page.

 

In ASCII Import dialog, user can check Append Filename to Worksheet Comment and uncheck Include File Path when Appending Filename.
