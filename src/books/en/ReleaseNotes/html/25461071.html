Support of Multi-level Table-like Tick Labels

(ORG-8556) (ORG-5345)

In Axis Dialog: X(or Y) Axis: Tick Labels: Bottom(or Top/Left/Right) tab, turn on the Table check box so that user can construct axis tick labels in a table format.

How to:

  1. Go to Axis Dialog: X/Y Axis: Tick Labels: Bottom/Top, check Table check box.
  2. More detailed table setting interfaces will show.
    See more details in How to Customize the Table-liked Tick Labels
  3. Right click the Bottom/Top N treenode to Hide/Add/Delete/Move the table row.
