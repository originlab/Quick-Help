More Baseline modes in PA: Straight Line, Use Script and Use Existing Dataset

(ORG-7728)

Use Straight Line

This option was only available for Integration gadget in the past. Now added to Baseline Mode of Peak Analyzer and also Quick Peaks gadget. It will connect two end points of the data plot and connect them as baseline.

Use Script to Search and Use Existing Dataset

In Peak Analyzer tool, when Baseline Mode is User-Defined, two more methods Use Script to Search and Use Existing Dataset are added.

In this way, user can define his own method to detect baseline anchor points according to his data using LabTalk script, calling X-Function or existing dataset. These customized methods can also be saved as a theme for batch peak analysis.
