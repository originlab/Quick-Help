Custom Color, Shape, Interior List per Data Plot

Customized Color/Shape/Interior Increment List for plot (ORG-2518, ORG-8795)

How to:

  1. Go to Plot Details: Plot Properties: Symbol tab.
  2. Set Edge Color/Fill Color/Shape/Interior to be Increment or Indexed by a column.
    You should check Custom Construction checkbox first for symbol increment/index.
  3. Go to Color/Shape/Interior List tab, check Use custom increment list checkbox to set your own list.

Note:

  • Edge Color, Fill Color in Symbol tab and Color in Line tab share the same Color List.
  • User can also right click in the gray area of the increment list and Load Increment List or Save Increment List for future usage.
