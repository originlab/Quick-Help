Origin 2017 Installation and Licensing

Licensing

  • For nodelocked Origin users (both single user, grouped user), no need to run as admin to do the licensing.
  • With internet access on, paste Product Key to activate the license automatically instead of the old way of generating a long licensing string and copy back into Origin, 
  • If no internet access or firewall issues, clicking Activate button to generate URL to manually get the long string license online.
  • Easier transfer of license: Just choose Help: Deactivate License... to deactivate license. Then go to new PC and use same Product Key to activate the new PC. No need to get new license file.


Note: When user purchases Origin 2017, a link will be sent to user's email account to login or signup on our website Originlab.com to register product and generate Product Key. Product Key will be sent to user's email address. 

User can also later login our website to view Product Key again.


Evaluation

User can easily switch between Origin vs OriginPro to see difference between them via Help: Run Evaluation as OriginPro menu. No need to run Modify program.

Converting from Evaluation to Product via Help: Convert Evaluation to Product menu. No need to run Modify program.
