Save Project without Data, Duplicate Workbook or Project without Data

(ORG-9099)

In previous versions, user has to save workbook or project as workbook/project templates and load it from File: Recent Workbooks or Recent Projects submenus to load the templates again. And use it from there.

So in Origin 9.1, we improved the menu to save user some steps and make it easier for users.

Save Project Without Data

File: Save Project Without Data... menu

This will save an OPJ file based on current workspace only with data removed. User can use it as project template and fill in new data.

Duplicate Project Without Data

File: Duplicate Project Without Data menu

This will duplicate current project without data. If the current project is dirty, user will be given a chance to save the project.

Duplicate Workbook Without Data

By duplicating workbook without data, user can quickly create a new workbook with the same structure and fill in new data to do the analysis. No need to save workbook as analysis template.
