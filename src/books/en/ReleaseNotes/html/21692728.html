Import Mat File Structure

 (ORG-8462,ORG-8274,ORG-1913)

The MATLAB file import dialog has been improved to support .MAT files with structure data. A new option named Import into Hierarchical Structure has been added.

Choose File: Import: MatFile(MAT)... to launch the dialog.

With the Import into Hierarchical Structure check box checked, only the variable list is first imported into a worksheet. You can then click on the individual variable names, displayed as linked text in the 1st column, to further import just the desired variables one by one, into new worksheets.
