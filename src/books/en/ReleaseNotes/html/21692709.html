Akima Spline Interpolation

 (ORG-3238)

Sample OPJ to download to try:   Akima Interpolation.opj

Akima Spline is added as a new data plot line connection method new line connection method, and also added as a new interpolation method.

Akima Spline Interpolation method is available in three interpolation tools. It shows its advantage when you would like to maintain the curve oscillation trend after interpolation, especially when there is an outlier. 

  • Interpolate Gadget
  • Interpolate/Extrapolate
  • Interpolate/Extrapolate Y from X

Interpolate Gadget

To use the Interpolate Gadget,  select Gadgets: Interpolate from the Origin menu, when a graph is active

 

Interpolate/Extrapolate

To use the interpolate/Extrapolate tool, select Analysis: Mathematics: Interpolate/Extrapolate from the Origin menu, when a worksheet or a graph is active


Interpolate/Extrapolate Y from X

To use the interpolate/Extrapolate Y from X tool, select Analysis: Mathematics: Interpolate/Extrapolate Y from X from the Origin menu, when a worksheet is active
