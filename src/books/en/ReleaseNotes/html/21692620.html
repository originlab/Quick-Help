Plot: User Defined: Menu to Access Customized Templates

(ORG-8840)

Â 

In the past, if a customized a graph was saved as a template, the template had to be accessed first by opening the Template Library tool.

In 9.1, a new fly out menu Plot: User Defined has been added, which will display all the user-created graph templates for easy access, no matter which category the graph template was saved into.
