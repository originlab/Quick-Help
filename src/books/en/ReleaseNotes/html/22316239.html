2D Integration Gadget (Pro)

(ORG-2460)

2D Integration gadget can integrate ROI in an image plot or a contour plot from XYZ data or matrix data.Â 

It also supports baseline in the integration. Baseline can be a constant or a fitted plane.

Integration results include:

  • Volume
  • Surface Area
  • FWHM for x and y
  • Centroid for x and y
  • Baseline

To use this gadget, make a contour plot or an image plot active, select Gadgets: 2D Integrate from the Origin menu.
