Analysis on Virtual Matrix based Plot

(ORG-487)

Sample OPJ to download to try:  Analysis on Virtual Matrix Based Plot.opj


In Origin 9.1, we can perform analysis on 3-D graphs or contours plotted from virtual matrix data. 3-D plots and contour graph can be plotted from virtual matrix from Origin 8.51 but no analysis are supported.

We support the following analysis for Virtual Matrix Based Plot

  • Nonlinear Surface Fit
  • 2D Interpolation
  • 2D Volume Integration
  • Matrix Surface Area
  • Custom Filter
  • Media Filter
  • Special Filter
  • 2D FFT
  • 2D IFFT
  • 2D Wavelet Decompose

How to

  1. Graph 1 is plotted from a virtual matrix. Active the graph, select Analysis: Fitting: Nonlinear Surface Fit  from the Origin menu
