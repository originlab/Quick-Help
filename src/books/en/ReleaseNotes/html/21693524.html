Importing OPJ

Import Origin Project File (OPJ) (ORG-9012)

 

A new X-Function has been added to facilitate importing data from another OPJ into the current OPJ.

When a worksheet is active, choose File: Import: Origin Project (OPJ)... to import selected columns in an OPJ file. Options are provided to select a particular book in the OPJ file, then all available sheets will be listed, allowing you to select a particular sheet, and then select particular columns from that sheet.
