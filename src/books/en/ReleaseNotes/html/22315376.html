Fit Comparison Improvements

  • Allow renaming model/datasets to better distinguish them 
  • Improve Compare Datasets to remember dialog setting for combined data

Allow renaming model/datasets to better distinguish them 

(ORG-8460)

When doing fit comparigon, we didn't allow user to rename models or datasets so sometimes it's hard for user to read the comparison results.

In Origin 9.1,

  • When Comparing Models, user can give two models better names to better distinguish them.

 

  • When Comparing Datasets, user can give two Datasets better names to better distinguish them as well.

Improve Compare Datasets to remember dialog setting for combined data

(ORG-8729)

When Compare Datasets for combined data, 9.1 will use dialog setting instead of fdf file, avoid the necessary to change in fdf file.
