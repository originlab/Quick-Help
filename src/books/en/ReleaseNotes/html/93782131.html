Subgroup control in Unbalanced Grouped Column Plot

ORG-14212

New system variable added to define filtering with categorical data


Before filtering, please notice that GMC and Kia's color is Green and Cyan. With new system variable @GMHC, users are able to control whether keep original legend color or not. @GMHC = 0, means filtered data legend remains the same color as before filter, while @GMHC =1 means not.

ORG-15292Categorical legend improvement related to connect line
