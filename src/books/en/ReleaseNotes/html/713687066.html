ORG-20573

CSV Connector only import 100 columns of data if column delimiter is comma

 When using a Data Connector to import CSV files, only the first 100 columns are imported.  Workaround is to use Data: Import From File: Comma Delimited (CSV). If the user does not want to save data with the project, he/she can choose to Exclude imported data when saving project (right-click on workbook title bar and choose Properties).

ORG-20050

Text object properties dialog in Matrix open as Matrix Properties

If right click a text object in Matrix window and choose Properties

==> Matrix Properties dialog opens.

Workaround: Ctrl+double click or Alt+double click on the text object to open Text Object dialog.
