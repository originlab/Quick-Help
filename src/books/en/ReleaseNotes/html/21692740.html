Improvements to Excel Import

More options in File: Import Excel dialog (ORG-8827)

Replace Existing Data option is added in 1st File Import Mode, allowing data to be imported into existing book, rather than create new book.

Exclude Empty Sheets checkbox has been added to leave out empty sheets in Excel file when importing.

Reimport of Excel File (ORG-8826)

In previous versions, importing an Excel file always resulted in the data being placed into a new workbook. It was also not possible to re-import a file into the same book that it was imported into previously.These limitations made it not possible to create analysis templates for processing Excel files.

In version 9.1, we have removed these restrictions. When importing Excel, the option to replace existing data is available, similar to other imports, Plus it is possible to re-import the file to bring in updated data if the file has changed since the last import.
