Data Labeling for 3D Bars

(ORG-5697)

Sample OPJ to download to try: 3D Bar with Labels.opj


Origin 9.1 supports label for 3D XYZ and XYY bars. You can use the Label tab in the Plot Details dialog to add or customized the label of 3D bars.  In this tab not only the content can be defined for the label by selecting the option from the drop down list of Label Form, but also the style, position and orientation of the labels can be customized.


Label tab in Plot Details dialog.

How to

The US Map is a matrix with 1's and 0's, displayed as a flat surface.

The population data is XYZ bars.

Labels have been turned on, and orientation of labels can be controlled in the Labels tab.(Double click on the labels to open the Plot Details dialog)
