Generate Patterned Data

(ORG-7792)

 

Origin 9.1 enable you to fill column with

  • Patterned/random numbers
  • Patterned/random date/time values
  • Patterned or random set of strings.

 

To fill the patterned data

  1.  Select a worksheet column or a range of cells in a worksheet column
  2. Right-click on the worksheet column and choose Fill Column with from the shortcut menu
