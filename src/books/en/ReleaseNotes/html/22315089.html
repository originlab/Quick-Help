Apply settings to Others

(ORG-7905)

There is Apply to Others node at the end of left panel, where user can apply one axis's tick label, line and ticks and grid line settings to other axes. It can be axes in same layer, same window, all windows in same folder or all windows in the project.

User can also Right click X Axis and Y Axis node on the left panel and choose Apply to Others... context menu to go to this node.

Select Others to easily set settings in multiple corresponding nodes

When user is on one node, user can click Select Others button to select matching node in other direction and customize their setting together.
