Gadgets: Batch Analysis

(ORG-8188)

Sample OPJ to download to try:  Gadget - new output for all curves.opj

 

New menu items have been added to the fly-out menu in Gadgets to output results for all curves, thus allowing batch analysis of multiple curves in a graph using gadgets.

After determining region of interest and customizing settings, choose New Output for All Curves or New Output for All Layers to do the analysis on all curves and output the results.
