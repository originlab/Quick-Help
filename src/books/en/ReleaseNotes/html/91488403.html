ORG-14316

More options in Rescale dropdown list on Scale tab


If user want to fix the From value and auto rescale axis when new data comes in, please choose Fixed From - Auto To.

Our old Fixed From option is now renamed Fixed From - Normal To, since it fixes from value. And not rescale to value till user click rescale to show all.
