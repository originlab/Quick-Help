Â (ORG-8473)

In Statistics tools that will generate plots, such as histogram or box chart, it's nice to show the same plot type in one graph so user can compare them in same scale.

So we added Arrange Plots of Same Type in One Graph checkbox under Plots node for those statistics that applies, such as those one or two sample t-test, paired t-test, etc.
