Graphing

JIRA Summary Details
ORG-8337 New Graph theme: Layer No Element Scaling added. To avoid elements in layer such as text, tick labels, axis titles from resizing when resizing layer, with graph window active, and press F7 to open Theme Organizer. And apply this theme to graph.

Import

JIRA Summary Details
ORG-7963 Improve NI TDM file import speed Improve the performance of NI TDM file importing. For example, for a 4.36GB file with 33 groups, it will cost 3.69 days to import the file before 9.0Sr2. But now in Sr2, only 2.27 hours will be cost.

Inteface

JIRA Summary Details
ORG-8319 Function Plot toolbar buttons are added. 4 toolbar buttons are added and grouped on Standard toolbar.
ORG-8173 Support custom menu with no window active From beta forum - http://beta.originlab.com/forum/topic.asp?TOPIC_ID=3705

Before the change, user menu can only be added to menu with workbook, graph or matrix window active.

ORG-8378 Provide more built-in button images for user defined toolbars Choose View: Toolbars... menu. On Button Group tab, there are more UserDefined button groups with different images.

DLL

JIRA Summary Details
ORG-8129 Need to include necessary files in installation for MOCA, both 32 and 64 bit Included MOCA DLL (both 32 and 64 bit) and examples in installation.
