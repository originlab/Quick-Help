Known Issues in 8.5.1 SR2



Analysis

JIRA Summary Details
ORG-2988 Paste Link to column label cells does not work. Users may copy a table in a hierarchy result sheet to a new worksheet via the Create Copy as New Sheet menu item. If a user then copies a cell from the new sheet and uses Paste Link to paste to a column label cell (e.g. Long Name), a link appears in the destination cell instead of a real value. Note, this does not occur when using Paste.
ORG-2988 Past Link to another workbook does not work with relative links. Users may copy a table in a hierarchy result sheet to a new worksheet. If a user then copies cells from the new sheet and uses Paste Link to paste into a new workbook, relative links appear in the destination cells instead of real values. Note, this does not occur when using Paste.

Excel

JIRA Summary Details
ORG-3084 Excel file type xlsm shows as an import option but is not yet supported. On computers with Excel 2007 or later installed, file type xlsm shows as an option in the Open Excel and Import Excel dialogs, but this file type is not yet supported. Excel file type xlsm will be supported in 9.0.
