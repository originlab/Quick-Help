Matrix

JIRA Summary Details
ORG-5198 Failed to get the right result with lots of function like cot() using x or y as argument in matrix book. Newly introduced in 8.6 SR2.

It exists in all OC-based functions (usually in internal.c), like acot(x), csc(x), secant(x), asec(x), acsch(x), degrees(x) etc.


LabTalk

JIRA Summary Details
ORG-5471 Run script which ends with "\" in Command Window cause Runtime error Newly introduced in 8.6 SR2.

Run this command in Command window, 

string fpath$ = system.path.program$+"Samples\Curve Fitting\";

"general operation failure" dumps to Message Log.

  • If removing the last "\", then no problem.
  • The problem does not exist when run from Script Window nor OGS file.
ORG-5477 Script using StringArray Append method produces Run Time Error This worked in versions previous to 8.6. 
stringarray sa;
sa.append("one|two|three|four","|");
loop(ii,1,sa.GetSize())
{
   sa.GetAt(ii)$=;
}
sa.append("oh, just five more things!");

It still works if you specify the delimiter: 

stringarray sa;
sa.append("one|two|three|four","|");
loop(ii,1,sa.GetSize())
{
    sa.GetAt(ii)$=; 
}
sa.append("one more thing","|");
sa.append("three more things"," ");
