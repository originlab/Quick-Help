Known issues in 8.5 SR0

Importing/Exporting

JIRA Summary Description
Transparency is not supported if graphs are exported to PDF, EPS or other vector formats.

Graphing

JIRA Summary Description
The "Column+Labels" 2d Graph button does not work in German OS. Labels can however be added using Plot Details once graph is created.
3D Plots with Error Bars does not work in German OS.

Licensing

JIRA Summary Description
ORG-3014 FLEXnet server with 8 digit solaris hostID doesn't work for Origin 8.5 and Origin 8.5.1. 12 digit works Failed to connect 8.5 to FLEXnet Server with 8 digit HostID. Workaround is to generate FLEXnet Server with 12 digit HostID.
ORG-1137 in7, start Origin in normal way, license message pops Win7, Starting Origin in normal way, license message pops.
