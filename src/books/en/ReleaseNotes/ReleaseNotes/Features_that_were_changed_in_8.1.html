Features that were changed in 8.1


Analysis

Data Manipulation

  1. Reduce Evenly Spaced X Data: Reduce data size by reduction factor and different sampling method for evenly spaced X data.
  2. Reduce X Duplicated Data: Reduce data size by removing duplicated X data points. For those duplicated X data, statistics values is used as Y.
  3. Reduce Clustered X Data: Reduce data size by different subgrouping methods and statics for clustered X data
