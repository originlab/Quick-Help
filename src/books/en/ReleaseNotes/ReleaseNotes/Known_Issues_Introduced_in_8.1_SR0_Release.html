Known Issues Introduced in 8.1 SR0 Release


Graphing

  1. When plotting data with X or Y error bars, if axis is set to log scale and the lower error bar value goes to a negative number, the error bar incorrectly draws to a positive number instead. ID=4170 (fixed in SR1)
  2. Offset Reciprocal scale type gave bad scaling. ID=15199 (fixed in SR3)
  3. Polar graph shows data points outside Y scale. ID=15225 (fixed in SR3)
