Menu Information

Plot: Probability: Probability Plot

Plot: Probability: Q-Q Plot

Brief Information

Create probability or Q-Q plot using specified distribution

Additional Information

Minimum Origin Version Required: 8.1 SR0

Command Line Usage

1. plot_prob -r 1 irng:=[QQplot1]"Q-Q plot"!B"Weight" method:=renard;

2. plot_prob irng:=[ProbPlot]Sheetl!A distr:=weibull;

Note: in Origin 2017 or ealier versions, this X-Function is not accessible from LabTalk script. To plot PP plot/QQ plot from script, you will need to use 

worksheet -px

For example, 

//to plot a probability plot
worksheet -px ? PPNormal plot_prob option:=0;

//to plot a Q-Q plot
worksheet -px ? QQ plot_prob option:=1;

Examples

  1. Create a new project and open the workbook which is located in <Origin Directory>\Samples\Statistics\Body.ogw.
  2. In the Male worksheet, highlight the Weight column, then select Plot: Statistics: Q-Q Plot from menu or click the Q-Q Plot button on the 2D Graphs toolbar to bring up the plot_prob dialog box.
  3. Choose the Normal in the Distribution drop-down list, check the Estimate from Data, and Method is to be Blom.
  4. Click the OK button to create the Q-Q plot.
