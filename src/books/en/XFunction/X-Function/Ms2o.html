Brief Information

Merge or move multiple matrix sheets to multiple matrix objects

Command Line Usage

1. ms2o

2. ms2o sheets:=1:3 oms:=Merge

3. ms2o sheets:="1,2" oms:=Merge

4. ms2o imp:=MBook1 sheets:="1,2,3" oms:=Merge
