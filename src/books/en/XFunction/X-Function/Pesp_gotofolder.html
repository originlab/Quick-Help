Menu Information

Right click on the file in the Favorites folder of Project Explorer: Go to Original Folder

Brief Information

Go to original folder from shortcuts window

Additional Information

X-Function not designed for Auto GetN Dialog.

Command Line Usage

pesp_gotofolder dump:=0

X-Function Execution Options

Please refer to the page for additional option switches when accessing the x-function from script

Variables

Display

Name 		I/O
and
Type 		Default
Value 		Description
dump

Input

int

0
 Go back to the original folder to where the shortcut page locates.

Description

This X-Function can be used to open the original folder that the shortcut page locates to. When the shortcut page is active, by using 
pesp_gotofolder dump:=0
in the Command window, it will go to the original folder.
