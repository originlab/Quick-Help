1.4.1.4 Image

GUI Control String

The control string is useless.The control is set to be Textbox with Interactive Image Selector button.

Variable Value Data

You can set the default data in Data field of the X-Function Bulider.

Variable Data Input Output Input/Output
<Input> useless The variable value will be put out to the Input Image. It's useless as input and the result will be put out to the active Image.
<new> no use The result will output to a new Image. The variable value is usless as input, and the result will output to a new Image.

Supportted Option String

  • A:-------Recalculate
  • G:-------Group
  • U:-------Output
  • V:-------Visible
