Brief Information

 Specify the settings for performing interpolation.

Description

This function is used to add a rectangle onto the plot to perform interpolation. The interpolation line displays in the rectangle directly, which will be updated automatically when you move the rectangle. For detailed information about quick interpolation, please refer to Interpolate Gadget in our User Guide.
