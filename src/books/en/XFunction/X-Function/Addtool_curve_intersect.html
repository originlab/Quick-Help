Gadgets:Intersect

Brief Information

Calculate intersection of two data plots within a region of interest

Additional Information

Description

This X-Function is used to add a rectangle onto the plot to perform a quick intersection. By moving this rectangle, you can observe the intersect points within this area. For detailed information, please refer to Intersect Gadget in Origin Help.
