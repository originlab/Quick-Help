Brief Information

Output Y value at a given X for all data plots in all layers of graph

Command Line Usage

1. gxy2w x:=10;

2. gxy2w x:=20 igp:=graph1 bname:=ExtractValues xname:=Length xunits:=mm;

3. gxy2w x:=20 n1:=1 n2:=0;
