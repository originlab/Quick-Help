Data: Import from File: EDF(EDF+,BDF,REC,HYP)

Examples

The following example shows you how to import the EDF data file C:\Osas2002plusQRS.edf:

Note that two new features are added to make data files more readable and manageable:

  • An annotation list that describes detailed data information of the selected files is shown in a new worksheet.
  • Intervals for each data column can be defined in the Column additional information window.
  1. Create a new project. Select Data: Import from File: EDF... from the Origin menu. This opens the EDF dialog.
  2. Go to C:\Osas2002plusQRS.edf. Then click the Add File(s) button. Make sure that the Show Options Dialog check box is checked.
  3. Click the OK button to close this dialog box. The impEDF dialog should be opened.
  4. In the File Information-osas2002plusQRS-signals branch, check all the check boxes including EDF Annotations.
  5. Click the interactive button next to Output, and then select the first column of the first sheet in Book1.
  6. Click the OK button. The selected data should be imported to the destination.
  7. A new worksheet named AnnotationLists, shown below, is added.
  8. Click in the Osas2002plusQRS worksheet. The Column additional information window is opened.
