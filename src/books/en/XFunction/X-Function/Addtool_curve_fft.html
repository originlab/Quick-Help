Brief Information

Description

This function is used to go to add a rectangle onto the plot to perform FFT. Moving this rectangle, you can observe the FFT effect of the data quickly in the FFTPREVIEW window. For more details about perform quick FFT on a plot, please refer to FFT Panel Tool in our User Guide.
