Brief Information

Calculate mean of a monotonic vector

Command Line Usage

1. vinc_check

2. vinc_check ix:=col(a)

3. vinc_check ix:=Col(A) tol:=0.1

4. vinc_check ix:=col(a) tol:=0.1 inc:=myinc
