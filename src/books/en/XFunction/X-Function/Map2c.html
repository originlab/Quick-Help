Brief Information

Combine amplitude and phase to create a complex matrix

Command Line Usage

1. map2c am:=[Mbook1]MSheet1!mat(1) pm:=[Mbook2]MSheet1!mat(1) cm:=<new> arc:=radians;

2. map2c am:=mat(1) pm:=mat(2) cm:=mat(3);
