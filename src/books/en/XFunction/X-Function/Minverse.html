Menu Information

Analysis: Mathematics: Inverse

Brief Information

Generate (pseudo) inverse of a matrix

Command Line Usage

1. minverse;

2. minverse im:=[MBook1]Msheet1!Mat(1);

3. minverse im:= [MBook1]Msheet1!Mat(1) om:=[MBook1]Msheet1!Mat(2);
