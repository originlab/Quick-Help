Menu Information

Analysis: Fitting: Simulate Curve

Brief Information

Generate curve data with specified fitting function

Additional Information

It is not accessible from script.

X-Function Execution Options

Please refer to the page for additional option switches when accessing the x-function from script

Variables

Display Name
Description

Category

Specifies a function category

Function

Specifies a fitting function

Parameter

Enter values for function parameters

X Minimum

The minimum X value

X Maximum

The maximum X value

X Points

The number of output curve points

Noise Level (%)

Add white noise to the curve

Output

The output XY range

Description

This tool can be opened from Analysis: Fitting: Simulate Curve..., or click the Simulate button on Function Organizer. It will simulate a curve which defined in the Function Organizer, and output curve data to worksheet columns.
