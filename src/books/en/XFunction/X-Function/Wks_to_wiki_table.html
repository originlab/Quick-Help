Brief Information

Output Worksheet As Wiki Table

Command Line Usage

1. wks_to_wiki_table file_path:="D:\Test\wikiTable1.txt" col:=0 row:=0;;

2. wks_to_wiki_table output_target:=1 headers:=2 col:=2 row:=5 first_col_bold:=1;
