Menu Information

Column: Set Sampling Interval

Brief Information

Set sampling interval for selected Y columns

Command Line Usage

1. colint rng:=Col(2) x0:=10 inc:=5;
2. colint rng:=Col(2) x0:=2.5 inc:=1.25 units:="s" lname:="Second";

X-Function Execution Options

Please refer to the page for additional option switches when accessing the x-function from script

Description

This X-Function is used to set Sampling Interval (Implicit X) for selected Y columns.

Examples

// This function sets internal X values for a column
// based on a start value and an increment
newbook;
colint rng:=1 x0:=-10 inc:=.01 units:=(msec) lname:=Time;

// This equivalent code uses column properties
wks.col2.xinit=-10;
wks.col2.xinc=.01;
wks.col2.XUNITS$ = (msec);
wks.col2.XNAME$ = Time;
