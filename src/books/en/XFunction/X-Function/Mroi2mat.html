Menu Information

Right-click on the ROI in the Matrix, select Create New

Brief Information

Create a new Matrix sheet with data copying from inside ROI.

Command Line Usage

1. mroi2mat quantity:=active;

2. mroi2mat -r 1 quantity:=stats trstats.n:=1;
