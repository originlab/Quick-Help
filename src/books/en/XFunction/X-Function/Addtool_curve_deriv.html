Gadgets: Differentiate

Brief Information

Perform differentiation on data plot within a region of interest

Minimum Origin Version Required: 8.5 SR0

Description

This X-Function is used to add a rectangle onto the plot to perform a quick differentiation. By moving this rectangle, you can observe the differentiation of the data plot in the Derivative graph window. For detailed information about differentiation, please refer to Differentiate tool in our User Guide.
