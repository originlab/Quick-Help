Menu Information

Custom Menu Organizer...

Brief Information

Opens a dialog box for customizing menus

Additional Information

Minimum Origin Version Required: Origin 8.1 SR0

Updated in Origin 8.6 SR0. X-Function not designed for Auto GetN Dialog.

Command Line Usage

1. customMenu;

2. customMenu imp;

X-Function Execution Options

Please refer to the page for additional option switches when accessing the x-function from script

If the default value of -1 is not specified, the Custom Menu Organizer dialog will open. Option list:

  • imp:Import
    Open the Import Menu Customization dialog.
  • exp:Export
    Reserved for later use.


Description

This X-Function is used to open the Custom Menu Organizer dialog and Import Menu Customization dialog.
