Brief Information

For internal use only

Command Line Usage

1. plotellipse;

2. plotellipse 1.0;

3. plotellipse 2.0 2.0;

4. plotellipse 0.1 0.1;

5. plotellipse a:=1.0 b:=1.0;

6. plotellipse a b NumPts XYOut; //specify all variables
