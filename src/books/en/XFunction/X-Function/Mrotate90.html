Specifies the degree and direction for the matrix rotation.
Option list:

  • ccw90:CCW90
    Rotates the matrix 90 degrees counter-clockwise.
  • ccw180:CCW180
    Rotates the matrix 180 degrees counter-clockwise.
  • cw90:CW90
    Rotates the matrix 90 degrees clockwise.
