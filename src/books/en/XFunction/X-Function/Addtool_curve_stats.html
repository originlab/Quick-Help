Description

This tool allows you to select a data range in the graph to perform the basic statistic using the ROI tool. The quantities to compute and the properties of the ROI box can be customized in the Preferences dialog, which can be open by clicking the triangle button on the right side of the ROI box. For more details about this tool, please refer to statistics panel tool in our User Guide.
