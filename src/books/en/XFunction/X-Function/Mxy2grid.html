Menu Information

Matrix: Generate Grid

Brief Information

Create 2 grid matrices with xy coordinates from input matrix

Command Line Usage

1. mxy2grid -r 2 ims:=[MBook1]MSheet1;

2. mxy2grid -r 1 x:=x+2;
