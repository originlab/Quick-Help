Menu Information

Analysis: Fitting: Simulate Surface

Brief Information

Generate surface data with specified surface function

Additional Information

It is not accessible from script. This feature is for OriginPro only.

X-Function Execution Options

Please refer to the page for additional option switches when accessing the x-function from script

Variables

Display Name
Description

Category

Show the function category

Function

Specifies a fitting function

Parameter

Enter values for function parameters

X Minimum

The minimum X value

X Maximum

The maximum X value

X Points

The number of X points

Y Minimum

The minimum Y value

Y Maximum

The maximum Y value

Y Points

The number of Y points

Noise Level (%)

Add white noise to the curve

Output

The output XYZ range

Description

This tool can only be opened by clicking the Simulate button on Function Organizer, when a function under Surface Fitting category is selected. It will simulate a surface and output XYZ data to worksheet columns.
