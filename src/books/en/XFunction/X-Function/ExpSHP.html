Menu Information

File : Export : ESRI Shapefile(SHP)

Brief Information

Export worksheet data as ESRI Shapefile

Command Line Usage

1. expSHP layout:=mxy fname:=%YMyMap.shp;

2. expSHP fname:="C:\Export\Sheet1.shp";
