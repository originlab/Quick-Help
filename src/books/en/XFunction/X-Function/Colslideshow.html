Menu Information

Column: Slide Show of Dependent Graphs

Brief Information

View slide show of graphs associated with specified columns

Additional Information

Minimum Origin Version Required: 8.1 SR0

Command Line Usage

colslideshow;

Description

This X-Function is used to view a slide show of all the graphs that depend on the selected column(s). Before running this X-Function, at least one column must be selected (highlighted) in the active worksheet.
