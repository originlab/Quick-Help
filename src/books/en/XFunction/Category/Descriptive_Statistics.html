2.13.1 X-Functions for Descriptive Statistics

Origin provides a collection of Descriptive Statistics X-Functions

Name Brief Example
colstats Perform statistics on columns
corrcoef Calculate correlation coefficients of the selected data Examples
crosstab Perform cross tabulation analysis Examples
discfreqs Calculate Frequency for discrete/categorical data
distfit Perform distribution fit.
diststats Execute distribution statistics on an XY range. Examples
freqcounts Calculate frequency counts
grubbs Perform Grubbs outlier test
kstest One sample Kolmogorov-Smirnov test for normality Examples
lillietest Lilliefors normality test Examples
mmoments Calculate moments on selected data Examples
moments Calculate moments on selected data Examples
mquantiles Calculate quantiles on selected data Examples
mstats Calculate descriptive statistics on selected data Examples
pcorrcoef Calculate partial correlation coefficients of selected data
qtest Perform a Dixon's Q-test for identification and rejection of outliers
quantiles Calculate quantiles on selected data Examples
rowquantiles Calculate quantiles on row(s) Examples
rowstats Descriptive statistics on row(s) Examples
stats Calculate descriptive statistics on selected data Examples
statsxy Statistics XY data using subgroup stats on X.
swtest Shapiro-Wilk Normality test Examples
twodbinning 2D Frequency Count/&Binning:Calculate frequencies on bivariate data
wstats Calculate descriptive statistics on whole sheet
