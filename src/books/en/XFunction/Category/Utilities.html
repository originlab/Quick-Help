2.14 Utilities


This chapter covers the following topics,and you can refer more X-Functions in these section pages:
  • X-Functions for File
  • X-Functions for System

In addition, Origin provides a collection of Utilities X-Functions

Name Brief Example
customMenu Open Custom Menu Editor Dialog.
newHTML Construct starter HTML dialog. Examples
themeApply2g Apply a theme to a graph or some graphs. Examples
themeApply2w Apply a theme to a worksheet or some worksheets. Examples
themeEdit Edit the specific theme file using the tools "Theme Editing" Examples
xop X-Function to run the operation framework based classes. Examples
get_plot_sel Get plot selections in data plot.
get_wks_sel Get selections in a worksheet. Examples
mkOPX Make an OPX file from ini settings.
newApp Create a new App.
