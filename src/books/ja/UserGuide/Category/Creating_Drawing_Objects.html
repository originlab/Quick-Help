グラフに直線または矢印を追加する

グラフに直線または矢印を追加するには
  1. 直線ツールまたは矢印ツールをクリックします。グラフウィンドウ上で、線の始点でクリックし、終点までドラッグします。 マウスボタンを離すと直線(または矢印)が表示されます。
  2. 垂直または水平の線または矢印を描画するには、線ツールまたは矢印ツールを使用してShiftキーを押しながらドラッグします。 線が垂直または水平で表示されます。 適切な向きになったらマウスボタンを離します。
Note：レイヤフレーム内に垂直線または水平線を描画する場合は、挿入：直線を追加：ダイアログを開くメニューを選択して、直線を追加：addlineダイアログを開きます。このダイアログでは、位置の値、線フォーマット、ラベルの形式、線の種類、および移動性を指定して、現在のグラフレイヤに垂直線または水平線を追加できます。
行を編集するには：
  • 線の始点または終点を調整するには、点を1回クリックすると表示される、線の開始と終了点のソリッドシンボルを使用します。シンボルにカーソルを合わせたら、ドラッグして線の開始点または終了点を変更できます。また、ポイント編集ミニツールバーボタンをクリックするか、コンテキストメニューからポイント編集を選択して、ポイント編集モードを開始できます。
  • 移動またはサイズ変更するには、ダイヤモンド型のハンドルが表示されるまで2回クリックします。ドラッグして移動するか、ハンドルをドラッグしてサイズを変更します。ハンドルをドラッグしながらCtrlキーを押すと、曲線の縦横比が維持されす。
  • ポイントを描画または移動しているときにShiftキーを押すと、線が垂直または水平にスナップします。
  • 回転させるには、丸いハンドルが周囲に表示されるまで3回クリックします。丸いハンドルをドラッグして回転させます。回転の中心を変更するには、図形の中央にあるカーソルをドラッグします。
  • 斜めにするには、三角形のハンドルが形の周囲に表示されるまで4回クリックします。三角形のハンドルをドラッグして傾斜させます。
  • 線をダブルクリックしてプロパティダイアログを開き、線種、クリック、移動、サイズ変更時の線、矢印、寸法、コードなどを編集します。

曲線矢印を描く

曲線矢印を描くには
  1. 曲線矢印ツール をクリックします。
  2. 曲線のアンカーポイントとなる位置を4つクリックします。最初のクリックで曲線の開始位置を、2回目と3回目のクリックで曲線の外形を、そして最後のクリックで矢頭位置を決めます。 このようにして、4回目のクリックをすると、曲線矢印が表示されます。 ポイントは、ベジエ(Bezier)曲線で結ばれます。
曲線矢印の編集：
  • 移動またはサイズ変更するには、ダイヤモンドハンドルが表示されるまでシングルクリックします。ドラッグして移動するか、ハンドルをドラッグしてサイズを変更します。ハンドルをドラッグしながらCtrlキーを押すと、曲線の縦横比が維持されす。
  • 回転させるには、丸いハンドルが周囲に表示されるまで2回クリックします。丸いハンドルをドラッグして回転させます。回転の中心を変更するには、図形の中央にあるカーソルをドラッグします。
  • 斜めにするには、三角形のハンドルが形の周囲に表示されるまで3回クリックします。三角形のハンドルをドラッグして傾斜させます。
  • 4回シングルクリックするか、ポイント編集ミニツールバーボタンをクリックするか、コンテキストメニューのポイント編集を選択してポイント編集モードを開始し、曲線の軌跡を調整します。
  • 曲線をダブルクリックしてプロパティダイアログを開き、線、矢印、寸法、または、クリック/移動/サイズ変更したときのコードなどを編集します。

距離計測ツールを追加する

距離計測ツールでラベルを追加するには
  1. プロット操作・オブジェクト作成ツールバーの距離計測ツールボタンをクリックします。
  2. グラフ上でクリックし、グラフ上の2点間をドラッグすると矢印が表示されます。
    • Shiftキーを押しながらドラッグすると、矢印を垂直または水平にできます。
    • 距離計測ツールは、データポイント > 軸 > 線オブジェクト（参照線を含む）にもスナップします。Ctrlキーを押すと、スナップが解除されます。
距離計測ツールのラベルを編集するには

ラベルは自動で矢印オブジェクトに接続されます。デフォルトのラベルのシンタックスは$(v,.2)で、2点間の距離を小数点以下2桁で表示します。また、2点間の水平距離（例： $(dx,.2)）や垂直距離（例：$(dy,.2))）、さらには線の傾きを表す$(dy/dx,.2)を表示することもできます。

  1. ラベルをダブルクリックすると編集モードになり、必要に応じてシンタックスを変更することができます。
  2. ラベルの位置を変更するには、クリックして選択し、青い選択ハンドルの一方または両方をドラッグします。 場合によっては、ラベルを更新するためにグラフウィンドウをリフレッシュ（標準ツールバーのリフレッシュボタンをクリック）する必要がある場合があります。


角度計測ツールを追加する

角度計測ツールでラベルを追加するには
  1. プロット操作・オブジェクト作成ツールバーの角度計測ツールボタンをクリックします。
  2. 3回シングルクリックして、セグメント、頂点、もう一方のセグメントを作成します。これにより2つのセグメントの間に円弧が表示されます。円弧の中心点に角度付きのラベルが表示されます。
角度計測ツールのラベルを編集するには
  • このオブジェクトの角度を変更するには、オブジェクトをクリックして菱形のハンドルを表示させます。ハンドルをドラッグして動かし、角度を変えます。また、角度が変わると角度ラベルも変わります。
  • ラベルは自動で円弧の中心点に接続されます。デフォルトでのラベルの構文は$(v,.2)で、これは2つのセグメント間の角度を示します。ダブルクリックして構文を変更します。 例えば、$(v, .3)は小数点以下3桁を表示し、$(v*pi/180, .2)は度をラジアンに変換して小数点以下2桁を表示します。
  • 角度計測ツールをダブルクリックしてプロパティダイアログを開き、線、矢印、寸法、円弧、または、クリック/移動/サイズ変更したときのコードなどを編集します。
  • ミニツールバーのボタンをクリックすると、円弧に矢印が表示されます。

連続直線(折れ線)を描く

連続直線を描く：
  1. 折れ線ツールをクリックします (「プロット操作・オブジェクト作成」ツールバー)。各アンカーポイントでクリックします。
  2. 折れ線ツールを終了するには、マウスでダブルクリックするか、ESCキーかEnterキーを押してます。

  • 移動またはサイズ変更するには、ダイヤモンド型のハンドルが表示されるまで2回クリックします。ドラッグして移動するか、ハンドルをドラッグしてサイズを変更します。ハンドルをドラッグしながらCtrlキーを押すと、線の縦横比が維持されます。
  • 回転させるには、丸いハンドルが周囲に表示されるまで3回クリックします。丸いハンドルをドラッグして回転させます。回転の中心を変更するには、図形の中央にあるカーソルをドラッグします。
  • 斜めにするには、三角形のハンドルが形の周囲に表示されるまで4回クリックします。三角形のハンドルをドラッグして傾斜させます。
  • 連続直線をダブルクリックしてプロパティダイアログを開き、線、矢印、寸法、塗りつぶし、または、クリック/移動/サイズ変更したときのコードなどを編集します。

  2. 1回クリックし、線を描くようにドラッグします。
  3. マウスボタンを離すと、操作が完了します。

四角形、円、多角形、自由閉曲線を描く

正方形、矩形、円、楕円、多角形、閉曲線をグラフに追加する
  1. ｢プロット操作･オブジェクト作成｣ツールバーの該当するボタンをクリックします。
  • 多角形を除く、すべてのオブジェクトは、グラフウィンドウ中の希望の位置でクリック＆ドラッグします。マウスのボタンを離すと、そのオブジェクトが表示されます。
  • 多角形の場合は、シングルクリックして多角形の角を描きます。最後の角をダブルクリックしてください。
  • 長方形/楕円形の代わりに完全な正方形または円を描くには、Ctrlキーを押しながらドラッグします。
オブジェクトを編集するには：
  • オブジェクトを移動またはサイズ変更するには、ダイヤモンドハンドルがオブジェクトの周囲に表示されるまで1回クリックします。ドラッグして移動するか、ハンドルをドラッグしてサイズを変更します。ハンドルをドラッグしながらCtrlキーを押すと、オブジェクトの縦横比が維持されす。
  • オブジェクトを回転させるには、オブジェクトの周囲に円のハンドルが表示されるまで、シングルクリックを2回クリックします。丸いハンドルをドラッグして回転させます。回転の中心を変更するには、図形の中央にあるカーソルをドラッグします。
  • オブジェクトを傾けるには、三角形のハンドルが図形の周囲に表示されるまで3回クリックします。三角形のハンドルをドラッグして傾斜させます。
  • 多角形と範囲の場合は、4回シングルクリックするか、ポイント編集ミニツールバーボタンをクリックするか、コンテキストメニューのポイント編集を選択してポイント編集モード（境界線に沿ったハンドルを取得）を開始し、さらに形状を調整できます。
  • オブジェクトをダブルクリックしてプロパティダイアログを開き、境界、塗りつぶし、位置、またはオブジェクトをクリック/移動/サイズ変更したときのコードをカスタマイズできます。

オブジェクトの編集

描画されるオブジェクトには、オブジェクトを繰り返しゆっくりクリックすることによって選択できる4つの編集モードがあります。

  • モードの順序はオブジェクトによって異なる場合がありますが、選択ハンドルの形状によって、アクティブなモードが示されます。
  • 編集オプションを無効にすると（例：グループ化の解除、プロパティのサイズタブ）、編集モードを順番にクリックすると、関連する編集モードは使用できなくなります。
モード 説明
ポイント編集
  • 正方形の選択ハンドル個別のポイントの移動
 サイズ変更
  • 菱形の選択ハンドル選択ハンドルをいずれかの許可されたサイズに動かします（例：垂直移動を無効にすると、垂直の寸法ではサイズを変更できなくなります）。
 回転
  • 丸形の選択ハンドル回転軸（例：十字線）を中心にオブジェクトを回転させます。
 ゆがみ
  • 三角形の選択ハンドルオブジェクトをゆがませることができます。

ポイント編集モードを開始するためのその他の方法

描画オブジェクト (解析ツールのROIボックスを含む) の形状を調整するポイント編集モードを開始するには、オブジェクトを4回シングルクリックするか、オブジェクトを右クリックしてポイント編集コンテキストメニューを選択するか、ポイント編集ミニツールバーボタンをクリックします。

オブジェクトをコピー&ペーストする

これは単一のオブジェクトまたは複数のオブジェクトに対してサポートされています。

オブジェクトをコピーするには
  1. ポインタツール を使ってオブジェクトをクリックまたはドラッグします。(選択ハンドルが現れます。)
  2. 複数のオブジェクトを選択するには、最初のオブジェクトをクリックしてから、CTRLキーまたはSHIFTキーを押しながら選択を続けます。あるいは、選択したいすべてのオブジェクトの周りに大きなボックスをドラッグすることもできます。
  3. Ctrl + Cを押すか、編集：コピーメニューを選択するか、選択したオブジェクトを右クリックしてコンテキストメニューのコピーを選択します。
他のウィンドウにオブジェクトを貼り付けるには
  1. コピー先のページで、オブジェクトを貼り付けたい場所でクリックします。
  2. Ctrl + Vを押す、または編集：貼り付けメニューを選択するか、右クリックしてコンテキストメニューの貼り付けを選択します。
  • 複数のオブジェクトを選択した後、オブジェクトエディタツールバーのグループツールバーボタンをクリックしてそれらをグループ化することができます。グループ化後は、単一のオブジェクトをクリックすると、グループ内のすべてのオブジェクトが選択されます。貼り付けはそれらをグループとして貼り付けます。貼り付け後にそれらを非グループ化して、各オブジェクトを別々に編集できます。
  • Origin 2018bより前のバージョンでは、貼り付けられたオブジェクトは、デフォルトで最後にペーストされたオブジェクトの背後に配置されていました。Origin 2018b以降では、貼り付けられたオブジェクトは最後に貼り付けられたオブジェクトの前に配置されます。オブジェクトを右クリックし、ショートカットメニューからオブジェクトの順序オプションの1つを選択することによって、オブジェクトの重なりの順序を変更できます。

線オブジェクトをコピーしてデータをワークシートに貼り付ける

線オブジェクト（直線、矢印、曲線矢印、折れ線、自由曲線）はどれでも、XYデータセットを生成することができます。

  1. グラフ内のオブジェクトを選択し、右クリックしてコピーを選択します。
  2. 空のワークシートセルをクリックして選択し右クリックして貼り付けを選択します。

XY座標のセットがワークシートに作成されます。

統計グラフでのグループ化を示すアスタリスクブラケットの追加

特に生命科学では、縦棒または横棒グラフを作成し、「アスタリスクブラケット」オブジェクトを使用して有意差を示すのが一般的です。ブラケットは、アスタリスクブラケットの追加ボタン（グラフにオブジェクトを追加ツールバー）を使用して、Originグラフに追加できます。

  1. アスタリスクブラケットの追加ボタンをクリックして、オブジェクトをアクティブなグラフウィンドウに追加します。
  2. オブジェクトを選択し、マウスでドラッグして再配置します。
    • 青いハンドルを使用して、ブラケットの「脚」のサイズと位置を調整します。
    • 赤いハンドルを使用して、ブラケットの「肩」に曲率を与えます。
    • ブラケット中央のテキストブロックは手動で編集して、アスタリスク（*）、p値などを追加できます。
    • 選択すると、ブラケットのミニツールバーボタンを使用して、ブラケットの向きを変えたり、線の太、括弧の種類を変更したりすることもできます。

線の終端の形状（円形、正方形、フラット）は、LabTalkのシステム変数@LCGによって制御されます。システム変数の値を変更する方法については、このFAQを参照してください。
