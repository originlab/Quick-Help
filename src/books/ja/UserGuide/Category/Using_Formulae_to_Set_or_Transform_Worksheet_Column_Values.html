この「値の設定」ダイアログボックスはワークシート列のデータまたはワークシートの複数列を作成するのに使用することができます。

「値の設定」ダイアログボックスを開くには

  1. 1列または複数列またはワークシート列内の範囲を選択します。
  2. メインメニューから、列： 列値の設定 あるいは、列： 複数列の値を設定 を選択します。

または

  1. 1列または複数列またはワークシート列内の範囲を選択します。
  2. 別の方法としては、ワークシート列を右クリックし、ショートカットメニューから「列値の設定」か「複数の列の値を設定」 を選択します。

または

  1. F(x) 列ラベルを右クリックして、ダイアログを開くを選択します。これは単一列に対してのみ利用できます。
このセクションで説明している項目
  • クィックスタート
  • 値の設定ツールのメニューとダイアログのオプション
  • 「値の設定」ダイアログボックスに数式を入力する
  • 列値の設定を行うためのユーザ定義式を作成する
  • 「LabTalkの変数と関数」ダイアログボックス
  • 「プロジェクト変数」ダイアログボックス
