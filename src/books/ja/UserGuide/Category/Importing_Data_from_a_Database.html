関連する動画はこちら:Import Data from a Database

Originは、ODBCとOLE DBによるデータベースへの接続をサポートしています。さらに、Originには、接続文字列を追加し、Origin内でSQLクエリを生成するためのシンプルなSQLエディタが含まれています。

Originのデータベース接続機能の概要については、次のOriginLabブログ投稿を参照してください。

  • Connecting to a Database and Importing Data
  • Things to Know About Origin Database Connectivity
