Originのプロジェクトファイル

Origin-Project-File

Originプロジェクトファイルは、すべての関連するデータを保存するファイル形式です。Originのセッション内で、インポートしたデータ、グラフやレイアウトウィンドウ、分析結果、結果に関する考察や分析手順のノート、内部保存したMS Excel(XLS)ファイルなどのすべてのデータにアクセスすることができます。さらに、MS Word文書ファイル、機器のデータファイル、画像ファイル、テンプレートファイルなどのOriginとは関係のないファイルをプロジェクトファイルに添付することができ、OPJファイルに添付したどんなファイルでもプロジェクトファイルを開いたときに取り出すことができます。

このセクションでは、Originプロジェクトファイルを参照、整理、保護する際に役立ついくつかの機能について説明します。

  • Windowsのファイルエクスプローラを使って、Originプロジェクトファイルのグラフをプレビューする。
  • Originのプロジェクトエクスプローラ(PE)で、Originプロジェクトファイルの内容を管理する。
  • Originプロジェクトファイルに保存されていないものと保存されていないものとデータのバックアップ方法を知る。
  • Originの検索ダイアログを使って、開いたプロジェクトファイル内の文字列または数値データを検索する。
  • Originプロジェクトファイルをパスワード保護する
このセクションで説明している項目
  • Windowsファイルエクスプローラを使用したプロジェクトのプレビュー
  • Originのプロジェクトファイルを開く、閉じる、バックアップする
  • プロジェクトエクスプローラを使ってウィンドウを管理する
  • Originプロジェクト内を検索
  • Originプロジェクトを保護する
