ノートウィンドウ

Notes-Window


ノートウィンドウは、テキストのみを入力できるOrigin子ウィンドウで、次のような場合に便利です。

  • テキスト、Originリッチテキスト、HTMLシンタックス、またはMarkDownシンタックスを使用して、情報の共有または記録に関するメモを作成します。
  • 分析手順を記録し、表やグラフを含む結果を表示します。HTMLまたはMarkDownのリンクシンタックスをレンダリングすることで、グラフをノートウィンドウに表示できます

Originのプロジェクトファイルは、数多くのノートウィンドウを持つことができます。 ノートウィンドウの内容をOriginプロジェクトファイルと一緒に保存することができますし、またノートウィンドウの内容をTXTファイルやHTMLファイル、Markdownファイルに保存することができます。

Origin 2022bから、ノートウィンドウは以下のモードをサポートします：

  • テキスト
  • Originリッチテキスト
  • HTML
  • MarkDown

HTML及びMarkdownモードでは、Originのグラフ、行列内の画像、解析テーブル、およびワークシートのセルを参照できるHTML/Markdownシンタックスを入力できます。ソースが変更されると、ノートウィンドウ内のリンクも更新されます。このHTML/Markdownモードでは、ノートウィンドウを分析レポートとして機能させることができます。
