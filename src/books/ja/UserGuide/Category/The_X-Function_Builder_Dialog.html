Xファンクションビルダダイアログ

Xファンクションビルダダイアログボックスは、Xファンクション(システムおよびユーザ定義)を作成したり、編集するのに使用します。「ツール」メニュー (「ツール：Xファンクションビルダ」)から開くことができます。

Xファンクションビルダダイアログボックスには、標準ビューとツリービューの2つのビューがあります。 標準ビューでは、Xファンクションに名前を付けたり、Xファンクションの変数を編集することができますが、ツリービューでは、より幅広いXファンクションのプロパティにアクセスすることができます。ダイアログボックスの一番上にある ツリービュー ボタン をクリックすることで、2つのビューを切り替えることができます。

このダイアログボックスには「ファイル」メニューと「ツール」メニューの2つのメニューがあります。「ファイル」メニューは、Xファンクションを作成、開く、保存するのに使用します。また、最近開いたXファンクションへのショートカットもあります。「ツール」メニューは、2つのビューを切り替えるのに使用することができ、現在のXファンクションをコードビルダで開くこともできます。ダイアログボックスツールバーは両方のビューで利用できます。
