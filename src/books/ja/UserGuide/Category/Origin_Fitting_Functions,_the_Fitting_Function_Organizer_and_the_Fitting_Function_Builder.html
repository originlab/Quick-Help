Originのフィット関数, フィット関数オーガナイザとフィット関数ビルダ

FitFunc-Organizer-Builder

関連する動画はこちら：ユーザ定義関数

Originで、すべての関数(組込みとユーザ定義の両方)は、フィット 関数オーガナイザ (FFO)にカテゴリー分けされ、統合されています。メニューからツール：フィット関数オーガナイザを選択するか、F9キーを押してFFOを開くことができます。

すべてのフィット関数は、 FDF (関数定義ファイル)で定義されます。FDFは、フィット関数について、カテゴリー情報(Origin2018以降)、フィットパラメータ、関数式、制約条件、その他の情報といったすべてのプロパティを保存します。組込み関数のFDFは、OriginのプログラムフォルダのFitFunc サブフォルダにあり、ユーザ定義関数は、ユーザファイルフォルダのFitFuncサブフォルダに保存されます。これらのFDFファイルは、メモ帳のようなテキストエディタで開いて、編集することができます。

フィット関数オーガナイザは、FDFファイルを整理したり、編集するのに便利なツールです。FDFファイルのすべてのセクションは、フィット関数オーガナイザの編集オプションに対応しています。関数を直接作成し、編集し、保存することができます。しかし、組み込み関数の場合、いくつかのセクションは編集できません。これらのオプションはフィット関数オーガナイザ内でグレーアウトしています。

ビルトイン関数の情報は、カテゴリーによる、Origin曲線フィット関数のリストをご覧ください。
