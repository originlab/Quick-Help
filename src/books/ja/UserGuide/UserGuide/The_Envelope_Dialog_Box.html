包絡線のタイプ

検索する包絡線のタイプを指定します。オプションリスト

  • 上方包絡線: 上側の包絡線のみを探します。
  • 下方包絡線: 下側の包絡線のみを探します。
  • 下方包絡線: 上と下両側の包絡線のみを探します。

スムーズポイント数

包絡線決定のためにスムージングが実行されます。この値はスムージングの次数を制御します。この値が大きければ大きいほど、曲線は滑らかになります。詳細については、包絡線アルゴリズムページをご覧下さい。
