ピークアナライザでピークをフィットする（OriginProのみ）

FitPeak-PA

 

関連する動画はこちら：Peak Fitting with Preset Peak Parameters

ピークのフィットラジオボタンがピークアナライザのスタートページのゴールグループで選択されている場合、ピークアナライザを使用して基線を作成し、入力データから基線を減算し、ピークを検索してピークにフィットを実行することができます。このモードに対応する典型的なウィザードマップは、次のとおりです：

「基線モード」ページでは基線のモードを選択して基線を作成します。それから「基線の処理」ページでは入力データから基線を減算し、「ピーク検索」ページではピークを検出します。最後に、「ピークのフィット」ページではピークと基線の非線形フィットを実行し、カスタマイズされたレポートを生成します。

ピークアナライザで実行されるフィッティングは、NLFitツールと同様にLevenberg-Marquardt（L-M）アルゴリズムを使用します。結果はレポートシートに出力されます。結果の詳細な説明は、「回帰と曲線フィット」の章の回帰結果の解釈セクションを参照してください。フィット処理を制御およびモニタリングしたい場合はピークフィットパラメータダイアログを使用し、これはピークのフィットページでフィット制御ボタンをクリックすると開くことができます。

ピークフィットの結果のほかに、ピークアナライザ はピーク属性もレポートします。ピーク属性はワークシートと結果のグラフに出力されます。ワークシートレポートはピークフィットページのレポート構成の設定ブランチでカスタマイズできます。 グラフレポートをカスタマイズするには、最初にピークフィットページのグラフ構成の設定ブランチを使用して希望の属性フィールドを選択します。選択した項目が結果グラフのラベルに表示されます。最後に、グラフ上のラベルをドラッグして希望の位置に移動します。ピーク属性の詳細についてはこちらのページを参照してください。
