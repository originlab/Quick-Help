シンボルマップとキャラクターマップ

Symbol-Character-Map

シンボルマップは、Originのインターフェイス全体で使用され、テキストオブジェクトまたはワークシートセルに特殊記号を追加します。この特殊記号は、グラフ、ワークシート、レイアウト ページ、ソートウィンドウに挿入できます。

Origin 2022b以降、新しい簡易ダイアログも利用できます。必要に応じて、長年使用されてきた詳細ダイアログを開くボタンが付いています。

簡易版シンボルマップ

Origin 2022b以降、リッチテキストが有効なワークシートセルや、セルノート、ノートウィンドウ、グラフのテキストラベル等の編集時にシンボルマップを選ぶと簡易版のシンボルマップが開きます。これは、頻繁に使用するシンボルにポイントアンドクリックですばやくアクセスできるようにするために追加された機能です。

加えて、ユーザはシンボルをすばやく追加できます (Origin 2023 以降、これには詳細シンボルマップダイアログでサポートされていないUTF-32のシンボルが含まれます)マップに1つまたは複数のシンボルを追加するための対応するコードポイントだけが必要です (ただし、以下のフォントの変更を参照してください)。

  • 挿入できるシンボルはギリシャ文字,数学欧語特殊,組み合わせ,矢印,その他　タブから選択できます。
  • 組み合わせタブは科学的表現で文字と結合されることが多い要素を挿入するために利用します。（例:”xに上線(¯)を追加する)
  • このダイアログのいずれかのタブから、もしくは詳細ダイアログから入力されたシンボルは最近使ったシンボルリストに追加されます。
  • ユーザがこのダイアログから目的のシンボルにアクセスできない場合は、(a) ダイアログの追加ボタンを使用してシンボルを追加する (次の箇条書き)、(b) 詳細ボタンをクリックして詳細なシンボルマップに移動する、(c) ワークシートのセル、テキストラベルなどに記号を直接入力するALT+X。
  • 「歯車」のアイコンをクリックすると、現在のタブにシンボルを追加したり、削除することができます。また、シンボルをドラッグすることにより、並び替えが可能なほか、工場出荷時の状態に戻したり、削除したデフォルトシンボルを再度追加することができます。

フォントの変更

  • このダイアログではフォントは変更できません。フォントはタブ内で事前に選択されています。
  • それぞれのタブで使うフォントを変える場合は、特別なデフォルトフォントダイアログを開きます。(ツール:特別なデフォルトフォント)ワークシートセルを編集する際、そのセルのアクティブフォントが使用され、一部のダイアログのシンボルがアクティブフォントで正しく表示されない場合があります（シンボルがあるはずの場所に空の四角形が表示するなど）。

シンプルマップのシンボルのユーザリストは、テキストファイル \User Files\SymbolMap.ini に保存されます。4桁の16進コードが、[Math]などのセクションごとに追加されます。「インストールされた」マップは、EXEフォルダ内の同じ名前のファイルに保持されます。工場出荷時の状態に戻すと、EXEフォルダのファイルにあるアクティブタブのセクションの内容がユーザリストに読み込まれます。

詳細シンボルマップ

Origin 2022bより前のバージョンでは標準だったシンボルマップですが、現在は「詳細」シンボルマップになっています。

簡易版シンボルマップで入力したい文字がみつからなかった場合やシンボルのコードポイントが分からない場合は、ダイアログ下の詳細ボタンをクリックして詳細シンボルマップを開くことができます。ただし、詳細シンボルマップダイアログはUTF-32をサポートしていません。

  • フォント: 各フォントには多少異なる文字があります。必要に応じてフォントを変更します。
  • Unicode: Unicode文字の方が種類が豊富です。有効にする場合、Unicodeにチェックを付けます。
  • Unicode値(16進): 16進数の4桁の文字コードがわかっている場合は、ボックスに入力して文字を選択できます。
  • エスケープ表記を使用: ここにチェックを付けてから文字を挿入すると、エスケープシーケンスを使用します。プロジェクトをOrigin 2017以前 (Unicode対応前) のユーザと共有する場合に必要です。
  • 挿入時にダイアログを閉じる: ダイアログを閉じずに複数の文字を挿入するには、このボックスのチェックを外します。
  • シンプルボタンをクリックして、簡易版シンボルマップダイアログに戻ります。

キャラクターマップ

リッチテキストが有効でないワークシートセルを編集する場合、キャラクターマップを利用できます (デフォルトでは、列ラベル行領域の単位およびユーザパラメータ行に対してのみリッチテキストが有効です)。キャラクターマップ（簡易版と詳細版の両方）はシンボルマップによく似ていますが、以下のような違いがあります。

  • 詳細キャラクターマップダイアログには、Unicodeとエスケープ表記を使用といった設定項目がありません。
  • 詳細キャラクターマップではフォントの設定ができじ、ドロップダウンリストはグレーアウトしています。特定のフォントの文字を挿入する必要がある場合は、(a) セルでリッチテキストを有効にする（セルを右クリックしてデータの書式を設定：リッチテキストを選択するなど）か、(b) セルを選択して、書式ツールバーフォントドロップダウンリストからフォントを選択し、その後、シンボルマップの詳細をクリックします。

Origin 2018から、ワークシートセルに特殊文字を入力する際にリッチテキストを有効にする必要がなくなりました。詳細は FAQ-149をご覧ください
