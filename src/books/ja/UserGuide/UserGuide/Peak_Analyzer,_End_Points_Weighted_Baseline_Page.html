ピークアナライザ, 重み付けした基線の終端ページ

PeakAnalyzer-WeightBaseline


このページはピークアナライザーの基線モードが最終ポイント重み付け に選択されているときのために追加されました。入力データの最終ポイント2つを使用してスムーズな曲線を作成します。

  • メニューの 解析: ピークと基線: ピークアナライザー: ダイアログを開くを選択します。
  • ウィンドウタイプ: ワークブック、グラフ

ダイアログテーマ

ダイアログテーマ こちらを参照してください ピークアナライザーテーマ.

ダイアログボックス内のオプション

最終ポイント(%) 基線作成に使う終点のパーセンテージを指定します。
基線のデータポイント数 基線作成に使うポイント数を指定します。入力データと同じ にチェックを入れると、基線のポイント数は入力したデータの数と同じになります。
