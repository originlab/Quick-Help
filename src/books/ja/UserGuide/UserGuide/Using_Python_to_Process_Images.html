Pythonを使用した画像の処理

Using-Python-in-Image-Processing

Originの組み込みPython機能に含まれているoriginproパッケージを使用して、イメージウィンドウの画像にアクセスして処理できます。

以下は、外部ファイルからの画像スタックをイメージウィ​​ンドウにインポートして、PythonパッケージのScikit-Imageを使用して画像の1つを反転する簡単なコードサンプルです。

import originpro as op
import numpy as np
from skimage.util import invert
#画像スタックをロード
fn = op.path('e') + r'Samples\Image Processing and Analysis\*.tif'
iw=op.new_image()
iw.from_file(fn)
print(iw.frames)
#3つの画像を取得
im2 = iw.to_np2d(2)
im2 *= 2
im2 = invert(im2)
#2番目の画像に戻す
iw.from_np2d(im2, 1)
iw.set_int('NAV',1)
