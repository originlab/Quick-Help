Xファンクションウィザード

新しいXファンクションウィザード ツールは、Xファンクションを作成する手順を示します。次の操作で開くことができます。

Xファンクションビルダダイアログの「ファイル：新しいXファンクションウィザード」を選択

または

Xファンクションビルダダイアログのツールバーの「新しいXファンクションウィザード」ボタンをクリック

Xファンクションウィザードには6つのページがあります。

入力変数の数

入力変数の数を指定します。
入力変数

すべての入力変数の名前とデータ型を指定します。
出力変数の数

出力変数の数を指定します。
出力変数

すべての出力変数の名前とデータ型を指定します。
関数の本体

このページに関数の本体を入力するか、空欄のままにしておき、後でコードビルダを使って、Xファンクションの本体を編集します。
変数のデフォルトデータ

入出力変数のデフォルト値を指定します。

詳細は、プログラミングヘルプのXファンクションの章をご覧ください。
