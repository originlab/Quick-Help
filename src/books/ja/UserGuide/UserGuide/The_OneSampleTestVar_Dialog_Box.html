入力

入力

入力データを指定します。

分散のカイ二乗検定

分散検定 検定される仮説の分散
帰無仮説 検定される帰無仮説(標本分散 = 検定分散 )
対立仮説 オプション：
分散 <>

標本の分散は仮説の分散と異なります。

分散 >

標本の分散は、仮説の分散より大きいです(上側片側検定)。

分散 <

標本の分散は、仮説の分散より小さいです(下側片側検定)。

有意水準 検定の有意水準を指定します。
信頼区間 標本分散の信頼区間を計算します。
%単位での信頼水準 信頼水準をスペース区切りで指定します(%)。このオプションは、信頼区間にチェックが付いている時のみ利用できます。

プロット

ヒストグラム 標本のヒストグラムをプロットするかどうかを指定します。
ボックスチャート 標本のボックスチャートをプロットするかどうかを指定します。
1つのグラフ中に同じタイプのプロットを配置する 同じタイプのプロットを同一グラフ内に表示するかどうか指定します。ヒストグラムではレイヤは垂直に並べられます。ボックスチャートでは水平に並べられます。

出力

プロットデータの出力 プロットされたデータを出力するワークシートを指定します（Originはプロットが作成された時に表形式データを生成します）。
結果の出力 検定結果を出力するワークシートを指定します。
