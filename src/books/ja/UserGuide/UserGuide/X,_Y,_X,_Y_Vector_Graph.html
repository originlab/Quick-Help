要求されるデータ

2つのX列と2つのY列（あるいは、2つのX列と2つのY列の部分領域）を選択する必要があります。

グラフ作成

データを選択します。

作図 > 特殊グラフ：XYXY型ベクトル を選択します。

または、

2DグラフギャラリーツールバーのXYXY型ベクトルボタンをクリックします。

テンプレート

VECTXYXY.OTP（Originのプログラムフォルダにインストールされています）

Note

ベクトルプロットは、Quiverプロットとも呼ばれます。

左端のX列とY列は、ベクトルの尾のXY座標を決定します。 2列目のX列とY列は、ベクトルの先頭のXY座標を決定します。
