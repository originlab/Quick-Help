（2D関数グラフ）のラベルタブ

PD-Dialog-2DFunc-Label-Tab

このタブでは、独立および従属変数の関数ヘッダと、関数名を設定します。このタブはテーブルで構成され、XおよびYのヘッダ情報を入力できます (例: ロングネーム、単位、コメント)。このヘッダ情報は関数グラフで表示できます。例えば、独立および従属変数のロングネームと単位はXおよびY軸のタイトルに表示でき、Yのコメントは凡例として表示できます。また、関数のショートネームで関数グラフのショートネームを指定できます。

関数グラフを作成した後でも、作図の詳細ダイアログの左パネルで関数プロットのショートネームとロングネームを変更できます。LNボタンをクリックすると、ショートネームからロングネームに切り替えできます。
