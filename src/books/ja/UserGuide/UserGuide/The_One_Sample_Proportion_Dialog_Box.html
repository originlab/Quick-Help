入力データ形式

入力データ形式
  • 要約
ダイアログボックスに手動でに入力する簡易統計を使用して検定を実行します。
  • 素データ
「成功」または「失敗」値を構成するデータセットを使用して検定を実行します。データセットには2つ以下の別個の値を含める必要があります。

入力データ

入力データ

(要約)

  • 成功数
成功の観測数
  • 標本サイズ
観測の合計数
入力データ

(素データ値)

  • サンプル
サンプルデータの範囲を定義
  • 成功
成功を示すデータセットの値

母比率検定

検定比率 検定される仮説の比率
帰無仮説 検定される帰無仮説(サンプル比率 = 検定比率 )
対立仮説 オプション：
比率 <> 検定比率
サンプル比率 = 検定比率 であるか調べるために、両側の比率検定が実行されます。
比率 > 検定比率
サンプル比率が検定比率 より大きいか調べるために、上側の比率検定が実行されます。
比率 ＜ 検定比率
サンプル比率が検定比率 より小さいか調べるために、下側の比率検定が実行されます。
有意水準 検定の有意水準を指定します。
信頼区間 真の(母)比率の信頼区間を計算します。
%単位での信頼水準 信頼区間を計算するのに使用する信頼水準を指定します。このオプションは、信頼区間にチェックが付いている時のみ利用できます。

検定手法

Originは、1群の比率の検定におけるp値と信頼区間の計算のために、2つの手法を使用します(正規近似と二項検定についての文書はアルゴリズムを確認してください)。

正規近似 正規近似は常に計算されます。一般に、成功 ≥ 10でサンプルサイズ - 成功 ≥ 10の時に、良い結果を得られます。近似の質は、サンプルサイズが増えると向上する傾向にあります。
二項検定 二項検定を使用します。

検出力解析

実際の検出力 検定の検出力を計算します（偽の時に帰無仮説を棄却する確率）。
有意水準 仮説検出力と/または実際の検出力にチェックが付いている場合にのみ利用できます。これは検出力の計算用の有意水準(またはアルファ値)です。
仮説の検出力 様々なサンプルサイズにおける検定の検出力を計算します。
仮説標本サイズ 仮説の検出力 にチェックが付いている場合のみ利用できます。検出力計算のための仮説の標本サイズを指定します。
