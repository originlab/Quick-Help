(作図の詳細)「ラベル」タブ（円グラフ）

ラベルタブ（作図の詳細）は円グラフのラベルを編集するためのものです。

フォント

円グラフのスライスにつけるラベルの、色やフォントサイズなどの書式を指定します。

フォントの色が自動に設定されている場合、フォントの色は次の方法に従うので注意してください。

  • 円/ドーナツグラフのスライス内にラベルがある場合、フォント色はそのスライスの色深度によって白または黒に設定されます。
  • スライスの外側にラベルを置くと、フォント色はスライスの色に従います。


自動の規則は、テキストオブジェクトのミニツールバーとプロパティダイアログで異なります。フォントの色が上記の2箇所で自動に設定されている場合、フォントの色は作図の詳細ダイアログの設定に従います。

フォーマット

ラベルの内容

ラベルとして何を表示するか指定します。

値を表示

ワークシートセルの値をラベルとして表示します。

パーセント

ワークシートセルの値の全体へのパーセントを表示します。
スライスタブで組み合わせ形式が選択されている場合、2番目のプロット（円、棒、ドーナツ）が追加され、すべてのスライスの組み合わせが表示されます。そしてドロップダウンリストが表示されます。

  • 合計を選択して、最初のプロットに結合されたスライスの実際のパーセンテージを表示します。
  • 個別を選択して、結合された全てのスライスのパーセンテージを表示します。2番目のプロットの全てのパーセンテージは最大100となります。
カテゴリーを表示

円のスライスごとに関連付けられたXを表示します。 結びついているX列が無ければ、行番号が使われます。

カスタム 表記法の入力するか、フライアウトリストから選択して、ラベルの表示を編集します。このボックスを選択した場合、その上のボックスは全て無効になります。また、次の表示フォーマットの設定も利用できません。

円グラフにはいくつか固有の表記があります。 

*C: カテゴリ
*v: スライスの値
*p: スライスのパーセンテージ
*rv: 半径の値           //半径の増分が無効な時は表示されません
*rp: 半径のパーセンテージ //半径の増分が無効な時は表示されません
*tv: 厚みの値      //厚みの増分が無効な時は表示されません
*tp: 厚みのパーセンテージ //厚みの増分が無効な時は表示されません

半径と厚みに関連する表記は、あらかじめ、円グラフの形状タブの半径の増分と厚みの増分を選択する必要があります。

ラベル表示フォーマット

表示フォーマットドロップダウンリストで数値フォーマットを設定します。†

  • 十進数:1000: 1, 1000, 1E6, 1E9
  • 科学:10^3: 10^0, 10^3, 10^6, 10^9
  • 工学:1k: 1.0, 1.0k, 1.0M, 1.0G
  • 十進数:1,000: 1, 1,000, 1E6, 1E9
  • 科学:1E3: 1E0, 1E3, 1E6, 1E9

小数点桁数

ラジオボタンとリストボックスを組み合わせて使用して小数点以下桁数を指定します。

有効桁数

ラジオボタンとリストボックスを組み合わせて使用して有効桁数を指定します。

† 指数表記への表記変更の契機となるしきい値は、｢オプション｣ダイアログボックス（ツール：オプション）の数値の表現形式タブ内で指定します。

表示位置

スライスの定位置に戻すのチェックボックスにチェックを付けると、円グラフのスライスラベルの位置は、放射方向と角度方向の2つで調整できます。

放射方向では、スライス外側、スライス内側、スライスの中心、円グラフの境界からラベルの位置を選択できます。

角度オフセットを使って角度方向のラベルオフセットを設定可能です。単位は半径のパーセンテージです。

角度方向では、スライスの開始位置、中心、終了位置からラベルの位置を選択できます。

割合が(%)未満の場合はラベルを非表示

この設定を使ってしきい値を設定し、設定したパーセント値より低いスライスのラベルを非表示にできます。

引出し線

独立引出し線 1つ以上の引出し線を独立してスタイルするには、このオプションを選択します。
オフセットが(%)を超える場合は引出し線を表示 データラベルとデータポイント間の距離がCritical Length,を超えている場合、リーダー線が表示されます。

有効な長さ = 指定されたパーセント値 (%) * (レイヤ高さ + レイヤ幅)/2

ラベルをドラッグしてグラフの位置を変更する場合は、スライスの定位置に戻すチェックボックスをオフにする必要があります。

引出し線をスライスまで延長(%) 引出し線をスライス内に延長するか決定し、どこまで延長するか指定します。単位は半径のパーセンテージです。
接続線 直線、2区分線、3区分線を選択します。
スタイル 実践、破線、点線、鎖線などから選択します。
幅 線の太さは、１ポイント(＝ 1/72インチ)を単位とします。
色 自動（デフォルト。ラベル色に従う）を選択するか色を指定します。
