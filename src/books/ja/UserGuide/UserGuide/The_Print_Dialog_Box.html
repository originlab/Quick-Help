「印刷」ダイアログボックス

印刷ダイアログは、印刷オプションを編集し、印刷を実行します。オプションの種類は、アクティブウィンドウタイプ(グラフ、ワークブック、Excelワークブックなど)に依存します。

内容

  • 1 『プリンタ』グループ
  • 2 『グラフを印刷する』、『印刷』、『印刷範囲』グループ
  • 3 『印刷部数』グループ
  • 4 OKボタン

『プリンタ』グループ

「プリンタ名」ドロップダウンリストからプリンターを選択します。希望のプリンタが使用可能なプリンタのリストにない場合は、Windowsのコントロールパネルからそのプリンタを追加する必要があります。

『プリンタ名』のドロップダウンリストで選択したプリンタドライバに特有なプリンタオプションを設定するには、プリンタ名の右にある｢プロパティ｣ボタンをクリックします。

選択したウィンドウをファイルに出力するには、『ファイルへ出力』チェックボックスにチェックを付けます。これはポストスクリプトファイルを作成するときに選択する必要があります。これはグラフ、関数グラフ、レイアウトページがアクティブなときに、グラフを印刷するグループのドロップダウンリストで現在を選択しているときのみ利用できます。

『グラフを印刷する』、『印刷』、『印刷範囲』グループ

  • グラフウィンドウ、関数グラフ、レイアウトページを印刷する場合
オブジェクト選択ドロップダウンリスト

プロジェクト内のグラフ、関数グラフ、レイアウトページを指定して、印刷します。ここで、現在のファイル、プロジェクト内のファイル(開いているまたはすべて)、アクティブフォルダ内のファイル(開いているまたはすべて)、アクティブフォルダとそのサブフォルダのすべてのファイルを選択できます。
オプションボタン

このボタンは、 「印刷詳細オプション」ダイアログボックスを開きます。
ポイントスキップの制御

印刷するグラフ上のワークシートデータポイント数を抑制するには、『ワークシートデータのポイントをスキップ』を選択します。このチェックボックスがチェックされると、『曲線あたりの最大ポイント数』のテキストボックスが現れます。各々のデータプロット上の印刷されるポイントの最大数を指定します。

こうすることにより、ポイントをスキップした印刷は、大量のデータセットを含むデータプロットでも高速に印刷することができます。このオプションは、印刷だけに関することなので実際のワークシートデータや画面表示には影響しません。また、このチェックボックスが選択されていると、精細な点線や網掛けを使用することはせず、プリンタのサポートするスタイルにこれらを変更します。

『ワークシートデータのポイントをスキップ』とそのテキストボックスと同様に、『行列データの最大ポイント数』とそのテキストボックスで、印刷されるグラフに表示される行列の数を制御します。
色をグレースケール化する/単色を使う

Originには、色つきグラフをモノクロ出力するオプションが2つあります。 『色をグレースケール化する』と『 単色を使う』です。 「ファイル：印刷」を選択するとき、どちらか一方のオプションが利用可能になっています。Originは、通常使うプリンタが、何色の色をサポートしているか調査して、チェックボックスのどちらかをアクティブにします。

2色だけしかサポートしていないプリンタに対しては、『色をグレースケール化する』チェックボックスがアクティブになります。

  • このチェックボックスにチェックが付いていないと、Originは、モノクロ印刷をします。この場合、軸やラベルは、それが設定されている色とは関係なく、黒で印刷されます。(Origin 6.0では、グラフの背景が黒で、軸が白に設定されていると、背景も軸も白く印刷してしまうため、出力結果が見えない状態でした。)
  • このチェックボックスを選択していると、Originは、画面で使われている色と同じ色をプリンタに送ります。

3色以上をサポートしているプリンタ(これはカラープリンタであるとは限りません。プリンタが何種類かの非ディザ濃淡をサポートしていることがあるからです。)では、『単色を使う』チェックボックスだけが利用可能です。

  • このチェックボックスにチェックが付いていないと、Originは、画面で使われている色と同じ色をプリンタに送ります。
  • このチェックボックスを選択していると、『色をグレースケール化する』で述べたように、モノクロ印刷します。
  • ワークシートまたは行列を印刷する場合
軸位置選択

『選択範囲のみ』チェックボックスを選択し、指定した領域のデータのみを印刷します。行と列の選択範囲を示すテキストボックスに希望の値を入力します。

また別の方法として、印刷ダイアログボックスを開く前に希望のワークシートや行列の(連続)範囲を選択しておきます。『選択範囲のみ』のチェックボックスを選択すると、領域範囲設定のテキストボックスに選択された領域が有効になります。
オプション

このボタンは、 オプションダイアログを開きます。
  • ノートウィンドウを印刷する場合
すべて

『すべて』のボタンを選択すると、ノートウィンドウに含まれるテキスト全てを印刷することができます。
ページ

ノートウィンドウの指定したページのみを印刷したい場合は、『ページ指定』のボタンを選択し、関連するテキストボックスに希望のページを入力します。
  • Excelワークブックがアクティブな場合、
印刷範囲

指定したページの範囲だけを印刷するには、『ページ指定』のボタンを選択し、関連するリストボックスに希望のページ範囲を入力します。
印刷対象

シート、選択範囲、ワークブック全体のいずれかをラジオボタンで選択します。

『印刷部数』グループ

｢部数｣欄に希望の印刷部数を入力するか、スピンボックスで選択します。

複数部数を丁合いしたい場合、部単位チェックボックスを選択します。

OK ボタン

このボタンをクリックすると、「印刷」ダイアログボックスの設定にしたがって印刷を開始します。
