データポイントの取得

Picking-Points

Originは、2Dグラフ上のデータポイントを取得する2つの方法を提供しています。

  • Xファンクションpickptsを使用
  • ポイントピックツールを使用する

Xファンクションpickpts でポイント取得

Xファンクションダイアログを開くには、グラフウィンドウをアクティブにして以下のコマンドを実行します。

pickpts -d;

このツールでは、データリーダーとスクリーンリーダーの2つのモードでデータポイント取得が可能です。必要に応じて出力する項目を編集できます。詳細は、このサンプルを参照してください。

データポイント取得ツールでポイント取得

Origin 2018bから、データポイント取得ツールを使用してグラフ上のデータポイントを取得できます。

2Dグラフ上のデータポイントを取得する方法

  1. グラフウィンドウをアクティブにして、データ: データポイントの取得を選択します。ポイントのピックダイアログが開きます。

    このダイアログでは、以下を確認できます。
    • 現在選択したデータポイントの実際のX/Y値
    • ピックしたポイントの合計数
    • 現在のデータポイントのデータソース情報

    • 青いクエスチョンマークをクリックして、ヒントパネルを表示し、ショートカットキー情報を見ることができます。

  2. 四角いクロスヘアでデータポイント上でダブルクリックして取得します。データポイントを取得すると、そのポイント上に丸いクロスヘアが表示されます。
  3. 完了ボタンをクリックして、次のような結果ワークブックを作成します。
