テーマプロパティのフィルタリング

ThemeRef-FilterProperties

このツールは、テーマを編集する別の手段です。カテゴリーフィルタグループまたはプロパティタイプフィルタグループのチェックボックスをつかて、テーマのプロパティグループを簡単に追加または削減することができます。

テーマプロパティのフィルタリングオプション

カテゴリーフィルター このグループには、プロパティのカテゴリを含めるか除外するのに使用できる9つのカテゴリーフィルタがあります。利用可能なカテゴリーには、テキストラベル、その他のラベル、軸スケール、軸スタイル、データプロット、背景、寸法、グローバルスタイル、その他があります。あるカテゴリーのプロパティを削除するには、対応するグループのチェックボックスを外し、フィルターの適用ボタンをクリックします。
プロパティタイプ・フィルター このグループには、テーマファイルから1つまたはいくつかのプロパティを含めるか除外するのに使用できる10個のプロパティタイプフィルタがあります。利用可能なプロパティタイプには、フォント、色、線、シンボル、塗りつぶし、サイズ、表示、境界、その他、テキストがあります。プロパティタイプを削除するには、このグループで対応するチェックボックスを外し、フィルターの適用ボタンをクリックして適用します。
プロパティの総数(サイズ) これは、テーマに保存されたオブジェクトプロパティの数を表示しています。テーマをプロパティから削除したり、追加すると数値が増減します。
プロパティ/値リスト この表には、テーマファイル内のすべてのプロパティを表示し、カテゴリーフィルタおよびプロパティタイプ・フィルタの変更に従って、自動的に更新されますここではプロパティを編集できないので注意してください。リストのデフォルトスタイルはツリービューです。 ダイアログの右にあるリストビューチェックボックスにチェックを付けると、リスト表示に変更できます。

また、ツリーのプロパティやブランチ部分で右クリックして、ショートカットメニューから削除を選択することもできます。フォルダ（ツリーノード）のアイコンを右クリックしてコンテクストメニューから追加を選択し、ツリーにプロパティ/ブランチを追加することができます。プロパティを削除または追加したあと、上記の総数が自動的に更新されます。

また、ショートカットメニューを使って選択したノードを展開または折りたたむことができます。

OK 選択された変更をテーマに適用し、ダイアログを閉じます。
キャンセル 全ての変更をキャンセルしてダイアログを閉じます。
フィルタを適用 カテゴリーフィルタ とプロパティタイプグループの変更に従って、プロパティをテーマファイルに含めるか除外します。
元に戻す デフォルトのダイアログ設定に戻します。
リストビュー このチェックボックスを選択してプロパティをリストビューで表示します。
コレクションを表示 このチェックボックスを選択して、コレクションのメンバーを持つツリーのブランチを表示します(例えば、レイヤ1, レイヤ2などを含むレイヤコレクション)。
