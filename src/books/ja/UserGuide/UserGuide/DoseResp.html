説明

パラメータpで指定される変数Hill勾配を持つ用量応答曲線

サンプル曲線

パラメータ

数：4

パラメータの名前:A1, A2, LOGx0, p

意味:A1 = 下側漸近線, A2 = 上側漸近線, LOGx0 = 中央, p = Hill傾斜

LOGx0 = -5.0 (vary), p = 1.0 (vary)

下側境界:none

上側境界:なし

スクリプトでのアクセス法
