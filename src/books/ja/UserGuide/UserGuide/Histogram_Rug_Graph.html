要求されるデータ

1つまたはそれ以上のY列（あるいは、Y列内のある範囲）を選択します。

グラフ作成

  1. データを選択します。
  2. メニューから 作図 > 統計：ヒストグラム+ラグ/分布+ラグ.

テンプレート

ヒストグラム+ラグ:HistogramRug.otp（Originのプログラムフォルダにインストールされています。） 分布+ ラグ: DensityRug.otp（Originのプログラムフォルダにインストールされています。）

ノート

  1. ラグの基本的な考え方は、データ点を軸上に投影し、それを軸の横の細い線として表現することです。
  2. このラグ表現は密度プロットではなく、各点を細い線で軸に投影するだけです。したがって、ラグ表現にはバンド幅がありません。
  3. X / Y方向の値が重複している場合は、罫線が重なります。
  4. ラグパネルをダブルクリックしてA軸 ダイアログのタブ ラグを開き、この種のグラフの詳細設定を確認できます。
