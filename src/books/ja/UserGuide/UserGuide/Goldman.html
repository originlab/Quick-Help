パラメータ

数：5

パラメータの名前:b, Nao, Nai, Ki, T

意味:

b = NaからKの比透磁率

Nao =外側のNaイオン濃度

Nai = 内側のNaイオン濃度

Ki = 内側のKイオン濃度

T = 温度 (ケルビン)

下側境界:Nao >= 0.0, Nai >= 0.0, Ki >= 0.0, T > 0.0.

上側境界: なし

Note:

ほとんどの場合、bのみを変更し、他の全ては値を指定して固定します。

スクリプトでのアクセス法

nlf_goldman(x, b, Nao, Nai, Ki, T )
