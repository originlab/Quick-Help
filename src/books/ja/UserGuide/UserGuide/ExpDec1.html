説明

一定時間パラメータの1次指数減少関数

サンプル曲線

パラメータ

数:  3

パラメータの名前: y0, A, t

意味: y0 = オフセット, A = 振幅, t = 時間定数 (崩壊定数)

下側境界: なし

上側境界: なし

派生パラメータ

減衰率: k=1/t1

半減期: thalf=t1*ln(2)

Note: 半減期は通常、慣例により記号で示されます。

スクリプトでのアクセス法

nlf_expdec1(x,y0,A,t)

FITFUNC\EXPDEC1.FDF
