要求されるデータ

ワークシートから１つのYデータかXY列範囲を選択します。

グラフ作成操作

データを選択します。

メニューから、作図：等高線図：2Dカーネル密度を選択します。

または、

2D グラフ ツールバーの2Dカーネル密度ボタンをクリックします。

plot_kde2 ダイアログボックスが開きます。
