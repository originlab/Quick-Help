「名前とコメント」ダイアログ

Name-and-Comments


名前とコメントダイアログを使用して、現在のワークシートの名前とコメントを管理できます。シートのタブを右クリックし、名前とコメントを選択するとダイアログが開きます。

名前とコメントダイアログ

ダイアログを開く

ワークシートタブ上で右クリックし、名前とコメントを選択します。

シートの名前とコメントダイアログ

このダイアログでロング/ショートネームとコメントを指定できます。表示するシート名のデフォルトオプションは、ショートネームです。このダイアログでショートネーム、ロングネーム、コメントを変更したら、CTRL+Zキーで変更を元に戻せます。
