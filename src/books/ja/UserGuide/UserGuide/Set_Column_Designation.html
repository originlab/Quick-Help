ワークシート列のプロット属性

WksCol-SetDesignation

すべてのワークシートの列は列属性（別名プロット属性）を持ちます。列属性は、プロットおよび分析操作中に列内のデータをどのように扱うかを決定します。

たとえば、デフォルトのOriginワークシートには、 "X"と "Y"の列が1つずつあります。これら2つの列に含まれるデータを選択し、（次の表で説明する理由でY列を選択する必要がある場合は、X、Y、Z：を参照）、プロットツールバーボタン（例：散布図 ボタン）をクリックします。 どの列のデータをX軸に対してプロットし、どの列をY軸に対してプロットするかを決定します。

既存のワークシートに列属性を適用するには、まずワークシート列を選択してから、

  • 右クリックして列フォーマットを選択し、プロットの属性を選択します。
  • または、右クリックして列XY属性の設定を選択し、X、Y、Z などを選択します。
  • 列 ツールバー のボタンを使うことも出来ます。


ファイルのインポート中に列指定を設定することもできます。 または、作図のセットアップダイアログボックスで作図を作成するときや、多くの析ダイアログボックス解の1つでデータ範囲を選択したとき。 または、新規をーくブックダイアログを使用してワークシートを作成するときにも設定が出来ます。

ワークシート列の属性

Originには、ワークシートの列データに適用できる9つの列指定があります。X、Y、Z、Xエラー、Yエラー、ラベル、グループ、または件名（ANOVAおよび一部の統計検定でのみ使用）、無属性。

X, Y, Z:

ワークシートの1つの列にのみX属性が割り当てられている場合、その列はワークシート内の、全てのY列(左方向と右方向)とZ列(右方向のみ)のX値として与えられます。

ワークシートの複数列 にX属性が割り当てられている場合、各X列のデータは、その右にある(ただし次のX列よりは手前の) YやZの列に対するXとなります。隣接しない列を選択する場合、選択されたX列は、選択されたY列と選択されたZ列のX値を提供します。選択範囲にX列が含まれていない場合、Originは選択されたY列またはZ列の左にある最初のX列を見つけます。何も存在しない場合、Originは選択されたY列の右側を見ます（Z列に関連付けられたX列の場合、Originは右側を見ません）。

Note： X列が指定されていないワークシートでは、等間隔のデフォルト値を作成（列：サンプリング間隔の設定）して表示（列：X列の表示）することができます。
Xエラー

1つの指定されたX列を持つワークシートでは、Xエラー列の値はワークシート内の列位置とは無関係にXエラーのサイズを設定します。

ワークシートが複数のX列を持っている場合、Xエラーバー列の値は、左にあって最も近いＸ列のデータに対する誤差データとして働きます。見つからない場合、選ばれたXエラーバー列の右にあるX列を捜します。隣接しない列を選択する場合、選択されたXエラー列は選択されたX列のXエラー値を提供します。 複数のX列が選択されている場合、Originは選択されたX列を左に見て、存在しない場合は右に見ます。選択範囲にX列が含まれていない場合、Originは選択したXエラー列の左にある最初のX列を見つけます。見つからない場合、選ばれたXエラーバー列の右にあるX列を捜します。
Yエラー

Y エラー列の値は、最初のY列のYエラーのサイズをYエラー列の左側に設定します。非隣接の列が選ばれた場合は、Yエラーバー列の左側でもっとも近いY列をその対象とします。
ラベル

ラベル列のテキストまたは数値は、その直ぐ左隣にあるY列値のデータラベルとして作図に用いられます。隣接しない列を選択する場合、選択されたラベル列は選択されたY列のデータラベルを提供します。複数のY列を選択した場合、ラベル列はそのすぐ左にあるY列と関連付けられます。

Note： この方法によるデータポイントのラベル付けは古い方法で、前後に互換性が残ります。Originの以前のバージョンでは、指定されたラベル列は作図の詳細ダイアログで別のデータセットとして表示されていました。Origin 2020bからは、指定されたラベル列は、実際のデータプロットの表示オプションとして単純に扱われ、共通のラベルタブで設定します。2Dグラフのデータラベルオプションについての詳細は、データポイントのラベル付けを参照してください。
グループ

選択された列の属性を、ANOVAツールなどの列の統計計算で使用するグループやファクターにします。
サブジェクト

選択した列を、繰り返し計測のあるANOVAで使用するためにサブジェクト列にします。
無属性

無属性の列は、ワークシートで選択されていてもプロットされません。なしと同じです。

列属性のシンボル

選択した列に指定を適用すると、括弧で囲まれた指定記号が列のショートネームの後の列見出しに追加されます。（無属性（なし）列には、列の短縮名のみが表示されることに注意してください。）


前述のように、ファイルインポートダイアログ（インポートウィザードなど）で、または新規ワークブック ダイアログで、新しいワークシートを作成するときに、属性を設定することができます。通常、これらのダイアログのリストコントロールは、2つのエラー属性と無属性の列属性を除いて、ワークシートの列見出しに表示されるものと同じ指定を使用します。

属性パターンの繰り返し

上記の指定記号を使用して、たとえばcoldesig ダイアログボックス または新規ワークブックダイアログボックスの列の構成 タブ で、一度に複数の列の指定を指定するために、ユーザー定義または既存の指定パターンを書くか、選択することができます。

かっこ内に記載されたXY属性は最後のワークシート列で繰り返されます。数値Nがかっこ内のパターンを先行するかNが1つのXY属性に先行するシンボルがある場合、そのパターンまたはXY属性がN回適用されます。例えば、(XY)を選択すると、列のXY属性は、“XYXYXY…”のような繰り返しパターンにセットされます。3(XY)と指定すると、XY属性は"XYXYXY"にセットされます。この場合、ワークシートの列数が6を超える場合、残りの列はすべてYとして指定されます（例： "XYXYXYYYYY ..."）

データインポート中の属性設定

データをインポートする際、インポートダイアログ（インポートウィザードオプションでインポートデータのXY属性を設定します。ドロップダウンリストから前もって作成したパターンを選択したり、属性を入力することができます。

データをインポートする前に新規ワークシートダイアログ （ファイル - 新規 - ワークシートメニューを選択して開く）を使用して新規ワークシートを作成する場合、同様の列属性コンボボックスを使用して新規ワークシートの列指定を設定できます。


Note: Originファイルインポートにデータコネクタを使う場合、アクティブシートの列属性は、インポートデータの属性設定に従って変更されます。OriginデータのインポートにSQLクエリを使う場合は例外で、アクティブシートの列属性の設定が保持されます。

ワークシートの属性設定

データがワークシートにインポートされ、複数列として整列された場合、列XY属性を設定する方法はいくつかあります。

「列のXY属性の設定」メニュー

単一列を選択してからその上で右クリックして、列XY属性の設定コンテキストメニューから、列のXY属性を設定できます。

複数列を選択した場合、それらの列の上で右クリックして単一のXY属性を選択するか、XY属性パターン（「XY]のような）を列XY属性の設定コンテキストメニューから選択します。次に、単一のXY属性かパターンが列が終了するまで適用されます。例えば、パターン「XY」を選択すると、列のXY属性は「XYXYXY...」となります。名前を付けて保存：カスタムコンテキストメニューから列属性のカスタマイズダイアログを開きます。

列属性ダイアログ

このダイアログで、ユーザ定義属性パターンを選択した列に設定します。

列ツールバー

列ツールバーにあるボタンを使って、選択した列に列XY属性を設定できます。

「列プロパティ」ダイアログボックス

列を選択し、そのうえで右クリックして、コンテキストメニューからプロパティ...を選択します。列プロパティダイアログのプロパティタブ が開きます。

作図と分析におけるXY属性の設定

デフォルトで、選択した元データを作図/分析した後に、前もってワークシート列でXY属性の事前設定を使います。また、インタラクティブデータ選択コントロールを使って、既存のXY属性を無視して、XY属性を設定します。

「作図のセットアップ」ダイアログのXY属性

作図のセットアップダイアログの中央のパネルで、素データとグラフの種類を選択してから、列XY属性を指定できます。ここでは、デフォルト指定モードとカスタム指定モードの2つの表示モードが使用可能です。

デフォルトの属性モードを設定すると、ワークシート列のXY属性が使われます。また、カスタム属性モードでは、列の前のチェックボックスを使って、XY属性を設定します。（X、Y、Z、ラベル、Xエラーバー、Yエラーバーなど）

詳細は、ヘルプをご覧ください。

分析ダイアログの「入力データ」選択

ほぼすべての分析ダイアログで、列XY属性を元のワークシートで設定しているかどうかに関わらず、インタラクティブデータ範囲セレクタボタンか、右向きの三角ボタンを使って、素ワークシート／グラフか入力データ要素のコンテキストメニューから列を選択することができます。

入力データの選択についての詳細は、入力データの指定をご覧ください。
