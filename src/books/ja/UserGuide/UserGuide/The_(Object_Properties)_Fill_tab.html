（オブジェクトプロパティ)塗りつぶしタブ

Ob-Prop-Fill-tab

このタブは矩形、円、多角形などの閉じたグラフオブジェクトで使用でき、塗りつぶしの色とパターンを設定できます。

塗りつぶし

塗り色

塗りつぶしの色を選択します。透明に表示するには、｢なし｣を選択します。デフォルトでは、明るい灰色で塗りつぶされます。

階調色

階調色のモードを指定します。

  • なし: 階調色による色の変更はありません。
  • 1色: 1色で黒から白に徐々に色が変化します。
  • 2色: 塗りつぶしの色が1色からもう1色へ徐々に変化します。

明るさ

これは階調色が1色になっている時のみ利用できます。 スライダーを使用して、黒から白への階調色の明るさ(％単位)を変更します。

2番目の色

これは階調色が2色になっている時のみ利用できます。 階調を作るための2色目を選択します。

透明度

これは階調色が2色になっている時のみ利用できます。デフォルトでは、塗りつぶし色とパターンの色は下の透明度の設定の透明度の値を共有します。自動のチェックを外すと階調色の透明度を個別に設定できます。

方向

これは階調色が1色または2色になっている時のみ利用できます。 塗り色がどのような方向から変化していくのかを指定します。

パターン

パターン

カラーセレクタと同様のパターンセレクタを使用してパターンを塗りつぶす方法を指定します。

独自の塗りつぶしパターンをパターンギャラリーに追加できます（ヒント: リストの最後に追加されます）。User Filesフォルダ (UFF)を開き、/Patternsフォルダを作成し、更に/Geologyと /Normalを追加します。カスタムパターンのEMF ファイルを適切なサブフォルダに保存します(例: /Patterns/Geology)。これでギャラリーにカスタムパターンが表示されます。

パターンの色

塗りつぶしパターンを選択すると、｢パターンの色｣ボタンが有効になります。このボタンをクリックすると、選択したパターンの色を設定するために使用できるカラーセレクタが開きます。

塗りつぶしパターンの太さ

塗りつぶしパターンで線の太さを指定します。 線の太さの単位は、１ポイント(＝ 1/72インチ)となります。

透明度

塗りつぶしの色を透過させます。スライダを動かすか、0（透過なし）から100（完全透過）の整数を入力します。

塗りつぶしのみ透過

透過率の設定を塗りつぶしのみに適用します。境界線の透過率は0になります。

デフォルトに設定

塗りつぶし設定を、全ての閉じたオブジェクトのデフォルトとして設定します。次回以降、閉じたオブジェクトを作成すると、この設定がデフォルトとして使用されます。

適用先


適用先ボタンをクリックして、境界と塗りつぶしの設定を、レイヤ内あるいはウィンドウ内の他のオブジェクトに適用します。

Note 1:透明度コントロールは、境界 タブのものと同じ透明度になります。

塗りつぶしの透明度％を変更すると、境界の設定にも、同じ透明度が適用されます。逆も同じです。

透明度を設定して塗りつぶしのみ透過にチェックを付けると、塗りつぶしにのみ透明度が設定され、境界の透明度は0に設定されます。

Note 2: Win98 と ME をご利用の場合、オブジェクトを含むグラフをコピー＆ペーストする際に、塗りつぶしパターンが受け側のアプリケーション（Word等）に貼り付けできない場合があります。 この場合は、次を実施して問題を修正してください。
  1. Originプロジェクトを保存して閉じます。
  2. スクリプトウィンドウ開き、以下を入力し、実行します。
    @ce=1 <Enter>
  3. Originプロジェクトファイルを再度開きます。
  4. 目的のアプリケーションにOriginのグラフをコピー/貼り付けします。

アプリケーション側でも塗りパターンが正常に表示されるはずです。
