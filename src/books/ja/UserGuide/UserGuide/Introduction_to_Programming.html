プログラミングイントロダクション

Originは、完全にプログラミングで制御可能なアプリケーションです。Originの操作に慣れてくると、自分の要望に合うようにOriginをカスタマイズしたくなるかもしれません。一般的なカスタマイズには次のことを行います。

  • 新しいインポート、分析、グラフ作成、エクスポートルーチンを作成してOriginに機能を追加する
  • データ処理操作を自動化する
  • Originで数値計算やシミュレーションを実行する

Originは2つのタイプのプログラミング環境を提供しており、それはLabTalkスクリプト言語とより高度なOrigin Cプログラミング言語です
