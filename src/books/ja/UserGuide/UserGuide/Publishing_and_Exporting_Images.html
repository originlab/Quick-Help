画像の印刷とエクスポート

Image-Publish-Export-Images

画像をイメージウィンドウからエクスポートする

  1. メニューのファイル：イメージのエクスポートを選択します。これによりCvExportダイアログが開きます。
  2. ファイルの種類を選択し、ファイル名とファイルパスを指定します。
  3. ファイルの種類 = AVIのとき、コーデックを選択します。
  4. OKボタンをクリックして画像やビデオをエクスポートします。

画像を行列ウィンドウからエクスポートする

  1. メニューのファイル：エクスポート：イメージを選択します。
  2. imageExpダイアログで、ファイルの種類ドロップダウンリストからファイル形式を選び、ファイル名を指定します。オプションダイアログを表示するにチェックを入れると、更にオプションを指定できます。保存ボタンをクリックします。
  3. オプションダイアログを表示するにチェックを入れた場合、expImageダイアログが開きます。エクスポートオプションを設定し、OKをクリックすると画像がエクスポートされます。

サポートされている画像の種類

次の形式で画像を保存することができます。

画像の種類 拡張子 ウィンドウの種類
ビットマップ *.bmp イメージ・行列
Graphics Interchange Format *.gif イメージ・行列
Joint Photographic Experts Group *.jpg, *.jpe, *.jpeg イメージ・行列
Zsoft PC Paintbrush Bitmap *.pcx 行列
Portable Network Graphics *.png イメージ・行列
Truevision Targa *.tga 行列
Adobe Photoshop *.psd 行列
Tag Image File *.tif, *tiff イメージ・行列
GeoTIFF （グレースケールGeoTIFFをデータとしてインポートにチェックを付ける） *.tif イメージ・行列
Audio Video Interleave *.avi イメージ

行列をAVIとしてエクスポートするには、
ファイル: エクスポート: ビデオを選択します。
