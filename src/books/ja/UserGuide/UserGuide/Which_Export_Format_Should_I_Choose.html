ファイル形式の選択の目安

ExpGraph-Format-Comprison

ターゲットアプリケーションによって選択する形式が異なります。 多くの場合、出版業者等で画像ファイル形式に対しる要件を設けているため、画像を作成する前に詳細を確認してください。

印刷出版に共通のフォーマット

拡張子 フォーマット名 特徴

TIF

Tag Image File

ラスター形式、ColorSpaceオプション、モノクロームからTrue Color、ファイル圧縮。

TGA

Truevision Targa

ラスター形式、256色からTrue Color。

EPS

Encapsulated Postscript

ベクター形式、 ColorSpaceオプション、True Color。

PDF

Portable Document Format

ベクター形式、 ColorSpaceオプション、True Color。

一般的に、スケーラビリティの点でベクター形式が好まれます。 ベクトル画像が無限にスケーラブルであることで合理的と感じるかもしれませんが、データセットが非常に大きい場合、画像を記述するために必要な情報はもかなり大きくなる可能性があり、注意が必要です。 また、ColorSpaceオプションがあるフォーマットでは、通常RGBよりもCYMKがよく使用されます。

Webパブリッシングに共通のフォーマット

Webで使用する画像を作成する際には、重要な考慮事項が2つあります。 以下の2点です。

  • ダウンロード速度を上げるには、ファイルサイズを最小化します。
  • 一般的に使用されているブラウザでサポートされている形式を選択します。

最も広くサポートされているWeb画像形式は、JPGとGIFの2つです。 PNGはライセンスが必要ないので、GIFの代替として一般的になっています。

拡張子 フォーマット名 特徴

JPG

Joint Photographic Experts Group

ラスター形式、グレースケールまたはTrue Color、調整可能な圧縮 (非可逆圧縮)。

GIF

Graphics Image Format

ラスター形式、256色、可逆圧縮。

PNG

Portable Network Graphics

ラスター形式、モノクローム、16または256色、True Color (24 bit RGB)、可逆圧縮。

SVG

Scalable Vector Graphics

ベクター形式、スケーラブル、XMLフォーマット

ダウンロード速度に対して色とディテールのバランスを確認してください。色密度および解像度が低いほど、ダウンロード速度は速くなります。 ただし、ダウンロード速度が遅くなることは考慮した上で色やシャープネスを高る場合もあります。

Webページはコンピュータの画面で表示されるので、モニターに近い解像度-72～120 dpi でエクスポートします。

ラスターまたはベクター形式のフォーマットです。 ベクター形式のエディタはあまりなく、一部のプログラムではエディタに読み込んだときにベクター形式の画像を「ラスター化」します。 詳細については、各アプリケーションのドキュメントを参照してください。
