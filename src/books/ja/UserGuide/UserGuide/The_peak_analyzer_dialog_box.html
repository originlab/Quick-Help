ピークアナライザダイアログボックス

PeakAnalyzer-Dialog


ピークアナライザダイアログボックスには、上側パネルと下側パネルの2つのパネルがあります(下図参照)。

上側パネルにはダイアログテーマのコントロールとウィザードマップがあります。テーマは、既存のテーマから設定をロードしたり、今後使用するために現在の設定をテーマファイルに保存することができます。一方で、ウィザードマップは、ウィザードの別のページに移動するために使用することができます。ウィザードマップ上にあるアイコンをクリックしてどのページにでも移動できます。ウィザードマップは色分けされており、現在のページ(緑色)、設定していないページ(黄色)、設定済みのページ(オレンジ色)、最後のウィザードページ(赤色)を表しています。

ピークアナライザの下側のパネルは、ウィザードの各ページで分析オプションを設定するのに使用します。ピークアナライザは、実際にはいくつかのXファンクションを組み込んだウィザードです。アクティブページにより異なるXファンクションが下側パネルに表示されます。パネル内のコントロールを使って、計算オプションを選択することができます。

パネルの間にいくつかボタンがあります。戻ると進むボタンは、ウィザードのページを移動するのに使用します。完了ボタンは、そのページより後のページをすべてスキップし、現在のテーマファイルでの設定で分析を実行します。一方、キャンセルボタンは、分析を実行せずにダイアログを閉じます。上側パネルの表示/非表示ボタンは、上側のパネルの表示と非表示を切り替え、 Xファンクションパネルの表示/非表示ボタンは、Xファンクションパネルの表示と非表示を切り替えるのに使用します。
