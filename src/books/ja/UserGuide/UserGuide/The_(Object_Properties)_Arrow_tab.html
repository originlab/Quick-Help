(オブジェクトプロパティ)矢印タブ

この矢印タブで、矢印の始点と終点の表示を編集します。

両方に矢先を選択する場合、同じ矢印の大きさチェックボックスにチェックを付けると、始点 と 終点 グループのどちらかで設定された幅 と 長さが両方の矢先に適用されます。

ポリライン オブジェクトとして個々のセグメントに矢印を追加するには、矢印の先端を選択する際に、各セグメントで矢印を表示 にチェックを入れます。

この デフォルトに設定 のチェックボックスにチェックを入れると、このタブで選択した内容が、以後作成される同じタイプのオブジェクト全てに適用されます。

Note: 透明度コントロールは、線 タブ と同様に、矢印の線と先端に同じ透過度を設定します。

線の透明度％を変更すると、矢印の先端にも同じ透明度が適用されます。逆も同じです。
