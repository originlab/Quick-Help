Xファンクションの技術

Xファンクションのコアコンポーネントは、指定した処理を実行するためにデザインされたOriginC関数です。Origin Cコードに加えて、Xファンクションには、名前、説明、入力および出力変数のデフォルト値、Xファンクションをメニューから実行すべきかどうかとその場合のウィンドウの種類、Xファンクションをスクリプトから実行すべきかどうかとその場合のシンタックスのヘルプなどの情報も含みます。

Xファンクションを使ったツールを作成する主なメリットとしては、ツールを作成する人がツールのGUIについて心配する必要がないということです。さまざまな入出力変数やそのデフォルト値、実際のデータ処理を行うOrigin Cコードなどの関数の詳細に集中するだけです。一度、Xファンクションを定義して、プログラムしてしまえば、Originは、自動的にプログラミング環境だけでなく、ユーザインターフェースからでもXファンクションを実行する手段をエンドユーザに提供します。

エンドユーザがXファンクションにアクセスできる別の方法には次のものがあります。

  • Xファンクションは、統一した編集オプションを使用して、すべての機能を利用できるようなダイアログボックスとして表示できます。Xファンクションを定義すれば、関数が起動するたびに、ダイアログボックスが自動生成されます。
  • Xファンクションは、解析メニューや、統計メニューなどのようにOriginメニューからアクセスでき、またユーザが作成したカスタムメニューに追加することもできます。
  • Xファンクションは、LabTalkスクリプトからプログラムでアクセスすることができ、コマンドウィンドウやラベルボタンなどから実行することができます。
  • Xファンクションのダイアログボックスは、ワークブック、行列、グラフウィンドウの一部として、特殊なXファンクションバーパネルで提供することができます。

OriginメニューからXファンクションにアクセスする

Origin 8の多くのメニューコマンド、特にデータ分析に使うメニューはXファンクションを元にしています。メニューコマンドのXファンクションを起動すると、Xファンクションがインタラクティブなダイアログボックスに表示され、入力データや実行パラメータを柔軟に変更することができ、結果/出力についても指定することができます。

いくつかのOriginメニュー -- 例えば 解析 や 統計 メニュー --は、自動的に利用可能なXファンクションツールを表示します。Originを実行するたびに、Originプログラムフォルダの \X-Functionサブフォルダとユーザファイルフォルダがスキャンされ、これらのメニューがその領域で見つかったXファンクションを表示します。各Xファンクションは、関数のパラメータに対するカスタム設定を含むテーマファイルと結びつけることができます。ツールで実行される多くのXファンクションはフライアウトメニューの機能を提供し、事前に保存したテーマから特定のテーマを選択することができます。下の図は、解析：信号処理：スムージングと選択して開くフライアウトメニューで、2つのテーマ<前回どおり> と、MySmooth が表示されています。

Xファンクションの作成者は、Xファンクション定義ファイルにフラグをセットシテ、そのフラグでXファンクションがメニューからアクセス可能か、さらにテーマを持つフライアウトサブメニューを表示するかどうか制御することができます。

グラフの挿入ショートカットメニューがXファンクションダイアログボックスを開き、ワークシートセルの中にOriginグラフを埋め込むことができます。

Originメニューは、プログラムでコントロールでき、ユーザ定義のXファンクションを呼び出すメニュー項目を作成することができます。プログラムでメニューをカスタマイズする方法の詳細については、Originlabのホームページのプログラミングに関するページを参照して下さい。

スクリプトからXファンクションにアクセスする

OriginのシステムXファンクションは、スクリプトプログラミング環境からアクセスできます。例えば、 (1) コマンドウィンドウ、 (2) OGS スクリプトファイルのセクション、 (3) ラベルボタンのようなオブジェクトに保存されたスクリプトXファンクションがLabTalkスクリプトから呼び出すことができるかどうかは、Xファンクションツリーの「使用のコンテキスト」ノードにある「LabTalk」チェックボックスの状態で決まります。「使用のコンテキスト」についての詳細は、Xファンクションビルダダイアログボックスのヘルプをご覧下さい。

新しいコマンドウィンドウには、スクリプトからXファンクションにアクセスするユーザインターフェースがあり、オートコンプリートおよびコマンド履歴のような機能もあります。詳細は、コマンドウィンドウの章を参照してください。

詳細は、プログラミングヘルプスクリプトからXファンクションにアクセスするにあります。
