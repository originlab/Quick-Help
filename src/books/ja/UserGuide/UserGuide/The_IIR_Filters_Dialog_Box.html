  • ウィンドウタイプ: ワークブック、グラフ
  • XファンクションスクリプトおよびGUIサンプル: dfilter(Pro)
ダイアログテーマ ダイアログテーマを参照してください。
再計算

分析結果の再計算を確認してください。
入力信号

入力信号を指定します。入力データの指定を参照してください。
応答タイプ
  • ローパス: 低い周波数成分を通します。
  • ハイパス: 高い周波数成分を通します。
  • バンドパス: 指定した範囲内の周波数成分を通します。
  • バンドストップ: 指定した範囲外の周波数成分を通します。
手法

フィルタのデザイン法を指定します。

  • Butterworth
  • 第一種Chebyshev
  • 第二種Chebyshev
  • 楕円
フィルタの次数

最小のチェックボックスにチェックを付けていると、Originは次数を計算します。チェックがついていない場合、自然数をフィルタの次数として指定します。
周波数仕様

IIRフィルタをデザインするうえでの周波数仕様を指定します。

  • ユニット
    • Hz: Hzを単位として使用して周波数仕様を指定します。
    • (0, 1)に標準化: 周波数は0から1の間の範囲で標準化されます。周波数(Hz)/(サンプル周波数(Hz)/2) で計算できます。ここで、サンプル周波数(Hz)/2 はナイキスト周波数です。
  • サンプル周波数: 信号のサンプル周波数を指定します（ユニットがHzのとき）。デフォルトで、Originはサンプルの周波数を対応するX列から検出します(このような列がある場合)。
  • パス、ストップ、カットオフ周波数。応答タイプ、手法、ユニットによって異なります。
振幅仕様(by dB)

振幅仕様について指定します。単位はdBです。

  • パスバンドリップル: パスバンドリップルをdBで指定します。
  • ストップバンド減衰: ストップバンド減衰をdBで指定します。
フォワード-バックフォワード フィルタリング

入力信号にフォワードとバックワードのフィルタリング両方を実行します。
出力結果

デザインしたフィルタやフィルタをかけた信号を含めて出力する結果を指定します。出力結果を参照してください。

  • SOS行列: デザインしたフィルタのSOS (Second Order Section)行列を出力します。
  • 零点・極・ゲイン: デザインしたフィルタの零点・極・ゲインを出力します。
  • 状態空間フォーム: デザインしたフィルタの状態空間フォームを出力します。
  • 係数: デザインしたフィルタの変換関数の係数を出力します。
  • 出力信号: デザインしたIIRフィルタを使用して出力した信号です。

プレビューパネル

フィルタ仕様
応答タイプ、手法、フィルタ次数、ユニットを変更すると、このプレビューはそれに応じて変化します。これにより、IIRフィルタのフィルタ仕様を簡単に見る事ができます。

フィルタを適用:
このプレビュータブはデザインしたIIRフィルタを元の信号に使った後の様子を示します。上のプロットには、元の信号とフィルタリングした信号が表示されます。下のプロットは度数分布を表示します。また、下のプロットで手動で垂線をドラッグすると、フィルタの周波数を手動で指定する事もできます。
