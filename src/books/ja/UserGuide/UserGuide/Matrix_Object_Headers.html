行列オブジェクトのヘッダ

行列オブジェクトのヘッダは、行列次数/ラベルの設定ダイアログ(メニューから行列：次数/ラベルを選択)で設定することができます。Zラベルタブは、ロングネーム、単位、コメントなどの行列ヘッダを入力することができる2Dグリッドの表です。ロングネームのようなこれらのヘッダは、行列サムネールに表示されるか、3Dプロット内のZ軸ラベルとして使われます。

 

行列オブジェクトラベルを編集するとき、セルを右クリックして次のオプションを開きます。

値を右方向に適用 現在の値を右方向のすべてのセルに適用
値を左方向に適用 現在の値を左方向のすべてのセルに適用
システムパラメータを表示する すべてのシステムパラメータを表示する
ユーザ定義パラメータを追加する ユーザ定義パラメータを追加します。
削除 ユーザ定義パラメータを削除します。
アクティブオブジェクト列に行く アクティブな行列オブジェクトに対応するこの表内の列に行きます。
