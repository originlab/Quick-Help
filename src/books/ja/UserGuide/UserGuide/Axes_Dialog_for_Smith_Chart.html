スミスチャートの軸ダイアログ

smith-axis-dlg

軸ダイアログはスミスチャートの軸と目盛を編集します。

目次

  1. 1 スケール
  2. 2 目盛ラベル
  3. 3 タイトル
  4. 4 グリッド
  5. 5 線

スケール

このタブでは、X 軸及びR軸に付けられる主目盛 と 副目盛 を制御します。

目盛ラベル

表示チェックボックスは特定の目盛ラベルを表示するかどうか指定します。

表示 このタブでは、目盛ラベル表示のコントロールか利用可能です。これは、一般的な2Dグラフの軸ダイアログの目盛ラベル表示と設定と同じですが、タイプ と 数式のオプションはありません。
フォーマット このタブは目盛ラベルフォーマットについて制御します。これは、一般的な2Dグラフの軸ダイアログの目盛ラベルフォーマットと設定と同じですが、テキストの折り返し と 位置合わせのオプションはありません。

タイトル

このタブでは、軸タイトルの設定を行います。

表示チェックボックスは、特定の軸タイトルを表示するかどうか指定します。

他の項目は一般的な2D軸ダイアログのタイトルタブと同じです。

グリッド

このページでは、X軸またはR軸のグリッド線の編集オプションを提供します。

主グリッドと副グリッド線はそれぞれ主グリッド線と副グリッド線のツリーノードでカスタム化できます。これら2つのツリーノードは対応する一般的な2D軸ダイアログの グリッド線タブと同じです。

折れ線

このタブでは、角度軸の線と軸目盛のスタイルを編集します。

表示

軸を表示するかどうかを指定できます。
色

ドロップダウンリストから線の色を選択します。
太さ

このコンビネーションボックスで、線の太さを入力または選択します。
