概要

ラーニングセンター ダイアログは、Originを使い始めのユーザに最適です。Originを起動すると開きます。

ラーニングセンターを開く
Originメニューからヘルプ：ラーニングセンター を選択します。
または、
キーボードの F11 キーを押します。

使用方法

サンプルグラフまたはサンプルプロジェクトを開く

グラフサンプル または サンプルプロジェクト を 開く セクションから選択し、Origin のサンプルを閲覧します。

沢山のグラフや分析のサンプルがあるので、リストボックスから、グラフサンプルまたはサンプルプロジェクトのサムネイルを選んでダブルクリックします。関係するフォルダが説明付きで開きます。データを入れ替えて、グラフや解析がどのようになるかを確認出来ます。

検索ボックスにキーワードを入力してグラフサンプルを検索できます。

Getting Started

ラーニング・リソース では、使い始めのユーザ向けに、操作チュートリアルや動画を提供しています。

ユーザファイルの転送

ダイアログ下側のユーザファイルの転送 ボタンを押すと、過去のUser Filesフォルダ を新規のフォルダに移動することが出来ます。
