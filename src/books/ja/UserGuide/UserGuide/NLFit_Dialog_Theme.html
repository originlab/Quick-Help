NLFitダイアログテーマ

NLFit-Dialog-Theme

ほとんどのOriginの分析ダイアログボックスと同様、NLFit ダイアログで設定した内容をテーマファイルとして保存することができます。保存した設定を呼び出すためには、ダイアログボックスのテーマをロードします。

NLFit ダイアログの上部にダイアログテーマのコントロールがあります。ダイアログテーマの項目の右のテキストボックスに、現在のダイアログテーマ名が表示されます。ここにテキストが表示されていない場合、テーマが使用されていません。テキストボックスの右にある三角形のボタンをクリックするとショートカットメニューが開き、ダイアログテーマの保存、ロード、削除を行ったり、設定をリセットできます。

<デフォルト>に保存

現在のテーマをNLFitダイアログのデフォルトテーマとして保存します。

保存

現在のテーマを保存します。

名前を付けて保存

現在のテーマを別の名前で保存します。選択するとダイアログボックスが開き、テーマ名を付けたり、テーマに保存する要素を選択できます。

テーマ設定

テーマに保存する設定項目を選択します(名前を付けて保存も同じ選択を使用できます)。詳細は、こちらをご覧下さい

システムデフォルト

NLFitダイアログのシステムデフォルトをロードします。

削除

ダイアログボックステーマを削除します。

比較

現在の設定と選択したテーマ名の設定を比較します。

<前回どおり>

最近実行した分析の設定をロードします。

Note:ダイアログボックステーマについての詳細は、このページを参照してください。
