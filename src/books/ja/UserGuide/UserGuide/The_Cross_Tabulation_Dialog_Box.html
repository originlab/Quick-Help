再計算

再計算

分析結果の再計算の設定を変更します。

  • なし
  • 自動
  • 手動

詳細情報は、 分析結果の再計算をご覧下さい。

入力データ

統計: 記述統計: クロス集計とカイ二乗 を選択します。

行 列からデータを選択して、行変数を指定します。
縦棒 列からデータを選択して、列変数を指定します。
レイヤ 列からデータを選択して、レイヤ変数を指定します。レイヤ変数を指定すると、行と列変数のクロス集計がレイヤ変数の各レベルで作成されます。
度数 度数データモードとしてデータを集計する場合、度数の列を選択して、度数の変数を指定します。

統計

分割表 このブランチは、どの統計で計算して分割表に出力を決定します。
  • 度数：度数、期待度数
  • パーセント：行度数、列度数のパーセント、総度数のパーセント
  • 残差：残差、標準残差、調整済残差

Note:これらの統計についての詳細は、イントロダクションとアルゴリズムのページをご覧ください。

有意レベル 以下の仮説検定には、有意水準を指定します。デフォルトの値は0.05です。

検定

カイ二乗検定 カイ二乗検定を行うかどうかを指定して、分割表内の変数間で有意な関連があるか決定します。

Note: これらの統計についての詳細は、イントロダクション、アルゴリズム、結果の解釈のページをご覧ください。

フィッシャーの正確確率検定 フィッシャーの正確確率検定を行うかどうかを指定して、分割表内の変数間で有意な関連があるか決定します。

Note: これらの統計についての詳細は、イントロダクション、アルゴリズム、結果の解釈のページをご覧ください。

関連性の尺度 このブランチは、どの統計で計算して、分割表の変数間の関連の強さを評価をするのかを決定します。このブランチは以下のチェックボックスを含みます。
  • ガンマ
  • ケンドールのタウbタウc
  • Phi
  • クラマーのV
  • C係数
  • Somer's D
  • ラムダ
  • 不確かさの係数

Note: これらの統計についての詳細は、イントロダクション、アルゴリズム、結果の解釈のページをご覧ください。

一致 このブランチは、どの検定を実行して、一致対データの一致を計測して検定するのかを決定します。このブランチは以下のチェックボックスを含みます。
  • カッパ
  • Bowker検定(Mcnemar検定[2*2])

Note: これらの統計についての詳細は、イントロダクション、アルゴリズム、結果の解釈のページをご覧ください。

他の測定 他の検定を実行するかどうか、および、他の統計を計算するかどうかを指定します。このブランチは以下のチェックボックスを含みます。
  • Cochran-Mantel-Haenszel
    条件付独立性検定、オッズ比の均一性検定、共通オッズ比を含む、複数分割表の関係を分析するかどうかを指定します。
  • 相対危険度：
    相対危険度を計算して、比較グループでイベントが発生するオッズに対する1つのグループでイベントが発生するオッズの割合を計測するかどうかを指定します。
  • オッズ比：
    オッズ比を計算して、イベントが発生するオッズに対するイベントまたは結果が発生するオッズの割合を計測するかどうかを指定します。

Note: これらの統計についての詳細は、イントロダクション、アルゴリズム、結果の解釈のページをご覧ください。

出力

結果ログの出力 結果ログを表示するかどうかを指定します。
モザイクプロット モザイクプロットを表示するかどうかを指定します。モザイクプロットのそれぞれの矩形の面積は、X変数の各レベルにおけるY変数の割合に比例します。
クロスタブレポート クロスタブ結果のためのシートを指定します。デフォルトは入力データがあるワークブックに新しいシートを作成して表示します。
プロットデータ モザイクプロットデータのためのシートを指定します。デフォルトは入力データがあるワークブックに新しいシートを作成して表示します。
