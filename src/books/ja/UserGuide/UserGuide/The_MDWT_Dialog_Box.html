ダイアログテーマ ダイアログテーマを参照してください。
再計算

分析結果の再計算を確認してください。
入力

入力信号を指定します。入力データの指定を参照してください。
ウェーブレットの種類

ウェーブレットの種類を指定します。

  • Haar: Haarウェーブレット
  • DBN: Daubechiesウェーブレット(N = ウェーブレット次数およびバニシングモーメントの数)
  • BiorNr.Nd: Biorthogonal (Nr = ウェーブレット次数または再構成のスケーリング関数; Nd = 分解の関数の次数)
拡張モード

終端拡張処理のモードを指定します。

  • 周期的: 入力信号を周期的と見なします。
  • ゼロパディング: 入力範囲の外側のデータポイントを0と見なします。
分解レベル

入力信号を分解するレベルの数を指定します。
複数係数データ

結果のレポートデータを指定します。出力結果を参照してください。
