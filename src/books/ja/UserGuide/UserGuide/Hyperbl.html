説明

双曲線関数さらに、酵素反応速度論のMichaelis-Mentenモデル

サンプル曲線

パラメータ

数：2

パラメータの名前:P1, P2

意味:P1 = 垂直漸近線, P2 = 水平漸近線

下側境界:なし

上側境界:なし

スクリプトでのアクセス法

nlf_hyperbl(x,P1,P2)

関数定義ファイル名

FITFUNC\HYPERBL.FDF
