概要

統計ガジェット (addtool_curve_stats Xファンクション)を使って、四角形で選択した関心領域(ROI)のデータについて、基本統計情報を得ることができます。

このダイアログで、次のことができます。

  • ROIの上に結果を表示
  • ROI内の平均と、n番目の標準偏差ラインを表示
  • ROIが移動した場合、結果を更新
  • グラフレイヤの中にあるプロットから、必要なデータプロットを簡単に選択
統計ガジェットを使うには

ツールを使うには、グラフがアクティブなときに、メニューからガジェット：統計を選びます。

平均ラインを表示する (meanline) 四角形に囲まれたデータの平均値線を表示するかどうかを指定します。デフォルトの設定は1です。

色 (color)
このドロップダウンリストで線の色を選択します。色の選択については、 データプロットの色を編集する をご参照ください。
標準偏差ラインを表示 (sdline) 四角形で囲われた標準偏差ラインを表示するかどうかを指定します。デフォルトの設定は0です（チェックなし）。
SDの倍数 (nSD)
標準偏差の線を引くための標準偏差の値の係数を指定します。
スタイル (style)
このドロップダウンリストで線のスタイルを選択します。
幅 (width)
このコンビネーションボックスで線の太さを入力または選択します。
色 (sdcolor)
標準偏差線の色を指定します。色の選択については、 データプロットの色を編集する をご参照ください。
最小/最大の線(極値）を表示 (extremumline) 四角で覆われたデータの平均ラインを表示するかどうかを指定します。デフォルトの設定は0です（チェックなし）。
スタイル (style)
このドロップダウンリストで線のスタイルを選択します。
幅 (width)
このコンビネーションボックスで線の太さを入力または選択します。
色 (color)
境界線の色を指定します。色の選択については、 データプロットの色を編集する をご参照ください。
値 (valForm)
ドロップダウンリストから、最大値および最小値ラベルの数量を指定します。：<X,Y>ではXの値とYの値両方をラベルに使用し、 Y ではＹの値のみをラベルに使用します。
線の長さ (linelength) 平均およびSD線の長さを指定します。以下のオプションがあります。
  • ROIボックス内部(0): ROI矩形内に直線を表示します。
  • レイヤまでの長さ(1): 拡大して、レイヤ全体に直線を表示します。
  • カスタム(2): 開始と終了で指定した範囲内に直線を表示します。
線ラベル位置 線のラベルの位置を指定します。以下のオプションがあります。
  • なし (0): 線を非表示にします。
  • 内部左 (1): 四角形内の左側に線ラベルを表示します。
  • 内部右 (2): 四角形内の右側に線ラベルを表示します。
  • 外部左 (3): 四角形外の左側に線ラベルを表示します。
  • 外部右 (4): 四角形外の右側に線ラベルを表示します。
ラベル形式：カスタム
  • 値 (0): それぞれの値で、平均と標準偏差をタグ
  • ラベル (1): それぞれのラベルで、平均と標準偏差をタグ
  • 両方 (2): 値とラベルの両方でタグ

ROIボックスタブの上中央に表示

平均 (mean) ROIの上部に、選択したデータポイントの平均値を表示するかどうか指定します。デフォルトの設定はチェックあり(1)です。
SD (sd) ROIの上部に、選択したデータポイントの標準偏差を表示するかどうか指定します。デフォルトの設定はチェックあり(1)です。
最小 (min) ROIの上部に、選択したデータポイントの最小値を表示するかどうか指定します。デフォルトの設定はチェックなし(0)です。
最小のX (xmin) ROIの上部に、選択したデータポイントの合計を表示するかどうか指定します。デフォルトの設定はチェックなし(0)です。
最大 (max) ROIの上部に、選択したデータポイントの最小値を表示するかどうか指定します。デフォルトの設定はチェックなし(0)です。
最大のX　(xmax) ROIの上部に、選択したデータポイントの合計を表示するかどうか指定します。デフォルトの設定はチェックなし(0)です。
合計 (sum) ROIの上部に、選択したデータポイントの合計を表示するかどうか指定します。デフォルトの設定はチェックなし(0)です。
カスタムラベル

このボックスにチェックを付けるとテキストボックスでカスタムラベルを定義できます。

  • Origin 2022b 以降のバージョンで可能です。
  • 以下のように、(a) リテラルテキスト、(b) エスケープシーケンス（ギリシャ文字の記号や上付き文字など）、（c）カスタム変数を組み合わせてカスタムラベルを作成します。Originのカスタムフォーマットを使用して小数点以下の桁数などを指定します。
  • カスタムラベルは保存され、新しい出力を選択すると更新されます。
  • カスタムラベルはダイアログテーマの一部として保存可能です。
ダイアログでの名称 カスタム変数 例
N n Count = $(n)
平均 mean \g(m) = $(mean, *4)
SD sd \g(s) = $(sd, .2)
最小 min Min = $(min, .2)
最小のX xminy X Min = $(xminy, .2)
最大 max Max = $(max, .2)
最大のX xmaxy X max = $(xmaxy), .2)
合計 sum \g(S) = $(sum, .1)

(wksName)

これは、ワークシートに追加(appendwks)が選択されたときのみ有効です。これはワークシートの名前を指定するのに使用します。

サンプル

  1. 新しいワークシートを作成します。
  2. <Origin Program Folder>\Samples\Data ManipulationにあるOriginのサンプルファイル not_monotonic_multicurve.datをインポートします。
  3. 列Bを選択し、作図：折れ線：折れ線を選択してグラフを作図します。
  4. グラフウィンドウをアクティブにして、Originメニューからガジェット： 統計： ダイアログを開くを選択し、統計: addtool_curve_statsダイアログボックスを開きます。
  5. デフォルトの設定のまま、OKボタンをクリックし、ROIボックスをグラフに追加します。ROIボックスのサイズと位置を調整してデータ範囲を以下のように選択します。
  6. 四角形の上部にあるボタンをクリックして、フライアウトメニューから新しい出力を選択します。結果は、デフォルトの出力先であるスクリプトウィンドウと結果ログに出力されます。
  7. 三角ボタン をクリックして、フライアウトメニューから、線にタグ を選択します。平均値付きの線が追加されます。

全ガジェットのROIボックスの表示は、グラフの右上にあるHボタンで制御でき、ガジェットの結果と共にグラフをエクスポートことができます。

アルゴリズム

このツールで、グラフウィンドウの曲線の記述統計を実行します。計算についての詳細は、 Xファンクションstatsのアルゴリズムをご覧ください。
