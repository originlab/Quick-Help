入力

入力信号を指定します。
ウェーブレットの種類と次数

ウェーブレットの種類と次数を指定します。

  • Haar: Haarウェーブレット
  • DBN: Daubechiesウェーブレット(N = ウェーブレット次数およびバニシングモーメントの数)
  • BiorNr.Nd: Biorthogonal (Nr = ウェーブレット次数または再構成のスケーリング関数; Nd = 分解の関数の次数)
拡張モード

入力信号の拡張に使います。

  • 周期的: 入力信号を周期的として扱います。
  • ゼロパディング: 入力範囲の外側のデータポイントを0として扱います。

近似係数のための出力先を指定してください。

詳細係数のための出力先を指定してください。
