平均のt検定

検定する平均 検定される仮説の平均値(平均が等しいときは0 )
帰無仮説 検定される帰無仮説：Mean1とMean2は、異なるとはいえません。
対立仮説 対立仮説：

平均1－平均2 <> 検定される平均の差

両側の2標本の検定を使用。平均1 と 平均2 が異なる。

平均1－平均2 > 検定される平均の差

上側の片側2標本の検定を使用。平均1 は平均2 より大きい。

平均1－平均2 < 検定される平均の差

下側の片側2標本の検定を使用。平均1 は平均2 より小さい。
有意水準 有意水準を指定します。
信頼区間 平均値の違いについて信頼区間を計算します。

計算の詳細は、アルゴリズム（2群のt検定）を確認してください。

%単位での信頼水準 信頼水準をスペース区切りで入力します。このオプションは、信頼区間にチェックが付いている場合にのみ利用できます。

検出力解析

実際の検出力 検定の検出力を計算します（偽の時に帰無仮説を棄却する確率）。
有意水準 仮説検出力と/または実際の検出力にチェックが付いている場合にのみ利用できます。これは検出力の計算用の有意水準(またはアルファ値)です。
仮説の検出力 様々なサンプルサイズにおける検定の検出力を計算します。
仮説標本サイズ 仮説の検出力 にチェックが付いている場合のみ利用できます。検出力計算のための仮説の標本サイズを指定します。

プロット

ヒストグラム ヒストグラムを作成します。
ボックスチャート ボックスチャートを作成します。
1つのグラフ中に同じタイプのプロットを配置 ヒストグラムの場合、レイヤは垂直に配置されます。ボックスチャートの場合、レイヤは水平に配置されます。
