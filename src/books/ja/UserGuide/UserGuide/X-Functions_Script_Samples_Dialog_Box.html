Xファンクションスクリプトサンプルダイアログ

「XFスクリプトダイアログ」は、すべてのXファンクションの使用法やある動作を行うサンプルスクリプトを集めたものです。「XFスクリプトダイアログ」を2つの方法で開くことができます。

  1. F11キーを押す。
  2. メニューから「ツール：Xファンクションスクリプト例」を選択する。

左パネルからXファンクションを選択し、右側パネルでXファンクションの使用法やサンプルスクリプトを表示または修正します(修正はユーザのXファンクションのみ)。

Note:  左パネルで、スクリプト例を持つXファンクションは、太字で表示されます。システムXファンクションに対しては、このダイアログで使用法やスクリプトサンプルを編集できません。これらのXファンクションの名前は青く表示されています。

ダイアログボックスのオプションは以下の通りです。

ドロップダウンリスト  

場所

システム組み込みXファンクション、ユーザ定義Xファンクション、グループで共有しているXファンクションを含む表示したいXファンクションの種類を指定します。

例:

Xファンクションの使用法やスクリプトサンプルを表示したいXファンクションの種類を指定します。
ボタン  

「保存」ボタン

システム組み込みXファンクション、ユーザ定義Xファンクション、グループで共有しているXファンクションを含む表示したいXファンクションの種類を指定します。

「実行」ボタン

スクリプトサンプルを実行します。

「閉じる」ボタン

ダイアログを閉じます。
または

下側の使用法/変数情報パネルを開いたり、閉じたりします。
