ファイル名コントロール

ショートネーム

Excelワークブックのショートネームを指定します。
ロングネーム

Excelワークブックのロングネームを指定します。

保存形式グループ

内部ラジオボタン

Excelワークブックをプロジェクト内部に保存する場合このラジオボタンを選択します。このラジオボタンを選択し、プロジェクトを保存すると、Origin内でExcelワークブックになされた変更は、元のExcelファイルには影響しません。元のExcelファイルになされた変更も、OriginプロジェクトのExcelワークブックに影響しません。
外部ラジオボタン

プロジェクトを保存するとき、アクティブなExcelワークブックと元のExcelファイルを接続するリンクを保存する場合、このラジオボタンを選択します。このラジオボタンを選択した後、Origin内でExcelワークブックになされた変更は、元のExcelファイルにも影響します。また、元のExcelファイルになされた変更も、OriginプロジェクトのExcelワークブックに影響します。

このボタンを選択してプロジェクトを保存すると、Originのワークブックに加えられた変更は、元のワークブックに反映されます。また、元のワークブックに加えられた変更は、Originのプロジェクトに反映されます。

このラジオボタンを選択すると、Originは、リンクされたファイルのパスフィールドにリンク先のファイルのフルパスとファイル名を表示します。
保存せず、読み取り専用でオープン

このチェックボックスは、『外部』ラジオボタンを選択した時のみ利用可能です。このチェックボックスが選択されていると、Originで開いたExcelワークブックは読み取り専用になります。これは、Origin内でExcelワークブックになされた変更は、元のExcelファイルには保存されないということです。しかし、元のExcelファイルになされた変更は、Origin内のリンクされたExcelワークブックに反映されます。

リンクされたファイルパスのフィールド

現プロジェクト(OPJ)パスとの相対位置

このチェックボックスは、Excelファイルの位置を現在のプロジェクトとの相対パスで保存するかどうかを指定します。例えば、プロジェクトファイルが c:\ に保存され、Excelファイルがc:\test\Data.xlsに保存されると、相対パスはtest\data.xlsとなります。相対パスを使うと、プロジェクトファイルとExcelファイルの可搬性が高くなります。

保存時にOPJパスに切り替えボタンをクリックすると、このチェックボックスは自動的に選択されます。
保存時にOPJパスに切り替え

このボタンをクリックすると、外部Excelファイルのパスを現在のプロジェクト(OPJ)のパスに変更します。Excelファイルの元のパスがプロジェクトのパスと同じでない場合、プロジェクトを保存したときに、Excelファイルの新しいコピーがプロジェクトパスに作成されます。

シート名とOriginインデックスのリスト

複数のシートからワークブックデータを作図する場合、Originは、 @SheetNumberという表記を使って作図されたデータを含むこれらのシートを表示します。SheetNumberの番号は、作図された順番に従って決定されます。ワークブックのシート名とそれに対応する SheetNumber の番号が、このリストボックスに表示されます。

Note: ワークブックの最初のシートは、常にこのリストボックスに現れ、それのOriginインデックスは空欄となります。
