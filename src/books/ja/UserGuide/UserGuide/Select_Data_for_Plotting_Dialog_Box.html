「作図データの選択」ダイアログボックス

プロット属性ボタン

プロット属性ボタンは指定したワークブックのデータの列プロット属性を指定するために使用します。

X

1つまたは複数のデータプロットに対するX値

Y

1つまたは複数のデータプロットに対するY値

Z

1つのデータプロットに対するZ値

Err

1つのデータプロットに対するYエラーバー エラーバーはエラーバーデータの左側にある最初に選択したYデータセットと結びついています。

ラベル

1つのデータプロットに対するデータラベル値 ラベルはラベルデータの左側にある最初に選択したYデータセットと結びついています。

表題行

データプロットに対する凡例テキスト 「オプション」ダイアログ(Excelワークブックがアクティブな場合「ウィンドウ：Originオプション」、その他のウィンドウの場合「環境設定：オプション」)の「Excel」タブで「凡例テキストをYの列から自動検索して作成する」にチェックが付いていると、Originは自動的にこの属性の割り当てを行います。

データプロットの関連ビューボックス

このビューボックスは、X, Yなどの値に対するデータ範囲選択を表示します。 表題行に対する例外は、範囲は次のシンタックスを使って表示されます。

StartColNameRowNum:EndColNameRowNum.

『表題』は、選択された行番号のみを表示します。

作図先ドロップダウンリスト

ワークブックから複数のY範囲を割り当てる場合、Originは選択されたXデータあるいは関連する行番号を基にして複数のデータプロットを作成します。このドロップダウンリストを使用するとデータプロットの表示を制御できます。

単一レイヤ

全てのデータセットを単一のレイヤグラフに作図します。

複数レイヤ

それぞれのデータプロットを同じグラフウィンドウ内の異なったレイヤに作図します。

複数ページ

それぞれのデータプロットを、それぞれ単一のレイヤグラフウィンドウに作図します。

作図後、ダイアログボックスを閉じるチェックボックス

このチェックボックスを選択すると、「作図」ボタンをクリックした後、ダイアログボックスを閉じます。

このチェックボックスのチェックされていない場合、ワークブックまたはグラフウィンドウがアクティブであれば、ダイアログボックスは開いたままになります。 ダイアログボックスを編集して、グラフにデータプロットを追加するか、ワークブックデータから新しいグラフを作成します。 ダイアログボックスの大きさを変更し、それが画面に占める量を小さくします。

プロット制御ボタングループ

プロット

このボタンをクリックし、表示ボックスのデータプロットでの選択に基づいて1つ以上のデータプロットを作成します。 ダイアログボックスが開いたときに選択したグラフタイプでデータがプロットされます。

Bookの変更

このダイアログボックスを開いた後でワークブックを変更する場合、「作図」ボタンが「ブックの変更」ボタンに変わります。 現在アクティブなワークブックにおいてデータセットの選択を可能にするには、このボタンをクリックします。 このボタンをクリックすると、｢作図｣ボタンに変わります。

プロットの追加

グラフウィンドウがアクティブである場合、｢作図｣ボタンは「プロットの追加」ボタンに変わります。 ｢作図データの選択｣ダイアログボックスに選択されているデータをグラフに追加する場合、このボタンをクリックします。

デフォルトを使用

このボタンは、｢作図：グラフの種類｣を選択する前に、ワークブックのデータを選択した場合、あるいはグラフツールバー上のグラフの種類のボタンをクリックした場合にのみ有効です。 この状態が満たされている場合でも、このボタンは、最初にダイアログボックスを開いた時にのみ使用できます。

このボタンをクリックすると、選択されたグラフの種類にOriginのデフォルト設定を利用するので、ダイアログボックスでXY属性を指定しなくて済みます。

このボタンをクリックした後、｢注意｣ダイアログボックスが開かれ、｢オプション｣ダイアログボックスの『Excel』タブにある『XY属性のデフォルト割り当て』チェックボックスでの設定を無効にするかが確認されます。(ワークブックがアクティブの場合、ウィンドウ：Originオプションメニュー、その他のウィンドウがアクティブの場合、「環境設定：オプション」メニューを選んで、｢オプション｣ダイアログボックスを開きます。)

  • 選択したデータを作図し、チェックボックスからチェックを取り除くには、｢はい｣を選択します。
  • データは作図し、チェックボックスは変更しない場合には、｢いいえ｣を選択します。
設定を保存

このボタンをクリックすると、『作図先』ドロップダウンリストでの選択が保存され、現行Originセッション中のダイアログボックスが後に使用できます。

クリア

このボタンをクリックすると、データプロットの関連ビューボックスの現時点での選択がクリアされます。

閉じる

このボタンをクリックすると、データを作図せずにダイアログボックスを閉じます。
