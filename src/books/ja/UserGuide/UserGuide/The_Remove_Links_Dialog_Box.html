リンクを削除ダイアログボックス

RemoveLinks-Dialog

Originワークシートにリンクまたは数式を挿入して再計算または自動更新が可能です。このようなツールを使用していないユーザーは、機能を無効にすることを心配する必要はありませんが、使用する人は、セル/列のリンクを一時的に無効にするためのワークフローが必要です。このダイアログはこのような場合に使用できます。


このツールを開くには

  • 編集：リンクを削除...を選択します。

このダイアログを使って、式やリンクを現在の値に置き換えます。

リンクを削除 式やリンクの削除対象を設定します。
  • プロジェクト
  • 現在のブック
  • 現在のシート
  • 現在の選択
セル式およびリンク チェックを付けるとセル式とリンクが削除され、値だけが残ります。

→ ワークシートセルリンクとは？
列の数式 チェックを付けると列の式を削除して、現在の列値を保持します。
→ 列の式とは？
Excel(DDE)リンク チェックを付けると ExcelのDDEリンクが削除され、値だけが残ります。
