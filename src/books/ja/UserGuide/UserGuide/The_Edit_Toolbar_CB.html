編集ツールバー

CB-Edit-Toolbar

編集ツールバーボタンは、コメント化、ファイルの検索とリセット、ブックマークを行います。

テキストの選択した行をコメントにします。

テキストの選択した行をコメント解除します。

指定したファイルを検索します。

指定したファイルの前の出現を検索します。

指定したファイルの次の出現を検索します。

指定したテキストを別のテキストで置き換えます。

最後の操作を繰り返します。

テキストエディタオプションダイアログを開きます。

オプションをそのデフォルトにリセットします。

最後の操作を元に戻します。

元に戻す操作をやり直します。

対応する括弧に移動します。

ブックマークをトグルします。

ブックマークを解除します。

次のブックマークに移動します。

前のブックマークに移動します。
