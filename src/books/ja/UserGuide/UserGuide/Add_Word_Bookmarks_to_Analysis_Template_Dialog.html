分析テンプレートダイアログにWordブックマークを追加

AddWord-Bookmarks-Dialog

分析テンプレートダイアログにWordブックマークを追加ダイアログボックスは、Wordテンプレートから分析テンプレートに、ブックマークを追加するために使います。これによって、バッチ処理結果が、レポート用Wordテンプレートの対応するブックマークセルに、直接出力されます。

分析テンプレートダイアログにWordブックマークを追加ダイアログを開いて、メニューのファイル: 分析テンプレートにWordブックマークを追加...を選択します。



分析テンプレートにWordブックマークを追加...ダイアログで、分析テンプレートにWordテンプレートからのブックマークを追加することが可能です。ブックマークを分析テンプレートに追加すると、新しいワークシートブックマークが自動で作成され、分析テンプレートに追加されます。追加のブックマークは、ワークシートブックマークの列ロングネーム)に、順番にに配置されています。ブックマークワークシートの列リンク(ロングネーム)を使って、レポートシートから、対応するデータ/グラフリンクを保存します。Graph Width (Long Name)列を使って、ポイント単位で挿入したグラフサイズを調整します。ワードテンプレートで使用したパスは ワードテンプレート ラベル行に追加されます。

一回ずつWordのレポートを作成する場合には、ファイルをデータシートにインポートし、 解析結果の準備が ブックマーク シートで整ったら、シートの左上にあるWord ボタン をクリックします。

分析テンプレートダイアログにWordブックマークを追加

Wordテンプレート 指定のWordテンプレート(.DOCX, .DOCM, .DOC, .DOTX, .DOTM, .DOT)をディスクからダウンロードします。
ブックマークの選択 ブックマーク
ブックマークリスト に表示されているブックマーク名をクリックして、ブックマークの入力を選択して、分析テンプレートに追加します。ブックマークの入力をサイドクリックして、選択解除します。
追加先 Wordブックマークを追加する分析テンプレートワークブックのソースを指定します。
  • アクティブな分析テンプレートウィンドウ
    アクティブな分析テンプレートワークブック(.OGW)を使用します。
  • 分析テンプレートのロード
    指定の分析テンプレートワークブック(.OGW)をロードします。
分析テンプレート 追加モードが、分析テンプレートをロードに設定されているときのみに利用できます。これを使って、分析テンプレートを指定して、Wordブックマークを追加します。
