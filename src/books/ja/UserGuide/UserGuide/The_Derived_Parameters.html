派生パラメータ

このページは、関数のパラメータ値から計算される追加のパラメータである派生パラメータを定義するオプションを提供しています。このページは任意ですので、進むボタンをクリックして設定を実行するために次のページに移動できます。フィットセッション中にNLFitダイアログでこれらの値を指定することもできます。

ダイアログボックス内のオプション

派生パラメータグループ 各派生パラメータの意味と単位を指定するオプションを提供します。 ここには次の列があります。 :
  • 名前
    すべての派生パラメータの名前の一覧
  • 意味
    任意で派生パラメータの意味を指定します。
  • 単位
    任意で派生パラメータの単位を指定します。
派生パラメータ定義式 派生パラメータの式をここに入力できます。以下は、派生パラメータの式を記述するサンプルです。wという関数パラメータが存在しているものとします。2つの追加の派生派他メータを以下のように定義できます。
sigma = w/2;
FWHM = sqrt(2*ln(2))*w;
