グラフページおよびレイヤ内のパンニング

Pan-GraphPage

パンニング（移動）はグラフページやレイヤ内、特にズームインやズームアウトした状態でグラフの別の箇所を見たい時にとても便利です。Origin/OriginProでは、例えばデータリーダなどのほかのツールとパンニングを併用できます。

グラフページでのパンニング

ズームパンニングツールボタンをクリックして有効にし、マウスの左ボタンを押しながらドラッグし、表示したいエリアに移動します。

Note: このパンニングツールはグラフページを動かすだけです。軸のスケールとは関係ありません。

グラフレイヤでのパンニング

スケール拡大画面移動ツールボタンをクリックするかXボタンを押した状態で、アクティブレイヤ内部のパンニングを有効にします。そして、マウスの左ボタンを押しながらドラッグして興味のある位置に移動します。Xキーを押しながら矢印キー使用すると同じ操作が可能です。

水平方向にパンニング
  • スケール拡大画面移動ツールボタン + Xキー押下 + マウスホイール
  • スケール拡大画面移動ツールボタン + Xキー押下 + +/- キー
  • スケール拡大画面移動ツールボタン + 右/左矢印 キー
  • Xキーを押しながらマウスホイール/マウスドラッグ
  • Xキーを押しながら+/-キー
  • Xキーを押しながら右/左矢印キー


軸をドラッグしてスケールを変更し移動：

  • X軸上でクリックして軸端に表示される赤いドットを水平方向にドラッグ
  • スケール拡大画面移動ツールボタンをクリックするかXキーを押しながら、Y軸を水平方向にドラッグ&ドロップ
垂直方向にパンニング
  • スケール拡大画面移動ツールボタン + Xキー押下 + Shiftキー押下 +マウスホイール
  • スケール拡大画面移動ツールボタン + Xキー押下 + Shiftキー押下 ++/- キー
  • スケール拡大画面移動ツールボタン + 上/下矢印 キー
  • XキーとShiftキーを押しながらマウスホイール/マウスドラッグ
  • XキーとShiftキーを押しながら +/- キー
  • Xキーを押しながら上/下矢印キー


軸をドラッグしてスケールを変更し移動：

  • Y 軸Y軸上でクリックして軸端に表示される赤いドットを垂直方向にドラッグ
  • スケール拡大画面移動ツールボタンをクリックするかXキーを押しながら、X軸を垂直向にドラッグ&ドロップ
両方でパンニング
  • スケール拡大画面移動ツールボタン + マウスドラッグ
Note: 軸範囲を再スケールするには再スケールツール ボタンを使用します。

パンニングモードの切り替え

パンニングツールがアクティブな時、スペースキーを押すと、パンニングモードが変更され、カーソルの形状が変わります。

以下の3つのパンニングモードがあります。

  1. デフォルトのパンニングモード
  2. 主目盛増分でパンニング
  3. 副目盛増分でパンニング
Note: 主/副目盛パンニングは矢印キーを使ってパンニングする時のみ可能です。
