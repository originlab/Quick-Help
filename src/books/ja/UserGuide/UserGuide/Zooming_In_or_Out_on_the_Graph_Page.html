1つのレイヤ内でズームインまたはズームアウト

グラフのページ全体を拡大する以外に、ページやレイヤのサイズを変えずに、軸範囲の一部を拡大・縮小したい場合があります。 Originのページズーム機能はグラフページの一部を拡大しますが、レイヤズーム機能はグラフ軸のスケールを変えて（軸の開始と終了のスケール値を変更）拡大表示します。

以下のように、グラフレイヤ内を拡大するためのツールが3つあります。

  • スケールイン および スケールアウト ツール
    • 軸の拡大表示: スケールインボタン + マウスドラッグ
    • 元に戻す: スケールインボタン をダブルクリックまたは スケールアウト ボタンをクリック
    • 拡大グラフを作成: スケールインボタン + Ctrl キー + マウスドラッグ
  • 軸のズームイン および 軸のズームアウトショートカットメニュー
    • 軸の拡大と縮小: 軸を右クリックしてショートカットメニューから軸のズームイン および 軸のズームアウト を選択して軸を再スケール（現在の軸方向で拡大及び縮小）
  • スケールのズームとパンツール
    • 軸の拡大:
      (1)X軸方向: スケールのズームとパンツールボタン(または Z キー) + マウスホイールまたは + / - キーあるいは、右クリック&左/右にドラッグ;
      (2)Y軸方向: スケールのズームとパンツール ボタン(または Z キー) + Shift キー + マウスホイールまたは+ / - キーあるいは右クリック&上/下にドラッグ
  • Z キー
    • 拡大: Z キー + マウスドラッグで現レイヤのXとY軸両方の拡大、縮小
