方向

環状ノードを角度方向に時計回りまたは反時計回りに配置する方法を指定します。

ノードの開始位置

ノード開始の角度を指定します。0に設定すると、ノードは3時の位置から始まります。

半径 (%)

ダイアグラムの半径を指定します。100に設定すると、レイヤの高さと同じになります。

比率レイアウト（コードダイアグラムのみ）

1つのリンクで入力と出力を表示します。これをチェックして、1つのリンク内の2つのノード間の交換を表示します。

事前モード、事後モード（コードダイアグラムのみ）

比率レイアウトを使用して流れをグラフ化する場合、正味の流れを評価する前に重みをグラフ化する、または正味の流れから生じる重みをグラフ化することができます。事前モードおよび事後モードラジオボタンを使用すると、どちらも表示できます。

上記の簡単な例では、事後モードでは、ノードBが5単位（40から45）を獲得し、ノードAが5単位（40から35）を失い、ノードCは正味の変化を示さないことがわかります。

順序（コードダイアグラムのみ）

ノードとリンク配置をコントロール

ノードの順序

円上のノードの順序を指定します。

ノードを次の順序で並べます。

  • なし：ソースデータの元の順序に従ってノードを配置します。
  • 名前：ノードを名前のアルファベット順に昇順または降順に配置します。
  • 合計重み：合計重み（入力と出力の両方を含む）に従ってノードを昇順または降順で配置します。
  • 最初に合計出力：昇順または降順の入力合計に従ってノードを配置します。
  • 最初に合計入力：昇順または降順の出力合計に従ってノードを配置します。
  • 出力/入力比：各ノードの出力と入力の比率に従って、ノードを昇順または降順に配置します。
  • 入力/出力比：各ノードの入力と出力の比率に従って、ノードを昇順または降順に配置します。

リンク順序

現在のノードから出力するリンクと現在のノードに入力するリンクの並べ方を指定します。比率レイアウトを選択した場合、このコントロールは使用できなくなります。

  • 最初に出力：現在のノードの最初の目盛りから始めて、最初に出力リンクを配置します。
  • 最初に入力：現在のノードの最初の目盛りから始めて、最初に入力リンクを配置します。


最初に入力または最初に出力を選択したら、以下に従って入出力リンクを配置するかどうかを決定できます。

  • 昇順の重み：リンクの重み値に従って、入出力リンクを昇順で配置します。
  • 降順の重み：リンクの重み値に従って、入出力リンクを降順で配置します。
  • ターゲットノードの位置：ターゲットノードの位置に合わせて入出力リンクを配置します。この場合、セルフリンクは最初に配置されます。

リンク描画順

コードダイアグラムの円内のリンクの描画順序を指定します。

  • ノードおよびリンク順による： ソースデータのノードとリンクの順序に従います。
  • 上部の大きなリンク：広いリンクを上に置きます。
  • 上部の小さなリンク：狭いリンクを上に置きます。

葉のバブル（階層エッジバンドリングのみ）

すべての葉のバブルを表示するかどうかを指定します。このボックスをオンにすると、シンボルタブが表示され、バブルを制御できます。

サブセットのノード（階層エッジバンドリングのみ）

葉またはサブセットのノードを表示するかどうかを指定します。このボックスをオンにすると、 ノード タブが表示され、ノードを設定できます。
