列散布図を作図する

Create-Column-Scatter-Plot

列の散布図を作成するには、1つ以上のYデータ列を選択して、メニューから作図：シンボル図：列散布図を選択します。

グループ化された列の散布図を作成するには、インデックスデータあるいは素データでグループ化ボックスチャートを作成します。メニューから作図：統計グラフ：ボックスチャート-インデックスデータあるいは、作図：統計グラフ：ボックスチャート-素データを選択し、開いたダイアログでグラフテーマをBox_Column Scatterとして作図します。ユーザガイド：グループ化ボックスチャート(インデックスデータ)ユーザガイド：グループ化ボックスチャート(素データ)
