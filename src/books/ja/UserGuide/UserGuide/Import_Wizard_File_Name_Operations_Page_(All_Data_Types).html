名前の変更

自動

インポートモードとファイル名に従って自動的にワークシートの名前を変更する
ファイル名をワークシート名にする

ワークシートの名前をインポートするファイルの名前にします。
変数でファイル名を指定する

ドロップダウンリストからワークシートの名前を変更する変数を選択します。これは、変数抽出のページのファイルのヘッダから抽出されたものです。
ファイル名をワークブック名にする

ワークブックのロングネームをインポートするファイルの名前にします。
変数でワークブック名を指定する

ドロップダウンリストからワークブックの名前を変更する変数を選択します。これは、変数抽出のページのファイルのヘッダから抽出されます。
ブックのロングネームのみを変更する

ワークブックのロングネームを変更して、ショートネームをデフォルトにすることができます。
ワークブック名の変更の際にパスを含める

このチェックボックスが選択されていると、ターゲットワークブックの名前をフルパスの名前にします。

名前の追加

ファイル名をワークブックコメントに追加する

インポートするファイル名をワークブックのコメントを保存します。(「ワークブックプロパティ」または「ワークブックオーガナイザ」のコメントセクションでアクセスできます。)
ファイル名を列コメントに追加する

インポートするファイル名をワークブックのコメントを保存します。(「列プロパティ」のコメントセクションおよびワークシート列のコメント行でアクセスできます。
ファイル名の追加の際にパスを含める

ワークブックとワークシート列のコメントにフルパス名を保存します。
NOTE：データソースページのデータタイプとしてユーザ定義を選択していると、このページは表示されません。
