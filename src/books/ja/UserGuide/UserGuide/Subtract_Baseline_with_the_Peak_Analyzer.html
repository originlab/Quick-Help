ピークアナザイザで基線を減算する

ピークアナライザの「スタート」ページの「ゴール」グループで「基線の減算」ラジオボタンが選択されている場合、ピークアナライザを使って基線を作成し、入力データから基線を減算できます。このモードに対応している標準的なウィザードマップは次のようになります。

「基線のモード」ページでは、基線のモードを選択して、基線を作成することができます。そして、「基線の減算」ページで入力データから基線を減算します。減算したデータおよび基線を出力するかどうかを決めることができます。
