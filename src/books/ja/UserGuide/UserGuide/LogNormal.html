簡単な説明

対数が正規分布しているランダム変数の確率密度関数

サンプル曲線

パラメータ

数：4

パラメータの名前: y0, xc, w, A

意味：y0 = オフセット, xc = 中央, w = 標準偏差, A = 面積

下側境界: xc > 0, w > 0

上側境界: なし

派生パラメータ

平均: mu = exp(ln(xc)+1/2*w^2)

標準偏差: sigma = exp(ln(xc)+1/2*w^2)*sqrt(exp(w^2)-1)

スクリプトでのアクセス法

nlf_lognormal(x,y0,xc,w,A)

関数定義ファイル名

FITFUNC\LOGNORM.FDF

カテゴリ
