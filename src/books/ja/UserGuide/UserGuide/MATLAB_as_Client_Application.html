クライアントアプリケーションとしてMathworks MATLAB® を利用する

MATLAB-client-app

MATLABを使う場合、Originのオートメーションサーバで、MATLABデータから自動的に論文用などの高品質なグラフを作成できます。例えば*、以下のタスクを実行する簡単なルーチンを作成できます。

  • Originサーバーアプリケーションにつないで、
  • 前もってカスタマイズしたOriginのプロジェクトファイルをロードします。このプロジェクトには、ワークブックとグラフが含まれています。
  • MATLABデータをワークシートに送って、自動でグラフを更新します。
  • グラフの再スケールして、
  • グラフ画像をシステムのクリップボードにコピーします。

このようなルーチンは、Originのプロジェクトファイルをプロジェクトテンプレートして使う際にとても便利です。プロジェクトを作成して元のワークブックとグラフを含むと、プロジェクトは自動で更新されて、希望のフォーマットで結果をエクスポートできます。

Originでこのサンプルを試すには、\Samples\COM Server and Client\MATLABにある「CreatePlotInOrigin.m」を呼び出してください。

  • Note: MATLABがご自身のPCにインストールされている必要があります。
