検索ツール(スタートメニュー)

Windows-like-Search-tool

Originインターフェース左下隅には、Originロゴの上に虫眼鏡がのったアイコンがあります。マウスカーソルをこのアイコン上にのせ、クリックするか、キーボードのF1キーを押します。検索パネルが開き、以下のカテゴリのOriginのリソースを全て検索できます。

  • プロジェクトファイル
  • サンプル: Origin Central
  • メニュー項目（ガジェット、ショートカットメニューを含む）
  • アプリ
  • ヘルプ（チュートリアル、ブログ、FAQなど）
  • ビデオ
  • Xファンクション

メニュー項目、結果パネルに表示されているアプリとXファンクションの場合、現在アクティブになっているウィンドウで使用できないものはグレーで表示され、呼び出すことはできません。 リストされたAppsの場合、アプリの前にあるマークはアプリの状況を表示します：1）ダウンロード可能な場合はチェックマーク 2）既にインストールされている場合は下矢印が表示されます。

キーワードを入力しなくても、最近のファイルと一般的なメニューのみがパネルに表示されます。検索ボックスの横にある設定ボタンをクリックして設定ダイアログを開き、検索の仕組みと結果の表示をカスタマイズします。

このダイアログでは、次のことを行うことができます。

  • 同時に入力された複数の語をどのように扱うかを指定します。1）「かつ」で検索：同時に使用するか、2）「または」で検索：いずれか1つを使用します。
  • 間違った単語を許可するファジー検索を実行するかどうかを指定します。
  • 結果をコンパクトに表示するか、サンプル上にマウスを置いたときにプレビューを表示するかを指定します。
  • 検索するカテゴリを指定します。説明されている青いヒントと同様に、1つの文字を使用して検索対象の1つの領域を制限することができます。
