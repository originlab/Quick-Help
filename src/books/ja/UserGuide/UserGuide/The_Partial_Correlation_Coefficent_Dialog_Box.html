制御変数

制御変数のデータを指定し、制御変数の効果は、上の変数の関係性を計るときに除外されます。

欠損値を除外

ペアワイズ 2列間の係数を計算するとき、欠損値があれば対応する2つのデータを除外します。
リストワイズ 欠損値がある場合、すべてのデータセットに対する全行を除外します。

結果表に有意性を表示

結果表に有意性を表示

レポートワークシートで、結果表に有意性を表示します。

有意相関のフラグ

有意相関のフラグ

有意水準0.05で有意な場合、結果表で偏相関係数にアスタリスクのマークを付けます。

偏相関係数値が有意水準0.05で有意であることを、相関シートに赤で示します。
