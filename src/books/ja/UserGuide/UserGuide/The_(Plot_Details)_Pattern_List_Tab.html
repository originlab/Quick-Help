(作図の詳細)「パターンリスト」タブ

PD-Dialog-PatternList-Tab


このタブは、パターンタブ の塗りつぶしパターンの設定を、推移またはインデックスを選択した時のみ使用できます。カラーリストをカスタマイズして、1つのデータプロット内で異なるデータポイントに別の色を設定できます。

Origin 2019 からカラーリスト タブの段階的に廃止します。 パターンリスト編集の際には、作図の詳細内の編集を行いたい要素のタブを開いて編集してください（例えば、棒グラフの場合はパターンタブ）。

カスタム推移リストを使用

このチェックボックスはデフォルトでチェックが外れていて、組込推移リストが適用されています。

このチェックボックスにチェックを入れると、パターンリストにあるパターン推移リストをカスタマイズできるようになります。

パターンリスト

パターンのリストを編集、保存、ロードなどできます。また、多くのオプションがコンテキストメニューに表示されます。

カスタマイズパターンリスト

パターンのカスタマイズ

リストの各行で、三角形ボタンをクリックして組込のシンボル内部を選択します。

パターンリストの並び替え

カラーリストの並び替えと同じ方法で、パターンリストで選択した行を、移動、追加、削除することが可能です。

推移リストの保存とロード

推移リストを保存したりロードする時と同じ方法で、右クリックをしてコンテキストメニューからパターンリストの保存やロードを選択することが可能です。

推移リストのコピーと貼り付け

カスタム推移リストを他のグラフにも適用しようとする場合、リストを右クリックして推移リストをコピーを選択します。他のグラフのパターンリストタブを開き、リスト上で右クリックをしてから、推移リストを貼り付けオプションを選択します。
