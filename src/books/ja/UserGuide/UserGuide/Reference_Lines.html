「参照線」タブ

AxesRef-RefLines


特定の値または範囲を識別するには、軸上に一定の値または計算値で複数の参照線を追加し、対の参照線の間の領域を塗ることができます。

一般的な2Dグラフで右Y軸と左Y軸の両方を表示し、表示タブで各軸を再スケールを設定すると、左と右の2つの垂直アイコンが表示され、左Y軸と右Y軸の参照線を個別に追加して編集できます。

レイヤ選択

参照線の編集前に、複数レイヤグラフのレイヤ切り替えにレイヤリストを使用できます。

参照線の値

参照線を追加するには、このテキストボックスにスペース区切りで値を入力するか、 このテキストボックスの隣にある、列の選択ボタン をクリックしてワークシートから参照する列を選択します。

ラベル形式

いったん参照線の値オプションで参照線を追加すると、このオプションが有効になり、これらの参照線に対するラベルを設定することができます。ワークシートから、スペースで区切られた値のリストを入力するか、ラベルの参照列を指定することができます。（このテキストボックスの横にある列選択ボタンをクリックします）

Note: このオプションは参照線の値オプションで値を入力するか列を選択することでのみ利用可能です。このオプションに入力した、もしくは選択した列に保存されたラベルは追加した参照線のラベルとして適用されます。（軸の開始/終了を除く）どんなラベルが入力・選択されても、ラベルチェックボックス　いずれかのラベルを入力または選択すると、追加参照線のラベルチェックボックスにチェックが入り、ラベルテキストが<自動>に設定されます。

交互塗りつぶし

交互塗りつぶし

例えば、RLine 1 ~ RLine 2, RLine 3 ~ RLine 4のように、各奇数の参照線と各偶数の参照線の間の領域を塗りつぶします。

交互塗りつぶしは軸の開始と終了を含みます

交互塗りつぶしは軸の開始と終了を含みます塗りつぶし順は軸の開始からRLine 1です。カスタム参照線が偶数である場合、軸の終了が最後の塗りつぶしになります。

計算値の軸スケール範囲内のデータを使用

プロットに統計基準線（平均値など）を追加する場合、このチェックボックスをオンにすると、軸範囲内のデータのみの統計値を計算できます。 このボックスをオフにすると、Originはデータセット全体の統計値を計算します。デフォルトでは、このチェックはオンになっています。

参照線の表示ボックス

全ての参照線リスト。参照線軸の開始と軸の終了は、デフォルトでこの表の上部に表示されます。軸の値セルに値を入力して、ここに線を追加することもできます（次を参照）。

軸の値

追加した参照線の表示名前付き範囲の使用については、このヒントを参照してください。

参照線はデータポイントの後ろまたは正面に表示できます。データ描画オプションを参照してください。

線

現在の軸値に参照線を表示するかどうかを指定します。

塗りつぶし

現在の参照線が塗りつぶされているドロップダウンリストから参照線を選択します。上の交互塗りつぶしチェックボックスを選択すると、この列にペアになった線が表示されていて、編集はできません。

極座標グラフでは、内側の中心円を塗りつぶすために、参照線の追加オプションがあります。

  • 参照線が軸の開始で、塗りつぶし=軸の開始を選択すると、円の中心から軸の開始線まで塗りつぶされます。
  • 参照線が軸の終わりで、塗りつぶし=軸の終わりを選択すると、円の中心から軸の終了線まで塗りつぶされます。

ラベル

参照線にラベルを表示するかどうか指定します。

ラベル位置

ラベルの位置を指定します。水平線と垂直線の両方で、可能な位置は、参照線のスタート、中-上、中間、中-下、最後です。

ラベルテキスト

参照線のラベルとして何を表示するのかを指定します。

  • デフォルトの設定は、現在の参照線のラベルとして位置の値を表示する<入力値>になっています。
  • <自動>を選択すると軸ダイアログの目盛ラベルタブの設定に従います (タイプ、表示、小数点以下の桁数の設定, 桁数)。
  • $(v, Formats)形式を使用したカスタム文字列を入力することもできます。ここで、vは位置値であり、Formats Originフォーマットで説明される構文を使用します。2つの特殊なケースがあります。
$(v,D99) // 目盛ラベルタイプ = 日付の場合、vは「MM / dd / yyyy hh：mm：ss」という形式のユリウス日の値 $(v,T99) // v = 目盛ラベルタイプ = 時間の場合、vはユリウス日の 「DDD：hh：mm：ss」の形式
  • キャリッジリターン/ラインフィードをラベルに挿入するには、CTRL + Enterを押します（これにより、文字%(CRLF)が挿入されます）。


デフォルトでは、ラベルオブジェクトには次のプロパティがあります。

  • ラベルは参照線の上部に配置されていますが、中央または下部に表示できます。
  • プロパティダイアログボックスのプログラミングタブに表示される名前。この名前はRL(X/Y)#で、 "X"または "Y"は軸を示し、 "#"は参照線リストに表示される順番です（例えば、RLX3はリストでX軸に関連付けられた3番目の参照線）。
  • レイヤーとスケールでグラフに接続されます。
  • (%, $)にリンク,置換レベルの1は、オブジェクトが変数vの値に動的にリンクされていることを意味します。

ボタン

表示ボックスの下に3つのボタンがあります。

挿入 表示ボックスで選択する前に、新規の参照線を挿入します。
削除 表示ボックスで選択されている1つまたは複数の参照線を削除します。(軸の開始と軸の終わり以外の参照線がある場合です)複数選択する場合、Ctrlキーを押しながら、削除したい参照線をクリックして選択します。
詳細... 参照線ダイアログを開き、参照線のスタイルを変更します。
Note: 挿入と削除ボタンを有効にするには、まず、値で参照線をクリアにしてください。

参照線の詳細ダイアログ

このダイアログは、軸ダイアログの参照線タブにある、詳細…ボタンをクリックすると開きます。参照線ダイアログを使用して線を追加し、値の種類（統計など）を設定して、各参照線のフォーマットやスタイルを編集します。

ダイアログが開いたら、メインの軸ダイアログは隠されます。適用ボタンをクリックすると、メインの軸ダイアログに戻ることなく、ダイアログ内のみ変更されます。設定を保存するためにOKボタンをクリックして（あるいはキャンセルをクリック）ダイアログを閉じると、軸ダイアログが表示されます。

軸選択

右上のラジオボタンで軸を切り替えて参照線の編集が可能です。

表示タブで左軸と右軸の両方を表示し、各軸を再スケールに設定すると、垂直方向のオプションが左軸と右軸の2つのオプションに分けられて、それぞれの軸で破断を個別に編集できます。

値の種類

参照線が指定する値の種類を選択します。カスタム参照線（軸の開始と軸の終わりを除く）についてのみ表示されます。

Origin 2022bでは、値の種類が値または式の場合、軸の値ボックスで、名前付き範囲を使用できます。（単独あるいは式の一部として）

さらに、参照線を追加する場合、名前付き範囲の定義位置がプロットデータのワークシート（データ行またはラベル行）であっても、名前付き範囲のスコープをプロジェクトにする必要があります。

値 軸の定数値における基準線を追加します。
数式 曲線の式または式で計算された計算値によって生成された参照線を追加します。
統計 統計値の基準線を追加します。

折れ線グラフ、散布図、線+シンボルグラフ、縦棒/横棒グラフ、ボックスチャートではミニツールバーの統計参照線を追加ボタンをクリックして追加することもできます。

  1. レイヤ内部をクリックしてレイヤを選択します。
  2. 統計参照線を追加ボタンをクリックして、リストから統計値を選択します。
  3. 必要に応じて、詳細ボタンをクリックして参照線ダイアログを開きます。

軸の値

参照線を追加する位置を指定します。カスタム参照線（軸の開始と軸の終わりでは無効）についてのみ表示されます。

値 定数値または名前付き範囲を入力します。名前付き範囲については、上述の値の種類の項目にあるヒントを参照してください。
式 LabTalk式を入力します。Originの多くのビルトイン関数を使用して式を作成します。編集ボックスの横にあるフライアウトボタンをクリックし、関数を選択して編集ボックスに挿入することができます。

式中には名前付き範囲を使用することができます。上述の値の種類の項目にあるヒントを参照してください。

Note：文字xとyは、それぞれ現在のx軸とy軸を参照する予約済みの変数です。y = x + 1の線をグラフに追加したい場合、軸の値編集ボックスに直接“x + 1”を入力します。
統計 Originに組み込まれた統計関数を組み込んだ式を入力します。編集ボックスの横にあるフライアウトボタンをクリックし、統計値を選択して編集ボックスに挿入することができます。

以下の統計の構文の説明を参照してください。

Note:

  1. プロットの統計情報を取得するには、統計関数内で関数plotdata()をネストします。例えば、 
    Min(plotdata(1,Y)); //最初のプロットのYデータの最小値
  2. トレリスプロットでは、plotdata(index, designation, h,v)をネストすることで、各パネルで統計情報を取得できます。例えば、 
    Min(plotdata(1, Y, h, v)); //各パネルの最初のプロットのYデータの最小値
    Origin 2021b以降、トレリスプロットなどのように区分が有効になっている場合（プロットレベルの作図の詳細の区分タブ）、レイヤレベルのミニツールバーの統計参照線を追加ボタンは各パネルの構文を自動的に処理します。
    同様に、plotdata(index, designation, *,v)によってすべてのパネル行のすべての水平パネルの統計情報を得ることができ、plotdata(index, designation, h,*)によって各パネル列のすべての垂直パネルの統計情報を得ることができます。統計関数でのplotdata(index, designation)のネストは、パネルを考慮せずにこのトレリスプロットのすべてのデータの統計情報を取得します。例えば、 
    Min(plotdata(1, Y)); //このトレリスプロットにおける最初のプロットのYデータの最小値
  3. パネルを折り返し、軸を反転させてXYを交換した後、異なる順序でパネルをミックスすることがあります。実際のパネルインデックスを把握するために、軸ダイアログの参照線タブをアクティブにすると、Originはトレリスプロット上にパネルインデックスラベルを表示します。

統計基準線は動的であり、たとえばデータ範囲を変更する、またはオブジェクトマネージャでデータプロットを表示または非表示にしたとき、データフィルタを追加したときのように、サポートするデータが変更され、再計算および再描画されます。

Note: 変数xとyを含む数式を別にして、軸の値入力ボックスに1つの値を指定する必要があります。データセットの場合、最初の要素のみが考慮されます。たとえば、軸の値入力ボックスに「sin(data(1,32))」と入力すると、x/y = sin(1)にのみ参照線が表示されます。

パネルに表示

参照線が表示されるパネルを指定します。トレリスプロットでのみ使用できます。パネルインデックスを区切るためにコンマを使います。

線

表示のチェックボックスは、ディスプレイ表にある線チェックボックスと一致しています。これを使って、参照線を表示したり非表示にしたりできます。

自動フォーマットチェックボックスのチェックを外すと、参照線の色、スタイル、太さをカスタマイズできます。

デフォルトの線種として設定

参照線の設定をしたら、その線種の設定をデフォルトとして設定できます。次回以降参照線を追加すると、保存した線種が適用されます。

2つの方法があります。

  • 参照線ダイアログでの左パネルで、保存したい参照線を右クリックしてコンテキストメニューからデフォルトの線種として設定を選択します。
  • グラフに戻り、保存したい参照線上で右クリックしてコンテキストメニューからデフォルトの線種として設定を選択します。

パターン

対になった参照線間の塗りつぶし

ドロップダウンリストから塗りつぶしを選択します。これは、表示表の塗りつぶしオプションと一致しています。交互塗りつぶしにチェックを付けた場合、このドロップダウンリストは使用できません。

  • 色と透明度のオプションを使用して、選択した線の塗りつぶしをカスタマイズします。
  • Origin 2020b以降、特定の参照線の左右両方を塗りつぶすことができます（以前のバージョンでは、塗りつぶしの色は線の片側のみに適用できました）。
  • Origin 2020bより前のバージョンでは、参照線の塗りつぶしは数値順に描画されていましたが（軸スケールの小さい方が最初に描画される）、2020bでは、参照線の表の順番で描画されます。この変更により、たとえば、重なり合う塗りつぶしパターンの色に違いが生じることがあります（以前は背後にあった色が前面に描画されることがあります）。Origin 2020b以降のバージョンで以前の描画動作を使用するには、@ARLS = 1に設定します。
Note: ダイアログのフォーマットのコピーとフォーマットの貼り付けで、参照線のスタイルを簡単にコピーすることができます。左パネルの参照線で右クリックして、フォーマットのコピーを選択し、貼り付けたい参照線の上で右クリックしてフォーマットの貼り付けを選択します。さらに、コンテキストメニューから、複製を選択して、新規の参照線を同じスタイルで作成できます。

ラベル

参照線のラベルを編集します。

表示 参照線のラベルを表示するか指定します。
テキスト 参照線のラベルとして何を表示するか指定します。前述のセクションで詳細を説明しています。
ラベルフォーム 参照線にどのラベルフォームを表示するか指定します。統計参照線でのみ使用できます。
  1. 値: 統計値をラベルとして表示します。
  2. 大きさ: 大きさをラベルとして表示します。
  3. 両方: 値と大きさの両方を、大きさ=統計値のフォーマットでラベルとして表示します。
  4. カスタム: カスタマイズしたテキストラベルを表示します。
プロットの識別 プロットを識別するために、ラベルに表示する名前を指定します。
位置 ラベルの位置を指定します。
