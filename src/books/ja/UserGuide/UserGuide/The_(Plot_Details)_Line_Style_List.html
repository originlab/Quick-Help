(作図の詳細)線のスタイルリスト

PD-line-style-list-tab

線のスタイルリストタブは、(a) カテゴリー値のワークシート列をグラフの線タブの線種として設定した場合、または、(b) 線種 = 推移に設定した場合に追加されるタブです。

デフォルトでは、線の種類のLabTalkリストを使用します。しかし、このリストは変更可能なので、必要に応じてスタイルを再配置したり削除できます。カスタム推移リストを使用のチェックを付けて有効にします。このリストの設定については、このページを参照して下さい。

Origin 2022b以降、カスタム破線を素早く定義でき、 これらのカスタム破線を線スタイル リストの作成に使用できます。
