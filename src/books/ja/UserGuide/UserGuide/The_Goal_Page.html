処理のゴールページ

FitFuncBuilder-Goal

処理のゴールページはフィット関数ビルダの最初のページで、これはフィット関数ビルダの処理のゴールを選択します。このページで、ユーザ定義関数を作成または編集することができます。異なる処理を選ぶと、ビルダの次のページが若干違ってきます。

ダイアログボックス内のオプション

新しい関数の作成 新しい関数を作成するゴールを選択します。関数は、ユーザファイルフォルダ \fitfuncフォルダに置かれます。
ユーザ定義関数の編集 ユーザ定義関数を編集するには、このゴールを選択します。ユーザファイルフォルダ\ fitfunc フォルダにユーザ定義関数があれば、このツールは、ユーザ定義関数を編集するオプションの下に、利用可能なユーザ定義関数をドロップダウンリストで表示し、そこから編集する関数を選択します。フォルダ内にユーザ定義関数がない場合、ユーザ定義関数が見つかりません。 最初に作成してくださいというメッセージが出力されます。

Fitting Function Libraryアプリ を使用すると、アドオンのフィット関数を参照できます。このアプリはOriginの最新バージョンにプレインストールされています。Originの組み込みライブラリ内に特定のフィット関数が見つからない場合は、OriginLabのWebサイトからダウンロードできるかどうかを確認してください。 処理のゴールページのリンクをクリックする、もしくはOriginワークスペースの右側にあるアプリギャラリーでアプリのアイコンをクリックしてアプリを起動します。
