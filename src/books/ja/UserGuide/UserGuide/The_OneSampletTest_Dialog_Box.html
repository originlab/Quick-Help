入力

入力データ形式

要約データ

標本の平均値、標準偏差、サンプルサイズを、"mean SD N" という形式（数字の間には、スペースを挿入）で入力します。

素データ

範囲を選択して、列を入力します。範囲の設定に関する詳細は、入力データを指定するをご覧ください。
入力

データを入力します。

平均のt検定

検定する平均 検定される仮説の平均値を入力すると帰無仮説と対立仮説の項目が更新されます。
帰無仮説 検定される帰無仮説

(平均 = 検定される平均)

対立仮説 対立仮説

オプション：

平均 <>検定される平均

両側の検定を使用。標本平均が仮説の平均と異なるか検定します。

平均 >検定される平均

上側の片側検定を使用。標本平均が仮説の平均より大きいか検定します。

平均 <検定される平均

下側の片側検定を使用。標本平均が仮説の平均より小さいか検定します。
有意水準 有意水準を指定します。
信頼区間 平均の信頼区間を計算します。

計算の詳細は、アルゴリズム（1群のt検定）を確認してください。

%単位での信頼水準 信頼水準をスペース区切りで指定します。 (1-)%で表わされます。信頼区間が選択されたときに有効です。

検出力解析

実際の検出力 検定の検出力を計算します（偽の時に帰無仮説を棄却する確率）。
有意水準 仮説検出力と/または実際の検出力にチェックが付いている場合にのみ利用できます。これは検出力の計算用の有意水準(またはアルファ値)です。
仮説の検出力 様々なサンプルサイズにおける検定の検出力を計算します。
仮説標本サイズ 仮説の検出力 にチェックが付いている場合のみ利用できます。検出力計算のための仮説の標本サイズを指定します。

プロット

ボックスチャート ボックスチャートを作図します。
ヒストグラム ヒストグラムを作図します。
1つのグラフ中に同じタイプのプロットを配置 ヒストグラムの場合、レイヤは垂直に配置されます。ボックスチャートの場合、レイヤは水平に配置されます。
