サンプル曲線

パラメータ

数：4

パラメータ名:y0, A1, a, b

意味:y0 = offset, A1 = Amplitude, a = Shape, b = Scale

下側境界:A1 > 0.0, a > 0.0, b> 0.0

上側境界:なし

派生パラメータ

Mean: mu=a*b

標準偏差：sigma=sqrt(a)*b

スクリプトでのアクセス方法

gamcdf( x, a, b)
