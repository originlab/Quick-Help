はじめに

ボックスチャートには特殊な凡例を作成できます。その凡例では、それぞれのエントリはボックスの要素、例えばボックス、ヒゲ、中央値線や平均線、パーセンタイル等に対応します。

ボックス要素に関する凡例は以下のグラフ形式で作成できます。

  • ボックスチャート
  • インデックスデータからのグループボックスチャート
  • 素データからのグループボックスチャート

凡例内の標準ボックスチャートパラメータの表示非表示は簡単に編集できます。

ボックスチャート要素の凡例を作成

ボックスプロットの凡例（上のセクションを確認してください）には、既存の凡例（データプロットに対する凡例オブジェクト）を削除して、ボックス要素に対する凡例を作成することができます。

ボックスチャート要素のデフォルト凡例はボックスチャートが変われば自動的に更新されます。

ボックスの要素に対する凡例を作成するには、legendbox ダイアログを使用します。このダイアログは次の手順でアクセスできます。

  • グラフウィンドウがアクティブな状態で、メニューからグラフ操作：凡例：ボックスチャートの内容を選択します。

または

  • グラフページを右クリックして、コンテキストメニューから凡例：ボックスチャートの要素を選びます。

または

  • 凡例オブジェクトを右クリックして凡例：ボックスチャートの要素と選択します。

Note：既存の凡例がボックスチャートの要素を既に含んでいる場合、このダイアログを使用すると凡例のエントリを更新します。

legendboxダイアログを開く以外の、ボックスチャートの要素に対する凡例を作成するアプローチがあります。

  • メニューから直接グラフ操作：凡例：凡例の再構築を選択するか、あるいはCtrl+Lキーを押してください。


legendboxダイアログ

このダイアログはボックスチャートの要素のために充実した凡例を作成するために使用されます。グラフ上にすでに凡例オブジェクトがある場合、ボックスチャートの内容を既存の凡例に追加します。ダイアログオプションの中には、特定のボックスチャートにのみ有効なものがあります。

このダイアログは legendbox Xファンクションを使用しており、どのボックス要素が凡例に表示されるのか指定するために使用します。ダイアログ内の項目は次の表に記載されています。

ボックス ボックスチャートの種類がボックス、ボックス+点列 重複、点列[左]+ボックス[右]、ボックス[左]+点列[右]、棒[左]+点列[右、ボックス半分[右]+点列[左]、ボックス半分[左]+点列[右]のいずれかの場合に利用可能です。これはエントリにボックスを表示するかどうか指定します。データプロットのシンボル形式は作図の詳細ダイアログにあるボックスチャートタブやパターンタブに倣います。凡例テキストはメニュー内のボックスIDとボックスの範囲を表示で指定された変数と範囲を使用します。
ボックスID このドロップダウンは、ボックスにチェックが付いている場合にのみ利用できます。<カスタム>を選択して、カスタムフォーマットを指定します。
  • ショートネーム
  • ロングネーム
  • 単位
  • コメント
  • カテゴリグループ列: ボックスチャートグループを決定するために使用されたカテゴリデータ列
  • <カスタム>
  • <なし>: ボックスIDなし通常のボックスチャートとグループ化ボックスチャートのデフォルトオプション
カスタムIDフォーマット これは、データIDで＜カスタム＞が選択されている場合のみ利用できます。次のいずれかから識別子を選択します。
  • ドロップダウンリストから組み込みのカスタムフォーマットを選択します。
  • コンボボックスの右側にある矢印ボタンをクリックします。
  • 独自のID文字列を直接入力します。詳細は@optionsの完全リストをご覧ください。
ボックスの範囲を表示 このチェックボックスは、ボックスにチェックが付いている場合にのみ利用できます。これを選択するとボックスIDのテキストの隣にボックス範囲の情報を追加します。ボックス範囲は作図の詳細ダイアログのボックスチャートタブにある範囲ドロップダウンリストで決まります。
ヒゲ このチェックボックスは、ヒゲが有効な時のみ利用できます。ヒゲを凡例内に表示するかコントロールします。表示されている場合、作図の詳細ダイアログの線タブ内のヒゲのスタイル設定でデータプロットのシンボルの設定が続きます。凡例テキストはヒゲ範囲設定（ボックスチャートタブのヒゲグループ内にある範囲 ドロップダウンリスト）のテキストを使用します。
棒 ボックスの種類が線、線 + データオーバーラップ、線(右) + データ(左)または、線(左) + データ(右) の時のみ有効なオプションです。これはデータの線へのエントリを凡例に表示するかどうか指定します。

データプロットのシンボル形式は作図の詳細ダイアログにあるボックスチャートタブやパターンタブに従います。凡例テキストはメニュー内の線IDと線の表示で指定された変数を使用します。

横棒識別子 このドロップダウンは、線にチェックが付いている場合にのみ利用できます。<カスタム>を選択して、カスタムフォーマットを指定します。
  • ショートネーム
  • ロングネーム
  • 単位
  • コメント
  • カテゴリグループ列: ボックスチャートグループを決定するために使用されたカテゴリデータ列
  • <カスタム>
カスタムIDフォーマット これは、線IDで＜カスタム＞が選択されている場合のみ利用できます。次のいずれかから識別子を選択します。
  • ドロップダウンリストから組み込みのカスタムフォーマットを選択します。
  • コンボボックスの右側にある矢印ボタンをクリックします。
  • 独自のID文字列を直接入力します。詳細は@optionsの完全リストをご覧ください。
棒の表示 このチェックボックスは、線にチェックが付いている場合にのみ利用できます。これを選択すると線IDのテキストの隣にボックス範囲の情報を追加します。ボックス範囲は作図の詳細ダイアログのボックスチャートタブにある範囲ドロップダウンリストで決まります。
エラーバー このオプションはエラーバーが有効な時のみ利用できます。エラーバーを凡例に表示するかコントロールします。
中央値線 このチェックボックスは作図の詳細ダイアログの線タブで 中央値線が有効な場合のみ選択できます。中央値線に対して凡例のエントリを表示するか指定します。データプロットシンボルは作図の詳細ダイアログの線タブの中央値線を表示し、その隣には中央値線とテキスト表示します。
平均線 このチェックボックスは作図の詳細ダイアログの線タブで 平均線が有効な場合のみ選択できます。平均線に対して凡例のエントリを表示するか指定します。データプロットシンボルは作図の詳細ダイアログの線タブの平均線を表示し、その隣には平均線とテキスト表示します。
最大

このチェックボックスは、ボックスチャートのパーセンタイルタブで最大にチェックがついている場合のみ利用できます。パーセンタイルポイントの最大値に凡例のエントリを表示するか指定します。データプロットシンボルは作図の詳細ダイアログのパーセンタイルタブの最大を表示し、その隣には最大値とテキスト表示します。

99% このチェックボックスは、ボックスチャートのパーセンタイルタブで99%にチェックがついている場合のみ利用できます。パーセンタイルポイントの99％に凡例のエントリを表示するか指定します。データプロットシンボルは作図の詳細ダイアログのパーセンタイルタブの99％を表示し、その隣には99％とテキスト表示します。
平均 このチェックボックスは、ボックスチャートのパーセンタイルタブで平均にチェックがついている場合のみ利用できます。パーセンタイルポイントの平均に凡例のエントリを表示するか指定します。データプロットシンボルは作図の詳細ダイアログのパーセンタイルタブの平均を表示し、その隣には平均とテキスト表示します。
平均値識別子 このチェックボックスは、ボックスチャートのパーセンタイルタブで平均にチェックがついていて、上述の平均にチェックがついている場合のみ利用できます。最初の4つは、ソースデータ列の対応する列ラベル行の変数を使用します。
  • ショートネーム
  • ロングネーム
  • 単位
  • コメント
  • カテゴリグループ列: ボックスチャートグループを決定するために使用されたカテゴリデータ列
  • <カスタム>: テキスト文字列を構成するためのカスタムIDフォーマットと一緒に使用
  • <平均>: 最初のプロットの平均値シンボルを使用
中央値 このチェックボックスは、ボックスチャートのパーセンタイルタブで中央値にチェックがついている場合のみ利用できます。 凡例のエントリに中央値のパーセンタイルポイントを表示するか指定します。データプロットシンボルはパーセンタイルタブの中央値を表示し、凡例テキストに中央値を表示します。
中央値識別子 このチェックボックスは、ボックスチャートのパーセンタイルタブで平均にチェックがついていて、上述の中央値にチェックがついている場合のみ利用できます。最初の4つは、ソースデータ列の対応する列ラベル行の変数を使用します。
  • ショートネーム
  • ロングネーム
  • 単位
  • コメント
  • カテゴリグループ列: ボックスチャートグループを決定するために使用されたカテゴリデータ列
  • <カスタム>: テキスト文字列を構成するためのカスタムIDフォーマットと一緒に使用
  • <中央値>: 最初のプロットの中央値シンボルを使用
1% このチェックボックスは、ボックスチャートのパーセンタイルタブで1%にチェックがついている場合のみ利用できます。 パーセンタイルポイントの1%に凡例のエントリを表示するか指定します。データプロットシンボルは作図の詳細ダイアログのパーセンタイルタブの1%を表示し、その隣には1%とテキスト表示します。
最小値 このチェックボックスは、ボックスチャートのパーセンタイルタブで最小にチェックがついている場合のみ利用できます。 パーセンタイルポイントの最小値に凡例のエントリを表示するか指定します。データプロットシンボルは作図の詳細ダイアログのパーセンタイルタブの最小を表示し、その隣には最小値とテキスト表示します。
カスタムパーセンタイル このチェックボックスは、ボックスの種類がデータではない時のみ利用できます。カスタム値でのパーセンタイルポイントを凡例のエントリを表示するか指定します。データプロットシンボルは作図の詳細ダイアログのパーセンタイルタブのカスタムパーセンタイルを表示し、その隣にはカスタムパーセンタイルとテキスト表示します。例えば、P=5 とすると、表示される凡例テキストは5%, 95% です。
データ このチェックボックスは、ボックスの種類がボックス/棒ではない時、または、列の散布図のとき利用できます。これはデータのエントリを表示するかどうか指定します。データポイントの形状は、作図の詳細ダイアログのシンボルタブの設定に従い、テキストはデータ識別子で決定されます。

Note: プロットのデータポイントがカテゴリー値の凡例や1ポイントずつの凡例に当てはまる場合、当てはまる項目の凡例エントリはボックスチャートの要素凡例に追加されます。

データ識別子 このチェックボックスは、ボックスの種類がボックス/棒ではない時、または、列の散布図で、さらに上述のデータにチェックがついているとき利用できます。最初の4つは、ソースデータ列の対応する列ラベル行の変数を使用します。
  • ショートネーム
  • ロングネーム
  • 単位
  • コメント
  • カテゴリグループ列: ボックスチャートグループを決定するために使用されたカテゴリデータ列
  • <カスタム>: テキスト文字列を構成するためのカスタムIDフォーマットと一緒に使用
  • <データ>: 最初のプロットのデータシンボルを使用
カスタムIDフォーマット これは、データIDで＜カスタム＞が選択されている場合のみ利用できます。次のいずれかから識別子を選択します。
  • ドロップダウンリストから組み込みのカスタムフォーマットを選択します。
  • コンボボックスの右側にある矢印ボタンをクリックします。
  • 独自のID文字列を直接入力します。詳細は@optionsの完全リストをご覧ください。
外れ値 ボックスチャートタブで外れ値を有効にした時のみ利用できます。凡例エントリーに外れ値を表示します。データプロットのシンボルスタイルは外れ値のデータポイントと同じで、隣に入るテキストは外れ値になります。
極端値 ボックスチャートタブで外れ値を有効にした時のみ利用できます。凡例エントリーに極値を表示します。データプロットのシンボルスタイルは極値のデータポイントと同じで、隣に入るテキストは極値になります。
平均値接続線 作図の詳細の接続線タブで平均接続線が有効なときのみ可能です。ボックスチャートの平均値の接続線の凡例エントリを作成します。データプロットシンボルは作図の詳細ダイアログの接続線タブの平均値接続線の設定に従います。また、平均値とテキスト表示します。
中央値線接続線 作図の詳細の接続線タブで中央値接続線が有効なときのみ可能です。ボックスチャートの中央値の接続線の凡例エントリを作成します。データプロットシンボルは作図の詳細ダイアログの接続線タブの中央値接続線の設定に従います。また、その隣には中央値とテキスト表示します。
データポイント接続線 作図の詳細の接続線タブでデータポイント接続線が有効なときのみ可能です。ボックスチャートに各変数のデータポイントの接続線の凡例エントリを作成します。データプロットシンボルは作図の詳細ダイアログの接続線タブのデータポイント接続線の設定に従います。また、データ値とテキスト表示します。データポイント接続線の色がカテゴリデータの列とインデックス化されている場合は、凡例テキストはカテゴリとなります。
パーセンタイル接続線 作図の詳細の接続線タブでパーセンタイル接続線が有効なときのみ可能です。特定のパーセンタイルの接続線の凡例エントリを作成します。データプロットシンボルは作図の詳細ダイアログの接続線タブのパーセンタイル接続線の設定に従います。また、パーセンタイルを凡例テキストに表示します。
