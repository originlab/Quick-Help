(作図の詳細) 「カラーリスト」タブ

PD-Dialog-ColorList-Tab

Origin 2020以降のバージョンでは、カラーリストタブはなくなりました。編集の際には、作図の詳細内の編集を行いたい要素のタブを開いて編集してください（例えば、散布図の場合はシンボルタブ）。

Origin 2019b 以前

この作図の詳細タブは、ポイント毎に色を変えてからインデックスのカラーオプションを選択した場合にのみ使用できます。


Origin 2019 からカラーリスト タブの段階的廃止を開始します。Origin 2019以降でカラーリストを編集するには、特定の要素のカラーコントロール - シンボルのエッジの色、線の色、パターンの塗りつぶしの色など - に移動して、その横にあるリストコントロール（下向き矢印）をクリックします。ポイント毎タブ（上の画像を参照）で、カラーリストの右端にある鉛筆アイコンをクリックすると、インクリメントエディタが開き、カラーリストにアクセスできます。
