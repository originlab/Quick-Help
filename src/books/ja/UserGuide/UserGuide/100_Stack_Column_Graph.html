要求されるデータ

少なくとも1つのY列、あるいは、その部分領域を選択する必要があります。関連付けられたX列がある場合はそれを使用し、そうでない場合は、Y列または行番号のサンプリング間隔が使用されます。

グラフ作成

データを選択します。

メニューから作図：棒、円、面積：100％積み上げ縦棒グラフを選択します。

または、

2D グラフ ツールバーの100%積み上げ縦棒グラフボタンをクリックします。

テンプレート

COLUMN.OTP (Originのプログラムフォルダにインストールされています。)

ノート

このグラフタイプは、積み上げ棒グラフを作成後、作図の詳細（レイヤ属性）の積み上げ形式タブで累積パーセントに正規化のチェックボックスを選択しても作図可能です。作図の詳細ダイアログのラベルタブで、ラベル形式としてカスタムを選択し、表示フォーマットとして $(p,.0)% と入力すれば、データラベルをパーセンテージの値として表示するこオプションを利用可能です。

ワークシートにX列とXエラー列の組み合わせを作ることで、100％積上げ横棒グラフにエラーバーを追加することが出来ます。詳細はこちらのページをご覧ください グラフにエラーバーを追加する　方法１- プロット属性を事前に設定.
