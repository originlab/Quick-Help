外部DLLを使用する

(高度なオプション)

組み込みのOriginフィット関数は、事前にコンパイルしたデータセットをフィットします。ユーザ定義のフィット関数の実行を向上させるには、外部DLLを使ってフィット関数を作成することができます。外部DLLで定義されるフィット関数は、Origin Cで定義したフィット関数よりも高速に実行できます。

Samples\Curve FittingフォルダにFitting with External DLL.opxというパッケージファイルを準備しています。opx ファイルをOriginワークスペースにドラッグ&ドロップすると、新しいフォルダがユーザファイルフォルダに作成されます。そのフォルダに、Microsoft Visual C++ ワークスペース、サンプルデータ、このようなDLLの作成方法があります。
