ユーザガイド：(オブジェクトプロパティ)線タブ

Ob-Prop-Line-tab


線 タブでは、オブジェクトの色、幅、タイプ 及び 透明度 を設定できます。

タイプの設定のカスタム破線の定義についての情報は、作図の詳細のグラフの線タブのヘルプページを参照してください。

直線のオブジェクト（矢印付の直線）では、位置合わせ で 垂直 または 水平 を設定出来、さらにレイヤ全体、全てのレイヤ または ページ全体 を選択出来ます。

この デフォルトに設定 のチェックボックスにチェックを入れると、このタブで選択した内容が、以後作成される同じタイプのオブジェクト全てに適用されます。

Note:透明度コントロールは、矢印 タブと同じ透明度を共有します。

線の透明度％を変更すると、矢印の先端にも同じ透明度が適用されます。逆も同じです。
