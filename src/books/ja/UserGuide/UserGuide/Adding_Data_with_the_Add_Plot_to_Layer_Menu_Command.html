「プロットをレイヤに追加」メニューコマンドを使ってデータを追加する


これは、既存の折れ線、散布図、線＋シンボル、縦棒グラフ、面積グラフにデータを追加する簡単な方法です。

  1. ワークシートデータを選択します。
  2. グラフウィンドウをクリックして、アクティブにします。
  3. メニューから、「グラフ操作：プロットをレイヤに追加：グラフの種類」を選びます。

Note:プロットをレイヤに追加する方法を設定するには(例えば、レイヤ内のプロットをグループ化する)、「プロットをレイヤに追加」メニューを選択したときに「作図のセットアップ」ダイアログボックスを開くようにLabTalkのシステム変数を設定することができます。

「プロットをレイヤに追加」メニューコマンドの動作を変更するには、コマンドウィンドウを開き、コマンドラインで以下のどちらかを入力します。

@LIP=0 <ENTER>//「作図のセットアップ」ダイアログボックスが開きます

@LIP=1 <ENTER> //デフォルト。 「作図のセットアップ」ダイアログボックスを開かずに、選択したデータセットを追加します。
