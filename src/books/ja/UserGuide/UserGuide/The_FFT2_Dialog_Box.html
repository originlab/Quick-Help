入力行列

入力信号の複素行列または実数行列を指定します。入力データの指定を参照してください。
虚数行列

入力行列が複素行列で実数行列と虚数行列に分かれている場合、入力行列の虚数行列を指定します。入力データの指定を参照してください。
振幅行列の規格化

振幅を1までに規格化するかしないかを指定します。
DC成分を中央に移動

この変数が選択された場合、DC成分は 結果行列の中央に配置されます。そうでなければ、DC成分は結果行列の四隅に配置されます。
