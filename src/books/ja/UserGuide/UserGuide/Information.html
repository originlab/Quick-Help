情報


Xファンクション「imgInfo」で表示できる画像のプロパティは以下の表の通りです。

プロパティ
説明
width イメージの幅
Height イメージの高さ
bits ピクセル当たりのビット数
colors 色の深さ
isgray 画像がグレースケールかどうか
order 色の順序

このXファンクションを使うには、解析する画像をアクティブにして、次のコマンドをコマンドウィンドウに入力します。

imgInfo

この関数の詳細については、OriginのXファンクションヘルプファイルにあるXファンクション「imgInfo」をご覧下さい。これはコマンドウィンドウで次のように入力すると直接表示することができます。

help imgInfo
