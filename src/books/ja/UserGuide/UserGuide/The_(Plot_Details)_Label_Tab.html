(作図の詳細)「ラベル」タブ


列のプロット属性がラベルに設定されている場合、作図の詳細ダイアログのデータセットレベルのラベルタブを使用できます。作図の詳細ダイアログの左側で選択されたデータラベルアイコンについて、散布図/折れ線や縦棒/横棒 のラベル付けに使用されます。つまり、ラベルはこの方法で追加できます。

ラベルタブは、データプロットのデータラベルのフォーマットおよび位置を編集するオプションを提供します。

元のグラフが散布図でデータプロットのアイコンが作図の詳細ダイアログの左側パネルで選択されている時は、別のラベルタブ があります。

フォント

フォント

このドロップダウンリストから希望のフォントを選択します。 デフォルトの選択は、「オプション」ダイアログボックスの「テキストフォント」タブの標準フォントドロップダウンリストで制御できます。

色

ラベルのテキスト色を選択します。

（ラベル列属性を使って）データポイントにラベル付けする方法の場合、ラベルは単一色になります。色推移または、データセットの値を使って色を割り当てを使ってデータポイントに色付けをする場合、ラベルの色とデータポイントの色を一致させるとよいでしょう。詳細は、こちらを、ご覧ください。

サイズ

このコンボボックスにサイズを入力または選択します。 サイズの単位はポイントです。

「白地」チェックボックス

このチェックボックスを選択すると、各データラベル値に「白地」の背景を表示します。

書式ボタン

太字、斜体、下線が提供されています。

回転

回転ボックスでの値は、データラベルの回転角度を指定します。

  • 正の値は、ラベルを反時計回りの方向に回転します。
  • 負の値は、ラベルを時計回りの方向に回転します。

水平オフセットと垂直オフセット

水平オフセット ボックスの設定は、データラベルの水平オフセットを指定します。

垂直オフセット ボックスの設定は、データラベルの垂直オフセットを指定します。

オフセットは、フォントの高さのパーセントで指定されます。

データラベルのオフセットをカスタム化するには、まずグラフへ戻ってデータラベルをドラッグ＆ドロップします。

接続先

このドロップダウンリストから、X軸下、Y軸左、X軸上、Y軸右を選択してデータラベルを整列します。 ラベルは、回転テキストボックスにセットした向きを維持しますが、指定した軸の位置に移動します。 なしを選択すると、結びついたデータポイントのXY座標にデータラベルを表示します。

位置合わせ

データポイントと結びついているデータラベルを左、中央、右のどれかに設定することができます。 例えば、 左 を選択すると、ラベルの左端がデータポイントのXY座標に合わせられます。(ヒント: Xオフセットと Yオフセットに連動した設定を使うことができます。 -- 以下参照)

数値表示フォーマット

Originフォーマットをご覧ください。サポートしているフォーマットには、日付と時間のデータがワークシート中にユリウス暦の値として数値で表示される限り、D（日付）とT（時間）のオプションがあります。
