ダイアログボックス内のオプション

Pythonパッケージの確認
(LabTalkスクリプト)		 このボックスにスクリプトを入力して、ユーザ定義のPythonフィット関数をインストールする際に、フィット関数の使用に必要なPythonパッケージのプロンプトを表示します。FDFファイルをOriginワークスペースにドロップすると、スクリプトが実行されます。このページは必ず入力してください（Python関数のみ）。
  • 必要なパッケージがインストールされていない場合、パッケージのインストールを許可するように求められます。
  • 必要なパッケージがすでにインストールされている場合、フィット関数をどのカテゴリに追加するか聞かれます。

以下がスクリプトのサンプルです。必要に応じて、必要なパッケージを括弧および二重引用符で囲みます。

[BeforeInstall]
if(Python.chk("pandas cv2(opencv-python)") > 1)
  return 1;// FDFをインストールしない

return 0;//続行
