関数

説明

一定時間パラメータの2次指数増加関数

サンプル曲線

パラメータ

数:  5

パラメータの名前: y0, A1, t1, A2, t2

意味：y0 = オフセット, A1 = 振幅, t1 = 時間定数 (崩壊定数), A2 = 振幅, t2 = 時間定数 (崩壊定数)

下側境界: なし

上側境界: なし

Note: t1とt2ではそれぞれ異なる初期パラメータが使用され、フィット結果として t2 > t1 が成り立つようになっています。

派生パラメータ

個別の増加率：

k1 = 1/t1

k2 = 1/t2

個別の倍加時間：

Td1 = t1*ln(2)

Td2 = t2*ln(2)

スクリプトでのアクセス法

nlf_expgro2(x,y0,A1,t1,A2,t2)

関数定義ファイル名

FITFUNC\EXPGRO2.FDF
