(列プロパティ ) 「ユーザーツリー」タブ

ColProperties-Dialog-UserTree-Tab

列プロパティダイアログのユーザーツリー タブでは、現在の列に関連するユーザーツリーを表示、編集することができます。

ユーザツリーを編集するには、ツリーまたはノードを右クリックし、ショートカットメニューから目的のコマンドを選択します。

名前の変更

このコマンドは名前と現在のツリーもしくはノードのラベルを変更します。ツリーノードの名前の変更ダイアログが開きます。ツリー名もしくはノード名は文字と数字のみサポートされています。1番目は文字で始まる必要があります。

削除

現在のツリーまたはノードを削除します。

ツリーの追加

このタブにユーザツリーを追加します。

ノードの追加

現在のツリーまたはノードに新しいノードを追加します。ツリーノードの追加ダイアログが開き、ノードの名前とタイプを指定できます。利用できるタイプは、数値、テキスト、ブランチです。

ノート:例えば、インポートにより自動的に追加されたようなノードなど、いくつかのノードの値は編集不可です。このような場合には、値は青字で表示されます。
