計算する値

選択されたビンの統計量を計算します。

最小 各ビンに対する最小値を出力します。
最大 各ビンに対する最大値を出力します。
平均 各ビンの平均を計算し、出力します。
中央値 各ビンに対する中央値を出力します。
合計 各ビンの合計を計算し、出力します。
カウント 各ビンのカウントを計算し、出力します。
パーセント度数 各ビンのパーセント度数を計算し、出力します。

計算する値の列

計算する値としてカウントを選択しているとき以外に利用可能なオプションです。計算する値を算出する列を指定します。カスタム化オプションは、現在のワークシートの任意の列を選択できます。

ベース列

計算する値の列でカスタム化を選択したときに利用できます。現在のワークシートの任意の列を選択できます。

ビン出力

指定したXビン範囲に対して出力するビン、ビン開始、ビン中心/ビン終端を指定します。デフォルトではこれらのボックスにはチェックは入っていません。

出力ワークシート

ワークシートに結果出力

各ビンYの小計カウント

以下の2つの条件がそろった時に利用可能です。

  • 出力ワークシートのチェックがついている。
  • 計算する値として、合計、カウント、パーセント度数が選択されている。

各ビンYの小計カウントを出力ワークシートに出力するか指定します。

出力行列

結果を行列に出力するか指定します。

行列プロット

出力行列にチェックが付いているときに利用可能です。

3D棒グラフ

出力された行列から3D棒グラフを作図します。
イメージプロット
