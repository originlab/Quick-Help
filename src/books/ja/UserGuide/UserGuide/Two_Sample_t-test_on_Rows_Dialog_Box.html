サポート情報

2群の行のt検定ダイアログボックスを開くには、

  • ワークシートがアクティブな状態で、統計：仮説検定：行のt検定（2群）...をクリックします。

再計算

再計算

分析結果の再計算を制御します。

  • なし
  • 自動
  • 手動

詳細情報は、 分析結果の再計算をご覧下さい。

入力データ

第1データ範囲 第1標本のデータ範囲（複数列）を指定します。各行がデータ変数を表します。
第2データ範囲 第2標本のデータ範囲（複数列）を指定します。各行がデータ変数を表します。


検定オプション

検定する平均 検定される仮説の平均値(平均が等しいときは0 )
対立仮説 対立仮説：
平均1－平均2 <> 検定される平均の差

両側の2標本の検定を使用。平均1 と 平均2 が異なる。

平均1－平均2 > 検定される平均の差

上側の片側2標本の検定を使用。平均1 は平均2 より大きい。

平均1－平均2 < 検定される平均の差

下側の片側2標本の検定を使用。平均1 は平均2 より小さい。

分散が等しいと仮定/Welch（分散が等しくないと仮定）

出力結果についてはこちらを確認してください：

結果は"t-statistic" ("Welch t"), "DF" ("Welch DF"), "P-Value" ("Welch P"), "Significant" ("Welch Sig")というロングネームで出力されます。

t統計量 選択して、t統計量を出力します。値の出力列を指定できます。各行は行中のデータのt統計量を表します。
自由度 選択して、自由度を出力します。値の出力列を指定できます。各行は行中のデータの自由度を表します。
P値 選択して、p値を出力します。値の出力列を指定できます。各行は行中のデータのp値を表します。
有意性 選択すると判断のフラグを出力し、値の列を指定します。値が1の場合、平均値の差は0.05レベルで有意で、0の場合、平均値の差は0.05レベルで有意に異ならないことを示します。
