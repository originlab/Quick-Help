インターネットライセンス

Internet-Lic

インターネットライセンスは、ネットワーク同時起動ライセンスと同様に、組織内の任意のPCにOriginをインストールし、購入したシート数と同じ数のPCで同時にOriginを使用できます。

インターネットライセンスはユーザ自身でライセンスサーバを用意する必要がないという点で、ネットワーク同時起動ライセンスと異なります。

n シート購入した場合、n 個のプロダクトキーが発行されます。これらのプロダクトキーを使用して、任意のPCにOriginをインストールしてアクティブ化できるので、Originを同時にn 台のPCで使用できます。ライセンス取り外し操作は不要で、他のPCで同じプロダクトキーを使用してアクティブ化した時点で、前使用したPCからは自動的にライセンスが取り外されます。

このライセンスの利用には、PCすべてがインターネットに接続されている必要があります。ただし、プロダクトキーを特定のPCに7日間まで固定するオプションを利用できます。これにより、出張などで組織外に出る場合に、ライセンスを中断することなく使用できます。

必ずご確認ください!!!

このライセンスは、使用するPCが常にインターネットに接続している必要があります。トラフィックの少ない安定したインターネット接続であるかを必ず確認してください。

シリアル番号を登録してプロダクトキーを取得

次のシリアル番号のシングルユーザパッケージの場合：

xFxxx-6xxx-70/71/72xxxxx

このページにある操作でシリアル番号を登録し、プロダクトキーを取得して下さい。

次のシリアル番号のマルチユーザパッケージの管理者の場合：

xFxxx-6xxx-76xxxxx

Originをユーザに配布する前に、あらかじめこのページにある操作でシリアル番号を登録し、プロダクトキーの配布方法を設定してください。

ライセンスのアクティブ化

  1. Originを起動します。ライセンスのアクティベーションダイアログが開きます。
  2. 以下にプロダクトキーを入力してアクティブ化をクリックしてください のオプションを選択し、プロダクトキーを入力し、アクティブ化をクリックします。
  3. バックグラウンドでライセンスプロセスが行われます。ライセンシングが成功すると、成功のメッセージが表示されます。

ライセンスプロセスが失敗した場合は、次の操作をお試しください。

  1. 何らかの理由でライセンスプロセスがうまくいかなかった場合、メモ帳が開き、リンクが表示されます。
  2. リンクをコピーして、インターネット接続のあるPCでブラウザソフトに貼り付けます。OriginLabのWebサイトが開きます。
  3. アカウントにログインすると、ライセンステキストが生成され、Webページに表示されます。
  4. ライセンスファイルテキストをコピーして、ライセンスのアクティベーションダイアログに戻り、貼り付けます。OKをクリックします。

ライセンスの取り外し

ライセンスの取り外しは不要です。前に使用したプロダクトキーを使って他のPCでアクティブ化すると、元々使用していたPCのライセンスは自動的に取り外されます。

インターネットに接続せずにOriginを使用

上限1週間の期間、特定のPCにライセンスを固定できます。この期間内は、インターネットに接続しなくてもOriginを使用できます。

Note: PCをインターネットから切断する前に、ライセンスを固定する必要があります。

  1. Originを起動します。ヘルプ：バージョン情報を選択します。
  2. ライセンスボタンをクリックします。
  3. プロダクトキーが入力されていない場合は、入力します。
  4. 「このPCのプロダクトキーをロックして、インターネットアクセスなしで最大1週間使用できます」にチェックを付けます。
  5. アクティブ化 ボタンをクリックします。
  6. PCをネットワークから切断します。
