Name

INIFile

Remark

The INIFile class provides methods to access data stored in initialization file.

Hierarchy

  • INIFile

Examples

EX1

void INIFile_ex1()
{
    INIFile ini1("Origin.ini", FALSE);    //for program path 
    INIFile ini2("Origin.ini");                        //for INI path
    int nNumLevels1 = ini1.ReadInt("Config", "NumLevels", 0);
    int nNumLevels2 = ini2.ReadInt("Config", "NumLevels", 0);
    printf("NumLevels in Program folder ini = %d\n", nNumLevels1);
    printf("NumLevels in ini folder ini = %d\n", nNumLevels2);
}
