Description

Scaled modified Bessel function exp(x)K/4(x)

Syntax

double bessel_k_nu_scaled( double x, int nu )

Parameters

x
[input] the argument x of the function
nu
[input] the argument of the function

Return

Returns the value of scaled modified Bessel function exp(x)K/4(x)

Examples

EX1

void bessel_k_nu_scaled_ex1()
{
        double dx[6] = {3.9,1.4,8.2,6.7,0.5,2.3};
        double duu[6] =  {-3,-2,-1,1,2,3};
        double dResults[6];
        for(int ii = 0; ii<6 ; ii++)
        {
                dResults[ii] = bessel_k_nu_scaled(dx[ii],duu[ii]);
                printf("%lf:   %lf\n",dx[ii],dResults[ii]);
        }
}
