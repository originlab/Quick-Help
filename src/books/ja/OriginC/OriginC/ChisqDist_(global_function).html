Description

Inverse Chi Square cumulative distribution function

Syntax

double ChisqDist( double dp, UINT nDof )

Parameters

dp
[input] the probability
nDof
[input] the number of degrees of freedom

Return

Returns inverse Chi Square cdf for probability, dp, if succeed, else -1

Examples

EX1

void    ChisqDist_ex1()
{
    double    dp = 0.95;
    UINT    nDof = 20;
    double    val = ChisqDist( dp, nDof);
    printf("ChisqDist(%lf, %d) = %lf\n", dp, nDof, val);
}

Remark

For historical reasons, this inverse Chi Square cumulative distribution function was misnamed; Now, we developed a new set of distribution functions, whose CDFs end up with *cdf(), such as chi2cdf(), PDFs with *pdf(), and Inverse CDFs with *inv().

This function performs the same functionality as chi2inv(), and we recommend the latter.
