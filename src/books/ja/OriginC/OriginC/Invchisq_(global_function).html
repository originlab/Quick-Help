Description

Chi-Square CDF

Syntax

double invchisq( double dChiSq, UINT nDOF )

Parameters

dChiSq
[input] Value of ChiSq (ChiSq can be any Real number) for which inverse ChiSq is returned.
nDOF
[input] Degrees of freedom where nDof>0.

Return

Reurns Chi-Square CDF for dChiSq and nDof.

Examples

EX1

void invchisq_ex1()
{
    double dChiSq=10;
    int nDof=5;
    double dP;
    dP=invchisq(dChiSq,nDof); // results in dP==0.9248
    printf("InvChisq(%lf, %d) = %lf\n", dChiSq, nDof, dP);
}

Remark

For historical reasons, this Chi Square cumulative distribution function was misnamed; Now, we developed a new set of distribution functions, whose CDFs end up with *cdf(), such as chi2cdf(), PDFs with *pdf(), and Inverse CDFs with *inv().

This function performs the same functionality as chi2cdf(), and we recommend the latter.
