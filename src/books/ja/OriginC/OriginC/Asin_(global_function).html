Description

Calculate the arcsine and return the result in radians.

Syntax

double asin( double x )

Parameters

x
[input] independent variable

Return

asin(x)

Examples

EX1

void    asin_ex1()
{
    double        x = -1.;
    double        val = asin(x);
    
    printf("asin(%f) = %f\n", x, val);
}
