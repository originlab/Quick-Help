Description

Get uints label of the matrix object.

Syntax

string GetUnits()

Parameters

None.

Return

Return the string of units label.

Examples

EX1

//Get uints label of the matrix object. 
void MatrixObject_GetUnits_ex1()
{
        MatrixLayer ml = Project.ActiveLayer();
        if ( ml )
        {
                MatrixObject mo = ml.MatrixObjects(0);         
                string str = mo.GetUnits();         
                out_str(str);
        }
}
