Description

Compute the Bessel function. The _y0, _y1, and _yn routines return Bessel functions of the second kind: orders 0, 1, and n, respectively.

Syntax

double _y1( double x )

Parameters

x
[input] Floating-point value

Return

The _y1 returns a Bessel function of x.

Examples

EX1

//This program illustrates Bessel functions.

#include <origin.h>

int test_Y1()
{
    double vv;
    
    vv = _y1(0);
    out_double( "_y1(0)=", vv );        //output should be _y1(0)=-- 
    ASSERT( is_equal(vv, NANUM) );
    
    vv= _y1(11);
    out_double( "_y1(11)=", vv );        //output should be _y1(11)=0.16371
    ASSERT( is_equal(round(vv, 5), 0.16371) );
    
    return 1;
}
