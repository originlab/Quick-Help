Description

Calculates x raised to the power of y.

Syntax

double pow( double x, double y )

Parameters

x
[input] base
y
[input] exponent

Return

x to the power of y

Examples

EX1

void pow_ex1( void )
{
   double x = 2.0, y = 3.0, z;

   z = pow( x, y );
   printf( "%.1f to the power of %.1f is %.1f\n", x, y, z );
}

//Output
//2.0 to the power of 3.0 is 8.0
