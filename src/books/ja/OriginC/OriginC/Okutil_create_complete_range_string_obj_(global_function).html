Description

Create a complete range string from book descriptor, sheet descriptor and object descriptor.

Syntax

OC_API void okutil_create_complete_range_string_obj(string* pstrDestination, LPCSTR lpcszBook, LPCSTR lpcszSheet, LPCSTR lpcszObject, BOOL bMustAddObjSep = false);

Parameters

pstrDestination
[output] Range string created
lpcszBook
[input] Book descriptor
lpcszSheet
[input] Sheet descriptor
lpcszObject
[input] Object descriptor
bMustAddObjSep
[input] If TRUE, "!" will be added even both book descriptor and sheet descriptor are empty

Return

None

Examples

EX1

void okutil_create_complete_range_string_obj_ex1()
{
        string strBook = "Book1", strSheet = "Sheet1", strObject = "Col(A)";
    string strRange;    
    okutil_create_complete_range_string_obj(strRange, strBook, strSheet, strObject);
    out_str(strRange);
}
