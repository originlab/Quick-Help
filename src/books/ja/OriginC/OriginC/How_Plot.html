Template and Plot IDs

For example, if you want to make a 3D scatter plot, please create a graph window with template of "3D" first. A Way to Get Template Name

Need to specify graph template in AddPlot method when plot in Origin C. To know the graph template name, please reference the following steps.

To get the template name for Surface plot.

  1. Plot this 3D graph in Origin panel, then right click on the title of this 3D graph window, choose Save Template As...
  2. In dialog there is a editbox named Template Name, this is the template name for the type graph.

For example, the template of Color Map Surface is "CMAP".
