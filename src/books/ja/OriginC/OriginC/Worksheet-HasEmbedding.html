Version

Minimum Origin Version Required: Origin 2017 SR0

Description

Checks if worksheet has any kind of embedding, graph, note, image etc.

Syntax

BOOL HasEmbedding( DWORD dwCntrl = WORKSHEET_HAS_EMBEDDING__DATA_CELL )

Parameters

dwCntrl
[input] WORKSHEET_HAS_EMBEDDING__DATA_CELL: checks if worksheet data area has embedding;
WORKSHEET_HAS_EMBEDDING__LABEL_CELL: checks if worksheet label area has embedding;
WORKSHEET_HAS_EMBEDDING__FLOATING: checks if worksheet has any cell linking to graph

Return

TRUE if worksheet has any kind of embedding, otherwise FALSE

Examples

EX1

void HasEmbedding_ex1()
{
    Worksheet wks = Project.ActiveLayer();
    if( wks && wks.HasEmbedding() )
        out_str("wks has embedding");
}
