Description

Show or Hide display of row header labels

Syntax

BOOL ShowLabelCategoryNames( BOOL bDisplay )

Parameters

bDisplay
[input] Show (TRUE) or Hide (FALSE) display of label row names

Return

TRUE if labels are displayed, FALSE otherwise

Examples

EX1

// Display Row header Labels
void Worksheet_ShowLabelCategoryNames_Ex1()
{
    Worksheet wks = Project.ActiveLayer();
    if(wks)
        wks.ShowLabelCategoryNames(TRUE);
}

EX2

// Toggle the display of Row header Labels
void Worksheet_ShowLabelCategoryNames_Ex2()
{
    Worksheet wks = Project.ActiveLayer();
    if(wks)
        wks.ShowLabelCategoryNames(!wks.IsLabelCategoryNamesShown());
}

Remark

Show or Hide display of Row header Labels.

These are Long Name, Units, Comment and Parameters.
