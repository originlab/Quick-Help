How to build GSL DLL

1. Donwload GNU Scientific Library win32 version source codes from:

http://gnuwin32.sourceforge.net/packages/gsl.htm


2. Download and install Visual C++ 2005 Express(Also works for VC2008) from:


http://msdn.microsoft.com/en-us/express/aa975050.aspx

For the description below, lets say we install it into c:\gsl\


3. Open c:\gsl\src\gsl\1.8\gsl-1.8\VC8\libgsl.sln


5. Build the solution in release mode.


6. Then you can got two dll "libgsl.dll" and "libgslcblas.dll" in the following folder:

(source code path)\VC8\libgslcblas\Release-DLL

(source code path)\VC8\libgsl\Release-DLL


7. We only need the libgsl.dll, so copy the libgsl.dll to your Origin C code location, for example c:\OC\

From

c:\gsl\src\gsl\1.8\gsl-1.8\VC8\libgsl\Release-DLL\ libgsl.dll

to

c:\oc\
