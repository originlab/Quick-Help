Description

to add import file info to page

Syntax

int tree_add_import_file_info( TreeNode & tnTarget, Page & pgSource, int nSrc = 0 )

Parameters

tnTarget
[output] the tree node to store info
pgSource
[input] the page including import file info
nSrc
[input]temporary variable, no useful for user

Return

return 0 if successful

Examples

EX1

// To see the results, please import some files and make this window active first, 
// for example, import *.dat file to worksheet.
void tree_add_import_file_info_Ex1()
{
        Page pg = Project.Pages();
        if( pg )
        {
                Tree tr;        
                tree_add_import_file_info(tr, pg);                    
                out_tree(tr); 
        }
}
