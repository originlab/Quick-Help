Description

Clear all values from specify TreeNode.

Syntax

void tree_clear_all_value( TreeNode & tr )

Parameters

tr
[modify]TreeNode to clear values

Return

Examples

EX1

void tree_clear_all_value_ex1()
{
    Tree tr;
    tr.Math.Score.dVal = 90.5;
    tr.Math.Bookname.strVal = "Advanced Math";
    tr.English.Score.dVal = 95.5;
    tr.English.Bookname.strVal = "English 1";    
    
    tree_clear_all_value(tr);
    out_tree(tr);
}
