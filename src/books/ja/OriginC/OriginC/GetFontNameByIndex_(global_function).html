Description

Get the name of font.

Syntax

string GetFontNameByIndex( int nIndex )

Parameters

nIndex
[input] Font index

Return

A string containing the font name.

Examples

EX1

void    GetFontNameByIndex_ex1()
{
    string    strFontName;
    strFontName = GetFontNameByIndex(0);
    printf("  Font Name is: \"%s\"\n", strFontName);
}
