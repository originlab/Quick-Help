Description

Generate bandblock Responses data.

Syntax

int create_bandblock_data( int nSize, double * pResponse, double dFc1, double dFc2 )

Parameters

nSize
[input] number of points of the bandblock Responses data
pResponse
[output] the result of bandblock Responses data
dFc1
[input] the lower bound of the frequency(has been normalized, so 0<dFc1<dFc2<0.5).
dFc2
[input] the upper bound of the frequency(has been normalized, so 0<dFc1<dFc2<0.5).

Return

Returns 0 for success or error codes for failure.

Examples

EX1

#include <..\OriginLab\fft.h>
void create_bandblock_data_ex1()
{
    vector vRes(8);
    create_bandblock_data(8, vRes, 0.1, 0.3);
}
