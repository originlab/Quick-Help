Description

Modified Bessel function I0(x)

Syntax

double bessel_i0( double x )

Parameters

x
[input] the argument x of the function

Return

Returns the value of Modified Bessel function I0(x)

Examples

EX1

void bessel_i0_ex1()
{
        double dx[6] = {0.5,1,3,6,8,10};
        double dResults[6];
        for(int ii = 0; ii<6 ; ii++)
        {
                dResults[ii] = bessel_i0(dx[ii]);
                printf("%lf:   %lf\n",dx[ii],dResults[ii]);
        }
}
