Description

Set the OperationManager as batch processing.

Syntax

void SetBatchProcessing(BOOL bOn = TRUE);

Parameters

bOn
[input] Whether to set OperationManager as Batch Processing (TRUE) or not (FALSE);default value is TRUE.

Return

Examples

//Set the OperationManager as batch processing and Output if the operation manager is batch processing. 
void OperationManager_SetBatchProcessing_ex1()
{
        OperationManager opManager;
    opManager = Project.Operations;
    opManager.SetBatchProcessing(TRUE);
    bool bRet;
    bRet = opManager.IsBatchProcessing();
    if(bRet)
        out_str("Is batch processing!");
}
