システムのXファンクション

OriginはシステムのためのXファンクションを提供します

名前 簡単な説明 サンプル
cd 現在の作業フォルダを変更したり表示します Examples
cdset 現在の作業フォルダに特定のインデックスを割り当てます。あるいは、全てのインデックスとそのパスをリストします Examples
changeUFF User Filesフォルダのパスを変更します Examples
debug_log デバッグのログファイルを作成します。OriginLab社に報告する必要がある問題がある場合のみ、オンにしてください Examples
dir 現在の作業フォルダ内にある、スクリプト(ogs)とXファンクション(oxf)をリストします
dlgChkList チェックボックスが付いているダイアログを開き、そのチェックがついた項目のステータスをダイアログを閉じた時に表示します Examples
group_server グループリーダーとメンバーのためにグループフォルダの場所を設定します
groupmgr グループリーダーがグループフォルダファイルを管理するためのツールです
instOPX OriginXMLパッケージをインストールします Examples
language Originの表示言語を切り替えます
lc Xファンクションのカテゴリをリストします。あるいは、特定のカテゴリ内にあるXファンクションを全てリストします Examples
lic Originにモジュールのライセンスファイルを追加します
lx Xファンクションをリストします(名前・キーワード・場所等) Examples
mkdir 現在の作業フォルダに新しいフォルダを作成します
op_change ツリーに保存されている操作オブジェクトを入手してセットします Examples
pb プロジェクトブラウザを開きます Examples
pe_cd プロジェクトエクスプローラのディレクトリを変更します Examples
pe_dir 現在のプロジェクトエクスプローラフォルダとワークブックをリストします Examples
pe_load Originプロジェクトを現在のフォルダ内にある現在のプロジェクトにロードします
pe_mkdir 新しいフォルダを作成します Examples
pe_move 特定のフォルダのページを別の指定したフォルダに移動します Examples
pe_path プロジェクトエクスプローラのパスを検索します Examples
pe_rename ページまたはサブフォルダの名前を変更します Examples
pe_rmdir プロジェクトエクスプローラのアクティブフォルダ内にあるサブフォルダを削除します
pe_save 現在のプロジェクトからOriginのプロジェクトファイルにフォルダを保存します Examples
pef_pptslide フォルダ内のすべてのグラフをPowerPointにエクスポート Examples
pef_slideshow 選択したフォルダ内にある全てのグラフのスライドショー(全画面表示) Examples
pemp_pptslide 選択したグラフをPowerPointのスライドへエクスポートします Examples
pemp_slideshow 選択したグラフのスライドショー(全画面表示) Examples
pep_addshortcuts お気に入りフォルダに選択したウィンドウのショートカットを作成します Examples
pesp_gotofolder このページが含まれている、元のフォルダに移動します
updateUFF ユーザファイルを転送します
ux 指定した場所のXファンクションリストを更新します
