ファイル向けのXファンクション


Originはファイルを実行するXファンクションを提供します

名前 簡単な説明 サンプル
cmpfile 2つのバイナリファイルを比較し、結果を印刷します
dlgFile ファイルを開くダイアログをポップアップして、ユーザにファイルを選ばせます Examples
dlgPath パスを開くダイアログをポップアップして、ユーザにパスを選ばせます Examples
dlgSave 名前を付けて保存ダイアログで保存するよう促します。 Examples
filelog ユーザの作業をメモや記録を交えながら文字列で作成する、.txtファイルを作成します Examples
findFiles フォルダーの下のファイルを検索します。 Examples
findFolders フォルダーの下のファイルを検索します。 Examples
imgFile ファイルを開くダイアログをポップアップして、ユーザにイメージを選ばせます Examples
lvvicopy Originの Sub-VIs をLabVIEW User.lib にコピーします
oftp スクリプトでFTPサーバにアクセスできます Examples
template_saveas グラフ/ワークブック/行列ウィンドウをテンプレートとして保存します Examples
window_saveas グラフ、ワークブック、行列ウィンドウをファイルに保存します
web2file Webページをローカルファイルにコピーします Examples
