メニュー情報

解析：数学：参照データの減算

概要

あるデータセットからほかデータセットを減算

コマンドラインでの使用法

subtract_ref iy1:=[Book1]Sheet1!(1,2) iy2:=[Book1]Sheet1!(3,4) oy:=[Book1]Sheet1!(5,6) // 1と2のXYデータから3と4列のXYデータを減算し、減算したXYの対を5と6列に入れる

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

説明

このツールはある曲線から別の曲線を減算する際に使用します。2つの曲線の座標が一致しない場合、減算が実行される前に、曲線は線形補間により補間あるいは補外されます。

サンプル

  1. \Samples\Curve Fitting フォルダのGaussian.datをインポートします。
  2. B列を選択し、作図：基本の2Dグラフ： 散布図を選択してグラフを作成します。
  3. 解析：数学：参照データの減算メニューを選択して参照データの減算ダイアログを開きます。
  4. 参照データの項目でC列を選択します。そして、OKボタンをクリックします。

すると、新しい曲線が作成されます。

キーワード:ベースライン補正, 減算
