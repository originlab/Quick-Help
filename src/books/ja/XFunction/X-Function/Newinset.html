概要

インセット付きで新しいグラフを作成します。

コマンドラインでの使用法

1. newinset inset:=2;

2. newinset inset:=3 name:="Three Insets" result:=GraphName;

3. newinset inset:=2 template:="Mytemplate.otp";

  • long:ロングネームのみ

  • ls:ロングネーム＆ショートネーム
    新しいグラフのロングネームとショートネームをセットします。

入力

string

新しいグラフページのテンプレートを指定します。

説明

この関数はインセットグラフ付きの新しいグラフを作成します。インセットグラフの数、グラフページのロングネームとショートネーム、グラフページを作成するのに使用するテンプレートを指定できます。
