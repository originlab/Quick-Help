概要

フォーマットされたワークシートセルをマスクする

追加の情報

必要なOriginのバージョン: Origin 2019

コマンドラインでの使用法

wcolor2mask;

wcolor2mask Mode:=1;

wcolor2mask Mode:=2;

入力/出力

Range

<active>

変換する対象 Mode

入力

int

0
 マスクするセルをどう決定するか指定します。

オプションリスト:

  • 0=背景色付きのセル
    色塗りされたセルをマスクします。
  • 1=色付きテキストのセル
    色付きのテキスト/数値のセルをマスクします。デフォルト色の黒は対象になりません。
  • 2=条件フォーマットのセル
    条件フォーマットのセルをマスクします。

説明

このXファンクションは、テキストや背景の色付けや条件フォーマットを含む編集されたセルをマスクするために使用されます。
