MatrixObject

GUI 制御文字列

制御文字列は無用です。コントロールは、インタラクティブレンジセレクタボタン付きテキストボックスでセットされます。このボタンは行列オブジェクトのみ選択できます。

変数値データ

Xファンクションビルダのデータフィールドにおけるデフォルトデータをセットできます。

変数データ 入力 出力 入力/出力
<unassigned> デフォルト値はMatrixObject値にセットされます。 不要 デフォルト値はMatrixObjectの値にセットされ、出力には無用です。
<input> 無用 結果の値は、入力MatrixObjectに入れられます。 入力には無用です。結果の値は入力MatrixObjectに入れられます。
<new> 無用 結果の値は、新しいMatrixObjectに入れられます。 入力には無用です。結果の値は新しいMatrixObjectに入れられます。
範囲文字列 デフォルト値はMatrixObject文字列値でセットされます。 変数結果の値はMatrixObjectに入れられます。 変数値はMatrixObjectから取得できます。実行後、結果値はMatrixObjectに入れられます。

サポートされているオプション文字列

  • A:-------再計算
  • G:-------グループ
  • U:-------出力
  • V:-------可視

詳細については、 オプション文字列を確認してください。
