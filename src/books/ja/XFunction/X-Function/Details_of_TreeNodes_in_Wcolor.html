Wcolorツリーノードに関する詳細


trrule

trruleツリーはwcolorXファンクションのルールを指定します。

シンタックス: trrule.Treenode:=<value>

サンプル:

  • trrule.cond1:=4 trrule.val11:=30;
ツリーノード ラベル データ型 デフォルト 説明
cond1 条件1 int 0 最初の条件のルールを指定します。

範囲 (列)の フォーマット = 数値 の時

  • 0: 等しい
  • 1: 不等
  • 2: より大きい
  • 3: より大きいまたは等しい
  • 4: より小さい
  • 5: より大きいまたは等しい
  • 6: 間
  • 7: 次のテキストを含む
  • 8: カスタム

範囲(列)の フォーマット = 日付 / 時間 の時

  • 0: 等しい
  • 1: 不等
  • 2: 後
  • 3: 以後
  • 4: 前
  • 5: 以前
  • 6: 間
  • 7: 次のテキストを含む
  • 8: カスタム
val11 double 0 trrule.cond1:=0 から 6のとき利用できます。条件の数値または統計/関数を設定します。データ形式が日付/時間の場合、ユリウス日数を指定します。
val12 double 1 trrule.cond1:=6（間を指定)したとき利用できます。条件の数値または統計/関数を設定します。データ形式が日付/時間の場合、ユリウス日数を指定します。
text1 string trrule.cond1:=7のとき利用できます。 セルに含まれるテキストを指定します。
custom1 カスタム trrule.cond1:=8のとき利用できます。 ユーザカスタムで条件を指定するのに使用します。
join 追加条件 int 0 2番目の条件を追加するか指定し、2つの条件の関係を指定します。
  • 0: なし
  • 1: And
  • 2: Or
cond2 条件2 int 0 trrule.join:=1 または 2のとき利用できます。 2番目の条件のルールを指定します。詳細は、cond1を参照してください。
val21 double 0 trrule.cond2:=0 から 6のとき利用できます。第二条件の数値または統計/関数を設定します。データ形式が日付/時間の場合、ユリウス日数を指定します。
val22 double 1 trrule.cond2:=6（間を指定)したとき利用できます。第二条件の数値または統計/関数を設定します。データ形式が日付/時間の場合、ユリウス日数を指定します。
text2 string trrule.cond2:=7のとき利用できます。 第2条件のセルに含まれるテキストを指定します。
custom2 カスタム trrule.cond2:=8のとき利用できます。 ユーザカスタムで第2条件を指定するのに使用します。
fill 背景 int 30067455 設定した条件に合う値のセルを色付けするためのツールです。内部カラーコードまたはcolor()関数でカラー名/RGB/HTML 16進カラーコードを使用して色を指定します。
font テキストカラー int 20132812 設定した条件に合う値のセルのテキスト色を指定します。内部カラーコードまたはcolor()関数でカラー名/RGB/HTML 16進カラーコードを使用して色を指定します。
