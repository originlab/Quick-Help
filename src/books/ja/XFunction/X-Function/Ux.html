概要

指定した場所のXファンクションリストを更新します

追加の情報

この機能はOrigin8.0 SR5で更新されています。詳細は、リリースノート のセクションをご覧下さい。

コマンドラインでの使用法

ux location:=user;

場所 location

入力

int
0
 更新するXファンクションを含むフォルダの位置を指定します。

オプションリスト

  • all:全て
すべての場所
  • exe:Exe
システムフォルダ
  • group:グループ
グループフォルダ
  • user:ユーザ
ユーザフォルダ
  • custom:カスタム
カスタムフォルダ
  • project:プロジェクト
プロジェクトフォルダ

説明

このXファンクションは指定した場所にあるXファンクションの一覧を更新するのに使用します。
