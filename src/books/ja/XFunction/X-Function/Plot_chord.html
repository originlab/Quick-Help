メニュー情報

作図＞カテゴリカル:コードダイアグラム

概要

コードダイアグラムを作成します

追加の情報

必要なOriginのバージョン: Origin 2021

コマンドラインでの使用法

plot_chord rowpos:=label label:="Long Name" colpos:=selcol1 ogl:=<new template:="chord">;

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト
値		 説明
入力 irng

入力

Range
<active>
 プロットのためのZ値または既存の仮想行列を指定します。
データレイアウト format

入力

int
0
 XおよびY値をセットする方法を指定します。

オプションリスト:

  • 0=Radio:yacross:行から列へのリンク
リンクは行の指定された値から始まり、リンクは列の指定された値で終わります。
  • 1=xacross:列から行へのリンク
リンクは列の指定された値から始まり、リンクは行の指定された値で終わります。
X値/Y値 rowpos

入力

int
<自動>
 Y（またはX）値の行を指定します。YまたはX値が指定されているかどうかは、データフォーマットの値で決まります。

オプションリスト:

  • 0=none:なし
  • 1=selrow1:選択中の第1行
  • 2=wksrow1:ワークシート中の第1行
  • 3=label:列ラベル
  • 4=custom:カスタム
Y/X範囲 rowrng

入力

Range
<unassigned>
 Y（またはX）値の行を指定します。rowpos が4の時のみrowrng が指定されます。
列ラベル label

入力

string
<unassigned>
 Y（またはX）値として使用される列食べるの文字列を指定します。rowpos が3の時のみlabel が指定されます。 スクリプトで列ラベル行を指定するには、1文字を使用する必要があります。
X値/Y値 colpos

入力

int
なし
 X（またはY）値の列を指定します。XまたはY値が指定されているかどうかは、データフォーマットの値で決まります。

オプションリスト:

  • 0=none:なし
  • 1=selcol1:選択中の第1列
  • 2=wkscol1:ワークシート中の第1列
  • 3=xcol:選択列の左にあるX列
  • 4=custom:カスタム
X範囲/Y範囲 colrng

入力

Range
<unassigned>
 X(もしくはY)値の列を指定します。colrngはcolposが4に設定されているときのみ指定する必要があります。


説明

このXファンクションは、コードダイアグラムの作図に使用されます。メニュー詳細はこちらもご参照ください。
