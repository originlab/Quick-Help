概要

プロジェクトエクスプローラ（PE）のサブフォルダを削除する。

追加の情報

この機能は、Origin 2017 SR0で更新されました。フォルダパスの指定が可能になっています。詳細は、リリースノート のセクションをご覧下さい。

コマンドラインでの使用法

1. pe_rmdir folder:= "Folder1";

2. pe_rmdir folder:= "/Folder1/Folder3" folpromt:=1 pgpromt:=1;

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト
値		 説明
サブフォルダ名 folder

入力

string

削除するフォルダの名前とパスを指定します。

フォルダ削除のプロンプト folpromt

入力

int
1

フォルダ削除に対してプロンプトを出すかどうかを指定します。TRUE(1)：Originはフォルダを削除する前にユーザに確認します。

ページ削除のプロンプト pgpromt

入力

int
1

サブフォルダにあるページの削除に対してプロンプトを出すかどうかを指定します。TRUE(1)：Originはページを削除する前にユーザに確認します。

説明

このXファンクションは、プロジェクトエクスプローラのサブフォルダを削除するのに使用します。"/FolderA/FolderB" や "../FolderB"といった絶対または相対パスを使用して現在のパスを変更できます。

Note: パス指定せずにフォルダ名を指定する場合、最初に親フォルダをアクティブにしてください。

サンプル

ルートディレクトリにあるmyDirというディレクトリを削除するには、以下のコマンドを実行します。

pe_cd; pe_rmdir myDir folpromt:=1 pgpromt:=1;

または

// Origin 2017 SR0からフォルダ名を直接指定できます pe_rmdir folder:= "/myDir" folpromt:=1 pgpromt:=1;
