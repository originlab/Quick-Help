概要

グラフレイヤの単位をセットします。

コマンドラインでの使用法

//このXFを実行する前に、アクティブグラフが存在することをチェックしてください。

laysetunit layer:=1 unit:=inch;

laysetunit igp:=Graph1 layer:="2:4" unit:=page;

laysetunit igp:=Graph1 layer:=2:4 unit:=page;

レイヤ layer

入力

string

<active>

操作するレイヤを指定します。Note: 複数レイヤを指定するには、コロン(:)を使って、操作する最初のレイヤと最後のレイヤを分けます。例えば、「1:3」は、1番目、2番目、3番目のレイヤという意味になります。
単位 unit

入力

int

ページの%

指定したレイヤの単位を指定します。
オプションリスト

  • page:0 ページの0%
    この単位は、ページサイズに相対的なレイヤサイズを維持するのに役立ちます。この単位が選択されていると、レイヤの左、上、幅、高さは、ページの高さと幅のパーセンテージで指定されます。
  • inch:1 インチ
    この単位が選択されていると、レイヤの左、上、幅、高さは、インチ単位で指定されます。
  • cm:2 cm
    この単位が選択されていると、レイヤの左、上、幅、高さは、cm単位で指定されます。
  • mm:3 mm
    この単位が選択されていると、レイヤの左、上、幅、高さは、mm単位で指定されます。
  • pixel:4 ピクセル
    この単位が選択されていると、レイヤの左、上、幅、高さは、ピクセル単位で指定されます。
  • point:5 ポイント
    この単位が選択されていると、レイヤの左、上、幅、高さは、ポイント単位で指定されます。

説明

このXファンクションは、いくつかのグラフレイヤの単位をセットするのに使用できます。
