メニュー情報

列：列の移動

概要

選択した列を移動します

コマンドラインでの使用法

1. colmove rng:=col(1) operation:=last;

2. colmove rng:=col(2):col(4) operation:=first;

3. colmove rng:=col(3) operation:=left;

4. colmove rng:=col(2) operation:=right;

5. colmove rng:=col(3) operation:=pos col:=col(2);

列 rng

入力/出力

Range

<active>

移動する列を指定します。

操作 operation

入力

int

first

列に実行する操作を指定します。
オプションリスト

  • first: 最初に移動 {0}
    選択した列をワークシートの一番左に移動します。
  • last:最後に移動{1}
    選択した列をワークシートの一番右に移動します。
  • right:右に移動{2}
    選択した列を1つ右に移動します。
  • left:左に移動{3}
    選択した列を1つ左に移動します。
  • pos:特定列の前に移動{4}
    col 変数で指定された列の前に移動します。
特定列 col

入力

列

<unassigned>

operation 変数が、pos(特定列の前に移動) にセットされているときのみ利用可能です。列を移動する際の目的列の位置を指定します。

説明

このXファンクションは指定した列を同じワークシートの新しい位置に移動して、列の再配置を行います。
