概要

データベース接続情報を表示

コマンドラインでの使用法

1. dbInfo; // dump to output winodw

2. dbinfo sql:=str; // still dump connection string but SQL string put into string variable str instead

3. dbinfo ds:=strc sql:=str; // no dump, get both strings
