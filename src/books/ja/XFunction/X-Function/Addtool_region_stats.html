メニュー情報

領域の統計

概要

グラフ上に四角形または円を配置して、基本統計を計算します。

追加の情報

必要なOriginのバージョン: 8.5 SR0

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト
値		 説明
設定 trGUI

入力

TreeNode

<unassigned>

 基本統計を計算する設定を指定します。

説明

このツールは、四角形または円のROIボックスをグラフに追加して、基本統計を計算するのに使用します。統計結果は、ガジェットダイアログに表示され、ROIボックスを移動すると自動的に更新されます。このツールについての詳細は、ユーザガイドの 領域統計ツールを参照してください。
