概要

グラフウィンドウをアクティブなグラフまたはレイアウトページに挿入します。

追加の情報

必要なOriginのバージョン: Origin 9.1 SR0以降


コマンドラインでの使用法

1. insert_graph_to_layer Graph1;//Graph1をBMPオブジェクトとしてグラフ/レイアウトウィンドウに挿入する


変数

表示名 変数名 I/Oとタイプ デフォルト値 説明
追加するグラフ名 igname

入力

string

<unassigned>

 グラフウィンドウやレイアウトページに挿入するグラフウィンドウのショートネームを指定します。
グラフオブジェクト名 ogname

出力

string

   この変数は読み取りのみです。グラフに挿入されたグラフオブジェクトの名前を有します。

説明

このXファンクションはアクティブなグラフ/レイアウトウィンドウにBMPオブジェクトとして挿入する際に使用します。

関連のXファンクション

insert_wks_to_layer, add_graph_to_graph
