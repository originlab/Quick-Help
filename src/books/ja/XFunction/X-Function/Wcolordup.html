メニュー情報

ワークシート: 条件フォーマット: 重複検出

概要

ワークシートのセルの範囲内で重複した項目をマークします

コマンドラインでの使用法

1. wcolordup rng:=[Book1]Sheet1! name:="C1R1-C0R0" fill:=32167617 font:=26830387;

2. wcolordup rng:=[Book1]Sheet1!4[1]:7[11] name:="C4R1-C7R11" fill:=color(LT Yellow) font:=color(Orange);

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

範囲 rng

入力

Range

<active> 条件フォーマットを適用するワークシート内のセルの範囲を指定します。
名前 name

入力

string

<unassigned> 選択された範囲の名前を指定します。
色を適用するシート extend

入力

int

0 条件指定でシート内すべての範囲に色を付けるかどうかを指定します。選択した場合、適用された色はすべての入力行範囲に拡大します。
背景 fill

入力

int

32167617 重複したセルをマークする背景の色を指定します。内部カラーコードまたはcolor()関数でカラー名/RGB/HTML 16進カラーコードを使用して色を指定します。
テキストカラー font

入力

int

26830387 重複したセルのテキストをマークする色を指定します。内部カラーコードまたはcolor()関数でカラー名/RGB/HTML 16進カラーコードを使用して色を指定します。

説明

このXファンクションを使用して、重複している値のセルに色を付けます。

関連 X ファンクション

wcolor, wheatmap
