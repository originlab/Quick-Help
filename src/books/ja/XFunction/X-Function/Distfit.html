メニュー情報

統計：記述統計：分布フィット

概要

データをパラメトリックな分布でフィットします

追加の情報

必要なOriginのバージョン: Origin 2015 SR0以降

これはOrigin Proのみの機能です。

コマンドラインでの使用法

1. distfit stat.moment:=0 stat.sum:=1 test.mtest.mean:=35 test.mtest:=1 dist.wb:=1 dist.gamma:=1 plot.plotinone:=1 ksmod:=1 plotdata:=<new>;//Carry out distribution fit on the active data range

2. distfit irng:=[Book1]Sheet1!B group:=[Book1]Sheet1!A stat.perc:=0 stat.sum:=1 stat.kurt:=1 test.mtest:=0 test.stest.var:=20.14 test.stest.cis:="" test.stest:=1 type:=1 dist.bino:=1 plot.cdf:=1 ks:=1 plotdata:=<new>;

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

データ範囲 irng

入力

Range

<active>

入力データ範囲（複数列可）を指定します。Origin 2020b以降、[Book]Sheet!(N1:N2) 形式の短縮構文を使用できます。ここで、N1は列の連続した範囲の開始の列番号、N2は終了の列番号です。[Book]Sheet!([Book]Sheet!N1:N2,[Book]Sheet!N3:N4) の形式の非連続データも使用できます。

グループ範囲 group

入力

Range

<unassigned>

グループ範囲を1列のみ指定します（オプション）。

記述統計 stat

入力

TreeNode

<unassigned>

指定した記述統計量を出力するかどうか指定します。各サブノードは、量の参照を制御します。利用可能なサブノードは、

  • perc: 分位数値（最小, 第1四分位数(Q1), 中央値, 第3四分位数(Q3), 最大, 四分位範囲(Q3 - Q1), 範囲(最大 - 最小)を含む）
  • moment:モーメント（N, N 欠損, 平均値, 標準偏差(SD), 平均のSE, 95%平均の信頼上限/下限を含む）
  • var:分散
  • sum:合計
  • skew:歪度
  • kurt:尖度
  • cv:変動係数
  • ss: 未修正二乗和
  • ss: 修正された二乗和
  • mode:モード(最頻値)
基本の検定 test

入力

TreeNode

<unassigned>

t検定またはz検定をするかどうか、検定条件を指定します。利用可能なサブノードは、

  • mtest: 平均値の検定を実行するか指定します。0: しない, 1: t検定, 2: z検定
    mtest.mean:検定される仮説の平均
    mtest.tail:対立仮説0: 平均<>検定される平均, 1: 平均 > 検定される平均, 2: 平均 < 検定される平均
    mtest.std: z検定で使用する標準偏差値を指定します。(.mtest = 2)
    mtest.cim: 平均の信頼区間を計算するかどうかを指定します。
    mtest.cim.level: パーセント値での信頼区間レベルを指定します。例：95は95%信頼区間です。
  • stest: 分散のカイ二乗検定を実行するか指定します。
    stest.var: 検定される仮説分散を指定します。
    stest.tail: 対立仮説を指定します。0: 分散 <> , 1: 分散 > , 2: 分散 < .
    stest.cis: 標本分散の信頼区間を計算するかどうかをを指定します。
    stest.cis.level: パーセント値での信頼区間レベルを指定します。例：95は95%信頼区間です。
分布タイプ type

入力

int

0

連続分布あるいは離散分布を実行するか指定します。

オプションリスト

  • 0:連続分布
  • 1:離散分布
分布 dist

入力

TreeNode

<unassigned>

入力データをフィットするために、1つ以上の分布を選択します。分布については、Originヘルプページを確認してください。サブノードは、分布を参照します。

連続分布：

  • norm:正規
  • logn:対数正規
  • wb:ワイブル
  • exp:指数
  • gamma:ガンマ

離散分布：

  • bino:二項
  • poi:ポアソン
標本サイズ sample

入力

string

"10"

二項分布が使用されるとき(dist.bino = 1)に利用可能で、分布の標本サイズを指定します。

プロット plot

入力

TreeNode

<unassigned>

結果シートに指定したプロットを出力するかどうか指定します。各サブノードは、分布を参照します。

  • box: ボックスチャート
  • Hist:: ヒストグラム
  • cdf: 累積分布関数プロット
  • pp: 確率プロット(連続分布で利用可能)
    pp.sig: 確率プロットの有意水準を指定します。
  • plotinone: 1つのグラフに複数データ範囲からのプロットを配置するか指定します。これは、複数データ範囲を比較する助けになります。ヒストグラムの場合、積上げ形式で同じビンサイズになります。
Kolmogorov-Smirnov ks

入力

int

0

Komogorov-Smirnov検定を実行するかどうかを指定します。

修正Kolmogorov-Smirnov ksmod

入力

int

0

修正Komogorov-Smirnov検定を実行するかどうかを指定します。

Anderson-Darling ad

入力

int

0

修正Anderson-Darling検定を実行するかどうかを指定します。

分布フィットレポート rt

出力

ReportTree

<new>

分布フィット結果を出力するワークシートを指定します。

プロットデータ plotdata

出力

ReportData

<optional>

プロットされたデータを出力するワークシートを指定します（Originはプロットが作成された時に表形式データを生成します）。

説明

Originヘルプの、統計>>記述統計の分布フィットの章を確認してください。

アルゴリズム

アルゴリズムは、このOriginヘルプファイルを参照してください。

キーワード:正規, 対数正規, ワイブル, 指数, ガンマ, 二項, ポアソン, 累積分布, 確率プロット
