Image

GUI 制御文字列

制御文字列は無用です。コントロールは、インタラクティブイメージセレクタボタン付きテキストボックスでセットされます。

変数値データ

Xファンクションビルダのデータフィールドにおけるデフォルトデータをセットできます。

変数データ 入力 出力 入力/出力
<input> 無用 変数値は入力イメージに出されます。 入力としては無用で、結果をアクティブなImageに出されます。
<new> 無用 結果は新しいImageに出力されます。 この変数値は、入力としては無用で、結果は新しいImageに出力されます。

サポートされているオプション文字列

  • A:-------再計算
  • G:-------グループ
  • U:-------出力
  • V:-------可視

詳細については、 オプション文字列を確認してください。
