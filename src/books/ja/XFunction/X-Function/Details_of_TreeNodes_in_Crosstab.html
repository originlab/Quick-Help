Crosstabツリーノードに関する詳細


ctable

ctableツリーは、どの統計で計算して分割表に出力するか指定します。

サンプル:

  • ctable.count=0;
ツリーノード ラベル データ型 デフォルト 説明
count 度数 int 1 この統計を分割表に出力するかどうかを指定します。
ecount 期待度数 int 1 この統計を分割表に出力するかどうかを指定します。
rcount 行カウントのパーセント int 1 この統計を分割表に出力するかどうかを指定します。
ccount 列カウントのパーセント int 1 この統計を分割表に出力するかどうかを指定します。
tcount 合計カウントのパーセント int 1 この統計を分割表に出力するかどうかを指定します。
res 残差 int 1 この統計を分割表に出力するかどうかを指定します。
stdres 標準化残差 int 1 この統計を分割表に出力するかどうかを指定します。
adjres 調整済み残差 int 1 この統計を分割表に出力するかどうかを指定します。


massocial

massocialツリーは、どの統計で計算して、分割表の変数間の関連の強さを評価をするのかを指定します。

サンプル:

  • massocial.gamma=1;
ツリーノード ラベル データ型 デフォルト 説明
ccoeff C係数 int 0 この統計を計算するか指定します。
phi Phi int 0 この統計を計算するか指定します。
cramer クラマーのV int 0 この統計を計算するか指定します。
lambda ラムダ int 0 この統計を計算するか指定します。
ucoeff 不確かさの係数 int 0 この統計を計算するか指定します。
gamma ガンマ int 0 この統計を計算するか指定します。
kendall ケンドールのタウbタウc int 0 この統計を計算するか指定します。
somer Somer's D int 0 この統計を計算するか指定します。
