概要

現在の作業ディレクトリにあるスクリプト(ogs)およびXファンクション(oxf)を一覧表示します。

追加の情報

このXファンクションは、自動GetNダイアログには対応していません。

コマンドラインでの使用法

1. dir;

2. dir pattern:="*.txt";

3. dir pattern:="a?c.*";

変数

 pattern

入力

string

<unassigned>

空白、*、abc (abc.*と同じ)、a*、a*.h、*.oxf、a?c (a?c.*と同じ)のような一致のパターン

引数無しのdirコマンドは、特殊で、ディレクトリだけでなく、oxfやogsのような実行可能なファイルのみを一覧表示します。

 

 cwd

出力

string

<unassigned>

現在の作業ディレクトリを取得

説明

現在の作業ディレクトリのファイルやサブディレクトリを一覧表示します。
