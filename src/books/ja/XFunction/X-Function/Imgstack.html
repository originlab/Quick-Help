メニュー情報

複数画像ファイル

概要

複数の画像ファイルに接続して1つのイメージウィンドウに表示します。

追加の情報

必要なOriginのバージョン: Origin 2022


コマンドラインでの使用法

1. imgstack path:=system.path.program$+"\Samples\Image Processing and Analysis";

2. imgstack src:=files fname:="C:\Myfile1.tif%(CRLF)C:\Myfile2.tif";

ソース src

入力

int
0
 インポートする画像ファイルのソースを指定します。オプションリスト：
  • 0=folder:指定したフォルダ内のファイル
  • 1=files:指定したファイル
画像フォーマット type

入力

int
0
 画像形式を指定します。ソース = 指定したフォルダ内のファイルの時利用できます。フォルダ内を検索して指定された画像形式の画像をインポートします。オプションリスト：
  • 0=tif:TIFF,
  • 1=png:PNG,
  • 2=jpg:JPEG
ファイル名 fname

入力

string
fname$
 インポートするファイルを指定します。ソース = 指定したファイルの時利用できます。
フォルダ path

入力

string

 画像形式が一致する画像のインポート元のフォルダを指定します。ソース = 指定したフォルダ内のファイルの時利用できます。

説明

このXファンクションは複数の画像ファイルを各ファイルを1フレームとした1つのイメージウィンドウとしてスタックするために使用できます。画像ファイルのインポートにはデータコネクタが使用されます。画像は、同じ寸法である必要があります。詳細は、このページを参照して下さい。
