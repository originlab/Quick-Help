メニュー情報

ガジェット：交差

概要

2つ以上のプロット上に四角形を置いて交差を実行します

追加の情報

必要なOriginのバージョン: Origin 8.5 SR0以降

変数

表示

名		 I/O
と
データ型		 デフォルト
値		 説明
設定 trGUI

入力

TreeNode
<unassigned>
 交差を実行するための設定を指定します。

説明

このXファンクションは、プロット上に四角形を置き、交点の確認を行います。四角形を移動すると、この領域内の交差する点を確認できます。詳細の情報は、Originヘルプの交差ガジェットを確認してください。
