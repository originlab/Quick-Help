概要

Y列の最後から欠損値を削除します。

追加の情報

このXファンクションは、自動GetNダイアログには対応していません。

コマンドラインでの使用法

trimright iy:=(1,2); // remove elements having the value NANUM from the right end of col(2)

説明

このXファンクションはY列の最後から欠損値を削除するのに使うことができます。

対応するX列に、最後からの欠損値が含まれている場合、Y列も削除されます。
