メニュー情報

分析テンプレートにWordブックマークを追加

概要

Wordテンプレートからのブックマークを分析テンプレートに追加する

追加の情報

必要なOriginのバージョン:2016SR0


コマンドラインでの使用法

1.指定したWordテンプレートからのブックマークを分析テンプレートに追加する

string TemplatePath = system.path.program$ + "Samples\Batch Processing\Sensor Analysis.ogw";

string wordPath$ = system.path.program$ + "Samples\Batch Processing\Sensor Analysis Report.dotx";

word2ogw word:=wordPath$ bookmark.Sel:={fittedGraph, RSquare} mode:=template template:=TemplatePath$;

変数

名 		変数

と

Wordテンプレート word

入力

string

<unassigned>

ロードするWordテンプレート (.DOCX, .DOCM, .DOC, .DOTX, .DOTM, .DOT) を指定します。

ブックマークを選択 bookmark

入力

TreeNode

<unassigned>

ツリーノードで分析テンプレートに追加するブックマークを指定します。

  • .Sel
    ブックマークを選択
追加先 mode

入力

int

0

Wordブックマークを追加する分析テンプレートワークブックのソースを指定します。オプションリスト

  • active{0}:アクティブな分析テンプレートウィンドウ
    アクティブな分析テンプレートワークブック(.OGW)を使用します。
  • template{1}：分析テンプレートをロード
    特定の分析テンプレートワークブック(.OGW)をロードします。
分析テンプレート template

入力

string

<unassigned>

追加先で分析テンプレートをロードが選択されている場合のみ利用できます。Wordブックマークを追加する分析テンプレートを指定します。

説明

この機能は、Wordテンプレートからのブックマークを分析テンプレートに追加します。このような分析テンプレートのバッチ処理では、レポートとしてWordテンプレートに結果を出力できます。
