メニュー情報

元のフォルダに移動

概要

このページがあった元のフォルダに移動します。

追加の情報

このXファンクションは、自動GetNダイアログには対応していません。

コマンドラインでの使用法

 

pesp_gotofolder dump:=0

 

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト
値		 説明
  dump

入力

int

0

ショートカットページの元のページがあるフォルダに移動します。

説明

このXファンクションは、ショートカットページがある元のフォルダを開くのに使用します。ショートカットページがアクティブなとき、コマンドウィンドウで次のコマンドを使います。

pesp_gotofolder dump:=0

元のフォルダに移動します。
