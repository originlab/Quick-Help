メニュー情報

編集: 検索/ 置換

概要

検索と置換ダイアログを開きます

追加の情報

自動GetNダイアログとして設計されていません。

必要なOriginのバージョン: 8.1 SR0以降

コマンドラインでの使用法

1. findreplace option:=0; //検索タブをアクティブにした検索と置換ダイアログを開く

2. findreplace option:=1; //置換タブをアクティブにした検索と置換ダイアログを開く

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

検索または置換 option

入力

int

0
 検索と置換ダイアログのどちらのタブをアクティブにするか指定します。

オプションリスト:

  • find:検索
    データを検索するための検索タブをアクティブにします。
  • replace:置き換え
    ある値を他の値に置き換えるために置き換えタブをアクティブにします。
検索内容 str

入力

string

0

検索する文字列を指定します。

説明

このXファンクションは、検索と置換ダイアログを開くために使用できます。このダイアログには検索タブと置き換えタブがあります。 前者は検索のために使用し、後者は置き換えを実行します。ダイアログのそれぞれのタブにいくつかのオプションがあります。
