メニュー情報

ファイル： エクスポート： ASCII...

概要

行列データをASCIIファイルとしてエクスポートします。

コマンドラインでの使用法

1. expMatASC path:="c:\signal";

2. expMatASC type:=all path:="c:\signal";

3. expMatASC type:=dat path:="c:\signal" separator:=comma;

4. expMatASC im:=[mbook1]1!1 path:="c:\signal";

  • All Files(*.*){3}

説明

expMatASC関数は、行列データをASCIIファイルにエクスポートします。これは、メニューまたはコマンドウィンドウから実行できます。

  • メニュー:ファイル： エクスポート： ASCII...を選択します。
  • コマンドウィンドウ: コマンドウィンドウに expMatASC -d と入力します。

出力形式は .dat, .txt, .csvが可能です。また、必要に応じて区切り文字またはその他のエクスポートオプションを指定することができます。

サンプル

次の例は、アクティブ行列をテキストファイル(c:\test.txt)にエクスポートするものです。

1Originメニューからファイル：エクスポート:ASCIIを選びます。これは、エクスポートオプションを調整するexpMatASC ダイアログボックスを開きます。

2「ファイルの種類」ドロップダウンリストから「テキストファイル *.txt」を選択します。「ファイルパス」に「c:\test.txt」と入力します。OK をクリックして、ダイアログを閉じます。

次のスクリプトコマンドは、ダイアログを開かずにアクティブ行列をASCIIファイル (c:\test.dat) にエクスポートします。

expMatASC path:="c:\test.dat" type:=0;
