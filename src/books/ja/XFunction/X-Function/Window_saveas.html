メニュー情報

ファイル: ウィンドウの新規保存

概要

グラフ、ワークブック、行列ウィンドウをファイルに保存します

コマンドラインでの使用法

1. window_saveas pg:=Graph1 fname:="OffsetY" filepath:="C:\My Windows";

変数

表示
名 		変数
名 		I/O
と
データ型 		デフォルト
値 		説明
ページ pg

入力

ページ

<active>
 ファイルとして保存するページを指定します。ページを指定しない場合、デフォルトでアクティブページが使用されます。
ファイルパス filepath

入力

string

<自動>
 ファイルパスを指定します。デフォルトのパスは、User Filesフォルダです。存在しないパスを指定するとスクリプトエラーを起こします。
ファイル名 fname

入力

string

<自動>
 ファイル名を指定します。デフォルトはページショートネームです。
ファイルの種類 ftype

入力

int

0
 ファイルタイプを指定します。
  • 0: ワークブック (*.ogwu), グラフ (*.oggu), 行列 (*.ogmu)
  • 1: ワークブック (*.ogw), グラフ (*.ogg), 行列 (*.ogm)
プレビューイメージ preview

入力

string

<unassigned>
 ページのコメントを指定します。
コメント comment

入力

string

<unassigned>
 ファイルのプレビューイメージを指定します。これはオプションです。
