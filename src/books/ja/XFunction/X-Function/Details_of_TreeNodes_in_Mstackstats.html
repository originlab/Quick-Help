Mstackstatsツリーノードに関する詳細

 

settings

trstats ツリーは mstackstats Xファンクションの全ての設定を指定します。

シンタックス: trstats.Treenode:=<value>

サンプル:

  • trstats.n:=1
ツリーノード ラベル データ型 デフォルト 説明
mean 平均値 int 1 シート内のすべてのオブジェクトの平均値を使って行列オブジェクトを生成するか指定します。
sd SD int 0 シート内のすべてのオブジェクトの標準偏差値を使って行列オブジェクトを生成するか指定します。
n N int 0 ソース行列シート内のオブジェクト数を使用して行列オブジェクトを生成するかどうかを指定します。
min 最小値 int 0 シート内のすべてのオブジェクトの最小値を使って行列オブジェクトを生成するか指定します。
max 最大値 int 0 シート内のすべてのオブジェクトの最小値を使って行列オブジェクトを生成するか指定します。
median 中央値 int 0 シート内のすべてのオブジェクトの中央値を使って行列オブジェクトを生成するか指定します。
sum 合計 int 0 シート内のすべてのオブジェクトの合計値を使って行列オブジェクトを生成するか指定します。
