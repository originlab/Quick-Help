GUI 制御文字列

制御文字列は無用です。コントロールは、インタラクティブセレクタボタンでセットされます。

変数値データ

Xファンクションビルダのデータフィールドにおけるデフォルトデータをセットできます。

変数データ 入力 出力 入力/出力
<new> 無用 結果の値は新しいWorksheetに入れられます。 入力としては無用です。結果の値は新しいWorksheetに入れられます。
<input> 無用 結果の値は、入力Worksheetに入れられます。 入力には無用です。結果の値は入力Worksheetに入れられます。

サポートされているオプション文字列

  • A:-------再計算
  • G:-------グループ
  • U:-------出力
  • V:-------可視
  • Z:-------自動サポート

詳細については、 オプション文字列を確認してください。
