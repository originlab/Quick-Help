概要

グラフ上のリンクテーブルをワークシートの内容に更新します。

追加の情報

これは、グラフ上の表をダブルクリックした時に、表の更新ボタンの動作をするコマンドです。このXファンクションは、自動GetNダイアログには対応していません。

コマンドラインでの使用法

1. wks_update_link_table;

2. wks_update_link_table iw:=[table1];

入力

int

1

 これは、表のリンクを更新したあとグラフをアクティブにするかどうかを指定します。

説明

このXファンクションはテーブル内の変更を更新します。

サンプル

グラフの標準メニューにある新しいリンクテーブルボタンを使ってテーブルを追加します。テーブルをアクティブにし、テーブルにいくつか変更を行います。そして、コマンドウィンドウに wks_update_link_table と入力します。 これは、グラフ上のテーブルの変更を更新します。
