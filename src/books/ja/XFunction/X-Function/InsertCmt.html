メニュー情報

ノートを挿入

概要

ワークシートのセルにノートを追加します

追加の情報

必要なOriginのバージョン: Origin 2022以降

自動GetNダイアログとして設計されていません。


コマンドラインでの使用法

1. insertcmt str:="This is a peak";

2. insertcmt str:="New Data" orng:=[Book1]Sheet1!col(1)[2];

 ノートが挿入されるワークシートセルを指定します。

このシンタックスを参照してください。

説明

このXファンクションは選択されたワークシートセルにノートを追加する場合に利用されます。もしくは

  • ワークシートセルをクリックしてミニツールバーからノートを挿入を選択する、もしくは
  • ワークシートセル上で右クリックしてノートを挿入を選択します。


insertcmtを実行してワークシートセルにノートを追加したのち、そのセルを選択することでさらに編集したり、ノートを削除することができます。

  • ノートにあるコメントを直接編集する場合は、 
    s.Notes$
  • ノートウィンドウで個別にノートを開く場合は、 
    s.ed = 2;
  • ノートを削除する場合は、 
    s.Notes$="";
