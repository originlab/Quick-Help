メニュー情報

作図 > カテゴリカル: トレリスプロット

概要

複数パネルのトレリスプロットをグループ変数を使用して作成します

追加の情報

必要なOriginのバージョン: Origin 2017

コマンドラインでの使用法

1. plot_group type:=scatter horz:=col(2);

2. plot_group type:=3 vert:=!3 color:=!4;

グラフタイプ type

入力

int

作成したいグラフの種類を選択します。

オプションリスト:

  • linesymb(2):Line+Symbol

  • bar(4):Bar
    横棒グラフ
  • bridge(5):Bridge Chart

  • stackbarp(9):100% Stacked Bar

水平 horz

入力

Range

<optional>

入力データをカテゴリ化するために使用されるグループ列を指定し、水平配置のパネルで表示されます。デフォルトでは、システム変数@NOPによって決定された50パネルまでです。

垂直 vert

入力

Range

<optional>

入力データをカテゴリ化するために使用されるグループ列を指定し、垂直配置のパネルで表示されます。 デフォルトでは、システム変数@NOPによって決定された50パネルまでです。

データポイントの色付けのための変数 color

入力

Range

<optional>

カラーマップするプロットに使う列を指定します。

グラフテンプレート template

入力

string

<自動>

グループ化グラフを作成するテンプレートを指定します。デフォルトで使用されるテンプレートはgrouped.otp です。(Originのプログラムフォルダにインストールされています)

説明

このXファンクションでトレリスプロットを作成します。入力データは指定したグループ列によって複数パネルにグループ化されます。詳細はトレリスプロットを参照してください。
