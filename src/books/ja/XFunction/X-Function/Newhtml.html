概要

HTMLダイアログを作成します

追加の情報

必要なOriginのバージョン: Origin 2017 SR2以降

コマンドラインでの使用法

1. newHTML;

2. newHTML -dp; \\ HTML dialog parentとしてコードビルダフレームを使用

3. newHTML title:=MyExample cname:=MyEx opj:=0;

4. newHTML resize:=1 dlgw:=500 dlgh:=500;

5. newHTML dlgtype:=1 title:=MyGraphExample cname:=MyGrphEx opj:=1 pgshow:=1 pgname:=MyGraph;

説明

このXファンクションは、HTMLダイアログを簡単に素早く作成する助けとなります。これはHTMLダイアログを作成するための簡単なオプションを提供しています。
HTMLダイアログについての詳細は、このチュートリアルを確認してください。

サンプル

  1. 新規プロジェクトでBook1を開きます。
  2. 列A、列Bを選択して右クリックし、ショートカットメニューから列値の一律設定：行番号を選択します。
  3. 列Bを選択して、Originメニューから作図> 基本の2Dグラフ：折れ線を選択してGraph1を作成します。
  4. プロジェクトをMyExample.opjとして保存します。
  5. Originメニューからウィンドウ：スクリプトウィンドウを選択します。
  6. スクリプトウィンドウで newHTML –d を入力し、ENTERを押してXファンクションダイアログを開きます。
  7. ダイアログで、下図のように設定し、OKをクリックしてダイアログを閉じます。
  8. プロジェクトを再度保存してOriginを終了します。
  9. MyExample.opjをダブルクリックして開きます。
  10. HTMLダイアログが開きます。
  11. 適用ボタンをクリックすると、Origin CメッセージボックスとWebサイトメッセージがひとつずつ表示されます。
