メニュー情報

解析: 数学: トレース補間

概要

XYデータのトレース/周期的な補間を実行します

コマンドラインでの使用法

1. interp1trace iy:=Col(2) method:=spline npts:=50;

2. interp1trace iy:=Col(2) method:=bspline;

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

ポイント数 npts

入力

int

<自動>

補間するポイントの数を指定します。ポイント数（npts）が指定されていない場合、デフォルトの<auto> = min( n*5, 1000)です。ここで、nは入力ポイントの数です。つまり、ポイント数nが200未満の場合、npts=5*n,、それ以外の場合はnpts=1000です。

説明

曲線が周期的な場合、アルゴリズムはX 座標の隣り合うデータポイントに基づいた曲線を補間するので、一般的な補間手法をは不適切です。これは周期的な傾向を無くす原因となります。

このXファンクションは、曲線を補間し、周期性を維持します。

サンプル

1. \Samples\Mathematics\ Circle.dat ファイルをインポートします。

2. データを選択し、散布図をプロットします。

3. メニューから解析：数学：トレース補間を選択し、ダイアログを開きます。

4. 手法で3次B-スプラインを選択し、ポイント数の編集ボックスには50を入力します。

5. OKボタンをクリックして実行します。
