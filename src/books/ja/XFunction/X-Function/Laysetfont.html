概要

レイヤ内のテキストの表示スケールを1に固定します。

追加の情報

必要なOriginのバージョン: 8.1 SR0

コマンドラインでの使用法

 

laysetfont option:=page;

 

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト
値		 説明
目的のグラフ option

入力

int

1

 どのレイヤのテキストの表示スケールを1にするかを指定します。

オプションリスト:

  • layer:現在のレイヤ
    アクティブグラフレイヤのみ
  • page:現在のページ
    アクティブグラフのすべてのレイヤ

 

説明

このXファンクションは、テキストの表示スケール係数を1に固定するために使います。いったん使用すると、 固定倍率が1になり、レイヤの大きさが変わったとしても、レイヤ内のテキストは常に元のサイズで表示されます。グラフがアクティブなときに利用でき、アクティブレイヤまたはアクティブグラフページ内のすべてのレイヤに対して適用できます。
