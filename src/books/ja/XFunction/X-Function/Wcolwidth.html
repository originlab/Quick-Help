メニュー情報

列を右クリックし、コンテキストメニューから、列幅を選択

概要

ワークシートの列の幅を更新します

コマンドラインでの使用法

wcolwidth irng:=[Book1]Sheet1!Col(3) width:=20;

変数

表示

名 		I/O
と
データ型 		デフォルト
値 		説明
列 irng

入力

Range

<active>

列を指定します。

幅 width

入力

double

7

列の幅を指定します。デフォルトフォントの比率に準じた値を入力します。wcolwidth が7の場合、システムデフォルトフォントの7文字を1つの列に表示できます。

説明

この関数は、ワークシート列の幅を更新します。
