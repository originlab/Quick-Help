メニュー情報

列の交換

概要

指定した2列の位置を交換します。

コマンドラインでの使用法

colswap rng:= [Book1]Sheet1!(3,1);

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト
値		 説明
列 rng

入力/出力

Range

<active>

 交換する列を指定します。2列以上選択されている場合、最も左の列と最も右の列が入れ替わります。

説明

このXファンクションは選択した2つの列を入れ替えます。
