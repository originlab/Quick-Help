Mroi2matツリーノードに関する詳細

 

settings

trstats ツリーはmroi2mat Xファンクションの全ての設定を指定します。

シンタックス: trstats.Treenode:=<value>

サンプル:

  • trstats.n:=1
ツリーノード ラベル データ型 デフォルト 説明
mean 平均値 int 1 すべてのオブジェクトから内部ROIの平均値を持つ行列オブジェクトを生成するか指定します。
sd SD int 0 すべてのオブジェクトから内部ROIの標準偏差を持つ行列オブジェクトを生成するか指定します。
n N int 0 ソース行列シート内のオブジェクト数を使用して行列オブジェクトを生成するかどうかを指定します。
min 最小値 int 0 すべてのオブジェクトから内部ROIの最小値を持つ行列オブジェクトを生成するか指定します。
max 最大値 int 0 すべてのオブジェクトから内部ROIの最大値を持つ行列オブジェクトを生成するか指定します。
median 中央値 int 0 すべてのオブジェクトから内部ROIの中央値を持つ行列オブジェクトを生成するか指定します。
sum 合計 int 0 すべてのオブジェクトから内部ROIの合計値を持つ行列オブジェクトを生成するか指定します。
