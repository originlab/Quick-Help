filltext

概要

指定した範囲のセルにランダムな文字を入力します。

コマンドラインでの使用法

1. filltext

2. filltext group:=52

3. filltext group:=10 rng:=[Book1]Sheet1!Col(A)

変数

グループ group

入力

int

26

英語のアルファベットで生成されるグループ
入力 rng

入力/出力

Range

<active>

ランダムな文字で入力する範囲を指定します。

説明

この関数は指定した範囲にランダムな文字を入力します。文字にグループ番号をセットすることができます。
