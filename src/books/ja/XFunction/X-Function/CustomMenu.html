メニュー情報

カスタムメニューオーガナイザ

概要

カスタムメニュー編集用ダイアログを開きます

追加の情報

必要なOriginのバージョン: Origin 8.1 SR0以降

Origin 8.6 SR0 にて更新Xファンクションは Auto GetNダイアログ用にデザインされていません。

コマンドラインでの使用法

  1. customMenu;
  2. customMenu imp;

変数

表示名 変数名 I/Oとタイプ デフォルト値 説明
メニューグループ group

入力

int

-1

この引数の値を指定しない場合(デフォルト値である-1を使用する場合)、カスタムメニューオーガナイザを開きます。オプションリスト

  • imp:インポート
    インポートメニューのカスタマイズ ダイアログを開きます。
  • exp:エクスポート
    後程使用するために確保済み

説明

このXファンクションはカスタムメニューオーガナイザダイアログとインポートメニューのカスタマイズダイアログを開くために使われます。
