メニュー情報

微分

概要

グラフに矩形を配置して、その範囲の微分を実行します。

追加の情報

必要なOriginのバージョン: 8.5 SR0

 

変数

表示

名		 I/O
と
データ型		 デフォルト
値		 説明
設定 trGUI

入力

TreeNode

<unassigned>

 微分を実行する設定を指定します。

説明

このXファンクションは、グラフに四角形を配置して、すばやく微分を実行するのに使われます。この四角形を移動することで、派生グラフウィンドウ内のデータプロットの微分を観測することができます。微分についての詳細は、ユーザガイドの 微分ツールを参照してください。

関連のXファンクション

differentiate, integ1, integ2
