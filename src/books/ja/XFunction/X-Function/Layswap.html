概要

2つのグラフレイヤの位置と順序を入れ替えます

コマンドラインでの使用法

// このXファンクションを実行する前に、少なくとも2つ以上のレイヤを持つアクティブグラフがあることを確認してください。

layswap igl1:=Layer1 igl2:=Layer2;

layswap igl1:=Layer1 igl2:=Layer2 reorder:=1;

変数

表示
名 		変数
名 		I/O
と
データ型 		デフォルト
値 		説明
レイヤA igl1

入力

GraphLayer

<unassigned>

最初のグラフレイヤの名前を指定します。

レイヤB igl2

入力

GraphLayer

<unassigned>

2番目のグラフレイヤの名前を指定します。

順序変更 reorder

入力

int

0

各レイヤのデータプロット位置の入れ替えに加え、レイヤの順序も入れ替えるか制御します。

説明

このXファンクションは、データプロットの位置を2つのレイヤの順序を入れ替えるために使用します。
