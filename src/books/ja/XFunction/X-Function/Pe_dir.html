概要

現在のプロジェクトエクスプローラフォルダとページをリストします

追加の情報

Xファンクションは Auto GetNダイアログ用にデザインされていません。

コマンドラインでの使用法

1. pe_dir;

2. pe_dir page:=W;

3. pe_dir name:=B*

4. pe_dir name:="B*" recursive:=1 display:=1 sensitive:=1;

5. pe_dir recursive:=1 display:=1 sep:=##;

  • 0

  • 1
サブフォルダを検索します。

入力

int

0

リストの表示形式を指定します。recursive=1の時利用できます。
オプションリスト

  • 0
ツリーリストとして表示します。
  • 1
説明付きツリーリストとして表示します。(フォルダでは、説明はCommentsで、ウィンドウでは説明はLongNameです)
Note:フォルダコメントに複数行ある場合、最初の行がツリーリストにに表示されます。
  • 2
フラットリストとして表示します。

入力

0

ページ名表示を大文字小文字の区別をするかどうか指定します。
オプションリスト

  • 0
大文字小文字を区別しない。
  • 1
大文字小文字を区別する。
区切り文字 sep

入力

string

///

説明の区切り文字を指定します。display=1 でフォルダとウィンドウの説明を表示するときのみ利用できます。

Note:左括弧（例えば、(, [, { , < ) を区切り文字として使用する場合、右側の区切り文字の右かっこに変換されます。

説明

このXファンクションは、現在のフォルダのページ/サブフォルダを表示するために使用されます。

サンプル

全てのサブフォルダとページのリストを表示する場合

pe_dir

全てのページのリストを表示する場合

pe_dir name:=*

全てのグラフページのリストを表示する場合

pe_dir page:=g

全てのワークシートページのリストを表示する場合

pe_dir page:=w

全ページ名を文字列変数に出力する場合

pe_dir oname:=stringName
