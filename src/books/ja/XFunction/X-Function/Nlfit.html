概要

fitフィットセッションを行います。

追加の情報

このXファンクションは、自動GetNダイアログには対応していません。

コマンドラインでの使用法

nlfit 0; // update fit cuves and recalculate chisqr

nlfit 1; // do only one iteration

nlfit; // fit until converge

説明

使用中のフィット関数で反復計算します。反復の回数と反復の手法を指定することができます。
