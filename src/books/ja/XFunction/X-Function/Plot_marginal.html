概要

さまざまな種類の分布投影グラフ（マージナルプロット）をプロットします。

コマンドラインでの使用法

1. plot_marginal -r 1 type:=kde_fill method:=thumb density:=binned top:=box right:=violin;

2. plot_marginal -r 1 type:=scatter_with_lr top:=dist_fill right:=violin gap:=5 size:=40 rugs:=1;

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

主レイヤ type

入力

int
0

グラフの主レイヤのグラフタイプを指定します。

オプションリスト

  • 0=scatter: 散布図
    選択したXYデータセットを全て散布図として、主レイヤにプロットします。
  • 1=scatter_with_lr: 線形回帰付き散布図
    選択したXYデータセットを全て散布図とし、それぞれの線形フィット線付きで、主レイヤにプロットします。
    このグラフタイプを選択すると、信頼帯と信頼水準の設定項目が利用できるようになり、線形フィット線に関する設定が可能になります。信頼帯は、線形回帰曲線の両側信頼区間を追加でき、信頼水準は、信頼帯の信頼水準を設定するために使用できます。
  • 2=kde_fill: カーネル密度塗りつぶし等高線
    このプロットタイプを選択すると、選択したすべてのデータセットが塗りつぶし付きのカーネル密度プロットとしてプロットされます。
    このプロットタイプを選択すると、帯域幅法、密度法、X/Yのグリッドポイント数が表示され、カーネル密度プロットの外観を制御できます。
  • 3=kde_line: カーネル密度等高線
    このプロットタイプを選択すると、選択したすべてのデータセットが、線のみのカーネル密度プロットとしてプロットされます。
    このプロットタイプを選択すると、帯域幅法、密度法、X/Yのグリッドポイント数が表示され、カーネル密度プロットの外観を制御できます。
信頼帯 conf

入力

int
1

主レイヤが線形回帰付き散布図の場合(type=scatter_with_lr)、このオプションを使用できます。線形回帰曲線に両側信頼帯を追加するかどうかを指定できます。

オプションリスト：

  • 0=偽
  • 1=真

double

主レイヤが線形回帰付き散布図の場合(type=scatter_with_lr)、このオプションを使用できます。信頼帯に対する信頼水準を指定します。
帯域幅の手法 method

入力 int
0

主レイヤがカーネル密度塗りつぶし等高線またはカーネル密度等高線の場合(type=kde_fill または type=kde_line)、このオプションを使用できます。2Dカーネル密度グラフの帯域幅計算手法を指定します。

オプションリスト：

  • 0=bivariate: 2変数のカーネル密度推定
  • 1=thumb: 経験則
密度の手法 density

入力

int
0

主レイヤがカーネル密度塗りつぶし等高線またはカーネル密度等高線の場合(type=kde_fill または type=kde_line)、このオプションを使用できます。XYグリッドを定義するカーネル密度の計算手法を指定します。

オプションリスト：

  • 0=exact:正確な推定
    Ks2densityの式に従って密度値を計算するオプションを選択します。大きいデータセットの場合、正確な計算には大規模な計算が必要になる可能性があります。
  • 1=binned:ビン化概算
    このオプションを選択して、密度値の概算を計算します。サンプルが大きいときに推奨されるオプションです。
X/Yのグリッド点数 grid

上レイヤ top

入力

int
ヒストグラム

グラフの上レイヤのグラフタイプを指定します。

オプションリスト

  • 0=hist: ヒストグラム
    全Xデータをヒストグラムとして上レイヤにプロットします。複数のXデータセットがある場合は、ヒストグラムを重ねてプロットします。
  • 1=stacked_hist: 積み上げヒストグラム
    全Xデータを積み上げヒストグラムとして上レイヤにプロットします。
  • 2=box: ボックスチャート
    全Xデータをボックスチャートとして上レイヤにプロットします。
  • 3=violin: バイオリン
    全Xデータをバイオリン図として上レイヤにプロットします。
  • 4=dist: 分布曲線
    全Xデータを分布曲線として上レイヤにプロットします。
  • 5=dist_fill: 塗りつぶし付き分布曲線
    全Xデータを塗りつぶし付き分布曲線として上レイヤにプロットします。
  • 6=histo_curve: 曲線付きヒストグラム
    全Xデータを分布曲線付きヒストグラムとして上レイヤにプロットします。複数のXデータセットがある場合は、隣に並べてヒストグラムをプロットします。

入力
