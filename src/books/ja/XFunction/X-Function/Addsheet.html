メニュー情報

概要

指定したワークブック/行列ブックにワークシート/行列シートを追加します。

追加の情報

Origin Version: Origin 8.0 SR6

コマンドラインでの使用法

 

1.addsheet book:=Book1 fname:=templates\analysis\Assays.ogw index:=2 active:=1 oname:=aa

2.addsheet book:=book1 fname:=origin.otw oname:=bb

 

変数

表示
名		 変数
名		 I/O
および
データ型		 デフォルト
値		 説明
シートを追加するブック book

入力

ページ
<active>
ワークシート/行列シートを追加するワークブック/行列ブックを指定します。
テンプレート名 fname

入力

string
fname$
新しいシートを作成するのに使用するテンプレートを指定します。これは、ワークシートテンプレート (*.OTW)、行列ブックテンプレート(*.OTM)、ワークブックファイル(*.OGW)、行列ファイル(*.OGM)にすることができます。テンプレートファイル名がフルパスで指定していない場合、そのテンプレートは<Originのインストールフォルダ> または <Originユーザフォルダ>にあるものと見なします。
テンプレートのシートインデックス index

入力

int
0
テンプレートのどのシートを追加するかを指定します。最初のシートは1です。0を使用すると、テンプレート内のすべてのしーとが追加されます。
追加したシートをアクティブ化 active

入力

int
1
追加したシートをアクティブにするかどうかを指定します。
追加するシートの名前 oname

出力

string
sname$
追加したシートの名前を保存する変数を指定します。

説明

addsheet関数は、指定したワークブックまたは行列ブックに、テンプレートを使って、ワークシートまたは行列シートを追加するのに使用します。テンプレート内の1つのシートまたはすべてのシートを選択することができます。
