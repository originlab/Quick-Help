メニュー情報

統計：生存分析：Weibullフィット

概要

生存データでワイブルフィットを実行

追加の情報

これはOrigin Proのみの機能です。

コマンドラインでの使用法

1.weibullfit irng:=(1,2) censor:=0 

2.weibullfit irng:=(1,2) censor:=0 conf:=99 rt:=<new name:= Weibull>

3.weibullfit irng:=(1,2) censor:=0  xmin:=1E-4 xmax:=200;

パーセンタイル線ツリーの詳細

  • パーセンタイル線ツリー

パーセンタイル線ツリーはパーセンタイル線のオプションを指定します。

シンタックス: percentile.Treenode:=<value>

ツリーノード ラベル データ型 デフォルト 説明
percentileType パーセンタイル線の種類 int 0 パーセンタイル線の種類を指定します。オプションリスト：
  • 0: パーセント(%)
    パーセンテージでパーセンタイル線を指定します。
  • 1: 時間
    時間でパーセンタイル線を指定します。
percentileValue パーセント値(%)/時間値 int 50 パーセンタイル線の種類としてパーセント(%)を選択した場合、パーセント値を指定します。時間を選択した場合、時間値を指定します。
