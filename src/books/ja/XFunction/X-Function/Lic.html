メニュー情報

モジュールライセンスの更新

概要

モジュールライセンスファイルをOriginに追加します。

追加の情報

必要なOriginのバージョン:Origin 8.1 SR0

コマンドラインでの使用法

lic lic:="valid license string";

有効なライセンスファイルテキスト
ライセンス名 name

入力

string

ライセンス名を指定します。Originが自動的にライセンス名をライセンスファイルから抽出するので、実際には必要ありません。

説明

Developer Kitは、複雑なダイアログボックス、フローティングツール、ウィザードを作成するためのユーティリティです。しかし、Developer Kitにアクセスする前に、OriginLabから有効なライセンスを取得し、Developer Kit をアクティブ化する必要があります。このXファンクションは有効なライセンスでDeveloper Kitをアクティブにするために使います。
