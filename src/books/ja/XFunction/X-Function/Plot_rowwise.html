メニュー情報

作図 > 2D: 線形グラフ

概要

行単位のデータからグラフを作成します

追加の情報

必要なOriginのバージョン: 2019 SR0以降

コマンドラインでの使用法

1. plot_rowwise iy:=[Book1]Sheet1!((,2),(,3),(,4)) ylabel:=[Book1]Sheet1!1 xdata:=label xlabel:=L type:=linesymb rd:=[<new>]<new>;

2. plot_rowwise iy:=[Book1]Sheet1!((,B),(,C),(,D)) ylabel:=[Book1]Sheet1!A xdata:=selrow1 type:=column;

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

行データプロットのデータ iy

入力

XYRange

<active>
 入力データの範囲です。1つ以上のY列で、任意でYエラー列も指定できます。
Yラベル ylabel

入力

vector

<unassigned>
 プロットの凡例として列を指定します。
Xデータ xdata

入力

string

2
 X軸の値を指定します。オプションリスト：
  • selrow1(0): 選択中の第1行
選択範囲の最初の行の値を使用します。
  • wksrow1(1): シートの第1行
ワークシートの最初の行の値を使用します。
  • label(2): 列ラベル
ワークシート内の列ラベルの値を使用します。
  • Custom(3):: カスタム
ワークシートの行をXデータとしてカスタマイズします。xrngで行を指定できます。
列ラベル xlabel

入力

string

<unassigned>
 行がX値として使用される列ラベルの文字列を指定します。
  • L: ロングネーム
  • U: 単位
  • C: コメント
X行範囲 xrng

入力

Range

<unassigned>
 Xデータにする行範囲を指定します。xdataが3に設定されている場合のみ、xrng指定できます。
作図形式 type

入力

int

0
 作図形式を指定します。オプションリスト：
  • Line(0):: 折れ線
  • Scatter(1):: 散布図
  • Linesymb(2):: 線+シンボルグラフ
  • Column(3):: 縦棒
  • Bar(4):: 横棒
プロット順 porder

入力

int

1
 カテゴリカルなY列に従ってプロット順を指定します。 カテゴリカルなY列をylabelに指定したときのみこのオプションは有効になります。

オプションリスト：

  • Row(0):: データ行の順序
ソースデータの行の順序に従って、中間シートの列をプロットします。
  • Cat(1):: カテゴリー順
列のYラベルのカテゴリー順に従って中間シートの列をプロットします。
出力 rd

出力

ReportData

[<入力>]<新規>
 計算したデータの出力先を指定します。
