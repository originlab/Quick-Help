概要

2つのデータセットの交点のX/Y座標を計算

コマンドラインでの使用法

intersect iy1:=[Graph1]1!1 iy2:=[Graph1]1!2;

intersect iy1:=col(2) iy2:=col(4) method:=1 npts:=100;

XYRange

 交点を検索する2つ目の曲線のXY範囲（曲線）を指定します。
手法 method

入力

int

0
 交点を検索するためにY値を補間する方法を指定します。3つの方法が利用できます。

オプションリスト

  • linear:線形{0}
  • spline:3次スプライン{1}
  • bspline:3次Bスプライン{2}

Note:これらの手法についての情報は、補間補外のヘルプページを確認してください。

サンプル

このLabTalkスクリプトサンプルは、2つのデータセットの交点を計算します。 

newbook;
col(1)=data(1,15,1);
col(2)=data(1,15,1);
col(3)=col(1)*2-7;
intersect iy1:=col(2) iy2:=col(3) method:=0 npts:=100
