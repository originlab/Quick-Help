画像インポートのツリーノードに関する詳細


ファイル情報

シンタックス: Info.Treenode:=<value>

サンプル: Info.TifAsData:=1 // グレースケールGeoTIFFを指定してデータとしてインポート

ツリーノード ラベル データ型 デフォルト 説明
データ型 グレースケールGeoTIFFをデータとしてインポート int 0 インポートした画像のサムネイルを表示するかどうかを指定します。


オプション

オプションツリーは、impImageXファンクションのインポートオプションを全て指定します。

シンタックス: Options.Treenode:=<value>

サンプル: Options.FirstMode:=0 // 第1ファイルのインポートモードを指定


ツリーノード ラベル データ型 デフォルト 説明
thumbnail イメージサムネールの表示 int 2 インポートした画像のサムネイルを表示するかどうかを指定します。
  • 0:しない
    インポート画像のサムネイルを表示しない
  • 1:する
    インポート画像のサムネイルを常に表示
  • 2:複数の場合のみ
    複数フレームの画像をインポートした場合のみサムネイルを表示
FirstMode 第1ファイルインポートモード int 0 最初のファイルのインポートモードを指定します。以下のオプションがあります。
  • 0:既存データを置き換える
    ターゲットウィンドウに存在するデータは消去されます。そして、新しいデータがインポートされます。
  • 3:ブックを新たに作って読み込む
    インポートされたファイル用に新しいブックを作成して読み込みます。
  • 4:シートを新たに作って読み込む
    インポートされたファイル用に新しいシートを作成して読み込みます。
  • 5:新しいオブジェクトの開始
    次数が既存のオブジェクトと同じである場合は、新しい行列オブジェクトを作成します。
Mode 複数ファイルのインポートモード(第1ファイル以外) int 5 第1ファイル以外のファイルのインポートモードを指定します。以下のオプションがあります。
  • 3:ブックを新たに作って読み込む
    インポートされたファイル用に新しいブックを作成して読み込みます。
  • 4:シートを新たに作って読み込む
    インポートされたファイル用に新しいシートを作成して読み込みます。
  • 5:新しいオブジェクトの開始
    インポートされたファイル用に新しい行列オブジェクトを作成して読み込みます。
names.FNameToSht ファイル名でシート名を変更 int 1 行列シート名をファイル名にするかどうかを指定します。
names.FNameToBk ファイル名でブック名を変更 int 1 行列ブック名をファイル名にするかどうかを指定します。
names.FNameToBkLnOnly ブックのロングネームのみを変更 int 0 names.FNameToBk が1のとき利用できます。1の場合、行列ブックのロングネームを変更します。
names.FPathToBk ブック名の変更の際にパスを含める int 0 names.FNameToBk が1のとき利用できます。行列ブック名にファイルパスを含めるかどうかを指定します。
