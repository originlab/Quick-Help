概要

ラベルをグラフのレイヤに追加します

コマンドラインでの使用法

1.label_layer type:=0 pos:=tro; //グラフレイヤの右上のラベルとしてアルファベットの大文字を順に追加

2. label_layer type:=2 custom:=Graph## pos:=1; // グラフレイヤ内左上にラベルGraph##を追加

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

タイプ type

入力

int

0
 ラベルの種類を指定します。

オプションリスト：

  • 0=capitalA:A
  • 1=a:a
  • 2=custom:カスタム
カスタム custom

入力

string

a$
 タイプでカスタムを選択した場合、次のように設定可能です。
  • a$ および A$は小文字、大文字の順序記号付けをします。
  • r$ および R$は小文字、大文字のローマ数字の番号付けをします。
  • n$ および # はインデックス番号を付けます。
  • h$ は、文字#を示します。
位置 pos

入力

int

0
 追加したラベルのレイヤj内の位置を指定します。

オプションリスト：

  • 0=tlo:左上の外側
  • 1=tli:左上の内側
  • 2=Tro: 右上の外側
  • 3=tri: 右上の内側
  • 4=tci:上部中央内側
  • 5=tco:上部中央外側
  • 6=tco:底部中央外側
フォント font

入力

TreeNode

<unassigned>
 フォントを指定します。

説明

このXファンクションは、グラフレイヤにラベルを追加できます。レイヤが複数ある場合、各レイヤに文字またはインデックス番号で順序付けたラベルを付けることができます。
