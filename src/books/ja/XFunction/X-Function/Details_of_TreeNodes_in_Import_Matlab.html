Matlabインポートのツリーノードに関する詳細

オプション

オプションツリーは、impMatlab Xファンクションのインポートオプションを全て指定します。

シンタックス：Options.Treenode:=<value>

サンプル：Options.bHierarchical:=1

ツリーノード ラベル データ型 デフォルト 説明
bHierarchical 階層構造にインポート int 0 これを選択すると、ワークシートには変数リストのみがインポートされます。それぞれの変数名は最初の列にリンクしたテキストとして表示されます。変数名をクリックすると、目的の変数を1つずつ新しいワークシートにインポートします。

Note:ワークブック/ワークシートへのみインポートできます。
bSaveFileInfo ワークブック中のファイル情報を保存 int 1 選択されていない場合、ファイル情報はオーガナイザに保存されず、再インポート機能は使用できませんが、インポートス速度は向上します。
ImpMode インポートモード int 0 オプションリスト：
  • 0: ブックを新たに作って読み込む
    それぞれのファイルを新しいブックにインポートします。出力が<新規>の場合、各ファイルごとに新しいワークブックが作成されます。出力として指定されたブックがある場合、新規で作成されたブック(ワークブックまたは行列ブック)は出力と同じタイプになります。
  • 1: 行の末尾に追加する
    インポートしたファイルを新しい行に入力します。ワークブックでのみサポートされているアクションです。
  • 2: 列を追加して読み込む
    インポートしたファイルを新しい列に入力します。ワークブックでのみサポートされているアクションです。
  • 3: 新たなシートを作成する
    それぞれのファイルを新しいワークシートにインポートします。全てのワークシートは全て同じワークブック内に組み込まれます。
  • 4: ブックを新たに作って読み込む
    それぞれのファイルをインポートする時に新しいワークブックを作成します。
  • 5: 行列シートを新たに作って読み込む
    ファイルをインポートする時に新しい行列シートを作成します。
  • 6: 行列ブックを新たに作って読み込む
    ファイルをインポートする時に新しい行列ブックを作成します。
TransMode 変数置換え int 0 インポート中にMatlabデータを置き換えるか指定します。

オプションリスト：

  • 0: なし
    データ構造を置換えずにインポートします。
  • 1:全変数
    インポートされたファイルのすべての変数を置換えます。
  • 2:1*N 次変数
    1*N 次の変数のみ置き換えます。
