メニュー情報

ノイズ

概要

画像にランダムノイズを付加します。

コマンドラインでの使用法

1. imgnoise r:=5;

2. imgNoise r:=10 img:=mat(1) oimg:=mat(2) c:=green;

変数

表示

名		 I/O
と
データ型		 デフォルト

範囲 r

入力

int

0

付加されるノイズの範囲のパーセンテージを指定します。
入力イメージ img

入力

Image

<active>

操作する原画像を指定します。デフォルトの入力はアクティブイメージです。
出力イメージ oimg

出力

Image

<input>

出力画像を指定します。デフォルトで、出力画像は入力画像と同じです。
チャネル c

入力

int

すべて

ノイズを追加するチャネルを指定します。

オプションリスト

  • all:全て
すべてのチャネルにノイズを付加します。
  • red:赤
Rチャネルのみノイズを付加します。
  • green:緑
Gチャネルのみノイズを付加します。
  • blue:青
Bチャネルのみノイズを付加します。

説明

imgNoise 関数は、入力画像にランダムノイズを付加します。すべてのチャネルまたは指定した色のチャネルにノイズを付加することができます。

サンプル

この例では、 imgNoise 関数を使って、入力画像のRチャネルにノイズを付加します。

  1. 画像がアクティブな場合、「イメージ：空間フィルタ：ノイズ」を選択し、Xファンクションのダイアログを開きます。
  2. ダイアログで、下の図のように設定を変更して、OK ボタンをクリックしてXファンクションのimgUserfilterダイアログを開きます。

    新しい画像が作成されます。いくつか赤いノイズが付加されたことが分かります。
