概要

nl fitセッションを終了します。

追加の情報

このXファンクションは、自動GetNダイアログには対応していません。

コマンドラインでの使用法

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト
値		 説明
NLFitレポートオプション 出力

入力

int

0

 レポートを出力する場所を指定します。
NLFit自動更新モード autoupdate

入力

int

AU_NONE

 フィットデータが変わったときの自動更新モード

オプションリスト:

  • AU_NONE:No Update
レポートを自動更新しない
  • AU_AUTO:Auto Update
レポートを自動更新する
  • AU_ON_COMMAND:Update on Command
ユーザがコマンドを入力したときレポートを更新します。

説明

主にレポートのプロパティを設定するためフィット後に実行します。
