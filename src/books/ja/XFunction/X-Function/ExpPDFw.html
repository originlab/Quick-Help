メニュー情報

ファイル：エクスポート：マルチページPDF形式で...

概要

ワークシートを複数ページのPDFファイルとしてエクスポートします。

追加の情報

必要なOriginのバージョン:8.1SR0

コマンドラインでの使用法

1. expPDFw export:=book filename:=<Long Name> path:="C:\My PDF Files" overwrite:=ask;

2. expPDFw export:=active filename:="<long name>" path:=D:\ overwrite:=ask tr2.PDF.pdf.ColorTranslation:=0. tr2.PDF.pdf.LineMode:=0.tr2.PDF.Compression.ClrImageCompression:=0. tr2.MultiPage.CoverPage.Text:="My Report" tr3.Check.Chk_Horz_Grids:=0. tr3.Check.Chk_Vert_Grids:=0. tr3.Check.Chk_Frame:=33554432.;

シートを選択 export

入力

int

0

この変数は、エクスポートするページを指定するのに使用します。デフォルトは現在のシートです。オプションリスト

  • active:現在のシート
    現在のワークシートがエクスポートされます。
  • book:ブック中のシート
    アクティブワークブック内のすべてのワークシートがエクスポートされます。
  • folder:フォルダ中のシート
    現在のフォルダ内のすべてのワークシートがエクスポートされますが、サブフォルダ内のシートは含まれません。
  • recursive:フォルダ中のシート(サブフォルダ適用)
    サブフォルダ内のシートを含み、現在のフォルダ内のすべてのワークシートがエクスポートされます。
  • project: プロジェクト中のシート
    プロジェクト内のすべてのワークシートがエクスポートされます。
  • Specified:特定
    ワークシートテキストボックス内で指定したワークシートをエクスポートします。
ワークシート pages

入力

string

<active>

エクスポートするワークシートを一覧表示します。エクスポートドロップダウンリストで、特定が選択されている場合に、このフィールドが編集可能になります。また、参照ボタンが利用可になります。参照ボタンをクリックして、プロジェクトブラウザシートダイアログを開き、エクスポートするワークシートを選択します。
ファイル名 filename

入力

string

  

エクスポートするPDFファイルの名前を指定します。以下のオプションがあります。

  • <ロングネーム>
    グラフページのロングネームをファイル名として使います。
  • <ショートネーム>
    グラフページのショートネームをファイル名として使います。
  • ＜シート名＞
    シート名のショートネームをファイル名として使います。
  • wks##
    "wks + インデックス"をファイル名として使います。
  • wks<pound>#
    "wks# + インデックス"と"#"の文字(<pound>で表現される)をファイル名とします。

Note:

  1. ファイル名はキーワードの組み合わせをサポートしています。例えば、「<ロングネーム>-multi」では、ワークシートのロングネームと文字列-multi がエクスポートするファイル名になります。
  2. "#" を表す <pound> は、グラフインデックスの抱き合わせ以外にも使用できます。例えば、ファイル名を、Wks<pound>2 とすると、表示ファイル名はWks#2 になります。
  3. 上記3つのオプションの1つから選択するだけでなく、目的のファイル名をキーボードから入力することもできます。
パス path

入力

string

  

エクスポートされるPDFファイルを保存するファイルパス
上書き overwrite

入力

int

3

同じパスに同じ名前を持つファイルが既に存在している場合のPDFファイルをエクスポートする方法を指定します。オプションリスト

  • rename:自動名前変更
    エクスポートするPDFに番号を付けて自動的に名前を変更します。
  • replace:置換え
    既存のPDFファイルを上書きして置き換えます。
  • skip:スキップ
    PDFファイルのエクスポートを避け、既存のファイルをそのままにします。
  • ask:尋ねる
    メッセージボックスを表示し、既存のPDFファイルを上書きするかどうかを尋ねます。

説明

このXファンクションは、画像または埋込/リンクグラフを含むワークシートを複数ページのPDFファイルにエクスポートするのに使用されます。

Note:
  1. レポートシートはPDFファイルにエクスポートできません。
  2. ワークシートは大きすぎる場合、1つのPDFファイルの複数ページにエクスポートします。

ツリーノード ラベル データ型 デフォルト 説明
CoverPage カバーページ 入力 TreeNode <unassigned> これはカバーページの右にあるチェックボックスにチェックを付けた場合のみ利用できます。エクスポートしたPDFファイルにカバーページを追加するかどうかを指定します。デフォルトで、カバーページはなしです。このブランチ内のツリーノードは、以下の通りです。(下記は、コロンの前のてきすとがツリーノード名で、コロンの後のテキストはツリーノードのラベルです。)
Text:テキスト
このテキストフィールドに入力するテキストがカバーページに表示されます。
Font:フォント
このブランチはカバーページテキストのフォントをセットするのに使います。
Name:名前
テキストのフォント名を指定します。
Size:サイズ
テキストのフォントサイズを選択または入力します。デフォルトサイズは18です。
Bold:太字
テキストを太字にするかどうかを指定します。このチェックボックスはデフォルトで、選択されています。
Italic:斜体
テキストを斜体にするかどうかを指定します。このチェックボックスはデフォルトで、選択されています。
Underline:下線
テキストに下線を追加するかどうかを指定します。このチェックボックスはデフォルトで、選択されています。
Pagination ページ番号付け 入力 TreeNode <unassigned> ページ番号付けの右側にあるチェックボックスにチェックが付いているときのみ、開くことができます。エクスポートしたPDFファイルにページ番号を追加するかどうかを指定します。デフォルトで、ページ番号の追加は無しです。エクスポートするPDFがカバーページを含む場合、カバーページを最初のページと見なしますが、ページ番号はカバーページに表示しません。このブランチ内のツリーノードは、以下の通りです。(下記は、コロンの前のてきすとがツリーノード名で、コロンの後のテキストはツリーノードのラベルです。)
FontP:フォント
このブランチはページ番号のフォントをセットするのに使います。
Name:名前
ページ番号のフォント名を指定します。
Size:サイズ
ページ番号のフォントサイズを選択または入力します。デフォルトサイズは18です。
Bold:太字
ページ番号を太字にするかどうかを指定します。このチェックボックスはデフォルトで、選択されています。
Italic:斜体
ページ番号を斜体にするかどうかを指定します。このチェックボックスはデフォルトで、選択されています。
Underline:下線
ページ番号に下線を追加するかどうかを指定します。このチェックボックスはデフォルトで、選択されています。
Top:上
エクスポートするPDFファイル内のページの上にページ番号を追加するかどうかを指定します。このチェックボックスはデフォルトで、選択されています。
Bottom:下
エクスポートするPDFファイル内のページの下にページ番号を追加するかどうかを指定します。このチェックボックスはデフォルトで、選択されています。
HorizPos:水平位置
ページ番号の水平位置を指定します。上および下の両方ともチェックが付いてない場合、PDFファイルにページ番号が無いので、ここで指定した水平位置は無効となります。以下のオプションがあります。
  • 0:偶数ページを左、奇数ページを右
    ページ番号は、偶数ページに対して左に表示され、奇数ページに対して右に表示されます。
  • 1:左
    すべてのページに対してページ番号は左に表示されます。
  • 2:右
    すべてのページに対してページ番号は右に表示されます。
  • 3:中央
    すべてのページに対してページ番号は中央に表示されます。これはデフォルトの選択です。

サンプル

  1. 新しいプロジェクトを作成し、メインメニューからファイル：インポート：カンマ区切りファイル (CSV)... を選択して <Origin プログラムディレクトリ>\Samples\Import and Export\donations.csv をインポートします。
  2. データがインポートされたワークシートをアクティブにし、ファイル：エクスポート：マルチページPDF形式で...を選び、expPDFwダイアログを開きます。
  3. ダイアログで、ファイル名をdonationsにし、パスを C:\My PDF Filesにします。画像の設定ブランチにある複数ページブランチで、カバーページにあるチェックボックスを選択し、このブランチを広げます。テキストフィールドにDonations と入力し、フォントサイズを72に変更します。その他のオプションはデフォルト値のままにします。
  4. OK ボタンをクリックします。すると、donations.pdf という名前のPDFがC:\My PDF Filesフォルダにエクスポートされます。
