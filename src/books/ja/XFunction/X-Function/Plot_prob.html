メニュー情報

作図: 確率: 確率プロット(p-p図)

作図: 確率: Q-Qプロット

概要

確率プロットまたはQ-Qプロットを指定の分布を使用して作成します

追加の情報

必要なOriginのバージョン: 8.1 SR0以降

コマンドラインでの使用法

1. plot_prob -r 1 irng:=[QQplot1]"Q-Q plot"!B"Weight" method:=renard;

2. plot_prob irng:=[ProbPlot]Sheetl!A distr:=weibull;

Note: Origin 2017以前のバージョンでは、このXファンクションをLabTalkスクリプトから呼び出せません。スクリプトからp-pまたはQ-Qプロットを作図するには、以下を使用します。 

worksheet -px

例えば、 

//確率プロットの作図
worksheet -px ? PPNormal plot_prob option:=0;

//Q-Qプロットの作図
worksheet -px ? QQ plot_prob option:=1;

サンプル

  1. 新しいプロジェクトを作成して、<Origin Directory>\Samples\Statistics\Body.ogwを開きます。
  2. Male ワークシートで、Weight列を選択し、作図: 確率: Q-Qプロットを選択するか、2Dグラフギャラリーツールバーで、Q-Qプロットボタンをクリックしてplot_probダイアログを開きます。
  3. 分布は正規分布にして、データより推定にチェックをつけてスコアー法はBolmにします。
  4. OKボタンをクリックして、Q-Qプロットを作成します。
