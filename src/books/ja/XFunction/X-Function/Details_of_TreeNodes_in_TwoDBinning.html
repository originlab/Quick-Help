TwoDBinningインポートのツリーノードに関する詳細

x

x ツリーは、Xビン範囲のオプションを指定します。

シンタックス：x.Treenode:=<value>

サンプル：

  • x.inc:=0.3
  • x.stepby:=1 x.intervals:=7
  • x.border.outleft:=1 x.border.outright:=1
ツリーノード ラベル データ型 デフォルト 説明
bin ビン範囲の指定 int 1 Xビン範囲を定義するパラメータを指定します。オプション:
  • 0:ビン中心
    Xビンの範囲はビンの中心で指定します。
  • 1:ビン終端
    Xビンの範囲はビンの終端で指定します。
min 最小ビン開始 double <自動> Xビン中心/開始の最小値を指定します。
max 最大ビン終端 double <自動> Xビン中心/終端の最大値を指定します。
stepby 増分 int 0 ビンを区分けする方法を指定します。オプション:
  • 0:ビンサイズ
    ビンサイズでビンを分けます。
  • 1:ビンの数
    ビンの数でビンを分けます。
inc ビンサイズ double <自動> 増分 が ビンサイズのとき編集可能Xビンの固定のステップサイズを指定します。
intervals ビンの数 int <自動> 増分 が ビンの数のとき編集可能Xビンの数を指定します。
periodical 周期的 int 0 データが周期的かどうかを指定します。
period 期間 double <自動> Xデータの期間を指定します。これは周期的にチェックがついているときに利用できます。
border.outleft 最小値より小さい外れ値を含む int 0 最小値より小さい外れ値を最小のビンに含めるかどうかを指定します。これは周期的にチェックがついていないときに利用できます。
border.outright 外れ値を含む/最大より大きい外れ値を含む int 0 外れ値を含むは、周期的が選択されているときに利用でき、外れ値が追加のビンに含まれるかどうかを指定します。最大より大きい外れ値を含むは、周期的が選択されていないときに利用でき、最大値より小さくない外れ値が最大のビンに含まれるかどうかを指定します。
border.cmin 最小値を別にカウント int 0 最小値を別に数えるかどうかを指定します。
border.cmax 最大値を別にカウント int 0 最大値を別に数えるかどうかを指定します。
order 出力ビン順序 int 0 出力ビンの並び順を指定します。オプション:
  • 0:昇順
    昇順でビンを出力します。
  • 1:降順
    降順でビンを出力します。
