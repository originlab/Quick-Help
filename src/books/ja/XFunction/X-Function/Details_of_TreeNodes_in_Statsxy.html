Statsxyツリーノードに関する詳細

 

値

quantitiesツリーは、計算、出力する値を指定します。

シンタックス: quantities.Treenode:=<value>

  • quantities.bincenter:=0
  • quantities.sd:=1
  • quantities.max:=1
ツリーノード ラベル データ型 デフォルト 説明
binstart ビンの開始 int 0 ビンの開始値を出力するか指定します。
binend ビンの終了 int 0 ビンの終了値を出力するか指定します。
bincenter ビン中心 int 1 ビン中心の値を出力するか指定します。
bin ビン int 0 ビン範囲を出力するか指定します。
posminy 最小のYのX値 int 0 ビン範囲の最小YデータのＸ座標値を出力するか指定します。
posmaxy 最大のYのX値 int 0 ビン範囲の最大YデータのＸ座標値を出力するか指定します。
mean 平均値 int 1 ビン範囲の平均値を出力するか指定します。
median 中央値 int 0 ビン範囲の中央値を出力するか指定します。
sd 標準偏差 int 0 ビン範囲の標準偏差値を出力するか指定します。
sem 標準誤差 int 0 平均の標準誤差を出力するか指定します。
lowercim 平均の下側95%信頼区間 int 0 平均の下側95%信頼区間を出力するか指定します。
uppercim 平均の上側95%信頼区間 int 0 平均の上側95%信頼区間を出力するか指定します。
variance 分散 int 0 ビン範囲の分散を出力するか指定します。
sum 合計 int 0 ビン範囲の合計値を出力するか指定します。
min 最小 int 0 ビン範囲の最小値を出力するか指定します。
max 最大値 int 0 ビン範囲の最大値を出力するか指定します。
count カウント int 0 各ビンの個数（カウント）を計算して出力するか指定します。
