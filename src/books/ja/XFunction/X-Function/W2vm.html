概要

ワークシートデータから仮想行列を作成します

追加の情報

必要なOriginのバージョン:8.5.1

vmname, coltype, rowtype 変数は、Origin 2015から利用できます。

コマンドラインでの使用法

1. w2vm irng:=[Book1]Sheet1 rowpos:=selrow1 colpos:=selcol1 ztitle:=VMObject;

2. w2vm irng:=1!2[1]:4[3] rowpos:=3 label:=L colpos:=2 ztitle:=MyVM;

入力

 XおよびY値をセットする方法を指定します。

オプションリスト

  • yacross:列方向のY{0}
    行をY値とし、列をX値として使用します。
  • xacross:列方向のX{1}
    行をX値とし、列をY値として使用します。
Y値/X値 rowpos

入力

int

<自動>
 Y（またはX）値の行を指定します。YまたはX値が指定されているかどうかは、データフォーマットの値で決まります。

行内では、Y（またはX）値は、昇順か降順である必要があります。

オプションリスト

  • none:なし{0}
    選択されたワークシートのセル範囲の1から始まる列番号を、Y（あるいはX）値として使用します。
  • selrow1:選択中の第1行{1}
    選択されたワークシートのセル範囲の1から始まる列番号を、Y（あるいはX）値として使用します。
  • label:列ラベル{2}
    ワークシートの列ラベルをY（またはX）値として使用します。labelで列ラベル行を指定します。ワークシートの範囲だけがirngで指定されていると、ワークシートの列ラベル行に対応している部分だけがY（またはX）値として使用されます。
  • custom:カスタム{3}
    ワークシート内で行を選択します。
Y/X範囲 rowrng

入力

Range

<unassigned>
 Y（またはX）値の行を指定します。rowpos がカスタムの時のみrowrng が指定されます。

rowrng はirngのワークシート名と同じである必要があります。

列ラベル label

入力

string

<unassigned>
 Y（あるいはX）値として使用される列ラベルを指定します。label変数は、rowposが列ラベルの時のみ指定できます。スクリプトで列ラベル行を指定するには、1文字を使用する必要があります。各列ラベル行を表す文字については列ラベル行の文字 を参照してください。
Y/X データフォーマット rowtype

入力

string

<unassigned>

X/Yデータの表示フォーマットを指定します。rowpos 変数が0（なし）にセットされている場合のみ、この変数を利用できます。詳細は、サポートされるOriginフォーマット表記をご覧ください。

X/Y値 colpos

入力

int

<自動>
 X（またはY）値の列を指定します。XまたはY値が指定されているかどうかは、データフォーマットの値で決まります。

列内では、X（またはY）値は、昇順か降順である必要があります。

オプションリスト

  • none:なし{0}
    選択されたワークシートのセル範囲の1から始まる行番号を、X（あるいはY）値として使用します。
  • selcol:選択中の第1列{1}
    選択の最初の列をX（またはY）値として使用します。
  • wkscol1:ワークシートの第1列t{2}
    ワークシートの最初の列をX（またはY）値として使用します。
  • xcol:X column to left of selection{3}
    irngの範囲の左にある最も近いX列をX値(またはY値)として使用します。irngの範囲の左側に少なくとも1つのX列があるときのみ指定できます。
  • custom:カスタム{4}
    ワークシート内で列を指定します。

説明

このXファンクションは、ワークシートデータから行列オブジェクトを作成するときに使用されます。仮想行列が作成された後、作図の詳細ダイアログとレイヤ内容ダイアログなどでアクセスして新しいグラフを作成できます。
