概要

画像をROIにトリミング

追加の情報

必要なOriginのバージョン: Origin 2022以降

コマンドラインでの使用法

1. cvcrop; // アクティブイメージウィンドウをトリミング

2. cvcrop igp:=Image1; // Image1をトリミング

3. cvcrop ogp:=<new>; // アクティブな画像ウィンドウを切り取り、結果を新しい画像ウィンドウに配置

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

説明

この機能は画像をROIにトリミングします。予め矩形ツールでROIを選択する必要があります。

アルゴリズム

計算には、OpenCVメインAPIの関数を使用します。OpenCVメインAPIヘルプファイルVersion 2.4.11.0を参照し、imgprocを確認してください。画像処理のトピックです。
