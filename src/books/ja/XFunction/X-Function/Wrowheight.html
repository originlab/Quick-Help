メニュー情報

行を右クリックし、コンテキストメニューから、行高

概要

行の高さを設定します

コマンドラインでの使用法

1. wrowheight [1] 3; // 1行目を3行の高さに

2. wrowheight [C:C] 2;// 行コメントを2行に

3. wrowheight i:=2![L:L] h:=3;// ラベル領域は、L:L, C:C

4. wrowheight ([2],[4],[6]) 1.5//部複数行を指定可

入力

double

1

行の高さを指定します。デフォルトフォントの比率に準じた値を入力します。wrowheight が1の場合、システムデフォルトフォントの1文字を1つの行に表示できます。

説明

この関数は行の高さをセットするのに使用されます。
