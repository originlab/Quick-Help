イメージプロファイル

概要

イメージプロファイル

追加の情報

必要なOriginのバージョン: Origin 8.5 SR0以降

Xファンクションは Auto GetNダイアログ用にデザインされていません。

コマンドラインでの使用法

  1. imageprofile;
  2. imageprofile 1 0;

変数

表示名 変数名 I/Oとタイプ デフォルト値 説明
グラフレイアウト layout

入力

int

0
 プロファイルの線をどの様に表示するか指定します。

オプションリスト

  • cline:同じグラフ 交差線
    元のグラフに垂直と水平の両方の線を表示します。
  • vline:同じグラフ 垂直線
    元のグラフに垂直線のみを表示します。
  • hline:同じグラフ 水平線
    元のグラフに水平線のみを表示します。
  • separate:別々のグラフ
    垂直プロファイル線と水平プロファイル線はそれぞれのグラフに独立して表示されますが、元のグラフには表示されません。

protectが1の場合、他のどのオプションを設定していても必ずseparateになります。

元のグラフを保護 protect

入力

int

1
 ツールを閉じるときに元のグラフを保護するかそうか指定します。

説明

このXファンクションはアクティブな等高線図に対してイメージプロファイルを作成する際に使用します。
