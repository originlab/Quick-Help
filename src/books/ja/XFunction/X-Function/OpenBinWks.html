概要

ヒストグラムプロット用のビンワークシートを作成または開きます

コマンドラインでの使用法

OpenBinWks nRecalculate:=2;

変数

表示
名 		変数
名 		I/O
と
データ型 		デフォルト
値 		説明
nRecalculate nRecalculate

入力

int

1

再計算モード

  • 0=なし
  • 1=自動
  • 2=手動

サンプル

 //ヒストグラムを作成してビンワークシートを開く
newbook;
col(B) = uniform(30);
plotxy iy:=(2) plot:=219 ogl:=<new template:=Hist>;
OpenBinWks nRecalculate:=2;

キーワード:ヒストグラム
