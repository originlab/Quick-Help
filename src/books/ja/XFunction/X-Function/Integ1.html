メニュー情報

解析： 数学： 積分

概要

XYデータを積分します

追加の情報

この機能はOrigin8.0 SR5で更新されています。詳細は、リリースノート のセクションをご覧下さい。 8.5.1で更新

コマンドラインでの使用法

  1. integ1 iy:=(Col(1), Col(2));
  2. integ1 iy:=Col(2) type:=abs;
  3. integ1 iy:=Col(2) area:=dArea;
  4. integ1 iy:=Col(2) rd:=[<active>]<new>!;
  5. integ1 iy:=[Book1]Sheet1!(A,B:E) rd:=[<active>]<new>!;

説明

このXファンクションは入力データに対して積分を実行します。数学的面積 (台形の代数的総計) または 絶対値面積 (絶対台形値の総計)を計算できます。欠損値がある場合無視されます。

サンプル

絶対面積を使用してデータを積分し、積分曲線をプロットしたい場合には、下記のように実行して下さい。

  1. <Originプログラムフォルダ>\Samples\Mathematics\Sine Curve.dat をインポートします。
  2. データ列を選択します。メニューから、解析：数学：積分 を選択し、Mathematics: integ1 ダイアログボックスを開きます。
  3. 領域の種類に絶対的面積を選択します。
  4. 積分曲線をプロットドロップダウンリストで新しいグラフを選択します。
  5. OKをクリックすると、ソースワークシートに積分値が入力された列が追加され、積分曲線のグラフが作成されます。
