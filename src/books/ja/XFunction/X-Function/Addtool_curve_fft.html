メニュー情報

FFT

概要

FFTを実行するためにプロットに四角形を追加します。

追加の情報

必要なOriginのバージョン: 8.0 SR5

 

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト
値		 説明
設定 trGUI

入力

TreeNode

<unassigned>

FFTを実行する設定を指定します。

説明

この関数は、FFTを実行するためにプロットに矩形を追加します。この矩形を移動すると、すぐにFFTPREVIEW ウィンドウに、データのFFTの結果が表示されます。クィックFFTツールについての詳細は、ユーザガイドのFFTパネルツール を参照してください。.
