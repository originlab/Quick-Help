概要

プロジェクトエクスプローラのディレクトリを変更します。

コマンドラインでの使用法

1. pe_cd ..;

2. pe_cd /;

3. pe_cd path:= "../subfolder1";

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト
値		 説明
パス path

入力

string

..

 移動するディレクトリのパス

説明

このXファンクションは現在のパスを変更するのに使用します。「/a/b」または「../b」のように絶対または相対的な方法で現在のパスを変更します。

絶対的な方法で指定するとき、プロジェクト名を省略することができ、「/」はルートフォルダに戻ることができます。

サンプル

現在のディレクトリを抜けるには、次のコマンドを使います。

pe_cd("../")

myDirという名前のディレクトリに移動するには、次のコマンドを使います。

pe_cd("C:\myDir")

現在のディレクトリを抜けて、myDirという名前のディレクトリに移動するには、次のコマンドを使います。

pe_cd("../myDir")
