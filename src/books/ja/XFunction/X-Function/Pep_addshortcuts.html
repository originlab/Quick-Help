メニュー情報

お気に入りにショートカットを追加

概要

お気に入りフォルダに選択したウィンドウのショートカットを追加する

追加の情報

このXファンクションは、自動GetNダイアログには対応していません。

必要なOriginのバージョン: 8.1 SR0

コマンドラインでの使用法

pep_addshortcuts dump:=1;

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト
値		 説明

 

 dump

入力

int

<unassigned>

 お気に入りに追加したショートカットの数を出力するかどうかを指定します。

説明

このXファンクションは、選択したウィンドウのショートカットをお気に入りフォルダに作成します。お気に入りフォルダが存在しない場合、このXファンクションを実行した後、Originは自動的にFavoritesという名前のフォルダを作成します。

サンプル

この例は、グラフのショートカットをお気に入りフォルダに追加するものです。

  1. プロジェクトファイル<Originインストールフォルダ>\Samples\Graphing\Customizing Graphs.opjを開きます。
  2. プロジェクトエクスプローラで、Edit Single Data Pointフォルダをクリックし、 Graph1Bを選択します。



  3. コマンドウィンドウで pep_addshortcuts dump:=1; を実行すると、Originはコマンドウィンドウに 1 shortcuts added to Favorites と出力します。



    そして、お気に入り というフォルダがGraph1Bのショートカットと一緒に作成されます。.
