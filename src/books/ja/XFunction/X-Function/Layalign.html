概要

元のレイヤに従って目的のレイヤを整列します。

コマンドラインでの使用法

layalign igl:=1 destlayer:=2 direction:=bottom;

layalign igp:=graph2 source:=1 destlayer:=3;

string

方向

direction

入力

int

left

整列を指定します。
オプションリスト

  • Left:左
    目的のレイヤの左辺を元のレイヤの左辺と揃えます。
  • top:上
    目的のレイヤの上辺を元のレイヤの上辺と揃えます。
  • Right:右
    目的のレイヤの右辺を元のレイヤの右辺と揃えます。
  • bottom:下
    目的のレイヤの下辺を元のレイヤの下辺と揃えます。

説明

このXファンクションは2つのレイヤをそれぞれのへるで揃えるのに使用します。
