何年にもわたるサンプリングの基準年 time

入力

int

0

エクスポートされたNetCDFファイルで時間変数を示す方法を指定できます。

  • 0=None: 時間変数を使用しません
  • 1=Begin: 開始年を時間変数として使用
  • 2=End: 終了年を時間変数として使用

説明

このXファンクションは、現在の行列または指定した行列シートをNetCDFファイルとしてエクスポートすることができます。
