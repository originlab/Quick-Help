概要

nlbegin Xファンクションが実行された後にフィットパラメータダイアログを開きます。

追加の情報

Xファンクションは Auto GetNダイアログ用にデザインされていません。

modal パラメータはOrigin9.1 SR0 で追加されました。

コマンドラインでの使用法

1.nlpara;

2.nlpara 1;//ダイアログ内で「OK」をクリックすると次のスクリプト行を自動的に実行

変数

表示名 変数名 I/Oとタイプ デフォルト値 説明
  modal

入力

int

0

 フィットパラメータダイアログのOKボタンをクリックした後にスクリプトの次の行(一般的にnlend Xファンクション)を自動的に実行します。1 はOKをクリックした後に自動的に次の行のスクリプトを実行します。 0 は次の行は自動的に実行されません。

説明

フィットパラメータダイアログを開きます
