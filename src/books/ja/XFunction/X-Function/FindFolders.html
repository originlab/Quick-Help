概要

フォルダを検索します。

コマンドラインでの使用法

1. findFolders path:="C:\c";

2. findFolders path:="C:\c" addpath:=1;

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト

サブフォルダを探すパス path

入力

string

path$

サブフォルダを探すパスを入力します。
すべての結果フォルダにルートパスを追加 addpath

入力

int

0

すべての結果フォルダにルートパスを追加
見つかったサブフォルダの数 n

出力

int

<optional>

見つかったサブフォルダの数を出力
CRLFで分けられる結果のフォルダ名 folder

出力

string

folder$

CRLFで分けられた結果のファイル名を出力

説明

このXファンクションは指定したパスと条件でフォルダを検索するのに使います。

サンプル

他のサンプルについては、XFスクリプトダイアログ(F11を押す)をご覧下さい。
