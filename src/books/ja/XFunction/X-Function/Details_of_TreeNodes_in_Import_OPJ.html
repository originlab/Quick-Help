OPJインポートのツリーノードに関する詳細

オプション

オプションツリーは、impOPJ ファンクションのインポートモードと列情報を指定します。

シンタックス: Options.Treenode:=<value>

サンプル： Options.FirstMode:=1 // 新しい列にインポート

ツリーノード ラベル データ型 デフォルト 説明
FirstMode インポートモード int 0 以下のオプションがあります。
  • 0: 既存データを置き換える
    目的のウィンドウに存在するデータは消去します。新しいデータがその中にインポートされます。
  • 3: ブックを新たに作って読み込む
    インポートする各ファイルに対して新しいブックを作成します。
  • 4: シートを新たに作って読み込む
    インポートするファイルに対して新しいシートを作成します。
  • 1: 列を追加して読み込む
    目的のウィンドウに新しい列を追加し、新しい列にデータをインポートします。

Include

オプションツリーのIncludeブランチはimpOPJ Xファンクションの列情報を特定します。

シンタックス: Options.Include.Treenode:=<value>

サンプル: Options.Include.comments:=0 // コメント列ラベルのない列にインポート

ツリーノード ラベル データ型 デフォルト 説明
type 列の種類 int 1 データをインポートする列の属性を指定します。
lname ロングネーム int 1 データをインポートする列のロングネームを指定します。
units 単位 int 1 データをインポートする列の単位を指定します。
comments コメント int 1 データをインポートする列のコメントを指定します。
params パラメータ int 0 データをインポートする列のパラメータを指定します。
udls ユーザ定義ラベル int 0 データをインポートする列のユーザ定義ラベルを指定します。
