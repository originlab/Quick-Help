メニュー情報

反転

概要

画像の色を反転します。

コマンドラインでの使用法

1. imgInvert oimg:=<new>;

2. imgInvert img:=mat(1) oimg:=mat(2);

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト

入力イメージ img

入力

Image

<active>

操作する原画像を指定します。デフォルトの入力はアクティブイメージです。
出力イメージ oimg

出力

Image

<input>

出力画像を指定します。デフォルトで、出力画像は入力画像と同じです。

説明

　imgInvert 関数は、指定したビットマップの色を反転し、写真のネガのようにします。この関数は、1ビット画像の色を反転するのに使うこともでき、白黒画像の白と黒を入れ替えます。

サンプル

この例では、 imgInvert 関数を使って、入力画像の色を反転します。

  1. 入力画像がアクティブなときに、「イメージ：調整：色の反転」を選択します。これはXファンクションのダイアログを開きます。
  2. Xファンクションダイアログで、下の図のように設定を変更して、OK ボタンをクリックしてダイアログを閉じます。

    新しい画像が作成されます。色が反転したのが分かります。

アルゴリズム

グレースケール画像に対して、

O(x, y) = - I (x, y);

カラー画像に対して、

O(x, y) は出力するピクセル値、I(x, y)は入力するピクセル値、Bitsは Bits/ Pixelです。

計算には、LEADTOOLS Main APIのL_InvertBitmap() を使用しています。詳細は、LEADTOOLS Main API Version 14 ヘルプファイルを参照し、トピックL_InvertBitmapを読んでください。
