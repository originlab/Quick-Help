メニュー情報

統計：記述統計：シート全体の統計

概要

シート全体の記述統計を計算する

コマンドラインでの使用法

wstats -r 2 quantities:="n|mean|sd|sum|min|median|max|lcl|ucl|var" identifier:="Name|Range";

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

値 quantities

入力

string

mean|sd|sum|min|median|max 計算し、出力する値を指定します。
  • n : N 合計
  • missing: N 欠損
  • mean:: 平均
  • sd: 標準偏差（SD）
  • sem:: 平均値の標準誤差（SE）
  • lcl:: 平均の下位95%CI
  • ucl: 平均の上位95%CI
  • var: 分散
  • sum: 合計
  • skewness: 歪度
  • kurtosis: 尖度
  • uss: 未修正平方和
  • css: 修正平方和
  • cv: ばらつきの係数
  • mad: 平均絶対偏差
  • sdx2: SDの2倍
  • sdx3: SDの3倍
  • geomean: 幾何平均
  • geosd: 幾何SD
  • mode: モード
  • harmmean:: 調和平均
  • min: 最小
  • imin: 最小値のインデックス
  • q1: 第1四分位(Q1)
  • median: 中央値
  • q3: 第3四分位(Q3)
  • max: 最大
  • imax: 最大値のインデックス
  • iqr: 四分位間範囲 (Q3-Q1)
  • range:: 範囲 (最大-最小)
  • cp: カスタムパーセンタイル
  • medad: 中央絶対偏差
  • rcv: ロバスト変動係数
