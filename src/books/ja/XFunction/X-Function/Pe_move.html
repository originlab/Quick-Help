概要

フォルダ内の指定したページを指定したフォルダに移動します。

コマンドラインでの使用法

1. pe_move move:=Folder1 path:=/Folder2;

変数

表示
名		 変数
名		 I/O
と
データ型		 デフォルト
値		 説明
ページ/フォルダ move

入力

string

 

既存のページまたはフォルダの名前
パス path

入力

string

 

ページの移動先の位置のパス

説明

このXファンクションは、現在のフォルダにあるページやサブフォルダを移動するのに使用します。

サンプル

現在のフォルダにあるワークシート名Bool1をルートディレクトリに移動するには、次のコマンドを使います。

pe_move Book1, /
