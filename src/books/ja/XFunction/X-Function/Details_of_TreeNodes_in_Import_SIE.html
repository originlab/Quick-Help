SIEインポートのツリーノードに関する詳細

オプション

オプションツリーは、impSIE Xファンクションのすべてのインポートオプションを指定します。

シンタックス: Options.Treenode:=<value>

サンプル: Options.sparklines:=0 // スパークラインを追加しない


ツリーノード ラベル

データ型

 デフォルト 説明
sparklines スパークラインの追加 int 2 データのスパークラインを追加するかどうかを指定します。
  • 0: しない
    スパークラインを追加しません。
  • 1: する
    スパークラインを追加します。
  • 2: する(50列未満)
    50列以下であればスパークラインを追加します。
UserParamChan チャネルパラメータに対する行を追加 int 0 このチェックボックスは、チャネルパラメータに対して、行を追加するかどうかを指定します。
UserParam 次元パラメータに対する行を追加 int 1 このチェックボックスは、次元パラメータに対して、行を追加するかどうかを指定します。
