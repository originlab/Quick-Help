メニュー情報

作図: カテゴリカル: グループ化浮動棒

概要

グループ化浮動棒グラフを作成

コマンドラインでの使用法

plot_gfloatbar -r 1 subgroup:=label label:=L type:=column;

追加の情報

必要なOriginのバージョン: Origin 2020以降

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

サブグループ subgroup

入力

int
0

サブグループに分ける方法を指定します。

オプションリスト：

  • 0=size:サイズによる
  • 1=label:列ラベルによる
サイズ size

入力

int
2

サブグループがサイズによるの場合、データサイズでサブグループを指定できます。

例えば、Size:=3とすると、行の3つの値がそれぞれサブグループとして表示され、浮動棒をプロットするために使用されます。
列ラベル label

入力

int
L

サブグループが列ラベルによるの場合、列ラベルでサブグループを指定できます。

オプションリスト：

  • L: ロングネーム
  • Units: 単位
  • Comments: コメント
プロットタイプ type

入力

int
2

グラフ作成の際のプロットタイプを指定します。

オプションリスト

  • 0=Column: 縦棒
  • 1=Bar: 横棒
出力データ rd

入力

ReportData

計算したデータの出力先を指定します。
