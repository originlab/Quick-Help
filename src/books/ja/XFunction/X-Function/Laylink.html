概要

いくつかのレイヤを1つのレイヤにリンクします。

コマンドラインでの使用法

1.laylink igp:=graph1 destlayers:="2:0" XAxis:=1;

2.laylink igp:=Graph1 igl:=2 destlayers:=3 YAxis:=1 unit:=link;

3.laylink igp:=graph1 destlayers:=2:0 XAxis:=1;

元レイヤにリンクするレイヤを指定します。。

Note: 複数のレイヤを指定する場合には、コロンを使って操作する最初のレイヤと最後のレイヤを分けます。例えば、「1:3」は、1,2,3番目のレイヤという意味になります。
X軸のリンク XAxis

入力

int

-1

X軸スケールにリンクするレイヤを指定します。
オプションリスト

  • -1
    現在のリンク設定を保持します。
  • 0
    X軸に元レイヤをリンクしない。
  • 1
    X軸に元レイヤをリンクする。

オプションリスト

  • 0

  • 1
    Y軸に元レイヤをリンクする。
単位 unit

入力

int

link

レイヤに対する単位を指定します。
オプションリスト

  • page: ページの%
    この単位は、ページサイズに相対的なレイヤサイズを維持するのに役立ちます。これが選択されると、レイヤの『左』、『上』、『幅』、『高さ』のテキストボックスの値は、グラフページの高さと幅のパーセンテージで表示されます。
  • inch:1 inch
    これが選択されると、レイヤの『左』、『上』、『幅』、『高さ』のテキストボックスの値は、インチで測られます。
  • cm:2 cm
    これが選択されると、レイヤの『左』、『上』、『幅』、『高さ』のテキストボックスの値は、cmで測られます。
  • mm:3 mm
    これが選択されると、レイヤの『左』、『上』、『幅』、『高さ』のテキストボックスの値は、mmで測られます。
  • pixel:ピクセル
    これが選択されると、レイヤの『左』、『上』、『幅』、『高さ』のテキストボックスの値は、ピクセルで測られます。
  • point:5 point
    これが選択されると、レイヤの『左』、『上』、『幅』、『高さ』のテキストボックスの値は、ポイントで測られます。
  • link:リンクされたレイヤの6%
    この単位は、ページサイズに相対的なレイヤサイズを維持するのに役立ちます。これが選択されると、レイヤの『左』、『上』、『幅』、『高さ』のテキストボックスの値は、グラフページの高さと幅のパーセンテージで表示されます。

説明

このXファンクションは、様々なレイヤを元のレイヤにリンクすることが出来ます。
