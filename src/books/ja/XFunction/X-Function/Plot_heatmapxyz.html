メニュー情報

作図 > 等高線図: ヒートマップ
作図 > 等高線図: 分割ヒートマップ

概要

一組のXYZデータからヒートマップを作成、あるいは、複数のXYZデータから分割ヒートマップを作成します。

追加の情報

必要なOriginのバージョン: Origin 2019b以降

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

詳細な情報

オプションツリーの詳細

  • X/Y ビン化範囲ツリー

x/y

xツリーはビン化範囲のオプションを指定します。一方、yツリーはYビン化範囲のオプションを指定します。

シンタックス: x.Treenode:=<value>

サンプル:

  • x.inc:=0.3
  • x.stepby:=1 x.intervals:=7
  • x.border.outleft:=1 x.border.outright:=1
ツリーノード ラベル データ型 デフォルト 説明
bin ビン範囲を指定 int 1 Xビン化範囲を定義する方法を指定します。オプション：
  • 0: ビン中心
    ビンの範囲はビンの中心で指定します。
  • 1: ビン終端
    ビンの範囲はビンの終端で指定します。
min 最小ビン中央/最小ビン開始 double <自動> Xビン中央/開始の最小値を指定します。
max 最大ビン中央/最大ビン終端 double <自動> Xビン中央/終端の最大値を指定します。
stepby 間隔 int 0 ビンの間隔を指定します。オプション：
  • 0: ビンサイズ
    ビンサイズで間隔を指定します。
  • 1: ビンの数
    ビンの数で間隔を指定します。
inc ビンサイズ double <自動> 間隔がビンサイズの場合に編集可能です。Xビンの固定のビンサイズを指定します。
intervals ビンの数 int <auto> 間隔がビンの数の場合に編集可能です。Xビンの数を指定します。
