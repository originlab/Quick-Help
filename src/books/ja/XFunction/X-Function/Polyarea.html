メニュー情報

多角形の領域

概要

閉じたプロット領域の面積を計算します。

追加の情報

必要なOriginのバージョン: 8.5 SR0

コマンドラインでの使用法

polyarea iy:=col(b) type:=math;

入力 iy

入力

XYRange

<active>

 入力XY範囲は閉じたプロットである必要があります。
面積の種類 type

入力

int

1

 面積の計算方法を指定します。

オプションリスト:

  • math:数学的面積
    面積の値は、説明セクションにある数式で計算され、正負を区別します。
  • abs:絶対面積
    絶対値で面積の値を計算します。常に正値となります。
面積 area

出力

double

<unassigned>

 結果の面積
