メニュー情報

ワークシート: 条件フォーマット: ヒートマップ

概要

条件フォーマットをワークシートのセルにヒートマップを使用して適用します

コマンドラインでの使用法

1. wheatmap rng:=[Book1]Sheet1!A:B name:="C1R1-C4R0";

2. wheatmap rng:=[Book1]Sheet1!A:B name:="C1R1-C2R0" trcolor.method:=0 trcolor.from:=31306751 trcolor.to:=32497615 trcolor.head:=19 trcolor.tail:=20;

3. wheatmap rng:=[Book1]Sheet1!A:B name:="C1R1-C2R0" trlevel.rescale:=0 trlevel.from:=5 trlevel.to:=30 trlevel.by:=0 trlevel.val:=3 trlevel.first:=6 trcolor.autocolor:=1 trcolor.method:=3 trcolor.name:="Soft Thermometer";

Xファンクションの実行オプション

スクリプトからXファンクションにアクセスする場合、追加のオプションスイッチについてのページを参照してください。

範囲 rng

入力

Range

<active> 条件フォーマットを適用するワークシート内のセルの範囲を指定します。Note:もし一連のテキスト値に従って条件フォーマットを適用したい場合には、この列をあらかじめカテゴリーに設定してください（この列を右クリックしカテゴリーとして設定に設定します。）
名前 name

入力

string

  選択された範囲の名前を指定します。
シートに色を付ける extend

入力

int

0 条件指定でシート内すべての範囲に色を付けるかどうかを指定します。選択した場合、適用された色はすべての入力行範囲に拡大します。このオプションは、範囲が1つの列の時のみ有効です。
レベル trlevel

入力

TreeNode

<unassigned> ヒートマップのレベルをセットします。詳細はこちらのページ をご覧下さい。
色 trcolor

入力

TreeNode

<unassigned> ヒートマップの色をセットします。詳細はこちらのページ をご覧下さい。

説明

レベルの設定に基づいて、ヒートマップを使いセルに色を付けます。
