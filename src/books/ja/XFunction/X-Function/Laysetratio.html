概要

レイヤの幅と高さの比をセットします。

コマンドラインでの使用法

1. laysetratio igp:=Graph1 destlayer:="1:0" ratio:=1.5;

2. laysetratio destlayer:="1" ratio:=1;

入力グラフ igp

入力

GraphPage

<active>

操作するグラフを指定します。
レイヤ destlayer

入力

string

<active>

操作するレイヤを指定します。Note: 複数レイヤを指定するには、コロン(:)を使って、操作する最初のレイヤと最後のレイヤを分けます。例えば、「1:3」は、1番目、2番目、3番目のレイヤという意味になります。
比率 ratio

入力

double

<unassigned>

各レイヤの新しい幅と高さの比を指定します。

説明

このXファンクションは、いくつかのグラフレイヤの幅と高さの比を設定するのに使うことができます。レイヤの幅と高さの比が指定した比よりも小さい場合、レイヤ幅は変更されず、レイヤの高さが新しい比に従ってリセットされます。レイヤの幅と高さの比が新しい比よりも大きい場合、レイヤの高さは変更されず、レイヤ幅が調整されます。
