バッチ処理のツリーノードに関する詳細

Word

wordツリーは、バッチ処理における全てのWordに関連するオプションを指定します。

シンタックス：Word.Treenode:=<value>

例：Word.export:=1

ツリーノード ラベル データ型 デフォルト 説明
export エクスポート先 int 1 分析レポートのエクスポートに使用されるファイルタイプを指定します。

オプションリスト

  • none{0}:なし
    ファイルにエクスポートしない
  • word{1}:Word
    Wordファイルにエクスポート
  • pdf{2}:PDF
    PDFファイルにエクスポート
  • all{3}:Word&PDF
    WordとPDFファイル両方にエクスポート
template Wordテンプレート string <unassigned>

ロードするWordテンプレート (.DOCX, .DOCM, .DOC, .DOTX, .DOTM, .DOT) を指定します。
path エクスポートパス String <unassigned>

Wordテンプレートから作成されたレ分析ポートをエクスポートする場所を指定します。
overwrite 上書き int 0

指定パスにある既存のファイルを上書きするかどうかを指定します。自動名前変更がデフォルトオプションです。

オプションリスト

  • rename{0}:自動名前変更
番号を付けて自動的に名前を変更します。
  • replace{1}:置き換え
既存のファイルを置き換えます。
  • skip{2}:スキップ
既存のファイルをスキップします。
  • ask{3}:尋ねる
上書きするかどうか尋ねるポップアップメッセージを開きます。
