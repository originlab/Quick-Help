Special Event Driven Macros

Special-Event-Driven-Macros

Several macros are triggered by system events. These macros are typically not defined when Origin starts. Define them when you need code to run at a particular time.

  • AutoExec
  • DocBegin
  • DocEnd
  • EndOpen
  • EndToolbox
  • ErrorProc
  • BeforePointProc(AfterPointProc)
  • PointProc
  • QuitToolbox
  • TimerProc
