Description

This function is used to determine whether an input is a text.

  • Since Origin 2019, a blank cell will be treated as a text.
  • Since Origin 2024, left aligned "--" will be treated as a text.

Syntax

int isText(string Str$ )

Parameters

Str$

The value you want to check.

Return

Return 1 if the input is a text or blank, else 0.

Example

string str1$=abc;
isText(str1$)=; 
//should return 1
string str2$=123;
isText(str2$)=; 
//should return 0
//to check if col(B)[1] is a text
isText(col(B)[1]$)=;
isText("")=;
//should return 1 for blank.
