Gampdf

Gampdf-func

Definition

Y = gampdf(X,a,b) computes the gamma probability density function at each of the values in X using the corresponding shape parameters in a and scale parameters in b.

Parameters

x (input, double)
a (input, double)
shape parameter .
b (input, double)
scale parameter .
