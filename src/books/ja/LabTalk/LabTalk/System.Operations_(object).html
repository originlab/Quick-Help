System.Operations

System-Operations-obj

Read and alter a toolbar spacer setting and an OLE in-place activation setting

 

LabTalk Object Type:

Utility

The operations object is a sub-object of the system object. The operations object properties allow you to read and alter a toolbar spacer setting and an OLE in-place activation setting which are set in the OPTION.CNF file. These settings can also be accessed in the Options dialog box.

 

Properties:

Property Access Description
System.operations.inPlace Read/write
numeric

Use OLE in-place activation when editing an Origin object that is embedded in a destination application. Note: OriginLab does not recommend using in-place activation to edit objects.
System.operations.useSpacer Read/write
numeric

Display a toolbar spacer below the menu bar. The toolbar spacer ensures that the toolbar area remains at a fixed width when changing the active child window. 1 = enable display, 0 = disable display.
