System.Script

System-Script-obj

Read and alter display-related settings

LabTalk Object Type:

Utility

The script object is a sub-object of the system object. The script object property allows you to read and set the Script window settings which is set in the OPTION.CNF file. This setting can also be accessed in the Options dialog box.

 

Properties:

Property Access Description
System.script.closePrompt Read/write
numeric

Display a prompt when closing the Script window: 1 = enable, 0 = disable. See also: Type object.
