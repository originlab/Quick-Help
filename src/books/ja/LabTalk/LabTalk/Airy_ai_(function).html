airy_ai

Airy-ai-func

Definition:

evaluates an approximation to the Airy function, Ai(x). It is based on a number of Chebyshev expansions.


For more information please review the s17agc function in the NAG document.

 

Parameters:

x (input, double)
The argument x of the function.
(output, double)
Approximation of the Airy function.
