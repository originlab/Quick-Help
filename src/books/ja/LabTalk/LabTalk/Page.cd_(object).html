Page.cd

Page-cd-obj

The page.cd object has properties and methods related to the Common Display settings on the Layers tab at the Page level of Plot Details dialog. The page object can be referred to using "page".

Properties

Property Access Description
page.cd.APPLLAYERSSTR$ Read/write
string		 A space-separated list of layers (identified by layer name or index) to include or exclude for Common Display.
page.cd.APPLLINKEDLAYERS Read/write
numeric		 Modifications to layer will affect: 1 = only its linked layers, 0 = not only its linked layers but other layers.
page.cd.APPLLINKEDLAYERSINCL Read/write
numeric		 The space-separated list of layers( APPLLAYERSSTR$ ) is to: 1 = include, 0 = exclude.
page.cd.axisscale Read/write
numeric		 Whether Axis Scale and Breaks will be shared between layers: 1 = share, 0 = not share.
page.cd.axisstyle Read/write
numeric		 Whether Axis properties( except Scale and Breaks ) will be shared between layers: 1 = share, 0 = not share.
page.cd.plotcolormap Read/write
numeric		 Whether Colormap settings(Level and Fill) will be shared between layers and plots in same layer: 1 = share, 0 = not share.
page.cd.panellayout Read/write
numeric		 Whether panel layout properties will be shared between layers: 1 = share, 0 = not share. Applicable primarily to Double-Y Trellis Plots.
page.cd.propslayer Read/write
numeric		 Whether layer-level properties (except Link Axes Scales and Layer Area) will be shared between plots: 1 = share, 0 = not share.
page.cd.propsplot Read/write
numeric		 Whether plot-level properties will be shared between plots of the same plot type: 0 = disable, 1 = All (default), 2 = Index, 3 = Name.
page.cd.propsplot$ Read/write
string		 Name string for metadata used in grouping. Empty string to when Index is selected.
page.cd.show Read/write
numeric		 Show or hide plots of the same Name or Index: 1 = enable, 0 = disable.
page.cd.show$ Read/write
string		 Name string for indicating layers. Empty string to indicate layers using index
