Syntax

wks.col.format

Type

numeric

Access

Read/write

Description

Column format: 1 = Numeric, 2 = Text, 3 = Time, 4 = Date, 5 = Month, 6 = Day of Week, 7 = Text and Numeric, 8 = Color, and 9 = Binary as listed in the Format drop-down list in the Worksheet Column Format dialog box.
