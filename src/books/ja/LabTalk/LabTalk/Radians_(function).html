Description

This function is used to convert degrees into radians.

Syntax

 double Radians(double angle)

Parameters

angle

The angle in degrees that need to convert into radians.

Return

Return the radians.

Example

double ff=radians(31.4);
ff=;  //Should return 0.54803338512622.
