Font

Font-func

The font(name) function returns a number corresponding to the font list index of the font specified by name.

int nn = font(some junk);
nn=; // should give 0 as font name given is not valid
nn = font(Arial);
nn=; // should be a >=1 value

// set current worksheet "standard" font
wks.font=font(Courier New);

In version:2021, font() supports convert between DWORD and index.

nFontIndex=font("Segoe UI");
nFontIndex=;
nFontDWORD=font($(nFontIndex));
nFontDWORD=;
font($(nFontDWORD))=;
