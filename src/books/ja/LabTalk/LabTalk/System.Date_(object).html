System.Date

System-Date-obj

LabTalk Object Type:

Utility

The date object is a sub-object of the system object. The date object property allows you to read and alter the default custom date formats. This setting can also be accessed in the Options dialog box.

 

Properties:

Property Access Description
system.date.customFormat1$ Read/write,
string

Read or set the first custom date format. Two custom date formats are available at the bottom of the Date Format drop-down list in the Worksheet Column Format dialog box.
system.date.customFormat2$ Read/write,
string

Read or set the second custom date format. See system.date.customFormat1$.
