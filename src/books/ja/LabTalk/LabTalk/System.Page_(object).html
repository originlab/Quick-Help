System.Page

System-Page-obj

Read and alter axis (a.k.a. layer) and object grid-related settings

 

LabTalk Object Type:

Utility

The page object is a sub-object of the system object. The page object properties allow you to read and alter axis (a.k.a. layer) and object grid-related settings which are set in the OPTION.CNF file. These settings can also be accessed in the Options dialog box.

 

Properties:

Property Access Description
System.page.axisSpacing Read/write
numeric

Set the spacing of the axis (layer) grid lines. The grid sizes are in 1/1000 of the page width. See also: System.Axis, Axis command.
System.page.objSpacing Read/write
numeric

Set the spacing of the object grid lines. The grid sizes are in 1/1000 of the page width.
