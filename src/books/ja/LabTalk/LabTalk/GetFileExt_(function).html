Description

Get the file extension from the full path.

Syntax

 string strExt$ = GetFileExt( string strFile$ )$

Parameters

strFile

file name string.

Return

Return the file extension or empty string if not a file name.

Example

strExt$=GetFileExt("origin.ini")$
