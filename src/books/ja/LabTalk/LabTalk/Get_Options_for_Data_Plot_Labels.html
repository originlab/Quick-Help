Options for Data Plot Labels

Get_Options_for_Data_Plot_Lables

 

 

-qb

Syntax: get name -qb state

Get the state of the font bold for the data label. If state = 1, the data label is font bold, if 0, not font bold.

-qi

Syntax: get name -qi state

Get the state of the font italic for the data label. If state = 1, the data label is font italic, if 0, not font italic.

-qu

Syntax: get name -qu state

Get whether the data label with underline. If state = 1, the data label is with underline, if 0, without underline.
