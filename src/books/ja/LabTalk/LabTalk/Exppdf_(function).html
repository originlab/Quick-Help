Exppdf

Exppdf-func

Definition

exppdf(x,lambda) returns the probability density function of the exponential distribution with rate parameter lambda, evaluated at the values in x.

where is the rate parameter. Mean parameter, Standard Deviation parameter or scale parameter

Parameters

x (input, double)
lambda (input, double)
