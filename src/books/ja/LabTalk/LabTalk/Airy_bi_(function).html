airy_bi

Airy-bi-func

 

Definition:

evaluates an approximation to the Airy function Bi(x). It is based on a number of Chebyshev expansions.

For more information please review the s17ahc function in the NAG document.

Parameters:

x (input, double)
The argument x of the function.
(x) (output, double)
Approximation of the Airy function
