EndToolbox

EndToolbox-macro

The EndToolbox macro is triggered after the getpts [options] npts [(message)] command has been issued and npts points have been selected. A point is selected when you (a) double-click or (b) select with the left or right ARROW key and press ENTER

  • on a data point of the active dataset (for the Data Reader tool).
  • on any point in the active graph window (for the Screen Reader tool).


If you choose to define your own PointProc macro after issuing the getpts command, you must call the EndToolbox macro yourself after npts points have been selected or it will not be triggered. This is because the call for the EndToolbox macro is contained in the PointProc macro that is defined by Origin when the getpts command is issued. The EndToolbox macro is deleted immediately after it is triggered and executed. For more information, see the getpts command
