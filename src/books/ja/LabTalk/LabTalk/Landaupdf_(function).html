Definition

prob = landaupdf(x,mu,sigma) returns the probability density function at each of the values in using the Landau distribution with location paramter and scale parameter .

Syntax

double landaupdf(double x, double mu, double sigma)

Parameters

x

Input, double.

mu

Input, double. Location parameter of the Landau distribution .

sigma

Input, double. Scale parameter of the Landau distribution .

Examples

landaupdf(3, 1, 2) = 0.1031123258564
