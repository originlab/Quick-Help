ライセンスサーバの変更

ライセンスサーバが機能していないか廃棄し、サーバホストを変更する必要がある場合は、システム移行のリクエストを行ってください。 リクエストが処理された後、メールで通知されます。その後、新しいライセンスファイルを取得し、新しいサーバでFLEXnetサービスをセットアップできます。FLEXnetセッアップの詳細については、こちらをご覧ください。

システム移行をリクエストする方法

  1. ご自身のWebメンバーシップアカウントでOriginLabのページにてログインします。
  2. 登録済のOrigin製品を表示 リンクをクリックします。
  3. 移行するシリアル番号にあるチェックボックスにチェックを付けます。
  4. もはや使われていないライセンスサーバーを報告ボタンをクリックします。
  5. フォームの各項目を入力し、ライセンスとインストールのサポートポリシーをお読みください。
  6. リクエストを送信します。

お使いのバージョンがサポート外の場合は、システム転送リクエストについてOriginLabにお問い合わせください 。

日本など米国/カナダ以外の国では、販売代理店にお問い合わせください。

リクエストが処理されたら、新しいライセンスファイルの取得および新しいサーバのセットアップに進むことができます。

  • WindowsまたはLinuxのFLEXnetセットアップ手順を参照してください。

キーワード:ライセンス, 同時起動, システム移行, 非アクティブ化, ディアクティベーションフォーム, ネットワークライセンス, サーバ変更

必要なOriginのバージョン: Origin 8.5SR0以降
