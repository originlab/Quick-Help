FAQ-426 たくさんのファイルをインポートする際、インポートのスピードが遅くなってきた場合どうしたら良いでしょうか。

Improve-Import-Speed

最終更新日:2015/02/04


データファイルをインポートするとき、ファイルの情報はワークブックに記録され、再インポートで使用されます。データファイルをインポートした後ワークブックオーガナイザ（ワークブックのタイトルバーを右クリックしオーガナイザを表示を選ぶ）を開くとインポートされたファイルの名前とパスが左側のパネルに表示されます。たくさんの表記（ファイル）がある場合、インポートのスピードは遅くなってきます。

インポートのスピードを改善するにはシステム変数(@npi) でインポートファイルの情報がワークブックに記録されないようにすることができます。しかしこの情報がないと再インポートできません。 

@npi = 1;  // インポートファイル情報の記録を抑制します


キーワード: インポート, スピード, 遅くなる, たくさんのファイル, npi, 再インポート
