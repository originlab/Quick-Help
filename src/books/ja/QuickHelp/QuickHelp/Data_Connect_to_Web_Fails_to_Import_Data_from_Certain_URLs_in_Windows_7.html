FAQ-995 Windows 7において「データ：Webに接続」メニューを使用すると、特定のURLからのデータインポートに失敗する

Data-Connect-to-Web-Fails-Windows-7

最終更新日: 2019/02/07

Windows 7のPCにOrigin 2019b以降のバージョンをインストールすると、『データ：Webに接続』メニューからデータコネクタを使用した際、HTTPSプロトコルのURLからのインポートが失敗することがあります。また、不可解なエラーメッセージが表示される場合があります（または表示されない場合があります）。

これはWindowsの特定の古い設定に関連しています。

こちらのMicrosoft Easy Fixウィザードをダウンロード、インストールして問題に対処してください：https://aka.ms/easyfix51044

問題の詳細な技術説明については、こちらをご覧ください：https://support.microsoft.com/en-us/help/3140245/update-to-enable-tls-1-1-and-tls-1-2-as-default-secure-protocols-in-wi

キーワード：データコネクタ、Webに接続、Windows 7、HTTPS、ダウンロード失敗、Error -50
