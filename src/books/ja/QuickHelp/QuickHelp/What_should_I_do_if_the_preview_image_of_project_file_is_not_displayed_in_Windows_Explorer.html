FAQ-1161 Windows エクスプローラーでプロジェクトファイルのプレビュー画像が表示されない場合どうしたらよいでしょうか？

OPJU-Preview-Image-Not-Display

最終更新日：2022/12/14

Windowsエクスプローラーの大または特大アイコン表示では、プロジェクトファイルが保存されるときに最後にアクティブだったグラフのプレビュー画像が表示されます。

Windowsエクスプローラーでプレビューイメージが表示されない場合は、最初にフォルダーオプションダイアログ (ファイル: オプションメニュー) の常にアイコンを表示し、縮小版は表示しないにチェックがついていないことを確認してください。

常にアイコンを表示し、縮小版は表示しないがオフの場合は、次のようにしてOPJHandler.dll の登録をお試しください。

  1. Windows のスタートメニューを開きます。
  2. 検索ボックスにcmdを入力します。コマンドプロンプトが表示されます。
  3. 右クリックして、管理者として実行を選択します。
  4. 開いたコマンド画面で以下のように入力します。 
     regsvr32 "<Origin Program Folder>\OPJHandlers.dll"

    たとえば、Origin 2023 をデフォルトパスにインストールした場合、次のように入力します。

    regsvr32 "C:\Program Files\OriginLab\Origin2023\OPJHandlers.dll"
  5. 成功したというメッセージ ボックスが表示されたら、PC を再起動します。
Note:

上記の解決策を試してもプレビューイメージが表示されない場合は、Kasperskyなどのウイルス対策ソフトウェアが OPJHandler.dll をブロックしていないか確認してください。
