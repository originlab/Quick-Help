FAQ-377 Originを起動したときだけ利用できる変数を宣言するには?

Declare-Variable-for-Session

最終更新日:2015/02/04

double型のような型識別子を持つ変数を宣言するとき、変数はセッション変数になります。例えば： 

// 以下で定義された変数はOriginプロジェクトに保存され
// 新規プロジェクトを開いたときは利用できません
cow = 3;
 
// 以下で定義された変数はOriginセッション内で有効ですが
// どのプロジェクトにも保存されません
double cow = 3;

詳細は、LabTalkガイドの変数のスコープの章を確認して下さい。

キーワード: グローバル, プロジェクト
