FAQ-916 Red Hat Enterprise LinuxにFlexnet Serverをインストールする際に"bad ELF interpreter"というエラーが表示されるのはなぜですか？

Bad-ELF-Interpreter-Err-RedHat-Linux-Concurrent

最終更新日:2018/1/14

Flexnet Serverは32ビットアプリケーションであるため、オペレーティングシステムライブラリの32ビット版が必要です。ライブラリファイルは、デフォルトで64ビットバージョンのRed Hat Enterprise Linuxにインストールされません。64ビットRed Hat Enterprise LinuxにFlexnet Serverをインストールすると、次のエラーメッセージが表示されることがあります。

/lib/ld-lsb.so.3: bad ELF interpreter: No such file or directory

この問題を解決するには、rootとして次のコマンドを実行して、redhat-lsbライブラリをインストールする必要があります。

yum -y install redhat-lsb
