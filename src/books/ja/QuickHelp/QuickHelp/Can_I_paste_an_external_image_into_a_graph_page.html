FAQ-218 外部の画像をグラフに貼り付けることはできますか?

Paste-img-to-Graph

最終更新日: 2019/1/6

編集：貼り付けを選択してコピーした画像を貼り付けられます。以下の値によって、EMFかDIB（環境依存型Bitmap）で貼り付けられます。 

system.display.pastemeta

スクリプトウィンドウ（メニューのウィンドウ: スクリプトウィンドウ）を起動し、次の実行します。 

system.display.pastemeta=1; // EMF画像として貼り付け
system.display.pastemeta=0; // bitmap画像として貼り付け

LabTalk system.display objectについては、system.displayをご覧ください。

Note：挿入：ファイルからの画像メニューコマンドを使用して、アクティブなグラフウィンドウに画像を挿入することもできます。メニューコマンドはsystem.display.pastemetaの値によって変わりません。

キーワード: 図, 注釈

必要なOriginのバージョン: Origin 8.5SR0以降
