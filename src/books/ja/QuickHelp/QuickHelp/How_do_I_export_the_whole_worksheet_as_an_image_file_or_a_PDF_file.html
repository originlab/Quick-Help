FAQ-456 ワークシート全体を画像ファイルまたはPDFファイルでエクスポートするにはどうしたら良いですか?

ExpWks-as-img-PDF

最終更新: 2018/07/31

イメージファイルにエクスポートするには

全てのワークシートをイメージファイルとしてエクスポートするには、ワークシートがアクティブな状態でファイル:エクスポート: イメージファイル形式でと操作します。

expWks Xファンクションダイアログが開きます。このダイアログはグラフエクスポートダイアログにエクスポート領域設定が加えられたものです。

ダイアログで、画像の種類ドロップダウンリストからエクスポートする画像の種類を選択します。

エクスポート設定ブランチを開き、エクスポート領域ドロップダウンリストで全体を選択します。(このオプションは表示範囲を選ぶことも、選択範囲だけを選ぶこともできます。)

出力画像の幅を設定でき、高さは縦横比を維持するように自動的に設定されます。

ワークシート内のセルグループは、カスタムレポートシートを作成するため、グラフ、画像、レポートテーブル、テキストを配置するため自由に統合することができます。

このカスタムレポートシートの作成機能は、ワークシートエクスポートファイルの幅を自由に設定する機能を持ち、大きなポスターなどのプレゼンテーション資料を作成できます。

複数ページをPDFファイルとしてエクスポート

全てのワークシートをPDFファイルとしてエクスポートする際は、ワークシートがアクティブな状態でファイル：エクスポート：マルチページPDF形式でと操作します。

これにより、expPDFｗ Xファンクションダイアログが開きます。

画像の設定：複数ページブランチではカバーページとページ番号付けを追加できます。

印刷設定のワークシート自身の設定を使うにチェックを付けておくと出力の様子はファイル：印刷プレビューで見た通りになります。この表示はファイル：印刷でオプションボタンをクリックすると編集できます。OKOKを押してオプションダイアログを閉じ、印刷ダイアログをキャンセルして閉じます。

ワークシート自身の設定を使うのチェックを外した場合、先ほどの印刷オプションでディスプレイの管理を行います。
