FAQ-817 2D線形グラフにドロップラインを加えるには、どのようにしたらよいでしょうか？

How-to-add-drop-lines-to-line-plot

最終更新日:2016/03/17

Origin 9.1より前では、2D 線形グラフに次の回避策でドロップラインを追加できます。

  1. プロットをダブルクリックして、作図の詳細ダイアログを開きます。
  2. 作図形式 を 線＋シンボル にします。（ダイアログの左下のドロップダウン）ドロップライン タブが作図の詳細に追加されます。
  3. ドロップライン タブをクリックして、ドロップラインの設定を行い、適用 をクリックします。
  4. 独立 したプロットのシンボルを削除するには、シンボル タブをクリックして、シンボルギャラリー（下図 プレビュー）を開き、ギャラリーの下にある空のセルを選択します。従属 グループのプロット (編集モード、グループ タブ）の場合は、シンボルタイプ を 推移 = なしに設定し、シンボル タブで、形状 を 0 シンボルなし にして 適用 または OK をクリックします。

Note: これらのシンボルを取り外す方法は、シンボルサイズ を 0 にするより良く機能します。また、こちらのページもご参照ください。FAQ 破線付折れ線＋シンボルプロット.

Origin 9.1以降では、作図の詳細のドロップラインタブから、直接簡単にドロップラインを追加することが出来ます。

キーワード：ドロップライン、線＋シンボル、シンボル、シンボルサイズ
