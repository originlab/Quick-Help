FAQ-330 どのようにカーネル密度グラフを作図したら良いでしょうか？

kernel_density_graph

最終更新日: 2023/04/26


1D カーネル密度グラフの場合

  1. メニューから、作図：統計：ヒストグラムを選択してヒストグラムを作図します。
  2. グラフをダブルクリックして、作図の詳細ダイアログを開きます。
  3. 左パネルでプロットを選択した状態で、右パネルで分布タブを開きます。分布曲線のドロップダウンリストから、カーネルスムースを選択します。
  4. データタブを開き、ビンを隠すにチェックを付けます。


2D カーネル密度グラフの場合

  • メニューから、作図>等高線図：2Dカーネル密度を選択します。


キーワード:カーネル密度, 1D, 2D, カーネル密度推定, 密度プロット, 線

必要なOriginのバージョン: Origin 2015 SR0以降
