FAQ-213 以前のバージョンで定義したカスタムカラーを使用するには、どのようにしたらよいでしょうか？

share-custom-color

最終更新：2019/01/07

以前のバージョンのOriginでカスタムカラーを定義して使用していて、同じPCにインストールしたアップグレードバージョンでも引き続き使用したい場合、以下の操作を行います。

  1. 環境設定メニューの、ユーザファイルの移行を選択し、ダイアログを開きます。
  2. ソースの場所を旧バージョンのユーザファイルフォルダにして、表示のセクションで設定（メニュー、ボタン...）のチェックを付けます。
  3. 左側のパネルから CustomColors.ini ファイルを選択し、 ボタンをクリックして右パネルに追加します。
  4. 移行ボタンをクリックし、CustomColors.ini ファイルを現在のOriginのUser Filesフォルダに移行します。


もちろんUser FilesフォルダにあるCustomColors.iniファイルをコピーし、新しいバージョンのOriginのUser Filesフォルダにペーストするという操作でもファイルを共有することができます。

キーワード: カスタムカラー, 共有, ユーザファイル移行, 設定の移行, 以前のバージョン, User Files

必要なOriginのバージョン: 9.0 SR0以降
