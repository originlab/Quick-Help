FAQ-354 錠前のアイコンの色を変更できますか？

最終更新日:2015/02/04

change-lock-color

Originで行った処理で、再計算の機能が使用されている場合、結果データセットと結び付けられている錠前のアイコンが表示されます。錠前のアイコンは、そのときの状況に応じて異なる色で表示します。

これらの色を変更したい場合、ユーザファイルフォルダを開き、Origin.ini ファイルを確認します。これをメモ帳などのテキストエディタで開き、[Color]セクションの以下の行を非コメント化して鍵の色を定義します。 

PadlockClean=0 255 0
PadlockDirty=255 255 0
PadlockBroken=255 0 0

そして、0から255までの赤、緑、青を表現する値である3つの数字を変更します。ここで、

PadlockClean は、再計算が必要ない状態です。デフォルトは緑色です。

PadlockDirty は、再計算があることを示します。デフォルトは黄色です。

PadlockBroken は、処理に問題が発生していることを示します。デフォルトは赤色です。

必要なOriginのバージョン:9.1SR0

キーワード: 色, 構成, origin ini, 鍵, ロック, 再計算, 再計算モード
