FAQ-143 レイヤの縦横比を軸範囲と同じにするには、どうしたら良いでしょうか?

最終更新日:2015/07/27

Set-Aspect-Ratio-of-Layer

XYスケール範囲の比率をXとY軸の実際の長差に合わせるには、

  1. メインメニューのフォーマット：作図の詳細(レイヤ属性)をクリックします。
  2. レイヤの大きさ・描画スピードタブでLink Axis Length to Scale with X:Y Ratioチェックボックスにチェックをしたら、編集ボックスに「1」が入力されていることを確認してください。
  3. OK をクリックして設定を保存します。

必要なOriginのバージョン:2016SR0

9.1以前のバージョンでは、以下のスクリプトにより、アクティブになっているレイヤの縦横比をXとYの軸範囲として設定できます。 

dx=x2-x1; // Xスケール範囲を検索
dy=y2-y1; // Yスケール範囲を検索
temp=layer.unit; // 現在のレイヤの単位を保存
layer.unit=2; // インチに変更
 
if(dx==dy) layer.height = layer.width; 
else
if(dx>dy) 
{
   layer.height=layer.height*dy/dx;
} 
else 
{
   layer.width=layer.width*dx/dy;
}
layer.unit=temp; // 元のレイヤの単位に戻す

キーワード: サイズ変更, 軸, 寸法, サイズ, ピクセル, 画像, 作図
