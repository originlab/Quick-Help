FAQ-927 新しいOPJUフォーマットを古いOPJに変換するにはどうすればよいでしょうか。

Convert-OPJU-Project-File-Format-to-Older-OPJ

最終更新日：2023/1/5

Origin 2018では、Unicode(UTF-8)に対応したプロジェクトファイルOPJUを使用します。OPJUファイルは古いバージョンOriginでは開くことができません(しかし、Origin 2018から2022bでは古いOPJフォーマットでファイルを保存することができます)。Origin 2023以降のバージョンではOPJ形式では保存できず、OPJUで保存する必要があります。

Originで、OPJUファイルが開けない場合は:

  1. OriginLabのこのページからOriginビューワをダウンロードします。
  2. ビューワファイルを解凍します。
  3. EXEファイルをダブルクリックするとビューワが起動します。(インストールは必要ありません)、ビューワのメニューでファイル: 名前をつけてプロジェクトを保存を選んで、OPJ形式で保存し直します。


Originビューワの詳細につきましては、OriginLabのこちらのページをご覧ください。

キーワード:後方互換, ファイル形式, プロジェクトファイル
