FAQ-922 電子メール経由で送信時のOriginファイル破損の防止

Email-incompatible-file-type-corruption

最終更新日:2018/2/28

電子メールの添付ファイルとして送付されたOriginファイルがOriginで開くことができない、またはOriginで次のようなエラーが出るという報告がいくつかあります。

File corruption: marker not at structure end: 000000000000001F File format error.Incompatible file type or file may be corrupted.

この問題は、一部の電子メールクライアントやウイルススキャナが添付のOriginファイルを微妙に変更したために壊れてしまうことがあるのが原因です。

この問題を軽減するために、Originファイルを電子メールに添付する前にZIP形式などに圧縮することを強くお勧めします。

キーワード:電子メール, 添付, プロジェクト, 破損, ファイル形式エラー
