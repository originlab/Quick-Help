FAQ-1031 FLEXnetサーバの開始時に "orglab exited with status 53" のエラーが表示されるのはなぜですか？

FLEXnet-Server-VC2012

最終更新日：2023/07/27

FLEXnet server バージョン 11.19以降は Microsoft Visual C++ 2015 Redistributable パッケージが必要です。これがインストールされていないと、FLEXnetサーバの開始時にログファイル内に次のようなメッセージが表示されます。 

orglab exited with status 53 (this status is reserved but will never be generated)

または 

 9:12:34 (lmgrd) orglab exited with status 255 ()
 9:12:34 (lmgrd) Since this is an unknown status, license server 
 9:12:34 (lmgrd) manager (lmgrd) will attempt to re-start the vendor daemon.


この問題を解決するには、

  • FLEXnet 11.16またはそれ以前のバージョンを使用している場合、11.19以降のバージョンにアップグレードしてください。最新のバージョンはこちらからダウンロード可能です。
  • 最新バージョンで実行してもエラーメッセージが表示される場合は、Microsoft Visual C++ 2015-2019 Redistributable Packageをご自身でダウンロードし、インストールしてください。
  • 32bit OSの場合、vcredist_x86-2015-2019.exeをダウンロードしてください。
  • 64bit OSの場合、vcredist_x64-2015-2019.exeをダウンロードしてください。

キーワード:ライセンス, ネットワーク, 同時起動, サーバエラー, err 255, status 53, 7,10015, ソケット未接続
