FAQ-677 LabTalkスクリプトで自分のグラフテンプレートを使用したグラフを作成するには？

最終更新日:2017/7/5

Graph-Template-LabTalk

作成したグラフの編集のあと、のちに同じ形式のグラフを作成する場合は、グラフテンプレートとして保存します。テンプレートを使用してグラフを作成する場合、作図：テンプレートライブラリを選択するか、worksheet コマンドを使用してスクリプトを実行します。

保存時にクローンテンプレートのマークにチェックをつけたかどうかで、作図の際のLabTalkシンタックスは異なります。

  • クローンテンプレートのマークにチェックを付けなかったテンプレートを使った作図のシンタックスは次のようになります。
    worksheet -p n templatename

    nの値には、テンプレートに合わせたプロットの種類を指定します。プロットオプションの利用可能な値は、プロット種類IDを確認してください。

    例えば、データを選択して以下を実行します。

    worksheet -p 200 MyTemplate;

    すると選択したデータが折れ線グラフ(200)の形式でテンプレートにプロットされます。

  • クローンテンプレートのマークにチェックを付けたテンプレートを使った作図シンタックスは次のようになります。
    worksheet -pa ? templatename

    クローンテンプレートでは、最初にデータを選択する必要はありません。ワークシートをアクティブにして次のように実行します。

    worksheet -pa ?MyMulti_Panel.otp

    ワークシート内のどのデータをプロットするかは、クローンテンプレートに情報があります。


キーワード:Labtalk, グラフテンプレート
