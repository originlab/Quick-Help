FAQ-879 引き出し線をシンボルの中央から開始するには?

Start-LeaderLine-from-Symbol-Center

最終更新: 2018/07/23

シンボルとラベルを結ぶ引き出し線はデフォルトでシンボルの縁から始まります。そして、線とシンボルの間には間隔が設定されています。シンボルの中心から引き出し線を引きたい場合は、次のようにします。

  1. 同じソースデータの散布図を追加します（レイヤ内容ダイアログか作図のセットアップダイアログを使用）。
  2. 新しいプロットのシンボルサイズを0にします（作図の詳細ダイアログのシンボルタブ）。
  3. 新しいプロットのラベルを有効にして引き出し線を表示します（作図の詳細ダイアログのラベルタブ）。
  4. 次のスクリプト 
    set %c -gtl 0;
    をコマンドウィンドウで実行し、引き出し線とシンボルの間隔を0に設定します。
ノート

上記のLabTalkはアクティブなグラフにのみ有効です。１つのデータに対して２つのグラフを作り、シンボルプロットが１番目ではない場合は、グラフをアクティブに出来ないので、このスクリプトは機能しません。
詳細はこちらを確認してください。
