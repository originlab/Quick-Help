FAQ-167 統合化されたグラフウィンドウのフォーマットを保存できますか?

MergedGraph-Save-Formatting

最終更新日:2015/02/04

グラフ操作：グラフウィンドウの統合メニューを使って、複数のグラフを1つにすることができます。開いたダイアログで、統合したいグラフウィンドウを選びます。このダイアログでは、統合後のグラフのプレビューを確認できます。

アクティブフォルダ内のグラフウィンドウを統合する場合、ダイアログを開いて統合するグラフを指定する時間を省略できます。このダイアログは他のOriginのダイアログと同様、ダイアログの設定をテーマとして保存でき、フライアウトメニューから利用できます。そのため、グラフの統合を行うときにメニューのテーマを使用することで時間を短縮できます。

例えば、4つのグラフから2x2の4区分グラフとして統合するには、以下のように操作します。

  1. 「グラフ操作：グラフウィンドウの統合」を選びます。
  2. 統合のドロップダウンリストが「現フォルダ中の全て（オープンのもの）」に設定してあることを確認します。
  3. 配置の設定のブランチを開き、行の数と列の数をそれぞれ2にします。
  4. 必要に応じて間隔などの設定をします。
  5. ダイアログ上のダイアログテーマボックスの右にある矢印ボタンをクリックします。名前を付けて保存を選び、テーマ名を2x2にします。
  6. OKをクリックして統合を行うか、統合を行わない場合はキャンセルをクリックします。

後で2x2のグラフを作成したい場合、「グラフ操作：グラフウィンドウの統合：2x2」を選びます。ダイアログは開かれずテーマの設定が使用され、アクティブフォルダ内の開かれているグラフが統合されて新しいグラフウィンドウが作成されます。

キーワード: 統合, テーマ, フォーマット
