FAQ-224 最後に行ったフィットのパラメータ値を新しいフィットの初期値として使う方法は?

Fitted-Result-as-Initial-Value

最終更新日:2015/02/04

分析結果の再計算機能を使って、パラメータを変更して、分析結果を更新することが出来ます。非線形フィットを実行すると、再計算モードは手動にセットされます。フィットボタンをクリックした後、緑色の錠前をクリックして、パラメータの変更を行うことが出来ます。

フィットボタンをクリックするか、または収束までフィットボタンをクリックしてOKボタンをクリックするかにより、使用される初期値がパラメータタブに表示されます。

フィットの正確さが向上し、前回のフィットで求めたパラメータ値でフィットを開始したければ、収束までフィット ボタンを最初にクリックし、それからOK ボタンをクリックします。

単にフィットボタンをクリックした場合は、パラメータの変更ダイアログを再度開いても、前回と同じ初期値が入力されています。


キーワード: 最終, 継続, 初期値, 非線形, フィット, 違い, 違う
