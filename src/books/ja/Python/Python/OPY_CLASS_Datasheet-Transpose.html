Description

Transpose a Datasheet

Syntax

Transpose(SelOnly = False)

Parameters

SelOnly
0: Transpose the Datasheet
1: Transpose the select range

Return

0 for success

Examples

EX1

import PyOrigin
#Assume a worksheet is active
wks=PyOrigin.WorksheetPages('Book1').Layers(0)
Col=wks.Columns(0)
Col.SetFormula('ran()')
Col.ExecuteFormula()
aa=wks.Transpose()
