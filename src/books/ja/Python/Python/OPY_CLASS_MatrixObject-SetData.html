MatrixObject::SetData

Description

Put two dimensional data into a specified MatrixObject

Syntax

SetData(pVar,  Row,  Col)

Parameters

pVar
The 2D array of data to be set into the column.
Row
Row number
Column
Column number

Return

True indicating set data successfully and False indicating it is failed.

Examples

EX1

import PyOrigin
pArr2D = [[3 for x in range(5)] for x in range(5)] 
MatrixPage=PyOrigin.MatrixPages('MBook1')
MatrixPage.Layers('MSheet1').MatrixObjects(0).SetData(pArr2D, 0, 0)
