Description

Indicate the position of this object in current container

Name

Index

Examples

EX1

import PyOrigin
#Create a new workbook with 3 sheets
wkPage = PyOrigin.ActivePage()
wksLayer3 = wkPage.Layers(2)
print(wksLayer3.Index)
wksLayer3.Index = 0 #Set the third sheet to be first
