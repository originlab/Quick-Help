Description

Retrieve the comments from an object that supports Comments.

Syntax

GetComments()

Parameters

Return

The comments text string

Examples

EX1

import PyOrigin
#Assume a worksheet is active
wks = PyOrigin.ActiveLayer()
print('The wks comments: %s' % wks.GetComments())
