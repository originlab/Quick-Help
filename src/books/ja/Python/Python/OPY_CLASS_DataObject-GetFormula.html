Description

Get the formula of the DataObject.

Syntax

GetFormula()

Parameters

Return

String with the DataObject formula.

Examples

import PyOrigin
wks=PyOrigin.WorksheetPages('Book1').Layers(0)
Col=wks.Columns(0)
Col.SetFormula('ran()')
Col.ExecuteFormula()   
print(Col.GetFormula())
