Www.originlab.com にて、Origin 2023の新機能ハイライト紹介とビデオをご覧ください。

カテゴリーごとの機能および改良

グラフ作成

新しいグラフ形式

  • 1つのグラフレイヤに2つの独立したY軸を表示
  • 三角座標のベクトルグラフ
  • 3D積み上げヒートマップ
  • 3D積み上げ曲面

一般

  • グラフエクスポート (詳細) <前回どおり>のホットキー
  • エクスポートパスをメッセージログのクリック可能なリンクとして出力
  • レイヤ、プロット、描画オブジェクトの選択可/不可の切り替え
  • 複数グラフをレイアウトページに統合する際の設定を追加
  • デフォルトの参照線スタイルを設定
  • 参照線ラベルの位置を追加
  • 軸の参照線の改善
  • 新しい積み上げグラフのオプション
  • バッチ作図でのショートネームの取り扱いを向上
  • ワークシートへ切り替え時にデータ強調表示モードをオフにするシステム変数
  • 副目盛ラベルのサイズを調整
  • より正確にレイヤをページに合わせる
  • 異なるサイズのレイアウト間で貼り付けるときにグラフとワークシートの寸法を維持
  • 作図 > 複数区分/軸：複数Y軸ダイアログにプロットタイプ = 縦棒を追加
  • データポイントの移動の改良
  • 色の作成ダイアログでの色補間機能の改良
  • 3D/等高線グラフの行列を別の行列に置換
  • オブジェクトマネージャでレイヤの名前変更

テキスト、凡例、ラベル

  • レイヤタイトルの改良
  • ページタイトルの追加
  • 角度計測ツール
  • ワークシート、テキストラベルでの正/負 (+/-) 記号のカスタム表示
  • 関数グラフの凡例エントリを非表示に
  • レイヤに他のプロットがある場合、関数プロットを軸タイトルにしない
  • ポイント毎の凡例でソート順を指定
  • 凡例にカスタムカラーブロックを手動で追加
  • シンボルマップダイアログの改良
  • 折り返しあり/なしのテキストのインプレース編集
  • アスタリスクブラケットの形状を追加
  • ボックスチャートにボックススケールを追加
  • ワークシートのセル置換フォーマットの機能強化
  • 凡例オブジェクトダイアログで余白を挿入するためのセパレータボタンを追加
  • 関数グラフの凡例に実際の式を表示

グラフ編集

  • LabTalkプロパティによるデータプロットの並べ替えオプション
  • 散布図、線+シンボルグラフポイントスキップオプションの追加
  • ミニ ツールバーを使って2D/3Dレイヤレベルで自動再スケールを制御
  • 交互に目盛りとラベルを表示
  • 短いZ軸をサポート
  • 等高線図の改善
  • リンクされたレイヤの縦棒/横棒/ボックスの間隔を個別に設定
  • サンバースト/ドーナツグラフの改善
  • グループ化ボックスチャートの列の値ごとにデータポイントを色を付け
  • 3D折れ線/散布図/トラジェクトリのスプライン接続をサポート
  • 3D棒のグラデーション塗りつぶしにパレットを適用

データの取り扱い

ワークシート

  • ワークシートの縞模様の行
  • ワークシートでのウィンドウ枠の固定
  • ワークシート保護のGUIサポート
  • ワークブックのシートを非表示にする
  • 非表示をスキップするためにワークシートを積み重ねる
  • ワークシートセルのリッチテキストの折り返し
  • ワークシート移動の使いやすさの向上
  • CTRLを押してセルを選択解除
  • CTRL+SHIFT+矢印キーで行/列の選択
  • URLリンクされたセルの整列とフォーマット
  • 列の削減 (wreducecols) のオプションによる統合
  • インテリセンス/オートコンプリートの改善
  • 列の数式の自動入力機能の改良
  • ステータスバーでラベル行選択時に統計を表示
  • URLセルリンク表記の編集を簡便化
  • 列選択の改良
  • ワークシートのセルに画像を挿入する際に方向の設定を追加
  • 列のアンスタック (wunstackcol) でグループ化情報の出力を制御
  • 選択したワークシート行をミニツールバーボタンで移動
  • 強制的にセルのキャッシュをクリア

ノート

  • ウィンドウの端に近い場合のセルノート配置位置の改善
  • セルノートのフォントと間隔の改善

インポート

  • CSVおよびExcelコネクタにインポート後のスクリプトを含む新しいオプションを追加
  • WGS84形式のシェープファイルのインポートをサポート
  • Excelの結合セルのインポートでの改善
  • ユーザ定義フィルタを編集するためのiwFilterダイアログへのGUIを追加
  • Webに接続ダイアログの改良

エクスポート

  • 行列を画像にエクスポートする際のGeoTIFFサポートの改善
  • ワークシートを画像にエクスポートの改良
  • 簡易版エクスポートでのプレビューオプション
  • ブラウザグラフを動画としてエクスポート (GIF, TIFF, AVI)

行列

XY座標追加、NetCDF

解析

一般

  • グループ列による正規化 (rnormalize)
  • 参照列によるXYの削減 (reducexy)
  • 入力データと同じシートにグループなどの変数変更をサポート
  • XYZZZ...列を1つのシートの複数行列オブジェクトに変換
  • 出力列のデータ識別子のより詳細な制御
  • 列の統計の出力における改良
  • 2群のt検定および (PSS) 2群のt検定の結果に脚注を追加
  • 分析ツールの入力の改善

ガジェット

  • ガジェットのボタンをラベルサイズの制御
  • クラスターガジェットでZ統計およびROI外のフェードアウトをサポート

画像処理

  • イメージウィンドプのROIからリンク行列を作成
  • ROI が非常に小さい場合の強度プロファイルの改善

プログラミング

LabTalk

  • LabTalk関連の機能追加 2023.

Python

  • Python関連の機能追加 2023.

その他

使いやすさの機能

  • フォルダとウィンドウ操作のツールバー
  • プロジェクトエクスプローラのフォルダにフォルダノートを追加
  • プロジェクトエクスプローラフォルダのミニツールバー
  • Origin 2023での新しいツールバーレイアウト
  • ウィンドウを整列機能を改良
  • ショートカットを改善して2つのフォルダからウィンドウを表示
  • ウィンドウサイズを復元
  • ズームとスケールインの簡易なホットキー
  • プロジェクトフォルダに簡単に保存するためのウィンドウをダイアログとして保存(windowsaveas) オプション
  • Originファイル全てについてUFFの最近のプロジェクト履歴に追加
  • OPJとしての保存サポートを終了し、OPJUとして保存
  • グラフ メーカーの結果で小数点のカスタマイズ
  • ラスター画像のデフォルトの色深度が増加
  • 画像、グラフ ウィンドウをAVI、GIF、TIFF としてエクスポート
  • セルの書式設定をExcelエクスポートでサポート
