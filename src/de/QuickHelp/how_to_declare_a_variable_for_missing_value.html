FAQ-378 Wie deklariere ich eine Variable zu einem fehlenden Wert?

Declare-Variable-for-Missing-Value

Letztes Update: 04.02.2015

Sie können eine Variable mit dem Ausdruck 0/0 deklarieren, um sie als einen fehlenden Wert zu definieren. Zum Beispiel:

misval = 0/0; misval =;//Should return --

Zusätzlich wird ein fehlender Wert in LabTalk als False ausgewertet. Zum Beispiel:

misval = 0/0; type $(misval?1:0);//Should return 0 (False)

Schlüsselwörter:fehlender Wert, deklarieren, Variable, false
