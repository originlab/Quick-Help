FAQ-1097 Nach dem Upgrade öffnen die Schaltflächen der Symbolleiste Import für ASCII-Dateidaten und Excel importieren unterschiedliche Dialoge

ascii-import-excel-button-change

Letztes Update: 09.04.2021

Datenkonnektoren stellen Origins neueste Importmethode dar. Da Anwender sich von den älteren auf X-Funktionen basierenden Importroutinen zunehmend den Konnektoren zugewandt haben, traf OriginLab die Entscheidung, die Schaltflächen ASCII-Dateidaten und Excel importieren auf der Symbolleiste Import neu zu kodieren, um einen direkten Zugriff auf to Text/CSV- und Excel-Datenkonnektoren zu bieten. Diese Änderung wurde mit dem Release von Origin 2021b umgesetzt.

Um diese Änderung zurückzunehmen, so dass eine oder beide der älteren X-Funktions-basierten Importroutinen, die mit diesen beiden Schaltflächen verbunden sind, wiederhergestellt werden, ändern Sie den Wert der Systemvariable @TIB.

@TIB = 1; // open impASC with "Import Single ASCII" button @TIB = 2; // open impMSExcel with "Import Excel" button @TIB = 3; // reconnect both impASC and impMSExcel to toolbar buttons

Eine Anleitung, wie Sie den Wert einer Systemvariablen ändern, finden Sie in dieser FAQ.
