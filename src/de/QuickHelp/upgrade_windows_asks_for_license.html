FAQ-789 After upgrading my Operating System Origin asks me for a license

Upgrade-Windows

Last Update: 11/10/2016

Upgrading your Operating System can delete the Origin license. In most cases, you can simply retrieve your single-user / multi-user Node-Locked license.

If you receive an error during obtaining a license stating that you have reached the maximum number of licenses, please fill out a System Transfer Request form with these instructions:

  1. Open a web browser to http://www.originlab.com/reginfo
  2. Log in
  3. Click on the serial number
  4. Click on Deactivate License or Transfer License
  5. Click on “request a license transfer” in the Note at the bottom of the page
  6. Submit the form

Note: If you purchased a Student Version license through OnTheHub, please contact the OriginLab Licensing Department at reginfo@originlab.com and include the following:

  • Original Origin Computer ID
  • New Origin Computer ID
  • OnTheHub Access Code
