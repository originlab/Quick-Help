FAQ-254 Wie kann ich einen einzelnen Datensatz mit mehreren Instanzen einer Funktion anpassen?

Single-Data-Multiple-Fit

Letztes Update: 04.02.2015

Führen Sie folgende Schritte durch:

  1. Drücken Sie mit einem aktiven Arbeitsblatt Strg + wählen Sie zwei oder mehr Datenbereichen in derselben Spalte aus. Verwenden Sie mit einem aktiven Diagramm das Hilfsmittel Regionaler Datenselektor , um zwei oder mehr Bereiche auszuwählen.
  2. Nachdem die Bereiche festgelegt sind, öffnen Sie den Dialog NLFit, indem Sie Analyse: Anpassen: Nichtlinearer Fit auswählen.
  3. Wählen Sie eine Funktion auf der Registerkarte Einstellungen unter Funktionsauswahl.
  4. Setzen Sie den Multi-Daten Fit-Modus auf Globaler Fit auf der Registerkarte Einstellungen in der Datenauswahl.
  5. Auf der Registerkarte Parameter können Sie Parameter teilen und auf der Registerkarte Code Bedingungen können Sie das Kontrollkästchen Lineare Nebenbedingungen aktivieren aktivieren und ggf. Bedingungen eingeben. Wenn beispielsweise mit Gauss angepasst wird, wobei jeder Peak dieselbe Breite besitzt, würden Sie das Kontrollkästchen Teilen für den Parameter w aktivieren. Wenn mit Boltzmann angepasst wird, bei dem grundsätzlich ein Boltzmann "über dem anderen sitzt", so dass die obere Asymptote (A2) des unteren Boltzmann mit der unteren Asymptote (A1) des oberen Boltzmann verbunden ist, würden Sie die Bedingung als A1_2 = A2 eingeben.

 

Schlüsselwörter:Wiederholung, Peak, wiederholen
