FAQ-14 Beim Start von Origin werde ich aufgefordert, es als Administrator auszuführen. Ich bin der Administrator dieses Computers, was also soll ich tun?

Run-Origin-as-Admin

Letztes Update: 04.02.2015

Um Origin als Administrator unter Windows 8, 7 oder Vista auszuführen, klicken Sie mit der rechten Maustaste auf das Origin-Programmsymbol und wählen Sie Als Administrator ausführen.
