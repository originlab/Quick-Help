FAQ-642 Welche Art von Gewichtung kann in der Anpassung verwendet werden?

weightInFitting

Letztes Update: 20.07.2018

Origin bietet eine Anzahl von Gewichtungsmethoden im Anpassungsprozess. Einige von ihnen verwenden Fehlerbalken als Wert für die Gewichtungen (eine zugewiesene Fehlerspalte muss festgelegt werden), einige von ihnen berechnen Mengen als Gewichtung.

  • Direkte Gewichtung
  • Instrumental
  • Statistisch
  • Beliebiger Datensatz
  • Varianz ~ y^2
  • Varianz = a*y^b
  • Varianz = c^b+a*y^b
  • Varianz = a*y^b*c^(tlast−t)
  • Varianz ~ yfit
  • Varianz ~ yfit^2
  • Varianz = a*yfit^b
  • Varianz = c^b+a*yfit^b
  • Varianz = a*yfit^b*c^(tlast−t)

Einzelheiten für jede Gewichtung finden Sie unter: Anpassen mit Fehlern und Gewichtung
