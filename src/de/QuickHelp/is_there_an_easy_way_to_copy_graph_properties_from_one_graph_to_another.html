FAQ-164 Gibt es eine einfache Möglichkeit, Diagrammeigenschaften von einem Diagramm auf ein anderes zu übertragen?

Share-Graph-Theme

Letztes Update: 07.01.2019

Zwei Methoden:

  1. Klicken Sie einfach mit der rechten Maustaste auf das Diagrammelement und wählen Sie Format kopieren im Kontextmenü. Sie können dann mit Format einfügen das Format auf ein anderes Diagramm oder einen anderen Layer im gleichen Diagramm übertragen. Sie können auch Bearbeiten: Format einfügen (Erweitert)... wählen, um das Format auf mehrere Diagramme anzuwenden.
  2. Wenden Sie Designs auf einer Grafikseite an. Designs bieten eine einfache Möglichkeit, Eigenschaften eines Diagramms zu kopieren und sie auf ein beliebiges oder alle Diagramme in einem Projekt oder sogar alle künftigen Diagramme anzuwenden (in Fall eines Systemdesigns). Um Designs zu verwenden, drücken Sie F7 oder wählen Sie Einstellungen: Designs verwalten, um den Dialog Designs verwalten zu öffnen und das richtige Design auf das Diagramm anzuwenden.

 

Schlüsselwörter:Design, Format
