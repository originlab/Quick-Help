FAQ-813 Wie zeige ich die Option X-Fehler im Dialog Diagrammeinstellungen?

Show-XErr-in-PS

Letztes Update: 24.01.2016

Standardmäßig werden im Dialog Diagrammeinstellungen Optionen für die X-, Y- und YEr-Diagrammzuordnung im mittleren Bedienfeld angezeigt. Um die Option X-Fehler zu zeigen, können Sie mit der rechten Maustaste in das mittlere Bedienfeld der Tabelle klicken und X-Fehlerbalken im Ausklappmenü auswählen. Sie können auch die Optionen yEr+ und yEr- einschalten, indem Sie Y-Fehlerbalken Plus/Minus im Ausklappmenü aktivieren.

Schlüsselwörter:Diagrammeinstellungen, X-Fehler, Y-Fehlerbalken Plus, Y-Fehlerbalken Minus
