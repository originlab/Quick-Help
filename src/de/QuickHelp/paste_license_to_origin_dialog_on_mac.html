FAQ-735 Wenn ich Origin auf einem Mac ausführe, warum kann ich den Lizenztext nicht von der Webseite in den Dialog von Origin einfügen?

Paste-License-to-Origin-Dialog-on-Mac

Letztes Update: 16.06.2015

Beachten Sie, dass die Tastenkombinationen zum Kopieren und Einfügen sich bei einem Mac (cmd + c und cmd + v) von denen bei einem Windows-Computer (Strg + C und Strg + V) unterscheiden. Wenn Sie den Lizenztext von einem Browser kopieren, der auf einem Mac-Betriebssystem läuft, indem Sie die Tasten cmd + c verwenden, sollten Sie die Tasten Strg + V verwenden, um den Text in den Origin-Dialog einzufügen.
