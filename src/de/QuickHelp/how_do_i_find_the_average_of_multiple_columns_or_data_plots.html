FAQ-494 Wie kann ich den Mittelwert mehrerer Spalten oder Datenzeichnungen finden?

Average-MultipleCuves

Letztes Update: 07.08.2015

Wenn die X-Werte für die Datensätze unterschiedlich sind, ist es wahrscheinlich, dass der Anfangswert von X (Xmin) und der Endwert von X (Xmax) jeder Kurve unterschiedlich sein wird. Sie möchten diese Kurven vielleicht in einem bestimmten Bereich mitteln, wie einen allgemeinen Bereich für die Kurven oder einen vollen Bereich von dem kleinsten Xmin bis zum größten Xmax aller Kurven.

Dies können Sie mit Hilfe von Analyse: Mathematik: Mehrere Kurven mitteln tun. Wählen Sie den gewünschten X-Bereich in der Auswahlliste Gemitteltes X aus.
