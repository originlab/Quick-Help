FAQ-393 Wie gebe ich Parameter für Origin C-Programme (LabTalk-Skript) über einen Dialog ein?

Dialog-input-parameter

Letztes Update: 04.02.2015


Manchmal müssen Sie beim Ausführen numerischer Berechnungen mit Ihrem eigenen Code einige Parameter verwenden. Befehle wie cin in C++ oder scanf in C werden jedoch nicht in Origin C unterstützt. In diesem Fall können Sie InputBox oder GetN dialog verwenden, in welche die Parameter mit einer einfachen oder komplexen Struktur eingegeben werden können.

Außerdem wird die Verwendung des Befehls GetNumber empfohlen, wenn Sie LabTalk-Skript verwenden möchten, um eine Zahlenfeld zu erzeugen.

 

Schlüsselwörter:InputBox, GetN, GetNumber, input, cin, scanf
