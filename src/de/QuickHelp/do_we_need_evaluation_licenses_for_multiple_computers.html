FAQ-3 Sind Demolizenzen für mehrere Computer erforderlich?

Share-DemoLic-for-All-PCs

Letztes Update: 04.02.2015

Sie können das gleiche Installationsprogramm verwenden, um die Demoversion von Origin auf jedem Computer zu installieren. Sobald Sie Origin auf einem Computer starten und den Lizenzierungsdialog nutzen, um eine Demolizenz anzufordern, können Sie diese Demolizenz auch auf allen anderen Computern verwenden. Wählen Sie auf den anderen Computer die Option "Ich habe bereits eine Demolizenz" in dem Dialog , der sich beim Start öffnet. Stellen Sie sicher, dass die E-Mail-Adresse eingegeben wird, an die die ursprüngliche Demolizenz gesendet wurde.

Schlüsselwörter: Demoversion
