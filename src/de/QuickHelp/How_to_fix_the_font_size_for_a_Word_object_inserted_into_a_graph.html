FAQ-1018 Wie korrigiere ich die Schriftgröße für ein Word-Objekt, das in eine Grafik eingefügt ist?

Fix-FontSize-for-Word-Obj

Letztes Update: 08.09.2019

Wenn Sie ein Word-Objekt in eine Grafik eingefügt haben und die Schriftgröße, die in Word-OLE festgelegt ist, beibehalten möchten, dann können Sie mit der rechten Maustaste auf das Word-Objekt klicken, um Eigenschaften auszuwählen. Der Dialog Dokumentationseigenschaften wird geöffnet. Gehen Sie zur Registerkarte Ansicht, um das Kontrollkästchen Relativ zur ursprünglichen Größe zu aktivieren.

Schlüsselwörter:Grafik exportieren, Word-Objekt, Schriftgröße, Word einfügen
