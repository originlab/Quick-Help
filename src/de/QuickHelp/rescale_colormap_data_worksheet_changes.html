FAQ-896 Wie skaliere ich die Farbabbildung neu, wenn die Daten im Arbeitsblatt sich ändern?

Rescale-Colormap-Data-Worksheet-Changes

Letztes Update: 12.01.2018

Wenn Daten für ein Konturdiagramm oder ein 3D-Oberflächendiagramm mit Farbabbildung sich ändern, kann man die Farbabbildung mit den folgenden Methoden einfach neu skalieren:

  1. Klicken Sie mit der rechten Maustaste auf das Diagramm und wählen Sie Farbabbildung: Ebenen festlegen, um den Dialog Ebenen festlegen zu öffnen. Klicken Sie auf Min./Max. suchen, um die Werte für Von und Bis auf Grundlage Ihrer Daten automatisch festzulegen. Klicken Sie auf OK, um die Änderung zu übernehmen.
  2. Klicken Sie doppelt auf die Zeichnung und gehen Sie zum Dialog Details Zeichnung. Ändern Sie auf der Registerkarte Farbabbildung den Modus Neu skalieren auf Auto. Wenn Sie die Daten jetzt ändern, wird die Farbabbildung automatisch neu skaliert.
  3. Sie können eine der folgenden Labtalk-Skripte testen: 
    • layer -axyz // Rescale colormap only.

    • layer -axy // Rescale both z axi and colormap.
    Weitere Einzelheiten zum Befehl zur Neuskalierung des Layers finden Sie auf dieser Seite.
