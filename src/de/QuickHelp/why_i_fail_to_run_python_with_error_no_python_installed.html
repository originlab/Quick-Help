FAQ-842 Warum lässt sich Python nicht ausführen und es wird die Fehlermeldung "Python nicht installiert" angezeigt?

Why_I_fail_to_run_Python_with_error_no_Python_installed

Letztes Update: 26.07.2018


In Origin 2017 hängt die Python-Konsole von einigen VC++ 2015 DLLs ab. Im Allgemeinen sind diese DLLs auf Ihrem System installiert und es wird ein Upgrade für sie auf die neue Version durchgeführt, wenn Sie Origin 2017 installieren.

Es kann passieren, dass eine andere Anwendung, die auch von diesen DLLs abhängig ist, bereits auf Ihrem System existiert, bevor Sie versuchen, Origin 2017 zu installieren. Sollte dies so sein, kann es vorkommen, dass das Upgrade dieser DLLs auf die neue Version aufgrund eines Konflikts oder einer Genehmigung fehlschlägt. Aus diesem Grund wird Ihnen die Fehlermeldung "Python nicht installiert" angezeigt, wenn Sie Python in Origin 2017 ausführen.

Folgende Schritte könnten helfen, das Problem zu beheben:

  1. Öffnen Sie die Windows Systemsteuerung und gehen Sie zu Programme und Funktionen.
  2. Scrollen Sie zu den Programmen Microsoft Visual C++ 2015 Redistributable(x64)-14.0.023026 und Microsoft Visual C++ 2015 Redistributable(x86)-14.0.023026.
  3. Klicken Sie doppelt auf die Programm, um die Dialoge zu öffnen.
  4. Klicken Sie auf die Schaltfläche Reparieren.
  5. Starten Sie Origin neu.

 

Origin-Version mind. erforderlich: Origin 2017

Schlüsselwörter:Python, Fehler, Python nicht installiert, reparieren
