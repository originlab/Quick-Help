FAQ-913 Wie wende ich ein Format auf mehrere Arbeitsblätter an?

Format-Multi-Worksheets

Letztes Update: 03.01.2018

Anwender können Arbeitsblattformate als Designdateien speichern und auf andere Arbeitsblätter anwenden.

Standardmäßig ist die Option Arbeitsblattformat als Design speichern in Origin deaktiviert. Um sie zu aktivieren, wählen Sie zuerst im Menü Hilfsmittel: Systemvariablen, klicken doppelt auf eine leere Zelle unter der Spalte Variable und geben dann WEF ein. Setzen Sie den Wert auf 1.

Passen Sie ein Arbeitsblattformat benutzerdefiniert an, ändern Sie z. B. die Zeilenhöhe, Spaltenbreite etc. Klicken Sie mit der rechten Maustaste auf die leere Fläche des Arbeitsblatts und wählen Sie im Kontextmenü Format bearbeiten. Klicken Sie im Popup-Dialog auf Arbeitsblatt Design Editor auf die Schaltfläche Speichern unter, um das Design zu speichern.

Um das gespeicherte Design auf mehrere Arbeitsblätter anzuwenden, z. B. alle Arbeitsblätter im aktuellen Ordner, wählen Sie im Menü Hilfsmittel: Designs verwalten, wechseln Sie zur Registerkarte Arbeitsblatt und wählen Sie das soeben gespeicherte Design aus. Wählen Sie in der Auswahlliste Design anwenden auf die Option Blätter im Ordner. Klicken Sie auf Jetzt anwenden, um das Format aus alle Arbeitsblätter im Ordner anzuwenden.
