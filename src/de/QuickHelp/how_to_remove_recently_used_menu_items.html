FAQ-873 Wie entferne ich zuletzt verwendete Menüelemente?

Remove-MRU

Letztes Update: 02.05.2019

Wenn Sie kürzlich ein Menüelement verwendet haben, wird dieses Element unten in der Menüliste aufgeführt. Es können bis zu 10 Elemente aufgelistet werden. Wenn ein Analysemenü verwendet wird, wird weiterhin ein "zuletzt verwendetes" Design als Unterelement unter diesem Menü geladen. Dies macht Ihnen den Zugriff auf die zuletzt verwendeten Einstellungen einfacher. Wenn Sie ein Analysedesign speichern oder installieren, wird es auch als Unterelement geladen.

Seit Origin 2017 SR1 haben wir die Systemvariable @MRU, um zu steuern, ob diese zuletzt verwendeten Elemente gezeigt und Designelemente in die Menüliste geladen werden. Sie besteht aus 3 Teilen:

1 = Zuletzt verwendete Elemente für das Menü Zeichnen aktivieren
2 = Zuletzt verwendete Elemente für das Menü Analyse aktivieren
4 = Laden des Designs in das Menü aktivieren

@MRU ist eine Kombination der steuernden Teile. Der Standardwert ist 7. Mit ihm werden beide "Zuletzt verwendet"-Menüs aktiviert (@MRU=1 für das Menü Zeichnen und @MRU=2 für das Menü Analyse) sowie die Elemente des Menüs Design geladen (@MRU=4).

Das Problem

Auf einigen Systemen kann die Ansammlung von zuletzt verwendeten Menüelementen eine leichte Verlangsamung zur Folge haben, wenn der Anwender versucht, ein Menü zu öffnen (Analyse und Statistik scheinen am anfälligsten zu sein). Wenn dies ein Problem ist, können Sie die zuletzt verwendeten Menüelemente global deaktivieren.

Um zuletzt verwendete Menüelemente global zu deaktivieren:

  1. Sie können die zuletzt verwendeten Menüelemente temporär deaktivieren, indem Sie das Skriptfenster öffnen (Fenster: Skriptfenster), das Folgende in der Befehlszeile eingeben und dann Enter drücken:
    @mru = 0
  2. Beachten Sie, dass dies temporär ist (nur diese Sitzung) und beim Neustart von Origin die zuletzt verwendeten Menüelemente wieder angezeigt werden.
  3. Sie können diese Funktion dauerhaft deaktivieren, indem Sie @mru = 0 im Dialog Systemvariablen festlegen (Einstellungen: Systemvariablen) eingeben.
  4. Um die zuletzt verwendeten Menüelemente wieder einzuschalten, löschen Sie @mru aus Ihrem Dialog Systemvariablen festlegen. Alternativ legen Sie Bits für Menüs fest, bei denen Sie die zuletzt verwendeten Elemente anzeigen möchten. Dazu verwenden Sie die Bit-Werte oben.
