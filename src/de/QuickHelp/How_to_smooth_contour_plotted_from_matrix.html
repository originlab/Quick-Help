FAQ-1188 Wie glätte eine Kontur, die auf Grundlage einer Matrix gezeichnet wurde?

Matrix-Contour-Smoothing

Letztes Update: 01.08.2023

Aktuell bietet nur das Konturdiagramm, das aus XYZ-Arbeitsblattdaten gezeichnet wird, Glättungsoptionen (Bedienelemente Glättung im Dialog Details Zeichnung).

Wenn Sie eine Kontur aus einer Matrix oder virtuellen Matrix erstellen wollen und glattere Konturlinien/Farbabbildungen des Gradienten erhalten möchten, müssen Sie:

  1. Ihre Matrix-/virtuelle Matrixdaten in XYZ-Spalten konvertieren:
    • Für Matrixdaten: Menü Matrix: In Arbeitsblatt konvertieren > Methode = XYZ-Spalten
    • Für virtuelle Matrix: Menü Arbeitsblatt: In XYZ konvertieren
  2. Erstellen Sie ein neues Konturdiagramm aus XYZ-Spalten.
  3. Sie können dann die Optionen unter Glättung im Dialog Details Zeichnung verwenden, um die neue Kontur zu glätten: Dialog Details Zeichnung > Registerkarte Konturinfo > Gruppe Glättung.

Schlüsselwörter:Kontur, Matrix, virtuelle Matrix, Glättung, Gradientenfarbe ändern, Konturlinie glätten
