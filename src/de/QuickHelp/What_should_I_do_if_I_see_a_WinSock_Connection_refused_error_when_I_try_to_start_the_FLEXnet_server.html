FAQ-1187 Was sollte ich tun, wenn ich beim Startversuch des FLEXnet-Servers die Fehlermeldung "WinSock: Connection refused" (Winsock: Verbindung verweigert) angezeigt bekomme?

WinSock-Connection-Refused-Server-Error

Letztes Update: 26.07.2023

Wenn Sie Ihren FLEXnet-Lizenzdienst starten möchten, dies aber fehlschlägt, und Ihnen folgende Fehlermeldung auf der Registerkarte LMTools > Server Status angezeigt wird:

System Error: 10061 “WinSock: Connection refused”

Dies wird durch den Server ohne Schreibzugriff auf das Directory der Protokolldatei verursacht. Normalerweise passiert es, wenn Sie versuchen, den Dienst von einem Windows-Account ohne Administratorrechte auszuführen.


Um dieses Problem zu beheben:

Sie müssen dem Server die notwendigen Zugriffsrechte einräumen. Weitere Einzelheiten finden Sie auf dieser Dokumentseite für Flexera.

Schlüsselwörter:Windows 10, Win10, Concurrent, Server, Floating-Lizenzserver
