FAQ-300 Wie führe ich eine Peakentfaltung durch?

peak-deconvolution

Letztes Update: 04.02.2015

Wenn die Intensität eine lineare Kombination von einzelnen Komponenten ist, können Sie die Kurvenanpassung anwenden, um die Peakzerlegung durchzuführen. Origin bietet zwei Hilfsmittel zum Anpassen von mehreren Peaks, von denen einige sich überschneidende Peaks sein können.

  • Wenn es keine Basislinie gibt, kann das Hilfsmittel Mehrere Peaks anpassen verwendet werden. Auf dieses Hilfsmittel wird über das Menü Analyse: Peaks und Basislinie zugegriffen. Es ermöglicht Ihnen, mit einem Klick die Position eines Peaks festzulegen und die Anpassung dieses Peaks basierend auf dem festgelegten Peakzentrum durchzuführen. Dieses Hilfsmittel verfügt auch über eine Option, um Origins Dialog NLFit für die erweiterte Anpassungssteuerung zu öffnen.
  • Wenn es eine Basislinie gibt, kann der Dialog Peaks analysieren verwendet werden. Öffnen Sie dieses Hilfsmittel, indem Sie auf Analyse: Peaks und Basislinie klicken. Legen Sie Ihr Ziel mit Peaks anpassen (Pro) fest (verfügbar in OriginPro). Dieses Hilfsmittel bietet mehrere Methoden für die Suche nach Basislinie und Peaks bei voller Kontrolle des Anpassungsprozesses. Um eine unterschiedliche Peakfunktion für jeden Peak anzupassen, klicken Sie auf die Schaltfläche Fit-Steuerung unten auf der letzten Seite des Assistenten.
