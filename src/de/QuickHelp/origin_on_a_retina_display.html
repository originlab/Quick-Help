FAQ-346 Wie behebe ich Anzeigeprobleme von Origin auf Bildschirmen mit hoher Auflösung?

Mac-Retina-Display-of-Origin

Letztes Update: 07.04.2022

Immer mehr Displays verfügen über viel höhere Bildschirmauflösungen als es für PC-Monitore üblich ist (bis zu 227 ppi vs. 96 ppi). Obwohl Origin bei einer nativen Auflösung ausgeführt werden kann, haben Tests gezeigt, dass die Verwendung einer Auflösung von 1920 X 1200 mit einer mittleren Skalierung (125%) für den Anwender beim Ausführen von Origin die am besten funktioniert.

Wenn das Display eine höhere Bildschirmauflösung hat, zum Beispiel ein QHD-Display mit einer Auflösung von 3200 X 1800, werden Schaltflächen und Text in Menüs bzw. Dialogen in Origin extrem klein angezeigt. Bitte versuchen Sie die folgenden Lösungen, um das Problem zu beheben:

Seit Origin 2022b wird die Fenstergröße und -position beim Start entsprechend der Bildschirmauflösung und DPI angepasst, so dass es einfacher ist, Projekte zwischen Laptop und hochauflösenden Desktopmonitoren zu teilen. Die Funktionalität wird mit der Systemvariable @SRWS gesteuert/eingestellt.

Lösung 1

  1. Wählen Sie Ansicht: Symbolleisten, um den Dialog zum Anpassen der Symbolleisten zu öffnen.
  2. Aktivieren Sie die Registerkarte Optionen. Aktivieren Sie die Kontrollkästchen Große Symbole verwenden und Skalierungssymbolleisten für größere Systemschrifteinstellungen.
Hinweis: Das Kontrollkästchen Skalierungssymbolleisten für größere Systemschrifteinstellungen kann die Funktionalität der geteilten Symbolleistenschaltflächen beeinträchtigen (Schaltflächen, hinter denen mehrere Hilfsmittel aufgelistet sind, wobei das zuletzt verwendete angezeigt wird).

Wenn Lösugn 1 keine Verbesserung bewirkt, versuchen Sie Lösung 2.

Lösung 2

Passen Sie die Computerauflösung und die DPI-Skalierung, wie folgt, an:

  • Verringern Sie die Computerauflösung auf 1900 X 1200.
  • Erhöhen Sie die DPI-Einstellungen, um ein großes Symbol zu haben.
Hinweis: Nach Ändern der Bildschirmauflösung und/oder der Systemskalierung sollten Sie sich bei Windows abmelden und wieder anmelden, um sicherzustellen, dass die neuen Einstellungen ordnungsgemäß zur Anwendung kommen.

Origin's Anzeigeproblem von Apples MacBook-Retina-Display

Apples MacBook-Retina-Display kann eine hohe Auflösung haben und würde daher Anzeigeprobleme in Origin verursachen. Versuchen Sie die obenstehenden Lösungen, um den Fehler zu beheben. Die besten Bildschirmeinstellungen zum Ausführen von Origin auf MAC sind ebenfalls eine Auflösung von 1920 X 1200 it einer mittleren Systemskalierung (125%).

Getestete Konfiguration:
Hardware: MacBook Pro, 2.0GHz Quad-Core Intel i7, 15.4" Retina-Display (2880 X 1800, 220 ppi).
Software: OS X Version 10.9.5, BootCamp 5.1, Windows 8.1, OriginPro 2015

Weitere Informationen zum Ausführen von Origin auf einem Mac finden Sie auf dieser Webseite.

Hinweis: Die letzten Versionen von Parallels und VMware Fusion unterstützen das Ausführen von Windows mit einem Retina-Display. Bitte lesen Sie in der Dokumentation Ihrer Software weitere Informationen.
