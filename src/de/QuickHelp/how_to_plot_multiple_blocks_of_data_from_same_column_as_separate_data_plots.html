FAQ-718 Wie werden mehrere Datenblöcke aus der gleichen Spalte als separate Datenzeichnungen gezeichnet?

plot-multiple-blocks-data

Letztes Update: 10.08.2015

Hierfür gibt es einige Möglichkeiten, abhängig von der Größe der Datenblöcke. Wenn Sie nur ein paar wenige kleine Datenblöcke auszuwählen haben, können Sie die STRG-Taste gedrückt halten und mehrere Bereichsauswahlen innerhalb einer einzelnen Spalte vornehmen.

Zum Zeichnen von mehreren Datenblöcken in der gleichen Spalte, getrennt von leeren Zeilen, wie unten zu sehen:

Sie können im Menü Worksheet: Daten aus Arbeitsblatt extrahieren wählen, um den Dialog Daten aus Arbeitsblatt extrahieren aufzurufen, und eine einfache Befehlszeile in dem Bearbeitungsfeld Bedingung eingeben, so dass Daten in allen Blöcken in derselben Spalten gleichzeitig ausgewählt werden:

Wenn die gewünschten Daten ausgewählt werden, können Sie sie direkt zeichnen, zum Beispiel um ein Boxdiagramm zu erstellen:

Sie können ausgewählte Daten auch maskieren und extrahieren oder über den Dialog Daten aus Arbeitsblatt extrahieren auf erweiterte Funktionen zugreifen.

Schlüsselwörter:extrahieren, maskieren, auswählen, mehrere Mappen, mehrere Teilbereiche
