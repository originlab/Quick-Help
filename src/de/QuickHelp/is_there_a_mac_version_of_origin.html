FAQ-12 Gibt es eine Mac-Version von Origin?

Mac-Origin

Letztes Update: 04.02.2015

Sie können Origin und OriginPro auf Macs installieren und ausführen, indem Sie eine Virtualisierungssoftware (wie Parallel), Boot Camp, CrossOver Mac oder WineBottler verwenden. Besuchen Sie diese Webseite, wenn Sie Einzelheiten nachlesen möchten.

Schlüsselwörter:Mac, installieren, Virtualisierung, Boot Camp
