FAQ-702 Wie kann ich ein Diagramm erstellen, das mehrere 3D-XYZ-Punktdiagramme in dem gleichen Layer enthält?

multiple-3D-XYZ

Letztes Update: 18.02.2019

Sie können die Datensätze im Arbeitsblatt mit der Zuordnung XYZXYZ... oder XYZZ... versehen, dann das Arbeitsblatt markieren und Zeichnen: 3D: 3D-Punktdiagramm im Menü wählen, um mehrere 3D-XYZ-Punktdiagramme im gleichen Layer zu zeichnen.

 

Schlüsselwörter:mehrere, 3D, Punkt, Layer
