FAQ-205 Was ist der Unterschied zwischen einem 2D- und 3D-Wasserfalldiagramm?

 

Letztes Update: 18.02.2019

2D-/3D-Wasserfalldiagramm

Das 2D-Wasserfalldiagramm wird über das Menü Zeichnen > 3D: Wasserfall erstellt. Das 3D-Wasserfalldiagramm wird über das Menü Zeichnen > 3D: 3D-Wasserfall erstellt. Beide Diagrammarten unterstützen die Y- bzw. Z-Farbabbildung.

Das 2D-Wasserfalldiagramm ist ein Pseudo-3D-Diagramm. Die meisten der Anpassungsoptionen ähneln denen eines 2D-Diagramms. Sie können die Projektion nicht ändern, und das Diagramm kann nicht gedreht werden. Sie können in 2D-Wasserfalldiagramme keine Achsenunterbrechungen festlegen.

Das 3D-Wasserfalldiagramm ist ein OpenGL-basiertes 3D-Diagramm. Die meisten Optionen der benutzerdefinierten Anpassung entsprechen denen von 3D-OpenGL-Diagrammen. Sie können gedreht oder gestreckt werden. Es werden zwei Projektionsmodi unterstützt, Perspektive und Orthographisch. Sie können die Projektion auf der Registerkarte Allgemeines des Dialogs Details Zeichnung festlegen. Sie können in 3D-Wasserfalldiagramme auch Achsenunterbrechungen festlegen.
