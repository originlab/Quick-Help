FAQ-317 Wie kann ich Daten oberhalb eines Box-/Balkendiagramms + Daten zeichnen?

Draw-Data-on-Top-of-Box

Letztes Update: 03.01.2020


Bei den Boxdiagrammvarianten Box + Datenüberschneidung und Balken + Datenüberschneidung können Sie die Daten oberhalb der Box bzw. des Balkens zeichnen:

  1. Klicken Sie zum Öffnen des Dialogs Details Zeichnung doppelt auf das Diagramm.
  2. Klicken Sie auf die Registerkarte Box und aktivieren Sie das Kontrollkästchen Daten oberhalb der Box/des Balkens zeichnen.

 

  • Weitere Informationen finden Sie unter Zeichnungsoptionen für Daten und Whisker.
  • Weitere Informationen zum Erstellen von Boxdiagrammen und Varianten (Balken, Violin, Säulen-Punkt etc.) finden Sie unter Boxdiagramme erstellen.

 

Schlüsselwörter:Boxdiagramm, Balkendiagramm, Daten oberhalb, Box + Datenüberschneidung, Balken + Datenüberschneidung
