FAQ-32 Warum wird beim Start von Origin eine Ablaufmeldung angezeigt?

Expiration-Message

Letztes Update: 12.12.2021

Wenn Sie ein ablaufendes oder verlängerbares Origin-Paket gekauft haben,

wird Ihnen die Ablaufnachricht eventuell deshalb angezeigt, weil sich das Ablauf- bzw. Verlängerungsdatum nähert. Bitte wenden Sie sich wegen der Verlängerung an den Support von OriginLab.

Sollten Sie ein dauerhaftes Origin-Paket gelauft haben,

stellen Sie sicher, dass Sie die Dauerlizenz (der Produktschlüssel ist mit "primary" gekennzeichnet) oder die ablaufende Lizenz (der Produktschlüssel ist mit "home" gekennzeichnet) verwenden. Letztere ist einer der Vorteile der jährlichen Softwarepflege. Sie wird mit der Softwarepflege ablaufen. Sollte dies der Fall sein, kontaktieren Sie bitte den Support von OriginLab, um Ihre Softwarepflege zu verlängern.

 

Hinweis:

Eine Ablaufmeldung kann beim Start von Origin auch angezeigt werden, wenn Origin nicht richtig lizenziert ist. Bitte lesen Sie diese Seite.
