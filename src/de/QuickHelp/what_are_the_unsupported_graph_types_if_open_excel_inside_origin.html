FAQ-715 Welche Diagrammtypen werden nicht unterstützt, wenn eine Excel-Arbeitsmappe in Origin zum Zeichnen geöffnet wird?

Open-Excel-UnsupGraph

Letztes Update: 09.04.2021

Origins Excel-Datenkonnektor erstellt eine Live-Verbindung zwischen Ihrem Origin-Projekt und einer externen Excel-Datei. Dadurch wird es einfacher, mit Excel-Dateien zu arbeiten, ohne sie in Origin zu öffnen. Wenn Sie daran gewöhnt sind, Ihre Excel-Dateien in Origin zu öffnen, und Sie ein Upgrade auf Origin 2021b oder höher vornehmen, lesen Sie bitte diesen Hinweis.

Wenn Sie eine Excel-Datei im Origin-Arbeitsbereich "öffnen", ist Ihr Zugriff auf die grafischen Darstellungsmittel von Origin eingeschränkt. Wenn Sie Zeichnungen mit dem Dialog Daten zum Zeichnen auswählen erstellen, können Sie mehrere Spalten als Y zuweisen (Y kennzeichnet die Spaltenzuweisung), aber Sie können nur eine Spalte als X oder Z auswählen.

Jeder Diagrammtyp in Origin erfordert eine bestimmte Kombination der Spaltenzuweisungen. Einige Diagrammtypen erfordern zusätzliche Beschriftungszeilen oder Fehlerspalten für die Eingabe. Andere müssen auf Grundlage einer Matrix, die in einer Origin-Matrixmappe gespeichert ist, oder einer virtuellen Matrix, die in einem Origin-Arbeitsblatt gespeichert ist, gezeichnet werden. Eine Zusammenfassung der Datenanforderungen nach Diagrammtyp finden Sie unter Origin-Diagrammtypen.

Die Diagrammtypen Vektor XYWG und Vektor XYXY können gezeichnet werden, wenn Sie einmal auf die Schaltfläche Standardwert verwenden im Dialog Daten zum Zeichnen auswählen klicken, um die Standardzuweisung zu verwenden.
