FAQ-272 Wie kann ich wiederkehrende Aufgaben automatisieren?

Automate-Repetitive-tasks

Letztes Update: 09.08.2018

  • Mit Origin können Anwender eine Stapelverarbeitung mehrerer Dateien oder Datensätze in Spalten mit Hilfe einer Analysevorlage durchführen.
    • Bitte lesen Sie folgende Tutorials: Stapelverarbeitung
    • oder sehen Sie sich dieses Video-Tutorial an: Stapelverarbeitung
  • Bei einigen der Analysehilfsmittel wie dem Anpassen können Sie eine Operation kopieren und einfügen, um die gleiche Analyse/Operation für andere ausgewählte Kurven zu wiederholen.
  • Entsprechend können die Operation/Analyse der aktuellen Y-Spalte/Zeichnung für alle Y-Spalten oder alle Zeichnungen wiederholen.
  • Zum Durchführen einer Peakanalyse per Stapelverarbeitung können Sie die Einstellungen Ihrer Peakanalyse im Hilfsmittel Peaks analysieren nach eigenen Wünschen festlegen und sie als Design speichern. Laden Sie es dann im Dialog Peakanalyse durch Stapelverarbeitung mit Design, um eine Stapelverarbeitung von mehreren Datensätzen durchzuführen.
  • Mit Minitools können Sie für mehrere Kurven in einem Diagramm eine Stapelanalyse durchführen. Nachdem Sie den gewünschten Bereich ausgewählt und die Einstellungen im Minitool an Ihre Bedürfnisse angepasst haben, wählen Sie Neue Ausgabe für alle Kurven oder Neue Ausgabe für alle Layer, um die Analyse für alle Kurven durchzuführen und die Ergebnisse auszugeben.
