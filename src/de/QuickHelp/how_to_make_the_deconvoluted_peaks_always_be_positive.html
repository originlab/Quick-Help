FAQ-964 Warum gibt es negative Peaks, wenn ich Peaks entfalte, und wie mache ich sie immer positiv?

deconvoluted-peaks-be-positive

Letztes Update: 29.06.2018

Manchmal, wenn Sie Peaks in Origin entfalten, kann es vorkommen, dass einige entfaltete Peaks, von denen erwartet wird, dass sie positiv sind, sich als negativ herausstellen.

Richtung der Peaks festlegen, um sie während der Entfaltung positiv zu belassen

  • Wenn die Basislinie nicht gut definiert ist, taucht sie womöglich einige der Gipfel unter, so dass die Tiefs als "negative Peaks" erkannt werden. In diesem Fall überprüfen Sie die Basislinie und passen Sie sie an, um negative Peaks zu vermeiden.
  • Wenn Sie das Hilfsmittel Peaks analysieren verwenden, um die Peakentfaltung durchzuführen, können Sie die Peakrichtung folgendermaßen beschränken:
  1. Der Dialog Peak-Fit Parameter kann geöffnet werden, indem auf die Schaltfläche Fit-Steuerung auf der Seite Peaks anpassen des Dialogs Peaks analysieren geklickt wird.
  2. Legen Sie die Parameter Fläche oder Amplitude auf 0 oder größer fest.
