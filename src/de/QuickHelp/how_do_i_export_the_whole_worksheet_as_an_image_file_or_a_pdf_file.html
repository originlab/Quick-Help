FAQ-456 Wie exportiere ich das gesamte Arbeitsblatt als Bilddatei oder PDF-Datei?

ExpWks-as-img-PDF

Letztes Update: 31.07.2018

Zum Exportieren als Bilddatei:

Um das gesamte Arbeitsblatt als Bilddatei zu exportieren, wählen Sie bei aktivem Arbeitsblatt den Menübefehl Datei: Export: Als Bilddatei.

Der Dialog der X-Funktion expWks wird aufgerufen. Dieser Dialog ist identisch mit dem Dialogfeld zum Exportieren einer Grafik mit dem zusätzlichen Bedienelement Exportbereich.

Wählen Sie in diesem Dialog den Dateityp für die exportierte Datei aus der Auswahlliste Bildtyp.

Erweitern Sie anschließend den Punkt Exporteinstellungen und wählen Sie Gesamt in der Auswahlliste Exportbereich. (Diese Option unterstützt auch das Exportieren des ausschließlich sichtbaren Bereichs eines Arbeitsblatts oder nur des Bereichs, der durch Ihre Auswahl festgelegt ist.)

Beachten Sie, dass Sie eine gewünschte Breite für die Ausgabe festlegen können und die Höhe automatisch festgelegt wird, um die Seitenverhältnisse beizubehalten.

Zellengruppen im Arbeitsblatt können beliebig zusammengefügt werden, um Diagramme, Bilder, Berichtstabellen und Text zu platzieren. Dies erlaubt Ihnen das Erstellen benutzerdefinierter Berichtsblätter.

Diese Möglichkeit zum Erstellen benutzerdefinierter Berichte in Kombination mit der Möglichkeit, die Breite eines zu exportierenden Arbeitsblatts beliebig festzulegen, ermöglicht Ihnen die Erstellung von Präsentationen mit Bildern in Postergröße.

Zum Exportieren als mehrseitige PDF-Datei:

Um das gesamte Arbeitsblatt als PDF-Datei zu exportieren, wählen Sie bei aktivem Arbeitsblatt Datei: Export: Als mehrseitige PDF-Datei im Origin-Menü.

Der Dialog der X-Funktion expPDFw wird aufgerufen.

Der Zweig Bildeinstellungen: Mehrere Seiten enthält Optionen zum Hinzufügen eines Deckblatts und einer Seitennummerierung.

Wenn Sie das Kontrollkästchen Eigene Einstellungen des Worksheets verwenden aktiviert ist, entspricht die Ausgabe derjenigen in der Ansicht unter Datei: Seitenansicht. Diese Ansicht kann im Menü über Datei: Drucken gesteuert werden. Klicken Sie auf die Schaltfläche Optionen, um Druckoptionen zu bearbeiten. Klicken Sie auf OK im Optionendialog und dann auf Abbrechen im Dialog Drucken.

Wenn Sie das Kontrollkästchen Eigene Einstellungen des Worksheets verwenden deaktivieren, wählen Sie anschließend, was Sie anzeigen möchten, mit den gleichen Bedienelementen wie denen unter den Druckoptionen.

 

Schlüsselwörter: Bereich, Bild, Poster, Präsentation, Format, groß
