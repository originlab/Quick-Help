FAQ-647 Wie identifiziere ich die Anpassungsergebnisse in der Berichtstabelle nach Datensatzidentifizierer?

Data-Identilier

Letztes Update: 21.07.2018

Datenidentifizierer unten wird verwendet, um die Informationen der Quelldatensätze in der Berichtstabelle festzulegen.

  • Bereich
    Die Bereichssyntax wird verwendet.
  • Mappenname
    Der Langname der Arbeitsmappe wird verwendet.
  • Blattname
    Der Arbeitsblattname wird verwendet.
  • Blattanzeigename
    Der angezeigte Arbeitsblattname wird verwendet.
  • Name
    Der Langname der entsprechenden Spalte, wenn es einen Langnamen gibt, wird verwendet. Ansonsten wird der Kurzname der Spalte verwendet.
  • Kurzname
    Der Kurzname der entsprechenden Spalte wird verwendet.
  • Langname
    Der Langname der entsprechenden Spalte wird verwendet.
  • Einheiten
    Die Einheit der entsprechenden Spalte wird verwendet.
  • Kommentare
    Der Kommentar der entsprechenden Spalte wird verwendet.
  • <Benutzerdefiniert>
    Zum Definieren eines Datenidentifizierers werden benutzerdefinierte Formate verwendet. Einzelheiten zu seiner Verwendung finden Sie unter Erweiterte Anpassungen des Legendentexts.
  • Identifizierer in Ergebnisblatt zeigen
    Viele Origin-Analyseoperationen geben Daten zusätzlich zu den minimierbaren Analyseberichtsblättern in einem Ergebnisblatt aus. Der Identifizierer wird in dem Ergebnisblatt verwendet.

Schlüsselwörter: Lineare Kurvenanpassung, Datenidentifizierer
