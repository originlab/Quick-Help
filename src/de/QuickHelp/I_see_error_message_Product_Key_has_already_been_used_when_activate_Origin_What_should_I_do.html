FAQ-1095 Wenn ich Origin aktiviere, sehe ich die Fehlermeldung "Produktschlüssel wurde bereits verwendet". Was soll ich tun?

Product-Key-Has-Already-Been-Used

Letztes Update: 09.04.2021

  • Wenn Sie versuchen, Origin zu aktivieren, und die folgende Fehlermeldung sehen, brauchen Sie eine Systemübertragung (Produktschlüssel vom alten Computer deaktivieren).
Produktschlüssel wurde bereits verwendet (Product Key has already been used)

Bitte lesen Sie weitere Anweisungen auf dieser Seite.

  • Wenn Sie versuchen, eine Lernausgabe zu aktivieren und folgende Fehlermeldung sehen:
Dieser Produktschlüssel wurde bereits auf dem Computer XXX aktiviert.
Bitte deaktivieren Sie ihn zuerst, damit Sie ihn auf diesem Computer aktivieren können.

Bitte senden Sie eine E-Mail an le@originlab.com, um eine Systemübertragung anzufragen. Bitte geben Sie Ihren Produktschlüssel und die E-Mail-Adresse an, die Sie zum Anwender der Lernausgabe verwendet haben.

Schlüsselwörter:Lizenz, LE, Lernausgabe, Lernversion, Fehler, kann nicht aktiviert werden, Produktschlüssel verwendet
