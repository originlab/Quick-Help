FAQ-757 Wie zeige ich die Farbskala als separate Farbbereiche an?

Color-Scale-Range

Letztes Update: 27.08.2015

Klicken Sie doppelt auf die Farbskala, um den Dialog Steuerung der Farbskala - Layer 1 aufzurufen, wobei die Option Layout im linken Bedienfeldoption markiert ist. Wählen Sie die Option Getrennt in der Auswahlliste Layout im rechten Bedienfeld. Die Anzahl der Farbbereiche wird bestimmt von der Spalte Ebenen auf der Registerkarte Farbpalette des Dialogs Details Zeichnung.

Origin-Version mind. erforderlich: 2016 SR0

Schlüsselwörter:Farbskala, Farbblöcke, Farbbereiche, getrennt, individuell
