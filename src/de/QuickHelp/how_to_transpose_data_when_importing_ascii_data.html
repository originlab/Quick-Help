FAQ-746 Wie kann ich beim Importieren von ASCII-Daten Daten transponieren?

Origin-Import-Transpose

Letztes Update: 24.06.2022

Origin unterstützt das Transponieren von ASCII-Daten beim Import.

Zum Starten:

  1. Erstellen Sie eine neue Arbeitsmappe (Matrix), indem Sie auf Datei: Neu: Arbeitsblatt (oder Datei: Neu: Matrix) klicken.
  2. Wählen Sie im Menü Daten: Aus Datei importieren: Einzelnes ASCII oder Daten: Aus Datei importieren: Mehrere ASCII. Wählen Sie dann eine .dat/.txt-Datei und fügen Sie sie zum unteren Bedienfeld hinzu. Stellen Sie sicher, dass das Kontrollkästchen Optionendialog zeigen aktiviert ist. Klicken Sie auf die Schaltfläche Öffnen.
  3. Aktivieren Sie in dem aufgerufenen Dialog im Zweig Dateistruktur das Kontrollkästchen Daten transponieren. Klicken Sie auf OK, um den Dialog zu schließen.

In den Versionen vor Origin 2019b befanden sich die Dialoge des ASCII-Imports unter Datei: Import: Einzelnes/Mehrere ASCII.

Vor 2019 wurden neue Arbeitsmappen über Datei: Neu: Arbeitsblatt erstellt.
Hinweis: Der Importassistent unterstützt diese Funktion noch nicht.

Schlüsselwörter:ASCII importieren, Datenimport, transponieren

Origin-Version mind. erforderlich: 2016 SR0
