FAQ-418 Wie öffne und bearbeite ich eine MySQL-Datenbank in Origin?

MySQL-in-Origin

Letztes Update: 04.02.2015


Origin verwendet ADO, um auf Datenbanken zuzugreifen. Wir haben auf unserer Webseite ein Tutorial zu dem Thema vorbereitet: Daten aus einer Datenbank importieren.

Es gibt einige standardmäßige OLE DB-Provider in Windows, die Sie direkt verwenden und mit Datenbanken wie Oracle und SQL-Server verbinden können. Für andere Datenbanken wie MySql jedoch können Sie den ODBC-Treiber im Hilfsmittel ADO verwenden, wenn Sie keinen Zugriff auf einen geeigneten OLE DB Provider haben. Um die Verbindung herzustellen:

  1. Zunächst brauchen Sie den MySQL Connector/ODBC auf dem Rechner, den Sie mit der Datenbank verbinden wollen. Sie bekommen ihn auf der Webseite von MySql.
  2. Im nächsten Schritt wird Ihre ODBC-Verbindung eingerichtet. Öffnen Sie die Systemsteuerung und den Ordner Verwaltung. (Benutzer von Vista müssen möglicherweise zur klassischen Ansicht wechseln, um den Eintrag Verwaltung zu sehen). Klicken Sie doppelt auf das Symbol "Datenquellen (ODBC)" und anschließend auf die Schaltfläche "Hinzufügen...". Geben Sie Ihrer Datenquelle einen Namen, den man sich gut merken kann, und testen Sie die Verbindung vor dem Speichern.
  3. Öffnen Sie das Hilfsmittel ADO, indem Sie Datei: Datenbankimport: Neu im Hauptmenü auswählen oder auf die Schaltfläche Anfragenbuilder oder SQL-Editor auf der Symbolleiste klicken. Erstellen Sie im Anfragenbuilder eine neue Datenquelle, indem Sie auf Abfrage: Datenquelle: Neu klicken. Wählen Sie auf der Registerkarte Provider Microsoft OLE DB Provider for ODBC Drivers und klicken Sie auf Weiter. Wählen Sie die ODBC-Datenquelle, die Sie gerade definiert haben, geben Sie Passwörter ein und wählen Sie einen Anfangskatalog aus. Klicken Sie auf OK. Im rechten Bereich des Anfragen-Builder werden Tabellen angezeigt. (Wählen Sie im SQL-Editor im Menü Datei: Neu, um den Dialog Datenverknüpfungseigenschaften, und nehmen Sie die gleichen Einstellungen vor. )
