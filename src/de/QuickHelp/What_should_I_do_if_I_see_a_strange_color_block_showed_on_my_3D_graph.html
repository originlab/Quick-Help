FAQ-1167 Was sollte ich tun, wenn ein seltsamer Farbblock auf meinem 3D-Diagramm gezeigt wird?

Strange-Color-Block-On-3D-Graph

Letztes Update: 29.05.2023

Wenn Sie mit 3D-OpenGL-Diagrammen spielen und einen seltsamen Farbblock auf dem Diagramm sehen, ähnlich wie im folgenden Bild, dann liegt das wahrscheinlich an einem Bug der AMD-Grafikkarte.

Das Problem wird in Origin 2023b behoben. Das heißt:

Setzen Sie die Systemvariable @OGLDP auf 1. Lesen Sie in dieser FAQ, wie der Wert einer Systemvariable geändert wird.

Hinweis: Diese Fehlerbehebung verändert den Rahmen des 2D-Rechtecks ein wenig vom Standard. Wenn Sie etwas Falsches bei dieser Änderungen feststellen, setzen Sie bitte @OGLDP = 0 (Standard), um sie auszuschalten.


Falls die obige Lösung nicht funktioniert, versuchen Sie bitte Folgendes:

  1. Wählen Sie im Menü Einstellungen: 3D-OpenGL-Einstellungen....
  2. Im Dialog 3D-OpenGL-Einstellungen deaktivieren Sie das Kontrollkästchen Anzeigeliste aktivieren im Zweig Einstellungen.
