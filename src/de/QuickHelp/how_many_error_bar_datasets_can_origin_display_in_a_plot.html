Y-Fehlerbalken zeichnen

Jede Spalte hat Diagrammzuordnungen wie X, Y, Z, Y-Fehler, Beschriftung etc. Wenn Sie eine Spalte als Y-Fehler setzen und dann die Y-Spalte auf der linken Seite und die Spalte YErr zusammen markieren, um sie zu zeichnen, wird die Spalte YErr als Y-Fehlerbalken gezeichnet. Sie können doppelt auf die Fehlerbalken im Diagramm klicken, um sie weiter im Dialog Details Zeichnung benutzerdefiniert anzupassen.

Unterschiedliche positive und negative Y-Fehlerbalken zeichnen

Wenn Sie verschiedene positive und negative Fehlerbalken haben, gibt es zwei Möglichkeiten:

  1. Dialog Diagrammeinstellungen verwenden
    • Stellen Sie sicher, dass nichts im Arbeitsblatt ausgewählt ist, und wählen Sie Ihren Diagrammtyp.
    • Klicken Sie im Dialog Diagrammeinstellungen mit der rechten Maustaste in das mittlere Bedienfeld, um die Spalten YErr+ und YErr- einzuschalten.
    • Legen Sie die Daten in X, Y, YErr+, YErr- zum Zeichnen fest.
  2. Zeichnen Sie zwei Y-Fehlerbalken und verbergen Sie dann einen Teil davon.
    • Setzen Sie Fehlerdatenspalten als Y-Fehler. Markieren Sie die Y-Daten und beide Fehlerdatenspalten, um sie zu zeichnen.
    • Klicken Sie zum Öffnen des Dialogs Details Zeichnung doppelt auf die Zeichnung.
    • Markieren Sie den entsprechenden Fehlerbalkeneintrag im linken Bedienfeld und deaktivieren Sie die Kontrollkästchen Minus oder Plus auf der Registerkarte Fehlerbalken.

X-Fehlerbalken zeichnen

Wenn eine Spalte auf X-Fehler gesetzt ist, wird sie wie beim Zeichnen des Y-Fehlerbalkens als X-Fehlerbalken gezeichnet.

Y-Fehlerbalken aus Datensatzstatistiken hinzufügen

Wenn es keine Fehlerdatenspalte gibt, können Sie Statistikfehler durch Berechnen einfacher Datensatzstatistiken hinzufügen. Wählen Sie bei aktivem Diagrammfenster Einfügen: Fehlerbalken.

Fehlerbalken als gefülltes Band zeichnen

Manchmal, wenn Daten sehr dicht sind, ist es schwierig, alle Fehlerbalken zu sehen. Vielleicht möchten Sie Fehlerbalken als ein gefülltes Band zeigen. Im Folgenden sehen Sie, wie Sie das machen.

  1. Klicken Sie im Diagramm mit Fehlerbalken doppelt auf einen Fehlerbalken.
  2. Setzen Sie die Verbindung auf Gerade, Spline etc., wobei die Fehlerdaten auf der linken Seite ausgewählt sind.
  3. Aktivieren Sie das Kontrollkästchen Fläche unter Kurve füllen.
  4. Setzen Sie auf der Registerkarte Muster die Füllfarbe und das Füllmuster, wie Sie möchten.

Fehlerbalken in 3D-Diagramm zeichnen

Wenn Sie ein 3D-Punktdiagramm aus XYZ-Spalten zeichnen, können Sie den Dialog Details Zeichnung öffnen und eine andere Spalte im Arbeitsblatt als Fehlerbalken festlegen. Origin unterstützt Fehlerbalken in X-, Y- und Z-Richtung für 3D-Diagramme.

Wenn Sie ein 3D-Punktdiagramm aus einer Matrixmappe zeichnen, müssen sich die Fehlerbalkendaten in einem Matrixobjekt im gleichen Matrixblatt befinden. Klicken Sie z. B. auf das Symbol D in der oberen rechten Ecke des Matrixblatts und wählen Sie Hinzufügen. Geben Sie in das neu erstellte Matrixobjekt die Fehlerbalkendaten ein. Wählen Sie dann Zeichnen: 3D: 3D-Punktdiagramm + Fehlerbalken.


Während es unter bestimmten Bedingungen nützlich ist, Fehlerinformationen auf diese Weise anzuzeigen, sollten Sie sich bewusst machen, dass die Hilfsmittel Lineare Anpassung und Nichtlinearer Fit nur einen Datensatz für den Y-Fehler beim Anpassen verwenden und dass ein Fehler als Versatz eines einzelnen Werts betrachtet wird - sowohl positiv als auch negativ. Dies bedeutet, dass jegliche “X-Fehlerbalken” und “Y-Fehlerbalken Minus” ignoriert werden und ein “Y-Fehlerbalken Plus” genauso wie ein “Y-Fehlerbalken Plus und Minus” behandelt wird.
Um eine lineare Anpassung mit X-Fehler durchzuführen, sollten Sie Lineare Anpassung mit X-Fehler verwenden, während für eine nichtlineare Anpassung mit X-Fehler Nichtlineare implizite Kurvenanpassung verwendet werden sollte. Bitte lesen Sie hier weitere Einzelheiten.
