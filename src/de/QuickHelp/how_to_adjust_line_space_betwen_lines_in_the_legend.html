FAQ-719 Wie passe ich den Zeilenabstand zwischen den Zeilen in der Legende an?

adjust-line-space-for-legend

Letztes Update: 21.09.2022

Wenn Sie den Abstand zwischen den Zeilen in der Legende anpassen möchten, können Sie mit der rechten Maustaste auf die Legende klicken, um Einstellungen... im Kontextmenü auszuwählen. Der Dialog Textobjekt wird geöffnet. Auf der Registerkarte Text in diesem Dialog wählen Sie für das Element Zeilenabstand (%) einen Wert in der Auswahlliste oder geben Sie direkt einen Wert in das Kombinationsfeld ein.

Bitte beachten Sie: Wenn Sie den Zeilenabstand (%) auf <Auto> setzen, folgt der Abstand zwischen den Linien in der Legende der Einstellung Zeilenabstand auf der Registerkarte Zeichensätze im Dialog Optionen.

Seit Origin 2023 können Sie ungleiche Zeilenabstände (d. h. Gruppierungen) innerhalb des Legendenobjekts erstellen. Dazu verwenden Sie die hier beschriebenen Methoden.
