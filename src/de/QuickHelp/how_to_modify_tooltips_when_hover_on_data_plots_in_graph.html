FAQ-972 Kann ich die Tooltipps modifizieren, wenn ich die Maus über Datenzeichnungen in meiner Grafik bewege?

modify-tooltips-on-data-plots-in-graph

Letztes Update: 13.07.2021

Origin 2017 fügte einen Tooltipp der Datenzeichnung hinzu, der angezeigt wird, wenn Sie mit der Maus über eine Datenzeichnung im Diagrammfenster fahren. Dieser Tooltipp ist recht einfach gehalten und informiert über Quellmappe, Blatt und Koordinatendatensätze. Er ist nicht individuell anpassbar.

Ab Origin 2019 gibt es einen zweiten Tooltipp, der standardmäßig für viele 2D-Diagrammtypen, 3D-Punktdiagramme, XYY- und XYZ-Balkendiagramme angezeigt wird. Er wird als Tooltipp der Datenpunkte bezeichnet, um ihn vom Tooltipp der Datenzeichnung zu unterscheiden. Der Tooltipp der Datenpunkte kann im hohen Maß angepasst werden.

In Zeichnungen, die beide Typen unterstützen, können Sie zwischen den beiden wechseln, indem Sie mit der rechten Maustaste auf den Tooltipp klicken und im Kontextmenü Ihre Wahl treffen.

Seit Origin 2021 können Sie zwischen Tooltipptypen wechseln, indem Sie auf diese Schaltfläche in der Minisymbolleiste auf Seitenebene klicken.

Tooltipps aktivieren oder deaktivieren

Um die Tooltipps für Datenpunkt und Datenzeichnung ein- bzw. auszuschalten, ändern Sie die Werte der LabTalk-Systemvariablen @PT und @PTI.

@PT=0; //Disable both data plot and data point tooltips @PT=1; //Enable data plot tooltips for 2D graph @PT=3; //Enable data plot tooltips for both 2D graph and 3D OpenGL graph (default) @PTI=0; //Disable only data point tooltips @PTI=1; //Enable data point tooltips (default)
Transparenz der Tooltipps ändern

Sie können auch das Maß der Transparenz der Tooltipps mit Hilfe der Systemvariable @TDT ändern. Dies möchten Sie möglicherweise dann tun, wenn Sie z. B. Ihren Tooltipp der Datenpunkte so verändert haben, dass er Bilder anzeigt. @TDT ist per Standard auf 220 festgelegt. Verringern Sie den Wert, um die Transparenz zu erhöhen, oder setzen Sie ihn auf 255, um die Transparenz zu deaktivieren.
