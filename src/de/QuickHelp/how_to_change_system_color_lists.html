FAQ-967 Wie ändere ich Systemfarblisten?

Change-System-ColorList

Letztes Update: 23.07.2018

Nachdem Sie Ihre eigene Farbliste erzeugt haben, können Sie sie in Origin als Systemfarbliste einrichten. Dazu verwenden Sie das Hilfsmittel Designs verwalten (Hilfsmittel: Designs verwalten). Zukünftig wird dann per Standard diese neue Farbliste unabhängig von dem von Ihnen gezeichneten Diagrammtyp verwendet.

Um zu erfahren, wie Sie Ihre eigene Farbliste definieren, indem Sie Farben von jeder Stelle Ihres Computerbildschirms auswählen, und sie dann als Standardfarbliste festlegen, sehen Sie sich bitte dieses Video-Tutorial an.

Ausführliche Anweisungen zum Ändern der Systemfarbliste in Origin 2016 und älteren Versionen lesen Sie bitte in diesem Blog.
