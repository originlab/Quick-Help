FAQ-125 Kann ich für mein Ternärdiagramm die Hilfsstrichsbeschriftungen auf 0 bis 100 festlegen?

TernaryPlot-Labels

Letztes Update: 04.02.2015


Ja, wenn Ihre Daten bereits auf "0 bis 100" normiert sind bzw. die Kriterien der Normierung von 0 bis 100 erfüllen (d.h., es gibt mehr Datenpunkte mit einer Summe von x, y und z, die 50 überschreiten). Die Hilfsstrichsbeschriftung des Ternärdiagramms wird automatisch auf 0 bis 100 gesetzt.

 

Origin-Version mind. erforderlich: Origin 9.0 SR0

In Vorgängerversionen von Origin konnten Sie dies manuell ändern, indem Sie die Einstellung für Dividieren durch als 0,01 auf der Registerkarte Beschriftung Hilfsstriche des Dialogs Achsen festlegen.

 

Schlüsselwörter:Ternärdiagramm, ternär, Achsenbeschriftung, normierte Daten, Beschriftung Hilfsstriche
