FAQ-981 Wie teile ich Projekte mit Anwendern älterer Versionen?

Share-projects-users-older-versions

Letztes Update: 02.11.2018

Beim Teilen von Origin-Projekten mit Kollegen, die eine ältere Version der Software haben, gibt es mehrere Aspekte, die Sie eventuell berücksichtigen müssen.

Inhalt

  • 1 Dateiformat OPJU vs. OPJ
  • 2 Vereinfachte Zellennotation (SCN)
  • 3 Zellenformel
  • 4 Neue Diagrammtypen
  • 5 Neue Hilfsmittel und Apps für die Analyse

Dateiformat OPJU vs. OPJ

Das neue OPJU-Dateiformat für Origin-Projekte wurde in Version 2018 eingeführt. Wenn Sie beabsichtigen, ein Projekt mit Anwendern älterer Versionen gemeinsam zu nutzen, müssen Sie vielleicht den Menübefehl Datei: Als Projekt speichern und dann *.OPJ in der Auswahlliste Als Typ speichern auswählen. (*1)

Beachten Sie, dass es nicht zu Verlust von Daten oder Grafiken führt, wenn Sie eine Datei in einem älteren OPJ-Format abspeichern. (*2) Lesen Sie die untenstehenden Abschnitte zu den neuen Funktionen, die nicht in älteren Versionen zur Verfügung stehen.

Sie können das Standarddateiformat mit Hilfe der Systemvariablen @OPJ in OPJ ändern.

Hinweis 1: Der Origin Viewer (Version 9.5.5 oder höher) kann die OPJU-Datei auch in eine OPJ-Datei verwandeln.

Hinweis 2: Das OPJU-Format ist in Unicode kodiert. Es enthält eine signifikante Erweiterung von Sonderzeichen sowie die Zeichen anderer Sprachen wie asiatische Buchstaben. Diese Unicode-Zeichen, die sich außerhalb der klassischen Kodierung befinden (d. h. nicht innerhalb der aktuellen "Codeseite") würde nicht richtig angezeigt (sondern als "?").

Vereinfachte Zellennotation (SCN)

Seit Version 2017 ist die vereinfachte Zellennotation (SCN) für neue Arbeitsblätter standardmäßig aktiviert. Dies führt zu Spaltenkurznamen wie A, B, C etc. wie in Excel. Dadurch sind die Spaltenformeln in der Zeile F(x) und im Dialog Werte setzen sehr viel einfacher. So ist es nun möglich, A*10 anstatt col(A)*10 einzugeben.

Wenn Sie Ihre Projekt mit aktivierter SCN teilen und eine einfachere Notation für die Spaltenberechnungen verwendet haben, sind die berechneten Daten in älteren Versionen noch sichtbar und verfügbar, aber Benutzer von älteren Versionen können die Werte nicht neu berechnen, indem sie die Eingabespalten ändern. Sie müssen die Formel bearbeiten/ändern, damit diese mit ihrer Version kompatibel ist.

Sie können die vereinfachte SCN für Ihre Arbeitsmappen ausschalten, wenn Sie das bevorzugen.

Zellenformel

Die Excel-ähnliche Zellenformel wurde mit Origin-Version 2018 eingeführt. Mit dieser Funktionen können Sie Formel in einzelne Zellen eingeben, z. B. =total(A1:B10).

Auch wenn Sie ein Projekt mit dem älteren OPJ-Dateiformat speichern, werden Benutzer von älteren Versionen nur die Ausdrücke der Zellenformel sehen. Sie sehen nicht die berechneten Werte und natürlich können sie die Berechnungen, die mittels einer Zellenformel durchgeführt wurden, weder aktualisieren noch ändern.

Neue Diagrammtypen

Jede neue Version von Origin umfasst typischerweise einige neue Diagrammtypen. Wenn Sie ein Projekt mit Hilfe der neuen Diagrammtypen erstellen, können Benutzer von älteren Versionen diese Diagramme weder anzeigen noch verändern.

Neue Hilfsmittel und Apps für die Analyse

Jede neue Version von Origin enthält üblicherweise neue Analysehilfsmittel und/oder neue Apps. Wenn diese Hilfsmittel und Apps in einem Projekt verwendet werden, werden Analyseberichte und andere numerische Ausgaben für Benutzer einer älteren Version sichtbar. Sie können allerdings keine Neuberechnungen der Ergebnisse durchführen, indem sie die Eingabedaten ändern.
