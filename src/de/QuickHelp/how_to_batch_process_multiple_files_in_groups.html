FAQ-761 Wie verarbeite ich mehrere Dateien in Gruppen per Stapelverarbeitung?

Batch-Process-Groups

Letztes Update: 20.10.2020

Um mehrere Datendateien in Gruppen per Stapelverarbeitung zu verarbeiten, können Sie eine Analysevorlage für mehrere Datenblätter erstellen, so dass Datendateien in der gleichen Gruppe in die entsprechenden Datenblätter der Analysevorlage importiert werden können. Sie erfahren in diesem Tutorial, wie Sie eine Analysevorlage für mehrere Datenblätter erstellen und die erstellte Vorlage im verwenden, um Dateien in Gruppen gestapelt zu verarbeiten.

Auch wenn die Methode nicht so flexibel oder leistungsstark wie die oben eingesetzten Operationen der Stapelverarbeitung ist, können Sie Import- und Analyseoperationen in der aktiven Arbeitsmappe "klonen". Dazu verwenden Sie die Vorgehensweise, die in diesem Tutorial beschrieben wird.

Origin-Version mind. erforderlich: Origin 2016 SR0

Schlüsselwörter:Stapelverarbeitung, Gruppen, gruppiert, klonen
