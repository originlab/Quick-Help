FAQ-739 Wenn ich Origin auf einem Mac über eine Virtualisierungssoftware ausführe, warum kann ich keine Schnelltasten von einer Mac-Tastatur verwenden?

Fail-to-Use-Shortcut-Keys-from-Mac-Keyboard

Letztes Update: 16.06.2015

Einige Tastenkombinationen unterscheiden sich möglicherweise, wenn eine Mac-Tastatur verwendet wird. Sie können zum Beispiel die Taste Entfernen auf Ihrer PC-Tastatur verwenden, um Daten zu löschen, tun dies aber mit fn + Del auf der Mac-Tastatur. Sie können sich in der Vergleichstabelle der Tastenkombinationen von PC- und Mac-Tastatur über diese Unterschiede informieren.
