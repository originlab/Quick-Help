FAQ-891 Wie kann ich die Anzeige der Dezimalstellen in den Analysehilfsmitteln festlegen?

Control-Digits-in-AnalysisTool

Letztes Update: 15.09.2017

Um die Dezimalstellen der numerischen Daten in einem Analyseberichtsblatt festzulegen:

  1. Klicken Sie auf den nach unten weisenden Pfeil hinter dem Zweigtitel oder klicken Sie mit der rechten Maustaste auf eine Zelle der Tabelle im Berichtsblatt.
  2. Wählen Sie im Kontextmenü Stellen....
  3. Legen Sie im Dialog Stellen im Bericht die Einstellungen für die Dezimalstellen fest.

Sie können das Kontrollkästchen Als Standard speichern aktivieren, um die Einstellung der Dezimalstellen bei diesem Analysehilfsmittel als Standard festzulegen. Alle Berichtsblätter, die mit diesem Analysehilfsmittel erzeugt werden, verwenden diese Einstellung.

Um Dezimalstellen global festzulegen:

Wenn Sie die Dezimaleinstellungen für alle Analysehilfsmittel festlegen möchten, nehmen Sie die Einstellung unter Stellen im Bericht auf der Registerkarte Zahlenformat im Dialog Optionen vor.
