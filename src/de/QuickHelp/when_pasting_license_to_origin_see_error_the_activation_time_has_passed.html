FAQ-798 When I paste my license into Origin I see "The activation time has passed". Why?

Activation-Time-Pass-Err

Last Update: 11/10/2015

When selecting "Get a license file now. This computer has internet access." the license must be entered into Origin within 15 minutes. If you see this message, you must restart Origin and obtain a new license.

When selecting "Get a license file. This computer does NOT have internet access." the license must be entered into Origin within 24 hours. If you see this message, you must restart Origin and obtain a new license.
