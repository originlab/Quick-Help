FAQ-180 Wie ändere ich die einzelne Breite der Säulen/Balken in einem Diagramm?

change-bar-column-width

Letztes Update: 04.02.2015


Sie können die Werte aus einer anderen Datenspalte verwenden, um die Abstände zwischen jeder/m Säule/Balken festzulegen. Auf diese Weise kann die Breite von Säule/Balken individuell gesteuert werden. Um dieses Bedienelement zu aktivieren, klicken Sie zum Öffnen des Dialogs Details Zeichnung doppelt auf das Diagramm. Gehen Sie zur Registerkarte Abstände und wählen Sie eine Datenspalte für Breite (in %) in der Auswahlliste. Sie können auch den Skalierungsfaktor festlegen. Weitere Informationen finden Sie in der Hilfe von Origin.

Schlüsselwörter: Säulen, Balken, zeichnen, Breite, Abstände

Origin-Version mind. erforderlich: 9.1 SR0
