FAQ-751 Ich sehe eine Meldung 'Lizenzdatei fehlt oder nicht korrekt', wenn ich Origin ausführe. Wo liegt der Fehler?

License-file-missing-or-incorrect

Letztes Update: 27.08.2015

Es besteht ein Konflikt zwischen Origins MSI Installer und Microsoft KB3072630 (https://support.microsoft.com/en-us/kb/3072630), das am 15. Juli 2015 herauskam. Wenn Sie bei einem Computer, auf dem KB3072630 installiert ist, den MSI Installer verwenden, um Origin zu installieren, öffnet Notepad die Meldung "Lizenzdatei fehlt oder nicht korrekt", wenn Sie versuchen, Origin auszuführen.

Die folgenden Anweisungen können verwendet werden, um dieses Problem zu lösen.

  1. Deinstallieren Sie Origin aus den Programmen und Funktionen.
  2. Klicken Sie auf Start, dann auf Ausführen, geben Sie regedit in das Feld Öffnen ein und klicken Sie dann auf OK.
  3. Finden und klicken Sie dann auf den folgenden Subkey in der Registry: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\Installer
    Hinweis: Wenn dieser Subkey nicht existiert, erstellen Sie ihn.
  4. Zeigen Sie im Menü Bearbeiten, dann auf Neu und klicken Sie anschließen auf den Wert DWORD.
  5. Geben Sie für den Namen von DWORD RemappedElevatedProxiesPolicy ein und drücken Sie dann Enter.
  6. Klicken Sie mit der rechten Maustaste auf RemappedElevatedProxiesPolicy und dann mit der linken Maustaste auf Modifizieren.
  7. Geben Sie im Datenfeld Wert 1 ein und klicken Sie dann auf OK.
  8. Installieren Sie Origin.
  9. Nachdem Origin erfolgreich installiert wurde, setzen Sie den DWORD-Wert von RemappedElevatedProxiesPolicy auf 0 zurück, um die Sicherheitslösung für KB 3072630 wieder zu aktivieren

Wenn Sie statt des MSI Installers den InstallShield Setup Installer verwenden und Sie außerdem die Meldung "Lizenzdatei fehlt oder nicht korrekt" beim Start von Origin erhalten, wendn Sie sich bitte an uns.
