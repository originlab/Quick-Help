FAQ-723 Wie ändere ich die Schriftgröße im Skriptfenster?

Set-Font-Size-in-Script-Window

Letztes Update: 12.12.2022

Seit Origin 2023 wird das Skriptfenster aktualisiert, um die Steuerung der Schriftgröße einzuschließen. Ändern Sie die Schriftgröße durch Drücken der Strg-Taste, während Sie mit dem Mausrad scrollen.

Bei älteren Versionen von Origin modifiziere Sie die Datei User Files\Origin.ini, wie unten auf dieser Seite gezeigt.
