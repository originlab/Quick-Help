FAQ-166 Wie passe ich alle Zeichnungen in einer Matrix von Streudiagrammen gleichzeitig benutzerdefiniert an?

Customize-Scatter-Matrix

Letztes Update: 08.01.2019

Eine Matrix von Streudiagrammen ist ein paarweise verbundenes Streudiagramm mehrerer Variablen, die in einem Matrixformat dargestellt werden. In einer Matrix von Streudiagrammen gibt es mehrere Layer.

Ab Origin 2016 wird für die Matrix von Streudiagrammen das gemeinsame Bearbeiten aller Zeichnungen unterstützt: Standard ist, dass alle benutzerdefinierten Anpassungen, die an einem Streudiagramm gemacht werden, automatisch auf alle anderen Streudiagramme angewendet werden und dass alle Anpassungen an einer Zeichnung auf der Diagonalen (Histogramme, Boxdiagramme) automatisch auf andere Zeichnungen auf der Diagonalen angewendet werden.

Hinweis: Um Matrizen von Streudiagrammen unabhängig in den Origin-Versionen 2016 und 2017 zu bearbeiten, müssen Sie Format: Seite wählen, zur Registerkarte Allgemeines wechseln und das Kontrollkästchen Unabhängige Layersteuerung aktivieren. Diese Option ist nicht in Origin 2018b oder später verfügbar (Zeichnungen können nicht unabhängig bearbeitet werden).

Weitere Möglichkeiten, alle Zeichnungen benutzerdefiniert anzupassen

Verwenden der Funktionen Format kopieren und Format einfügen

  1. Passen Sie den Typ, die Größe und die Farbe des Symbols oder die Hintergrundfarbe in einem Layer benutzerdefiniert an. Klicken Sie mit der rechten Maustaste auf diesem Layer auf Format kopieren: Symbol, Linie und Füllung, Farben oder Hintergrund und so weiter.
  2. Klicken Sie dann auf die weiße Fläche des Diagramms, klicken Sie mit der rechten Maustaste und wählen Sie Format einfügen, um die Einstellung auf alle anderen Layer zu übertragen.

Verwenden des Hilfsmittels Layerverwaltung zum Anpassen der Hintergrundfarbe

  1. Wählen Sie Grafik: Layer-Verwaltung, um den Dialog zu öffnen, und gehen Sie zur Registerkarte Anzeige.
  2. Markieren Sie all die Layer, die in der Layerauswahl aufgeführt werden, und legen Sie dann die Hintergrundfarbe der Layer fest.
