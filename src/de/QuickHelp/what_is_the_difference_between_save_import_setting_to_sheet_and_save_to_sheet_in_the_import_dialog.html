FAQ-969 Was ist der Unterschied zwischen "Importeinstellungen im Blatt speichern" und "Speichern in <Blatt>" im Importdialog?

Save-to-Sheet-Theme-Diff

Letztes Update: 19.09.2018

Sie haben vielleicht bemerkt, dass es zwei ähnliche Optionen im Ausklappmenü Dialogdesign gibt, um Ihre Importeinstellungen im Blatt zu speichern, wenn Sie Daten mit Hilfe des Importdialogs importieren. Wenn Sie zum Beispiel eine ASCII-Datei mit dem Dialog ImpASC importieren und auf die dreieckige Schaltfläche rechts von Dialogdesign klicken, sehen Sie diese beiden Menüelemente:

  • Importeinstellungen im Blatt speichern
  • Im <Blatt> speichern

Was ist der Unterschied zwischen diesen beiden?

“Im <Blatt> speichern” speichert Ihre Importeinstellungen in ein X-Funktions-Dialogdesign, das erneut geladen werden kann, wenn Sie diese X-Funktion für den Import verwenden. Wenn Sie jedoch keine X-Funktion verwenden und eine Datendatei beispielweise per Drag&Drop importieren, können die Einstellungen nicht verwendet werden.

“Importeinstellungen in Blatt speichern” wird nun eingeführt, um dies zu ermöglichen. Die Option speichert die Importeinstellungen in einem Seitenfilter, der automatisch verwendet werden kann, wenn eine Datei per Drag&Drop in ein Arbeitsmappenfenster gezogen wird.
