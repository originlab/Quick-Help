FAQ-754 Wie klone ich ein Diagramm mit der gleichen Datenstruktur aus einem bestehenden Diagramm?

Clone-Graph

Letztes Update: 17.03.2021

Klicken Sie bei aktivem Diagrammfenster mit der rechten Maustaste auf die Fenstertitelleiste, um Template speichern unter auszuwählen, oder gehen im Menü zu Datei: Template speichern unter..., um den Dialog Utilities\File:template_saveas zu öffnen. Aktivieren Sie das Kontrollkästchen Als klonbare Vorlage markieren, wählen Sie Kriterien, die der Datenstruktur entsprechen, in der Auswahlliste Spalte stimmt überein mit und klicken Sie auf die Schaltfläche OK, um das Diagramm als eine klonbare Vorlage zu speichern.

Danach klonen Sie ein Diagramm auf Grundlage dieser klonbaren Vorlage mit neuen Daten der gleichen Datenstruktur. Dazu markieren Sie alle zu zeichnenden Daten und wählen im Menü Zeichnen: Meine Vorlagen. Fahren Sie fort, indem Sie die gespeicherte klonbare Vorlage auswählen, gekennzeichnet von einem davor stehenden Schafsymbol zur Unterscheidung von einer gewöhnlichen Diagrammvorlage, um ein Diagramm mit genau den gleichen benutzerdefinierten Anpassungen wie die Vorlage zu erstellen.

Sie können auch die Vorlagenbibliothek wählen, um eine Vorschau von der Vorlage in dem Dialog anzuzeigen und zu bestimmen, welche Vorlage zum Zeichnen verwendet werden soll.

Einzelheiten finden Sie auf dieser Seite.
