FAQ-370 Wie liste ich alle Skript-Variablen auf?

List-Variables

Letztes Update: 04.02.2015

Um alle Sitzungsvariablen aufzulisten, geben Sie Folgendes im Skript- oder Befehlsfenster ein und drücken Sie dann Enter:

list a

Um alle Projektvariablen und numerischen Sitzungsvariablen aufzulisten, geben Sie bitte Folgendes ein:

list v

Um alle Projektvariablen und Zeichenkettenvariablen der Sitzung aufzulisten, geben Sie bitte Folgendes ein:

list vs


Weitere Einzelheiten finden Sie unter dem Befehl LIST.


Hinweis: Der Befehl ed öffnet den Dialog LabTalk Variablen und Funktionen. In diesem Dialog können Sie auswählen, welche Variablen Sie anzeigen lassen möchten: System-, Projekt-, Zeichenkettenvariablen usw.

Schlüsselwörter: Bearbeiten, Sitzung, Projekt
