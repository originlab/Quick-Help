FAQ-1110 Was soll ich tun, wenn Origin abstürzt, wenn ich einen Link aus Excel einfüge?

Paste-Link-from-Excel-Make-Origin-Crash

Letztes Update: 30.11.2021

Wenn Ihr Origin beim Kopieren von Daten aus einer Excel-Datei und ihrem Einfügen in ein Origin-Arbeitsblatt (Rechtsklick auf eine Zelle und Auswahl von Link einfügen) abstürzt, erreicht Ihr Excel-Dateipfad eventuell die Microsoft-Beschränkung, die ein Problem für die Funktion Link einfügen verursacht.

Um dieses Problem zu umgehen,

kürzen Sie bitte den vollen Pfad der Excel-Datei. Sie können

  • die Excel-Datei in einen Speicherort mit kürzerem Pfad verschieben ODER
  • der Excel-Datei und dem Blatt einen kürzeren Namen geben.

Wichtig ist, dass die Kombination auf Dateipfad, Dateiname und Blattname weniger als 250 Zeichen hat.

 

Hinweise:

Das Kopieren von Daten aus Excel und das direkte Einfügen in ein Origin-Arbeitsblatt haben womöglich ein ähnliches Problem, wenn der Dateipfad zu lang ist. Dies liegt außerhalb von Origins Kontrolle, da Excel eventuell einen internen Mechanismus verwendet, um Daten aus einer Datei abzurufen, anstatt große Mengen direkt in der Zwischenablage abzulegen.
