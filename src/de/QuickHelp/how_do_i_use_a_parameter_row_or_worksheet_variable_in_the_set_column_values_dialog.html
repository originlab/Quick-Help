FAQ-497 Wie verwende ich eine Parameterzeile oder Arbeitsblattvariable im Dialog Werte setzen?

Use-Variable-in-SetColValue

Letztes Update: 04.02.2015

Sie müssen im Dialog Werte setzen das Feld Skript vor Anwenden der Formel verwenden, um einem Double den Wert der Parameterzeile zuzuweisen.

double dd = %(col(a)[P1]$);

Die Formel könnte dann sein:

col(a)/dd;

Um auf andere Arbeitsblattköpfe zuzugreifen, lesen Sie bitte Zeichen der Spaltenbeschriftungszeile.
