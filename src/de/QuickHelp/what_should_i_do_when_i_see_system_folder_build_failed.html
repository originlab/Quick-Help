FAQ-19 Was soll ich tun, wenn die Fehlermeldungen "System Folder Build Failed" oder "X-Function failed to execute" angezeigt werden?

System-Folder-Build-Failed

Letztes Update: 07.01.2019

Wenn die Fehlermeldungen "System Folder Build Failed" oder "X-Function failed to execute" in Origin angezeigt werden, bitte:

  1. Wählen Sie Fenster: Skriptfenster im Menü.
  2. Geben Sie das Skript unten im Skriptfenster ein und drücken Sie Enter, um den OCTemp-Ordner zu löschen, so dass OriginC alle neuen Quelldateien verwendet. 
    del -occlean
  3. Führen Sie das Skript unten im Skriptfenster aus, um den Wert des Anwenderdateiordnerpfads in der Registry zu löschen. Danach können Sie den Anwenderdateiordner zurücksetzen, wenn Sie Origin das nächste Mal starten. 
    del –path
  4. Schließen Sie Origin.
  5. Starten Sie Origin neu. Legen Sie einen leeren neuen Ordner als Anwenderdateiordner fest.
Hinweise:

Um benutzerdefinierte Dateien aus einem früheren Anwenderdateiordner zu übertragen:

  1. Wählen Sie Einstellungen: Anwenderdateien übertragen. Der Dialog Anwenderdateien übertragen wird geöffnet.
  2. Wählen Sie in dem geöffneten Dialog die Option Festgelegter Ordner in der Auswahlliste Quellenverzeichnis. Aktivieren Sie das Kontrollkästchen Rekursiv. Verschieben Sie die Dateien in das rechte Bedienfeld und klicken Sie auf die Schaltfläche Übertragen.


Wenn durch diesen Vorgang die Probleme nicht gelöst werden, können Sie versuchen, die Operation Reparieren auszuführen (dazu klicken Sie auf das Symbol "Dateien hinzufügen oder entfernen" in Ihrer Apps- oder Programmliste). Stellen Sie sicher, dass Sie das Installationsprogramn bis zum Ende ausführen, starten Sie dann Origin neu und prüfen Sie, ob Ihr Problem gelöst wurde.

Sollte diese Maßnahme fehlschlagen, wenden Sie sich bitte an den Technischen Support von OriginLab.
