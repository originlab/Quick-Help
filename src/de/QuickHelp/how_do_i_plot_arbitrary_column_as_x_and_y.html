FAQ-381 Wie zeichne ich eine beliebige Spalte als x und y?

Plotxy-PlotDesignation

Letztes Update: 04.02.2015

Jedes Datenspalte ist mit einer Diagrammzuordnung verbunden, die schnelles und einfaches Zeichnen ermöglicht. Wenn Sie jedoch die zu zeichnenden Spalten nicht markieren, wird der Dialog Diagrammeinstellungen geöffnet, in dem Sie zuweisen können, welche Spalte X und welche Y ist etc., unabhängig von der Diagrammzuordnung im Arbeitsblatt.


Origin besitzt auch eine X-Funktion plotxy, die Sie über Skript verwenden können. Geben Sie im Befehlsfenster, das Sie im Menü Fenster: Befehlsfenster öffnen, plotxy ein und drücken Sie die Leertaste, um sich von der automatischen Vervollständigung bei den verschiedenen Argumenten der Funktion helfen zu lassen.


Beispiele:

// To create a new scatter plot: plotxy iy:=(col(1), col(2)) plot:=201; // To create a new layer in the active graph: plotxy [Book1]Sheet1!(A,C) ogl:=new;

Schlüsselwörter:Grafik, Layer, erstellen
