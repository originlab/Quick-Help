FAQ-862 Wie aktualisiere ich einen Datenfilter, wenn ich Änderungen an meinen Daten vornehme?

how-to-update-data-filter

Letztes Update: 06.01.2019

Origin unterstützt das Filtern von drei Datentypen: (1) Datum-Zeit, (2) Numerisch und (3)Text. Die Filterungsoptionen hängen von dem Datentyp ab, der gefiltert wird (z. B. Datum-Zeit-Daten werden gefiltert, indem ein Datum oder ein Datenbereich festgelegt wird). Wenn Sie einen Filter anwenden, verbergen Sie diese Zeilen der Arbeitsblattsdaten, die Ihre Filterbedingungen nicht erfüllen. Gefilterte Daten können dann gezeichnet und analysiert werden. Dies macht Datenfilter für explorative Analysen extrem hilfreich, insbesondere im Umgang mit großen Datensätzen.

Beachten Sie, dass Sie den Filter erneut anwenden müssen, wenn Sie Änderungen an den gefilterten Daten in einer Arbeitsblattspalte vornehmen, wie es z. B. auch bei MS Excel der Fall ist. Wenn der Filter erneut angewendet werden muss, wird das Filtersymbol gelb angezeigt. Sie können den Filter erneut anwenden, indem Sie auf die Schaltfläche Datenfilter erneut anwenden auf der Symbolleiste Worksheet-Daten klicken.

Allgemeine Lösung

Sie können eine automatische Filteraktualisierung bei Datenänderung einstellen, indem Sie Folgendes tun:

  1. Klicken Sie im Arbeitsblatt mit der rechten Maustaste in die graue Fläche auf der rechten Seite der Datenspalten und wählen Sie Einstellungen im Kontextmenü aus.
  2. Klicken Sie auf die Registerkarte Skript und geben Sie Folgendes in das Feld Skript ein: 
    wks.runfilter();
  3. Aktivieren Sie das Kontrollkästchen neben Bei Änderungen im Bereich und geben Sie einen Bereich für die Spalte(n) ein, in denen die Daten aktualisiert werden. Hilfe bei dieser Syntax erhalten Sie unter Das Skriptfeld.
  4. Klicken Sie auf OK.


Jegliche spätere Änderungen an den gefilterten Daten, einschließlich dem erneuten Import der Datei, löst eine Filteraktualisierung aus.

Eine weitere Lösung für den Dateiimport

Viele Dialoge für Importoptionen für den Import (Daten: Aus Datei importieren) enthalten das Kontrollkästchen Skript nach jeder importierten Datei, das verwendet werden kann, um die Methode wks.runfilter() auszuführen.
