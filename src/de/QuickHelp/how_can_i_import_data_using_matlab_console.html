FAQ-435 Wie importiere/exportiere ich Daten aus/in eine MATLAB-Arbeitsbereich?

Matlabconsole-Importdata

Letztes Update: 08.01.2019

In Origin können Sie die MATLAB-Konsole verwenden, um Daten aus MATLAB zu importieren und sie mit der grafischen Bedienoberfläche und Befehlen in Arbeitsblatt oder Matrix abzulegen. Sie können außerdem Daten aus Origin in den MATLAB-Arbeitsbereich exportieren, und zwar als direkt aufrufbare Variablen. Diese Funktion macht es erforderlich, dass sowohl Origin als auch MATLAB auf dem gleichen Rechner installiert sind.

Um die MATLAB-Konsole aufzurufen, wählen Sie Konnektivität: MATLAB-Konsole im Origin-Menü. Der Dialog MATLAB-Konsole wird aufzurufen. Führen Sie desktop in dem Bedienfeld aus. Dieser MATLAB-Befehl öffnet den Standarddesktop von MATLAB. Nach dem Importieren der Datendatei in MATLAB wechseln Sie zurück zum Dialog MATLAB-Konsole in Origin und klicken Sie auf die Schaltfläche Importieren, um den Dialog MATLAB-Import aufzurufen. Wählen Sie die gewünschten Daten und klicken Sie Import als Matrizen oder Import als Worksheet, um die Daten in Origin zu importieren.

Umgekehrt wählen Sie, um Daten von Origin auf den MATLAB-Desktop zu exportieren, Konnektivität: MATLAB-Konsole im Origin-Menü. Der Dialog MATLAB-Konsole wird aufgerufen. Führen Sie in dem Feld desktop aus, um den Standarddesktop von MATLAB zu öffnen. Klicken Sie jetzt auf die Schaltfläche Export, um das Exportieren der Daten in den MATLAB-Arbeitsbereich zu initiieren.
