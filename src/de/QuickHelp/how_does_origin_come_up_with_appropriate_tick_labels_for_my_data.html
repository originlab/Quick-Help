FAQ-384 Wie erstellt Origin passende Hilfsstrichsbeschriftungen für meine Daten?

Tick-Labels-by-LT

Letztes Update: 04.02.2015

Die Probleme beim Einstellen von korrekten Achsenskalierungen und Berechnen von sinnvollen Hilfsstrichsbeschriftungen werden von dem Befehl limit in LabTalk geregelt. Zum Beispiel:

newbook; range rax = 1!col(a); rax = data(17.3,23.6,.25); range ray = 1!col(b); ray = normal(rax.GetSize());
limit ray;  // find limiting values for dataset // limit.xmin = minimum x value // limit.xmax = maximum x value // calculate the appropriate increment value and adjust begin and end values begin = limit.xmin; end = limit.xmax; ticks = 5; limit.size =; limit.ymin =; limit.imin =; limit.ymax =; limit.imax =; limit.xmin =; limit.xmax =; ty Beginning with From : $(begin), To : $(end), Steps : $(ticks); limit -r begin end ticks increment; ty Limit command creates From : $(begin), To : $(end), Increment : $(increment); 
plotxy (1,2);

Die Anfangs-, End- und Inkrementwerte können dann verwendet werden, um jeweils Von, Bis und Inkrement einer Achse festzulegen.


Alle Standarddiagrammvorlagen von Origin haben einen Rand von 8%, der zu dem Bereich der X- und Y-Daten vor Anwendung der oben stehenden Berechnung hinzugefügt wird. Die Eigenschaft kann modifiziert werden mit

layer.x.rescalemargin
oder
layer.y.rescalemargin
