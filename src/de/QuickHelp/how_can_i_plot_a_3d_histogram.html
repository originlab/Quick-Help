FAQ-320 Wie kann ich ein 3D-Histogramm zeichnen?

Plot-3D-Histogram

Letztes Update: 04.02.2015


Sie können das Hilfsmittel 2D-/Klasseneinteilung verwenden, um ein 3D-Histogramm zu zeichnen.

 

  1. Öffnen Sie ein Arbeitsblatt mit zwei Variablen als X- und Y-Spalte.
  2. Wählen Sie im Menü Statistik: Deskriptive Statistik: 2D Häufigkeitszählung/Klasseneinteilung.
  3. Stellen Sie in dem geöffneten Dialog sicher, dass Ausgabematrix aktiviert ist.
  4. Erweitern Sie den Zweig Matrixdiagramme und wählen Sie 3D-Balken.


Sie können sich auch das Beispiel auf der Seite Grafikgalerie ansehen.

 

Schlüsselwörter:2D-Histogramm, 3D-Histogramm, 2D-Klasseneinteilung, 2D-Häufigkeitszählung
