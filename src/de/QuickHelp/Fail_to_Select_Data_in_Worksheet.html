FAQ-924 Warum kann ich eine Spalte oder einen Datenbereich in einem meiner Arbeitsblätter nicht markieren?

Fail-to-Select-Data-in-Worksheet

Letztes Update: 05.04.2018

In seltenen Fällen kann es vorkommen, dass ein Anwender Daten im Arbeitsblatt nicht markieren kann:

  • Wenn mit dem Maus über den Kurznamen einer Arbeitsblattspalte gefahren wird, wird der Cursor nicht in den Auswahlmodus versetzt (der Zeiger verwandelt sich nicht in einen nach unten weisenden Pfeil).
  • Wenn Sie auf den Spaltenheader klicken, wird die Spalte nicht ausgewählt.
  • Sie können keinen Zellenbereich im Arbeitsblatt auswählen.

Sollte dies der Fall sein, öffnen Sie das Skriptfenster (Fenster: Skriptfenster), während das Arbeitsblatt mit dem Problem aktiv ist, geben Sie Folgendes ein und drücken Sie Enter:

worksheet -es

Ab Origin 2018b wird dieser Befehl ausgeführt, wenn eine gespeicherte Datei geladen wird, und sollte solche Probleme zukünftig verhindern.

Schlüsselwörter:Auswahl, Maus, Ziehen
