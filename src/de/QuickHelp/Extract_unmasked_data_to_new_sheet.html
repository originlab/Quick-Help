FAQ-1009 Wie extrahiere ich unmaskierte Daten in einem neuen Arbeitsblatt?

extract-unmasked-data-to-new-sheet

Letztes Update: 14.05.2019

Wenn Sie auf die Datenpunkte in einem Diagramm eine Maskierung anwenden, werden die maskierten Daten im Diagramm verborgen oder gezeigt (Wahl des Anwenders) und mit einer Kontrastfarbe im Arbeitsblatt hervorgehoben.

Während die maskierten Punkte im Diagramm verborgen werden können, werden sie nie im Arbeitsblatt verborgen. Wenn Sie also das Arbeitsblatt auswählen und Ihre Auswahl dann kopieren und in ein neues Arbeitsblatt einfügen, werden Sie feststellen, dass auch Ihre maskierten Daten kopiert wurden.

Eine einfache Methode zum Ausgeben von nur unmaskierten Daten ist:

  1. Als Erstes verwenden Sie das Hilfsmittel Datenmarkierer , um alle Daten im Diagramm (unwichtig, ob maskiert oder unmaskiert) auszuwählen. Nur die Daten, die unmaskiert sind, werden ausgewählt.
  2. Sie können das gesamte Arbeitsblatt auswählen, dann kopieren und in ein neues Arbeitsblatt einfügen. Nur die unmaskierten Daten werden in Ihr neues Blatt kopiert.

Schlüsselwörter:Datenmarkierer, Daten kopieren, Zeilen kopieren, maskieren
