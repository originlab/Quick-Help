FAQ-207 Wie wird eine X-Spalte von Textwerten gezeichnet?

XCol-Text-Plotting

Letztes Update: 04.02.2015

Das Standardformat (Text & Numerisch, Numerisch etc.) einer Spalte ist Text & Numerisch. Wenn Sie Textwerte in eine Spalte eingeben und importieren, erkennt Origin sie als solche. Wenn diese Werte als X-Werte gezeichnet werden, zeichnet Origin gegen Zeilennummern, d.h. einen Startwert von 1 und ein Inkrement von 1. Jeder entsprechende Textwert wird dann zum Beschriften der Hilfsstriche verwendet.

Wenn Sie in dem Diagramm doppelt auf diese Hilfsstrichsbeschriftungen klicken, können Sie in dem sich öffnenden Dialog Achsen sehen, dass die Hilfsstrichsbeschriftungen dem Typ Text aus Datensatz entsprechen und der aufgeführte Datensatzname der Name dieser X-Spalte ist.

Wenn Sie eine Mischung aus Text und numerischen Werten in Ihrer X-Spalte hatten, zeichnet Origin je nach dem jeweiligen Prozentanteil entweder alle Werte als Text oder es zeichnet die numerischen Werte und behandelt den Text als einen fehlenden Wert.
