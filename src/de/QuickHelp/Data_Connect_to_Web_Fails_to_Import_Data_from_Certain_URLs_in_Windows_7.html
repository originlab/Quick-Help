FAQ-995 Daten: Die Verbindung zum Internet schlägt beim Importieren von Daten von bestimmten URLs in Windows 7 fehl

Data-Connect-to-Web-Fails-Windows-7

Letztes Update: 07.02.2019

Wenn Sie Origin 2019b oder höher auf Windows 7 installiert haben und Datenkonnektoren über Daten: Mit Web verbinden verwenden, kann es passieren, dass der Import von Daten von URLs, deren Protokoll HTTPS ist, fehlschlägt. Möglicherweise (oder auch nicht) wird Ihnen eine kryptische Fehlermeldung angezeigt.

Sie bezieht sich auf bestimmte nicht mehr aktuelle Einstellungen in Windows.

Sie können den Assistenten Microsoft Easy Fix herunterladen und installieren, um das Problem anzugehen: https://aka.ms/easyfix51044

Eine weiterführende technische Erklärung des Problems finden Sie hier: https://support.microsoft.com/en-us/help/3140245/update-to-enable-tls-1-1-and-tls-1-2-as-default-secure-protocols-in-wi

Schlüsselwörter:Datenkonnektor, Mit Web verbinden, Windows 7, HTTPS, Download fehlgeschlagen, Error -50, Fehler
