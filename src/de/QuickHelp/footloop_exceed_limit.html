FAQ-388 Was kann ich tun, wenn die Größe eines For-Loop-Befehls die Beschränkung überschreitet?

Letztes Update: 04.02.2015

for-loop-exceed-limit

In LabTalk liegt die Beschränkung der Skriptlänge eines For-Loop-Befehls bei 3072 Zeichen, einschließlich weißer Leerstellen. Wenn das Skript zu lang ist, können Sie den Befehl define verwenden, um zuerst ein Makro zu definieren und dann das Makro in dem Loop-Befehl aufzurufen.

Beispiel:

// Define three macros define Part1 {ty one;}; define Part2 {ty two;}; define Part3 {ty three;}; // Call macros within for loop int ii=0; for (ii=0;ii<3;ii++) { Part1; Part2; Part3; }

Schlüsselwörter:Loop, Makro definieren, Zeichenbeschränkung, For, Makro
