FAQ-817 Wie füge ich Ankerlinien zu einem 2D-Liniendiagramm hinzu?

How-to-add-drop-lines-to-line-plot

Letztes Update: 17.03.2016

Vor Origin 9.1 machte das Hinzufügen von Ankerlinien zu einem 2D-Liniendiagramm folgenden Workaround erforderlich:

  1. Klicken Sie zum Öffnen des Dialogs Details Zeichnung doppelt auf die Zeichnung.
  2. Legen Sie für Diagrammtyp die Option Linie + Symbol fest (Auswahlliste unten links). Die Registerkarte Ankerlinien wird hinzugefügt.
  3. Klicken Sie auf die Registerkarte Ankerlinien, konfigurieren Sie Ihre Ankerlinien und klicken Sie dann auf Anwenden.
  4. Zum Entfernen der Zeichnungssymbole für eine unabhängige Zeichnung klicken Sie auf die Registerkarte Symbol und öffnen Sie die Symbolgalerie (Vorschau unten) und dann auf eine leere Zelle (unten in der Galerie). Für abhängige gruppierte Zeichnungen (Modus Bearbeiten, Registerkarte Gruppe) setzen Sie den Symboltyp auf Inkrement = Kein und auf der Registerkarte Symbol die Form auf 0 Kein Symbol. Klicken Sie auf Übernehmen oder OK.

Hinweis: Diese Methoden zum Entfernen der Zeichnungssymbole funktionieren oft besser als Symbolgröße auf 0 zu setzen. Dieser Artikel FAQ-818 Warum funktionieren gestrichelte Linien nicht in meinem Linien-Punktdiagramm? könnte Ihnen auch weiterhelfen.

Seit Origin 9.1 können Sie Ankerlinien mit Hilfe der Bedienelemente auf der Registerkarte Ankerlinien im Dialog Details Zeichnung direkt zu einem Liniendiagramm hinzufügen.
