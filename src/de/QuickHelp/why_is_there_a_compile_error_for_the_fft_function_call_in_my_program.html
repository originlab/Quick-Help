FAQ-398 Warum wird ein Kompilierungsfehler für den Aufruf der FFT-Funktion in meinem Programm gemeldet?

FFT-Compile-Error

Letztes Update: 04.02.2015


Manchmal kompilieren Sie eine Origin C-Funktion, die die FFT-Funktion aufruft (zum Beispiel die Funktion auf unserer Webseite) und sehen folgenden Kompilierungsfehler:

Fehler, Funktion FFT@IAAAAAAAMAAABAAAMAAABAAA wurde aufgerufen, konnte aber während des Verknüpfens nicht gefunden werden.

Das bedeutet, dass der Körper der FFT-Funktion nicht gefunden wurde. Mit den folgenden zwei Schritten können Sie das Problem lösen:

  1. Stellen Sie sicher, dass Ihr Programm die Datei fft_utils.h enthält, die sich unter <Origin Program Folder>\OriginC\System befindet.
  2. Fügen Sie die Datei <Origin Program Folder>\OriginC\Originlab\fft_utils.c zum aktuellen Arbeitsbereich hinzu.

Stichwörter: Arbeitsbereich, Fourier-Transformation
