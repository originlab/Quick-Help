FAQ-478 Wie richte ich mein Arbeitsblatt ein, so dass es immer m x n Zellen hat?

Resize-Wks

Letztes Update: 04.02.2015

Öffnen Sie den Dialog Arbeitsblatteigenschaften durch Auswahl von Format: Worksheet oder klicken Sie alternativ mit der rechten Maustaste rechts neben die letzte Spalte im Arbeitsblatt und wählen Sie Einstellungen. Sie können auch F4 drücken. Setzen Sie auf der Registerkarte Größe die Zeilennummer und Spaltennummer. Deaktivieren Sie auf der Registerkarte Verschiedenes das Kontrollkästchen Automatisches Hinzufügen von Zeilen und aktivieren Sie das Kontrollkästchen Blatt auf Fenster anpassen. Klicken Sie auf OK, um den Dialog zu schließen, und dann die Größe des Arbeitsmappenfensters zu ändern.
