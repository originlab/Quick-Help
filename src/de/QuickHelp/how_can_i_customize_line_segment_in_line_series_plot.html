FAQ-807 Wie passe ich die Liniensegmente in einem Kurvenschardiagramm benutzerdefiniert an?

Customize-Line-Series

Letztes Update: 03.01.2016

Das Kurvenschardiagramm erzeugt ein verborgenes Arbeitsblatt, das die tatsächlichen Quelldaten enthält. Um die Liniensegmente in einem Kurvenschardiagramm benutzerdefiniert anzupassen (Farbe der Liniensegmente oder Linienstil von jedem Segment ändern etc.), können Sie die Indexspalte(n) zu dem verborgenen Arbeitsblatt hinzufügen und die Indexspalte(n) im Dialog Details Zeichnung festlegen. Wenn Sie zum Beispiel die Farben der Liniensegmente ändern möchten:

  1. Klicken Sie zum Öffnen des Dialogs Details Zeichnung doppelt auf das Kurvenschardiagramm. Klicken Sie auf die Schaltfläche Arbeitsmappe unten im rechten Bedienfeld. Eine Arbeitsmappe wird geöffnet.
  2. Fügen Sie in der geöffneten Arbeitsmappe am Ende eine neue Spalte ein. Füllen Sie die Indexfarben für jedes Paar von 2 Zeilen.
  3. Kehren Sie zum Kurvenschardiagramm zurück, Öffnen Sie den Dialog Details Zeichnung erneut. Wählen Sie auf der Registerkarte Linie Farbe: Nach Punkten: Index: <die neue Spalte, die Sie gerade hinzufügt haben>.
  4. Passen Sie die Symbolfarbe oder andere Merkmale nach Bedarf an.
