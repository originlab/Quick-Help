FAQ-460 Wie konvertiere ich ein Arbeitsblatt mit Daten, die dem Stil einer Matrix entsprechen, in eine Matrix oder ein XYZ-Arbeitsblatt?

Convert-VirtualMatrixWks-to-XYZ

Letztes Update: 04.02.2015

Wenn Sie Ihre Arbeitsblattdaten direkt in eine Matrix konvertieren und die Abstände Ihrer X- und/oder Y-Daten nicht äquidistant sind, kann es vorkommen, dass der Befehl Worksheet: In Matrix konvertieren: Direkt fehlschlägt, falls die Schwankung in Ihren X- oder Y-Daten über einem festgelegten Toleranzwert liegt.

In diesem Fall müssen Sie zunächst den Befehl Worksheet: In XYZ konvertieren verwenden. Sie erhalten ein neues Blatt mit drei Spalten: X Y Z. Anschließend können Sie die XYZ-Daten in eine Matrix konvertieren, indem Sie den Befehl Worksheet: In Matrix konvertieren: XYZ Gridding... verwenden. Der Gridding-Prozess erstellt neue, interpolierte Daten in einem X/Y-Gitter mit gleichmäßigen Abständen.

Sie können das Interpolieren der Daten vermeiden, die Z-Spalte auswählen und ein Konturdiagramm direkt aus den XYZ-Daten erstellen, ohne die Daten in eine Matrix konvertieren zu müssen.

Schlüsselwörter: zufällig, ungleich, getrennt, nichtlinear, Schritt, Abstand, Gridding, gleichmäßig, Toleranz, äquidistant
