FAQ-871 Hat Origin eine "IF"-Funktion?

Set-Col-IF-function

Letztes Update: 06.12.2018

Ja, seit Origin 2019 haben wir eine if()-Funktion hinzugefügt.

Angenommen, Spalte A enthält Werte mit einigen fehlenden Werten. Geben Sie if(A=1/0, mean(A), A) in Zelle F(X) von Spalte B ein. Dann wird B mit Werten aus Spalte A gefüllt, falls es Werte in A gibt; Spalte B wird mit dem Mittelwert von Spalte A gefüllt, wenn die entsprechende Zeile in A gefüllt mit fehlenden Werte oder leer ist.

Vor Origin 2019 verwenden Sie bitte den Operator "?".

Um die gleiche Berechnung wie oben durchzuführen, geben Sie A=1/0? ein. Mean(A):A in Zelle F(x) der Spalte B


Ein kompliziertes Beispiel: Ich habe eine Reihe von Zeilennummern in Spalte A und möchte Spalte B mit konditionalen Sequenzen füllen, so dass, wenn der Wert in col(A)

  • 1<=col(A)[i]<=7 ist, dann wird col(B)[i] mit "Low" gefüllt.
  • 7<col(A)[i]<=17 ist, dann wird col(B)[i] mit "Mid" gefüllt.
  • 17<col(A)[i] ist, dann wird col(B)[i] mit "High" gefüllt.

Geben Sie A>=1&&A<=7?"Low":(A>7&&A<=17?"Mid":"High") in Zelle F(x) von Spalte B ein.


Siehe auch Vergleich von Excel- und Origin-Funktionen.

Schlüsselwörter:IF-Funktion, Werte festlegen unter Bedingung, konditionaler Operator
