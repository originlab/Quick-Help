Wenn Sie Origin neu kaufen und noch nicht registriert haben

Wenn Sie einen Einzellizenz haben

Nach dem Kauf erhalten Sie eine E-Mail mit dem Betreff, die Ihre Origin-Seriennummer und einen Link zur Registrierung enthält.

  1. Klicken Sie auf den Link zur Registrierung. Sie gelangen zur Origin-Webseite. Melden Sie sich an.
  2. Nach der Anmeldung auf der Webseite werden der Code Ihrer Produktregistrierung und Ihre Seriennummer angezeigt. Klicken Sie auf die Schaltfläche Registrieren, um diese Seriennummer zu registrieren und Ihre Produktschlüssel zu erzeugen.

Wenn Sie statt des Links den Registrierungscode haben,

  1. melden Sie sich auf der Webseite originlab.com an.
  2. Klicken Sie auf der Willkommensseite auf den Link Mein Produkt registrieren - Version 2017 (9.4).
  3. Geben Sie auf der Seite Mein Produkt registrieren den Registrierungscode ein und klicken Sie auf die Schaltfläche Weiter, um diese Seriennummer zu registrieren und Ihre Produktschlüssel zu erzeugen.

Weitere Einzelheiten finden Sie auf dieser Seite.

Wenn Sie der Administrator einer Gruppenlizenz sind
  1. Als Administrator sollten Sie die erste Person sein, die die Seriennummer registriert und den Mechanismus zum Verteilen der Produktschlüssel einrichtet. Bitte lesen Sie auf dieser Seite die einzelnen Schritte zum Registrieren und Verteilen der Produktschlüssel nach.
Wenn Ihre Organisation eine Gruppenlizenz kauft und Sie berechtigt sind, diese zu nutzen
  1. Bitte wenden Sie sich an den Origin-Administrator oder die IT-Abteilung Ihrer Organisation, um den Produktschlüssel zu erhalten.

Wenn Ihre Seriennummer bereits registriert ist

Besuchen Sie diese Seite.

Wenn Sie ein Upgrade von Origin 2016 oder älter auf 2017 vornehmen

  1. Melden Sie sich auf der Webseite originlab.com an.
  2. Klicken Sie auf der Willkommensseite auf den Link Wartungscenter.
  3. Klicken Sie im Wartungscenter auf den Link Registrierung für ein automatisches Upgrade auf die neueste Version.
  4. Auf der Seite mit der Liste der Registrierungen werden alle Seriennummern aufgeführt, die für ein Upgrade zur Verfügung stehen. Wählen Sie die gewünschte Seriennummer aus und klicken Sie auf den Link Produkt registrieren.
  5. Ihr Registrierungscode und Ihre Seriennummer werden auf der aufgerufenen Webseite angezeigt. Klicken Sie auf die Schaltfläche Registrieren, um zu registrieren und Produktschlüssel zu erzeugen.

Wenn Sie ein Upgrade von Origin 2017 auf 2018 oder höher vornehmen

Ab Origin 2018 haben verschiedene Versionen die gleiche Lizenz. Das bedeutet, dass die neuere Version alle Lizenzen der alten Version mitsamt ihren Produktschlüsseln und ihrem Nutzungsstatus übernehmen wird.

Wenn Sie eine Einzellizenz haben

Der Produktschlüssel der neuen Version ist der gleiche wie der Schlüssel für Origin 2017.


Wenn Sie der Administrator einer Gruppenlizenz sind

Für Ihre Seriennummer wird automatisch ein Upgrade auf die neuere Version durchgeführt. Alle lizenzverbundenen Einstellungen der vorherigen Version (einschließlich des Mechanismus zum Verteilen der Produktschlüssel und der E-Mail-Fragmente) werden auf die neuere Version angewendet. Sie können Ihre Anwender über das Upgrade informieren, sobald Sie bereit sind. Im Folgenden ist beschrieben, wie Sie die Produktschlüssel finden.

  1. Melden Sie sich auf der Webseite originlab.com an.
  2. Klicken Sie auf den Link Meine Lizenz(en) unter Ihrem Portal.
  3. Wählen Sie den Link der gewünschten Seriennummer aus. Beachten Sie, dass für seine automatisch ein Upgrade durchgeführt wird.
  4. Klicken Sie auf den Link Alle Produktschlüssel. Sie können dann die Produktschlüsselinformationen auf der aufgerufenen Seite einsehen.


Wenn Sie der Anwender einer Gruppenlizenz sind

Der Produktschlüssel der neuen Version ist der gleiche wie der Schlüssel für Origin 2017.
