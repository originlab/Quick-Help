FAQ-638 Wie erstelle ich ein eingesetztes Diagramm?

create-inset-graph

Letztes Update: 08.01.2019

Klicken Sie auf die Schaltfläche Eingefügtes Diagramm hinzufügen oder Eingefügtes Diagramm mit Daten hinzufügen auf der Symbolleiste Diagramm. Verwenden Sie dann das Hilfsmittel Achsenskalierung vergrößern, um einen anderen Datenbereich zu zeigen.

Eine andere Möglichkeit (mit mehr Schritten) dies zu tun, besteht in der Auswahl von Einfügen: Neuer Layer (Achsen): Untere X + Linke Y, um einen zweiten Layer hinzuzufügen. Wählen Sie den neuen Layer und verändern Sie seine Größe. Positionieren Sie ihn am der gewünschten Stelle im ersten Layer und nehmen Sie die notwendigen Änderungen in dem gezeichneten Y-X-Bereich vor usw.

Standardmäßig blockieren eingesetzte Diagramme nicht die Zeichnungen in den ursprünglichen Diagrammen. Wenn Sie die Sicht auf ursprüngliche Zeichnungen blockieren möchten, so dass diese nicht durchzusehen sind, wählen Sie Format: Seite im Menü. Aktivieren Sie auf der Registerkarte Layer die Option Layer nacheinander zeichnen in der Gruppe Zeichenreihenfolge. In älteren Versionen befindet sich dieses Bedienelement auf der Registerkarte Allgemeines.

Schlüsselwörter:Eingesetztes Diagramm, durchsehen, sich überschneiden, Layer
