Beschriftung von Achsenhilfsstrichen benutzerdefiniert anpassen

Um die Schriftart, Gewichtung und Farbe der Beschriftungen von Achsenhilfsstrichen zu ändern, können Sie doppelt auf die Beschriftungen der Achsenhilfsstriche klicken oder im Menü Format: Achsen: X/Y/Z-Achse wählen. Gehen Sie zur Registerkarte Beschriftungen der Hilfsstriche und ändern Sie die Einstellung auf der Registerkarte Format.


Wenn Sie nur ein Mal auf die Beschriftung der Hilfsstriche klicken, können Sie die Symbolleiste Format verwenden, um Schriftart, Gewichtung oder Farbe zu ändern.

Beschriftung von einzelnen Hilfsstrichen benutzerdefiniert anpassen

Methode 1: Spezielle Hilfsstriche im Dialog Achsen verwenden

Ab Origin 2015 können Sie auch spezielle Hilfsstriche verwenden, um einzelne Hilfsstrichsbeschriftungen benutzerdefiniert anzupassen. Wenn Sie z.B. ein Datenarbeitsblatt wie das folgende haben:

A(X) B(Y)
a 1
b 2
c 3
  1. Kopieren Sie die obenstehenden Daten und fügen Sie sie in Origin ein. Markieren Sie diese zwei Spalten und wählen Sie Zeichnen: Linie: Liniendiagramm, um ein Liniendiagramm zu zeichnen.
  2. Klicken Sie doppelt auf die X-Achse, um den Dialog Achsen zu öffnen. Wechseln Sie zur Registerkarte Spezielle Hilfsstriche und klicken Sie dann doppelt auf die 3. Zelle der Spalte col(Bei Achsenwert). Geben Sie 2,35 ein. Klicken Sie doppelt auf die 3. Zelle der Spalte col(Beschriftung) und geben Sie spezielle Hilfsstriche in der Zelle ein.
  3. Klicken Sie auf die Schaltfläche Details..., um den Dialog Spezielle Hilfsstriche zu öffnen. Wählen Sie 2,35 im Feld Spezielle Hilfsstriche bei. Gehen Sie zur Registerkarte Beschriftungsformat, um die Beschriftung der speziellen Hilfsstriche zu formatieren.
Methode 2: Texte aus Arbeitsblatt verwenden

Wenn Zellen in einem Arbeitsblatt direkt ausgewählt werden, um sie zu formatieren, z.B. kursiver oder farbiger Text, und anschließend ein Diagramm gezeichnet wird, dessen Hilfsstrichs- oder Textbeschriftungen mit dem formatierten Text erstellt werden, dann fehlt die Formatierung in dem Diagramm. Damit die Textformatierung auch bei Diagrammen angewendet wird, müssen Sie den Modus Rich Text aktivieren, bevor Sie die Zellen bearbeiten.

Wenn Sie z.B. ein Arbeitsblatt haben, das Daten wie im Folgenden enthält:

A(X) B(Y)
red
apple		 1
orange
orange		 2
green
banana		 3

Sie möchten, dass die Beschriftungen der X-Achsenhilfsstriche die Textfarbe wiedergeben.

  1. Wählen Sie Format: Worksheet, um den Dialog Arbeitsblatteigenschaften zu öffnen.
  2. Setzen Sie auf der Registerkarte Format Anwenden auf auf Daten und aktivieren Sie dann das Kontrollkästchen Rich Text.
  3. Klicken Sie doppelt in die Arbeitsblattzelle, so dass Sie den Text in der Zelle bearbeiten können. Verwenden Sie die Schaltfläche Schriftfarbe auf der Symbolleiste Format, um die Textfarbe festzulegen. Markieren Sie z.B. das Wort Red und wählen Sie dann Rot über die Schaltfläche Schriftfarbe.
  4. Markieren Sie die Y-Spalte und wählen Sie den Diagrammtyp Ihrer Wahl oder gehen Sie zurück zu einem vorhandenen Diagramm und wählen Sie Fenster: Aktualisieren, um es zu aktualisieren.

Hinweis 1: Nachdem Rich Text für das Arbeitsblatt aktiviert wurde, müssen Sie doppelt in die Zelle klicken und den Text markieren, anstatt einmalig zu klicken, um die Zelle für die Formatierung auszuwählen.

Hinweis 2: Wenn Sie Text im Modus Rich Text formatieren und diesen dann als Datenbeschriftung verwenden, greift die Formatierung auch beim Diagramm.

Hinweis 3: Der mehrzeilige Text in der Spalte A(X) wird als zwei Zeilen angezeigt, wenn die Werte als Beschriftungen der X-Achsenhilfsstriche angezeigt werden. Wenn Sie die Werte in einer Zelle eingeben und mehrere Zeilen erstellen möchten, verwenden Sie Strg + Enter, um eine neue Zeile zu beginnen. Sie können auch die Option Textumbruch für die Beschriftung der Hilfsstriche verwenden. Einzelheiten dazu lesen Sie auf dieser Seite der Schnellhilfe.
