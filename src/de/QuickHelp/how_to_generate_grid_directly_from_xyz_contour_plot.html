FAQ-758 Wie erzeuge ich das Gitternetz direkt aus einem XYZ-Konturdiagramm?

Generate-grid

Letztes Update: 14.08.2015

Klicken Sie mit der rechten Maustaste auf das Konturdiagramm, das Sie aus XYZ-Daten erstellt haben, und wählen Sie Gitternetz erzeugen, um den Dialog zu öffnen. Legen Sie in dem aufgerufenen Dialog Gitternetz erzeugen: xyzgrid die Anzahl der Gitternetzpunkte und die Bereiche entlang der X- und Y-Bereiche fest und wählen Sie eine Datenausgabeform, um die interpolierten Konturwerte in einem Arbeitsblatt oder einer Matrix auszugeben.

Beachten Sie, dass Konturwerte für einen Bereich außerhalb des Eingabedatenbereichs als fehlende Werte behandelt werden.

Einzelheiten finden Sie auf dieser Seite.

Origin-Version mind. erforderlich: 2016 SR0
