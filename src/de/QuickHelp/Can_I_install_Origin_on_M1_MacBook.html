FAQ-1099 Kann ich Origin auf einem M-X MacBook (M1, M2 etc.) installieren?

Install-Origin-on-Mac-M1

Letztes Update: 19.02.2024

Ja, Sie können Origin auf einem Mac mit einem Apple M-X Chip (M1, M2 etc.) installieren und ausführen.

  • Sie müssen eine mit einem Apple Chip kompatible virtuelle Maschinensoftware verwenden, um Origin auf einem Mac auszuführen. Wir empfehlen Parallels Desktop.
  • Da Origin Windows-basiert ist, müssen Sie außerdem Windows auf der virtuellen Maschine installieren.
  • Da der M1-/M2-/M3-Chip ARM-basiert ist, müssen Sie Windows 11 auf ARM installieren.

Sollte Origin nach Befolgen der obigen Anweisungen noch immer Bei Absturz beim Start oder während des Arbeitens oder anderen ungewöhnlichen Verhaltensweisen, wie sehr geringe Geschwindigkeit, Rechtsklick funktioniert nicht etc.:Lesen Sie bitte diese Seite, um das Problem zu beheben.

Der Kohärenzmodus im Parallels Desktop kann zu unerwarteten Problemen bei Origin führen. Falls Sie das Gefühl haben, dass Origin nicht stabil ist, verlassen Sie den Kohärenzmodus und versuchen Sie es erneut.
