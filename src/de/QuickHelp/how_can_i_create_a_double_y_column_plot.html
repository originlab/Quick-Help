FAQ-622 Wie kann ich eine Säulendiagramm mit doppelter Y-Achse erstellen, so dass die zwei Säulenscharen jeweils ihre eigene Y-Achse haben?

double-Y-column-plot

Letztes Update: 04.09.2022

Wenn Sie XYY-Daten haben und ein Säulendiagramm mit doppelter Y-Achse erstellen wollen (d.h. gemeinsame X-Achse, individuelle Y-Achsen und Anzeige nebeneinander), können Sie den Schritten unten folgen:

Methode 1: Ein Layer

Markieren Sie die XYY-Daten und wählen Sie Zeichnen: Mehrere Felder/Achsen: Säulen mit doppeltem Y, um direkt ein Säulendiagramm mit doppelter Y-Achse zu erstellen. Diese zwei Säulendiagramme werden in einen Layer gezeichnet, basieren aber auf unterschiedlichen Y-Achsen.

Methode 2: Zwei Layer

Markieren Sie die XYY-Daten und wählen Sie Zeichnen: Mehrere Felder/Achsen: 2Y, Säulen, um direkt ein Säulendiagramm mit 2 Y zu erstellen. Diese zwei Säulendiagramme werden in unterschiedliche Layer mit unterschiedlicher Y-Achse gezeichnet.
