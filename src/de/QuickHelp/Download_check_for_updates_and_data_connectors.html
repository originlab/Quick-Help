FAQ-1003 Warum kann Origin keine Verbindung zum Internet herstellen, um nach Updates etc. zu suchen?

Download-updates-data-connectors

Letztes Update: 30.01.2023

Wenn Sie Probleme in einem der folgenden Bereichen erleben:

  • Hilfedateien erhalten/&Nach Updates suchen ...
  • App-Center und Apps herunterladen bzw. installieren
  • Lernzentrum aktualisieren
  • Datenkonnektoren zum Importieren von Daten aus dem Internet verwenden


Es kann mit der Verwendung eines Proxy-Servers in Ihrem Netzwerk zu tun haben. Viele Organisationen setzen Proxy-Server ein, um bestimmte Netzwerke durch ein kontrolliertes Portal zu routen. Standardmäßig kann es sein, dass Origin nicht den Einstellungen des Proxy-Servers entspricht und daher von den laufenden, mit dem Internet verbundenen Aktivitäten blockiert wird, die für verschiedene Funktionen erforderlich sind.


Seit Origin 2019 können Sie die Proxy-Server-Einstellungen von Origin ändern, um zu prüfen, ob die Probleme damit behoben werden:

  1. Wählen Sie im Menü Einstellungen: Optionen (Hilfsmittel: Optionen in Origin 2019).
  2. Navigieren Sie im Dialog Optionen zur Registerkarte Sonstiges.
  3. In der Gruppe Proxy gibt es drei Optionen:
    • Direkte Verbindung umgeht jegliche Proxy-Server - Origin versucht, eine direkte Verbindung zum Ziel herzustellen.
    • Systemkonfiguration verwenden setzt die gleichen Proxy-Server-Einstellungen ein, die üblicherweise von Webbrowsern und vielen anderen netzwerkbasierten Softwareprogrammen auf Ihrem System verwendet werden.
    • HTTP-Proxy-Server verwenden ermöglicht eine fest programmierte Proxy-Server-IP-Adresse und einen festzulegenden Port.
      Wenn Sie einen Proxyserver verwenden, der eine Authentifizierung erfordert, verwenden Sie die Syntax: 
      [username]:[password]@[server name|server ip]:[port]

      Wenn der Benutzername oder das Passwort "prozentkodierende, reservierte Zeichen" enthält, ersetzen Sie sie mit prozentkodierenden Zeichen. (Hinweis: Prozentkodierung wird seit Origin 2021b unterstützt). Wenn zum Beispiel Benutzername = user@a.com, Passwort = pwd, Servername = proxy und Port = 3129, dann sollten Sie schreiben:

      user%40a.com:pwd@proxy:3129


Es wird sehr empfohlen, dass Sie zuerst die Option Systemkonfiguration verwenden auswählen, um zu sehen, ob die Probleme damit behoben werden (Sie müssen Origin neu starten, damit die neue Einstellung wirksam werden kann). Sollten die Probleme noch nicht behoben sein, müssen Sie vielleicht eine fest programmierte Proxy-Server-Adresse und einen Port in HTTP-Proxy-Server verwenden eingeben (auch hier ist hinterher ein Neustart erforderlich). Sie sollten Ihre IT-Abteilung zurate ziehen, um die geeignete Adresse und den zu verwendenden Port in Erfahrung zu bringen.

Sollten diese Änderungen in Origin noch immer keine Lösung darstellen, könnte Ihre Windows- oder Antivirus-Firewall Origin blockieren. In diesem Fall sollten Sie die Dokumentation für die relevante Firewall zur Hand nehmen, um nach Methoden zu suchen, wie Sie Origin den Zugang ermöglichen können.
