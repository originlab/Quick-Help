FAQ-22 Wie erfahre ich, ob ein Service Release für mein Origin verfügbar ist?

Check-Available-Patch

Letztes Update: 30.01.2023

Innerhalb einer Version veröffentlicht OriginLab üblicherweise ein oder mehrere Service Release, um Fehler zu beheben, die in der Release-Version gefunden wurden. Dieses Service Release wird als Patch bereitgestellt. Üblicherweise ist auch ein neuer Installer basierend auf diesem Service Release verfügbar.

Wichtig: Um den Patch auszuführen, müssen Sie Origin einmal ausgeführt haben, wie während des Lizenzierungsprozesses oder nachdem Origin lizenziert wurde.

Zwei Optionen, ein Service Release zu erhalten:

  • Hilfedateien erhalten/Nach Updates suchen
Wenn Ihr Origin registriert ist, können Sie Hilfe: Hilfedateien erhalten/Nach Updates suchen wählen, um zu prüfen, ob ein Patch für Ihre Origin-Version und den Build verfügbar ist. Sollte ein Patch verfügbar sein, wird es im Dialog Nach Updates suchen angezeigt, und Sie können auswählen, das Patch herunterzuladen. Am Ende des Downloadprozesses werden Sie gefragt, ob Sie Origin schließen möchten, um das Patch auszuführen, oder ob Sie das Patch später ausführen möchten. Es werden Informationen bereitgestellt, wie das Patch zu einem späteren Zeitpunkt ausgeführt wird.
  • Laden Sie das Patch von der OriginLab-Webseite herunter.
OriginLab bietet Patches auch zum Download im Support-Bereich der Webseite von OriginLab an. Bevor Sie prüfen, ob ein Patch verfügbar ist, führen Sie Origin aus und wählen Sie Hilfe: Über Origin. Beachten Sie die Version und die vollständige Build-Nummer im Dialog Über Origin. Die Downloadseite der Patches auf der Webseite von OriginLab bietet Anweisungen zum Speichern und Anwenden des Patches. Es wird auch ein Link bereitgestellt, über den Sie die Release Notes des Patches einsehen können.

Hinweis: Wenn Sie eine Gruppenlizenz haben und ein Service Release verteilen möchten, lesen Sie bitte diese Seite.
