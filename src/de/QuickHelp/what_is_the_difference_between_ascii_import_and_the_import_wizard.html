FAQ-411 Wo liegt der Unterschied zwischen dem ASCII-Import und dem Importassistenten?

ASCIIImport-ImpWiz

Letztes Update: 06.01.2019

Der Importassistent unterstützt den Import von ASCII- und einfachen Binärdateien (ein optionaler Headerabschnitt gefolgt von einer sequenziellen binären Struktur). Der Importassistent gibt Ihnen viele Optionen zum Importieren und bietet eine Importvorschau, so dass Sie Ihre Einstellungen allmählich anpassen können. Der Importassistent unterstützt außerdem den Import von Daten über die Zwischenablage. Der Assistent kann zum Erstellen von benutzerdefinierten Filterdateien für jeden Ihrer Dateitypen verwendet werden. Sobald Sie Ihre(n) Filter erstellt haben, können nachfolgende Dateien einfach per Drag&Drop in den Origin-Arbeitsbereich importiert werden.

Das Menü Daten: Aus Datei importieren: Einzelnes ASCII eignet sich gut zum Importieren von nicht sehr komplexen ASCII-Dateien. Sie möchten diese Menüoption vielleicht am ehesten verwenden, wenn Ihre Datei nur aus wenigen Headerzeilen besteht (einer kurzen Beschreibung für die Datei und dann Namen und Einheiten für die Spalten) und die Datenspalten ordentlich getrennt sind. Sie können auch den Dialog Optionendialog zeigen öffnen und einige Optionen ähnlich wie im Importassistent festlegen, dies jedoch ohne Vorschau.

Das Menü Daten: Aus Datei importieren: Mehrere ASCII funktioniert genauso wie bei Einzelnes ASCII, allerdings ist Optionendialog zeigen standardmäßig aktiviert.

Der Dialog für die Importoptionen von Einzelnes und Mehrere ASCII kann Ihre Einstellungen in dem Arbeitsblatt oder als eine Designdatei speichern.

Im Ausklappmenü des Dreiecks neben dem Dialogdesign werden die Einstellungen in dem Blatt gespeichert, wenn Sie die Option Speichern in <Blatt> wählen. Sie können die Arbeitsmappe als Vorlage speichern. Origin verwendet diese Einstellungen dann während des ASCII-Imports, auch wenn Sie den Optionendialog nicht öffnen.

Wenn Sie die Einstellungen als Design speichern, werden sie unabhängig von jeder Vorlage extern gespeichert. Sie können viele solcher Designs wie diese lokal speichern und das richtige Design für die aktuelle Datei im Ausklappmenü des Dialogs impASC vor dem Import wählen. Sie können auch über das Menü Daten: Zuletzt verwendete Importe gehen, um diese Designs auszuwählen.
