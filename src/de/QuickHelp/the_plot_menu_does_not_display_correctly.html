FAQ-853 Was tue ich, wenn das Menü Zeichnen nicht korrekt angezeigt wird?

The-Plot-Menu-does-not-Display-Correctly

Letztes Update: 14.07.2018

Es gibt einige Berichte, dass es beim Anzeigen des Menüs Zeichnen (das auf Symbolen basierte Menü) Probleme gab. Dieses Menü wurde in Origin 2017 hinzugefügt. Die Probleme können mit dem kürzlichen Windows 10 Fall Creators Update, 2017 (Version 1709) zusammenhängen. Stellen Sie sicher, dass Ihr Windows 10 nicht diese Version hat.

Microsoft hat nach dem Windows 10 Fall Creators Update weitere Updates herausgegeben. Das Anwenden dieser Updates löst das Problem der Langsamkeit möglicherweise.

  • Gehen Sie zu Windows Update und wenden Sie alle verfügbaren Updates an.


Sie können auch zum alten Zeichenmenü zurückkehren:

  • Öffnen Sie das Skriptfenster (Fenster: Skriptfenster), geben Sie folgenden Befehl ein und drücken Sie dann Enter:
@DPPM = 1

Sollte dies das Problem lösen, können Sie das Popup-Menü Zeichnen permanent über den Dialog Systemvariablen festlegen deaktivieren.


Siehe auch: LabTalk-Systemvariable @DPPM
