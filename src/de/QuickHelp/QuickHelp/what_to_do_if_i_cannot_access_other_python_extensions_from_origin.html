FAQ-406 Was kann ich tun, wenn der Zugriff auf weitere Python-Erweiterungen von Origin aus nicht möglich ist?

 

Letztes Update: 04.02.2015

access-python-extensions

Das eingebettete Python in Origin umfasst alle Basismodule. Wenn Sie allerdings weitere Python-Erweiterungen installiert haben (z.B. PyQt4) und auf diese von dem in Origin eingebetteten Python aus zugreifen möchten, müssen Sie immer den entsprechenden Installationsordner zu der Systempfadliste hinzufügen, bevor Sie sie importieren.

Ein Beispiel zum Hinzufügen des Installationsordners zur Systempfadliste finden Sie im Abschnitt A Note to Use Python Extensions im Kapitel Working with Python in der LabTalk-Anleitung.

Schlüsselwörter:Python, Python Binding, Python-Erweiterung, PyQt4, Systempfad
