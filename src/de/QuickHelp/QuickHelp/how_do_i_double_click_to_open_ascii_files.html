FAQ-415 Wie wende ich den Doppelklick an, um ASCII-Dateien in Origin zu öffnen?

double-click-open-ASCII-files

Letztes Update: 20.07.2017

Sie können doppelt auf sie klicken, um eine ASCII-Datei in Origin zu öffnen, indem Sie Origin als das Standardprogramm einrichten.

  1. Klicken Sie im Windows Explorer mit der rechten Maustaste auf eine ASCII-Datei und wählen Sie Öffnen mit: Programm auswählen.
  2. Navigieren Sie, um Origin##.exe im Origin-Programmordner auszuwählen. Aktivieren Sie Dateityp immer mit dem ausgewählten Programm öffnen und klicken Sie auf OK.

 

Für Origin 2017 und ältere Versionen müssen Sie eine Stapeldatei einrichten, um eine ASCII-Datei in Origin durch einen Doppelklick zu öffnen.

  1. Öffnen Sie das Skriptfenster über Fenster: Skriptfenster.
  2. Wählen Sie Datei: Neu im Menü des Skriptfensters und geben Sie das folgende Skript ein:
start C:\Progra~1\OriginLab\<Program Folder>\<Version.exe> -r {impasc %1;};
Hinweis: C:\Progra~1\OriginLab\<Program Folder>\ ist der Ordner, in dem Ihre Origin-Version installiert ist ("C:\Progra~1" sollte für "C:\Program Files" verwendet werden). Wenn Sie zum Beispiel die Origin-Version 2016 haben, würden Sie eingeben: "start C:\Progra~1\OriginLab\Origin2016\Origin93.exe -r {impasc%1;};"
  1. Wählen Sie Datei: Speichern unter und legen Sie den Dateityp: auf Alle Dateien (*.*) fest. Benennen Sie die Datei mit einer .BAT-Erweiterung um (z.B. imp2org.bat).
  2. Klicken Sie im Windows Explorer mit der rechten Maustaste auf eine ASCII-Datei und wählen Sie Öffnen mit: Standardprogramm auswählen.
  3. Navigieren Sie zu der .BAT-Datei, die Sie in Schritt 3 gespeichert haben, und aktivieren Sie das Kontrollkästchen Dateityp immer mit dem ausgewählten Programm öffnen. Klicken Sie dann auf OK.
