FAQ-141 Wie kann ich Layer zu einer Diagrammseite hinzufügen und anordnen?

Add-and-Arrange-Layers

Letztes Update: 03.07.2018

Verwenden Sie das Dialogfeld Layer Management, indem Sie im Menü Grafik: Layer-Verwaltung wählen.


Auf der Registerkarte Hinzufügen können Sie einen neuen Layer zu Ihrem Diagramm hinzufügen.

Wählen Sie in der Auswahlliste den Typ und klicken Sie anschließend auf die Schaltfläche Anwenden. Der neue Layer wird im Dialog Layerverwaltung im Vorschaubereich angezeigt.

Hinweis: Um Layer zu löschen, markieren Sie den oder die Layer im Listenfeld Layerauswahl, öffnen durch Klicken mit der rechten Maustaste das Kontextmenü und wählen Layer löschen.

 

Auf der Registerkarte Anordnen können Sie Anordnungen der Felder erstellen.

  • Legen Sie die Anzahl der Zeilen und Anzahl der Spalten fest, um alle Layer in Ihrem Diagramm anzuordnen.
  • Zusätzliche Layer werden automatisch hinzugefügt, wenn das Kontrollkästchen Weitere Layer für das Gitter hinzufügen aktiviert ist.
  • Um nur die ausgewählten (markierten) Layer in der Layerauswahl auf der linken Seite anzuordnen, aktivieren Sie das Kontrollkästchen Ausgewählte Layer anordnen.
  • Legen Sie die Abstände (in % der Seitengröße) fest und klicken Sie anschließend auf die Schaltfläche Anwenden. Die Diagrammvorschau wird aktualisiert. Klicken Sie auf Rückgängig, wenn Sie mit der Anordnung nicht zufrieden sind.

Hinweis: Alle Operationen, die auf das Vorschaudiagramm angewendet wurden, werden nur auf das aktive Diagramm angewendet, nachdem Sie auf OK geklickt haben.

Schlüsselwörter: ausrichten, verschieben, vertauschen, Größe ändern, anpassen
