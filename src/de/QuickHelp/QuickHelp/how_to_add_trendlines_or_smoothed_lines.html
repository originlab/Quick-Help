FAQ-947 Wie füge ich eine Trendlinie oder geglättete Linie für ein 2D-Punktdiagramm hinzu?

Add-Trendlines

Letztes Update: 18.03.2021

Wenn Sie ein 2D-Punktdiagramm haben und eine Trendlinie oder eine geglättete Linie einfügen möchten, können Sie

  • Klicken Sie auf die Zeichnung und wählen Sie die Schaltfläche Angepasste Kurve hinzufügen auf der Minisymbolleiste. Wählen Sie Lineare Anpassungskurve in der Auswahlliste.

ODER

  • die App 2D Smoother verwenden. Bitte beachten Sie, dass wir auch die App 3D Smoother zur Verfügung stellen, die zum Glätten von 3D-Daten aus XYZ-Spalten oder einer Matrix verwendet werden kann.

 

Wenn Sie eine geglättete Linie zu einem 2D-Punktdiagramm hinzufügen möchten, können Sie auch

  • das Standardhilfsmittel Glätten verwenden.

Schlüsselwörter:Trendlinie, geglättete Linie, Glättung
