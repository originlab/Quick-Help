FAQ-777 Wie kann ich Daten mit höchster Präzision mit Hilfe von Strg + C kopieren?

Full-Precision-Copy

Letztes Update: 09.08.2017

Beim Kopieren und Einfügen von Daten ist es üblich, die Tastenkombinationen Strg + C (Kopieren) und Strg + V (Einfügen) zu verwenden. Dies erhält aber möglicherweise die numerische Präzision nicht bei, wenn Sie Daten in externen Anwendungen einfügen. Wenn Sie die volle numerische Präzision erhalten möchten, verwenden Sie die Tastenkombination Strg + ALT + C (Kopieren, volle Genauigkeit) (Einfügen gelingt über die Standardtasten Strg + V).

Beachten Sie, dass Strg + C problemlos funktioniert, wenn Sie Daten nur zwischen Origin-Arbeitsblättern kopieren und einfügen. Die Daten werden mit höchster Präzision kopiert und eingefügt. Wenn Sie jedoch in eine externe Anwendung kopieren-einfügen und die Tastenkombination Strg + ALT + C etwas mühsam finden, können Sie Origin modifizieren, so dass die einfachere Kombination Strg + C Origin-Daten mit höchster Präzision kopiert. Dazu ändern Sie den Standardwert der Systemvariablen @CWACC.

  • Um das Standardverhalten nur für die aktuelle Origin-Sitzung zu ändern, öffnen Sie das Skriptfenster, geben Sie Folgendes ein und drücken Sie dann Enter:
@CWACC=1
Mit dieser Änderung wird Strg + C das Tastaturkürzel zum Kopieren von Daten mit höchster Präzision in der Arbeitsmappe und Strg + Alt + C kopiert die Daten unter Verwendung der angezeigten Stellen.
  • Damit dieses Verhalten bei jedem Start von Origin zum Tragen kommt, lesen Sie bitte diese FAQ.
