FAQ-680 Welche Anzahl von Zeilen ist maximal in einem Arbeitsblatt/einer Matrix zugelassen?

Letztes Update: 11.03.2015

Maximum-Number-Rows

Die maximale Anzahl der Zeilen in einem Arbeitsblatt/Matrixblatt beträgt 90.000.000.
