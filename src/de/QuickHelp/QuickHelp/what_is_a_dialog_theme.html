FAQ-351 Was ist ein Dialogdesign?

Dialog-Theme

Letztes Update: 06.08.2018

Dialogdesigns bieten Ihnen die Möglichkeit, Einstellungen in dem Dialog extern für zukünftige Verwendungen zu speichern. Sie können mehrere Designdateien für einen Dialog speichern und auf diese Weise den Dialog benutzerdefiniert anpassen, damit es in verschiedenen Modi mit verschiedenen Daten verwendet werden kann. Designdateien werden im Unterordner der Designs in Ihrem Anwenderdateiordner gespeichert.

Klicken Sie in einem Dialog, der Designs unterstützt, auf die dreieckige Schaltfläche rechts von dem Dialogdesign, um auf das Ausklappmenü zuzugreifen, mit dem die Dialogdesigns verwaltet werden.

Abhängig von dem Dialog und der Tatsache, ob Zugriff auf diesen Dialog vom Menü aus besteht, werden von Ihnen gespeicherte benutzerdefinierte Designs als zusätzliche Ausklappelemente im Hauptmenü angezeigt. In einem Untermenü ausgewählte Designs werden ausgeführt, ohne dass der Dialog geöffnet wird, es sei denn die Shift-Taste wird dabei gedrückt. Das Design kann auch in der Auswahlliste Dialogdesign oben im Dialog ausgewählt werden.

Die zuletzt im Dialog verwendeten Einstellungen werden als Design unter <Zuletzt verwendet> gespeichert. Abhängig davon, wie der Dialog im Menü aufgerufen wird, wird entweder das Standarddesign oder das zuletzt verwendete Design oder ein angegebenes Design beim Öffnen des Dialogs geladen.

Dialoge, die das Speichern von Designs unterstützen, haben auch eine Option zum Speichern von Einstellungen als Standard. Wählen Sie die die Option Speichern als <Standard> im Ausklappmenü zum Speichern Ihrer Einstellungen als neuen Standard. Sie können die Standardeinstellungen jederzeit mit der Option Systemstandard im Ausklappmenü auf die internen/Systemeinstellungen zurücksetzen.

Schlüsselwörter:Einstellungen, speichern, Speicherung, wiederholen, Standard, benutzerdefiniert, konfigurieren
