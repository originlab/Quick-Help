FAQ-214 Wie ändere ich den Diagrammtyp?

Change-Plot-Type

Letztes Update: 04.02.2015

Im Dialog Diagrammeinstellungen können Sie jeden gültigen Diagrammtyp frei ändern. Um den Diagrammtyp zu ändern:

  1. Markieren Sie eine Datenzeichnung im Feld Liste Diagramme (im unteren Bedienfeld).
  2. Wählen Sie einen Typ in der Liste Diagrammtyp (im mittleren Bedienfeld).
  3. Klicken Sie auf die Schaltfläche Ersetzen.
  4. Klicken Sie auf OK, um den Diagrammtyp zu ändern.
