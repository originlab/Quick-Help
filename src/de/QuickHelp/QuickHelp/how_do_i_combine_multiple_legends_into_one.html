FAQ-155 Wie kann ich mehrere Legenden zu einer zusammenfassen?

Combine-Multi-Legends

Letztes Update: 04.02.2015

Jeder Layer in einer Grafik hat seine eigene Legende. Sie können jedoch eine Legende für die gesamte Grafik erstellen, wenn Sie folgende Schritte durchführen:

  1. Wählen Sie Grafik: Legende aktualisieren.
  2. Setzen Sie Ganze Seite als Ziel und verwenden Sie den Aktualisierungsmodus Rekonstruieren.
  3. Wählen Sie Eine Legende für alle in der Auswahlliste Legende.
  4. Klicken Sie auf OK.

Alternativ können Sie diesen Schritten folgen:

  1. Klicken Sie mit der rechten Maustaste und wählen Sie im Kontextmenü die Option Eigenschaften Zeichnung (oder Einstellungen).
  2. Wählen Sie im Dialog Details Zeichnung auf der linken Seite die Ebene der Grafik aus (der Ordner ganz oben in der Verzeichnishierarchie). Wählen Sie rechts im Dialog die Registerkarte Legenden/Titel.
  3. Aktivieren Sie das Kontrollkästchen Zeichnungen aus allen Layern beim Erzeugen der Legende einschließen und klicken Sie dann auf OK. Jede Legende der Seite zeigt nun Informationen aus allen Layern an.
  4. Sie können alle außer einer Legende löschen.

 

Schlüsselwörter: zusammenfügen, Box, einzeln
