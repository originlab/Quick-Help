FAQ-269 Wie definiere ich eine gewöhnliche Differenzialgleichung für eine Anpassung?

Define-ODE

Letztes Update: 04.02.2015

In Origin können Sie eine gewöhnliche Differenzialgleichung (ODE) definieren, indem Sie NAG-Funktionen aufrufen. In dem Tutorial Mit einer gewöhnlichen Differenzialgleichung anpassen erfahren Sie, wie Sie eine gewöhnliche Differenzialgleichung der ersten Ordnung mit Hilfe der Runge-Kutta-Methode anpassen. Eine ODE oder ein ODE-System höherer Ordnung können auf entsprechende Weise definiert werden.

Schlüsselwörter:Anpassung, Fit, gewöhnliche Differenzialgleichung, ODE, System, NAG, Runge Kutta

Origin-Version mind. erforderlich: 9.1 SR0
