FAQ-371 Wo finde ich Beispiele und Hilfe zum Programmieren?

LT-Script-Example

Letztes Update: 04.02.2015

In dem Ordner <Origin Installation Directory>/Samples/LabTalk Script Examples/> befinden sich viele LabTalk-Skriptdateien (mit OGS-Dateierweiterung). Sie können jede dieser Dateien im Code Builder öffnen (über das Menü Ansicht: Code Builder) und sich die Skripte ansehen.


Um die Beispiele im Befehlsfenster auszuführen, befolgen Sie die unten beschriebenen Schritte:

  1. Verwenden Sie die X-Funktion cd, um das aktuelle Arbeitsverzeichnis im Ordner dieser LabTalk-Skriptdateien festzulegen, zum Beispiel: 
    cd 2;
  2. Listen Sie alle Dateien in diesem Verzeichnis mit Hilfe der X-Funktion dir auf: 
    dir;
  3. Dieser Schritt ist optional. Um eine Datei im Code Builder zu öffnen, können Sie ed.open(fileName.ogs) verwenden wie in folgender Eingabe: 
    ed.open(embed_graphs.ogs);
  4. Um ein Beispiel auszuführen, wählen Sie eine Datei aus der Liste aus, geben Sie einfach den Dateinamen im Befehlsfenster ein und drücken Sie Enter. Ein Beispiel: 
    embed_graphs;

Das Handbuch zur Programmierung LabTalk Scripting Guide, die Referenz zur LabTalk-Sprache und Skriptbeispiele sind allesamt sinnvolle Mittel zum Lernen von LabTalk.
