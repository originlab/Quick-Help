FAQ-737 Wie tausche ich Dateien zwischen Mac und Windows aus, wenn Origin auf einem Mac läuft?

Exchange-Files-between-Mac-and-Win

Letztes Update: 16.06.2015

Verschieben Sie die Datei nur per Drag&Drop zwischen den Partitionen von Mac und Windows. Die virtuelle Maschine erstellt eine Kopie, wenn eine Datei in die jeweilige Richtung gezogen und "fallen gelassen" wird. Sie können daher leicht eine OPJ-Datei öffnen, indem Sie die Datei per Drag&Drop aus einem Mac-Ordner direkt in den Origin-Arbeitsbereich ziehen. Auf ähnliche Weise können Sie ein Diagramm aus Origin in einen Ordner in der Windows-Partition exportieren und die exportierte Datei zu Publikationszwecken per Drag&Drop in eine Mac-Anwendung ziehen.

Bitte beachten Sie, dass Drag&Drop nur von Parallel und VM Fusion unterstützt wird; es wird nicht von Virtual Box, Wine und Crossover unterstützt.
