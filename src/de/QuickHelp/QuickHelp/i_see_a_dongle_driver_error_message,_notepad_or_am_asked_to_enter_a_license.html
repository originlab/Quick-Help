FAQ-711 Ich sehe eine Fehlermeldung des Dongle-Treibers bzw. Notepad oder werde gebeten, eine Lizenz eingegeben. Wo liegt der Fehler?

Dongle-driver-error

 

Letztes Update: 18.05.2022

Warum wird Notepad geöffnet, wenn ich versuche, mein mit Dongle verwaltetes Origin auszuführen?

Warum wird mir "Kein Treiber gefunden" angezeigt, wenn ich meinen Dongle verbinde?

Warum wird mir "Das System kann die angegebene Datei nicht finden" angezeigt, wenn ich meinen Dongle verbinde?

Warum wird mir eine Ablaufwarnung von 3 Min. angezeigt?

Warum werde ich gebeten, eine Lizenz beim Start von Origin einzugeben? Bei Auftreten dieses Problems wenden Sie sich an uns.

Hinweis: Wenn Sie kürzlich eine Windows-Aktualisierung durchgeführt haben, starten Sie Ihren Rechner bitte neu, bevor Sie den aktualisierten Treiber installieren (unten). Dies löst das Problem möglicherweise, ohne dass der Treiber installiert werden muss.

Aktualisierte Dongle-Treiber sind jetzt zum Ausführen von mit Dongle verwalteten Origin und OriginPro verfügbar. Dieser neueste Treiber behebt folgende Probleme:

  • Konflikt zwischen den Dongle-Treibern von Origin/Pro 8.x und 9.x , wenn Sie auf dem gleichen PC installiert sind
  • Kompatibilität von Origin/Pro 8.x und 9.x auf Windows XP 64Bit, Windows Vista 32Bit und 64Bit, Windows 7 32Bit und 64Bit und Windows 8 32Bit und 64Bit
  • Die Installation über den Installer von Origin/Pro 8.5 SR1 ist nicht erfolgreich
  • Probleme des Standby-Modus von Windows bei Origin/Pro 7.5, 8.x und 9
  • Kompatibilität von Origin/Pro 7.5 auf Windows Vista und Windows 64Bit


Die folgenden Anweisungen können verwendet werden, um den aktualisierten Dongle-Treiber zu installieren.

  1. Schließen Sie Origin/Pro, wenn es ausgeführt wird.
  2. Entfernen Sie den Dongle aus dem USB-Port.
  3. Laden Sie diesen Dongle-Treiber herunter und speichern Sie ihn auf Ihrem Computer.
  4. Klicken Sie doppelt auf die Datei Install.exe, um den Installer des Treibers auszuführen.
  5. Aktivieren Sie das Kontrollkästchen Deinstallieren ("Uninstall") und klicken Sie auf Deinstallation starten ("Begin Uninstall"). **Dies ist notwendig, um den alten Treiber zu entfernen, falls er installiert ist bzw. nicht korrekt installiert ist.
  6. Klicken Sie auf Fertigstellen ("Finish"), wenn die Deinstallation abgeschlossen ist.
  7. Klicken Sie erneut doppelt auf die Datei Install.exe, um den Installer des Treibers auszuführen.
  8. Aktivieren Sie beide USB-Kontrollkästchen (sowohl 2 als auch 3). Wählen Sie nur den Parallel-Dongle, wenn Sie die Software Parallels verwenden. Klicken Sie auf Begin Install (Installation starten).
  9. Klicken Sie auf Fertigstellen ("Finish"), wenn die Installation abgeschlossen ist.
  10. Schließen Sie den Dongle an dem USB-Port an und warten Sie auf die Windows-Meldung Neue Hardware gefunden, um die Installation abzuschließen.
  11. Starten Sie Origin.


Wenn Sie beim Start von Origin aufgefordert werden, eine Lizenz einzugeben, wenden Sie sich bitte an uns.
