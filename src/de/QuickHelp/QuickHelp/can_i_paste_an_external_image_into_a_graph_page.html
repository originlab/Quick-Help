FAQ-218 Kann ich ein externes Bild in eine Diagrammseite einfügen?

Paste-img-to-Graph

Letztes Update: 06.01.2019

Ja, der Befehl Bearbeiten: Einfügen fügt ein Bild, das Sie kopiert haben, ein. Es wird als ein Bild des Typs EMF oder DIB (Device Independent Bitmap) eingefügt, abhängig von dem Wert der folgenden Eigenschaft:

system.display.pastemeta

Öffnen Sie das Skriptfenster (Fenster: Skriptfenster) und geben Sie Folgendes für die aktuelle Sitzung ein:

system.display.pastemeta=1; // paste as an EMF system.display.pastemeta=0; // paste as a bitmap

Informationen zum LabTalk-Objekt system.display finden Sie unter system.display.

Hinweis: Sie können ein Bild in das aktive Diagrammfenster auch mit dem Menübefehl Einfügen: Bilder aus Dateien... einfügen. Der Menübefehl wird von dem Wert für system.display.pastemeta nicht beeinträchtigt.

Schlüsselwörter:Bild, Anmerkung

Origin-Version mind. erforderlich: 8.5 SR0
