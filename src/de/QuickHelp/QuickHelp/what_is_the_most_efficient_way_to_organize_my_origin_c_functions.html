FAQ-397 Wie kann ich meine Origin C-Funktionen am effektivsten organisieren?

Organize-OriginC-Func

Letztes Update: 04.02.2015

Wenn Sie benutzerdefinierte Origin C-Funktionen entwickeln, möchten Sie diese sicherlich auch zukünftig in neueren Versionen verwenden. Daher ist es möglicherweise am geschicktesten, wenn Sie all diese Funktionen in einem Unterordner speichern, der sich nicht im Origin-Installationspfad befindet und auch nicht im Anwenderdateiordner Ihrer aktuellen Version. Eine Möglichkeit wäre, sie an einem Speicherort für benutzerdefinierte Ordner zu speichern, wie dem Bereich 'Meine Dokumente' in einem Unterordner mit dem Namen 'Meine Origin C-Funktionen'.


Um Ihre Origin C-Dateien zu verwenden, müssen Sie sie dem Origin C-Arbeitsbereich hinzufügen, sie kompilieren und verknüpfen.

  • Klicken Sie mir der rechten Maustaste auf den Zweig User im linken Bereich des Code Builders und wählen Sie Dateien hinzufügen oder wählen Sie im Menü Datei: Öffnen. Hinweis: Es gibt ein Kontrollkästchen "Zu Arbeitsbereich hinzufügen" im Dialogfeld Öffnen. Dies muss aktiviert sein.
  • Klicken Sie auf die Schaltfläche Kompilieren, um das Kompilieren und Verknüpfen durchzuführen.
  • Sie können auch alle Origin C-Dateien im Zweig User in einer Datei des Origin C-Arbeitsbereichs speichern (*. .ocw). Wählen Sie Datei: Arbeitsbereich öffnen, um ihn zukünftig zu öffnen.


Wenn Sie wollen, dass Ihre Origin C-Funktionen in jeder Origin-Sitzung zur Verwendung im Code Builder mit Ihren Origin C-Dateien im Arbeitsbereich verfügbar sind, klicken Sie mit der rechten Maustaste auf Origin C Workspace und wählen Sie Bei Start erstellen.


Lesen Sie bitte dieses Tutorial und erfahren Sie mehr zur Organisation von und den Zugriff auf Origin C-Funktionen.
