FAQ-322 Wie kann ich Daten aus verschiedenen Gruppen in einem Boxdiagramm zeigen?

Show-Data-from-Groups-Within-One-Box-Chart

Letztes Update: 04.02.2015


Seit Origin 9.1 können Sie eine andere Spalte für den Index oder die Farbabbildung der Datensymbolfarbe verwenden.


Gehen Sie zu der Registerkarte Details Zeichnung: Symbole auf Ebene der Zeichnung. Setzen Sie die Auswahlliste Symbolfarbe auf Index/Farbpalette und legen Sie eine Spalte als Referenz fest.

Hinweis: Sie müssen zuerst den Typ Daten oder Box + Datenüberschneidung auswählen, um die Daten zu zeigen.

 

Schlüsselwörter: Index, Farbabbildung, Boxdiagramm, Daten, Gruppen
