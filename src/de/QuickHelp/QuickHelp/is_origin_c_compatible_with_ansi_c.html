FAQ-395 Ist Origin C kompatibel mit ANSI C?

ANSI-C-Compatibility

Letztes Update: 04.02.2015

Ja, abgesehen von kleineren Ausnahmen unterstützt Origin C eine nahezu vollständige ANSI C-Sprachsyntax, einschließlich der meisten ANSI C-Elemente (Programmstruktur, Deklarationen und Typen, Ausdrücke und Zuweisungen, Aussagen, Funktionen, Makros, Aufzählungen, Typdefinitionen und Syntax). Weitere Informationen entnehmen Sie bitte der Origin C-Hilfe.
