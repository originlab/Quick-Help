FAQ-376 Wie lösche ich eine Variable?

Del-Variable

Letztes Update: 04.02.2015

Verwenden Sie den Befehl delete. Um eine lokale Variable zu löschen, geben Sie Folgendes ein:

del -al varName

Um eine Projektvariable zu löschen, geben Sie Folgendes ein:

del -v varName

Sie können auch Platzhalter verwenden, um alle Variablen, die mit 'var' beginnen, zu löschen:

del -v var*

Hinweis: Der Befehl ed öffnet den Dialog LabTalk-Variablen und -Funktionen. In diesem Dialog können Sie überprüfen, welche Variablen Sie anzeigen möchten, wie z.B. Session, System, String usw. Wählen Sie eine Variable aus und klicken Sie auf die Schaltfläche Löschen, um die ausgewählte Variable zu löschen.
