FAQ-697 Wie kann ich ein Diagramm mit einer bestimmten Größe in PowerPoint einfügen?

Paste-Graph-with-CertainSize

Letztes Update: 03.01.2019

Manchmal, wenn Sie eine gewünschte Diagrammseitengröße im Details Zeichnung festlegen und dieses Diagramm dann in PowerPoint, Word oder andere Anwendungen einfügen, stellt sich heraus, dass die Diagrammgröße nicht diejenige ist, die Sie festgelegt haben. Dies liegt daran, dass Origin die Seite nicht standardmäßig bei 100% kopiert. Um das Diagramm mit der bestimmten Größe einzufügen,

  1. wählen Sie Einstellungen: Optionen, um den Dialog Optionen zu öffnen.
  2. Wechseln Sie zur Registerkarte Seite. Ändern Sie den Wert von Verhältnis in 100 (Standard ist 40). Dadurch wird Ihr ursprüngliches Diagramm mit einer Größe von 100% kopiert. Klicken Sie auf OK, um den Dialog zu schließen.
  3. Klicken Sie doppelt auf das Diagramm, um den Dialog Details Zeichnung zu öffnen.
  4. Wählen Sie im linken Bedienfeld die Zeichnungsebene aus. Aktivieren Sie die Registerkarte Drucken/Druckbereich im rechten Bedienfeld. Legen Sie die gewünschten Einheiten, Breite und Höhe fest. Klicken Sie auf OK, um den Dialog zu schließen.
  5. Klicken Sie jetzt mit der rechten Maustaste auf die Diagrammseite und wählen Sie Seite kopieren. Wechseln Sie dann zu PowerPoint oder Word, klicken Sie mit der rechten Maustaste auf eine Folie und wählen Sie unter den "Einfügeoptionen" “Grafik (G)”.
