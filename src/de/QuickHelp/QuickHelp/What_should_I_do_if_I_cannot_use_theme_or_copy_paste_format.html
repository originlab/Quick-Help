FAQ-1159 Was soll ich tun, wenn ich kein Design verwenden oder das Format kopieren und einfügen kann?

Cannot-Use-Theme

Letztes Update: 14.11.2022

Wenn Sie die folgenden Probleme sehen:

  • Format kopieren/einfügen funktioniert nicht.
  • Klicken Sie mit der rechten Maustaste auf den Diagrammlayer und wählen Sie Format als Design speichern, um den Dialog zu öffnen. Im unteren Bedienfeld werden keine Eigenschaften aufgeführt, nur eine leere Tabelle.

Das liegt daran, dass Origin temporäre Dateien wie OTF0000.tmp im Temp-Ordner erstellen würde und der Ordner voll ist.

Um dieses Problem zu beheben:

  1. Geben Sie %temp% im Ordnerbrowser von Windows ein, um den Temp-Ordner zu öffnen.
  2. Löschen Sie alle Dateien mit dem Namen OTFXXXX.tmp
