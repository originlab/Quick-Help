FAQ-223 Wie setze ich für Parameter die gleichen Schranken beim Anpassen von mehreren Datensätzen?

Fitting-Multiple-Data-Param-Bounds

Letztes Update: 14.07.2018

Sie können zuerst die Anpassungsparameter von mehreren Datensätzen teilen und dann die Grenzen festlegen. Danach können Sie das Teilen der Parameter beenden, so dass sie die gleichen Grenzen haben.

Wenn Sie den Parameter y0 teilen möchten, können Sie auf der Registerkarte Parameter das Kontrollkästchen Teilen (nur verfügbar, wenn Globaler Fit als der Fitmodus für mehrere Datensätze gesetzt ist) für alle beliebigen y0-Parameter wie y0 oder y0_2 aktivieren. Die Parameterliste wird aktualisiert, so dass nur y0 mit dem aktivierten Kontrollkästchen Teilen gezeigt wird. Wenn Sie dann auf die Registerkarte Schranken gehen, können Sie auf das Element für Untere Grenzen klicken und deaktivieren entweder in < oder in <= ändern und dann doppelt auf den UG-Wert klicken und einen Wert eingeben. Wiederholen Sie die Schritte für Obere Grenzen. Sie können zurück zur Registerkarte Parameter gehen und das Kontrollkästchen Teilen deaktivieren. Sie können jetzt auf der Registerkarte Schranken sehen, dass y0, y0_2 etc. alle die gleichen unteren und oberen Grenzwerte haben.

Schlüsselwörter: multi, viele, mehrere, teilen, gemeinsam nutzen
