FAQ-224 Wie verwende ich die Parameterwerte aus der letzten Anpassung als neue Startwerte?

Fitted-Result-as-Initial-Value

Letztes Update: 04.02.2015

Mit der Funktion Neuberechnung der Analyse können Sie Parameter ändern und die Analyse aktualisieren. Wenn Sie eine nichtlineare Anpassung durchführen, wird der Modus Neu berechnen auf Manuell gesetzt. Nachdem Sie auf die Schaltfläche Fit geklickt haben, können Sie Parameter ändern, indem Sie auf das grüne Schloss klicken.

Sie können die verwendeten Initialisierungsparameterwerte auf der Registerkarte Parameter sehen. Sie hängen davon ab, ob Sie zuvor auf Fit oder die Schaltflächen Fit bis konvergiert und OK geklickt haben.

Wenn Sie wissen, dass Sie später möglicherweise sehen wollen, ob die Genauigkeit des Fits erhöht werden kann, und mit der Anpassung fortfahren und dabei mit den zuvor gefundenen Parameterwerten starten wollen, klicken Sie zuerst auf die Schaltfläche Fit bis konvergiert und dann auf die Schaltfläche OK.

Wenn Sie beim erneuten Öffnen des Dialogs zum Ändern der Parameter nur auf Fit klicken, sehen Sie die gleichen Initialisierungswerte, die beim vorherigen Anpassen verwendet wurden.

 

Schlüsselwörter:final, fortsetzen, Initialisierung, nichtlinear, nicht-linear, Anpassung, Differenz
