FAQ-836 Wie exportiere ich Diagramme in Windows-Unterordner mit der gleichen Ordnerstruktur wie in Origins Projekt Explorer?

Export-Graphs-in-SubFolders

Letztes Update: 19.09.2016


Angenommen, Sie haben zwei Ordner im Projekt, Ordner A und Ordner B. In beiden Ordnern befinden sich ein Diagramm, Diagramm A bzw. Diagramm B. Manchmal möchten Sie vielleicht beide Diagramme entsprechend der Ordnerstruktur exportieren, so dass Sie etwas erhalten wie:

--<Projektordner>/Diagramme/OrdnerA/DiagrammA.jpg
--<Projektordner>/Diagramme/OrdnerB/DiagrammB.jpg

Ab Origin 2017 wird ein neues Schlüsselwort eingeführt, dass eine perfekte Lösung für diesen Fall darstellt: der <PE-Pfad>. <PE-Pfad> bedeutet, dass Diagramme in Windows-Unterordnern mit der gleichen Ordnerstruktur wie in Origins Projekt Explorer exportiert werden. Das obige Beispiel soll zu Veranschaulichung dienen. Um Diagramme erwartungsgemäß in Unterordner zu exportieren:

  1. Wählen Sie bei aktivem Diagramm Datei: Grafiken exportieren im Origin-Menü. Der Dialog expGraph wird geöffnet.
  2. Wählen Sie die Option <Projektordner><PE-Pfad> in der Auswahlliste Pfad.
  3. Legen Sie die anderen Einstellungen, wie gewünscht, fest. Klicken Sie zum Exportieren auf OK.

Schlüsselwörter:Diagramme exportieren, Ordnerstruktur, Pfad exportieren
