FAQ-280 Wie führe ich eine Analyse auf eine spezielle Kurve in meinem Diagramm mit mehreren Kurven durch?

Analyze-CertainCurve-of-MultiCurve

Letztes Update: 04.02.2015

Beim Durchführen von Analysen auf gezeichnete Daten wird im Analysedialog automatisch die AKTIVE oder die AUSGEWÄHLTE Datenzeichnung als EINGABE zugewiesen.

Die aktive Datenzeichnung kann in der Statusleiste unten rechts im Arbeitsbereich abgelesen werden. Die aktive Datenzeichnung kann geändert werden, indem Sie im Menü auf Daten klicken und aus der Datenliste auswählen.

Anstatt die aktive Datenzeichnung zu ändern, können Sie auch nur eine der Grafiken auswählen, für die Sie die Analyse durchführen möchten. Klicken Sie auf eine Ihrer Kurven (Datenzeichnungen), um sie auszuwählen. Bei einer Gruppe von Datenzeichnungen werden durch Klicken auf eine Zeichnung alle ausgewählt. Sie müssen nur noch einmal auf eine spezielle Kurve klicken, um nur diese Kurve auszuwählen. Diese ausgewählte Datenzeichnung wird dann als Eingabe erkannt, wenn sich das Analysedialog öffnet.

Schlüsselwörter:aktiv, auswählen, mehrere, anpassen, Grafik, Diagramm, Layer
