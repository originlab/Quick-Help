FAQ-864 Wie kann ich unterschiedliche Bedingungen für die gleiche Spalte haben und weitere Zeichnungen und Analysen machen?

how-to-lock-data-filter

Letztes Update: 27.03.2017

Grafische und analytische Ergebnisse, die aus gefilterten Arbeitsblattdaten erzeugt werden, werden aktualisiert, wenn Daten oder Filterbedingungen sich ändern. Daher, wenn Sie Vergleiche von Grafiken und Analysen, die mit unterschiedlichen Filterbedingungen der gleichen Arbeitsblattdaten erzeugt wurden, durchführen möchten, indem Sie sie nebeneinander legen, müssen Sie den Menübefehl Worksheet: Spalten kopieren in (die X-Funktion colcopy) verwenden und die relevanten Spalten, die für die Analyse und die grafische Darstellung gebraucht werden, in ein neues Arbeitsblatt kopieren. Danach "sperren" Sie den Filter im neuen Arbeitsblatt. Auf diese Weise "friert" ein bestimmtes Ergebnis ein, so dass Sie Änderungen an den ursprünglichen Arbeitsblattdaten oder Filterbedingungen vornehmen und ein neues Ergebnis erzeugen können.

Weitere Informationen zu dieser Vorgehensweise finden Sie in dem Tutorial: Filter für Analyse blockieren oder sehen Sie sich das Video: Diagramme mit Filterbedingungen dynamisch aktualisieren an.
