FAQ-825 Kann ich das Layout an PowerPoint senden?

Can_I_send_layout_to_PowerPoint

Letztes Update: 30.05.2016


Obwohl das Layoutseitenfenster als ein “Anzeigefeld” für Diagramme und Arbeitsblätter dient, die in separaten Unterfenstern bearbeitet werden, ist das Layout technisch keine Grafik. Deswegen ist es nicht möglich, das Layout als ein OLE-Objekt in andere Anwendungen einzufügen bzw. sie an diese zu senden (wie Microsoft Word oder PowerPoint). Es kann aber als Bild eingefügt oder gesendet werden.

Wenn Sie ein Layout in eine andere Anwendung als Bild einfügen möchten:

  1. Aktivieren Sie ein Layoutfenster, drücken Sie Strg+C in Origin und Strg+V in der anderen Anwendung, um das Layout als Bild einzufügen.


Wenn Sie ein Layout an PowerPoint senden möchten, könnte auch das Hilfsmittel Grafiken an PowerPoint senden hilfreich sein. Sie können Folgendes tun:

  1. Aktivieren Sie ein Layoutfenster und klicken Sie dann auf die Schaltfläche Grafiken an PowerPoint senden , um den Dialog ped_pptslide zu öffnen.
  2. Wählen Sie Bild unter Grafiken einfügen als anstatt Eingebettetes Objekt.
  3. Anschließend können Sie das Layout als ein Bild an PowerPoint senden, indem Sie auf OK klicken.
