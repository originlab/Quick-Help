FAQ-7 Kann ich die Demoversion in mein gekauftes Produkt umwandeln?

DemoLic-to-Product

Letztes Update: 16.05.2019

Ja. Es gibt drei Möglichkeiten, um die Demoversion in ein vollwertiges Produkt umzuwandeln (Seriennummer erforderlich):

Inhalt

  • 1 Wenn Sie eine Seriennummer mit Node-Locked-Lizenz haben
    • 1.1 Methode 1 - Wenn Sie noch keine Demolizenz haben
    • 1.2 Methode 2
  • 2 Wenn Sie eine Seriennummer mit Concurrent-/Dongle-Lizenz haben

Wenn Sie eine Seriennummer mit Node-Locked-Lizenz haben

Methode 1: Wenn Sie noch keine Demolizenz haben
  1. Wählen Sie im Dialog Optionen für Demolizenz, der aufgerufen wird, wenn Sie die Demoversion starten, die letzte Option Für diesen Computer aktivieren und klicken Sie auf OK.
  2. Geben Sie im Dialog Lizenzaktivierung den Produktschlüssel ein, um die Demoversion in das vollwertige Produkt umzuwandeln und es zu aktivieren.
Methode 2:
  1. Wählen Sie Hilfe: Demoversion in Produkt umwandeln im Menü.
  2. Geben Sie den Produktschlüssel im nun aufgerufenen Dialog Lizenzaktivierung ein. Die Demoversion wird in das Produkt umgewandelt und dieses dann aktiviert.

Wenn Sie eine Seriennummer mit Concurrent-/Dongle-Lizenz haben

  1. Schließen Sie Origin (Pro), wenn es ausgeführt wird.
  2. Wenn Sie Windows 7 und Vista verwenden:
    • Klicken Sie auf die Startschaltfläche von Windows und wählen Sie Programme | OriginLab Origin(Pro) | Origin(Pro) reparieren oder deinstallieren. Der Installationsassistent (InstallShield Wizard) wird gestartet.
    Wenn Sie Windows 8 verwenden:
    • Drücken Sie die Taste Windows auf der Tastatur.
    • Klicken Sie mit der rechten Maustaste auf einen leeren Bereich des Desktops.
    • Klicken Sie auf die Schaltfläche Alle Apps unten rechts im Desktop.
    • Wählen Sie Origin(Pro) reparieren oder deinstallieren, das auf dem Desktop angezeigt wird. Der Installationsassistent (InstallShield Wizard) wird gestartet.
    Wenn Sie Windows 10 verwenden:
    • Klicken Sie auf die Startschaltfläche von Windows und wählen Sie Alle Apps | OriginLab Origin(Pro) | Origin(Pro) reparieren oder deinstallieren. Der Installationsassistent (InstallShield Wizard) wird gestartet.
  3. Wählen Sie die Option Programm ändern und klicken Sie auf Weiter.
  4. Klicken Sie auf "Ja", wenn Sie gefragt werden, ob die Seriennummer geändert werden soll.
  5. Geben Sie Ihre Seriennummer ein und schließen Sie den Änderungsprozess ab. Klicken Sie auf Fertigstellen, wenn Sie fertig sind.

Schlüsselwörter:Demoversion, Testversion, in Produkt umwandeln, Seriennummer
