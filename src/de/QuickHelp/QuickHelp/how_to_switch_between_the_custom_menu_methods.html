FAQ-857 Wie erstelle ich benutzerdefinierte Menüs?

Switch-Menu-Method

Letztes Update: 12.04.2021

Um benutzerdefinierte Menüs zu erstellen:

Wählen Sie im Menü Einstellungen: Benutzerdefinierte Menüs verwalten. Lesen Sie auf dieser Seite ein Beispiel, das Schritt für Schritt zeigt, wie Sie benutzerdefinierte Menüs erstellen.

Die Einstellungen sind in einer omc-Datei gespeichert. Dadurch können Sie Ihre benutzerdefinierten Menüs mit anderen teilen.

Seit Origin 2020 wird Volles Menü/Kurzmenü (die in einer cnf-Datei sind) im Menü nicht mehr gezeigt. Wenn Sie den Eintrag Kurzmenü wieder herstellen möchten:

  1. Setzen Sie @scnfm = 1 im Dialog Systemvariablen. Ausführliche Anweisungen finden Sie auf dieser Seite.
  2. Ändern Sie den Wert MenuLevelByOMC in 0 wie in Origin.ini im Anwenderdateiordner (UFF). 
    [Config] MenuLevelByOMC=0
  3. Stellen Sie sicher, dass sich keine omc-Dateien im Anwenderdateiordner befinden.

Weitere Einzelheiten finden Sie auf dieser Seite.
