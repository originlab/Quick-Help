FAQ-765 Wie definiere ich Projektvariablen schnell im Dialog Werte setzen und greife dort auf sie zu?

Define-ProVar-in-SetValues

Letztes Update: 27.08.2015

Um eine Projektvariable im Dialog Werte setzen zu definieren,

  1. Wählen Sie Variablen: Projektvariablen: Mehr..., um den Dialog Projektvariablen zu öffnen. Definierte Projektvariablen werden hier aufgelistet und können ausgewählt werden.
  2. Klicken Sie auf die Schaltfläche +, um Ihre Projektvariable im Dialog Variablen definieren zu definieren.

Wenn sie definiert ist, können Sie die Projektvariable im Menü Variablen: Projektvariablen finden, auf das Sie schnell zugreifen können.

Origin-Version mind. erforderlich: Origin 2016 SR0

Schlüsselwörter:Projektvariable definieren, Werte setzen
