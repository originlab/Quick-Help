FAQ-409 Wie deaktiviere ich Sparklines beim Import von ASCII-Dateien?

Turn-off-Sparklines-when-Import

Letztes Update: 11.06.2021

Der ASCII-Dateiimport kann möglicherweise langsam sein, wenn die Datei viele Spalten hat und für jede eine Sparkline erzeugt wird. Sie können Sparklines beim Import ein- oder ausschalten.

Die Seite Datenspalten des Importassistenten und die Importoptionen für den Import einzelner und mehrerer ASCII-Dateien haben ein Bedienelement, das auf Ja, Nein oder Ja (wenn weniger als 50 Spalten) gesetzt werden kann.

Wenn Sie einen Datenkonnektor verwenden, um CSV- oder MS-Excel-Daten zu importieren, können Sie die Sparklines beim Import durch die Einstellung @IMPS = 0 deaktivieren. Siehe FAQ-708 Wie ändere ich permanent den Wert einer Systemvariablen? für Informationen zum Ändern des Werts einer Systemvariable.

Schlüsselwörter:langsam
