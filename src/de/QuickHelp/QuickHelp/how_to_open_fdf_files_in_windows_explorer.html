FAQ-630 Wie öffne ich FDF-Dateien im Windows Explorer?

open-FDF

Letztes Update: 21.07.2018

Wenn auf Ihrem Computer der Adobe Acrobat Reader installiert ist, kann neben Funktionsdefinitionsdateien von Origin (.FDF) ein Adobe Acrobat Reader-Symbol angezeigt werden. Wenn Sie doppelt auf die Datei klicken, versucht Acrobat Reader, erfolglos die Datei zu öffnen. Um die Datei in einem Texteditor zu öffnen, können Sie mit der rechten Maustaste auf die Datei klicken und im Kontextmenü Öffnen mit auswählen. Sie können Notepad oder einen anderen Texteditor aus dem zugehörigen Dialog auswählen.

Hinweis: Es gibt auch in diesem Dialog ein Kontrollkästchen, mit dem Sie Dateien diesen Typs immer mit Hilfe der festgelegten Anwendung öffnen können.
