FAQ-315 Wie kann ich Ausreißer in einem Boxdiagramm in einer Linie ausrichten?

Align-Outliers-in-Line-Box-Chart

Letztes Update: 04.02.2015


Seit Origin 9.1 werden Ausreißer standardmäßig auf einer Linie im Boxdiagramm gezeichnet. Die Optionen sind im Dialog Details Zeichnung verfügbar.

  1. Gehen Sie zu der Registerkarte Details Zeichnung: Boxdiagramm auf Ebene der Zeichnung.
  2. Aktivieren Sie das Kontrollkästchen Ausreißer.
  3. Aktivieren Sie Ausreißer auf einer Linie, um sie auszurichten.

 

Schlüsselwörter:Ausreißer, Boxdiagramm, in einer Linie, ausrichten
