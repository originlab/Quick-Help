FAQ-1004 Was kann ich tun, wenn Origin keine Verbindung zu R herstellen kann?

Origin-cannot-connect-to-R

Letztes Update: 06.05.2019

Wenn Sie auf Ihrem System R installiert haben, Origin aber meldet, dass R nicht installiert ist bzw. R nicht gefunden werden kann, gibt es eine einfache Möglichkeit, diesen Fehler zu beheben.

Hintergrund

Wenn Origin aufgefordert wird, R zu starten, sucht es zuerst in der Windows Registry nach dem Registry Key für den Installationsordner von R. Falls es den Key nicht findet, sucht es nach der Windows-Umgebungsvariable R_HOME.

In einigen Fällen richten die R-Installationsprogramme für Windows den Registry Key nicht ein oder Anwender überspringen seine Einrichtung. Außerdem kann es vorkommen, dass das Installationsprogramm von R die Windows-Umgebungsvariable R_HOME nicht einrichtet.

In diesem Fall kann Origin R nicht finden und geht davon aus, dass es nicht installiert ist.

Lösung

Die Lösung besteht darin, eine Windows-Umgebungsvariable mit dem Namen R_HOME hinzuzufügen, die auf den Installationsordner von R verweist.

Angenommen, R wurde in einem Ordner auf der Festplatte installiert wie: C:\Program Files\R\R-3.6.0, dann befolgen Sie diese Schritte:

  1. Schließen Sie Origin, falls es gerade ausgeführt wird.
  2. Suchen Sie über die Windows-Suche nach den "erweiterten Systemeinstellungen" (ohne Anführungszeichen) und öffnen Sie Erweiterte Systemeinstellungen anzeigen (Systemsteuerung).
  3. Wählen Sie im Dialog Systemeigenschaften die Registerkarte Erweitert.
  4. Klicken Sie auf die Schaltfläche Umgebungsvariablen....
  5. Klicken Sie entweder unter Benutzervariablen für... oder Systemvariablen auf die Schaltfläche Neu.... (Siehe Hinweis unten.)
  6. Geben Sie für Variablename R_HOME ein.
  7. Geben Sie für Variablenwert den Installationspfad für R auf Ihrem System ein (z. B. C:\Program Files\R\R-3.6.0).
  8. Klicken Sie auf OK, um die Dialoge zu schließen.


Wenn Sie das nächste Mal Origin starten und versuchen, eine Verbindung zu R herzustellen, sollte dieser Vorgang erfolgreich sein.


Hinweis: Ob Sie die Umgebungsvariable zu Ihren Benutzervariablen oder den Systemvariablen hinzufügen. hängt von Ihren Anforderungen ab. Falls nur Sie R auf Ihrem Computer nutzen, reicht das Einrichten einer Benutzervariable wahrscheinlich. Falls jedoch andere Ihren PC verwenden, müssen Sie sie vielleicht als Systemvariable festlegen. Das finden Sie am ehesten durch einen Test heraus.
