FAQ-26 Was ist der Autorisierungscode und wofür brauche ich ihn?

Authorization-Code

Letztes Update: 04.02.2015

Der Autorisierungscode ist eine Code von acht Zeichen, der verwendet wird, um der OriginLab-Webseite über Ihre Anforderung einer Lizenz zu informieren. Wenn Sie den Dialog der Lizenzierung verwenden, um Origin zu lizenzieren, wird der Autorisierungscode automatisch zur Lizenzerzeugung an die Webseite von OriginLab weitergeleitet. Für den manuellen Abruf (beispielsweise im Fall von Computern ohne Internetzugang) müssen Sie diesen Autorisierungscode jedoch eingeben, wenn Sie eine Lizenz anfordern möchten.

Schlüsselwörter:Lizenzierung
