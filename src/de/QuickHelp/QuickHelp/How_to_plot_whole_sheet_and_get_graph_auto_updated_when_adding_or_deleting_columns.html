FAQ-1191 Wie kann ich mein Diagramm automatisch aktualisieren, wenn ich Spalten hinzufügen oder lösche?

Auto-Update-Plot-When-Add-Remove-Col

Letztes Update: 29.11.2023

Wenn Sie alle Daten in einem Arbeitsblatt zeichnen (indem Sie oben links in die Zelle klicken oder alle Spalten markieren), können Sie Folgendes festlegen, um Ihr Diagramm automatisch zu aktualisieren, wenn Spalten im Quellarbeitsblatt hinzugefügt, eingefügt oder gelöscht werden:

  1. Aktivieren Sie das Diagrammfenster, das Sie einrichten möchten.
  2. Wählen Sie im Menü Fenster: Skriptfenster.
  3. Geben Sie im Skriptfenster das folgende Skript ein und drücken Sie Enter, um es auszuführen: 
    layer.groupn.allCols = 1
    wobei n die n-te Zeichnungsgruppe im aktiven Diagrammlayer ist.


Einige Einschränkungen, die Sie kennen sollten ...

  • Diese Methode erfordert, dass die Zeichnungsgruppe aus einer einzelnen X-Spalte und allen Y-Spalten rechts von dieser X-Spalte besteht. Andere Strukturen wie XYXY... werden nicht unterstützt.
  • Falls Sie viele Y-Spalten löschen, so dass nur noch eine Y-Spalte zurückbleibt, wird die Gruppe der Zeichnung automatisch aufgelöst. Wenn Sie später neue Spalten hinzufügen, wird das Diagramm automatisch aktualisiert, aber die Gruppeneinstellungen (Farbliste, Form, Inkrement etc.) gehen verloren.

Schlüsselwörter:Gruppenzeichnungen, Zeichnung dynamisch aktualisieren, Spalte hinzufügen, Spalte entfernen, Spalte einfügen, Spalte löschen
