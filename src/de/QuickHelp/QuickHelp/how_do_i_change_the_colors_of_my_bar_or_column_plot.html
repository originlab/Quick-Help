FAQ-179 Wie ändere ich die Farben meines 2D-/3D-Balken- oder Säulendiagramms?

Letztes Update: 25.12.2017

Change-Color-for-Bar-Column

Es bestehen zwei Möglichkeiten:

Möglichkeit 1: Spezielle Punkte erstellen

Halten Sie die Strg-Taste gedrückt, während Sie auf eine der Säulen oder Balken in Ihrem Diagramm klicken. Es wird nur eine Säule bzw. ein Balken für die Bearbeitung markiert.

Alternativ können Sie auf eine Säule oder einen Balken klicken (wodurch alle Säulen/Balken markiert werden), eine Pause machen und dann ein zweites Mal auf einen einzelnen Datenpunkt klicken.

Verwenden Sie die Schaltfläche Füllfarbe auf der Symbolleiste Stil, um diesem Datenpunkt eine neue Farbe zuzuweisen, oder klicken Sie doppelt auf den Punkt, um den Dialog Details Zeichnung zu öffnen und alle Eigenschaften zu ändern.

Möglichkeit 2: Eine andere Spalte als Farbindexdaten verwenden

Legen Sie die Steuerung der Füllfarbe der Säulen/Balken durch Index, basierend auf einem anderen Datensatz, oder durch Inkrement fest. Passen Sie die Farbe für jede Komponente dann auf der Registerkarte Farbliste benutzerdefiniert an.

Schlüsselwörter: Strg, Gruppen, separat, Farbliste, Index, Inkrement, spezieller Punkt, einzelner Balken, individueller Balken
