FAQ-436 Kann Origin die MATLAB-Datei mit Strukturdaten importieren?

Import-MAT-with-Structure

Letztes Update: 06.01.2019

Wählen Sie Daten: Aus Dateien importieren: Matlab(MAT)..., um den Dialog impMatlab aufzurufen.

Bei aktiviertem Kontrollkästchen In hierarchische Struktur wird zunächst nur die Variablenliste in ein Arbeitsblatt importiert. Sie können dann auf die einzelnen Variablennamen klicken, angezeigt als verknüpfter Text in der 1. Spalte, um weiterhin nur die gewünschten Variablen, eine nach der anderen, in neue Arbeitsblätter zu importieren.

Schlüsselwörter: MAT, Strukturdaten, Hierarchische Struktur
