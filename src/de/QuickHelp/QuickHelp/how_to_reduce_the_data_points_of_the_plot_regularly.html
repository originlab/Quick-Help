FAQ-958 Wie reduziere ich die Datenpunkte der Zeichnung regelmäßig?

Reduce-Data-Points

Letztes Update: 08.01.2019

Es gibt zwei Möglichkeiten, Datenpunkte der Zeichnung regelmäßig zu reduzieren:

Methode 1:

  1. Wählen Sie bei aktivem Diagrammfenster Format: Zeichnung, um den Dialog Details Zeichnung zu öffnen.
  2. Gehen Sie zur Registerkarte Ankerlinien. Aktivieren Sie Punkte auslassen im Block Anzeige von Datenpunkten. Legen Sie die Frequenz fest. Hinweis: Wenn 5 in das Textfeld eingegeben wird, dann wird nur der erste von je 5 Datenpunkten angezeigt; die anderen 4 Datenpunkte werden übersprungen.

Methode 2:

  1. Gehen Sie zum Quellarbeitsblatt, markieren Sie die Spalte mit den Quelldaten und wählen Sie im Menü Arbeitsblatt: Zeilen reduzieren, um den Dialog Zeilen reduzieren zu öffnen.
  2. Setzen Sie in diesem Dialog die Reduktionsmethoden auf N Zeilen löschen, dann M Zeilen überspringen, Zeilen löschen auf 4, Zeilen überspringen auf 1, Ausgabe auf <neu>.
  3. Klicken Sie auf OK, um die Datenzeilen zu reduzieren. Markieren Sie die Ergebnisspalten und zeichnen Sie ein neues Diagramm mit weniger Datenpunkten.

Schlüsselwörter:Punkte reduzieren, Punkte löschen, Punkte überspringen
