FAQ-1005 Warum ist das exportierte Diagramm viel kleiner als die Seitengröße, die in Origin festgelegt ist?

export-graph-size-small

Letztes Update: 10.05.2019

Wenn Sie ein Diagramm als Vektorformat wie EMF, PDF, EPS oder WMF exportieren, sehen Sie vielleicht, dass das exportierte Bild viel kleiner geworden ist als erwartet.

Lösung 1

Erweitern Sie den Zweig Exporteinstellungen: Erweitert und wählen Sie Größe beibehalten im Dialog Grafik exportieren.

Lösung 2

Im Fall einer kleinen Wahrscheinlichkeit funktioniert die obenstehende Lösung nicht.

Das Problem wird wahrscheinlich verursacht von

Das Diagramm wird aus einem eingebetteten Diagramm in der Software eines Drittanbieters wie Microsoft Word oder PowerPoint neu gespeichert. Das heißt, das Diagramm wurde irgendwann von Origin in Word/PPT kopiert, dann aus Word/PPT neu geöffnet und in einer neuen Sitzung von Origin neu gespeichert.

Um die Seitengröße zurücksetzen:

Öffnen Sie im Menü Fenster: Skriptfenster und führen Sie das folgende LabTalk-Skript mit dem aktiven "problematischen" Diagramm aus:

doc -e p { page.info.system.export.auto=1;}
