FAQ-471 Wie vermeide ich, dass die Eingabedatenzeichenkette sich im Arbeitsblatt automatisch in numerische Werte verwandelt?

Avoid-Data-Numeric

Letztes Update: 04.02.2015


Manchmal geben Sie eine Zeichenkette wie "001315472" in eine Zelle ein und möchten nicht, dass die Zeichenkette als numerische Werte erkannt wird. Sie können das Kontrollkästchen Text während der Dateneingabe nicht in numerische Daten konvertieren im Dialog Spalteneigenschaften aktivieren.

Bitte lesen Sie Einzelheiten in diesem Abschnitt über Text während der Dateneingabe nicht in numerische Daten konvertieren.
