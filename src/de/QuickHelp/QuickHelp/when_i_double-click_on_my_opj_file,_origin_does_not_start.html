FAQ-17 Was muss ich tun, wenn ein Doppelklick auf meine OPJ-Datei Origin nicht startet?

Double-Click-Open-Failed

Letztes Update: 04.02.2015

Dieses Problem besteht seit Windows Vista und nicht ausschließlich für Origin. Normalerweise lautet die Meldung There is a problem sending the command to the program (Es gibt ein Problem beim Senden des Befehls an das Programm). Der Fehler tritt nur auf einigen PCs aus. Noch haben wir keine Lösung dafür. Testen Sie folgende Vorschläge. Möglicherweise hilft Ihnen einer dieser Vorschläge weiter.

Vorschlag 1

Legen Sie die Standardanwendung fest, die beim erneuten Öffnen der OPJ-Datei verwendet werden soll.

  1. Klicken Sie mit der rechten Maustaste auf die Origin-EXE und wählen Sie Als Administrator ausführen, um Origin einmal auszuführen, und schließen Sie das Programm dann.
  2. Klicken Sie mit der rechten Maustaste auf eine Origin-Projektdatei (OPJ) und wählen Sie Öffnen mit: Programm auswählen.
  3. Wählen Sie Origin und aktivieren Sie die Option "Dateityp immer mit dem ausgewähltem Programm öffnen".
  4. Die 64-Bit-Version von Origin wird seit 9.0 unterstützt. Wenn Ihr Betriebssystem 64 Bit ist, sind sowohl die 32- als auch die 64-Bit-Version standardmäßig installiert. Einigen Anwendern zufolge hat es geholfen, wenn das Programm OPJ-Dateien standardmäßig mit Origin9_64.exe geöffnet hat, zu Origin9.exe zu wechseln oder umgekehrt.

Vorschlag 2

Löschen Sie die DDE-Einstellungen in der Registry.

  1. Führen Sie Origin als Administrator aus und wählen Sie Fenster: Skriptfenster, um das Skriptfenster zu öffnen.
  2. Führen Sie das Skript "doc -ddde 1" aus. ddde steht hier für die Deaktivierung von DDE.
  3. Befolgen Sie die Popup-Meldung, Origin zu schließen und es einmal als Administrator auszuführen. Schließen Sie Origin.

Vorschlag 3

Versichern Sie sich, dass die Kompatibilitätseinstellungen der Origin-EXE-Datei nicht geändert wurden.

  1. Verlassen Sie Origin und suchen Sie die Origin-EXE-Datei im Windows Explorer.
  2. Klicken Sie mit der rechten Maustaste auf die Origin-EXE-Datei und wählen Sie im Kontextmenü Einstellungen.
  3. Stellen Sie sicher, dass auf der Registerkarte Kompatibilität alle Kontrollkästchen deaktiviert sind, und klicken Sie auf OK.
  4. Klicken Sie mit der rechten Maustaste auf die Origin-EXE und wählen Sie Als Administrator ausführen, um Origin einmal auszuführen.

Wieder gilt für Anwender von Origin 9.0 und 9.1, wenn Ihr Betriebssystem 64 Bit groß ist und Sie beide Versionen, 32 und 64 Bit, installiert haben, sollten Sie obige Schritte für beide EXE-Dateien durchführen (Origin9.exe und Origin9_64.exe).

 


Wenn Sie zwei Versionen von Origin auf dem gleichen PC installiert haben, z.B. Origin 9.0 und Origin 7.5 (Windows Vista, 7 oder 8) und bei einem Doppelklick auf eine OPJ-Datei Origin 7.5 anstatt Origin 9.0 gestartet wird, können Sie Folgendes versuchen:

  1. Verlassen Sie sowohl Origin 7.5 als auch Origin 9.0.
  2. Suchen Sie die EXE-Datei von Origin 9.0 im Programmierungsverzeichnis von Origin.
  3. Klicken Sie mit der rechten Maustaste auf die EXE-Datei und wählen Sie Als Administrator ausführen.
