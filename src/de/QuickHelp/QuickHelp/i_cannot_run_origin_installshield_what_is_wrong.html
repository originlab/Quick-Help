FAQ-854 Warum wird mir eine Fehlermeldung angezeigt, wenn das Origin-Installshield ausgeführt wird? Wo liegt der Fehler?

cannot-install-by-installshild

Letztes Update: 06.02.2017

Wenn Sie das Installshield von Origin ausführen, um Origin zu installieren, und unten eine Fehlermeldung sehen, dann sind Sie womöglich auf einen Konflikt zwischen Installshield und Windows OS gestoßen.

Fehlercode:-5006: 0x80070020
Fehlerinformation
...

Bitte verwenden Sie den MSI Installer zum Installieren von Origin.
