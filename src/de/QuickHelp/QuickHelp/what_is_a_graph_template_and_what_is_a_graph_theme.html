FAQ-161 Was ist eine Diagrammvorlage, und was ist ein Diagrammdesign?

GraphTemplate-and-GraphTheme

Letztes Update: 16.03.2021


Eine Diagrammvorlage ist eine Datei, die alle Spezifikationen eines Diagramms wie die Anzahl der Layer, Achseneinstellungen und Diagrammstile beinhaltet. Die Vorlagendatei enthält keine Daten, obwohl sie Grafikelemente wie Pfeile, Rechtecke, benutzerdefinierte Textbeschriftungen etc. enthalten kann. Alle Origin-Diagramme werden aus einer Vorlage heraus erstellt.


Sie können jede Vorlage benutzerdefiniert anpassen und die Änderungen für zukünftige Verwendungen speichern. Wählen Sie Datei: Vorlage speichern unter. Das Bearbeitungsfeld des Vorlagennamens enthält den Namen der Vorlage, aus der das Diagramm erzeugt wurde. Ändern Sie den Namen, um eine neue Vorlage zu erstellen, oder lassen Sie den Namen wie er ist und ersetzen Sie auf diese Weise die vorhandene Vorlage. Um in Ihre Vorlage zu zeichnen, wählen Sie Zeichnen: Meine Vorlagen. Wählen Sie dann Ihre Vorlage in der Menüliste oder klicken Sie auf Zeichnen: mit Template, um zu Ihren benutzerdefinierten Diagrammvorlagen zu navigieren.

Seit Origin 2021b bietet Origin erweiterte Diagrammvorlagen, mit denen Sie spezielle Diagrammtypen zeichnen können. Prüfen Sie die Einzelheiten in Ihrer Vorlagenbibliothek.

Ein Diagrammdesign ist eine Zusammenstellung von Diagrammeigenschaften, die in einer XML-Datei extern gespeichert wird. Ein Diagrammdesign kann jederzeit auf jedes Diagramm angewendet werden und Objekteigenschaften wie Layergröße, Seitenfarbe, Symboltyp und -farbe etc. ändern. Wählen Sie Einstellungen: Designs verwalten. Mehrere Diagrammdesigns können nacheinander auf ein Diagramm angewendet werden. Designdateien können zum Erstellen neuer Designs kombiniert werden. Wählen Sie ein Systemdesign, das automatisch auf alle neu erstellten Diagramme angewendet wird.


Eine Möglichkeit sich den Unterschied zwischen einer Diagrammvorlage und einem Diagrammdesign klar zu machen, ist, dass eine Vorlage zum Erstellen eines Diagramms verwendet wird, während ein Design hinterher angewendet wird, um das Aussehen eines Diagramms auf schnelle Weise zu ändern. Ein Systemdesign kann auch allgemein verwendet werden, so dass Sie viele Diagrammvorlagen mit der gleichen Einstellung, wie z.B. nach innen gezeichnete Hilfsstriche, nicht erneut speichern müssen.

Wenn Sie Origin 2019 oder höher verwenden, können Sie OriginLabs kostenlose App Theme Preview testen. Die App enthält Designs für Achsen, Boxdiagramme, Farbschemata, Konturdiagramme und mehr, die Sie auf Ihr Diagramm anwenden können. Designs können einzeln oder in Kombination angewendet werden, um ein vollkommen neues Design anzulegen. Außerdem -- wie aus dem Namen der App abzuleiten -- können Sie alle Änderungen in der Vorschau ansehen, bevor sie auf Ihr Diagramm angewendet werden.

Schlüsselwörter:Einstellungen, benutzerdefiniert anpassen, Vorlagenbibliothek, Template-Bibliothek, Designs verwalten
