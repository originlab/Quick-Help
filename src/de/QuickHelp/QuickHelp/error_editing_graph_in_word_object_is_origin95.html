FAQ-923 Fehler beim Versuch, ein Origin-Diagramm in MS Word zu bearbeiten: "The program used to create this object is Origin95"

Error-editing-graph-in-Word-object-is-Origin95

Letztes Update: 03.01.2019

Wenn Sie Bearbeiten: Seite kopieren verwenden und ein Diagramm in MS Word einfügen, können Sie dieses Diagramm später durch Doppelklick auf das Diagramm in Ihrem Word-Dokument bearbeiten. Mit einem Verfahren, das als Object Linking and Embedding (OLE) bezeichnet wird, identifiziert Word das Objekt als ein Origin-Diagramm. Es öffnet das Diagramm in einer Instanz von Origin, in der Sie das Diagramm in der Origin-Umgebung bearbeiten können. Danach wählen Sie Beenden und zu Dokument zurückkehren. Sie werden sehen, dass Ihr eingebettetes Diagramm in Ihrem Word-Dokument aktualisiert wurde.

Origin 2018 führte die neuen Unicode-kompatiblen Dateitypen ein, einschließlich denen für das Origin-Projekt (OPJU) und für das Origin-Diagramm (OGGU). Wenn Sie Origin 2018 oder höher verwenden, um ein Origin-Diagramm in ein Word-Dokument einzubetten, und das Dokument dann einem Kollegen geben, der Origin 2017 oder älter verwendet, kann er bzw. sie das Origin-Diagramm nicht durch OLE bearbeiten. Stattdessen erhält Ihr/e Kollege/in womöglich die Meldung: "The program used to create this object is Origin95. That program is either not installed on your computer or it is not responding ..."

Während es für Benutzer von Origin 2018 keine andere Lösung gibt, als das Diagramm einzufügen, können Benutzer mit Origin 2018b die permanente Systemvariable @OPJ festlegen, um den älteren OGG-Dateityp beim Kopieren einer Diagrammseite zu verwenden.

Um Origin 2018b oder höher zu konfigurieren und den älteren OGG-Dateityp in die Zwischenablage zu kopieren:

  1. Wählen Sie Einstellungen: Systemvariablen.
  2. Geben Sie im DiaIog Systemvariablen festlegen Variable = OPJ ein (kein vorangehendes "@"), setzen Sie dann Wert = 2 oder = 3 und klicken Sie dann auf OK.

Jetzt verwendet der Menübefehl Bearbeiten: Seite kopieren standardmäßig das ältere OGG-Dateiformat und ermöglicht den Benutzern der älteren Versionen der Software die Bearbeitung ihrer Diagramme in einem MS Word-Dokument. (Beachten Sie, dass der Diagrammtyp von der älteren Origin-Software unterstützt werden muss).
