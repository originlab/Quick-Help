FAQ-736 Kann ich ein Origin-Diagramm als eingebettetes OLE-Objekt in Word einfügen, wenn Origin auf einem Mac ausgeführt wird?

Paste-an-Origin-Graph-to-Word-on-Mac

Letztes Update: 13.05.2016

Die Objektverknüpfung und -Einbettung (OLE) erlaubt Anwendern das Kopieren und Einfügen von Origin-Diagrammen als eingebettete Objekte in andere Anwendungen (wie Microsoft Word oder PowerPoint). Sie können hinterher doppelt auf das eingebettete Objekt klicken, um es in Origin zur weiteren Bearbeitung zu öffnen.

Die OLE-Einbettung wird nur für das Betriebssystem Windows unterstützt. Sie ist nicht für Mac OS verfügbar. Auch wenn Microsoft Office-Anwendungen, die auf einem Mac ausgeführt werden, sie innerhalb ihrer Gruppe von Anwendungen unterstützen, ist die Funktion nicht als allgemeiner Mechanismus für Anwendungen wie Origin verfügbar.

Sie haben die folgenden zwei Optionen beim Ausführen von Origin auf einem Mac:

  1. Microsoft Office ist auf einer Windows-Partition installiert
    In diesem Fall können Sie unter Verwendung von Boot Camp oder einer Virtualisierungssoftware wie Parallels die OLE-Einbettung der Origin-Diagramme in Microsoft Office-Anwendungen durchführen.
  2. Microsoft Office oder andere Anwendungen zur Dokumentbearbeitung sind auf einem Mac installiert
    In diesem Fall können Sie den Menübefehl Bearbeiten: Seite als Bild kopieren in Origin verwenden und das Bild dann in andere Anwendungen einfügen. Beachten Sie, dass dieser neue Menübefehl in Origin-Version 2016 oder später verfügbar ist. Für ältere Versionen von Origin müssen Sie das Diagramm als eine Bilddatei exportieren und das Bild dann in die andere Anwendung einfügen.
