FAQ-316 Wie kann ich die Matrix von Streudiagrammen, die mit dem Hilfsmittel Korrelationskoeffizient gezeichnet wurde, benutzerdefiniert anpassen?

Customize-Scatter-Matrix-with-Correlation-Coefficients

Letztes Update: 04.02.2015


Sie können die Diagrammvorlage verwenden, um benutzerdefinierte Einstellungen der Matrix von Streudiagrammen zu speichern, die mit dem Korrelationskoeffizient gezeichnet wurde.


Führen Sie das Hilfsmittel Korrelationskoeffizient bei einer Matrix von Streudiagrammen aus. Klicken Sie doppelt, um eine eingebettete Zeichnung zu öffnen, passen Sie sie benutzerdefiniert an und speichern Sie die Diagrammvorlage. Das nächste Mal, wenn Sie den Korrelationskoeffizienten ausführen, werden die einzelnen Zeichnungen entsprechend geändert.

 

Hinweis: Um weitere Optionen wie Histogramm oder Boxdiagramm in diagonalen Zellen oder in der Matrix gezeigte Statistiken zur Verfügung zu haben, können Sie die Matrix von Streudiagrammen über das Menü Zeichnen erstellen (Zeichnen: Statistikdiagramme: Matrix von Streudiagrammen). Informationen darüber, wie Sie die Matrix von Streudiagrammen des Menüs Zeichnen benutzerdefiniert anpassen, finden Sie im Tutorial.
