FAQ-27 Braucht mein Computer immer eine Internetverbindung, um Origin auszuführen?

Run-Origin-with-Internet

Letztes Update: 04.02.2015

Nein. Sie brauchen nur zum Abschließen der Node-Locked-Lizenzierung einen Internetzugang (Sie können den Lizenzierungsvorgang auch von einem anderen Computer aus durchführen, wenn Ihr Origin-Computer über keinen Internetzugang verfügt). Außerdem brauchen Sie Internetzugang, wenn Sie Hilfe: Nach Updates suchen ausführen. Mit diesem Befehl können Sie in Origins Update-Center überprüfen, ob ein Patch oder eine aktualisierte Hilfe für Ihr Origin verfügbar ist.

Schlüsselwörter:Lizenzierung
