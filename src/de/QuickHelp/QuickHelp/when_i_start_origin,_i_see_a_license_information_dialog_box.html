FAQ-37 Beim Start von Origin wird ein Dialog zu den Lizenzinformationen angezeigt, der mich auffordert, einen Hostnamen oder eine IP-Adresse einzugeben. Wie fahre ich fort?

Lic-Info-Dialog-Shows-when-Start

Letztes Update: 04.02.2015

Wenn beim Start von Origin der Dialog der Lizenzinformationen angezeigt wird, der Sie auffordert, einen Hostnamen oder eine IP-Adresse einzugeben, dann wird Ihr Origin von einem Lizenzdienst verwaltet. Wenn Sie den Lizenzierungsvorgang noch nicht abgeschlossen haben, müssen Sie den Hostnamen oder die IP-Adresse des Lizenzservers eingeben. Wenn Sie vorher Origin problemlos ausführen konnten, nun aber den Dialog mit den Lizenzinformationen sehen, wird Ihr Lizenzdienst vielleicht nicht ausgeführt oder Ihr Computer kann nicht mit dem Lizenzdienst verbunden werden. Wenden Sie sich an ihn, um dieses Problem zu lösen.
