FAQ-1107 Wie füge ich separate Datums- und Zeitspalten in einer Spalte zusammen?

merge-date-time

Letztes Update: 20.07.2021

Sie befinden sich womöglich einmal in einer Situation, in der Ihre Datums- und Zeitdaten sich in separaten Spalten (z. B. Jahr, Monat, Tag, Stunde, Minute) befinden, und Sie die Komponenten zu einem einzelnen Datums-/Zeitwert (einzelne Spalte) zusammenfügen müssen.


Sie können Origins Funktionen Date( ) und Time() verwenden, um einen Julianischen Datumswert in Werte setzen zu erhalten.


Im obigen Beispiel funktionieren beide der folgenden Ausdrücke in Ihrer Formel in F(x)/Werte setzen (es kann weitere geben):

//concatenate the 1st 5 columns as string and specify what date format the string is.  //It will return julian date date(A$+"/"+B$+"/"+C$+" "+D$+":"+E$, "yyyy/M/dd HH:mm") //date(year, month, day) returns integer part of julian date,  //time(hour, minute, second) returns fraction part of julian date.  //So they should be added together to get complete julian date date(A,B,C)+time(D,E,0)

Beachten Sie, dass im zweiten Beispiel die "0" in time(D,E,0) gebraucht wird, weil es keine Spalte für "Sekunden" gibt. Hätte es diese Spalte gegeben, hätte es im Beispiel time(D,E,F) gehießen.

F(x)/Werte setzen gibt einen Julianischen Datumswert aus, dessen Anzeige Sie ändern können:

  1. Klicken Sie doppelt auf den Spaltenheader oder klicken Sie mit der rechten Maustaste und wählen Sie Eigenschaften....
  2. Setzen Sie das Format auf Datum und die Anzeige auf, zum Beispiel, 7/20/2021 HH:mm und klicken Sie auf OK.
