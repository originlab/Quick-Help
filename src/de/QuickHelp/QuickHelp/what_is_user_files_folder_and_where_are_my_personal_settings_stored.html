FAQ-355 Was ist der Anwenderdateiordner, und wo sind meine persönlichen Einstellungen gespeichert?

UFF-and-Personal-Settings

Letztes Update: 03.01.2019

Jeder Benutzer eines Rechners wird, wenn er Origin zum ersten Mal ausführt, aufgefordert, einen Pfad festzulegen (Anwenderdateiordner), der dann zum Standardverzeichnis für diesen Origin-Benutzer wird. Sowohl Einstellungen als auch benutzerdefinierte Dateien wie Designs, Vorlagen und Projektdateien werden in diesem Verzeichnis gespeichert. Der Benutzer muss über Lese- und Schreibrechte für dieses Verzeichnis verfügen.

HINWEIS:

Hinweis: Wenn Sie sich für ein mobiles Laufwerk entscheiden, müssen Sie dafür sorgen, dass es während des gesamten Verwendungszeitraums von Origin zur Verfügung steht. Wenn Sie dagegen ein Netzwerk wählen, wirkt sich die Netzwerkleistung auf die Leistung von Origin aus.

 

Wie finde ich meinen Anwenderdateiordner?

Es gibt mehrere Möglichkeiten, Ihren Anwenderdateiordner zu finden.

  1. Seit Origin 2017 wählen Sie Hilfe: Ordner öffnen: Anwenderdateiordner im Menü wählen. Der Ordner wird im Windows-Explorer geöffnet.
  2. Wählen Sie Einstellungen: Optionen im Origin-Menü. Auf der Registerkarte Systempfad Der Anwenderdateiordner ist hier aufgeführt.
  3. Wählen Sie Fenster: Skriptfenster im Menü. Geben Sie %Y= ein und drücken Sie Enter. Der Pfad wird im Skriptfenster abgelegt.
  4. Vom Windows-Startmenü aus Wählen Sie Alle Programme: OriginLab: Origin ##: Anwenderdateiordner, der Shortcut ist hier angegeben.
