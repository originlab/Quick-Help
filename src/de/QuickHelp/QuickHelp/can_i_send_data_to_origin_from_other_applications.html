FAQ-506 Kann ich Daten aus anderen Anwendungen wie LabVIEW an Origin senden?

send-data-from-other-application

Letztes Update: 04.02.2015

Origin kann als Automatisierungsserver (COM, Component Object Model) verwendet werden, wobei andere Anwendungen mit Hilfe von Methoden und Eigenschaften mit Origin kommunizieren, die von Origin bereitgestellt werden. Jede Anwendung, die COM-Programmierung unterstützt, wie z.B. Visual Basic, Microsoft Excel, National Instruments LabVIEW usw., kann als Client-Anwendung zur Verbindung mit Origin verwendet werden. Solche Anwendungen können Daten von Origin erhalten und zu Origin senden und auszuführende Befehle an Origin senden.
