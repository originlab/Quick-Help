FAQ-916 Warum wird mir der Fehler "bad ELF interpreter" beim Installieren des Flexnet Servers auf Red Hat Enterprise Linux angezeigt?

Bad-ELF-Interpreter-Err-RedHat-Linux-Concurrent

Letztes Update: 14.01.2018

Der Flexnet Server ist eine 32-Bit-Anwendung und erfordert daher eine 32-Bit-Version der Bibliotheken des Betriebssystems. Die Bibliotheksdateien sind standardmäßig nicht auf einer 64-Bit-Version von Red Hat Enterprise Linux installiert. Wenn Sie den Flexnet Server auf einer 64-Bit-Version von Red Hat Enterprise Linux installieren, erhalten Sie möglicherweise die folgende Fehlermeldung:

/lib/ld-lsb.so.3: bad ELF interpreter: No such file or directory

Um dieses Problem zu lösen, müssen Sie die Bibliothek redhat-lsb installieren, indem Sie diesen Befehl als Root ausführen:

 yum -y install redhat-lsb
