FAQ-1132 Wie erstelle ich eine graue Legende eines Boxdiagramms?

customize-box-legend-to-gray

Letztes Update: 11.04.2022

Das Legendensymbol eines Boxdiagramms folgt dem ersten Boxstil per Standard. Wenn Sie die Box in der Legende benutzerdefiniert anpassen möchten, erstellen Sie zum Beispiel eine graue Boxlegende.


Schritt:

  1. Klicken Sie mit der rechten Maustaste auf die Legende und wählen Sie Eigenschaften im Kontextmenü.
  2. Ersetzen Sie die Syntax der Standardlegende mit Folgendem: 
    \l(1,B, BorderColor:#808080 PatternFill:#C0C0C0) %(1,@V"Box_R") \l(1,W, lineColor:#808080) %(1,@V"Box_W") \l(1,MDL, lineColor:#808080) %(1,@V"Box_MDL") \l(1,M, edgeColor:#808080) %(1,@V"Box_M") \l(1,O, edgeColor:#808080) %(1,@V"Box_O")

  • Sie können die Farbe von Box/Rand/Punkt benutzerdefiniert anpassen, indem Sie die entsprechende Hex-Farbe #808080 ändern.
  • Erfahren Sie mehr über die Legendenanpassung des Boxdiagramms auf dieser Seite.
