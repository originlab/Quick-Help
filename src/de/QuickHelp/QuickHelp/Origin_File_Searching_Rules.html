FAQ-1071 Welche Regeln gelten in Origin zum Suchen von Dateien?

OriginFile-Searching-Rules

Letztes Update: 15.09.2020


Standardmäßig legt Origin Standarddateien wie Vorlagen, fdf-Dateien etc. im Systemordner (Installationsordner) ab. Wenn Sie jedoch eine Datei mit dem gleichen Namen im Anwenderdateiordner speichern, wird die Datei zuerst im Anwenderdateiordner gesucht.

Anwenderdateiordner > Systemordner

Falls es einen Gruppenordner gibt, ist Origins Suchreihenfolge folgende:

Anwenderdateiordner > Gruppenordner > Systemordner

Schlüsselwörter:relativer Pfad, vollständiger Pfad, absoluter Pfad, Suchreihenfolge, Standardpfad, Standardordner, Anwenderdateiordner
