FAQ-710 Warum ist meine OPJ-Datei so groß, und warum nimmt Origin die gesamte CPU meines Computers in Beschlag?

OPJ-so-large-and-take-full-cpu

Letztes Update: 14.04.2015

Wenn Sie eines dieser Probleme erleben:

  • Sie arbeiten mit einer spezifischen Origin-Projektdatei (OPJ), und Origin nimmt die gesamte CPU des Computers in Beschlag.
  • Die Größe einer OPJ-Datei ist unverhältnismäßig groß.
  • Wenn Sie auf die Symbolleistenschaltfläche mit dem gelben Rechteck klicken, schlägt die Neuberechnung fehl und die Schaltfläche wird rot.

Diese Probleme werden womöglich von den ausstehenden Operationen verursacht. Führen Sie die untenstehenden Schritte durch, um die ausstehenden Operationen zu löschen.

Öffnen Sie die spezifische OPJ. Rufen Sie dann das Skriptfenster auf (Fenster: Skriptfenster), geben Sie das folgende Skript ein und drücken Sie Enter. Origin listet die Operationen im Skriptfenster auf. (Ausstehende Operationen sind Operationen ohne Eingabe und Ausgabe.)

list op;

Geben Sie dann das folgende Skript ein (und drücken Sie Enter), um diese ausstehenden Operationen zu löschen und die OPJ-Probleme zu lösen.

delete -op;

Schlüsselwörter:große OPJ, gesamte CPU, Neuberechnung schlägt fehl

Origin-Version mind. erforderlich: Origin 8.5.1
