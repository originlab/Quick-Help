FAQ-616 Mit welchen unterschiedlichen Methoden ist es in Origin möglich, den Mittelwert zu berechnen?

Es gibt mehrere Möglichkeiten, einen Mittelwert bzw. den Durchschnitt für einen Datensatz zu ermitteln.

Schnelle Statistikwerte in der Statusleiste

Wenn Sie einen Zellenbereich im Arbeitsblatt auswählen, werden einige Statistikwerte, einschließlich dem Mittelwert (Durchschnitt), unmittelbar in der Statusleiste unten im Origin-Arbeitsbereich angezeigt. Klicken Sie mit der rechten Maustaste auf die Leiste und wählen Sie die Statistikwerte im Kontextmenü aus, die angezeigt werden sollen, oder kopieren Sie sie in die Zwischenablage. Weitere Informationen finden Sie unter  Die Statusleiste.

Der Mittelwert einer Spalte

Der Mittelwert für eine besondere Spalte kann durch Auswahl von Statistik: Deskriptive Statistik: Spaltenstatistik im Origin-Menü ermittelt werden. Sobald die Berechnung durchgeführt wurde, gibt Origin die Ergebnisse in einem neuen Arbeitsblatt aus. Der Mittelwert für jede ausgewählte Spalte wird in einer separaten Zelle in der Spalte Mittelwert angezeigt.

Einzelheiten können Sie unter Der Dialog Spaltenstatistik nachlesen.

Der Mittelwert über mehrere Zeilen

Um den Mittelwert für eine Reihe von Datenzeilen in einem Arbeitsblatt zu ermitteln, wählen Sie Statistik: Deskriptive Statistik: Zeilenstatistik. Standardmäßig gibt Origin die Ergebnisse in neuen Spalten des ursprünglichen Arbeitsblatts unmittelbar rechts von den ausgewählten Daten aus.

Einzelheiten können Sie unter Der Dialog Zeilenstatistik nachlesen.

Der Mittelwert mit Hilfe der Mittelwertfunktion

Definieren Sie einen Datensatz (z.B. Spalte A), und sein mittlerer Wert wird von der LabTalk-Funktion Mean() ausgegeben. Zum Beispiel:

Mean(col(A));

Der Mittelwert für einen Anteil (ROI, Region-of-Interest) einer Datenzeichnung

Sie können die Statistikwerte (einschließlich Mittelwert) des ausgewählten Bereich eines Linien-, Punkt-Linien-, Punkt- oder Säulendiagramms mit Hilfe des Minitools Statistik ermitteln.

Weitere Einzelheiten finden Sie unter Das Minitool Statistik.

Der mittlere Wert eines Datensatzes für eine Gruppe von Datenzeichnungen

Beim Berechnen des Durchschnitts von mehreren Kurven (Analyse: Mathematik: Mehrere Kurven mitteln) fügt Origin die Durchschnittskurve als ein grünes Liniendiagramm zu dem Layerinhalt des aktiven Diagrammfensters hinzu. Um auf die Daten zuzugreifen, klicken Sie doppelt auf die Durchschnittskurve. Der Dialog Details Zeichnung wird geöffnet Klicken Sie auf die Schaltfläche Worksheet.

Weitere Einzelheiten finden Sie unter Mehrere Kurven mitteln
