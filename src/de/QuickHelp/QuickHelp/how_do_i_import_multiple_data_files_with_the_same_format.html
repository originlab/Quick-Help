FAQ-413 Wie kann ich mehrere Datendateien mit demselben Format importieren?

Import-Multi-Data

Letztes Update: 06.01.2019

Beide Methoden, Importassistent (Daten: Aus Datei importieren: Importassistent) und Mehrere ASCII (Daten: Aus Datei importieren: Mehrere ASCII), ermöglichen die Auswahl von einer oder mehreren Dateien. Alle Dateien importieren mit der gleichen benutzerdefinierbaren Gruppe von Einstellungen.

  • Informationen zum Importassistenten finden Sie unter Der Importassistent.
  • Informationen zum Import von mehreren ASCII finden Sie unter Optionen des ASCII-Imports benutzerdefiniert anpassen.

Beachten Sie, dass Sie Filter (*.OIF-Dateien) mit dem Importassistent erstellen oder Designs in Mehrfachimport ASCII speichern können, wodurch Sie eine bestimmte Reihe von Importoptionen in eine Datei speichern können. Auf diese Weise ist es möglich, komplexere Dateien (Importassistent) per Drag&Drop zu importieren oder einfach ein Design unter dem Menübefehl "Zuletzt verwendet..." auszuwählen.

Schlüsselwörter:wiederholen, automatisieren, Automatisierung, Stapel, Filter, OIF, Design, Drag&Drop
