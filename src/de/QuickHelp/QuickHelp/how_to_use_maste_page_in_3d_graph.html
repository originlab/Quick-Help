FAQ-866 Wie verwende ich die Masterseite im 3D-Diagramm?

how_to_use_Maste_page_in_3D_graph

Letztes Update: 05.07.2018

Momentan werden 3D-Diagramme im OpenGL-Modus (Standard) nicht von der Masterseite unterstützt. Es gibt zwei Wege, dies zu umgehen:

  • Verwenden Sie eine Layoutseite.

oder

  • Schalten Sie den Modus für 3D-OpenGL-Diagramme aus, indem Sie die Systemvariable @OGL auf 0 setzen und das Diagramm erneut zeichnen.

Beachten Sie, dass das 3D-Diagramm, das nicht OpenGL ist, nur die Matrixdaten unterstützt. Methode 1 ist im Allgemeinen bequemer.
