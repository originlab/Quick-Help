FAQ-1087 Was soll ich tun, wenn meine GUI-Anpassungen JEDES MAL beim Start verloren gehen?

Toolbar-WindowSettings-not-Save

Letztes Update: 05.03.2021

Origin speichert die Fenster- und Bedienoberflächeneinstellungen, die Sie letztes Mal verwendet haben. Wenn Sie die folgenden Probleme sehen:

  • Symbolleiste "verschwindet" jedes mal beim Start von Origin
  • Origin erinnert sich nicht an die zuletzt verwendeten Fenstereinstellungen
  • Im Projekt Explorer gemachte Einstellungen gehen beim Start von Origin verloren

Versuchen Sie bitte, den Registry Key zu entfernen, und starten Sie Origin neu:

  1. Drücken Sie Windows + R, um den Dialog Ausführen von Windows zu öffnen.
  2. Geben Sie regedit ein und klicken Sie auf OK, um den Registrierungs-Editor zu öffnen.
  3. Gehen Sie zum Ordner HKEY_CURRENT_USER\Software\OriginLab\Origin 9.8\International\ToolBars\.
  4. Klicken Sie mit der rechten Maustaste auf den Ordner im linken Bedienfeld und wählen Sie Löschen.
  5. Starten Sie Origin neu.

 

Hinweise:

Bitte beachten Sie, dass die Lösung oben für GUI-Anpassungen gilt, die JEDES MAL verloren gehen. Wenn Sie nur die Standardeinstellungen der Bedienoberfläche wiederherstellen möchten, versuchen Sie die untenstehenden Schritte.

  1. Wählen Sie im Menü Ansicht: Symbolleisten.
  2. Klicken Sie auf der Registerkarte Symbolleisten auf die Schaltfläche Erneut initialisieren.

Schlüsselwörter:Einstellungen speichern fehlgeschlagen, PE, Standardeinstellungen, Symbolleiste, zuletzt verwendet
