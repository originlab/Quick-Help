FAQ-714 Wie ich Layer gruppieren?

Set_same_size_label

Letztes Update: 16.06.2015

Sie können verknüpfte Layer als eine Gruppe behandeln, um alle Layer gleichzeitig zu verschieben und in der Größe zu verändern, wenn Sie einen der Layer (Haupt- oder Unterlayer) in der Gruppe aktualisieren. Die Funktion Layer verknüpfen kann dagegen nur Änderungen des Hauptlayers wiedergeben.

Angenommen, Sie haben ein Mehrfachdiagramm mit neun Feldern, in denen alle Layer unverknüpft sind. Befolgen Sie die Schritte unten, um die benutzerdefinierte Anpassung durchzuführen:

  1. Wählen Sie bei aktivem Diagrammfenster Grafik: Layer-Verwaltung.
  2. Zuerst werden bevorzugte Layer als eine Gruppe verknüpft. Wählen Sie Layer2, Layer4 und Layer5 im Feld Layerauswahl, gegen Sie zur Registerkarte Verknüpfung und setzen Sie Verknüpfung mit auf 1. Klicken Sie auf Übernehmen.
  3. Als Nächstes ordnen Sie alle Layer in einem gewünschten Layout an. Wählen Sie alle Layer im Feld Layerauswahl und gehen Sie zur Registerkarte Anordnen, aktivieren Sie das Kontrollkästchen Verknüpfte Layer als Gruppe behandeln und passen Sie das Layout mit 3 Zeilen und 2 Spalten an. Stellen Sie sicher, dass das Kontrollkästchen Achsenrahmen zeigen deaktiviert ist. Klicken Sie auf Anwenden und dann auf OK, um den Dialog zu schließen. Die gruppierten Layer werden in einem Feld angeordnet.
