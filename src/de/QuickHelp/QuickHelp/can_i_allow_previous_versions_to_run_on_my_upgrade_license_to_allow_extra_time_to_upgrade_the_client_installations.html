FAQ-40 Kann ich zulassen, dass vorherige Versionen auf meiner Upgradelizenz laufen, damit mehr Zeit für die Clientinstallationen zur Verfügung steht?

PreVersion-On-NewLic-FLEX

Letztes Update: 04.02.2015

Ja. Mit dem Upgrade der Origin-Lizenz ist es möglich, ältere Versionen auf der gleichen Lizenz auszuführen (Anwender der Version 8.0 müssen SR2 oder höher ausführen. Anwender der Version 7.5 müssen SR7 ausführen.) .Weitere Informationen
