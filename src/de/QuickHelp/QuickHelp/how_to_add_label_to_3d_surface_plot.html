FAQ-132 Wie füge ich eine Beschriftung zu einem 3D-Oberflächendiagramm hinzu?

Add-Label-on-3D-Surface

Letztes Update: 05.07.2018

Wenn Sie eine Punktbeschriftung zu einem 3D-Oberflächendiagramm hinzufügen möchten, können Sie Folgendes tun:

  1. Geben Sie die XYZ-Koordinatenwerte der gewünschten Punkte in ein separates Arbeitsblatt ein.
  2. Verwenden Sie den Dialog Layerinhalt oder den Dialog Diagrammeinstellungen, um den Punkt als 3D-Punktdiagramm zu dem Oberflächendiagramm hinzuzufügen.
  3. Öffnen Sie den Dialog Details Zeichnung und gehen Sie zur Registerkarte Beschriftung der 3D-Punktdiagramms, aktivieren Sie das KontrollkästchenAktivieren, um die Datenbeschriftungen zu aktivieren und weitere benutzerdefinierte Anpassungen vorzunehmen.
  4. Um eine Verbindungslinie hinzuzufügen, aktivieren Sie das Kontrollkästchen Verbindungslinie zeigen, wenn Versatz (%) überschreitet und passen Sie die weiteren Einstellungen in der Gruppe Verbindungslinie nach Bedarf an.

Die einzelnen Schritte können Sie in diesem Tutorial nachlesen.
