FAQ-811 Wie kann ich ein X-Funktionsmenü, z.B. Entfaltung, einblenden?

Turn-on-XF-Menu

 

Letztes Update: 05.01.2016

Es kommt vor, dass Sie einige Standard-X-Funktionen als nützlich empfinden, aber sie werden nicht im Origin-Menü angezeigt, wie z.B. die Entfaltung. Um sie einzuschalten,

  1. Öffnen Sie das Hilfsmittel X-Funktion erstellen, indem Sie im Menü Hilfsmittel: X-Funktionen erstellen wählen.
  2. Wählen Sie Datei: System-X-Funktionen öffnen. Suchen Sie die System-X-Funktion, die Sie einschalten möchten, und laden Sie sie.
  3. Wählen Sie Hilfsmittel: Baumansicht, um sie im Baummodus anzuzeigen.
  4. Erweitern Sie den Zweig Nutzungskontext. Wählen Sie Hauptmenüs in der Auswahlliste Menüs.
  5. Wählen Sie Datei: Speichern unter..., um sie als eine benutzerdefinierte X-Funktion zu speichern.

Hinweis: Sie können die benutzerdefinierte X-Funktion im Menü auf die gleiche Weise zeigen.
