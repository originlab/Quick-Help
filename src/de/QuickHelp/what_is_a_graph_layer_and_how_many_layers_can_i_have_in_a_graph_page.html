FAQ-140 Was ist ein Diagrammlayer, und wie viele Layer kann eine Diagrammseite enthalten?

Graph-Layer-Defination

Letztes Update: 08.01.2019

Ein typisches Diagramm besteht aus Folgendem: einem Satz X- und Y- (und optional Z-) Koordinatenachsen, einer oder mehreren Zeichnung(en) und zugehörigen Achsentiteln, Textbeschriftungen und Zeichenobjekten.


Diese Elemente stellen eine verschiebbare, in der Größe veränderbare Einheit namens Layer dar. Eine einzelne Origin-Diagrammseite besitzt zwischen einem und 1024 Diagrammlayer (ein Diagramm mit 0 Layern ist nicht definiert, denn wenn Sie alle Layer aus der Diagrammseite löschen, wird das Diagrammfenster gelöscht).


Da der Layer eine unabhängige Einheit ist, kann er praktisch unbegrenzt auf der Diagrammseite verschoben und angeordnet werden. Layer eines Diagramms mit mehreren Layern können teilweise überdeckt (wie bei einem "eingesetzten" Diagramm"), völlig überdeckt (wie bei einem Diagramm mit "doppelter Y-Achse") oder sie können nebeneinander angeordnet sein (wie bei einem "Mehrfach"diagramm).


Obwohl ein Diagramm auch mehrere Layer haben kann, ist immer nur ein Layer gerade aktiv. Der aktive Layer wird durch ein "gedrücktes" Layersymbol n gekennzeichnet, das sich in der oberen linken Ecke des Grafikfensters befindet. Im folgenden Diagramm ist Layer 3 der aktive Layer, wie man zum einen durch das gedrückte Symbol von Layer 3 und zum anderen im Menü unter Ansicht: Zeige: Markierung bei aktivem Layer sehen kann.


Wenn Sie auf die verschiedenen Bereiche Ihres Diagramms klicken, ändert sich der aktive Layer abhängig davon, in welchen Layer geklickt wird. Wenn Sie dieses Verhalten nicht wollen, können Sie Origin so konfigurieren, dass ein Diagrammlayer nur über Klicken auf das Layersymbol aktiviert werden kann. Dazu wählen Sie Format: Seite. Dieser Menübefehl öffnet den Dialog Details Zeichnung, in dem das Symbol für die Seite (das Diagramm) auf der linken Seite des Dialogs angezeigt wird. Wählen Sie die Registerkarte Layer und aktivieren Sie dort das Kontrollkästchen Layer nur durch Symbole aktivieren. (Hinweis: Die Einstellung funktioniert nur für das aktuelle Diagramm.)
