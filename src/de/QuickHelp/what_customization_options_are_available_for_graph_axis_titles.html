FAQ-113 Welche Anpassungsoptionen sind für Achsentitel von Diagrammen verfügbar?

Options-for-Axis-Titles

Letztes Update: 29.03.2021

Siehe auch: FAQ-154 Wie passe ich die Standardachsentitel und -legende benutzerdefiniert an?

Legen Sie fest, welche Zeichnungsinformationen verwendet werden sollen.

Wenn Sie ein Diagramm erstellen, zeigt der Y-Achsentitel den Namen (Langname, falls vorhanden) und die Einheiten der ersten Spalte, die als Y gezeichnet wird. Wenn mehr als eine Y-Spalte gezeichnet werden, können Sie steuern, welcher Spaltenname für den Y-Achsentitel verwendet wird. Dafür:

  1. Wählen Sie im Menü Format: Seite. Aktivieren Sie die Registerkarte Legenden/Titel.
  2. Ändern Sie den Indexwert unter Index der Zeichnungen in autom. Achsentiteln.

Der Y-Achsentitel zeigt den Namen und die Einheiten der verbundenen Y-Spalte an.


Hinweis: Wenn Sie keine Einheiten wollen, deaktivieren Sie das Kontrollkästchen Einheiten zeigen....

Legen Sie fest, welche Metadaten verwendet werden sollen.

Per Standard werden der Name der Zeichnung (Langname, falls vorhanden) und die Einheiten als Achsentitel angezeigt. Sollten Sie andere Informationen der Daten als Achsentitel verwenden wollen, können Sie doppelt auf den Achsentitel klicken. Es wird %(?Y) angezeigt. Ändern Sie dies mit @Options in

  • %(?Y, @LC) --- falls Sie die Spaltenkommentare der gezeichneten Y-Spalten als Achsentitel verwenden möchten. Wenn es sich um die X-Achse handelt, können Sie %(?X,@LC) verwenden.
  • %(?Y, @WL); --- der Langname der Arbeitsmappe der gezeichneten Y.

Bitte lesen Sie die Abschnitte Achsentitel und Vollständige Liste der @Optionen der Tabelle der Textbeschriftungsoptionen, um mehr zu erfahren.

Tief-/hochgestellt, Griechisch, Abbildung Symbole

Achsentitel sind einfach eine spezielle Textbeschriftung im Diagramm, ähnlich allen anderen Textbeschriftungen im Diagrammfenster. Sie können doppelt auf den Achsentitel klicken, um literalen Text einzugeben und auch die Schaltflächen Griechisch, Hochgestellt, Tiefgestellt etc. auf der Symbolleiste Format verwenden, um speziellen Text in den Achsentitel einzugeben. Sie können im Bearbeitungsmodus auch mit der rechten Maustaste in das Bearbeitungsfeld klicken und im Kontextmenü Abbildung Symbole wählen, um Sonderzeichen, Unicode etc. einzufügen.
