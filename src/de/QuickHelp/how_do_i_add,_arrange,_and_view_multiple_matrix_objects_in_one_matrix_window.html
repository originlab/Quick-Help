FAQ-459 Wie kann ich mehrere Matrixobjekte in einem Matrixfenster einfügen, anordnen und anzeigen?

Manipulate-MatrixObject

Letztes Update: 04.02.2015

Das Matrixfenster kann wie die Arbeitsmappe mehrere Blätter enthalten. Jedes Matrixblatt kann mehrere Matrixobjekte enthalten - analog zu den Spalten in einem Blatt. Ein Matrixobjekt entspricht einem zweidimensionalen Datenfeld, das entweder Datenwerte oder Bilddaten enthalten kann.

Diese Fähigkeit, mehrere Blätter und Objekte innerhalb jedes Blattes zu enthalten, erlaubt eine effektive Verwaltung von Daten und Ergebnissen in Mappen und daher eine bessere Organisation und eine bessere Übersicht.

Die Matrixmappe besitzt ein Blatt und ein Matrixobjekt in diesem Blatt. Die Anzeige 1/1 im Fenstertitel zeigt an, dass Sie das erste von einem Objekt in diesem Blatt ansehen. Beachten Sie das D in der oberen rechten Ecke, das den Datenmodus anzeigt.




Die folgende Matrixmappe, MBook2, besitzt ein Blatt mit dem Namen "Flower" und vier Matrixobjekten. Die Anzeige 1/4 im Fenstertitel zeigt an, dass Sie das erste von vier Objekten in diesem Blatt ansehen. Beachten Sie das "I" in der oberen rechten Ecke, das den Bildmodus anzeigt. Wenn Sie auf "I" klicken, öffnet sich ein Menü, in dem Sie aus den vier Matrixobjekten wählen können. Sie können mit diesem Menü außerdem neue Objekte hinzufügen.




Klicken Sie mit der rechten Maustaste auf die Titelleiste des Fensters und wählen Sie Miniaturbild zeigen. Dies fügt oben Miniaturbilder aller Matrixobjekte hinzu, mit denen Sie zwischen den verschiedenen Matrixobjekten in diesem Matrixblatt wechseln können.

Eine einzelne Matrixmappe kann beispielsweise eine Reihe Matrixblätter und innerhalb eines jeden Matrixblattes mehrere Matrixobjekte wie ein Bild und die roten, grünen und blauen Komponenten des Bildes enthalten. Als ein weiteres Beispiel kann sich in einem Matrixblatt ein Matrixobjekt befinden, das komplexe Daten enthält, sowie weitere Matrixobjekte, die reale, imaginäre, Phasen- und Amplitudenkomponenten der komplexen Daten enthalten.

Schlüsselwörter: Matrixmappe, Blatt, Layer, Objekt, Slider
