FAQ-445 Wie richte ich eine Diagrammseite auf mehreren Blättern ein und drucke sie aus?

Set-Print-Page-Size

Letztes Update: 08.01.2019

Wählen Sie im aktiven Diagramm Format: Seite aus. Der Dialog Details Zeichnung wird geöffnet. Wählen Sie die Registerkarte Drucken/Druckbereich, falls sie nicht bereits aktiv ist. Um sicher zu stellen, dass die Werte in den Textfeldern Breite und/oder Höhe in der Gruppe Dimensionen die Größe des ausgedruckten Diagramms steuern, deaktivieren Sie die Kontrollkästchen Breite und/oder Höhe in der Gruppe Standarddruckbereich des Druckers verwenden. Andernfalls passt Origin die Diagrammseite der Druckerseite unabhängig von den Einstellungen der Gruppe Dimensionen an.


Origin druckt das Diagramm über zwei Seiten (weil 20 Zoll /10 Zoll = 2), wenn die Breite einer Diagrammseite auf 20 Zoll und die Höhe auf 8 Zoll gesetzt ist, während die Breite der Druckerseite 10 Zoll und die Höhe 8,5 Zoll beträgt, und das Kontrollkästchen Breite (Gruppe Drucker verwenden...) deaktiviert und das Kontrollkästchen Höhe aktiviert ist.


Aktivieren Sie das Kontrollkästchen, um Druckbereich markieren an den Rändern des druckbaren Bereiches des Druckers anzuzeigen. Hinweis: Sie können Schneidemarken auch drucken, indem Sie das Kontrollkästchen Druckbereich markieren im Dialog Weitere Druckoptionen aktivieren (klicken Sie auf die Schaltfläche Optionen im Dialog Drucken).
