FAQ-706 Einige meiner Diagramme werden im EPS-Format nicht korrekt exportiert. Kann man dieses Problem umgehen?

alternative-export-EPS-method

Letztes Update: 03.08.2015

Falls Ihr Diagramm hoch- bzw. tiefgestellten Text, Text mit Sonderzeichen (griechisch) oder Skriptzeichensätze, eine eingebettete Windows-Metadatei bzw. eine erweiterte Metadatei enthält oder die Origin-Funktion 'Nur innerhalb des Rahmens' verwendet, wird Ihr Diagramm möglicherweise nicht korrekt aus Origin exportiert, wenn Sie im Menü Datei: Seite exportieren verwenden und die Option Encapsulated PostScript (*.EPS) im Listenfeld Speichern als verwenden.

Als Erstes sollten Sie überprüfen, ob die Transparenzfunktion eingeschaltet ist oder nicht. Da das EPS-Format selbst Transparenz nicht unterstützt, Transparenz aber trotzdem in Origin eingeschaltet werden kann, bevor ausgewählt wird, welches Format exportiert werden soll, kann das exportierte EPS-Bild verzerrt erscheinen.

Sollte dies nicht auf die Transparenz zurückzuführen sein, verwenden Sie die unten in Teil B beschriebene Vorgehensweise, um Ihre EPS-Datei in Origin zu erstellen. Bitte beachten Sie, dass Sie zuerst einen PostScript-Druckertreiber auf Ihrem Computer installiert haben müssen, bevor Sie die Anweisungen in Teil B abschließen können. Wenn Sie nicht bereits einen PostScript-Drucker auf Ihrem Computer installiert haben, befolgen Sie bitte die Anweisungen in Teil A.

Hinweise:
  1. Sie müssen nicht wirklich einen PostScript-Drucker an Ihrem Computer anschließen, um einen PostScript-Druckertreiber zu installieren, weil Sie "File: (Ausgabe in Datei umleiten)" für "Vorhandenen Anschluss verwenden". Folglich können Sie einen beliebigen PostScript-Druckertreiber wählen, der die Ausgabe des EPS-Formats unterstützt. Es wird ebenfalls empfohlen, dass Sie einen PostScript-Druckertreiber wählen, der eine benutzerdefinierte Seitengröße unterstützt, so dass Sie über maximale Flexibilität beim Festlegen der Dimensionen des Rahmenfelds der ausgegebenen EPS-Bilddatei verfügen.
  2. Die alternative Methode zum Erstellen einer EPS-Datei hat die folgenden Beschränkungen: 1) Das Diagramm muss im Hochformat sein. Querformat wird nicht unterstützt. Für Drucker mit einer benutzerdefinierten Seitengröße bedeutet dies, dass die Seitenhöhe immer größer oder gleich der Seitenbreite sein muss. 2) EPS-Dateien, die mit dem PostScript-Druckertreiber von Adobe erstellt wurden, können nicht mit einem der Adobe-Softwareprodukte geöffnet werden. Abgesehen von diesen beiden Einschränkungen stellen die unten dargestellten Schritte eine weitläufig verwendete und robuste Vorgehensweise dar.

Teil A

  1. Installieren Sie einen der PostScript-Druckertreiber. Sie können zur Webseite der Adobe-Software gehen, nach Druckertreibern suchen und die geeigneten Treiber für Ihr Windows-Betriebssystem auswählen. Laden Sie den Treiber dann herunter und installieren Sie ihn. Stellen Sie sicher, dass "File: Ausgabe in Datei umleiten" für "Vorhandenen Anschluss verwenden" festgelegt ist, wenn Sie den Treiber installieren. Nach Installation des Druckertreibers fahren Sie mit Teil B fort, um mit Hilfe der alternativen Methode eine EPS-Bilddatei in Origin zu erstellen.
  2. Während des Installationsprozesses des Druckertreibers werden Sie aufgefordert, eine *.PPD-Datei (PostScript Printer Definition) bereitzustellen. Die *.PPD-Datei für den Drucker HP DesignJet 3500CP unterstützt sowohl das EPS-Ausgabeformat und eine benutzerdefinierte Seitengröße. Erwiesenermaßen funktioniert dies gut, wenn ein Adobe PostScript-Druckertreiber verwendet wird, um eine EPS-Bilddatei zu verwenden. Adobe-Software bietet diese PPD-Datei (sowie viele andere) unten auf der gleichen Webseite, auf der Sie den Adobe PostScript-Druckertreiber heruntergeladen haben.

Teil B

Sobald Sie einen PostScript-Druckertreiber installiert haben, folgen Sie der unten beschriebenen Vorgehensweise zum Erstellen einer EPS-Datei Ihres Origin-Diagramms oder Ihrer Layoutseite.

  1. Wählen Sie Datei: Drucken. Dieser Menübefehl öffnet den Dialog Drucken.
  2. Wählen Sie in der Auswahlliste Name der Gruppe Drucker einen PostScript-Drucker aus.
  3. Klicken Sie auf die Schaltfläche Einstellungen. Der Dialog Eigenschaften wird geöffnet. Unterschiedliche Druckertreiber haben unterschiedliche Eigenschaften, aaer im Allgemeinen müssen Sie die Optionen, die in den Schritten 4 bis 6 festgelegt sind, auswählen.
  4. Wählen Sie die Option Hochformat in der Gruppe Ausrichtung aus.
  5. Wählen Sie die Registerkarte PostScript (oder bearbeiten Sie die PostScript-Optionen).
  6. Wählen Sie Encapsulated Postscript (.eps) in der Auswahlliste des PostScript-Ausgabeformats. Möglicherweise wird eine Warnmeldung angezeigt, die Sie daran erinnert, das Kontrollkästchen In Datei umleiten im Dialog Drucken zu aktivieren.
  7. Klicken Sie auf OK im Dialog der Warnmeldung.
  8. Wechseln Sie zur Registerkarte Papier (oder bearbeiten Sie die Optionen von Papier/Seitengröße).
  9. Wählen Sie das Papierformat, um die Größe des EPS-Rahmenfelds festzulegen. Wählen Sie Benutzerdefiniert und legen Sie ggf. Ihre eigenen Abmessungen fest.
  10. Klicken Sie erneut auf OK, um den Dialog Eigenschaften zu schließen.
  11. Aktivieren Sie das Kontrollkästchen In Datei umleiten in der Gruppe Drucker des Dialogs Drucken.
  12. Bestätigen Sie mit OK, um den Dialog Drucker zu schließen. Der Dialog In Datei drucken wird geöffnet.
  13. Wählen Sie den gewünschten Ordner in der Auswahlliste Dateityp aus.
  14. Geben Sie den Dateinamen (*.eps) in das Textfeld Dateiname ein.
  15. Klicken Sie auf Speichern, um den Dialog zu schließen und die EPS-Datei zu erstellen.

Um eine EPS-Datei eines zusätzlichen Diagramm- oder Layoutfensters zu erstellen, aktivieren Sie das gewünschte Fenster. Wählen Sie Datei: Drucken und wiederholen Sie die Schritte 11 bis 15.
