FAQ-502 Wie kann ich eine Arbeitsblattzelle mit einer anderen verknüpfen?

Link-Cells

Letztes Update: 04.02.2015

Markieren Sie die Zelle(n) aus, die Sie kopieren möchten, und wählen Sie Bearbeiten: Kopieren oder drücken Sie Strg+C. (Es wird nur eine Auswahl von zusammenhängenden Zellen unterstützt.) Klicken Sie in die Zielzelle und wählen Sie Bearbeiten: Verknüpfung einfügen oder drücken Sie Alt+Strg+V. Diese Menüs sind auch aus dem Kontextmenü aufrufbar, welches mit einem Rechtsklick auf eine Arbeitsblattzelle geöffnet werden kann.

Wenn Sie eine Arbeitsblattzelle bearbeiten, die einen verknüpften Wert besitzt, werden Sie die verwendete Schreibweise bemerken. Wenn Sie z.B. den Wert in Zeile 6 der Spalte A einfügen, sehen Sie Folgendes: Zelle://A[6]. Wenn Sie den Wert in Zelle A[6] ändern, zeigen alle Zellen mit Zelle://A[6] diesen Wert an.

Schlüsselwörter:Wert, Referenz, automatisch, Aktualisierung, Ergebnis, Parameter
