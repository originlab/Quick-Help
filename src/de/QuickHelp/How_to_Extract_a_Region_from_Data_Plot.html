FAQ-1184 Wie extrahiere ich einen Bereich der Datenpunkte aus der Datenzeichnung?

Extract-Plot-Region

Letztes Update: 29.06.2023

Wenn Sie an einem Bereich der Datenzeichnung interessiert sind und die Datenpunkte in neue Spalten oder ein neues Blatt extrahieren möchten, um eine weitere Analysen durchzuführen, können Sie das Hilfsmittel Datenmarkierung, das Hilfsmittel Datenselektor oder das Minitool Cluster verwenden.

  • Datenmarkierung verwenden
  • Datenselektor verwenden
  • Minitool Cluster verwenden
