FAQ-812 Wie zeichne ich subtrahierte Daten im Ergebnisdiagramm?

Plot-Subtracted-Data-in-Graph

Letztes Update: 05.01.2016

Wenn Sie im Hilfsmittel Peaks analysieren eine Basislinie in einem Spektrum entdecken und subtrahieren, bevor Sie auf die Schaltfläche Fertig klicken, kann es sein, dass die subtrahierte Kurve angezeigt wird. Sobald Sie jedoch einmal auf die Schaltfläche Fertig geklickt haben, werden Sie sehen, dass die Kurve zur ursprünglichen Kurve zurückkehrt. Um das Spektrum mit der subtrahierten Basislinie im Ergebnisdiagramm anzuzeigen:

  1. Klicken Sie auf der Seite Basislinie subtrahieren des Hilfsmittels Peaks analysieren auf die dreieckige Schaltfläche neben dem Bedienelement Subtrahierte Daten und wählen Sie <Eingabe>: XY = Eingabe XY.
  2. Klicken Sie auf die Schaltfläche Fertig. Die Daten mit der Basislinienkorrektur ersetzt Ihre ursprünglichen Daten und werden im Ergebnisdiagramm angezeigt.

Hinweis: Die Standardposition der Subtrahierten Daten sind neue Spalten im Quellarbeitsblatt. Der Grund, warum die Subtraktion nicht bei der Standardeinstellung durchgeführt wird, liegt darin, dass wir die Quelldaten nicht überschreiben möchten, während das Ergebnisdiagramm auf den Quelldaten beruht. Das Diagramm, das Ihnen angezeigt wurde, nachdem Sie während der Peakanalyse auf die Schaltfläche Subtrahieren geklickt haben, ist nur ein Vorschaudiagrammfenster. Das Fenster wird weg, nachdem Peaks analysieren geschlossen wurde.
