FAQ-292 Wie kann ich Berechnungen mit Hilfe von Datenspalten aus Arbeitsblättern ausführen?

Math-Between-Cols

Letztes Update: 04.02.2015

Verwenden Sie den Dialog Werte setzen. Um den Dialog zu öffnen, wählen Sie Spalte: Spaltenwerte errechnen oder markieren Sie eine Spalte und klicken Sie auf die rechte Maustaste, um Spaltenwerte errechnen im Kontextmenü auszuwählen.

Dieser Dialog besitzt Beispielformeln sowie eine Liste aller standardmäßigen mathematischen Funktionen. Sie können auf die Schaltfläche Funktionen suchen und einfügen klicken, um den Dialog Funktionen suchen und einfügen aufzurufen und Schlüsselwörter einzugeben, mit denen Sie eine gewünschte Funktion ganz leicht finden.

Jede Spalte in Ihrem Arbeitsblatt kann eine damit verbundene Formel besitzen. Sie können die Formeln anderer Spalten anzeigen, indem Sie auf die Schaltflächen und klicken. Sie können dann die Spalten scrollen.

 

Schlüsselwörter:Datensatz, Vektor, mathematisch, einfach
