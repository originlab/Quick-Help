FAQ-325 Wie kann ich eine Verteilungskurve über mein Histogramm legen?

Overlay-Dist-Curve-on-Histogram

Letztes Update: 04.02.2015

  1. Klicken Sie zum Öffnen des Dialogs Details Zeichnung doppelt auf das Diagramm.
  2. Stellen Sie sicher, dass die Datenzeichnung auf der linken Seite des Dialogs ausgewählt ist. Sie sehen dann auf der rechten Seite die Registerkarte Daten. Auf dieser Registerkarte gibt es eine Auswahlliste des Verteilungskurventyps, aus der Sie auswählen können.

Wenn Sie die Daten dem Arbeitsblatt für eingeteilte Daten hinzufügen möchten, aktivieren Sie das Kontrollkästchen Verteilungskurven hinzufügen.

 

Schlüsselwörter: Histogramm, Verteilung, Overlay, Einteilung, überlagern
