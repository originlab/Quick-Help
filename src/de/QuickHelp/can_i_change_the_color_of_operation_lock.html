FAQ-354 Kann ich die Farbe des Operationsschlosses ändern?

Letztes Update: 04.02.2015

change-lock-color

Wenn in Origin die Neuberechnung für eine Operation verwendet wird, wird ein Schlosssymbol angezeigt, dass mit den Ergebnisdatensätzen verbunden ist. Das Schlosssymbol kennzeichnet mit verschiedenen Farben unterschiedliche Situationen.

Um die Standardfarbanzeige für diese unterschiedlichen Situationen zu ändern, öffnen Sie den Anwenderdateiordner und navigieren zu der Datei Origin.ini. Öffnen Sie sie mit Notepad und finden Sie den Abschnitt [Color], unter dem sich die Zeilen zur Definition der Schlossfarbe befinden:

PadlockClean=0 255 0 PadlockDirty=255 255 0 PadlockBroken=255 0 0

Ersetzen Sie die drei Zahlen, die die Werte für Rot, Grün und Blau zwischen 0 und 255 darstellen. Ihre Bedeutung:

PadlockClean bedeutet, dass keine weitere Neuberechnung erforderlich ist. Standard ist grün.

PadlockDirty bedeutet, dass Neuberechnungen ausstehen. Standard ist gelb.

PadlockBroken bedeutet, dass die Neuberechnung in dieser Operation nicht funktioniert. Standard ist rot.

Origin-Version mind. erforderlich: 9.1 SR0
