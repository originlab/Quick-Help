FAQ-8 Ich habe Origin mit einer falschen Seriennummer installiert. Wie kann ich das korrigieren?

Modify-SN

Letztes Update: 10.05.2021

Sollte die Software Origin oder OriginPro mit einer inkorrekten Seriennummer auf Ihrem Computer installiert worden sein oder Sie haben ein Upgrade von Origin auf OriginPro durchgeführt, können Sie die installierte Seriennummer folgendermaßen korrigieren/ändern:

  1. Schließen Sie Origin (Pro), wenn es ausgeführt wird.
  2. Wenn Sie Windows 7 und Vista verwenden:
    • Klicken Sie auf die Startschaltfläche von Windows und wählen Sie Alle Programme | OriginLab Origin [Version] | Origin [Version] reparieren oder deinstallieren. Der Installationsassistent (InstallShield Wizard) wird gestartet.
    Wenn Sie Windows 8 verwenden:
    • Drücken Sie die Taste Windows auf der Tastatur.
    • Klicken Sie mit der rechten Maustaste auf einen leeren Bereich des Desktops.
    • Klicken Sie auf die Schaltfläche Alle Apps unten rechts im Desktop.
    • Wählen Sie Origin [Version] reparieren oder deinstallieren, das auf dem Desktop angezeigt wird. Der Installationsassistent (InstallShield Wizard) wird gestartet.
    Wenn Sie Windows 10 verwenden:
    • Klicken Sie auf die Startschaltfläche von Windows, scrollen Sie durch die Liste der Apps und wählen Sie OriginLab Origin [Version] | Origin [Version] reparieren oder deinstallieren. Der Installationsassistent (InstallShield Wizard) wird gestartet.
  3. Wählen Sie die Option Programm ändern und klicken Sie auf Weiter.
  4. Klicken Sie auf "Ja", wenn Sie gefragt werden, ob die Seriennummer geändert werden soll.
  5. Geben Sie Ihre Seriennummer ein und schließen Sie den Änderungsprozess ab. Klicken Sie auf Fertigstellen, wenn Sie fertig sind.
