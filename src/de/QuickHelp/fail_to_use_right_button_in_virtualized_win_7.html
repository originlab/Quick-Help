FAQ-740 Warum kann ich die Schaltfläche Rechts nicht verwenden, wenn ich ein virtualisiertes Win7 auf Mac ausführe?

Fail-to-Use-Right-Button-in-Virtualized-Win-7

Letztes Update: 16.06.2015

Unter Umständen ist es beim Ausführen von Windows 7 auf einem Mac nicht möglich, die rechte Schaltfläche zu verwenden. Um das Problem zu beheben, können Sie Origin als Administrator ausführen, indem Sie OriginPro 2015 (oder eine entsprechende andere Version) im Suchfeld des Startmenüs von Windows 7 eingeben und Strg + Shift + Enter drücken.
