FAQ-392 Wie kann ich meine Origin C-Datei verschlüsseln?

Encrypt-Origin-C-File

Letztes Update: 15.06.2015

Um eine Origin C-Datei in Origin zu verschlüsseln, können Sie entweder direkt eine verschlüsselte OCZ-Datei erstellen oder eine vorhandene *.c- oder *.cpp-Datei als *.ocz-Datei verschlüsseln.

Anwender sollten vor der Verschlüsselung immer ein Backup von ihrem Code machen, da die verschlüsselte Datei nicht wiederhergestellt werden kann, sobald das Passwort vergessen ist.

Um eine verschlüsselte OCZ-Datei zu erstellen:

Wählen Sie Datei: Neu im Code Builder oder klicken Sie auf die Schaltfläche Neu auf der Symbolleiste Standard. Der Dialog Neue Datei wird geöffnet. Wählen Sie oben links im Bedienfeld OCZ-Datei und geben Sie einen Dateinamen unter Dateiname: ein. Klicken Sie dann auf OK. Der Dialog Passwort wird aufgerufen. Geben Sie ein Passwort unter Neues Passwort ein und bestätigen Sie das Passwort unter Neues Passwort bestätigen. Klicken Sie zum Fertigstellen auf OK. Die erstellte OCZ-Datei wird standardmäßig im OriginC-Unterordner unter dem Anwenderdateiordner gespeichert.

Eine bestehende Quelldatei als verschlüsselte Datei speichern

Zuerst öffnen Sie die Datei im Code Builder. Wählen Sie dann Datei: Speichern unter..., um den Dialog Speichen unter aufzurufen. Fügen Sie die Erweiterung .ocz rechts hinter dem Dateinamen hinzu und klicken Sie auf Speichern. Der Dialog Passwort wird aufgerufen. Geben Sie das Passwort ein und bestätigen Sie es, um die vorhandene Quelldatei in eine verschlüsselte *.ocz-Datei zu konvertieren.

Wenn eine verschlüsselte OCZ-Datei in der Origin-Sitzung geöffnet ist, können Anwender die *.ocz-Datei erneut als nicht verschlüsselte *.c oder *.cpp speichern, indem sie im Menü auf Datei: Speichern unter klicken und einen Dateityp in der Auswahlliste Dateityp auswählen.
