FAQ-24 Warum funktioniert Origins Funktion "Nach Updates suchen" nicht?

check-for-updates-in-Origin

Letztes Update: 04.02.2015


Wenn Ihr Netzwerk für die Verwendung eines Proxyservers eingerichtet wurde, kann es passieren, dass Nach Updates suchen in Origin nicht funktioniert. Wenn Sie die entsprechende Information in das Applet der Internetoptionen eingegeben wird, kann Origin wieder eine Verbindung zu unserer Site für die Updates herstellen.

Wenn Sie den Internet Explorer ausführen, können Sie Hilfsmittel: Internetoptionen auswählen. Ansonsten wählen Sie Start: Einstellungen: Systemsteuerung und starten das Applet der Internetoptionen.

  1. Wechseln Sie zur Registerkarte Verbindungen.
  2. Klicken Sie auf die Schaltfläche Lan-Einstellungen.
  3. Aktivieren Sie das KontrollkästchenProxyserver für LAN verwenden.
  4. Geben Sie die Adresse und den Port Ihres Proxyservers ein. (Fragen Sie Ihren Systemadministrator nach diesen Informationen.)
  5. Klicken Sie auf OK.
  6. Klicken Sie auf OK.

Führen Sie Origin aus und klicken Sie auf Hilfe: Nach Updates suchen.
