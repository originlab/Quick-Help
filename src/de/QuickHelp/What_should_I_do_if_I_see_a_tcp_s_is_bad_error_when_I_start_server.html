FAQ-1160 Was sollte ich tun, wenn ich beim Starten des Servers die Fehlermeldung "tcp_s is bad" sehe?

Tcp-s-Is-Bad

Letztes Update: 15.11.2022

Wenn der Origin-Lizenzserver nicht gestartet werden kann und Sie den folgenden Fehler im Fehlerbehebungsprotokoll sehen:

tcp_s is bad!!! Exiting

Dieser Fehler wird dadurch verursacht, dass Sie zwar eine Portnummer für den Vendor Daemon orglab in der Lizenzdatei angegeben haben, dieser Port jedoch bereits von einem anderen Prozess verwendet wird.

Um dieses Problem zu beheben:

Wenn Origin sich nicht auf der anderen Seite der Firewall befindet, besteht keine Notwendigkeit, eine Portnummer für orglab festzulegen. Ansonsten stellen Sie sicher, dass Sie einen freien, ungenutzten Port verwenden. Lesen Sie auf dieser Seite weitere Informationen, wie Sie Origin Zugriff durch die Firewall ermöglichen.
