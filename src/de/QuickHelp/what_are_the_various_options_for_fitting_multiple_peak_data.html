FAQ-299 Welche Optionen habe ich zum Analysieren von Peakdaten?

Tools-for-PeakAnalysis

Letztes Update: 04.02.2015

Origin bietet mehrere Hilfsmittel, um eine Peakanalyse durchzuführen. Im Folgenden finden Sie eine allgemeine Anleitung dazu, welches Hilfsmittel für welchen Datentyp geeignet ist. Weitere Einzelheiten zu jedem Hilfsmittel finden Sie in der Hilfe.


Minitool Peakanalyse

  • Zugriff über das Menü Minitools bei aktivem Diagramm
  • Ideal für grafische Analysen von Peakdaten, die eine geringe Anzahl von Peaks enthalten
  • Grafische Auswahl eines bestimmten Bereichs im Diagramm (ROI), um einen Teilbereich der gezeichneten Daten zu analysieren
  • Mehrere Methoden zum Erkennen und Subtrahieren der Basislinie
  • Option zum Anpassen von Peaks mit Hilfe verfügbarer Peakfunktionen (nur OriginPro)
  • Speichern von Einstellungen in einem Design für die wiederholte Analyse von ähnlichen Daten
  • Keine Stapelverarbeitung möglich



Hilfsmittel Mehrere Peaks anpassen

  • Zugriff über Analyse: Peaks und Basislinie: Mehrere Peaks anpassen, entweder bei aktivem Arbeitsblatt oder Diagramm
  • Ideal zum Anpassen von Peaks an Daten mit einer geringen Anzahl von Peaks, die deutlich separat sind
  • Auswahl von Peakzentren durch Anklicken; Durchführen der Peakanpassung mit der Option, zum Dialog NLFit zu wechseln
  • Keine Korrektur der Basislinie möglich



Peaks analysieren

  • Zugriff über Analyse: Peaks und Basislinie: Peaks analysieren, entweder bei aktivem Arbeitsblatt oder Diagramm
  • Ideal für die Peakanalyse von Datensätzen mit einer großen Anzahl von Peaks
  • Assistentenoberfläche mit Vorschaufenster zur schrittweisen Anzeige der Ergebnisse
  • Mehrere Analyseziele: Peaks suchen, Peaks integrieren, Peaks anpassen (nur OriginPro)
  • Mehrere Methoden zum Erkennen und Subtrahieren der Basislinie
  • Speichern von Einstellungen in einem Design für die wiederholte Analyse von ähnlichen Daten
  • Ausgabe umfasst mehr als 15 Peakeigenschaften bei Anpassung von Peaks
  • Zuweisen von unterschiedlichen Peakfunktionen zu unterschiedlichen Peaks
  • Flexibles Teilen von Peakparametern und Anwenden von Grenzen und Nebenbedingungen beim Anpassen
  • Über den Dialog Peaks analysieren gespeicherte Designs können für die Stapelanalyse über das Menü Analyse: Peaks und Basislinie: Peakanalyse durch Stapelverarbeitung mit Design verwendet werden.

 

Schlüsselwörter:auswählen, suchen, Integration, integrieren, Wiederholung, Bereich, Basislinie, anpassen, benutzerdefiniert, Entfaltung
