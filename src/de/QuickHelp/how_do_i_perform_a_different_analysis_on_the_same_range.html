FAQ-278 Wie führe ich eine andere Analyse für denselben Bereich durch?

DifferentAnalysis-on-SameRange

Letztes Update: 09.08.2018

Wenn Sie eine Analyse auf einen ausgewählten Datenbereich in Ihrem Diagramm ausführen, werden die Marker zu Analysemarkern und können in einer neuen Analyse nicht als Eingabebereich verwendet werden.

Eine Möglichkeit besteht darin, auf das grüne Schloss zu klicken und dann Eingabedaten mit Datenmarkern zeichnen zu wählen. Jetzt wird ein neues Diagramm mit den Markern gezeichnet. Auf dem Diagramm können Sie jetzt Analysen durchführen.

Schlüsselwörter:Marker, Schloss, Selektor, Bereich
