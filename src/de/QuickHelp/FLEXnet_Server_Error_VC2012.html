FAQ-1031 Warum wird mir die Fehlermeldung "orglab exited with status 53" angezeigt, wenn ich versuche, den FLEXnet-Server zu starten?

FLEXnet-Server-VC2012

Letztes Update: 27.07.2023

FLEXnet-Server Version 11.19 oder höher erfordert Microsoft Visual C++ 2015 Redistributable Package. Falls dieses nicht installiert ist, sehen Sie die folgende Fehlermeldung in der Protokolldatei beim versuchten Start des FLEXnet-Servers:

orglab exited with status 53 (this status is reserved but will never be generated)

ODER

 9:12:34 (lmgrd) orglab exited with status 255 () 9:12:34 (lmgrd) Since this is an unknown status, license server 9:12:34 (lmgrd) manager (lmgrd) will attempt to re-start the vendor daemon.


Um dieses Problem zu beheben:

  • Wenn Sie FLEXnet 11.16 oder eine ältere Version verwenden, führen Sie bitte ein Upgrade auf Version 11.19 oder höher durch. Sie können die neueste Version immer hier herunterladen.
  • Wenn Sie die neueste Version ausführen, aber noch immer diese Fehlermeldung angezeigt bekommen, müssen Sie das Microsoft Visual C++ 2015-2019 Redistributable Package manuell herunterladen und installieren:
  • Für das 32-Bit-Betriebssystem laden Sie bitte vcredist_x86-2015-2019.exe herunter.
  • Für das 64-Bit-Betriebssystem laden Sie bitte vcredist_x64-2015-2019.exe herunter.
