FAQ-1152 Ich erhalte die Fehlermeldung "Lizenzmanager: kann nicht initialisieren: Keine SERVERzeilen in Lizenzdatei. Was soll ich tun?

Cant-Initialize-Cant-Find-Lic-File

Letztes Update: 21.09.2022

Wenn das Starten des FLEXnet-Lizenzservers fehlschlägt und Sie die folgende Fehlermeldung in der Log-Datei erhalten:

10:38:30 (lmgrd) license manager: can't initialize:Cannot find license file.
10:38:30 (lmgrd) The license files (or license server system network addresses) attempted are
10:38:30 (lmgrd) listed below. Use LM_LICENSE_FILE to use a different license file,
10:38:30 (lmgrd) or contact your software provider for a license file.
10:38:30 (lmgrd) Filename: "C:\flexlm\Originlicense.lic"
10:38:30 (lmgrd) License Path: "C:\flexlm\Originlicense.lic"
10:38:30 (lmgrd) FlexNet Licensing error:-1,359
10:38:30 (lmgrd) System Error:2 No such file or directory
10:38:30 (lmgrd) Using license file "C:\flexlm\Originlicense.lic""


Um dieses Problem zu beheben:

Bitte aktivieren Sie das Kontrollkästchen LMTOOLS ignoriert Umgebungsvaruables des Lizenzdateipfads auf der Registerkarte Service/Lizenzdatei der LMTOOLS.

Schlüsselwörter:concurrent, -1,359, keine Initialisierung, keine Verbindung zum Lizenzserver, kann Lizenzdatei nicht finden, Systemfehler 2, Datei oder Verzeichnis wie diese fehlen
