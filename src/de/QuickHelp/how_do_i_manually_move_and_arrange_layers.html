Layer verschieben oder in der Größe verändern

  1. Markieren Sie den Layer, so dass 8 Elemente um den Layerrahmen herum angezeigt werden. Ziehen Sie an den Elementen oder verändern Sie die Größe des Rechtecks, um den Layer zu verschieben/zu verändern. Halten Sie die Strg-Taste beim Verändern der Größe gedrückt. Dann bleibt das ursprüngliche Seitenverhältnis erhalten. Es gibt mehrere Methoden, um den Layer zu markieren:
    • Klicken Sie auf die leere Fläche innerhalb des Layerrahmens.
    • Drücken Sie die Alt-Taste und klicken Sie innerhalb des Layers. Wenn es viele Zeichnungen im Layer gibt oder es sich um ein Konturdiagramm handelt, funktioniert ein einzelner Klick nicht, da dadurch nur eine Zeichnung anstatt des Layers ausgewählt wird.
    • Drücken Sie die Strg-Taste und klicken Sie auf die Achse, um den entsprechenden Layer auszuwählen.
  2. Wählen Sie im Menü Format: Layer..., um den Dialog Details Zeichnung - Layer zu öffnen. Gehen Sie zur Registerkarte Größe und Performance, um die Layergröße genauer festzulegen.
    • Bei verknüpften Layern können Sie Einheiten als % der verknüpften Layer und Links, Oben, Breite und Höhe relativ zum Hauptlayer festlegen. Wenn Sie z. B. möchten, dass Layer 2 sich mit Layer1 überschneidet. dann können Sie zu Layer 2 mit Layer 1 verknüpfen. Setzen Sie dann die Größe von Layer 2 auf % des verknüpften Layers und Links, Oben, Breite und Höhe auf 0, 0, 100, 100.
    • Wenn die Einheiten des verknüpften Layers % des verknüpften Layers ist, dann wird er mit verschoben, wenn Sie den Hauptlayer verschieben oder in der Größe verändern.

Mehrere Layer verschieben oder in der Größe verändern

  1. Drücken Sie die Tasten Shift+Alt und klicken Sie auf mehrere Layer und verschieben Sie sie alle zusammen bzw. verändern Sie ihre Größe.
  2. Verknüpfen Sie zuerst die Layer und verschieben Sie sie dann zusammen.
    1. Wählen Sie im Menü Format: Layer..., um den Dialog Details Zeichnung - Layer zu öffnen.
    2. Wählen Sie im linken Bedienfeld einen Layerknoten. Wählen Sie die Registerkarte Achsen verbinden. Verknüpfen Sie ihn mit dem Layer (Haupt), mit dem Sie ihn verschieben möchten. Hinweis: Der erste Layer im Diagramm kann nicht mit einem anderen Layer verknüpft werden, daher gibt es keine Registerkarte Achsen verbinden für ihn.
    3. Wechseln Sie zur Registerkarte Größe und Performance. Wählen Sie unter Einheiten die Option % des verknüpften Layers.
    4. Klicken Sie auf OK.
    5. Wenn Sie den Layer, mit dem dieser Layer verknüpft ist, jetzt verschieben bzw. in der Größe verändern, werden sie gemeinsam verschoben oder in der Größe verändert.
    6. Sie können auf der Registerkarte Größe und Performance auch das Kontrollkästchen Das Ziehen zur Neupositionierung positioniert auch den Hauptlayer neu aktivieren. Wenn Sie dann diesen Layer verschieben/in der Größe verschieben, wird auch der Hauptlayer verschoben.

Um die Verschiebung in horizontaler oder vertikaler Richtung zu beschränken, drücken Sie die Shift-Taste, während Sie an den Layern ziehen.

Beim Verschieben/Verändern der Größe von einem Layer werden auch andere Layer verschoben/in der Größe verändert

Dies liegt daran, dass die Layer miteinander verknüpft sind und die Einheit des Unterlayers auf % des verknüpften Layers gesetzt ist. Siehe den obigen Abschnitt. Wenn Sie nicht möchten, dass sie zusammen verschoben werden,

  1. wählen Sie im Menü Format: Layer..., um den Dialog Details Zeichnung - Layer zu öffnen.
  2. Wählen Sie den Layerknoten im linken Bedienfeld und gehen Sie zur Registerkarte Achsen verbinden, um zu prüfen, mit welchem Layer er verknüpft ist. Wechseln Sie zur Registerkarte Größe und Performance, um das Menü zu prüfen. Prüfen Sie, ob die Einheit % des verknüpften Layers ist. Ändern Sie sie in eine andere Einheit, wie z. B. % der Seite, Zoll etc.
  3. Klicken Sie auf OK. Wenn Sie jetzt den Hauptlayer verschieben, verschiebt sich dieser Layer nicht mit. Wenn Sie diesen Layer verschieben, verschiebt sich der Hauptlayer nicht mit.

Layer von einer Stelle aus verschieben, in der Größe verändern und anordnen

Es gibt ein Hilfsmittel für die Layerverwaltung, um Diagramme mit mehreren Layern von einer Stelle aus zu verwalten. Vergessen Sie nicht, auf die Schaltfläche Übernehmen auf jeder Registerkarte zu klicken, bevor Sie auf OK klicken. Ansonsten werden die Änderungen womöglich nicht wirksam.

  1. Wählen Sie im Menü Grafik: Layerverwaltung.
  2. Gehen Sie zur Registerkarte Anordnen, um sie in Feldern anzuordnen, z. B. MxN-Felder.
  3. Gehen Sie zur Registerkarte Größe und Position, um sie zu verschieben, auszutauschen, auszurichten oder ihre Größe zu verändern.
