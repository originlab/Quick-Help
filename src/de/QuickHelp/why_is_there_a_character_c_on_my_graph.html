FAQ-934 Warum befindet sich der Buchstabe "C" auf meinem Diagramm?

Character-C-Shows-on-Graph

Letztes Update: 09.12.2021

Wenn der Buchstabe "C" unerwartet auf Ihrem Diagramm gezeigt wird, bedeutet dies, dass das Origin, das Sie verwenden, entweder ungültig oder eine Raubkopie ist. Bitte wenden Sie sich an OriginLab, um eine legale Software zu erhalten. Sie können auch eine Demoversion von unserer Webseite herunterladen.

Wenn Sie eine gültige Lizenz haben, aber noch immer ein großes "C" auf dem Diagramm angezeigt wird,

versuchen Sie bitte Folgendes:

  • Laden Sie die neueste Version herunter und installieren Sie sie.
  • Klicken Sie mit der rechten Maustaste auf das Symbol der Origin-Software und wählen Sie Als Administrator ausführen, um Origin zu öffnen.
  • Wenn Sie eine rechnergebundene Lizenz (Node-locked) haben, wählen Sie Hilfe: Lizenz aktivieren, um die Lizenz zu reaktivieren.

Falls die obigen Lösungen das Problem nicht lösen können, wenden Sie sich bitte an uns.
