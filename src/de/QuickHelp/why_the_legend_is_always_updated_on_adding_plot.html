FAQ-156 Wie ändert sich die Legende beim Hinzufügen/Entfernen von Zeichnungen?

Legends-Update-Adding-Plot

Letztes Update: 08.01.2019

Seit Version 9.1 aktualisiert Origin die Legende standardmäßig, wenn eine Zeichnung zu einem Diagramm hinzugefügt wird.

Wenn Sie die Aktualisierung der Legende beim Hinzufügen von Zeichnungen deaktivieren möchten:

  1. Wählen Sie bei aktivem Diagramm Format: Seite im Origin-Menü. (Oder drücken Sie F2.)
  2. Wechseln Sie dann zur Registerkarte Legenden/Titel auf der rechten Seite.
  3. Wählen Sie die Option Kein in der Auswahlliste Modus Legende. Diese Option wird mit der Diagrammvorlage gespeichert.

Damit die Legende beim Hinzufügen/Entfernen von Zeichnungen rekonstruiert wird,

  1. wählen Sie bei aktivem Diagramm Format: Seite im Origin-Menü. (Oder drücken Sie F2.)
  2. Wechseln Sie dann zur Registerkarte Legenden/Titel auf der rechten Seite des Dialogs Details Zeichnung.
  3. Wählen Sie in der Auswahlliste Aktualisierungsmodus der Legende die Option Beim Hinzufügen/Entfernen neu erzeugen.
