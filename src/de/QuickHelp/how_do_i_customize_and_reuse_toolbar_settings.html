FAQ-337 Wie passe ich Symbolleisteneinstellungen benutzerdefiniert an und verwende sie erneut?

Customize-Reuse-Toolbar

Letztes Update: 04.02.2015

Das Menü Ansicht: Symbolleisten öffnet den Dialog zum Anpassen von Symbolleisten. Hier können Sie einstellen, welche Symbolleisten angezeigt werden sollen. Auf der Registerkarte Befehle haben Sie Zugriff auf alle verfügbaren Symbolleistenschaltflächen, einschließlich einiger Schaltfläche, die nicht standardmäßig verfügbar sind.

Wählen Sie beispielsweise Spalte in den Kategorien und sehen Sie rechts dann die Schaltfläche Spalte tauschen. Sie können diese Schaltfläche per Drag&Drop auf eine vorhandene Origin-Symbolleiste ziehen.

Wenn Sie nur Schaltflächen von einer Symbolleiste zu einer anderen verschieben möchten, müssen Sie den Dialog zum Anpassen der Symbolleisten nicht öffnen. Halten Sie einfach die Alt-Taste gedrückt und ziehen Sie die Schaltfläche per Drag&Drop auf eine andere Symbolleiste. Wenn Sie sie an einer anderen Stelle als auf einer Symbolleiste "fallen lassen", wird sie nicht mehr angezeigt.
