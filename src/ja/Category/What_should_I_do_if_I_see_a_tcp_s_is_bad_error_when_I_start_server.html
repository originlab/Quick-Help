FAQ-1160 サーバの起動時に「tcp_s is bad」というエラーが表示された場合はどうすればよいですか?

Tcp-s-Is-Bad

最終更新日：2022/11/15

Originライセンスサーバを起動できず、デバッグログに次のエラーが表示される場合

tcp_s is bad!!! Exiting

これは、ライセンスファイルでベンダーデーモンorglabのポート番号として指定したものが別のプロセスで既に使用されていることが原因です。

この問題を解決するには、

Originがファイアウォールで制限されていない場合、orglabのポート番号を指定する必要はありません。それ以外の場合は、空きポートを使用してください。ファイアウォールを介してOriginアクセスを許可する方法の詳細については、 このページを参照してください。

キーワード:FLEXlmサーバ, 接続できない, ポート番号, orglab exited with status 28, EXITING DUE TO SIGNAL 28, 同時起動ライセンス, lic
