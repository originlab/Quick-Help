FAQ-765 値の設定ダイアログでプロジェクト変数を素早くて定義してアクセスするには？

Define-ProVar-in-SetValues

最終更新日:2015/08/27

値の設定ダイアログでプロジェクト変数を定義するには、

  1. メニューから変数：プロジェクト変数：詳細...を選択し、プロジェクト変数ダイアログを開きます。ここに表示されるプロジェクト変数の表示や削除が可能です。
  2. + ボタンをクリックして開く変数定義ダイアログでプロジェクト変数を定義します。

定義したプロジェクト変数は、変数：プロジェクト変数メニューから使用できるようになります。

必要なOriginのバージョン:Origin 2016 SR0

キーワード:プロジェクト変数の定義, 列値の設定
