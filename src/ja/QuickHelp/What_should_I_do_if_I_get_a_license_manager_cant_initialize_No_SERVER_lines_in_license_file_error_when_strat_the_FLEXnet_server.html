FAQ-1131 FLEXnetサーバを開始したとき"license manager: can't initialize:No SERVER lines in license file." というエラーが表示されます。どうしたらよいでしょうか？

LIC-Server-Error-1366

最終更新日：2022/3/14

ライセンスサーバを開始できず、ログファイルに以下のエラーメッセージが表示された場合


12:07:37 (lmgrd) license manager: can't initialize:No SERVER lines in license file.
12:07:37 (lmgrd) License Path: "C:\OriginLicenseServer\Licenses\orglab.lic"
12:07:37 (lmgrd) FlexNet Licensing error:-13,66


以下の方法をお試しください。

  1. メモ帳（または他のメモ帳エディタ）でライセンスファイルを開きます。SERVER 行があるか確認します。ない場合、OriginLabのWebサイトからライセンスファイルを取得してください。
  2. ファイル：名前を付けて保存メニューでライセンスファイルを再保存します。名前を付けて保存ダイアログで、エンコードがANSIであることを確認します。

キーワード:ライセンス, -13,66, can't initialize:No SERVER lines, ライセンスサーバシステムに接続できません
