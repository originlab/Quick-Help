FAQ-842 「Pythonがインストールされていません」というエラーが表示され、Pythonを実行できません。どのようにしたらよいでしょうか?

Why_I_fail_to_run_Python_with_error_no_Python_installed

最終更新: 2018/07/26


Pythonコンソールは、VC++ 2015 DLLに依存しています。一般的に、これらのDLLはシステムにインストールされており、Origin2017をインストールすると、新バージョンにアップグレードされます。

Origin2017をインストールする前に、これらのDLLに依存する別のアプリケーションが既にシステムにある場合があります。この場合には、競合のために、これらのDLLを新バージョンにアップグレードすることができません。従って、Origin2017でPythonを実行すると、「Pythonがインストールされていません」というエラーが表示されてしまいます。

この場合、次の方法で修復することができます。

  1. Windowsコントロールパネルを開き、プログラムと機能をクリックします。
  2. スクロールして、プログラムMicrosoft Visual C++ 2015 Redistributable(x64)-14.0.023026 とMicrosoft Visual C++ 2015 Redistributable(x86)-14.0.023026を探します。
  3. プログラムをダブルクリックして、ダイアログを開きます。
  4. 修復 ボタンをクリックします。
  5. Originを再起動します。


必要なOriginのバージョン: Origin 2017 以降
