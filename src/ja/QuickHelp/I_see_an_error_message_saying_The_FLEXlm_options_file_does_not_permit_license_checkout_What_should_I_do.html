FAQ-1121 "The FLEXlm options file does not permit license checkout"というエラーメッセージが表示されます。どうしたらよいでしょうか？

Opt-File-Not-Permit-Lic-Checkout

最終更新日：2021/12/30

Originがライセンスサーバに接続しようとすると以下のようなメッセージが表示される場合

License Status: 112

The FLEXlm options file does not permit license checkout.

これは、ライセンスサーバーマネージャーがサーバーにアクセスできるユーザを決定するためのオプションファイルにより、そのマシンが許可されていないことが原因です。

この問題を回避するには、ライセンス管理者にオプションファイルについてお問い合わせください。

キーワード:err 112, サーバに接続できない, 3分, ライセンス期限切れ, ネットワーク, 同時起動ライセンス, フローティングライセンス, チェックアウを許可しない
