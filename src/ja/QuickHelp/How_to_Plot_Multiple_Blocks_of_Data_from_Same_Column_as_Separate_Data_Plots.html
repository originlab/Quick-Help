FAQ-718 同じ列にある複数ブロックのデータを異なるデータプロットとして作図するにはどのようにしたらよいでしょうか？

plot-multiple-blocks-data

最終更新日:2015/08/10

データのブロックのサイズに応じて、異なる方法があります。少しのデータブロックで作図する場合、CTRLキーを押しながら、単一列内で複数範囲の選択を行います。

下図のように、同じ列のデータのブロックが空の行で分けられている場合、

メインメニューのワークシート：ワークシートクエリを選択してダイアログを開き、選択条件編集ボックスに以下のコマンドを入力して同じ列内の全てのブロックのデータを選択します。

目的のデータが選択されたら、作図します。下図はボックスチャートを作図したサンプルです。

ワークシートクエリではマスクや、選択データの抽出などの他の機能も使用できます。

キーワード:抽出, マスク, 選択, 複数ブロック, 複数サブ範囲
