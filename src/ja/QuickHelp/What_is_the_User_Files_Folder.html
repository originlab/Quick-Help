FAQ-15 OriginはUser Filesフォルダを尋ねます。これは何でしょうか?

UserFilesFolder

最終更新日: 2019/1/3

User Filesフォルダ (UFF)は、Originに関連したファイルを保存したり、開くためのデフォルトのフォルダです。

User FIlesフォルダを開くには：
  1. ヘルプ: フォルダを開く: ユーザファイルフォルダを選択して直接フォルダを開きます。

または、

  1. 環境設定：オプションをクリックして、システムパスタブを開きます。
  2. ここで、User Filesフォルダ行を選択してから、コピーボタンをクリックして、パスをコピーします。このパスを使って、ウィンドウズエクスプローラにある現在のユーザファイルフォルダを開きます。


User FIlesフォルダを変更するには：
  1. 環境設定：オプションからシステムパスタブを開き、表からUser Filesフォルダの入力を選択して、変更ボタンをクリックするか、スクリプトウィンドウでコマンドchangeUFF -d;を実行します。これにより、User FIlesフォルダの変更ダイアログが開きます。
  2. ブラウズボタンを使って、新しい場所を指定してOKをクリックします。
  3. 変更を反映させるには、Originを再起動します。

キーワード:インストール、User Files、UFF、カスタムファイル、ユーザファイル
