FAQ-896 ワークシートのデータを変更した場合カラーマップを再スケールするには？

Rescale-Colormap-Data-Worksheet-Changes

最終更新日:2018/1/12

等高線図または3Dカラーマップ曲面図では、データが変更されたとき以下の方法で簡単にカラーマップを再スケールできます。

  1. プロット上で右クリックして、カラーマップ: レベルの設定を選択してレベルの設定ダイアログを開きます。データに基づいて、開始と終了の値を自動的に設定するには、最小/最大の検索ボタンをクリックしてください。OK をクリックして、変更を適用します。
  2. プロットをダブルクリックして、作図の詳細ダイアログを開きます。カラーマップ/等高線タブを開き、再スケールモードを自動 に変更します。これで、データを変更したときにカラーマップが自動で再スケールされます。
  3. 以下のLabTalkスクリプトの一つをお試しください。 
    • layer -axyz // カラーマップのみ再スケール
		
    • layer -axy // Z軸とカラーマップを再スケール
    レイヤの再スケールコマンドの詳細は、このページを参照してください。
