FAQ-1189 “HOSTNAME specified in the license file is not available in the local network database” エラーが表示されるのはなぜですか？修正方法は？

Hostname-Not-Available-Linux-Server

最終更新日：2023/8/1

Linux OS (RHEL8.8 など)上でFLEXnetライセンスサーバをセットアップすると、デバッグログに次のエラー メッセージが表示されます。

(lmgrd) Unknown Hostname: <HOSTNAME> specified in the license file is not available in the local network database.

これは、サーバマシン上のhostsファイルの問題が原因で発生します。サーバがホスト名を認識できず、正しく起動できません。


この問題を解決するには、

  • 次の行を追加して、ホストファイル (/etc/hosts) を編集してください。
<Server_Ip_Address >   <HOSTNAME>

キーワード:flexnet, フローティングサーバ, 仮想サーバ, ライセンス, lic, ライセンスが利用できません, KVM
