FAQ-1097 アップグレード後、インポートツールバーボタンの単一ASCIIインポートおよびExcelインポートで、これまでと異なるダイアログボックスが開きます。

ascii-import-excel-button-change

最終更新日:2021/4/9

データコネクタはＯriginの最新インポートメカニズムです。従来のXファンクションベースのルーチンの代わりにコネクタに移行するために、インポートツールバーの単一ASCIIインポートボタンとExcelのインポートボタンを再コーディングして、テキスト/ CSVおよびExcelデータコネクタに直接アクセスできるようにしました。これはOrigin 2021bから変更されています。

この変更を戻して、従来のXファンクションベースのインポートルーチンに再関連付けされるようにするには、システム変数@TIBの値を変更します。 

@TIB = 1; // 「単一ASCIIインポート」ボタンでimpASCを開く
@TIB = 2; //「Excelのインポート」ボタンでimpMSExcelを開く
@TIB = 3; // impASCとimpMSExcelの両方をツールバーボタンに接続

システム変数の値を変更する方法については、このFAQを参照してください。
