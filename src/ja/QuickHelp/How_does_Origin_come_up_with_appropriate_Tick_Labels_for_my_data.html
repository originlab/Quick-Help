FAQ-384 データに対して適切な軸ラベルを表示するにはどうしたら良いですか?

Tick-Labels-by-LT

最終更新日:2015/02/04

正確な軸スケール設定および適切な軸ラベルの問題は、LabTalkのlimitコマンドで扱います。例えば、 

newbook;
range rax = 1!col(a);
rax = data(17.3,23.6,.25);
range ray = 1!col(b);
ray = normal(rax.GetSize());
 
limit ray;  // データセットの制限値を検索する
// limit.xmin = 最小x値
// limit.xmax = 最大x値
 
// 適切な増分の値を計算し、開始と終了の値を調整します
begin = limit.xmin;
end = limit.xmax;
ticks = 5;
limit.size =;
limit.ymin =;
limit.imin =;
limit.ymax =;
limit.imax =;
limit.xmin =;
limit.xmax =;
ty Beginning with From :$(begin), To :$(end), Steps :$(ticks);
limit -r begin end ticks increment;
ty Limit command creates From :$(begin), To :$(end), Increment :$(increment);
 
plotxy (1,2);

開始・終了・増分の値は、軸の開始、終了、増分をセットするのに使用できます。


すべてのOriginの標準的なグラフテンプレートには、上記の計算を行う前に、XおよびYの範囲に8%の余分を付けています。このプロパティは、以下のもので修正できます。

layer.x.rescalemargin
または
layer.y.rescalemargin


キーワード: 範囲、自動スケール、軸
