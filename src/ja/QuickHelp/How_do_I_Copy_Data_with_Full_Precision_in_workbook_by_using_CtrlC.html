FAQ-777 Ctrl + Cを使って完全な精度でデータをコピーするには？

Full-Precision-Copy

最終更新日:2017/8/9

データをコピーして貼り付ける際の一般的な方法は、キーボードのCTRL + Cでコピーし、CTRL + Vで貼り付けするものです。しかし、外部アプリケーションに貼り付けを行う場合、この方法では完全精度を保つことができません。完全数値精度を保ったままコピーするには、CTRL + ALT + Cでコピーします（貼り付けは通常通りCTRL + Vです）。

Originワークシート間でのデータコピーであれば、CTRL + Cで問題ありません。データは完全精度でコピー、貼り付けされます。しかし、外部のアプリケーションにコピー/貼り付けの際にCTRL + ALT + Cを使用するのが面倒な場合、CTRL + Cで完全精度のデータをコピーできるようにOriginの設定を変更できます。これには、システム変数@CWACCのデフォルト値を変更する方法があります。

  • 現在のOriginセッションでのみデフォルトを変更するには、スクリプトウィンドウを開き、次のように入力してEnterキーを押します。
@CWACC=1
この変更により、Ctrl + Cはワークブックのデータを完全な精度でコピーするショートカットキーになり、Ctrl + Alt + Cは表示されている桁数を使用してデータをコピーします。
  • この動作をOriginを起動するたびに有効にするには、このFAQを参照してください。
