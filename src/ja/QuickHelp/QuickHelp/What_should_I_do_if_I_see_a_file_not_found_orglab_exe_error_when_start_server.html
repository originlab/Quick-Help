FAQ-1139 FLEXnet サーバを開始すると"File not found, orglab.exe"というエラーが表示されます。どうしたらよいでしょうか？

Orglab-File-Not-Found

最終更新日：2022/9/20

ライセンスサーバを開始できず、ログファイルに以下のエラーメッセージが表示された場合

9:42:08 (lmgrd) License server manager (lmgrd) startup failed:
9:42:08 (lmgrd) File not found, orglab.exe


lmgrd.exeと同じフォルダ内にorglab.exeファイルがあるか確認してください。LMToolsのConfig Servicesタブでlmgrd.exeのパスを確認できます。

ない場合

  1. こちらからorglab.exe をダウンロードしてください。
  2. ファイルを解凍し、 orglab.exeをlmgrd.exeがあるフォルダに置きます。
  3. FLEXnetサーバを再起動します。


orglab.exe はあるが、flexlmサーバがファイルを見つけられない場合

  1. Originライセンスファイルをテキストエディタで開きます。
  2. 2 行目にorglab.exeファイルへのパスを追加して、ライセンスファイルを次のようにします。 
    SERVER_NAME HOSTID
VENDOR orglab "C:\OriginLicenseServer\Servers\orglab.exe"
  3. FLEXnetサーバを再起動します。

キーワード:ライセンス, orglab, File not found, ライセンスサーバを開始できない, lmgrd
