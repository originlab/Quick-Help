FAQ-891 各分析ツールでの数字の表示桁を制御するにはどうすればよいですか?

Control-Digits-in-AnalysisTool

最終更新日:2017/9/15

分析レポートシートで数値データの桁数を指定

  1. レポートシート上の各表の後にある下矢印をクリックまたは、テーブル上のセルを右クリックをします。
  2. コンテキストメニュー内で桁数表示法を選択します。
  3. レポートでの桁ダイアログで、桁数を指定します。

デフォルトとして保存のチェックをつけると、分析機能のデフォルトとして保存できます。分析レポート機能で作成されたレポートシート全てでこの桁数設定が使用されます。

グローバルに桁数を指定

全ての分析機能で桁数設定を変更したい場合、オプションダイアログの数値の表現形式タブでレポートでの桁を指定します。


キーワード:レポートシート数字, 小数点以下, 有効桁数, レポート表
