FAQ-888 LabTalkを使用して特殊テキストオブジェクト（軸のタイトルなど）のプロパティを設定する方法は？

Set-Special-Text-ByLT

最終更新日:2017/8/31

Originには、下表のような軸タイトルなどの特殊なテキストオブジェクトがあります。

オブジェクト オブジェクト名
下X軸タイトル xb
上X軸タイトル xt
左Y軸タイトル yl
右Y軸タイトル yr
背面Z軸タイトル zb
正面Z軸タイトル zf

このような特殊なオブジェクトでは、objectName.=を実行してすべてのプロパティを取得または設定できます。

xb.fsize = 20; //下X軸タイトルのフォントサイズを設定

yl.fillcolor=2; //左Y軸タイトルにボックスフレームを追加し、塗りつぶしの色を赤に設定 
yl.transparency=50;//透過率を50%に設定
Note: Run objectName.= ではほとんどすべてのオブジェクトプロパティダイアログを確認できます。総合的なプロパティについては、このページで説明しています。


キーワード:LabTalk, テキストオブジェクト, 軸タイトル, プロパティ
