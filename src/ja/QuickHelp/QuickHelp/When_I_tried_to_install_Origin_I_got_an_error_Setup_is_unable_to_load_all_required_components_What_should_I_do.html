FAQ-1126 Originをインストールしようとすると、「必要なすべてのコンポーネントをロードできません」というエラーが表示されました。どうしたらよいでしょうか？

Setup-Unable-Load-Comp

最終更新日：2022/5/7

次のようなエラーが発生してOriginインストールできないことがあります。

必要なすべてのコンポーネントをロードできません

これは、インストールファイルが壊れているか不完全であることを示しています。あるいは、インストールの一部がローカル セキュリティ要素によってブロックされています。次の解決法をお試しください。

  • OriginLabのWebサイトからインストールファイルを再ダウンロードします。実行する前に、ダウンロードが完了していることを確認してください。
  • 完全なセキュリティクリアランスでインストールファイルを実行します。管理者アカウントでコンピュータにログオンしてください。また、インストール中だけウイルス対策ソフトウェアを一時的にオフにして、ブロックされないようにすることができます。
