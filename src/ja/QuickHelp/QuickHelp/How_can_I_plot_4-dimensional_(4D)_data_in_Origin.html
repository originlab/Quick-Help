FAQ-184 Originで4次(4D)データをプロットできますか?

Plot-4D-Data

最終更新日: 2019/2/18

プロットしたいデータが4列ある場合は、最初の3列をXYZ散布図としてプロットし、次に4列目を使用してシンボルの色を設定できます。

  1. 3列目をZに設定してからこの列を強調表示し、作図> 3D：3D散布図を選択します。
  2. プロットをアクティブにした状態で、フォーマット：プロットを選択し、作図の詳細ダイアログを開きます。
  3. シンボルカラーをポイント毎＞カラーマップに設定し、列のリストから4列目を選択します。(下記のnoteを参照 )

これをさらに一歩進め、XYZデータを行列に変換してから、散布点を通るワイヤフレームプロットとしてプロットすることもできます。

Note:

データセットの値を使ってデータプロットの色を制御する3つのオプションがあります。

  1. インデックス：このオプションを使用すると、Origin カラーパレットの色の順序（インデックス番号）を示す値を持つデータセットを指定できます。
  2. RGB直接：RGB合成値は、赤、緑、青の3成分成分から計算されます。R、G、およびBコンポーネント値は0から255の範囲です。
  3. カラーマップ：このオプションを使用して、2D Y値または3D Z値の範囲とそれに関連する色のスケールの間のマッピング関係を作成します。作成したカラーマップに基づいて、2D Y値や3D Z値がプロットのデータポイント色要素として利用されます。

他のインデックス作成オプションを使用すると、このアイデアを拡張して、3D XYZプロットの最大8次元のデータを表すことができます。

  1. Xデータ
  2. Yデータ
  3. Zデータ
  4. サイズ - col（列名）オプションの1つを選択
  5. エッジの色 - col（列名）オプションの1つを選択します（2Dシンボルタイプの場合のみ）
  6. 塗りつぶしの色 - col（列名）オプションの1つを選択します（2Dシンボルタイプの場合のみ）
  7. 形状 - col（列名）オプションの1つを選択します（2Dシンボルタイプの場合のみ。3Dシンボルタイプは5つの形状のみをサポートし、データセットを使用したシンボル形状の指定はサポートしません）
  8. 内部 - col（列名）オプションの1つを選択します（2Dシンボルタイプの場合のみ）

追加の列はすべて、ワークシートのZ列のすぐ右側になければなりません。


キーワード: プロット, 4d
