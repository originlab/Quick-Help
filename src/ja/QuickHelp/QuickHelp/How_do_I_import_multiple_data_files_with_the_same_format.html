FAQ-413 複数のデータファイルを同じ形式でインポートするには、どうしたら良いでしょうか?

Import-Multi-Data

最終更新日: 2019/1/6

インポートウィザード（データ：ファイルからインポート：インポートウィザード）と複数ASCII（データ：ファイルからインポート：複数ASCII）の両方の方法で、1つ以上のファイルを選択でき、すべてのファイルが同じカスタマイズ可能な設定グループを使用してインポートされます。

  • インポートウィザードについては、インポートウィザード を参照してください。
  • 複数ASCIIインポートについては、ASCIIインポートオプションのカスタマイズを参照してください。

インポートウィザードでフィルタ（* .OIFファイル）を作成したり、複数ASCIIインポートでテーマを保存したりすることで、特定のインポートオプションのセットをファイルに保存することができます。これにより、より複雑なファイルをドラッグアンドドロップ（インポートウィザード）したり、「最近利用した」メニューコマンドからテーマを選択してインポートすることができます。

キーワード ： 繰り返し、自動化、自動化、バッチ、フィルタ、OIF、テーマ、ドラッグアンドドロップ
