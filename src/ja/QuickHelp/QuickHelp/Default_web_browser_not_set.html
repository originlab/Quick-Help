FAQ-787 Originのライセンスをオンラインで入力しましたが、起動できません。どうしたらよいでしょうか?

Default-Web-Browser

最終更新：2019/01/03

PCに、デフォルトのウェブブラウザの設定がないか、デフォルトのウェブブラウザを開く際に、ファイルの関連付けがなされていません。

デフォルトのウェブブラウザをを設定するには：

下記のウェブブラウザ（一般的なもの）から、デフォルトにしたいものを選択して、インストラクションに従ってください。

  • Microsoft Internet Explorer:コントロールパネルまたは設定からインターネットオプションを開きます。 Internet Explorerのオプションメニューからプログラムタブのデフォルトに設定をクリックし、もう一度Originを起動してください。
  • Mozilla FireFox:メニューから「オプション」→「オプション」を選択します。詳細（タブ）をクリックして、システムデフォルトにある「一般」タブで「Firefox が既定のブラウザか確認する」にチェックを入れてから、Originを再起動します。
  • Google Chrome:Chromeの右上にある3本の水平バーをクリックして、メニューを開き、「設定」を選択します。ボタンをスクロールします。デフォルトブラウザの下で、「Google Chromeを規定のブラウザにする」をクリックして、Originを再起動します。
ファイルを関連付けるには：

デフォルトウェブブラウザが設定されていることを確認した後、まだ解決しないようでしたら、「ファイルの関連付け」をデフォルトウェブブラウザで修正して、ファイルを開く必要があります。

Windows Vista, 7, 8, 8.1,10

  1. コントロールパネルを開いて、「既定のプログラム」をクリックします。
  2. 「ファイルの種類またはプロトコルのプログラムへの関連付け」をクリックします。
  3. .htmと.htmlまでスクロールします。「現在の既定プログラム」がご自身のデフォルトウェブブラウザに設定されていることをご確認ください。
  4. もし、設定されていない場合、「プログラムの変更」をクリックして、 .htmと.htmlの両方に、ご自身のウェブブラウザを設定します。最後までスクロールすると、「プロトコル」の項目が見つかります。FTP、HTTP、HTTPSが、全てデフォルトのブラウザになっていることをご確認ください。
  5. Originを再起動します。
