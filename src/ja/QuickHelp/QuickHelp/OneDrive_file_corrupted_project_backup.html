FAQ-1127 OneDriveに同期されたファイルを開くと、エラーが発生します：ファイルが破損しています

OneDrive-file-corrupted-project-backup

最終更新日：2022/02/08

Microsoft OneDriveアカウントに同期されたフォルダからOriginファイルを開くと、一部のユーザは、ファイルが破損しているが、以前のバージョンのファイルは引き続き使用できる可能性があるというエラーメッセージが表示されると報告しています。ユーザは、ヘルプ：フォルダを開く：プロジェクトのバックアップを選択してバックアップファイルを取得するように案内されます。

ユーザーからの報告は次のとおりです。

  • エラーメッセージに対してOKをクリックすると、ファイルは正常に開き、そのまま表示される。
  • ただし、ユーザがファイルを保存しようとすると、Originはデフォルトで\ Backupに保存します。これには、OneDriveフォルダを参照し、その場所にファイルを保存する必要があります。

このエラーメッセージは、ユーザアカウントに完全な管理者権限がある場合は表示されません。詳細はまだ調査中ですが、ユーザから、通常のユーザアカウントから権限が昇格された管理者アカウントに切り替えることで問題が解決したようだと報告されています。

キーワード：破損, 開けない, バックアップ, cloud, クラウド, 管理者, 権限, 管理者権限
