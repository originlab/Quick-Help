FAQ-388 forループのサイズの上限を超える場合は、どのようにしたらよいでしょうか？

最終更新日:2015/02/04

for-loop-exceed-limit

LabTalkでは、for ループコマンドのスクリプト長は、スペースを含む3072文字までに制限されています。もし、スクリプトが長すぎる場合、define コマンドを使用してマクロを定義し、ループ内でマクロを呼び出します。

例： 

// 3つのマクロを定義
define Part1 {ty one;}; 
define Part2 {ty two;}; 
define Part3 {ty three;}; 
 
// forループでマクロを呼び出す
int ii=0; 
for (ii=0;ii<3;ii++) 
{ 
Part1; 
Part2; 
Part3; 
}

キーワード: ループ, loop, マクロ定義, 文字数制限, for, マクロ
