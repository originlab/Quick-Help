FAQ-792 Origin2016を使って、FLEXnetサーバー に接続しようとすると、「Your FLEXlm license file does not allow the requested feature.」というメッセージが表示されますが、なぜでしょうか?

FLEXnet-Server-Error

最終更新日:2015/11/02

Origin2016の同時起動版には、FLEXnet server version 11.13.0.2か、それよりも新しいバージョンが必要になります。

FLEXnetサーバーバージョンが古い場合、次のようなエラーメッセージが表示されます。
"Your FLEXlm license file does not allow the requested feature".

OriginのFLEXnetサーバーを次のインストラクションに沿って、アップデートしてください。

http://www.originlab.com/index.aspx?go=support&pid=3215
