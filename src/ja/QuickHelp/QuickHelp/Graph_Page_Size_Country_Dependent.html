FAQ-217 印刷時グラフサイズを国によって変更する方法は？

Graph-Page-Size-Country-Dependent

最終更新: 2018/08/03

グラフを印刷するとき、Originはその国の標準を参照してデフォルトグラフサイズをカスタマイズします。US、カナダ、メキシコでは、幅6400 ピクセル(10.67インチ) 高さ4800 ピクセル(8インチ)、US、カナダ、メキシコ以外では、幅6779ピクセル (11.3 インチ) 高さ4760 ピクセル(7.933 インチ) となります。

設定を変更するには以下のように操作します。

  1. Windowsシステムの場所をセットします（コントロールパネルの時計、言語、および地域）。地域がUnite State と設定されているとします。
  2. Originを開き、グラフを作成します。このグラフをアクティブにして、灰色の領域で右クリックし、ページをプリンタに合わせるを選択します。このオプションにより、システムの地域の設定を参照して、グラフの印刷サイズを自動調整します。
  3. グラフが印刷に適当なサイズであることを確認したら、ウィンドウ: スクリプトウィンドウをクリックして以下のLabTalkスクリプトを入力します。そして、スクリプトを選択してEnterキーを押し、空のグラフを作成します。 
    page.width=;
page.height=;

    以下のスクリプトを実行して設定することもできます。スクリプトウィンドウを閉じます。 

    page.width=6400
page.height=4800
  4. グラフをアクティブにして、ファイル: 印刷を選んでグラフページを印刷します。


Note: 特定のグラフで、ページをプリンタに合わせるを選択したあとのみ、標準グラフページサイズは動作します。このオプションなしの他のグラフは、国固有のページサイズに従わず、グラフテンプレートの設定に従います。


キーワード: グラフページサイズ, 印刷, 国依存

必要なOriginのバージョン: Origin 2015 SR0以降
