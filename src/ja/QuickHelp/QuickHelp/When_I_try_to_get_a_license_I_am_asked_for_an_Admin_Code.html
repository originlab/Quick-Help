FAQ-35 ライセンスを取得するとき、アクセスコードを尋ねられます。これは何ですか、なぜ入力しますか?これはなぜですか、どのように修正したらよいですか?

Admin-Code-for-Licensing

最終更新日:2015/02/04

アクセスコードは、複数ユーザパッケージにシリアル番号ラベルだけでなく、ライセンス証書にも記載されています。ソフトウェアの管理者だけがアクセスコードを入力でき、ライセンスアクセス制限をセットアップできます。また、ユーザもライセンス認証処理中にアクセスコードを入力する必要があります。ライセンスアクセス制限について質問があれば、 サポートまでお問い合わせください。
