FAQ-1152 "license manager: can't initialize:Cannot find license file."というエラーが表示されます。どうしたらよいでしょうか？

Cant-Initialize-Cant-Find-Lic-File

最終更新日：2022/9/21

FLEXnet ライセンスサーバを開始できず、ログファイルに以下のようなエラーメッセージが表示された場合

10:38:30 (lmgrd) license manager: can't initialize:Cannot find license file.
10:38:30 (lmgrd) The license files (or license server system network addresses) attempted are
10:38:30 (lmgrd) listed below. Use LM_LICENSE_FILE to use a different license file,
10:38:30 (lmgrd) or contact your software provider for a license file.
10:38:30 (lmgrd) Filename: "C:\flexlm\Originlicense.lic"
10:38:30 (lmgrd) License Path: "C:\flexlm\Originlicense.lic"
10:38:30 (lmgrd) FlexNet Licensing error:-1,359
10:38:30 (lmgrd) System Error:2 No such file or directory
10:38:30 (lmgrd) Using license file "C:\flexlm\Originlicense.lic""


この問題を解決するには、

LMTOOLSのService/License FileタブでLMTOOLS ignores license file path environment variablesにチェックを付けてください。

キーワード:同時起動, -1,359, 初期化できない, ライセンスサーバに接続できない, ライセンスファイルが見つからない, System Error 2, そのようなファイルまたはディレクトリは存在しません
