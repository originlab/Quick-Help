FAQ-486 Origin 8.5で作成したワークシート上のテキストラベルが、新しいバージョンで開いたときに大きく表示される場合、どのようにしたら良いでしょうか？

text- label-bigger

最終更新日:2015/02/04

Origin 8.5で作成したワークシート上のテキストラベルが、新しいバージョンで開いたときに大きく表示されることがあります。

この問題を解消するには、関連するOPJ/OTW ファイルを開き、メインメニューからウィンドウ：スクリプトウィンドウと選択します。スクリプトウィンドウに以下を貼り付けます。 

doc -e w
{
	page.textsizecorr=1;
	doc -uw;
}

スクリプトウィンドウでこのスクリプトを選択し、Enterキーを押します。この操作は、Origin 8.6以降のバージョンで動作します。

キーワード: ワークシート, テキストラベル, サイズ, 大きくなる, Origin 8.5
