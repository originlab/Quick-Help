FAQ-836 Originのプロジェクトエクスプローラと同じフォルダー構成で、Windowsのサブフォルダーにグラフをエクスポートするには？

Export-Graphs-in-SubFolders

最終更新日:2016/09/19


プロジェクトにAとBという2つのフォルダーがあるとします。また、graphA、graphBというグラフが、それぞれのフォルダーに1つずつあります。次のように、プロジェクトフォルダーの構造に沿って、両方のグラフをエクスポートしたい場合は、

--<project folder>/graphs/folderA/graphA.jpg
--<project folder>/graphs/folderB/graphB.jpg

<PE Path>という新しい表記を使うと便利です。<PE Path>を使って、Originのプロジェクトフォルダーにあるフォルダー構造に沿ってウィンドウズサブフォルダーにグラフをエクスポートできます。上記の場合を例にすると、サブフォルダーにグラフをエクスポートするには、

  1. グラウフィンドウをアクティブにして、ファイル： グラフエクスポートを選び、expGraphダイアログを開きます。
  2. パスのドロップダウンリストから、オプションの<Project Folder><PE Path>を選択します。
  3. 必要があれば、その他についても指定します。OKをクリックしてエクスポートします。

キーワード:グラフエクスポート、フォルダ構造、エクスポートパス
