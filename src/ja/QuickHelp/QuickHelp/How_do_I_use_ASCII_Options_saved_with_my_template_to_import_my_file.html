FAQ-410 テンプレートにASCIIインポートのオプションを保存するにはどうしたら良いでしょうか?

Save-Import-Settings

最終更新日: 2019/1/6

  1. ASCIIインポートのカスタマイズ
    1. データ：ファイルからインポート：単一のASCIIまたはデータ：ファイルからインポート：複数のASCIIを選択します。
    2. ファイルを選択するために開いたASCIIダイアログで、オプションダイアログを表示のチェックボックスがオンになっていることを確認します。
    3. ファイルを選択して開くをクリックします。これにより、impASCダイアログボックスが開きます。
    4. 必要に応じてインポート設定をカスタマイズします。
    5. OKをクリックします。（Note：Origin 9.0以前では、OKをクリックする前に、ダイアログテーマボックスの右にある矢印ボタンをクリックして、<シートに保存>を選択してください。
    6. インポート後、インポート設定はワークシートにあります。
    7. さらに分析することができます。
  2. インポート設定をテンプレートに保存
    • メニューのファイル：テンプレートの新規保存...を選択します。
    • データをインポートした後にワークシート内で分析と計算が行われるため、ワークシート内に緑色のロックが表示される場合は、ファイル：ブックを分析テンプレートとして保存...を選択します。
  3. カスタマイズされたASCII設定で新しいデータをインポートします。
    1. ファイル：開くを選択して保存したテンプレートテンプレートを読み込みます
    2. データ：ファイルからインポート：単一のASCIIまたはデータ：ファイルからインポート：複数のASCIIを選択します。
    3. ASCIIダイアログでは、オプションダイアログを表示の隣に<sheet>が表示されます。これは、ワークシートにインポート設定があることを意味します。 チェックを外してインポートすると、カスタマイズされた設定でデータが自動的にインポートされます。確認して開くをクリックすると、カスタマイズされた設定が読み込まれた状態でImpASCダイアログが開きます。


テンプレート保存の詳細については、[1].を参照してください。
分析ダイアログボックスのテーマの詳細については、ダイアログテーマ.を参照してください。
