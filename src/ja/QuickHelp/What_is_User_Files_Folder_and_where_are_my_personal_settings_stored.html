FAQ-355 ユーザフォルダとは何ですか、個人用の設定はどこに保存されていますか?

UFF-and-Personal-Settings

最終更新日: 2019/1/3

マシンの各ユーザは、Originを初めて起動したときに、そのOriginユーザのデフォルトの場所になるパス（(ユーザファイルフォルダ）を指定するように指示されます。設定、テーマ、テンプレート、プロジェクトファイルなどのユーザーカスタムファイルはこの場所に保存されます。この保存場所には、ユーザーに対する読み取りおよび書き込み権限が必要です。

NOTE:

リムーバブルドライブを選択した場合、そのドライブはOriginが使用されている間ずっと利用可能でなければなりません。ネットワークドライブを選択した場合、ネットワークパフォーマンスはOriginのパフォーマンスに影響します。


ユーザーファイルフォルダを見つけるにはどうすればよいですか。

ユーザーファイルフォルダを見つける方法はいくつかあります。

  1. Origin 2017以降、ヘルプ：フォルダを開く：ユーザーファイルフォルダメニューを選択してください。フォルダがWindowsエクスプローラで開きます。
  2. 環境設定：オプションメニューを選択します。システムパスタブをクリックします。User Filesフォルダがそこにリストされています。
  3. ウィンドウ：スクリプトウィンドウメニューを選択します。％Y =と入力してEnterキーを押します。パスはスクリプトウィンドウに表示されます。
  4. Windowsのスタートメニューから、すべてのプログラムを選択します：OriginLab：Origin ##：ユーザファイルフォルダショートカットがそこに表示されます。
