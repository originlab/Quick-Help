FAQ-292 ワークシートデータ列の計算はどのように実行したらよいですか?

Math-Between-Cols

最終更新日:2015/02/04

値の設定ダイアログを使うダイアログを開くには、メニューから列：列値の設定を選択するか、列を選択して右クリックし、列値の設定を選択します。

このダイアログには、組み込みの数学関数だけでなく、サンプルの数式もあります。関数の検索と挿入ボタンをクリックして、ダイアログを開き、キーワードで関数を検索し、入力することができます。

ワークシートの各列は、それに関連する数式を持つことができます。他の列の式を表示するには、 と ボタンをクリックして、選択列を移動します。


キーワード: データセット, ベクター, 数学, 単純

必要なOriginのバージョン:2015SR0
