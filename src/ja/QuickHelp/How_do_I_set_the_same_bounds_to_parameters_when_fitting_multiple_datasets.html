FAQ-223 複数データセットをフィットするとき、パラメータを同じ境界条件にするには、どうしたら良いでしょうか?

Fitting-Multiple-Data-Param-Bounds

最終更新: 2018/07/14

Originでは複数のデータセットのフィットパラメータを共有して、境界値として設定できます。そして、パラメータの共有を停止することで、境界値のみを共通化できます。

パラメータy0を共有したい場合は、パラメータタブでy0のパラメータに共有チェックボックスをクリックします。 (複数データフィットモードがグローバルフィットの時のみ有効)。パラメータのリストは、共有にチェックが入っていると、y0のみを表示するように更新されます。境界をクリックすると、下側境界を不可から < または <= のいずれかに変更でき、LB値をダブルクリックして値を入力できます。上側境界に対して繰り返します。そして、「パラメータ」タブに戻り、「共有 」チェックボックスのチェックを外します。「境界」 タブで、y0, y0_2などはすべて同じ上限および下限に設定されていることを確認してください。

キーワード: 複数, 共有, パラメータ
