FAQ-821 積み上げグラフにレイヤを交換/挿入するには？

replace-layer

最終更新: 2018/07/31

新規のデータにレイヤを交換する。

作図済の積み上げグラフにある一つのレイヤに、違うデータを入れ替える場合、新規のレイヤを埋め込む必要はなく、単純に交換したいレイヤ上のプロットを選択して右クリックしてメニューを開きます。メニューからX/Yを変更 を選択して、さらに交換したい新しいデータを選びます。

好感したいデータが別のワークブックやワークシートにある場合は、グラフウィンドウの左上にある、レイヤ番号でダブルクリックし、レイヤ内容 ダイアログを開きます。ダイアログの右側パネルから削除したいプロットを選択して中央の削除矢印をクリックします。左側パネルから追加したいプロットを選択し、中央の追加矢印をクリックして右側パネルに移します。

新規グラフでレイヤを交換する。
  1. 不要なレイヤを選択し、キーボードの削除 キーを押すと削除出来ます。
  2. 交換したいグラフウィンドウにあるレイヤをコピーして、積み上げグラフウィンドウで貼り付けします。
  3. 目的のグラフウィンドウのいずれかのレイヤを右クリックし、フォーマットのコピー: 寸法を選び、貼り付けるレイヤを選択し、右クリックしてフォーマットを貼り付けるを選んで、貼り付け先のレイヤの設定を変更します。
  4. 画面の右下にあるオブジェクト操作 ツールバーで貼り付けたレイヤを正しい位置に移動します。
積み上げグラフにレイヤを挿入する

Origin 2016 SR0から、追加機能アプリを利用することが出来ます。指定したレイヤを交換するには、まず、Layer Stack Manager アプリ opx file をダウンロードし、ドラッグ＆ドロップでOriginiにインストールします。OriginアプリのWebページにある Description フィールドを読んで、積上げグラフにレイヤを挿入するアプリの利用方法をご確認ください。

キーワード:レイヤの置き換え, レイヤの挿入
