FAQ-435 MATLABワークスペースでデータを入力/出力するには？

Matlabconsole-Importdata

最終更新日: 2019/1/8

Originでは、MATLABコンソールを使用し、MATLABからデータをインポートしてワークシートや行列に入力することができます。反対に、Originからは、MATLABワークスペースで直接呼び出せる変数としてデータをエクスポートできます。 この機能は、1台のコンピュータにOriginとMATLABの両方がインストールされている必要があります。

MATLABコンソールを起動するには、メニューの接続：MATLABコンソールを選択しMATLABコンソールダイアログを開きます。パネルでdesktopを実行します。このMATLABコマンドは、通常のMATLABデスクトップを開きます。MATLABにデータをインポートした後、OrignのMATLABコンソールに戻り、インポートボタンをクリックするとMATLABインポートダイアログが開きます。既存データを選択し、ワークシートにインポートまたは、行列にインポートをクリックし、Originにデータをインポートします。

逆に、OriginからMATLABデスクトップにデータをエクスポートするには、Originメニューから接続：MATLAB コンコールを選択して、MATLABコンソールダイアログを開きます。パネル内でデスクトップを実行し、通常のMATLABデスクトップを開きます。あとはエクスポートボタンをクリックすれば、MATLABワークスペースにデータをエクスポートし始めます。

キーワード：MATLABコンソール, データインポート, 変数をエクスポート, MATLABワークスペース
