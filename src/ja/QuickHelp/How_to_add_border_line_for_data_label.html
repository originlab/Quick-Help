FAQ-730 データラベルに境界線を追加するには？

add-border-for-label

タイムラインプロットのような、ラベルを設定したプロットを作成するとき、データラベルの周りに境界線を追加したい場合があります。その際は、以下のようにしてみてください。

  1. ラベルを設定したグラフをアクティブにします。メニューから、フォーマット：プロットを選択し、作図の詳細ダイアログボックスを開きます。
  2. ラベルタブの「枠」ドロップダウンリストでBox かShadowを選択し、境界色ボタンと塗り色ボタンで色を変更します。
  3. 余白(フォント高さの%)コンボボックスの値を変更し、ラベルの上下左右にある余白を調整します。
  4. OK ボタンをクリックします。

Ctrl キーを押しながらデータポイントをダブルクリックし、作図の詳細ダイアログを開きます。この操作で特定ポイントが追加されます。左側パネルでこの特定ポイントを選択した状態でラベルタブに移ると、前述の4つのオプション、枠、境界色、塗り色、余白(フォント高さの%)がデフォルトで自動にセットされます。このことは、特定ポイントの設定がプロット全体の設定に従うことを意味します。必要に応じて、これらのオプションをカスタマイズすることもできます。

キーワード：境界線, データラベル, 枠, ボックス, 影
