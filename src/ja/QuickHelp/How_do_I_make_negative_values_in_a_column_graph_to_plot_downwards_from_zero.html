FAQ-178 縦棒グラフの負の値を0から下向きに表すにはどうしたら良いでしょうか?

Plot-Negative-Cloumn-from-0

最終更新日:2015/02/04

X 軸をY＝0の位置に配置する場合、

  1. メインメニューのフォーマット：軸スケール：X軸を選ぶか、X軸をダブルクリックして軸ダイアログを開きます。
  2. 軸と軸目盛タブを開き、左パネルで下アイコンが選択されていることを確認します。
  3. 軸位置ドロップダウンリストから位置＝を選択します。
  4. パーセント/値テキストボックスに0 を入力します。
  5. OKをクリックします。


下向きの負の縦棒グラフをX軸に表示したい場合、Y=0の線を有効にするなどの操作が必要です。

  1. メインメニューのフォーマット：軸スケール：X軸を選ぶか、X軸をダブルクリックして軸ダイアログを開きます。
  2. 軸と軸目盛タブを開き、左パネルで下アイコンが選択されていることを確認します。
  3. 軸位置ドロップダウンリストから下を選択します。
  4. グリッド線タブを開きます。
  5. 追加の線グループのY=0チェックボックスにチェックをします。もしこのチェックボックスにチェックが付いていない場合、Yスケールで最小のY値から上の方向に棒が作図されます。



キーワード: ボタン, 縦棒グラフ, 縦棒プロット, 横棒グラフ, 上向き, 下向き

必要なOriginのバージョン:2015SR0
