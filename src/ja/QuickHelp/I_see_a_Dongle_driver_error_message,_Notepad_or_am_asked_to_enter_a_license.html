FAQ-711 ドングルドライバのエラーメッセージやメモ帳が表示され、ライセンスを入力するよう要求されます。何が悪いのでしょうか?

Dongle-driver-error


最終更新日：2022/5/18

ドングル版のOrigin を実行しようとするとメモ帳が開くのはなぜですか?

ドングルを接続すると「ドライバが見つかりません」と表示されるのはなぜですか?

ドングルを接続すると、「指定されたファイルが見つかりません」と表示されるのはなぜですか?

3 分間で期限が切れるという警告が表示されるのはなぜですか?

Originの起動時にライセンスの入力を求められるのはなぜですか? この問題については、お問い合わせください。

Note: 最近のWindowsアップデートを使用している場合は、更新されたドライバー(下記) をインストールする前にコンピューターを再起動してください。これにより、ドライバをインストールしなくても問題が解決する場合があります。

ドングルの Origin および OriginPro を実行するために、新しいドングル ドライバが利用できるます。この最新のドライバは以下の問題を修正します。

  • 同じPCにインストールされているOrigin/Pro 8.x と 9.x のドングル ドライバー競合する。
  • Windows XP 64bit、Windows Vista 32bitおよび 64bit、Windows 7 32bitおよび 64bit、Windows 8 32bitおよび 64bitでのOrigin/Pro 8.x および 9.x の互換性。
  • Origin/Pro 8.5 SR1 インストーラで正常にインストールできない。
  • Origin/Pro 7.5、8.x、および 9のWindowsのスリープ（スタンバイ）モードの問題。
  • Windows Vista および Windows 64bitでの Origin/Pro 7.5 の互換性。


次の手順で、新しいドングルドライバをインストールできます。

  1. Origin(Pro)が起動している場合は終了します。
  2. ドングルをUSBポートから取り出します。
  3. ドングルドライバをダウンロードして保存します。
  4. Install.exeファイルをダブルクリックしてドライバのインストーラを実行します。
  5. Uninstallにチェックを付けてBegin Uninstallをクリックします。**これは、古いドライバがインストールされていたり、不適切にインストールされている場合に、それを一掃するために必要なステップです。
  6. アンインストールが完了したらFinishをクリックします。
  7. 再度Install.exeファイルをダブルクリックしてドライバインストーラを実行します。
  8. USB チェックボックスの両方(2と3)にチェックを付けます。Parallelsソフトウェアを使っている場合、Parallelドングルのみ選択します。Begin Installをクリックします。
  9. インストールが完了したらFinish をクリックします。
  10. ドングルをUSBポートに接続し、Windowsの新しいハードウェアの追加ウィザードでインストールが完了するまで待ちます。
  11. Originを起動します。


Originの起動時にライセンスの入力を求められた場合お問い合わせください。
