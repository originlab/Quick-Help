FAQ-969 「インポートした設定をシートに保存」とインポートダイアログの「シートに保存」は何が違うのでしょうか？

Save-to-Sheet-Theme-Diff

最終更新：2018/09/19

インポートダイアログを使用してデータをインポートするとき、ダイアログテーマのフライアウトメニューには、インポート設定をシートに保存するための2つのオプションがあります。たとえば、ImpASCダイアログでASCIIファイルをインポートし、ダイアログテーマの右側にある三角形のボタンをクリックすると、次の2つのメニュー項目が表示されます。

  • シートにインポート設定を保存
  • <シート>に保存する

これら2つの違いは何ですか？

「<シート>に保存する」は、インポート設定をX-Functionダイアログテーマに保存します。このテーマは、このX-Functionを使用して読み込むときに読み込むことができます。ただし、X-Functionを使用しない場合、たとえばインポートするデータファイルをドラッグアンドドロップすると、その設定は使用できません。

一方、この不足をカバーするために、「シートにインポート設定を保存する」が導入されています。インポート設定をページフィルタに保存します。このフィルタは、ファイルをワークブックウィンドウにドラッグアンドドロップすると自動的に使用されます。

キーワード：インポート設定の保存、ドラッグアンドドロップ
