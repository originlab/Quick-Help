FAQ-1177 フィット関数で入力データに関連する定数を定義するにはどのようにしたら良いでしょうか？

Define-Constant-Relate-to-Input-Data

最終更新日：2023/6/6

ユーザ定義フィット関数で入力データに関連する定数がある場合、それをパラメータとして定義して初期式に保存できます。

例えば、以下のフィット関数を定義する場合 

y = y_min + (y_max-y_min)*exp(-k*x)

ここで、k は計算するパラメータ、y_max および y_min は入力データに関連したパラメータ定数で、y_max はYデータセットの最大値、y_min は最小値です。

Originで定数y_max およびy_min を定義する方法

  1. フィット関数ビルダーの変数とパラメータページで、k、y_max、y_min をパラメータとして定義します。
  2. パラメータ初期化ルーチンページで、三角形のボタンをクリックして希望の式を選択し、y_maxとy_minの初期式を指定します。
  3. 固定パラメータの初期化を許可するにチェックを付けて、 y_maxおよびy_minを初期値に固定します。

キーワード:ユーザ定義関数, 初期値, 非線形曲線フィット, 定数, メタデータ
