FAQ-346 高解像度のスクリーンで、Originの表示に問題がある場合は、どうしたらよいでしょうか？

Mac-Retina-Display-of-Origin

最終更新日：2022/4/7

一般的なPCモニター (最大 227 ppi vs 96 ppi)よりも高い解像度のディスプレイが増えてきています。Originはネイティブ解像度での機能ですが、中システムテキストスケール(125％)で、 1920 X 1200の解像度を使用するとよいとのテスト結果が示されました。

高解像度のディスプレイで、例えば、Origin 3200 X 1800の解像度でQHD表示をする場合、メニューやダイアログのボタンとテキストが、Originで極端に小さくなってしまいます。この問題を解決するには、次を行ってください。

Origin 2022bから、画面の解像度とDPIに応じてウィンドウのサイズと位置が調整されるので、ノートPCと高解像度のデスクトップモニター間でのプロジェクト共有が簡単になります。この機能は、システム変数@SRWSを使って制御/調整できます。

方法1

  1. 表示: ツールバー...を選択して、カスタマイズダイアログを開きます。
  2. 「オプション」タブをアクティブにします。大きいアイコンを使用 と システムフォント設定のスケールツールバーチェックボックスを選択します。
Note:システムフォント設定のスケールツールバーチェックボックスは、分かれたツールバーボタンの機能の妨げになります。(複数ツールボタンは、最後に使ったツールの後ろに重ねられます。）

方法1では解決しない場合、方法2を試してください。

方法2

PCの解像度とDPIスケールを以下のように調整してください。

  • PCの解像度を1900 X 1200より低くしてください。
  • DPIを上げて、大きいアイコンにします。
Note: 画面の解像度やシステムテキストスケールを変更した後は、Windowsをログアウトしてから、再度ログインして、新しい設定を反映させる必要があります。

AppleのMacBook Retinaディスプレイで、Originのディスプレイに問題が起こる

AppleのMacBook Retinaディスプレイは高解像度で、Originのディスプレイに問題が起こることがあります。このときは、次のようにして解決します。MacでOriginを起動するに推奨されるスクリーンの設定は、「中」システムテキストスケール(125％)で、解像度1920 X 1200です。

テスト環境:
ハードウェア: MacBook Pro, 2.0GHz Quad-core Intel i7, 15.4" Retina display (2880 X 1800, 220 ppi).
ソフトウェア: OS X Version 10.9.5, BootCamp 5.1, Windows 8.1, OriginPro 2015.

MacでOriginを起動する方法についての詳細情報はこのwebページを確認してください。

Note:Retina ディスプレイでWindowsの起動をサポートしている最近のParallelsとVMwareです。詳細は、お使いのソフトウェアの資料もご参照ください。
