FAQ-1163 データコネクタにファイル拡張子を追加するには?

Add-File-Extension-Data-Connector

最終更新日：2023/1/28


アプリウィンドウのコネクタタブでインストール済みのコネクタを確認できます。データコネクタアイコンを右クリックしてファイル拡張子の編集...を選択します。

ダイアログで、ワイルドカード文字*（アスタリスク）の後に拡張子を記述することで、拡張子を追加できます。リスト内の拡張子はそれぞれ;（セミコロン）で区切ります。

拡張子を記述する際に誤ってスペースや空白文字を含めないように注意してください。また、;と*の間にスペースを入れないでください。

キーワード:ファイルの種類, カスタマイズ
