FAQ-214 作図形式を変更するには、どのようにしたら良いでしょうか？

Change-Plot-Type

最終更新日:2015/02/04

作図のセットアップダイアログボックスで、自由にグラフタイプを変更することができます。グラフタイプを変更するには

  1. 『プロットリスト』ボックス内のデータプロットを選択します(下側パネル)。
  2. 『 グラフタイプ 』リストでグラフを選択します(中央のパネル)。
  3. 「置き換え」ボタンをクリックします。
  4. OK ボタンをクリックして、プロットタイプを変更します。

キーワード: 変更, 作図形式, 作図タイプ, 作図のセットアップ
