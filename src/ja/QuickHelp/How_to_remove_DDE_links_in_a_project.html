FAQ-1134 プロジェクト内のDDEリンクを削除するには?

Remove-DDE-Link

最終更新日：2023/5/5

プロジェクトにDDE (Dynamic Data Exchange) リンクがある場合 (たとえば、ワークシートのセルが外部Excelファイルのセルにリンクされている場合)、Excelファイルにアクセスできない他のユーザとopjuを共有するのが難しいことがあります。

代わりに、数式/リンクを削除ダイアログを使用して、プロジェクト内のすべてのDDEリンクを値に変換後、他のユーザと共有できます。

  1. メニューから編集: リンクを削除...を選択します。
  2. Excel(DDE)リンクにチェックが入っていることを確認します。
  3. リンクを削除ドロップダウンリストからプロジェクトを選択してOKをクリックします。
  4. プロジェクトを保存します。

Note: このダイアログは、セルの式/リンクおよび列の式を値に変換するために使用することもできます。


Excel ファイルへのアクセス無しでそのようなプロジェクトファイルを取得した場合、Originでプロジェクトファイルを開くと、DDEリンクの開始ダイアログが開き、リンクを再度確立するか削除するかを尋ねられます。プロジェクト内のDDEリンクが非常に多い場合、ダイアログがポップアップし続け、非常に煩わしく、プロジェクトを開くのも難しくなります。

DDEリンクの開始ダイアログを表示せずにプロジェクトを開くには、プロジェクトファイルを開く前にシステム変数@dden=1を設定します。 システム変数@ddenの詳細はこちら

キーワード:excel, ddeオブジェクト, 壊れたリンク, opjを開く, dde確認, dde削除
