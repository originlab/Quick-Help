FAQ-406 Python の拡張モジュールにOriginからアクセスできない場合どうしたら良いですか？

最終更新日:2015/02/04

access-python-extensions

Origin内に組み込まれたPythonには、基本的なモジュールが含まれています。しかし、他のPythonの式をインストールしていて、Originに組み込まれたPythonからそれにアクセスしたい場合、インポート前にインストレーションフォルダをシステムパスリストに追加する必要があります。

システムパスリストにインストールフォルダを追加するサンプルは、LabTalkガイドの、Python と一緒に操作するの章にあるPython 拡張モジュール使用のための注意のセクションを確認してください。

キーワード: python, python binding, python拡張モジュール, PyQt4, システムパス

必要なOriginのバージョン:2015SR0
