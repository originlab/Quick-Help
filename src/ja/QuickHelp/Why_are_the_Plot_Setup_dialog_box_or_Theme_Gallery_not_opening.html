FAQ-341 なぜ「作図のセットアップ」や「テーマオーガナイザ」が開かないのですか?

PlotSetup-Never-Appear

最終更新日:2015/04/13

ワークシートで何も選択しないで、作図ツールをクリックしたり、作図メニューを選択すると、Originは、プロットを構成する「作図のセットアップ」ダイアログを表示します。

「作図のセットアップ」ダイアログは、通常インストール中にシステムに登録されるActiveXコントロールを使っています。登録がうまくいかないと、「作図のセットアップ」ダイアログは表示されません。これはインストール中のセキュリティ設定によりActiveXコントロールがブロックされるために起こります。このコントロールはテーマギャラリーでも使われています。

スクリプトウィンドウを使ってOrigin内でこのコントロールを登録することができます。Originウィンドウメニューからスクリプトウィンドウ (または単にAlt+3を押す)を開きます。次のスクリプトをウィンドウにコピーします。 

%A = system.path.program$;
run -e regsvr32 "%Avsflex8l.ocx";

スクリプトウィンドウでこの2行のスクリプトを選択し、Enterキーを押します。登録を確認するメッセージまたは失敗を示すメッセージが表示されます。失敗した場合には、十分な権限を得られるようシステム管理者にご相談ください。

登録成功のメッセージが表示されたのに「作図のセットアップ」ダイアログやその他のダイアログが開けない場合は、OriginLabのサポートチームにお問い合わせください。


キーワード: 失敗、登録
