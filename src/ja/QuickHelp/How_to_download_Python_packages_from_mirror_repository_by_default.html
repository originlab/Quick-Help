FAQ-1075 デフォルトでミラーリポジトリからPythonパッケージをダウンロードするにはどうすればよいですか？

Download-Python-Package-from-MirrorRepository

最終更新日：2020/11/11

以下を使用してPythonパッケージをダウンロードしてインストールする場合、

  • LabTalkコマンド pip install SomePackage
  • メニューから接続:Pythonパッケージ > インストールボタンを選択

以下の手順に従って、デフォルトのミラーリポジトリを切り替えることができます。

  1. pip.iniファイルを検索します。（非表示となっています） C:\ProgramData\pip\pip.ini
    ない場合は、自分で作成します。
  2. pip.iniを開き、次のように目的のミラーリポジトリを構成します。 
    [global]
index-url = https://mirrors.aliyun.com/pypi/simple
  3. iniファイルを保存します。

キーワード：python, pypi, pip config, ミラー PyPi リポジトリ, インストール
