FAQ-151 Originプロジェクトのパスをテキストラベルに追加するには、どうしたら良いですか?

Add-Project-Path-as-Label

最終更新日:2015/02/04


グラフを印刷したときに情報を参照するため、Originプロジェクトのディスク上のパス/位置をグラフに追加したいかもしれません。

これを行うために、最初にプロジェクトをディスクに保存します(まだ保存されていない場合)。

そして、次のいずれかを行います。

方法1:

  1. ラベルを配置したい場所で右クリックし、コンテキストメニューからテキストの追加... を選択します。
  2. ラベル編集ボックスで、次の文字列を追加します。
    %X%G
    必要に応じて、別の文字を追加し、外側をクリックして編集モードを抜けます。
  3. ラベルを右クリックして、コンテキストメニューからプログラム制御... を選びます。
  4. 「変数(%,$)へリンク」チェックボックスを選択し、OKをクリックします。
    NOTE:バージョン8.5.1以降では、変数(%,$)へリンク,置換レベルドロップダウンを 1 にし、OKをクリックします。
  5. ラベルを右クリックして、コンテキストメニューからプロパティ... を選びます。
  6. 編集ボックスで、テキストを変更し、以下のように読むことができます。
    \v(%X%G)
    そして、OKをクリックしダイアログを閉じます。
    NOTE:プロパティダイアログに \v() を入力する必要があります。インプレースの編集モードで、ラベルを直接このように変更すると動作しません。

方法2:

  1. ラベルを配置したい場所で右クリックし、コンテキストメニューからテキストの追加... を選択します。
  2. ラベル編集ボックスで、次の文字列を追加します。
    %X%G
    必要に応じて、別の文字を追加し、外側をクリックして編集モードを抜けます。
  3. ラベルを右クリックして、コンテキストメニューからプログラム制御... を選びます。
  4. オブジェクト名 のプロパティを Text1 のようにします。
  5. 「変数(%,$)へリンク」チェックボックスを選択し、OKをクリックします。
    NOTE:バージョン8.5.1以降では、変数(%,$)へリンク,置換レベルドロップダウンを 1 にし、OKをクリックします。
  6. スクリプトウィンドウを開き、次のコマンドを入力します。
    <Object name from step 4>.verbatim=1
    そしてEnterキーを押します。

キーワード: \, \v, 逐語的, 場所, ディスク, OPJ

必要なOriginのバージョン:7.5
