FAQ-1187 サーバの起動時に「WinSock: Connection refuse」というエラーが表示された場合はどうすればよいですか?

WinSock-Connection-Refused-Server-Error

最終更新日：2023/7/26

FLEXnetライセンスサービスを開始しても動作せず、LMTools > Server Statusタブで以下のようなエラーメッセージが表示された場合

System Error: 10061 “WinSock: Connection refused”

これは、サーバにログファイルのディレクトリへの書き込みアクセス権がないことが原因で発生します。これは通常、管理者以外のWindowsアカウントからサービスを実行しようとしているときに発生します。


この問題を解決するには、

必要なアクセス権限をサーバに適用する必要があります。詳細は、Flexera のページを参照して下さい。

キーワード:Windows 10, Win10, コンカレント, サーバフローティングライセンス
