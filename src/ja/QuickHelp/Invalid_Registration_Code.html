FAQ-910 OriginLabから提供された登録コードを入力しましたが、このコードは無効であると表示されます。

Invalid-Registration-Code

最終更新日：2022/12/1

登録ID（9文字）と登録コード（16文字）は異なります。正しい情報を入力しているか確認してください。

登録コードは電子メールで送信され、シリアル番号をアカウントに登録する際に使用します。

この問題は、登録コードをコピー/貼り付けした場合、OriginLabのウェブサイトへのリンクを使用した場合双方で起こることがあります。
登録コードは半角16文字であることを確認してください。そうでない場合、お問い合わせください。
