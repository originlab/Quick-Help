FAQ-754 既存グラフから同じデータ構造のグラフのクローンを作成するには？

Clone-Graph

最終更新日：2021/3/17

目的のグラフウィンドウをアクティブにして、ウィンドウタイトルバーを右クリックしてテンプレートの新規保存を選択するか、ファイル：テンプレートの新規保存を選択して、Utilities\File:template_saveasダイアログを開きます。 クローンテンプレートとしてマークのチェックを付け、列の一致条件ドロップダウンリストからデータ構造一致条件を選択し、OKをクリックしてクローンテンプレートとしてグラフを保存します。

この操作の後、このクローンテンプレートから、同じデータ構造の新しいデータを使用してグラフのクローンを作成できます。プロットするデータをすべて選択して、メニューの、作図：ユーザテンプレートから、羊のアイコンが付いている保存したクローンテンプレートを選択します。するとテンプレートとして編集と全く同じグラフを作図できます。

テンプレートライブラリを選択してダイアログでテンプレートをプレビューでき、作図にどのテンプレートを使用するか決定できます。

詳細は、このページ をご覧下さい。

必要なOriginのバージョン: 2016 SR0以降

キーワード:クローン, 同じ, コピー, 複製, レプリカ
