FAQ-393 ダイアログでOrigin Cプログラム（LabTalkスクリプト）のパラメータをどのように入力すれば良いですか？

Dialog-input-parameter

最終更新日:2015/02/04


独自のコードで数値計算したときのパラメータが必要になることがありますが、C++言語のcin やC言語での scanf といったコマンドは、Origin Cではサポートされません。このような場合、InputBox またはGetN ダイアログ を使用し、単純なあるいは複雑な構造のパラメータを取得できます。

また、LabTalkスクリプトを使用して番号取得のボックスを構成したい場合、GetNumber command の使用を推奨します。


キーワード:InputBox, GetN, GetNumber, 入力, cin, scanf,
