FAQ-1057 HTMLレポートでローカルCSSおよびその他のファイルにリンクするにはどうすればいいですか？

link-css-html-reports

最終更新日：2020/05/22

ローカルファイル（*.css, *.js, イメージファイル)にリンクし、ファイルへのフルパスを使用すると、アセットがHTMLレポートに読み込まれない場合があります。たとえば、次のようなスタイルシートの読み込みに失敗したとします。

<link rel="stylesheet" href="C:\path\to\files\my_styles.css">

パスが確実に正しい場合は、パスの前にfile:///追加してみてください。例えば、

<link rel="stylesheet" href="file:///C:\path\to\files\my_styles.css">

これにより、アセットの読み込みの問題が解決するはずです。そうでない場合は、最初にファイルにアクセスする権限があるかどうかを確認する必要があります。

キーワード：html、レポート、css、javascript、スタイル、img、イメージ、リンク、ファイル、パス
