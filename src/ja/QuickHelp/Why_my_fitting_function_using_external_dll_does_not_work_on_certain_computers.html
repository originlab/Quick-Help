FAQ-1077 外部dllを使用したフィット関数が特定のコンピュータで機能しないのはなぜですか？

dll-compatibility

最終更新日：2020/11/19

Originには、vcredist_x64_2008、2010、2012、および2015-2019のみが事前インストールされています。そのため、2013などの別のバージョンのVisual Studioを使用してdllをビルドしている場合、vcredist_x64_2013がインストールされていないと、他のマシンで正しく機能しない可能性があります。

互換性を持たせるために、プロジェクトのプロパティの構成プロパティ：C / C ++：コード生成ページでランタイムライブラリをマルチスレッド(/MT)に設定する必要があります。

キーワード:dll, フィット
