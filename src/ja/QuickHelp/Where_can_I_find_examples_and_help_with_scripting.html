FAQ-371 スクリプトのサンプルやヘルプはどこにありますか?

LT-Script-Example

最終更新日:2015/02/04

多くのLabTalkのスクリプトファイル（拡張子OGSのファイル）が、以下のフォルダ<Origin Installation Directory>/Samples/LabTalk Script Examples/にあります。コードビルダからこれらのファイルを開くことができ、（表示：コードビルダから開けます）スクリプトを見ることができます。


コマンドウィンドウからどの例を実行するにしても以下の手順に従ってください。

  1. Xファンクションcdを使って現在作業中のディレクトリを特定し、それからこれからのLabTalkスクリプトファイルを特定します。 
    cd 2;
  2. ディレクトリ内のファイルをdirXファンクションを使用して全てリストします。 
    dir;
  3. このステップは任意です。コードビルダ内でファイルを開くにはed.open(fileName.ogs)を使うことができます。例えば、以下のように扱います。 
    ed.open(embed_graphs.ogs);
  4. 例として表示するにはリスト内から一つを選びファイルの名前をコマンドウィンドウに打ち込んだ後にEnterを押します。例えば、
    embed_graphs;

LabTalk スクリプトガイド、LabTalk 言語参照とスクリプト例はLabTalkを学ぶ上でとても役立ちます。

キーワード: LabTalk, マクロ, OGS
