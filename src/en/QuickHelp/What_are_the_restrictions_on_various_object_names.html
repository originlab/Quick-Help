4.7 FAQ-365 What are the restrictions on various object names?

Restrictions-on-Object-Name

Last Update: 2/4/2015

Each Origin column and workbook has a Short Name and a Long Name. The Short Name for column is restricted to 18 alpha-numeric characters and the short name for workbook is restricted to 13, whereas Long Names are not restricted. And Origin worksheet name is limited to 32 characters, including spaces.


Please note that the sum of Book Short Name + Column Short Name must be <= 23

Please refer to the Origin Limitations page to learn more limitation in Origin.
