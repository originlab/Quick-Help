6.106 FAQ-1187 What should I do if I see a “WinSock: Connection refused” error when I try to start the FLEXnet server?

WinSock-Connection-Refused-Server-Error

Last Update: 7/26/2023

If you try to start your FLEXnet license service but fail, and see following error message in LMTools > Server Status tab:

System Error: 10061 “WinSock: Connection refused”

It is caused by the server without the write access to the directory of the log file. It usually happens when you are trying to run the service from a non-administrator Windows account.


To fix this problem,

You will need to apply necessary access privileges to the server. Refer to this Flexera document page for details.
