1.159 FAQ-816 How do I insert graphs as new slides in my currently opened PowerPoint presentation?

Insert-Graphs-into-Current-Presentation-in-PPT

Last Update: 3/26/2021

To insert Origin graphs into an opened PowerPoint file, follow these steps:

  1. Make sure the PPT presentation file has been saved once, thus giving it a name (you can have done further edits to it without saving them).
  2. Click the Send Graphs to PowerPoint button to open the Send Graphs to PowerPoint App.
  3. Beside Starting Slide Index, check the End box to add the graph as a new slide to the end of the PowerPoint presentation.
  4. Check the box next to Send to Opened PPT File.
  5. Click Send to append the specified graph(s) to your PowerPoint file.
  6. You can continue to edit the PowerPoint file as needed, and keep sending new graphs to that file as additional slides; or use the App dialog's Starting Slide Index and Slide Mode controls to replace existing slides.


Refer to this tutorial to learn more about how to sending graphs to an existing PowerPoint file.
