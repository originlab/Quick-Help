1.197 FAQ-921 How to show plots smoother?

Show-Plot-Smoother


Last Update: 1/4/2019

You can turn on Anti-Aliasing to show the plot smoother.

If you want to turn on Anti-Aliasing globally

You can set AntiAliasing theme as system default theme.

  1. Select Preferences: Theme Organizer.
  2. In the Graph table, locate AntiAliasing theme. Right click on it and select Set as System Theme.

If you want to turn on Anti-Aliasing for several graphs

You can locate the AntiAliasing theme in the Theme Organizer dialog, choose a desired applicable level from the Apple Theme To drop-down list, and click Apply Now button.

If you want to create a user-defined theme with Anti-Aliasing on

  1. Select Preferences: Theme Organizer.
  2. Right click inside the Graph table and select Add New to add a new theme. If you already have an user-defined theme, right click on it and select Edit.
  3. In the Theme Editing dialog that opens, right click in the table and select Add: Page if it doesn't have a Page node.
  4. Right click on the Page node and select Add: Display if it doesn't have a Display sub-node.
  5. Right click on the Display node and select Add: Anti-alising.
  6. Check Anti-aliasing checkbox that is added.
  7. Save this theme.

Keywords:AntiAliasing, smooth, graph theme
