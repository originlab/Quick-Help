6.70 FAQ-916 Why do I see "bad ELF interpreter" error when Installing Flexnet Server on Red Hat Enterprise Linux?

Bad-ELF-Interpreter-Err-RedHat-Linux-Concurrent

Last Update: 9/20/2022

Flexnet Server for Linux is a 32-bit application and thus requires 32-bit version of Operating System libraries. The library files are not installed on 64-bit version of Red Hat Enterprise Linux by default. If you Install Flexnet Server on a 64-bit Red Hat Enterprise Linux you may get the following error message:

/lib/ld-lsb.so.3: bad ELF interpreter: No such file or directory

To resolve this problem, you will need to install the redhat-lsb library by running this command as root: 

 yum -y install redhat-lsb
