6.31 FAQ-35 When I try to get a license, I'm asked for an Access Code. What is this and why do I see this?

Admin-Code-for-Licensing

Last Update: 2/4/2015

The Access Code is provided on the License Certificate for multi-user packages as well as on the serial number label. A Software Administrator can enter the Access Code and set up the License Access Restrictions. A user also needs to enter the Access Code during licensing process. If you have any questions about the License Access Restrictions, please contact OriginLab Support.
