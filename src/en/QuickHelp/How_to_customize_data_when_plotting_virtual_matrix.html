1.170 FAQ-833 How to plot with virtual matrix?

how-to-customize-data-when-plotting-virtual-matrix

Last Update: 2/18/2019

In Origin, you can create a contour or 3D graph from a range of cells in a worksheet, which is treated as a virtual matrix.
You have the following two options to customize data when you plot a graph from a virtual matrix:

1. Creating a virtual matrix

In this scenario, you can make a plot as well as create a virtual matrix:

  1. Select the worksheet cells block, and then click the Plot menu, such as Plot > 3D : 3D Colormap Surface or another graph types that support creation from virtual matrix, to open the Plotting: plotvm dialog.
  2. Select a way to set X values and Y values for the graph in Data Layout item.
    Note: If the radio of Y across columns is selected, it means that it will use the values in a row as the Y-values and the values in a column as the X-values. Conversely, if the radio of X across columns is selected, it means that it will it will use the values in a row as the X-values and the values in a column as the Y-values.
  3. Select the X and Y data range from the drop down lists of Y value in and X value in.
    Note: If None is selected in the drop down list, row index or column index will be X or Y data when creating virtual matrix.
  4. Click OK button, a virtual matrix will be created with the plot.

2. Selecting an existing virtual matrix

In this scenario, you can use an existing virtual matrix to make a plot:

  1. Click the Plot menu to open the Plotting: plotvm dialog.
  2. Click the triangle button in Input item, and then select an existing virtual matrix in the fly-out menu.

You can use Virtual Matrix Manager to manage the existing virtual matrices. If you want to launch the Virtual Matrix Manager dialog, you can select Tool: Virtual Matrix Managerâ€¦ from Origin menu or double click on any plot created by virtual matrix to open Plot Details dialog, and then click Workbook button at the bottom.

In Virtual Matrix Manager dialog, you can:

  1. Delete an existing virtual matrix by right clicking the virtual matrix in the left panel and selecting Delete.
  2. Modify the name, data title and data format of the virtual matrix in the dialog.
  3. Go to the worksheet which generated the current virtual matrix by clicking Go to Worksheet button.
  4. Go to the graph which was plotted the current virtual matrix by selecting a graph in the right panel and clicking Go to Graph button.

You can get the range of virtual matrix easily in Plot Details dialog after creating a plot from a virtual matrix:

  1. Double click the plot to open the Plot Details dialog.
  2. Go to the left panel and put the cursor on the name of plot.
  3. A tooltip will show up to tell you the range of virtual matrix.

In Origin, the virtual matrix can be also accessed and changed the properties by vmatrix object in Labtalk.
