5.78 FAQ-1094 What should I do if Origin not responding when exporting graphs or choosing menu?

No-Responding-Anti-Virus

Last Update: 1/27/2023

If Origin does not function for unknown reasons, including but not limiting to following situations:

  • No response when exporting graphs,
  • No reasponse when choosing a menu,
  • Shows an error message External call execution or Fail to find xxx.DLL when fitting or creating data sheet from a graph file...


Please check if the anti-virus software cleans up some of Origin's DLL files.

  1. If yes, reinstall required DLL or repair Origin (select Repair or Uninstall Origin from Windows Start menu and select Repair in the install wizard).
  2. Add Origin to anti-virus software's white list.
