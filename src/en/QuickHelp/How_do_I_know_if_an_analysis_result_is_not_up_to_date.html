3.44 FAQ-277 How do I know if an analysis result is not up to date?

Recaculation-Status

Last Update: 7/20/2018

The color of the lock on the Result Sheet and Graph Window indicates the status.

  • If the result is up to date, the lock color is green .
  • If the result is in need of an update, then the lock color is yellow .
  • If there is a problem with the update, then the lock turns red.

If the source data changes the lock will turn yellow, to indicate a recalculate is needed. You can left click the lock to Recalculate or Change Parameters.

The Recalculate button on the Standard toolbar, , also indicates if an update is needed. It will also turn yellow, . Clicking this button will update any result in the project that needs updating. If you hold the mouse over the button, a message will appear telling you how many updates are needed.
