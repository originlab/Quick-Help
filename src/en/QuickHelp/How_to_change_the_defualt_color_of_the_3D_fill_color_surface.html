1.169 FAQ-832 How to change the specific color in the 3D color map surface?

how_to_change_the_specify_cplor_in_the_3D_color_map_surface

Last Update: 5/4/2018

If you want to change the specify color of in the 3D color map surface into what you want, you can follow the steps below:

  1. Double click on the plot of 3D colormap surface to open the Plot Details dialog, and then go to Colormap/Contours tab.
  2. Click the Fill header to open the Fill dialog, and then select Load Palette radio.
  3. Uncheck the box of Link to Palette.
    Note: If you select this check box to link the color filling to the selected palette, it will disable editing the single color under the Fill column by clicking.
  4. Click OK button and return to the Plot Details dialog, and then you can click on any single color under the Fill column and change the color in the pop-up dialog box.
