5.86 FAQ-1169 Can I pull individual windows such as graphs and worksheets out of the main Origin interface and float them on a second monitor?

Float-Windows

Last Update: 3/30/2023

Yes. Origin 2023b introduced the Floating Window feature which allows child windows in Origin to be pulled out of the main Origin workspace as a separate window. Go to Window: Float Windows to enable or disable this feature.

Floating window will be auto pinned so that it will be kept when switching folders in Project Explorer. This is helpful for users who work with different folders in project.

Please see our blogpost for more information on this feature:
