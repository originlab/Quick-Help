2.102 FAQ-989 How to display number as 10 000 instead of 10000?

Display-thousand-separator-as-space

Last Update: 1/9/2019

To display thousand separator as space rather than "," or other character, for example, display 10000 as "10 000", you can

  1. Double click on the column header to open the Column Properties dialog. Set Format as Numberic and Display as Decimal: 1,000. Click OK.
  2. Select menu Preference: Options. In the dialog that opens, go to Numeric Format tab. Set Separators to 1 000.0.
