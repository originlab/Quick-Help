3.45 FAQ-278 How do I perform a different analysis on the same range?

DifferentAnalysis-on-SameRange

Last Update: 8/9/2018

When you perform analysis on a selected range of data in your graph, the markers turn into analysis markers and cannot be used as the input range in a new analysis.

One option is to click the green lock and then choose Plot Input Data with Data Markers. This will create a new plot of the data along with the original data markers. You can now perform analysis on this graph.
