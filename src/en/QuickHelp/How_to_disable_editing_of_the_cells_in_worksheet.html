2.98 FAQ-973 How to disable editing of the cells in worksheet?

Disable-Edit-Cells

Last Update: 9/13/2018

Disable Clicking to Edit

Highlight the cells in worksheet, right click and choose Format Cells from context menu. In the opened dialog, choose Disable Editing check box

Notes: The option is only about the click to edit. You can still change the values in cells with Set Column Values tool or Fill Column With tool

Disable Editing for All Operation

  1. Highlight the cells in worksheet which you want to keep editing and run script 
    lay -les 1
  2. Then run script below to protect the whole sheet except those we set in step 1 
    lay -lw hex(2);
