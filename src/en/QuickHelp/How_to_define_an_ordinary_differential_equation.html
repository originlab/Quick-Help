3.40 FAQ-269 How to define an ordinary differential equation for fitting

Define-ODE

Last Update: 2/4/2015

Origin allows you to define an ordinary differential equation (ODE) by calling NAG functions. See the tutorial Fitting with an Ordinary Differential Equation, which shows how to fit a first-order ordinary differential equation using the Runge Kutta method. A higher order ODE or ODE system can also be defined in a similar way.

Keywords:fit, ordinary differential equation, ODE, system, NAG, Runge Kutta

Minimum Origin Version Required: 9.1SR0
