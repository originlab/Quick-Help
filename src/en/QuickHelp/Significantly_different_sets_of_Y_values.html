1.115 FAQ-668 Is it possible to plot 2 significantly different sets of Y values on the same X scale where each each Y data set has its own Y axis?

Last Update: 9/4/2022

Significantly-Different-Sets

The Double Y graph can be created in Origin by highlighting two Y datasets in a worksheet and selecting Plot > Multi-Panel/Axis: Double-Y, Plot > Multi-Panel/Axis: Double-Y Column or Plot > Multi-Panel/Axis: Double-Y Column-Line Symbol. A graph with two different Y axes (one on the right, the other on the left) sharing the same X axis will be created, with only one layer.

You can also create a 2Ys Y-Y graph by selecting the menu Plot > Multi-Panel/Axis: 2Ys Y-Y, Plot > Multi-Panel/Axis: 2Ys Column or Plot > Multi-Panel/Axis: 2Ys Column-Line Symbol, which has two graph layers(each has one Y axis).
