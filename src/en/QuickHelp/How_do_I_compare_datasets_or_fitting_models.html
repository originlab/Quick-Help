3.11 FAQ-234 How do I compare datasets or fitting models?

Compare-Datasets-FittingModels

Last Update: 2/14/2017


To compare two datasets that are fit to the same function, select Analysis : Fitting : Compare Datasets from the Origin menu. This uses an F-test to determine whether the two datasets are significantly different from each other. You must first fit each of the datasets using Origin's Fitting tools. For more information, see The Compare Datasets Dialog Box.

To compare the results of two models fit to the same dataset, select Analysis : Fitting : Compare Models from the Origin menu. This test will determine which model is the best one. The model comparison is done using three different tests: F-test, Akaike Information Criterion (AIC) and Bayesian Information Criteria (BIC). You must first fit the dataset to each of the two models using Origin's Fitting tools. For more information, see The Compare Models Dialog Box.

Origin also provides a tool to rank fitting functions within a given category and fit data with the function which gives the best result. Select Analysis: Fitting: Rank Models from Origin menu. For more information, see Fit and Rank Multiple Functions.
