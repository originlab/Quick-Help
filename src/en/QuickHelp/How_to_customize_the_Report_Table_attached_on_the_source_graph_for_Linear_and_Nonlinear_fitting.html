3.132 FAQ-770 How to customize the Report Table attached on the source graph for Linear and Nonlinear fitting?

Custom-ReportTable-LR-NLFit

Last Update: 10/13/2016

You can customize formats of Report Table and save it as a template. Both Linear Fit and Nonlinear Curve Fit dialog have options to load this template and specify which quantities to be shown in the Report Table. Please refer to this page for details.

Minimum Origin Version Required: Origin 2016 SR0
