3.133 FAQ-774 Can I add fitting equation with parameter values to the graph in nonlinear curve fitting?

add-equations-to-graph

Last Update: 10/13/2016

Yes. To add fitting equation with parameter values to the source graph,

  1. Open Nonlinear Curve Fit dialog, select Output on the Settings tab.
  2. Expand Optional Report Tables branch on the right panel, select Equation with Values for Equation in Notes.

Note: the Equation with Values option is also available in Quick Fit Gadget, Linear Fit, Linear Fit with X Error, and Polynomial Fit dialog.

Minimum Origin Version Required: Origin 2016 SR0
