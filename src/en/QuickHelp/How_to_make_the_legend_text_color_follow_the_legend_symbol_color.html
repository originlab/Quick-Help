1.133 FAQ-726 How to make the legend text color follow the legend symbol color?

same-text-and-symbol-color-in-legend

Two ways to make the legend text color follow the legend symbol color such as

  • Right-click on the legend object and choose Legend: Text Color Follows Plot.
  • Set the font color as Auto on the Format toolbar or right-click the legend to select the Properties from the context menu to set the color to Auto in the Text tab of Text Objects dialog.

Minimum Origin Version Required: 2016 SR0

Keywords:legend text color, legend symbol color
