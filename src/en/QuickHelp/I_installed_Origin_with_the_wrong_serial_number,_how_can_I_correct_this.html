6.7 FAQ-8 I installed Origin with the wrong serial number, how can I correct this?

Modify-SN

Last Update: 11/7/2023

If the Origin or OriginPro software has been installed on your computer with an incorrect serial number or you have upgraded from Origin to OriginPro, you can correct/change your installed serial number as follows:

  1. Close Origin(Pro) if it is open.
  2. If you are using Windows 7 and Vista,
    • Click the Windows Start button and select All Programs | OriginLab Origin [version] | Repair or Uninstall Origin [version]. This starts the InstallShield wizard.
    If you are using Windows 8,
    • Press the Windows key on your keyboard.
    • Right-click in a blank area of the desktop.
    • Click the All Apps button at the bottom right corner of the desktop.
    • Select Repair or Uninstall Origin [version] that displays on the desktop. This starts the InstallShield wizard.
    If you are using Windows 10,
    • Click the Windows Start button, scroll down the app list and select OriginLab Origin [version] | Repair or Uninstall Origin [version]. This starts the InstallShield wizard.
    If you are using Windows 11,
    • Click the Windows Start button, then select the arrow next to All apps to expand that pane. Scroll down the app list and select OriginLab Origin [version] | Repair or Uninstall Origin [version]. This starts the InstallShield wizard.
  3. Select the Modify/Change option and click Next.
  4. Click "Yes" when asked if you would like to change your serial number.
  5. Enter your serial number and complete the Modify/Change process, clicking Finish when done.
