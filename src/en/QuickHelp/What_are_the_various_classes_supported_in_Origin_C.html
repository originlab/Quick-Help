4.36 FAQ-396 What are the various classes supported in Origin C?

Classes

Last Update: 2/4/2015

In addition to standard data types, Origin exposes many of the internal object classes that make up the windows and objects that appear in Origin's Graphical User Interface (GUI). For a complete list of classes along with methods that can be used with those classes, consult the Origin C Classes Reference and tutorial on accessing internal Origin objects by Origin C.
