3.74 FAQ-312 Does Mann Whitney Test support raw data in Origin?

Mann-Whitney-Test

Last Update: 2/4/2015

Select Statistics: Nonparametric Tests: Mann-Whitney Test (Pro only) to open dialog

You can perform Mann-Whitney test on either Indexed or Raw data, chosen from the Input Data Form drop-down list.
