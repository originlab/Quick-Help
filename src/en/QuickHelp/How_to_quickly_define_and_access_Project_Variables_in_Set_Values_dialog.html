3.129 FAQ-765 How to quickly define and access Project Variables in Set Values dialog?

Define-ProVar-in-SetValues

Last Update: 8/27/2015

To define a project variables in the Set Values dialog,

  1. Select Variables: Project Variables: More... to open the Project Variables dialog. Defined project variables are listed here allowing you to view and delect.
  2. Click + button to define your project variable in the Define Variables dialog.

Once defined, you can find the project variable under Variables: Project Variables menu which can be quickly accessed.

Minimum Origin Version Required: Origin 2016 SR0

Keywords:define project variable, set column values
