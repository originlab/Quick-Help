1.20 FAQ-132 How to add label to 3D surface plot?

Add-Label-on-3D-Surface

Last Update: 7/5/2018

If you want to add point label to a 3D surface plot, you could do the following:

  1. Put the XYZ coordinate value(s) of the desired point(s) into a separate worksheet.
  2. Use the Layer Contents dialog or Plot Setup dialog to add the point as 3D Scatter to the surface plot.
  3. Open the Plot Details dialog and go to the Label tab of the 3D scatter plot, check Enable to activate the data labels and do further customization.
  4. To add a leader line, check Show Leader Line if Offset Exceeds(%) checkbox and customize other settings as needed in Leader Line group.

For more detailed steps, you could refer to this tutorial.


Minimum Origin Version Required: 2016 SR0
