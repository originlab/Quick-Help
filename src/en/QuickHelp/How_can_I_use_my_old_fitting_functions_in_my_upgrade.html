6.19 FAQ-21 How can I use my old fitting functions in my upgrade?

Transfer-User-Defined-Function

Last Update: 7/4/2018

The Transfer User Files dialog is designed to help upgrade users transfer their files, including user-defined fitting functions, from Origin 7.5 and more recent versions to their Origin upgrade. To open this dialog, click Tools on the main menu, and then click Transfer User Files. See details here.
