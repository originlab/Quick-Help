5.60 FAQ-920 How can I use English in reports and graphs when Origin is running in non-English OS?

Use-English-in-Reports-Graphs

Last Update: 1/3/2019

If you run Origin in a non-English OS, Origin will use the OS language and its Date and Time format by default. However, you may want to use English in graph labels and analysis report sheets to keep the whole page format consistent. To use English in graphs and reports,

  1. Select Preference: Options.
  2. In the Numeric Format tab, check Use English in Reports and Graphs checkbox.

Keywords:Chinese OS, Japanese OS, German OS, English date-n-time

Minimum Origin Version Required: 2018b
