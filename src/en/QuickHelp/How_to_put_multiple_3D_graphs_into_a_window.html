1.207 FAQ-950 How to put multiple 3D graphs into a window?

Combine-3D-graphs

Last Update: 9/5/2023

Currently, Origin's 3D graph window doesn't support multiple layers.

But you can put multiple 3D graphs into the Layout window

Origin 2021b and Later

Choose Graph: Merge Graphs to Layout... menu and pick graphs. 2D and 3D graphs can be merged together.

Before Origin 2021b

You will need to manually do it.

  1. Choose File: New: Layout menu or click New Layout toolbar button to start a new layout window.
  2. Choose Insert: Graph... or click Add Graph button on Layout toolbar to insert each graph manually.
