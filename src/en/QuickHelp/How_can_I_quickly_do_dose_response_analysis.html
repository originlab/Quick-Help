3.24 FAQ-252 How can I quickly carry out dose response analysis?

doseResp-analysis-template

Last Update: 2/4/2015


There is a built-in analysis template which can be used for dose response analysis in Origin.

You may click File: New: From Template: Assays.ogw to open this template and follow the instructions in the Note worksheet.

With this analysis template, you only need to import your raw data and then will be able to get many dose response analysis results including the calibration plot and fitted results.

Keywords:Dose Response, analysis template, built-in template

Minimum Origin Version Required: 9.0 SR1
