1.297 FAQ-1188 How to smooth contour plotted from matrix?

Matrix-Contour-Smoothing

Last Update: 8/1/2023

Currently only contour graph plotted from XYZ worksheet data has smoothing options (Smoothing controls in Plot Details dialog).

If you create a contour from matrix or virtual matrix, and want to get smoother contour lines/gradient colormap, you will need to

  1. Convert your matrix/virtual matrix data to XYZ columns:
    • for matrix data: Matrix: Convert to Worksheet menu > Method = XYZ Columns
    • for virtual matrix: Worksheet: Convert to XYZ menu
  2. Create a new contour plot from XYZ columns.
  3. Then you can use Smoothing options in Plot Details dialog to smooth the new contour: Plot Details dialog > Contour Info tab > Smoothing group.

Keywords:contour, matrix, virtual matrix, smoothing, gradient color change, smooth contour line
