3.147 FAQ-891 How can I control digits display in each analysis tool?

Control-Digits-in-AnalysisTool

Last Update: 9/15/2017

To Specify digits of numeric data in an analysis report sheet

  1. In the report sheet, click on the down arrow after the branch title or right-click on one cell of the table.
  2. Select Digits... in the context menu.
  3. In the Digits in Report dialog, specify the digit settings.

You can select Save as Default checkbox to save the digits setting as this analysis tool's default. All report sheets generated by this analysis tool that after will use this digits setting.

To specify digits globally

If you want to control the digits settings of all analysis tools, specify the Digits in Report setting on the Numeric Format tab of the Options dialog box.
