1.177 FAQ-856 Why Origin fails to insert equation to graph?

Fail-to-Insert-Euation

Last Update: 3/29/2023

Origin 2023b builds in a SVG-based LaTeX equation tool. This new Insert LaTeX Equation tool does not require users to manually install any extra libraries or softwares. To get rid of problems caused by LaTeX App, it is highly recommended that you upgrade your Origin to version 2023b or later and try the new Insert LaTeX Equation tool. Please refer to this document for more information.

If you still want to stay with LaTeX App, please keep on reading.

For Origin 2023 and ealier versions

Insert Equation tool requires Equation Editor to be installed, which was included in previous versions of Microsoft Word. If you encounter the following error when you try to open Insert Equation tool in Origin:

Origin provides an app LaTeX for equation editing.

It requires applications MiKTeX and OAutotrace be installed.

Would you like to use it?

See more details on LaTeX page. (here "LaTeX" shows as a link: https://www.originlab.com/FileExchange/details.aspx?fid=221)

You can later set system variable @IEE=0 to toggle back.

If you installed the January 2018 Public Update (PU)

Equation Editor (Microsoft Equation 3.0) has been removed from all versions of Microsoft Office products. To use Insert Equation tool in Origin, please consider using LaTex app as mentioned in the message or installing the third-party App MathType

To use free Latex App
  1. When a graph is active, click Insert Equation button from the Tools toolbar
  2. Click Yes when you saw the dialog.
  3. The app and the two applications, Mikitex and OAutotrace, will be installed automatically. Once they are installed, relaunch Origin.
  4. When a graph is active, click Insert Equation button from the Tools toolbar


To use Mathtype (Not Free)
  1. Install MathType in advance
  2. When a graph is active, click Insert Equation button from the Tools toolbar
  3. Click No when you saw the dialog.


Notes:To decide the behavior of Insert Equation button , Set @IEE=0 to toggle the reminder dialog
