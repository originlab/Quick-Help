1.14 FAQ-125 Can I set tick labels as 0 to 100 for my ternary plot?

TernaryPlot-Labels

Last Update: 1/11/2017


Yes, if your data is already normalized to "0 to 100" or meets the criteria of 0 to 100 normalization (i.e. there are more data points with a sum of x, y, and z exceeding 50), the tick label of the ternary plot will be set as 0 to 100 automatically.


Minimum Origin Version Required: Origin 9.0 SR0

A simple solution is to set the Divide by Factor to 0.01 in the Tick Labels tab of the Axis dialog.
