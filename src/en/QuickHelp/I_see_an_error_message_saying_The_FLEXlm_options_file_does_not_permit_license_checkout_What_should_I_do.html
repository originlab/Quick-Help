6.93 FAQ-1121 I see an error message saying "The FLEXlm options file does not permit license checkout". What should I do?

Opt-File-Not-Permit-Lic-Checkout

Last Update: 9/20/2022

If you see following message when trying to connect your Origin to the license server:

License Status: 112

The FLEXlm options file does not permit license checkout.

It is caused by that your license server manager set up an options file to determine who have access to the license service, and you are not on the list.

To fix this issue, please contact your manager to check the options file being used.
