3.87 FAQ-325 How do I overlay a distribution curve on my histogram plot?

Overlay-Dist-Curve-on-Histogram

Last Update: 1/2/2020

If you have already plotted a histogram and want to add a distribution curve on it, you can

  1. Double-click on your graph which will open the Plot Details dialog.
  2. Make sure the dataplot is selected from the left hand side of the dialog, and then on the right hand side you will see the Distribution tab. On this tab, there is a Curve Type drop-down list from which you can select the desired distribution curve type, and then do further customization with the options shown.

You can also check the Add Distribution Curves checkbox on the Data tab, if you wish the data added to the Bin Worksheet.

To plot a histogram with a distribution curve directly, you can

  1. Highlight the source column(s) to select Plot: Statistical > Distribution

By default, the Curve Type is Normal set in the Distribution tab of Plot Details dialog.
