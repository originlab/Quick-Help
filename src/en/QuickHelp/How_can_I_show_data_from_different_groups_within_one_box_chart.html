3.84 FAQ-322 How can I show data from different groups within one box chart?

Show-Data-from-Groups-Within-One-Box-Chart

Last Update: 10/13/2016


From Origin 9.1, you can indexing or mapping another column for data symbol color.


Go to Plot Details: Plot Properties: Symbol tab. Expand Symbol Color drop-down list, go to By Points tab, select a dataset from the Indexing or Color Mapping list to be the reference.

Note: You need to set Type to be Data or Box+Data first in the Box tab to show the data.

Keywords:indexing, color mapping, box chart, data, groups
