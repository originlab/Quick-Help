3.101 FAQ-631 How to report FWHM for each peak in a single dataset Gaussian fit?

FWHM-for-multi-peak-curve

Last Update: 7/20/2018

To report the full width at half maximum (FWHM) for each peak in a single dataset Gaussian fit, select Analysis: Peaks and Baseline: Peak Analyzer and perform the Gaussian fitting, and you will get FWHM value for each peak from the PeakProperties result sheet.
