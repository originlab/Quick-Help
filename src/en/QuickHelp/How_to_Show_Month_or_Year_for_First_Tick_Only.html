1.299 FAQ-1192 How to Show Year/Quarter/Month for First Tick only to Avoid Duplicate Display?

Show-YearMonth-at-1stTick

Last Update: 3/17/2024

When you plot date and time dataset in same period, you might want to show year/quarter/month only one time and place it at the firstly-appeared tick.

Since Origin 2024b, Origin supports \n to wrap text for date display, After you inserted \n in the Custom Display format, the part after \n will only show one time for same period.

Note: \n format could be used multiple times. For example, if the axis tick is 12/1/2023, 12/15/2023, 12/30/2023, you can set Custom Display to dd\nMMM\nyyyy, the tick label only show "Day" only for 2nd and 3rd tick, since both "Month" and "Year" repeat with that of 1st tick.
