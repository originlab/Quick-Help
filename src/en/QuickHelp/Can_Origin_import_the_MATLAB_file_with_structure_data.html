2.22 FAQ-436 Can Origin import the MATLAB file with structure data?

Import-MAT-with-Structure

Last Update: 1/7/2019

Choose Data: Import from File: Matlab(MAT)... to launch the impMatlab dialog.

With the Import into Hierarchical Structure check box checked, only the variable list will be firstly imported into a worksheet. You can then click on the individual variable names, displayed as linked text in the 1st column, to further import just the desired variables one by one, into new worksheets.
