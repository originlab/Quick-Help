3.167 FAQ-1156 How to get "exact P values" in ANOVA tools?

get-Exact-P-Values

Last Update: 4/18/2023


From Origin 2023, P values less than 0.0001 will be shown as < 0.0001 in report of ANOVA tools to indicate significance.

If you want to get "exact P values" in the reports, please change the value of system variable @SPD to be 0 with steps below.

  1. Choose menu Preferences: System Variables
  2. In the opened System Variable dialog, enter SPD and 0.
  3. Restart Origin
  4. Recalculate the results in your saved OPJ or run the statistical tools again
