6.98 FAQ-1130 How to disable or restrict borrowing of license?

Lic-Server-Borrow-Restriction

Last Update: 9/20/2022

To change the maximum time a user can borrow a license from FLEXnet server:

By default, the maximum allowed borrow time is 7 days. If you require a shorter or longer maximum borrow time (up to one month), please contact OriginLab and we will regenerate a new license file with the required borrowing time for you.

To disable borrowing of license or restrict the borrowing feature to a limited group of users:

You will need to create an options file to set the restriction for borrowing feature. Please see detailed description and examples on this page, Example 4 and Example 5.
