6.16 FAQ-18 I am receiving missing DLL errors. How can I resolve this?

Resolve-Missing-DLL

Last Update: 5/16/2022

Origin is built using Microsoft (MS) Visual C++, and thus it requires certain MS DLLs to run. These DLLs need to exist on the computer where Origin runs. You may see an MS DLL error at Origin startup on a computer without these DLLs, especially if you use MSI to install Origin.

To fix this problem, follow steps on this FAQ to reinstall required DLLs.
