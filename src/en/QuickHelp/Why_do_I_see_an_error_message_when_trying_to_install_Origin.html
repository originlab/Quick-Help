6.8 FAQ-9 Why do I see an error message when trying to install Origin?

Install-Origin-Error-Message

Last Update: 7/3/2018

If you see an "error installing ikernel.exe" message when running the Origin Setup, this is usually caused by your Windows log in account not having installation permissions. Please try the following solution:

  1. Go to the Flexera website.
  2. Click Knowledge Base.
  3. Select Installation/Application Readiness Products for Product Group.
  4. Select InstallShield from the Product drop-down menu.
  5. Enter Q111519 in the Keyword(s) edit box and click Search.
  6. The item Initialization Error Troubleshooting describes possible causes for this error message. Follow the steps to troubleshoot the problem.
