1.286 FAQ-1159 What should I do if I cannot use theme or copy paste format?

Cannot-Use-Theme

Last Update: 11/14/2022

If you see following problems:

  • Copy/Paste Format does not work
  • Right click on graph layer and select Save Format as Theme to open dialog. No properties are listed in the bottom panel, only an empty table

It is because Origin would create some temporary files such as OTF0000.tmp under the Temp folder and the folder is full.

To solve the problem,

  1. Type %temp% in Windows Folder Browser to open the Temp folder
  2. Delete all the files named as OTFXXXX.tmp
