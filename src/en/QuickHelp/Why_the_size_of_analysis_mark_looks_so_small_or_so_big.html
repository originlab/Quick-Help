3.51 FAQ-284 Why do analysis markers look so small or so big?

Unusual-Marker-Size

Last Update: 2/4/2015


The size of analysis markers can be controlled by the menu item Data: Analysis Markers:Tiny/Small/Medium/Large which can be accessed from four different places:

  1. right-click on current layer (do not select the layer) to select Analysis Markers from shortcut menu.
  2. right-click on the analysis markers to select Analysis Markers from shortcut menu.
  3. left-click on the lock icon to select Analysis Markers from shortcut menu.
  4. select Data: Analysis Markers from main menu.


However, after fitting, if you find that the analysis markers in the output are smaller or bigger than usual, this may be caused by the resolution of your default printer.

First, you can get the resolution of your default printer by running system.resolution = in the command window. After you get the value of the resolution, please set the proper x/y value of page resolution using the script below. 

page.resx= 
page.resy=
