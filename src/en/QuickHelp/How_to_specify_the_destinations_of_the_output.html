3.110 FAQ-648 How to specify the destinations of the output?

destinations-output

Last Update: 2/4/2015

To specifies the destination of output books, select the items below in the drop down list besides books

  • <none>: Do not output report worksheet tables.
  • <auto>: The workbook that is automatically picked by the program.
  • <source>: The source data workbook.
  • <new>: A new workbook.
  • <existing>: A specified existing workbook.

If <new> is selected as destinations, you can define name in BookName box. What's more, Result Log, Script Window, Notes Window are available for output.
