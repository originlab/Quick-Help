2.104 FAQ-995 Data: Connect to Web Fails to Import Data from Certain URLs in Windows 7

Data-Connect-to-Web-Fails-Windows-7

Last Update: 2/7/2019

If you have Origin 2019b or later installed on Windows 7 and are using Data Connectors via Data: Connect to Web, you may observe a failure to import data from URLs whose protocol is HTTPS. You may (or may not) also observe a cryptic error message.

It is related to certain outdated settings in Windows.

You can download and install this Microsoft Easy Fix wizard to address the problem: https://aka.ms/easyfix51044

For an indepth technical explanation of the issue, see: https://support.microsoft.com/en-us/help/3140245/update-to-enable-tls-1-1-and-tls-1-2-as-default-secure-protocols-in-wi
