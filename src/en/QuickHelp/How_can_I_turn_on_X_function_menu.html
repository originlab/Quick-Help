5.47 FAQ-811 How can I turn on X function menu, deconvolution for example?

Turn-on-XF-Menu


Last Update: 1/5/2016

Sometimes you find some built-in X-Functions useful but they don't appear on Origin menu, deconvolution for example. To turn them on,

  1. Open X-Function Builder by selecting Tools: X-Function Builder.
  2. Select File: Open System X-Functions.... Find and load the system X-Function you want to turn on.
  3. Select Tools: Tree View to view it in Tree mode.
  4. Expand Usage Context branch. Select Main menus from Menus drop-down list
  5. Select File: Save As... to save it as a user-defined X-Function.

Note: you can also show user-defined X-Function in menu the same way.
