1.241 FAQ-1019 How to customize the font color for the labels of a Pie/Doughnut chart?

FontColor-Labels-of-Pie

Last Update: 9/19/2019

By default, when you plot a pie/doughnut plot, the labels of wedges will show as Black. If you want to change the font color of these labels, you can

  • Double-click on one label to open the Labels tab of Plot Details dialog. Expand the Font control panel, use the Color Chooser to specify the Font Color. Please note, once you have set Font Color to be Auto, the label color will follow a smart way as below:
  • Click on one label on the graph, a mini toolbar will appear to let you fast edit the label properties(including Font Color). Follow the steps below to customize the font color for selected label object:
    If you have set the Font Color to Auto, it means the color of current label will follow the Font Color setting in the the Labels tab of Plot Details dialog.
    If you have changed the font color(including changing it to Auto) of the selected label object, you can use the Apply Formatting to button on the mini toolbar to apply the new color to all other labels in current pie/doughnut graph. Please note, this change will be synchronized to the Font Color option in the the Labels tab of Plot Details dialog.
  • Right-click on one label to open the Annotation dialog. In this dialog, you can use the Font Color option to customize the color for current label.
    If the color has been set to Auto, it means the color of current label will follow the Font Color setting in the the Labels tab of Plot Details dialog.
    Once you have changed the color for current label to a certain color(just as "Red"), you can use the Apply to drop-down list to apply the font color to other labels in current plot/layer/graph. Please note, this change will be synchronized to the Font Color option in the the Labels tab of Plot Details dialog too.
