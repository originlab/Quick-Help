3.77 FAQ-315 How can I align outliers in a line in box chart?

Align-Outliers-in-Line-Box-Chart

Last Update: 2/4/2015


From Origin 9.1, Outliers will be drawn in line in box chart by default. The options is availble in the Plot Details dialog

  1. Go to Plot Details: Plot Properties: Box tab
  2. Check Outliers checkbox.
  3. Check Outliers in Line checkbox to line them up.


Keywords:outliers, box chart, in line, align
