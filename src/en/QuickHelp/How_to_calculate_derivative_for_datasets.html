3.117 FAQ-687 How to calculate derivative for all datasets in graph?

Last Update: 11/5/2015

Calculate-Derivative

Active the graph, Select Gadgets: Differentiate in Origin menu, click OK to apply the Gadget on the graph, the derivation operation will be applied to all the datasets, and the preview will be shown in a new graph window. Or use Analysis: Mathematics: Differentiate tool to get the derivative for datasets specified.
