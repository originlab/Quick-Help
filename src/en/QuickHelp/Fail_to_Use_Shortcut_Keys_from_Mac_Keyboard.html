5.38 FAQ-739 When running Origin on a Mac through Virtualization Software, why can't I use some shortcut keys from a Mac keyboard?

Fail-to-Use-Shortcut-Keys-from-Mac-Keyboard

Last Update: 6/16/2015

Some shortcut keys might be different when using from a Mac keyboard, for example, to clear data, you use Delete hot key in PC keyboard, but fn + Del from Mac keyboard. You can refer to the comparison table between hot keys in PC and Mac keyboard for details.
