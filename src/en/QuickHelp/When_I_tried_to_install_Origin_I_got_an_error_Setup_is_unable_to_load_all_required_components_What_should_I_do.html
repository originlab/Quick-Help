6.97 FAQ-1126 When I tried to install Origin, I got an error "Setup is unable to load all required components." What should I do?

Setup-Unable-Load-Comp

Last Update: 5/7/2022

If you fail to install Origin software and get following error:

Setup is unable to load all required components.

It indicates that the installation file you have is either corrupted or incomplete; or that parts of the installation are being blocked by local security elements. Please try following steps to troubleshoot it:

  • Re-download the installation file from OriginLab website. Make sure the download is completed before running it.
  • Run the installation file with full security clearance. Please log on your computer with an administrator account. And you can temporarily turn off the antivirus software during installing to make sure it wouldn't be blocked by any security element.
