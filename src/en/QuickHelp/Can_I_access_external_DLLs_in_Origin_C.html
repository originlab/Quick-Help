4.30 FAQ-390 Can I access external DLLs in Origin C?

External-DLL

Last Update: 2/4/2015

Yes. Origin C functions can make calls to functions in external DLLs created by C, C++, or Fortran compilers. To do this, your source file must contain an include directive for the header file which declares the functions in your external DLL. Please read more in the Origin C Help about accessing an external DLL.
