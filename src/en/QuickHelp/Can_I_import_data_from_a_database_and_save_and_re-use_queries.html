2.11 FAQ-419 Can I import data from a database and save and re-use queries?

DatabaseAccess-Toolbar

Last Update: 11/15/2019

For an introduction to importing data from a database, please refer to the OriginLab blog post Connecting to a Database and Importing Data.

  • To save a defined query in the SQL Editor Dialog Box, choose File: Save Connection and Query As. Alternately, you can Save to Active Worksheet (Good option if you want to save that information but donâ€™t want to import data immediately).
  • To reuse an existing query, click the Load ODQ File button to browse the query file. You also can open the ODQ file directly in SQL Editor by (a) choosing Data: Connect to Database: Open, then browsing to the file or (b) from the Editor menu, choose File: Open.
