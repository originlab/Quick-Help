3.126 FAQ-729 How can I put the results of Statistics on Columns in the source worksheet?

Output-Results-Destination

Last Update: 10/13/2016

User can perform statistics on columns and put the results in the new columns of the source worksheet with the Statistics on Columns tool.

  1. Choose menu Statistics: Descriptive Statistics: Statistics on Columns
  2. In the opened dialog, choose Output tab, expand the Quantities branch, set settings as the image blow

Minimum Origin Version Required: 2016 SR0

Keywords:Descriptive Statistics, source worksheet, mean, standard deviation, columns
