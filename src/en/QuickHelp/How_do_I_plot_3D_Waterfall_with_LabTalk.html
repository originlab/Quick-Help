1.138 FAQ-749 How do I plot 3D Waterfall with LabTalk?

Water-Fall-LabTalk

Last Update: 4/4/2018

You can use X-Function plotxy to plot a 3D Waterfall with LabTalk, and use set command to control the color mapping

Please look at the last example in the page, which is a detailed example to create 3D waterfall with Y color mapping or Z color mapping.
