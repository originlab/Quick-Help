4.41 FAQ-402 Why is my newly created X-Function not working from script?

XF-Name-Conflict-with-LT

Last Update: 2/4/2015

When a command is executed in script, the execution order of priority is LabtTalk command > OGS file > X-Function. So, when you create a new X-Function, and the name may be conflicting with a LabTalk command name or an OGS script file name in the current working directory, which makes the X-Function fail to work from script, please re-name the X-Function and try again.


Keywords:script, ogs, precedence, priority, name, conflict
