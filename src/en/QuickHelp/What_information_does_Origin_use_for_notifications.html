6.53 FAQ-793 What information does Origin send to the website for notifications?

What_information_does_Origin_use_for_notifications

Last Update: 10/14/2016

When checking for updates to Apps, or when fetching Origin software-related notifications, Origin sends ONLY the following information:

  • Origin version number
  • Product language (English, Japanese, German)
  • Origin bit version (32- or 64-bit)


Origin DOES NOT send any personal information about the user, the product serial number, etc.

Keywords:privacy, notification, Apps, flag, status bar

Minimum Origin Version Required: 2016 SR0
