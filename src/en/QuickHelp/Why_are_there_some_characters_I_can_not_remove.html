1.203 FAQ-944 Why are there some characters I can't remove, such as Legend, XB, XT, YL, YR?

Remove-edit-mode-labels

Last Update: 11/4/2018

Sometimes, your graph might have some irremovable labels, such Legend, XB, XT, YL or YR.

Actually, they are not text contents, but the text object name for the graph elements such as axis labels, which only show up under the Button Edit Mode.

To escape this mode,

  • Select Edit: Edit Mode to uncheck this item.

Keywords:axis label, XB, XT, YL, YR, unmovable label, Legend, Edit Mode
