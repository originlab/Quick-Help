4.54 FAQ-723 How to change the font size in Script Window?

Set-Font-Size-in-Script-Window

Last Update: 12/12/2022

Beginning with Origin 2023, the Script Window is updated to include font-size control. Change font size by pressing Ctrl while scrolling the mouse wheel.

For previous versions of Origin, modify the User Files\Origin.ini as instructed at the bottom of this page.
