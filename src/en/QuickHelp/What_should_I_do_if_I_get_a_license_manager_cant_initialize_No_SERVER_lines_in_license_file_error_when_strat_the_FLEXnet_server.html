6.99 FAQ-1131 I get a "license manager: can't initialize:No SERVER lines in license file." error when strat the FLEXnet server. What should I do?

LIC-Server-Error-1366

Last Update: 9/20/2022

If you fail to start the license server, and get the following error in the log file:


12:07:37 (lmgrd) license manager: can't initialize:No SERVER lines in license file.
12:07:37 (lmgrd) License Path: "C:\OriginLicenseServer\Licenses\orglab.lic"
12:07:37 (lmgrd) FlexNet Licensing error:-13,66


Please try the following solution:

  1. Open the license file in Notepad (or other text editor). Make sure the SERVER line exists. If not, please retrieve a license file from OriginLab website.
  2. Resave the license file by menu File: Save as. In Save as dialog, make sure encoding is ANSI.

Keywords:license, -13,66, can't initialize:No SERVER lines, Cannot connect to license server system
