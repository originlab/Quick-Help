6.62 FAQ-900 I am qualified for auto-upgrade. How to activate my free upgrade to the new version?

Auto-Upgrade-to-New-Version

Last Update: 5/4/2019

If you are installing Origin on the same computer:

Install the new version with same serial number. Origin will activate itself without having to manually input any product key.

If you installing the new version on another computer:

Log in to the OriginLab website and find available product keys under the serial number.

If no product keys are available, you need to deactivate the old version from the other computer in order to re-use that product key for the new version.
