1.300 FAQ-1193 What should I do if the graph object pasted in PPT fail to update after editing?

PPT-Graph-Fail-to-Update

Last Update: 3/20/2024

If you copy an Origin graph (menu Edit: Copy Page, or shortcut key Ctrl+J) and paste it as an OLE object into PowerPoint, you can double click the graph object to re-open and edit it in Origin. Once done, you can close Origin to return to PPT and the graph will update accordingly.

If you upgrade your Origin to version 2024 SR1 and find that the graph in PPT fails to update, follow solution below to solve this problem:

  1. Change system variable @CWM = 1.
  2. If the above solution doesn't solve the problem, please also set system variable @CE = 1.

Keywords:PPT, paste graph, OLE, paste special, not update, Office 2021, Origin 2024
