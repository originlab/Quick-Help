1.40 FAQ-156 How does the legend change when adding/removing plots?

Legends-Update-Adding-Plot

Last Update: 3/9/2021

Origin forces the legend to be updated when adding plot to a graph by default.

To prevent the forced update of the legend when adding a plot:

  1. When the graph is active, select Format: Page from Origin menu. (or Press F2)
  2. Choose the Legends/Titles tab on the right panel.
  3. Select None from the Legend Update Mode dropdown list. This option will be saved with the graph template.

To make the legend reconstruct when adding/removing plots,

  1. With the graph is activated, select Format: Page from Origin menu. (or Press F2)
  2. Choose the Legends/Titles tab on the right panel of the Plot Details dailog.
  3. Select Reconstruct when Adding/Removing from the Legend Update Mode drop-down list.


Note:

Origin 2020b had fixed the following bugs:

  • If Legend Update Mode is set to Update when Adding and when you add new data to the active layer of a multi-layer graph, legends in the non-activa layers well update.
  • If Legend Update Mode is set to None and you add a function plot to the graph, the legend will update.

If you see problems above, please update your Origin to the latest version.
