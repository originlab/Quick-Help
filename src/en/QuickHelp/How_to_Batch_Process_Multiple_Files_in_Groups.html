3.128 FAQ-761 How to batch process multiple files in groups?

Batch-Process-Groups

Last Update: 10/20/2020

To batch process multiple data files in groups, you can create a multi-data sheets analysis template so that data files in the same group can be imported into corresponding data sheets in analysis template. To learn how to create a multi-data sheets analysis template and how to use the created template to batch process files in groups, please refer to this tutorial.

While not as flexible or powerful as the Batch Processing operations linked to above, a second option is to "clone" import and analysis operations in the active workbook, using the procedure outlined in this tutorial.

Minimum Origin Version Required: Origin 2016 SR0

Keywords:Batch process, groups, grouped, clone
