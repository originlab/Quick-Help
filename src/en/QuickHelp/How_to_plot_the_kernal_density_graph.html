3.92 FAQ-330 How to plot the kernel density graph?

kernel_density_graph

Last Update: 4/26/2023


For 1D Kernel Density Graph

  1. Create a histogram first (From menu Plot > Statistical :Histogram)
  2. Double-click on your graph which will open the Plot Details dialog.
  3. Make sure the data plot is selected from the left panel of the dialog, and then on the right panel select the Distribution tab. Choose Kernel Smooth from the Distribution Curve Type drop-down list
  4. Go to the Data tab and check Hide Bins.


For 2D Kernel Density Graph

  • Choose Plot > Contour : 2D Kernel Density menu
