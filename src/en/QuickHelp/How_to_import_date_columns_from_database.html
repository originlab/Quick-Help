2.69 FAQ-794 How do I import date data from database?

DB-Import-Date

Last Update: 11/15/2019

When you import date columns from a database, you may find that the date format is lost. To keep the date format while importing,

  1. Select Preference: Options from Origin menu to open the Options dialog.
  2. Go to Numeric Format tab, select the appropriate format for the Date Format for Database Import drop-down list. You can also enter a custom date format using Origin's date-time format specifiers, in the edit box directly.
