1.47 FAQ-164 Is there an easy way to copy graph properties from one graph to another?

Share-Graph-Theme

Last Update: 1/7/2019

Two methods:

  1. Simply right-click on the graph element and choose Copy Format from the context menu. You can then Paste Format on another graph or other layer in same graph. You can also choose Edit: Paste Format (Advanced)... to paste to multiple graphs.
  2. Apply themes on a graph page. Themes offer an easy way to copy features of one graph and apply them to any or all graphs in a project or even all future graphs (in the case of a System Theme). To use themes, press F7, or select Preferences: Themes Organizer, to open the Theme Organizer dialog and apply the proper theme on the graph.
