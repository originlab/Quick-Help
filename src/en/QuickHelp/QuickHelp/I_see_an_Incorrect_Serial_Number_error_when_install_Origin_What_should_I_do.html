6.84 FAQ-1102 I see an "Incorrect Serial Number" error when install Origin. What should I do?

Incorrect-Serial-Number

Last Update: 5/24/2021

If you see Incorrect Serial Number error message when install Origin,

|


Please double check if the input serial number is exactly the one you received from OriginLab. The serial number is in the form of XXXXX-XXXX-XXXXXXX

  • We always recommend copying the serial number from email sent from OriginLab and pasting it to the installer wizard If possible.
  • If you need to input the serial number manually, make sure all characters are typed in English input method. Do not miss the character "-" connecting between segments.
