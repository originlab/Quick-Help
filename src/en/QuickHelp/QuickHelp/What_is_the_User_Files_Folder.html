6.13 FAQ-15 Origin asks me for a User Files Folder. What is this?

UserFilesFolder

Last Update: 1/3/2019

The User Files Folder (UFF) is the default folder for saving and opening Origin-related files.

To open your User Files folder:
  1. Select Help: Open Folder: User Files Folder menu to open the folder directly.

or

  1. Click Preference: Options in the main menu, go to the System Path tab.
  2. Select User Files folder row to click Copy button to copy the path, and then use this path to open the current User Files Folder in Windows Explorer.


To change the User Files folder:
  1. Open Preference: Options> System Path tab, select the User Files Folder entry in the table, then the click the Change button. or you can run the command
    changeUFF -d;
    in Script Window. This opens the Change User Files Folder dialog.
  2. Use the Browse button to point to a new location and click OK.
  3. You will need to restart Origin for this change to take place.
