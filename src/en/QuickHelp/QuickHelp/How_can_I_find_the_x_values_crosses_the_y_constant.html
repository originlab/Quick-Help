3.115 FAQ-673 How can I find the x values where my data plot crosses the y=constant such as x axis?

Intersection-Point-Values

Last Update: 10/21/2021

There are several ways:

  1. Download and use the levelcrossing APP to find the X values where the input curve crosses a specified y value.
  2. Use the built-in levelcrossing x-function.
  3. With graph window active, use File: New: Function Plot: 2D Function Plot... to add a plot with specified Y value in current graph. Then use Gadget: Intersect to find the intersect points of the curve and function plot.

Note: To find y values of a given X value, please use Gadget: Vertical Cursor.

Keywords:Intersect, cross, x-axis
