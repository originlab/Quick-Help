1.240 FAQ-1018 How to fix the font size for a Word object inserted into a graph?

Fix-FontSize-for-Word-Obj

Last Update: 9/8/2019

If you have inserted a Word Object into a graph, want to keep the Font Size set in the Word OLE, then you can right-click on the Word object to select Properties to open Documentation Properties dialog, go to View tab to check the Relative to Original Size check box.

Keywords:export graph, Word Object, Font size, insert Word
