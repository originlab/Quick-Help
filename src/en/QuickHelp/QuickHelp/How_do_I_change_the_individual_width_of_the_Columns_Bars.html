1.59 FAQ-180 How do I change the individual width of the columns/bars in the graph?

change-bar-column-width

Last Update: 2/4/2015


You can use the values in another data column to control the spacing between each column/bar. This allows you to control the column/bar width individually. To enable this control, double click on the data plot to open the Plot Details dialog box. Go to the Spacing tab and select a data column for Width (in %) from the drop-down menu. You can also set the Scaling Factor. For more information, please refer to this help file.

Keywords:column, bar, plot, width, control, spacing

Minimum Origin Version Required: 9.1 SR0
