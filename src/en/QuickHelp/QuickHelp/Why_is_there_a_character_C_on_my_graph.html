1.201 FAQ-934 Why is there a character "C" on my graph?

Character-C-Shows-on-Graph

Last Update: 12/9/2021

When the unexpected character "C" shows on your graph, that means the Origin copy you are using is invalid or pirated, please contact OriginLab to get a legal copy. Or you can download our evaluation version from our website.

If you do have a valid license but still have big "C" on the graph,

please try the following:

  • Download the latest version and install it.
  • Right click on the Origin icon and select Run as Administration to launch Origin.
  • If you have a node-locked license, select Help: Activate License to reactivate the license.

If above solutions cannot solve the problem, please contact us.
