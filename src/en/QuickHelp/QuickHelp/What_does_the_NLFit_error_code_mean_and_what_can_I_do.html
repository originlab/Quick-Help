3.14 FAQ-239 What does the NLFit error code mean and what can I do?

NLFit-ErrorCode

Last Update: 2/4/2015

If errors occur in nonlinear fitting, you will see an error code in your fit report. It is in the Fit Status field in the Statistics table.

For more information, please see Fit Status Code in NLFits Fit Result.
