1.293 FAQ-1176 Why my exported image is different from the Origin graph window?

Export-Graph-Different-From-Inside-Origin

Last Update: 6/1/2023

If your exported image looks different from the one in Origin Graph window, e.g., unexpected lines/objects, it is usually because there is another window with the same short name in the project file.


To workaround this problem:

  • Duplicate the current Graph window by on the Standard toolbar, or right click on Graph window title bar and select Duplicate. And then export the duplicated graph again.


  • If you see different font size on the exported image, please refer to this FAQ.
  • If your copied image has extra white space, please refer to this FAQ
