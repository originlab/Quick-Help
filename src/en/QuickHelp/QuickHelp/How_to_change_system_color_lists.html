1.220 FAQ-967 How to change system color lists?

Change-System-ColorList

Last Update: 1/4/2019

After build your own color list, you can choose to set it as system color list in Origin, by Theme Organizer tool (Preferences: Theme Organizer). In the future, no matter what kind of graph type you plot, this new color list will be used by default.

To learn how to define your own color list by selecting colors from any place on the computer screen and then set it as default color list, please refer to this video tutorial.

You can also refer to this blog for detailed steps of changing increment list in Origin.
