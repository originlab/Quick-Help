5.40 FAQ-743 How do I sort windows by multiple attributes under Project Explorer?

Sort-Multiple-Attributes

Last Update: 10/20/2022

You can do nested sorting by multiple attributes in Project Explorer by holding down SHIFT key when clicking column headers.

For instance, say you want to sort by window type ascending and then by window long name descending.

First click the Type column header to sort by Type. Arrow will show next to Type. Up arrow means ascending. Down arrow means descending. If you see a down arrow, you can click the Type column header again to change it to an up arrow. Then hold the SHIFT key and click the "Long Name" column header once or twice so that down arrow shows next to Long Name. Here Long Name is a secondary sort key.

Note: The Layer Contents dialog, Plot Setup, and Column/Dataset Browser also have this feature.
