5.58 FAQ-912 Restoring Display of Folders in Lower Panel of Project Explorer

Restoring-Display-Of-Folders-In-Lower-Panel-Of-Project-Explorer


Last Update: 3/19/2018

Starting with Origin 2018b, by default folders are not displayed in the lower panel of Project Explorer.

To restore the display of folders in the lower panel, set the System Variable @PEF=1 (0 is default and means hide folders).

To learn how to set the System Variable @PEF=1, please see: Setting Your Preferences with System Variables
