1.135 FAQ-732 How to align Y axis at certain value for a multi-Y graph?

align-axis-at-certain-value


For Multi-Y graph, the first layer is parent layer by default and other layers are child layers. Double-click on the child layer to open Plot Details dialog, go to Link Axes Scales tab, select Align for Y Axis Link group, enter the desired value in the Y= text box.

Keywords:align, alignment, Axes start from same value, align axes, same axis start, axis beginning
