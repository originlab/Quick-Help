Changing the license server

If your license server is no longer functioning or is being decommissioned and you need to change the server host, please submit a System Transfer Request form. After your request is processed, you will be notified via email. You can then obtain a new license file and set up the FLEXnet service on the new server. Here is more information on FLEXnet Setup.

How to submit a System Transfer Request form

  1. Log in to the OriginLab website with your web membership account
  2. Click the Manage My License(s) link
  3. Check the check box in front of the Serial Number you would like to transfer
  4. Click the Report a license server no longer in use button
  5. Complete the form and read the Licensing and Installation Support Policy
  6. Submit the request

Please contact OriginLab about your System Transfer request if your version is no longer supported.

If you are outside of US/CAN, please contact your local distributor.

Once your request is processed, you can proceed to obtain a new license file and set up the new server.

  • See the FLEXnet setup instructions for Windows or Linux.
