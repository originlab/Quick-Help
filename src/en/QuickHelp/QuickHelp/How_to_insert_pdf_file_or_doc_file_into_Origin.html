5.73 FAQ-1027 How to insert pdf file or doc file into Origin?

Insert-Word-or-PDF

Last Update: 3/19/2024

  1. Create a new layout window by clicking New Layout button on the Standard toolbar. You can right click on the window title bar and select Properties to rename it as you desire.
  2. Select Insert: OLE Object from menu.
  3. Select Create from File and select the file by clicking Browser button.
  4. Click OK to insert the document as an object to the layout window.

Keywords:insert word file, doc file, insert pdf file, insert object, layout
