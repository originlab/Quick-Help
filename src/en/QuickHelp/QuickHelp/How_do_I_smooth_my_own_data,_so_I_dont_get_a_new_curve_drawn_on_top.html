3.65 FAQ-302 How do I smooth my own data, so I don't get a new curve drawn on top?

Smooth-Data-without-NewCurve

Last Update: 2/4/2015

When you select Analysis: Signal Processing: Smooth, the default Output is <new>, meaning a new smoothed curve is generated. If you want your original curve smoothed, then choose <input>: XY=inputXY.

If the Recalculate Mode is set to Auto or Manual, you will see a message output to the Script Window: "Unable to establish autoupdate. Recalculate mode is set to NONE." The Recalculate feature is not available when the output is the same as the input.

Keywords:output, input, new
