1.121 FAQ-683 How do I rescale only the X or Y axis of a graph?

rescale-x-or-y-only

Last Update: 7/6/2018

When you plot worksheet data, the graph layer's axes are automatically scaled to include all data points. Subsequently, if you add or remove plots from graph layer, axes scales automatically adjust to accommodate changes to the scale range.

To rescale only X or Y axis:

  • With the graph activated, click the Rescale X or Rescale Y button.


Otherwise, you can open the Scale tab of Axis dialog to set the Rescale mode of the current axis to be Fixed. Then, this axis will be prevented from rescaling when there is any plot been added or removed from the current graph layer.
