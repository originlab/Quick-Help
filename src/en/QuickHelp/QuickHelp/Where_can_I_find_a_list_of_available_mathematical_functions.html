3.55 FAQ-290 Where can I find a list of available mathematical functions?

MathFunctions-in-SetColValue

Last Update: 2/4/2015

The Set Values dialog is used to define a mathematical expression that fills a column with values or executes more complex scripts. To open the Set Values dialog box, select a worksheet column or a range of cells in a worksheet column, and then choose Column: Set Column Values... from the menu. Alternatively, right-click on the worksheet column and choose Set Column Values... from the shortcut menu.

To search functions, click on the Search and Insert Functions button or select from Function: Search and Insert... to bring up Search and Insert Functions dialog. You can input keywords to search and use scroll bar to browse through the searched results, then double click on the desired function to insert into Formula box. You can also click the function description to go to the detailed function documentation page.

Keywords:mathematics, formula, math

Minimum Origin Version Required: 2015 SR0
