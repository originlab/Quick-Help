6.60 FAQ-898 What is new in Origin 2018 for Node-locked license?

New-in-2018-NodeLocked

Last Update: 11/12/2017

From version 2018 onwards the node-locked license is cross-version, similar to how it works currently for network versions. So you can use the same product key to activate multiple versions on the same computer. For now, this means same product key for 2017 and 2018.

Refer to this page -> "If yes..." section for detail steps of upgrading.
