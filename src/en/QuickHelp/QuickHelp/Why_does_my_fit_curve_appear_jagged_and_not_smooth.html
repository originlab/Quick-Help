3.16 FAQ-241 Why does my fit curve appear jagged and not smooth?

Jagged-Fit-Curve

Last Update: 8/8/2018

If the resulting fit curve is not sufficiently smooth, the X data settings used to generate the fit curve should be modified.

To do this open the NLFit dialog by selecting Analysis: Fitting: Nonlinear Curve Fit, then the Settings tab, and then select Fitted Curves. Expand the Fitted Curves Plot branch and modify settings for the X Data Type, which changes the X scale, and/or increase the number of Points, which is the number of points used to draw the fit curve.
