3.78 FAQ-316 How can I customize the scatter matrix plotted with the correlation coefficient tool?

Customize-Scatter-Matrix-with-Correlation-Coefficients

Last Update: 2/4/2015


You can use the graph template to save customized setting of the scatter matrix plotted with the correlation coefficients tool


Run correlation coefficient tool with a scatter matrix plotted; double-click to open an embedded plot; customize it and save the graph template. Then the next time when you run correlation coefficient tool, the individual graphs will be changed accordingly.


Note:To have more options like histogram or box chart in diagonal cells or what statistics to be shown in the scatter matrix, you can plot the scatter matrix from the plot menu (Plot:Statistics:Scatter Matrix). To customize the scatter matrix available from the plot menu, please refer to the tutorial
