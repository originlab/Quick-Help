2.66 FAQ-692 How to set values of a cell or a subrange?


Last Update: 5/16/2019

Origin 2018 and on supports cell formula. You can define cell formula like Excel and then drag to fill similar formula for a subrange.This is also supported in user-defined parameter rows.

To make absolute references to a cell value, place square brackets around the cell's row index number (e.g. "=B1+C[1]"). This way when you drag the corner to auto fill, B1 will change to B2, B3, ... while C[1] will be fixed.

In older versions, to set values for a subrange of a column or multiple columns, you can select the subrange, right-click and select Set Column Values or Set Multiple Column Values. Use the range/column controls at the top of the dialog box to specify the subrange and for calculating the formula entered in the column formula edit box.

Please refer to the Set Values dialog documentation for details.

Minimum Origin Version Required: 9.1 SR0
