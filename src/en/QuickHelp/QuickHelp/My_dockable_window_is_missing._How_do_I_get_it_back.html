5.2 FAQ-338 My dockable window is missing. How do I get it back?

Get-Dockable-Window-Back

Last Update: 3/30/2022

If the normal menu option or keyboard shortcut fails to display a dockable window, select View: Toolbars to open the Customize Toolbar dialog. Click the Reinitialize button . This will reinitialize all toolbars and dockable windows. (You will lose any custom toolbar configurations you have.)


A Special Note on the Data Display Window:

If the Data Display tool fails to display when selecting tools such as Data Reader or Annotation, you should be able to restore it by setting LabTalk system variable @BDDS = 1 (show Data Display window). Note that the window's dock state can be controlled by setting @BDDF = 0 (dock) or = 1 (float).

For information on changing the value of a system variable, see this FAQ.
