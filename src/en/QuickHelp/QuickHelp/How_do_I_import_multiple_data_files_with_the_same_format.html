2.6 FAQ-413 How do I import multiple data files with the same format?

Import-Multi-Data

Last Update: 1/7/2019

Both the Import Wizard (Data: Import from File: Import Wizard) and the Multiple ASCII (Data: Import from File: Multiple ASCII) methods allow you to select one or more files and all files will import using the same customizable group of settings.

  • For information on the Import Wizard, see The Import Wizard.
  • For information on Multiple ASCII import, see Customizing the ASCII Import Options

Note that you can create filters (*.OIF files) with the Import Wizard or save Themes in Multiple ASCII Import that allow you to save a particular set of import options to a file. This makes it possible to drag-and-drop more complex files (Import Wizard) or import by simply choosing a Theme from a "most recently used" menu command.
