1.176 FAQ-846 How to avoid drawing small error bars inside open plot symbols?

Draw_small_error_bars_inside_open_plot_symbols

Last Update: 8/7/2018

When creating a scatter plot with error bars and open symbols with fill color (no transparence), error bars smaller than the symbol size will be drawn inside the symbol by default. The error bars are then overlapped by the symbol and thus cannot be seen. To avoid this, open the Layer Contents dialog (double-click on layer icon on top left of the graph), and simply change the plotting order so that the error data is listed above the Y data.

Keywords:plot order, layer contents
