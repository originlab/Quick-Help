1.122 FAQ-697 How can I paste graph into PowerPoint with the certain size?

Paste-Graph-with-CertainSize

Last Update: 1/3/2019

Sometimes, if you specify a desired graph page size in the Plot Details dialog, when paste this graph into PowerPoint, Word or other applications, you may find that the graph size is not the one you specified. That is because Origin does not copy the page at 100% by default. To paste the graph with the certain size,

  1. Select Preference: Options to open the Options dialog
  2. Activate the Page tab. Change the Ratio value to 100 (default is 40). This will copy 100% size of your original graph. Click OK to close the dialog.
  3. Double click on the graph to bring up the Plot Details dialog.
  4. Select Graph layer on the left panel. Activate the Print/Dimensions tab on the right panel. Specify the desired Units, Width and Height. Click OK to close the dialog.
  5. Now right click on the graph page and choose Copy Page. Then go to PowerPoint or Word, right click on one slide and in the “Paste Option” select “Picture (U)”.

Keywords:Graph size, Paste Graphs, Other software

Minimum Origin Version Required: 8.0
