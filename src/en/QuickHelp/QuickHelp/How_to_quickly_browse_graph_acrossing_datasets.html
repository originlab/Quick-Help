1.252 FAQ-1045 How to quickly browse graph acrossing datasets?

Browse-Graphs-Accrossing-Datasets

Last Update: 4/9/2020


Origin 2020b supports Graph Browser to quickly browse a set of graphs. That is,

  1. Put all datasets in the same worksheet.
  2. Select menu Plot: Browser.
  3. Use left panel to switch acrossing different datasets.
