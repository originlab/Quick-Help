5.61 FAQ-922 Preventing Corruption of Origin Files When Sending Via Email

Email-incompatible-file-type-corruption

Last Update: 2/28/2018

Some users have reported that Origin files transferred as email attachments fail to open in Origin, or Origin reports errors similar to: 

File corruption: marker not at structure end: 000000000000001F
File format error. Incompatible file type or file may be corrupted.

This issue can arise because some email clients and/or virus scanners may subtly alter Origin file email attachments rendering them corrupt to Origin.

In order to mitigate this issue, it is strongly suggested that Origin files be zipped (or otherwise compressed) prior to being attached to emails.
