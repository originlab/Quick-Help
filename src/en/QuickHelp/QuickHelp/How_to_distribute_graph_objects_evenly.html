1.187 FAQ-882 How to distribute graph objects evenly?

Distribute_Graph_Objects

Last Update: 9/12/2017


If there are multiple objects in your graph, and you want to distribute them evenly vertically or horizontally.

You can try to:

  1. Select the multiple objects.
  2. Click the Distribute Vertically button or Distribute Horizontally button .

Then the position of the top and the bottom (rightmost and the leftmost) selected object will be kept, and distribute selected objects vertically (horizontally) to space at equal distances from each other.

Keywords:graph object, distribute vertically, distribute horizontally

Minimum Origin Version Required: 2018 SR0
