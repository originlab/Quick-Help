6.29 FAQ-32 Why do I see an expiration message when I start Origin?

Expiration-Message

Last Update: 12/12/2021

If you purchased an expiring or renewable Origin package,

You may be seeing the expiration message because the expiration or renewal date is approaching. Please contact OriginLab Support for renewal.

If you did purchase a perpetual Origin package,

Make sure you are using the perpetual license (the product key is marked as "primary") or the expiring one (the product key is marked as "home"). The latter is a benefit from the annual Maintenance service. It will expire along with the Maintenance. If so, please contact OriginLab Support to renew your Maintenance.


Note:

An expiration message may also display at Origin start up if the Origin is not properly licensed. Please refer to this page.
