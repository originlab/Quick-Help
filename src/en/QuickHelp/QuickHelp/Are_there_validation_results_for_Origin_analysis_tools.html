3.156 FAQ-1002 Are there validation results for Origin's analysis tools?

Compare-Origin-Output-with-NIST

Last Update: 4/22/2019

From Origin 2019b, sample projects add a group of projects that compare analysis output from key Origin tools against published National Institute of Standards and Technology (NIST) results. You can select Help: Learning Center, set Samples in as Validation on Analysis Samples tab, and then find the desired comparison projects.
