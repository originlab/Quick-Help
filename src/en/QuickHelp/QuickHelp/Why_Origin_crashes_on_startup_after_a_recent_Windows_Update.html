6.76 FAQ-1086 What should I do if Origin crashes on startup?

Crash-after-Windows-Update

Last Update: 2/19/2024

If Origin starts having following problems on startup,

  • crashes on startup just after splash screen, or
  • other issues where Origin misbehaves, like operates very slowly, right click does not work

the problem is probably caused by redistributable package is not properly installed. Please try the following to solve it:

Origin 2023b and later Origin 2022b and 2023 Origin 2022 and earlier
  1. vcredist_x86-2015-2022.exe
  2. vcredist_x64-2015-2022.exe
  1. vcredist_x86-2015-2022.exe
  2. vcredist_x64-2015-2022.exe
  3. vcredist_x64-2012.exe
  1. vcredist_x86-2008.exe
  2. vcredist_x86-2010.exe
  3. vcredist_x86-2012.exe
  4. vcredist_x86-2015-2022.exe
  5. vcredist_x64-2008.exe
  6. vcredist_x64-2010.exe
  7. vcredist_x64-2012.exe
  8. vcredist_x64-2015-2022.exe

Please search all of them in the "Uninstall or change a program" list.

  • If a required DLL is found, right click on it and choose Uninstall from context menu. If prompted, restart PC.

    Keep only one DLL for each version. That is, if you find multiple DLLs of the same version are installed, for example, 2015 and 2015-2019, or 2017 and 2015-2019, uninstall the redundance.

  • Download all the required DLLs on above table and install them.
  • Once done, restart your PC for the changes to take effect.
  • Now try Origin again (Run Origin as administrator if needed)

Keywords:crash, fail to execute, not function, Start up, startup, Repair, cannot open, right click, slow
