3.39 FAQ-268 How to estimate initial parameters for rational functions

Estimate-Initial-Parameters-for-Rational

Last Update: 7/20/2018

Origin provides a built-in OriginC function ocmath_rational_params_init to estimate initial parameters for rational functions. The built-in fitting function rational5 in the Rational category shows how to call this function to estimate its initial parameters in the Parameter Initialization box of Function Organizer

Keywords:fit, rational, estimate, initial, parameter

Minimum Origin Version Required: 9.1SR0
