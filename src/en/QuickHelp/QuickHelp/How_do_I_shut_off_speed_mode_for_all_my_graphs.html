1.101 FAQ-620 How do I shut off speed mode for all my graphs?

shut-off-speed-mode-for-all

Last Update: 4/26/2022

In order to turn on/off speed mode for all graphs in folder, project or all instances, activate a graph window and select Graph:Speed Mode to open the speedmode dialog.
