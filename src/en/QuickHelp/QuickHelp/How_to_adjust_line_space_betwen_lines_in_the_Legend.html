1.131 FAQ-719 How to adjust line space betwen lines in the Legend?

adjust-line-space-for-legend

Last Update: 9/21/2022

If you want to adjust the line space between lines in the legend, you can right-click the legend to select Properties... from the context menu to open the Text Object dialog. In the Text tab of this dialog, for the Line Spacing(%) item, select a value from the drop-down list or enter a value in the combo box directly.

Please note, when you set the Line Spacing(%) to <auto>, the space between the lines in the legend will follow the setting of Line Spacing in the Text Fonts page of Options dialog.

Beginning with Origin 2023, you can create irregular line spacings (i.e. groupings) within the legend object using the methods described here.
