6.91 FAQ-1119 I see an error message saying "Your FLEXlm license file does not match your OriginLab serial number". What should I do?

Server-Lic-Not-Match-SN

Last Update: 12/28/2021

If you see following message when trying to connect your Origin to the license server:

License Status: 9

Your FLEXlm license file does not match your OriginLab serial number.

Please contact your license server manager to check the correct serial number and license server information.

  • If you installed Origin with the wrong serial number, please follow steps on this page to correct it.
  • If your serial number and server information are correct but still see this problem, there may be some license issue on the server side. Please let the server manager refer to this page to fix it.
