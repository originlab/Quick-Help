2.132 FAQ-1163 How do I add a file extension to a Data Connector?

Add-File-Extension-Data-Connector

Last Update: 1/28/2023


Open Apps: Connectors to see a list of installed Data Connectors. Right-click the Data Connector and select Custom File Extensions....

In the dialog, you can add your extension by using the wildcard character * (asterisk) followed by the extension. Each extension in the list is separated by the ;(semicolon) character.

Be careful to not have erroneous spaces or blank characters in your extension. There should not be a space between ; and *.
