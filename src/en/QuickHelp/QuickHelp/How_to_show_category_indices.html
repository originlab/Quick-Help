2.114 FAQ-1043 How do I know the categorical values if I set a column as categorical?

Show-Category-Indices

Last Update: 9/22/2021


When you set a worksheet column as categorical, that is, right click on a column and select Set as categorical, the column values are mapped to a series of indices. In all post-processing and graphing involving this categorical column, Origin will treat the categorical values as index numbers internally. To see the corresponding index numbers, click the upper left corner of worksheet and click Show Category Indices button on the mini toolbar that appears.
