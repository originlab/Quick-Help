1.110 FAQ-652 How to check if my data plots are grouped, if not how to group them?

check-if-plot-are-grouped

Last Update: 7/2/2018

When highlighting multiple datasets and plot them together, they are grouped automatically. To check if data plots are grouped or not, double click on the graph to open Plot Details dialog. Expand LayerN node on the left panel of the dialog and select the 1st data plot node. If there is Group tab, it means data plots are grouped in this layer.

If you want to group multiple data plots, double click the corresponding layer icon to open Layer Contents dialog. On right panel of the dialog, select the data plots you want to group. Click the Group button.

Note:

  • There can be multiple data plot groups on a layer and Group tab only appears for 1st data plot in each group. There is only Offset tab for other data plot in the group.
  • Data plots must have the same plot type and be contiguous in the list to be grouped.
