1.150 FAQ-776 How to edit annotations belong to same plot or plots in same group?

edit-annot-group

Last Update: 10/17/2016

To quickly edit the annotations belong to same plot or plots in same group, you can first edit an annotation object by following the instructions in this page, and then select Plot or Grouped Plots in Apply to drop-down list as needed.

Minimum Origin Version Required: 2016 SR0
