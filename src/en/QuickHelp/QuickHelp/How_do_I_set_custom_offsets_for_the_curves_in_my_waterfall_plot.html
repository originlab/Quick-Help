1.74 FAQ-203 How do I set custom offsets for the curves in my waterfall plot?

Set-Offset-for-Waterfall

Last Update: 1/8/2019

2D waterfall graph

X and Y offsets between adjacent plots is set from the Waterfall tab of the Plot Details dialog.

  1. Select Format:Layer.... The Plot Details dialog will open with the layer selected on the left-hand side of the dialog.
  2. On the right-hand side of the dialog, on the Waterfall tab, you can modify both the X and Y Offset values.


Note: Alternatively, the plot viewing angle can be modified graphically by selecting the graph layer and manipulating the selection handles.

3D waterfall graph

You can skew the graph graphically and set the X Offset and Y Offset values in the Miscellaneous tab of Plot Details dialog.
