1.172 FAQ-838 How to transpose a virtual matrix?

Transpose-VM

Last Update: 9/19/2016

All virtual matrices in a project will list in the Virtual Matrix Manager dialog. To transpose a virtual matrix,

  1. Select from menu Tools: Virtual Matrix Manager to open the Virtual Matrix Manager dialog. You will find all virtual matrices list in the Virtual Matrices box on the left.
  2. Select the desired virtual matrix and right click, select Duplicate as XY Transpose from the context menu. This will duplicate and transpose the selected virtual matrix.

Keywords:Virtual Matrix, transpose, duplicate

Minimum Origin Version Required: Origin 2015
