1.141 FAQ-755 How to set global unit display format for axis title?

Global-Unit

Last Update: 3/29/2021

From the main menu, go to Preference: Options... to open Options dialog. Go to the Axis tab, select a unit display format by clicking the button next to Unit Display for Axis Title edit box and choose a style from the fly-out menu.

Note that you can also customize the notation my manually editing the displayed string. For instance, if you want to display the unit as in Unit you can input in <Unit> in the edit box.

For more details, please refer to this page.

Minimum Origin Version Required: 2016 SR0
