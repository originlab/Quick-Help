1.180 FAQ-866 How do I use Master page in the 3D graph?

how_to_use_Maste_page_in_3D_graph

Last Update: 7/5/2018

Currently 3D graph in OpenGL mode (which is default) does not support Master page. There are two work-arounds:

  • Use a Layout page.

or,

  • Turn OFF the 3D OpenGL mode by setting @OGL system variable to 0 and plot the graph again

Note that non-OpenGL 3D graph only supports matrix data, the method 1 may be generally more convenient.

Keywords:Master page, 3D graph
