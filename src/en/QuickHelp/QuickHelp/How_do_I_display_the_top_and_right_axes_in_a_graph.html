1.4 FAQ-114 How do I display the top and right axes in a graph?

Last Update: 4/19/2019

Display-Top-Right-Axis

Display Top and Right Axes in Current Graph

A single graph layer is made up of 4 axes (left Y, bottom X, right Y and top X). By default only bottom and left axes show. Here are different ways to show top and right axis.

  • Since Origin 2017 Sr1, choose Preferences:Theme Organizer. On Graph tab, select All Axes On and click Apply Now to turn all axes on. The top and right axis will also show ticks.
  • In Theme Organizer, there is also an Opposite Lines theme, which will show top and right lines without ticks.
  • For 2D graphs in Cartesian coordinates, choose View: Show: Frame to show the layer frame. Beginning with Origin 2019b the frame will follow the line width and color of the graph axes.
  • Double click any axis line. Select Line and Ticks tab. Select Top and Right on the left panel, and check Show Line and Ticks.
  • Double click any axis line. Select Grids tab. Select Vertical and Horizontal on the left panel, and check Opposite.

Display Top and Right Axes for Future Graphs

  • Since Origin 2017, we added a built-in graph theme called All Axes On. So choose Preferences: Theme Organizer menu. On Graph tab, right-click on the All Axe on theme and choose Set as System Theme from the context menu.

    You can also hold the CTRL key down and select more than one theme and then right click and select the Combine context menu item. This will combine the chosen themes and create a new theme. You can then suitably name that theme and then right click and set that theme as your system theme.

  • In older Origin, there was a built-in graph theme called Opposite Lines. You can set it as system theme. There are no ticks on top and right axes.

    Or you can create a graph theme with top and right axes on and set it as system theme. E.g plot a simple graph. Turn the top and right axes on first. Then click within layer to select the layer frame. Right click and choose Save Format as Theme. In Theme Organizer, set it as system theme. Many layer properties are saved in the theme which are not related to show top and right axes. It may be hard for you to edit the theme so it may not work for all cases.
