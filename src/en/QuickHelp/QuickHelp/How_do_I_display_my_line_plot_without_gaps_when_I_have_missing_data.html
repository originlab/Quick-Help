1.66 FAQ-189 How do I display my line plot without gaps when I have missing data?

Display-no-Gap-for-Missing-Data

Last Update: 1/8/2019


To display the line connection across missing values, do the following when the graph window is active:


  1. Select Format: Page... to open the Plot Details dialog
  2. Go to the Display tab.
  3. Check the Connect line across Missing Data checkbox.
