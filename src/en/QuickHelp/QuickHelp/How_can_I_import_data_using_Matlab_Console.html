2.21 FAQ-435 How to import/export data from/to MATLAB Workspace ?

Matlabconsole-Importdata

Last Update: 1/8/2019

In Origin, you can use MATLAB Console to import data from MATLAB and put them into worksheet or matrices with the graphical interface and issuing commands. On the other hand, you can export data from Origin to MATLAB workspace as variables to be called directly. This feature requires both Origin and MATLAB be installed on the same computer.

To start the MATLAB Console tool, select Connectivity: MATLAB Console from the Origin menu to bring up the MATLAB console dialog. Run desktop in the panel. This MATLAB command will open the standard MATLAB desktop. After importing the data file into MATLAB, go back to the MATLAB Console dialog in Origin, click Import button to bring up the MATLAB Import dialog. Choose the desired data and click Import as Matrices or Import as Worksheet to import the data into Origin.

Conversely, to export data from Origin to MATLAB desktop, select Connectivity: MATLAB Console from the Origin menu to bring up the MATLAB console dialog. Run desktop in the panel to open the standard MATLAB desktop. Now click Export button to initiate exporting data into MATLAB workspace.
