5.1 FAQ-337 How do I customize and reuse toolbar settings?

Customize-Reuse-Toolbar

Last Update: 7/29/2015

The View: Toolbars menu opens the Customize Toolbar dialog. From here you can control which toolbars to display. On the Button Groups tab you can access all available toolbar buttons, including some which are not available by default.

For example, select Column in Button Groups and you will see the Swap Column button. You can drag-n-drop this button onto any existing Origin Toolbar.

If you simply want to move buttons from one toolbar to another, there is no need to open the Customize Toolbar dialog. Simply hold down the Alt key and drag-n-drop the button onto another toolbar. If you drop it anywhere other than a toolbar, it will display as a separated floating button.
