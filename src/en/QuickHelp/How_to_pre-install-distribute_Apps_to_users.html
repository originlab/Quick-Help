6.108 FAQ-1194 How to pre-install/distribute Apps to users?

preinstall-apps

Last Update: 5/22/2024

For multi-user packages, it is convenient to deploy Apps (and updates) in a silent way

Pre-install the apps for the end users have never had Origin installed

  • IT creates a folder named Auto Install in the Origin setup folder (with Setup.exe or MSI)
  • Download the apps files (*.opx) and save them in the Auto Install folder
When the end users install Origin from the Origin setup folder, the Auto Install folder will be copied to the end users' Origin Program Folder. When Origin runs, contained app files (*.opx) are auto-installed.

Post-distribute the apps for ends users have had Origin installed

  • Set up Group Leader(s) and Group Members
    1. The first step is to set the Group Folder for the group leader machine.
    2. Next, you can set the Group Folder for the member machines.
  • Distribute the Apps
    1. Create a folder named Auto Install in the User Files folder on the group leader machine
    2. Download the apps files (*.opx) and save them in the Auto Install folder
    3. Use Group Folder Manager to distribute the apps files (*.opx)
    4. Now when the Origin group member starts their Origin, they will see the apps installed on their Origin
