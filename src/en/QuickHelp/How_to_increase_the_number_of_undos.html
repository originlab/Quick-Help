5.54 FAQ-880 How to increase the number of undos?

increase-number-undo

Last Update: 1/4/2019

From Origin 2018 SR0, by default, Origin supports 100 times of UNDO operations (Ctrl + Z).(The number is 10 before Origin 2018).

However, if you find it is not enough, you can use the system variable @UN to increase the number of UNDOs.

  • To set the number of UNDOs to be 200 in the current Origin session only, open the Script Window, type the following, then press Enter:
@UN=200
  • To increase the number of UNDOs to be 200 in the future Origin sessions, choose Preferences: System Variables to open Set System Variables dialog. Set UN to 200
Note: Increasing the number of UNDO operation may affect the performance of Origin. We recommend that you do not increase the number of undo levels beyond 200.
