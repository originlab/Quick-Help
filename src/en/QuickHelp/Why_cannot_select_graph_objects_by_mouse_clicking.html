1.223 FAQ-984 Why can't I select graph objects by mouse clicking?

block-mouse-clicking-on-graph

Last Update: 12/18/2018

If you cannot select graph objects (including axes, plots, labels, drawing objects, layer frame, .etc) by mouse clicking, you may disable mouse clicking on them unconsciously.

To enable it, set page.noClick to 0. That is,

  1. Select Window: Script Window.
  2. Enter the following script in the Script Window and press Enter to run it.
page.noClick = 0

Keywords:got blocked for dragging, cannot click, can't select, Ctrl + select
