1.56 FAQ-175 How to plot a 3D ternary scatter graph?

Plot-3D-Ternary-Scatter

Last Update: 2/19/2019

Make sure that your columns are designated as X, Y, Z and Z. To do that, right-click on the column and choose Set As from the context menu. Then highlight the XYZZ columns and choose menu Plot > 3D : Ternary Symbol to plot the graph.

You can also add a 3D ternary scatter plot to an existing 3D ternary surface from the Layer Contents dialog.
