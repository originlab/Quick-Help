6.89 FAQ-1116 What should I do if I see a "Location is not available" error after install Origin?

Access-Is-Denied

Last Update: 12/30/2021

If you see following "Location is not available" error message after installing Origin:

C:\Users\username\AppData\Roaming\Microsoft\Windows\StartMenu\Programs\OriginLab Origin #### is not accessible.

Access is denied.

It is because you installed Origin with a non-administration account and chose to install it for current user only. That is,

  1. Logged in Windows with a non-admin account
  2. . Right-clicked on the Origin setup.exe and selected Run as Administrator. Logged in an admin account.
  3. Process according to the installation wizard. On the All Users or Current User?' page, chose Current User Only
This is the step that caused the problem. At that time, the setup program regarded the admin account as "Current User" rather than the one you currently logged in. So the shortcuts will be created under the admin account's folder C:\Users\<AdminUserName>\AppData\Roaming\Microsoft\Windows\StartMenu\Programs\OriginLab Origin2022, which cannot be opened by the current non-admin account.


To solve this problem:

  1. Double click the Origin setup.exe to launch the Origin Setup wizard.
  2. Select Repair/Upgrade.
  3. On the All Users or Current User? page, chose All Users
