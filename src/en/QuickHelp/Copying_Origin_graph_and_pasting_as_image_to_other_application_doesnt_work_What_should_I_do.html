CtrlC-CtrlV-Not-Work

Last Update: 6/17/2022

Origin 2022 improved the behavior of copying graph page as an image. If you press Ctrl + C to copy a graph and then use Ctrl + V to paste it to other applications, e.g. chat apps such as Slack, QQ, google chat, you may find it not functioning.

To fix this issue,

  1. Activate any graph window. Select menu Edit: Copy Graph as Image.
  2. In the dialog, select DIB from Image Format drop-down list and Copy once.

Now you can use Ctrl + C to copy graphs to other applications.
