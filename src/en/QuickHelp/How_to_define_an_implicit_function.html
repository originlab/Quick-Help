3.36 FAQ-265 How to define an implicit function?

Implicit-Function

Last Update: 7/20/2018

To use a function in Nonlinear Implicit Curve Fit tool, the function has to be put in Implicit category.

Select Tools: Fitting Function Builder to open the Fitting Function Builder, in the Name and Type page, select Implicit for Function Model, then you can define an implicit function with this wizard.

Keywords:implicit function, fit

Minimum Origin Version Required: 9.0SR0
