1.216 FAQ-959 How to make X axis have two different groups of tick labels?

Two-groups-X-tick-labels

Last Update: 1/6/2019

To add another group of X tick labels independent from the existing X axis,

  1. With the graph window activated, add one more layer only with top X axis by clicking on the Add Top-X Layer or selecting Insert: New Layers(Axes): Top-X(Linked Y Scale and Dimension)
  2. Move the top X axis down to the position of bottom X axis to make these two axis lines overlap completely.
  3. Double-click on the X axis tick labels of the new added layer to customize the tick labels as you desired.

Note that these two X axes are not linked together.

For more information about double-X axes, please refer to this blog.
