1.123 FAQ-700 How can I turn off Actively Update Plot?

Turn-Off-Update-Plot

Last Update: 3/30/2015

When you change data value in a worksheet/matrix, by default it will automatically update the corresponding data plot if exists without the need to active the graph window again.
If you want to turn off actively update plot, you can go to View:Actively Update Plots to uncheck this item. When you save the opj again, it will keep the settings for Actively Update Plots.

Keywords:Update plot,autoupdate, turn off
