3.169 FAQ-1183 How to get the Z data in a specific X/Y value in Contour plot?

Interpolate-Z-From-XY-In-Contour

Last Update: 8/3/2023

To get the Z value from a pair of specified XY data in a XYZ contour graph, you can use Interpolate Z from XY tool.

  1. Select menu Analysis: Mathematics: Interpolate Z from XY.
  2. In the dialog that opens,
    • XY Values to interpolate: select XY columns you want to calculate the Z values for.
    • Input: select XYZ columns from which the contour is created.
    • Method:
      • Not select smoothing option in contour: use Method = Triangle
      • Select smoothing option in contour: use Method = Random TPS and TPS Smoothing Factor should be Smoothing Parameter you set in the contour.

Keywords:interpolation, Contour value, XYZ Contour, calculate Z
