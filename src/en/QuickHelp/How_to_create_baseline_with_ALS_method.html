3.134 FAQ-778 How to create baseline with Asymmetric Least Squares method?

Create-baseline-with-ALS

Last Update: 5/8/2018

You can create baseline with Asymmetric Least Squares (ALS) method by Peak Analyzer:

  1. On the Baseline Mode page, select the Asymmetric Least Squares Smoothing Baseline (ALS) as the Baseline Method. Click Next.
  2. On the Asymmetric Least Squares Smoothing Baseline page, adjust the parameters to make the baseline optimal, then click Finish button to generate the baseline.

Please refer to this tutorial for details.

Note: the Asymmetric Least Squares method is Pro only.

Minimum Origin Version Required: Origin 2016 SR0
