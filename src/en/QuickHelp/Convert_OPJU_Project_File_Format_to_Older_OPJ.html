2.89 FAQ-927 Is there some way to convert the new OPJU file format into the older OPJ format?

Convert-OPJU-Project-File-Format-to-Older-OPJ

Last Update: 1/5/2023

Origin 2018 introduced a new Unicode-compliant (UTF-8) project file type -- the OPJU file. These files cannot be opened in earlier versions of Origin (though, it should be noted that Origin 2018 through Origin 2022b retain the ability to save files in the older OPJ format). New features in Origin 2023 and later require the OPJU format and therefore cannot save as OPJ.

If you cannot open an OPJU file with your Origin software, do this:

  1. Download the Origin Viewer by visiting this page on the OriginLab website.
  2. Unzip the contents of the Viewer file download.
  3. Double-click on the EXE to run the Viewer (no installation required), then choose File: Save Project As from the Viewer menu and re-save the project as an OPJ file.


For more information on the Origin Viewer, visit this page on the OriginLab website.
