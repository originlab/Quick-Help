1.149 FAQ-773 How to plot the bounds of 90% confidence intervals in probability or Q-Q plot?

Plot-90Bounds-QQ

Last Update: 2/18/2019

Highlight the desired Y column and go to menu Plot > Statistical: Probability Plot/Q-Q Plot to open Plotting: plot_prob dialog. Select a desired distribution function from Distribution drop-down list and specify the Score Method. Make sure Confidence Band checkbox is checked, input desired confidence interval in percent in Confidence Level(%) edit box. Click OK to make the plot with confidence band.

Minimum Origin Version Required: 2016 SR0

Keywords:confidence band, confidence interval, probability plot, Q-Q plot
