3.130 FAQ-766 How to generate a customized Word report after batch processing?

Word-repo-Batch-Process

Last Update: 8/27/2015

You can create a Word Template so that the batch processed results can be outputted to this template and generate a customized Word report automatically.
To learn how to create a Word template and use the created template in batch process, please refer to this tutorial.
