1.55 FAQ-174 How to move the axis plane manually in a 3D graph?

move-3d-axis-plane

Last Update: 1/8/2019


In an OpenGL based 3D graph, you can move the axis plane(s) manually by drag and drop. In order to do that:

  1. Double click on the graph layer to open the Plot Details - Layer Properties dialog. (Or click Format: Layer from top menu).
  2. Go to the Planes tab, for the plane(s) you would like to move, select % From Bottom for the Position drop-down menu(s).
  3. Click OK to apply and close the Plot Details dialog.
  4. Click on the graph layer within the cube box to enable the frame control, in the example below, the Position of all three axis planes are changed to % From Bottom hence all three frames (green, blue and red) are shown
  5. Click on the colored border frame area of the axis plane you would like to move, the axis plane will then be highlighted and a perpendicular arrow will show up:
  6. Click on the arrow and drag to move the axis plane

In the Planes tab of the Plot Details - Layer Properties dialog, you can directly enter the percentage(for % From Bottom option) or arbitrary value (for At Position = option) to precisely change the position the corresponding axis plane.
