1.114 FAQ-664 My dashed lines appear incorrectly or not at all in Page view, Print Preview, or my printout. What do I do?

Dashed-Lines-Incorrectly

Last Update: 1/3/2019

If you are experiencing this problem, try the following:

Select Preference: Options and go to the Graph tab. In the group labeled Origin Dash Lines, check the box marked Use Origin Dashes in Page View and uncheck the box Scale Dash Pattern by Line Width. At this point check to see if Page view or Print Preview appears correct or try printing again.
