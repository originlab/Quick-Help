4.42 FAQ-404 How can I run Python in Origin?


Last Update: 9/20/2016

use-python-in-Origin

Origin provides embedded python environment, so that you can either run python in Origin (support both command line and .py file), and use a PyOrigin module to access Origin from Python.

To get started, you may refer to the Working with Python chapter in the LabTalk Guide.

Keywords:Python, Script, PyOrigin

Minimum Origin Version Required: 2016 SR0
