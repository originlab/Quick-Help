1.181 FAQ-868 How do I remove the line border or red rectangle from the legend?

Customize-Legend-Border

Last Update: 1/8/2019

Removing the line border

  1. Hold down the CTRL key and double click on the legend
    OR
    Right-click and select the Properties... context menu
  2. In the dialog that opens, switch to the Frame tab
  3. Set the Frame drop-down to None

Note: In older versions, the dialog that opens has no tabs, and the drop-down is named Background.


Removing the red rectangle

In some graph templates shipped with Origin, a red rectangle is drawn around the legend entry for the active dataset. Various operations such as curve fitting pick up the active dataset by default. Some users find it helpful to have this red box present within the legend. The red box will not be included in your graph export.

If you wish to turn off the red box, do the following:

In Origin 2018 and newer

  • Right click on the legend and unselect Legend:Indicate Active Dataset context menu


OR

  • Right click on the legend and choose Update Legend... context menu. In the dialog, uncheck Indicate Active Dataset checkbox.

In Origin 2017 and older

  1. Select Format: Page menu. Go to the Legends/Titles tab and uncheck Indicate Active Dataset at the bottom.
  2. Optionally save the template for future use, with the File: Save Template As menu.
