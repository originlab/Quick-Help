1.239 FAQ-1017 How to recover the deleted labels for a pie chart?

recover-deleted-labels-for-pie

Last Update: 9/2/2019

Sometimes, you might have deleted some labels for pie chart as your own needs. If you want to get these deleted labels back, you can double-click on graph to open the Plot Details dialog, select the plot level at the left panel and then go to the Labels dialog, check the Hide Label if Percentage Less than(%) check box to recover the labels.

Minimum Origin Version Required: Origin 2020

Keywords:label, pie chart, delete labels, show labels, recover, restore
