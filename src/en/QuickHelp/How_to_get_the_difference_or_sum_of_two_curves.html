2.94 FAQ-961 How to get the difference or sum of two curves?

Math-of-two-curves

Last Update: 9/12/2018

To get the difference or sum of two curves,

  1. Activate the graph with the two curves, select Analysis: Mathematics: Simple Curve Math... to open the Simple Curve Math dialog.
  2. Select one plot as Input1 (the minuend/summand), set Operator as Subtract/Add, set Operand as Reference Data, and select another plot as Reference Data (the subtrahend/addend).
  3. Select <new>: New XY as Output to store the result. Click OK button to get the result.

Keywords:difference, sum, subtract a plot, add a plot, math on curves
