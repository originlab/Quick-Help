3.131 FAQ-769 Can I pre-process peak data, such as exclude unwanted data, before each analysis process?

Batch-PA-Script-before-each-Process

Last Update: 10/13/2016

Yes, you can run specified scripts before each process to pre-process data before each peak analysis. Refer to this tutorial for the detailed instruction.

Note that Batch Peak Analysis Using Theme dialog also provides options to run script after each process and at the end of all processes. Besides, almost all peak analysis settings, including Snap to Spectrum; Relative Baseline; Sequentially Initializing Parameter Values, can be saved in theme to help generate results in batch processing.

Minimum Origin Version Required: Origin 2016 SR0

Keywords:Batch process, peak analysis, pre-process, script before each process
