6.36 FAQ-40 Can I allow previous versions to run on my upgrade license to allow extra time to upgrade the client installations?

PreVersion-On-NewLic-FLEX

Last Update: 7/10/2018

Yes. For the Concurrent Network package, the upgraded Origin license will allow previous versions to run on the same license. For example, the new v2017 license will allow v2016, v2015, v9.1, v8.5... to run until all the client installations are updated to Origin2017. (For 8.0, users must run SR2 or higher. For 7.5, users must run SR7.). More Information.
