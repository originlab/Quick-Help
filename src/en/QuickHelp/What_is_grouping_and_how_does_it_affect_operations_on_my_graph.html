1.80 FAQ-210 What is grouping and how does it affect operations on my graph?

Grouping-Mechanism

Last Update: 7/30/2015

When you select more than one column of data for plotting they will be grouped. When dataplots are grouped various display properties such as line color, line type, symbol shape, border and fill colors or patterns, and other display properties for different plot types will automatically increment.


The Group tab of the Plot Details dialog allows you to edit these incremental lists. You can also edit the display properties of grouped data plots independently by setting the Edit Mode to Independent.
